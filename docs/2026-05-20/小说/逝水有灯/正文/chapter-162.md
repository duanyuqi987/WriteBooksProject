# 第162章 寒风里的台阶

所属部卷：第四部：失去的美丽
时间：大二冬天
地点：女生公寓外台阶

## 诗歌插页 P162：原创仿唐绝句

台阶风冷照衣单，
文件怀中夜色寒。
一句朋友难安放，
灯前两影各为难。

苏晚枝抱着文件袋，手指冻得有点红。她看出陆允川有话要说，便停在台阶下。

“你说吧。”她说。

陆允川反而卡住。风从公寓楼侧面吹过来，把地上的枯叶卷到台阶边。

“我们最近是不是好一点了？”他终于问。

苏晚枝没有立刻回答。这个问题太轻，也太重。轻的是语气，重的是后面藏着的期待。

“我觉得现在这样还可以。”她说。

“还可以是什么意思？”

苏晚枝闭了闭眼，像把一句不想说的话压下去，“就是普通同学，普通社团朋友。”

陆允川点头，却没有真正接住。他又问：“那以后呢？”

苏晚枝看向公寓门口。门里有人刷卡进去，暖黄的灯漏出来一点，又很快被门合上。

“陆允川，我不想每次关系刚平一点，你就要问以后。”她的声音很低，“我会累。”

他脸上热了一下，冬夜里那点热显得很狼狈。

“对不起。”他说。

“你不用每次都道歉。”苏晚枝抱紧文件袋，“你只要别一直问。”

说完她上了台阶，刷卡进门。陆允川站在外面，第一次觉得“对不起”也可能是一种让别人继续回应的方式。
