# 第095章 她没有接受

时间：2014年6月中旬

地点：楚州师院

卷次：卷五：各自南北

两周后。支教出发前两天。许若琳发来一条QQ消息。

"告诉你一个消息。"

顾临风正在图书馆看《中国古代文学》——讲到《史记》的"刺客列传"。他把书推开了一点。打了两个字。

"什么。"

"那个人没追到。"

他打字的手停在键盘上面。没有回。

许若琳继续发。"苏晚晴没有接受。她就说不想谈。"

他看着屏幕。对话框上方的"正在输入——"闪了几秒钟。然后又停了。然后许若琳又发了一条"你别想多。我就是随便跟你说一声。"

他打了两个字。"知道了。"

关掉对话框。把手机放回桌面。

下午的阳光从图书馆的窗户斜射进来——照在《中国古代文学》的页码上。页边有他做的笔记——"豫让——士为知己者死"。他把书翻到下一页。然后上一页——发现上一页的内容好像没看进去。又翻回来了。

她在屏幕那头的文字——"她就说不想谈"。五个字。他读了三遍。

她就说不想谈。

为什么不想谈。跟他有关系吗。因为那个约定吗。还是只是不喜欢那个人。还是她也在等——等什么。等时间。等毕业。等他。

他不能问。也不能猜。

他把手机放口袋。低头看了一会儿书。同一页看了大概十分钟——一个字都没看进去。然后抬头看了一会儿窗外。图书馆的窗外是槐树的枝干。四月的槐树长满叶子了——绿油油的，密得看不到对面的楼。他把窗户又推开了几寸。风进来了。风里有香樟树的花香——淡淡的，跟操场那边的味道一样。

他没有说话。但那天晚上他去了操场。

操场里人比平时少。下午刚下了一场阵雨，跑道上有几处积水。积水是透明的，反射着路灯的光。他绕着积水走。一圈。两圈。三圈。四圈。五圈。六圈。

比平时多了三圈。

走第七圈的时候他停下来了。站在操场最外道上。抬头看天空。云层很厚。没有星星。月亮被遮住了——只透出一圈模糊的毛边。他把手揣进兜里——兜里有三颗在操场上捡的小石子。

他把石子丢进跑道排水沟的缝隙里。三颗依次落进去。咚咚咚。然后沿着操场走了出去。走到操场门口的时候灯熄了——啪地一声，整个跑道暗掉。他没有回头。

回到宿舍。李博在看新闻。阿坤在打副本。小伍在写信——这次是给初中同学写的，信纸折了三折，上面写了"展信佳"三个字。

"今天又走圈了？"李博抬头看了一眼他鞋底的红色颗粒。

"嗯。"

"今天有什么想说的吗。"李博的视线从眼镜片后面看过来。

"没有。"

"那就行。"

顾临风去洗脸。经过水池的时候碰到了隔壁宿舍的一个人——光着膀子在刷牙。牙膏沫滴在了地上。那人嘴里咕噜咕噜漱了一口，然后把水吐了——"你天天晚上出去。谈恋爱呢。"

"走圈。"

"有区别吗。"

顾临风拧开水龙头。冷水泼到脸上的时候——脑子里闪过一个念头——她没有接受那个人。她不想谈。不是因为那个人不够好。也许是因为——但这不属于他该去想的范围。他不能想。他把水龙头关掉。擦了脸。走回了512。

手机亮了一下。是许若琳——"出发那天记得带晕车药，山路很颠。"

"好。谢谢。"

"不用谢。"

李博的书页又翻过一页。阿坤的键盘又响了。小伍又吸了一下鼻子。512的生活——照常。

他躺回床上。枕头下面的手机是烫的。他把手伸进枕头下面——碰了碰手机壳。热的。许若琳的短信还在屏幕上——"出发那天记得带晕车药，山路很颠。"他打了两个字——"晚安"——但删掉没发。他发的是"好。谢谢。"

然后按灭了手机。黑暗里天花板那条裂缝还在原来的位置。他数了数裂缝的分叉——一共三条。跟一年前一样。三条裂缝。十二公里。十一站。一个人。没什么变化。也没什么不好。
