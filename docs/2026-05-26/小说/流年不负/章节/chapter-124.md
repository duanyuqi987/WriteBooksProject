# 第124章 是她

时间：2018年3月

地点：楚州，顾临风公寓

卷次：卷七：流年未断

百分之百是她。

苏晚晴。青阳的。楚州师大。在小学教书。每一条都对上。

他把这四条信息写在了备课笔记本上。纸是浅黄色的，横线很密。四行字，每个字的间距都一样。笔是红色的——批作文的红水笔。他用红笔写了一个他想了六年没敢确认的事实。红笔写出来的字看起来比黑笔更重。更像是真的。

青阳——对。那个有月湖的小县城。银杏树。水塔。第五道跑道。元旦的礼堂。

楚州师大——对。他在这座城市的另一端。她在另一端。中文楼二楼和三楼的距离。两年。他不知道她在头顶。

小学教师——对。她站在讲台上。面对一群八九岁的孩子。讲《荷花》。讲乘法口诀。讲拼音。她的声音还是那个在走廊上说"你也来透气"的声音。轻的。稳的。尾音微微往上提。

名字——苏晚晴——对。三个字。苏——晚——晴。他用这个顺序在脑子里念了很多遍。有时候拆开念。苏。晚。晴。每个字都有具体的笔画。三横一竖。日字旁加一个免。日字旁加一个青。

他把笔放下了。

世上不会有第二个苏晚晴。这个名字太特别了。高中的时候全年级只有她一个人叫这个名字。每次月考张榜，他总是在成绩单上先找这个名字——通常会出现在前二十名的位置。找到了，他会用手指在那个名字上点一下。假装在找自己的名字。然后停一秒。然后才找自己的名字。

他那时候大概在五十名左右。离她隔了三十个人的名字。三十个人。六十多分。一个操场的距离。

现在隔了一张茶餐厅的桌子。周六下午。一张木桌。七十厘米宽。一壶绿茶。一壶茉莉花茶。两个白瓷茶杯。面对面。

他对着这四行字坐了大概十分钟。窗外的路灯亮了。一只飞蛾撞在纱窗上，发出轻微的沙沙声。飞蛾的翅膀在纱窗上扇了一下——又撞了一下。然后飞走了。

然后他在四行字下面又加了一行——

她知不知道是我。

问号写得很重。笔尖在纸上压出了一个小坑。

他不知道答案。

她听到顾临风这个名字的时候停了一下。她说"好"。

这可以有太多种解释。

第一种——她记得他。高中同学。后排那个不怎么说话的男生。有点印象。但仅此而已。所以答应——反正也是回青阳相亲，见个同学也没什么。

第二种——她记得他。但不只是记得。高考后他给她打过电话。说了什么他记不清了。只记得月湖边的晚风和她站在对岸的身影。她可能觉得——都过去六年了。没什么了。所以答应了。

第三种——她根本没往心里去。青阳一中的顾临风。是那个顾临风吗。不一定。重名的人多了。她已经想不起他的脸了。所以随口说了"好"。

他没法确定是哪一种。

这让他失眠了。

凌晨一点。他躺在床上。天花板上的裂缝像一个问号。他已经盯着这条裂缝看过几百次了——大一开学、考研成绩出来、代课第一天——每次失眠都在看它。

凌晨两点。他起来倒了杯水。水是凉的。宿舍里只有冰箱的嗡嗡声。

凌晨两点半。他把窗帘拉开了一点。外面是很深的黑。对面楼的灯全灭了。路灯光是橘黄色的，在地上画了一个圆。他的窗台上有灰。他用手指抹了一下——一道印子。

凌晨三点。他坐起来。靠在床头。被子堆在腿边。他拿起手机看了看——没有消息。放下。又拿起来。打开天气预报——明天多云转晴，气温九到十七度。关掉。放下。

她在决定答应相亲的那个瞬间，脑子里闪过的是什么。

是一张模糊的、六年前的、坐在后排的男生的脸。

还是什么都没有闪过。

他不知道。

他只知道周五晚上一定要洗头。村衫最下面的扣子要找刘老师借针线缝一下。皮鞋要擦一擦——鞋柜最底下有鞋油，已经干掉了，不知道还能不能用。

凌晨三点半。外面有鸟叫——大概是夜莺，声音很细，像在试探什么。

他躺下来。把被子拉到胸口。

闭上眼睛。

六年前他删掉她号码的那天晚上，也是这样的夜晚。手机屏幕亮着。光标停在删除键上。他按下去，然后翻了个身，面朝墙，墙上的涂料起皮了，他用手指一片一片地抠——抠到指甲缝里全是白灰。

那个时候他觉得——删了就删了。反正也不会再联系了。

现在——她要坐在他对面了。

隔着茶餐厅的一张桌子。她要抬起头看他。他也要抬起头看她。

八年。从高一教室到茶餐厅。从十六岁到二十四岁。从不敢对视到必须对视。

他翻了个身，面朝墙。

墙上的涂料还在起皮。

他没有去抠。

天快亮的时候他才睡着。

他在四行字下面又加了一行——她知不知道是我。问号写得很重。笔尖在纸上压出了一个小坑。然后他把笔放下了。窗外的飞蛾已经不撞纱窗了——大概飞走了。纱窗上留了一小块灰——飞蛾翅膀上的鳞粉。

他站起来。在屋里走了两圈。从书桌走到床边。从床边走到厨房。厨房的水龙头在滴水。他拧了一下——不滴了。然后走回书桌。坐下。看着那五行字。忽然用笔在"苏晚晴"三个字旁边画了一个圈。然后把笔记本合上。放进抽屉里。
