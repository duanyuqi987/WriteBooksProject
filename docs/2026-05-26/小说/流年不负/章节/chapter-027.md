# 第027章 图书馆

时间：2011年6月初

地点：校图书馆

卷次：卷二：窗边人影

六月初。周六下午。

青阳一中的图书馆不大。在教学楼后面一排平房里。三间教室那么大。窗户朝南。下午有阳光照进来。

顾临风周六没有回家。他爸说这周在工地上有事，让他自己待在学校。

他把作业写了一半。然后去了图书馆。

进门的时候管理员阿姨看了一眼他的学生证。"高一三班。"她念了一声。放他进去了。

图书馆里人不多。周六住校的学生不到二十个。

他往里走。走到第三排书架的位置。

然后停住了。

苏晚晴在。

她坐在靠窗的位置。面前摊着一本物理参考书。书很厚。封面是蓝底白字——《高中物理完全解读》。

她低着头。右手拿笔。左手按在书页上。

阳光从窗外照进来。照在她的右半张脸上。睫毛在眼睑下投了影子。

她没有看到他。

顾临风往后退了两步。退到书架后面。从书架上抽了一本书。不知道是什么书。拿在手里。

然后他绕到第三排桌椅的另一端。最左边靠墙的位置。离她大概三张桌子的距离。

他把书放在桌上。翻开。

书是《中国古代史》。他翻到第三章——"秦汉时期的政治制度"。

他一个字都没看进去。

从书架间的缝隙可以看到她。她还在做物理题。右手在草稿纸上画东西。画了一会儿停住。抬头想了想。又低头画。

她把头发拢到耳后。发绳是淡绿色的。

这个动作她做了三四次。每次都一样——右手放下笔，左手把头发拢到耳后，然后右手拿起笔继续写。

管理员阿姨在门边打着瞌睡。阳光一点点往西移。图书馆里很安静。只有翻书的声音和偶尔的咳嗽声。

她在做物理题。做了大概四十分钟。

然后她站起来。去书架上找了一本别的参考书。翻了两页。又放回去。又找了一本。翻了几页。夹在胳膊底下回了座位。

她经过的时候往顾临风这个方向看了一眼。

他低下头。盯着《中国古代史》第七十三页。眼睛不眨地看。

她走过去了。

他听到她的椅子刮地板的声音。然后是翻书的声音。

他把头抬起来。

他的书还在"秦汉时期"。

下午四点半。她开始收拾东西。把参考书放回书架。把笔放进笔袋。把草稿纸对折夹进书里。

站起来。背上书包。小鱼挂件晃了两下。

走出图书馆。

门关上了。

顾临风放下《中国古代史》。走到她刚才坐的座位前面。

桌上有一小块橡皮擦的碎屑。

他看了两秒。

然后走回去。把《中国古代史》放回书架。

出门的时候管理员阿姨叫住他。

"那个同学——你借的书呢。"

"没借。"

"那你来干嘛的。"

"看看书。"

阿姨看了他一眼。没再问。

出了图书馆。外面的阳光还是很亮。水泥地上有热气在往上蒸。

顾临风走回宿舍。

晚上他在日记本上写了两行字。

"她在图书馆做物理题。做了将近两个小时。很认真。她的发绳是淡绿色的。"

"秦汉时期的政治制度——一个字都没记住。"

合上本子。

窗外有人在打篮球。球弹在地上的声音一下一下的。越来越远。
