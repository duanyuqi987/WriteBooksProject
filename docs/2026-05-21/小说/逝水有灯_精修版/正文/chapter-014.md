# 第014章 理科名单旁观

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：新分班后的理科教室

## 诗歌插页 P014：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
公告栏边被胶带粘住的名单角还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，分班这件事本来没有那么多可以讲的心跳。教务处把名单贴出来，学生看完就走，胶带粘不住就再贴一条。公告栏前面最多站五分钟，然后该搬课本的搬课本，该找座位的找座位。这个程序对所有人来说都是一样的。

对程砚来说，分班最大的好处是他不用再面对坐在前排的叶澄催他交作业了——虽然叶澄去了文科班，但程砚很快发现新班主任陈老师催作业的功力比叶澄还要高两个等级。"这个人连你少写一个步骤都能看出来，"程砚在课间跟陆允川抱怨，"而且他是直接走到你座位边俯下身看你的本子，根本不给你装傻的机会。"

但程砚分班后真正关心的不是陈老师。他在开学第二周就发现陆允川的状态不太对。不是成绩下滑的那种不对——陆允川的物理考得比上学期还好了——而是他的注意力结构变了。以前在老三班，陆允川的注意力是分散的，会在叶澄催作业的时候抬头，会在后排有人讲笑话时跟着笑。现在他的注意力被压缩成了一个扇形，扇顶在靠窗的角落，扇形范围覆盖前中排靠走廊那片区域，其他角度全被屏蔽了。

"你有没有发现你看黑板的时间比以前少了？"程砚在课间问他。
"黑板有什么好看的。"
"那你总在看那个方向。那边又没黑板。"
"我在看窗外。"
"你在看窗外看了一整节数学课，陈老师那节课讲了四个图像变换，你笔记里只记了两个——你觉得他会放过你？"程砚把他手里捏的那支笔抢过来，放在桌上，"你认真听一节行不行？反正她在教室里又不会跑。"

程砚不是没有分寸。他知道陆允川在暗恋谁，他只是在把握一种尺度：既不让陆允川觉得他在戳穿，又不让陆允川以为自己藏得很好。因为对程砚来说，帮朋友藏一个显而易见的秘密需要的演技，比帮朋友分析一道三角函数题难得多。

叶澄的旁观位置更远，但她的视角更客观。她去交表格时经过理科一班教室，从窗户往里看了一眼。陆允川正低着头抄板书，样子很认真。但叶澄注意到他的笔盒——他以前在老三班，笔盒永远放在课桌右上角，现在它被挪到了左上角。因为右上角的视野会被挡，左上角能看到的前排范围更大。

"你换手写字了？"她课间在走廊里碰见陆允川时问了一句。
"没有。"
"那你笔盒怎么换到左边了？"
陆允川低头看看手里拿着的笔盒，过了两秒才说："右边抽屉塞满了。"
"你右边抽屉里放的是你的笔记本、剪刀、透明胶带，"叶澄盯着他的课桌方向看了一瞬，"去年你就塞得更满。塞不下笔盒？"

她没有等答案。她只是把他那个小动作登记下来，像在登记表上多写了一行字。叶澄最厉害的地方在于她从来不说"我知道你在想什么"，她只说"我知道你在做什么"。

何敏和蒋雯，夏知遥的两个朋友，对这件事一无所知——或者说，她们知道的程度刚好在"隐约觉得后排有个男生总是往这边看"的水平线上。她们不可能不知道，因为陆允川偷看的技术对于旁观者来说实在太明显了。

"你有没有觉得那个靠窗的男生老是看这边？"何敏在课间问蒋雯，声音压得很低。
"哪个？"
"就是后排那个，书皮包得特别好那个。"
"哦，陆允川？"蒋雯顺着何敏的视线回头看了一眼——正好对上陆允川的目光，对方像被烫了一下似的低头翻书。"他应该是在看窗外吧，你这个自恋狂。""我哪里自恋了——等一下他刚才是不是低头了？""你不要吓人家。"两人笑成一团，何敏说"他是不是喜欢你"，蒋雯说"你脑子里除了喜欢没别的事了吗"。这个话题就这样不了了之——因为对她们来说这只是课间闲聊的零碎材料，不值得认真对待。

夏知遥没有参与这场讨论。她在改错题，钢笔在纸上划了一道线，把选错的化学答案涂掉，在旁边重新写了一个B。

很多年以后叶澄在微信上问程砚："你当年为什么老敲老陆？你不敲他会怎样？"

程砚回了很长一段话："因为他那种性格，不敲一敲他就会一直待在壳里。喜欢一个人不敢说，考好了不敢高兴，难过了不敢让人看出来。我不是要推他去表白，我是想让他至少能承认自己在乎——在乎一个人不丢人。你坐在壳里头闷着想她一辈子，你累不累？"

叶澄回了一句："那你为什么不让他在壳里？他也许舒服。"
程砚说："他在壳里不舒服。他只是怕出来了更不舒服。"
叶澄沉默了一会儿，说："你比我想的要细。"
程砚回了一个咧嘴笑的表情。
