# 第159章 短暂和好旁观

所属部卷：第三部：寒风里的花
时间：七月中旬
地点：诗社活动室

## 诗歌插页 P159：原创现代诗

她递出一块饼干，
不是递出从前。
她愿意轻松地笑一下，
也愿意在笑声后，
把箱子封好，
把距离放回原位。

苏晚枝也感觉到那天气氛轻了一点。陆允川没有抢着帮她拿所有东西，没有把每句话都接成私人话题，也没有在她递饼干后露出那种过分郑重的神情。

这让她松了一口气。

她不是想把关系弄僵。一个社团里的同学，如果每次见面都绕路，太累，也太难看。她希望他们能重新变得普通：一起做事，偶尔开玩笑，事情结束各自回去。

可她也清楚，普通需要两个人一起维护。她递饼干时有一点犹豫，最后还是递了。不是因为心软到要撤回边界，而是因为边界不必让人变得僵硬。

陆允川接得很平常。她看见了，也记住了。

封箱时，胶带声音很响。苏晚枝按住箱角，陆允川拉胶带，两个人配合得不错。她说“这样挺好”，说的是箱子，也有一点点说的是这种状态。

只是这“一点点”不能被放大。她在心里提醒自己，也希望陆允川能听懂。

当天晚上，她回宿舍后洗手，指腹还有纸页划过的细痕。那些细痕比情绪可靠。它们证明那天最主要的事，是整理旧书。
