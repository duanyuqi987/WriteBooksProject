# 第123章 教室的灯之后

所属部卷：第三部：寒风里的花
时间：同一天夜里
地点：二号教学楼空教室

## 诗歌插页 P123：原创仿宋小令

寒灯边，晚声轻，
半窗灯影落无凭。
问旧年、几回执念，几回分明。
人去不须追远路，
水流仍向晓光生。

墙上的诗拼贴树在周五晚上完成了。从地板到天花板，树冠散成七根枝杈，挂了三十多张白纸片。每张纸片上一首短诗，字的大小不一——苏晚枝故意没有统一字号，说"树叶本来就没有两片一样的"。树干的基底是一句长诗，用毛笔写在一条窄长的牛皮纸上，从天花板悬下来，字的颜色是墨绿："诗不是装饰，也不是替人显得文雅。它更像一个缓冲，让人在说重话之前先停一下。"

这是陆允川写的。不是抄苏晚枝的——是他自己改过的版本。他把"我"改成了"人"，把"像一盏灯"改成了"像缓冲"。苏晚枝看完后说："比原句好。你比我更擅长抽象转具体。"

"这句话本来就是你教我的。"

"什么时候？"

"第一次诗社活动。你在读诗会上说：'写诗之前应该先看见东西。看见一朵花、一个人、一盏灯。而不是看见别人诗里的花、人、灯，再抄一遍。'"

"你还记得？"

"全部。"

周六下午，诗社新一期社刊的发布会在301教室举行。来的人不多——苏晚枝、叶澄、陆允川、祁野、沈让，还有一个新加入的大一女生叫小孟，学设计的，戴一顶贝雷帽。叶澄带了一袋饼干——不是上次被祁野说"像在嚼纸"的那种，是新款，奶油味的。

"这次饼干什么来头？"祁野拿起一片。

"丹麦进口。你再敢说像纸——"

"不像纸。像嚼砂纸。"

叶澄伸手去抢饼干袋，祁野举高了不让她够到。小孟在旁边小声问沈让"他们是不是一对"，沈让说"不是，但你最好别参与进去"。

苏晚枝站在树墙前面，拿着一张稿纸做讲解。她没有念稿子——稿子只是用来挡住手抖的。她解释了七个树枝分别代表什么，然后指着树根部分说："树根这里不是诗。是一句话：'这面墙献给每一个在夜里找空教室的人。'"

叶澄举手："你是说——这栋楼九点半以后不关门？"

"不关门。301的门锁是坏的。欢迎来。"

"那清洁阿姨发现怎么办？"

"清洁阿姨三个月前就发现了。她没上报。"苏晚枝笑了一下，"她还帮我们擦了桌子。昨天我进来时桌面上的灰没了，讲台上多了一卷新的垃圾袋。"

散会后大家陆续离开。祁野和叶澄在走廊里开始了针对饼干包装袋的持续性拌嘴，声音渐渐远去。小孟和沈让走在最后，沈让在给小孟讲"301教室的灯管为什么一直闪——因为线路接头松了"。

苏晚枝一个人站在树墙前面。陆允川本来已经走到门口，回头看见她的背影，停住了。

"你不走？"他问。

"再待一会儿。"

"要不要开灯——"

"一盏。就够了。"

陆允川走回来，站在她旁边。她脸上的表情不是成就——他见过叶澄做完实验时的成就脸，也见过祁野投进绝杀球时的成就脸。苏晚枝不是那样。她是在确认——确认这面墙上每一个纸角的位置、每一条树枝的弯曲角度、每条字缝之间的呼吸节奏。她在确认这个空间终于有了体温。

"你这个周末回宿舍会想什么？"苏晚枝问。

"想树干上那句话。'诗不是装饰'。"

"别想那句话。那句话是你的。想点别的。"她指了指树冠最上面那根树枝。那根枝条上只挂了一张纸片。很小的字，剪成叶子的形状。

陆允川踮脚看。纸片上写的是一行他没见过的小字：

"等我收集够所有树叶的时候——苏晚枝"

"这是什么？"他问。

"是诗拼贴树上面最后一片叶子。但不是结尾——是在等的。等诗社以后每一期社刊出一首好诗，就从这里加一片叶子。"她指了指树枝上的空白空间。"这里、这里、这里——都是留给未来的。"

"你打算让这棵树上长多少片叶子？"

"直到我都收集够。或者——"她停了一下。"——直到我毕业的时候，把这面墙留给下任社长。跟他说：树已经种好了，叶子你来加。"

陆允川看着那个空白空间。想象它被填满的样子。

"我觉得你可以先摘一片上去。"他说，"你不一定非要等别人。你自己的诗也可以挂在上面。"

"我自己的已经放上去了。树根那句话就是我写的。'献给每一个在夜里找空教室的人'——这句话写的是我自己。"

教室里安静了一会儿。日光灯管在头顶嗡鸣着，闪了一下。苏晚枝伸手去关灯。

"走吧。"她说。

两个人走到走廊。苏晚枝转身锁门——她从布袋里拿出了一根新的链条锁，锁在门把手上，钥匙拧了两圈。

"你终于给秘密基地上了锁。"

"对。树墙不能被人蹭坏。"她把钥匙放进布袋的内层口袋，那个口袋是专门缝的，边角缝得不平整，看得出是手工活。

下楼时陆允川的手机震了。是祁野发来的："叶澄的新饼干其实挺好吃。只是我不方便当她面承认。你们那边搞好没有？食堂夜宵开始了。"

"搞好了。马上来。"陆允川回了消息。然后对苏晚枝说："祁野说饼干其实好吃。"

"他肯定不会当着叶澄的面说。"

"对。他说不方便当面承认。"

"你这两个朋友——"苏晚枝摇了摇头。她没有说完，但脸上的表情是软的。

出了教学楼，路灯下能看到食堂窗口的灯还亮着。夜风吹过二号教学楼前面的梧桐树，叶子沙沙响了几声。陆允川回头看了一眼301教室的窗户——黑色的，但能想象那面墙上有一棵树在暗中慢慢等着叶子。
