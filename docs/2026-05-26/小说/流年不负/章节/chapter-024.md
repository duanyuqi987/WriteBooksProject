# 第024章 篮球赛

时间：2011年5月初

地点：学校篮球场

卷次：卷二：窗边人影

五月初。学校组织班级篮球赛。通知贴在公告栏上——白纸黑字——"高一年级班际篮球赛——抽签对阵：理科三班 VS 理科六班——周一下午第三节课后——南篮球场——"

高一理科三班对高一理科六班。下午第三节课后——下午四点半——操场南边的水泥篮球场。场地不好——水泥地——有几道裂缝——缝里填了石子——球弹上去会变向——上一届的学长说"不怪地——你打不准别怨场地。"篮板是木头的——漆掉得斑驳——篮圈有点歪——靠左——向右投更容易进。

顾临风打前锋。他个子不算高——一米七五——但跑得快——第一步蹬出去的时候力量在脚尖——加速快——抢球也狠——别人抢篮板站着等——他是跳起来摘。今天穿了白色旧回力球鞋——鞋底磨平了一块——鞋带重新系过——拽紧了——鞋面上的"回力"两个字磨掉了一半——只剩"回"。

赵一鸣打后卫——吨位大——背身卡位稳——嗓门大——场上喊起来盖半场。"传球——这边——对——对——"他在场上指挥——手举着——方向——前——后——"突破——"嗓门快哑了——说完喝水——过一会儿又哑——再喝。他投球不准——但传球可以——半场长传——落点往往刚好——偶尔传太过——飞出底线——自己骂一声。

篮球场边围了不少人。自己班的女生来得多一些——许若琳在——拿了一瓶水——站在篮架旁边。别的班有路过停下来的——不认识——看两分钟走人。体育老师今天当裁判——穿着那个红黑条的运动服——脖子上挂着哨子——不是平日上课那个哨子——是金属哨——银的——吹起来更尖——"吡——吡——"

顾临风在场上跑着。额头有汗——顺着鬓角往下——到下巴——将滴——手臂一抹——校服袖子卷到手肘上面——胳膊上青筋可见。防守——对六班那个高高的中锋——比他高半头——背身——挤他——他压膝盖——重心往下——侧身防——不犯规——手举着——球传过去——他没抢到——对方投了——没进——他转身抢篮板——拿住。

上半场打完。三班落后六分——22比28。从场上下来的时候腿有点沉——手腕微疼——刚才和对方抢球打了一下手背——红了——过一会儿会青。

中场休息。阳光斜了——操场东边旗杆的影子拉得老长——已经过了篮球架——落到跑道外——煤渣上的影子边缘模糊。赵一鸣蹲在篮球架下——拿瓶农夫山泉往脸上倒——水顺着下巴滴到地上——"六班那个中锋——太高了——你下半场往前顶——上半场防得好——但他转身慢——"说着他拿手指在地上比画位置——"你打他右手边——他左手断球差——右脚也慢——往那边突破——"顾临风擦了一把汗——"知道了——"然后拿起地上的水瓶——拧开——往嘴里倒——喉咙里泛起水的凉意。

然后他往场边看了一眼。站在人群的第二排中间——她站在那里。

浅绿色T恤——不是校服——棉布——袖子到胳膊一半——校服外套系在腰间——袖子在腰前打了个简单的结——不紧——绳子尾巴垂着。她站在许若琳旁边——应该是许若琳拉她来的——许若琳喜欢看篮球——班赛每次都到——手里那瓶水也是准备给赵一鸣的——赵一鸣自己不知道——打完才发现——说"谢谢"——脸红了一下。

苏晚晴没在看球——不是那种盯着球追的人——目光在场上的队员之间扫——扫的幅度不大——从左到右——然后停了一停——又扫回来。扫到顾临风这边的时候——目光停了一下。大概——不到一秒——和上次不同的——这次她眼里有焦点——不是空扫。然后移开了——去看许若琳——许若琳在跟旁边一个女生说话——"最后的配合——挺精彩的——"

顾临风转过头。把水瓶拧紧——盖子拧了一圈又一圈——多拧了几下——然后放在地上——站起来——走了两步——弯下腰——手撑膝盖——看着场上那边——那边的对方队员——几个在补充水分——趁机抓节奏。

下半场开始。顾临风像变了个人。不是技术变了——是脚底下——好像地收紧了——球在手里更轻——跳的时候膝盖往上——第一步过对方——转身——接到赵一鸣的界外球——运——切——在罚球线附近顶着那个比他高的中锋投——球擦了下指尖——进了。跑得比上半场还快——从后场反击——接球——带球——起跳——抢篮板的时候跳得比平时高了至少五厘米。赵一鸣传球过来——篮下没人——他接球——上篮——进——很干脆——没碰框。场边自己班的人喊了——不知道是谁——"三班——加油——"嗓子粗——喊出来之后自己咳嗽——旁边有人笑了。他落地的时候差点往她那边看。忍住了——头扭到一半——硬转回去——跑回后场防守。

比分追上来——一分一分地追——三四分——擦平——反超——又被反超——追。水泥地上的摩擦——球鞋刮地——"吱嘎——"赵一鸣在下面喊——"最后一个——防好——"声音劈了——哑。最后三分钟——33比34——差一分——六班领先。

赵一鸣带球过半场——被对方两个人包夹——人矮了半头——球压低——差点被掏——他把球从人缝里传出来——双掌一推——力度够了——方向也正。

顾临风接球。三分线外——脚尖对着弧线外大概半步——面前是空的——防守的人被赵一鸣拉走了——剩下那个中锋离他两步——来不及扑。场上——这一刻很慢——球在他手里——手上刚才被碰的地方还痛着——但球的重力感——正好——呼吸——出手。跳起来——脚离地——腿伸直——右手推送——手腕——食指——方向。球在空中——转了——自旋——在下午五点的光里——皮面上的小颗粒——看不到——但会有。弧线——抛物线——高——从三分线——"唰——"进了。空心。网底都没怎么动——只是穿过——干净的——轻飘飘地——然后网绳晃了一下。场边炸了——有人喊——喊得语无伦次——许若琳在拍手——不知谁的手拍到赵一鸣的头——"好球——"赵一鸣冲过来——手肘撞了一下顾临风的肩——大喊——"好球——好球——"嗓门哑了——破——但很响。

顾临风落地。右脚踝稍震了一下——不怕。嘴角动了一下——往场边瞥了一眼——苏晚晴。她在拍手。没有跟别人一起喊——只是拍手——手腕平——掌心轻碰——几下——然后停了。那个拍手的角度——不是朝向某个特定方向——是——"比赛——好球"——的方向。但她确实在拍。

比赛继续——六班发底线球——对方全场紧逼——他们有点急——中线传球——球被赵一鸣拍出了边线——重新发。最后十几秒——六班投篮——后仰——高了——碰到篮脖子——弹出——赵一鸣抢到了最后一个篮板——双手抱紧——运了两步——长传后场——球往天上一抛——哨——终场哨响。体育老师的哨子——长——响彻半个操场——"吡——"

36比34——三班赢了。两分之差。

顾临风弯着腰——手撑着膝盖——喘气——汗从额角顺着鼻子边缘一滴滴落到水泥场上——地上的灰被滴上汗——变成小小一个深色的点——噗——然后化开。腿上的乳酸和心跳——突突——在耳鼓。赵一鸣跑过来——脸上全是汗——汗在笑容里——眼睛眯成线——拍他后背——巴掌——砰——"你下半场打鸡血了吧——那个三分太牛了——那个三分——平时你投十进三——今天投五进四——进的全是空心的——怎么打的——你要么是吃了什么——要么是——"他停了——看了顾临风一眼——"要么是什么刺激的——不说了——"

顾临风直起腰——右手擦了把汗——擦完——汗在手指上滑。"运气好——"

他往场边看。许若琳的那瓶水不见了——给了赵一鸣——赵一鸣正咕嘟咕嘟往嘴里灌——半瓶下去——打了个嗝。但苏晚晴——已经不在了。人群散了——篮球架边的台阶上——两个男生还坐着系鞋带——旁边——一个女的在捡空水瓶——是三班那个文体委员。篮球架下的影子被夕阳拉长——地面上一道长黑。苏晚晴刚才站的位置——现在空着——留下一个鞋印——白帆布鞋那种平底形状——在浮灰上——淡得明天早上风一吹就没了。

赵一鸣顺着他的目光看了一眼——拍了拍他的肩——"走了——回去洗澡——晚上还要上自习——"顾临风从地上捡起水瓶——拧盖——喝了一口——水是温的——被午后的太阳晒了很久——不好喝——但解渴。

晚自习——他坐在座位上——数学作业收上去了——物理笔记没写——桌面上空白。左膝盖磨破了皮——下午那会儿滑地救球蹭的——皮破了——裤子上有个小洞——不深——能看到里面——粉红色的新肉。右手手腕——隐隐酸痛——投篮的时候——那一推——用太多力——但球进了——值了。腿很酸——在学校板凳——一坐就不想起来——明天会更酸。

但他觉得今天挺好。苏晚晴在第二排——做物理题——那个加速度求向心力的题——应该是在改今天下午测验错的那道。马尾——安静——后背挺直——小指翘着——手里的笔——蓝色。她下午在场边看了多久？她有没有看到那个三分——那个球——空心进——她拍手的时候——是不是因为看见了那个——还是只是许若琳在拍——她跟着拍。他不知道——这些事——只能猜——但今天他确实投进了几个好球——那个三分——空心——那个篮板——那个上篮——裁判吹了哨——赵一鸣拍着他说好球——她——在场边站着——手里的水——没给他——给了他一个拍手。

翻开物理练习册——画了一个受力分析图——滑轮——绳——木块——斜面。摩擦力方向——沿斜面向下——画好了——对了——继续下一题。下课铃响——赵一鸣拿着两瓶水走过来——"给你一瓶——今天的MVP——"顾临风接过——"不是MVP——""你说了不算——我说的——"他拧开瓶盖——喝了一口——水是凉的了——小卖部冰箱刚拿的——瓶壁上一排排水珠。好喝。
