# 第171章 最重的话之前

所属部卷：第四部：失去的美丽
时间：大二冬末
地点：男生宿舍书桌前

## 诗歌插页 P171：原创现代诗

夜里键盘很轻，
轻到不像能伤人。
他把一句话删了又写，
光标闪着，
像一根针，
等他亲手把线穿错。

东门误会之后，陆允川安静了两天。安静不是消气，而是把话压在心里，越压越硬。

苏晚枝没有找他。诗社群里她照常发通知，语气清楚，没有多余情绪。陆允川看见她和宣传部同学讨论横幅尺寸，胸口又闷起来。

晚上宿舍熄灯前，祁野提醒他：“别在情绪上头的时候发消息。”

“我知道。”

“你每次说知道，我都想把你手机没收。”

陆允川笑了一下，却没有把手机放远。熄灯后，宿舍只剩电脑屏幕的蓝光。他坐在书桌前，打开和苏晚枝的聊天框，打下“我是不是让你很烦”。

删掉。

又打：“你到底有没有认真看过我？”

删掉。

最后，他写：“你是不是觉得，别人喜欢你就活该被你这样晾着？”

这句话出现时，他自己也停了一下。他知道太重。可那一刻，他更想让她疼一下，好证明自己不是唯一疼的人。

他按下发送。
