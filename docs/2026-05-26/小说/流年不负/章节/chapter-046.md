# 第046章 元旦晚会

时间：2011年12月31日

地点：学校礼堂

卷次：卷三：此心藏久

十二月三十一日。高二元旦晚会。

学校礼堂在操场东边。平时是用来开年级大会的——主席台上放一排桌子，校长副校长坐一排，讲话。今天不一样。主席台上挂了红灯笼，一排，十六个。横幅是红底黄字——"青阳一中2012年元旦文艺汇演"。

下午四点。各班按班级入座。高二年级坐中间——最好的位置。高一坐前面靠左，高三坐后面。老师坐最边上。

顾临风跟着队伍进场的时候，已经很难找到好位置了。赵一鸣在前面挤，回头拽了他一把。"这边这边。"他们在第七排靠过道的位置坐下。

椅子是折叠的。铁架子，木座面。坐上去会吱一声。他的椅子叫了一声。

礼堂的灯光变暗了。台上的灯光亮了。两个主持人走上来。男的穿着黑色西装，女的是白裙子，声音在礼堂里回荡。"尊敬的各位领导、各位老师——""亲爱的同学们——"

顾临风在看节目单。一张粉红色的打印纸。节目一个一个列着。舞蹈。独唱。相声。诗朗诵。魔术。中间偏后的位置——"合唱《明天会更好》"。

表演者：高二三班。

她的班级。

他把节目单折了一下。赵一鸣在旁边说"第一个是舞蹈——街舞"。他没听见。

前面几个节目他没怎么看进去。唱歌的走了调，相声的包袱没响，诗朗诵的同学忘了词——在台上卡了五秒钟。台下有人起哄。诗朗诵的同学脸红了。

然后是高二三班的合唱。

主持人报幕。幕布拉开。台上站了三排人。男生穿白衬衫黑西裤，女生穿白色连衣裙。灯光打在每个人脸上。化过妆的脸在灯光下显得有点不真实。

苏晚晴在第二排靠右的位置。

顾临风坐直了。

她的头发今天放下来了——不是马尾。是披肩发。发梢打在肩膀上。她没有戴眼镜——大概是戴了隐形眼镜。脸上有淡妆。腮红很浅。嘴唇上涂了一层淡淡的颜色。

灯光打在她脸上。她的眼睛很亮。不是因为化妆。那一瞬间她往台下扫了一眼。很快。然后收回去。

顾临风手里的节目单被捏皱了。

音乐响起。钢琴的前奏。一个男生领唱——"轻轻敲醒沉睡的心灵，慢慢张开你的眼睛——"然后所有人开始合唱。

"看看忙碌的世界是否依然——孤独地转个不停——"

她的声音在合唱里听不出来。混在三十多个人的声音里。但他知道她在唱。她的嘴唇在动。她的眼睛看着前方。不是看台下。是看礼堂后方——虚的某个点。

"春风不解风情——吹动少年的心——"

她的嘴角动了一下。

顾临风盯着台上。旁边赵一鸣在拍手——跟着节奏。有人在后面跟唱。但顾临风什么都听不见了。他的眼睛里只有第二排靠右那个位置。

灯光变了一种颜色——暖黄的。

她的头发在灯光里是褐色的。发梢有一点卷——大概是扎马尾扎久了，放下来以后有一个自然的弧度。她的耳垂上什么都没有。没有耳钉。很干净。

"让我们的笑容——充满着青春的骄傲——"

她笑了。

不是在笑谁。是在笑——大概是这个歌让她觉得开心。或者舞台的灯光让她觉得很亮。嘴角弯了一点。眼角也跟着弯了。

顾临风低下头。把节目单又折了一道。

深呼吸了一下。然后抬起头继续看。

合唱结束了。幕布拉上。掌声响了很久。有人在吹口哨。

幕布再拉开的时候，台上的人集体鞠躬。第一排蹲下来，第二排半蹲，第三排站着。她的脸在二排右边。她在往台下看。她的目光扫过中间区域的时候，顾临风把脸偏了一下。

不是躲。是怕被她看到自己在看她。

后面几个节目他都没怎么看。脑子里就是刚才那个画面——灯光打在她脸上。她化了淡妆。眼睛很亮。

散场的时候九点多。

赵一鸣在耳边说"班里还有活动"。他没反应过来——"班级活动？""不是。老刘说十点之前回教室。有茶话会。"

从礼堂往教学楼走。路灯下面所有人都在说话。讨论刚才的节目。说街舞队的谁谁谁好帅。说主持人的裙子好好看。说诗朗诵忘词那个太尴尬了。

顾临风走在人群中。没说话。

赵一鸣在旁边讲相声的包袱为什么没响。"节奏不对。他的节奏快了一秒。快一秒整个场的反应就跟不上。你看——"顾临风"嗯"了一声。赵一鸣知道他根本没在听。

到教室楼下的时候，赵一鸣忽然停住。

"你是不是觉得。你配不上她。"

顾临风没有停。继续往上走。走了三步。然后停住。没有回头。

"你再说一遍。"

"我没说什么。"

赵一鸣追上来。两个人并排上楼。楼梯间的灯忽明忽暗。有一个灯泡坏了，每闪一下赵一鸣就骂一句。走到三楼的时候灯彻底不闪了——黑了一大段。什么都看不见。只有楼上漏下来的一点光。

黑暗中。顾临风的声音从前面传来。

"不是觉得配不上。"

停了停。

"是觉得——她还不知道我是谁。"

赵一鸣没有说话。

两个人摸黑上到四楼。教室里的灯亮着。桌椅被推到两边，中间空出了场地。苹果、橘子、瓜子堆在桌子上。有人带了一个小音箱——在放歌。周杰伦的《晴天》。

顾临风在门口站了片刻。

苏晚晴已经换回了校服。马尾重新扎起来了。淡妆还没卸。她坐在窗边的桌子上，手里拿着一个橘子，在剥皮。许若琳在旁边跟她说话。她点了点头。

她剥了橘子。白色的橘络被她一根一根撕掉。撕得很仔细。

然后把橘子掰成两半。一半放到许若琳手里。另一半自己吃。吃了一瓣。然后停了一下——好像在想什么。

赵一鸣在后面推了顾临风一把。

他走进去。找了个角落坐下。从果盘里拿了一个苹果。没吃。放在手里翻来翻去。

音箱里换了一首歌——《十年》。有人起哄"换一首换一首——元旦放这个不吉利"。放歌的人说"我又不会不吉利。我又不分手。"

苏晚晴在另一边笑了一下。

她的笑声不大。别人都听不出来。隔了两排桌椅的距离。但他的耳朵动了。和去年秋天走廊上她笑的那一声很像。

他把苹果放到桌上。苹果滚了一下，撞到橘子又停住了。

零点。有人喊了一声"新年快乐"。所有人都跟着喊。教室里热闹起来。瓜子壳吐了一地。苹果核扔在垃圾桶外面。有男生在打架——是真打还是闹着玩分不清。

顾临风没有喊新年快乐。他只是坐在角落里。看着热闹的人群。

偶尔——大概两次——他的视线晃到窗边。她在跟许若琳碰水杯。她的杯子里是白开水。她喝了一口。

然后她往角落方向看了一眼。

只是扫过。不是看谁。

但他把"2012"写在了桌面上。用手指。写得很轻。没有痕迹。
