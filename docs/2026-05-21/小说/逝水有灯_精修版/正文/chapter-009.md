# 第009章 新课本的气味旁观

所属部卷：第一部：初秋三楼
时间：高一开学周
地点：教室后排靠窗处

## 诗歌插页 P009：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
透明书皮里压住的气泡还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，新课本的气味会显得没有那么沉重。那天的教室里，三十八个人有三十八种和书皮搏斗的方法：有人用订书机封口，订歪了又拔出来重订；有人不用胶带只用技巧把四角折进去，结果书皮鼓得像气球；有人干脆放弃包皮，把新书往抽屉里一塞，趴下睡觉。方老师在讲台上说"新学期新气象"，被底下的一片塑料沙沙声彻底盖过去了。

叶澄对陆允川的第一印象来自讲台上收表格的那个早上。"他交表格的时候把纸角抚平了两次才递给我，"叶澄后来对程砚说，"那种人肯定是怕给别人添麻烦。""'怕给别人添麻烦'，你就直接说他心思重。"程砚说。"心思重不是坏事，但心思重的人容易自己扛。"

包书皮那天，叶澄的观察位置最好——她是班长，可以名正言顺地从前往后走。她发现陆允川的书皮包得比任何人都好，四角没有一处翘起，塑料膜和封面之间没有一颗气泡。但她同时也发现，他的注意力不在书上——书皮包好之后他没有翻开看一眼，而是看向黑板旁边的成绩单。那个目光的方向很明确，路程很短，去了又回来。

"他在看成绩单。"叶澄课后对程砚说。"他不是在看成绩单，"程砚正往嘴里塞饼干，"他是在成绩单上找一个人的名字。""你怎么知道？""因为我坐他旁边，他的眼睛会投球——每次都投到同一个篮筐。"

程砚的观察方式比叶澄粗暴，但更准确。他不是会分析微表情的人，他只会统计——统计陆允川一天看手机的频率（基本为零）和看某个方向的频率（远高于零），然后做一个减法，得出一个差值。这个差值就是他判断的依据。

"我那天故意把胶带粘到手指上叫了一声，你猜怎么着？夏知遥回头笑了一下，然后老陆那个表情——"程砚在宿舍里复述这段时停顿了一下，像是在挑选措辞，"不是说他在看，是说他看得忘了自己也在被别人看。那种忘我程度，做物理卷子的时候都没出现过。"

夏知遥那一笑是被动的。她被声音吸引，回头看到了一个男生正在和透明胶带打架，觉得好笑，就笑了。笑完不到一秒就转回去继续做事。她的笑不是信号，不是暗示，不包含任何需要被破译的信息。她之后也不会记得自己笑过，因为这件事在她的一天里太小了，小到在她记忆里连一秒都占不满。

陆允川不会同意这个判断。他会觉得那一笑是一把钥匙——但钥匙只能开一个锁，而他想开的门太多了。这就是旁观者和当事人之间的区别：旁观者能看到一把锁的范围，当事人会把自己整个青春当作门。

但夏知遥不是没有自己的厚度。那天蒋雯和何敏在课间讨论谁的化学笔记记得好，何敏说"知遥的字最好认，去问她借"，夏知遥就把笔记本翻出来递给她们，什么都没说。笔记本封面贴满了透明胶带补丁，何敏接过去时补丁边角翘起一块，夏知遥用手指按了按，把它重新粘牢。那个动作和她所有的动作一样——不解释、不表演、不等待任何人注意。她只是把一块翘起的胶带按平。

叶澄晚自习后经过陆允川座位时停下来，直截了当地说："我知道你在看谁。没关系。但你要知道人家没有义务配合你没说出来的剧本。"她说完就走了，语气和催交作业时一模一样。路过讲台时顺手把粉笔槽推进去半截，粉笔灰轻轻飘起来，在日光灯下一闪。

程砚晚自习前也和陆允川聊过一次。不是在教室，是在水房——陆允川在洗杯子，程砚在接水。"老陆，我说真的，"程砚靠在门框上，水瓶搁在窗台上，"你喜欢她是你的自由，但你能不能至少——多和我们说几句话？你不说也行，但你至少别一个人消化。你这消化系统装不下那么多。"

陆允川把杯子从水龙头下拿开，甩了甩水："你这是在关心我？"
"废话。我不关心你，你吃红烧肉那块是我夹的，我到现在还饿着呢。"
陆允川笑了一声。程砚看见他笑了，把水瓶从窗台上拿起来："行了，走吧，杯子别甩了，再甩你那袖子就湿透了。"

旁观不是议论是非，是替当事人分担他们无法开口的重量。叶澄替陆允川想了一句话没问出口的问题，程砚替他想了一个没讲出口的笑话。他们都还年轻，都不一定做对了，但他们都在试着不让他一个人。
