# 第187章 名字

时间：2021年5月
地点：楚州妇幼保健院
卷次：卷十：不负少年

第二天下午。阳光从窗户投进来。病房里的消毒水味道淡了一些。

苏晚晴靠在枕头上。女儿睡着了。攥着小拳头。嘴巴微微张着。呼吸很轻——轻到几乎听不到。但她一直在听。

顾临风坐在床边。膝盖上摊着一个本子。上面写了好几个名字。有的划掉了，有的圈起来。

"顾予。"他念了一个。"给予的予。"

她摇了摇头。"太正式了。"

"顾棠。海棠的棠。"

"像男孩子。"

"顾知。知道的知。"

"也像。"

"顾溪。"

"太软了。"

他把本子合上了。看着她。"你想叫什么。"

她转头看着窗外。窗外有一排杨树。树叶在风里翻着白色的背面。

然后她转回头。看着他。

"顾念。"

"哪个念。"

"念念不忘的念。"

他愣住了。

窗外的杨树还在翻叶子。白色的背面、绿色的正面、白色的背面。

"念念不忘——"他重复了一下。"必有回响。"

"嗯。"

她说的声音和平时不一样。轻了一个度。

"这句话——你知道哪来的吧。"

"知道。王家卫的电影。"

"那你还用。"

"我用不是因为电影。"

她把女儿身上的小被子掖了掖。被角翻出来了，她用手指把它塞进去。掖完之后又掖了一遍。

然后抬头。

"因为他说的念念不忘，是香港。是六十年代。是另一个故事。"

"我说的念念不忘——"她顿了顿。

"是高一走廊饮水机前面。你排队站在我后面。你的杯子是红色的。我的杯子是绿色的。你杯子里的水洒地上了。你把杯子背到身后。然后看了我一眼。"

顾临风没有说话。

他记得那天。2010年9月。开学第二周。他去走廊接水。饮水机是那种桶装水的——蓝色桶。热水那边有个小红灯。冷水那边是蓝灯。她也在。她站在热水那边——绿色杯子。他站在她后面排队。轮到她接水的时候，她弯了一下腰。马尾往前扫——发尾扫在他的胸口上。他往后退了一步。又退了一步。然后她接完水转身——头发又转了一圈。目光碰在了一起。大概半秒。

他杯子里的水洒在了地上。开水——烫了一下他的手指。他没管。把杯子背在身后。然后看了她一眼。她的嘴角动了一下——不是笑，但也不完全不是。那个弧度——嘴角往上跑了大概两毫米。然后她从他旁边走过去。走了几步。马尾还在晃。

他站在原地。手背上的烫伤后来起了个泡。赵一鸣问他怎么烫的。他说"喝水烫的"。没说是因为看一个人。

"我以为你不记得。"

"我记得。"她停了停。"那天你的杯子是红色的——杯盖上有个笑脸。头发比现在短——大概到眉毛。校服的袖口卷了两道。你接水的时候从来不按'停止'。水满了溢出来——洒一地。然后你用鞋底蹭。蹭来蹭去——越蹭越花。"

他看着她。说不出话。

她的手指在女儿的被子上慢慢划着。划了一个圈。

"还有——"她继续说。"2019年5月。月湖边上。你跟我说'我喜欢你，喜欢了三年'。"

"不是三年。"

"我知道。九年。"

"但你当时说的是三年。"

"那是在月湖边。"

他低下头。手握着床沿。

"所以——"她说。"念念不忘。念是顾念的念。不忘是——我们没有忘。"

他看着女儿的小脸。

"顾念。"

他把这两个字念了一遍。很慢。像在咬文嚼字。又像在练一个新词——以后会念很多次的词。

"好听吗。"

"好听。"

"比顾棠好。"

"比顾棠好。"

他伸出手。用食指摸了摸女儿的手指——五根小手指弯着。关节处有一圈一圈的小褶。

"顾念。爸爸以后给你们改卷子，先批你妈妈的，再批你的。"

苏晚晴把脸转过去。看着窗外。眼圈有一点红。但没让眼泪掉下来。她从来不轻易流眼泪。

"还有一个原因。"

她没转回头。对着窗外说。

"我妈叫苏念慈。"

"你妈——"

"嗯。外婆叫念慈。外孙女叫顾念。中间隔了一代人。但念字还在。"

她转回头。看着他。

"我小时候不喜欢我妈的名字。觉得老气。念慈——像民国时代的人。小时候同学问我你妈叫什么——我说'阿姨'。不说名字。后来你送我诗集之后——我才觉得'念'这个字好。念不是想念的念。是挂念的念。是心里有一个人。是一个人走了很远，走了很久。回头看——那个人还在原地。还在教室后排。还在月湖边。还在301路公交车站牌下——举着伞。"

他嗯了一声。

病房里。加湿器喷出的水雾在阳光下形成一道很小的彩虹。转瞬即逝。然后又被喷出来。又形成。又消失了。短暂的彩虹——每次大概两秒。两秒。消失。又两秒。像一个循环。念念不忘——必有回响。回响之后——是新的念念不忘。

"还有一个——"他说。

"什么。"

"'念'字拆开。上面是'今'——今天。下面是一个'心'——心在底下。'今'在上，'心'在下。意思是——今天的心。现在的心。不是昨天。不是明天。就是今天。每一天。每一天都在心里。"

她看着他。看了几秒。然后低头看着女儿。手指在女儿的脸颊上轻轻划了一下。从太阳穴划到下巴。很轻——大概跟用笔的时候一样轻。

"那就顾念。"

"顾念。"

他把本子翻到新的一页。用圆珠笔写下两个字。

顾念。

笔迹很用力。墨油在纸上鼓了一下。

然后把这一页撕下来。折成小方块。塞进了床头柜的抽屉里。抽屉里有她的病例本、一包纸巾、还有两个橘子——赵一鸣昨天送来的。

"等她长大了。我会告诉她——"

"告诉她什么。"

"告诉她你的名字是妈妈取的。妈妈念大学的专业是教育学，但取名字的天赋是遗传——你外婆叫念慈，她教了一辈子语文。你妈妈叫晚晴——她教我怎么在雨天等人。你叫顾念——你教我怎么不迟到。"
"告诉她——你妈当年跟爸爸在月湖边走了两圈。一圈是爸爸跟着妈妈。一圈是并排走的。"
"告诉她——你奶奶给你妈妈取了'晚晴'这个名字，爸爸给你取了好几个名字，但最后还是你妈妈定了——顾念。因为你妈妈最知道——念这个字读起来重，写起来轻。"
"告诉她——你的名字写在爸爸当年的草稿纸上——背面有物理公式。那个公式爸爸高考没做出来。但是你的名字爸爸写了很多遍——比任何公式都熟。"

她看着他。眼圈红了。但这一次，她没有转过头去。

"我教什么。"

"你教我怎么做一个不怂的人。怎么从教室后排走到前排。怎么从看一个人到告诉她。怎么从一个人走月湖到三个人一起吃早餐——两个鸡蛋你吃一个她吃一个。"

她看着他。然后把手伸过去。在他手背上拍了一下——不重。

"别在孩子面前说自己怂。"

"还没醒。"

"醒着也不准说。"

"好。不说。"

他把抽屉推回去。抽屉有响声——木头磨着轨道。

窗外的杨树还在翻着叶子。风不大。但叶子一直在动。绿色的。白色的。绿色的。
