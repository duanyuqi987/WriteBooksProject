# 第040章 成绩单下滑多年后

所属部卷：第一部：初秋三楼
时间：高一期末考试后
地点：公布成绩的公告栏前

## 诗歌插页 P040：原创七言小诗

暑假蝉鸣各闭关，
一排名单各淡斑。
年少不知人自力，
错将备策作无关。

多年后陆允川重写成绩单下滑，先把自己写得很委屈——写到他为了一份她根本不知道的能量消耗准备好了备选教案，而他连递出去的机会都没有。他重写这段时是秋天——窗外银杏叶黄了一半，他泡了一杯速溶咖啡放在键盘旁边，咖啡凉了也没喝。写完从头读到那一行——"她靠自己就行了"——然后将所有关于"被需要"的委屈一并删除。因为在三十岁的年纪重写这件事，他终于能对十七岁的自己说一句话：你觉得可惜，是因为你把喜欢当成了一种服务——你觉得服务被拒绝了就等于感情不成立。但感情和服务是两套完全独立的系统。她能在暑假里靠自己的能力做完三百道错题——这种能力的展示本身就是她身上你最应该尊重的东西。你总不能说"我喜欢你，但你最好等一等——等我准备好教辅"。那不是喜欢，那是抢在她自己的进步前面想占一个位置。

多年后陆允川在工作中听过一个类似的描述——他的一位同事在座谈会上分享职业转型时说"以前在教研组时带过的那个学生，期末考砸了，我说要不要补课——那学生说不用，然后自己把整个暑假关在房间里做完了所有练习册。开学测验直接考了年级第一。我当时的心情是又高兴又失落——高兴是因为学生独立，失落是因为发现学生会飞了，不需要你托"。散会后陆允川拦住那位同事聊了几句——会议室外面走廊上，饮水机咕噜咕噜地烧着水，两人站在饮水机旁边聊了大概十分钟。陆允川说他也经历过一模一样的感觉，不过不是老师，是同学。同事笑说这种替别人难受超过替自己难受的性格，适合当老师不适合当暗恋者——暗恋需要一点自私，而他又太不自私了。

叶澄后来在他的修改稿上又批了一段，她说："这一章的核心矛盾不是'她成绩下滑你心疼'，而是'她不需要你你也心疼'。前者是担心，后者是觉醒。前者温暖但可有可无——班里每个人或多或少都会关心一下考差的同学。后者是在拆掉你的自我认知——原来你没有你想象的那么重要。但这很健康。拆掉一个建立在幻想上的自我认知，比让它一直骗你到三十岁——要健康得多。"

在定稿时陆允川写下一句话："她成绩下滑的那个期末，我整个暑假心里压着一道七十三名的排名。而她用三百道题的体力把那个排名清零了。我没有任何功劳——这是这件事最骄傲的地方。因为我喜欢的那个女孩不需要我的救生圈——她自己游过去了。而我唯一做的事是站在岸边一直看着她游——为她每一次划水在心里鼓掌。那掌声很响，但只有岸上的风听见了。"
