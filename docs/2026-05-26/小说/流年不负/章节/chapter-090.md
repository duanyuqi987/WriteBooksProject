# 第090章 诗

时间：2014年3月

地点：楚州师院图书馆

卷次：卷五：各自南北

三月。图书馆三楼靠墙第四个座位。

他每天下午都在那里。上完课以后——有时候是四点，有时候是五点——走到图书馆，爬三层楼，坐到那个位置。楼梯是大理石的，扶手是木头的。扶手被无数人的手摸得发亮。三楼走廊里贴着"静"字的告示牌——红底白字，塑料的，左上角缺了一小块。

窗外是槐树的枝干，还没有长叶子，光秃秃地伸向灰色的天空。三月的阳光还不太暖——从窗户斜进来，在桌面上铺成一道白色的长方形。他把笔记本放在那道白光里面。墨水的字在光下面有种很深的蓝。

桌上放着一本黑色封面的笔记本——没有横线的那种。封皮是布纹的，四个角磨白了。他在左边摊着《现代诗歌选读》——从图书馆借的，书脊上用索书号标签贴着"I226/12"。右边是笔记本。看几页，写几句。

他在图书馆的座位周围有几种固定的声音。左边第三个座位是一个考研的学姐——每天翻书的声音很大，哗啦啦的。对面是一个看杂志的男生——偶尔会低声笑出来，大概是看到了什么笑话。斜后方是一个戴耳机听英语的女生——耳机漏音，隐约能听到"listen to the conversation"。管理员推着小车经过——车轮在地板上发出嗡嗡的声音，停在新书架前面，开始一本一本地往上摆。

风格在变。

高一的时候他写古体诗。七律。模仿唐诗三百首的句式。押韵。对仗。每一联都工整。意象离不开"月""柳""雁""泪"。"那年九月风初凉""三载同窗未敢言"。那些诗工整、规矩——像他当时看她的那种目光——隔着三排课桌，不敢越界。

但现在不一样了。

他开始写更自由的东西。韵脚没了。对仗没了。句子变短了——有时只有三四个字。分行也变得随性——断在让人不舒服的地方。

意象也在变。不再是"月""柳""雁"。变成了——

"灯"——图书馆的台灯，绿色的灯罩，照亮半张桌面。拉绳开关垂下来，末端有一个黄色的塑料坠子。他有时候会用手指拨一下那个坠子——坠子摇晃，影子在桌面上来回摆动。

"站台"——师院南门口的公交站，301路停靠的那一站。站牌的油漆剥落了一小块，露出里面银色的铁皮。站台上的地砖有一块裂了——裂缝里塞着一个烟蒂。

"走廊"——教学楼那条长长的走廊，窗外的槐树影子投在地上。走廊尽头是厕所——门口挂着蓝色的门帘，门帘上有漂白水洗过的白渍。走廊中间有时候会有一个女生坐在窗台上看书。

"抽屉"——宿舍书桌的抽屉，里面放着他从高中带来的笔记本。抽屉边缘有被钥匙刮出的白色痕迹。拉开的时候木头和木头摩擦发出涩涩的声音。

"操场"——晚上走圈的那些跑道，红的，塑胶的，踩上去有回弹。跑道边上的排水沟盖板有一块松了——踩上去会咯噔一下。

"窗户"——窗外远处的宿舍楼，灰色的楼体在雨中发暗。那栋楼的某一扇窗上贴着红色的窗花——大概是过年时候贴的，一直没撕。

他写了一句——"走廊的尽头是另一条走廊"。

写完以后自己看了很久。不知道自己在写什么。走廊的尽头为什么不能是一个房间、一扇门、一片操场。为什么偏偏是另一条走廊。他把笔停住。看着那几个字。图书馆的暖气嗡嗡响。窗外的槐树枝在风里晃了一下——影子在桌面上移了一寸。

但他没有划掉。

他把笔尖悬在纸上，又写了下一句。笔尖在纸上停了一下——墨水洇出一个小蓝点。然后手动了。然后是下一句。写到一半的时候笔芯快没水了——字迹变淡。他从笔袋里换了一支新的。旧的放进笔袋——笔尖还没盖。笔袋里被染了一小块蓝色。

那天下午他写完了一整首诗。

诗的名字没想好。他在标题的位置留了一个空。

他把诗从头到尾看了一遍。字迹有深有浅——因为中途换了笔芯。有些地方改了——划掉的句子旁边补了新的。改了以后又划掉，换了第三种写法。纸面上有几处涂改——有的用横线划掉，有的涂成了黑方块。有一个词被圈起来，打了三个问号。然后放开笔。甩了甩手腕。手腕的关节咔嗒响了一下。

然后翻开桌子上的那本《现代诗歌选读》。里面有一篇海子的诗——"从明天起，做一个幸福的人"。他看了五分钟。阳光在书页上移动了一小截。他把书合上了。海子的诗太亮了。自己的太暗了。海子写麦田、写阳光、写幸福。他写走廊、写站台、写没有拨出去的电话。海子的世界是打开的。他的世界是一只锁着抽屉的手——拉不开，也不想拉开。

他把笔记本合上。夹着笔的那一页有点鼓。纸页的边缘有他手指上的汗印——在纸上留下了一道浅浅的皱纹。

窗外天快黑了。槐树的枝干在暮色里变成了剪影——黑黑的线条印在深蓝色的天上。图书馆里的灯越来越亮——绿色的台灯像一排豆子，沿着长条桌一个一个亮过去。对面的男生收起了杂志。斜后方的女生把耳机摘下来塞进书包。管理员的小车在走廊尽头转弯——拐进了史学书架区。

他站起身。座位椅面是木头的——坐久了有点硬。他把书还到架子上。《现代诗歌选读》塞回原来的位置——I226区，第四层。笔记本夹在胳膊底下。背上书包。

走出图书馆。外面的风还是冷的。他把校服的拉链拉到下巴。路灯已经亮了——橘色的光洒在图书馆门口的台阶上。台阶上的地砖有点松——踩上去晃了一下。

他站在台阶上看了三秒钟的天空。天是深蓝色的。有一颗星星——很淡。图书馆门口有几个学生结伴出来——在讨论什么。两个人的影子在路灯下拉得很长。然后往不同的方向走。然后转身往宿舍走。
