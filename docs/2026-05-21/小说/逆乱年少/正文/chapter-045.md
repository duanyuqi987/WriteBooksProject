# 第045章 空校园

时间：2011年五月一日，劳动节假期

地点：江城大学校园

卷次：第二卷：花谢的时候

五一假期来得很快。四月三十号下午，宿舍楼就开始空了。郭向阳是四月二十九号晚上走的，说他妈要他回去吃小龙虾。周明和陈磊各自走了，一个回孝感，一个回黄石。

林玉起没有回家。母亲打电话来说路费太贵，他也说路费太贵。母子俩互道了几次"那算了吧"，就把电话挂了。

他不是不想回去。但三个半小时的大巴，来回两天，在家待一天，他觉得不划算。而且——他没有说出来——他想看看空了的校园是什么样子。

五月一号早上，他被安静叫醒了。平时六点半走廊里就有拖鞋声和水声，今天什么声音都没有。他躺在床上看着上铺的木板，木板上郭向阳贴的科比海报还在，科比的扣篮姿势凝固在半空中。

食堂只开了一个窗口。打饭的阿姨认出他来，多打了半勺番茄炒蛋。他端着饭盒坐到靠窗的位置——那是苏安晴平时坐的位置。他不是故意的，至少他跟自己说不是故意的。只是那个位置光线好。

食堂空得像一个演完戏的舞台。平时排队排到门口的那个窗口，现在只有他和另一个女生在不同的窗口，隔着十几张空桌子。

那个女生他认识。不是认识，是经常看见。她也在封丘人那个群里——但他不敢上去问。他只是远远地看了她一眼，然后低头吃自己的饭。

五月二号，他在图书馆待了一整天。图书馆里的人比平时少了一大半。空调还没开，窗子都敞着，偶尔有一阵风穿堂而过，带来玉兰树叶子的气味。

下午四点，他去操场跑步。操场上只有一个男生在踢足球——一个人对着空球门踢，踢一脚，自己去捡回来，再踢。远看像个无声的默片。

他跑了五圈，停下来的时候看见一个身影往文华路那头走——苏安晴。

也留下来了。

她背着一个帆布包，走得不快，好像校园里所有的路都不需要赶。林玉起在操场边站着，汗水从额头上流下来。他想跟上去——操场到文华路，直道，不到一百米。但他没有。

不是懦弱。是不知道说什么。五一食堂，五月空校园——这些话太轻了，像打招呼，不像说话。他想说的话太重了，又不能说。所以他站在原地，看着那个背影拐进图书馆，门上的玻璃反了一下光，然后什么都看不见了。

晚上，他在笔记本上写：

> 空了的校园
> 让每一个背影
> 都更清楚
> 也更远

郭向阳发了一条短信："学校没人吧？给你带了小龙虾，回去别馋。"

林玉起回了一个字："好。"

他把手机放在床头。窗外的校园安静得像一本书翻到了空白的那一页。五月二号，假期还有两天。他不知道明天还会不会在食堂看见那个身影。但他知道，从明天早上起床开始，他又会往食堂的方向多看几眼。

不是刻意的。是习惯。
