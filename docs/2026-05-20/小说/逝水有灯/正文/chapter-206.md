# 第206章 毕业照里的空位之前

所属部卷：第四部：失去的美丽
时间：大四毕业季
地点：大学图书馆前台阶

## 诗歌插页 P206：原创现代诗

台阶被太阳晒热
所有人都在找位置
我也找
只是这一次
不再找离你最近的那一级

毕业照安排在图书馆前台阶。那天阳光很好，好到让人睁不开眼。各班学生穿着学士服从两边往中间挤，帽穗总是掉，领口也总是不平。有人举着手机自拍，有人到处借别针。

陆允川提前到了十分钟，站在树荫下看名单。拍照顺序贴在公告栏上，胶带边缘被太阳晒得翘起。他所在班级排在苏晚枝她们班后面，中间隔了一个专业。这个安排没有任何意义，却还是让他多看了一眼。

祁野拍着他的肩：“等会儿站后排，显高。”程砚在旁边整理帽穗，说：“你别听他的，后排太阳最晒。”三个人为站位争了几句，语气轻松，像终于把许多沉重的事留在了别处。

苏晚枝从图书馆侧门出来，和叶澄一起抱着几束花。她看见陆允川，点了一下头。陆允川也点头，没有走过去帮忙，也没有问她要不要一起拍单独合照。叶澄把花分给同学，苏晚枝低头整理袖口，动作很慢。

摄影老师举着喇叭喊：“先按班级排，不要乱站，空位后面补！”人群笑起来，台阶上乱成一片。陆允川站在原地，听见“空位”两个字，心里轻轻动了一下。原来毕业季最常见的词，也能让人想到告别。
