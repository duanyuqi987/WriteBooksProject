# 第050章 元旦晚会多年后

所属部卷：第一部：初秋三楼
时间：高二元旦前
地点：布置中的教室

## 诗歌插页 P050：原创七言小诗

旧台空椅各凝尘，
一曲余音二十年。
年少不知辞岁意，
错将散场作新篇。

多年后陆允川重写元旦晚会，先把自己写得很委屈——写到他终于能在元旦晚会散场后的空教室里站了十秒钟，觉得自己升了一级——从六字对话升级到了七字对话——七个字能构成一个完整的行为句。写完以后看了三遍，把"升级"这个比喻删掉，改为"那次我知道了说话不需要靠本子递过去"。

因为多年后他开始真正理解那七个字意味着什么——"你的那一段很稳"。他对她的表现做了评价。这句话平常得像同事们开会后互相说的一句"你的报告还行"。正因为这么平常，这才是他那时需要的突破——不是告白的突破，而是把对方当成一个普通人去说话的突破。之前他把她当成一个必须小心翼翼对待的对象——像端着一杯满水——手不敢抖，话不敢多说。但在元旦晚会的散场里，她裹着围巾问他节目怎么样——她开门见山地问了一个意见征求式的问题——这是最普通不过的社交动作。他接住了——并且把"挺好"升级到了"你的那一段很稳"——这不浪漫、不感人、不值得被写进任何暗恋故事里——但这恰恰是它最需要被写进去的部分：因为他第一次用对待普通人的语气对她说了话。他不再把她当成对象，而是在三十秒内把她当成一个可以向自己征求意见的普通人。

程砚在他修改稿边页上批说:"这七个字是你所有话语里最不浪漫的一句。但它是最主动的一句。因为之前的六字全部是呼应——只有这一次你是把自己的观点放进去了。把观点放进去，就是把这个人的脑袋放进对话里去了——不只是手和脚和反射。恭喜你的脑袋在元旦晚会上首次对你认为很重要的人开了口。"

叶澄批得简洁，只有一排字:"你以前怕说错话所以只说短话。你这次说了一句长一点的短话——因为你在说这条话之前没有担心对错。这是她让你放松的——她让你在她的面前成为一个不需要自我审查的人。这件事比一百次告白都更深刻。"

多年后的那个元旦——三十三岁的陆允川请了一天假，在家看书。他在那本旧诗抄本里发现了一张压扁的红纸屑——是当年的"恭贺新禧"横幅剪纸掉下来的碎屑。他把它放在摊开的诗抄本最后面——不是收藏，是送别。这五十章是从高一的九月写到了高三前夕。这部小说是从一个人在教室里第一次看见另一个人的早晨开始的。他写了报到单、书皮、分班名单、作业本、操场、值日、成绩排名、诗抄本、晚会——他把她写进了无数个学校空间里——描述，再重写，再删掉委屈。在删掉最后一遍的时候他意识到这个小说已经不再是为了纪念谁而写的了。它是为了记下那个年代里存在过的一种极其缓慢、极其恒久、极其单向但对单方面的注意力来说成立——的聚焦方式。

聚焦方式没有结果。但聚焦本身是真实的。教室是真实的，阳光是真实的，法桐叶每一年都在落——而他在那些早晨记下的每一笔，都是真实的。

全书第一部"初秋三楼"终章，结尾句：

他把红纸屑夹进诗抄本最后面，合上封面。窗外的路灯亮了——橘色的一盏，灯罩里面有一只飞蛾在绕着灯泡转，影子被投在灯罩内壁上忽大忽小。他想起那年在教室里第一次看见她时，她站在靠窗的位置，日光灯照着她的头发。那道光很细，细到他写了五十章才把它写下来。现在他写完了。他关掉台灯，让房间暗下来。小区外面有人骑着自行车经过——车铃叮叮地响。他去阳台浇花，水从叶子上往下滴，浇到第三盆的时候水壶空了，他把壶倒过来在栏杆上磕了两下，壶底掉出几滴水珠。没有任何仪式——只是把活过的那几年收纳进一本书里。书名叫《逝水有灯》。水一直在流，灯一直亮着——不是为他亮的，但他一直在看。
