# 第056章 最后一场模拟考之前

所属部卷：第一部：初秋三楼
时间：高三春天
地点：县一中考场

## 诗歌插页 P056：原创现代诗

准考证压在透明笔袋下，
铅笔削得很尖。
最后一场模拟考
不像最后，
更像一扇门前
大家同时停了停脚。

最后一场模拟考前，学校把所有人按考号重新分了考场。

陆允川在三楼东侧，靠窗第二列。桌角贴着考号，透明胶带反光。他把准考证、黑笔、2B铅笔、橡皮一一摆好，又检查了一遍笔芯。高三春天的风从窗缝里进来，带着一点潮气。

程砚在隔壁考场，临进门前还在背化学方程式。路过陆允川门口时，他举起透明笔袋晃了晃：“活着出来。”

陆允川说：“你也是。”

他们都笑了一下，却没有像以前那样多开玩笑。最后一场模拟考不是真高考，但它离高考太近，近到每个人都不敢轻视。

夏知遥的考场在楼下。陆允川是在公告栏前看见的。她的考号和名字在另一张纸上，离他不远，也不近。他没有特意去找她，只是查自己考场时顺便看见。这个“顺便”让他心里平静许多。

进考场前，广播提醒考生准备入座。走廊里全是透明笔袋摩擦校服的声音。陆允川坐下后，发现桌面有一道旧划痕，像一条不完整的线。他用手指摸了一下，立刻收回来。

监考老师拆卷，纸张声在教室里响起。卷子发到手里时，他先写姓名和考号。笔尖落下去，比高一开学时稳很多。

窗外有人跑过操场，脚步很快。陆允川没有抬头。他把试卷翻到第一页，心里只剩一个很朴素的念头：把会做的做对。
