# 第188章 诗社告别之后

所属部卷：第四部：失去的美丽
时间：大三开学
地点：诗社活动室

## 诗歌插页 P188：原创仿宋小令

罗衣轻，旧梦醒，
小窗深处有余灯。
问少年、几番执念，几番无凭。
人去不须追远影，
水流仍向明朝生。

余波总比事件慢。陆允川退出诗社，把借来的几本书还回架上。之后，陆允川把细节一遍遍放回心里比较，像检查一张做错的卷子，越检查越不肯交卷。他回到宿舍以后还在想书脊上贴歪的编号纸，想得越具体，心里越乱。

他后来明白，青春不是一场持续高烧。高烧之间有很多低温的日子，正是那些低温，让人有机会活下来，也有机会在多年后把话说得平和一点。所以这一章不只写一件事，也写那件事在岁月里怎样慢慢改变形状。那些形状有时锋利，有时柔软，最后都被他放进更宽的生活里。

街边摊的油烟慢慢升起来，把人的脸隔得有些不真切，像回忆自己加了一层雾。陆允川站在那样的声音里，忽然觉得自己并不是故事中心，只是众多走过校园的人之一。这个发现当时没有让他轻松，反而让他更想抓住一点能够证明自己的东西。

陆允川退出诗社，把借来的几本书还回架上。之后，陆允川把细节一遍遍放回心里比较，像检查一张做错的卷子，越检查越不肯交卷。他后来反复想起这一段，不是因为它多么宏大，而是因为书脊上贴歪的编号纸。具体的物件比誓言诚实，它不替人辩解，也不替人作证，只安静地留在光里。

他把台灯调暗，房间里的影子跟着软下来，话也像终于可以少说一点。他想说的话有很多，可真正到了嘴边，只剩下几句平常的话。年轻时他总觉得平常太轻，装不下喜欢；后来才知道，太重的喜欢常常先把自己压弯。

夜里停电过几分钟，整栋楼忽然安静，连平时最吵的人也压低声音。黑暗让他短暂松了一口气，因为没有人能看清他脸上的表情。这些细枝末节在当时没有被他认真看见。他忙着判断自己有没有被在意，忙着从别人的一句话里找温度，反而忽略了日子本身怎样一格一格向前挪。

“有些话你晚点懂也行，别在没懂的时候先把关系弄坏。”程砚这样说时，语气并不郑重，甚至带着一点不耐烦。陆允川当时听不进去，只觉得朋友站得太远。后来他才知道，太近的人往往只能看见火，站远一点的人才看得见整间屋子正在冒烟。

关系在这里往前挪了一点，也往旁边偏了一点。离开不是惩罚谁，而是给自己拆掉一个每日复发的触点，但余波比事件本身更慢，也更难收拾。陆允川那时没有能力准确命名这种变化，只能把它写进纸页，写得越满，越像害怕空白。

程砚后来承认，他也不知道怎样劝才算对，只能尽量不让陆允川把自己弄得太难看。祁野说退社可以，别退化成人形日记。朋友的声音并不总是温柔，有时甚至粗糙，可它们让他看见世界并没有随着一段关系变窄。

陆允川最晚才学会的一件事，是别人的日子也同样拥挤。她们不是站在原地等待他理解的谜题，而是在自己的课程、家庭、朋友和未来里努力腾挪的人。这些周边生活把爱情从高处拉回地面。它提醒陆允川，心动不是唯一的课程，失落也不是唯一的作业；人总要在很多不浪漫的事情里，慢慢学会怎样对待一件浪漫的事。

多年后他终于愿意承认，她不需要在他的青春里承担一个完整解释。她可以不解释，也可以继续往前走。苏晚枝看到空出来的登记栏，沉默了一会儿。这部分事实后来才被陆允川补进小说。补进去以后，故事没有变得轻松，却变得诚实了一些。

远处操场有哨声，传到这里时已经被教学楼磨软，只剩一点淡淡的回响。他把这一幕和诗歌放在同一页，并不是为了显得文雅。诗对那时的他来说，是一块可以暂时放下重物的石头；重物还在，只是手终于能松开片刻。

原创仿宋小令写完以后，他在下面空了一行。那一行空白比诗句更安静，像把解释留给未来，而不是逼当时的任何人马上回答。

他写小令时最怕写得太满，所以故意留出转折。那些转折像成年后才学会的礼貌，不把自己的伤口直接推到别人面前。因此这些插页不是装饰，也不是为了让青春显得更漂亮。它们更像一排小灯，照着他从自我感动走向自我审查的路；灯光不强，但足够让人看清脚下有水。

后来他才明白，年轻时最容易把强烈当成深情，把忍耐当成道德，把被忽略当成命运给自己的考验。他越长大，越不愿把这段青春说成谁辜负谁。辜负当然存在，可它不是全部；全部里还有胆怯、误会、虚荣、善意和没学会好好告别的年纪。

下午的课间很短，短到只够从教室走到水房再走回来。可他偏偏在这十分钟里想了很多事，想未来，想成绩，想某个名字，也想自己是不是从一开始就把普通日子看得太像预兆。这一类日常在小说里反复出现，是因为陆允川后来相信，真正改变一个人的不是某个爆炸瞬间，而是爆炸之后仍要买饭、交作业、赶公交、给手机充电的无数普通时刻。

多年后重写这一章，他把原来一整段自怜改成了两个人的动作。改完以后，人物没有那么可怜，故事却更宽了。他没有因此变得完全宽容，也没有把所有疼痛都解释成成长礼物。他只是终于愿意承认，自己和别人都曾经笨拙，而笨拙不必永远被写成罪。

她把目光转向窗外，看了几秒树影，再回头时语气已经恢复平常。这个动作很小，小到放进当年的日记里可能会被嫌不够痛。可真正能留下来的，常常正是这些不够痛的小动作，它们让人相信那一年确实活过。

在这一章的末尾，陆允川没有得到答案。书架很快被别人补满，像生活从来不缺位置。他只是比上一章多看见了一点自己，多承认了一点别人，多放松了一点攥紧的手。

他曾经以为自己失去的是某个女孩，后来才知道，他更难接受的是自己的付出没有换回想象里的结局。那一刻并没有答案，只有风把窗帘吹起一点，又缓缓落下。故事继续往前走，像水流过桥洞，没有替任何人停下，却把所有经过都带成了更低的声音。
