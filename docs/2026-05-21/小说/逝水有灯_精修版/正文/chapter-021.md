# 第021章 操场边的单词之前

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P021：原创现代诗

我把早晨放低，
让一粒尘从课桌边慢慢经过。
你的名字没有被我喊出，
只在单词书页角折起的三角旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写操场边的单词抵达之前的坡度。晚自习前的操场还有白天剩下的热。看台水泥摸上去发烫——手按上去能感觉到那种从地底往上返的热量，像摸一块刚从太阳底下搬回来的砖。塑胶跑道被夕阳照出暗红色，像一条慢慢冷下来的河。远处篮球场上还有人在投篮，篮球砸在水泥地上发出闷闷的"咚咚"声，间隔很长——投一个，弹两下，滚进草丛。陆允川摊开单词书，嘴里念着"abandon"，眼睛却跟着跑道边的影子移动。单词书的纸页被晚风吹起来，他用大拇指压住，拇指边缘嵌进纸页里，留下一个湿湿的指印。

念到第三遍"abandon"时他发现自己连中文意思都忘了——"abandon"是放弃，但他脑子里浮现出来的中文是"抛弃"。这两个词在中文里不太一样，但他也懒得去查了。他又看了眼书角那个被折起的三角——那是程砚折的，上周两人在看台上背单词，程砚说折个角好翻页，折得歪歪扭扭，三角形的两条边都不一样长，折痕还偏了。这个角现在支棱着，像一个不肯被书本压平的念头。他用指甲把那个折角重新压了压，压平了不到一秒它又翘了起来。

"你是来背单词的还是来看夕阳的？"程砚从旁边的台阶上转过来，手里的单词书翻到了第七页——明显比陆允川的进度快。程砚背单词的办法很粗暴：先读三遍中文，再读三遍英文，然后闭上眼睛在脑子里把十个单词编成一个离谱的故事。比如"abandon放弃"+"absorb吸收"+"abstract抽象"——他会编成"一个被放弃的人吸收了很多抽象的道理"。他自己也知道这个故事毫无逻辑，但越离谱越记得住。

"我背到第四个了。"陆允川说。
"你背到第四个，手里拿着的是第三页。第三页第一个单词是abandon，中文意思放弃。你在放弃这个单词上停了五分钟——你想说明什么？"
"说明我背得很认真。"
"说明你在看跑道。"

操场上有人在跑步。不是校队的，是几个因为体育课被占而自己来补跑的高二学生。他们跑得不快，呼吸声粗重，脚步声参差不齐——有人踩在塑胶跑道上声音闷，有人踩在煤渣边缘上声音脆。跑道上除了他们还有一圈慢走的人——夏知遥和蒋雯。蒋雯说今天的物理作业太多，需要走几圈放空脑子。她一边走一边比划，像是在背公式，手指在空中画了好几个圈。夏知遥没有跑步，只是慢慢地走，手里拿着一瓶水——农夫山泉，瓶身被捏得有点凹进去了——偶尔喝一口，拧瓶盖的时候瓶口发出很小的"嗤"一声，气流声在空荡荡的操场上听得格外清楚。她的影子被夕阳拉得很长，从跑道里面斜着伸到草坪上，和草坪边缘的白色界线相交成一个钝角。

陆允川的目光在单词书和跑道之间来回切换，频率越来越不规律——三秒看书，六秒看跑道，再看书时发现同一行读了两遍。他开始在心里给自己找借口：背单词需要远眺保护视力，看书太近会加深近视，塑胶跑道的颜色刚好是暗红色，看着能放松眼球。他甚至想出一个非常科学的说法——"暗红色光的波长最长，对眼部肌肉的放松效果最好"——虽然他知道这是编的。每一个借口都在骗自己，但他需要骗——因为不骗的话他没法继续坐在这看台上。坐不住意味着下次就不来了，而不来的话，那五米的距离就变成了永久的。

晚自习前的时间到了，操场边的路灯亮起来——先是闪了两下，然后才稳住，发出橘黄色的光。跑道上的影子从一道变成了两道——人和灯各给了一道，一深一浅，深的往东斜，浅的往西斜，交叠在一起像一对没有对上焦的底片。蒋雯和夏知遥走完了三圈，从西看台下面经过时蒋雯抬头看见了他，挥了一下手："嘿，背单词啊？明天英语小测加油！"她的声音在空旷的操场上荡了一圈才消散。陆允川举起手里的单词书回了一句"你们也加油"。蒋雯笑了笑继续走。夏知遥没有抬头，她低着头在用脚踢跑道边沿的一块碎石子——石子不大，差不多指甲盖大小，被她踢得滚了两圈停住，她收脚绕过去，继续走了。她的运动鞋踩在塑胶跑道上几乎没有声音。

陆允川把目光收回单词书。他翻到第四页，逼自己从"abandon"隔壁开始重新背。新的单词是"absorb"——吸收。他把这个词抄在手上，中文意思也抄了一遍，手心写得满满的。他告诉自己：下一次来操场背单词，不抄满一页不许走。

但他也知道，下一次还是会在同一个角度看同一个方向，手中的单词书翻到同一页，在同一个"abandon"上停下来。不是因为"abandon"这个词有多重要——单词书上五百个词，它只是恰好排在第一个——而是因为从"放弃"开始背，每次都能给自己一个重新开始的感觉。就像每天早上从宿舍走到教室的路，一样的路灯、一样的台阶、一样被踩得发亮的树叶渣，但迈第一步的时候总觉得自己今天可以比昨天好一点。
