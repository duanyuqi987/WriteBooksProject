# 第189章 诗社告别旁观

所属部卷：第四部：失去的美丽
时间：同一周末
地点：诗社活动室

## 诗歌插页 P189：原创现代诗

空出来的位置
没有立刻长出答案
有人坐下
有人搬水
桌面上仍有铅笔灰

苏晚枝周末值班时，把陆允川留下的文件夹重新看了一遍。流程表写得很细，投影线放在哪个抽屉、备用马克笔还有几支、活动结束后谁负责锁门，都列得清清楚楚。她看着看着，心里有一点酸。

她不否认陆允川曾经认真。认真本身没有错。让她累的是那份认真后来总要她回应，总要她承认它的重量。现在这份认真被整理成了流程表，放在桌上，不再追着她要一句评价，她反而能安静地说一声：他其实做了很多。

叶澄在旁边贴海报，见她发怔，问：“怎么了？”

“没事。”苏晚枝合上文件夹，“他把东西交得挺完整。”

叶澄低头把胶带扯断：“这样也好。”

两个人都没有继续说下去。活动室里有些话不用讲透。讲透了，就又会变成另一种压力。苏晚枝把钥匙挂回门后的钩子，蓝色塑料牌轻轻晃了两下。她忽然想起陆允川离开时没有回头叫她，也没有把告别说得像一场审判。

晚上读书会来了七个人，比预计多两个。新生坐在陆允川常坐的位置，拿着本子记大家推荐的书。苏晚枝给她倒水时，心里那点空慢慢落下去。空不是惩罚，有时只是给后来的人留座。
