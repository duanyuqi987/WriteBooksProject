# 第093章 许若琳

时间：2014年5月

地点：支教社团活动

卷次：卷五：各自南北

支教社团出发前有三次培训。培训地点在师院大学生活动中心二楼——一间有落地镜的舞蹈教室。教室是木地板，走路的时候有点咯吱响。落地镜占了整面墙。镜面有刮花的痕迹——大概是以前练舞的时候不小心碰到的。每次去都要脱鞋。门口摆了一排鞋子——运动鞋、帆布鞋、凉鞋。鞋子里有鞋垫——有些鞋垫是红色的，有些是灰色的。每个人脱鞋的动作都很熟练——单脚站、解鞋带、把鞋放整齐。顾临风每次都坐在最后一排——背靠着镜子。镜子里能看到自己的后脑勺。

第一次培训讲安全事项。一个老师站在投影幕布前面讲——"山区有蛇""不要单独行动""随身带手电筒"。屏幕上放的照片是大山里的土路和石头房子。第二次讲教案编写。发了打印的模板——"教学目标""教学重难点""教学过程"。每人一份，纸张是A4的白纸，最下面印着"全省高校支教联盟制"。第三次讲山区教育的现状。PPT上放了数据——"山区小学入学率""留守儿童比例"。有一页放了照片——一个山区小学的教室：泥墙、破课桌、黑板上写着"好好学习"。

第三次培训结束以后，大家在门口穿鞋。有人坐在排椅上穿——慢悠悠地系鞋带。有人单脚站着跳着穿。许若琳也在——她是第二次培训才出现的，坐得离顾临风不远。她穿的是帆布鞋——白色的，鞋带是花的。鞋底脏了——大概踩过泥。

"你也来支教社团了？"她系着鞋带，抬头看他。鞋带系了一半——左手拉着鞋带，右手还没动。

"嗯。第三分队。"

"第三分队是跟理工学院的。你认识那边的人吗。"

"不认识。"

"那你挺勇的。"她站起来，拍了拍膝盖上的灰。裤子上有些微小的尘屑在灯光里飘起来。"我是第一分队——师大的那队。"

"嗯。"

"苏晚晴也在第一分队。"许若琳说这句话的时候正在把鞋带塞进鞋子里。她低着头，看不到顾临风的表情。她的声音不大——舞蹈教室里回声有点重，尾音被放大了。

"嗯。"

"她教三年级的语文。我教英语。"

"嗯。"

许若琳把脚塞进去了，抬起了头。她的眼睛里有一丝笑意——不是那种好玩的，是那种看穿了什么的笑。"你怎么每次提到她都是'嗯'。"

"不知道说什么。"

"行吧。"她笑了一下。笑的时候眼睛眯起来。牙齿露了一排。"不过我倒是可以偶尔给你传递一下情报——她在师大的近况。你要是想知道的话。"

"不用。"

"真的不用。"

"嗯。"

顾临风穿好了鞋。鞋带拉得太紧——脚背有点勒。他松开了一点。站起来。把旁边的墙当椅子——靠了一下。墙壁是凉的。他后背贴着墙。

但许若琳还是说了。每次社团活动或者培训碰到面，她总会在谈话的某一刻忽然把话题带到苏晚晴——"晚晴最近在准备教师资格证"——"她实习教三年级语文"——"她说小朋友很可爱"——"她上次在食堂点了一道糖醋排骨然后说太甜了"——"她的室友说梦话，说的是'同学们坐好'，梦里面还在上课"——"她学了骑自行车，天天在师大里面的梧桐路下面练习"。

她说的时候像在闲聊。语气很轻。说完就看别处——不盯着他看。有时候她会在说之前先看他一眼——看看他在不在听。然后接着说。说完以后低头系鞋带或者翻书包。假装刚才的话只是随便提起的。

顾临风每次都是一样的反应——不做声。不多问。不接话。

但他把每一句话都记住了。

她准备教师资格证。笔试考了两门——教育学和心理学。她实习教语文。第一堂课紧张得声音发抖——但讲完了全本。糖醋排骨太甜了。小朋友很可爱。室友说梦话在喊"同学们坐好"。她学会了骑自行车。每一句都像是从另一个人的人生里截取下来的碎片——碎的，不完整的。他不能拼，也不敢拼。只能收着——把它们塞进那个叫记忆的抽屉，拉开的时候里面有光，关上的时候灯灭了。

有一次许若琳说——"晚晴学会了骑自行车，在师大里面骑，把膝盖摔破了皮。"

她说完这句话的时候正在整理书包。书包带子打了个结——她低头解结。

他那天晚上在操场多走了一圈。第五圈。走得很慢。第五道上的白色分界线在灯光下有微弱的光。不是因为他担心膝盖破了皮——是因为他想象了她骑车的样子。马尾。浅蓝色外套。骑在师大的梧桐路下面。然后摔了。膝盖破了皮——她会自己上红药水吗。

他不知道。也不会问她。

他只知道——她在十二公里外的另一个校园里，学会了骑自行车。膝盖上多了一块擦伤。跟所有普通的大学生一样在上课、吃饭、睡觉。他住在十二公里外的另一个校园里，这些不是他应该知道的——但许若琳还是说了。

他没阻止。也没问更多。

五月中旬。最后一次培训前。许若琳忽然在QQ上说了一句话——"她好像瘦了一点。"

"为什么。"

"不知道。可能是实习太累了。"

"嗯。"

"你不是说不关心吗。"

"随口问的。"

"随口问——也是关心。"

顾临风把QQ对话框最小化了。窗外是五月傍晚的光——金黄色的，斜斜地照在对面楼房的墙上。玻璃反着光。对面楼房的阳台上晾了几床被子——被套是碎花图案的。远处有人在大声喊——"快点——食堂要没菜了——"

他没有告诉许若琳——那天晚上他查了师大的地址。在地图上找到了师大北门。手指在地图上来回滑动——把师大北门放大、缩小、再放大。然后退出了。

十二公里。十一站路。四十分钟公交。301路。他从手机备忘录里翻出来又看了一遍。

他没有问许若琳——能不能帮我带句话——能不能告诉她我在师院挺好的——能不能告诉她我考了第三名——能不能告诉她我其实每天都——

他没有问。他关了电脑，去了操场。

那天晚上操场上人很少。大概是因为明天要下雨——空气里的湿度大。他在第五道走了五圈。走完以后在饮水台喝了一口水。回到宿舍。阿坤在打游戏，李博在看书，小伍在写信。他躺在床上。枕头下面的手机压着一个号码和一个名字。

天花板上的裂缝还在。
