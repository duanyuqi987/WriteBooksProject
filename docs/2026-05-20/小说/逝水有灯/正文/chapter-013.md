# 第013章 理科名单之后

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：男生宿舍与新教室

## 诗歌插页 P013：原创仿宋小令

红榜定，晚风停，
旧班灯火换新名。
问同窗、几分靠近，几分伶仃。
一页书声翻未稳，
少年心事怕人听。

换班后的第一晚，陆允川给家里打电话。

宿舍走廊信号不好，他站在窗边，听母亲问新班怎么样，老师凶不凶，吃饭习不习惯。他一一回答，说都挺好，说理科班人多，说老师看起来很负责。母亲在那边停了一下，又问：“有没有认识的同学？”

陆允川看向走廊尽头。那里挂着一盏昏黄的灯，蚊虫围着灯罩乱撞。

“有。”他说，“程砚也在。”

他说的是事实，却不是全部。可有些名字没办法说给家里听，说出来就像把一件还没成形的东西拿到亮处，会显得轻，也会显得不懂事。

第二天早读，理科教室比原来的教室更挤。前排同学读英语的声音整齐，后排总有人压低声音交换橡皮和涂改带。陆允川把单词书翻开，余光却总往座位表那里飘。邵老师把座位表贴在讲台侧面，他每次抬头，都能看见那张纸。

程砚看不下去，把自己的书立起来挡住他的视线：“你再看，座位表都要怀孕了。”

“你能不能正常说话？”

“不能。”程砚说，“正常说话你不听。”

上午第三节物理，老师讲运动学。陆允川听到“相对位置”四个字，笔尖停在纸上。他觉得这几个字很准，又觉得自己用课本概念解释心事实在荒唐。

下课后，夏知遥从前排走到讲台交作业。她经过后排时，手里的作业本滑了一下，最上面一本差点掉下来。陆允川手指动了一下，没来得及伸出去，旁边另一个女生已经帮她扶住。

“谢谢。”夏知遥对那个女生说。

陆允川低下头，继续抄物理公式。那一瞬间的失落很小，小到不值得写进任何大事里。可少年人的心事往往就靠这些小失落长大。它们像桌肚里没放平的书，一开始只是硌一下，久了就让人总觉得哪里不舒服。

晚上他在硬皮本上写：同班不是答案，只是把问题放近了一点。

写完以后，他把这行字划掉。不是因为不对，而是太像结论。那时的他还没有资格下结论。他只是在新教室里，笨拙地学习怎样把目光收回来。
