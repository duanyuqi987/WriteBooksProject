# 第141章 第四次见面
时间：2018年4月中旬
地点：楚州月湖边
卷次：卷八：旧诗新读

周六下午两点。顾临风站在楚州月湖的入口处，手插在口袋里。阳光从柳树叶子间漏下来，在地上晃。

苏晚晴从公交站方向走过来。深蓝色长裙，白色帆布鞋。马尾扎得比上次高了一点。

"到了。"她说。

"嗯。"他说。

她看了一眼湖。"这就是你说的那个地方。"

"楚州的月湖。跟青阳那个同名。"

她点点头，没说话，往湖边走过去。

湖不大。一圈大概一公里。湖边的柳树种了有些年头了，枝条垂到水面上。水是绿的，不清，能看到小鱼在水面下翻。几只野鸭子从芦苇丛里游出来，看了看他们，又游回去了。

她走到湖边的一块石头上，站住。石头是半圆的，表面被磨得很平。她低头看了看，然后坐下去。

他没跟过去。站在两步以外，看着她。

她在石头上坐了一会儿。腿收起来，胳膊环着膝盖。湖风吹过来，把她的刘海吹到眼睛上。她没去拨——就让头发在脸上停着。

"这个地方。"

"嗯。"

"跟青阳的月湖很像。"

"我看到的时候也这么想。"

她转过头来看他。"你什么时候发现这个湖的。"

"去年。骑车经过这里，看到路牌上写着月湖路。就拐进来看了看。"

"然后你就记住了。"

"嗯。"

她转回去，继续看水面。一只水蜘蛛从石头边划过去，在水上留下四道细纹。

他从口袋里拿出两瓶水。一瓶递给她。

她接过去。"又是农夫山泉。"

"习惯了。"

她拧开盖子喝了一口。瓶口压过嘴唇的地方留了一小圈水印。

"你知道我第一次在月湖边坐是什么时候吗。"她说。

"高中？"

"高二下学期。月考之后。我考得不好。"

"你考得不好？"

"物理。七十二分。"

他没说话。他记得那张成绩单——物理那一栏，她的名字排在中间偏后。他排在前十。

"我一个人去的月湖。"她说。"在湖边坐了一个小时。"

"然后呢。"

"然后回家了。"

"那次我没看到你。"

她转过头。"你怎么可能看到我。你又不认识我。"

他说不出话。

她看了他两秒。"骗你的。我知道你认识我。"

又一只野鸭子飞起来。翅膀拍在水面上，啪啪响了三声。

他走到石头旁边，在她左边坐下来。石头够大，两个人坐还有空。中间隔了大概一掌的距离。

"那次相亲。"他说。"你进门的时候，我手心里全是汗。"

"我知道。"

"你怎么知道。"

"你握手的时候手是湿的。"

他没说话。耳朵尖开始发红。

她把水瓶放在石头边上。水面上的波纹一荡一荡的，晃到了石头下面。

"你那天穿了一件白衬衫。"她说。"领子没翻好。"

"你怎么不告诉我。"

"后来想告诉你的。但你已经走了。"

"我走得太快了。"

"嗯。"

一只鸟从柳树上跳到另一棵柳树上。树枝晃了几下，又不动了。

她忽然笑了一下。不是大笑——就是嘴角往上弯了一点。他从来没见她这么笑过。连高三毕业那晚坐在长椅上也没有。

"你笑什么。"

"没什么。"她说。"就是在想，这算是第四次见面。"

"第四次。"

"相亲第一次。咖啡馆第二次。书店第三次。这里第四次。"

他算了算。"对。第四次。"

"四次了。"

"嗯。"

她站起来，拍了拍裙子上的灰。"走吧。绕湖走一圈。"

他也站起来。

湖边的石板路有些不平。她的帆布鞋踩在上面，没有声音。他的运动鞋踩上去，也没有声音。

太阳往西偏了一点。阳光从背后照过来，把两个人的影子投在湖面上——一高一低，挨得很近。

柳条垂到水面，又被风吹起来。一根柳条上的叶子碰了一下另一根——三片叶子碰在一起，又分开。

湖对面的亭子里有两个老人在下棋。棋子落在棋盘上，啪嗒啪嗒地响。隔着半个湖面传过来，声音已经很轻了。

她走在前面。他走在后面。走了大概二十步，她停下来等了他一下。

"走快点。"

"好。"

他赶上去。两个人并肩走着。路过一棵特别粗的柳树时，她伸手碰了一下树干。树皮粗糙，在她的指尖上留下一点白印。

"这棵柳树。"

"怎么。"

"跟青阳月湖那棵差不多粗。"

"你记得。"

"记得。"

她把手收回来，继续往前走。

经过亭子的时候下棋的老人刚好收棋。一个老头把象往前推了一步——"将军。"另一个老头看了看棋盘，把老帅往右移了一格——"没死。"将军的那位想了想——把車往回拉了三步。"算了——不杀了。天快黑了。"

两个老人把棋子收进布袋。红棋和黑棋混在一起，哗啦啦地响。布袋的口子收紧了——抬起来扛在肩上。她从他们身边走过去——那个老人看了她一眼，又看了一眼跟在她后面的他。嘴角动了一下，没说话。

他们走过亭子。亭子后面有一棵歪脖子柳树——树干弯到水面上方。一只翠鸟正站在树干上——蓝绿色的羽毛。它在盯水面。忽然扎进水里——水面开了个洞——翠鸟从洞里弹出来，嘴上叼了一条小银鱼。鱼尾甩了两下。飞走了。

"翠鸟。"她说。

"青阳月湖边也有。"

"你知道翠鸟的窝吗。在水边的土洞里。很小——洞口只有拳头大。"

"你找过。"

"小时候跟我爸去找过。我爸说翠鸟的窝要耐心找——它们把洞口藏在水草后面。那天我们找了两个多小时。最后在一丛芦苇根的后面找到了。我爸说——你看，好找的地方都不是真的。"

他听着。手在口袋里捏着那个青石子。

"你爸呢。"

"前年走了。肺病。"

他没说话。脚步慢了半拍。

"那棵翠鸟窝还在。"

"在月湖边上。"

"对。石桥往右走。过了那排水杉。第三丛芦苇根后面。洞口朝东——早上太阳照进去的时候，洞口是亮的。"

"你后来一个人去过。"

"嗯。每年都去一次。今年还没去。"

"一起去。"

她看了他一眼。点了头。没说话。

绕湖一圈走完了。回到入口处。公交站牌在马路对面，红底白字。

"下周？"他说。

"什么。"

"下周。还来这里。"

她没立刻回答。风从湖上吹过来，吹动她裙子下摆。

"好。"

她上了公交车。车门关上之前，她回过头看了他一眼。

车门关上了。车开走了。

他在公交站牌下站了一会儿。湖面上又飞起一只野鸭子。这次只有一只。
