# 第099章 省城南湖旁观

所属部卷：第二部：六月以后
时间：大一九月
地点：大学南湖凉亭

## 诗歌插页 P099：原创现代短诗

那时我们都以为，
结束会有清楚的门。
后来才知道，
门只是普通下午的一阵风。
有人走向远处，
有人学着把远处还给远处。

苏晚枝是第一次当生活委员。临时当的——开学那天班主任在群里问"谁愿意负责班费"，群里沉默了大概三十秒。苏晚枝在那三十秒里想了两件事：第一，她妈是大学会计，她从小学一年级起就帮妈妈用Excel记账；第二，如果没人举手，班主任可能会点名。所以她举手了。

事后她发现自己举手的那个时间点，和季朗发第一条群消息的时间点相差不到一秒。季朗发的是"老师我推荐苏晚枝"。后来她问季朗怎么知道她的名字——他们当时才认识两天。季朗说因为新生名单是按学号排的，"你学号是女生第一号，我扫了一眼就记住了。"

"你这个记账方式太细了。"季朗在旁边看着她那个巴掌大的笔记本——不是电子表格，是纸质笔记本，翻开的页面被铅笔划成了细密的格子。上面密密麻麻写满了每个人的签子数量：张XX，牛肉3竹签，玉米1铁签。李XX，鸡翅2铁签，豆腐1竹签。每行末尾还用小字标注了单价和合计。"烧烤而已，不用这么精确吧。"

"班费是大家的。每根签子都是成本。"苏晚枝头也不抬，笔尖在本子上飞快地动——她的字很小，像印刷体，每个数字都写得很清楚，"你知道鸡翅多少钱一串吗？"

"不知道。"

"六毛。竹签两分。总成本六毛二分。一个人多吃两串就是一块二。四十个人就是四十八块。班费总共才收了一千。烧烤只是第一项开销，后面还有秋游、聚餐、打印复习资料。"她终于抬了一下头，把垂到额前的一缕头发别到耳后，"如果我每项都差四十八块，到期末班费就会出现一个让人无法解释的缺口。"

"……你以后适合当财务总监。"

苏晚枝没有理他。她的注意力分成了两半——一半在记账上，一半在观察人。她发现今天来的四十多个人里，有几个特别有意思的。

一个是陆允川。他自我介绍说"高考英语作文差点跑题，最后还是没跑"。这句话把全班逗笑了——有人笑得拍了一下大腿，有人喊"英语作文跑题是祖传技能"。但苏晚枝注意到他说这句话的时候嘴角是平的——他没有把这句话当笑话讲。他是把一件真实发生过的事当成"简介"来说，像在陈述一个已经归档的技术参数。这不是幽默，是某种已经消化之后的坦诚。她妈教她看账本的时候说过一句话：真正的数字从不撒谎，但自己可以决定怎么看它。陆允川就是这样——他把那些差点绊倒自己的东西，变成了可以轻松说出口的事。不是释然，是把伤口晾干了再收起来。

另一个是那个站在人群外圈啃玉米的男生——苏鹤。他是二班的，但不知怎么混进了一班的团建。她记得他是怎么来的——团建开始前十分钟，季朗在走廊上喊了一句"有没有人来南湖烧烤"，苏鹤正好拎着水壶路过，季朗说"你也来"，苏鹤说"好"。然后就跟来了。全程没有主动跟任何人说话，自我介绍只有一句话："苏鹤，二班。"然后就坐到凉亭边上去啃玉米了。他是那种——不需要别人关注，也不在意别人不关注——的人。苏晚枝在他身上看到了跟自己完全不同的生存方式：她是靠精确和记账来找到安全感的，事事都要有据可查；他是靠沉默和距离，把自己的存在感压缩到最小，好像这样就不会有人找他麻烦。

还有一个，季朗。他一直在说话。跟这个人说他在床上铺床单的心得——"床单要塞进去，不能只盖在床垫上，不然睡一晚上就会皱成一团"；跟那个人说高中军训叠被子的秘诀——"教官教的不是叠被子，是折纸，角度要对齐"。他没当一回事，但每个人都被他说得笑了一次。他是那种天然的气氛调节器——不需要多么幽默的段子，只要不停地说话就够了。苏晚枝数了一下，在烧烤的前二十分钟里，季朗至少跟七个人搭过话，话题涵盖了床单铺法、军训叠被子、牛肉和羊肉的区别、以及"为什么土木系的女生比例是全校最低的"。

散场的时候，苏晚枝把最后一批用完的竹签收进垃圾袋，数了数——一百二十四根。跟她账本上的数字完全一致。她把账本合上，满意地点了点头。

季朗从旁边递过来一瓶没喝完的矿泉水。"喝不喝？"

"这是你的。"

"我喝不完了。你帮我喝掉，别浪费。"

苏晚枝接过水瓶，拧开瓶盖喝了一口。水是温的。

"你刚才一直在观察人。"季朗忽然说。

苏晚枝差点被水呛到。"你怎么知道？"

"因为你记账的时候眼神不在本子上。你在看人。"

"……你观察力这么好干嘛用在了奇怪的地方。"

"不奇怪。我以前也想当生活委员。但我懒得记账。"

"那你现在当什么？"

"什么也没当。"季朗笑了，"我准备竞选的唯一职位是班级会议出席率最高的人。"

苏晚枝没忍住笑了出来。她把水瓶盖拧好，站起来，把垃圾袋提起来准备去扔。季朗跟在她后面，两个人一前一后走过南湖边上的柳树。柳影在水面上晃着，夕阳把凉亭的倒影拉得很长。

"你觉得陆允川这个人怎么样？"季朗忽然问。

"为什么问这个？"

"因为他是唯一一个自我介绍时说了实话的人。'差点跑题但没跑'——这不是笑话。这是一段被消化过的故事。"

苏晚枝停了一步。她回头看了季朗一眼。"你刚才说你想当生活委员是骗我的吧。你其实想当心理委员。"

"被你发现了。"季朗笑得坦荡，"但我说的都是真的。包括那句——他自我介绍的时候，我看到了他嘴角是平的。你也看到了，对吧？"

苏晚枝没有回答。但她刚才观察到的，跟季朗一模一样。
