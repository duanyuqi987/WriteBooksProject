# 第020章 前排和后排多年后

所属部卷：第一部：初秋三楼
时间：高一春天
地点：物理课后的教室

## 诗歌插页 P020：原创七言小诗

传本旧事已成尘，
一瞬相邻误此生。
年少不知交臂过，
错将传处作归程。

多年后陆允川重写前排和后排，先把自己写得很委屈——写到那两本作业本在传递过程中贴在一起时他用了一整段的比喻，什么"像两条鱼在洋流里短暂相触"，写完读到一半就划掉了。太酸了。两条鱼至少还是活物，两本作业本只是摩擦力、相对运动和手与手之间的物理传递。他看了看自己划掉的那行字，笔迹从实线变成了锯齿状的波浪——他划掉的时候下手太重，纸面被他划出了三道平行的凹痕。物理老师如果看到他这么写，会让他把比喻改成受力分析。

多年后他在一份办公邮件里看到有人把文件传错了顺序——附件一和附件二的标题被贴在一起发出去，同事故意回复了所有人，在邮件正文里画了一个箭头说"这两个附件是不是在谈恋爱"。陆允川盯着那封邮件看了好几秒，然后莫名想起了那次传作业。两本作业本叠在一起的几秒钟，在物理时间的维度上甚至不值得被标记——它比一个喷嚏还短，比他后来任何一次正式的、有话要说的交谈都微不足道。但他之所以还记得，不是因为那几秒钟真的多重要，而是因为在他十七岁的时候，那就是他离她最近的——不是距离上的近，是意义上的近。那时候他没有什么别的可以拥有的，所以连几秒钟的封面贴着封面都被他算成了共同经历。他甚至给那几秒钟起过一个名字：在脑子里叫它"叠面事件"。一个只有他一个人知道的事件。

程砚后来在一次喝酒时忽然提起这件事。那是在一家烧烤店里，桌面上铺着一次性塑料布，烤串的竹签堆了一小堆。程砚喝了两瓶啤酒，脸上有点红了，用竹签在桌上划拉了两下，忽然说："你还记得高一物理课传作业本吗？""记得。""你那时候每次传作业之前都要摸一下黑板。手上沾了粉笔灰，然后递本子的时候手指白白的。我当时以为你有洁癖——你知道为什么我这么想吗？因为你摸完之后还看手指，像在检查指甲缝里有没有灰。后来才反应过来你是在给自己找借口——手上有灰的话递本子的动作可以慢一点。"陆允川放下啤酒杯，没有说话。程砚替他补了一句："你那个灰量，够写满半块黑板了。你每节课摸两次黑板，一周五天物理课，一个学期二十周——你算算你一共拿掉多少粉笔灰？"

叶澄在他的修改稿边上批了另一句话。她读完《前排和后排》这一章后说："你把传本子写得太重要了。重要不是因为本子重要——本子就是本子——而是因为十七岁的人没有别的可以重要的事。你没钱、没工作、没朋友圈、没有需要负责的人。你手里只有作业本。所以你把作业本当成了全部意义的载体。这不是幼稚，这是现实条件下的最优解。"

陆允川觉得她说得很对。十七岁时能拥有的东西太少了：一张课桌、一本练习册、一支快用完的圆珠笔、每天从后往前传的那几秒钟。他不可能在那时候就拥有成年人的从容——从容需要时间站在你这边，而时间在十七岁的时候还没开始站队。所以他只能在作业本上找意义。那本作业本上有一处蓝色墨点——他后来确认过，那是钢笔水从笔尖漏出来的，圆圆的，可能是考试时不小心戳到纸面留下的。墨点本身什么都不是，但它是她的书桌、她的笔试、她捏笔时无意间的一个偏差——它是一整个世界的一扇窗，而他十七岁时刚好就站在那扇窗户前。

改完这章之后他接到叶澄的电话，说她要结婚了。对象是她研究生导师介绍的一个建筑师，话不多，但每次叶澄加班到很晚都会去接她。"他话不多，但做事很稳。"叶澄在电话里说，语气和当年在教室里提醒他填表格时一模一样。"那就好。"陆允川说。"你那个小说还在写？""还在。""写到哪了？""写到作业本传过去了。""传过去就对了。本子总归要传过去。传过去才能批。"

挂了电话，他走到窗前站了一会儿。窗外是城市的夜景，路灯很亮，照得人行道上的地砖一片橙黄。他忽然想到——叶澄说得对，本子总归要传过去，传过去才能批。他回到桌前，在修改稿上又加了一行字：多年后回头看，当年那两本叠在一起的作业本没有被任何人注意到——物理老师没注意，前排的人没注意，她本人也没注意。只有他注意到了，并且留到了今天。这不是命运的伏笔，只是一个人为了撑过自己的青春而制造的微小的仪式感。但那也没关系——仪式感只要能撑住一个人，就是有效的。就像当年摸黑板沾粉笔灰，明知那个借口很拙劣，但每次做完这个小动作之后，递本子的手确实更稳了一些。
