# 第047章 元旦晚会

所属部卷：第一部：初秋三楼
时间：高二元旦前
地点：布置中的教室

## 诗歌插页 P047：原创仿唐绝句

满座歌声逐岁新，
一班灯下各天真。
少年不解寻常意，
只把同台作近邻。

事情真正发生时，并没有陆允川想象得那么郑重。那天晚上全校晚会在礼堂举行。红色的壁灯，台下一排折叠椅，天花板的红色条幅被空调暖风吹得轻轻晃动。理科一班的合唱是第四个上台。陆允川站在后排偏左的位置，按照叶澄教他的那套方法——他把整班当成一个乐队，她是一只中音号。合唱开始的时候声音响起全班一起出——"友谊地久天长"的旧旋律。他在歌声中听见了她的声音——不是单独挑出来的，而是从整体中剥离出一条细线——那线很稳，没有飘，没有颤，比排练时准了一个调。他眼角的余光看见她站在前排右边第四个——头发是用黑皮筋盘起来的，穿了一件深蓝色的晚会针织衫。

他在自己唱到"友谊二字时嘴型变形了一下。程砚在他旁边低声说了句——嘴唇不动只鼓气——"你再走调我就踢你脚后跟。"陆允川收了下颌继续唱，第二节音准恢复了。

合唱结束后全班人往侧台撤。侧台狭窄——挤得所有人站成一条线。夏知遥被挤到他右边一处紧靠着他后面旁排的位置。她的手臂靠着他的背紧在墙边上大概十几秒——她的胳膊肘隔着她的毛衣和羽绒服压在他的背处肩胛角位置。她没有任何反应——只是在等节目结束散场。他的背上肩胛骨一片温烧——是从她那件深蓝色针织衫传过来的体温。那十几秒钟里他没有躲开，也没有故意往那边靠——他站在那里让它自然压下，然后被后面的人一挤松开了。

晚会结束后大家回到教室收拾横幅。教室里撒满了彩带碎片和气球残骸，踩脚底下发出塑料爆裂的声响。陈老师在讲台前说"今天大家表现很好——放一个元旦假三天，回来迎接期末考试"。蒋雯拿着手机到处找人合影——把夏知遥和其他几个女生一起拉到教室前门拍了几张。陆允川站在后排收椅子——把折叠椅摞成二垒。他听见她的声音从前门那边传过来:"我们站在哪面——在横幅下面？对——'恭贺新禧'的光有点晃。"他听着她同女同学们的谈话，心里觉得很平静——演出结束后的平静是那种所有人都松懈后的散场平静。

他收拾好东西，背上书包往门口走。经过前门时正好撞上她——她站在门边拿围巾，围巾的另一端被夹在门把手里。他帮她把围巾从门把手上摘出来，递给她。她说"谢谢"，他说"不客气"。

这时教室里几乎没有人了——蒋雯去了厕所，程砚在外面走廊接家里的电话。夏知遥停下来，忽然问了一个不是"谢谢"的问题:"你觉得今晚节目怎么样？""挺好的。""你觉得我们班唱得齐不齐？""齐——特别是你的那一段很稳。"她突然被点出来低头笑了一下——没有回答——然后把围巾裹上脖子说"元旦快乐"。他说"元旦快乐"。她推开前门走出去了，门扇在她背后玻璃上轻轻地晃了两下，灯映出门外她走远的背影。

陆允川在教室里站了大概十秒钟。他发现自己刚才说了一句"你的那一段很稳"——这是他对她说过的最直接、最主动、最接近于对她个人说的话。而他同时也是在全班歌唱完之后才说出口——他花了整整一个学期才从"你的笔掉了"升级到"你的那段很稳"。这种进步如果说出去大概会被所有人笑——十七岁的人在元旦晚会后的散场教室里站着感慨自己从六个字升级到十二个字——但他确实很骄傲。

他走出教室时程砚追上来:"我刚才在走廊听你说什么'你的那一段很稳'。""对。""那是我听过你对她说的最像人话的一句。""什么意思？""之前你对她说的那些话都像在检查作业——'你捡''谢谢''没事''你这边拖过了'——那句是第一次你在说她这个人。"两个人一起往学校门口走。程砚又说了一句——这次没开玩笑——"元旦快乐老陆，明年你争取说到二十个字。""那要看到时有什么可说的。""肯定有——每天都有值日都有传作业，一天一句话也能凑够两千字。"陆允川把书包甩到肩上，跟着路灯往前走。今晚的月亮是细弯的——细弯但是亮。
