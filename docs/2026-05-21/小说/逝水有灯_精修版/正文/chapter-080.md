# 第080章 毕业照边缘多年后

所属部卷：第二部：六月以后
时间：六月中旬
地点：学校操场主席台前

## 诗歌插页 P080：原创七言小诗

窗过长廊日影深，
一行旧字压衣襟。
少年未解离分意，
先把沉默写到今。

大三那年夏天，陆允川在宿舍整理电脑文件，在一个叫"旧照片"的文件夹里看到了那张毕业照。

文件夹里一共三百多张照片，绝大部分是大学拍的——社团活动、寝室聚餐、期末考试前在图书馆通宵的合影。高中照片只有两张：一张是毕业照，一张是程砚给他拍的单人照。

他点开毕业照。

分辨率不算高，放大到一定程度就糊了。但他还是能找到自己——最后一排最右边，眼睛没有看镜头，校服袖口被风吹起来了一块。旁边程砚脸上憋着笑。再往前看，第二排中间，夏知遥手指停在耳侧，正在别头发。

他把照片缩小，又放大，又缩小。动作和很多年前一模一样。不同的是，现在做这个动作的时候，心跳是平稳的。

他想起那天拍照的细节。摄影师的喇叭声、程砚说摄影师像物理老师引发的那场笑场、快门重拍的两次、前排女生差点被横幅绊倒。还有他自己——快门按下的那一瞬间，他往第二排看了一眼。就一眼。在几百张面孔里，他的目光偏了大概两厘米的角度。两厘米在照片上几乎看不出来，但在他自己的记忆里，那是一个完整的画面：阳光在夏知遥的耳侧亮了一下，她把散下来的头发别过去。

他把照片关了。不是逃避。是看够了。

晚上他给程砚打电话。两个人在聊各自最近的烦心事——程砚厂里换了一个新主管，特别事多。陆允川在设计院画图画到眼睛疼。聊着聊着，程砚忽然说："你还记不记得拍毕业照那天？"

"记得。"

"那天我说摄影师长得像物理老师，你憋笑憋得脸都红了。照片出来以后你的表情特别诡异——又想笑又往下看。"

"你还记得？"

"废话。我拍的。"程砚那边传来打火机的声音——他在点烟，"好多年了啊。你那时候还老想着夏知遥。现在呢？"

"不了。大学都没联系过。"

"一次都没有？"

"一次都没有。"

程砚沉默了片刻。"其实挺可惜的。不是可惜你和她没在一起——那不可惜。可惜的是你这三年把很多时间花在了想一个人身上。想又不行动，想了白想。"

"你那时候不是也让我把题做完再说吗。"

"那是高三。高考之后你可以行动了，但你没行动。"

"因为高考之后我发现，我不行动是对的。"

"怎么说？"

陆允川靠在椅背上，看着宿舍窗外那棵银杏树。树叶正绿着，离秋天还远。"高考之后第二天，我在家里整理试卷，翻到一张高一的英语卷。上面有我用铅笔写过又擦掉的她的姓。"

"你还干过这种事？"

"干过。当时觉得很羞耻。但高考之后重新看到那个铅笔印，我突然觉得——我喜欢的可能不只是夏知遥。我喜欢的是我自己想象出来的那个夏知遥。"

电话那头程砚没说话。

"真正的夏知遥是一个具体的人。她有自己的压力、自己的计划、自己的朋友、自己的高考志愿。她在走廊里跟林曼有说有笑，她在毕业照上笑得紧张但认真，她差点被横幅绊倒被叶澄扶了一把。这些事情都跟我没关系。但我觉得这才是对的。她本来就跟我没关系。"

"你什么时候想明白这点的？"程砚问。

"不知道。不是某一天突然想明白的。是慢慢就不想了。"

程砚把烟掐了。"行吧。比我想的要靠谱。"他清了清嗓子，"那你现在有喜欢的人吗？"

"没有。"

"有喜欢的事吗？"

"画图。把一座桥的结构算清楚，比喜欢一个人简单。"

"废话。桥又不会拒绝你。"

陆允川笑了。他在毕业很多年之后，才学会了用"事情"来形容自己的过去——不是"伤痛"，不是"青春"，不是"遗憾"。是"事情"。发生了、记住了、过去了的事情。

挂了电话以后，他把那张毕业照拖进一个新文件夹，名字叫"高中"。这个文件夹里还有几张照片：县一中老校门的照片、程砚拍的他的单人照、叶澄的英语错题本扫描件。他把文件夹压缩了，存进移动硬盘。

然后他关上电脑，去阳台站了一会儿。大学城里的夜比县城的夜亮得多——到处都是路灯和宿舍楼的灯光，看不到北斗七星。但他知道北斗七星在天上，和很多年前在院子里洗脸时看到的是同一组星。只不过现在看不见了。

那天晚上他做了个梦。梦里回到拍毕业照的操场，他还是站在最右边，风把袖口吹起来。快门响之前的一瞬间，他回头看了一眼——不是看第二排，是看整个操场。主席台、横幅、蓝色凳子、站在他旁边的程砚、前面几排的叶澄和林曼和夏知遥、还有远处器材室墙上那道被雨淋出来的水渍。

快门响了。他醒了。

窗外的银杏树在晨风里抖了一下，一片叶子都没掉。
