# 第054章 高三倒计时旁观

所属部卷：第一部：初秋三楼
时间：高三九月
地点：教室前排

## 诗歌插页 P054：原创现代诗

她也看见那个数字，
每天少一。
别人以为她稳定，
她知道稳定是用很多慌
一点点压出来的。
黑板不问谁喜欢谁，
只问今天有没有背完。

夏知遥讨厌别人说她“肯定没问题”。

这句话听起来像夸奖，其实很重。好像只要她考不好，就是辜负了大家对稳定的想象。高三开始后，亲戚、老师、同学都爱这么说。她每次只能笑笑，回到座位后把错题本摊得更开。

黑板右上角的数字每天少一，她比谁都清楚。她的目标学校在省城，分数不低。母亲说尽力就好，可电话里那句“别太累”后面总藏着没说出的希望。

有一次午休，她实在太困，趴在桌上睡着了。醒来时，数学卷子还压在手下，最后一道题的辅助线画歪了。林曼说刚才有人从旁边经过，脚步很轻。

“谁？”夏知遥问。

“不知道，可能交作业的。”林曼说。

夏知遥没有追问。高三里每个人都在赶路，路过就是路过，不一定有含义。她越来越不想把同学之间的细节解释得太多。解释会耗力，而她的力气要留给题。

晚自习前，她去黑板前帮值日生改倒计时。粉笔在指间断了一截，白灰沾到指甲缝里。她写下 278，退后看了看，又把 8 的下半圈补圆。

陆允川坐在后排低头做题，没有看她。

这个发现让她稍微轻松。不是因为她讨厌被看见，而是因为一个人终于把目光放回自己的卷子上，对彼此都好。

她回到座位，翻开英语作文模板，在第一行写下：keep moving。

写完后，她觉得这两个英文词很普通，却很适合高三。继续走。不要把每一道目光都背在身上，也不要把每一次疲惫都说成失败。
