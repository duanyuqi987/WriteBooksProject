# 第207章 毕业照里的空位

所属部卷：第四部：失去的美丽
时间：大四六月
地点：大学图书馆前台阶

## 诗歌插页 P207：原创仿唐绝句

松线微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

事情落到当天，没有被命运单独照亮。

班级合影拍完之后，是自由合照时间。图书馆台阶变成了一个开放式的摄影棚，到处是举着手机蹲着找角度的人。女生们三三两两挽着手，男生们勾着肩，快门声此起彼伏。

陆允川站在台阶右边缘，学士服搭在胳膊上，正把袖口那根烧断的线头捻干净。夏知遥去了西草坪找叶澄，祁野被同寝室的人拉去拍寝室合影。他一个人站在那儿，太阳把学士帽下面的头发晒得发烫。

苏晚枝从台阶左侧走上来。她不是来找他的——她在找自己班的人。两个人在第三排和第四排台阶之间，挡不住彼此的视线。

她先看见了他。

两个人都没有刻意靠近也没有刻意避开。中间隔着一排同学，有人在换镜头，有人在补口红，有人蹲在地上捡掉落的帽穗。

苏晚枝穿学士服的样子和平时不太一样。学士服宽宽大大，把她整个人罩得只露出脸和一小截小腿。她的帽穗拨到了右边——已经有人帮她拨过了。

陆允川不知道该不该上前。一个星期前他们在面馆把该说的话都说完了，现在他站在毕业照的台阶上，面对一个已经道过歉也接受了道歉的人，突然不知道该说什么开场白。

苏晚枝替他开了。"你站那么边上干嘛？"

"凉快。"

"这里是太阳最晒的位置。"

"我帽子厚。"

苏晚枝笑了一下。不是"算了"的笑，是"你还是一样"的笑。

"你那根线呢？"她指了指他的袖子。

"烧掉了。夏知遥借的打火机。"

"夏知遥随身带打火机？"

"她说点蜡烛用的。"

"什么蜡烛？"

"我没问。"

旁边一个女生跑过来拉苏晚枝："晚枝，我们班那边开始拍了，快来！"

"等会儿。"苏晚枝说，"一分钟。"

女生看了陆允川一眼，识趣地先跑了。

"你等下有空吗？"苏晚枝问。

"有。大把。"

"那拍完班级合照，过来跟我们班合张影。就站在后排随便哪个位置。别挑。别想。"

陆允川愣了一下。"你确定？"

"没什么确定不确定的。"她说，"毕业照又不是结婚照。多一个人少一个人，过十年谁还记得。"

"好。"

"行。那我过去了。记得站后排，右边第三个，最不起眼的位置。"

她转身往自己班级的方向跑，学士服下摆扬起来，像一个不合身的披风。跑了三步又回头："对了，拍完别马上走。我介绍你认识一下我室友。"

"你室友？"

"对。她总问我那个写了两百封信的男生长什么样。"

陆允川还没来得及回答，她已经消失在一群穿学士服的人中间了。

祁野不知道什么时候回来了，手里多了两瓶水。他把一瓶递给陆允川："刚才那个是苏晚枝？"

"对。"

"她跟你说什么？"

"让我去跟她们班合影。"

祁野拧瓶盖的动作停了一下。"那你答应了？"

"答应了。"

"牛逼。"今天的第三次。

"你现在说这两个字的频率越来越高了。"

"因为毕业了，所有不合逻辑的事都能用'牛逼'一笔带过。"祁野喝了口水，"你要不要先去买瓶水洗把脸？你额头上全是汗。"

陆允川去了图书馆一楼的洗手间。镜子里的自己穿着白衬衫，学士服脱了搭在胳膊上，领口扣子松了一颗。他扣紧，洗了把脸，用纸巾擦干。镜子上方有一行小字：节约用水。

他走出去的时候，苏晚枝班级的合照已经快拍完了。他按照她说的，站到后排右边第三个。旁边一个戴眼镜的男生问他哪个班的，他说物管的。眼镜男生说哦，二班的吧。他说嗯。

快门响了。这一次他没有捏袖口。

拍完之后，苏晚枝拉着他袖子走到台阶旁边。一个扎马尾的女生站在那里，手里端着杯奶茶。

"这就是我室友，鲁萱。"

"你好。"鲁萱上下打量了他一遍，"你就是那个陆允川？"

"是我。"

"看着不像是会写两百封信的人。"

"那你觉得会写两百封信的人长什么样？"

鲁萱想了想。"戴眼镜？穿深色衣服？眼神比较忧郁？"

"我今天没戴隐形。"

"啊？"

"所以也算戴眼镜。"

苏晚枝在旁边笑了。鲁萱说："你比我想象的有意思。晚枝说的那个沉默深情的版本是假的吧？"

"沉默是真的。"陆允川说，"深情不好说。"

"为什么不好说？"

"因为后来发现，深情和偏执之间的差别，大概就是一个人的事还是两个人的事。"

鲁萱喝了口奶茶，看他的眼神变认真了一点。"行。这句话像是个会写信的人说的。"

苏晚枝拉了拉陆允川的袖子。"你等下聚餐在哪？"

"北门蜀味轩。六点。"

"那还有两小时。"

"你要干嘛？"

"校门口那个打印店还开着。你把这两百章的书稿打一份给我。"

"现在？两百章？"

"不是全部。就打印面馆那一章。"

陆允川看着她。"你要那个干嘛？"

"留个底。"苏晚枝说，"万一以后你想抵赖。你说你在书里把我写成了一个人——我要自己验证。"

他答应了。

打印店的小哥看见两百多页的文档时问了一句"毕业论文吗"。陆允川说"不是"。小哥没再追问，调好边距开始打。

激光打印机一张一张吐出来，带着温度和油墨的味道。苏晚枝站在旁边，第一页出来的时候她拿起来看了一眼，读到前两句就放下了。"我回去看。在这里哭出来太丢人了。"

"会哭？"

"不知道。以防万一。"

两百多页打完了。陆允川付了钱，把纸摞整齐递给她。挺厚的一沓，比毕业论文还厚。

"走了。"她说，"聚餐别喝太多。你酒量不好。"

"你知道我酒量不好？"

"大二那次班级聚会，你喝了三杯啤酒就趴在桌上睡了两个小时。我在旁边那桌。"

陆允川想了想。"你当时在旁边？"

"在。你趴下的时候把醋碟打翻了，醋洒了一桌。喻可擦了五分钟。"

"你怎么不叫我？"

"因为我觉得你睡着的时候比醒着的时候安静。"她把稿子抱在胸前，"走了。照片我发你。"

她往宿舍方向走了。太阳已经西斜，影子拉得很长，和台阶的阴影混在一起。

陆允川在原地站了一会儿，直到手机响了。程砚问他在哪，说聚餐的菜单要提前定，问他有没有忌口。他说没有。程砚说好，然后挂了。

他把手机放回口袋。学士服的袖口那根线头已经不在了，但烧过的痕迹还在——一小截稍微发硬的纤维，摸上去和其他部分不太一样。

有人喊他名字。是班上同学，叫他过去拍小组合照。

他走过去。这一次不用站最边上。中间偏左的位置刚好空着。
