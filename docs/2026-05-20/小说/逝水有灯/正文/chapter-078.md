# 第078章 毕业照边缘之后

所属部卷：第二部：六月以后
时间：六月中旬
地点：学校打印店门口

## 诗歌插页 P078：原创仿宋小令

照初成，日偏长，
边角人影带微光。
问少年、几番放大，几番收藏。
笑声散在打印店，
一纸青春各自忙。

毕业照电子版先传到班级群里。

那时网速慢，图片一点点刷出来，先是主席台，再是前排老师，最后才出现站在后排的人。陆允川坐在打印店门口的塑料凳上，盯着手机屏幕等了很久。

照片加载完后，他第一眼找夏知遥。她在第二排，表情很稳。然后他找自己，在最边上，脸很小，旁边是笑得过分用力的程砚。

程砚凑过来看：“我这张不错。”

“你挡到我了。”

“你站边上还能怪我？”

陆允川把照片放大。放大以后，画面变糊，他和夏知遥之间隔着许多同学、老师和夏天的光。他盯了一会儿，忽然觉得自己的动作很傻。照片不是地图，不能因为放大就缩短距离。

打印店老板喊：“洗照片的过来登记尺寸！”

大家一窝蜂挤进去。陆允川也洗了一张全班照，又多洗了那张和程砚的合照。程砚付钱时抢着掏零钱，说以后发达了这张能卖高价。

“卖给谁？”

“卖给未来的你，让你看看我年轻时多有精神。”

陆允川笑了。他把照片装进薄薄的纸袋里，纸袋边缘很快被汗洇软。

回家路上，他没有再放大那张全班照。手机屏幕黑下去后，他才发现手指有点酸。原来一个人用力看一张照片，也会累。
