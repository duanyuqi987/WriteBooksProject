# 第082章 十二公里

时间：2013年9月

地点：楚州师院

卷次：卷五：各自南北

开学第一周，他做了一件事。

用宿舍里阿坤的电脑查了公交路线。

阿坤的笔记本是联想牌的，屏幕有点小。他搬了凳子坐在阿坤的桌子前面，打开浏览器。输入框里打了八个字——楚州师院到楚州师大。

回车。

页面跳出来一条结果。301路。11站。大约四十分钟。

起始站：师院南门。

终点站：师大北门。

中间九站——市图书馆、市体育馆、中山路口、公园西门、工商银行、人民医院、新华书店、广场东路、师范大学。

他把十一站的站名一个一个记在手机备忘录里。按了七八下键。手机键盘很小，他的拇指有点大，按错了三次，删了重新打。

打完以后他又看了一遍。

301路。师院南门到师大北门。全程大约12公里。

他盯着屏幕上的数字看了十几秒钟。

然后他又搜了一个页面——楚州公交线路图。地图是绿色的底色，公交路线用不同颜色的线标出来。301路是蓝色的——一条蓝色的线从地图左下角弯弯曲曲地往右上角延伸，穿过楚河上的中山桥，拐进老城区，最后停在标注"楚州师大"的红点旁边。他把地图放大了。师院南门是一个蓝色的小圆圈。师大北门是另一个蓝色的小圆圈。两个圆圈之间——是那条蓝色的线。

他看了大概五分钟。

阿坤从上铺伸头过来。"找什么路子呢？想出去逛？"

"随便看看。"

他把浏览器关了。把电脑还给阿坤。站起来，走到窗边。

窗外那栋远处的宿舍楼还是灰色的。天空有点阴。楼下有人在晒被子，用一根竹竿拍打着被芯。啪啪的声音隔着玻璃传进来，闷闷的。

他拿出手机。翻到备忘录——301路、11站、12公里。

然后把手机合上，放在桌上。

下午他去上了第一节专业课——现代汉语。教室在三楼，能坐一百多人。他坐在靠窗的第三排。老师姓刘，五十多岁，说话很慢。阳光从窗户斜进来，在课桌上切出一道明暗分界。他把笔记本放在明的那半边。

"现代汉语——研究的是我们每天都在用的语言。"

他在笔记本上写下"现代汉语"。然后在本子右下角画了一个圈。又在圈外面画了一条线。

下课的时候他收拾书包。经过学校南门。南门有两扇铁栅栏，门口对面就是公交站。

他把脚步放慢了。

站牌是绿色的铁牌子，上面印着线路图。301路排在第三行。站牌的杆子底下锈了一截——大概是冬天撒了融雪剂。站牌顶端落了鸟粪——白色的，干了。

他站在站牌前面。逐个看那个站名——市图书馆、市体育馆、中山路口、公园西门、工商银行、人民医院、新华书店、广场东路、师范大学。每一个名字对应一个他没去过的地方。市体育馆——大概是圆形的建筑，顶上有个大钟。人民医院——他在楚州的第一天就看到过，白色的大楼，楼顶有个霓虹十字架。新华书店——不知道跟青阳那家是不是连锁的。师范大学——倒数第二站，离终点站只差一站。

师大北门。终点站。

他伸出手摸了一下站牌的边沿。铁皮是热的——被太阳晒了一下午。

一辆公交车开过来了。蓝色的车身，车头贴着红字——"301"。前门打开。车身的引擎嗡嗡地响着。排气管冒出一股黑烟。车上下来三四个人——一个提菜的阿姨、两个背书包的学生、一个夹着公文包的中年人。又上去五六个人。前门关上。车发动。往东边开走了。

往师大方向开走了。

他没有上车。

他站在站牌旁边看了大概两分钟。等第二辆301路开过去——这一辆人少，车窗里只有两三个乘客。他看到靠窗的位置空着好几个。然后转身，走进了校门。

回了宿舍。

李博在整理选课表。"你选修课选了吗？"

"还没。"

"抓紧。有的课名额满了。对了——你想选什么方向的？"

"都行。"

李博看了他一眼。"都行是什么意思。"

"什么有意思就选什么。"

他选了最多的选修课。中国古代文学、西方哲学导论、书法基础、中国历史地理。周一到周五每天都有课，周六上午也有两节。选修课比必修课多了一倍。

李博看了他的选课表。"你疯了吧。选这么多。"

"闲着也是闲着。"

晚上熄灯以后，室友们在聊高中的事。阿坤说他高中翻墙出去上网被逮过三次。小伍说他高中班花长得像刘亦菲。李博说他高中是班长兼学生会主席。

顾临风躺着没说话。

"临风你呢？你们高中什么样？"

"普通的。"

"有没有什么特别的事？"

"没有。"

他把枕头翻了一面。枕套凉凉的。手机压在枕头下面。屏幕应该是黑的。

天花板上那条裂缝还在。他看了几分钟。闭上了眼睛。

301路车的路线在他脑子里过了一遍——师院南门、市图书馆、市体育馆、中山路口、公园西门、工商银行、人民医院、新华书店、广场东路、师范大学、师大北门。

每站之间大概三四分钟。离开师院南门后，第一个站是市图书馆。图书馆门口有个喷泉——他在网上搜过图片，喷泉是圆形的，中间立着一个不锈钢雕塑。公交车会停在喷泉正对面。上车的人里有学生模样的——大概是去师大的。到市体育馆的时候，车会绕着体育馆的转盘转半圈。到中山路口的时候会经过一座桥——楚河上的中山桥。桥下的水是绿的。到公园西门的时候会在一个十字路口等红灯。红灯大概四十秒。到工商银行的时候路边有一排小吃摊——煎饼果子、烤面筋、炸串。到人民医院的时候有人推着轮椅上下车——司机把车身降下来，翻出踏板。到新华书店的时候门面是红色的——跟青阳不一样，青阳的是蓝色。到广场东路的时候路两边种着银杏树——秋天会变黄。到师范大学的时候——那是倒数第二站，离师大北门只差一站。师大北门是终点站。下了车往右拐——就是校门。校门是灰白色的。

他把路线背下来了。闭着眼睛都能数。

十一站。十二公里。四十分钟。

有时候他会在脑子里坐一遍这趟车。买票——投币两块钱。坐在靠窗的位置。右手边的窗外会依次掠过那些站名对应的建筑。每过一站，车上的人少几个。到师大北门的时候——车停了。下车。站在师大校门口。然后呢。

然后没有然后。

他睁开眼睛。天花板上的裂缝还是那条裂缝。

第二天下午他又经过了南门的公交站。站牌下面站着三个等车的人——一个拎着菜的老太太、一个戴着耳机的中年人、一个抱着书的女生。女生的马尾扎得有点松，几根碎头发落在脖子后面。他看了那个马尾一眼。然后移开了视线。301路来了。三个人都上去了。车开走。

他又站了几分钟。第二辆301路开过去的时候，他数了车上的人数——大概十二三个。有人拉着扶手站着。有人靠窗坐。靠窗的那个位置——第三个窗户——坐着一个扎马尾的女生。不是她。他把视线移回来。

他转身进了校门。门口的保安看了他一眼——大概是认出他了。他在门口站太多次了。

保安没说话。他也没说话。

走回宿舍的路上，他在心里把那十一个站名又默了一遍。没有漏。没有错。还是那十一个站名。

十二公里。他没有去过一次。

他遵守了约定。
