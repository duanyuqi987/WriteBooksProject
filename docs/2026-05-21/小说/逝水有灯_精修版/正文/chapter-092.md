# 第092章 夏天的聚餐

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P092：原创仿唐绝句

风色微明照旧尘，
少年别后识前因。
一声铃尽人潮散，
半页风光不问春。

菜上到第四道的时候，话题从大学转到了高中。

"你们还记不记得高二那次运动会？"程砚夹了一块回锅肉，筷子上全是红油，"周长海跑接力赛最后一棒，眼看就要第一了，结果他跑错道了。"

"我没跑错！隔壁班的抢了我内道！"周长海抗议。

"行行行。反正最后是第四名。体委哭了一整节课。"

"体委哭是因为摔伤了膝盖，不是因为成绩。"叶澄纠正。

"你记得也太清楚了。"

"我是运动会记录员。所有成绩都要记。"

圆桌中间的玻璃转盘被人转了一圈，菜盘子从这头转到那头。回锅肉、水煮鱼、麻婆豆腐、炒青菜、一盆酸菜粉丝汤——汤面上浮着几颗花椒，有人舀汤的时候没注意，喝到了麻椒籽，表情瞬间变了形。

陆允川坐在程砚旁边，默默吃菜。桌上聊到了夏知遥。是林曼先提起来的——她说夏知遥今天不能来是因为她妈那边的亲戚从深圳过来了，说要给她庆功。她爸也从深圳打了电话回来，说了很多话。

"她爸很久没联系她了？"叶澄问。

"差不多两年。中间就过年打过一次电话。但这次考上大学，他突然很热心。说让她研究生考到深圳去，深圳机会多。"林曼放下筷子，"知遥说他还记不住她具体读的是什么专业。说是建筑设计，他说成'建筑工人'。"

一桌人安静了一瞬。

"她爸真这么说？"程砚皱起眉头。

"真的。知遥跟我说的时候在笑，但我听着一点都不好笑。"

陆允川放在筷子上的手指动了一下。然后他夹了一块鱼，低头挑刺。

"她不在乎她爸怎么说。"杜雨薇忽然开口，声音不大但很坚定，"她跟我说过——他想说什么就说什么，但她的路她自己走。"杜雨薇今天穿了一条碎花裙子，说话的时候筷子还举在半空中。

"你怎么知道？"林曼问。

"因为我也问过她类似的问题。她说她爸对她的了解，停在她十岁那年。但问题是她现在已经十八岁了。十年之间的空白，不是打几个电话就能填回来的。"杜雨薇把筷子放下，"她说这些话的时候语气很平，像是在陈述事实，不是在抱怨。"

叶澄喝了一口水，没有接话。但她的表情里有一种无声的认可——她知道杜雨薇说的是真的。那次在楼梯口她帮夏知遥拉书包拉链时，看到的那行字——"想考去一个有江的城市"——或许不是为了浪漫，而是为了远离一些需要远离的东西。

程砚举起茶杯。"那为她说一句——前程似锦。"

全桌人都举起了杯子。玻璃转盘停了。酸菜粉丝汤里的热气慢慢地升起来，把所有人的脸都罩了一层薄薄的白雾。

陆允川碰了杯，喝了茶。然后他放下杯子，把筷子拿起来，继续挑鱼刺。这次挑得很仔细——一根一根，全部挑干净。然后他把那块鱼肉放进程砚碗里。"你刚才一直说想吃鱼。"

程砚低头看了看碗里的鱼肉。"你把刺都挑了？"

"嗯。"

"靠。你这样我都不好意思损你了。"

"那你别损。"

"不行。我忍不住。你这人就是太好了——对谁都好。你要是坏一点，我还能多骂你两句。"程砚把鱼肉塞进嘴里，含含糊糊地说，"好吃。你自己也吃。"

陆允川夹了一块鱼放到自己碗里。这次没有挑刺，直接吃了。
