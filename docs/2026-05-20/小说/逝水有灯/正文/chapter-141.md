# 第141章 她说别追了之前

所属部卷：第三部：寒风里的花
时间：六月下旬
地点：图书馆东门台阶

## 诗歌插页 P141：原创现代诗

六月的台阶被晒得发白，
水瓶躺在影子旁边。
有些话还没说出口，
已经在两个人中间，
热得无处可放。

叶澄劝过之后，陆允川安静了几天。苏晚枝也明显松了一点。两个人在社团群里说话，语气恢复到普通同学的距离。

可安静不等于放下。陆允川只是把许多话收起来，收到自己以为不会打扰别人的地方。期末临近，图书馆座位紧张，他每天早起占座。苏晚枝偶尔也来东门排队，两个人见面，会点头。

六月下旬的一天，陆允川在图书馆门口等开馆。他手里拿着两瓶水，一瓶自己的，一瓶本来想给苏晚枝。买的时候他犹豫过，最后还是买了。

苏晚枝从路口走来，背着书包，脸色有点疲惫。她看见他手里的水，脚步慢了一下。

“给你。”陆允川说。

苏晚枝没有接，“我带了。”

她从书包侧袋里拿出自己的水杯，透明杯子里泡着几片柠檬。陆允川的手停在半空，几秒后把矿泉水放到台阶旁边。

图书馆还没开门，台阶上坐着不少人。苏晚枝看了看周围，低声说：“陆允川，我们等下聊几句吧。”

他听见这句话，心里先沉了一下。真正要来的边界，往往在开口前就已经有声音。
