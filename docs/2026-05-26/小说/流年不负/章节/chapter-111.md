# 第111章 等成绩

时间：2017年2月

地点：青阳县

卷次：卷六：平行轨道

二月。寒假。青阳。

考研成绩要二月中旬才出。等待的这段时间——每天早上醒来第一件事是摸手机，看有没有成绩公布的推送。没有。然后放下手机。翻个身。再睡二十分钟。然后起床。日子被拉得又长又细。像冬天窗玻璃上往下淌的冷凝水——缓慢的，看不出来在动的，但其实一直在往下滑。

顾临风回了家。他家住在青阳县机械厂家属院，五楼的两居室。房子是九十年代的——楼道里的墙皮有些地方鼓了包。出了电梯——不，没有电梯。楼梯间是水泥的，扶手的绿漆被摸得发亮。他妈把他高中那间屋子收拾了一遍——换了新床单，擦过桌子，窗帘也换了。深蓝色，拉上的时候屋里很暗。暗得像洗照片的暗房——只有窗帘边透进来一线白光。书桌还是那张——三合板的，右边桌腿有点晃，他高中时在底下垫了一张折了四折的纸。纸还在。

早上他睡到九点多。起来吃一碗稀饭配咸菜——他妈腌的萝卜条，切得粗粗的，咸得下饭。然后翻翻手机——朋友圈滑了几页。再躺回去。被子里的热气还没散，重新钻进去的时候整个人缩成一团。

他妈在客厅打电话，跟张阿姨聊了一个多钟头。电话线从电视柜那里拉到茶几旁边——他妈坐在沙发上，一边讲电话一边织毛衣。谈话内容他听不清楚——断断续续地飘进来的几个词："也是""还没呢""考研""你说这孩子的性格——""那姑娘挺好的""我也急"。声音隔着两扇门——一扇妈妈的卧室门，一扇他房间的门。两扇都是木头的——空心板，隔音不好。但他不想听清。

下午他去县城转了转。羽绒服拉链拉到下巴——风还是能找到缝往里钻。

青阳县这些年变了不少。主干道拓宽了——原来的四车道变成了六车道，路边新装了护栏。路两边新开了好几家店——一家连锁快餐、一家奶茶店、一家"数码维修"。那个修鞋的大爷还在老地方——坐在马扎上，面前摆着个木箱子。箱子里的东西还是那些——钉子、锤子、碎皮、一管黄色的万能胶。大爷老了很多。头发全白了。脸上的皱纹比以前深了一倍——尤其是鼻翼到嘴角那两道。他正在修一只女式皮鞋——鞋跟断了，他往断面涂上胶水，用力按紧。按着大概过了三十秒。送到箱子左边——那里已经摆了五六只修好的鞋。

新华书店没变。门脸还是那个门脸。门楣上的红色大字褪色了——"新华书店"四个字，其中"华"的最后一竖掉了一块漆。门口的台阶磨得发亮——青石板上的纹路已经被踩平了。门把手是铜的——被握得发亮。他在门外站了一会儿。透过玻璃门能看到里面——收银台换了新人，一个大姐在低头看手机。没进去。

往前走。经过那条有老槐树的巷子。

树还在。冬天的槐树光秃秃的。枝干伸向灰色的天。那些枝干上有几个鸟窝——空巢，黑色的，树枝搭成的。巷子口的奶茶店关了——门上贴了张纸。"旺铺招租。联系电话138xxxx。"纸被雨水泡过，边角卷起来，字迹模糊了一半。旁边那家包子铺还在。门口的蒸笼冒着白汽——一层一层的竹笼屉堆得比人还高。包子铺老板还是那个光头大爷。他戴着白围裙——围裙上沾了面粉——拿着长筷子往蒸笼里夹包子。

他在巷子口站着。脚踩着人行道上的红砖。有一块砖松了——踩上去会晃一下。他把脚挪开了。

风吹过来。巷子底传来什么东西被风卷起来的声音。沙沙的。大概是落叶。或者是塑料袋。

有人从巷子里走出来。一个中年女人——个子不高，穿深蓝色羽绒服，羽绒服的袖子有点长，盖住了大半只手掌。手里提着一袋子菜——塑料袋，半透明的，能看到里面有芹菜、土豆、一把葱。葱叶子从袋口伸出来，被风吹得乱晃。

是苏晚晴的妈妈。

他认出来了——走路的姿态没变。身子往前微微倾着。脚步节奏很快——小碎步的。是那种常年操持家务的女人走路的方式。高二那年他在新华书店门口见过一次，苏晚晴跟她走在一起。母女俩走路的姿势很像——都是稍微内八。当时苏晚晴提着一个袋子跟在她后面——袋子里大概是新买的参考书。

苏妈妈在巷子口拐了个弯，往东边走了。那边是菜市场。她大概还要去买什么——豆腐或者鸡蛋。她的皮鞋踩在水泥地上发出细密的响声。

他没动。胳膊垂在两侧。手指在口袋里攥着。

过了一会儿。巷子里又有脚步声。这次不用看——他知道是谁。

白色羽绒服。马尾。

苏晚晴。

她手里提着一个超市塑料袋，上面印着"永辉超市"——青阳县三年前开的那家。里面装了牛奶——一升装的纯牛奶，白色盒子；一袋切片面包；几袋速冻水饺——猪肉白菜馅的。袋子有点重，她的肩膀微微往那边斜。她站在巷子口——离他大概二十米——低头在塑料袋里摸了一下。大概是在确认钱包。摸了两下。确认了。抬头。

然后她拐进了老槐树巷子。从巷子口往里走了几步——走过那棵老槐树的时候低头躲了一下低垂的树枝。白色的衣服擦过灰黑色的树皮。

隔着一条街。马路上没有车。冬天的下午，小县城很安静。安静到能听见对面包子铺的笼屉被揭开——水蒸气嗤地一声。能听见她的鞋子踩在巷子里的石板地上——轻轻的，一下一下的。

十四步。

他从街这边看过去。她的马尾比照片上低了一点——不是扎高了，是普通的高度。白色羽绒服领子立起来——领子上的毛毛有点乱。耳朵露在外面——冻得有一点红。耳垂比高中时候大了一点——大概是冬天的原因。或者是他记错了。那时候他没有那么多次盯着她的耳朵看。她走路的速度不快——不急。手里提着的塑料袋晃一下晃一下的。面包大概在里面滚了一下——她用手稳住了袋子。

她在巷子里走了大概三十米——经过那盆铁皮文竹。弯腰把文竹盆边的落叶捡起来——两片黄的。放进旁边的小垃圾桶。直起腰。拐了个弯。进去了。那个弯后面是第三家。门口有春联——去年的，红纸褪成了粉色。"吉祥如意"四个字还能看清。"意"字的心字底掉了半边。那是她家。

他站在马路对面。没有任何动作。没有喊"苏晚晴"。没有跑过去。没有从口袋里摸出手机。没有挥手。没有咳嗽。没有往前迈一步。

只是站着。像路边的那棵槐树——光秃秃地站着。唯一动的只有被风吹起来的围巾角。

公交站广告牌在身后。上面贴着某品牌饮料的广告——一个穿红裙子的女人举着玻璃杯笑。女人的牙很白。广告右下角被撕了一块——大概是有人想记电话——撕得不齐。

风钻进他的领口。后颈起了一层鸡皮疙瘩。他把围巾紧了紧——围巾绕到第三圈的时候毛线抽丝了。他用手指把那根丝扯断了。

她拐进去的那个弯——巷子里第三家。门口有一盆铁皮文竹。花盆是土褐色的，盆底垫了一个塑料托盘。那大概就是她家。她家的门是铁皮的——刷了暗红色的漆。门上有个猫眼。窗台上放着两个空花盆。她大概在里面换鞋。把拖鞋从鞋架上取下来——白色的拖鞋，跟高一的同款。她妈在厨房喊她——"牛奶放冰箱里——""知道啦——"脱下来的白色羽绒服挂在门后。

他站在街这边。看着那盆文竹。竹叶黄了两片——刚才已经被她摘掉了。现在盆里只剩绿的。绿的叶子在冬天的风里微微发抖。

十四步。

他想——如果他走过马路。走向那条巷子。脚踩在那块松了的红砖上——晃一下。然后敲开那扇暗红色的铁皮门。她会有什么反应。门开了——她站在门口。穿着一件灰色的家居毛衣。手里还攥着刚从冰箱里拿出来的牛奶盒。脸上的表情——先是"谁啊"的茫然，然后是"——顾临风？"的不确定。然后——然后可能是一种说不出话的沉默。她妈从厨房探出头来——"谁呀——"

"阿姨你好。我是顾临风。是苏晚晴的同学。高中同学。青阳一中。2010级。2013届。就坐在她斜后方——倒数第三排。"

然后呢。她妈大概说"哦——是同学啊——进来坐——"。她呢——她大概愣在那里。手里攥着牛奶盒。嘴唇动了动——可能是"你怎么来了"。可能是"好久——"。可能什么也没说。只是看着他。

然后他坐下来。接过她妈倒的水。看着茶几上放着那盆文竹的亲戚——另一盆文竹。墙上挂着的全家福——她爸她妈和她，她穿着高中校服。茶几底下放着几本教科书——三年级的语文课本。

然后他们开始说话。说了什么——不知道。说了这八年。或者没有。或者只是说了句"好久不见"——然后发现这句话后面没有准备好的下一句。

他笑了。很轻。笑自己还跟高中时一样——把所有可能性在脑子里排练一遍。然后永远选最不可能的那种。

他转身走了。帆布鞋踩在红砖上——那块松了的砖又晃了一下。

往家的方向。往南。南边是机械厂家属院。路上经过一家卖煤的——黑黑的煤块堆在墙角，上面盖了一层塑料布。经过一个五金店——店门口挂着几圈电线。经过一棵被锯掉一半的法桐——剩下半截树干上长出了几根新枝。

经过新华书店的时候停了一下。书店门口贴了张海报——"寒假特惠，全场八折"。海报右下角有店长的手写——"寒假没有结束——你可以继续假装在学习。"下面画了个笑脸。这笔字跟三年前一样——还是那支记号笔。墨水的颜色没变。

他看了两秒。然后继续走。口袋里的手机震了一下——李博发了条微信。"成绩快出了吧？紧张不。"他回："还行。"把手机放回口袋。

回到家的时候天色已经暗了。冬天的天黑得早。五点半——窗外已经能看到远处路灯的轮廓亮了。他妈还在厨房。炒菜的声音嗞啦啦的——蒜苗下锅碰到热油的那一瞬间爆出一声脆响。屋里有油烟味——排风扇开到最大，嗡嗡转着。他换了鞋。拖鞋在脚底下硬邦邦的——鞋底被暖气烤干了。把羽绒服挂进衣柜。衣柜里没几件衣服——他的衣服都在楚州宿舍里。在床边坐下。床单是新的——深蓝色。他用手摸了摸。棉布发凉。

电话响了。他妈接的。话筒拿起来的时候电话线在桌腿上弹了一下。

"喂——张阿姨。嗯——在家呢。挺好的。嗯——你上次说的那个姑娘——是——叫什么——哦——苏晚晴——"

顾临风把耳机从桌上拿起来。塞进耳朵。手机的播放列表打开——第一首就是孙燕姿的《开始懂了》。点了播放。声音调大。前奏的钢琴和鼓点一起冲进耳膜——压过了客厅的电话声。歌词唱到"我竟然没有调头——"。他把声音调大了一格。再大了一格。

窗外青阳县城的路灯亮起来。他在这里过了十七个寒假。这是第十八个。街景几乎一样——路灯还是那几盏路灯——里面是高压钠灯，黄澄澄的。电线还是那几根电线——从两根电线杆之间垂着。广告牌换了几茬——这次是新能源汽车的。唯一不同的是她刚才走进巷子里的背影。跟十七岁那年一样——也是白色的衣服。也是马尾。也是隔着一条街。

如果那年他走过去了。如果那年他敲了那扇暗红色的铁皮门。如果那年他说了"嗨，苏晚晴"。如果他大一开学那天在楚州师大校门口等她。如果他点过那个赞。如果他考完之后往右拐而不是往左。

但是没有如果。只有十四步。十四步他从来没走过。这辈子大概也不会走了。

他翻了个身。把被子拉过来。被子是新的——他妈刚缝的。蒙住头。黑暗和棉布的味道一起把他裹住。耳机里的歌还在唱。

那些没有喊出来的句子又在脑子里打转。像抽屉里那封没寄出去的信。像高三那年空白的草稿纸——正面密密麻麻写的是《论语》的翻译题，反面只有一个"S"。像朋友圈里那组照片——第四张：白衬衫、马尾、笑。没有按下去的赞。

他闭上了眼睛。眼皮后面是黑暗。黑暗里有一双白色的帆布运动鞋踩进巷子。鞋底在石板地上留下的声音——轻轻的一下、一下。她进去了。拐了个弯。不见了。

客厅的电话还在说。张阿姨的声音透过门缝传进来——"苏晚晴——那姑娘可好了——当老师——长得好——性格也好——你儿子肯定喜欢——"

他听到了。"苏晚晴"三个字像掉进水里的石子——水面只动了一秒，然后恢复平静。但石子永远沉在底下。

耳机里的歌放到了副歌。孙燕姿唱："不能回头——怎么回头——"

他翻了个身。膝盖撞到墙。骨头跟砖头撞在一起的时候——疼。疼得眼睛闭得更紧了。疼得像一个句号。
