# 第042章 诗抄本

所属部卷：第一部：初秋三楼
时间：高二冬天
地点：男生宿舍下铺

## 诗歌插页 P042：原创仿唐绝句

夜雪未成窗已寒，
小灯照纸字微残。
不题名字题风月，
风月偏知少年难。

诗抄本第一页，陆允川写得很端正。

他先写日期，再写天气：阴，北风。然后抄了几句课本里的诗，抄到第三行时，忽然觉得别人的句子太稳，稳得不像自己的心。他在下面空了一行，写了自己的四句。

楼外寒枝压夜灯，
书声深处有微冰。
不知谁把冬天过，
一页风来一页疼。

写完以后，他盯着“疼”字看了很久，觉得太直白，想改，又不知道改成什么。最后他用笔在旁边画了一个小点，算是暂时放过。

程砚下床倒水，路过时瞥见：“你这不是抄诗，是自己给自己添堵。”

陆允川把本子往里挪：“你别看。”

“我不看也知道。”程砚压低声音，“你别把诗写成未发送消息。写诗可以，别写完就等世界回你。”

陆允川没说话。

这句话他听懂了。诗抄本里每一个没写出的名字，都像一条没发出去的消息。它看似安静，其实一直等待某种回音。可他又舍不得不写。不写，心里就像被冬天的风塞满；写了，至少能让那些风从笔尖漏出去一点。

熄灯后，他躲在被子里用小手电补了两行：

我把名字藏进风里，
风没有替我告诉谁。

手电光很弱，照得纸面发黄。陆允川写完，忽然有点困。他把诗抄本放到枕头下，像把一个小小的秘密压住。

第二天早晨，枕头边露出一角深蓝色。程砚看见了，只把它往里推了推，没有再笑。
