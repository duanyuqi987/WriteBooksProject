# 第033章 高一结束了

时间：2011年7月下旬

地点：教室

卷次：卷二：窗边人影

七月下旬。暑假前一天。

下午不用上课。班主任说了几句假期注意事项——注意安全、不要玩水、按时完成作业。然后发暑假作业。厚厚的一沓。每一科一本。

教室里闹哄哄的。有人在讨论暑假去哪里玩。有人在问要不要补课。有人已经开始收拾书包。

赵一鸣在跟后桌说暑假要去城里舅舅家。说来也是做作业。然后去网吧。

"你暑假去哪。"赵一鸣问顾临风。

"回家。帮我爸做点事。"

"闷不闷。"

"还行。"

赵一鸣把一个笔记本塞到他书包里。

"暑假作业借你抄。"

"你不做。"

"我到时候再说。你先拿去。"

两个人把东西收拾好。赵一鸣背起书包。

"我先走了。我妈在校门口等我。"

"好。"

"暑假回来见。"

"好。"

赵一鸣走了。顾临风没走。

教室里的人越来越少。先是走了几个。然后走了一帮。最后只剩下零零散散三四个人。日光灯亮着。吊扇还在转。

苏晚晴在座位上。在收东西。把课本一本一本放进书包。物理参考书放在最上面。书包鼓鼓的。拉链拉到一半卡了一下——她拽了两下才拉上。小鱼挂件在书包拉链上保持了平衡。没掉。

她站起来。背上书包。

走到前门口。停下。回头看了一眼教室。

她的目光扫过自己的座位——第二排靠窗。扫过黑板——上面还留着老师写的最后一节数学课的板书，没擦。扫过吊扇——还在转。然后她转身走出门。

脚步声在走廊里越来越远。

顾临风站起来。

教室里只剩他一个人了。

日光灯嗡嗡响。吊扇的叶片影子在地上转。黑板上写着数学公式——二次函数、顶点坐标、对称轴。课桌排得很整齐。四十来张。每一张桌面上都有学期留下的印记——圆珠笔印、涂改液的白点、贴纸的残迹。

他往前走。

走到第二排靠窗的座位前。苏晚晴的座位。

桌面上很干净。只有一道淡淡的铅笔印——可能是画辅助线留下的。椅背上还挂着一个东西——发绳断了。大概是没带回去。黑色的。断了三分之一。

他站在她的座位旁边。

站了几秒钟。

窗外银杏树绿得很浓。七月了。叶子叠着叶子。阳光从叶缝间漏下来。像碎金子。

他没有坐下去。

只是站在那里。看着窗外。从这个角度看到的银杏和平时不一样——他在后排看的时候银杏在窗框左下角。从这里看银杏占了大半个窗。

她每天看到的就是这个画面。

他看了大概一分钟。然后转身往教室后面走。

走到自己座位。

弯腰。从桌洞里拿出最后一本书。

然后走出教室。

走廊很长。很空。阳光从走廊的窗户斜照进来。他的影子在地上拖了很长。到楼梯口的时候他又回头看了一眼教室——门是开着的。窗帘在风里动了动。

出了教学楼。太阳很大。

校门口的横幅换了一块——不是开学时那块"热烈欢迎2010级新生"。是"2011年暑期安全须知"。红底白字。布有点褪色了。

他往车站走。

书包里装着暑假作业。赵一鸣的笔记本在夹层。日记本在最底下。日记本里夹着一张成绩单——他趁没人注意从墙上撕下来的。撕得不是很整齐。缺了一个角。

到车站。上了公交车。靠窗坐下。

窗外。青阳县城在车窗外面往后退。熟悉的街道、小店、招牌、树。阳光照在车窗上泛起白花花的一片反光。

他低头。看窗外。直到车开出县城。

高一结束了。

窗外稻田一片绿。远处有山。天很蓝。

车在国道上开了一个小时。到了镇上车站。他下车。往家的方向走。

书包有点沉。
