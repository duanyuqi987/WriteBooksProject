# 第012章 理科名单

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：新分班后的理科教室

## 诗歌插页 P012：原创仿唐绝句

红榜风前字未干，
少年偷看几回还。
同窗不等同心近，
一尺书桌隔万山。

新班第一天，大家搬书像搬家。

原来的教室在三楼东侧，理科班换到西侧。距离不远，可每个人都像从一个旧世界迁到另一个旧世界。书箱、脸盆、试卷袋、没喝完的水，全堆在走廊里。班主任站在门口喊：“先找座位，不要堵门！”

陆允川抱着一摞书进教室时，最先看见的是黑板上的座位表。名字用粉笔写得很急，竖排横排都不齐。他找到自己，在后排靠窗。然后他没有立刻抬头，却知道夏知遥的名字在前排偏中间的位置。

程砚坐在他旁边，把书往桌肚里塞，塞到一半卡住了。他用膝盖顶了一下，书哗啦掉出来两本。

“理科班第一课，”程砚说，“桌肚容量守恒。”

陆允川弯腰替他捡起一本物理练习册。程砚接过来，忽然用下巴点了点前排：“你看，距离不远吧？”

陆允川不说话。

“也不近。”程砚又补了一句。

新班主任姓邵，声音不大，但每句话都像用尺子量过。她让大家做自我介绍，不许超过二十秒。轮到夏知遥时，她站起来，说自己叫夏知遥，喜欢英语，希望以后和大家一起进步。声音清楚，尾音有一点紧。

陆允川低头看草稿纸，在纸角写了一个“夏”字，又很快涂掉。涂得太用力，纸面破了一点。

轮到他时，他站起来，说：“我叫陆允川，来自青石镇。”

班里有人回头看他。青石镇离县城远，很多人只听过名字。陆允川感觉自己的耳朵热起来，后面准备好的话全忘了，只补了一句：“希望跟上大家。”

他坐下时，程砚小声说：“挺好，至少没说希望跟上前排。”

陆允川差点笑出来，又忍住。前排的夏知遥正在整理铅笔盒，没有回头。她的马尾落在校服领子上，发梢被窗风吹起一点。

那一天，他第一次明白，同一个班并不等于同一条路。路可以并排很久，也可能从一开始就各有方向。只是十六岁的他还没学会看方向，只会看距离。
