# 第132章 银河

时间：2012年7月10日，夜晚（接上章）

地点：李家沟村小操场

卷次：第六卷：雪落无声

苏安晴走到教室拐角的时候停住了。

她回头看了一眼林玉起——他还坐在乒乓球台下面，低头在本子上写着什么。然后她抬起头，往天上看。

"林玉起。"

他抬头。

"你过来看。"

林玉起合上本子走过去。苏安晴站在操场的正中间，仰着头。

"这里能看到银河。"

他也仰起头。

是真的。

一条光带从东边的山脊上横跨过来，穿过整个天顶，铺到西边去了。不是照片里那种五彩斑斓的银河——肉眼看到的银河更素淡一些，像一层发光的云，也像一道很宽很淡的乳白色的河。无数的星点密密地挤在那条河里，分不清哪个是星星、哪个只是光的碎屑。

林玉起在书里读过一百遍银河，今晚是第一次亲眼看见。

"我小时候在外婆家见过。"苏安晴说。她的声音比平时低，像是怕惊动什么。"外婆家在信阳的山里。夏天睡在院子里，一睁眼就是满天星星。后来搬到城里就看不到了。"

"你外婆还在吗？"

"不在了。初三那年走的。"

林玉起没有说话。

苏安晴又看了一会儿，然后指着头顶偏北的方向："那是织女星。那边那颗是牛郎星。"

"你懂星？"

"高中天文社的。办了一年就倒闭了。但我记住了一些。"

她说话的时候没看他，一直看着天。

"小时候外婆跟我说，银河是王母娘娘用簪子划出来的。这边是织女，那边是牛郎。每年七月七，喜鹊搭桥，他们才能见一面。"

"七月七快到了。"

"还有一个月。"

夜风从山沟里翻上来，吹动了操场边上的那根旗杆绳。绳子打在铁杆上，发出"当当"的声音，很轻，像远处有人在敲钟。

"林玉起。"

"嗯。"

"你觉得他们值不值得？"

"谁？"

"牛郎织女。一年只见一面。值得吗？"

苏安晴的这个问题问得很突然，也很认真——不是随便聊天的语气，是她真的在想这个问题。

林玉起想了一会儿。

"值不值得不是他们能选的。"

苏安晴偏过头看了他一眼。光线太暗，看不清她的表情。但她点了一下头。

"也是。"她说。

她把手插进外套口袋里，往女生宿舍那间教室走去。走了几步，没回头，说了一句：

"你的诗可以写写银河。比写虫叫有意思。"

林玉起站在操场正中间，头顶是银河，脚下是泥土，四周是山。

他忽然很希望时间停在这一刻。

>
银河横天过，少年立土场。
不敢问来日，只怕夜不长。
>
