# 第195章 夏知遥的消息多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：县城 / 线上

## 诗歌插页 P195：原创仿唐绝句

旧名微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

多年后陆允川回县城过年，在超市的冷冻区碰到了一个人。

不是夏知遥。是张敏——高中坐在夏知遥前排、英语笔记记得最整齐的那个女生。张敏推着一辆购物车，车里放了速冻水饺、两袋汤圆和一瓶老抽。她的头发剪短了，穿着厚厚的羽绒服——县城冬天还是那么冷。

"陆允川？"

"张敏——你也在家过年。"

"对。刚从深圳回来。你变了好多——眼镜换了？"

"换了。度数涨了。以前的镜框太窄。"

"不是窄不窄的问题——以前你是黑框，现在是银的。看着稳重了。"

两个人在冷冻柜旁边聊了一会儿。冷链的压缩机嗡嗡响，冷气从柜门缝里往外漏。张敏说她在深圳做跨境电商——"每天跟老外发邮件，英语用到吐"。然后她提了一句让陆允川差点没握住购物车扶手的话。

"对了——你还在高中群吗？那个群去年解散了。"

"解散了？"

"嗯。群主清了一批长期不说话的人。剩下的全是发拼多多链接和投票的。群主自己也受不了了。上个月直接解散了。你退得早——不知道后面的事。"

"我大三就退了。"

"我知道。你退群那天我们几个在私聊里讨论过。"

"讨论什么？"

"讨论你是不是因为夏知遥退的。有人说是。有人说不是——'陆允川从来不发言，可能早就不想待了'。后来夏知遥本人也在私聊里提过一次。"

陆允川的手停住了。"她说什么？"

"她说——'他退群之前我都没跟他说过话。希望他退得轻松一点。'"张敏把一袋速冻水饺放回购物车。"就这一句。没有再多了。"

"'退得轻松一点'——她原话是这样？"

"对。原话。"

陆允川握着购物车扶手的手指松了一下。不是紧张——是觉得十七岁那年所有没开口的话，在这一句里被轻轻托了一下。不是回应。是托了一下。

"那她现在在哪儿？"

"在南京。研究生毕业以后留校了。做辅导员。去年结的婚——对象是她研究生同学。我没去婚礼，看了照片。她穿旗袍。很漂亮。"

"挺好的。"

"你呢？结婚了吗？"

"结了。妻子是我工作以后认识的——不是同学。"

"干什么的？"

"在出版社做编辑。"

"那很配。你写诗。她编书。"

陆允川笑了一下。"不一定配。有时候她嫌我的诗太散。"

"什么散？"

"结构散。她读完之后说——'情绪到了但落点没到'。然后帮我改。她改诗比我自己改得狠。"

张敏也笑了。"那说明她在乎。不在乎的话，谁会帮你改诗。"

冷冻区的广播响了——"超市二楼家电特卖，满500减50"。两个人不约而同地往购物车里看了一眼。张敏的车里有汤圆。陆允川的车里有酱油和鸡精——妻子叫他买的。

"那我先走了。老婆在家等着酱油炒菜。"陆允川说。

"好。新年快乐。"

"新年快乐。"

他推着车走了几步。然后停下来。

"张敏——夏知遥知道我退群是因为她吗？"

"可能知道。也可能不知道。她只说了一句'希望他退得轻松一点'。没问过原因。"

"那她有没有——"

"没有。她没有问你要不要加回来。也没有让我传话。她很清楚退群是一个人的决定——不是两个人的对话。"

陆允川点了点头，然后推着车走向零食区。

晚上在家吃完年夜饭，陆允川坐在沙发上。妻子在厨房洗碗——碗筷碰撞的声音很轻，水流声一阵一阵。电视开着但没有看——春节联欢晚会的声音被调成了背景。

他打开手机。在微信通讯录里搜了"夏知遥"三个字。搜索结果只有一条——是高中时候加的好友。头像还是系统默认的灰色人物剪影。朋友圈一条横线——不知道是被屏蔽了还是根本没发过。

他点开对话框。最后一条消息是高三毕业那天的——她发的："毕业快乐！" 他回的是："你也是。"

就这六个字。再也没说过话。

他的手指在输入框上放了一会儿，没有打字。

妻子洗完碗从厨房走出来。她用围裙擦着手上的水。"在看什么？"

"一个老同学——高中同学。好多年没联系了。"

她想了一下，然后问："女的？"

"嗯。"

"女朋友？"

"不是女朋友。是——以前喜欢过的人。"

"表白过吗？"

"没有。"

"那你现在找我申请——你什么意图？"

"没有意图。"他把手机横过来给她看。"你看。十二年没说一句话。刚才想发个新年祝福——然后发现不知道说什么。不是尴尬——是真的没有话。"

妻子在他旁边坐下。她把围裙叠好放在沙发扶手上。"没有话就不要硬说。你以为祝福是礼貌——但对方可能觉得是打扰。尤其你们十二年没说话。突然跳出来发一句'新年快乐'——对方要回什么？'你也是'？然后就没了。跟高三毕业那天一摸一样。"

"你怎么知道高三毕业那天我们说了什么？"

"我猜的。高三毕业就是那种场景——所有人都说'毕业快乐'。然后真的毕业了。真的快乐了吗？不一定。"

陆允川把手机放下。"你说得比我好。"

"因为你还在想她的部分。我在想整个场景。"妻子站起来，把靠垫拍松。"你们男人永远在纠结要不要再往前一步。女人一般想的是——有没有这一步的必要。"

"那你觉得有没有必要？"

"你自己已经回答了。你刚才说——'不知道说什么'。不想说话就是没必要。"

电视里一个相声演员正在模仿方言。笑声一阵一阵的。陆允川把手机锁屏，放回茶几。过年的鞭炮声在窗外很远的地方响了几声——城里禁放，但还是有人偷偷放。

年初三那天，他在县城的老街走了走。县一中还在老位置——大门换了，操场铺了塑胶跑道。以前他蹲过的那根单杠已经不在了——整个单杠区都被拆了，换成了一片花坛。花坛里种了矮牵牛——冬天开不了，叶子都缩成一团。

他在花坛边上站了一会儿。不是缅怀。就是站了一会儿。然后他掏出手机拍了一张花坛的照片。发给了祁野。

祁野秒回："这什么。"

"县一中操场。以前单杠的位置。现在种花了。"

"好看吗。"

"不好看。矮牵牛。冬天缩了。"

"那你拍它干什么。"

"因为以前那里有根单杠。高三毕业那天我在单杠底下蹲了半小时。渴得要死。可乐没拧开。"

"我知道。你跟我说过。"

"今天看到单杠没了。变成花了。就想拍一下。"

祁野过了几秒才回。"恭喜。"

"恭喜什么。"

"恭喜你看到的是单杠没了——不是夏知遥还在拍照。"

陆允川没有回这条消息。他把手机放回外套口袋。风吹过来，矮牵牛的枯叶子哗啦啦响。他沿着塑胶跑道走了半圈。跑道上有几个放寒假的高中生在跑步——校服跟以前不一样了，颜色变了。但跑步的姿势是一样的——男孩子跑得快，女孩子落在后面喘气。

他从操场的小门走出去。门卫室的窗户上贴了一张告示——"访客请登记身份证"。他把身份证掏出来又放回去。算了。不进去了。

回家路上经过一家奶茶店。以前高中时候这里卖的是珍珠奶茶——三块五一杯。现在改成了连锁品牌——"一点点"。他点了一杯四季春茶。无糖。去冰。服务员在单上写了"陆先生"三个字——字很工整。

他拿着茶在路边喝了一口。然后拿出手机打开备忘录。新建了一条。

标题没写。正文就一段：

"今天去县一中看了一圈。操场还是那个朝向。单杠没了——上面种了矮牵牛。高三毕业的时候我在单杠下面蹲了半小时。现在想想——当时以为自己在'暗恋'，其实就是不敢。不敢面对'我说了她会怎么回'这个问题。不敢承担她说'不'的那个瞬间。把不敢包装成深情——是我十七岁最擅长的事。

张敏说夏知遥后来提过我退群的事，说的是'希望他退得轻松一点'。谢了。退得确实轻松。只是晚了好多年。

不是为了这个写的。是刚才在奶茶店等茶的时候，脑子里突然冒出一个念头——如果十七岁那年我在操场上走过去，不是为了表白，只是为了说一句'可不可以合照一张'。那现在电脑里的那个文件名大概会是'2018-06-20-高中合照-夏知遥'。而不是'高中合照-已归档'。

但想这个也没用。茶凉了。"

他把备忘录合上。没有保存。手指在"删除"按钮上停了一下。然后删了。

回到家里，妻子正在整理书架。她把那些歪着角的书一本一本地摆正。摆到那本《中国诗歌选》的时候，她停了一下——那本还是歪的。

"这本要不要摆正？"

陆允川看了看。"不用。让它歪着。大三贴的标签——贴歪了就歪到底。"

"行。"她把书推回去。书脊从书架边缘凸出来一个小角度。正好挡袖口。

他走过去站在书架前面。眼睛扫过那些书脊上的编号纸——有的正有的歪。大一贴歪的还歪着。大三贴正的正了。毕业后买的书没有编号——干干净净。他看着这些书，忽然觉得书架就像一个时间切片——每一个歪角都是一年。

"你今天心情不错。"妻子说。

"不算不错。就是——松了一口气。"

"松什么气。"

"高三欠自己一句话。今天去操场看了一圈。发现单杠已经拆了。原来觉得欠了一个答案——现在发现答案不用找了。因为出题的人已经走了。考场都拆了。"

妻子走过来站在他旁边。她看了书架几秒。"你知道你为什么娶了我吗？"

"为什么？"

"因为我不像你的诗——你的诗永远在找落点。我是一个落点已经在了的人。你不需要找。"

陆允川看着她。然后伸手把那本歪着的诗集又往外拨了一下。更歪了。

"这又是什么操作？"

"标记。标记我跟你说过这句话。"

"什么话？"

"你是落点。"
