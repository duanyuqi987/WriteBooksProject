# 第130章 苏老师

时间：2012年7月6日，上午

地点：李家沟村小高年级教室

卷次：第六卷：雪落无声

第一天上课。

早上七点半，孩子们陆陆续续来了。小的七八岁，大的十二三岁，穿的衣服普遍大一两号——有的是哥哥姐姐穿剩下的，有的是家里人故意买大的，"能多穿两年"。

高年级班在中间那间教室。黑板是水泥抹的，刷了一层黑漆，漆面已经磨花了，写上去的字带着一层灰蒙蒙的底。

苏安晴站在讲台上。她把教案放在讲桌左边，粉笔盒放在右边，然后拿起一根白粉笔在黑板正中间写了自己的名字——

苏 安 晴。

她的粉笔字写得比钢笔字好看。三个字端端正正，横平竖直。最后一横收笔的时候，粉笔在黑板上磕了一下，断了一小截。

她弯腰把断掉的那截粉笔捡起来，放回盒子里。

然后转过身。

"同学们好。我叫苏安晴。这个暑假由我来给大家上语文课和英语课。"

底下没有声音。十四张脸，十四双眼睛，有些好奇，有些紧张，有些还没睡醒。

坐在第一排的一个女生先开了口："苏老师好。"

然后其他孩子也跟着喊了——"苏老师好。"七嘴八舌的，声调不齐。有个男孩喊得特别大声，喊完之后自己不好意思了，把头埋进胳膊里。

苏安晴在讲台上站了两秒。

林玉起坐在教室最后面——他没课，但他想来看。

他看见苏安晴的眼睛亮了一下。那种亮不是激动，是一种很安静的、很确定的亮。像煤油灯灯芯刚被拨正的那一刻。

"谢谢，"她说，"那我们开始上课。"

她翻开课本。第一篇课文是《桂林山水》。

"请大家打开课本第一页。我们先读一遍课文。我读一句，你们跟着读一句。"

"人们都说——'桂林山水甲天下'——"

她读得很慢，每个字都咬得很清楚。声音比平时说话高了一点，但不尖。被教室的墙壁弹回来，带着一点点回音。

孩子们跟着读。不整齐，但声音很大。那个刚才喊"苏老师好"的男孩叫李铁蛋，读得最卖力，青筋都冒出来了。

林玉起坐在后面，看着她一页一页地翻课本，看着她在黑板上写生字、画插图——鸡还是像长了四条腿的土豆——看着孩子们慢慢放松下来，开始举手回答问题。

他忽然想到一件事：他认识苏安晴两年多了，这是他第一次看见她站在讲台上的样子。

跟平时走的那个苏安晴不太一样。

但又确实是同一个人。

下课后他去水房打水，碰见教低年级语文的赵一琳。赵一琳说："苏安晴上课上得真好。她是不是从小就相当老师？"

林玉起想了想，说："我不知道。"

他真的不知道。他认识她两年多，却发现自己对她知道得太少了。
