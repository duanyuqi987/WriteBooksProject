# 第136章 她变了

时间：2018年3月周六下午

地点：楚州"一茶一坐"茶餐厅

卷次：卷七：流年未断

她比以前爱笑了。

说话的时候不再那么拘谨。会有手势——手掌摊开、指尖轻点桌面、或者双手抱在茶杯两边取暖。讲她们班上的学生的时候——有一个男生上课老是学猫叫。她学他学的样子——嘴唇缩成小圆，发出一个很闷的"喵"。然后自己先笑了。他愣了一下。然后也笑了。八年前她不会在别人面前学猫叫。连笑都是抿着嘴的。

"那个学生叫王浩然——坐在最后一排。每次我转身在黑板上写字他就来一声。回头他就趴着。装睡。全班都笑了。"她说着摇了摇头。手掌在桌面上摊开一下。然后收回去抱住杯子。动作很快。

"后来呢。"

"我让他站到讲台上来叫——他就老实了。叫不出来了。"她又笑了。眼睛弯成两道细缝。

"你变多了。"

"嗯。"

"我是说——好的那种。"

"我知道。"她点了点头。没有客套。大概是教了六年小学——每天对着四十五个八九岁的孩子——不得不学会放松。

他注意到她手上有两个小茧——拇指和食指之间。捏粉笔的地方。跟她高中时候的手不一样。那时候她的手白白的，手指很细。现在还是细的，指尖上有粉笔磨出的茧子。

"你看我的手干嘛。"

他抬头。"没什么。"

"你看很久了。"

"粉笔磨的。"

她低头看了看自己的手指。翻过来——手心朝上。掌心里也有茧。洗不掉的粉笔灰嵌在指纹里——灰白的。"没办法。每天写板书。"

他把自己的手翻过来给她看。同样的茧。同样的位置。粉笔灰在指纹里嵌了两年——用什么洗都洗不掉。只是他的手比她的更大、更厚、青筋更多。

她看了看他的手。没说话。然后又看回他自己的手。然后抬起头看着他。

"你别一直看我啊。"

"习惯了。"

说完就后悔了。他低头看着杯里的茶叶——茶叶已经完全沉在杯底了。他假装在看茶叶的脉络。其实什么都没有在看。他在等她的反应。

她没有介意。她低下了头。马尾的发梢滑到前面，遮住半边脸。然后她伸手把头发别到耳后。耳朵尖红了一点。

空气里茶香已经淡了。取而代之的是从厨房飘出来的炒饭的味道。茶餐厅开始供应晚餐了。有人在隔壁桌点了煲仔饭——锅巴的焦香混在茉莉花香里。

"你没怎么变。"她说。声音压低了。

"什么。"

"看人的习惯。高中你就是这样。一直看我。但你站在走廊上我看不到——我知道你在看我。"

他不知道。

他以为他把这件事藏得很好。站在走廊尽头假装看窗外——她在教室里，应该看不到。但她知道。她不但知道他在看窗外。她还知道窗外是围墙。围墙外面是马路。马路上什么都没有。一个男生站在走廊尽头对着围墙发呆——他不是在看风景。

"你怎么知道的。"

"窗户有反光。"

她笑了。是那种"赢了"的笑——嘴角往一边翘，眼睛亮一点。然后她伸出食指，在桌面上点了点——像在画一个光路图。"你去饮水机打水——每次我刚好路过。你去办公室交作业——每次都走我座位这边。你早上到教室——每次都比我晚几分钟。你是算好的。"

他低头。手指在茶杯杯沿上画了一圈。茶已经完全凉了——凉得有点涩。水面反射着头顶暖黄的灯光。竹编灯罩的影子在水面上晃了一下。然后他抬起头。

"你还记得这些。"

"我记得很多。"

她的声音很轻。但眼睛没有躲闪。茶餐厅的暖光灯照在她眼睛里——把她的瞳孔染成了一小片琥珀色。

窗外路灯全亮了。一排橘色的灯光把街景分成一格一格的。茶餐厅里的光线变得暖了一些。音响里换了一首曲子——萨克斯。慢悠悠的。

她指着窗外那棵樱花树。"三月份要落了。我们学校也有两棵——在后操场。每年都落得一地。学生把花瓣捡起来——夹在作业本里交上来。"

"我们学校也是。不过是一棵——长在教学楼后面。"

"青阳那个——月湖边的樱花是不是更多。"

"对。湖边有一排。"

"你还记得月湖。"

"记得。"

"湖边的石凳还在吗。"

"应该还在。我过年回去的时候看到过。"

她点了点头，没有说话。眼睛看着窗外的路灯。路灯下有个塑料袋被风吹起来——翻了一下，然后又落下去。

"我有时候会梦到月湖。"她说。没有看他。

他喉咙动了一下。端起杯子——才发现茶已经喝完了。只剩杯底有一小片完全泡开的茶叶——孤零零地贴在内壁上。

她的茉莉花茶——那一壶也空了。但两个人都没有叫续。

窗外风吹了一下。樱花树剩下的几片花瓣落了——落得很慢。有一瓣贴在茶餐厅的玻璃上——粘了两秒，然后被风带走，飘到马路那边去了。

窗外路灯全亮了。一排橘色的灯光把街景分成一格一格的。茶餐厅里的光线变得暖了一些。音响里换了一首曲子——萨克斯。慢悠悠的。她指着窗外那棵樱花树——说三月份要落了。她们学校也有两棵。在后操场。每年都落得一地。学生把花瓣捡起来——夹在作业本里交上来。

"我们学校也是。"他说。"不过是一棵——长在教学楼后面。""青阳那个——月湖边的樱花是不是更多。""对。湖边有一排。""你还记得月湖。"他的声音低了下去。"记得。"她点了点头。没有说话。眼睛看着窗外的路灯。路灯下有个塑料袋被风吹起来——翻了一下。然后又落下去。
