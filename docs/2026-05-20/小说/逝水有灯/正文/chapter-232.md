# 第232章 诗歌索引

所属部卷：第五部：逝水有灯
时间：整理全稿夜里
地点：自己的书房

## 诗歌插页 P232：原创仿唐绝句

百页诗灯次第排，
删繁留影到窗台。
旧名不入新行里，
只许风声缓缓来。

陆允川把诗歌索引打印出来，一共十几页。打印机卡了一次纸，他把纸抽出来，边角皱了，只好重新打。纸张从机器里吐出来时带着热气，油墨味在书房里散开。

他拿红笔逐首检查。现代诗里删掉太像宣言的句子，仿唐绝句里删掉过于漂亮的词，仿宋小令里删掉那些把自己写得太无辜的转折。有些诗原本看起来顺口，删完反而短了，空了，却更接近那一章该有的气息。

第082章迟到表白那首，他把“长恨”改成“晚风”。第142章拒绝那首，他把“她心如铁”整句划掉，改成“台阶微凉”。第202章最后争论那首，他保留“感动不能替我答应”的意思，却不把它写成口号。

整理到凌晨，红笔水快没了。陆允川把索引摊在桌上，像摊开一段被重新编号的人生。诗歌给了他表达的入口，也给了他节制的边界。四句写不下的，就不要硬塞；一行说不准的，就让正文去承担。

他在索引首页写下：诗不替人物辩护，只替场景留灯。写完这句，他把红笔扣上，听见笔帽轻轻一响。
