# 第118章 楼下等她之后

所属部卷：第三部：寒风里的花
时间：大二五月
地点：女生公寓楼下圆石凳

## 诗歌插页 P118：原创仿宋小令

石凳边，晚声轻，
半窗灯影落无凭。
问旧年、几回执念，几回分明。
人去不须追远路，
水流仍向晓光生。

诗社会议开完后，苏晚枝说了一句让陆允川在原地呆了半秒的话。

"明天不用来等我。明天我要去市区看建筑系的一个展览，晚上才回来。"

"什么展览？"

"青年建筑师作品展。市立美术馆。系里组织的，包车。"她把社刊样稿收进布袋，"后天如果你方便，可以继续来等我。"

"好。"

陆允川从人文楼走回宿舍的路上，把这个"继续来等我"在心里翻了好几遍。她说得很平淡，像是在说"后天食堂继续开"。但他听出了一种默认——默认等人这件事已经被纳入了她的时间表，像课程、像画图、像诗社活动一样，是有固定位置的事。

回到宿舍，祁野正在用手机看篮球比赛回放，声音放到最大。徐远在打游戏，键盘声噼里啪啦。

"你今天穿得挺整齐。"祁野瞥了他一眼，"白衬衫——你平时不穿白衬衫。"

"今天诗社开会。"

"开会要穿白衬衫？诗社什么时候变企业了？"

"我想穿。"

祁野放下手机。"你是去诗社开会，还是穿给谁看？"

"苏晚枝。我下午去5号楼接她。"

宿舍里安静了两秒。然后祁野说："所以她看到你穿白衬衫了？"

"看到了。"

"她说什么了？"

"什么都没说。但她戴了一个银杏叶的胸针。"

"这跟白衬衫有什么关系？"

"去年她去食堂吃饭时也戴了那个胸针。那天她穿了一件墨绿色的毛衣。所以墨绿配银杏是她喜欢的一个搭配。白衬衫能配任何胸针。"

祁野靠在床头看着天花板，像是在计算什么。"陆允川，你知道你刚才说了什么吗？你记得她去年穿什么毛衣，记得她喜欢什么胸针，记得她搭配的颜色。你今天穿白衬衫，是因为你想成为她审美体系里一个可搭配的部分。"

陆允川说不出话。

"我没有批评你的意思。"祁野把目光收回来，"我就是想确认：你知道自己在做什么。"

"我知道。"

"那就行。"

第二天晚上，陆允川没去5号楼。他坐在图书馆里写诗社提案——"等字考"。苏晚枝发了朋友圈：美术馆展览的照片，一张是一个模型照片，建筑外立面用了大面积的玻璃和木材，配文是"老师问：为什么用木材？设计师答：因为木材会老。"另一张是一群建筑系学生站在展厅门口的合影。苏晚枝站在第二排中间，还穿着昨天那件白衬衫——不对，不是同一件，这一件的领口没有银杏叶胸针。

他去过夜她还没回来。十点四十二分她在群里发了"已返校。大家晚安。"陆允川在十二点关上手机上屏时还在想：她今天有没有在那个展览上想到什么跟我有关的东西。

周五六点五十分。陆允川再次出现在5号楼圆石凳前。这次石凳空着。他坐下，膝盖上摊着一本书。五分钟后，玻璃门打开，苏晚枝走出来。

"今天准时。"她说。

"你也是。"

"因为图提前画完了。"她从布袋里拿出一管卷着的纸，"透视图终稿。昨晚上画到十二点——从美术馆回来以后脑子的空间透视感特别敏锐，一口气画完了。"

陆允川接过她的图纸夹在腋下。她继续说："我在展览上看到一句话，可能可以放到'等字考'那一期社刊。"

"什么话？"

"一个做了四十年木结构的老师傅说的：'木头最懂等。它长在那里等你砍，砍下来等你晒，晒干了等你用，用上了等你老。'"

"这不太像老师傅说的话。"

"对。我加工了一下。"

"原来那句是什么？"

"木材需要时间。"

他们沿着老路往人文楼走。陆允川发现这条路已经变成了一条很熟悉的路：5号楼→经过篮球场→经过东食堂→经过打印社→拐到人文楼。总共大概八分钟。苏晚枝沿途会指着一两样东西说今天的变化：篮球场的网换新的了，食堂门口新贴了海报，打印社的刘阿姨把门楣上的照片换了——以前是她和她丈夫的合影，现在换成了一张打印社门口花盆的特写，黄色的月季。

"她终于换了照片。"苏晚枝说，"前几年一直是那张合影。她说换了照片不代表忘了人，只是换了一种记法。"

陆允川看着打印社门楣上的月季照片。他想：苏晚枝注意这些变化——网、海报、照片——不是因为观察力强，是因为她每次走这条路都在看。这和他在楼下等她是一样的性质：不是偶尔发生，是已经被纳入了时间表的固定事件。

"你在想什么？"苏晚枝问。

"想你的'等字考'。提案我写了三分之二。"

"剩下的三分之一呢？"

"在等你。"

"等我？"

"等你告诉我那个老师傅到底说了什么。还有今天圆石凳旁边花坛里的三色堇开了几朵。"

苏晚枝推开人文楼玻璃门，回头看了他一眼。那个目光他之前没见过——不是审视，不是礼貌，不是疲惫。是一种他还没有名字的表情。像是她发现了一个细节，而这个细节是她没有预料到的。

"三朵。"她说。"五瓣的紫色的。你数花干什么。"

"因为等人需要眼睛有地方放。"
