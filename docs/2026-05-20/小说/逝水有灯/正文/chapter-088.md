# 第088章 录取通知书之后

所属部卷：第二部：六月以后
时间：七月下旬
地点：陆家小院

## 诗歌插页 P088：原创仿宋小令

红封展，晚风长，
母亲灯下数行囊。
问少年、远方初近，旧梦微凉。
纸上城名分两路，
窗前仍有饭茶香。

通知书到家后，母亲看了三遍。

她先看学校名，又看报到时间，再看学费和住宿说明。看完以后，她去柜子里拿出一个旧本子，开始记要准备的东西：被子、凉席、脸盆、饭盒、证件照。

陆允川坐在旁边，忽然觉得大学不是一个抽象的远方，而是一堆具体物件。母亲问他要不要买新箱子，他说旧的也行。母亲说第一次出远门，还是买个结实点的。

父亲在院子里打电话给亲戚，声音不高，却能听出一点压着的高兴。陆允川拿着通知书走到东屋，墙角的试卷还在，红色信封放在桌上，像从另一种生活寄来的门票。

程砚打来电话，说自己的通知书也到了，学校离他不算远。两个人互相损了几句，又沉默下来。高中真的要散了，这件事直到通知书到手才变得确定。

晚上，陆允川打开班级群，群里全是通知书照片。夏知遥那张也在，拍得很端正，旁边放着一支笔。她去的城市在地图上离他不近，也不算遥不可及。可他们都明白，不同方向就是不同方向。

陆允川没有私聊她。

他把自己的通知书拍照，发给程砚，又发给母亲留存。然后把红封放回塑料袋，压在书桌最上面。那一晚，他睡得很晚，却没有写诗。

远方已经开始变成行李，他需要先学会收拾。
