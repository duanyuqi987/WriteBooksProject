# 第155章 南湖问答多年后

所属部卷：第三部：寒风里的花
时间：多年后的某个夏天
地点：城市公寓书房

## 诗歌插页 P155：原创仿唐绝句

湖灯微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

多年后的一个夏夜，陆允川坐在自己的书房里，重读了那一晚的日记——不是那本旧日记本，而是他后来新建的一个文档，专门存放那些"可以回头看但不必反复翻"的记忆。

旧日记本已经被他收到老家的柜子里了。压了两本教材的抽屉后来也换了新的，那两本教材毕业时卖给了二手书店。但南湖问答一直在。不是作为伤口，是作为一个参照点——"那是叶澄跟我说真话的晚上"。

他给叶澄发了一条消息："还记得南湖那次吗？"

叶澄的回复来得不快也不慢，二十年如一日："哪次？你约我好多次。"

"就六月末那次。你让我把日记本合上。"

"哦。记得。怎么了？"

"没什么。就是今天翻到了当时的记录，忽然觉得你那时候说的话，有一半我是后来才真的听懂。"

"哪一半？"

"关于'方向是错的'那一半。你说如果我的方向是'成为一个不再想她的人'，那方向就是错的。我当时觉得这话很有道理，但没真的理解。后来我用了很多年才明白，你说的是：你不需要成为一个'不再想谁'的人，你需要成为一个'想自己'的人。"

叶澄过了一会儿才回："你那晚听懂的比我想象的多。"

"不多。刚好够用。"

他没有再追问什么。退出微信，把那个文档又往下翻了翻。

南湖之后的事情他记不太清楚了——不是因为不重要，是因为实在太多了。期末考试像每年必然到来的雨季，考完一门又来一门；然后是暑假回家，母亲做了他爱吃的红烧肉，父亲问他学业怎么样，他说还行；然后是新的学期，新的专业课，新的设计作业，新的烦恼——烦恼的内容换了，但烦恼的密度没有变少。生活不允许他沉浸在一场问答里太久。祁野的泡面借条越积越多——从三包变成了五包，祁野说"你要是再不还我就按复利计算了"；程砚的实验论文投了期刊，《机械工程学报》，初审过了二审还在等消息；苏晚枝考上了研——建筑学硕士，去了南京，走的那天她在301教室的树墙下站了很久，最后只带走了一片叶子；夏知遥换了第三份兼职——从奶茶店到图书馆再到一家小型设计公司的前台——然后找到了正经工作，在深圳一家金融机构做风控。

每个人都往前走了一截。他也走了一截。不是大步流星，是每天挪一点。像一个生了锈的齿轮被一点一点滴了油：少翻一页旧日记，多上一节课；少发一条试探消息，多跑一圈操场；少追问一个"为什么"，多煮一碗泡面还给祁野——虽然从来没真还清过。

这种挪法不壮观，但管用。

他现在还记得南湖那晚的一个细节：叶澄走的时候回了头。她说的那句话是关于"方向"的，但那个回头的姿势更重要——她说完了觉得还需要补一句，于是回头了。不是为了再说教什么，是怕他一个人走在回去的路上又想偏。

当时他没意识到这是一种关心。他当时只觉得被拆穿了有点难堪。

后来他见过很多人关心别人的方式。有的是送礼物，有的是陪喝酒，有的是说一百句"你很好"。但叶澄只给了他三件事：一句话、一瓶水、一个回头。

他现在觉得，三件事足够了。

也是多年后，他才理解了祁野那晚说的"收据论"。祁野说关系干净了就不叫关系叫收据，当时他觉得这是个很聪明的比喻，后来他才真明白这句话在说什么。

祁野说的"收据"不是贬义词。收据也是有用的东西，证明交易发生过。但它不是关系。关系是不需要收据的。你对一个人好，不需要保留证据；你从一个人那里得到善意，不需要开票报销。但陆允川年轻时就是在开票——他对苏晚枝的每一次付出都记了账，对每一次冷落都做了分录，最后的财务报告就是那本旧日记：入不敷出，严重亏损。

祁野自己后来也跟一个人在一起了。那个人不喜欢玩游戏，祁野就少玩了一些。程砚问他是怎么想通的，祁野说："没想通。就是觉得跟她待着比抽卡有意思。"

这个答案比任何哲学都准确。

至于苏晚枝——她后来过得很好。考上了想去的学校，做了想做的工作，遇到了想在一起的人。她没有出席他的青春审判法庭，因为她从来就不该坐在被告席上。

陆允川是后来才意识到这一点的：他一直把她放在自己故事的中心位置，却没问过她愿不愿意参演。被擅自安排角色是很累的，尤其当你被安排成"需要负责让一个男生成长"的时候。

苏晚枝只是路过了他，然后继续走她自己的路。他的成长不是她的责任，从来不是。

陆允川重读旧稿的时候删掉了那些过重的词——'命运'、'注定'、'永远'、'放不下'。这些词写的时候很有力，回头看却很虚。真正有力的是什么呢？是长椅扶手上掉的那块漆，是叶澄拧开水瓶时的那个动作，是程砚说"需要的时候说一声"的那条消息，是祁野说"三包泡面有一包你半夜偷泡的"那句揭穿。

这些细节不负责抒情。但它们比所有的抒情都经得起时间。

他把那晚的湖灯也留在了文档里。湖灯不是暗号，不是象征，只是那天确实在场的东西。叶澄拍了一张糊掉的照片，他还记得她说"糊的更好看"。这大概也是一种哲学：清晰的东西太硬，模糊一点才留有余地。

他把文档翻到最新的一页，在空白处打了一行字：

"南湖之后，他还是会在一些夜里想起一些事。但不问结果了。只是想起来，然后翻个身继续睡。祁野的呼噜声很大，程砚的闹钟永远比所有人早十分钟。日子就是这样接上去的。"

打完这行字，他看了看时间。快十二点了。明天还要上班，有个方案要交。

他关掉电脑，去厨房倒了杯水。窗外城市的夜景密密麻麻，没有南湖那么安静，也没有虫声。但他也不需要那些了。

手机震动了一下。是叶澄发来的："对了，你上次欠我的那顿火锅，什么时候兑现？"

他笑着回："周末。你挑地方。"

"人均两百以下的我不去。"

"你上次不是说人均一百就够了？"

"那是上次。现在通货膨胀了。"

"好。两百。"

他把手机放在床头柜上，关了灯。

黑暗里他想了想南湖那夜，想了想那块掉漆的扶手，想了想叶澄回头时路灯正好照亮她的左半边脸。然后他翻了个身。

祁野不在旁边打呼噜了。但他偶尔打视频过来，第一句话永远是"你还欠我一颗卤蛋"。

二十年了。卤蛋的利息大概够开一家养鸡场了。

但祁野从来没真让他还过。因为那包泡面、那颗卤蛋、那瓶带回来的水，都是不需要收据的东西。

他把被子拉了拉，闭上眼睛。

湖灯在记忆里还亮着，但已经不需要他去看它了。
