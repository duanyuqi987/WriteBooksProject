# 原始学习书系统参考文件（存档）

本目录存放的是原 `project-to-learning-doc` 技能系统的参考文件。该项目已将核心定位切换为"小说写作台"，这些文件不再被主流程引用，仅作历史存档。

## 文件说明

| 文件 | 原用途 |
|------|--------|
| `document-types.md` | 学习书/源码书/网站知识书/教师用书等 8 种文档形态蓝图 |
| `source-handling.md` | 书籍、网站、代码库、项目文件夹和混合素材的来源处理规则 |
| `writing-system.md` | 长文写作系统：章内节奏、算法写法、受众校准、代码讲解比例 |
| `html-output.md` | 单文件 HTML 输出规范：章节 ID、目录、元信息、占位章节 |
| `iteration-and-quality.md` | 分批写作协议、字数估算、质量检查清单、完成标准 |

这些规则针对"资料转学习书"场景，与小说写作无直接关联。如需使用原学习书功能，可通过 `SKILL.md` 历史版本恢复。
