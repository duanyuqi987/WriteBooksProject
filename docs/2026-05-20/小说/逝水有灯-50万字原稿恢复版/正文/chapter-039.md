# 第039章 成绩单下滑旁观

所属部卷：第一部：初秋三楼
时间：高二期中后
地点：班级后墙成绩栏

## 诗歌插页 P039：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
成绩栏上被红笔圈出的名次还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，成绩单下滑会显得没有那么沉重。教室、走廊、操场里站着许多人，每个人都有自己的迟到、作业、饭卡和回家路。

叶澄后来记得的不是陆允川的沉默，而是他那天总在看成绩栏上被红笔圈出的名次。她觉得那样的眼神很熟，像很多人刚开始喜欢谁时都会有的样子：认真，又有点不讲道理。

叶澄路过后墙，看到陆允川还站着，就把一张草稿纸递给他。纸上写着几道同类型题，她说，先会一道是一道。

程砚看得更直接。程砚把他拉到座位上，翻出物理卷子，说从这道题开始，别把喜欢当借口，也别把借口写得像喜欢。他不喜欢把话说得太柔，因为太柔的话陆允川听了会继续躲。

“掉了就补，别站这儿给成绩单守灵。”程砚说。“我知道。”陆允川说。“你不知道，你现在脸上写着天塌了。”程砚说。

夏知遥在这一章里也不该只是一个被看的背影。夏知遥只看了一眼成绩单，就回座位整理错题。她没有炫耀，也没有安慰谁，手里的红笔一下一下，把错误改得很清楚。她有时会因为题做不出皱眉，有时会在朋友讲笑话时笑出声，有时也会因为家里电话而沉默很久。

她没有义务知道陆允川每一次停顿。就像陆允川也不知道她书包里那张没交的申请表，不知道她冬天手指容易冻红，不知道她回宿舍前常在水房多站一会儿。

期中成绩贴出来那天，后墙前比公告栏还挤。红笔圈出的排名像一个个小伤口，大家嘴上说随便看看，脚却都不肯走。

在旁观者眼里，陆允川因为分心掉了十几名，他盯着夏知遥稳定的名字，第一次觉得喜欢也会让人羞愧。这件事只是一天里很短的一折。可短并不等于无意义，它只是提醒陆允川：意义不能只从自己这边生长。

后墙下的垃圾桶里塞满废纸，几张揉皱的卷子露在外面。有人假装开玩笑说完了完了，声音却发虚。陆允川听着那些声音，忽然不想再把失控说得很浪漫。

叶澄晚自习后对他说，喜欢谁不是错，但别让别人替你承担你没说出口的戏。她说完就去交作业，留下陆允川站在门口，手里捏着一本卷了角的练习册。

成绩单贴出来后，班主任没有立刻批评，只让大家把卷子拿出来订正。可越是这样，陆允川越坐不住，仿佛那张纸还贴在后墙上，隔着一整个教室看他。

夏知遥在前排问老师一道化学题，问完后把步骤重新写了一遍。她的稳定不是天生的，陆允川看见她草稿纸上密密麻麻的计算，才知道自己曾经把别人的努力看得太轻。

程砚把他的错题本翻到空白页，说先别谈心，谈心不能提分。陆允川想反驳，程砚已经把第一道题抄了上去，笔画很重，像把他从发呆里敲醒。

晚自习下课后，陆允川没有立刻走。他把后墙那张成绩单看了一遍，又把视线收回来。名字仍在那里，名次也仍在那里，可他终于愿意从第一道错题开始，而不是从一场情绪开始。

旁观不是冷漠，而是把故事从一个人的胸口挪开一点。挪开以后，夏知遥有了自己的呼吸，程砚有了自己的烦躁，叶澄也有了不愿明说的担心。

陆允川不喜欢被旁观，因为旁观会让他的深情显得没有那么独一无二。但正是这种不独一无二，让后来的人生有了出路：大家都年轻过，也都误会过，不必把一次笨拙写成终身审判。

从叶澄的角度看，陆允川的心事并不神秘，只是太年轻，所以总以为别人都该看懂。他越沉默，周围人越不知道该不该靠近。

夏知遥也许注意到了一点异常，也许没有。她那天要处理的不是谁的暗恋，而是自己的作业、座位、朋友和回宿舍前必须打完的一通电话。

如果只写疼，这一章会很薄；加上成绩单、错题本、红笔、物理卷，它才重新有了重量。那重量不是压人，而是让读者相信他们确实在这里生活过。

陆允川把错题本拿在手里，先把物理卷订正完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把卷子订正完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把一道化学题重新列了三遍步骤，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

班级后墙里最明显的不是心跳，而是红笔盖被按上的咔哒声。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，晚饭里凉掉的米饭把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。错题本、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

从旁边看，陆允川的心事并不神秘。它只是少年人常有的笨拙：怕被看见，又怕完全不被看见，于是每个动作都带着一点别扭。

夏知遥也许注意到了一点，也许没有。无论哪一种，她都不该被写成答案。她有自己的订正物理卷，也有不想向任何人解释的沉默。

这一段把喜欢从成绩栏前拉回错题本。名次不会因为难过自动升上去，陆允川必须把羞愧折成一道道能订正的题。

如果只写疼，这章会很薄；加上错题本、红笔盖被按上的咔哒声和晚饭里凉掉的米饭，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在班级后墙发呆。”陆允川把错题本收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

这一章的火被旁人的目光压低了。压低以后，才看得见灰里的温度，也看得见每个人不被爱情叙事吞掉的完整生活。
