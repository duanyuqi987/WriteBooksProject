# 第017章 寒假

时间：2011年1月至2月

地点：石桥镇、青阳县城

卷次：卷一：初见秋风

寒假是突然就空的。

前一天晚上还在教室里背书——期末的最后一天——教室里日光灯嗡嗡响——同桌还在问"这道题的受力方向到底往哪"——第二天早上一睁眼——不用起床了。宿舍没了。食堂没了。耳边只有他妈在楼下走来走去的拖鞋声——啪嗒啪嗒——从厨房走到院子——从院子走到厨房。他缩在被子里——被面是蓝格子——被里是棉布——厚——压在身上暖和。睁开眼看了看天花板——没有日光灯——是白墙和一道裂痕。又闭上——过了好久——可能是半小时——又可能是两个钟头。楼下有人说话——是他妈和隔壁刘婶——"你家临风放假了？""放了。""考得怎样——""不清楚——还没告诉我。"

没人催他起床。他翻了个身——脸埋在枕头里——枕巾上有洗衣粉的味。

第二天他去镇上逛。石桥镇是一条主街——青石板铺的——从头到尾走十分钟。快到过年了——街上有卖年货的——挂在墙上的春联、铺在地上的年画——红纸、铜版纸、轴子——对联有用金粉写的——"迎春接福"四个字在太阳底下闪。街口卖豆腐的胖阿姨今天没出摊——天冷——买菜的人少。五金店门口坐着一个老头在晒太阳——帽子扣在脸上——椅子是竹椅——仰躺在墙上——只见膝盖和手——手抱在胸前——手背上有褐色的老年斑。杂货铺子门前摆了一排烟花炮仗——红皮的大地红、花皮的电光炮。旁边是一桶一桶的烟花——上面标着"火树银花""彩珠筒"——纸壳色彩鲜亮——好看。两个小孩蹲在炮仗前面——手指在各种花色上来回摸——眼珠子转——嘴巴微微张着——手指从左边划到右边——没敢碰。老板在里面喊了一声——"不买别动手——"小孩把手缩回去——站起来——跑了。跑了几步回头看了一眼——然后又跑。

第三天他开始无聊了。坐在堂屋里——翻了几页物理课本——几页纸——然后放下。看电视——按遥控器——中央一套、二套、三套——放了半个钟头——关了。在院子里站了会儿——看天——天是灰白的。伸出手——手背上起了皮——冬天皮肤干燥——虎口那裂了一道小口——不疼。他妈说擦点蛤蜊油——他没擦。

然后就开始了漫长的寒假日子。每天下午骑车出门。

他那辆车是旧的——凤凰牌二八大杠——他爸淘汰下来的。车架上的漆是墨绿色——后叉那里的漆掉了一块——露出锈褐。车座是人造皮革——裂成几瓣——里面发黄的泡沫从裂缝翻出来——坐上去泡沫会轻微地侧移。车铃铛锈了——按不下去——只能用手拧——他也没拧过——觉得按铃太用力骑过去自然会让路人听到车轮的声音——或者不用让——石桥镇没什么人。刹车还行——后刹的刹车皮还厚——雨天刹不住——晴天还行。他把车从门洞里推出来——轮胎碾过门槛——咯噔。右腿一跨——坐上去。踩一脚——链条咯噔一下入位——滑行——脚用力往下蹬——链条转起来了。

骑着车走。石桥镇出来是一条省道——省道很窄——只有两条车道——两边是杨树——冬天杨树秃——树枝细密——像一群竖着的指头——指的方向都是右侧——因为长年刮北风。路面上——白色标线已经斑驳了——水泥地上补过沥青——一块块深黑。汽车不多。偶尔一辆中巴经过——车体蓝色——轮子溅起路面上的石子和灰尘——噗——从他身边过去——排出一阵黑柴油烟。他往右偏——车轮压在白色标线上骑——线只有几毫米宽——他能在上面骑行二十多米不掉——轮子的橡皮抓地力还不错。

第一周骑了两次县城。沿着省道——经过水泥厂的东墙——水泥厂两根大烟囱——正在冒烟——浓白的烟往上直——升到半空——然后被风吹斜——灰白色——消失在冬季天空里。石桥河也经过——河水量少——水流极慢——水面上有些冰晶——河对岸有个老人在放羊——羊趴在枯草丛里不动——老人拿竹竿在地上敲了两下——羊站起来——走了两步——又趴下了。他继续骑——大概二十五分钟后到县城的边缘——路上看到一中操场——操场上没人——煤渣跑道落了一层薄灰——篮球架上挂了个破网——一个篮圈歪了。

他去县城。不买东西。不去新华书店。不找同学。就骑着车在街上绕一圈。县城的街道是棋盘式——不太好绕——他先经过中心街——工商银行门口——那个旋转门在转——几个穿羽绒服的人走出来——嘴里的白气一股股的。再经过菜市场——下午的菜市收了大半——地上是烂菜叶——和冰碴子混在一起——踩上去嘎吱响。然后经过老街——骑到巷口。减速——慢到车轮快刹住了。往里看一眼。

冬天老槐树还是光秃秃的。树枝朝天——主干裂着——裂纹里夹了些残雪——两个星期前的雪还没全化——白。树根上那堆湿叶——冻得硬邦邦——黑色的——被冻在青石板上面。巷口的穿堂风大——电线吹得呜呜响——电线是铝线——绷在电杆之间——风过的时候有金属的细响——嗡嗡——像刀在钢管里擦过。青石板路面冻着一层薄冰——浮在石面上——透明——能看到冰下青石的老纹。

那扇对开的木门还是关着的。

有时候他在旁边停一下——假装鞋带散了。坐到路边的石墩上——蹲在自行车前——低头——把右脚的鞋带拉开——重新系。系完右脚——左脚也松开——也系——故意把交叉的绳结揉慢些——抽紧——拇指拉过耳环——拉成蝴蝶。两边的蝴蝶大小不一样。系完——站起来——拍拍裤子上沾的炮仗灰——推着车往前走几步——骑上去——用力蹬——先慢——快出巷口的时候回头看了一下——那扇门还是关着的。

有一次他骑过去。巷口有动静。

苏晚晴在巷口收被子。就她一个人。她穿一件红格子棉睡衣——衣摆到膝盖——袖子很长——手全裹在袖子里只露五个手指尖。头发没扎——披在肩上——发尾有一点卷——大概是扎马尾久了留下的自然弧度。她怀里搂着一床被子——蓝格子棉被——被套的蓝格子和顾临风那床洗得发白的被套同一种格子——也许是在同一家店扯的布——也许是县城的纺织厂出的同一个花色。她踮起脚——双手把被子往竹竿上搭——竹竿的高度差不多到她的额头。手劲不够——被子在竹竿上滑了一下——差点掉在地上。她去捞——手指隔着被子夹不住——最后还是抱住了。然后用下巴抵住被子——双手举高——翻过竹竿——被子搭上去了——在竿子上是斜的——她拽了拽角落——弄平。被面上有处淡黄——洗不掉的旧渍——可能是什么时候的茶汤——放在了里边。

然后她抬起头——往巷口的方向看了一眼。眼睛扫过来的方向——正对着巷口的石阶——一个影子。她的目光在巷口停留了一下——大概一两秒。

顾临风已经骑出去了。他在她抬头前的那一瞬间——猛蹬了两脚——车轮转得飞快——链条咯咯响——拐弯的时候车把晃了一下——差点撞上路边的垃圾桶——绿色的铁皮垃圾桶——上面印着"爱护环境"。他脚撑地——车停了——重起——车身稳过来——继续骑。手心里是汗。手指拉刹车的时候滑了一下。心跳很快——在胸腔里嘭嘭地——不是因为差点摔倒。

从那以后他再经过那条巷口的时候——不减速。直接骑过去。但在巷口拐弯的地方——头会飞快地往右偏一下——像被什么钩住了——然后立刻转回来——看路——脚不停。一秒钟。眼球的快转。

那个红色格子的棉睡衣——棉裤——没扎的马尾——怀里抱着的蓝被子。记在了脑子里。

寒假一共有二十五天。他骑了大概十次县城。每次都经过那条巷子。有时候在巷口骑得很快——不让头偏。有时候骑到那里就想——算了——然后头还是偏了一下。

老槐树在正月的某一天——被人贴了一张红"福"字——方形的红纸——金粉字——倒着贴。墙角堆了一摞旧砖头——不知道谁家砌墙剩下——砖头裂了。窗台上多了几个柿饼——柿饼上结着白色的果霜——在冬天看上去像盖了层薄雪。巷子里偶尔有狗跑出来——黄色的土狗——对着他叫——汪汪——他骑车不理——狗跟了几步——跑了。

寒假第二个星期。他上午在家写寒假作业。英语作业——抄单词每个五遍——抄到第六课——手腕酸了——甩了甩——继续。下午骑车出门。傍晚回来——天色是灰橘——太阳掉到后面山腰——山的剪影很硬。推车进门洞——把车靠墙停好——进屋——他妈问他去哪了——"出去逛逛——"

"天天逛。镇上有什么好逛的。"

"随便骑骑。"

"那你顺便帮我去买包盐。街口那个小超市——你看看还开没开着——"他嗯了一声——又推车出去了。买盐——一块二一包——找的零钱三枚一毛硬币——揣在兜里——跑起来在兜底叮叮当响。

第三个星期过年了。他家门贴了对联——红纸黑字——上联"门迎百福"——下联"户纳千祥"——是他爸在镇上买的——两块钱一副——印刷的——字边上有些金粉闪点。堂屋里摆了一盘瓜子——一盘花生——一盘糖果——水果硬糖——透明的那种——绿色的薄荷味——橙色的橘子味——他小时候爱吃——现在不了——还是抓了一颗放在嘴里——甜。有人来串门——隔壁刘婶——拿着一个搪瓷盆——"这是过年糕——端过来给你们尝尝——"他妈接过去——"你太客气——"。他倒了茶——说"过年好——"然后回二楼。坐在桌前。翻那本牛皮纸本子。翻了两页——抽出那张写了诗的——《初见》——又在看——看到"手心融尽一痕霜"——把纸慢慢折回去。什么时候能给她看——不知道。想了——然后不想。

寒假最后一天。他把作业本摊在床上挨个数——语文、数学、英语——英语作业少了两本——补交就完了——找——床上——翻了被子和枕头——枕头上落了几根头发——短。没有。桌上——作业本堆里——拿出来一本本翻——不是。书架——把词典挪开——词典很重——旁边没夹着。最后在床头和墙的夹缝里找到了那两本英语作业。掉进去的。抽出来——纸面上积了灰——灰盖住了扉页上的名字——他用湿抹布在封面上擦——灰变成了灰水——顺着抹布纹流下。放在窗台晾干——窗台上有道水印——下午才全干——压出几道长波浪。干了之后——抖了抖——塞进书包——书包拉链拉到最顶。

明天开学。高一第二个学期。理科班。

他睡前把闹钟定到五点四十五。早读前多背十个英语单词——至少。关了灯——屋里暗了。窗外的石拱桥在月光下发白——石头桥身——月亮在天上被云罩了一半——另一半清白的——云很慢——在月亮前面滑。楼下堂屋的门——夜风推了一下——门框上贴的那张对联纸——在黑暗中啪啪轻响——红纸白墙。他听着这个声音——想着明天的教室——理科三班的教室——三楼的走廊走到头是哪扇门——门的颜色——座位表贴没贴——第二排靠窗会坐谁。然后闭上眼睛——呼吸平稳。

闹钟在五点四十三的时候提前亮了他手机屏幕——他没按——已经在黑暗中睁开了眼。
