# 第090章 秋日诗会

时间：2011年11月下旬，星期六下午

地点：人文楼三楼小礼堂

卷次：第四卷：暑假的信

十一月下旬，诗社办了一场"秋日诗会"。何南星在公告栏上贴了海报，手绘的——几片银杏叶的简笔画，配上两行毛笔字："秋天快过去了，带一首诗来。"

地点在人文楼三楼小礼堂。那间教室不大，平时用来开班会或者小型的学术讲座，能坐五六十人。何南星带着几个社员提前去布置——把桌椅摆成环形，中间空出来做"朗读区"，墙上贴了几张旧报纸，上面用毛笔抄着古诗。

何南星说这叫"氛围"。"你读诗的时候看着白墙没感觉，但如果你看到墙上有一句'落霞与孤鹜齐飞'，你念下一句的时候声音都会不一样。"

林玉起帮他在报纸上抄了王维的《山居秋暝》——"空山新雨后，天气晚来秋"。他的毛笔字不算好，但抄古诗时笔画会自然放慢，字就有了工整的意思。何南星说好。诗社的秘书叫孙晓月，中文系大二的一个女生，做事认真，拿了签到表挨个登记。她给的每人发了一张编号纸条，说今天要评"最佳朗诵"，匿名投票。

下午两点，人陆陆续续来了。比上学期多了不少人——何南星的海报贴到了教学楼和食堂门口，有人是被"带一首诗来"那句话吸引的。郑学然来了，带了自己的旧体诗，写在方格稿纸上，从书包里掏出来的时候小心翼翼地展开。副社长罗远也来了，他是历史系大三的，人很随和，什么诗都能点评两句。会场里坐了大概四十个人，比上学期翻了一倍。

林玉起坐在靠后的位置，手里拿着笔记本——里面是准备读的诗稿。他一直在找一个人。

她不在。

何南星开场说了几句话，然后按编号顺序让人上台。有人念了海子的《面朝大海春暖花开》，有人念了顾城的《远和近》，有人念了徐志摩的《偶然》，也有人念了自己写的——好坏参差，但何南星带头鼓掌，每首结束都他先说一句"谢谢"。

第七个人上台的时候，后门被轻轻推开了。

苏安晴走进来，贴着墙站，没有往前挤。她今天穿了那件浅蓝色的短袖——天气凉了她还穿着短袖，大概是走得太急忘记了冷。她在人群后面踮了一下脚，往台上看了一眼。

林玉起看见她了。他的心跳忽然加快，然后变慢了——他意识到她大概是来看诗会的。也许是她从哪张海报上看到的，也许是宋乔告诉她的（宋乔参加了校刊的社团，消息灵通）。

轮到他上台的时候，他深吸了一口气。手里捏着笔记本，纸页被手心汗湿了一点。他站在"朗读区"——一块被空出来的木地板，有点旧，踩上去微微发翘。头顶的日光灯照下来，把木地板的纹路照得很清楚。

他翻开笔记本，找到了那首诗。然后开始读。
