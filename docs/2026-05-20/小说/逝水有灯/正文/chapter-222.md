# 第222章 重回县一中

所属部卷：第五部：逝水有灯
时间：出差顺路回乡
地点：县一中翻新的校门

## 诗歌插页 P222：原创仿唐绝句

新门高处映晴光，
旧径重来草色长。
三楼仍有读书响，
不为当年一页伤。

校道两旁的树长高了，地面重新铺过，雨水顺着排水沟往操场方向流。陆允川沿着主路慢慢走，胸前访客牌轻轻碰着衬衫扣子。高一教学楼外墙刷成浅灰色，走廊栏杆也换了新的。

他找到三楼东侧教室。门牌不是当年的蓝底白字，换成了透明亚克力。透过窗户，里面正在上自习。学生们低着头写题，黑板右上角有倒计时数字，粉笔字清楚得让人心里发紧。

陆允川没有靠太近，只站在走廊尽头。教室里没有夏知遥，也没有十七岁的自己。坐在那里的是另一届学生，他们有自己的排名、作业、喜欢的人和不能说出口的烦恼。这个发现让他有一点失落，也有一点轻松。

下课铃响后，学生涌出来，有人背单词，有人讨论午饭。一个男生差点撞到他，赶紧说对不起。陆允川说没事，往墙边让了让。

他没有拍教室照片。手机拿出来又放回去。某些地方如果真的重要，就不必强行留影。它们已经把你送到这里，剩下的路要你自己走。
