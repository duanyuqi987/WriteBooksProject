# 记忆快照

## 当前总体记忆

《逝水有灯》是一部温柔现实青春成长小说。它从县一中初遇写到高考后空落，再写到大学里新的喜欢与失衡，最后在多年后整理旧日记和诗歌时完成释怀。

## 本轮精修记忆

- 第001-006章已完成首轮减法精修。
- 开篇不再用大量“多年后解释回忆”，改为从陆允川到县一中报到、三楼教室、旧草稿纸、程砚的薄荷糖、夏知遥的水杯和新课本书皮展开。
- 夏知遥在第004章获得旁观视角：她有公交、家庭、怕被起哄、座位边界和自己的学习压力，不再只是陆允川日记里的秋天。
- 程砚的功能从“说正确话”调整为用玩笑、饭盆、薄荷糖和草稿纸提供粗糙陪伴。
- 第007-030章已完成第二轮减法精修，五组事件从抽象总结改为连续校园现场：新课本书皮、分班红榜、物理作业互改、操场背单词、楼梯间归还黑色笔记本。
- 夏知遥在第009、014、019、024、029章获得更稳定的旁观视角：她不是“被照亮的人”，而是在处理课业、家庭电话、边界感和自己的紧张。
- 程砚在第007-030章继续承担粗糙陪伴功能，用薄荷糖、冰棍、饭盆、红笔和玩笑把陆允川从过度解释里往回拉。
- 第031-060章已完成第三轮减法精修，六组事件从模板回忆改为可见现场：走廊旧伞、成绩红榜、深蓝诗抄本、元旦节目单、高三倒计时、最后一场模拟考准考证。
- 夏知遥在第034、039、049、054、059章继续获得旁观视角：她会处理未具名善意的不适、成绩压力、舞台紧张、高三疲惫和模拟考失误，不承担陆允川的单向叙事。
- 程砚在第031-060章的功能更具体：用毛巾、热水、作文模板、考后玩笑和不太好听的话，把陆允川拉回现实生活。
- 第061-090章已完成第四轮减法精修，六组事件从“回忆自我感动”改为可见现场：校门口水瓶、东屋墙角试卷、网吧聊天窗口、毕业照边缘、河边迟到表白、镇上邮政所录取通知书。
- 夏知遥在第064、069、079、084、089章继续拥有主体视角：她会被家人接出考场、整理自己的试卷、保存毕业照、处理被表白后的压力和边界、拿到自己的通知书。
- 程砚在第061-090章承担现实边界功能：催陆允川出门、陪他网吧通宵、提醒“不知道就先别发”、在表白后确认是否逼迫对方。
- 第091-120章已完成第五轮减法精修，六组事件从大学开端的“回忆解释”改为具体现场：县城小饭馆升学宴、新生宿舍与南湖、图书馆前诗社传单、南门封校取社刊样张、食堂二楼第一次吃饭、女生公寓楼下等待。
- 夏知遥在第094章获得后续旁观视角：她在亲戚饭桌上处理录取、亲戚问话和礼貌祝福，不再作为陆允川聚餐受伤的缺席符号。
- 苏晚枝在第104、109、114、119章获得主体视角：她负责登记、社刊、吃饭边界和教室打扫，感谢陆允川的细心，但明确不让礼貌和合作变成承诺。
- 祁野开始承担大学线现实边界功能：用宿舍、饭卡、澡堂、炒饭和难听但准确的话提醒陆允川，等待和细心不能变成向对方讨债。
- 第121-150章已完成第六轮减法精修，六组关系失衡事件从模板解释改为具体现场：二号教学楼空教室、男生宿舍阳台连续电话、校园超市物资袋、南湖凉亭劝告、图书馆东门台阶拒绝、东门外小饭馆争执。
- 苏晚枝在第124、129、134、144章继续获得主体视角：她处理签名、社团会议、买物资和明确拒绝，边界来自具体压力，不是冷淡标签。
- 叶澄在第136-140章不再只承担说理功能，而是带着社团表格、活动流程、主持词和自身疲惫介入，提醒陆允川别把别人写成自己的钥匙。
- 祁野在第127、128、148、149章继续承担现实边界功能：抢下连续拨号的手机、指出“催债”、在饭桌争执后把水杯带出来，但也表现出朋友会累。
- 后续章节应延续此标准：少总结，多场景；少命运感，多具体行动；“多年后”章节也必须落到旧物、文件袋、旧照片、同学群、旧书箱等可见载体。
