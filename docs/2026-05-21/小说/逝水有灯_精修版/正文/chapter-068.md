# 第068章 墙角的试卷之后

所属部卷：第二部：六月以后
时间：高考后第二天
地点：陆家小院的东屋

## 诗歌插页 P068：原创仿宋小令

风入旧窗轻，
纸上有余声。
欲问前程何处去，
水边灯影自分明。
回首不须惊。

晚上吃完饺子，陆允川回到东屋，终于决定把床上那堆试卷彻底清理掉。

他从厨房拿了一个纸箱——装过苹果的，箱底还有一片干了的苹果叶子。他把纸箱放在墙角，开始把试卷往里面放。

首先放进去的是高三的模拟卷。这些卷子他反复做了两遍，边角已经磨得发毛。数学模拟卷第二张，立体几何那道题他后来去办公室问了老师，终于搞清楚了辅助线的画法。他把这张放在最上面——它算是一个证据，证明他确实努力过。

然后是高一高二的期中期末卷。这些卷子分数参差不齐，好的时候一百一十多，差的时候九十出头。有一张英语卷只考了八十九分，作文被老师画满了红圈。他当时在卷子背面写了一段很矫情的话，大概意思是"我一定要超过平均分"，后面加了三个感叹号。现在看起来，平均分他都记不得了。

手机响了。程砚在群里发了一张照片——他自己的书桌，上面也堆着一堆试卷。配文："我妈说今天之内不收拾完就全部卖给收废品的。"

叶澄回复："你拍张照的功夫已经能收完三张了。"

程砚："你怎么每次都这么精准打击。"

叶澄："因为你不是真的需要安慰，你只是想让人看你有多惨。"

周长海："被看穿的感觉好吗"

程砚："滚。"

陆允川笑了。他回了一条："我的也还没收完。正在往箱子里放。"

程砚："你准备留着还是扔？"

陆允川："留着。"

程砚："我也是。我妈说留着占地方，我说我以后说不定要拿出来怀念。"

叶澄："你们男生能不能不要这么矫情。"

程砚："你先说你是不是全扔了。"

叶澄隔了半分钟才回："……留了几张。"

程砚："哈哈哈哈哈你看！"

叶澄没有回复。过了一会儿她发了一张照片——她的"留"那摞大概只有五张。最上面是一张英语作文，标题是"My Dream"，作文分是满分。下面是批语："论点清晰，逻辑严密，语言流畅。继续保持。"

陆允川放大照片看了看。他记得这篇作文——叶澄写的是想当老师，教孩子们用语言去看更大的世界。她那天在烧烤摊说过一样的话。

他把手机放下，继续往箱子里放试卷。放到一半，手指碰到那张英语卷上的铅笔印——那个被擦过的"夏"字起笔。他这次没犹豫，直接放进了箱子。

有些东西不需要被特殊处理。它就在那里，是一张纸，曾经写过一个字，后来擦掉了。仅此而已。

箱子塞满的时候，他拿了胶带准备封口。手停在半空中，又放下来——他去书桌上拿了一支马克笔，在箱盖上写了四个字：

"高中三年。"

字写得很潦草，跟程砚那张纸条上的字差不多。他把胶带贴上，纸箱推到墙角。

站起来的时候膝盖咔嗒响了一声。他蹲太久了。

推开房门，院子里已经黑了。母亲在看电视，声音从客厅漏出来。他走到院子里，在井边洗了一把脸。井水冰凉，顺着下巴滴进领口。

他站在院子里抬头看天。县城的夜空能看到星星，不多，但比城市里多。北斗七星挂在天顶，勺子把指着北边。

母亲从客厅探出头来："外面蚊子多，早点进来。"

"马上。"

他抹了一把脸上的水，转身回屋。路过厨房的时候看到灶台上还剩了半盘饺子，旁边扣着一碗醋。他用手指捏了一个塞进嘴里，凉的，但味道还是那个味道。

回到东屋，他看了一眼墙角那个纸箱。封好了，推到角落里，上面没有任何标记——除了他写的"高中三年"四个字。

他坐在床上，打开手机，看到夏知遥在班群里发了一条消息——这是她高考后第一次在群里说话：

"有人知道教育局在哪吗？我的准考证地址要改。"

叶澄回复了她，附了地址和需要带的材料。夏知遥回了一个"谢谢"。

陆允川看着这条消息，手指在屏幕上停了片刻。然后他把班群设成了免打扰，把手机充上电，关了台灯。

黑暗里，墙角那个纸箱安静地坐着。三年装在里面，封了胶带，明天也不会被打开。
