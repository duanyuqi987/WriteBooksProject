# 第041章 诗抄本之前

所属部卷：第一部：初秋三楼
时间：高二秋天
地点：教室后排的课桌抽屉

## 诗歌插页 P041：原创现代诗

我把早晨放低，
让一行诗从课桌边慢慢经过。
你的名字没有被我喊出，
只在硬皮抄本里被铅笔写在目录页的角落旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写诗抄本抵达之前的坡度。高二秋天的一个下午，语文老师布置了一项作业：每人准备一个硬皮抄本，每周抄写一首诗，可以抄课本、课外诗集或者自己写的。期末要交。陆允川放学后去校门口书店买了一个黑色硬皮本，封面是仿皮纹的，摸上去有细密的颗粒，边角包了铜色的金属护页。书店老板从柜台下面翻出来给他——因为黑色硬皮本放在最底下一层货架，被一摞彩色笔记本压住了——老板一边往外抽一边说"这个质量好，就是销量不好，学生都喜欢买带卡通封面的"。

他在本子的扉页写上"诗抄本 高二(一)班 陆允川"。写完以后翻到目录页，空的——只画了横线。他看着那一页空目录想了一秒钟：如果这本本子里抄的每一首诗都能找到一个对应的意向——窗外法桐的落叶、雨天值日的拖把、操场边的夕阳——那他就等于是在用诗做暗号，每一个暗号对应到一个只在两个人之间成立的编码系统。这个想法出现以后他自己把它按住了——因为那套编码的收信人根本不知道有这么一套编码簿存在。但没关系：他抄诗不是为了出被动接收，他抄诗是为了在自己十七岁的日记旁边开一扇隔壁的窗——日记记录事实，诗记录事实下面层的涌流。

程砚发现他买了诗抄本以后问他："你这本能写多少页？""一百二十页。""一周抄一首，一学期大概二十周，你最多抄二十首。剩下的空白页你打算怎么用？"陆允川没想好。程砚替他答了："剩下那一百页你应该用来写小说——把你每天看人家看了几秒、说了几个字、捡了多少本子全都写成章回体小说，封面不要叫诗抄本，应该叫纪实文学。"

叶澄听说诗抄本的事以后反应不同。她过来借本子翻了一下——封面的铜色护页在日光灯下反出一道细长的亮光——看到扉页上是空的，只有一句目录注释。她说了一句话："抄诗的时候不要选太压抑的——你本身已经够闷了，再抄顾城的诗你会整个人沉进桌子里去的。"陆允川反问："那你建议抄什么？""抄夏天的、操场的、有人笑的。""那抄什么？余光中的？""什么都可以——只要里面有个'灯'字。""为什么是灯？""因为暗的地方需要灯。"说这话的时候叶澄用手指在桌面上画了一个圆——大概是一盏灯的形状——然后把手收回去了。

他把这句话记在本子里——不抄诗，只记了一行的笔记：叶澄说，暗的地方需要灯。然后合上本子。接下来一周他需要交出第一首诗。他还没想好抄什么，但他确定目录的某个位置会留着一行空，那行空不标诗名、也不标页数，只留着——对应一项不便说明标题的私人索引。
