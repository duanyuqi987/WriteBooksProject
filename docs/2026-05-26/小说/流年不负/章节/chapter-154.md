# 第154章 赵一鸣知道了
时间：2018年6月
地点：电话
卷次：卷八：旧诗新读

六月初。楚州开始热了。宿舍没有空调，顾临风把电风扇开到最大档。扇叶嗡嗡嗡地转，把桌上的物理卷子吹得哗哗响。

手机响了。来电显示——赵一鸣。

他按下接听。还没来得及说"喂"——电话那头的声音已经炸出来了。

"我——靠——"

他把手机拿远了几厘米。赵一鸣的声音穿透力很强——从北京到楚州，跨了大半个中国，音量一点没减。

"你什么情况！你俩在一起了你都不跟我说一声！还是我从许若琳那儿听来的！"

"你这不是知道了。"

"我知道应该从你嘴里知道！从许若琳那儿算怎么回事——她说苏晚晴发了个朋友圈——我就想什么叫'我们在一起了'——我点进去一看是你们两个站在月湖边拍的影子——两只鞋。我一看那个运动鞋我就知道是你。"

他低头看了看自己脚上的运动鞋。灰白色的。穿了两年了，鞋底磨平了一块。确实是照片里的那双。

"你什么时候看到的。"

"昨天晚上。昨晚我看到之后马上想给你打电话——当时快十二点了，我怕你睡了。今天一早就打。"

"你不用上班？"

"我在洗手间打的。"

他无语了一秒。

赵一鸣在电话那头深吸了一口气。然后发出了一声笑——不是平时那种嘻嘻哈哈的笑。是另一种——声音沉了一点。

"说真的。哥们。"

"嗯。"

"等了八年。"

"嗯。"

"值了。"

这两个字——顾临风的妈妈在电话里说过，他在月湖边的亭子里想过，苏晚晴在微信里打过。现在赵一鸣又说了。

"你知不知道你这么说话让我有点不习惯。"

"怎么——我不能正经吗。"

"不是不能。是太少了。"

赵一鸣在电话那头干笑了两声。然后语调忽然降下来——认真了。

"你还记得高一那年。你第一次告诉我苏晚晴这个名字的时候吗。"

"记得。九月下旬。体育课。我们在操场上坐着。你问我有没有喜欢的姑娘。我说——三班有个叫苏晚晴的。马尾上总扎着一条浅蓝色发圈。"

"你那时候连人家名字的全称都是查了座位表才知道的。"

"嗯。"

"我当时说什么来着。我说——三班的？那你追啊。你说——不追。怕影响她学习。我当时差点把篮球砸你头上。"

"你确实砸了。"

"你躲开了。"

"我没躲。球从耳朵边上飞过去了——擦了一下。肿了两天。"

赵一鸣笑了一声。然后又沉下来。

"其实我挺佩服你的。忍了八年。从高一忍到大四。从大四忍到工作。中间一点没断过。我换了好几个女朋友了。你没换过。"

"我没女朋友。"

"你有——你在心里有。你的女朋友在月湖边——在成绩单上——在你的蓝色笔记本里。这些不算吗。"

他没接话。扇叶的风吹过来。热风——比不吹还闷。但他没去调风扇。

"赵一鸣。"

"嗯。"

"谢谢你。"

"谢我什么。"

"大学那几年。你每次从外面回来——都会提苏晚晴。'今天在三食堂看到苏晚晴了''她新换了副眼镜''她参加了学校的演讲比赛'——你以为我不知道你在干什么。"

赵一鸣在电话那头安静了片刻。

"你知道。"

"知道。你帮我盯着她。帮我打听她的消息。你从来没说过——但我都知道。"

"知道你怎么从来不提。"

"不用提。我记在心里。"

扇叶嗡嗡嗡地转着。桌上那张物理卷子从桌面上滑到了地上。他没弯腰去捡。

"还有一件事。"赵一鸣的声音更低了。"大四毕业那年。苏晚晴签了楚州一小的合同。学校发了公告。我拿手机拍了照发给你——说'随便发发'。"

"我记得。"

"你不是随便看的。你收到之后过了三个礼拜——你也签了楚州的学校。"

"我的专业本来就是师范。"

"师范毕业生全省哪里都可以去。你偏偏选了楚州。"

他没反驳。

"赵一鸣。你是对的。"

"什么。"

"高中那会儿你说过一句话。你说——以后你俩肯定在一起。我还记得那次。是高三上学期。体育课下课后。我跟在你后面回教室。你忽然回过头跟我说'你俩以后肯定在一起'。你是怎么知道的。"

赵一鸣想了想。"她看你的时候。你没有看她。但她看你的眼神跟你一模一样。"

"你怎么看出来她看我——"

"苏晚晴的座位在第一排靠窗。她前面的窗户能反光。反光的范围刚好是你那条走廊。每次你从走廊上经过——不管是在跟谁说话还是自己走——她都从窗户上看你。只看大概两秒。然后低头继续写字。"

两秒。跟他在笔记本上看到她的时间一样——三秒看诗，两秒看他。

"你怎么从来没告诉过我。"

"因为你每次跟我说苏晚晴的时候——耳朵都红到耳根。我怕告诉你这个，你整个人会烧起来。"

他没说话。耳朵尖的温度开始升高。

"赵一鸣。"

"嗯。"

"你真够义气。"

"不然呢。你以为我是看着你追不到才不管的。我能帮多少帮多少。虽然能帮的不多——大学生活费不够我找你借过好几次——"

"你借了三百。还了一百五。剩下的一百五我不要了。"

"别。这钱我记着。等你结婚的时候包红包还你。"

两个人在电话里安静了一会儿。不是尴尬——是那种聊了很多年之后老朋友之间会有的一种沉默。不需要特意说什么了。

"说点正经的。"赵一鸣的声音收起了戏谑。"你跟她在一起——好好对她。她是个好姑娘。比我认识的所有姑娘都好。你知道我等这一天等了多久吗——不是等你告诉我。是等她跟你说——好。"

"你等她说好。"

"对。高三毕业那晚她说了——'我需要想一想'。我想她需要想多久。想过要一年。两年。三年。她想了六年。"

"不是六年。她也等了。"

"我知道。从高二开始。你坐在后排看她的时候——她也在看你的笔记本。"

他又说不出话了。空调没有。风还是热的。风扇的嗡嗡声填满了整个房间。他想起了高二那年——自己把笔记本翻开摊在桌上。低头写"那年九月风初凉"的时候。走廊上有脚步声。他没抬头。脚步声停了——他以为是有人路过。继续写。把"长"字上的那一捺拉得格外用力——捺得纸都凹了进去。

那脚步声是她的。

"你什么时候回来。请你吃饭。"他说。

"暑假吧。回去找你们。叫上许若琳。"

"好。"

"挂了。洗手间的换气扇太吵了。再不出来我同事以为我掉厕所了。"

"你先出来的。"

电话挂掉。风扇还在吹。他把地上的卷子捡起来。卷子上有一道题——一个质量为2kg的物体在水平面上受到一个8N的拉力。求加速度。F=ma。a = 8 / 2 = 4米每二次方秒。他把答案填上。

手机又亮了。不是微信——是赵一鸣的短信。短信。2018年了。居然还有人发短信。

"刚才忘了说。你的诗我偷看过。高二那年翻你笔记本的时候——第一页就是《初见》。我没看完——怕被你发现。只看到了第一句。那年九月风初凉。剩下的没敢看。你把秘密藏了八年——我帮你藏了七年。现在不用藏了。"

顾临风盯着手机屏幕。然后把手机翻过来扣在桌上。屏幕朝下。扣了大概十秒。翻过来。打了两个字。

"谢谢。"

然后犹豫了一下，又打了四个字——

"兄弟。真的。"

发送。手机放进口袋里。风扇开到二档。嗡嗡声更大了。他站起来去关窗户。窗户外面的月亮很圆。六月十五——阴历还差几天。月亮是个大半圆，右上方缺了一小块。月湖水面上应该也有这个月亮。

他想着。苏晚晴也在看这个月亮吗。
