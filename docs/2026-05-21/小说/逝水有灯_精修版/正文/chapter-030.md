# 第030章 第一次说话多年后

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：教室前门和讲台之间

## 诗歌插页 P030：原创七言小诗

初语犹在人已移，
六字如山各自知。
年少不知寻常重，
却将捡本比旧诗。

多年后陆允川重写第一次说话，先把自己写得很委屈——写到他蹲下去帮她捡作业本的时候觉得那是命运在门框边安排的一个位置，写到手背碰到她手指的零点一秒等同于整个宇宙压缩成一道光，写了一大段关于"手指温度"的比喻，什么"像春天的河水漫过河滩的石子"。写完以后自己看了两遍，把那一页按住——不是撕，是用手掌整个地盖住纸面，像在给一锅滚烫的汤盖上锅盖。书桌上的台灯是暖黄色的，灯罩边缘积了一层薄灰，他写这段的时候窗外的城市正在下小雨，雨打在空调外机上发出嗒嗒嗒的金属回音。这是第一次他在重写时没有删自己十七岁的心事。因为他终于承认了一件事：那六个字尽管在时间长度上不值一提——在物理老师的秒表上连一个完整的秒数都占不到——但在他的生命周期里确实占据了一个节点。那是他和她在没有任何第三方建议、没有任何排练、没有任何理由的情况下自然发生的第一段对话。他不小心撞上了她蹲在地上的背影，然后嘴巴自己动了。发音清楚——"你捡吧"三个字没有破音。没出错——没说成"你捡"，也没说成"让给我"。也没有事后让自己想钻地缝——他说完以后虽然耳朵热了，但脚还站在地上，没有跑开。十七岁的他再也做不出比这更好的六个字了。他后来参加过大型演讲、做过工作汇报、在婚礼上说过致辞——没有哪一次比那六个字更让他满意。

多年后他在办公楼电梯里跟人抢按开门键时，手指碰到了对方同事的手——一个三十三岁的已婚男人，手上戴着婚戒，戒指是铂金的，在电梯灯下闪了一下。电梯间里弥漫着中央空调吹出来的微凉空气和淡淡的消毒水味，地板上有上午保洁拖过后留下的半干水痕。那人说"你按"，他说"你按"，两人在电梯门要关的瞬间同时按下去，四根手指在同一个按钮上碰了一下，然后各自退开。面无表情，各看各的楼层指示。那人还按了手机看了一下时间，手机屏幕的光映在他脸上，解锁界面是一个小孩的照片。他忽然在那零点一秒的指关节相碰里想起了高一的那次。但没有任何心跳——不过是两个人同时在同一个位置按同一个按钮，纯属概率——每天在全世界的电梯里大概发生几百万次。人在三十三岁的时候终于能做到面无表情地处理肢体碰触，甚至可以在一秒后把这件事完全忘掉。而十七岁的时候——十七岁他花了三节晚自习来回想她手指的温度，并且在日记本里用了"宇宙"和"光"做比喻。三十三岁的他回想起来，觉得自己很好笑，但没有觉得丢人。因为十七岁的"宇宙"是真宇宙——只是他的宇宙在那一年只有教室那么大。

程砚后来问他："你还记得那六个字吗？""记得。""说实话，一般人问'你记得跟谁说的第一句话吗'，大部分人早忘了。""因为一般人的第一句话太长了——'你叫什么名字''你从哪个初中毕业的''你现在住校吗'——太多了。六个字好记。"

叶澄在修改稿上批了一段更长的。她说："你第一次和她说话，不是因为你想说话。是因为地下洒了本子。你不需要准备好话题去接近她——环境替你创造了接触，你只需要做一个人最基本的反应——弯腰帮她捡起来。这件事之所以成立，不是因为你有多勇敢，而是因为你在不做任何准备的情况下，做的第一反应是帮人捡东西。这不是什么浪漫桥段，但比浪漫更难——因为它证明你在她的世界里出现时，本能是好的。十七岁的你也许不够勇敢，但你够善良——你该少骂自己一点。"

这段话陆允川看了很多遍。每一遍都没划线，直接贴在了修改稿最后一页的背面，折了角——他用拇指指甲沿着纸边压出一道深深的折痕，反复压了三次，确认折角不会自己弹回来。折角代表这句话是他会反复回来翻看的位置。这么多年过去，那些他觉得最笨拙、最不够大方的瞬间，在别人眼里不过是一个善良的人在做一件很小的事。那些他从头复盘到每一帧画面的片段，在真实的世界里只是一次没有人注意的弯腰。那些他自己嫌走路步幅太小的日子，在别人看来他只是在走——走出门，走下蹲，捡起本子，站起来，看着她的眼睛说一句"你捡吧"。他终于知道那六个字就够了。
