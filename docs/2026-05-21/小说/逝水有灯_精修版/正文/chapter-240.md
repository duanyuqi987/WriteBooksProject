# 第240章 同学群的婚讯多年后

所属部卷：第五部：逝水有灯
时间：多年后
地点：陆允川书房

## 诗歌插页 P240：原创仿唐绝句

残粉微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

多年后陆允川重写同学群的婚讯那一章时，纸稿空白处有一行铅笔字。又是程砚写的。

"老陆：这段差点意思。你在婚礼上的状态是对的——坐在陌生人桌，喝完了咖啡、吃了鲈鱼、聊了婚宴物流——但你没写自己在想什么。我猜你在想：如果一切没有变，今天站在新郎位置上的人是谁。别删这行字。这是你心底的事实。即便你没有写在书里。"

陆允川看着这行铅笔字看了很久。程砚的笔迹比以前稳了——不再是大学时那种歪歪扭扭的批注体，变得更圆、更从容。他在所有的修订稿上都留下了这类批注。有时候是在页边写一两句，有时候是直接在某个词上画圈标注不准。像一个不需要发薪水的第三方编辑。

他把程砚的铅笔字用手机拍下来，发给他："又是你写的？"

程砚秒回："什么时候写的——我忘了。可能上次去你家借火锅的时候。"

"你在我书稿上写了多少东西？"

"不知道。你慢慢找。应该有七处。或者八处。"

"你是怕我的书写得不够好？"

"不是。你的书写得够好了。我是怕你把自己的部分写漏了。"程砚停了停，"你用整本书写了别人——苏晚枝、我、祁野、叶澄、夏知遥——但你很少写自己。你把自己藏在叙述者后面。但在这一页，在婚礼这个场景，你应该站出来。不是作为主角——是作为那个十年后仍然能被旧群婚讯唤起的真实的陆允川。"

陆允川没有回复。他关掉聊天窗口，回到文档。

他重新打开婚礼那章的稿件。看着那行被程砚铅笔标注的段落。然后在段落上面加了一句：

"他看着喻可发在群里的电子请帖。新郎的方框眼镜反着暖黄色的灯光。他想：'遇到对的人就什么都变了'——这是今天群里她说的最像口号的一句话。但后来他细品——也许真的是。他不知道自己算不算那个对的人。但有一点他是知道的——十年过去他不再急着成为任何人的答案。"

删了。太像总结了。

改成："新郎站起来敬酒。白酒很辣。他喝了半杯。"

保留。不再改了。

终稿确认以后他关了文档。把程砚的铅笔字拍照备份。然后翻开一本新日记本——这是他自从旧的那本写完以来第一次开新日记本。牛皮原色的。和旧的那本一样。

在扉页写：《逝水有灯》终稿完毕。感谢程砚的七处或八处批注。其中这一处——写在婚礼那页——是最受益的。

写完合上。新日记本放进书架。

书架的"终稿"那一格已经填满了：牛皮日记本、毕业照、出租屋照片、《逝水有灯》书稿。旁边是程砚存的那半瓶白酒——今晚不喝。改天。

他关了书房的灯。走廊里只有冰箱的嗡鸣。
