# 第118章 相亲

时间：2018年2月（农历正月初三）

地点：青阳县 顾临风家

卷次：卷六：平行轨道

正月初三。家里来了亲戚。舅舅、舅妈、表姐带着两岁的儿子。屋里一下多了五个人。小孩在地上爬来爬去，把茶几下面的瓜子翻了一地。

顾临风坐在沙发角上。手里端着水杯——没喝，水凉了。

舅妈问："临风——工作了？"

"代课。还在读书。"

"研究生？"

"嗯。"

"好呢好呢。有出息。"

他点了点头。把水杯放下。

他妈在厨房和客厅之间来回走。端菜，摆筷子，招呼人。嘴一直没停。到了饭桌上才安静一小会儿。饭桌上舅舅说青阳县的房价涨了——城东新楼盘开盘八千一平米。"比三年前涨了一半。""咬咬牙吧。过几年还得涨。"

舅妈说表姐的单位今年招了新人，表姐被调去带徒弟。"带新人——累得很。"表姐补充道："现在的应届生连Excel都不会用。我说你大专读了什么。她说'读了电商'——我说电商你Excel都不会用。"

大家都笑了。顾临风也笑了一下。很轻。

然后舅妈忽然转向顾临风。"你给你妈省点心吧——二十三了还没对象？我们家表姐二十三岁孩子都生了。你呢——"

"他整天忙学习。"他妈替他接话。"考研、代课、写论文——没心思谈对象。"

"女的不等。"舅妈正色道。"好姑娘谁都抢着要。你等呀等的——三十岁着急了——哭都没地方哭。"

他妈放下筷子。看着他。"听你舅妈的。"

他说："知道了。"

舅妈又絮叨了几句。然后话题转回孩子身上——表姐的孩子把勺子扔到地上，哇哇哭起来。表姐弯腰去捡。

顾临风趁机去厨房添水。他妈跟进来。

"临风。你张阿姨说——那姑娘姓苏——苏晚晴。也是青阳的。在楚州一小教书。小学老师。你下个月回楚州见一面。"

他手里的水杯没有端稳——不是没端稳。是手轻微地缩了一下。像手指忽然碰到烫东西。其实热水没有烫到他——水是温的，已经不烫了。但他还是一震。水晃了一下，溅到灶台上。

他抽出抽纸。擦灶台。擦得很快。

"姑娘叫苏什么来着。苏晚晴。名字挺好听的。"他妈接着说。"你张阿姨说过年前问你意见——我说他肯定同意——你肯定同意对吧。"

顾临风的筷子停在半空中。不是筷子真的在手上——是在脑子里。他正在夹菜——不，他正在擦灶台——不，他站在厨房里，手里拿着抽纸。抽纸上洇了一小片水印。

"苏晚晴。"

这个名字从别人的嘴里说出来，是一种很奇怪的感觉。这个名字陪伴了他八年。在课本空白处写过，在黑皮笔记本里写过，在那封没寄出去的信上写过，在他考研报名的时候想过，在他每天骑车经过楚州一小校门的时候想过。八年来这三个字一直是他一个人的秘密——一个只有他自己知道的坐标。

现在他妈说出来。在厨房里。混着油烟味和抽纸擦灶台的沙沙声。像一张揉在抽屉底下的旧纸条被翻出来。摊平。读出来。"苏晚晴。"三个字。不对——他妈说出来的第三个字是"晴"。二声。然而这么多年来他默念这两个字的前三个声音时从来不会有二声——他默念的名字是轻声。不是"苏晚晴"。是"苏晚晴——"。后缀的音符永远落下来——很轻。浮在喉咙里。像一句话还没说完就咽下去了。

他站在厨房里。一只手拿抽纸。另一只手扶着灶台边沿。

他妈在旁边说——"名字挺好听的。应该是个好姑娘——"

"不认识。不认识的。"他的声音干干的。像嗓子被什么擦了一下。

"不去。我不想去。"他又加了一句。

"为什么？"

"不想相亲。我自己找。"

他妈以为他就是逆反。笑着拍了他一下。"找？你去找去——你要能找到我早抱孙子了。"

他没回嘴。

把擦过灶台的纸扔进了垃圾桶。

饭吃完。

大家移到客厅喝茶。舅舅在翻手机。舅妈在看电视里重播的春晚小品。表姐在哄孩子睡觉。

顾临风坐在饭桌边上没起身。桌上的盘子快要空了——红烧排骨还剩两块。油麦菜剩了三片叶子。酱焖豆角剩了一小堆——凉了一两个小时，油结成了块。

他望着那个凉掉的盘子。

苏晚晴。

她大概也在青阳过年。她家的除夕桌上——是她妈做的菜。大概是红烧鱼、炒青菜。跟以前一样——先吃素再吃肉。她妈妈也一定会说："你年纪不小了——该找对象了。"

她大概也说"妈——不急"。

一模一样。

两个二十三岁的年轻人——在同一座县城过年。隔了几条街、一棵老槐树、那家关了门的奶茶店。在不同的餐桌上吃着各自不同的妈妈做的菜。被各自不同的妈妈催婚。独自在心里回答：不是不想找，是我喜欢的人你不认识。我也不好意思告诉你。因为我已经八年没跟她说过话了。

下午五点多。舅舅一家走了。他妈去厨房收拾碗筷。

顾临风一个人坐在沙发上。电视开着——静音。画面上在播一档旅游美食节目——介绍一个西藏的酥油茶。主持人正在喝。表情很夸张。无声的。

他拿出手机。打开微信。

输入框。输入了"S"——通讯录自动补全。苏晚晴三个字弹出来。他盯着看了一小会。

然后退出了。把手机放回口袋里。

眼睛回到电视屏幕上。酥油茶的镜头已经过了。现在是拍藏民在跳舞。男的穿白袍子。女的戴红围巾。音乐大概很热闹——但电视是静音的。所以画面上的人在无声地跳。

他的手指在腿上来回地动。
