# 第015章 第一首诗

时间：2011年1月初

地点：顾临风家中（石桥镇）

卷次：卷一：初见秋风

元旦假期第三天的下午。

顾临风坐在二楼自己房间的书桌前。天气晴了。下过两晚霜之后，天空是淡白的蓝色——不太深——像被水冲过的衬衫。窗户开着一条缝——窗框下的墙上有蚂蚁洞——冬天没蚂蚁——但去年夏天的蚂蚁爬过的痕迹还在——一道浅浅的泥线。冷风从缝里钻进来——人的手背能感觉到那股气流的窄边。窗帘被吹得一鼓——然后落到窗台上。窗台下面的书桌——桌面——松木旧桌面——他写作业的台灯旁边多了一个新的橡皮擦。四方的——还没用过。外面的麻雀在叫。叫一串——停下——然后又一串。有一只落在窗框外的位置——影子从窗帘布上掠过——很快——一个黑影——没了。

桌上摊开的——那本旧词典。纸页泛黄——淡到老稻草的颜色。有些页面边上被虫蛀了小孔——不是活虫——是以前留下的。翻书的时候能听到书脊松了——纸和线之间轻微分离——纸页在被翻动时会发出干纸磨干纸的声音——沙——像脚踩干树叶。内页有他以前用铅笔画的——第一次翻字典查两个字——"晓"和"阑"——那时候压着写——字小小的——在这个分册页的右上角。中间夹着那张纸方块——从书页之间掉出来——落在桌面上。蓝墨水在纸上看去颜色淡了一点——过了一个多月了。他画的那些墨迹的纹路。有些地方——墨吃进纸——而纸在折角处——墨在上面形成了细微的裂纹——像干涸的河床底。

蓝墨水——《初见》。

他把纸方块拆开。纸折久了——折痕是白的。沿着折痕——纸被压成四个小格——折线分上下左右。摊平之后用手掌在上面抚了三遍。第一遍——手掌按着中线往下推——纸没完全平——折起的地方反抗着——耸起了一层微小的弧形。第二遍——用力——手掌在折痕上来回——纸受热——和手上的潮气——抵抗变小了。第三遍——轻——把纸角捋一下。现在能平着看了。

钢笔还搁在桌面上——笔帽没套。上一次写完之后忘了套上笔帽——笔尖上的残余墨水就干在那里——笔尖那条缝——原来是湿亮的蓝——现在细缝里凝固着蓝色的墨粒。他用手指抹了一下笔尖——蓝的——细细的粉——弄在指腹上。然后再把笔放回笔帽——笔帽插上——"嗒"轻一响。桌面右手边的两个小指印——是蓝墨水染的——刚才弄的时候碰上的。

他拿起那张纸。从头到尾看了一遍。

那年九月风初凉。青阳旧道桂花香。马尾蓝衫窗前影。日光落肩半寸长。

初雪来时人未语。手心融尽一痕霜。此心已在阑珊处。不问春风问晓光。

读了。然后放下。阳光从窗玻璃照进来——照在纸面上——纸上粗糙的纤维看上去比在阴影里清楚——起毛——一点点白丝。然后他从抽屉里拿出那个牛皮纸封面的新本子——"新年愿望"那页已经翻过去了。他翻到第二页——没有字——干净。他从笔筒里重新拿起钢笔。拧开墨水瓶。英雄牌碳素墨水。小玻璃瓶——瓶盖拧开的时候很紧——里面的垫圈是橡皮的——瓶口有一层干掉的墨垢——瓶身标签已经翘起来了——翘起的地方是胶水干了之后的皱缩——字"碳素墨水"印在泛白的纸标上不全了。他把钢笔尖伸进瓶口——金属尖碰到瓶口——轻碰撞的"叮"。拇指和食指捏住橡皮吸管——挤了两次——气泡从一个吸嘴里吐上来——墨被吸进管里——管里的墨水高度到了中间。把笔尖从瓶子里拔出来——刮了刮瓶口——多余的墨刮回瓶里。然后开始写。

在牛皮纸本上。米黄纸。沙沙——笔尖在纸面上细微地磨着。墨水在笔尖下面摊开——蓝是深蓝——没上次洇——好一点。

第一行。引号。"那年九月风初凉。"——写"九"的时候那一撇收得太快了。"青阳旧道桂花香。"——"香"的"日"字里那横有点歪——不管。往下。

第三句。"马尾蓝衫窗前影。"——写"马尾"的时候，他笔顿了一下。手腕停着。然后才把"蓝"字提起来——"蓝衫"——撇笔和捺笔。黑色墨水——在纸上干得还挺快。写"窗前影"的时候用的是平常速度。

四句写完了。他把笔搁下。看着这四行。第一联写的初见——九月一号。高一三班的前门。她从门走进来——头发在九月的阳光里是褐色的——浅蓝T恤。还有她走进来的时候，周围闹哄哄的教室——他桌上的铅笔盒没动。

他坐直了。继续写。

第二联。"初雪来时人未语。手心融尽一痕霜。"——十二月的第一场雪。她站在窗边把窗开一缝——右手伸出去接雪。雪花落在她手心里。她收手低头。雪融——水滴在手指上——顺着食指往下滑。他在两米外看她侧脸——她没有说话——他没往前走。这些都在诗里。

第三联。"此心已在阑珊处。"——此心。什么是"心"——他不知道——在一个说心里话但不说出来的东西吧——在灯火稀落的、阑珊的那个地方。阑珊——是灯火将灭未灭的那些地方。他初三语文课里读过辛弃疾的词——"蓦然回首——那人却在，灯火阑珊处。"老师说过——阑珊不是繁华——灯火阑珊是最后的——细的——残留的——快暗了。他觉得这两个字好看——有火和书——组合在一起——灯火快灭的时候最静。不对——他的"阑珊"是用在他十六岁的教室最后一排——那地方也是静静的——一直——从没人点过灯。

第四联。他看了看。写"不问春风问晓光"——几乎一笔连成。晓光——早晨的光。早晨——太阳出来了。她的名字——苏晚晴。晚是傍晚。晴是雨后天晴——太阳出来。他的诗里——春风吹来的不是他要问的——他想问的是早晨的光——清晨。这句话放在最后是押韵——"霜"——"光"——第一声——从平到平。

他从"晚晴"里取了一个"晴"字——把晓接在她面前。中间隔了一个长夜。如果她能看到这几行——她是不会看懂的——她已经习惯做那些不像答案的物理题了。

他写完了全诗。把笔放下。然后把这一页撕下来——撕得慢——大拇指压着装订线——齿脊——沿着纸和胶——一道道锯齿往上拆。整整的一个左边——撕到最上方的顶端——纸脱离了。纸从根部出来的刹那听到了那个声音——像蚊子在飞——嘎。有几个小豁口——按平。

把纸拿起来。对折。再对折。折了个方块。拿在手里捏了捏——捏出直角。然后站起来——走到书架前——从第二层——把那本旧词典抽出来。书脊上有压痕——还带点点绿漆——包装早没了。他打开词典——翻到有"阑珊"字的页码——"阑"字那一条——有红色的印章——印刷行线——反正好多年了。他把纸方块夹在这一页。薄纸和词典纸的重量不一样——翻过"阑珊页"的时候会有一道弯。词典合拢。放回第二层靠左。和其他书挤在一起。退后一步。书架上有十几本书——这些书从小到大陪他——学校里的课本年年换——只有家里的书还是那些旧书——几本父亲的旧小说——还有他那本蓝皮的初中练习册——《几何初步》——六年级的书——现在还挤在一角。这本词典的书脊是深绿色的。从四岁就看到它了——他妈用来压咸菜缸的——后来被他拿来当最早的识字书。现在夹着诗——在里头沉。

窗外的麻雀不知什么时候换了位置。它们落到电线上。电线由一根水泥柱引过来——左墙头一根横过瓦顶。三根电线——一根上落着三只麻雀——一根上落了一只。另外那只——大概停了一下就飞走了。风一吹。电线往下一荡又回来——麻雀身体斜了斜——翅膀张开又收。但没飞——它把脑袋歪向左边——然后又转回——又歪。往下的风又来了——麻雀尾巴往上一翘。风停。抖抖毛。然后三只一起从电线上弹走了——飞起时翅膀扑扇了几拍——声音像手掌拍过棉被——一簇——然后消失。

他把钢笔帽旋上——拧到底。墨水瓶瓶盖旋紧——手指在瓶盖上压了十五秒——关得紧紧——怕沾氧。纸屑扫进手心——收进桌底下的垃圾桶里。然后下了楼。

晚饭吃的饺子。他妈包的——芹菜猪肉馅。芹菜是昨天从市场买来的——水芹——茎细——洗的时候泥留在底部——摘了好一会。面皮自己手擀的——擀出来的皮厚薄不均——有几张中间太薄——包的时候手指压得又用力——煮时就破了。破了三个。他妈一个个地捞出来——用筷子把皮夹回来放在小碗里——一边念叨:"这皮太薄了——你爸就说我手没轻重——我说那个擀面棍不好用——"他爸没说话——埋头吃着——饺子蘸醋——醋是镇上醋厂卖的散醋——装在一个老酱油瓶里——酸味淡——咸味重。他爸吃了二十一个——顾临风数的——因为他坐他爸对面——他爸每夹一个——那个碗里就少一个——他无心数到了。他自己吃了十四个。他妈吃了九个。还剩六个——他妈说:"明天早上煎着吃——"油煎饺子——用猪油——到第二天皮会脆——但馅干了些——他记得那个味道。

晚上他回二楼。夜色已沉。窗外远处——石桥边的那户人家又亮起了灯——黄色的——透过冬白的薄雾——模糊。窗玻璃上结了一层薄薄的水汽——是屋里的暖气碰着冷玻璃凝的。他站在书架前——手指在那本旧词典深绿色的书脊上碰了一下——温——字典一直夹在书群之间——比空气暖和。然后缩手。关灯——开关在门边——"啪"。躺在床上。盯着天花板。天花板上有一个水印——是左边角落那个——夏天屋顶漏水留下的——圆形的。他妈说过了年就请人补屋顶。水印在暗处看起来像一张脸——侧面——往下看——眼窝凹进去的那个位置刚好有一个更深的灰点。他翻了个身——把脸埋进枕头里。枕头上有昨天太阳晒过的味道——晒过的棉花——枕芯拍松了——脸埋进去能陷下一坑。又翻了个身——吸气——"初雪来时——"没有往下想——把诗放在书架上了。呼吸的声音——吸气——呼出——两股气流。楼下的灯也灭了。他爸他妈睡了。堂屋的木门被夜风推了一下——门框上贴着的那张去年的门神画——卷起一边——纸画在风里——碰在门上——哒——哒。不是人敲的——风捶的。他在这样一个夜里——还有两天寒假就结束了——开学——是三班的教室——第二排靠窗。

他把眼睛闭上。眼皮在暗处——闭着也能感觉到眼睛后面有隐隐的光——窗外的远灯——眼睑里的红色的——毛细血管——透过来了。然后慢慢暗了。睡着了。
