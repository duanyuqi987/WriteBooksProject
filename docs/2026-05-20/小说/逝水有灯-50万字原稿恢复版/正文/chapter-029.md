# 第029章 第一次说话旁观

所属部卷：第一部：初秋三楼
时间：高一末
地点：楼梯转角

## 诗歌插页 P029：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
笔记本封面上一道浅浅的折痕还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，第一次说话会显得没有那么沉重。教室、走廊、操场里站着许多人，每个人都有自己的迟到、作业、饭卡和回家路。

叶澄后来记得的不是陆允川的沉默，而是他那天总在看笔记本封面上一道浅浅的折痕。她觉得那样的眼神很熟，像很多人刚开始喜欢谁时都会有的样子：认真，又有点不讲道理。

叶澄站在楼梯平台等同桌，看见这一幕后没有起哄，只把快掉下来的练习册往陆允川怀里推了推，说先拿稳。

程砚看得更直接。程砚在后面憋笑，憋到走出半层才忍不住，说你刚才那句谢谢像背课文，标点都背出来了。他不喜欢把话说得太柔，因为太柔的话陆允川听了会继续躲。

“你的本子。”夏知遥说。“谢谢。”陆允川说。“下楼慢一点，书容易滑。”夏知遥说。

夏知遥在这一章里也不该只是一个被看的背影。夏知遥正从下面上来，弯腰把本子捡起。她没有翻开，只用手掌拍掉封面上的灰，递过来时说，你的本子。她有时会因为题做不出皱眉，有时会在朋友讲笑话时笑出声，有时也会因为家里电话而沉默很久。

她没有义务知道陆允川每一次停顿。就像陆允川也不知道她书包里那张没交的申请表，不知道她冬天手指容易冻红，不知道她回宿舍前常在水房多站一会儿。

楼梯转角常年有风，尤其晚自习前后，整栋楼的人一齐上下，脚步声像一阵急雨。墙角贴着消防示意图，边缘被人抠起一点。

在旁观者眼里，夏知遥捡到陆允川掉下的笔记本，递给他时只说了一句你的本子。这件事只是一天里很短的一折。可短并不等于无意义，它只是提醒陆允川：意义不能只从自己这边生长。

楼梯间的灯坏了一盏，光从上一层斜下来，人影被拉得很长。有人抱着篮球从旁边挤过去，橡胶味和汗味混在一起，把那句谢谢衬得更加短。

叶澄晚自习后对他说，喜欢谁不是错，但别让别人替你承担你没说出口的戏。她说完就去交作业，留下陆允川站在门口，手里捏着一本卷了角的练习册。

那一摞书其实不重，重的是他下楼时总想快一点。前面有人堵住楼梯，他停得太急，最上面的笔记本便从臂弯里滑出去，像替他做了一件他一直不敢做的事。

夏知遥递本子时，指尖没有碰到他的手。陆允川却仍然僵了一下，僵得程砚在背后差点笑出声。她说完“下楼慢一点”，就侧身让过一群往上跑的男生。

回到座位后，陆允川把“谢谢”两个字在草稿纸上写了一遍，又觉得自己傻，揉成团塞进桌斗。纸团滚到最里面，和几张旧小测挤在一起。

叶澄晚自习后问他，本子没摔坏吧。她问得自然，陆允川却答得很慢。程砚在旁边替他说，没坏，人差点坏了。三个人都笑了，笑声很轻。

旁观不是冷漠，而是把故事从一个人的胸口挪开一点。挪开以后，夏知遥有了自己的呼吸，程砚有了自己的烦躁，叶澄也有了不愿明说的担心。

陆允川不喜欢被旁观，因为旁观会让他的深情显得没有那么独一无二。但正是这种不独一无二，让后来的人生有了出路：大家都年轻过，也都误会过，不必把一次笨拙写成终身审判。

从叶澄的角度看，陆允川的心事并不神秘，只是太年轻，所以总以为别人都该看懂。他越沉默，周围人越不知道该不该靠近。

夏知遥也许注意到了一点异常，也许没有。她那天要处理的不是谁的暗恋，而是自己的作业、座位、朋友和回宿舍前必须打完的一通电话。

如果只写疼，这一章会很薄；加上楼梯扶手、笔记本、练习册、坏掉的灯，它才重新有了重量。那重量不是压人，而是让读者相信他们确实在这里生活过。

陆允川把笔记本拿在手里，先把笔记本夹回书堆，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把书抱稳，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把本子递回来时没有翻开里面的字，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

楼梯转角里最明显的不是心跳，而是脚步撞在水泥楼梯上的回声。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

晚饭后，塑料袋里晃动的酸奶把人从心事里拽出来。程砚说再大的事也得先回教室，陆允川拎着袋子，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。笔记本、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

从旁边看，陆允川的心事并不神秘。它只是少年人常有的笨拙：怕被看见，又怕完全不被看见，于是每个动作都带着一点别扭。

夏知遥也许注意到了一点，也许没有。无论哪一种，她都不该被写成答案。她有自己的捡起掉落的本子，也有不想向任何人解释的沉默。

这一段真正留下的不是一句谢谢有多重，而是本子被捡起、书重新抱稳、楼梯重新通行。生活没有为他停住，却给了他一次学会分寸的机会。

如果只写疼，这章会很薄；加上笔记本、脚步撞在水泥楼梯上的回声和晚饭后塑料袋里晃动的酸奶，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

这一章的火被旁人的目光压低了。压低以后，才看得见灰里的温度，也看得见每个人不被爱情叙事吞掉的完整生活。
