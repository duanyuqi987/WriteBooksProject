# 第013章 元旦之前

时间：2010年12月31日

地点：高一三班教室

卷次：卷一：初见秋风

十二月三十一号。今年的最后一天。

下午学校在操场开了个新年联欢会。每个班出一个节目，高一三班的是合唱。唱的是《明天会更好》。班主任说这首歌积极向上。音乐老师站在下面指挥——手举得很高。唱到"伸出你双手"的时候，有人举了手，有人没举。

苏晚晴站在女生第二排左侧。

她唱的时候嘴张得不大。脸上的表情很淡。但她把每句歌词都唱到了——嘴型对得上。风把她的刘海吹开了，露出一小截额头。

顾临风站在男生最后一排。他唱没唱自己不确定。嘴在动。声音卡在喉咙里。

联欢会四点结束。各班带回。班主任站到讲台上说了几句过年的话——假期安全、少放鞭炮、写寒假作业。底下一片"知道了知道了"，然后哄堂大笑。班主任也笑了笑，一挥手:"走吧。"

教室空了。

有人跑得快。椅子倒了一把，没人扶。教室门被推开之后就没关，风灌进来，把桌上的几张试卷吹到了地上，贴着地面滑出去老远。

顾临风没走。

他坐在座位上。面前摊着一本英语书。那一页从上到下全是单词的英文释义。他看了一行。没看第二行。

教室里的日光灯关掉了一半——值日生走之前关的。暗的那半边的桌椅像泡在水底，轮廓模糊。

他站起来。

从倒数第三排——往后门方向——走出去了。

然后他又回来了。从前门进来的。

教室空无一人。四十二张课桌，四十二把椅子。黑板擦干净了，连粉笔槽都清过了。讲台上的粉笔盒盖上了盖子。

他顺着过道往前走。走到第二排靠窗。停住了。

苏晚晴的座位。

桌上什么都没有。书全收走了。椅背上也没有书包。桌面擦过——应该是放学那会儿她自己擦了。桌面右下角有一道铅笔印子。划得很轻，不使劲看看不出来。

窗外的光从她座位前面的那扇窗户照进来。冬天的光很薄。打在她的桌面上。木纹的纹路在光里格外清楚。有一块木头节痕——一个小圆圈。像眼睛。

他伸出了手。没有碰桌面。手悬在桌子上方停了一下。然后收回去。

转身走回了自己的座位。

坐在倒数第三排。往左前方看——四排+两个过道。第二排靠窗的位置是空的。椅子推得整齐。桌上的木头眼睛朝着他。

他看了很久。

然后站起来，走到门口。

手伸向开关。教室南边的灯已经关了，就差北边这一排。他的手放在开关上面。

最后一刻。他转头看了一眼那个空座位。

按下了开关。灯灭了。教室暗下来了。桌椅变成了暗色的剪影。窗外有一点天光——灰蓝色的。但很快就暗了。

他关上门。锁簧咔嗒一声。

走廊里很静。别的班早就锁了门。他一个人下了楼梯。三十九级。到一楼的时候，传达室的老头在听收音机。中央人民广播电台——"现在是北京时间六点整"。后面跟了一段音乐。

他推开门，走出教学楼。天已经全黑了。校门口的灯亮着。梧桐树的叶子掉光了，光秃秃的树枝指向天空。

门卫大爷在传达室里冲他点了头。他也点了点。

校门外面的街上很热闹。有人放着一首流行歌，声音很大。街边的小卖部红彤彤的，挂了一排鞭炮模样的装饰，红的黄的。几个小学生在挑糖果。

二〇一〇年的最后一天晚上。

顾临风往车站方向走。

走得很快。手插在口袋里。左边的口袋里有一张纸——是他下午联欢会的时候从节目单上撕下来的。上面油印着高一三班的节目——《明天会更好》。字是宋体的。

他攥着那张纸，走了一路。纸被手心的汗洇得有点软。

汽车站。末班车六点二十。车上没几个人。他坐到靠窗的位置。旁边座位上没有人。玻璃是冷的。他把额头贴在玻璃上，凉的。

车启动了。

窗外县城灯火一片。红灯笼、白灯管、黄色的路灯连成线。有户人家在阳台上挂了一串彩灯，一闪一闪的。车拐了个弯。那些灯就看不见了。
