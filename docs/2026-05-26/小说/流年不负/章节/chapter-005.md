# 第005章 挂件

时间：2010年10月

地点：高一三班教室

卷次：卷一：初见秋风

十月。天凉了一些。

早晚温差拉到了十度以上。早上出门不穿外套扛不住。有人已经穿上了秋衣，领口从校服领子里面露出来一个边。蓝的、灰的、白的。女生开始在校服外面套一件薄毛衣。红的、紫的、黑条横纹的。

校门口的梧桐树开始掉叶子。值日生每天早上扫一堆——三两个人拿着大扫帚，从校门口一直扫到教学楼。扫帚是竹枝扎的，在地上划出刷刷刷的声音。叶子堆在垃圾筐里，中午又掉一地。值日生骂一句。下午继续扫。

课间的时候，大家都去走廊晒太阳。秋天的太阳不晒人，暖融融的。晒在不锈钢栏杆上，栏杆热了一些。有人靠着栏杆聊天——聊周末去哪玩、聊明天调不调座位、聊哪个老师今天作业布置太多。有人趴在窗台上睡觉。脸对着阳光那面。过几分钟翻一面。

教室里前后两个门都开着。穿堂风从前往后吹，带着楼下烧烤摊的烟味——烤面筋的味道。不知道是谁在课间小卖部买了根烤面筋带回来，边走边吃。签子扔在后门垃圾桶里。上面还有半截没吃干净的。

顾临风坐在座位上翻数学课本。翻得很慢。一页一页翻。每页看个三五秒。其实没在看书。手指放在书页上——食指夹在前一页和后一页之间。拇指压在页面上。这是他看书的习惯姿势。在家里磨豆腐的时候他姐教他认字就是这么翻书本的。

数学课本的页角有点卷。他用手指从外侧往内侧捋了捋。捋平了。又卷了。又捋。

他旁边的人已经到走廊上站了。赵一鸣站在栏杆边跟后座的人在说什么——说着说着笑了起来，笑声从窗户传进来。顾临风没起身。

他桌上的那张试卷——数学期中模拟卷——分数不是很高。七十一分。红色水笔打的。旁边有赵一鸣的卷子——六十六。赵一鸣看了分数之后撇了撇嘴。"无所谓。反正不是真的考试。"把卷子窝成一团塞进抽屉。过一会儿又掏出来。把卷子展开，用手掌压平。皱的地方用手指来回刮。

就在这时。

浅蓝色从眼角晃过去了。

是苏晚晴。去后门那边扔废纸。

她手里捏着一团草稿纸。白色的横格纸，团得很紧。擦过他课桌旁边的时候，校服袖口拂过桌角。她的手指按着纸团——拇指和食指之间夹着。

她经过他座位的时候，书包从椅背上垂下来。书包带子很长，书包底部悬在她腰下面大约一个拳头的位置。拉链上挂着一个东西——

陶瓷小鱼。黄豆大小。白色的底，上面画了两道蓝线。鱼尾巴和鱼身的连接处很细——可能就是一两毫米的瓷。线上穿的是红线。不是新红线——洗褪过很多次的红。

书包蹭到了他课桌的桌角。帆布擦过木头的边缘。小鱼晃了两下。撞在桌边上，发出一声很轻很脆的响。瓷碰木头。像两个手指尖轻轻碰了一下。声音细得只有在这附近才能听见。后排正在聊天的那两个人肯定没听到。

然后她走过去了，把废纸扔进垃圾篓。弯腰的时候马尾往前滑过肩膀。然后直起身。转身往回走。

小鱼还在晃。从左晃到右，又从右晃到左。晃的轨迹是一个拉长的"8"字——摆锤运动。幅度越来越小。最后一次晃到左，弹到右。右到左。停了一下。又摇了摇。然后彻底不动了。

顾临风捡起来的试卷还捏在手里。手指紧了紧。纸没折。手指捏的位置湿了一点。

他盯着那个鱼看。直到她走回自己的座位。小鱼在她身后晃。走到第二排的时候书包蹭过她自己的椅背。鱼又响了一声。她拉开椅子坐下。书包自然地落在椅背上。鱼贴在书包上，白色的小点。

那天下午的课他没怎么注意听。

下午第一节课是英语。英语老师穿着高跟鞋走进来，鞋跟敲在水泥地上——嗒嗒嗒。她翻开课本，板书了一黑板的单词。顾临风应该背单词的。他背了两个——contribute。distribute。第三个是attribute。他三个背混了。拿起一本英语单词本翻了一下。

第二节课化学。化学老师在讲"离子方程式"——配平。黑板上写了几个方程的步骤。顾临风在笔记本上画了一条鱼。画完一看——不像鱼。像一块石头。拿橡皮擦了。橡皮屑留在笔记本上。他吹了一下。碎屑飞起来了。落到了前排那个男生的背上。那个男生伸手在后面挠了挠。没发现。

那个陶瓷小鱼——黄豆大小。小到一般人不注意就看不清。女生挂的挂件在学校里多了——塑料小星星、指甲盖大小的毛绒熊、街上饰品店里买的那种彩绳编的小鞋。绿豆大一点的挂件很少。豆子大的瓷鱼更没见谁挂过。大部分人看不出她书包上到底挂了什么。也许一直以来就没人看清过。

鱼身上的蓝线可能是手工画的——两条线不是完全平行的，右边那条略微弯了一点。像画的时候手抖了一下。不是工厂里机器印的标准印花——瓷片外面是薄薄的手绘白底。看起来有些年头了。底部的白色有点泛黄，发着旧东西特有的颜色。

不是新东西。

后来几天他特意留意了那个鱼。课间她走动的时候——去讲台交数学作业、去黑板前面看明天值日的名单、去第二排后面的座位找某个女生借一支修正液——小鱼都会晃。不同的晃法。走路快的时候鱼撞在书包拉链上来回弹，像敲小鼓——叮叮叮叮。走路慢的时候鱼只是轻摇，两边摆一次，再慢慢收回来。摇晃的幅度比走路的步子慢一拍。她穿过中间过道的时候转弯了——鱼往转角的方向甩了一下。回正的时候又弹回来。走路停的时候鱼还在晃。她停住了。鱼没有——还在那里荡着。

有一次她在教室门口跟人说话。站在门槛上——半只脚在门里半只脚在门外。风吹过来。走廊的风和教室里的穿堂风撞在一起。书包碰到门框上——铁皮门框。小鱼撞出一声脆响——叮。金属的共鸣，很短。她自己好像没听到。继续跟那个女生说话——那个女生在跟她借英语笔记本。她嗯了一声，回到座位上翻书包。把笔记本递过去。

顾临风听到了。

他正在抄数学笔记。抄的是黑板上的例题。例题抄完了一个。第二个抄到一半——x²+3x+2=(x+2)(x+1)。黑板上画了一个坐标系。他的笔在黑板上画的地方停了一下。那个"叮"的声音在他耳朵里多停了半秒。

然后他继续抄。第二个例子抄完了。

周六下午没有课。宿舍里只有两个人。另外四个人回家了。赵一鸣也回了——他家在县城，骑自行车十五分钟就到。走之前他把枕头底下的漫画压了压。"我星期天晚上回来。帮我看好位置——别让隔壁的人睡我床上。"然后钥匙一甩。走了。

顾临风没回去。石桥镇太远，来回车费十几块。十几块能吃两个星期的加餐。他爸上次给的两百块还剩五十六块。上次给他爸打电话——公用电话，在宿管老师那边——他爸问他钱够不够。他说够了。他爸说"那就行"。然后挂了。他站在电话前面停了几秒钟。宿管老师看了他一眼。"打完了？""嗯。"

宿舍很安静。对面床铺上，留下来的那个男生——刘洋——戴着耳机听歌。耳机是从中关村的旧货摊上淘的。线是用黑胶布缠过的。偶尔哼一两句，不成调。在哼周杰伦的《稻香》。哼了两句忘了。重来。又哼了两句。放弃。手机屏幕的蓝光映在他脸上。上铺的木板下面是粗麻布。楼下有人在打水，水桶落地的声音闷闷地从墙外传过来。

他坐在床上，膝盖撑着下巴。手肘搁在膝盖上。手指交叉在一起。脚边放着搪瓷饭盆。中午吃剩的半碗米饭还在里面。没洗。筷子搁在盆沿，一只在上面，一只滑下去了。

那个陶瓷小鱼。

小到一般人不会挂。黄豆大小。

挂在书包拉链上。

是路边摊买的塑料件早就断了、磨坏了、变色了。班里课代表——那个英语课代表——头绳上的塑料小熊不到一周就掉了，散开成一截红皮筋。哪有什么黄豆小的东西能挂这么久。瓷的不一样。瓷的好好地挂在那里。洗褪过的红线。没断。没变色。除了那么点微小得几乎看不到的泛黄。

他一直想这件事。

想到后来，他翻开数学课本。看了两页。第一页是例题。第二页是练习题。什么都没看进去。翻页的时候手指不知道翻的是第几页。课文上的字是一行一行的，但他脑子里的"字"不在书本上。

又翻了一页。

上铺的人翻了个身，床咯吱响了一下。刘洋把耳机摘下来——"你在干嘛呢。我听到你在翻书。翻了半天。"然后又把耳机戴回去了。不一会儿耳机漏出来的音乐又响起来了。这回换了一首。陈奕迅的。没听清歌名。

他脱下外套。外套袖子蹭到了床边的铁架子。架子凉凉的。他打了个颤。然后把外套叠了两下，放在床尾。

窗外的天暗下来了。傍晚的云是灰蓝色的。太阳已经落在教学楼后面看不见了。操场上的灯还没亮。有几个影子在操场上动——大概是在跑步或者散步的人。有一只鸟飞过去。灰色的。停了一下。又飞了。

顾临风把书合上。侧身躺下来。脸对着墙。枕头上有一根线头。他拈起来。在手指上绕了一圈。然后扔掉——扔到床下。线头太轻，飘了一下。落在地上一道阳光的边缘。

墙上贴的那张课程表还在。数学、语文、英语、物理。课表是蓝色圆珠笔手抄的。字迹不好看——他的字不算好，有些笔画写得很短。星期五最后一节是班会。旁边用铅笔写了一行小字——"期中考试复习"。是他自己写的。周日晚上写的时候铅笔头断了。所以那些字的后半截是用没尖的笔头划上去的。

他闭上了眼睛。

然后睁开。

从枕头底下摸出语文课本。翻开——书的纸张在黑暗的宿舍里有一点凉。封面。内页。封底内页。那两个划了横线的字还在——苏晚晴。划的横线压在字上。中间那部分破了。破口有点毛。不知道是被枕头的压力磨的还是他自己后来又翻看过。

他把书合上。放回枕头底下。这次放得很平。放的时候手指在枕头上按了一下——确认书在下面。没有折角。

外面的风变大了，吹得窗户框在响——墙缝里的旧木头，吃风的时候嘎一声。那个留在宿舍的男生摘了耳机。站起来——脚踩在地上，赤脚——把窗户关上了。窗户关上的时候窗帘往里吸了一下。然后往外鼓。玻璃窗的窗扣生锈了。他转了两下才扣上。扣上之后回头看了一眼。"要下雨了。"

顾临风嗯了一声。

雨大概八点钟开始下的。打在铁皮屋顶上。先是很细的沙沙声——大约半分钟——然后突然声音很大。噼里啪啦密集地砸下来。从四周往中间聚。铁皮屋顶上的水顺着檐口往下淌，砸在下水管道里，声音很沉。间或有大滴的雨——大概是从树叶上汇下来的——打在窗户上比普通雨声重。

宿舍门没关严——门缝里漏进来一阵雨后的泥土味。刘洋把门推了一下。锁舌嘣地弹进框里。

他躺着听雨。雨丝斜打在窗户上，顺着玻璃往下滑。窗外的路灯在雨线里变成了一个模糊的橙色点。透过雨水看路灯——光晕是放射状的。像隔着磨砂玻璃。

后来雨小了。铁皮屋顶上滴水的节奏从密密的连成一片变成单个单个的。一颗。两颗。一颗。一颗。然后两颗之间的间隔越来越长。最长那颗隔了大概八秒。然后停了。

路灯的光重新变得清楚了一些。雨后的路灯特别亮——灯泡下面水汽还在飘。有一个女生打着一把红伞从路灯下面走过去。影子拉得很长。不到天亮是不会干的。

刘洋睡着了。呼吸很均匀，偶尔带一点鼾——从鼻子里吹出来的那种。耳机还挂在脖子上。头顶的床不响了。

顾临风翻了个身。被子拉到肩膀上面。蓝格子被套在暗处看不清颜色。被套的边角有点卷——他把卷的角卷了往回折。然后把脸埋进枕头里。枕头上昨天留的牙印还在——大概是睡着了咬的。印子是两排，上下各一。中间有块牙膏味——大概刷牙没洗干净就睡了。

外面的树在滴水。一滴一滴打在水泥地上。铁皮屋顶的檐口还在滴——比树上的滴得慢。滴。滴。滴。

他记得那个声音。黄豆大小的瓷鱼撞在木桌边上的声音——不是铁也不是石头。是瓷的。细的声音。比现在窗外的滴水声还细。但比滴水声更清楚。更紧。叮。就只有那么一声。

他翻了一个身。又翻回来。

周而复始。

第二天早上醒来的时候，他发现自己把枕头底下的语文课本抽出来了。书摊开在胸前。封底朝上。划了横线的名字正对着天花板。他坐起来。把书合上。重新放到枕头底下。枕头在书上面。

窗外是洗过一夜的天空。蓝得发亮。树叶上还挂着水珠。太阳照上去的时候反光。亮晶晶的。像塑料片。

然后他穿上衣服。洗脸。刷牙。出门。周一早上。又是新的一周了。
