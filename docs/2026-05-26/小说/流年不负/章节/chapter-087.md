# 第087章 寒假回家

时间：2014年1月

地点：青阳县

卷次：卷五：各自南北

一月中旬。寒假。青阳县。楚州的气温降到零度——晚上水龙头的水管有时会冻住。早上洗脸的时候水是冰的——冰到手指头。

从楚州坐大巴回青阳——三个半小时。楚州长途汽车站在城西。候车厅里飘着泡面和汽车尾气的混和味。他买了票——四十五块，比九月来时贵了五毛。售票窗口后面坐着一个中年女人——头发染过，发根有些霜白。她把票从窗口递出来的时候手指上有一个创可贴。过检票口的时候检票员咬着一根牙签——牙签在嘴角晃着——把票撕了一半还给他。他把那半张票收进裤兜。

大巴停在老站台。行李舱开着——他把装得鼓鼓的蛇皮袋塞进去。旁边排队的人推了他一下——"不好意思"——他回头看了一眼又继续走。上了车——坐在倒数第三排靠窗的位置。座位套是灰蓝色的，上面有几道圆珠笔划的印子——蓝色的、红色的。还有一块黑色的污渍——大概是之前谁洒了咖啡或酱油。靠背的调节杆坏了——椅子一直往后倒，椅背斜得离谱。他用膝盖顶住前面的座位——膝盖顶着椅背，大腿绷着。车开出城区。经过师院南门的时候——那个公交站牌在晨光里是绿的。301路的站牌——第三行。他看了两秒。车没有停。然后经过市图书馆——喷泉关了，冬天水放掉了，池底是灰色的水泥。经过市体育馆——圆形的建筑外面挂了一条横幅"2014新年全民健身跑"。经过新华书店——还没开门，卷帘门拉着。经过人民医院——有人在门口推轮椅进急诊室。窗外的每一个站名他都在手机备忘录里记过——每一个地标他都认得。大巴沿着301路的路线走了一段——这一段重叠的路是他每天在脑子里开的班车路线——唯一的区别是大巴不会在师大北门停。车速在中山路口放慢——他看了一眼中山桥下的楚河水——冬天，水面很平，颜色是深绿的。

高速公路上风景单调——灰色的护栏、枯掉的灌木、偶尔的蓝色路牌。路牌上写着"距青阳 85km""距青阳 40km""距青阳 12km"。每看到一块路牌他的身体都会坐直一点——颈椎被拉直了。从十二公里到零——他在楚州也用同一个数字丈量过另一段距离。但那十二公里他一次都没有走过。这十二公里他在回家——而那十二公里在平行地等待。

到青阳是中午十二点。大巴拐进汽车站。站外的路还是那条路。路右边多了一家奶茶店——"茶の道"，粉色招牌，店门是玻璃的。高中时候没有奶茶店——那时候只有一个卖绿豆沙的推车，三轮铁皮推车，"绿豆沙，一块五一杯"。

从汽车站出来。走回家的路上经过了县城的主街。街上的铺子大部分还是老样子——修鞋摊的大爷还在马扎上晒太阳，缝纫铺门口摆着一台旧缝纫机，新华书店的门脸没变——蓝色招牌上落了点灰。冬天的县城灰扑扑的，街面上有鞭炮放过后的红纸屑——大概是前几天有人结婚。纸屑被风吹到路边的水沟里——水沟里的水是绿色的，上面漂着红纸屑和白沫。

到一个十字路口的时候——他停了步。右手边巷子拐进去——走五分钟——是老槐树巷口。他没有拐进去。站在十字路口看了三秒钟。一个骑电动车的大姐从他旁边拐过去——喇叭嘀了一声。他往家的方向走。

到家是下午三点。进院子的时候他拖着的行李箱在水泥地上还发出轰隆隆的声音。他家在机械厂家属院——五层楼房，外墙是灰白色的，有些地方墙皮翘起来像卷起的纸。院子里停了他爸的电动车——后座上盖着一块黑色的雨布。晾衣绳上挂着腊肉和腊鱼——腊肉是红白相间的五花肉，在冬阳下有点发亮。他妈在厨房里炖鸡——姜片和八角的味道飘满整个院子。他爸在院子里晾腊肉——用竹竿穿了一排排骨肉，袖子卷到手肘，冻得胳膊发红。"回来了。"他爸把竹竿架好。"嗯。"他说。

他妈从厨房探头。围裙上沾了面粉——手里还拿菜刀。菜刀上有几片蒜。"瘦了。"说完给他妈手里的菜刀还是握着的——看了他一眼——"晚饭要多吃点"。然后转身回去继续炖鸡。他走回自己房间。

回到自己房间。门上的锁还是那把——铜黄色的小挂锁。拧开了——锁芯有点涩。房间和四个月前走的时候一模一样——床上的被褥收在衣柜里，只剩一张床垫。桌子上的课本还在——摆得整齐——《五年高考三年模拟》的封面褪色了，曾经在上面演算过的很多草稿还夹在页缝间。窗台上落了薄灰。他用手抹了一下——留下一条干净的印。窗外能看到院子里的那棵柿子树——冬天叶子落光，枝干在灰色天空像用铅笔描出来的。树上还挂着两颗干柿子——黑瘪瘪的，被鸟吃了一半剩一半挂在枝头。

晚上吃饭——四菜一汤。红烧鸡块——酱油色浓厚，鸡皮是光滑的金黄色。蒜蓉油麦菜——绿油油的，蒜末的香味很冲。西红柿炒蛋——蛋还是炒得太老了。凉拌黄瓜——醋放多了。他妈给他夹了三回菜。"吃——大学食堂哪有这种。多吃——。"他爸不怎么说话，喝酒。酒是散装白酒——从塑料壶里往玻璃杯倒的，倒的时候会翻点白泡。他爸端着酒杯抿一口、咂嘴。

"在学校怎么样。"

"还行。"

"成绩呢。"

"班里第三。"

他爸举杯的手停了一下——举到一半停了。"第三？不赖。"把杯子端上来——然后嘴角动了动。不是笑——是那种想笑又不好意思笑的嘴角弧度。

三天以后——腊月二十三，小年。县城里的年味很浓——到处人挤人，街上铺子摊子都摆到了马路边。主街两边挂了红灯笼——塑料外壳的，里面装小灯泡，晚上会亮。灯笼有些歪了——风吹得它们在铁丝上打转。店铺门口贴着"春节大促销"海报——红纸黑字，浆糊还没干透，在风中微微翘着角。卖鞭炮的摊子摆在路边——红纸盒堆成小山。鞭炮的种类有十几种——小鞭、二踢脚、花炮、窜天猴。有人在摊子前面挑——随手拆了一包放了一挂小鞭，噼噼啪啪的，硝烟呛得路人侧目。街上的人踩着炸过的碎红纸屑走来走去。

他去县城街上给他妈买东西——对联、福字、窗花。他妈说买两副就行，大门口一幅，厨房门口一幅。他穿着那件新买的棉袄。拉链拉到下巴。手揣兜里。走得很慢。

经过新华书店——门脸没变，蓝色招牌，门口贴着"寒假特惠"横幅——这张横幅从高一他第一次看到现在居然还在用，"惠"字的那一点被洗得快看不见了。玻璃橱窗里摆着几本新书——《明朝那些事儿》和《哈利波特与死亡圣器》。经过修鞋摊——老头今天没出摊，马扎空着，用一根绳子拴在旁边的树上。马扎上的皮带裂了口——露出里面的白棉垫。经过那条有老槐树的巷口——他往里看了一眼。巷子深处有枇杷树的轮廓。

他继续往前走。在对联摊子前面停下来。摊子摆在路边——一块塑料布铺在人行道上，对联一摞一摞码在上面。每摞对联上搁一块石头压着。摊主是个白头发老头，坐在小马扎上。老头穿军绿色棉袄，领口露出灰色的毛衣边，手上戴着线手套——手指头位置磨出了洞。"随便看随便挑——两块钱一副——三块钱两副——"他把对联摊在塑料布上。金粉字在红纸上闪光——"春回大地""万事如意""家和万事兴""吉祥如意"。一股油墨的味道。他翻了翻——没有买。红纸金粉沾在手指上——他用拇指搓搓，金粉在拇指肚上留下亮晶晶的一片。搓着搓着手指。这时他站起来。

他正要走。抬头往街对面看了一眼。

然后他不动了。他的手停在半空里——金粉还粘在拇指上。

街对面是一家糕点铺。"老李糕点"——木头匾额，金漆边都掉了。门口摆着炒货摊——瓜子、花生、炒栗子。铁皮炉子里烤着红薯，焦甜的味飘了半条街。炉子上排着一圈红薯——皮烤得焦黑，裂了口，露出黄澄澄的瓤。一个穿白色羽绒服的女人站在摊子前面——白羽绒服是短款的，领口有一圈褐色的人造毛——正在接炒货摊主递过来的塑料袋。旁边站着一个中年女人。中年女人穿着深紫色棉袄，头发烫了小卷——像螺丝的卷，在她耳边颤着。正跟摊主讨价还价——"栗子怎么又贵了五毛——去年才八块——""过年了嘛——什么都涨了嘛——"

白色羽绒服。马尾。侧脸转过来的一瞬间——鼻梁的弧度，下巴的线条——太熟悉了。熟悉到就像一件一直藏在心里埋在底层的东西被一下子挖了出来。他认出了她的鼻子——挺，但鼻尖显圆。也认出了她的下巴——不尖，是一个圆弧形。她跟旁边的人说了一句话。嘴动了动。他没听见说的什么——街上的声音太杂了。但那个口型他认得。

是苏晚晴。

她站在糕点铺门口，也在办年货。她从摊主手里接过一袋糖炒栗子——栗子还冒着热气。低头跟身边的妈妈说了一句话——嘴动了动露出她的牙齿——大门牙有一点点不平，是歪的。然后她们往巷子那边走。走了几步。拐进去了。

那条巷子——有老槐树的巷子。她家的方向。

马尾在拐弯的时候甩了一下。幅度不大——就是她走路时惯常的摆动。跟她高一在走廊里端着水杯走过的样子一模一样。皮筋是黑的——好像跟高中用的一样的。

全程大概十几秒。有可能更长一点——十五秒左右——从认出她到她拐进巷口。

他站在街对面。没有动。手不知道什么时候在对联摊上拿起了一张"福"字——红底金粉做的，被他攥得紧紧的。金粉掉了些在手上。纸页边缘把他的掌心硌出一道红印。脚钉在地上——隔着一条街，二十米左右，两个人站在世界的两个边缘。她没有看到他。他看了她全程。

街上的人还在走。电动车喇叭还在滴滴答答。小孩还在笑。鞭炮还在响。这一切声音都变成了一层嗡嗡的背景音——像把头沉进水里以后听岸上的声音。

那个穿白色羽绒服的身影在巷口了——拐进去——不见了。那个马尾还在他的视网膜上留着一道残影——那条黑色皮筋兜着的发梢在空气里画完最后一条弧线。

然后街面上恢复了正常。

摊主抬头看了他一眼。"小伙子——你要不要——"

他低头看着手里那张纸。金粉被捏花了——"福"字的一半碎成了小亮片。纸上有手指攥出来的汗迹——汗迹在金粉上融成小小的亮痕。他把纸放回去。对摊主说了声"不好意思"。转身走了。

他走的不是她那个方向。他往反方向走了——经过菜市场后面绕回去，多走了快十分钟。菜市场下午还开着——卖鱼的摊子在地上泼水，鱼腥味混着水汽扑在脸上。卖肉的摊主把砍刀在砧板上剁——当当当——声音震得街道都在响。剁下来的猪骨头扔进塑料袋。他跨过地上的水。继续走。

走的不是她那个方向。他绕了另外一条路——从菜市场后面拐过去，多走了大概十分钟。菜市场下午还开着——卖鱼的摊子在往地上泼水，水带着鱼腥味流到路面上。他跨过去。继续走。

回到家。他妈在厨房里炸春卷。铁锅里咕嘟咕嘟翻滚着油——油花在春卷皮周围翻着白色的沫。油锅里的春卷炸得金黄——春卷皮边缘有焦黄的皱，脆得发光。豆腐干和韭菜碎在油里面翻滚着——嗞嗞响着——香味充满整个院子，从烟囱口往外飘。他经过厨房的时候他爸在院子里劈柴。斧头高高举起——当——木头被劈开，清脆的一声，飞出一截碎木屑。劈好的柴码在墙根——整整齐齐的一排。

"买到对联了吗。"他妈在厨房里喊。

"忘了。"他站在厨房门口。

"忘了？"他妈从厨房探头——围裙上全是面粉，手上沾满了油——举着筷子，筷子头上还夹着一块没翻完的春卷。"我说了对联要买你还忘了——"

"明天去买。"

他走进房间。把门关上了。坐在床垫上。床垫的弹簧硌着屁股——中间那块弹簧早就塌下去了，人坐着会陷在那。手心里还有刚才捏对联时沾上的金粉。他用拇指搓了搓指腹——金粉簌簌往下掉——在昏暗的冬日照到的那一绺光里慢慢飘、慢慢落。金粉在手指上像磨碎的星星——每一个细亮片都折着从窗户射进来的那绺光。

窗外的天是灰色的——那层灰让一切都变得平淡。屋檐上蹲了一只麻雀——歪着头，跳了两下，看了他一眼，然后飞走了。麻雀飞的时候蹬了一下瓦——一块碎瓦屑掉在窗台上，啪的一小响。

她。白色羽绒服。那个马尾在拐弯时甩的那一下——幅度不大。她拎着一袋栗子——还是糖炒的，热乎的——跟着她妈，步子没变——拐进那条有老槐树的巷子。对于她来说，那可能就是小年办年货的一个再普通不过的下午——买袋栗子，跟妈妈讨价还价一下，然后回家。如果她知道街对面站着的那个攥着对联发愣的人就是她亲口说了"暂时不要联系"的那个人——她会不会在拐进巷口之前转过头看一眼。也许她会的。也许她不会。也许她正在想别的事——学校的事、支教的事、寒假过后要考的教师资格证——脑子里全跟青阳的那个下午没关系。他不知道。隔着一条街，他离她太远了。

他没有追上去。他做到了——跟往常一样，远远地看着。然后走开。然后回家帮他妈洗菜。

他把手心里的金粉拍掉——拍了两下——金粉掉在地上，地上有一个细亮的点。站起来。把棉袄挂到椅背上——椅背的木头边角有一点磨损了。然后去厨房帮他妈洗菜。水龙头的水很冷——井水，冰得手指尖发红，骨节发僵。他把白菜叶子一片一片掰下来——白菜帮子上的淤泥要搓好几下才能洗干净，用拇指一点点刮掉。每掰一片白菜水就会溅到灶台上一点点。他妈在旁边切姜——菜刀落在砧板上果决又急促的当当当声。切好的姜片码在旁边——码得很齐。砧板上有几片切下来多掉的——挤在一角。旁边还搁着那盘春卷皮——还剩半摞。厨房里蒸汽和油烟混合着——他妈的身影在蒸汽里显得模糊，动作却清晰——她用长筷子夹住春卷放进油里，春卷在触油的瞬间炸起一层酥。油锅里的油花溅出一滴——正好溅在旁边没切的大蒜上。他妈又夹了一只放下去。

"再洗一棵。"

"嗯。"

他又拿了一棵。掰开。水继续流。
