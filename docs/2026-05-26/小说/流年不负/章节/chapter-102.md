# 第102章 她在一所小学

时间：2015年10月

地点：楚州

卷次：卷六：平行轨道

十月中旬。楚州的秋天很短。

树叶黄了几天就掉光了。图书馆外面那一排银杏，叶子落得满地都是。踩上去沙沙响。清洁工每天扫三趟，扫完了又落一层。

顾临风在图书馆看考研政治。看了两个小时。中间去接了一次水。饮水机在二楼走廊尽头，冷水龙头坏了，只出热水。他端着杯子等了半分钟，让水凉一点。回来继续看。

手机震了一下。

微信群。高中同学群。平时是死的。偶尔有人往里面扔一条链接——微商广告、拼车信息、寻人启事。他从来不回。

今天是活的。

许若琳发了三张照片。

第一张：一间教室。墙上有板报。黑板上画着一个巨大的"三年级"。板报上用彩色粉笔画了一只蝴蝶。蝴蝶翅膀上的粉笔颜色还没擦干净——大概是用手指抹的。讲台上放着一盒粉笔和一个黑板擦。黑板擦上沾满了白灰。

第二张：一群小孩在操场上做操。队伍歪歪扭扭的。前排有个小胖子动作特别大——胳膊甩起来比旁边的孩子高出来一截。后排有个男生根本没做——蹲在地上看操场边上的蚂蚁。红色塑胶跑道上拉着一道长长的影子。

第三张：一个女老师站在讲台上。侧脸。马尾。白衬衫。手里拿着一本语文书。衬衣袖口往上卷了两道。她的头发有一些碎发从马尾里散出来，贴在脖子上。

第三张是苏晚晴。

顾临风的拇指停在屏幕上。停的位置刚好盖住了她的半边脸。他把拇指移开。又放上去。又移开。

许若琳在照片下面配了几行字。

"今天去楚州一小看晚晴。她在那里实习。教三年级语文。"

"那些孩子很喜欢她。下课都围着她叫苏老师。有个小姑娘说苏老师最漂亮——全班都鼓掌了。"

"晚晴变了好多——比以前会笑了。站在讲台上的样子特别好看。"

群里开始冒泡。

"哇，苏晚晴当老师了。"

"小学老师好啊——有寒暑假。"

"她在哪个小学来着？"

许若琳回复："楚州一小。就在楚州师大后面那条街。"

顾临风盯着这几个字。

楚州一小。

楚州师大后面那条街。

他把照片放大。

苏晚晴站在讲台上。侧脸。马尾扎得比高中时候高了一点——原来是在后脑勺偏下，现在到了耳朵上方的位置。白衬衫袖子挽到手腕。手腕上什么饰品都没有——没有手表，没有手链。她以前也没有。手里拿着一本语文书——大概是三年级的。封面是浅绿色的，右上角印着一只小燕子。她的嘴在动——应该是正在讲课。嘴角有一点弧度。不是笑。是说话时自然带出来的那种弧度——嘴巴张到一半收回来，在"an"和"ang"之间过渡。

他看了很久。久到手机自动锁屏了。

解锁。又看了一遍。

第二张照片里她在操场上。站在队伍前面带操。动作很标准——举手、转身、蹲下。后面的小孩子们跟着做。这个做错了那个做慢了。前排有个小女生把方向搞反了——别人举左手她举右手。苏晚晴停下来，走过去，把她的手从右边挪到左边。动作很轻。像搬一只小碗。

阳光很好。她的影子落在塑胶跑道上。旁边是一个小女生的影子——踮着脚在学她的动作。两个影子一高一矮，在红色跑道上排在一起。

他的手指在屏幕上停着。

教室里那张。黑板旁边的墙上贴着一张手抄报——彩色粉笔画的，上面写着"三年级（2）班"。下面贴了几张学生的作业。字迹有大有小。有一张上面画了一个太阳——旁边歪歪扭扭地写着"我的老师苏老师"。太阳有八条光线，长短不一。最长的那条画弯了，像一根面条。旁边还有一个小孩画的小人——两根辫子，脸上点了三个点——两只眼睛和一张嘴。小人旁边写着"苏老师，你好像公主"。

他不知道她在楚州一小实习。不知道她教三年级。不知道她扎马尾的高度变了。不知道那些孩子在作业本上画了她。

现在是知道了。

他把照片看完。退出去。回到桌面。打开浏览器。在搜索框里输了几个字。

楚州一小。

地址：楚州市城西路126号。

点击地图。学校的卫星图像在屏幕上慢慢加载出来——一块灰色的矩形，旁边是一个绿色的操场。看了一会儿。然后关掉浏览器。

把手机翻过来放在桌上。屏幕朝下。过了几秒又翻过来。又看了一眼第三张照片。她的侧脸。白衬衫。微张的嘴。

然后把手机重新翻过来。屏幕朝下。不动了。

继续看考研政治。

看了大概十分钟。一个字没看进去。同一行字看了五遍。"物质决定意识"六个字在视网膜上晃过去又晃过来。他的眼睛在字上走，手放在桌上。手边的钢笔没盖上盖——干了。笔尖结了薄薄一层墨膜。他把笔在纸上划了两下，不出墨。

他重新翻开笔记本。翻到空白页。在左上角写了四个字——楚州一小。

然后划掉了。

划得很重。笔尖把纸划破了。纸破开的那道口子——从左上角一直拉到中间。像一条锯齿状的闪电。

他把这一页撕掉。撕得不对——从左往右，纸边上留下一条不规则的齿。他把齿也撕整齐了。叠了两折。扔进桌下的垃圾桶。

垃圾桶是空的。纸落在底上。孤零零的一张。垃圾桶是塑料的，浅蓝色，边缘有一道裂口——去年被阿坤踢出来的。那张纸在里面摊开着。透过那层薄薄的横格纸能看到桶底。上面写着的四个字——从空白的那一面对折压在里面——看不到了。

下午他去食堂吃饭。西红柿炒蛋。土豆丝。一个馒头。跟昨天一样。跟前天也一样。打菜的大妈看了他一眼。"小伙子——多吃点肉。看你瘦的。"他点了点头。多打了一份红烧肉。三块钱。咬了一口。油有点大。吃完了。

把碗端到回收口。白瓷碗在金属台面上碰了一下。铛。碗里的菜汤晃出来溅在回收台的边缘。他用袖子擦掉了。

回宿舍的路上经过校门口。秋天的傍晚来得早。五点半的天已经有点暗了。他往右拐了一下——那边是往城西路的方向。走了几步。停了。

站在人行道上。风灌进领口。他把围巾紧了紧。

然后转身。继续往宿舍走。步速比平时快了一点。

李博在宿舍里打游戏。屏幕上的英雄正在团战。噼里啪啦的特效声音。李博一边点鼠标一边骂队友。"老顾——你怎么脸有点白。"

"没怎么。"

"感冒了？"

"没。"

他坐下。打开台灯。拿出考研政治。把书翻到第三章。手指沿着目录往下点——找到了刚才看的那个小节。

李博隔着桌子看了他一眼。没追问。然后又把注意力转回游戏。"辅助——辅助上来——辅助你在干嘛——"

晚上十点。熄灯。

他又翻出手机。打开许若琳的微信朋友圈。往下翻。翻过了李博的女朋友晒饭，翻过了小伍转发的考研帖子，翻过了阿坤的公司聚餐照。三张照片还在。他看了第三张。苏晚晴的侧脸。

然后关掉手机。屏幕的余光在黑暗里弥留了几秒。然后全黑了。

天花板上那条裂缝还在。

窗外的路灯穿过窗帘。光很薄。落在地上。黄澄澄的一片。那片光里有铁丝网和树枝的影子——像窗外的世界被压成了一张平面图贴在宿舍地板上。

楚州一小。城西路126号。

从他学校骑车过去大概二十分钟。公交车四站——坐12路。走路四十分钟。

他闭了一下眼睛。

眼前是她站在讲台上的侧脸。马尾比高中高了一点。白衬衫袖子挽到手腕。手拿语文书。嘴在动——在讲课。台下的孩子们仰着头看她。有个小女孩在画她的脸——两个点作眼睛，一条线作嘴。她不介意。

他没有去。

天花板上的裂缝在月光里。他盯着那道裂缝。盯着。然后翻了个身。脸冲着墙。墙上的小孔还在。黑的。什么都看不到。他伸手摸了一下——指尖触到冰凉的墙灰。那个孔的大小刚好能放进一根食指。他没放进去。把手缩回被子里。
