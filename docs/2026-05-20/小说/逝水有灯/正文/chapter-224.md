# 第224章 重回县一中旁观

所属部卷：第五部：逝水有灯
时间：同一天
地点：县一中翻新的校门

## 诗歌插页 P224：原创现代诗

保安记不住每个校友
他只记得
访客牌少了一张
又还回来一张
秩序很轻
却托住很多人

保安姓冯，在县一中门口值了六年班。每天进出的人很多，送饭的家长、维修师傅、来开会的老师、偶尔返校的校友。陆允川在登记本上写名字时，他只扫了一眼，确认身份证号和电话填全，就把访客牌递了过去。

这种校友他见过不少。有人一进门就拍照，有人站在教学楼前讲很久，有人问以前的老师还在不在。冯师傅不觉得奇怪。学校对学生来说是青春，对他来说是工作地点，早晚开门、检查车辆、提醒来访者不要乱进教室。

陆允川还牌时，神色很平静。冯师傅接过来，挂回钩子上，又继续接电话。等电话挂断，他在登记本后面画了个勾，表示人已经离校。

这一个勾对陆允川来说也许很轻，对学校来说却很重要。来过的人要离开，外面的人不能随便闯入，里面的学生要继续上课。青春再有光，也不能让现在的人替它停下。

下午放学，冯师傅站在门口维持秩序。学生从校门里涌出来，背包、试卷、水杯和笑声混在一起。他早已不记得上午那个校友的脸，只记得访客牌都还齐了。很多故事被门口这样轻轻合上。
