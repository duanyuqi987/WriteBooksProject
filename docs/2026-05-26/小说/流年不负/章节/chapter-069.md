# 第069章 等待

时间：2013年6月9日

地点：家里

卷次：卷四：夏夜湖光

六月九号。

高考完的第一天。

顾临风躺在床上。眼睛睁着。天花板上的水渍还在。去年春天就在的那块。形状像一只摊开的手掌。

他盯着水渍看了很久。

手指数了一遍。一根。两根。三根——食指的位置。中指的位置。无名指的位置——那个位置有一小块新晕开的印子。昨晚下雨了？

手机在枕头边。

黑屏。

他把手机拿起来。按了一下旁边的键。屏幕亮了。时间——八点十七分。没有消息。

放到枕头边。

翻了个身。

左侧睡。腿蜷起来。左边的膝盖压住被子的一角。被套是蓝灰色的——他妈上周刚换的。洗衣粉的味道还挺浓。

手机又亮了。他翻身去看——10086。余额不足提醒。他把手机的提示音关了。屏幕的光在黑暗里刺了一下眼——他醒了以后把窗帘拉得很低，房间很暗。

侧回仰面。

看着天花板。

三年前的九月一号。她穿着浅蓝色的T恤走进教室。阳光打在她侧脸。他在课本上写了一个S——写了一半——划掉了。那天他穿的是灰色T恤。他记得自己穿的灰色T恤。她不记得。

他说"我记得"的时候她停了一下。

那句话停住的空隙——有一秒左右——比"三年"更重。

天花板上的水渍在视线里慢慢失焦。

他把手放在胸口。胳膊交叉压在锁骨下面。能感觉到自己的心跳。不快。但每次跳都闷闷的。像有人在胸腔里锤棉花。

九点钟。

九点半。

十点。

他起来上厕所。穿着拖鞋走到客厅。他妈在厨房里择菜。豆芽。她一根一根把豆芽尾掐掉。

"起来了？"

"嗯。"

"学都不考了还睡这么晚。"

他没说话。走进洗手间。把门关上。没上锁。门缝底下能看到光——客厅的光亮着。

他在马桶上坐着。没上厕所。就是坐着。

洗手台上有一瓶洗手液。蓝色的。牌子没注意过。他拧开水龙头。用凉水冲了手。冲了很久。手凉透了才关水龙头。

回到床上。

手机亮了。

赵一鸣的短信。

"怎么样了"

四个字。顾临风看了这四个字大概看了十几秒。没回。

他把手机翻面盖在床上。屏幕朝下。

然后又把手机拿起来。翻回来。打开QQ。眼睛盯着联系人列表上那个灰色的头像。灰色的。离线状态。

退出QQ。

把手机扣回床上。

午饭的时间到了。他妈推开门。手里还捏着一根葱。

"吃饭了。"

"不饿。"

"不饿也得吃。早饭就没吃。"

"不饿。"

他妈走过来。伸手摸他的额头。手上有葱味。凉的。

"发烧了？"

"没有。"

"你脸有点白。"

"没睡好。"

他妈站在床边看了他一会儿。嘴动了动。没问。她大概从儿子脸上的某个地方——也许是眉毛的角度、也许是嘴唇干的纹——判断出了这个"没睡好"不是需要她追问的"没睡好"。

她走出去了。

门没关紧。留了一条缝。

他看到那条缝里透进的光。中午的阳光是白亮的。白中带黄。

门缝里的光从左边慢慢移到了右边。光在动——他没动。

下午两点。

他从床上坐起来。

下床。走到书桌前。书桌上堆着复习资料——《五年高考三年模拟》《高考语文必备》《英语作文模板》。所有东西都堆在左边。右边清出了一小块地方——那是他以前写诗的时候放纸的地方。

他拉开书桌的抽屉。第二格。

里面有个笔记本。横条的。封面是牛皮纸的颜色。没有图案。只有角上磨白了。

翻到中间。

一页一页翻过去。

每页纸上都有划掉的字。划掉的比他留下的多。有的整首诗都被划掉了——横线、竖线、交叉线——下面工工整整地重新抄了一遍。

有一页他只写了两个字。

"晚晴"

没有划掉。

他看了很久。然后把本子合上了。

放回抽屉。

他走到窗口。把窗帘拉开一半。楼下的街道上没有什么人。三十四五度的下午，路面上的柏油被晒得有点软——一个骑自行车的人骑过去，车胎在路面上留下浅浅的痕。很快消失了。

他换了一双鞋。出门。

没有明确的目的地。只是走。走到巷口拐过新华书店——今天的书店没什么人。暑假的学生还没涌出来。玻璃橱窗上映着他一个人的影子。灰色的T恤。头发挡着眉毛。

他往里面看了一眼。书架之间没有人。

继续走。走到月湖路口的时候他的脚步慢下来。从这里能看到月湖的一角——湖水上阳光碎成一片。柳树的叶子在太阳下发白。湖边的木椅上坐着两个人——一个老头一个老太太——离得太远看不清脸。

他没有往湖边去。

在路口的冷饮摊买了一瓶矿泉水。拧开喝了一口。温水——冰柜里的水早就不冰了，太阳晒在冰柜玻璃上把里面的冷气都赶走了。

他把水瓶拿在手里，沿着月湖路往南走。走了大概二十分钟。路过一些店铺——关了门的台球室、一家五金店、一家贴着"店面转让"的面馆。县城在正午的时候总是这样——安静得像所有人都在午睡。

走到尽头再折回来。

又经过月湖路口。湖上的老人还在。太阳从头顶往西偏了。他看了一下手机——下午一点半。

往回走。

到家的时候他妈在客厅沙发上靠着眯了一会儿。电视机开着，静音。屏幕上的字幕在滚动。他没有惊动她，直接走回房间。门关上了。鞋没脱。

躺在床上。枕头上还有早上枕的凹痕。他把手搭在额头上。胳膊肘压着枕头边。

手机又亮了。

他翻身去看——10086。余额不足提醒。刚才在外面的时候也收到了一条。

他把手机的提示音关了。屏幕的光在黑暗里刺了一下眼——他醒了以后把窗帘拉得很低，房间很暗。

侧回仰面。天花板上的水渍还在。

QQ消息。

他从床角捞过手机。屏幕亮着。通知栏里有一行字——"苏晚晴请求添加您为好友"。

他的手顿了一下。大拇指悬在屏幕上方。

附加消息：我是苏晚晴。

他把屏幕按亮。点开QQ。同意。

页面跳转到聊天框。

她的头像是月亮的图片——他放大看。不是系统自带的月亮。是自己拍的。拍的月湖。

聊天框上方显示"对方正在输入..."

停了。

又显示。

又停了。

又显示。

他把手机握在两只手里。两个大拇指悬着。

等了大概三分钟。

最后一条消息到了。

就一个字。

嗨。
