# 第049章 一张小纸条



时间：2008年十二月下旬，晚自习第一节



地点：十班教室第三组和讲台边



卷次：第三卷：雪落在两栋楼之间



2008年十二月下旬，晚自习第一节，十班教室第三组和讲台边，夹在数学卷子里的小纸条先把这一天按进了十班的日常里。一张提醒补公式的小纸条被误传成暧昧，十班的玩笑第一次越过边界。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

段璟想解释，常俊凡先把纸条收回，告诉大家别把普通帮忙说歪。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看夹在数学卷子里的小纸条，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

曹蔚也在这一天被推到前面。曹蔚的特点是组织能力强，成绩普通但很能收场，班级杂事常落到他手里。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“别传了，纸条上就一道公式。”曹蔚说。

“我还以为有大事。”付小川说。

“你们的大事，很多时候只是别人的麻烦。”常俊凡说。

“纸条是我问她借的。”段璟说。

“笑别人之前，先看看自己卷子空了几道题。”叶书兰说。

小纸条在十班不是稀罕事。有人传题号，有人借橡皮，有人写下课去不去水房。

真正麻烦的是，纸条一旦被起哄，就会从一件小事变成谁也不好收拾的热闹。

段璟看见纸条角露出来时，第一反应不是好奇，而是上一次被误会后常俊凡皱起的眉。

他把纸条递回去，手指没有多停。这个动作太普通，普通到旁边人找不到起哄的缝。

宋乔看了他一眼，像在确认他这次终于学聪明了。

付小川张了张嘴，又自己闭上，把玩笑咽回去，低头装作研究橡皮屑。

常俊凡接过纸条时只说了声谢谢，声音不大，却把边界放得很清楚。

十班从这些小事里慢慢学会，热闹不是想要就要，别人的难处也不是拿来传的。

一张小纸条没有悬在空处。小纸条折痕旁边沾着的一点粉笔灰，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，一张小纸条带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

叶书兰进门时不一定立刻说话，鞋底在水泥地上一停，一张小纸条带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在曹蔚快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。一张小纸条越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。夹在数学卷子里的小纸条让他想说话，又逼他先低头看题。

曹蔚被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，一张小纸条留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。曹蔚路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为一张小纸条翻身，谁还在被窝里背单词，旁边人都听得见。

县城晚上暗得早，小店门口的灯一亮，纸条上的玩笑就显得没有白天那么轻。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理一张小纸条里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：夹在数学卷子里的小纸条、一只冻红的手、一句老师点名、一块还没扫干净的雪。

事情是从数学卷子发下来开始的。

晚自习第一节，陶建国把批完的单元测试卷发下来，教室里弥漫着一股刚印出来的油墨味。段璟翻到自己那张卷子，看见最后一道大题旁边用红笔写了两个字：公式。他没有往下看完，因为那两个字的意思他很清楚——应用题的公式列错了，答案对了也是蒙的。

他翻了翻课本，发现三角函数那页折了角，上面补的公式是上一周抄的，字迹已经模糊了。他侧头往左前方看了一眼——常俊凡的桌上摊着数学错题本，本子边缘露出半张草稿纸，草稿纸上密密麻麻写满了公式推导。

他没有直接开口，而是从草稿本上撕下一角，写了几个字：sin2θ的公式你还记得吗？然后把纸片折成小方块，用手指轻轻推到了桌子边缘。

常俊凡正在订正选择题，感觉到旁边的动静，低头看了一眼桌角。她没立刻拿，先抬头瞄了一眼讲台上的陶建国——陶建国正低头批作业，眼镜滑到鼻梁中间。确认安全后，她把纸片拈起来，展开，看了一眼，在下面写了三行公式。字很小，但每个符号都写得很清楚，sin和cos的笔画分得干干净净。她把纸片折回去，用食指和中指夹着，从椅子侧面递了过去。

整个过程不超过十五秒。

但纸条在传回来的路上出了问题。常俊凡递回来时，段璟的同桌刚好转身翻书包，胳膊肘撞到了段璟的手腕。纸条从指缝间滑出去，掉在了过道上。

付小川从后排探过头，目光落在纸条上，脸上浮起一个意味深长的笑。"喔——"他把这个"喔"拖得很长，长到旁边两三排的人都转头看过来。

段璟弯腰把纸条捡起来，动作很快，但已经来不及了。付小川已经用气声朝他前后左右的人说了一句："有人传纸条。"

这四个字在十班这种地方，跟"有人谈恋爱"是一个意思。

后排几个男生开始低声起哄。有人用笔敲桌子，有人扭头往常俊凡那边看。何南星不在起哄的人里，但他也没反应过来该怎么帮，只能把身体坐直了一点，试图挡住后排往前的视线。

段璟握着那张纸条，手心里出了一点汗。他知道只要展开让大家看——看，就一道公式——起哄就会停。但他没有。因为他想起来，上一次有人被传谣的时候也是这样想的：我解释清楚就行了。结果解释完那天，谣言没有停，只是换了一个说法继续传。

常俊凡在这时候转了过来。

她没有站起来，没有提高声音，甚至没有看那些起哄的人。她只是侧过半个身子，把手里正在做的数学卷子翻过来，用笔尖指着最后一道大题，对段璟说了一句话，声音不大不小，刚好够前后两三排听见："公式你抄完了吗？这道题等一下老师要讲，你得先看懂这一步。"

起哄声低了一半。

不是因为她说了什么特别的话，而是因为她说话的语气太平常了。跟平时讲解题一样，跟平时催交作业一样，跟平时告诉付小川"别闹了"一样。这种平常本身就是在说：你们觉得有猫腻的事，在我这里就是一道题。

付小川把嘴闭上了。他不是被吓到，而是被常俊凡的语气提醒了一件事——他刚才的笑，可能真的笑早了。

段璟趁机把纸条展开，摊在桌面上。纸条上除了他写的那句询问和三行公式回复之外，什么都没有。没有多余的字，没有画圈，没有表情符号，甚至没有"不客气"。就是干干净净的数学公式。

旁边几个探头的人看到了，表情从兴奋变成了无聊，从无聊变成了不好意思。后排最后一声起哄收住之后，教室里安静了几秒。日光灯管的嗡嗡声又浮上来，填满了刚才被笑声占住的空间。

宋乔从前排扭过头，看了段璟一眼。那个眼神不是责备，也不是同情，更像在说：你这次总算没有一头撞上去。她把头转回去之前，嘴唇动了动，段璟读出了那两个字：收好。

段璟把纸条夹进数学课本的三角函数那一页。手指按在纸片上，能感觉到纸的背面有笔尖压出的凹痕——那是常俊凡写字时留下的力度。三行公式，每一行都在纸上按出了形状。

第二节晚自习的课间，曹蔚从后排走到前面来倒水，路过段璟时停下来，弯腰看了一眼他课本里夹的纸条角。"你那个公式，"他用气声说，"中间那一行少抄了一个括号。sin2θ = 2sinθcosθ，不是sin2θ = 2sinθcos。"

"我知道。"

"你知道就好。"曹蔚直起腰，端着搪瓷杯继续往前走。走了两步又回头，"刚才是付小川先起哄的，你别怪他。他不是故意的。"

"我没怪他。"

"那就好。"曹蔚走到饮水机前面，拧开水龙头。热水冲进杯子里，白气升起来，罩住了他的脸。"你说我们班这些人，怎么什么事都能往那个方向想？"

段璟没有回答这个问题。他看了看自己的草稿纸，上面用铅笔写着常俊凡刚才递来的公式，他抄了三遍。第三遍的时候，sin的s写歪了，像一条滑倒的蛇。

曹蔚端着热水回来时，在他旁边停了几秒，忽然压低了声音说了一句话，让段璟后来记了很久："其实大家不是想看笑话，是不知道该怎么面对自己也有这种念头。起哄最响的人，往往是心里最慌的。"

段璟抬头看他。曹蔚的表情不是玩笑，也不是说教，而是一种在十班里少见的沉稳。他拍了拍段璟的肩膀，走回了自己的座位。

晚自习第三节，马立群来教室收英语作业。她从第一排开始收，收到第三排时在常俊凡身边停了一下。常俊凡把作业本递给她，她翻开看了一眼，点了点头，合上放进怀里那摞本子最上面。

收到段璟这里时，马立群的手指在本子边缘停了几秒。她没有翻开，只是看着段璟说了一句："听录音了吗？"

"听了，每天都听。"

"那你试试，明天上课我让你念哪段你就念哪段。"

段璟点了点头。马立群走后，付小川在后面用气声问："你今天怎么答得这么硬气？"

段璟说："因为我真的听了。"

付小川没再说话。过了一会儿，一张纸条从后排传过来，这次是付小川折的。段璟展开看，上面写着一行字，笔迹潦草到几乎认不出：刚才对不住。段璟看完，把纸条折回去，在上面加了一行字传回去：下次你别光道歉，先把公式背会。付小川收到纸条后在后面压着嗓子笑了一声，很短，像被掐住脖子的鸭子。

第三节晚自习下课时，段璟去水房接水。水房地上照例是几排湿脚印，暖气管在墙角低声咕噜。他拧开水龙头时，发现常俊凡也在——她站在水池另一边，正把搪瓷杯里的旧茶叶倒进水槽，茶叶渣在搪瓷壁上留下一圈深色的痕迹。

两个人都没说话。水龙头的声音在窄小的空间里被放大，像隔在两个人之间的一小片帘子。

最后还是段璟先开了口："刚才的事，对不住。"

常俊凡把杯子放在水龙头底下冲刷，没有转头。"你有什么好对不住的。"

"纸条是我先写的。"

"你写的是公式，没写别的。"她把水龙头拧紧，杯底残留的水珠滴进水槽里，"别人怎么想不是你能控制的。"

段璟不知道怎么接这句话。他低头看着自己杯子里的水位慢慢升上来，热气扑在脸上，有一瞬间镜片起了雾。

常俊凡转过身准备走，经过他身边时停了一下。"但是下次——"她把杯盖拧紧，"下次如果还是公式，不用折那么小，直接递过来就行。越小心越像有事。"

段璟愣了一下，然后说好。

常俊凡推开纱门走出去时，冷风从门缝里灌进来，段璟打了个寒噤。他站在水池前停了一会儿，忽然发现自己在笑。不是那种被人逗的笑，而是一种从胸口往上涌的、压不下去的释然。

收拾书本的时候，段璟发现数学课本里夹的那张纸条不知什么时候被他自己又拿出来看了一遍。三行公式之外，他发现常俊凡在纸的最下面还写了一行极小的字，小到他刚才一直没注意到：对了，你的sin写得像一条蛇。后面画了一个箭头，箭头指向的地方写了一个示范的s。

段璟看着那一行小字，在空教室里忽然笑了一下。

他自己也不知道为什么笑。可能是因为那个箭头画得很认真，连箭头尾巴上的羽毛都画了出来——虽然那不是羽毛，只是她写字时有把横线稍微带弯的习惯。也可能是因为，在一个差点被全班起哄的晚上，她在跟他说公式之外，还顺便提醒了他一件事：把字写好看。

晚自习结束铃响时，段璟把那张纸条从课本里拿出来，想了想，夹进了文具盒最下面那层。那里已经躺了一张纸——上次英语改错答案那张。两张纸叠在一起，一张写满单词，一张写满公式。放在一起看，像他这段时间学习进度的小型展览。

走出教室时，外面的风停了。操场上那盏路灯照常亮着，光晕里没有冰粒也没有雪花，只有一盏灯安安静静地照着水泥地面。常俊凡和宋乔走在前面，两个人的影子被灯光拉得很长，并排向前移动，偶尔分开，偶尔重叠。

宋乔的声音从前面传过来，被风吹散了一半："你今天那句'这道题等一下老师要讲'，我觉得可以载入十班史册。"

"为什么？"

"因为这句话杀伤力比拍桌子还大。"

常俊凡没有回答。她走了几步，把手从口袋里抽出来，调整了一下书包的带子。然后段璟听见她用一种比平时淡一点的语气说了一句话，像是自言自语，也像是说给风听："我只是不想他把事情搞砸。"

宋乔没有说话，但她侧头看了常俊凡一眼。那个眼神在路灯下看不真切，但段璟觉得里面有笑。

段璟放慢了脚步，让自己和前面的两个人之间多隔了几步。付小川从后面追上来，跟他并排走，把棉袄的帽子扣在头上，帽子边上的一圈人造毛在路灯下闪着微微的光。

"下次再有这种事，我绝对不起哄了。"付小川说。

"行。"

"但你要告诉我，那纸条上到底写的什么。"

"数学公式。"

"就数学公式？"

"就数学公式。"

付小川沉默了一会儿，然后把帽子往后一推，露出整张脸。"那也挺好的，"他说，"数学公式比什么都有用。"

段璟低头看了一眼自己的文具盒，夹层里那两张纸片安安静静地躺着。他不知道这两张纸能保留多久——它们会被水打湿，会被橡皮屑蹭脏，可能哪一天换文具盒的时候就被夹在旧东西里扔掉了。但在2008年冬天的一个晚自习之后，它们确实躺在那里，像两件不起眼的证据，证明有些事情不是没有被说破，而是不需要被说破。

一张小纸条之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

曹蔚没有因此变成特别醒目的人，可在段璟眼里，曹蔚从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，这件小事让段璟学会在别人起哄时先保护对方，而不是急着证明自己。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
