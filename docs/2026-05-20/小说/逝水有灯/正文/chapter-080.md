# 第080章 毕业照边缘多年后

所属部卷：第二部：六月以后
时间：多年后
地点：照片文件夹

## 诗歌插页 P080：原创仿唐绝句

旧照重开像素低，
边中人物各东西。
同框不是旧盟约，
只是青春站一席。

多年后，陆允川把毕业照扫描进电脑。

纸质照片保存得不好，边角有点卷。扫描后放大，所有人的脸都糊成一小片。程砚仍然很好认，因为表情太夸张。夏知遥站在第二排，陆允川站在边上，中间隔着半个班级。

以前他总爱放大这张照片。放大到画面失真，也要确认他们确实在同一张图里。现在他觉得那样有点辛苦。同框不是缘分的证明，只是学校安排大家在同一天站到同一块操场上。

可这并不让照片变得无意义。

它仍然珍贵。因为那里面有整整一个班，有晒白的操场，有老师的白衬衫，有程砚欠揍的笑，有夏知遥年轻的脸，也有一个站在边缘、不知道怎样和青春告别的自己。

陆允川给文件命名：2012毕业照。

他没有再裁剪，没有单独截出谁。完整的照片比截出来的部分更诚实。青春不是只属于某一次凝望，它属于一整群人挤在太阳下，听摄影师喊一二三。

他把照片放进“高中照片”文件夹，关掉预览。那张曾经被反复放大的边缘，终于回到了它该在的位置。
