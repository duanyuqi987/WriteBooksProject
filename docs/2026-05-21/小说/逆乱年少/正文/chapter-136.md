# 第136章 告别

时间：2012年8月3日，上午

地点：李家沟村小教室及操场

卷次：第六卷：雪落无声

支教最后一天。

早上起来，天气很好，天蓝得像洗过的瓷碗底。山里的风吹过来带着早晨特有的凉意和草叶上露水的味道。但没有人觉得今天是好天气。

上午是最后一节课。苏安晴没有讲新课。她让每个孩子站起来说一句话——随便说什么。

"我想苏老师明年还来。"

"我学会查字典了。"

"老师你脚好了吗？"

"外面的大学长什么样？"

铁蛋是最后一个站起来的。他站起来，手撑着桌沿，想了很久。久到旁边的同学开始拽他衣角。

"苏老师，"他说，"你走了以后，我还能给你写信吗？"

苏安晴站在讲台上。她把手撑在讲桌边缘上，手指按在教案上。

"能。"

"你怎么收到？"

"我把地址写给你。"

她从讲台的粉笔盒下面抽出一张纸，上面已经提前写好了学校的地址。她把纸对折了一下，走到铁蛋面前，放在他桌子上。

铁蛋低头看了看那张纸，把它小心地夹进了课本第一页里。

然后他开始哭。

不是大声嚎啕——是那种憋着憋着憋不住了，眼泪从眼眶里涌出来，顺着腮帮子往下淌。他也不擦，就低着头看着课本，看着那张写着地址的纸。

然后第二个孩子哭了，第三个，第四个。女生哭出了声音，男生拼命忍着但眼睛红了。

苏安晴看着这十四张脸。

然后她也哭了。

不是嚎啕，也不是嚎啕的反面——就是眼泪安安静静地流下来。她没有擦，也没有转身，就那么站在讲台上，面对着十四个孩子，让眼泪淌了一脸。

教室里安静了大概十秒。

然后坐在中间排的女生刘小云站起来，走到讲台边上，把手里攥着的一颗水果糖放在讲桌上。"苏老师，给你糖。"

然后孩子们一个接一个地站起来，往讲桌上放东西。一个苹果、半袋瓜子、一张画了画的纸、一颗核桃、一朵路边摘的野花——不知道什么时候摘的，花瓣都蔫了，但每一瓣都还在。

讲桌被铺满了。

苏安晴低头看着一桌子的东西。她张了张嘴想说什么，但什么都没说出来。

林玉起站在教室后门外。他没有进去。他知道这个时刻不属于他。

他看见苏安晴的背——很瘦，肩膀在微微发抖。她的头发用一根深蓝色的发圈扎着，脖子上被晒出了一道分明的印子——上半截白，下半截黑。

那道晒痕是这一个月留下的。

>
黑板字未干，讲台泪已满。
一月同窗短，半生谁会忘。
