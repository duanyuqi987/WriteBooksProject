# 第144章 独阅

时间：2012年11月初，晚上

地点：江城大学图书馆三楼自习区

卷次：第六卷：雪落无声

十一月初的一个晚上。

苏安晴约他在图书馆见面——"你上次说好的，第一首给我看"。所以林玉起带着小本子来了。

图书馆三楼自习区。晚上八点半，人不多。靠窗的位置空着大半，窗外的路灯把光投进来，在桌面上铺了一层淡淡的橘色。

苏安晴已经坐在那里了。面前摊着一本教育学的教材。她看见他来了，把教材推到一边，腾出位置。

"带来了吗？"

林玉起把本子放在桌上。一共三首。他用那支钢笔抄了三遍——每首都是一遍一遍地写，写到满意为止。最后一遍的字迹很工整，看得出来写得很慢。

苏安晴翻开本子。

第一首。关于支教的山路。她看了一分钟。没说话。翻到第二首。关于银河和夜晚。看了一分半钟。翻到第三首。关于——没有题目，也没有意象里的山和星星。只有几行很直白的句子。第一行是"你是人群里我看得最久的人"。

苏安晴在这第三首上停了很久。

她的手指按在本子的边缘上，指甲是修剪过的，干干净净的，没有涂指甲油。她读诗的时候嘴唇会微微地动一下——不是念出来，是跟着默念。

然后她轻轻地把本子合上了。

"第三首——"她开口，然后又停住了。

林玉起等着。

"第三首你写的是谁？"

这句话她问得很轻。不是质问，不是试探。是一个真问题——她真的想知道，但她不确定自己该不该这么问。

窗外的风把图书馆窗户的把手吹得轻轻响了一下。

林玉起没有立刻回答。他看着苏安晴按在桌边的手指——指尖因为用力而有点发白。

"你知道的。"他说。

这三个字说出口之后，他立刻后悔了——太直接了。但同时又觉得松了一口气——终于说了一句直接的话。

苏安晴没有看他。她低头看着合上的本子封面。

自习区的空调嗡嗡地响着。隔壁桌一个女生在翻书，纸页哗啦了一下。

"林玉起。你知道吗——"她把本子推回来给他。"——你写诗的时候比你说话的时候勇敢。"

"我知道。"

"那你为什么不一直写诗。"

这个问题不在他的预料之内。

苏安晴终于抬起头来看着他。她的表情很复杂——不是喜怒哀乐中的任何一种，是所有这些的混合物，被一张冷静的脸严密地盖住了。

"你的诗写得很好。"她说。"这支笔没白买。"

然后她把教育学的教材拿回来，翻到刚才看到的那一页。

"我明天要考试。不跟你聊了。"

她把书拿起来挡在面前，挡住了半张脸。但从侧面能看到她的耳朵——红了。

林玉起注意到了。

他没有说什么，把本子装回包里，站起来，走了两步。

"苏安晴。"

"嗯？"

"你刚才说的那句话——'你写诗的时候比你说话的时候勇敢'——这句话我会记住。"

书后面没有声音。

他走了。走到楼梯口的时候回头看了一眼。苏安晴已经把书放下了。她在看他离去的方向。

两个人的目光在图书馆的灯光里碰了一下。

然后她低下头，继续看书。

>
三首诗给你，一支笔写君。
说话总太怯，纸上敢真心。
>
