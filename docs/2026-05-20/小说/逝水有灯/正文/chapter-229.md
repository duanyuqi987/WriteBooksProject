# 第229章 南湖再见旁观

所属部卷：第五部：逝水有灯
时间：同一天傍晚
地点：大学南湖凉亭

## 诗歌插页 P229：原创现代诗

新生把帐篷撑开
不知道这里曾有人难过
她只关心
横幅有没有歪
报名表会不会被风吹走

那天傍晚，诗社新一届负责人在南湖边试摆招新帐篷。她叫陈予宁，大二，手里拿着一卷透明胶和一摞报名表。凉亭里坐着一个穿衬衫的男人，她看了一眼，以为是附近培训的老师，便没有打扰。

帐篷支架不好扣，两个学弟弄了半天。陈予宁蹲下去重新对孔，手指被夹了一下，疼得吸了口气。她抬头看见那个男人还坐在凉亭里，看湖面，看手机，又看柱子。她觉得有点奇怪，却很快被横幅尺寸拉回现实。

“字歪了。”她对学弟说，“往左一点。”

对她来说，南湖是招新地点，是活动照片背景，是晚上开会前顺路买饭的地方。她不知道多年前这里有过问答、沉默、误会和劝告。她也不需要知道。校园的每一处都被很多人使用，谁都不能用自己的故事把它占满。

等陆允川离开后，陈予宁把报名表压在文件夹下面，防止被风吹散。她在群里发消息：明天九点集合，别迟到。

南湖边的灯一盏盏亮起来。新的年轻人开始忙自己的事。旧故事退到旁边，没有消失，也不再挡路。
