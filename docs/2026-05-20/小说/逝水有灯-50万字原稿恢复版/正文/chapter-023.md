# 第023章 操场边的单词之后

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P023：原创仿宋小令

风过三楼，
纸页小窗明。
欲把旧年藏一笔，
却听铃后有人行。
灯下水声轻。

操场边的单词之后，陆允川表面上没有变化。早读照旧，跑操照旧，作业本照旧从前排传到后排，可他自己知道，有一处地方被轻轻碰过。

铃声响起来时，操场上的人群像被一根线牵着往教学楼走。陆允川合上书，页角被他无意识折出一个三角，像给某个没发生的瞬间做了记号。

他把单词书页角折起的三角反复想了几遍，每想一遍都添一点不存在的意思。到第三遍时，他自己也觉得过分，便打开练习册，硬把目光压到题目上。

“你这单词背得挺远，眼神都跑两百米了。”程砚说。“我看她们是不是要上楼。”陆允川说。“你管得比值周老师还宽。”程砚说。

程砚把自己的单词本拍到他膝盖上，叫他随机抽查。陆允川抽到的第一个词是 patient，他愣了一下，程砚说，正好，你先学这个。

晚饭时食堂排队很长，铁勺敲在不锈钢盆上，声音一阵一阵。陆允川端着饭找位置，程砚抢先坐下，把一半桌面让给他。

“你别一有事就不吃饭。”程砚把筷子递过来，“感情没进展，胃先退学，不划算。”陆允川接过筷子，笑了一下，笑得很短。

叶澄从主席台旁边经过，抱着广播站的稿子。她看见他们坐在看台上，远远喊了一句快上自习，别等铃响才跑。

夏知遥并不知道这些后续。夏知遥和同伴从看台下经过，手里拿着水杯。她笑的时候没有看这边，声音被风吹散，只留下几个轻快的尾音。她有自己的课表、错题、家里打来的电话和不想解释的疲惫，不可能替陆允川安置所有心情。

这一点，是陆允川很晚才懂的。当时他只懂把一个小动作翻来覆去，像把一颗糖含到没有味道，还舍不得吐掉。

跑道边有卖烤肠的小摊，油烟越过围栏飘进来。有人边跑边喊借饭卡，有人在看台最上面弹吉他，音不准，倒也热闹。陆允川坐在这堆声音里，心事显得没那么孤单。

夜里回到宿舍，他在日记本上记了几行，又怕别人看见，把本子压到枕头下面。窗外的风刮过晾衣绳，衣架轻轻碰撞，像有人替他把没说完的话敲碎。

看台最下一级积着一层灰，陆允川坐下前用书拍了拍，灰尘扬起来，被夕阳照成细小的金色。程砚嫌他讲究，直接坐下，裤子上立刻多了一道白印。

夏知遥从跑道边经过时，鞋带松了。她停下来弯腰系鞋带，同伴站在旁边等她。陆允川把视线挪回单词书，下一秒又忍不住看过去，像做错题的人偷看答案。

广播里放着晚自习前的通知，声音有点失真。叶澄抱着稿子从主席台下来，经过他们时说，背单词可以，别把操场坐成你们家客厅。程砚立刻举手保证三分钟后撤退。

回教室的路上，人群从操场往楼梯口挤。陆允川把单词书夹在胳膊下，页角一直硌着皮肤，他没有抚平它，像舍不得抚平那一点荒唐的标记。

他把这一天拆成很多小段：哪一句话是真的，哪一个动作是自己添上的，哪一点失落其实来自期待太高。拆到最后，他发现能留下的并不多，可那一点点留下来的东西反而更干净。

寝室灯灭以后，走廊还有脚步声。陆允川闭上眼，脑子里反复出现白天的画面，出现到后来也疲惫了，只剩一道模糊的光。人总会在疲惫里比在激动里诚实一点。

事情过去以后，操场西侧看台又恢复原样。单词书还在，塑胶跑道也还在，只有陆允川看它们的眼神变了一点，像有人在同一页纸上多压了一道折痕。

晚自习结束，程砚拍着他的肩说：“别复盘了，再复盘明天早读都开始了。”陆允川把书合上，终于跟着人群往外走。

如果只写疼，这一章会很薄；加上单词书、塑胶跑道、水杯、晚自习铃，它才重新有了重量。那重量不是压人，而是让读者相信他们确实在这里生活过。

陆允川把单词书拿在手里，先把这一页单词背完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把单词背完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她弯腰系好鞋带又跟同伴继续往前走，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

操场西侧看台里最明显的不是心跳，而是晚自习铃从教学楼里散出来。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，围墙外烤肠摊飘来的油烟把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。单词书、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情过去以后，操场西侧看台又恢复原样。人群散开，桌椅归位，只有陆允川自己还把单词书想得太久，像把一页纸反复折到发白。

夜里他把这件事写进日记本，写到一半停住。原本想写“命运”，最后改成了“今天”，因为今天已经足够具体。

这一段把心事放回操场。单词要背，铃声会响，夏知遥会继续走向自己的同伴，陆允川也要跟着人群回到晚自习。

如果只写疼，这章会很薄；加上单词书、晚自习铃从教学楼里散出来和围墙外烤肠摊飘来的油烟，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在操场西侧看台发呆。”陆允川把单词书收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

这一章收在平常处：灯灭了，楼道静下来，陆允川睁着眼，终于承认今天不是命运给他的暗号，只是生活递来的一个片段。可片段也会发光，只要他不把光攥得太疼。
