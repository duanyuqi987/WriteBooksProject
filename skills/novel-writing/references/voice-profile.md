# 声纹系统

为每个角色和叙事者建立可复用的声纹档案。深度融合 brand-voice 的 voice-profile 方法论。

## 声纹档案模板

```
声纹档案：<角色名/叙事者>
=====================

角色身份：[一句话]
叙事功能：[这个声音在故事中的作用]

## 节奏
- 句长偏好：[短句为主(10-20字) / 中等(15-25字) / 长短交替]
- 段落节奏：[密集紧凑 / 舒展从容 / 张弛交替]
- 标点习惯：[多用句号 / 偶用省略号 / 不用感叹号]

## 用词范围
- 词汇量级：[狭窄(核心300词) / 中等 / 宽广]
- 特色词汇：[角色特有的词语、口头禅、称呼习惯]
- 禁用词汇：[这个角色绝对不会说的词]
- 方言/语气词：[哪些语气词是ta的标志？]

## 句式特征
- 偏爱句式：[陈述 / 疑问 / 反问 / 祈使]
- 复杂句频率：[从不 / 偶尔 / 经常]
- 省略习惯：[经常省略主语 / 对话省略引导语 / 完整句]

## 对话特征
- 发言长度：[短(1-5字) / 中(10-30字) / 长(50+字)]
- 打断习惯：[经常打断别人 / 等人说完 / 不参与]
- 沉默方式：[用动作代替回答 / 转移话题 / 直接沉默]
- 情绪外露：[藏不住情绪 / 喜怒不形于色 / 用行动表达]

## 标志性表达
- 开场方式：[ta最常说的第一句话是什么风格？]
- 收尾方式：[ta如何结束一段对话？]
- 冲突反应：[面对挑衅/质疑，ta的典型反应是什么？]
- 温暖时刻：[ta如何表达关心？（可能很别扭）]

## 禁止事项
- [这个角色绝不做的事 1]
- [这个角色绝不会说的话 2]
```

## 叙事者声纹

叙事者（第三人称有限/全知/第一人称）也应该有声纹。

### 叙事者声纹额外维度

```
## 叙事距离
- [贴得很近(能听到角色内心) / 保持距离(只看行为和对话) / 飘忽不定]
- 何时拉近：[关键时刻/情感时刻]
- 何时拉远：[过渡/背景交代/群体场景]

## 知识边界
- 叙事者知道什么：[有限（跟随某个角色）/ 全知（知道所有人的想法）/ 事后回忆]
- 叙事者不知道什么：[故意留白的内容]

## 评论倾向
- 是否评论角色行为：[从不(冷白描) / 偶尔(春秋笔法) / 经常(说书人风格)]
- 评论方式：[含而不露 / 直接评判 / 反讽]

## 时间感
- 叙事时间：[顺叙 / 偶有倒叙 / 多时间线]
- 时间压缩：[详细描写 / 概括带过 / 跳跃]
```

## 声纹之间的对比

好的小说中，不同角色的声纹应该有可辨识的差异。在人物设置阶段做声纹对比：

| 维度 | 主角 | 配角A | 配角B | 反派 |
|------|------|-------|-------|------|
| 句长 | 短句 | 中长句 | 短句 | 长短交替 |
| 口头禅 | "算了" | "你知道吗" | （无） | "你觉得呢" |
| 情绪外露 | 藏不住 | 完全内敛 | 时隐时现 | 表演性外露 |
| 发言长度 | 短 | 长 | 中 | 长(说教) |
| 冲突反应 | 沉默 | 讲道理 | 退缩 | 攻击 |

如果两个角色的声纹过于相似，考虑合并角色或重新设计差异化声纹。

## 声纹测试

写完一章后，做声纹测试：

1. 遮住对话的说话人标注
2. 你能仅从对话内容判断这是谁在说话吗？
3. 如果判断不出，声纹差异化不够

## 声纹的演变

角色在故事中成长，声纹也应随之变化：

- 第一卷：角色用［初始声纹特征］
- 经历关键事件后：新增［新的表达习惯］
- 故事尾声：声纹反映角色已完成的变化

在角色卡变更记录中追踪声纹演变。
