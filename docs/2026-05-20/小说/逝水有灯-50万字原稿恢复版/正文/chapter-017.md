# 第017章 前排和后排

所属部卷：第一部：初秋三楼
时间：高一春天
地点：物理课后的教室

## 诗歌插页 P017：原创仿唐绝句

青山微明照旧尘，
少年心事隔江春。
回看不见当时路，
一树风声认故人。

事情真正发生时，反而没有他预想得那么郑重。老师让同学互换作业批改，夏知遥的本子从前排传到后排，却没有传到陆允川手里。周围还有人说笑、推门、找座位，生活没有给这件事单独停表。

有些日子只有过完很久才显出轮廓。当时只是上课、下课、等人、错过、回宿舍，后来才发现这些动作排在一起，已经是一条完整的河。所以这一章不只写一件事，也写那件事在岁月里怎样慢慢改变形状。那些形状有时锋利，有时柔软，最后都被他放进更宽的生活里。

湖边的柳枝贴着水面，偶尔被风掀起，露出一小块正在晃动的天。陆允川站在那样的声音里，忽然觉得自己并不是故事中心，只是众多走过校园的人之一。这个发现当时没有让他轻松，反而让他更想抓住一点能够证明自己的东西。

老师让同学互换作业批改，夏知遥的本子从前排传到后排，却没有传到陆允川手里。他后来反复想起这一段，不是因为它多么宏大，而是因为红笔在纸上划出的细线。具体的物件比誓言诚实，它不替人辩解，也不替人作证，只安静地留在光里。

朋友把饮料瓶推到他手边，没有劝，只用瓶底在桌上轻轻磕了一下。他想说的话有很多，可真正到了嘴边，只剩下几句平常的话。年轻时他总觉得平常太轻，装不下喜欢；后来才知道，太重的喜欢常常先把自己压弯。

宿舍里有人洗头，有人背单词，有人对着游戏输赢骂得很响。陆允川坐在这些声音中间，忽然觉得自己的难过没有想象中那么特殊，却也没有因此减轻多少。这些细枝末节在当时没有被他认真看见。他忙着判断自己有没有被在意，忙着从别人的一句话里找温度，反而忽略了日子本身怎样一格一格向前挪。

“真要写诗就好好写，别拿诗当未接电话的替身。”程砚这样说时，语气并不郑重，甚至带着一点不耐烦。陆允川当时听不进去，只觉得朋友站得太远。后来他才知道，太近的人往往只能看见火，站远一点的人才看得见整间屋子正在冒烟。

关系在这里往前挪了一点，也往旁边偏了一点。最近的距离仍隔着几排课桌，他第一次体会到无能为力的具体形状。陆允川那时没有能力准确命名这种变化，只能把它写进纸页，写得越满，越像害怕空白。

叶澄没有急着站队，她只是把问题递回给陆允川，让他自己看见手上的绳结。程砚说你连作业本都碰不到，还谈什么未来。朋友的声音并不总是温柔，有时甚至粗糙，可它们让他看见世界并没有随着一段关系变窄。

他后来很珍惜这种笨拙的具体。比如一张写错座号的考场表，一支漏墨的笔，一条总是发不出去的消息。它们不解释爱情，却能解释一个人怎样在爱情之外继续活着。这些周边生活把爱情从高处拉回地面。它提醒陆允川，心动不是唯一的课程，失落也不是唯一的作业；人总要在很多不浪漫的事情里，慢慢学会怎样对待一件浪漫的事。

她的温柔是真的，犹豫也是真的。年轻时的陆允川只肯相信前半句，于是后半句就慢慢变成了伤。夏知遥认真改题，只把那天当成普通课堂。这部分事实后来才被陆允川补进小说。补进去以后，故事没有变得轻松，却变得诚实了一些。

楼道里有人提着热水壶经过，壶盖碰着壶身，声音细得像一句没说完的话。他把这一幕和诗歌放在同一页，并不是为了显得文雅。诗对那时的他来说，是一块可以暂时放下重物的石头；重物还在，只是手终于能松开片刻。

原创仿唐绝句写完以后，他在下面空了一行。那一行空白比诗句更安静，像把解释留给未来，而不是逼当时的任何人马上回答。

四句写完，余下的话只能留在纸外。这样的限制对年轻时的他反而有用，因为它让他第一次知道，不是所有情绪都必须说到尽头。因此这些插页不是装饰，也不是为了让青春显得更漂亮。它们更像一排小灯，照着他从自我感动走向自我审查的路；灯光不强，但足够让人看清脚下有水。

有些路走错了，也不能只骂路。人要承认，是自己曾经太想从一个方向走到所有答案里。他越长大，越不愿把这段青春说成谁辜负谁。辜负当然存在，可它不是全部；全部里还有胆怯、误会、虚荣、善意和没学会好好告别的年纪。

雨后的操场有一种塑胶和泥土混在一起的味道。鞋底踩上去会带起一点水，他低头看着那些短暂的脚印，心里想，原来痕迹也会很快干掉。这一类日常在小说里反复出现，是因为陆允川后来相信，真正改变一个人的不是某个爆炸瞬间，而是爆炸之后仍要买饭、交作业、赶公交、给手机充电的无数普通时刻。

修订时他给她多留了一段视角。不是为了洗白谁，而是为了提醒自己，任何被喜欢的人都不该被压缩成喜欢者的考题。他没有因此变得完全宽容，也没有把所有疼痛都解释成成长礼物。他只是终于愿意承认，自己和别人都曾经笨拙，而笨拙不必永远被写成罪。

他没有立刻解释，只低头把鞋带重新系了一遍，明明鞋带并没有松。这个动作很小，小到放进当年的日记里可能会被嫌不够痛。可真正能留下来的，常常正是这些不够痛的小动作，它们让人相信那一年确实活过。

在这一章的末尾，陆允川没有得到答案。窗外有人打球，篮球砸在地面的声音一下一下传来。他只是比上一章多看见了一点自己，多承认了一点别人，多放松了一点攥紧的手。

真正的成长不是突然不疼，而是疼的时候仍能看见别人也有边界，也有疲惫，也有不属于他的去处。远处有人喊了一声名字，不是喊他，他却还是停了一下。故事继续往前走，像水流过桥洞，没有替任何人停下，却把所有经过都带成了更低的声音。
