# 第091章 夏天的聚餐之前

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P091：原创现代诗

桌布上的油点亮了一下，
又被风扇吹暗。
有人把未来说得很远，
有人只顾把筷子拆开。
夏天不替谁作证，
它只把一杯温水放在桌角。

升学宴前一天，陆允川被母亲从床上喊起来。院子里晒着两件白衬衫，一件是他明天要穿的，领口有一点黄，母亲端着搪瓷盆搓了很久。

“小饭馆订好了，你爸说别太贵。”母亲说。

他坐在门槛上回短信。程砚问他去不去同学那桌，后面跟了一个“别装”。陆允川看着屏幕，想起夏知遥前一天在群里回了一句“大家顺利”，短短四个字，被许多人的祝福顶到上面，很快找不到了。

他没有去翻聊天记录，只把手机扣在膝盖上。母亲让他试鞋，他站起来，鞋后跟磨脚，走了两步就皱眉。母亲说新鞋都这样，忍一忍。

小饭馆下午还没开席，老板娘在门口洗青菜，塑料筐里积着水。墙上贴着褪色菜单，红烧鱼后面改过两次价。陆允川提前过去帮父亲搬饮料，一箱雪碧压在手心，凉得发疼。

饭局还没开始，他已经觉得累。不是因为要见谁，而是因为所有人都准备在明天用热闹替他确认一次未来。他忽然明白，自己对夏知遥的那点惦记，不能再藏在高考、通知书和祝福后面，让这些东西替它说话。
