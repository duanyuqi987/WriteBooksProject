# 第037章 成绩单下滑

所属部卷：第一部：初秋三楼
时间：高二期中后
地点：班级后墙成绩栏

## 诗歌插页 P037：原创仿唐绝句

红榜无声照晚灯，
少年名字忽低层。
抬头不敢看前列，
一纸排名胜冷风。

排名表贴出来时，教室里先安静了一瞬。

然后人群涌到后墙。有人笑，有人叹气，有人挤进去看一眼又立刻出来。陆允川站在外围，迟迟没有动。程砚比他快，挤进去扫了一圈，回来后表情收住了。

“你自己看。”程砚说。

陆允川走过去。红纸上密密麻麻都是名字和分数，他先找夏知遥。她的位置很稳定，和上次差不多。然后他找自己，找了两遍，才在更靠下的地方看见。

掉了十二名。

这四个字没有写在纸上，却在他脑子里响得很重。旁边有人说这次物理太难，有人说排名没什么。陆允川听不进去。他盯着自己的名字，觉得那三个字忽然变得陌生。

邵老师进教室后，让大家回座位。她没有当众批评谁，只说一次考试说明不了全部，但能说明最近的状态。说到“状态”时，陆允川低下头。

课间，程砚把一张草稿纸推过来，上面写着：十二名，不是世界末日。

陆允川没笑。

程砚又写：但你要是继续这样，下一次就是地理位置重新规划。

陆允川把纸折起来，放进书里。他知道程砚是在救场。可羞愧这东西很难被一句玩笑接住。它像粉笔灰，落在桌面上，一擦就散，却总觉得手上还有。

夏知遥从前排经过，去讲台交卷。她没有看成绩栏，也没有看他。陆允川却忽然不敢抬头。他第一次意识到，喜欢并不会让自己变好。至少不会自动变好。有时候它还会让人用漂亮理由逃开本来该做的事。

那天晚自习，他把硬皮本收进书包最底层，拿出物理错题本。第一道题看了十五分钟，仍然不会。他没有再写诗，只在题号旁边写：从这里开始补。
