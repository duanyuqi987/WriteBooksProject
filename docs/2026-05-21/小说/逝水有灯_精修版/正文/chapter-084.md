# 第084章 迟到的表白旁观

所属部卷：第二部：六月以后
时间：六月末
地点：河边护栏旁

## 诗歌插页 P084：原创现代短诗

那时我们都以为，
结束会有清楚的门。
后来才知道，
门只是普通下午的一阵风。
有人走向远处，
有人学着把远处还给远处。

夏知遥拿着志愿确认表走出校门的时候，林曼在校门口等她。

"你碰到陆允川了？"林曼问。

"你怎么知道？"

"我从机房里出来的时候看到你们在走廊里说话。"

夏知遥把确认表折了两折塞进书包侧袋。"说了几句。他问我报了什么学校，我说建筑。他说他是大类招生。然后他说祝我大学顺利。我说你也是。"

"就这些？"

"就这些。"

林曼跟她并肩走在人行道上。两边的香樟树遮住了一部分太阳，树影落在她们脚边，斑斑驳驳。

"他没说别的？"

"没有。"

"你失望吗？"

夏知遥看了林曼一眼。"为什么失望？"

"因为他什么都没说。三年了，一句话都没说。"

"你以为他不知道？"夏知遥把书包带子往肩上拉了拉，"他早就知道该怎么做了。只是需要时间。"

"什么意思？"

"他在走廊里叫了我的名字。就叫了一声'夏知遥'。然后隔了两秒，说'祝你大学顺利'。"夏知遥踢开脚边一颗小石子，"那两秒里他想说什么，我知道。他也知道我知道。但他最后选择说了一句'祝你大学顺利'。这就是答案。"

"谁的答案？"

"我们两个人的。"

林曼消化了一下这句话。然后说："你这么说话太绕了。"

"不绕。他听懂了。"

"你怎么知道他听懂了？"

夏知遥停下脚步，转过身来看着林曼。阳光从她背后照过来，把她的脸罩在阴影里看不清表情，但声音很清楚。"因为他不是一个需要被拒绝的人。他是一个需要被确认的人。"

那天晚上，程砚回到家以后，在班群里发了一张河边的照片——夕阳下的河面，蓝色的护栏，远处一个收竿的老大爷。配文："今天河边的风很舒服，瓜子很咸。"

叶澄回复："你是在炫耀你有瓜子吃吗。"

周长海："谁的瓜子？"

程砚："我买的。分了半包给陆允川。"

周长海："那他人呢？"

陆允川隔了一会儿回了一条："在。肉吃多了，躺着。"

程砚："红烧肉？"

陆允川："嗯。我妈做的。"

林曼在群里看了一会儿，然后切换到夏知遥的私聊窗口。

林曼：我好像有点懂了。

夏知遥：懂什么？

林曼：程砚发的那张照片。河边的护栏。他跟陆允川在那里坐了很久吧。

夏知遥：应该是。

林曼：你怎么不问问？

夏知遥：问什么？问他们聊了什么？不需要。程砚发这张照片就是在告诉我。他不光是在跟陆允川说话。

林曼盯着屏幕上这行字，愣了好几秒。然后她打了一行字：你们这些人真累。话都不直接说，还要靠河边的照片来传消息。

夏知遥：不累。直接说到处都是刺。绕着说，大家都保住了体面。

林曼想了很久，回复了四个字：那我也是。

她这句话不只是回答夏知遥。也是在回答她自己很久以前在笔记本上写过的那个"陆"字——她对他那一点朦胧的好感，从来没有说出口过。不是因为害怕，是因为她知道那个人从来不在她的方向上。

她把手机放到一边，翻开了下午从学校拿回来的志愿填报指南。省城师范大学，第三页。专业：汉语言文学。她把那一页折了一个角。
