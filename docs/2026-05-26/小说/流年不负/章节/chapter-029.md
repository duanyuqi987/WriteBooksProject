# 第029章 借笔

时间：2011年6月下旬

地点：教室

卷次：卷二：窗边人影

自习课。下午最后一节——离放学还有四十分钟。教室里只有翻书和笔在纸上沙沙的声音。吊扇在头顶转——转得很慢——扇叶把灯管的影子搅成一圈一圈的——在课桌上一会儿亮一会儿暗——暗暗淡淡。六月的下午很闷——气压低——人趴在桌上会觉得皮肤粘在木面上——抬起来的时候有一层微汗。窗外——蝉——突然叫了一声——从银杏树上传过来——尖锐——不过一秒——又短了——像它叫了自己也忘了为什么要叫。

顾临风在做数学题——立体几何——直线与平面的夹角——空间向量法——设平面法向量n——用方向向量a——sinθ=|a·n|。笔——是黑色中性笔——笔杆磨得发亮——手指捏的地方——那层外包的塑料磨掉了一些——露出下面浅灰的底色。

第二排——一声叹气。不是平时那种叹——是——在喉咙里——把气压着——只放了一小部分——像气话但又不算——只是——没写出来——然后叹了。苏晚晴低头——看手上的笔袋——浅蓝色的——抽绳笔袋——她拉了一下——拉开——低头——在里面翻——拿出来一支——拔开——划——不出水——甩了甩——再划——还是不出。放回去。再翻——手指在笔袋里拨——没有别的了——今天出门急——只带了两支——都写干了？她把笔袋拉上——拉绳——收紧——搁在桌角。

然后她回头。

"谁有多余的笔——"

声音不大——不算问整个班——只是——往后面扫了几排——三四排——能听到的人。她目光扫过——从左到右——落在他所在的第三排——扫到他——停了一下——不太长——大概是等待响应——只要有人回应——她就接。

顾临风的笔袋里有三支笔。一支黑的——就是手里这支。一支蓝色圆珠笔。一支红色圆珠笔——芯是晨光的——红色——墨还剩半管。他几乎没有想——右手伸进笔袋——抽出那支黑色中性笔——不是正在写的那根——是从笔袋内侧抽出来的另一支——新笔——刚用两天——笔芯几乎全满——笔壳干净——笔帽上的标签还没撕——"0.5——黑"——白色的那圈纸。他递过去——笔头朝自己。手——从第三排右侧伸出去——过边道——悬在半空。她看见了——身体——往右转了一下——伸手——手指——轻巧——够到笔端——接过去。

"谢谢。"

声音落在空气中。从她那边——到他——大概不到两秒。他摇了摇头——嘴角——没有——只是嘴轻轻抿了一下——然后收回手。她把笔转了过来——拔开笔帽——笔帽吸的那点空气——"啵"——然后落——开始写。马尾——垂在椅背上——还是那条黑色发绳——上次的深蓝色换了——大概新买了一根——黑的比蓝的粗一点。她写的姿势——右肩微沉——笔在纸上——一字一字走——他的笔在她手里——笔尖——触在纸面上——从上面——从后边——隔了——五排——过道——他看得出那是一支新笔——很熟悉的黑——墨——目前处于湿润的轨道上——从纸面的纤维往外渗——干干净净——但那个词——应该是——公式——一个分母——下面——正在写——'r-'——然后接着往下。

手放在桌面上——没动。然后收回——放下——指腹在桌面上轻轻敲了一下——食指——先敲——指腹——然后是第二个指节——慢——然后收了——仿佛只是换气——不是有意义的信号。他看向自己的文具——从桌洞里拿出那支红芯圆珠笔——拔开——在草稿纸上试——断墨——划了几下——很粗的干划——发吱——不出水——甩手——又划——终于出一点点红——像是粉红——稀释——像血——水——混过的细线——用手指压了压笔芯——再划——稍微好一点——勉强能写——就是写着写着又断。

赵一鸣从旁边凑过来——用小声模式的嘴——没停——"你那支黑笔呢——"——"借人了——"他答——声偏——反正没有解释。赵一鸣往前面看了一眼——什么都没有——然后说"哦"——然后抽出自己的笔——从笔袋拔出一支蓝的——"用我的——""你用什么——""我还有一支——"赵一鸣从上衣口袋摸出另一支——杆上印了一个"福"字——大概是过年分的——"这个是去年过年从超市拿的——能用——"顾临风接过来——蓝笔——有点粗——笔杆磨得——可能赵一鸣用了好一阵——但出水没问题。

自习课还剩下四十分钟。他用那支断墨的红笔做了一道数学大题——辅助线——划了错——擦——纸擦破了两个小洞——纸面——两个小圆——白白的纤维毛——露出来。他继续用那支——写——擦——划——洞越扯越大——拿起赵一鸣的蓝笔——继续做完——答案——X=——算是做好了。赵一鸣在旁边偷瞄了一眼——"做出来了？"——"嗯。"

下课铃响。椅子——座椅上的重量离开——风吹——长下课的声音——走廊——脚步声——说话声——有人喊——"食堂待会有好心人帮我带饭——"

苏晚晴站起来——转身——手里握着那支黑色中性笔——朝他的座位走过来。她的步伐——不紧——不慢——从前排——过过道——经过空椅子——到她与他的桌之间——大概——两步——然后——一步——然后——站住——左手——手指——轻轻按在他的桌沿——另一只手——把那支笔递过来——放在桌上。笔是横放的——笔帽朝左——笔尖朝右——笔中段在桌子的纹路上——和桌板纹——对齐——偏了点——但大概——整齐。他抬头——看到她——脸——很近——近到他看到——她眉毛尾部的小杂毛——然后她——额头——她说了两个字——"还给你——谢谢——"

"不客气——"他——说——声音——平稳——自然——他没察觉到自己的声音——比平时——略低——也许。她转身——马尾——往右肩甩——肩——往左——走了。到走廊——被另一个女生截住——好像问食堂的事——"——今天周三——有没有——"

顾临风拿起那支笔——笔帽上——有一个淡淡的指印——是拇指的——不太清楚——像——像玻璃上冬天碰过留下的那种轻微印——在笔帽上方——圆——略椭圆——细看——有汗——干了之后的——微痕。他把笔帽拧开——又拧上——没有擦——然后把这根笔——和他的——铅笔——旧的红芯——赵一鸣那根——放回笔袋——但这根黑的——单独放了个位置——铅笔盒里层——最深处——贴着小虎队的贴纸——旁边——那个位置。自从那天——他一直用这支笔——比用其他中的任何笔都更留神用——拧开——写完——关上——不咬笔帽——不用来干任何别的；写到后来——大概——快到六月底——笔芯开始——出来的时候不是顺畅的——而是间发的——出水断断续续的——像——像早上的雾——淡——他写——一写——断——搓笔——再写——又断。赵一鸣路过:"你那把伞不换——笔也不换——你这个人——笔芯都干了——"

"还没用完。"

赵一鸣看了他一眼——看了他两秒——看了看那笔——然后转头——没再说——伸手在他肩膀上放了放——轻轻——像拍走灰尘。六月的最后——在那支笔实在不出来的那天——他在日记本上写——"她今天借了我的笔——黑色的——还回来的时候——上面有个指印——我一直没有擦——我把那支笔用到最后一滴墨——写出来的字越来越淡——像冬天的晨雾——浅——看不清——但那个指印——还在——笔帽上——她的——我认得出。"

翻一页——"她借笔的时候——回头看着我说的——'谁有多余的笔'——然后看着我——虽然只是在问我这里有没有笔——但她就是看着我了——那个——眼睛——深色——和她声音——一句话里——有那么一点点沙——我接得很快——几乎是——她话还没结束——笔已经拿出去了。快得像怕她反悔——但她不会——她只是——需要一支笔——而已。"

写完——合上本子——窗外——有人又——打篮球了——嘭——嘭嘭——弹在水泥地——节奏——空——然后远。他把那支干掉的笔——放在铅笔盒最里面——贴着角落的那支已经没笔芯的圆珠笔芯旁边——推了推——让它刚好——靠着——不晃。铁皮铅笔盒——小虎队的贴纸——吴奇隆——下嘴唇——原来看上去像笑——现在从左边看——像在问——"你——知道。"

晚自习——日光灯——嗡嗡——她做物理题——手里的笔换了一支——粉色杆——不是他的——他的干了——她的新笔——可能又去小卖部买的——可能她觉得自己总丢笔——多准备了几支——谁知道。

顾临风用的——是赵一鸣那支——今天下课后他刚买一支新的——和赵一鸣原来的样——因为他的那根拿走了——就放了——晚上到教室——桌上——原来那支——赵一鸣没拿走——旁边还有个捏瘪的方便面袋。他把物理练习册打开——开始——力学——牛顿第二定律——F=ma。做不出来的时候——他就看看那支黑笔——放在铅笔盒最里面的——那个——笔帽上——有一个——干了很久的——在她手里——曾经——湿的——指印。然后低头——继续做题。
