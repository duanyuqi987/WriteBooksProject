# 第044章 诗抄本旁观

所属部卷：第一部：初秋三楼
时间：高二冬天
地点：男生宿舍下铺

## 诗歌插页 P044：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
诗抄本里被墨水洇开的句号还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，诗抄本会显得没有那么沉重。教室、走廊、操场里站着许多人，每个人都有自己的迟到、作业、饭卡和回家路。

叶澄后来记得的不是陆允川的沉默，而是他那天总在看诗抄本里被墨水洇开的句号。她觉得那样的眼神很熟，像很多人刚开始喜欢谁时都会有的样子：认真，又有点不讲道理。

叶澄后来借过这本诗抄本，看见边角密密麻麻的小字，只说了一句，诗可以救人，也可以把人困住，看你怎么用。

程砚看得更直接。程砚说完就去洗漱，牙刷杯碰在脸盆架上叮当响。陆允川盯着那句“别欠你的”，心里有点不舒服，却没有反驳。他不喜欢把话说得太柔，因为太柔的话陆允川听了会继续躲。

“你这本子是给语文老师看的，还是给自己壮胆的？”程砚说。“抄诗怎么了？”陆允川说。“抄诗没事，别把人家抄成欠你的。”程砚说。

夏知遥在这一章里也不该只是一个被看的背影。夏知遥并不知道自己被写进那么多空白。那天她在教室里背政治提纲，手指压着书页，一条一条往下划，神情认真得近乎朴素。她有时会因为题做不出皱眉，有时会在朋友讲笑话时笑出声，有时也会因为家里电话而沉默很久。

她没有义务知道陆允川每一次停顿。就像陆允川也不知道她书包里那张没交的申请表，不知道她冬天手指容易冻红，不知道她回宿舍前常在水房多站一会儿。

冬天的宿舍很早就黑，窗缝漏风，铁床架一碰就凉。陆允川坐在下铺，把被子搭在膝盖上，台灯罩出一块小小的黄光。

在旁观者眼里，陆允川开始抄诗，唐诗、宋词、现代诗混在一本本子里，所有空白处都隐约指向夏知遥。这件事只是一天里很短的一折。可短并不等于无意义，它只是提醒陆允川：意义不能只从自己这边生长。

窗外有人倒热水，水流进暖瓶时发出空空的回声。宿管阿姨在楼道里喊别说话了，声音从门缝钻进来，压住一屋子没长大的秘密。

叶澄晚自习后对他说，喜欢谁不是错，但别让别人替你承担你没说出口的戏。她说完就去交作业，留下陆允川站在门口，手里捏着一本卷了角的练习册。

他抄唐诗时喜欢把字写得瘦一点，抄宋词时又忍不住拖长最后一笔。程砚说他这不是练字，是给心事换衣服。陆允川没理他，只把台灯往里挪，免得光照到别人床上。

有一页现代诗写到一半，宿舍突然熄灯。他在黑暗里停住笔，墨水在纸上多洇了一点。上铺有人问谁还有手电，另一边有人说别写了，明天还要早操。

夏知遥的名字从来没有直接出现在诗抄本里，却又到处都是。一个“她”、一处“窗边”、一行“冬天的课桌”，都像绕路写出的地址。多年后陆允川再看，才觉得这种绕路既真诚又危险。

叶澄借本子那天没有翻太久，只停在一首仿宋小令前，说这句还可以，但你别把所有空白都留给同一个人。陆允川当时没答，心里却记了很久。

旁观不是冷漠，而是把故事从一个人的胸口挪开一点。挪开以后，夏知遥有了自己的呼吸，程砚有了自己的烦躁，叶澄也有了不愿明说的担心。

陆允川不喜欢被旁观，因为旁观会让他的深情显得没有那么独一无二。但正是这种不独一无二，让后来的人生有了出路：大家都年轻过，也都误会过，不必把一次笨拙写成终身审判。

从叶澄的角度看，陆允川的心事并不神秘，只是太年轻，所以总以为别人都该看懂。他越沉默，周围人越不知道该不该靠近。

夏知遥也许注意到了一点异常，也许没有。她那天要处理的不是谁的暗恋，而是自己的作业、座位、朋友和回宿舍前必须打完的一通电话。

陆允川把钢笔拿在手里，先把这一页诗抄完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把这页写完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她在教室前排背政治提纲，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

男生宿舍下铺里最明显的不是心跳，而是暖瓶水灌满时的空响。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，泡面调料包被撕开的味道把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。钢笔、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

从旁边看，陆允川的心事并不神秘。它只是少年人常有的笨拙：怕被看见，又怕完全不被看见，于是每个动作都带着一点别扭。

夏知遥也许注意到了一点，也许没有。无论哪一种，她都不该被写成答案。她有自己的抄诗和改句子，也有不想向任何人解释的沉默。

这一段让诗抄本不只装暗恋，也装宿舍的冷、台灯的光和同伴的提醒。诗可以安放心事，但不能替他向别人索要答案。

如果只写疼，这章会很薄；加上钢笔、暖瓶水灌满时的空响和泡面调料包被撕开的味道，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在男生宿舍下铺发呆。”陆允川把钢笔收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

这一章的火被旁人的目光压低了。压低以后，才看得见灰里的温度，也看得见每个人不被爱情叙事吞掉的完整生活。
