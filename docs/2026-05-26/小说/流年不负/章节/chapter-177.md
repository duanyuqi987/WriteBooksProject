# 第177章 兑现
时间：2019年5月1日傍晚
地点：青阳县月湖
卷次：卷九：执手之约

2013年6月8号晚上。

高考刚结束。英语考试结束铃响起之后大概四个小时。青阳一中的一群高三毕业生在月湖边聚会。有人带了啤酒，有人在唱歌，有人在湖边的长椅上打电话给家里——"考完了。还行。不知道。"

苏晚晴坐在湖边最偏远的一条长椅上。不是聚会的那片区域——是沿着湖边走大概三百米，一棵大柳树下面。周围很暗。只有远处水塔顶上的红灯每隔两秒闪一下。

她在那里坐了很久。顾临风找到她的时候，她正把一颗小石子往湖里扔。石子飞了三下——在水面上弹跳了两次，第三次沉了下去。

"你怎么来了。"

"赵一鸣说你在湖边。"

他站在长椅旁边。她没叫他坐。他自己坐下来了。离她有半个椅面的距离。湖风吹过来的时候，她头发上沾的一小片柳絮飞过来——落在他的袖口上。他没拿掉。

那天晚上他们说了很多话。从高考物理最后一道题说到小时候的事。她说她小学一年级走丢过一次——一个人走了三公里路，后来在菜市场被邻居大妈捡到。他说他小时候在镇上河沟里摸鱼——摸到过一条两斤重的鲫鱼，回家被他妈红烧了，上面放的葱段比鱼还多。

后来她说了一句话。

"不管去了哪里。以后都要回来看这个湖。"

不是对着他说的。不是对着湖说的。是自言自语——声音很轻，轻得跟湖面上的雾气一样。但他听到了。

他点了点头。

没说话。只是点头。

她不知道他点了头——因为她没有在看他。她在看着水塔上的红灯。红灯闪了一下又灭了。闪了一下又灭了。

但他在心里说了"好"。在他高三写的那本暗恋日记里——最后一页，最底下那一行——他用很细很细的铅笔写了四个字:

"我也回来。"

2019年5月1号傍晚。同一把长椅。同一棵柳树。同一个湖。同一座水塔。同一盏两秒一灭的红灯。

他穿着黑色西装。她穿着白色婚纱。

他们的膝盖上沾着同样的长椅铁锈——她的左边膝盖上一小片褐红，他的右边膝盖上也有一片。两个锈渍隔着大概二十厘米——就是他当年坐下来时跟她隔的距离。但现在隔着的不是距离——是已经填满了的空间。

"你回来了。"她说。

"你也回来了。"

他没有点头。这次不用点头了。

她低下头。从婚纱口袋里掏出一样东西。一个本子。

旧笔记本。蓝色硬壳封面，边角磨得发白。封面上用黑色钢笔写着"流年"两个字。字迹不是他的——笔锋偏窄，起笔轻收笔重。是她的。

翻开。里面不是他的诗。是她写的。从2010年10月开始——

"十月。周三。物理又没考好。看到他在本子最后一页写了一道题的解法。不是给我写的。但我知道那是给我的。"

"十一月。跑操的时候他跑在我后面。他鞋带没系紧——跑出去十几米鞋后跟踩到了鞋带。我想过去帮他系。但没动。"

"十二月。下雪了。他在窗外看着我的手——我在用手接雪花。我其实知道他在看。手故意在窗外多停了两秒。不多——就两秒。"

"2011年1月。他写了一句诗。'那年九月风初凉'。我经过他座位的时候看到了。他赶紧把本子翻到下一页。但我已经记住了。"

"2012年3月。运动会。他在打篮球。我在观众席第四排。每次他回头往楼上看——我就低头假装看书。"

"2013年6月8号。他说了。他说了。他说他知道——"

字迹中断了。那一行最后几个字是歪歪扭扭的——可能是写得很快，可能是手在抖。

然后后面空了很多页。大概空了三年——从2013年夏天到2016年的冬天。

她继续往后翻。翻到中间某页。

"2016年12月。楚州。在街上看到一个人骑车过去了。骑得太快没看清脸。但背影像他。我知道我在骗自己——那是楚州，不是青阳。但还是追出去几十米。然后停下了。不是他。"

"2018年3月。妈说给我安排了相亲。青阳的，在楚州当老师。我本来要推掉的——也不知道为什么没推。"

"2018年3月28号。周六。下午三点。我坐在茶餐厅的窗边等他。心里想——那个敲门的人会是谁。任何一个人都行。只要不是陌生人。"

"推门进来的是他。"

"是他。是他。是他——"

三个"是他"。一个比一个用力。字迹穿透了三层纸。纸的背面能摸到钢笔字的凸痕。

顾临风看着这本日记。从她手里接过来的时候，手在抖——不是冷，是每个字都在往他心里压。他从来不知道她也写了日记。他以为只有他在记录。他以为那些目光都是单向的。他以为雪落在她手心的时候，她什么都没察觉。

"写了好久。"她说。声音很轻很轻。

"从2010年开始。跟你的第一首诗同年。你写诗。我写这些。你写在笔记本最后一页。我写在这个本子第一页。"

"画侧脸的那张——"

"在这里。"

她翻到第一页反面。一张折叠的画夹在页缝里——纸已经发黄了，折痕很深。她把那张纸掏出来。展开。

男生的侧脸。画得很粗糙——看得出不是专业的。线条有些生硬，下颌角的比例不对，耳朵的位置偏高了一点。但侧脸的轮廓——从额头到下巴那一条线——像他。

"大二画的。凭记忆画的。画完一直没拿出来——怕画得不像。"

他看着那幅画。手指在纸上轻轻划了一下。碳粉触感很干——放了太多年，碳粉已经不沾手了。

"像吗。"

"像。"

他把日记本合上。放回她手里。她的手指按在封面的"流年"二字上。

2013年6月8号晚上。她坐在这条长椅上，说了"不管去了哪里，以后都要回来看这个湖。"当时他没回答——但他始终记得。

后来的一年、三年、六年——他去了楚州、读了师范、考了编制、租了房子、买了二手车。在每一次经过月湖路的时候都往湖的方向看一眼。在每一个春节回青阳的时候都来湖边坐一会儿。在写每一首诗的时候都想着——什么时候能带她回来。

六年。从一轮高考结束的夏夜到一场婚礼结束的傍晚。从一个刚刚被表白的十八岁女生到穿婚纱的二十六岁女人。从两个站在红绿灯前不敢牵手的人到两个坐在长椅上彼此看着对方日记本的人。

六年不短。

但终归是兑现了。

她靠在椅背上。仰头看着暗下来的天。五月的星星不亮——只有几颗最亮的能在城市边缘的夜空里被看见。

"兑现一个约定用了六年。"

"你还想用多少年。"

"一辈子。够不够。"

他没回答。把她的手握过来。两只戴戒指的手扣在一起——戒指碰到戒指，"叮"的一声。比订婚宴上碰杯的声音更轻。比她在厨房里碗碰碗的声音更脆。但本质上是同一种声音——两个金属环碰到一起，震动了十几圈碎钻和一整圈素铂金的共鸣。

柳树在天色完全暗下来之后变成了一个剪影。湖水的颜色消失了——变成一片深色的平面，映着水塔的红灯。红点在水面上的倒影被拉长成一条红色的线——从岸边直通湖心。像一条用灯光铺成的路。

兑现。不仅仅是回来了。是穿着结婚礼服回来的。是用一整个青春的沉默换来的。

她把婚纱下摆拉好。在长椅上站起来——膝盖又响了一声，跟苏父一模一样。她走到湖边，蹲下。手指在湖水里划了一下——水面很凉，五月的湖水还带着冬天的寒意。指尖碰到水面的瞬间，波纹散开，把水塔的红色倒影搅乱了。红点变成红丝——在水面上缠了几圈，然后慢慢恢复成原来的圆形。

他在身后看着。西装外套已经脱了——只穿白衬衫。衬衫袖口卷到小臂，领带被风吹歪了——他懒得扶。风吹过柳条的时候，几根柳丝从后面扫过他的肩膀。他没有回头。因为他知道那是柳树，不是人。

她把手指从湖水里拿上来。水从指尖滴下去——滴滴答答。她站起来。回到长椅上。

"湖水还是冷的。"

"冬天在湖边求婚的时候更冷。"

"忘了吗——你跪在石子上。膝盖上的泥我拍了半天。"

"没忘。石子还在——可能又多了一颗。"

她笑了一下。嘴角弯的时候右手无名指上的碎钻闪了一下——湖对岸的灯光恰好照过来。

兑现。不是某个具体的动作。是所有这些事都发生过了——泥和石子、诗和戒指、日记和画、眼泪和茶、红灯和柳絮。所有这些事都发生过了，都在同一个湖边发生过了。从十八岁到二十六岁。从"我需要想一想"到"好"。

她现在二十六岁。他二十六岁。六年前他们站在这个湖边——不知道后来会发生什么。不知道会有一场相亲。不知道他会在同一个湖单膝跪下。不知道两本红本本会放在抽屉最里面。不知道今天会在婚礼之后偷偷溜出来——穿着礼服和婚纱，膝盖上沾着长椅的铁锈。

不知道。但回来了。

而月湖还是这样——水不深、湖不大、柳树还是那几棵。只是有一个约定在等了六年之后终于变成了两个人并肩走的十个脚印。从长椅到湖边的十个脚印。来回。
