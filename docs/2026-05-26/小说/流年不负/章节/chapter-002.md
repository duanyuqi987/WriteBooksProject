# 第002章 第一周

时间：2010年9月第一周

地点：青阳县一中教室、食堂

卷次：卷一：初见秋风

开学第一周的早晨总是慌慌张张的。

六点二十，宿舍里的灯一亮，六个人同时开始找袜子。灯管闪了两下才全亮——日光灯，启动慢。光很白，照得水泥地上有了一层灰白。水房挤满了端着牙缸的人，泡沫滴在地上。水龙头只出了三个。另外两个坏了。等水的人和刷牙的人挤在一起。顾临风刷牙的时候对着墙。牙膏是中华牌的，薄荷味。牙膏皮管子从后面往上卷。卷到一半挤不出来了。他用力捏了两下，捏出来一点点。

隔壁水龙头下的男生在洗头。低着头，头发浸在冷水里。他旁边的台面上放了一小袋洗衣粉——白猫牌的。撕开的口子拿夹子夹着。

七点进教室。从宿舍到教室要穿过半个操场。操场边的草地上有露水，踩过去鞋底湿了。鞋是白色的回力鞋，鞋头磨得有点毛。晨读七点十分开始。

教室里已经坐了人。领读的是英语课代表，一个短头发的女生。她站在讲台上翻开英语书，念了一句，底下稀稀拉拉跟着念。前排声音大，后排几乎没人张嘴。课代表的英语发音带着点口音——"book"的尾音发得太重。

顾临风走到倒数第三排。

停了一下。

往左前方看了一眼。

第二排靠窗。

浅蓝色T恤换成了白衬衫。袖口卷了两道。马尾还是马尾。她在翻英语书，手指顺着单词往下滑。指尖在纸面上移过的时候留下了一个浅浅的压痕。嘴唇动着，没出声。嘴唇的形状每个单词都不一样。

他把书包放下。拿出英语书。翻到同一页。她翻到哪一页他是怎么知道的——刚才经过她课桌旁边的过道时瞥了一眼。上面有个蓝色圆圈圈住的单词——"distance"。

英语单词一排一排的。他不知道她在念哪一个。但他知道她在念。声音在空气中传不过来。隔着四排桌子、两个过道、十几个人的背影。

早读课是四十分钟。中间有人打呵欠，长长的，到一半又吞回去了。有人把书竖起来挡着脸，眼睛其实是闭着的——在假读。英语课代表往台下看了几眼，没说什么。下课铃响的时候，英语课代表合上书走了。教室里倒下去一片——有人趴桌上睡，有人伸懒腰打哈欠。

她没趴。

她拿出一本小本子。本子不大，比巴掌长一点。深蓝色封皮。上面写了什么。笔是蓝色的圆珠笔。写几行停下来看一会儿，然后继续写。写完之后看了看，手指在纸面上从左往右滑了一下——像是在检查。然后翻到下一页。从侧脸看过去，她的表情没什么变化。但手指翻页的动作很轻。不急。

接下来的几天，顾临风每天都来得比前一天早一点。

周二七点到的。周三六点五十。周四六点四十五。周五六点四十——那时候教室的灯还没全开。只有靠窗那排的灯亮着，另一边还是暗的。暗处坐着两个在补作业的男生，头埋得很低，互相借着看。

早起这件事对他不难。在石桥镇的时候天不亮就得起来帮他妈推豆腐磨——每天早晨，石磨吱扭吱扭的声音就是闹钟。现在不需要磨豆腐了，反而觉得早上多出来的时间不知道怎么用。

他不是最早的那个。但已经不是倒数第三个进教室的了。

进教室的第一件事——把书包放下，抬头，往左前方看一眼。

在——然后他开始拿书、翻页、低头。整个过程不超过五秒。如果有人这时候盯着他看，不会觉得他在看什么特别的东西。他看第二排靠窗的眼神和他的表情一样平常。

不在——他坐在座位上不会动，但眼睛会往门口看。看进来的人影。浅蓝色或者白色。等那个影子走到第二排靠窗。然后他翻开书。如果她已经来了——他是怎么知道她来了而自己没看到的——可能是刚才在走廊上迎面走过去了没注意，可能是他去水房接水的当口她刚好进来了。他会在心里数：从自己进教室到看到第二排有人，中间隔了多少秒。数不清。

有一次她没来。周四上午。第二排靠窗的位置空了一整个早读课。桌上的书还是昨晚自习课那样摞着。她同桌帮她把卷子写好名字放在桌面上。顾临风翻了五页英语书，五页里背了三个单词。

第三节课做到物理题的时候，听见后门有人轻敲了一下。开门的同学说了声进来。是她。说"报告"，声音有点沙。物理老师的粉笔停在半空。点了个头。她绕到第二排坐下。顾临风的笔停了。停了大概十秒。然后继续写。那道题横线上写了三个字，写错了。用铅笔擦掉。擦了又写。

后来赵一鸣告诉他——苏晚晴那天上午请了半天假，去了一趟医院。没说什么病。赵一鸣说这话的时候在拆一包干脆面，调料粉撒了一桌。"你关心这个干嘛？"

"没关心。"

"那你问个什么呢。"

"随便问的。"

赵一鸣把干脆面揉碎了塞进嘴里，嚼得嘎嘣嘎嘣响。没再问。

中午在食堂吃饭。

食堂在教学楼后面，一个铁皮屋顶的大平房。里面摆了上百张白桌子、蓝塑料凳。桌子边角磕坏了，凳子腿有的弯了。地面是水磨石的，湿的地方反光。打饭窗口四个，排的队歪歪扭扭。三号窗口最近的菜式贴在玻璃上——周一土豆丝、周二炒豆芽、周三红烧茄子、周四可能会换。玻璃上还有上个月没撕干净的旧纸条。

顾临风一般在三号窗口排。菜便宜一点。一块钱一个菜。两个菜两块，饭五毛。土豆丝、西红柿炒蛋、偶尔有红烧肉。打菜的是个胖阿姨，拿大铁勺的手稳得很——舀下去的时候多，抖两下就少了。

周四中午，他端着饭盆找位子的时候，看到了她。

她坐在靠墙的角落里，一个人。面前两个包子，一瓶矿泉水。矿泉水的瓶盖放在包子旁边。她吃包子不用手拿——用筷子夹着吃。

她先拿起左边的包子咬了一口。素馅的——皮薄，能看到里面的菜色。韭菜鸡蛋。吃完了一个。然后拿起右边的。肉馅。猪肉大葱。

先吃素的。再吃肉的。

她吃东西很慢。咬一口，嚼一阵子。不着急。周围有人在扒饭、有人筷子在盘子里搅得哗哗响。有个男生跑过去撞了一下她的椅背。包子晃了一下。她没回头。

顾临风在隔了五六排的地方找到了一个空位。坐下。低头吃自己的饭。

土豆丝有点咸。

他把菜吃完了。饭剩了一半。筷子横放在饭盆上。然后站起来去倒剩饭。倒饭的桶在食堂门口，铝皮的。里面已经堆了半桶。有几只苍蝇在桶沿上爬。

从那以后，他中午吃饭的时候会下意识地往靠墙的角落看一眼。有时候她在，有时候不在。在的时候永远是两个包子、一瓶矿泉水。永远先吃素的。

有一次她的包子掉了一个。素的那个——从筷子间滑下去了，跌到饭盆里。饭盆里有一层汤水——西红柿蛋花的。包子皮泡软了。她愣了一下。筷子停在空中。然后她用筷子把那个泡软的包子夹起来放在盆边。没有吃。继续吃肉的。吃完肉包子，喝了一口矿泉水。然后把那个泡软的包子拿起来。外面的皮已经烂了。她把烂掉的那层皮剥掉，里面还是白的。吃了。

顾临风看着那一整套动作。低头扒了一口饭。饭凉了。

有一次她写字。

是在周五的自习课上。物理老师没来——办公室门口贴了通知，"王老师因病请假，本节自习"。大家自己看书。有几个在后面说话了，声音压得很低，嗡嗡的——像隔着一层玻璃在外头说话。

她在写物理题。头微侧着，左手扶着本子。右手握笔，蓝圆珠笔。本子是那种横格本，一行一行整整齐齐的。

写到某一行的时候，她停了一下。笔尖在纸上晃了晃。然后笔继续往下走。

她的小指翘起来了。

不是那种很夸张的。就是握笔的时候，小指自然离开了纸面，翘出一个很小的弧度——大概离纸面三五毫米。她自己好像没注意到。继续写。小指翘着。左手在右手动的时候不动。右手在左手按住本子的时候一笔一笔往下走。

顾临风的笔不知道什么时候停了。

他看着那个翘起来的小指头。

窗外的光照在她右手边。铅笔盒的影子落在本子上。影子一动不动。只有小指头——那个翘起来的弧度——像是什么东西被轻轻托起来了。

直到旁边的同学翻了一页书，响声让他回过神来。他低头看自己的物理书。那一页讲了什么，他没记住。页脚有一道钢笔划的细线——不知道什么时候划上去的。

周五晚上熄灯后，他躺在下铺。头顶是上铺的床板，木头的，中间有一道裂缝。裂缝的宽度刚好能让他看到上铺床垫底下的粗麻布。刘洋在翻身的时候裂缝就窄了一下。

宿舍里有人说了梦话。不知道是谁。含含糊糊的几个字。像是"不要"。

孙磊的呼噜停了。然后又开始了。

走廊里周老师的脚步声——查夜。步子比平常轻。大概十一点。

他想了一些事。

然后又觉得那些事不值得想。

但还是想了。

外面起了风。窗户没关严，窗帘被吹得一鼓一鼓的。鼓起来的时候月光全进来了。落下去的时候月光又没了。一明一暗。一明一暗。

他翻转了一下身。枕头的这一面凉一些。

然后闭上了眼睛。
