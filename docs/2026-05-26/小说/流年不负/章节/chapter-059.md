# 第059章 最后一天

时间：2013年6月4日

地点：教室

卷次：卷三：此心藏久

六月四日。高考前三天。

这是最后一天在教室上课。

黑板上用白色粉笔写着一个醒目的"3"——数字很小，但写在黑板正中央，旁边的字都被擦了。只有它。三。三天后高考。

上午第一节是老刘的课。班主任。教语文。

他走进来的时候没有拿教案。手里端着一个茶杯——茶叶泡了太久，水是深褐色的。他把杯子放到讲台上。看着台下一张张脸。沉默了大概十秒。教室里的声音一点点收敛了。到完全的安静。

他说。

"三年了。"

停了停。

"你们是我带的第四届。每一届到了这个时候，我都不知道该说什么。说多了你们嫌烦。说少了你们嫌敷衍。所以我不说多了。就说三句。"

"第一句。把东西带齐。准考证。身份证。2B铅笔。黑色签字笔。橡皮擦。尺子。这些东西提前准备好。不要到时候慌。"

"第二句。考场上遇到不会的题不要纠结。先跳过去。做会的。最后再回来。高考比的不是谁会做的题多——是比谁犯的错误少。"

"第三句。"

他停顿了一下。把茶杯端起来喝了一口。

"第三句。高考很重要。但不是人生的全部。六张卷子决定不了你是谁。你是一个什么样的人，是这三年每一天决定的。不是那三天。"

"好了。我说完了。你们自己看书。有事到办公室找我。没事也可以来。"

没有人出声。教室后面的"拼搏改变命运"的横幅歪得更狠了——最后一个图钉也掉了。布耷拉了一角下来。老刘走过去，把图钉重新按上。按了两下。手背上的青筋很明显。

他走出去之后。教室里的安静持续了大概三秒。然后有人开始收拾东西。把抽屉里攒了一年的卷子往外拿。语文卷。数学卷。英语卷。理综卷。模拟卷。真题卷。专项卷。厚厚一沓。有人翻着看——这张是六十八分，那张是九十二分，这一张写了一整面草稿纸的推导过程。翻到最后没耐心了，把卷子一把捏起来扔进垃圾桶。垃圾桶满了。有人把卷子抱到走廊上——堆成一堆。像一垛废纸山。

顾临风没有扔东西。

他把抽屉里的东西一样一样拿出来。高一上学期期中成绩条——三十一名。他看了它很久。高二上学期期中——十五名。高三第一次月考——第九名。高三第三次月考——第六名。每一次的成绩单。折了角。按时间排列。他拿起这些纸。放进了一个牛皮纸信封里。

然后是物理笔记本。那一本他用三个晚上重誊过的——每一道右手定则的题旁边有红色小注。每一道易错题后面有简化解法。第一页的右下角用很小的字写着"给她"。这本笔记本他在枕头下面放了半年多。每天晚上手电筒的光照在上面。他没有给出去。

他把笔记本拿出来。放到桌面上。用手掌压了压。

然后放进了书包。今天他给不给出去。还没决定。

下午。物理老师来上了一节课——最后一节物理课。讲的是考试技巧——选择题做完就涂答题卡，不要等全部做完再涂。大题不要跳步骤——少了步骤扣步骤分。公式写对也能得公式分。物理老师是个瘦高个。戴着眼镜。说话永远不快。下课铃响的时候。他放下粉笔。转过身。对大家鞠了一躬。然后走了。粉笔灰还飘在他的肩膀后面。

化学课——最后一节化学课。老师是女的。年轻。课上到一半忽然就停了。她说"算了不讲了。你们自己看看公式吧"。然后在黑板旁边站了一阵。她靠在墙边。眼镜摘下来擦了擦。大概是粉笔灰进了眼睛。重新戴上后，在讲台上放了一袋大白兔奶糖。"一人一颗。考场上困了就含一颗。""别吃太多——会黏牙。"大家笑了一下。笑声很轻。没有人去抢。一个一个轮流去拿。顾临风也拿了一颗。白色的糖纸。上面印着胖兔子。他放进了口袋里。

下午最后一节是自习。放学铃声响了。四点半。平常四点半不会有特别的感觉——只是一个普通的放学的钟声。每一次放学钟声响起——大家收拾书包。回家或去食堂。明天继续。后天继续。但今天是最后一次在这间教室里听到它。明天开始放假。六月七号直接去考场。

有人开始往外走。有人没动。有人在教室里面面相觑——一种不知道该怎么告别的气氛弥漫开来。没有哭的。没有抱的。没有喊口号的。就是安静的。一种比哭更淡但比开心更深的安静。

苏晚晴开始收拾东西。

她把课桌上的课本一本一本放进了书包。语文。数学。英语。物理——那本她写了擦擦了写的物理书。化学。生物。她所有笔记本——整整齐齐一摞。她把铅笔盒打开——铁皮的，面上印着小花猫。从高一用到现在。盖子有点变形——每次合上都有一点缝隙。她把笔倒出来——黑笔两支。红笔一支。铅笔一支。橡皮擦一块。削笔刀一个。尺子一把。全部装进笔袋。铅笔盒空了。她把它的盖子盖上。摸了一下盒面上的小花猫。放进了书包。最后一件事。她把她桌面上贴的名字贴撕下来。"苏晚晴"三个字。铅笔写的。已经模糊了。以前她打算换一张，但一直没换。撕的时候边角撕破了一点。她把纸片折了一下——很小的一片。放进了铅笔盒。

然后她把椅子推到桌子底下。站在自己桌前。看了这个地方几秒。窗边的位置——她坐了三年。从这个窗户看出去能看到银杏树。能看到操场。能看到天空。高一的时候她坐在这里，窗外的银杏还是绿的。高二的时候她在这里看银杏黄了又落了。高三搬到走廊边——不再看银杏了，但她还是习惯性地看窗外。她往外看了一眼——最后一眼。银杏全绿了。六月的银杏，叶子是深绿色的。厚厚的。一层一层叠在一起。她深吸了一口气。转身往前门走。

顾临风在第四排的座位上坐着。

他没有收拾。他看着她的整个过程。她拉开书包拉链。她把课本一本一本放进去。她撕下名字贴的动作。她看窗外的最后一眼。

她转过身，往门口走。经过他座位旁边的时候——和平时一样——隔了一米。她的马尾从他眼前晃过去。头发梢差一点点碰到他的桌子。她的书包拉链上那个小鱼挂件晃了两下。淡青色的小鱼。釉面上反射了窗外的一点亮光。

她走到教室门口。停了一下。右手搭在门框上。

然后她转过身。

往教室里看了一眼。

不是看某个人。是看整间教室。黑板。课桌。讲台。窗外的银杏。墙壁上的横幅——"拼搏改变命运"——右下角图钉又掉了，布又歪了。她的目光从这些东西上一一扫过去。最后——在第四排的位置——停了不到一秒。

然后她转回身。走了出去。脚步声在走廊上渐渐远去。

顾临风坐在座位上。手放在桌子上。右手的食指在桌面上按了一下。他没有站起来。没有追。他只是在她的脚步声完全消失之后，低下头。把额头顶在桌面上。头顶着发凉的木板。眼前是黑漆漆的。他在黑漆漆里闭了一会儿眼睛。

教室里的人越来越少了。有人陆陆续续往外走。赵一鸣站起来。想叫顾临风——看到了他趴在桌上。他没叫。拍了拍他的肩膀。拍了两下。然后把什么东西放在顾临风的桌角上——可能是糖，可能是纸条。然后他也走了。

教室里只剩顾临风一个人。

日光灯还亮着。吊扇嗡嗡地转。窗外的银杏在风里发出沙沙的声音。后排墙上那横幅耷拉着的一个角在电扇的风里轻轻摆动。黑板上用白色粉笔写着的那个"3"还是安的——没有人擦它。

他抬起了头。然后慢慢站起来。走到她的座位旁边。她的桌面是空的——比任何时候都空。没有课本。没有笔记本。没有铅笔盒。没有透明水杯。没有名字贴。只剩木桌面的纹理。和一年一年沉淀下来的、三年来杯子放上去造成的那个水印——透明的圆圈。很淡。要光线刚好从窗外照进来才能看到。现在刚好有这个光线。他看到了。

他把手放在那个水印上方，没有碰到桌面。距离大概两厘米。阳光从窗户照进来，照在水印上，也照在他的手背上。他把手收回去。放进口袋里。

然后他转过身。看着整间教室。

他要把这间教室里的一切都记住。

她的位置。二排靠窗。高一和高二坐了两年。她趴在桌上的样子。她的马尾松松地搭在肩膀上。物理笔记本上绿色的小鱼——不，是书包上。她笑的时候眼角先弯。嘴角慢半拍。她唱《送别》的声音——今宵别梦寒。她吃包子的顺序。先吃素的。再吃肉的。好的留到最后。

窗外的银杏全绿了。黑板右上角——没有倒计时了——被"3"取代了。黑板上那个"3"写得很稳。每一笔都是白色。在黑色上很清晰。

他要把这些都记住。因为明天开始就再也看不到了。

他走回自己的座位。背上书包。拿起桌上赵一鸣放的那张纸条。上面写着：等你。楼下。糖我没吃。给你存着。他把纸条折起来放进口袋。然后走到门口。手放在灯的开关上。准备按下去。

在关灯之前。他最后往教室里看了一眼。日光灯的白光。四十二张桌椅的排列。拉歪了的横幅。窗外的绿银杏叶。阳光在木地板上铺了一块金色。

啪。

灯灭了。

他关上门。走在走廊上。五楼的走廊很长。灯光隔一盏亮着一盏。他的脚步声在天花板上制造出回音。一下。又一下。在走廊尽头他拐进楼梯间——向下一层——再下一层。走过了四楼。走过了他和她在楼梯间对视过的那层台阶。走过了三楼——高一的位置。走过了二楼。走到一楼大门口。外面的天还很亮。六月的傍晚，太阳还在半天上。地上是暖的。空气是暖的。是夏天的味道。

他站在教学楼门口。仰头看了一眼五楼。那间教室的窗户是黑色的。

他低头继续走。走过操场的主席台。走过那棵银杏。走过她摔倒过的沙坑。走过他跑过无数次的第五道跑道。走过路灯——那个夏天冬天晚上都亮着的路灯。走过食堂旁边的水房。走过图书馆——那个隔了三张桌子的窗户。走过一切有她的印记的地方。

然后走到了校门口。

赵一鸣在那儿等着。靠在校门口的石柱上。嘴里叼着一根棒棒糖。看到他来，把糖拿下来。

"你能慢一点吗。我站了半个小时。"

"你在等我。"

"废话。不等你等谁。"

两个人往外走。谁也不想回头。但走到校门口的大牌子前面——"青阳县第一中学"——赵一鸣没忍住。回头了。看了校门好几秒。

"三年了。"

"嗯。"

"后天考场见。"

"嗯。"

两个人往不同的方向走。赵一鸣往东——他家在县城。顾临风往汽车站——回镇上。

走了几步。顾临风停住。从口袋里拿出那包纸巾——绿色的心相印。十二月图书馆那包。她没有解释为什么给他。他想保存到什么合适的时候用它。现在似乎是时候了——他抽了一张擦了擦眼镜。但擦完后把剩下的纸巾放回了口袋。还有用。再看一眼青阳一中那个牌子。灯亮了——白色的。校门口那盏灯永远是最早亮的。照得校门的铁栅栏泛着光。

他转身上了班车。靠窗坐下。车开出去的时候，窗外的树一棵一棵往后退。和三年差不多一个场景。那个时候他是从镇上来县城上高中的。现在他要从县城返回镇上了。三天后参加高考。高考后他可以告诉她了。

他在车窗上看到自己的脸。模糊的。外面是后退的树。他看了几秒。然后闭上眼。手在口袋里攥成了拳头。又松开。
