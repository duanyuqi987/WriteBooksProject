# 第007章 新课本的气味

所属部卷：第一部：初秋三楼
时间：高一开学周
地点：教室后排靠窗处

## 诗歌插页 P007：原创仿唐绝句

纸影微明照旧尘，
少年心事隔新春。
回看不见当时路，
一角书声认故人。

事情真正发生时，并没有陆允川想象得那么郑重。新书发下来那天下午，全班都在包书皮。方老师让人把摸底成绩单贴在黑板上，大家都挤过去看。陆允川趴在桌上给物理书包皮，手指沿着书脊压平，一抬头，刚好看见夏知遥站在成绩单前。

她侧对着他，校服还没发到手，穿着一件白色的长袖T恤。她看成绩单的方式和别人不一样——不踮脚，不往前挤，站在人群外面等了大概十秒钟，等前排的人让出一个缝隙，才轻轻靠过去。她显然不是在看自己的排名，因为她的手指在名单上往下划了两下就停了，然后转身走回座位。整个过程不超过十五秒。陆允川手里的透明胶带差点从指间滑出去。胶带脱手落到桌面上，啪地粘住。他低头去揭，指甲抠了两下才揭起来，胶带已经粘了一层灰。他把脏了的那截折掉，重新扯了一截新的——手指很稳，脑子却还停在刚才的画面里。他说不清为什么那十五秒会让自己停住。也许是因为她看成绩单的方式——不踮脚，不往前挤——和他自己缩在人群外面假装削铅笔的样子完全相反。她好像不需要假装什么。

"包这么认真，准备把数学书供起来？"程砚没抬头，他正和自己的书皮较劲，"书皮不平容易翘。""你在跟书皮说话还是跟我说话？"程砚终于抬头看了陆允川一眼，"跟你说话呢——但你眼睛的方向告诉我你可能也没在听。"

陆允川把目光收回到课桌。程砚的透明胶带粘在手指上撕不下来，疼得龇牙咧嘴，把手举到陆允川面前："帮我弄一下，这玩意儿比我前女友还黏人。"陆允川替他撕掉胶带，忍不住笑了一声。胶带撕下来时带起了一小片皮肤上的汗毛，程砚嘶了一声，甩了甩手指。就在这时，夏知遥在前排听见动静回头看了一眼，抿了一下嘴角——那个表情介于"好笑"和"这人怎么这么笨"之间，大概持续了不到一秒就转回去了，继续低头写名字。

陆允川把那不到一秒钟的笑记得太清楚。清楚到多年后程砚问他"你到底是从什么时候开始的"，他脑子里第一个跳出来的画面就是这个——不是报到那天的走廊，不是她的声音，而是程砚被胶带粘住手指疼得叫唤时，她回头抿的那一下嘴角。

班级一直按自己的步调往前走。有人嚷嚷着裁纸刀不够用，有人把书皮包反了拆了重来，有人在课本背面画小人的脸——画的是物理老师，三角眼，鼻子又窄又尖，周围一圈人憋着笑偷看。前排的蒋雯在用橡皮擦课桌上的铅笔印，擦了半天没擦干净，蘸了一点水继续擦。有人从书包里掏出一袋薯片，撕开以后香味飘了半个教室，立刻有两三个人回头问"什么味的"。没有哪一件事为陆允川停下来。也正因为没有停，这一刻才有了形状：它不是被单独聚光照着的，而是夹在透明胶带的撕拉声、削铅笔的沙沙声和程砚的叫疼声中间，勉强留下了一点痕迹。

午饭后他回到教室，大部分人都趴在桌上睡觉，教室里安静下来。他悄悄绕到黑板前，在成绩单上找到了她的名字——排在第十三，不高不低，数学那一栏的分数和他差不多。这个发现没有任何实质意义，只是让他突然有一种不太讲理的亲近感：她在这一科上也和他一样困在中间地带。

下午发新练习册，叶澄一本一本往后传，传到夏知遥那一排时念了名字。前排同时伸出两只手，她笑着说了一句："一个一个来，我又不是食堂打饭的。"旁边几个人笑了。练习册一本本落在桌上，封面还带着油墨味，崭新的纸页硬邦邦的，翻起来哗哗响。有人翻开第一页就发现中间装订歪了，举起来给叶澄看，叶澄说你先用着，明天去教务处换。

晚自习前，教室里又响起一片尺子划过塑料书皮的声音。陆允川把最后一本——语文书——包完，书角压了很久才松手。他把包好的课本放回抽屉，手指碰到抽屉底部一枚生锈的订书针，扎了进去。他把订书针拔出来扔进垃圾袋，看一眼指尖上的红点，觉得自己今天扎的这一下大概和别的事有关。

放学后程砚拽他去操场散步——程砚的原话是"再闷在教室里你就发芽了"。跑道上有人在跑步，胳膊下面夹着计时表，呼出的白气在路灯下散了。程砚踢着跑道边的石子说："你今天包书皮的专注度可以参加职业技能大赛了，你在紧张什么？"陆允川说："我没紧张。""你每次说没紧张的时候鼻子都会皱一下，你照过镜子吗？"陆允川下意识摸了一下鼻子，程砚看到了，笑出声来。

他没有反驳，因为程砚没有恶意。他只是不知道怎么把"包好书皮以后会觉得自己至少做了点什么"解释成一个合理的答案。跑道尽头有人在练起跑，哨子一响就冲出去，跑了二十米又折回来。程砚看了一会儿，说："你看那个人，每次起跑都比别人慢半拍，但跑着跑着就追上去了。"陆允川知道他是在拐着弯说话，没有戳穿他。

回到教室，晚自习快开始了。教室里重新坐满了人，日光灯嗡嗡响。他坐在后排靠窗的位置，把包好的课本一本一本摞整齐，书脊对齐桌边。夏知遥在前排翻课本，透明的书皮在日光灯下反了一下光，正照到他这边。他眯了眯眼，没有抬手挡。
