# 第175章 最重的话多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：公寓书房

## 诗歌插页 P175：原创仿唐绝句

残墨微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

多年后陆允川在一次年底清理旧物时，翻出了那本日记本。

那天是十二月二十九号，再过两天就是元旦。他把书房里的旧纸箱一个个拆开——搬家时封起来的，一直没时间整理。纸箱上有灰，拆开时空气里浮起细小的尘埃，在台灯的光柱里慢慢旋转。

封面已经磨得不成样子，边角卷起，有几页的纸角被水泡过——大概是某个雨天书包进了水。翻开到那一页——密密麻麻的删除线，黑墨压得很深，有几处几乎划破了纸背。纸页背面可以看到凸起的笔画，像盲文一样。

他坐在书房的地板上看了很久。地板上铺着地毯，是妻子选的浅灰色。窗外的城市噪音闷闷地传进来——远处有工地施工，打桩机的节奏一下一下，像远处有人在敲一面很大的鼓。

妻子端着茶杯路过，停下来看了一眼。茶杯冒着热气，是刚泡的红茶。"你在看什么？"

"大学时候的日记。"

"有什么特别的吗？"

"有好几页写满又全部划掉了。划得很重——笔尖把纸都划破了。"

"写的什么？"

陆允川想了一下。"骂人的话。骂一个叫苏晚枝的人——那时候觉得她对不起我。写了很多，写了一页又一页，然后全部划掉了。"他指着一条特别深的删除线，"你看到这条没有——这条下面原本写的是'你知不知道你的一句我很累比所有的付出加起都重——'句子没写完整就划掉了。以我那时的力气，如果发出去，我现在一定后悔。"

妻子坐到他旁边的地板上，接过日记本，自己翻了翻。她在翻页的时候看到后半页——没有删除线的部分。

"这一面写的是什么？"

"是对我自己说的话。"

她轻声读了一段，然后停下来。"写的人是二十三岁的你——但语气不像二十三岁。"

"像几岁？"

"像——比那时候老了十岁的人，回过头来教那个二十三岁的自己怎么不恨。"

"那不是老。"陆允川说，"那是我刚写完那些恨，就知道那些恨不对。脑子里的警钟响了——可能是祁野帮我的，也可能是叶澄提前预设的——总之在删除线之后的那一瞬间，我已经不是二十三岁的我了。可能是二十三岁零两个小时。"

妻子把日记本合上，放在他膝盖上。本子的重量很轻——几十页纸而已——但膝盖上感到的却是一种奇怪的沉。不是物理重量，是时间在上面压出的密度。

"你年轻时候真是一场风暴。"

"对。现在风平浪静了。"

"是因为写够了删除线？"

"是因为有人在我写删除线的时候没走。"

妻子没有追问是谁。她端着自己的茶杯站起来，走到窗边。外面下起了小雨，雨点打在空调外机上，叮叮当当的，像有人在远处弹一种没有调好的琴。

多年后重写那些重话，他不再需要替自己辩解，也不再需要把它们从记忆里铲除。那些重话是存在的——在删除线下面，在划破纸的裂缝里。他不需要否认自己曾经那么恨过——但也不需要让那种恨定义自己。

删除线本身已经是答案。它说明了一个人在情绪最冲动的时候选择停手——不是怕发送——是怕那些话会落在另一个人面前，让她也背上他当年的重量。

他在"不再分析的日常"文件夹里加了一行：

"年底翻旧日记。删除线还在。划得很重。纸破了。那些话没被发送——没被任何人看到。所以几十年后，我不想起来那时有多恨——我想起来的是祁野半夜坐起来跟我说：'你现在写出来的不是狠，是没地方放的疼。'"

他把本子放在书架上。跟那本《中国现代诗选》并排——苏晚枝铅笔写的"好"还在里面。两种笔迹——一个用铅笔轻写，一个用黑墨乱划——隔着几十厘米的书架空间，互相对望。

他想，这两种笔迹其实是他青春的两面。一面是付出、期望、反复和划掉的恨。一面是一个"好"字——某人愿意说好，但不愿意再往前了。两面都是真的。两面都不需要撒谎。

窗外的天已经黑了。他关掉书房的灯，走到客厅。妻子已经在沙发上睡着了，毯子滑到了地上。他捡起来给她盖好，然后去厨房倒了杯水。

祁野前一阵发了条朋友圈："清理旧物。发现一张大学时候写的便签。上面画了一只王八，旁边配字——'陆允川正在写新的不可能成功的策划案'。我这个预判能力不应该只是打游戏的。"

程砚在下面回："你那张太模糊了。我抽屉里有他当年的心理状态时间线——跟实验数据一样精准。时间戳、行为、结论俱全。"

叶澄没有参与。但她单独给陆允川发了一条私信——她永远是发私信的：

"看到祁野的朋友圈了。翻出了那些删除线吗？"

"翻出来了。"

"什么感觉？"

"像一个很久以前的凌晨两点——我坐在同一张桌子的同一侧，同样的台灯，同一个日记本。外面有人打水，脚步声远去。我要么正在划一条我认为太重的话，要么刚写完，准备去水房洗脸。"

"现在你终于可以坐在沙发上，不用台灯，不用日记本，不需要划任何字，也能对它说：晚安。不用等到天亮。"

"对。"

他把水喝完。把杯子放进水槽。窗外城市灯火的海洋比当年的操场大灯更壮观，但没有他需要抬头去看的那一盏。

此刻泡面桶早换了垃圾桶。垃圾桶早换了城市。那些删除线下面的话——永远没机会被任何人读到。被他亲手执行了终审判决。
