# 第180章 你也在省城

时间：2014年3月20日，复试当天

地点：省城师范大学+综合大学→回校火车上

卷次：第八卷：没有答案的答案

复试那天的天气好得不像话。

林玉起早上六点钟出门的时候，东边的天刚泛出鱼肚白。他在校门口的面包店买了一个豆沙包和一杯豆浆，边走边吃。包子是凉的，但豆浆是热的，冷热交替的感觉让他打了个寒颤。

他坐最早的一班城际大巴去省城。车上人不多，他选了个靠窗的位置，把复试通知又看了一遍。面试教室是文学院三楼302，时间是上午十点。

车窗外的田野在三月里绿了一层薄薄的绒毛。他看了一会儿，把目光收回来，闭上眼睛，在脑子里过最后一遍准备的答案。

师范大学的校门比江城大学小，但更旧，有一种沉下来的底气。林玉起到的时候才八点半，他在校门口的石凳上坐了一会儿，看着来来往往的学生。这里以后就是他的学校了——如果复试顺利的话。

复试进行得比想象中快。笔试题不难，面试老师姓魏，五十多岁，头发花白，戴一副银框眼镜。问他的问题有三个，前两个是关于现当代文学的，第三个是："你为什么选择这个学校？"

林玉起想了想，没有说"因为这里的现当代文学专业好"或者"因为导师阵容强"——这些都是对的，但不是真的。

"因为省城离我想去的地方近。"他说。

魏老师看了他一眼，没有追问"想去的地方"是哪里。他只是在评分表上写了几个字，然后说："好，下一个。"

出来的时候林玉起的后背是湿的。不是因为紧张，而是因为说了真话。真话永远比标准答案更让人出汗。

手机在口袋里震了一下。苏安晴。

"复试结束了吗？"

"刚出来。"

"顺利吗？"

"还行。你呢？"

"下午。现在还在图书馆等。"

"不用紧张。讲吴小蔓。"

"好的。讲吴小蔓。"

下午三点，苏安晴的复试也结束了。她没有打电话，发了一条消息："讲完了。老师眼睛红了。"

林玉起看着那条消息，笑了出来。他在师范大学门口的公交站等车的时候，对着手机屏幕笑的。旁边有个大妈看了他一眼，他赶紧把笑容收了收。

两个人约好在省城火车站碰面，一起坐火车回江城。林玉起到得早，在候车室里找了个座位。候车室里有一股方便面和花露水混合的气味，广播里的女声每隔五分钟报一次车次，声音很平。

苏安晴来了。她从候车室的另一头走过来，背着一个蓝色的双肩包，头发被风吹得有点乱。她的脸颊微微发红——不是化妆，是赶路赶的。

"你跑过来的？"林玉起站起来。

"走快了一点，"她喘了口气，"怕你等太久。"

火车是绿皮慢车，省城到江城，三十七分钟。他们面对面坐着，中间隔着一张小桌板。窗外是三月的田野、旧厂房、偶尔一条小河。

"真的讲了吴小蔓？"林玉起问。

"讲了。"

"然后呢？"

"然后三个老师都看着我。中间那个女老师摘了眼镜擦了一下。"苏安晴停下来，"我是不是说太多了？"

"不多。"

她想了想，说："复试最后他们问我为什么选省城。我说——"

她停顿的时间比平时长。窗外的田野换成了一排灰扑扑的厂房，烟囱在冒白烟。

"我说，因为这里有我想做的事，也有我想靠近的人。"

林玉起握着水杯的手停住了。

苏安晴没有看他。她盯着窗外，嘴唇抿着，像她在大一那年小卖部门口看价签的样子。

"我说的是实话，"她说，"面试不就是要说实话吗？"

绿皮车哐当哐当地往前开。三十七分钟，不长不短，刚好够一个人把一场暗恋从大一背到大四，然后发现对方手里也有同样的草稿——她只是比他更沉得住气，把草稿藏了更久。

"都过了。"林玉起说。这句话不是在陈述成绩，是在确认别的什么。

苏安晴终于转过头来。

"都过了。"她说，"你也在省城。"
