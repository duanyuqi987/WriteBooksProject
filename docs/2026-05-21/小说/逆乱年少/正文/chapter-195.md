# 第195章 走吧

时间：离校当天下午五点多

地点：校门口→大巴车站

卷次：第八卷：没有答案的答案

苏安晴站在路灯下，看了一眼自己的行李箱。箱子靠轮子的那面沾了一块泥，大概是在花坛旁边蹭的。她蹲下来，用手掌把那块泥拍掉了。然后站起来，拍了拍手掌上的土。

"走吧。"她说。

这两个字她说了很多年。在敬老院里对李奶奶说"我们走了"，在支教的山村小学里对吴小蔓说"老师走了"，在元旦晚会后台对宋乔说"你快点走吧，下场要开始了"。每一个"走吧"都有明确的前进方向。但今天这一个，方向还没有完全确定——大巴是去封丘的，但封丘只是一个中转站，省城才是下一站。而省城之后是什么，她不知道。

林玉起把行李箱从路灯柱上拿起来，拉杆抽出来——拉到第三节的时候卡了一下，他用力拉了两次才拉到位。这个箱子快退休了，这是它最后的贡献。

大巴站就在校门口往南走三百米。三百米的路，两边是梧桐树和关着的铺面——面包店、奶茶店"甜度"、打印店、修手机的铺子，全都因为毕业季过去了而关门歇业。打印店的老板叫老齐，在门口贴了一张纸条："毕业季已过，九月营业。祝前程似锦。"

林玉起和苏安晴并排走在这三百米的路上。两个行李箱的轮子在地上滚出不同的声音——林玉起的箱子轮子旧了，是咕噜咕噜的声音。苏安晴的箱子新一点，是刷刷的声音，像翻书的纸页。

走到面包店门口的时候，苏安晴停了一下。玻璃门上贴着一张红纸："毕业后，面包会有的，牛奶也会有的。——老板娘康姐。"

"康姐每年毕业都贴这句话。"苏安晴说。

"大一刚来的时候你在这里买过面包吗？"

"买过。"她说，"买了一袋全麦切片面包，五块钱。吃了一周。"

"我也买过。"林玉起说，"同一款。全麦切片，五块钱。"

"那是全校最便宜的面包。"

他们两个同时笑了一下。不是那种哈哈大笑，是嘴唇往上提了一个很轻的弧度，在心里完成了一个默契——大一那年九月，两个从封丘来的新生，在不同的时间去同一家面包店买了同样的最便宜的面包。四年后他们并排走在这条路上，这个发现比任何情话都更真实。

走到两百米的时候，苏安晴忽然开口了："其实面试的时候我还有一句话没说。"

"什么话？"

"复试那天老师问我'你为什么选省城'，我说了'有我想做的事，也有我想靠近的人'。然后那个老师又问了一句——'是男朋友吗'。"

林玉起握着行李箱拉杆的手收紧了。她从来没有在任何人面前定义过他们之间的任何关系。

"你怎么说的？"

苏安晴没有立刻回答。她走路的速度慢了一点，然后恢复了正常。

"我说——'是一个会先在省城等我的人'。"

她说完继续往前走。林玉起在她旁边走了三步，终于消化了这句话。

"会先在省城等"。她没说"是"，也没说"不是"。她说"会"——未来时。"先在省城"——他自己说过的，他考到了省城。"等我"——她会来。

六个字，每一个都经过了苏安晴的挑选。不多不少。

大巴站到了。那辆绿白相间的长途大巴已经在站台旁边等着了。发动机是发动着的，突突突地响，排气管在冒淡淡的青烟。司机姓高，五十多岁，靠在车门边抽一根烟，看见两个人拖着箱子过来，把烟掐灭了。

"去封丘？"

"是。"林玉起把车票拿出来。

高师傅看了一眼，接过票。他打开车厢侧面的行李舱，帮林玉起把箱子塞进去。苏安晴的箱子小一些，和高师傅一起推进去，塞在了林玉起的箱子旁边。两个箱子在幽暗的行李舱里挨在一起——一个旧得掉漆，一个深蓝色，看起来很新。像他们两个人，来自同一个县城，走了不同的路，现在要去同一个方向。

苏安晴登上车门的踏板，林玉起在后面等着她上去。

她没有回头，只是在上车的时候说了一句："你坐哪？"

"你旁边——有空就坐你旁边。"
