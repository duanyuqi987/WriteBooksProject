# 第092章 支教社团

时间：2014年5月

地点：楚州师院

卷次：卷五：各自南北

五月初。学校公告栏贴了一张海报——浅蓝色的底，黑色的字。全省高校支教联盟暑期支教行动招募志愿者。海报右下角印着联盟的标志——一个圆圈，里面是一双手捧着一本书。

顾临风在公告栏前面站了好几分钟。

海报上说——暑期支教，为期两周，地点在太行山区的几个乡镇小学。联盟高校包括楚州师大、楚州师院、楚州理工学院等六所学校。报名方式——填表、面试、分配。

他注意到海报左下方有一个分组表格。上面列出了六所学校的分队编号——楚州师大是第一分队。

苏晚晴在师大。许若琳之前说过她加了支教社团。如果她参加——他大概知道她在第几分队。

他盯着"楚州师大"和"第一分队"这两个词看了很久。公告栏的玻璃反光——他的脸映在分组表格上方，模糊的一块。

他的手伸进书包侧兜，摸到一支笔。然后又把笔放回去了。

其实他可以想办法调队。可以给面试官说想去第一分队。可以编个理由——"有个高中同学在师大"。但他没有。因为见到她应该说什么。她说了不要联系。如果他忽然出现在她的支教队伍里——她会怎么想。他会打乱她的节奏。他也打乱了自己的。

当天晚上他在宿舍的电脑上填了报名表。志愿一栏——"地无所谓"。理由——"想去山里教孩子"。特长——"语文"。他打得很慢。一个键一个键地按。阿坤从背后经过——端着一碗泡面——"又填什么表呢。"

"支教。"

"暑假不回家了？"

"就两周。"

"你还真是闲不住。"

面试在第二教学楼一间空教室里。面试官是两个大三的学长——一个男生一个女生。男生的胡子刮得很不干净，下巴有一道青色的印子。女生的头发很短，戴着一副红框眼镜。

"你为什么要参加支教。"

"想做点有用的事。"

"你有什么特长。"

"语文。会写毛笔字。"

"对山区孩子有什么了解。"

"不了解。想去了解。"

"如果分配到的学校条件很差——没有热水、没有网络——你怎么办。"

"不是去度假的。"

面试官互相看了一眼。女生在表格上写了几个字。然后说"可以了。结果会发短信通知。"

三天后。短信来了——"顾临风同学，你已被录取为暑期支教志愿者。分配分队：第三分队。支教地点：李家沟小学。"

第三分队。

不是第一分队。

他把短信看了两遍。然后把手机放在桌上。

第三分队——是楚州理工学院的队伍。跟楚州师大不在同一个分队。李家沟小学在太行山另一个镇上——离她那个支教点大概隔了三十多公里的盘山公路。

其实他是想去有她的那队的。

但他没有提。报名的时候他在志愿栏写了"地点无所谓"。面试的时候他没有问分队的事。接到分配通知后他也没有提出调队申请。他回复了短信——"收到。确认参加。"就四个字。

四个字。比十一个站名短。比十二公里短。他把手机放在桌上，然后拿起来又看了一遍短信。第三分队。李家沟。太行山深处。三十多公里的盘山公路——坐小车大概要四五十分钟。比301路还短一点。但301路他没有坐过。这趟盘山公路他大概会坐很多次——接下来的两年暑假，如果还去支教的话。

去不了有她的那队——也没关系。他还是会去。他不是为了她才报支教的。至少——不完全是。那个小地方有二十一个孩子。他教了他们《静夜思》。他们问他"老师你家离这里多远"。他会在两年后、三年后还想去——不是为了在太行山的盘山公路上"偶遇"她。是为了那些孩子。也为了自己——证明给自己看，他不是只会等。

晚上阿坤问了他一句——"支教队分到哪了。"

"第三分队。李家沟。"

"远不远。"

"太行山里面。大概五六个小时的车。"

"就你一个人去？没有认识的人吗。"

"没有。"

李博从对面床铺往这边看了一眼。他没有说什么。他知道顾临风说的"没有认识的人"是跟谁在比——第一分队是师大的队伍。

他比谁都清楚。
