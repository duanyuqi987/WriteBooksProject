# 第219章 旧日记本旁观

所属部卷：第五部：逝水有灯
时间：搬家当天
地点：搬家纸箱旁

## 诗歌插页 P219：原创现代诗

朋友只看见纸箱
看不见纸箱里的年份
他弯腰帮忙
说这一箱很重
我说是纸

程砚是下班后赶来的。他穿着衬衫，袖口卷到手肘，进门第一句话是：“你这哪是搬家，你这是转移资料室。”陆允川正蹲在地上给最后一箱贴胶带，听见这句，笑了一下。

程砚搬起写着“旧纸资料”的箱子，皱眉：“这里面什么？砖头？”

“日记本，旧相册，还有一堆证件复印件。”

程砚没有追问日记本。他们认识太久，知道有些东西不适合在搬家现场摊开。下楼时电梯慢，两个成年人抱着纸箱站在楼道里，灯光白得发冷。程砚忽然说：“你以前写东西挺狠的。”

陆允川愣了一下：“你看过？”

“没看过。”程砚把箱子往上托了托，“但你那时候说话就够狠，写下来估计更狠。”

这句玩笑不轻不重，却刚好。陆允川没有辩解，只说：“后来删了不少。”

“删可以，别删到只剩自己委屈。”程砚看着电梯数字往下跳，“人不能靠改稿把自己改成好人。”

电梯门开了，搬家车停在雨后的路边。程砚把箱子放上车，拍了拍纸板上的灰。旁观的朋友不替他忏悔，也不替他开脱，只在关键处提醒一句：别把真实修成好看的样子。
