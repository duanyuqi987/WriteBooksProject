# 第147章 雪仗

时间：2012年12月20日，下午

地点：江城大学北区操场

卷次：第六卷：雪落无声

下午雪停了。

天还是灰蒙蒙的，但雪地上反射的光很亮，亮得有点刺眼。北区操场上聚了很多人——打雪仗的、堆雪人的、拍照的。南方来的大一新生最兴奋，有个广东的男生在雪地里打滚，被同宿舍的人拍了视频说要发到QQ空间。

林玉起从教学楼出来，往宿舍走。路过北区操场的时候停住了。

因为他在人群中认出了一个声音。

不是说的话——离得太远听不清。是笑声。

苏安晴的笑声。

苏安晴在操场上打雪仗。

她和宋乔、还有另外两个女生——一个是室友方敏，另一个是她们教育学院的班长周莹——四个人在操场东南角。苏安晴蹲在一棵梧桐树后面，手里捏了一个雪球，探头看了看对面的位置，然后猛地站起来把雪球扔了出去。

扔中了。宋乔的背上炸开一朵白色的花。

宋乔尖叫了一声，抓起地上的雪反击。苏安晴往旁边躲，踩空了一步，整个人摔在了雪地里。她没有立刻起来，而是躺在雪地上笑——笑得很大声。

林玉起从来没见过苏安晴这样笑。

平时的她是克制的。笑也是微笑，说话也是斟酌过的，每一个表情都落在可控的范围内。即使是支教的时候在乒乓球台上说的那些话，她的声音也始终保持在一种稳定的频率上。

但现在的苏安晴不一样。

她躺在雪地上，头发散了，围巾歪了，手套掉了一只，脸上沾了雪。她在笑——眼睛眯成了一条缝，嘴大张着，笑得身体都在抖。宋乔扑过去往她身上撒雪，她一边笑一边用手挡，但挡得完全不用力。方敏在旁边拍照，周莹笑得弯下了腰。

林玉起站在操场边上的梧桐树下，看着。

他意识到一件事：这才是苏安晴本来的样子。平时那个冷静的、克制的、做什么都先想好的苏安晴，是她后天学会的。这个在雪地里笑得肆无忌惮的，才是出厂设置。

苏安晴从雪地上坐起来，拍掉头发上的雪。然后她看见了林玉起。

他们隔着半个操场的距离。

苏安晴站起来。她抓了一把雪，在手里捏了两下。

朝着他的方向扔了过来。

雪球飞了大概七八米就散了——捏得不够紧，在半空中解体成一片雪的粉末。没打中。差了起码三米。

但两个人都笑了。

林玉起弯腰抓了一把雪，捏实了，朝她扔过去。他的准头也不行，但比苏安晴强一点，雪球落在了她脚边上，溅起的雪末子扑到了她的裤腿上。

然后宋乔插手了。宋乔从侧面包抄过来，手里捏着两个大雪球，往林玉起身上招呼。两个都中了。一个打在背上，一个打在肩膀上。然后是方敏。然后是周莹。四个女生围攻一个人。

林玉起没有反击的余地。他只能往后退，一边退一边胡乱扔几把雪。雪地上踩出了一大片乱七八糟的脚印。

笑了大概十分钟。或者更久。没人看表。

最后苏安晴叫停了。她站在操场中央，双手撑着膝盖喘气，额头上有细细的汗。"不行了。打不动了。"

她把掉在地上的手套捡起来，拍了拍上面的雪。

然后她朝林玉起走过来。脸上还带着刚才笑过的痕迹——眼角有点红，嘴巴还是咧着的，没有完全收回去。

"你怎么那么准——"她说，"——准地不准。"

"你在表扬我吗。"

"你可以这么理解。"

她把手套戴回去，发现少了一只——左手的。林玉起四下看了看，在雪地里找到了那只蓝色的毛线手套，捡起来递给她。

"谢谢。"

她把左手套戴上，活动了一下手指。

"林玉起。"

"嗯。"

"你刚才笑了。"

"你也笑了。"

"我知道。"她看了看操场上那一地狼藉的脚印。"我好长时间没这么笑过了。从上初中以后，好像就——"

她没有说完。但林玉起听懂了。

宋乔在远处喊苏安晴——"走啦！冻死了！"

苏安晴朝她们挥了挥手，然后回头看林玉起。

"走了。"

"嗯。"

她走出了几步，又回头。脸上那个笑已经完全收住了，但眼睛还带着刚才的亮度。

"雪里看你——"她说，"——还挺帅的。"

然后她转身跑了。跑得很快，在雪地上踩出了一长串深深的脚印。

林玉起站在原地。

他看着那串脚印，看着残留在手套里的雪化成水，在指缝间慢慢干掉。

>
雪中一笑声，惊破两年静。
少年始知君，原来不是冰。
