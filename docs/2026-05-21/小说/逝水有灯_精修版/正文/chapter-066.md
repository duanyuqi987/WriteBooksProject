# 第066章 墙角的试卷之前

所属部卷：第二部：六月以后
时间：高考后第二天
地点：陆家小院的东屋

## 诗歌插页 P066：原创现代诗

我把水放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
墙角那摞卷起边的试卷没有替谁回答，
只把那一年轻轻照亮。

高考后第二天，陆允川睡到九点才醒。

阳光已经从东窗户照进来，落在床脚那堆课本上。他翻了个身，闭上眼，又睁开——确认今天不用去学校，不用背书，不用做题。然后才慢慢坐起来。

母亲在院子里喂鸡。听见他房门响，从窗户探进头来："锅里有粥。自己盛。"

"嗯。"

他趿着拖鞋去厨房。粥是白粥，旁边放了一碟咸菜和半个咸鸭蛋。他盛了一碗，坐在厨房的小凳子上吃。母亲从院子里进来洗手，看见他在吃，说："今天是第二天了，你可以出去玩玩。别老闷在屋里。"

"去哪？"

"程砚不是老打电话来吗。"

陆允川想起昨晚程砚在烧烤摊上说的——今天去打球。他还没回程砚消息。

吃完饭他回到东屋，拉开书包拉链。课本、卷子、草稿纸、笔芯，哗啦一下全倒在了床上。他坐在地上，背靠着床沿，面对这摊纸愣了一下。

三年。这些东西就是他三年的全部。

他把数学卷子捡出来。高三上学期的期中卷，一百零二分，选择题错了四道，立体几何那道画错了辅助线，程砚那次说他"辅助线画AC不是BD"。他把这张卷子放到一边。

又捡出一张语文卷。作文的题目是《我的理想》，他写了八百字，结尾套了一句鲁迅的名言。老师批语是："论点分散，但文笔有进步。"他记得那次作文分不高，但叶澄说比上次好。

物理卷。化学卷。英语卷。每一张都能唤起一些记忆——不是关于题目的，是关于做那张卷子那天发生的事。物理卷那天程砚在课上被叫上黑板做题，结果做错了，下来以后一直念叨。化学卷那天叶澄给了他一颗薄荷糖。

他把卷子按科目分成几摞，分到一半又打乱了。因为有一些卷子他记不清是哪天做的，有一些他甚至不记得自己做过。

母亲推开房门，手里端着一碗切好的西瓜。西瓜是井水冰过的，碗底汪着一层淡红色的汁水。

"这些还要不要？"她看着满床的纸。

"先放着。我自己来。"

"要扔就扔，要留就留，别堆着。"母亲把西瓜碗放在窗台上，"热了就吃。"

她出去了。房门没关严，从门缝里漏进院子里的阳光和鸡叫。

陆允川拿起一块西瓜咬了一口。冰凉的，很甜。他吃完一块，翻到一张英语卷子——高一下学期的期末卷，姓名栏写着"陆允川"，旁边有一个被橡皮擦掉的铅笔印，隐隐约约能看出"夏"字的起笔。

他想起来了。那次考试他和夏知遥分在同一个考场，他坐在她后面两排。考完以后他在卷子上用铅笔写了她的姓，然后马上擦掉了。橡皮擦得很用力，纸都擦毛了，但还是留下了印子。

他把这张卷子合上，放到"犹豫"那一摞——这是他临时起意加的分类。不是"留"也不是"扔"，是"犹豫"。

手机响了。程砚发来消息：醒了吗？打球去不？

陆允川擦了擦手，打字：几点？

程砚秒回：现在。操场等你。

陆允川：等我十分钟。

他把手里的西瓜吃完，换了件T恤。出门前回头看了一眼床上那堆纸——它们在阳光里安安静静地躺着，像退潮后留在沙滩上的贝壳。各色各样，有些完整，有些缺了角。

房门关上，那些纸被留在背后，被窗外的风轻轻掀了一下页角。
