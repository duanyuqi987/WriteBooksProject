# 第167章 牵手的误会

所属部卷：第四部：失去的美丽
时间：大二冬夜
地点：学校东门外小路 / 食堂

## 诗歌插页 P167：原创仿唐绝句

炉火微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

事情落到第二天中午，陆允川最终没有忍住。

他在食堂碰到苏晚枝的时候，她正一个人吃饭，面前一碗牛肉面。他端着餐盘坐到她对面，语气比平时硬。

"昨天晚上你在东门？"

苏晚枝抬头看了他一眼。"对。买烤串。"

"旁边有人。"

"我同学。怎么了？"

"——那个扶你肩膀的人。"

苏晚枝的筷子停住了。她慢慢把筷子放下，擦了擦嘴角。

"你看到了。"

"对。"

"那你想问什么？"

"我想问——"陆允川的句子到嘴边又缩了回去，但最终还是挤了出来，"那个人是谁？你们是什么关系？"

苏晚枝看着他。食堂里的噪声很大，但这两个人之间的空隙忽然很安静。

"那个人是通讯社的学长。我们刚做完一个采访回来，我的鞋踩到下水道盖子的缝，差点崴了。他扶了一下。整个过程不到一秒。"她一字一句地说，"你还有什么想问的？"

陆允川张了张嘴。苏晚枝不给他说话的时间，继续说道：

"你知道最让我累的是什么吗？不是跟你说明情况。是每次发生了正常的事情之后，都需要对你说明。我上个晚自习，路上遇到学长一起走了一段，我就得告诉你。我站得离什么人近了一点，我就得跟你解释。你知道这叫什么吗？这叫——我需要把你随时放在心上。"

她的声音不高，但每一个字都像被压了很久才轮到的。

"我不是不喜欢被别人放在心上。但你把我放在心上的方式，是一种审核。审核我有没有跟别人接触，审核我的行为符不符合你的预期。你想过没有——我连跟你是什么关系都说不清楚，就已经在被你审核了。"

陆允川低头看着自己的餐盘。米饭已经凉了。

"我没有想审核你。"

"但你做的，就是审核。"苏晚枝的声音缓了一点，但分量不轻，"你说你很难受——我理解。但我的难受你不理解：我需要反复地向一个没有承诺过我任何东西的人证明我的清白。这是最消耗人的关系。"

"我没有让你证明——"

"你没有用嘴说。但你用眼神说。用沉默说。用你的情绪说。"苏晚枝把椅子往后推了一点，"你知道我昨晚为什么没有主动告诉你这件事吗？因为我觉得不需要。一个学长在马路上扶了一个差点崴脚的学妹，需要上报给你吗。"

陆允川没有回答。

苏晚枝站起来，端起餐盘。"我还有事。先走了。"

她走了两步，又回头。

"如果以后每件事都要这样解释，我会很累。而且不值得——不是因为你不值得，是这样解释本身不值得。"

陆允川一个人坐在食堂里。碗里的牛肉面还剩一半。

他把筷子放下。

然后拿起手机，给叶澄发了一条消息："我跟她说了。她解释了——学长，扶她是因为差点崴脚。过程不到一秒。她说她很累。"

叶澄秒回："她觉得累是对的。"

"我知道。但我不去问她的话，我自己会想很多。"

"你想很多是你的问题。你让她解释之前，有没有想过：她为什么要向你解释？你们两个之间，有谁给过谁解释的义务？"

陆允川看着屏幕，没有打字。叶澄又补了一条：

"你这是把事情反过来了。本来是你看到一秒就进了死胡同——现在好了，你进去之后，把她也拉进来了，还嫌她解释得不够主动。"

他没有回。把手机装进口袋，端着餐盘走向回收处。

下午，他在图书馆碰到叶澄。叶澄刚自习完，正在整理书包。

"有空吗？"他问。

"说吧。"

"我不明白。我真的不是故意要审核她。我只是——看到那一下，心里就空了。然后就一个劲地想——她是不是有别人了，她是不是讨厌我到连告都懒得告诉我。"

叶澄把书包拉链拉上，靠在书架旁听他。她的头发被日光灯映得有点灰白，但表情清晰。

"你看到的那一秒，你解读成什么？"

"解读成亲密。"

"但实际呢？"

"实际是她的鞋踩进缝里差点崴了。"

"那你为什么不先信那一秒可能只是偶然，而是一上来就信它代表什么？"

陆允川想很久。

"因为我怕她是别人的。"

"所以呢？怕不是理由。怕只能解释你为什么这样做，不能证明你这样做是对的。"

叶澄站起来。两人的对话在学术问答的浅淡气息中结束。

"今晚去操场跑几圈。什么都别想，跑累了回宿舍睡觉。明天等心情平了再说。"

"好。"

"还有——"她在他转身时叫住他，"你扪心自问，你介意的，真的只是那一秒吗？还是介意——她可以没有你就过得很正常。"

陆允川的肩膀松了半寸。他没有回答，但叶澄也没有等他的回答。

她推开门走了。冬夜的冷气从门缝里涌进来，又很快被暖气赶回去。

他在原地站了很久。不是因为叶澄那句话扎心——是因为他找不到反驳的理由。

介意的不只是那一秒。介意的是她没有他也可以好好吃饭好好走路好好跟人说话。介意的是她的世界里他没有占据所有的位置。介意的是她不是一个属于他的答案。

而她没有义务成为任何一个"属于他"的东西。
