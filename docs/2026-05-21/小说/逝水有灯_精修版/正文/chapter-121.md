# 第121章 教室的灯之前

所属部卷：第三部：寒风里的花
时间：同一天夜里
地点：二号教学楼空教室

## 诗歌插页 P121：原创现代诗

我把教室灯放在黄昏里，
不让它替任何人回答。
风从校园的一侧经过，
带走一句没有说稳的话。
后来我学会把手松开，
让灯照着路，也照着空出来的地方。

晚上九点二十，苏晚枝在诗社群里发了一条消息："谁现在有空？二号教学楼301教室，帮我搬一下桌子。教室后墙的板报要重新排版，我一个人搬不动。"

陆允川正在宿舍看当代文学课的论文。手机亮了一下，他看了一眼消息。三秒钟内祁野和叶澄都没回复——叶澄应该在洗澡，祁野大概率在打球。他回了三个字："我来。"

九点二十七分，陆允川到二号教学楼楼下。这栋楼是学校最老的教学楼之一，外墙贴了米黄色的瓷砖，走廊地面铺的是水磨石。走廊里的灯是日光灯管，白得发冷，有一根在走廊尽头一直闪。清洁工刚刚拖过地，水痕弯弯曲曲地延伸到楼梯口，拖把桶搁在角落，塑料桶身被撞出了几道凹印。

三楼走廊更安静。301教室的门开着，里面只有前排的灯亮着。苏晚枝站在教室后面，面对着六张并排的铁架课桌。每张桌子都很大，是用来画工程图的绘图桌，空置已久，桌面上积了一层灰。

"你来了。"她没回头，声音从教室后面传过来。

"这些都要搬？"

"对。六张全部移到靠窗的位置。"她终于转身，下巴朝窗户方向抬了抬，"给后墙腾出一整面白墙。新板报的主题是'诗与空间'——我想用整面墙做一个诗歌拼贴。"

"像地图那样？"

"不是地图。是拼贴——每首诗剪成不同大小的纸片，贴在墙上形成一个大的图案。图案本身是一首诗的形状。"

陆允川走到第二排，卷起袖子。"什么图案？"

"一棵树。从地到天花板。"

绘图桌很重。铁架的，四只脚贴着防滑橡胶垫，在水泥地上拖动时发出沉闷的低音。陆允川搬第一张，桌子歪了一下，桌面差点滑落。苏晚枝赶紧扶住另一头。

"一起搬。一张一张的。"她说。

两个人一人抓一头，把第一张桌子贴着地面挪到窗边。摆好后苏晚枝用脚推正桌角的位置，推到和窗框平行。

"这个桌脚要对齐窗框的线——"

"否则空间线不直。我知道。"陆允川替她说完了。

苏晚枝愣了一下。"你怎么知道我的下一句？"

"因为你每次对空间的精确度都有这个要求。上次社刊样稿你说——"

"'页码和边距要对齐裁切线——不是差不多，是精确。'"

"对。"

第二张桌子搬到窗边时，陆允川的手心开始出汗。第三张桌子搬完，他停下来喘了口气。苏晚枝从讲台抽屉里翻出一瓶没开过的矿泉水，递给他。

"这里怎么会有水？"

"我以前放这里的。这个教室我经常来。"她把另一瓶拧开自己喝了一口。"大二上学期吧，有次考试周需要安静的地方复习——图书馆挤满了人，宿舍太吵，我在二号教学楼转了一圈，发现301的门锁是坏的。进来以后觉得这里好——桌面上有灰，说明没人来。"

"所以这是你的秘密基地。"

"不算秘密。只是大家不知道这个教室不上锁。"她把水瓶放下来，走到后墙前，用手指在墙面上比划。"树从墙角这里长出来——树干沿着墙的中线往上。树枝分开，每一枝上挂五六片纸——每片纸是一首短诗。"

"那树干用什么？"

"用长条的诗句。一整句作为树干，从上到下。"

"什么句子？"

苏晚枝把手从墙上收回来，想了片刻。"还没确定。可能需要一句够长的话——长到足以撑起一棵树。"

"我有个建议。"

"说。"

"'诗不是装饰，也不是替人显得文雅。'"

苏晚枝转头看他。"这是你自己写的？"

"是我抄的你写的。你去年的社刊前言。"

"这句话不对。太像标语了。"

"那换一句。"陆允川走到墙前面，用手指在墙的中线上比了比。"不管句子是什么——树是从地往天花板长的。但字要从上往下排。这样读者从树根往上看——看的是字从天上落到地上。"

苏晚枝盯着他指的那条线。"你说得对。"

"哪部分？"

"全部。'字从天上落到地上'——这个写进你的'等字考'提案里。"她蹲下去，用手指在踢脚线的位置画了一个小记号，"树的根在这里。"

搬完六张桌子后已经快十点半了。走廊里日光灯闪得更厉害，那根坏灯管已经完全罢工，让走廊中间段落入半明半暗。陆允川去洗手间冲了冲手上的灰，回来时发现苏晚枝站在教室中央，关了后排的灯，只留前排一盏。

"最后看一下效果。"她说。

偌大的教室空出了一整面白墙。六张绘图桌整齐排在窗边。前排的台灯照着讲台，光线落在粉笔槽的碎粉笔头上。苏晚枝站在阴影里，白墙前，构图像一张有意安排的彩色照片。

"明天开始贴纸。"她说。

"我可以来帮忙。"

"好。"

她关了最后一盏灯。教室陷入完全的暗。走廊里远处清洁工推拖把桶的声音一下一下地传过来，像钟摆。

"走吧。"苏晚枝说。

两个人沿着昏暗的走廊往楼梯走。陆允川走在后面，看着她的背影刚被走廊灯照亮又被下一截阴影遮住，交替出现。她推开楼梯间的防火门时回头说了一句："晚上的教学楼比白天安静很多。空教室的灯不用全部亮——亮一盏就够了。"

"所以你今天只开了前排的灯。"

"故意的。"她走进楼梯间，鞋底在台阶上落得很轻。"这栋楼九点半以后就没人了。只亮一盏灯的话，从外面看会以为是清洁员忘了关。不会有人来。"

"原来如此。"

"对。秘密基地的第一条规则：灯光不要超过一盏。"

陆允川在楼梯间门口停了一下，回头看了一眼301教室紧闭的门和门下透出来的那线暗光——是走廊尽头的那个应急灯。他说不清为什么，但他发现自己很想在接下来的几个晚上都来这个教室，哪怕只是坐在角落里看书。不是为了碰见谁。是因为这个空间里有一种他想要的东西：安静、精确、有序。她说得对——灯光不要超过一盏。
