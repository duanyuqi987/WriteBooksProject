# 第131章 推门

时间：2018年3月周六下午3:00

地点：楚州"一茶一坐"茶餐厅

卷次：卷七：流年未断

风铃响了。

她抬起头。

目光从手机屏幕移到他身上。手机还亮着。屏幕上是她没看完的一篇文章——标题写着什么育儿经验。左手食指还停在屏幕上方。

顾临风的手还放在门把上。门在他身后缓缓合上——弹簧很紧，门框撞在胶条上发出一声闷响。风铃又震了一下。他感觉到自己的手心是湿的。门把上留下了淡淡的汗印。

她的表情在他推门的这三秒钟里做了三次变化。

先是皱眉——像在辨认。眼睛微微眯起来。睫毛扑了一下。手机放在桌上——屏幕朝下。头歪了一点点。嘴唇动了一下——大概是想说什么但没说。那个表情像一个在火车站人群中忽然觉得看见了熟人——但不能确定的人。嘴唇翕动了。没有声音。大概是一个字的嘴型——"顾"——没念出来。

他往前走了一步。阳光从窗子打在他身上——浅灰衬衫、深灰夹克、头发有点翘。脚步很慢。怕走快了会吓到她。又怕走慢了会让她觉得他在犹豫。

然后她的眉头松开了。

眉头展开的过程很慢。不是那种恍然大悟——而是那种确认。像在看一个记忆里的轮廓慢慢跟眼前这个人的轮廓重合。从眉骨的弧度、到鼻梁的高度、到下巴的线条。从无法确定——到好像可以确定——到真的可以确定。从高一教室后排那个低着头的男生——到现在站在她面前的这个人。一样的眉骨。一样的鼻梁高度。一样的下巴线条。只是比以前更瘦了。轮廓更硬了一点。

她的嘴唇微微张开了。嘴角动了一下。

然后她笑了。

慢慢地。嘴角往上弯。很慢——好像不确定是不是该笑。然后又弯了一点。然后整张脸都松了。

不是那种久别重逢的激动。是一种——好像是这样的。好像本该是这样的。好像这么多年就是该在一个普通的周六下午的茶餐厅里，推开门的人是他。

他也笑了。

嘴角不自觉地上扬。左边先动——然后右边跟上。幅度不大。但眼睛里的笑意比嘴角明显。他感觉到自己的眼尾出现了细纹。二十四岁的皱纹。被粉笔灰和失眠的夜晚磨出来的。

阳光从窗外照进来，照在两人之间的地板上。那是一块很大的方形光斑。他站在光斑的这一边。她坐在光斑的那一边。阳光在他们之间画了一道明亮的分界线。阳光里有微尘——细细的。在空气里飘来飘去。永远落不了地。

他往前走。踩进了光斑里。鞋底的灰在地板上留下一个很淡的脚印。阳光照在他的鞋面上——黑色的皮鞋。擦了油的。但鞋头上还有一道划痕。

她在看他。没有移开目光。

不像高中时候。高中她总是很快移开目光。走廊上那几次对视——她和他目光相遇不会超过一秒就会低下去。他假装看窗外。她假装经过。两个人都装着。装了三年。现在她看着他走进来，目光稳稳地——不移开。

他的喉咙动了一下。想说什么，但嘴唇动了动没有发出声音。喉咙里好像有什么东西卡住了。一团气。或者一整个高中的秋天。

她先开口了。

声音很轻。比高中时厚了一点点——大概是当了六年班主任老师磨出来的，但尾音还是一样的。微微往上提。那种尾音——他记了八年。走廊上她跟许若琳说话时。食堂里她跟同学打招呼时。教室里她回答老师问题时。都是这个尾音。微微往上提半度。像一句话结束了但还没完全结束。

她说的话混在茶餐厅的背景音乐里——听不太清。他只看到她的嘴唇在动。嘴唇的形状跟高中时一样。下唇比上唇薄一点点。说话的时候嘴唇会微微往外翻一下。很轻。几乎看不出来。

然后他听清了。

"顾临风。"

她叫了他的全名。连名带姓。像高中班主任点名时念出的自己的名字。

不是"你是"。不是"好久不见"。就三个字——顾临风。声音很轻。尾音在空气里挂了零点几秒。然后落下。像一根羽毛从高处飘下来——落在地上。没有声音。但听得到。

他站在她面前大概三秒钟。三秒钟里他想了什么——他也说不清。只觉得自己站在一个等了太久的画面里。这个画面他想过无数次。在深夜的手电筒光里。在高考前最后一天的教室里。在楚州公寓的窗前。现在画面动起来了。不是静止的。她在看着他。嘴唇合上了。眼睛一眨不眨的。然后他点了点头。

"嗯。"

声音比想象的沙。今天第一天开口说话——跟谁都没说过。从早上到现在，他没跟任何人说过话。他清了清嗓子。手在裤兜里攥了一下。大拇指的指甲掐在食指的指节上——疼。他松开了。

"苏晚晴。"他说。

他也叫了她的全名。也是连名带姓——就像他在纸上写过的很多遍一样。但这次是说出来的。对着她说出来的。两个字。苏。晚。晴。舌头顶住上颚。吐气。声音落定。这个声音传进了她的耳朵里——不是响在纸上。是响在空气里。在这个有阳光的茶餐厅里。有竹编灯罩吊在头顶。有窗外的樱花豆叶。有门上的风铃刚刚停住。有她的面前一杯水还温着。

她听了以后，眼睛亮了一下。

不是哭。是亮。像灯忽然调大了一格。发光的部分不只是瞳孔——是整只眼睛。眼角。眼白。虹膜。都亮了一下。然后又回到正常的亮度。但那零点几秒的光——他看到了。

然后她低下头。看着桌上那杯喝了一半的水。杯子里的水面微微晃动——是她刚才端的时候手在抖——或者是他推门时候风铃震动传到了桌面上。水面在杯子里转了一小圈。然后恢复了平静。她低下头的动作也很熟悉——高中时她每次跟他对视完都会低下头。不同的是高中时候是躲闪。现在是——整理。她需要一秒钟整理一下自己的表情。把翘起来的嘴角往下压一压。没压住。嘴角还是弯的。

她抬起头的时候，嘴角还是弯的。

"坐。"她说。指了指对面的椅子。椅子的把手是藤编的。藤条有点松了。缠在扶手上——最外面一圈翘起来了。好像刚才一直在等人来。

他把夹克的拉链拉开。拉链往下滑——声音很轻。像一根针划过布面。脱下一只袖子。再脱另一只。搭在椅背上。夹克的里子是格子图案的——红黑相间。洗了很多次。颜色淡了。椅子被拉开——木腿在地板上发出一声轻响。

他坐下。

双手放在桌上。手指交叉——然后松开——然后放在膝盖上。膝盖上能感觉到裤子的布料——棉的。有点薄。然后手又放回桌上。放在靠近自己的这一侧。离她的手大概半臂的距离。

她看着他坐下来。眼睛跟着他从门边走到桌边——从站着到坐下。这个过程中她的目光一直在——像在确认他是真的。从头到脚。从脚到头。看得不快不慢。最后目光落回他的脸上。停住了。

服务员走过来。围裙是绿色的。口袋里插了一支笔。菜单拿在手里——是竹简式的。用麻绳绑着。翻开来有竹片的清香味。

"先生喝什么。"

他看了她一眼。她面前还是那杯水。

"绿茶。"

"您呢——"服务员转向她。

"茉莉花茶。"

服务员走开了。水壶放下来——里面有烧水的咕噜声。热气从壶嘴冒出来。茶壶是紫砂的——深褐色。上面刻了几笔竹子。笔触很简单——两节竹干。几片叶子。

两人面对面坐着。木桌的宽度大概七十厘米。

近到他能看见她鼻尖上有一点很小的光——窗外光线反射的。近到他能看见她鼻翼两侧有很淡的雀斑——几粒。以前高中时坐在后排看不到。现在在这么近的距离里看到了。近到她能看见他领口磨毛的纹路。白色的线头翘在外面。近到能数清她眨眼睛的次数——一分钟大概七八次。

两个人同时张了张嘴。又同时闭上了。他吸了一口气。她拨了一下碎发。然后同时动了嘴唇。又同时停下来。

茶还没上。茉莉花还没有泡开。绿茶还在壶里安静地卷着。

窗外的樱花树枝在风里晃了一下。一只麻雀落在枝头。枝头往下沉了一点点。麻雀转头看了看天。飞走了。枝头弹回来。

一片嫩绿的叶子贴在了玻璃上。很快又被风带走了。
