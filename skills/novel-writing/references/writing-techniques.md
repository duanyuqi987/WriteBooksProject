# 小说写作技法库

本库收录 10 种核心写作技法，源自《红楼梦》等经典小说的技法分析，可直接在写作中运用。

## 使用方式

- **novel-outline** 规划章节时，为每章标注建议使用的技法
- **novel-draft** 写正文时，根据场景类型自动匹配技法
- **novel-review** 审查时，检查技法使用是否得当

---

## 一、不写之写

> 不着笔墨，尽得风流。某些内容故意不写，反而比写出来效果更好。

### 核心原理
- 读者在脑补过程中投入想象力，产生更强的阅读参与感
- 留白给读者制造的悬念和疑问驱动继续阅读
- 有时候"写什么"不如"不写什么"重要

### 适用场景
- 多人场景中某个关键人物保持沉默（如薛宝钗在刘老老逗笑全场时不笑）
- 情感高潮处只写动作不写内心
- 重要事件通过侧面反应来暗示，不正面描写
- 对话中的"不说"比"说"更有信息量

### 操作要点
1. 决定不写什么：提前规划哪些内容要留给读者想象
2. 用周围人物的反应来暗示被省略的内容
3. 确保不写的内容读者能通过上下文感知到
4. 不写 ≠ 忘了写，要有意识的选择

### 使用自查
- 这段内容如果删掉，读者会注意到吗？
- 删掉后，读者能否从前后文中推出被删的内容？
- 被省略的内容所产生的空白，是否比填满更有力量？

## 二、一击两鸣（一笔两用）

> 通过一件事同时描绘两件或多件事。一支笔作千百支用。

### 核心原理
- 过渡段落不浪费，每个看似日常的细节都在埋线
- 表层对话/动作 + 深层信息传递二合一
- 高效的叙事密度，用最少字传达最多信息

### 适用场景
- 日常场景中暗含权力关系（如周瑞家的回话暗含王夫人掌实权）
- 丫鬟/下人的行为暗示主子之间的关系
- 一次对话同时推进多条线索
- 过渡段不浪费，同时做场景切换、人物出场的铺设

### 操作要点
1. 写每个过渡场景时自问：这个场景除了表面内容，还能暗示什么？
2. 丫鬟、仆人、路人等次要人物的言行可以折射主角之间的关系
3. 闲聊不闲，每一句闲话都应有深层含义或伏线
4. 让一次行动同时完成多个叙事任务

### 使用自查
- 这个场景删掉后，除了表面情节，读者还会失去什么信息？
- 能否让这段过渡同时暗示至少一个深层信息？

## 三、草蛇灰线（伏延千里）

> 像在草地看蛇，似断非断。把不引人注目的线索精心插入描述中，将来再作发展。

### 核心原理
- 提前埋下读者不会立即注意到的线索
- 线索之间保持"似断非断"：不是连续出现，但读者回想时能连起来
- 回头读第二遍时，读者会发现处处是暗示

### 适用场景
- 人物命运的预言（如宝玉梦游太虚幻境看判词）
- 关键道具的反复出现（如通灵宝玉前七回反复提起但不细写）
- 人物体质的暗示（如黛玉容易致病）
- 贯穿全文的意象和象征

### 操作要点
1. **提前规划伏线**：在 outline 阶段就确定全书的伏线地图
2. **自然植入**：伏线要融入日常描写中，不能突兀
3. **渐进揭示**：一条伏线从埋下到兑现，中间至少再出现 1-2 次
4. **记录追踪**：每条伏线记录在 `伏笔/` 状态文件中，明确初始埋入、中途提示、最终兑现的位置
5. **分层次**：
   - 大伏线（跨卷）：影响全书走向的伏笔
   - 小伏线（跨章）：影响一卷或数章的伏笔
   - 微伏线（单章内）：在单章内埋设和兑现的伏笔

### 伏线状态

| 状态 | 含义 |
|------|------|
| `planned` | 已规划，尚未埋入 |
| `planted` | 已埋入某章 |
| `reminded` | 已在中途提示 |
| `paid_off` | 已兑现 |
| `abandoned` | 已废弃（记录原因） |

### 使用自查
- 这条伏线埋入时是否自然（不会让读者觉得刻意）？
- 中途是否提示过（避免读者遗忘）？
- 兑现时是否与埋入时呼应？

## 四、春秋笔法（史笔）

> 含而不露的评判。用看似客观中立的叙述传达批判或褒贬。

### 核心原理
- 只记录事实，让读者自己判断
- 看似中立的描述中隐含作者态度
- 通过选材（写什么不写什么）传达评判

### 适用场景
- 描写权势集团的行为（如贾政轻松为贾雨村谋官）
- 社会批判（如官场腐败、人情世故）
- 人物评判：不直接说好坏，通过他的行为和他人反应来呈现

### 操作要点
1. 叙述保持表面的客观中性
2. 通过"结果"来暗示"原因"的不合理
3. 用细节呈现而非直接评判
4. 让读者自己去"发现"问题，而不是被告知

### 使用自查
- 如果去掉直接评判的句子，读者还能感受到作者的态度吗？
- 是否在用事实说话，而不是在说教？

## 五、白描

> 用最朴素、最不经意的方式呈现画面。只用可感知的细节，不修饰不渲染。

### 核心原理
- 五感描写（看、听、闻、触、味）代替抽象概括
- 具体动作代替情绪标签
- 朴素的语言产生最强的感染力
- 大场面用群体白描产生画面感（如湘云喷茶、黛玉笑岔气）

### 适用场景
- 群体场景（多人各具特色反应的场面）
- 日常生活的细腻描写
- 情绪传达（不写情绪却让读者感受到情绪）
- 人物出场和退场

### 操作要点
1. 只写看到、听到、摸到的，不写感觉到的
2. 每个人物的动作要有辨识度（湘云的喷茶 vs 黛玉的笑岔气 vs 宝玉的滚到怀里）
3. 具体动作 + 人物特征 = 个性化白描
4. 白描群像：每个人物的反应不同，通过不同反应刻画人物性格

### 使用自查
- 这个段落中，读者能不能"看到"画面？
- 有没有直接用情绪词（"高兴""悲伤"）代替具体的动作和细节？
- 群体场景中，每个人物的反应是否有辨识度？

## 六、层峦迭翠 / 两山对峙

> 在不同位置处理类似题材，既不着力也不重复。对比、对照无处不在。

### 核心原理
- 相似情境的对比产生意义
- 平行结构的互文效果
- 两次相似的选择反映人物变化

### 适用场景
- 两段相似情境产生对照（如两次葬礼、两次重逢、两次离别）
- 回目对仗（如林黛玉焚稿断痴情 / 薛宝钗出闺成大礼）
- 人物镜像（两个人物互为对照）
- 前后呼应的情节结构

### 操作要点
1. 在 outline 中标注需要进行对照的章节
2. 相似情境中写出不同的细节：同样的场景，不同的结果
3. 让人物在相似处境中做出不同选择，展示人物成长或沦落
4. 对照不限于情节，也包括意象、对话、场景

### 使用自查
- 前后两个相似场景，读者能否感受到"变化"？
- 对照是否过于直白（好的对照应该让读者自己发现）？

## 七、反衬法

> 言在此而意在彼。表面写的意思和真正的意思相反或不同。

### 核心原理
- 似贬实褒（如宝玉的"西江月"判词）
- 似褒实贬
- 喜景写悲（以乐景写哀情）
- 表面的言行与真实的想法相反

### 适用场景
- 引入主角时的"反写"（看起来不好实际上是好的）
- 他人眼中的主角和真实的主角不同
- 在喜庆场景中写悲剧预感
- 角色的口是心非（说反话）

### 操作要点
1. 确定"表面"和"内里"之间的张力
2. 表面写得越充分，内里的反转越有力
3. 不是让角色说谎，而是让角色说的话有双重解读

### 使用自查
- 是否有人物说了反话或做了表面和内心不一致的事？
- 场景氛围和实际内容是否有反差（喜景写悲 / 悲景写喜）？

## 八、避繁文法

> 对反复出现的行为不做过细描写，通过偶尔的侧面交代来保持读者的认知。

### 核心原理
- 反复写同样的内容会让读者疲劳
- 通过他人所见进行"一句话交代"，省略重复描写
- 保持读者的认知但不消耗读者的注意力

### 适用场景
- 人物习惯性行为（如黛玉常哭，只偶尔通过紫鹃的观察交代）
- 重复性事件（不每次都详细写，而是通过"后来""一年一月"来概括）
- 背景信息更新（人物状态的渐进变化）

### 操作要点
1. 第一次发生时详细描写，建立读者认知
2. 后续通过侧面交代（"素日知道""看惯了"）来维持认知
3. 用"后来""谁知""竟"等词自然过渡
4. 只有当习惯发生变化时才再次详细描写

### 使用自查
- 是否每次都在详细写重复性行为？是否可以改用侧面交代？
- 侧面交代时是否保持了读者对这件事的认知？

## 九、繁章波折法

> 在短小段落内制造多重波折，层层递进，一波三折。

### 核心原理
- 短小的场景也充满戏剧张力
- 对话中的不断转折制造阅读快感
- 每一句话都可能改变场景的情绪和方向

### 适用场景
- 情感冲突场景（如黛玉撞见宝玉宝钗在一起时说"我来得不巧了"）
- 对话交锋（每一句都有转折和张力）
- 关键场景的渐进升级

### 操作要点
1. 确定场景的情绪起点和终点
2. 规划中间的情绪波动（至少 3 次转折）
3. 每次转折由对话或动作触发
4. 转折不宜过长，1-3 句话即可

### 使用自查
- 短短一个场景中，情绪走向是否有至少 2-3 次变化？
- 每次转折是否推动了场景的戏剧张力？
- 转折是否有铺垫（不是突兀地转折）？

## 十、暗透法

> 让一个人物说出另一个人物心里想的事。通过他人之口揭示潜藏的心理活动。

### 核心原理
- 不直接写 A 的心理，让 B 替 A 说出来
- 增加叙事的多层次感
- 通过旁人的观察来揭示主角的内心世界
- A 心里想的 → B 嘴上说的 → 读者悟到的 → A 的反应

### 适用场景
- 情感线索的暗示（如莺儿说宝玉的玉和宝钗的锁是一对）
- 不便于直接心理描写时的替代手法（第三人称有限视角）
- 通过仆从、朋友之言揭示主角隐藏的心思

### 操作要点
1. 先确定 A 心里想的是什么
2. 选择谁替 A 说出来（这个人要天然和 A 亲近或关注 A）
3. B 说出后，A 的反应是重点：否认、沉默、转移话题、脸红
4. 不要写得太直白，让读者自己去联系

### 使用自查
- 被说出来的"心里话"是否足够微妙（不是直白的心声）？
- B 说出后，A 的反应是否有层次？
