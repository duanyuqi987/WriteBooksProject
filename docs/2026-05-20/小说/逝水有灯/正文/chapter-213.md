# 第213章 南方出租屋之后

所属部卷：第五部：逝水有灯
时间：第二天清晨
地点：南方城市出租屋

## 诗歌插页 P213：原创仿宋小令

雨未歇，闹钟催。
旧照合成灰。
地铁人潮向早归。
心事让晨辉。

第二天早上，陆允川被闹钟叫醒时，电脑还开着。屏幕进入睡眠，鼠标一动，旧照片文件夹重新亮出来。他盯了两秒，先把昨晚没接满的水倒掉，又把盆放回水龙头下面。滴答声很准，像另一个闹钟。

他没有再翻照片。洗漱、换衬衫、找工牌、确认地铁路线，这些事一件接一件，把昨夜的情绪挤到墙角。楼下早餐店排着队，蒸汽从窗口往外涌。陆允川买了一个鸡蛋和一杯豆浆，边走边拆吸管，差点把豆浆洒到文件袋上。

公司电梯里人很多，大家低头看手机。有人讨论绩效，有人问周末团建，没人知道他昨晚在出租屋里看见了几年前的苏晚枝和夏知遥。成年后的好处和坏处都在这里：世界不再围着一段感情转，也不会因为你终于不疼了就给你掌声。

中午，他把“旧日”文件夹同步到硬盘。进度条从百分之一慢慢走到百分之百。他看着那个绿色完成提示，忽然觉得自己并不是把过去封存，而是给它找了一个不用日日占用桌面的地方。

下班时雨停了。街面还湿，路灯映出一小片亮。他没有绕路去喝酒，也没有给旧人发消息，只在便利店买了新的水龙头垫圈。
