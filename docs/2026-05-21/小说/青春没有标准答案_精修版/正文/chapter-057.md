# 第057章 高二楼的反击



时间：2009年一月上旬，大雪继续下的下午



地点：两栋教学楼之间、食堂侧门



卷次：第三卷：雪落在两栋楼之间



2009年一月上旬，大雪继续下的下午，两栋教学楼之间、食堂侧门，砸在十班值日牌旁边的雪团先把这一天按进了十班的日常里。高二楼反击更猛，雪团越过空地，十班值日牌被砸歪。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

梁一帆不服，赵远航想带人绕到食堂侧门，段璟第一次主动拦住同学。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看砸在十班值日牌旁边的雪团，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

梁一帆也在这一天被推到前面。梁一帆的特点是爱运动，反应快，雪仗里第一个按不住手。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“他们砸咱们牌子。”梁一帆说。

“从食堂那边绕过去。”赵远航说。

“别绕，绕过去就真成打架了。”段璟说。

“段璟说得对。”常俊凡说。

“谁再下楼，下午自习全站后面。”陶建国说。

高二楼的反击更有准头，几个雪球从二楼斜着飞下来，落点都在高一楼门前那块最热闹的地方。

十班值日牌被砸歪时，梁一帆脸色一下变了，像被砸中的不是牌子而是班级脸面。

赵远航说可以从食堂侧门绕过去，高宇马上说我知道那条路，几个人眼睛都亮了一下。

段璟拦住他们时声音并不大，但他站在台阶中间，没有让开。

常俊凡看了他一眼，马上接话，说现在绕过去就不是雪仗，是找事。

陶建国赶到时，鞋底带着雪水。他没有先骂，只让所有人把手里的雪放下。

高二楼上还有人不服气，隔着空地喊了几句，被他们自己的老师吼回去。

那片空地忽然安静下来，只有雪还在下，像刚才那些喊声并不是它弄出来的。

高二楼的反击没有悬在空处。值日牌旁边散开的雪渣，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，高二楼的反击带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

陶建国进门时不一定立刻说话，鞋底在水泥地上一停，高二楼的反击带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在梁一帆快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。高二楼的反击越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。砸在十班值日牌旁边的雪团让他想说话，又逼他先低头看题。

梁一帆被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，高二楼的反击留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。梁一帆路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为高二楼的反击翻身，谁还在被窝里背单词，旁边人都听得见。

高二楼的喊声压过来，高一楼这边也不肯立刻低头，楼道里全是湿鞋印。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理高二楼的反击里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：砸在十班值日牌旁边的雪团、一只冻红的手、一句老师点名、一块还没扫干净的雪。

食堂侧门那条路不太好走。两旁是冬青丛，雪盖了厚厚一层，踩上去脚会陷进去半截。赵远航在前面探路，高宇跟在后面，两个人的鞋底都浸了雪水。梁一帆站在教学楼拐角往上看，二楼窗口探出几颗脑袋，高二的人还在找下一个靶子。

"他们还没完。"梁一帆说，声音比他自己预期的要低。

段璟抬手拦住赵远航。这个动作他后来想了很久——不是因为当时想清楚了，而是因为他没想清楚就做了。

"你干嘛？"赵远航停住，一只脚还在雪里。

"绕过去就变了。"段璟说，"现在还是雪仗，绕到侧门就不是了。"

赵远航看了看他，又看了看梁一帆。梁一帆没说话，嘴唇抿成一条线。冬青丛里的雪被风刮下来，落在几个人肩上，没有人去拍。

常俊凡从走廊那边跟过来，校服领子竖着，哈出的气在面前散成一小团白雾。她没有跑到跟前，隔着三四步就停了。

"他说得对。"她只说了这四个字，然后看着梁一帆。

梁一帆不是能被一句话劝住的人。但他能看懂人的立场。常俊凡很少替人站队，所以当她替段璟的这句话站队时，分量比她说出来的那四个字重得多。

"行。"梁一帆把手里攥着的雪球往冬青丛里一甩，雪球打在叶子上，簌簌落了一地碎雪。

赵远航脚从雪里拔出来，在水泥地上跺了两下："那牌子被砸了怎么办？咱们十班的牌子，就让人家这么砸？"

"牌子可以扶正。"常俊凡说，"处分扶不了。"

沈曼青已经把歪掉的值日牌扶起来了。牌子上沾了雪水，铝皮边缘翘起一小块。她用袖口擦了擦，发现擦不干净，又把牌子转了个方向，让砸瘪的一侧朝墙。

"这样看不出来。"她说。

陶建国赶到时，食堂侧门那条路上的雪已经被踩实了好几处。他没有问路是谁踩的，只看了一眼值日牌。

"高二的，还是你们自己碰的？"

"砸的。"周明澈说，"不是我们自己碰的。"

"砸的值日牌。"陶建国把"值日牌"三个字咬得很重，"哪个年级的值日牌被砸了，就是哪个年级的脸被人打了。你们打算怎么把脸捡起来？"

梁一帆把脸转开。他本来想说"打回去"，但这话在陶建国面前说不出口。

"捡起来不是靠打赢。"陶建国说，"是靠让人家用别的东西记住你。"

他没有解释"别的东西"是什么。但段璟后来想，陶建国在那一刻已经把话说完了，只是当时没人能听懂。

走廊里的脚步声渐渐散掉。有人去办公室搬卷子，有人去小卖部买笔芯，有人在楼道拐角小声议论高二楼哪些人扔得最准。

付小川从走廊那头过来，手里提着两个暖水瓶，一个灰的一个红的。他自己的和常俊凡的——常俊凡刚才去帮沈曼青挂牌子了，把瓶子落在了水房门口。

"段璟，"他把灰瓶往段璟怀里一塞，"你帮人家提了好几次热水了，这次我替你先提了。别谢我，谢我就见外了。"

"你嘴能不能歇一会儿。"

"歇了谁跟你说常俊凡今天的围巾是什么颜色？"付小川凑过来，"浅灰的，带两道蓝杠。跟你的那块灰手帕是一个色系——别以为我没看见你那块手帕。"

段璟把手里的暖水瓶往地上一顿，热水在瓶里晃了一下，发出闷闷的一声。

付小川笑了，笑容里没有恶意，纯粹是发现了一件他认为早就该被发现的事。"行行行我不说了。但你那个眼神，讲真的，全班就你自己觉得别人看不出来。"

他说完就走了，红瓶还拎在手里，走路时暖水瓶一晃一晃，瓶底在走廊地面上磕出有一搭没一搭的声音。

段璟站在原地，把灰瓶拎稳了。瓶身上用白油漆写着"常俊凡"三个字，字体比别人的小一圈，笔画很细，像怕浪费油漆似的。

他忽然想起上次帮她打水时，她说了声谢谢，没有抬头。那次他差点问了一句别的什么，最后说出口的却是"水温不够，再烧一会儿"。段璟在心里骂了自己一句：下次能不能至少说一句比水温有用的话。

但这个"下次"什么时候来，他也不知道。他只是每次碰到她的时候，都觉得自己像在做一道自己还没学到的题——明知道解题方法应该不难，可就是找不到那个辅助线该往哪儿画。

段璟没有跟着散。他还站在食堂侧门旁边，脚边是踩得乱七八糟的雪。常俊凡从走廊那头走回来，手里多了一个搪瓷杯。杯子冒着热气，金银花茶的味道在冷空气里格外清楚。

"你没去搬卷子。"段璟说。

"先喝口热水。"她把杯子往他那边推了一寸，没有递到手上。

段璟接过来喝了一口。金银花茶不甜，微苦，但热。他想起小时候在城关镇外婆家，每次感冒外婆就泡金银花茶，说"苦了就对了，苦的才管用"。

"刚才，"他顿了顿，"你说'他说得对'——"

"因为你说得对。"常俊凡把杯子拿回去，自己也喝了一口，"不是因为别的。"

这句话里面有很多层意思。段璟只敢拆到第一层。

下午的数学课，黑板上有一块反光。陶建国写到立体几何的辅助线时，粉笔刚好打在光区里，后排看不清。有人举手说"老师反光"，陶建国停下来，把黑板擦翻了个面，用背面压着光区又写了一遍。

段璟盯着黑板，脑子里却在算另一件事：刚才他拦赵远航时，常俊凡站在他身后三四步。那三四步的距离，比他们两个座位之间的直线距离要近。

宋乔在下课时凑到常俊凡桌前，把下午的事重新过了一遍："梁一帆真就听你的了？"

"他听的是道理。"

"道理谁不会讲？赵远航讲了半天他也没听。"

常俊凡把错题本合上："话和话不一样。有理的话和占理的话也不一样。"

宋乔托着下巴想了一会儿："所以段璟说那句的时候占理？"

"他是讲道理。"常俊凡顿了一下，"我只是帮他把道理变成大家能听懂的。"

宋乔没有追问，但她嘴角弯了一下。她发现自己正在学常俊凡的另一种本事：把一件本来很好看穿的事说成不好看穿的。

晚自习之前，陶建国让周明澈把值日牌挂回去。牌子挂在门边，被砸瘪的一侧朝墙，正面看和原来一样。梁一帆从旁边经过时，在牌子前面停了半秒。

"下次呢？"他像是问自己，又像是问刚好经过的段璟。

"下次再说下次的。"段璟说，"今天的事还没完。"

确实还没完。晚自习第二节，王启民进来通知：汶川地震的捐款还在继续，各班自愿，捐多少记在班级名单上。十班上次捐了四百多，这次马立群建议大家再凑一点，至少别比上次少。

常俊凡从口袋里摸出五块钱，放在桌面右上角。段璟看见她把钱折了两折，是早上从家里带的早点钱省下来的。

"你早上没吃饭？"段璟问。

"吃了。少买了一个包子。"

段璟从自己的饭卡余额里算了算，也掏了五块。

付小川从后排探过来，往桌上放了一个钢镚，又放了一个，啪啪两声。然后他拿起段璟刚放的五块钱，用手指弹了一下："这个月你是不是又得找我借饭卡？"

"不借。"

"那你怎么吃饭？"

"少吃一个包子。"

付小川笑了，把钱放回去，又多加了一个钢镚。

捐款名单从第一排传到最后一排。传到梁一帆时，他写了名字，后面跟了十块。那个数字在名单里不算最大，但段璟注意到了。梁一帆写字时眉头皱着，像在跟谁较劲。

下晚自习后，宿舍楼道里的日光灯灭了一半。段璟端着搪瓷杯去水房接热水，在走廊上看见常俊凡和宋乔一起往女生宿舍那边走。她们的背影被走廊尽头的光切成两个剪影，宋乔在说着什么，常俊凡偶尔应一声。声音传不过来，但段璟看见常俊凡抬手把宋乔的围巾往上拉了一下。

水房里人不多。热水器的红灯亮着，出水口嘶嘶地响。段璟把搪瓷杯接满，热气从杯口往上涌，他低头对着杯口哈了一口气，看见自己的呼吸和热水汽混在一起。

隔壁宿舍有人在用收音机听新闻。信号不太稳，声音断断续续的，隐约能听出是关于奥运总结的内容。北京奥运已经过去五个月了，但封丘一中还有人把"同一个世界同一个梦想"写在作文里。段璟想，梦想不梦想的不好说，同一个教室倒是真的。

他端着杯子回到宿舍时，梁一帆正趴在床上，用被子蒙着头。付小川坐在下铺剪指甲，剪子在手指间咔咔响。

"他睡了？"段璟小声问。

"没睡。"付小川说，"在数今天被高二砸了多少个雪球。"

"我没在数。"梁一帆的声音从被子里闷闷地传出来，"我在想明天怎么把值日牌钉死。"

"用钉子？"付小川停下剪子，"你认真的？"

"钉子不行，墙是砖的。"段璟说，"明天找根铁丝缠一圈。"

梁一帆从被子里探出半个头："你会缠？"

"试试。"

这不是什么大事，但梁一帆把头缩回被子里之前说了声"行"。那个"行"字很短，听起来却比白天的雪球和喊声要扎实。

熄灯后，宿舍陷入一种拥塞的安静。上铺有人翻身，下铺有人咳嗽。段璟闭着眼睛，把今天的事从头过了一遍：值日牌被砸、赵远航想绕食堂侧门、他伸手拦住了、常俊凡说"他说得对"、陶建国说"让人家用别的东西记住你"。

"别的东西"到底是什么？段璟想不出来。他只是觉得，今天拦住的那一步，比任何时候往前走的一步都更沉。

远处操场边的路灯还亮着一盏。光从窗帘缝隙里挤进来，在天花板上画了一道细细的白线。段璟盯着那道线，迷迷糊糊地想到：封丘一中的冬天，大概就是这样一层一层地冷下来，然后在某个你不知道的时刻，突然回暖一点。

高二楼的反击之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

梁一帆没有因此变成特别醒目的人，可在段璟眼里，梁一帆从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，高二和高一的玩笑开始有了年级对抗的味道，老师们往楼下赶。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
