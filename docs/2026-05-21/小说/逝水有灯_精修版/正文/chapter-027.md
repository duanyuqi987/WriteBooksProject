# 第027章 第一次说话

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：教室前门和讲台之间

## 诗歌插页 P027：原创仿唐绝句

一语初逢半日迟，
檐前谁记旧年时。
少年不解平常意，
却把闲声唱作诗。

事情真正发生时，比陆允川在草稿纸上排练的任何场景都更短。那天下午课间，物理课代表抱着一摞作业本往教室里走，在门口被人撞了一下——本子从手上滑出去，散了一地。陆允川刚好从教室出来，弯腰帮捡。他蹲下去捡起最靠近自己的两本，抬头时发现对面蹲下的是夏知遥。她也同时在捡。她的动作比他快——已经捡了三本摞在腿上，正伸手去够第四本。

她的手指尖快碰到那本作业本的封面时，和他的手碰上了。不是指尖碰指尖——只是她的手背轻轻蹭到了他的大拇指外侧，凉凉的，可能因为刚从水房出来洗过手。两个人同时把手收回去，又同时说了一句"你捡"。然后停了半秒——都不知道谁该继续捡。

陆允川先说了一句话，声音不大，像把一块石子轻轻放到地面："你捡吧。"她把那本捡起来，摞进自己手里，站起来，说"谢谢"。两个字很平，没有任何多余的语气——就像对任何一个帮她捡东西的同学那样。陆允川也站起来，把自己捡的两本放到她摞的顶上，说了声"没事"。然后她转身走回座位把本子放下，去水房了——她的水杯还放在水房的窗台上。

这是他们之间第一次说话。完整的对话只有三个词——"你捡吧""谢谢""没事"。加起来六个字，时间不超过四秒钟。但陆允川把这六个字拆成了无数个片段：她的"谢谢"里有没有笑？没有，但声音比平时软了半度。他的"没事"音量够不够？够，不过听上去像是先吞了半个音节才说出口的。

程砚在教室里目睹了整个过程。等陆允川回座位后，他说了一句话："六个字——你捡吧、谢谢、没事。你知道你今天的外交发言用了多少字吗？六。你知道你昨天草稿纸上排练的对话加起来多少字吗？大概两百。"陆允川把课本翻开："你数过了？""我没数你的草稿纸，但我数过对方——她说完'谢谢'就转身了。你觉得她会在意那六字台词吗？她不会。所以你也不用。"程砚继续说，"但恭喜你，你终于跟她说话了。从今天开始你算是在她的世界里正式注册了——用六个字。"

那天晚自习前，陆允川在日记本上写了很长一段，措辞极其隆重——用了"命运""转折""奇迹"这些词。写完以后看了一遍，把这一页撕掉了。重新翻开新的一页，只写了一行字：今天下午课间，帮她捡了三本物理作业本。她说谢谢，我说没事。他把这一页合上，不再往下写。那六个字不需要续写任何句子——它们本身就是完整的。
