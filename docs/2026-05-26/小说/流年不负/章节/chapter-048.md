# 第048章 操场风波

时间：2012年3月

地点：操场

卷次：卷三：此心藏久

三月中旬。春天刚来。

操场边的柳树发了新芽。嫩绿色的。风一吹就摆。有几个高一女生在柳树下面自拍，比了一个V字手。手机举得高高的，脸往左边歪。

下午体育课。高二三班。

体育老师让跑三圈热身后自由活动。顾临风跑完了三圈——第五道。他在第五道跑。习惯了。第一圈不快，第二圈匀速，第三圈冲刺了一下。

然后他走到操场边，坐在台阶上。

赵一鸣去打篮球了。他叫顾临风一起去。"三缺一。""不打。""你每次都这样。""嗯。"

他一个人坐在台阶上。旁边放了一瓶水。农夫山泉。一块五一瓶。喝了一口。瓶盖没拧紧，水洒了几滴在裤子上。他没管。

操场上有人在踢足球。门柱是铁的，球踢上去咣一声。有人在练跳远。跑几步。起跳。落地。沙坑里的沙子溅起来。旁边有女生在体前屈——双手抱着脚踝，头往膝盖上压。

他在看这些人。目光从足球场移到沙坑。从沙坑移到跑道。

然后他看到了苏晚晴。

她在操场的另一边。第四道跑道上。她旁边有一个男生。

两个人并排走着。很近。大概隔了半只胳膊的距离。

男生很高。比顾临风高了半个头。短头发。穿的是白色T恤。校服外套搭在胳膊上。他手里拿着她的水杯——那个透明的杯子。他的另一只手在指操场后面的水塔。大概在说水塔有多高。

苏晚晴仰头顺着他的手看水塔。然后笑了一下。

她的笑。那种脸上先笑眼角再跟上来的笑。

顾临风坐在台阶上。手里攥着农夫山泉。瓶身被捏得凹进去了。

他没有站起来。也没有往那边走。就坐在那里。看着第四道跑道上的两个人。他们在走得很慢。走了大概一百米。然后转弯。继续走。一边走一边聊天。

那个男生帮她把水杯递给她。她接过去喝了一口。然后拧上盖子。男生又把水杯接回去了。很自然。没有客气。没有谢谢。就像这件事已经做过了很多次。

他们在操场上走了三圈。

顾临风坐在台阶上。手里那瓶水一直没喝。阳光照在他身上。三月的阳光不热，但晒久了也有点烫。他的脖子后面出汗了。

下课铃响了。体育老师吹哨集合。各回各班。

苏晚晴跟那个男生说了两句话。然后往女生队列走。那个男生往另一个方向走了——不是高三三班的队列。是另一个班。

顾临风站起来。把水瓶扔进垃圾桶。水还是满的。溅了一下。

他走路没有看前面。差点撞到篮球架上。赵一鸣在后面喊他。"你怎么了。走路不看路。""没事。"

他回了教室。坐在座位上。假装看物理书。翻到第二十六页——电磁学。右手定则。他低头看着那个图。电流方向。磁场方向。手心方向。看了五分多钟。那一页没翻过去。

他的眼睛往左前方偏了一下——控制住了。偏到一半收回来。盯着课本。

苏晚晴从前门进来。坐到自己的位置上。她跟许若琳聊了几句。说了什么他听不清楚。然后她开始做化学题。笔在纸上走着。她的小指还是翘着的。跟第一天他看到的一模一样。

他没有像往常那样用余光看她。

他故意不看。

眼睛只盯着物理书。右手翻了一页——第二十七页。左手在桌子底下攥成拳头。指甲掐在手心里。掐了四个印。

晚自习。他做了一张数学卷子。

做了两个小时。选择题。填空题。计算题。证明题。全部做完了。最后一道大题有点超纲——他想了想。跳过。然后翻到下一张。继续做。

这套卷子。他一般做一个半小时。今天做了两个小时。

做完之后。他没有对答案。把卷子折起来夹进练习册里。笔放回铅笔盒。啪一声盖上。

赵一鸣在旁边看了他一眼。

"物理作业你做了没。"

"做完了。"

"借我抄——"

他把物理作业推到赵一鸣桌上。赵一鸣翻开。抄了两行。又停下来。

"你抄错题了——这是昨天的。"

顾临风看了一眼。把作业拿回来。从桌肚里翻出今天的作业。推到赵一鸣面前。全程没说一个字。

赵一鸣没再问什么。低下头抄作业。抄完，把作业本还给他。在作业本的最后一页，赵一鸣用圆珠笔写了两个字。

"有事。"

顾临风看到那两个字。把作业本翻到下一页。盖住了。

晚自习结束。回宿舍的路上。赵一鸣没有再说任何话。两个人在路灯下走。影子一长一短。飞虫还在灯泡边绕。

快到宿舍楼下的时候。顾临风先开口了。

"明天早上别叫我。"

"为啥。"

"想多睡会儿。"

"你从来不说这种话。"

顾临风没有回答。推开宿舍楼的门。上楼。楼梯间那个灯修好了——新换的灯泡。白炽灯。很亮。照得台阶上一清二楚。

他进门。洗漱。躺下。熄灯。

手电筒没有打开。

他在黑暗中翻了个身。上铺的板子没响——他翻身很轻。外面有人在唱歌。周杰伦的《安静》。隔壁宿舍。唱了一半停了。有人在骂"还他妈不睡觉——"。然后安静了。

窗帘那条缝。月光漏进来。他盯着天花板上的荧光星星——上届留下来的。去年还能发一点光。今年彻底不亮了。就是两张绿色的塑料片贴在天花板上。

他没有想什么。

脑子是空的。

没有她。没有操场。没有水杯。没有白色T恤的男生。

但他也睡不着。

凌晨一点多。他从床上坐起来。走到窗前。把窗帘拉开了一点。窗外是操场——从这个角度能看到操场的一角。第四道的跑道在月光下是一条灰色的线。草皮上有一块一块暗的痕迹。去年秋天拔草的时候留下来的——没拔干净。新草还没长出来。

那个男生帮她拿水杯的样子。

很自然。

没有谢谢。

他放下窗帘。坐回床上。手在被子外面放了一会儿。凉的。春寒还很明显。三月晚上的青阳还是挺冷。

他拿起枕头底下的语文课本。翻到夹着《暗恋》的那一页。手电筒没开。黑暗中看不清字。但他背得出来每一个字。三载同窗未敢言。千回目送只偷痕。

他把书合上。放回去。

躺下。

第二天早晨。他没有吃早饭。直接去了教室。坐到座位上。翻开英语书。

苏晚晴来的时候。他没有抬头。

她从前面经过。马尾。浅绿色外套——春天换的新外套。书包拉链上那个小鱼挂件还在。晃了两下。她的脚步声在讲台那里停顿了一下——好像往他的方向看了一眼。

他没有抬头。

她走到自己的座位上。坐下。翻书。背单词。

整整一天。他没有往窗边看。

整整一天。
