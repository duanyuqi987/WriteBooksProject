# 第114章 黑板报边角



时间：2010年七月上旬，期末倒计时前



地点：教室后墙、英语办公室门口



卷次：第五卷：未来在黑板上倒计时



2010年七月上旬，期末倒计时前，教室后墙、英语办公室门口，刘青岚抄在黑板报边角的一句英文世博标语把2010年的春夏推到十班面前。刘青岚把世博英文标语抄到黑板报边角，被马立群拿来讲英语表达。

她英语好，总分却上不去，觉得自己的长处没有用；常俊凡让她别用总分抹掉进步。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

刘青岚也在这段日子里显出自己的位置。英语听力好，说话轻，常在别人情绪后面补一句温柔的话。群像不是名单，群像是每个人都有自己的难处。

“英语好有什么用，总分还是不够。”刘青岚说。

“有用。长处不是万能，但它能带你往上走。”马立群说。

“先保住长处，再补短板。”常俊凡说。

“我愿意用物理换一点英语。”段璟说。

“你想得倒美。”刘青岚说。

黑板报边角总容易翘，透明胶贴上去，过一会儿又被风掀起一点。

世博专题、英语标语、竞赛名单和暑假倒计时挤在后墙上，像十班这几个月的缩影。

有人觉得黑板报耽误复习，有人觉得后墙不能太难看。争到最后，还是要有人拿尺子压住纸边。

常俊凡负责把文字压缩，周明澈负责看版面，宋乔负责把起哄的人骂回座位。

段璟拿透明胶时，指尖被胶带黏了一下。很小的疼，却让他从走神里回到眼前。

后墙下面铺着旧报纸，粉笔灰、红纸屑和剪坏的边角混在一起。

老师路过时没有立刻评价，只说别贴太满，留白也是版面的一部分。

黑板报边角终于压平时，晚自习铃也响了，所有人又被卷子拉回去。

马立群这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

刘青岚被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

罗雨桐用尺子压住黑板报边角，讲台粉笔盒里只剩半截白粉笔。十班的热闹还在，只是每个人都比去年更会藏心事。

赵远航剪掉多出来的一截卡纸，后墙的错题纸被风掀起一角。老师从门口经过，原本散开的声音立刻往回收。

陈柏言把透明胶重新贴平，竞赛名单的纸角已经微微卷起。这样的细节不会被写进红榜，却会留在很多人的夏天里。

张若楠擦掉后墙旁边的粉笔灰，新华书店塑料袋在手里沙沙响。这点动作很小，却把那天的压力落到了实处。

刘青岚用尺子压住黑板报边角，桌洞里的练习册露出红色封皮。旁边的人没有追问，翻书声把难堪遮住。

王子越剪掉多出来的一截卡纸，树莓摊旁边的纸盒被太阳晒软。常俊凡看见了，没有替谁下结论，只把题号重新写清楚。

黑板报边角之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

黑板报边角的一句英文，让偏科不再只是短板，也成了一个人还能抓住的长处。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，刘青岚抄在黑板报边角的一句英文世博标语还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
