# 第141章 她说别追了之前

所属部卷：第三部：寒风里的花
时间：六月下旬
地点：图书馆东门台阶

## 诗歌插页 P141：原创现代诗

我把台阶水瓶放在黄昏里，
不让它替任何人回答。
风从校园的一侧经过，
带走一句没有说稳的话。
后来我学会把手松开，
让灯照着路，也照着空出来的地方。

六月下旬的一个傍晚，陆允川收到了一封电子邮件。发件人是夏知遥。自从高考结束以后，他们就没有联系过。她去了北方一所大学读金融，QQ不再上线，朋友圈三天可见。最后一次聊天是高中毕业典礼那天，她在走廊里跟他说"以后常联系"，然后就上了她爸爸的车。

邮件标题是："好久不见。看到你发的诗了。"

正文很短：

"陆允川：

我们学校文学社团转发了你们诗社的公众号推送。我看到了一首署名LX（是你吧？）的诗，写得比以前好太多了。我记得你高中的时候写《风铃》，秦老师让你在班上念。你念的时候声音抖，大家都听出了。

你现在还在写诗吗？我偶尔写。但写得不好。金融专业不教这个。

我的大学同学问我高中是什么样子的。我讲了你的名字。他们问你是不是高中喜欢我。我说可能是吧。他们说那你现在有女朋友了吗。我说不知道。

有吗？

夏知遥"

陆允川在图书馆东门台阶上坐着看完了这封邮件。台阶是水磨石的，被无数学生的鞋底磨得发亮。旁边搁着一只水瓶——他刚打完球回来，水瓶壁上还挂着水珠。

他把邮件读了三遍。第一遍，他注意到她的写作风格变了——高中时候她写的纸条都不加标点，像在喘气。现在邮件加了标点，句子完整，语感不同了。三年时间改变了她的断句方式。第二遍，他注意到了那句"他们问你是不是高中喜欢我。我说可能是吧"——她把他的暗恋概括为"可能是吧"。这个概括让他觉得：当年他觉得沉重到不敢开口的事，在对方那里只是一个不确定的推测。第三遍，他注意到了最后一行："有吗？"

两个字。像踩水花一样轻。

他没有立刻回复。他把水瓶拧开喝了一口。水温了。六月的下午温度把冰水变成了温水。

程砚刚好打来电话。

"干嘛呢？"

"在图书馆台阶上坐着。"

"又发呆？"

"收到一封邮件。夏知遥的。她问我有没有女朋友。"

"你怎么想的？"

"不是怎么想——是她加了一句话：当年你是不是喜欢我。我说不是——我说可能是吧。她说的。"

"所以你现在知道她当年其实也感觉到了。只是不确定。"

"对。不确定。"

电话里安静了一会儿。程砚的声音把安静打破："那你现在要告诉她你有苏晚枝了。然后——我觉得就完了。不需要再谈以前。以前那些——已经解决了。"

"什么东西解决了？"

"你的状态。你高中的时候是'暗恋但不敢说'。你现在不是了。你现在有苏晚枝。你跟苏晚枝在一起不是因为夏知遥不在——是因为你改变了。你不会再暗恋三年不说。这个改变是你自己的，跟夏知遥无关。跟她写不写这封邮件也无关。"

陆允川看着台阶前方正对的花坛。花坛里是橙色的万寿菊，开得很密。

"那我怎么回？"

"如实回。'有女朋友。叫苏晚枝。诗社的。祝你好。'不超过三行。"

"三行太短了。"

"够了。说多了会让她以为你还想聊。"

挂掉电话后，陆允川开始打字。打了几遍又删除。他发现自己不是在犹豫内容——是在犹豫语气。他想让语气听上去平静但不冷淡、直接但不伤人、真实但不八卦。最后他写了六行，比程砚建议的三行多了三行：

"夏知遥：

收到你的邮件很高兴。我有女朋友了，叫苏晚枝，是大学诗社的社长。她是一个会用《说文》解释人名字的人。

高中那些事——你猜得对。但那时候我不太会表达。现在已经会了。是她教我的。

祝你在北方一切都好。

陆允川"

他点下发送。然后站起来，拿起水瓶，走回宿舍。路上他把手机掏出来，把夏知遥的邮件转发给了苏晚枝。加了一句话："下午收到了这个。回了。想让你知道。"

苏晚枝十分钟后回："我看到了。你的回信写得不错——'是一个会用《说文》解释人名字的人'。这句介绍我喜欢。"

然后她又发了一条："她问你的高中——你说'已经会了'。是我教你的。这句话我没教过。是你自己学会的。这个功劳我不冒领。"

"你教了我直接说感受。"

"那是我教的祁野方法。你拿去用了。好用吗？"

"好用。"

"那就好。"
