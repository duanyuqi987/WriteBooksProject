# 第152章 南湖问答

所属部卷：第三部：寒风里的花
时间：七月初
地点：南湖河边

## 诗歌插页 P152：原创仿唐绝句

湖边旧椅半生纹，
一问低回到夜昏。
若把改过当凭据，
仍将他意锁吾门。

陆允川看着那三个词，先指了“道歉”。

“我道歉过。”

叶澄点头，“道歉是说你知道错了，不是要求别人马上舒服。”

他又指“改变”，“那改变呢？”

“改变是你之后少做什么，多做什么。不是你拿给她看的成绩单。”

陆允川被这句话堵住。他原本想说自己这几天没有打电话，也没有去找苏晚枝，可说出口之前就明白，那些事不能拿来讨赏。

叶澄把笔帽扣上，“你真正想问的是，改了以后她还会不会理你。”

陆允川低声说：“我不知道怎么接受没有结果。”

“那就先接受不知道。”叶澄说，“别急着把不知道变成她亏欠你。”

这句话很轻，却让他耳根发热。南湖的水面有小虫掠过，划出一道细线，很快消失。

陆允川问：“那我是不是很自私？”

叶澄没有马上回答。她把水杯拿起来喝了一口，才说：“你不是一直自私。但你难过的时候，会只看见自己。这就够让别人累了。”

他低下头。湖边长椅的漆皮翘起来，像一小片没有撕干净的旧纸。他终于没有再问“她会不会原谅我”。那个问题现在问出来，只会把压力换个方向送出去。
