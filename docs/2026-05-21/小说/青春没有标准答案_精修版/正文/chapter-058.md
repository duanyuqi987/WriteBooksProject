# 第058章 十班被点名



时间：2009年一月上旬，雪仗被叫停后



地点：高一十班教室和年级办公室门口



卷次：第三卷：雪落在两栋楼之间



2009年一月上旬，雪仗被叫停后，高一十班教室和年级办公室门口，黑板上被写下的十班两个字先把这一天按进了十班的日常里。王启民在年级会上点名十班，要求参与雪仗的学生写说明。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

罗雨桐画下雪仗场面，却被误会在幸灾乐祸；十班内部开始争谁先动手。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看黑板上被写下的十班两个字，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

罗雨桐也在这一天被推到前面。罗雨桐的特点是敏感细腻，爱画画，能把别人忽略的场面记下来。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“高一十班，名字我记住了。”王启民说。

“记住不一定是好事。”付小川说。

“我只是画下来，不是觉得好玩。”罗雨桐说。

“画下来也许比吵有用。”常俊凡说。

“下午每个人写清楚自己做了什么。”陶建国说。

下午第一节还没上，王启民已经在年级会上把高一十班点了出来。

十班两个字落在黑板上时，全班都不说话了，连付小川都把手从抽屉里拿了出来。

罗雨桐本来在草稿纸上画雪仗，画到一半才发现旁边人都在看她。

她把纸盖上，说我不是觉得好玩，我是怕以后大家只记得谁挨骂，不记得事情怎么变大的。

常俊凡说那就把过程写清楚。她这句话让教室里少了一点互相推责任的声音。

陶建国让每个人写说明，先写自己做了什么，再写别人做了什么。顺序不能反。

段璟在纸上写自己捏了雪团、扔了一个、拦了赵远航。他写到第三项时，忽然有点不自在。

原来一个人可以同时做错一点事，也做对一点事，班级里很多判断都没有那么干净。

十班被点名没有悬在空处。黑板上被写下的班级名字，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，十班被点名带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

王启民进门时不一定立刻说话，鞋底在水泥地上一停，十班被点名带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在罗雨桐快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。十班被点名越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。黑板上被写下的十班两个字让他想说话，又逼他先低头看题。

罗雨桐被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，十班被点名留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。罗雨桐路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为十班被点名翻身，谁还在被窝里背单词，旁边人都听得见。

被点名以后，楼前空地安静得很快，只有屋檐下的雪水一滴一滴往下落。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理十班被点名里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：黑板上被写下的十班两个字、一只冻红的手、一句老师点名、一块还没扫干净的雪。

段璟在说明纸上先写了"捏雪团一个"，又写了"扔雪团一个"，然后把笔放下了。第三项"拦赵远航"他不知道该不该写——写了好像在邀功，不写又确实做了。

前排有人把笔摔在桌上，啪的一声。是个平时不怎么说话的女生，她的纸还空着，不知从何写起。

罗雨桐的画夹在英语书里。画上是一个人弯腰捡雪的样子，旁边有歪歪扭扭的值日牌和二楼窗口的几颗脑袋。画得不精细，但一眼就能看出是哪一天。这张画本来只是她课间随手画的，结果被旁边的人看到，传了两圈，最后传到了王启民耳朵里。

"画是画得挺好。"王启民说，"但你们班现在最不需要的，就是把这事当成热闹看。"

罗雨桐的脸一下红了。她不是那种会当场顶嘴的人，只是在低头的时候把嘴唇咬得很紧。

常俊凡在下课后走到她桌前，没说话，先把罗雨桐桌上的铅笔灰用小刷子扫进手里。

"我不是觉得好玩。"罗雨桐终于开口，声音很小。

"我知道。"常俊凡把铅笔灰倒进垃圾桶，"你不是那种人。"

罗雨桐抬头看她。常俊凡已经转身走回自己座位了，步子不快，像什么都没发生。但她那三个字——"我知道"——比任何安慰都管用。罗雨桐后来想，被人看准了，原来是这种感觉。

但事情没有因为画解释清楚就过去。年级组办公室里，王启民把十班的名单摆在桌上，用红笔在参加雪仗的名字旁边各画了一个勾。十班被点名的事很快传遍整个年级，连小卖部的阿姨都听说了——"高一十班，雪仗打得最凶的那个班？"

付小川中午去打酱油时听见这话，回来学了一遍，把人笑倒一半。

"笑什么笑。"周明澈说，"明天就传到校长那儿了。"

"传到校长那儿也好。"付小川把酱油瓶放在窗台上，"至少校长知道咱们班不是只会做题。"

"你这种自信是哪来的？"

"天生的。"付小川说，然后被周明澈用一本练习册拍了后脑勺。

写说明这件事，全班都卡住了。卡住的不是不知道怎么写，是不知道怎么写得既诚实又不坑同学。陶建国要求每人写"先写自己做了什么，再写别人做了什么，顺序不能反"，这个要求看起来简单，真写的时候才发现难。

赵远航在纸上写了三行，又全部划掉。他旁边的男生凑过去看了一眼，被他用胳膊肘挡开了。

"你写了什么？"那个男生问。

"什么都没写。"赵远航说。他划掉的那三行是：我跟梁一帆扔了雪球。我也想去食堂侧门。段璟拦住了我。这三件事都是真的，但写在纸上就像在卖队友，哪怕被卖的队友做的事全班都知道。

梁一帆倒是写得很快。他的纸上只有几行字，笔画很重："我参与了雪仗。我扔了四个雪球。我想去食堂侧门。段璟拦住了我。以上属实。"

他把纸交给周明澈时，周明澈扫了一眼："你就写这么点？"

"够了。"梁一帆说，"又不是写作文。"

宋乔从女生那边传回来的消息是：马立群在办公室说了一句"十班这次闹得不像话"，但陶建国接了一句"我班的学生我知道，他们不是闹事，是闹情绪"。两句话放在一起比较，谁向着十班一目了然。

"陶老师还是偏心咱们的。"宋乔趴在常俊凡桌边说。

"不是偏心。"常俊凡把笔放下，"他是知道事情怎么起来的。知道起因的人说话不一样。"

段璟在旁边听见了这句话。他突然想，陶建国说的"让人家用别的东西记住你"，和常俊凡说的"知道起因的人说话不一样"，好像是同一个意思——都是在说：不要只看结果，要把事情从头看一遍。

他在说明纸上加了一行："高二先朝高一扔的雪球，扔得不重，但先扔了。"

这行字加完，他自己看了一遍，又看了一遍。然后他确定了一件事：今天他要说的不止这些。

下午第三节课后，陶建国把全班写的说明收齐，在讲台上一张一张地看。他看的时候教室很安静，只有暖气管偶尔响一下和翻纸的声音。

看完最后一张，他把那摞纸放在讲台上，没有立刻说话。窗外的天色已经暗了大半，日光灯还没开全。陶建国的侧脸有一半在暗处。

"大部分写的是实话。"他终于开口，"有些人只写了自己做了什么，有些人把别人的也写了。写别人不是错，但你写别人的时候，先想清楚你是想帮他，还是想推他。"

他的目光在教室里扫了一遍，这次没有在任何一张脸上停顿。但每个人都知道他在看谁。

晚自习之前，段璟在走廊上碰到了罗雨桐。她正把那张画对折，准备塞进书包最里层。

"不贴出来？"段璟问。

"贴出来就真成热闹了。"她把画压平，"等过了这阵子再说。"

"画得挺好的。"

罗雨桐看了他一眼，忽然问："你觉得我把梁一帆的脸画得像吗？"

"耳朵画大了。"

罗雨桐笑了一下，那个笑很小，但和下午被误解时咬着嘴唇的样子完全不同。她打开画给段璟看："耳朵是故意画大的——他听见高二喊话的时候，耳朵比谁都竖得直。"

段璟仔细看了一眼，还真是。梁一帆在那张画里侧着身子，一只耳朵画得比另一只大了一圈，像收音机天线朝向高二楼的方向。罗雨桐看到的细节，比全班任何人一句话看到的都多。

"你这张画比十份说明都有用。"段璟说。

"那你帮我在陶老师那里说一下？"

"你自己说。"

罗雨桐又笑了一下，这次笑容里有一点不好意思，也有一点确定。她把画折好，塞进书包，拉链拉到头。

教室后排的暖气管又咣当响了一声，靠墙的几个男生同时用腿夹了一下管子——这是十班男生冬天里练习出来的条件反射，像青蛙被戳了腿。

"谁去拧一下阀门？"周明澈头也不抬地说。

"我去。"付小川站起来，走到后排暖气管旁边，蹲下去拧了两圈。他拧完站起来的时候，膝盖上多了一道白灰印子。

"拧好了？"

"拧好了，但马上又会响。"付小川拍拍膝盖上的灰，"这管子跟咱们班一个毛病——按住了能安静一会儿，但迟早还要闹。"

旁边几个人笑出声来。梁一帆隔着两排座位喊了一句："你说谁是管子？"

"没说谁，你别自己对号入座。"

笑声又大了。连平时不太参与后排闹腾的陈柏言也弯了弯嘴角。段璟注意到，黑板上的"十班"两个字还没擦掉——王启民用白粉笔写的，字体方正，像贴在墙上的告示。但粉笔灰已经开始往下掉，第一个字"十"的横画已经缺了一块。

他没去擦。他觉得留着也有好处——人在被盯着的时候，反而会想清楚什么该做什么不该做。

沈曼青从书包里拿出一本《新华字典》，翻到中间，找了一个字，然后在草稿纸上写了下来。付小川路过时瞥了一眼："你在查什么？"

"'责任'的'责'字怎么写。"

"你不是每次语文都考前五吗？"

"不是查写法。"沈曼青把字典合上，"是查它的本义。字典上说，'责'本义是索取，引申为要求自己。"

付小川愣了一下，拍拍她的桌面："你写说明写出哲学来了。"

"不写清楚不行。"沈曼青说，"如果连自己做了什么都不敢承认，以后哪个老师还敢信你？"

她说这话时语气很淡，但付小川收起了笑。他觉得沈曼青从雪仗那天起好像一直在往前站——不是站到人群中间，而是站到人群和混乱之间。

晚自习第一节，马立群来讲英语卷子。他讲到完形填空时，在黑板左侧写了一个长句，粉笔断了两次。第二次断的时候，后排有人笑了一声，马上又憋住了。

马立群转过身，手里捏着半截粉笔："笑什么？你们班被点名了还笑得出来？"

笑声更大了。这次没有憋，全班都笑了。连马立群自己也没忍住，把断粉笔往粉笔槽里一丢，说："算了，至少你们心态好。"

段璟看着黑板上那个断成两截的粉笔句，忽然觉得十班就是这样：被点名了会觉得丢脸，但丢脸之后该笑还是会笑，该闹还是会闹，只是笑和闹的分寸比以前多了一点。

这个"一点"很小，但很重要。

下晚自习后，走廊上的人慢慢散掉。段璟看见常俊凡站在教室门口，手里拿着自己那份说明，在等陶建国。

她的纸上只写了半页，每个字都像刻上去的。段璟经过时扫到一句："班长没有管住人，是我的问题。"然后下面那句是："但我不认为先动手的人是错的——先动手的人只是比别人先觉得这件事该有人做。"

段璟停了一下，没有停太久。他继续走，一直走到走廊尽头，拐进宿舍楼之前回头看了一眼。常俊凡还站在门口，走廊的灯光把她和门框的影子拉得很长。

段璟想，这个世界上有些人会把"责任"写成"我的问题"，同时又把"勇气"写成"这件事该有人做"。这两个词她写在同一张纸上了，而且没有涂改过。

他觉得自己今天又多看懂了她一点。只一点，但比昨天的全部都要多。

熄灯后，段璟在被窝里把手电筒拧到最暗那一档。他翻开单词本，看见"sincerely"旁边多了一个小三角——上次他在草稿纸上画的，不知什么时候抄到这里来了。

他盯着那个三角看了很久，然后把单词本合上。

明天还不知道会发生什么，但从今天写的那些说明和那张被折好的画来看，十班不会因为被点了一次名就散掉。恰恰相反，被点了名的班，往往比没被点名的班更知道自己是谁。

十班被点名之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

罗雨桐没有因此变成特别醒目的人，可在段璟眼里，罗雨桐从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，常俊凡让大家先把事实说清楚，别把责任推给最会闹的人。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
