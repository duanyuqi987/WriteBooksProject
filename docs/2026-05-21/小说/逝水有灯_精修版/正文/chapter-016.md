# 第016章 前排和后排之前

所属部卷：第一部：初秋三楼
时间：高一春天
地点：物理课后的教室

## 诗歌插页 P016：原创现代诗

我把早晨放低，
让一场雨从课桌边慢慢经过。
你的名字没有被我喊出，
只在作业本边缘一处蓝色墨点旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写前排和后排抵达之前的坡度。春天的物理课总在下午最困的时候。窗外树叶刚冒新绿，阳光透过新芽在课桌上投下稀稀疏疏的光斑，教室里却全是圆珠笔划过纸面的沙沙声——物理老师让大家当堂完成一道受力分析题，限时五分钟。吊扇还没开，空气有些闷，前排有人拿练习册当扇子扇风，纸页哗啦哗啦地响。物理老师背着手在过道里踱步，皮鞋咔咔地敲在水磨石地面上，走到谁旁边谁就赶紧把头低下去。

陆允川做完了。他的作业本上画着一个物体和三个箭头——重力、弹力、摩擦力，每个力都标了大小和方向。他把本子合上，等着老师喊"从后往前传"。

"你做完了？"程砚从旁边探过头来，手里还捏着笔，纸上只有两个歪歪扭扭的力——箭头画得像蚯蚓，标注的数字5N写到一半没水了，笔迹断断续续，"这摩擦力方向到底是往左还是往右？""看相对运动。""我知道看相对运动——但你这三个箭头画的比我的标准十倍，你是不是背了步骤？"陆允川把本子翻过来，盖在桌上。程砚笑了一声，没再偷看，低头继续和摩擦力掰扯。他捏着圆珠笔在纸上戳了两下，笔尖不出水了，在纸面上划出两道白印。他把笔甩了甩，一滴墨水甩到了自己校服袖口上，骂了一声"靠"，然后从笔袋里换了一支新的。

物理老师说了一声"传"，整个教室开始动。从后排往前排传，每列一摞。坐在他前桌的男生把本子往后一伸手，头也不回，手指张了张——意思是快放上来。陆允川把自己的本子递出去，前桌接了，摞在下面继续往前传。第二排接过去，第三排再接过去。他的目光跟着自己的本子往前移动——它被一只手接过去，叠在下面，再被另一只手接过去。传到第三排时他看见自己的本子被搁在了另一本旁边——那一本的封面边角有一处蓝色钢笔水的墨点，圆圆的，像一滴没擦干净的雨。那是夏知遥的作业本。他的本子和她的被叠在了一起。封面贴着封面，页码对着页码。前桌的人把整摞往讲台上"啪"地一放，他才发现自己刚才屏住了呼吸。

"你看题还是看本子？"程砚回头看了一眼他视线的方向。他刚才也在传本子，但他的注意力百分之九十在摩擦力上。
"都看。"
"那你倒是把第三题看对——你刚才说摩擦力等于三牛，但题里给的是五牛。"程砚说着，拿笔帽点了点自己草稿纸上的计算过程，"你连受力分析步骤都说对了，最后代入数字却代错了。你这种错误——"他顿了顿，"不在纸上。"
陆允川低头看了一眼自己的作业本——刚才他回答程砚的摩擦力值时确实说了三牛，而题目写的是五。
"你在想什么？"程砚压低声音。
"想错了。"
"你不是想错了，你是根本没想。"程砚把笔帽往桌上一丢，"三牛是你编的，你上课的时候脑子没在题上。"

叶澄从前排回头借橡皮。她已经把作业本交上去了，现在正在帮英语老师收拾一叠散开的单词卡——卡片是从鞋盒里倒出来的，上面写满了单词和中文释义，有的边角已经卷起来了。她拿着橡皮敲了敲陆允川的桌角，说了一句"别把别人的本子当成你的草稿纸看"。语气平平的，像在提醒他把水杯挪开别碰湿作业本一样。陆允川接过橡皮，低头在自己的一处错误上擦了擦，发现她说的"错误"可能不在纸上。橡皮擦过的地方纸面起了毛，他把毛屑吹掉，那口气呼出去的时候莫名觉得连叹息都算不上。

春天下雨，窗外开始飘细雨，雨点打在法桐新叶上，发出细碎的啪嗒声。雨水顺着窗玻璃往下淌，把外面的树影搅成一片模糊的绿。教室里有人叹气——因为下午的体育课多半要改成室内自习了。后排两个男生开始讨论如果真的下雨就去体育馆占羽毛球场，其中一个已经把手伸进抽屉里摸羽毛球拍了。陆允川听着这些声音，看着雨水在玻璃上画出的那些歪歪扭扭的线，忽然发现自己的作业本还没有被传回来。其实已经传回来了——埋在前排那摞批完的本子里，而他在等的不是自己的分数，是另一个人翻开自己本子时的样子。

夏知遥这时还只是人群里的一个名字。她把本子交出去时没有回头，只低声提醒同桌："最后一道题单位别漏了，他上次扣了我三分。"她说得很轻，像把一枚图钉按进软木板。同桌哦了一声，赶紧翻到最后一页补写了"N"。夏知遥看了一眼，点了点头，然后从笔袋里摸出一支红笔，开始对黑板上的标准答案。陆允川听见了她的语气、内容、以及她说话时手指压在作业本边角的小动作——所有这些都比他自己那道摩擦力题记得更牢。他甚至注意到了她用红笔的习惯：先在草稿纸上划一下看看有没有水，然后再往本子上写。
