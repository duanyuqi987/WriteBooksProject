# 第082章 迟到的表白

所属部卷：第二部：六月以后
时间：六月末
地点：河边护栏旁

## 诗歌插页 P082：原创仿唐绝句

雨色微明照旧尘，
少年别后识前因。
一声铃尽人潮散，
半页风光不问春。

他最后还是没有说。

不是没有机会。填志愿那天，他们全部回到了学校。机房里的旧电脑风扇转得嗡嗡响，每个人坐在屏幕前，把自己的志愿一个一个填进系统。程砚坐在他旁边，一边填一边嘟囔："机械工程、机械设计——这些名字都一样吧？——算了先填进去。"

机房里弥漫着一股旧电脑散热时特有的焦糊味。头顶的日光灯管有一根在闪，忽明忽暗的，没人去管。坐在陆允川后面的男生在打电话——"妈，那个学校代码是多少？我找不到了。"前面一排的女生小声讨论着专业排序的问题，声音压得很低，像怕别人听见又怕自己记错。

陆允川的第一志愿是一个沿江城市的大学，第二志愿是省城的大学。他填完以后没有立刻提交——把页面从头到尾检查了两遍，确认没有填错学校代码，然后点了"提交"。

系统弹出一个对话框："提交后不可修改，确认提交？"

他点了"确认"。

程砚也提交完了，把椅子往后一仰，伸了个懒腰。"搞定。剩下听天由命。你第一志愿是江大？"

"嗯。"

"那个城市我去过一次。小时候跟我爸出差。"程砚回忆了一下，"江边风很大，吹得人站不稳。不过夜景挺好看。"

"你第一志愿呢？"

"南方工大。我爸研究了三天，说这个学校就业率最高。"程砚顿了一下，"不过我偷偷改了第二志愿。"

"改成什么？"

"省内的理工。杜雨薇报的也是省内的。"

陆允川看了他一眼。程砚耸了耸肩。"没跟她说。说了她会骂我没出息。但其实我觉得省内也挺好的，离家里近。"

从机房出来的时候，他碰到了夏知遥。她刚从另一间机房出来，手里拿着一张打印好的志愿确认表。夏日的走廊里没什么人，阳光从窗户斜进来，照在她手里的A4纸上。

"你填完了？"她先开口。

"嗯。第一志愿是沿江那边的一个学校。"

"我也是。"她低头看了一眼自己的确认表，然后又抬头，"不过我报的是建筑。你呢？"

"还没定专业。大类招生，进去再分。"

"那挺好的。可以先了解一下。"她停了一下，"你考得不错。叶澄跟我说了你的分数。"

"你也是。五百八十五。"

"还差一点才到我妈的目标。她想让我考六百分。"夏知遥笑了一下，那个笑里有无奈，但不多，"不过够我想去的学校了。所以还行。"

两个人站在走廊里，窗外有学生在操场上打篮球，球砸到地面的声音一下一下传过来。

陆允川的手指在裤缝边上弯了一下又伸直。他想起程砚在河边说的话——"你要是想告诉她，就趁现在。"

"夏知遥。"他叫了她的名字。

"嗯？"

走廊里的光线很亮，亮到他能看清她脸上因为天热而起的细微汗珠。她的表情很平静，是在等他说下一句话的平静。

"夏知遥。"

"嗯？"

他看着她手里的那张确认表。建筑学，五年制。他忽然意识到，如果他现在不说，等她去了南方，他们之间的物理距离就会变成一千多公里——高铁也要坐五六个小时。

"……没什么。就是想说，祝你大学顺利。"

夏知遥看着他，点了下头。"你也是。"她把手里的确认表折了一下，塞进了书包侧袋里。书包是灰色的，拉链上挂着一个很小的建筑模型挂件，像是某个大学开放日发的纪念品。

她转身走了。走出走廊拐角的时候，那张志愿确认表不知怎么从侧袋里滑出来一角，被风吹得卷起来，她用手按住了。动作很自然，像按住了某个不需要再展开的东西。

陆允川站在走廊里，看着她的背影消失在拐角。然后他转过身，往另一个方向走。

程砚在校门口等他。"说了吗？"

"没有。"

"为什么？"

陆允川把书包甩上肩膀。"因为我在说出那句'祝你大学顺利'的时候，她的表情一点都没有变。不是冷漠——是那种'听到了，收到了，会礼貌回应'的表情。那种表情就是我想要的答案。"

程砚沉默了几秒。然后他把手搭在陆允川肩上。"走吧。去河边坐坐。"

他们又去了那条河边。护栏上的蓝漆还是那样，剥落了几块。河水还是那样，慢慢地流。对岸钓鱼的人换了一个——是个老大爷，鱼篓里已经装了两条鲫鱼。

"你说得对。"陆允川靠着栏杆，声音很轻，"有些话不说，不是因为怕被拒绝。是怕说了以后连现在这种状态都维持不了。但我今天发现，我跟她现在也没有什么状态。我们就是同学。前后桌过的同学。仅此而已。"

程砚没说话，等着他继续。

"我在走廊里叫她名字的时候，她转过头来的表情——就是那种'哦，你还有话要说'的表情。不是期待，也不是紧张。就是等我把话说完的表情。"陆允川把瓜子壳捏在手里，捏得咔咔响，"那时候我就明白了。她心里从来没有把我放在那个位置。"

"你现在才发现？"

"早就发现了。但今天是确认。"

程砚从兜里掏出手机，翻了几下，然后把屏幕转过来给陆允川看。是他和杜雨薇的聊天记录。最近一条是杜雨薇发的一段语音，程砚点开——"程砚我警告你，你要是再偷偷改志愿，我就把你删了。"

陆允川愣了一下，然后笑了。"她怎么知道的？"

"我哪知道。女人的直觉吧。"程砚把手机收回去，自己也笑了，"我都没敢回。到现在还晾着呢。"

"那你真删了第二志愿？"

"没有。先放着。到时候再说。"程砚嗑了一颗瓜子，"反正录取又不是我填什么就给什么。"

河上有一艘捞垃圾的小船慢慢开过去，船尾拖着一道浅浅的波纹。船上的工人用长竿网兜把水面上漂的塑料袋和枯枝捞起来，动作不紧不慢，像在做一件做了很久的事。

程砚从兜里掏出一包瓜子——不知道他什么时候买的。他撕开包装，倒了一半给陆允川。"吃点东西。你每次心情不好的时候都不吃东西。"

"我今天没有心情不好。"陆允川接过瓜子，嗑了一颗。是五香味的，有点咸。

"那你是什么？"

"说不上来。不是轻松，也不是难过。就是……确定了。"

程砚嗑着瓜子，看着河水。"确定了就好。确定了比一直悬着强。"

河风吹过来，把瓜子壳从栏杆上吹下去，飘到水面上，很快就被流水带走了。远处的夕阳正往山后面沉，把河面染成一整片橙红色。

陆允川把手里最后一颗瓜子嗑完，把瓜子壳扔进旁边的垃圾桶。他拍了拍手上的盐粒，说："走。回去吃饭。我妈今天做了红烧肉。"

"你妈做的红烧肉比我妈做的好吃。"

"那你来我家吃。"

"行。"

两人沿着河边往回走。陆允川走的时候，回头看了一眼那段蓝色护栏——漆剥落的地方露出一块一块的锈红色铁。河水还是那样流，不急不缓。他想，很多年后他可能不会再记得今天走廊里夏知遥的表情，但会记得这段护栏的颜色，和程砚把瓜子倒在他手心里时那个随意的动作。
