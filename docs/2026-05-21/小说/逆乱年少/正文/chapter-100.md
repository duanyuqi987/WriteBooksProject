# 第100章 第一个电话

时间：2012年一月二十三日，除夕夜

地点：封丘县，林玉起家

卷次：第四卷：暑假的信

除夕夜，封丘县城被鞭炮声塞满了。

从下午五点开始，零零星星的响声就没断过，天完全暗下来之后，四面八方同时炸开——北边是纺织厂宿舍区的鞭炮，南边是二中家属院，东边是刚盖好的新小区。整个县城弥漫着硫磺味，空气被烟染成了淡蓝色。路灯照下来，烟在光柱里缓缓翻涌。

林玉起和父母吃完年夜饭。八个菜——母亲做了红烧鱼、炖鸡、炸春卷和几样素菜，父亲从橱柜里拿出一瓶放了半年的白酒，给他倒了小半杯。父亲说大二了可以喝一点。他喝了一口，辣得皱眉，母亲笑他"还早着呢"。

吃过饭，母亲去厨房洗碗，父亲靠在沙发上看春晚。林玉起回了自己房间。他把门关上，坐在床边，手里握着手机。短信一条接一条地涌进来——高中同学、大学室友、文学社认识的人——郭向阳发了一条很长的拜年短信，一看就是秦佳禾帮他编辑的，因为结尾有个颜文字笑脸。付小川发的是"新年快乐，下学期别蹲图书馆了，来物理实验室玩我请你吃泡面"。何南星群发了一条带打油诗的贺春词——字数和唐诗一样工整。连郑学然都发了一条——"祝新年创作丰收"。林玉起一条一条回复——"新年快乐""同乐""明年见"。回复完以后列表里还剩下一个人没有发来消息也没有收到他的回复。

他打开通讯录翻到苏安晴的名字。他没有存她的照片——通讯录头像留白。只有名字和一串十一位的数字。那个号码他存了一年半，但拨出去的次数是零。之前他们只在QQ上聊过——有限的几次，在群里、私下也只有那次"谢谢你"——纯文字、静止的、没有声音。

他盯着那串数字看了很久。窗外鞭炮声忽然密集起来——大概是谁家放了一挂五千响的。电视里春晚的开场曲从客厅隐隐传来，父亲在外面说了一句"今年的阵容没有去年好"。母亲的拖鞋声在走廊里来来回回。

他按下了拨号键。

嘟——嘟——嘟——嘟——每一声都像有人用手指轻轻叩他的胸口。然后电话接通了。

"喂？"

苏安晴的声音。在电话里和面对面有一点点不一样——电话里的声音更近，像是贴着耳朵在说话。她那边也很吵——有隐约的鞭炮声、电视机声，还有一个女人的声音在喊"安晴，过来吃饺子"。

"是我。林玉起。"

"我知道是你。来电有名字。"

她笑了一下——笑声很轻，但他听见了。电话线把那个笑声从县城那头传过来，没有任何损耗。

"新年快乐。"他说。

"新年快乐。"

然后沉默了两拍。不是尴尬的那种沉默，而是两个人都在想下一句话该说什么。窗外又炸了一挂鞭炮，这次更近——大概就在楼下，震得窗户嗡嗡响。

"你在干嘛？"他问。

"没干嘛——刚吃完饺子。你呢？"

"也是——吃完饭在看电视。我爸睡着了。"

"我爸也是。"她又笑了，"春晚还没到相声呢就睡着了。"

又沉默了一拍。然后两个人同时开口，又同时停下来。

"你先说——"

"你先——"

苏安晴轻声笑了，说："下学期开学什么时候？"

"二月十八——十九号。你那边呢？"

"差不多。也可能十九号。"

"那——你会回图书馆四楼吗？"

"会。"她回答得很快，快到像不需要思考。"我说的——说了会就会。"

林玉起握着手机的掌心出汗了。窗外鞭炮声还在响，但他已经听不太清楚了。他只能听见电话那头苏安晴的呼吸声——很轻，像图书馆四楼门被推开时带进来的那阵风。

"开学有什么事可以找我。"他说完这句话以后在心里用力骂了自己——这句话太弱了，太同学化了。但他确实不知道说什么。

"好。你也是——有事找我。"

她那边有开门的声音，然后那个女人的声音又响起来了："安晴，谁的电话？"

"同学。聊两句就挂。"

苏安晴压低了声音："那我先挂了。新年快乐——不是群发的。"

"不是群发的。新年快乐。"

她先挂断了。林玉起看着手机屏幕上的通话记录——苏安晴，通话时间：三分钟。不是群发短信，是真的电话。三分钟。这三分钟里他说了什么——新年快乐，你在干嘛，下学期你去不去图书馆。不是什么重要的话。但这些不重要的话加在一起就是——我想你了。他想她。

他把手机放到枕头底下，在笔记本里接着上一行继续写：

"大二过半，我还在看她，她也还在看我——但'看'和'看'之间隔着什么，我说不清楚。这层东西不是墙，墙可以敲或推倒；不是海，海可以航；不是天空，天空可以用眼神穿过去。它很薄——有时觉得一伸手就能碰到对面。但每次刚要伸直手指，我们同时收回去了。对——同时——这才是让我说不清楚的地方。今天三分钟的电话，我挂掉以后手心全是汗，但胸口那个位置是暖的。那个位置，以前叫'苏安晴所在的方向'，今天我叫它——'快到了'。"

房间外的鞭炮声慢慢稀了。远处偶尔还有一两声，像在梦里回响。父亲在沙发上发出均匀的鼾声，母亲轻手轻脚地收拾茶几上的瓜子壳。林玉起把那本蓝皮笔记本合上，从抽屉里拿出那个包着报纸的永和豆浆纸杯，在手里转了一圈，放回去。大二过半。他离出口还有多远——不知道。但今晚，他第一次听见她的声音穿过除夕夜的硫磺味抵达了他这边。不是群发短信，不是QQ消息。是她接起电话说的那一声"喂"——带着一点惊讶，带着一点笑意，没有任何推挡。

他在笔记本底画了一条横线——

第四卷完。
