# 第060章 最后的凝视

时间：2013年6月4日傍晚

地点：教室

卷次：卷三：此心藏久

其实他没有马上走。

赵一鸣以为他在收拾书包。其实他关上门之后，在教室外面站了一会儿。走廊里已经没什么人了。五楼的走廊很长，从头到尾大概要走一百步。他走了一步。两步。三步。然后停住。转身面对那扇门。

门是木头的。上面有一个玻璃小窗。玻璃上贴了一张纸——"高三（三）班"。纸是开学时老刘用毛笔写的。墨迹已经洇开了。高三那个"三"的第三道横，中间断了一点——大概是写的时候笔没墨了。

他把手放在门把上。门把是铜的，被无数人拧过，磨得发亮。他拧开门，轻轻推开。走进黑暗的教室。没有开灯。外面的光从窗户照进来。傍晚六点多的光，灰蓝色，从对着水塔的窗户洒进来，在地板上画了几个方形的亮块。他站在门口的光与暗的分界线上。

然后慢慢往里面走。走过讲台。走过第一排。走过第二排。走到第三排——她的位置。那个空了的位置。椅子翻在桌面上——四脚朝天。和所有座位一样。桌面是空的。上午那个水印还在——光线从侧面打过来的时候隐约能看到。一个很淡的圈。他用指尖碰了一下那个水印。凉的。木质本身的凉。

他拉开那把翻着的椅子，放下。椅子腿在地板上蹭出了一声很轻的响。他坐下去。坐在她的椅子上。面前是她的课桌。桌面上的木纹在灰蓝色的光线里像一条一条河。他低头往桌子里面看——抽屉也是空的。她真的把所有东西都带走了。没有留下半张纸。没有留下一个字。唯一留下的就是这个玻璃杯放了三年磨出来的水印。

他坐了很久。

窗外的光线在一点一点变。灰蓝色的光开始往深蓝色转。天空像一块正在被染色的布。从浅到深。从亮到暗。他想起高一开学第一天。她从前门走进来的时候。浅蓝色T恤。马尾。阳光打在她的脸上。那天的阳光也是从这个窗子照进来的。角度几乎一模一样。三年了。阳光还是那个角度。人已经不一样了。

他也想起她在走廊饮水机前转过身来的那个瞬间——一秒的目光对视。她的眼睛是深黑色的。那年秋天。走廊上的兰花开了。她端着一杯热水。他假装路过。想起她趴在桌上脸埋在胳膊里的样子——物理考了五十八分的时候。那天下午的夕阳也是橘色。照在她弓起的背上。想起她站在元旦的昏暗灯光下唱《送别》——长亭外古道边。那是他第一次看到她化了妆的样子。想起她在图书馆窗外下雪的时候看雪——隔着三张桌子。

想起她在走廊上靠墙说"你也来透气"——带着兰花香的三月傍晚。那是她第一次主动跟他说话。

想起她把那瓶水喝了——拧开盖子仰头喝了两口。第一口。第二口。瓶身上那两座山的图案被捏得有点变形。想起她今天下午撕下名字贴放进铅笔盒时的动作。撕得很慢。边角撕破了一点点。她把纸片折好。放进了铅笔盒。想起她站在门口转身看了一眼教室——最后落在第四排的、不到一秒的那一眼。

他在这把椅子上坐了不知道多久。灰蓝色的光变成了深蓝色。然后又暗了一层。水塔上的红灯开始闪——三秒一下，红光从西边窗户打进来，落在天花板上。一闪——天花板红了一角。一灭——又黑了。一闪。一灭。像心脏在跳。

他从书包里拿出笔和本子。翻开错题本的最后一页。那个他已经写了大半的本子——封面上是一个他自己画的棋盘格，里面每一页密密麻麻的算式、批注、箭头。他翻到空白的那一页。借着窗外微弱的红光，开始写字。

不是算式。

是诗。现代诗。

题目叫做《空座位》。

他写得不多。二十几行。每个句子都很短。他把三年里想说但没有说过的那些话写了下来。不是情书。不是表白。是一个人在高三最后一天的傍晚坐在她的椅子上对着空教室写下的东西。

写完最后一个字，他把笔放下。从头到尾看了一遍。然后撕下来。折了两折。和《初见》《暗恋》《操场》放在一起。

现在有四首诗了。

他站起来，把椅子重新翻到桌面上。和所有椅子一样——四脚朝天。然后走到自己的座位。也把自己的椅子翻上去。讲台上有一截粉笔头。他捡起来。在黑板上写了一个很淡的字。淡到明天早晨可能就看不见了。

"再会。"

然后把粉笔放到粉笔槽里。拍了拍手上的灰。背上书包。走到教室门口。最后看了一眼整间教室——在窗外红灯的红光里。她的座位。他的座位。四十二张桌椅。被拉歪的横幅——"拼搏改变命运"。黑板上那个"3"。一切。

然后他关上门。这一次是真的关了。

走下楼梯。每一步都踩得很慢。从五楼到四楼。四楼的班级也在打扫——有人在往外搬书。纸箱子堆在走廊上。从四楼到三楼。高一的班级——早就放学了。教室空空的。从三楼到二楼。从二楼到一楼。走到教学楼门口的广场上。他可以感觉到心跳在胸腔里——不是快。是重。每一步都像踩在自己的心跳上。

广场上人很少。天已经黑了。只有路灯和远处水塔上那个红灯。操场上的第五道跑道在黑暗中是深红色的一条线。他站在那里看着那条线看了很久。然后转身上了班车。

回到镇上。家是那扇掉了一点漆的老门。推门进去——妈在厨房里。锅里煮着面条。他爸坐在布沙发上看报纸。眼镜挂在鼻尖上。看到他回来。"回来了。""嗯。""后天好好考。""嗯。"他回到自己的房间。把门关上。

房间很小。单人床靠墙。书桌上的台灯是暖黄色的。他把书包放在床上。靠着床头坐了下来。从书包里拿出那个牛皮纸信封——成绩条、笔记、写着"15→6"和"不管旁边站着谁"的最后一页。拿出那四首诗——《初见》、《暗恋》、《操场》、《空座位》。把它们在膝盖上摊开。四张纸。一张发黄。一张还白着。一张折痕很新——刚才写的。手电筒放在床头柜上。他没开它。只是用手摸了一下它的塑料壳。

窗外的月亮是半轮。光从窗帘缝里漏进来。照在地板上一小条。和宿舍那条几乎一模一样。他想起高一的第一个晚上也是这样——他睡不着，在语文课本封底内页写了两个字，然后划了。现在三年过去了。那两个字还在那本书上——划了，但没划烂。透过墨痕还能隐约看清笔画。

他闭上眼睛。在心里把过去三年过了一遍。然后他对自己说了一句话。

"高考后。告诉她。"

然后他睁开眼。翻身起来坐在桌前。从书包里拿出语文课本——上面的错题本——还有物理那本翻了很多遍的笔记。翻开。开始做最后一遍看知识点。今晚不做新题。只是把一些容易错的看一看。看到十点半。

考前三天。这次他不着急。不紧张。不害怕。他已经等了三年。最后三天而已。

他放下课本，关了台灯，躺下。窗帘缝里的月光还在。他盯着那把书桌上的椅子——椅子上搭着校服外套。校服的颜色在黑暗里看不出。但那是蓝色。三年四季都穿着的那件。明天不会再穿它了。后天也不会。高考的时候规定穿便装。

他伸手把校服拿过来。叠好。放在枕头旁边。明天收进衣柜里去。

然后翻过身。闭上眼。

过两天。他会站在她面前。他会把那三年没说出口的话一句一句地拆开——不是一次说完——慢慢地。一件一件来。从他高一九月第一天看到她开始。从他用指甲在名单上划她的名字开始。从他写的每一首诗——他看到她的每一天。那些还没说出口的话。他会把它们都说出来。

不管她听不听得懂。

不管结果。

他翻了个身。用被子蒙住了半张脸。嘴角动了一下。然后睡着了。

窗外很安静。后院那棵槐树上停了一只夜鸟。叫了两声。不叫了。然后只剩远处水塔那盏红灯还在闪。三秒一下。一直闪。一直闪到了天亮。天亮的鸟又叫了。这次的叫声是另一种鸟——叫声更脆更短。阳光从窗帘缝里漏进来。落在他枕头旁边的校服上。校服上的蓝色在早晨的光里很淡。像被洗了很多次之后的颜色。确实是洗了很多次。三年。
