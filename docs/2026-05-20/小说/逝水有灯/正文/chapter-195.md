# 第195章 夏知遥的消息多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：旧同学群

## 诗歌插页 P195：原创仿唐绝句

旧群沉处有微光，
车站新墙雨后凉。
不是故人曾薄我，
少年自把梦铺长。

多年后，陆允川换手机，旧同学群的聊天记录只剩一部分。那张车站照片因为缓存失效，已经打不开，只留下一个灰色方框和“图片已过期”的提示。他盯着那行字看了一会儿，觉得很合适。

许多事本来就会过期。不是被谁删掉，也不是被谁否认，而是时间到了，系统不再替你保存那么清楚。你若非要重新下载，就要承认自己是在主动翻找。

他没有去找夏知遥要原图，也没有在群里提起那年她回县城的事。后来同学们陆续结婚、生子、换工作，群里偶尔热闹一阵，又安静很久。夏知遥的名字出现过几次，和其他人一样，普通地落在消息列表里。

陆允川慢慢明白，旧人不是用来证明自己曾经深情的证据。夏知遥如此，苏晚枝也是如此。她们在他的青春里出现过，却都不该被他写成某种功能：一个负责初恋，一个负责遗憾。

他把那条“图片已过期”的提示截下来，放进素材文件夹，文件名叫“旧群”。不是为了纪念某个人，而是为了记住那种轻轻放过的能力。过期的图片没有回来，生活却因此腾出一格新的内存。
