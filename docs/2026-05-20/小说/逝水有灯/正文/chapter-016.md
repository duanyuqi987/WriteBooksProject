# 第016章 前排和后排之前

所属部卷：第一部：初秋三楼
时间：高一春天
地点：物理课后的教室

## 诗歌插页 P016：原创现代诗

粉笔灰落在公式旁边，
前排传来红笔的声音。
后排有人等一本作业，
像等一封并不存在的信。
桌子之间只有几步，
心却绕了很远的路。

物理老师让大家互换作业批改时，教室里先是一阵乱。

前排往后传，后排往前递，作业本在空中交错。有人喊名字，有人找红笔，有人把别人的本子拿反了。陆允川坐在后排，看见夏知遥把自己的练习册放到同桌手上。

那本练习册封面是淡蓝色，右上角贴着姓名贴。陆允川只看了一眼，就低头翻自己的本子。他知道自己不该期待一本作业会传到自己手里，可心里还是不争气地算了一下座位距离。

程砚把红笔咬在嘴里，含混地问：“你看什么？”

“没什么。”

“没什么你耳朵红？”

陆允川把作业本推给他：“改题。”

程砚拿过来，看见第一题后面的空白，啧了一声：“你这不是没做吗？”

“忘了。”

“你不是忘了，你是把脑子借给前排了。”

陆允川没反驳。他听见前排有女生笑，接着夏知遥低声说了一句什么，声音被翻本子的响动盖过去。他不知道她说的是题目，还是别的。正因为不知道，才更容易胡乱想。

作业本一排排传下来。夏知遥的那本在中间被另一个男生拿到，对方认真改题，红笔很快划出几道线。陆允川看着那几道红线，心里冒出一种很小的失落。失落得没有道理，却真实。

老师在讲台上说：“错的地方要写原因，不要只打叉。”

陆允川低头给程砚的本子写批注。程砚的字很潦草，他看了半天才认出一个“加速度”。他忽然觉得自己也像一道没写清条件的题，明明没有资格要求答案，却总想让别人看懂。

下课铃响，作业本又传回去。夏知遥接过自己的练习册，翻了翻，看见红笔批注后皱了一下眉。她没有回头，也没有任何特别的表情。

陆允川把自己的本子塞进桌肚，手背蹭到一层粉笔灰。那灰很轻，一吹就散，可他没有吹。他只是把手握起来，像握住一个没有发生的小故事。
