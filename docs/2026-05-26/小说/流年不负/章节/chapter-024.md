# 第024章 篮球赛

时间：2011年5月初

地点：学校篮球场

卷次：卷二：窗边人影

五月初。学校组织班级篮球赛。

高一三班对高一六班。下午第三节课后。操场南边的水泥篮球场。

顾临风打前锋。他个子不算高，一米七五。但跑得快。抢球也狠。

赵一鸣打后卫。嗓门大——场上喊得最响的就是他。

"传球——传——这边——"

篮球场边围了不少人。自己班的多一些，别的班有路过停下来的。

顾临风在场上跑着。额头上有汗。校服袖子卷到手肘。鞋是回力的，鞋底磨平了一块。

上半场打完。三班落后六分。22比28。

中场休息。赵一鸣坐在篮球架下面喝水。

"你下半场往前顶。六班那个中锋很高。但他转身慢。"

顾临风擦了一把汗。

"知道了。"

然后他往场边看了一眼。

她站在那里。

浅绿色T恤。校服外套系在腰间。站在人群的第二排。旁边是那个圆脸女生——许若琳。大概是许若琳拉她来的。

苏晚晴没在看球。她在看场上的人——目光扫着场上休息的队员。扫到顾临风这边的时候停了一下。大概不到一秒。然后移开了。

顾临风转过头。把水瓶子拧紧。

下半场开始。

顾临风像变了个人。

他跑得比上半场还快。抢篮板的时候跳得比平时高了至少五厘米。球弹到篮筐弹出来，他挤开对方两个防守——抢到篮板，转身，后仰——球进了。

场边自己班的人在喊。

他落地的时候差点往她那边看。忍住了。

比分追上来。一分一分地追。

最后三分钟。33比34。差一分。

赵一鸣带球过半场。被对方包夹。他把球传出来。

顾临风接球。三分线外。

面前是空的——防守的人被赵一鸣拉过去了。

他跳起来。

出手。

球在空中划了一道弧线。

进了。

场边炸了。

有人喊。有人拍手。赵一鸣冲过来拍他肩膀。

"好球"

顾临风落地。嘴角动了一下。往场边瞥了一眼。

苏晚晴在拍手。

不是对着他的方向。就是跟着旁边的人一起拍。

但她确实在拍手。

比赛继续。六班发底线球。三班全场紧逼。最后十几秒——对方投篮不中。赵一鸣抢到篮板。运了两步把球往后场一抛。

终场哨响。

36比34。三班赢了。

顾临风弯着腰喘气。手撑着膝盖。汗滴在水泥地上。

赵一鸣跑过来。

"你下半场打了鸡血吧——那个三分太牛了——平时你投十进三——今天投五进四——"

顾临风直起腰。擦了把汗。

"运气好。"

他往场边看。

她已经走了。

人群散了。场边空空的。许若琳也不在了。只有篮球架下的影子被夕阳拉长。

赵一鸣顺着他的目光看了一眼。拍了拍他的肩。

"走了。回去洗个澡。晚上还要上自习。"

顾临风把球衣往下扯了扯。跟着赵一鸣往宿舍走。

晚自习。

他坐在座位上。左边膝盖磨破了皮。右手手腕有点疼。腿很酸。

但他觉得今天挺好的。

苏晚晴在第二排做物理题。马尾很安静。后背挺直。

她下午在场边看了多久。她有没有看到那个三分。她拍手的时候是不是真的在拍。

他不知道。

但他觉得自己今天确实投进了几个好球。

翻开物理练习册。画了一个受力分析图。画错了。涂掉。重画。

下课铃响。

赵一鸣走过来，手里拿着两瓶水。

"给你一瓶。你今天MVP。"

"不是MVP。"

"你说了不算。我说的。"

顾临风接过水瓶。拧开盖子。

水是凉的。

好喝。
