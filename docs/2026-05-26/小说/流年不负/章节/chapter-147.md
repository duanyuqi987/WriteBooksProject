# 第147章 "那个小鱼还在吗"
时间：2018年4月中旬
地点：楚州月湖边
卷次：卷八：旧诗新读

天快黑了。两个人坐在亭子里。这次亭子里没有拉二胡的老人。石凳空着，石桌上的棋盘蒙了一层薄灰。

顾临风在石凳上坐了一会儿。手在膝盖上放平。然后收回来，又放平。

苏晚晴坐在他对面。帆布包放在旁边的石凳上。包是米白色的，边角磨得有点发白。

"你那个包。"他说。

"怎么。"

"用了很久了吧。"

"大一买的。六年了。"她伸手摸了一下包的边角，"底边有点开了。补过一次。"

"里面装了什么。"

"书。水杯。纸巾。钥匙。还有一些小东西。"

"什么小东西。"

她没回答。把手从包上拿开。

他想了一会儿。手指在石桌上叩了两下。叩完又叩了两下。

"那个小鱼。"他说。

她看着他。

"还在吗。"

天边最后一点光沉下去了。路灯还没亮——正是昏暗交替的那一瞬。她的脸在暮色里轮廓模糊。但她的眼睛是亮的。

她没说话。拿过包。拉开拉链。

拉链的声音在安静的亭子里响了三秒。齿轮咬合——咯哒咯哒咯哒。

她把手伸进去。伸到包的最里面。手指越过一本书，越过一个水杯，越过钥匙串，越过一包纸巾。

然后在包的底部——内层口袋——停住。

她从内层口袋里取出一个小东西。攥在手里。攥得很紧。

然后她把手伸到他面前。手心向上。五指慢慢张开。

掌心里躺着一只陶瓷小鱼。

黄豆大小。

灰白色的身体。尾巴边缘有一点缺——被时间磨损了。肚子中间有一道细细的裂纹——没有断，只是裂了。像瓷器上那种冰裂纹——碎了又没碎。裂纹沿着鱼身的弧度弯下来，弯到尾巴尖。

他的目光停在那条鱼身上。停了很久。

路灯亮了。亭子外面第一盏。黄光从亭子外面照进来，照在她手心里。鱼身上的裂缝在光里更清楚了。

"裂了。"他说。

"高一的时候就裂了。"

"怎么裂的。"

"书包蹭到桌角。那天下午回家才发现。鱼在书包的外侧口袋里，包蹭到桌角的时候咯噔一声。我以为是铅笔断了。"

"然后你打开看了。"

"嗯。看到这条鱼。身上多了一道裂纹。"

"你没扔。"

"没扔。"

他伸出手。指尖碰到鱼尾巴。鱼在她手心里躺了二十年，温度跟她手心一样温。顺着裂缝的纹路慢慢滑——从鱼尾到鱼肚子，再滑到鱼头。

"高一（三）班。你的座位是靠走廊第三排。"

她点点头。

"桌角是铁包边的。刷了蓝漆。漆磨掉了一块——底下是铁。橙红色，有点锈。"

她抬头看他。"你怎么知道——"

"我后来去看过。你毕业以后。"他说，"高一暑假。有一天我一个人回了学校。教室开着。我走进去，找到你的桌子。"

她没说话。手里的小鱼还躺着。鱼眼睛是两颗小黑点——笔画上去的。这么多年了，黑点还在。

"桌角确实少了一块漆。"他说。"我站在那里——站了很久。大概五分钟。"

"你在想什么。"

"在想你那天下午——包蹭到桌角的声音。在想你回到家发现鱼裂了。在想你有没有心疼。"

她把手收回去，放在膝盖上。小鱼还在手心里。裂纹在路灯下泛着淡淡的黄。

"心疼了。"她说。"但没哭。"

"为什么不来找我。"

"找你干嘛。"

"告诉你鱼裂了。"

"给了就是给了。裂了也是给了。"

他说不出话。耳朵尖开始发红。

她重新低头看小鱼。"这条鱼。我一直放在笔袋里。高中三年。大学四年。工作这一年。换了好几个笔袋——但鱼一直在。"

"笔袋。"

"嗯——不是放在外面给人看的。就是在笔袋里面。每天打开笔袋的时候，先看到的是铅笔和红笔，然后看到它。"

"每天都能看到。"

"每天。"

他双手撑着膝盖。手指在膝盖上摩挲。湖风从亭子外面吹进来——四月晚上的风，凉凉的，带着水汽。空气里有柳树叶子被晒了一天之后散出的微苦味道。

"你为什么一直留着它。"

她转过头看着他。路灯的光在她眼睛里晃了一下。

"因为它是你送的。"她没有停顿——说这句的时候也没有犹豫。

六个字。说得很轻。但每个字都清清楚楚。

他低下头。两只手交叠在膝盖上。手指握得很紧。指甲抠进肉里。

"顾临风。"

"嗯。"

"你知道我收到那条鱼的时候在想什么吗。"

"在想什么。"

"我在想——这个人。他从运河边上捡到一颗陶瓷小鱼——黄豆那么大。他没有扔。他把它揣在口袋里，带到学校来。然后用一节语文课，趁老师转过身在黑板上写字的时候，从后排走到我座位旁边，把鱼放在我桌上。"

"你看到了——"

"我没看到是你放的。但你放在我桌上的那一刻，正好有一束阳光从前门那里照进来。我在桌面上看到一个人影——就是一只手——放下一颗小东西——然后马上缩回去。"

"你以为是——"

"我当时不知道是谁放的。后来猜到了。"

"怎么猜到的。"

"你往后退的时候。书包带子勾到了后排的椅子背。椅子往后倒了一下——没倒到地上，椅子腿在空中停住了——你又伸手接住了。椅子落地的时候声音很小——咕咚一声。"

"你听到了。"

"听到了。"她说。"但我没回头。因为刚好上课铃响了。老师转过身来继续写字。"

她把鱼翻过来。鱼肚子朝上。鱼肚子下面有一个更小的裂口——细得像一根头发丝。路灯的光从裂缝中间透过去——很小很小的一线。

他看着她手心里的鱼。看着那道裂缝。想起运河边那个下午——他在捡石子的时候看到这颗小东西。它的眼睛是两个小黑点。黄豆一样大。他把它捡起来，在水里洗了洗，揣进左边口袋里。左边口袋放了整节语文课。

"你怎么知道是运河——"

"你老家在运河边上。"她说。"赵一鸣说的。高二有一次课间聊天，赵一鸣说你们一起去运河边玩过。"

赵一鸣。又是赵一鸣。

他把手伸进口袋里摸手机。摸到了，没拿出来。又把手机放回去。

"你是不是觉得我很傻。"他说。

"什么。"

"为了一条黄豆大的鱼。"

她收拢手指，把小鱼攥在手心里。

"你不傻。"她说。然后把手放回包里。把小鱼重新塞进内层口袋。拉链拉上。咯哒咯哒咯哒。

"谢谢。"他说。

"谢什么。"

"你没扔。"

她站起来。把包背好。

"不谢。"她说。"它本来就是我的。"

她走出亭子。他在亭子里坐了两秒。然后站起来跟上去。他们的影子在路灯下叠在了一起。
