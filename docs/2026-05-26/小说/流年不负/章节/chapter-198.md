# 第198章 流年

时间：2023年1月
地点：楚州
卷次：卷十：不负少年

下午。天空暗了下来。云层是铅灰色的。低低地压在城市上面。风不大。但冷——那种潮潮的冷，渗骨头的冷。

然后开始下雪了。

开始很小。像空中飘着白色的灰尘。不仔细看看不见。慢慢地大了。雪花变大了——指甲盖大小。一片一片地往下落。不是急急的那种。是很慢的。飘下来的。有点像有人在很高的地方往下撒碎纸。

这是楚州2023年的第一场雪。

苏晚晴在阳台上。阳台是封闭的——推拉玻璃窗。她把窗户推开一条缝。冷风灌进来。她没缩手。伸手——接了一掌雪。雪在她手心里化成水。和十二年前在那个教室窗户前一样。

她从角落里拎起那盆茉莉花。花盆里的土已经很干了——冬天她很少浇水，怕烂根。土面开裂了。裂成了一小块一小块的。她往盆里倒了新土——用小铲子铲了两铲。土是湿的。黑的。有一股泥土的腥味。然后把土压了压——用手指摁平了。又倒了半杯水。水从土面上慢慢地渗下去。没有声音。

花盆里有一片枯叶子。她弯腰捡起来。放进旁边的垃圾桶里。她手指上沾了泥。没擦。就这么糊着。泥在指腹上干了一层——很薄，像涂了一层膜。她闻了闻手指——泥土的味道。

顾临风在客厅里。他把浅灰色笔记本从书架上取下来。翻开第一页。看着自己昨天写的诗。

七律。《不负》。

他从头到尾又默念了一遍。然后站起来。也没有说"我来念给你听"。就拿着本子走到阳台门口。阳台门是推拉式的。他推开门。

冷风扑脸。淡淡的雪花落在他的头发上。几片。白的。在黑头发里一眼就能看出来。

她蹲在花盆前面。手扶着花盆。回头看到他。

"冷。你出来干嘛。"

"给你念个东西。"

"念什么。回屋里念。"

"就这里。"

她站起来。因为蹲久了膝盖响了一下。她扶着阳台的窗台。看着盆里新换的土——土里有她刚刚浇的水，正在往下渗，表面颜色变深了。

他站在阳台上。窗外的雪从推开的窗户缝里飘进来。落在他的肩膀上。他把笔记本打开。清了清嗓子——这个动作有些多余。大概是为了给自己壮胆。念自己的诗——以前是打死都不敢的事情。

"《不负》。"

"这是一首七律。"

他念了——

那年九月风初凉，
笔底春秋墨未央。
少日藏心无一字，
中年执手话偏长。
曾经别路千山外，
到底归途一水旁。
流年未必无情物，
不负当时一片心。

他念得很慢。一字一顿。念到"流年未必无情物"的时候声音低了一点。念到最后一句的时候几乎是在读给自己听的。

念完了。他把笔记本垂下来。手垂在身体一侧。

她没有说话。

她手里还攥着那把铲土的小铲子。塑料柄的。上面沾着土。她手指上的泥蹭到了铲柄上。

雪又飘进来两片。落在她已经换了新土的花盆里。白色的。在黑色的土上很显眼。

然后她往前走了一步。把头靠在他肩上。额头抵着他的颈窝。手上还有泥。蹭到了他的白衬衫肩头——一道灰黑色的泥印。她没发现。或者发现了但没管。鼻子埋在他衬衫的领子下。呼吸是热的。

他没动。让她靠着。左手举着笔记本。右手空着。过了一会儿右手抬起来——放在她背上。轻的。像放在一件易碎的东西上面。

他看着她肩头衬衫上那道泥印。没有拍掉。

窗外的雪还在下。落得很安静。飘在空调外机上——没声音。飘在楼下的冬青树上——落上去就不动了。白点越来越多，冬青树从绿色变成了斑斑点点的白绿色。

对面楼。有个老人趴在窗台上看雪。旁边楼有个小孩在阳台上喊:"下大了——"然后被大人叫回去了。

阳台上的茉莉花没有叶子可以接雪——只剩几根枝条，光秃秃的。雪落在枝上。落了一点。化了。又落了一点。

"肩膀上蹭了泥。"

她嗯了一声。没抬头。他知道她没打算擦。

"你当年写的《初见》——也是冬天。也是下雪。第一场雪。"她的声音透过衬衫传上来——闷闷的。能感觉到她说话时嘴唇在他肩头的布料上摩擦——很轻的震动。

"对。第一场雪。2010年12月。雪不大——跟今天差不多。下了大概两个小时就停了。我趴在窗台上看。窗子外面有一条电线——雪落在电线上。积了一条白线。然后一阵风。雪全掉了。"

"你在石桥镇那个小房间里。窗外能看到石桥。石桥东边第二个桥墩缺了一块——被河水冲刷的。但那天没水——河床干了好几年了。桥底下有一棵野构树——光秃秃的。一只鸟停在上面。然后飞走了。"

"你怎么知道我窗外能看到石桥。连桥墩缺口都知道。连野构树都知道。"

"你说过。大学时在信里说的。你描述了你的房间——书桌靠窗、窗外是石桥、河床干了、东边第二个桥墩缺了一块、底下长了一棵构树。你写了好几张——密密麻麻。字很小——像是在怕信纸不够用。"

"那封信你收到了。"

"收到了。"

"我以为你没收到。因为我后来写信你都不回了。写了好几封——都石沉大海。我以为你收到第一封之后觉得这个人太烦了——就不回了。"

"回了。每一封都写了回信。写完了。折叠好了。装进信封。信封上写了你的地址——楚州师院中文系。甚至在信角上写了'亲启'。然后在信封背面粘上了邮票。没寄出来。"

"为什么。"

"怕你觉得我太主动。一个女生主动写信给一个男生——那个男生三年不敢跟她说话。如果她主动了——他会不会觉得不对。会不会觉得——她不是那个安静的、他看了三年的苏晚晴了。"

他沉默了。窗外的雪把他的沉默填满了。一片。又一片。轻轻地打在玻璃上。没声音。

他的手在她背上又拍了拍。像拍一个小孩子。又像在拍一个做了错事不知道该不该认的人。

"你写了什么。"

"忘了。大概就是——楚州的冬天冷不冷。你加衣服了没有。再就是——'你的诗我看了'。"

"就这些。"

"就这些。"

窗外的雪越来越大了。从指甲盖大小变成了硬币大小。阳台的玻璃上蒙了一层雾。他伸手指在雾上画了一下——画了一道线。雾被划掉了——外面看得更清楚了。雪把院子里的地面全盖住了。白的。以前地上的裂缝、狗屎、烟头——全看不见了。

他转头看她。她还靠在他肩上。呼吸均匀。眼睛闭着。睫毛在抖。不是哭——是风吹的。阳台窗户还开着一条缝。

他伸手把窗户推上了。玻璃窗合拢的声音闷闷的——槽里有密封条。

"进屋。"

"再站一会儿。"

又站了几分钟。雪在外面落。暖气管在阳台角落里嘶嘶响。

她从他肩头上抬起头。他的衬衫肩头有一个灰印——泥干了。她伸手拍了拍。泥印没拍掉——搓了两下才淡了一些。

"衬衫毁了。"

"不要紧。我还有几件白的。"

她转身把花盆搬进屋里。搬的时候小铲子掉在地上了——当的一声。她弯腰捡起来。铲子把上刻着一朵小花——茉莉花。他刻的。大概是去年。不记得了。

然后她说了一句话。对着花盆。声音很轻。大概不是对他说的。但房间里很安静——他听到了。暖气嘶嘶响。窗外雪落无声。厨房里的水龙头大概没关紧——隔几秒滴一滴。花盆里的新土正在吸那半杯水——湿痕从中间往四边扩散。

"十二年。"

没有上下文。就这三个字。但她不需要上下文。他也从来不需要她的上下文。十二年前在教室后排——他不需要她回头就知道她在。她也不需要他说就知道他在看。十二年后在这间屋子里——他不需要她解释"十二年"是什么意思。他知道。是十二年。是从第一首诗到最后一首诗。是从一个人放学回家走冷巷子到三个人围着一张餐桌喝粥吃煎蛋。是从"此心已在阑珊处"到"流年未必无情物"。是从"怕你觉得我太主动"到"不怕了——都嫁给你了"。

他听懂了。

他走过去。把那盆茉莉花从她手里接过来。放在阳台的角落里。然后蹲下来。把洒水壶拿起来。往土里又浇了一圈水。水壶把上刻着那朵小花——去年刻的。她说像茉莉。他说就是茉莉。
