# 第176章 月湖边
时间：2019年5月1日傍晚
地点：青阳县月湖
卷次：卷九：执手之约

婚礼结束之后。宴会厅里还在热闹——有人在喝酒，有人在打包剩菜，有人在拉着新郎新娘拍照。

顾临风在她耳边说了一句话。声音很低。被大厅里的音乐和人声盖过了大半，但她听清了。

"去月湖。"

她没有问为什么。把头上的白纱摘下来，放在化妆间的椅子上。高跟鞋换了平底鞋——帆布鞋，那双穿了很久的，鞋带洗得发灰。然后两个人从酒店后门偷偷溜了出去。

穿着礼服和婚纱的两个人走在青阳县傍晚的街道上。

五月的晚风不冷不热。太阳开始往西边落——不是那种红彤彤的落日，是淡金色的。阳光斜斜地洒在街道上，把路边的梧桐树叶照成半透明。街上的行人不多——有人侧眼看他们，一个穿婚纱的新娘和穿西装的新郎在街上走，画面有些滑稽。

一个骑自行车的大爷经过——车铃铛响了两下。大爷回头看了他们一眼，说了一句"年轻人——结婚啊？"说完没等回答就骑走了。

月湖入口处。那棵老柳树还在。树干比去年粗了一圈——看不出肉眼的变化，但顾临风觉得它比六年前更粗了。枝条全绿了，垂到水面上，风吹过来就晃。柳条尖触到水面的时候激出一圈圈小波纹。

湖边的长椅还是那把。铁的椅脚生了更多的锈——红色的锈渍在灰色铁管上蔓延开来。椅面是木条的——有三根换了新的，颜色比旧的浅，能看出痕迹。但坐上去还是那种微凉而硬的触感。

苏晚晴先坐下来。婚纱的下摆在长椅边缘铺开——白缎上沾了椅脚的锈渍。她看了一眼。没拍。顾临风坐在她右边。西装裤管蹭到了椅脚上的锈，也沾了一小块褐红色。他也没拍。

湖面在落日下泛着金色。

是那种不是亮金也不是暗金——是水本身的深绿上浮着一层薄薄的暖色。金光从湖心向岸边蔓延，遇到波纹就碎成小块，小块又重组——不断地碎不断地重组。几只野鸭子在水面上慢慢地划——有两只靠得很近，头对着头，像在交换什么东西。

远处的水塔还站在那里。塔身是红砖砌的，几十年的风吹雨淋把红色洗成了暗棕色。塔顶是圆的，上面蹲着一盏灯——一盏老式航空障碍灯，红色的。太阳还没完全落下去，所以灯还没亮。

"六年前。"她说。"你在这里说暂时不要联系了。"

"不是你说的？"

"是我说的。"

她说完看了他一眼。他低头看着湖面。

"当时你说——'以后不会了'。"

"我说了吗。"

"说了。2013年8月。高考后。就在这个长椅上。我坐了左边，你坐了右边——跟现在一样。"

他当然记得。每个字都记得。她说"我们需要暂时不要联系了。到了大学，各自好好读书。"他说"好。"一个字的回答。但说完之后他在心里把所有没说出来的话全部过了一遍——像翻一本没有字的书，每页空白但很重。

"以后不会了。"他说。

这次的话跟六年前不一样。六年前那四个字是隐忍。现在这四个字是肯定。声调一样，音量一样，说话的人一样。但字已经变了——

六年前是"以后不联系了。"现在是"以后不会了。"不联系和不会——差了一个"会"字。这个"会"字用六年等来的。

她把头靠在他肩上。

婚纱的肩部面料滑滑的——缎面。他西装的肩垫硬硬的——劣质海绵。两种面料蹭在一起，发出很轻微的摩擦声。她的头不重——他几乎感觉不到重量。但肩上的感觉是温的。像高中冬天教室里暖气片忽然热起来那一瞬间——不是烫，是慢慢的、从外往里渗的暖意。

柳树的影子在水面上越来越长。从岸边到湖心——整片水面从淡金色开始往橙红色过渡。太阳沉下去的速度比中午快——像有人在天上加速拉窗帘。

"高一那年的秋天。我们在走廊接水。第一次对视。你端着水杯回去的时候，水洒了一路。"

"你看到了。"

"看到了。"

"我以为你没注意。"

"我注意了。你看我的时候我在看你。但我不敢让你知道——"

她的声音还是那么轻。好像在说一件很小的事。

"高二冬天的第一场雪。你站在窗边伸手接雪花。我站在你后面两米。那个画面我写了三首诗——都不好。全揉了。"

"你当时为什么不说话。"

"不知道说什么。"

他低头看着自己放在膝盖上的手。手指上有粉笔磨出来的薄茧——在黑板上写字留下的。她手上也有——位置不一样，她的在拇指和食指之间，因为拿粉笔的姿势跟他不一样。

"如果高三毕业你没表白。"

"我会后悔一辈子。"

她把头在他肩上蹭了蹭。不是撒娇——是点头。头在肩上动了一下的那个动作，不像点头也不像摇头。更像是一种确认——确认他说的是真的。

湖面上泛起了一层很淡的雾。傍晚的温差把水汽从湖面蒸起来——白色的，很轻，贴着水面飘。透过那层雾看到对岸的柳树变成了水墨画里的背景——虚的，但有形状。

水塔顶上的灯亮了。

不是全亮——是闪。一红一灭，一红一灭。节奏很慢，大概两秒一个周期。红灯映在湖面上——在深绿色的水面上投下一个变形的红点。红点晃动，被风吹成不规则的椭圆。跟六年前那晚一样——高考后她坐在长椅上说"我需要想一想"的那个晚上，水塔上的灯也这样闪。

"每次水塔的灯亮起来。我就想起你。"

他转头看她。她看着湖面。声音很低，像在自言自语。

"在楚州的时候。在宿舍阳台上能看到远处一个水塔。不是这一个——但也是红色的灯。每个晚上我都看着那个灯——然后想起青阳的月湖。想起高考后那晚。想起你坐在我旁边。"

她停了一下。

"后来相亲。我在茶餐厅等你。从窗户往外看——远处也有一个水塔。不是青阳的也不是楚州的。是另一种形状。但顶上也有一盏红灯。我当时想——如果你不来。我就当这个灯从来没亮过。"

"我来了。"

"嗯。你来了。"

她把手从他膝盖上拿开。伸进自己的婚纱口袋——婚纱居然有口袋，是她在裙摆内侧缝的，很小，只能放一张卡片。她从里面掏出一张照片。

民政局门口拍的。今天早上——出门前。两个人站在台阶上，穿着便装。他穿的蓝色衬衫，她穿的白毛衣。阳光从左边打过来，两个人没有看镜头——在看彼此。她的嘴角弯着。他的眼里有光。

"今天早上拍的。你还没看过。"

他接过照片。照片的边缘在晚霞中泛着淡红色——像加了一层极薄的滤镜。照片上的两个人站在一月的阳光里——不，那是五月。但他觉得他任何时候看她都是这个样子——从十六岁到现在，她在他眼里的样子从来没变过。不是脸没变——是那种感觉没变。

"这张照片——"

"嗯。"

"照得比结婚证上那张好。"

"因为没看镜头。"

他把照片翻到背面。背面白纸上有钢笔字——她的字:"2019.5.1。青阳民政局。我们结婚了。"

她把她爸的紫砂壶、她把妈妈的卤牛肉、她把他妈的红烧排骨、她把赵一鸣的领带、她把许若琳的眼泪——她把这些全都浓缩在这行字上。

他把照片放进西装内袋——胸口的位置。跟诗稿放在同一个口袋。

太阳完全落下了。水面上的金色褪尽。只剩下水塔上一个红色的点。一闪一灭。一闪一灭。再过两个小时月亮会出来——五月初的月亮不圆，但很亮。月光碎在湖面上，跟路灯的光搅在一起。那将是他们第六次在这个湖边看月亮——第一次在高考后，第二次在相亲后，第三次在求婚时，第四次在领证后，第五次在订婚前，第六次在今天。

柳树梢头最后一道光也消失了。天从橙红变成深蓝。

她站起来。婚纱下摆拖在地上——沾了草籽和一小片落叶。落叶是柳叶——细长、尖头、边缘有点焦。她从裙摆上把叶子摘下来，放在长椅扶手上。叶子在风里翻了个身。

"不管去哪。以后都要回来看这个湖。"

"我知道。"

"六年前说的。今天再确认一遍。"

"确认了。"

湖水在夜色里沉下去了。但水塔上的红灯还在闪。一红一灭。一闪一闪。像水塔在每隔两秒眨一次眼睛。
