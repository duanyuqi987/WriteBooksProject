# 第004章 初秋三楼旁观

所属部卷：第一部：初秋三楼
时间：2010年九月
地点：县一中三楼东侧教室

## 诗歌插页 P004：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
窗外法桐落下的第一片黄叶还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，初秋三楼会显得没有那么沉重。教室、走廊、操场里站着许多人，每个人都有自己的迟到、作业、饭卡和回家路。

叶澄后来记得的不是陆允川的沉默，而是他那天总在看窗外法桐落下的第一片黄叶。她觉得那样的眼神很熟，像很多人刚开始喜欢谁时都会有的样子：认真，又有点不讲道理。

叶澄在前排帮老师分表格，经过他们这排时敲了敲桌角，提醒新同学先写家庭住址，别把县城和乡镇填反。

程砚看得更直接。程砚把书包往桌上一扔，拉链没拉好，几本练习册哗啦滑出来。他笑得很轻，倒没有恶意，只是把陆允川那点紧张照得无处可放。他不喜欢把话说得太柔，因为太柔的话陆允川听了会继续躲。

“你这人真有意思，刚来就盯着窗外。”程砚说。“我只是看看座位。”陆允川说。“座位在屋里，窗外又不发课本。”程砚说。

夏知遥在这一章里也不该只是一个被看的背影。夏知遥进门时没有抬头，她抱着两本书，发梢被风吹得贴在脸颊边。她先问班长空座在哪里，声音不高，像怕惊动还没安顿好的早晨。她有时会因为题做不出皱眉，有时会在朋友讲笑话时笑出声，有时也会因为家里电话而沉默很久。

她没有义务知道陆允川每一次停顿。就像陆允川也不知道她书包里那张没交的申请表，不知道她冬天手指容易冻红，不知道她回宿舍前常在水房多站一会儿。

报到那天，三楼东侧教室的门一直敞着，风从走廊穿过去，把黑板上残留的粉笔灰吹成淡淡一层。陆允川把行李箱靠在墙边，先低头看鞋尖上的泥，再看讲台上摞着的登记表。

在旁观者眼里，陆允川从乡镇中学来到县城，第一天报到时在靠窗座位看见夏知遥走进教室。这件事只是一天里很短的一折。可短并不等于无意义，它只是提醒陆允川：意义不能只从自己这边生长。

窗框底下积着一线灰，手指摸过去会留下白印。班主任在门口叫名字，叫错一次，教室里就有人低低笑一声。陆允川跟着笑，又很快收住，怕自己笑得不像这个地方的人。

叶澄晚自习后对他说，喜欢谁不是错，但别让别人替你承担你没说出口的戏。她说完就去交作业，留下陆允川站在门口，手里捏着一本卷了角的练习册。

班主任拿着花名册站在门口，念到陆允川的名字时停了半秒，问他是哪个乡镇来的。他答得很快，答完才发现自己的普通话带着一点方言尾音，耳根立刻热了。

夏知遥坐下后把书放到桌角，又从笔袋里取出一支黑色水笔。她没有加入旁边女生的闲聊，只在表格上慢慢写家长电话，写错一个数字，又安静地涂掉。

午饭时程砚带他去食堂，指给他看哪个窗口米饭少、哪个窗口阿姨手稳。陆允川端着餐盘跟在后面，第一次觉得县城学校也不是传说里的光鲜，它一样有排队、油烟和找不到座位的狼狈。

晚自习前，叶澄把多余的登记表交给老师，回座位时顺手关了半扇窗。风小下来，法桐叶贴着玻璃滑过去，陆允川低头写名字，最后一笔压得很重。

旁观不是冷漠，而是把故事从一个人的胸口挪开一点。挪开以后，夏知遥有了自己的呼吸，程砚有了自己的烦躁，叶澄也有了不愿明说的担心。

陆允川不喜欢被旁观，因为旁观会让他的深情显得没有那么独一无二。但正是这种不独一无二，让后来的人生有了出路：大家都年轻过，也都误会过，不必把一次笨拙写成终身审判。

从叶澄的角度看，陆允川的心事并不神秘，只是太年轻，所以总以为别人都该看懂。他越沉默，周围人越不知道该不该靠近。

夏知遥也许注意到了一点异常，也许没有。她那天要处理的不是谁的暗恋，而是自己的作业、座位、朋友和回宿舍前必须打完的一通电话。

陆允川把报到单拿在手里，先把报到单填完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把报到单填完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把家庭电话写错一格后安静涂掉，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

三楼东侧教室里最明显的不是心跳，而是粉笔灰被风吹起的轻响。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，食堂铁盘里的番茄炒蛋把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。报到单、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

从旁边看，陆允川的心事并不神秘。它只是少年人常有的笨拙：怕被看见，又怕完全不被看见，于是每个动作都带着一点别扭。

夏知遥也许注意到了一点，也许没有。无论哪一种，她都不该被写成答案。她有自己的填报到表，也有不想向任何人解释的沉默。

这一段不靠告白推进，而靠报到单、食堂队伍和晚自习把人稳稳托住。陆允川会尴尬，会饿，也会在夏知遥转身以后继续学着适应这个班。

如果只写疼，这章会很薄；加上报到单、粉笔灰被风吹起的轻响和食堂铁盘里的番茄炒蛋，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在三楼东侧教室发呆。”陆允川把报到单收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

这一章的火被旁人的目光压低了。压低以后，才看得见灰里的温度，也看得见每个人不被爱情叙事吞掉的完整生活。
