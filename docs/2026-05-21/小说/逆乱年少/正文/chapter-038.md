# 第038章 面试

时间：2011年4月15日，下午

地点：江城大学大学生活动中心304

卷次：第二卷：冬天的图书馆

四月十五日，周五下午。大学生活动中心304门口排了十几个人的队。

林玉起提前一小时到的。他以为自己是第一个，结果前面已经站了五个人——四个女生一个男生，有的在低头看稿子，有的在小声念着什么。每个人手里都拿着一张打印好的报名表，捏得纸边已经出了褶子。

他排在第六位。走廊里的椅子不够坐，他被挤到墙角站着。从三楼窗户看出去，能看见操场——跑道上有田径队的人在训练，穿着红色背心，一圈一圈地跑。四月的下午，阳光从窗户照进来，在走廊地板上投下一个个长方形的亮块。

面试的时间比预计的要长。前面五个人进去又出来，每个大概十分钟。轮到他的时候，已经下午两点四十分了。

"林玉起？"门口一个短头发的女生喊他的名字。

"在。"

"该你了。"

他深吸一口气，走进304。

房间不大，一张长桌，三把椅子。正中坐着蒋慧——大三的学姐，看起来不太好惹的样子。左边是孟书恒，戴着黑框眼镜，文件夹打开在面前。右边是赵今越，手里拿着一支笔，面前的笔记本上已经记了密密麻麻的半页字。

"林玉起，文学院，汉语言文学，大一。"蒋慧念完报名表上的基本信息，抬起头看着他。"你是中文系的，为什么要来支教？"

"我来自封丘县城。"他开口了。第一句话说出来，就发现自己的声音有点发干。"我家不在村里，但是县城的教育资源和城里比还是有差距的——我中考的时候，英语听力也是用那种很老的录音机放的，杂音很大。去年高考，我们县有好几个成绩不错的人因为英语听力失分，本来能上一本的最后去了大专。"

蒋慧看了一眼孟书恒。孟书恒的表情没有变化。

"你在报名表上写——你想亲眼看见孩子们需要什么，然后写出来。你觉得写作能改变什么？"孟书恒推了推眼镜。

"不是改变。是记录。"林玉起说。"鲁迅写过很多乡下的事——祥林嫂、阿Q、闰土。他没有改变那些人的命运，但他让更多人看到了。看到了，就可能有更多人去改变。"

赵今越在本子上记了一笔。蒋慧换了一个问题："你觉得自己能在那种环境里待一个月吗？没有空调，没有热水器，厕所是旱厕。你做好准备了吗？"

"做好了。"他停了停，又说，"我高中三年冬天都是用冷水洗脸的。不是因为我家没有热水器——是因为我妈说我体质不好，洗冷水增强抵抗力。"

这个回答不在他准备的稿子里。但他话说出口以后觉得，比稿子上写的任何一句话都真实。

面试官三个人交换了一个眼神。

"如果你被分在同一个支教点，然后跟你分到一起的人里有你不太喜欢的人，你会怎么处理？"蒋慧问。

问题很尖锐。这不是一个能在网上搜到标准答案的问题。林玉起想了大概五秒钟：

"我会跟他一起吃一顿饭。"

"就这？"

"一顿饭不能解决所有问题，但至少可以让你们知道——对方也是个活人，也有自己讨厌的事情和喜欢的东西。剩下的，在相处的每一天里，一点一点磨。"

蒋慧点了头。幅度不大，但能看出来是一种认可。

"你有什么特长？"赵今越问。

"我会写诗。现代诗和格律诗都会一些。"

"格律诗？"孟书恒来了兴趣，"就是那种押韵、对仗——"

"五言、七言。绝句、律诗。平仄、对仗、用韵。我读杜甫比较多，他的《秋兴》八首我能背六首。"

孟书恒笑了一下，转过头对蒋慧说："这个好。支教的时候可以给小孩开个诗词课。不是那种正儿八经的，就是教他们玩——对对子，猜字谜，读李白。"

"你自己觉得你能教小学语文吗？"蒋慧问。

"我觉得能。小学语文最重要的不是教知识，是教兴趣。一个小孩如果觉得读书有意思，后面就不用教了——他会自己读。我五年级的时候老师让我们背《静夜思》，全班只有我一个人去翻了李白别的诗来看。"

面试结束了。他从304出来的时候，走廊里还坐着七八个等待面试的人。太阳已经偏西了，阳光从走廊的玻璃窗照进来，把地板上的长方形亮块拉得更长。

他走到楼梯口，靠着扶手缓了口气。他觉得自己表现还可以——不是那种可以拿满分的面试，但至少没有结巴，没有手心出汗湿了报名表，没有说出"因为她去了"这种荒唐话。

他不知道苏安晴的面试是什么时候。也许比他早——上周六？也许是同一天但不同的时间段。他们在同一栋楼里，她先进去，他后进去。也许她出去的时候他还没来，也许她刚走下楼梯他就从上楼。

他下楼以后，在大活一楼的公告栏前面又停了一下。支教团的招募海报还在右下角，只是旁边多了一张新的公告："暑期支教面试通知 —— 面试结果将于下周一公布，请各位同学留意大学生活动中心公告栏。"

下周一。三天以后。

他把公告拍了下来，发给了郭向阳。郭向阳回了一个大拇指的表情，然后问："怎么样，面试官没有看出来你喜欢苏安晴？"

他回："没有。"

实际上他觉得蒋慧看出来了。那个问题——"如果你被分在同一个支教点，然后跟你分到一起的人里有你不太喜欢的人"——她问的时候没有看他，看的是桌上的文件夹。但她在"同一个支教点"上稍微停顿了一下。

他不敢往下想。
