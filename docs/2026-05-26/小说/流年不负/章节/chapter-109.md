# 第109章 大四

时间：2016年9月

地点：楚州师院

卷次：卷六：平行轨道

二零一六年九月初。大四。

开学的第一天。顾临风走进校门。低头看了一下自己的脚——四年前也是这双帆布鞋走过的路。鞋是新的、旧的一直换了四双。路还是那条路。

512宿舍。四个床位三个满了。靠门的上铺空着——阿坤彻底搬走了。李博在床上贴了一张纸条："此铺待租，月租零元，包热水。"没人来。

小伍在翻手机。"阿坤说他找到工作了——在老家一个公司做文员。一个月两千八。"

"挺好。"顾临风说。

李博从上面探出头来。"兄弟——你想好了吗。考研还是找工作。"

"考研。"

"哪个学校。"

"楚州师大。"

李博愣了一下。"楚州师大？不是——你成绩还可以。干嘛不报个更好的。"

"离家近。"

李博看着。张了张嘴。没再说话。

顾临风打开电脑。笔记本电脑是他去年暑假打工买的——戴尔的旧款，银灰色外壳。屏幕亮度已经不太好了。他打开浏览器，输入网址。

研究生招生信息网。

登录。注册信息。填写。

姓名。身份证号。本科院校——楚州师范学院。毕业时间——2017年7月。

报考院校。

他在搜索框里打入"楚州师范大学"。

页面跳转。学校的招生简章在最上面。红色标题。下面是一行小字——"楚州师范大学研究生院欢迎您。"

他往下翻。专业目录。文学院。中国古代文学。招生人数：6人。

他把鼠标移到"填报"按钮上。

窗外有人在打篮球。声音很响——球撞篮板、落地、弹起来。有人在喊"好球"。

他没有马上填。

他在想一个问题。

为什么要考这个学校。

楚州师大在省内的确不是最好的。中文系的师资也不如省师院。考楚州师大——可以说是"求稳"。也可以说是"躺平"。还可以说是"离家近"。

但他知道不是。他知道自己为什么要填楚州师大。因为她在楚州师大。因为她去年在楚州师大教育学院。因为她现在大概还在那里——或者已经毕业了，在楚州一小教书。

但他不会承认。连对自己都不会。他对自己说的是——楚州师大离家近。他对自己说的是——楚州师大考上的概率更大一些。他对自己说的是——楚州是个好城市，他想留在那里。

他不需要看她离校的日期。他不需要查她宿舍的门牌号。他甚至不需要知道她现在还在不在学校里。他只需要把这个名字填上去——"楚州师范大学"——就好像把一条线绑在一条轨道上，让两根铁轨在同一座城市里并行。

但两根铁轨不会交汇。

他会读他的研究生。她会在她的小学教书。他们之间依然隔着那条街——二十分钟车程。依然没有任何联系。

但是至少——是同一所学校了。

他点了提交。

屏幕弹出一个确认框。"确定报考楚州师范大学中国古代文学专业吗？"

鼠标指针在"确定"上停着。

然后他点了。

确认框消失。页面跳转——"报名信息已提交，请等待现场确认。"

他把胳膊放在桌上。手撑着下巴。看着屏幕。

窗外又有鸟叫。不是麻雀了——是八哥。黑色的黄的嘴。在窗台上跳了两下飞走了。

他低头看着键盘。手指放在键面上。没有打字。只是放着。

很久。

李博在背后看到了。"所以你真的报楚州师大了？"

"嗯。"

"为什么。"

"离家近。"

李博没有追。但他看顾临风的眼神里有几秒的停留。然后他把视线转回自己的电脑。"反正兄弟支持你。"

晚上他躺在床上。新被子。旧的。他妈把高一的被子重新缝了一遍——加了一层棉絮。比原来厚了。他把被子盖到胸口。手搭在肚子上面。

新床单。洗过一次。还有洗衣液的味道。但不是她洗衣服的那个牌子。他高一住宿舍的时候发现苏晚晴衣服上的洗衣液有一种特别的味道——他一直没找到它叫什么。

他翻个身。膝盖撞到墙上。那面墙凉凉的。

他在心里想——现在报了她的学校。然后呢。

见到她怎么办。她说"你怎么在这儿"怎么办。她说"真巧"怎么办。她说"我快要毕业了"怎么办。

他没有这些问题的答案。但他还是报了。

不是因为想见她。不是因为要追她。不是因为觉得还有机会。只是因为——他觉得在同一所学校里待着，呼吸同一座城市的空气，看着同一排法国梧桐叶子黄了又绿，已经是一件事了。他已经想好了——他不会去找她。不会刻意去教育学院门口等她下课。不会打听她住哪栋宿舍。

他只是想在楚州师大的校园里走一走。想象她走过的路。她曾经在图书馆三楼靠窗的位置看过的日落。她曾经在哪个食堂吃过午饭。她曾经在哪条林荫道上看书。

然后他过他的日子——上课、泡图书馆、写论文、代课。她过她的日子——教书、改作业、下班骑车回家。井水不犯河水。两条轨道平行延伸。不交汇。

他闭上眼睛。睡着了。

梦里有一条很长的走廊。两边的门都关着。他在走廊里走，脚步声在墙上弹来弹去。走了很久——走廊还是那一条。门还是那些门。
