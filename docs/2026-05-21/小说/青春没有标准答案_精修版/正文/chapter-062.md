# 第062章 月考排名



时间：2009年一月下旬，期末模拟后



地点：十班后墙和英语办公室门口



卷次：第三卷：雪落在两栋楼之间



2009年一月下旬，期末模拟后，十班后墙和英语办公室门口，被手指按出印子的成绩条先把这一天按进了十班的日常里。期末模拟排名出来，雪仗后的轻松被分数再次压回去。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

刘青岚英语进步，却总分仍不显眼；段璟物理上来，语文又拖住。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看被手指按出印子的成绩条，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

刘青岚也在这一天被推到前面。刘青岚的特点是英语听力好，说话轻，女生宿舍里常补最后一句。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“英语涨了，总分还是没动。”刘青岚说。

“一科一科来，不要用总分抹掉进步。”马立群说。

“我语文又掉了。”段璟说。

“你作文开头太急。”常俊凡说。

“他说得对，急着感动自己。”叶书兰说。

排名表贴上去的时候，教室先安静了两秒，随后才有人装作随便地往后墙走。

有人只看自己，有人先看同桌，有人嘴上说无所谓，眼睛却在前二十名里来回找。

段璟找到自己的名字后，手心里出了一点汗。他没立刻找常俊凡，怕自己找得太明显。

常俊凡看排名很快，看完就回座位，把试卷翻到错题页，像把自己从墙上那张纸里领回来。

邓桥在旁边一遍遍算单科差距，铅笔头敲着桌面，敲得前排都忍不住回头。

刘青岚把英语卷子折好，声音很轻地说，进步也得有人看见才行。

马立群把听力题重新放了一遍，磁带机卡了一下，班里没人笑，因为大家都知道笑完还得听。

这一天的分数把十班切开，又把他们拉回同一间教室。谁都不能只靠名次活，也不能假装名次不存在。

月考排名没有悬在空处。后墙红榜的纸角和同学们压低的呼吸，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，月考排名带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

马立群进门时不一定立刻说话，鞋底在水泥地上一停，月考排名带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在刘青岚快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。月考排名越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。被手指按出印子的成绩条让他想说话，又逼他先低头看题。

刘青岚被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，月考排名留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。刘青岚路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为月考排名翻身，谁还在被窝里背单词，旁边人都听得见。

有人说将来要去新乡、郑州，也有人只盯着红榜上那一行小字，先把下一张卷子改完。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理月考排名里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：被手指按出印子的成绩条、一只冻红的手、一句老师点名、一块还没扫干净的雪。

后墙的排名表贴在一张红色底纸上，用胶带粘了四角。胶带是段璟贴的——周明澈让他帮忙，他把左边两个角对齐了，右边两个角偏了半厘米。偏了的那半厘米后来被好几个人用手指来回抚过，像反复摸一块不会愈合的疤。

贴排名表的时候是上午第三节下课。陶建国把成绩单交给周明澈时什么也没说，只把折着的纸放在他桌上，用手指点了点纸的左上角——意思是"角要对齐"。周明澈找了段璟帮忙，两个人一个扶着纸，一个撕胶带。撕胶带时用力太猛，胶带头贴回了胶带卷上，段璟用指甲抠了五秒钟才抠开。

"你抠胶带的水平跟你写作文的开头一样——太急。"付小川在后排说。

段璟没理他。但胶带抠开之后，他在贴第四个角的时候故意放慢了一倍速度，慢到周明澈说"你手停了吧"，他才往下一按。

段璟在排名表上从下往上找自己。先找到名字的是刘青岚——她的英语从第三十一蹦到了第十九，但总分还趴在第四十二。一张红纸同时告诉了她两件事：你进步了，但还不够。她的手指在英语那一栏上停了很久，指腹在"19"那个数字上轻轻磨了又磨。

"英语涨了十三名。"常俊凡站在她旁边，盯着刘青岚的手指说。

"总分只涨了四名。"刘青岚把手放下来。

"涨了就是涨了。"常俊凡说，"十三名不是往总分前面走了四名的代价，是下一回还能往上走的原因。"

刘青岚没有回答，但她把成绩条从口袋里掏出来，在英语分数旁边画了一个小星星。星星是五个角都画满了的那种，不是那种只画一笔敷衍的。

段璟找到自己名字时，手心里出了一点汗。物理涨了，语文掉了——作文扣了十五分，比上次多扣了四分。这四分把他从全班第十八拉到了第二十二。他想，原来四分之一张卷子的分差可以改变四个名次，就像半个字可以在他心里来回滚几百遍。

"你作文开头太急。"常俊凡的声音从他肩膀后面传过来。她已经看完自己的排名了——全班第九，稳定得不像话——然后往回走的路上顺便看了他的成绩条。

"你怎么知道我开头急？"

"你上次作文写'人生就像一场考试'，这次写'时光荏苒，白驹过隙'。"常俊凡把他成绩条上的语文栏指给他看，"这两个开头都有一个毛病：还没说清楚要写什么，先急着把最大的词搬出来。"

段璟低头看自己的成绩条。她连他上次作文的开头都记得。

"那怎么改？"

常俊凡从窗台上拿了一支粉笔，在窗玻璃上写了三个字："先写事。"然后用手擦掉了。擦完她回了座位，翻出错题本开始分类——她有一页专门整理"作文扣分类型"，分为"开头太远""中间跳步""结尾拔太高"和"字迹"四栏。

邓桥坐在段璟旁边，从排名表下来以后就没说话。他的成绩条被铅笔头压在桌上，纸角有七八道指甲按出的白痕。他把物理选择、化学填空、数学大题的分数反复加了三遍，每一遍都差两分。

"别算了。"段璟说。

"我没算总分。我在算单科差距。"邓桥把笔放下，"数学差三分就进前十了。化学差两分。"

他的声音没有抱怨，只是在陈述数字。段璟忽然发现，邓桥是这个班上对自己最精确的人——他知道自己哪一科需要多少分，就像会计对账，一边入一边出，永远在算平衡。但分数不像账，分数背后还有运气、状态和临场那道完形填空的出题人心情。

马立群抱着录音机走进教室。他的英语录音机是封丘一中目前最老的一台——磁带仓的盖子掉了，他用橡皮筋勒着；暂停键按下去有时候弹不起来，要用指甲往上拨一下。今天他放的是期末模拟的听力复盘，磁带机转了不到一分钟就卡住了。马立群用指甲一拨暂停键，磁带又重新转起来。

"上次卡了有人笑，"马立群说，"今天不笑了？"

班里没人答话。不是因为纪律变好了，而是因为大家已经习惯了——封丘一中的东西总要修修补补，暖气管、录音机、值日牌、也包括每次考试以后需要重新拼起来的信心。

马立群把听力题的第三遍重放完，走到刘青岚桌边，用手指敲了敲她的英语卷子。

"你的听力，全对。"

刘青岚抬起头。

"你是怎么听的，可以把方法写在错题本上，我下节课让你讲两分钟。"马立群说完就拎着录音机走了，磁带仓的橡皮筋弹了一下，差点打中他自己的手腕。

刘青岚低头看英语卷子，那只小星星旁边又多了一条短短的横线——她自己加的。段璟从侧面看见，觉得那条横线像一条起跑线。

食堂中午的菜谱破天荒地没换：白菜炖粉条连着第三天登场。付小川端碗坐下时说："我怀疑食堂大师傅跟白菜签了合同。"梁一帆接了一句"你替白菜打官司吧"。两个人一唱一和，把周围几个人的筷子都笑歪了。

常俊凡今天吃饭时没像平时那样看错题本，而是端着碗听宋乔讲隔壁班怎么排的名次。宋乔的语速配合着夹菜的频率——夹一下白菜说一句排名，夹一块鸡块又说一句。常俊凡偶尔点一下头，那种"点在信息真正的缝隙里"的点头，宋乔已经能识别了。

段璟打饭时故意排到了邓桥后面。邓桥端碗的时候还在自言自语"差两分"，段璟说"你再算这碗白菜就凉了"。邓桥把碗端起来喝了一口汤，然后把成绩条折好塞进了口袋——不是扔掉，是暂时收起来。

晚自习前，常俊凡把错题分类方法借给了三个人：刘青岚一份打印好的表格、段璟一张手写的索引、邓桥一个活页本。三样东西合在一起就是完整的"常俊凡式学习法"：表格负责统计数据，索引负责定位题型，活页本负责随时插入新的错误。

段璟接过那张手写索引时，发现纸的边缘裁得不齐——常俊凡是用直尺和裁纸刀自己裁的。裁纸刀是文具店里最便宜的那种，刀片有点松，所以纸边起了毛。段璟用拇指沿着毛边摸了一遍，觉得这条不齐的边缘比买来的任何本子都好看。

"你帮邓桥做活页本的时候，也在里面写了例题？"段璟问。

"写了两道。"常俊凡把活页本最后一页翻给他看，"一道是容易忘加单位的，一道是抄错符号的。这两种错误看起来简单，考试的时候最容易犯。"

她给邓桥的活页本最后一页写着两道例题，旁边用了红蓝两色标注。红色标的是"错因"，蓝色标的是"正解"。排列得比黑板上的板书还清楚。

段璟在操场和常俊凡擦肩而过时想问她为什么要帮每个人分类，但他看了她一眼，忽然不需要问了——她的表情跟雪仗那天检查梁一帆有没有捏太硬的雪球时一模一样。那不是一个在付出的人的表情，那是一个总觉得别人漏了什么应该补上的人的表情。

下午有一阵子，教室里的日光灯因为电压不稳闪了三次。第一次闪的时候大家都抬头看灯管，第二次有人"啧"了一声，第三次没人抬头了——十班对坏东西的容忍度一向很高。梁一帆说这是"封丘一中的自适应能力"，赵远航说你这措辞可以写进作文。梁一帆回了两个字："写屁。"

段璟在灯闪的间隙里把作文重写了一遍。这次他的开头是："后墙的胶带偏了，我贴的，偏了半厘米。"常俊凡说先写事，他写了事。他发现自己一旦不急着把最大最亮的词摆出去，句子反而走得更远——像走路，抬头看目标的时候容易踩空，低头看脚下的时候反而能走出直线。

梁一帆也站在后墙看了排名。他的英语上次全班垫底，这次进步了七名。但他的英语老师马立群什么都没说，只是在发卷子时把他那张放在第一页，用红色笔圈了一道听力题的序号——那题全班只有他一个人做对了。梁一帆看见了红圈，但没说话。他把卷子折起来时，折痕刚好穿过那个红圈的中心，像瞄准镜的十字。

"你那个题是怎么听对的？"付小川凑过来。

"那个单词我认识。"梁一帆说。

"什么单词？"

"bridge。"梁一帆念得不太标准，中间的d发成了t。但他不在乎，因为他确实认识这个单词——他在黄河大堤上看到过一座老桥，桥头的石碑上刻着"bridge"的英文注释，那是他寒假帮外婆拉煤时无意中看到的。

段璟听见这段对话，想起了自己在公交车上写"常"字被涂掉的那个瞬间。每个人都在用自己的方式记住一些东西——一座桥、一个字、一个偏了半厘米的胶带、一个没被说破的英语单词。

宿舍熄灯后，邓桥又把手电拧开了。他不是在背单词，是在把成绩条上的数字重新算一遍。这一次他在每科旁边标注了"差距来源"：数学是"大题第三问步骤分丢了"，化学是"选择题粗心"，语文是"阅读题没看清设问"。他写得比老师批改还要详细，像在给分数做尸检。

段璟从下铺探头看了一眼，小声说："邓桥，你知道你算这些的样子像什么吗？"

"什么？"

"像陈柏言列扫雪计划表。"

邓桥愣了一下，然后把手电往下照，光柱在段璟脸上晃了一下："那你写作文开头的样子像付小川撕胶带。"

两个人在黑暗里各自笑了一声。隔壁床的赵远航翻了个身，说你们再不睡明天政治课打瞌睡别怪我没叫你们。

晚自习铃响后，教室里的日光灯还是老样子——嗡嗡响几秒才全亮。段璟翻开作文本，在第一行写了两个字："今天。"然后停下来。常俊凡说"先写事"，他就从今天发生在教室后墙的事写起：红色的底纸、偏了半厘米的胶带、被很多人用手指来回抚过的纸角。

写了三行，他把"今天"两个字划掉，改成了"红榜上的胶带偏了"。然后在旁边加了一行小字："她记得我上次作文的开头。"

他没有把这句话圈起来，也没有涂掉。他用透明文件袋压在那行字的上面——就是昨天在文具店买的那个——然后翻开数学练习册。

明天还有物理课，后天有英语听写，下星期是真正的期末考试。排名只是期中的一次暂停，不是终点站的名单。段璟知道，就像他知道自己喜欢一个人的时候，也应该像考试一样：先看清题目，再选择答案，不因为想得分就乱蒙。

月考排名之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

刘青岚没有因此变成特别醒目的人，可在段璟眼里，刘青岚从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，常俊凡没有安慰所有人，只把错题分类方法借给需要的人。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
