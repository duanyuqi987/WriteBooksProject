# 第027章 第一次说话

所属部卷：第一部：初秋三楼
时间：高一末
地点：楼梯转角

## 诗歌插页 P027：原创仿唐绝句

风影微明照旧尘，
少年心事隔新春。
回看不见当时路，
一角书声认故人。

事情真正发生时，并没有陆允川想象得那么郑重。夏知遥捡到陆允川掉下的笔记本，递给他时只说了一句你的本子。

楼梯转角常年有风，尤其晚自习前后，整栋楼的人一齐上下，脚步声像一阵急雨。墙角贴着消防示意图，边缘被人抠起一点。

陆允川抱着一摞书下楼，最上面的笔记本滑出去，先撞到扶手，再啪一声落在台阶上。那声音不大，他却像被叫了名字。他越想装得平常，动作越显得多余。手指碰到纸边、桌角、书脊，哪一样都比他的表情诚实。

夏知遥正从下面上来，弯腰把本子捡起。她没有翻开，只用手掌拍掉封面上的灰，递过来时说，你的本子。

“你的本子。”夏知遥说。“谢谢。”陆允川说。“下楼慢一点，书容易滑。”夏知遥说。

程砚在后面憋笑，憋到走出半层才忍不住，说你刚才那句谢谢像背课文，标点都背出来了。

他当然知道程砚是在提醒他。可人在心事最热的时候，很难分清提醒和打断，只觉得旁人都站在一条更凉的岸上。

叶澄站在楼梯平台等同桌，看见这一幕后没有起哄，只把快掉下来的练习册往陆允川怀里推了推，说先拿稳。

班级仍按自己的速度往前走。老师催交作业，值日生擦黑板，后门口有人借校牌，有人因为饭卡找不到翻遍整个书包。没有哪一件事为陆允川停下来。

也正因为没有停，这一刻才显得可信。笔记本封面上一道浅浅的折痕被他看见，被他放大，又被日常一点点压回原来的大小。

回到教室，陆允川把笔记本放进抽屉，过一会儿又拿出来。封面上那道折痕很浅，浅到别人根本不会注意，他却用指腹摸了好几遍。

陆允川那时还不会把喜欢说成一句完整的话。他只会绕路、停顿、假装翻书，像很多笨拙的少年一样，把靠近做成一串别人未必能读懂的小动作。

楼梯间的灯坏了一盏，光从上一层斜下来，人影被拉得很长。有人抱着篮球从旁边挤过去，橡胶味和汗味混在一起，把那句谢谢衬得更加短。

那一摞书其实不重，重的是他下楼时总想快一点。前面有人堵住楼梯，他停得太急，最上面的笔记本便从臂弯里滑出去，像替他做了一件他一直不敢做的事。

夏知遥递本子时，指尖没有碰到他的手。陆允川却仍然僵了一下，僵得程砚在背后差点笑出声。她说完“下楼慢一点”，就侧身让过一群往上跑的男生。

回到座位后，陆允川把“谢谢”两个字在草稿纸上写了一遍，又觉得自己傻，揉成团塞进桌斗。纸团滚到最里面，和几张旧小测挤在一起。

叶澄晚自习后问他，本子没摔坏吧。她问得自然，陆允川却答得很慢。程砚在旁边替他说，没坏，人差点坏了。三个人都笑了，笑声很轻。

这一刻真正有力量的，不是心跳有多响，而是它嵌在一堆普通事情里。有人催他让路，有人问他借笔，有人把椅子碰到他膝盖；世界越不配合，他越意识到自己不能把心事演得太满。

夏知遥没有给他特殊回应，也没有刻意冷淡。她只是按照自己的节奏把事情做完。这个事实在当时让陆允川失落，许多年后却让他松了一口气，因为真实的人本来就不该只负责别人的想象。

真正撞上这一刻时，陆允川反而没有准备好的台词。他只听见自己衣料摩擦书包的声音，还有程砚在旁边压低的一声笑。

夏知遥的反应很平常，平常到几乎让他失望。后来他才懂，这种平常不是冷淡，而是她没有被他的想象绑架。

陆允川把笔记本拿在手里，先把笔记本夹回书堆，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把书抱稳，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把本子递回来时没有翻开里面的字，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

楼梯转角里最明显的不是心跳，而是脚步撞在水泥楼梯上的回声。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

晚饭后，塑料袋里晃动的酸奶把人从心事里拽出来。程砚说再大的事也得先回教室，陆允川拎着袋子，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。笔记本、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

真正发生的那一刻，陆允川没有准备好的台词。他只来得及把笔记本攥紧，又很快松开，怕自己连手上的动作都显得用力。

夏知遥没有给他特殊回应，也没有故意回避。这个平常的事实让他短暂失落，却也让这一章不必靠夸张成立。

这一段真正留下的不是一句谢谢有多重，而是本子被捡起、书重新抱稳、楼梯重新通行。生活没有为他停住，却给了他一次学会分寸的机会。

如果只写疼，这章会很薄；加上笔记本、脚步撞在水泥楼梯上的回声和晚饭后塑料袋里晃动的酸奶，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在楼梯转角发呆。”陆允川把笔记本收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

章节落下去时，他没有得到答案，却比早晨多明白一点：一个人可以被打动，但不能因为被打动，就把世界看成只围着自己转。
