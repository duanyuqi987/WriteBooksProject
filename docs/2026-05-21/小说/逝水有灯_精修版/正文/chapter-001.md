# 第001章 初秋三楼之前

所属部卷：第一部：初秋三楼
时间：2010年九月
地点：县一中三楼东侧教室

## 诗歌插页 P001：原创现代诗

我把早晨放低，
让一线水从课桌边慢慢经过。
你的名字没有被我喊出，
只在窗外法桐落下的第一片黄叶旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写初秋三楼抵达之前的坡度。报到那天，三楼东侧教室的门一直敞着，风从走廊穿过去，把黑板上残留的粉笔灰吹成淡淡一层。陆允川把行李箱靠在墙边，先低头看鞋尖上的泥，再看讲台上摞着的登记表。

他把新发的校服外套搭在椅背上，又把桌斗里的碎纸一点点捡出来，像这样做就能让自己快些属于这个班。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。

那天真正的事还没有发生，窗外法桐的叶子却先被他记住了。那片黄叶很小，小到放在任何人的日子里都不显眼，偏偏在陆允川这里有了重量。

"你这人真有意思，刚来就盯着窗外。"程砚从后排探过头来。
"我只是看看座位。"陆允川把目光收回来。
"座位在屋里，窗外又不发课本。"
"你不懂，我先看看风景。"陆允川声音不大，但嘴角动了一下。
程砚哼笑一声，没再追问，转过身去和旁边的人聊球赛了。

程砚的座位靠过道，他坐下时椅子发出刺耳的刮擦声。他把书包塞进桌斗时，拉链卡在半截，拽了两下才拽开，里面掉出一包没拆封的饼干。他撕开袋子，掰了一半递给陆允川："没吃早饭吧？看你脸都白了。"陆允川接过饼干，没说自己早上五点就醒了，在汽车站的长椅上坐了一个小时才敢过来。

教室渐渐坐满。叶澄是班长，她从前排站起来，抱着一摞淡绿色的表格一张张往后传，传到他这排时轻声说："家庭住址写现住址，别写成老家。"她手指点了点表格上的栏目，又补充："县城的地址在第一行，乡镇的在第二行，有人上午填反了，多写了两遍。"陆允川点点头，心想自己差点也要填反。

课间十分钟被铃声切得很碎。有人挤去小卖部买面包，有人趴在桌上补觉，有人拿着水杯一路跑向水房。后排两个男生在争论昨天晚上的球赛，声音大到班主任从门口探头看了一眼。陆允川站在这些声音中间，心里那点紧张没有消失，却被周围的噪音稀释了一些。

夏知遥这时还只是人群里的一个名字——她进门时没有抬头，抱着两本书，发梢被风吹得贴在脸颊边。她走到讲台边问班长空座在哪里，声音不高，像怕惊动还没安顿好的早晨。陆允川看见了，却没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子，又一行行划掉。留下来的，是教室里真实的光、真实的灰、真实的不知道该把手放在哪里。

班主任姓方，四十出头，头发梳得一丝不苟。他夹着一个蓝色文件夹走进来，没有拿花名册念名字，而是让每个人走到讲台前报名字——他说这样记得快。轮到陆允川时，他报完名字，方老师看了他一眼，问："哪个乡镇的？"陆允川答得很快，答完才意识到自己的普通话里夹着一点乡下口音，耳朵立刻烧起来。方老师倒没说什么，只是在本子上记了一笔，让他回去坐着。

夏知遥找到靠后的座位，把书包挂在椅背上，从笔袋里翻出一支黑色水笔。旁边女生凑过来问她暑假去了哪里，她简单应了句"就在家"，便低头继续填表格。写到家长电话那一栏时，她停了一瞬，像在确认号码，写完以后轻轻把笔放下。

第一节下课后，程砚拉着陆允川去食堂认路。"你记住了，"他在排队时指着窗口，"三号窗口阿姨手不抖，四号窗口饭多但菜少，一号窗口回回都排最长——因为她们班的几个女生爱站那边。"陆允川端着餐盘跟在后面，第一次觉得县城学校也不是传说里的光鲜，它一样有排不完的队、呛人的油烟和端着饭找不到座位的狼狈。坐下以后，程砚一边往饭上浇菜汁一边说："你在镇上吃的什么？"陆允川说："家里带。"程砚筷子一停："那你今天是第一顿食堂？"陆允川点头。程砚笑了一声："那你得先练胃。"

晚自习前，叶澄把收齐的登记表交给方老师，回座位时看见窗户还开着，顺手推了半扇。风小下来，窗外的法桐叶贴着玻璃滑过去。陆允川低头在表格最后一栏签名字，最后一笔压得比前面都重。

陆允川后来才知道，所谓开始并不是一个明确的按钮，而是很多个微小选择叠在一起。他没有说话、没有靠近、没有确认，可他已经在心里给那个人留出了一块地方——一片黄叶落下来，刚好掉在那块地方的边界上。

晚自习结束，程砚收拾好书包说："走了，明天早读别迟到，方老师记名字的。"陆允川说了声好，但没有立刻站起来。教室里的人渐渐走光，值日生拖着扫帚从过道里经过，扫帚的竹枝刮过地砖，发出干燥的沙沙声。

他把桌上的表格又看了一遍，确认每一项都填对了，才把笔帽合上，起身往宿舍走。楼道里风从窗缝挤进来，把他的校服袖口吹起一点。他下意识按住，像按住一句还没成形的话。

回到宿舍，室友们已经在分瓜子吃。有人问他叫什么名字、住哪个镇，他一一答了。灯灭以后，走廊里还有拖鞋声和压低的笑声。陆允川把被子拉上来，闭上眼。今天发生的事很小，可他知道明天会不由自主地走进同一间教室，坐在同一个位置，看向同一个方向。
