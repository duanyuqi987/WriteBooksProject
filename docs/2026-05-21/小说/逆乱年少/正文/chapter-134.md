# 第134章 家访

时间：2012年7月20日，上午

地点：李家沟村至山沟深处的小路

卷次：第六卷：雪落无声

李铁蛋两天没来上课。

他是个黑黑瘦瘦的男孩，九岁，在低年级班。上课的时候永远举手最快——虽然答对的概率不高。他坐在第一排靠窗的位置，那个位置空了两天，像一个缺了的门牙。

苏安晴问村长李德厚。李德厚说，铁蛋家住在前面的山沟里，叫核桃沟，"走路要三里地"。

"他家里还有一个奶奶，一个弟弟。他爹在外面打工，娘走了。可能家里有事。"

"我去看看。"苏安晴说。

"那路不好走。"

"三里地。没事。"

她往包里装了两本书、一瓶水、一包饼干。走到校门口的时候，林玉起跟了上来。

"我跟你去。"

"不用。"

"我走前面。"

苏安晴看了他一眼，没再拒绝。

七月下旬的山里，太阳很毒。但山路上有树荫——核桃树、柿子树、枣树，叶子密密地叠在一起，把日光筛成了碎片。路确实是土路，被雨水冲出了深深浅浅的沟壑，最窄的地方只能放下一只脚。

苏安晴走在林玉起后面。她走得不快，但很稳。脚踩在碎石上的时候会先试探一下再落下去。

"铁蛋在课堂上特别爱举手。"苏安晴边走边说。"有时候明明不会也要举。我叫他回答，他站起来想了半天说——'老师，我忘了'。其他同学笑，他也不生气，坐下继续举手。"

"他喜欢你。"

"他喜欢被人叫起来回答问题的感觉。"

走了大概二十分钟，过一个坎的时候，苏安晴的右脚踩到了一块松动的石头。

石头翻了。她的脚腕往内扣了一下。

她闷哼了一声，一只手扶住旁边的树干。

"崴了？"

"等一下。"

她试着踩在地上。踩下去的时候眉头皱了一下，但很快就松开了。

"能走。"

"别逞强。"

她站了两秒，把重心换到左脚上，然后看了林玉起一眼。

"那你扶一下。"

林玉起把手伸过去。苏安晴抓住他的小臂——手指很凉，握力比他想象的大。

以这样的姿势走了三里路剩下的最后一里。

路上苏安晴说了很多关于铁蛋的事——铁蛋写了十篇关于他奶奶的作文，篇篇不一样；铁蛋把英语单词"apple"念成"阿婆"；铁蛋从家里带了核桃给每个支教老师，每人一颗，用报纸包着。

林玉起听着。他发现苏安晴说起学生的时候话比平时多得多，语速也快了，中间几乎没有停顿。不像平时那个每句话都要先想三秒的苏安晴。

到了核桃沟。铁蛋家是三间石头房子，院子里晒着核桃。铁蛋的奶奶在院子里择菜，看见两个年轻人走进来，愣了一下。

苏安晴松开林玉起的手臂，往前走了一步——右脚落地的时候嘴角动了一下，但声音一点没变：

"阿姨您好，我是铁蛋的老师。他两天没来上课了，我来看看他。"

她说这句话的时候站得很直，笑得很好。

林玉起站在她身后一步远的地方，看着她。

>
山路三里远，石滑脚一崴。
谁言少年弱，师心比路长。
>
