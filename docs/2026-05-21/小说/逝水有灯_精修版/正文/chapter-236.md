# 第236章 同学群的婚讯之前

所属部卷：第五部：逝水有灯
时间：某个普通午休
地点：公司茶水间

## 诗歌插页 P236：原创现代诗

我把咖啡粉放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

同学群的婚讯发生之前，生活先把场面摆好。

午休时间，陆允川在公司的茶水间等热水。咖啡机是新装的——不算新款，但比他出租屋那个烧水壶要高级两档。茶水间的窗台上放着几盆绿植——是谁从公司开业时摆到现在的不死鸟，叶片肥厚，边缘长了一排小芽，没人浇水也没人管，照样活着。他撕开速溶咖啡的包装，倒进纸杯，然后把纸杯放在出水口下面。咖啡机嗡了一声，开始加热。热水注入纸杯的瞬间，咖啡粉被冲得旋转起来，形成一个细小的褐色漩涡。

手机震了。群聊。那个他已经静音了大半年的大学班级群。

是大学班级群。这个群平时很安静，一个月大概冒泡一两次，发的内容不外乎是哪家公司招人、哪里有便宜的房子租。今天不同——消息连着跳了十几条。

他点进去。红底黄字的表情包铺满了屏幕，像过年。往上翻了好几屏才翻到第一条消息。是喻可发的："同学们！我要结婚了！下个月十五号！"

纸杯满了。他端起来回到工位，继续看群。

喻可发了一张电子请帖。配乐是钢琴曲，背景是海边的婚纱照。她旁边站的新郎戴着方框眼镜，笑得很开。请帖上写着两人的简单介绍——新郎是工程师，新娘在做市场，去年认识，今年求婚。

有人在群里说："喻可！我记得你大学的时候说三十岁之前不结婚！！"

喻可回："计划赶不上变化。"

然后她补了一条："遇到对的人就什么都变了。"

陆允川看着这一条消息。他没有跟队发祝福——不是因为不想，是因为他喝完咖啡再发。

祝福的队伍已经排了四十多楼。他往下翻，在中间看见一个熟悉的头像。是苏晚枝。她发的是："祝福可！永远幸福！"后面跟着三个红色的爱心。不多不少，标准的朋友圈语言。

又往下翻了几条，看见夏知遥发了一个大红包——群红包，总金额二十块，分成十个包，封面写着"喻可新婚快乐"。被秒抢光了。抢到的人纷纷在群里发谢谢。

陆允川抢了一个。三块二。不多。

他把咖啡喝完。纸杯底有没化开的咖啡粉——几小粒黑色的颗粒浮在最后一口水里，晃了几下也没沉下去。他把最后一口喝完，把纸杯扔进垃圾桶。

然后往回翻，重新看了苏晚枝那句"祝福可！永远幸福！"

他给她发了私聊："你什么时候结婚？"

发完就后悔了。但撤不回了。超过两分钟。

她很快回了："你问这个干嘛？"

"看到群里喻可的婚讯。"

"我的不急。你要随礼？"

"对。提前存钱。"

她发了个白眼的表情。"八字没一撇。先吃喻可的喜酒。"

"你去吗？"

"去。她请了全班同学。场地在城南。"

"那你坐哪桌？"

"新娘朋友桌。你呢？"

"新娘朋友桌坐不下我这种——边缘人物。我坐陌生人桌。最方便。"

"你又来了。"

"我认真的。我最怕在婚礼上被安排到熟人桌——问你现在在哪上班、结婚没、买房没、小孩多大。陌生人桌只需要点头和夹菜。"

苏晚枝发了一长串笑着的表情包。然后停了一下，发了一条文字："你怕的不是问题。你怕的是回答那些问题的人——是你自己不知道怎么定义自己的现状。"

陆允川看着这条消息。这判断太准。准到他不确定是自己教给她的，还是她自己想出来的。

"你自己总结的？"

"总结的。跟你在面馆说的话合在一起。"

"哪句？"

"'接不住。'你说我'接不住'。但后来我发现——你也在接。只是你接的方式是沉默和回避。"

他把纸杯从扔掉的垃圾桶里又捞了出来。不是要捡，是要看看干透以后杯底残留咖啡粉的纹路。一圈深褐色的粉痕，中心是白色的。

"你是对的。"他回。"所以喻可的婚礼我去。"

"好。到时见。"

"到时见。"

他把纸杯重新扔进垃圾桶。午休还有十五分钟。他把手机调成震动，打开电脑，继续早上没改完的那份标书。
