# 第157章 短暂和好

所属部卷：第三部：寒风里的花
时间：七月中旬
地点：诗社活动室

## 诗歌插页 P157：原创仿唐绝句

旧书翻处有微尘，
一笑轻轻不许深。
短日晴窗能共事，
莫将共事认归心。

整理到第三箱时，陆允川翻出一本没有封面的旧刊，扉页上写着一行很歪的字：南湖诗社内部资料，勿借不还。

苏晚枝看见，笑了一下，“这句像叶澄写的。”

“叶澄字没这么丑。”

“她听见会让你重排目录。”

两个人都笑了。笑声很短，却让活动室里的空气松了一点。陆允川低头贴标签，把那本旧刊放进“保留”一栏。

中午，叶澄还没回来。苏晚枝从包里拿出两块饼干，放了一块在桌边，“你要吗？”

陆允川看了看她，“谢谢。”

他接过饼干，没有说“你还记得我没吃早饭吗”，也没有把这件事看得很重。饼干就是饼干，一块有点碎的苏打饼干，边缘掉着屑。

下午封箱时，两个人一起按住纸箱。胶带从箱面拉过去，声音很响。苏晚枝说：“这样挺好。”

陆允川知道她说的是整理旧书。他也希望她说的是他们之间。可这次，他没有把希望说出来。

短暂的轻松留在活动室里，像窗边一小块阳光。它照了一会儿，就够了。
