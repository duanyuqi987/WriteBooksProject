# 第201章 最后一次争论之前

所属部卷：第四部：失去的美丽
时间：大四开学后
地点：校外面馆

## 诗歌插页 P201：原创现代诗

最后一次
并不会提前挂牌
它只是坐在热汤旁边
把筷子摆正
等人把旧话说完

大四开学后，学校周围的店铺换了一批。东门外那家面馆还在，招牌褪色，门口多了一个扫码点单的牌子。陆允川去打印毕业材料，出来时下雨，便进店避一避。

苏晚枝也在。她坐在靠墙的小桌前，面前是一碗番茄鸡蛋面，旁边放着一把湿伞。两个人对视时，都有一点措手不及。店里人不多，老板在后厨切葱，电视机声音开得很小。

陆允川本可以点完面坐到另一边。可苏晚枝先开口：“你也来吃饭？”语气平常，像把路让出一半。陆允川点头，问：“这里有人吗？”她摇头。

他坐下后，两个人先谈了很安全的事：实习、论文、答辩安排。汤气往上冒，把桌面上的塑料菜单熏出一层雾。陆允川拿纸巾擦了一下，又不知道该把纸巾放到哪里，只好攥在手里。

过去那些话像面馆墙上的旧油渍，不说也在。陆允川以为自己已经练习得足够平静，可苏晚枝说“毕业照下周拍，你们班也在图书馆前吧”时，他还是听见胸口轻轻一紧。

争论还没有开始。它先以一碗热汤、一把湿伞、一个没选好的座位，坐到了两个人中间。
