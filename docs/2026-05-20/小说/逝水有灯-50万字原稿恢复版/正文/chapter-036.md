# 第036章 成绩单下滑之前

所属部卷：第一部：初秋三楼
时间：高二期中后
地点：班级后墙成绩栏

## 诗歌插页 P036：原创现代诗

我把早晨放低，
让一场雨从课桌边慢慢经过。
你的名字没有被我喊出，
只在成绩栏上被红笔圈出的名次旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写成绩单下滑抵达之前的坡度。期中成绩贴出来那天，后墙前比公告栏还挤。红笔圈出的排名像一个个小伤口，大家嘴上说随便看看，脚却都不肯走。

陆允川从上往下找自己的名字，找到时喉咙发紧。他又往上看，夏知遥的名字稳稳停在那里，像一根没有被风吹动的线。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。

那天真正的事还没有发生，成绩栏上被红笔圈出的名次却先被他记住了。它太小，小到放在任何人的日子里都不显眼，偏偏在陆允川这里有了重量。

“掉了就补，别站这儿给成绩单守灵。”程砚说。“我知道。”陆允川说。“你不知道，你现在脸上写着天塌了。”程砚说。

程砚把他拉到座位上，翻出物理卷子，说从这道题开始，别把喜欢当借口，也别把借口写得像喜欢。

叶澄路过后墙，看到陆允川还站着，就把一张草稿纸递给他。纸上写着几道同类型题，她说，先会一道是一道。

课间十分钟被铃声切得很碎。有人挤去小卖部买面包，有人趴在桌上补觉，有人拿着水杯一路跑向水房。陆允川站在这些声音中间，心里那点紧张没有消失，却被生活稀释了一些。

夏知遥这时还只是人群里的一个名字。夏知遥只看了一眼成绩单，就回座位整理错题。她没有炫耀，也没有安慰谁，手里的红笔一下一下，把错误改得很清楚。陆允川看见了，却没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子，又一行行划掉。留下来的，是教室里真实的光、真实的灰、真实的不知道该把手放在哪里。

后墙下的垃圾桶里塞满废纸，几张揉皱的卷子露在外面。有人假装开玩笑说完了完了，声音却发虚。陆允川听着那些声音，忽然不想再把失控说得很浪漫。

如果说喜欢在这里有了开端，也不是因为命运突然发声，而是因为一个少年开始过分留意普通事物。他还不懂，留意可以是温柔，也可能慢慢变成对自己的消耗。

成绩单贴出来后，班主任没有立刻批评，只让大家把卷子拿出来订正。可越是这样，陆允川越坐不住，仿佛那张纸还贴在后墙上，隔着一整个教室看他。

夏知遥在前排问老师一道化学题，问完后把步骤重新写了一遍。她的稳定不是天生的，陆允川看见她草稿纸上密密麻麻的计算，才知道自己曾经把别人的努力看得太轻。

程砚把他的错题本翻到空白页，说先别谈心，谈心不能提分。陆允川想反驳，程砚已经把第一道题抄了上去，笔画很重，像把他从发呆里敲醒。

晚自习下课后，陆允川没有立刻走。他把后墙那张成绩单看了一遍，又把视线收回来。名字仍在那里，名次也仍在那里，可他终于愿意从第一道错题开始，而不是从一场情绪开始。

这一章要把脚步放慢，慢到能听见课桌被拖动时的短促摩擦声。事情还没到来，可人物已经在日常里摆好了位置：谁靠窗，谁靠门，谁总替别人递一张纸。

陆允川后来才知道，所谓开始并不是一个明确的按钮，而是很多个微小选择叠在一起。他没有说话、没有靠近、没有确认，可他已经在心里给那个人留出了一块地方。

事情还没有真正发生时，陆允川已经在心里排练了好几种反应。可排练没有用，现实总会把人推到更笨的位置，让他先拿错成绩单，再说错一句无关紧要的话。

那天放学，他没有立刻回宿舍，而是在班级后墙外多站了一会儿。风把物理卷吹得轻轻晃动，他忽然意识到，自己等待的并不是答案，只是一个能让明天继续期待的理由。

陆允川把错题本拿在手里，先把物理卷订正完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把卷子订正完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把一道化学题重新列了三遍步骤，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

班级后墙里最明显的不是心跳，而是红笔盖被按上的咔哒声。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，晚饭里凉掉的米饭把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。错题本、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情还没到来，成绩栏已经先进入他的视线。他当时不懂这种提前的留意有多危险，只觉得只要自己足够安静，就不会打扰任何人。

放学后他在班级后墙外多站了一会儿，听见楼下有人喊集合。那一声很普通，却把他从想象里拉出来，让他记起自己还要回宿舍、洗杯子、准备明天的早读。

这一段把喜欢从成绩栏前拉回错题本。名次不会因为难过自动升上去，陆允川必须把羞愧折成一道道能订正的题。

如果只写疼，这章会很薄；加上错题本、红笔盖被按上的咔哒声和晚饭里凉掉的米饭，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在班级后墙发呆。”陆允川把错题本收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

本章最后，事情仍悬在半空。陆允川收好书，跟着人群往外走。楼道里风从窗缝挤进来，把他的校服袖口吹起一点，他下意识按住，像按住一句还没成形的话。
