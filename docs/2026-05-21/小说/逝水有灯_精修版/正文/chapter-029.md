# 第029章 第一次说话旁观

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：教室前门和讲台之间

## 诗歌插页 P029：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
门口散落的作业本还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，物理课代表把作业本散了一地这件事对大多数人来说只是一次课间小事故。课代表赵一磊自己慌了三秒——"完了完了"念了两遍——蹲下去捡的时候不小心把手磕在了门框上，疼得吸了一口气，但还是继续捡。旁边的同学帮忙捡了几本就算了，有人一边捡一边说"赵一磊你这抱本子的技术得练练"，赵一磊回了一句"你去抱抱看，二十几本"。班主任陈老师恰好经过，看了一眼说了句"小心点"，就走了——每天这种小事在走廊上太多了，不值得停下来。前后门的过道每天产生上百次交互——传卷子、借尺子、归还捡到的校牌——百分之九十九的交互都在发生后的十分钟内被遗忘。当天晚上如果有人问你"下午课间发生了什么"，大概谁也说不出"物理课代表把作业本撒了"。但陆允川会记住那一次，因为那一次刚好踩在剩下百分之一里——而且还是那百分之一的最核心，像子弹正正打在靶心。

程砚从座位上远距离观察了整个过程。他后来跟叶澄复述时是这么说的："老陆蹲下去帮她捡本子的那个动作，比他在操场上做任何运动都要迅速。平时体育课让他冲刺他磨磨蹭蹭，捡本子他零点三秒就到了现场——比下课铃还快。"叶澄说："你别老拿他的紧张开玩笑。""我不是开玩笑，我是在记录他难得迅速的一次反射。这属于正常人类行为。"程砚把手一摊，"再说了，他最后不就说了六个字嘛。六个月了才攒够六个字，你觉得我嘲笑得动吗？我都替他省着了。"

叶澄的观察方式完全不同——她注意的不是陆允川的速度，而是他的嘴唇。她当时正从办公室回来，手里拿着班务日志，站在讲台边上看到了整个过程。她说："本子撒在地上的时候陆允川正从后门出来。他看见人蹲下去了，嘴巴张开了一下——上嘴唇和下嘴唇分开了大概一厘米——我以为他要说'小心'或者'要我帮忙吗'——但他把嘴闭上了。闭上了大概两秒钟，嘴唇抿成一条线，然后直接蹲下去捡了。他不是不想说话，他是觉得从嘴巴里说出来的字不如从手里递出去的本子安全。说话有语法错误的风险，但捡本子不会。递本子是一个纯物理动作——递到了就是递到了，不存在'我说错了什么'的可能性。"叶澄总结完以后又加了一句："他这种人值得等——因为他虽然慢，但他选字选得比别人认真。六字里的每一个字都经过了至少三个月的筛选。"

何敏不小心踢翻了脚边的水杯——水洒了一桌，她的笔记本湿了一片。全班的目光都集中到第三排的泼水现场，而前门的弯身捡物事件被自然地掩盖了。陆允川和夏知遥蹲在地上捡本子的那四秒钟，在教室里其他人的记忆里连一根线条都划不出。

只有蒋雯，因为坐在靠前的位置，看到了那个瞬间。她后来问何敏："你有没有觉得陆允川对夏知遥有点不一样？""怎么不一样？""就——他帮她捡本子的时候，他手碰到她的手就立刻缩回去了。平时男生碰到女生都不会反应这么大——你昨天撞到程砚的胳膊，他都懒得缩回去。""那是因为程砚反应慢，他缩什么缩？"何敏正在拧抹布，根本没注意这个话题的方向。蒋雯自己也没再往下想——因为在她十七岁的判断体系里，"多帮一次忙"完全可以是"这人比较热心"，远远够不着"这个人有心事"。

多年后程砚在聚会上讲起这件事时用了一句话概括："你知道为什么青春文学里有那么多捡本子的桥段吗？因为当你在那个年纪看到她的脸和你的脸在同一摞书上方对齐的时候，你就是控制不住地会把这两张脸想象成某种对齐——哪怕你们在数学坐标里压根不在同一个象限。"
