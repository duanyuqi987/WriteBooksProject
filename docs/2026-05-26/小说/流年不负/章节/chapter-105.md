# 第105章 朋友圈

时间：2015年12月

地点：楚州师院宿舍

卷次：卷六：平行轨道

十二月了。楚州入冬。

宿舍没有暖气。晚上冷得手指尖发僵。李博买了个暖水袋。灌满热水，抱着睡觉。

顾临风没有。他裹着被子看书。

手机放在枕头旁边。屏幕黑着。

考研政治背到第三章——马原。生产力决定生产关系。他背了三遍。第一遍划出来。第二遍默读。第三遍闭眼回忆一遍。然后翻到下一页。

窗外的风很大。呼呼地灌进玻璃缝里。窗帘在动。他把被角又掖了掖。

十点十五。李博已经睡了。暖水袋从被窝里滑出来掉在地上——他没醒。

顾临风背完五页。合上书。

拿起手机。

他本来只是看时间。十点十六。

然后他点开了微信。

朋友圈。许若琳。

九张照片。

他一张一张往下翻。

第一张：一棵圣诞树。大概是商场里的。装满了彩灯。

第二张：一盆火锅。红汤的。筷子在里面捞东西。

第三张：三个人在火锅店端着饮料碰杯。许若琳在最左边。中间是另一个不认识的女生。

第四张——

他停了下来。

第一张合唱照片。苏晚晴和许若琳。苏晚晴穿着白衬衫，最上面两颗扣子没扣。马尾。站在操场上。阳光很亮。她在笑。不是高三毕业照上那种收着的笑。是松开的——嘴巴张着，眼睛弯弯的。

她旁边是一群孩子。穿校服。在做操。

楚州一小操场。

第五张。苏晚晴站在大课间的队伍前面。领操。双臂平伸。一个标准的侧平举。后面的孩子们跟着做——有的手举高了，有的慢了半拍。

第六张。教室里。苏晚晴蹲在一个小女孩旁边，指着她桌上的作业本。嘴动着——大概在讲题。小女孩咬着笔头。

第七张。黑板。黑板上写着"祝苏老师生日快乐"——粉笔字，歪歪扭扭的，一看就是孩子们写的。

第八张。苏晚晴捧着一束纸折的花站在黑板前面。眼泪在眼眶里——没掉下来。她在笑。又哭又笑的那种。

第九张。所有人合影。她站在最中间，被孩子们围着。一个女生在背后给她比了个兔耳朵。

顾临风把第四张照片放大。

她的笑容比以前松弛了。

高三那年在考场外面她笑过一次——收着、浅的、嘴弯了一点点。她回头看操场的时候他站在花坛后面看到过。

现在不是。现在她站在那里，阳光铺在脸上，笑得像一个没有心事的人。

他的拇指在屏幕上来回滑动。放大、缩小、再放大。

许若琳的配文：

"去楚州一小找晚晴蹭教师食堂。伙食不错！看到她被一群小萝卜头围着叫苏老师，老母亲般的欣慰。晚晴真的适合当老师——温柔又有耐心。生日快乐呀我的苏老师。"

他在底下看到了几条点赞。赵一鸣、几个高中同学。苏晚晴本人没有评论。

他看了很久。

久到手机屏幕暗下去。他又点亮。又看了一会儿。

没有点赞。

他把手机关掉。翻过去。放在枕头旁边。

窗外风还在刮。窗帘一下一下地鼓起来。

他躺在黑暗里。天花板上那条裂缝应该还在——太黑了看不见。

十二月。楚州的冬天很冷。

她的手大概也冷。站在操场上带操。初冬的风吹过来。白衬衫下面应该加了保暖衣。她领操的时候手伸得很直——照片上看得出来。

那些孩子叫她苏老师。声音大概很响——一大群小学生喊起来，操场上能听到回音。

她收到了纸折的花。收到了写歪了的"生日快乐"。收到了一个五年级男生偷偷塞给她的糖。

她大概很开心。

他的嘴角动了一下。在黑暗里。

然后翻了个身。

手机里那些照片还在。那个放大过的笑容还在。

他没有点赞。

那天晚上他做了个梦。梦见了楚州一小的操场。阳光很亮。她站在队伍前面带操。他在校门口站着。隔着铁栅栏。她没有看到他。然后下课铃响了。孩子们涌过来。把栅栏挡住了。她不见了。

早上醒来他洗了把脸。镜子里的脸有点白。

他擦了擦脸上的水。

背上书包去了图书馆。
