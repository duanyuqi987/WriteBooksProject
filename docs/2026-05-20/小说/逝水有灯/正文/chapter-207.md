# 第207章 毕业照里的空位

所属部卷：第四部：失去的美丽
时间：大四毕业季
地点：大学图书馆前台阶

## 诗歌插页 P207：原创仿唐绝句

台前衣袂映晴晖，
一格空阶不必追。
快门声中人各立，
风将帽穗送斜飞。

苏晚枝她们班先拍。陆允川站在台阶下等，手里拿着程砚塞给他的矿泉水。摄影老师让第一排蹲下，第二排站直，后排往中间靠。有人迟到，队伍里空出一个位置，大家喊名字，笑声一层一层传开。

那个空位刚好在苏晚枝斜后方。陆允川看见她回头望了一眼，又把身体让开一点，给迟到的女生留出路。女生提着学士服跑来，气喘吁吁地站进去，苏晚枝伸手替她把帽子扶正。

很普通的一幕。没有暗号，没有多余的意味。可陆允川看着，心里忽然平静下来。她在她的队伍里，照顾她身边的人，完成她自己的毕业照。这件事与他有关，又不只与他有关。

轮到他们班时，祁野把他推到第三排边上。“站这儿，脸小。”陆允川笑骂了一句，还是站过去。隔壁班正在散队，苏晚枝和叶澄从台阶另一侧下来，经过他们班队伍前。苏晚枝没有停，只轻声说：“毕业快乐。”

陆允川回：“毕业快乐。”

摄影老师开始倒数：“三、二、一。”快门声响起时，风把他的帽穗吹到眼前。他没有去看苏晚枝走到哪里，也没有在人群中找她的背影。照片里，他站在第三排靠边的位置，旁边恰好空着半级台阶，是刚才有人临时去接电话留下的。

那半级空位留在照片里，没有被修掉。
