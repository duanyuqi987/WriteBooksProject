# 第142章 走廊

时间：2012年10月中旬

地点：江城大学综合教学楼三楼走廊

卷次：第六卷：雪落无声

周三下午。教育心理学。

林玉起上完课从教室出来，在走廊里碰见了苏安晴。

这已经是第五次了。开学到现在，第五次在走廊里碰见。

前四次都只是停下来打个招呼——"课多吗""嗯""走了"——像一个被缩减到极限的仪式。但这次不一样。

苏安晴先看见了他。她站在楼道拐角的地方，手里拿着一个牛皮纸信封，好像在等人。

"林玉起。"

他走过去。

"正好碰见你。"她把手里的信封递过来。"陈老师让我转交你的——支教实践证明。盖了章的那个。我正好去拿我的，顺便把你的也拿了。"

"谢谢。"

林玉起接过信封，捏在手里，没急着拆。

楼道里人来人往。下课铃刚打过，学生从各个教室里涌出来，走廊里站满了。有人抱着书站着聊天，有人靠着墙看手机，有人在公告栏前面看新贴出来的讲座海报。

苏安晴没走。

以前的她会说"我还有个课"或者"走了"，然后挥挥手离开。但这次她靠在走廊的墙上，把手里的书抱在胸前，看着林玉起。

"你的诗呢？"

"什么诗？"

"上次你说改好了给我看。快一个月了。"

林玉起没说话。他确实改好了。一本小本子都写满了。但每次想发给她，都能找到一个新的理由——这首还不行，那首需要再改，这首太直白了，那首太隐晦了。

苏安晴看着他。她没有催，但也没有把目光移开。

"你怕给我看。"她说。

这句话不是问句。

"我——"

"你觉得我会评价吗？"

林玉起没有否认。

苏安晴把书从胸前放下来，换到另一只手上。"我不会评价的。不是因为我客气。是因为我评不出来。你写的诗比我能看懂的好。"

走廊里的人渐渐少了。上课铃还没打，但大部分人都进了教室。

"那这样吧。"苏安晴说。"你不要把你的诗当成诗给我看。"

"当成什么？"

"当成——"她想了两秒，"——当成你写的东西。"

"那不是一样吗。"

"不一样。'诗'是要被评价的。'你写的东西'只是你写的。"

林玉起看着她。他忽然想起来，这些话和李家沟那个晚上——她说"可以写得更直接一点"——是连贯的。她在用不同的方式说同一件事。

"好。"他说。

"好是什么意思？"

"明天我发给你。"

苏安晴点了一下头，推开门进了隔壁教室。门关上之前她回头看了一眼——不是微笑，但嘴角有一点点弧度。

那是十月十六号，周三。

走廊里很亮。秋天的阳光从走廊尽头的窗户里斜射进来，把地板砖照得反光。林玉起站在那道光里，手里捏着那个牛皮纸信封，站了一会儿。

然后他打开手机QQ，翻出这一个月写的那几首诗。从头到尾看了一遍。删了四首。留了三首。

他没等到明天。当天晚上就发过去了。

>
纸书不可藏，诗成何须饰。
谁言独行者，不敢示真心。
>
