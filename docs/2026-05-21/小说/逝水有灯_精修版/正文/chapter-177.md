# 第177章 朋友的烟

所属部卷：第四部：失去的美丽
时间：凌晨
地点：宿舍楼后门

## 诗歌插页 P177：原创仿唐绝句

残烟微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

祁野没给陆允川抽烟。他没递，陆允川也没要。只有他自己夹着那支烟——烟头的红光在冷空气里缓缓后移，照亮了他放在砖缝上的拇指。

"你以前抽过几次？"陆允川问。

"很少。一包烟能放三个月。"祁野把烟按在砖缝里。

"那今天为什么点？"

"因为你需要的不是我说'别想了'。你需要的是有人陪你站在一个没什么用的地方，什么也不说，待一会儿。抽烟可以让人什么都不说。"

他碾了碾墙上的烟痕，然后用鞋底轻轻碾灭了烟头。烟头落在地上时只剩一点红，亮了两秒，灭了。

天开始露出灰蓝色的边。食堂的灯已经亮了，能隐约看见后厨的师傅在搬菜。

"陆允川。"祁野忽然开口，声音不重，但很稳。

"嗯。"

"你可以丢脸——丢脸没关系，我陪你一起丢。但你别把人家也拖进来陪你丢脸。"

陆允川转过头看祁野。祁野没有看他——他望着远处食堂的灯光。

"我什么意思你懂吗？"祁野继续说，"你写的那些删除线底下的话——是在生她的气。她做的事是没答应当你的女朋友。这不是她的错。如果你因为被她拒绝而说出更难听的话，那些话会反过来让你丢脸——不是在她面前，是在你自己面前。有些话说出来以后，你自己都会看不起自己。我只是不想让你最后变成那样。"

陆允川没有说话。风从洗衣房那边吹过来，洗衣粉的味道浓了一股。

"你觉得我已经丢脸了吗？"他过了很久才问。

"丢了很多次。"祁野说，然后顿了顿，"但丢了还能捡回来。只要你不再追着人家道歉。"

"这两天我一直在想——我对苏晚枝做的那些事。送书、等楼下、质问、写那些话——如果我站在她的角度，我会不会害怕？"

"会。"祁野回答得很快，"你不要假装不知道。你知道。所以你后来才会划掉——才会写那些自省的话——才会想道歉。但她需要的不是你反复道歉。她需要的是你不做了。道歉是开口，不做是闭嘴。你要学会闭嘴。"

天更亮了一点。食堂那边传来的声音多了一个层次——除了倒车的提示音，还有锅铲碰锅沿的声音。清晨的味道慢慢取代了洗衣粉味。

手机在陆允川口袋里震了一下。是程砚发的消息："你们在哪？"

陆允川回："宿舍楼后门。"

过了一会儿程砚出现了。他套着一件棉睡衣，头发翘着一撮。看到祁野手里的烟头痕迹，他皱了皱眉。

"你又抽了。"

"嗯。"

"你不是说你戒烟了吗。"

"没戒。就是很少抽。"祁野说。

程砚走到他们旁边，也靠在墙上。他看了一眼陆允川——陆允川没有哭，但眼睛有点红。不是哭——是失眠加冷风吹的。

"昨晚你们都写了些重话——日记那边的事我知道。"程砚说。

"祁野跟你说的？"

"不是。你昨晚说梦话。大概凌晨两点你说了两遍'划掉'。我在实验室待晚了没睡着。"

陆允川低下头。他不知道自己在梦话里还说了什么。但程砚没有问细节。

"我发那条消息是怕你半夜冲动发出去。"程砚说，手指了指屏幕，"不是说你没资格难过——是你难过的方式容易伤到别人。发出去以后你肯定会后悔。与其后悔发射了，不如先别发射。"

陆允川点了点头。

又有三轮车在食堂那边倒车——声音在清晨的安静里显得格外清晰。卸菜的人是食堂大师傅——陆允川认识他，他的标志是不管多冷总戴副橡胶手套。

"走吧。"祁野从墙上直起身，"天快亮了。食堂马上开门——今天有鸡蛋灌饼。"

"你怎么知道？"

"周三。每周三食堂都做鸡蛋灌饼。"

"你对食堂菜单记得比专业课还清楚。"

"当然。我的记忆力只能用在正事上。"

程砚走了一步又停下。"陆允川。"

"嗯。"

"后面这些天——如果还想写那些重话，先给我或祁野看一眼。不是审查——是帮你看看哪些话是你真想说的，哪些只是不甘心。不甘心的可以删——不急着删也没关系，划掉就好，像上次那样。"

"好。"

三个人从后门拐进楼里。走廊里已经有了早起的同学——有人拎着水壶去打水，有人在楼道口穿鞋，有人正站在走廊尽头低头发消息。生活正在恢复它平时的节奏。

上楼梯的时候，陆允川忽然对祁野说："谢谢。为这个凌晨的烟。"祁野在台阶上停了一下，没有回头。

"不用谢。不过你刚才说今天会有鸡蛋灌饼——如果真的有，加一个鸡蛋给你。算是庆祝。庆祝什么不知道——可能是庆祝你没把那些话说出来。"

"如果没鸡蛋灌饼呢？"

"那就庆祝我戒烟。两种都有积极意义。"

程砚在背后跟上来，一边扯了扯他那件穿反的棉睡衣，一边说："你们确实可以成立一个'如何用不靠谱方式应对陆允川的情感危机'互助小组。口号就是——'鸡蛋灌饼永远有。烟不要抽。'"

祁野笑了。陆允川也笑了。天亮了——已经是完全的清晨。早上第一节课还有两个小时。
