# 第107章 信

时间：2016年1月

地点：楚州师院宿舍512

卷次：卷六：平行轨道

一月中旬。期末考之前。宿舍空了三个——阿坤搬出去了，小伍去图书馆通宵，李博回家了一趟。李博走的时候说"后天回来——帮我把床单收一下"。他床上摊着一本翻开的考研英语真题集。

512只剩顾临风一个人。

下午四点。阳光从窗户斜进来，在桌子上摊开一片橘黄色的光。光里能看到细小的灰尘在飘——被窗户缝里透进来的气流推着，浮浮沉沉的。暖气不够足，屋里还是冷。暖气片是铸铁的老式款——刷了银粉，表面有一层凹凸不平的漆纹。摸着是温的——不烫手。他穿着棉拖鞋。脚趾在鞋里缩着。鞋底磨薄了，脚后跟能感觉到水泥地面的凉意。

桌上有考研政治、一本古代文学史——封面是仿古的线装花纹，但其实是普通平装本，书脊已经快翻断了——半瓶墨水。墨水是英雄牌的。蓝黑色。瓶盖上结了一层干涸的墨渍。玻璃瓶的底部有一层沉淀——深蓝到近乎黑色。

他拉开抽屉。

抽屉上了锁。一把小挂锁——铜的，锁面磨得发亮。钥匙挂在脖子里——挂在一条深蓝色尼龙绳上。从高中养成的习惯。记笔记的本子、写的诗、还有那张高考准考证——上面他的照片已经褪色了，脸变成了模糊的浅绿色。

锁弹开。发出清脆的一声"咔"。他翻了翻——笔记本。几支铅笔——短得只剩半截的。一枚生锈的校徽——青阳一中的校徽，背面有别针的地方长了一层白色氧化层。

然后从最底下抽出来一沓稿纸。横条格，20×20，师范学院的专用稿纸。左上角印着楚州师院的校徽。稿纸放了很久——边角有点发黄，上面压出来的那道折痕已经很深了。纸面上有淡淡的墨水味和纸张老化的酸味。

他铺开稿纸。用手掌把纸面上的褶皱压平——从中间往两边，一遍一遍地。直到纸比较服帖地铺在桌面上。第一页。拿起那支英雄钢笔。笔身是黑色的塑料，笔帽是不锈钢的——有点松，拧开的时候需要用点力。拧开笔帽。笔尖在纸的左上角点了点。墨水洇开一个细小的蓝点。墨水从笔尖往外扩散的速度不快——稿纸的吸水性中等。圆点边缘有细微的毛刺——墨水沿着纸纤维往外爬的样子。

然后他写了第一行。手腕贴着纸面。笔尖在纸上游动。

"苏晚晴："

写完之后停了一下。看着这几个字。自己的字迹。自己写的她的名字。"苏"字草字头有点歪——太多年没在信纸上写这个字了。"晚"字的"日"旁偏窄。"晴"字的"青"那一笔横画太长——长到了他都不想承认是偶然。

"你好"太客套——像写给陌生人的开头。"好久不见"太像要抱怨——好像在说"你怎么不联系我"。"见字如面"太文绉绉——他不是那种会在信里写成语的人。

他划掉了这几个字。用横线划的——从左往右，压在这三个字上面。重新写。

"很久没跟你说话了。其实高考后就没说过。"

第一段。写考研。

"我大三了。在准备考研。报的是古代文学方向。每天在图书馆坐六个小时。政治背了三遍。英语真题做了十套。专业课看了一大半。不知道能不能考上。"

写到这里他的笔停了。这一段太干了。像是在填一张表格——"姓名：顾临风。职业：考研考生。每天学习时长：6小时"。他没划掉，但把"不知道能不能考上"涂了一下——改成"希望吧"。"希望吧"写完之后又觉得太软。又在旁边补了一句——"尽力了。"三个字写完。然后把"希望吧"划掉了。现在那段结尾变成了——"专业课看了一大半。尽力了。"

第二段。写实习。

"去年在楚州二中实习。教初一。站在讲台上的感觉——很奇怪。像小时候玩游戏，忽然自己变成了老师。"

他把"像小时候玩游戏"划掉了。他觉得信不应该用比喻句。然后继续写。

"我对他们说的话，跟我当年的老师对我说的话——差不多。'把手机收起来。'说完之后觉得自己老了。"

他在"老了"旁边画了一个小圈。然后继续往下加——"但站在讲台上朗读《春》的时候，底下很静。连后排的男生都抬头了。那一瞬间觉得——这份工作不坏。"

第三段。写青阳。

"过年的时候回了趟青阳。经过那条有老槐树的巷子。树还在。比记忆中矮了。大概是树老了，或者是我长高了。巷子口新开了一家奶茶店。卖的东西叫'初恋的味道'。我买了杯柠檬水。"

他在"初恋的味道"和"我买了杯柠檬水"之间留了一个空行。这个空行不是忘了写。是故意的。他不想让这两个句子挨得太近。挨近了会暴露他想说什么。

第四段。

"在书店门口站了一会儿。新华书店——不知道还叫不叫这个名字。以前周六去县城买参考书，每次经过那条巷子都会往里看一眼。"

写到这里，笔停了。

手指悬在纸上。笔尖离纸面还有大概半毫米。墨水在笔尖中缝里凝着。他知道自己在写什么。写的不是巷子，不是奶茶店，不是考研。他写的是——我还记得你。我记得你家在哪条巷子。我记得你从书店出来跟你妈提东西走进去的样子——塑料袋子是白色半透明的，里面装了几本书，袋子勒在手指上勒出红印。我记得你书包上晃着的小鱼——那条鱼的尾巴会左右摆。我记得你以为没人看到的时候会自己哼歌——声音很小，只有嘴唇在动。

但他不能写"我还记得你"。

他在纸上写了"我记得"。笔触很轻——不像在写，像在试探纸面。然后划掉了。在旁边补了一句"青阳县的冬天还是那么冷"。

窗外有鸟叫。可能是麻雀。楚州冬天的麻雀不多。偶尔有一只落在窗台上——爪子踩在窗台的金属边上发出很轻的嗒嗒声。鸟把头偏了一下，往窗户里看了一眼，然后飞走了。

他继续写。

第五段。

"不知道你现在怎么样。听许若琳说你在楚州一小实习，教三年级语文。那些孩子应该很喜欢你。你以前就不太会拒绝人——小孩大概看准了这一点。他们让你写板书你写板书，让你画小花你画小花，让你学猫叫你大概也会学。然后全班笑得前仰后合。然后你也笑——笑得比他们还大声。"
"希望是这样。"

写到这里他的手停了一下。笔尖在"希望是这样"下面画了一条短横——收笔时往上一挑。

赵一鸣前几天发微信说了一句话。大意是晚晴好像谈过恋爱，分开了。

那行字在他的脑子里闪了一下——然后就没了。他没法把它接在"希望是这样"后面。"听说你分开了"不是不能写——而是写了就变成另外一种含义。一种他不想让这封信带上的含义。这封信不是要趁虚而入。不是要在她脆弱的时候敲窗。这封信只是想告诉她——我还在这里。我一直在这里。无论你旁边有没有人。

他不能写这个。

他把笔停在半空。一滴墨水从笔尖渗出来——重力把它拉成一个小小的半球形。然后突然——表面张力被打破——滴落下去。落在稿纸的那个位置——第五段的末尾。洇成一个小点。圆圆的，边缘有细微的毛刺。墨水沿着纸纤维往外扩散。蓝黑色的小圆点周围是一圈淡蓝色的光晕。

他下面写了三个字——划掉了。又写了几个字——又划掉。笔尖在纸上走了一下，然后后退。像一个人在雪地里留下了几个脚印然后又走回到起点。

稿纸上有七八处涂改。有的是划掉的——一条线从左往右，把整个字压着。有的是涂黑的——笔尖在字上面反复地画圈，直到原字完全看不见。有一个地方他用指甲把纸抠出了一个洞——大概是他想写下那个字，但手指在拒绝。指甲划过去的时候纸纤维被推开——留下了一个豌豆大小的洞。透过那个洞能看到桌面——木纹密密的。

第六段。很短。只写了一句。

"其实我一直想给你发个消息。不知道说什么。就算了。"

然后他在"就算了"后面加了一个逗号。又把逗号划掉了——改成句号。写成句号是对的。"就算了"后面不该有任何接续——包括逗号。

第七段。更短。

"你还记得我吗。大概记得吧。毕竟也才过了不到三年。三年不长。但也够忘掉一个人了。"

他写完眼睛挪开了。看着窗户。窗户外面的天空从橘黄色变成了灰蓝色。冬天天黑得快。四点半到五点——天就暗了。

下面他没有继续写。

他没有写"我还喜欢你"。没有写"我经常翻你的朋友圈但从来不点赞"。没有写"你穿白衬衫很好看——你大一在操场上那件旧的白的不知道还在不在。高三毕业典礼那天你穿的那件新的——领口有淡蓝色小花。你家人给你拍的照片里你站在校门口的台阶上。那时候你笑得不太自然。但现在在楚州一小你的笑容松弛了。这是我在许若琳朋友圈里看到的——每张都看了很多遍。每次看完都会把手机屏幕往下压。"

他没有写"你扎马尾的背影是我的青春"。

他没有写——"我在图书馆翻古代文学史的时候想到的不是《诗经》《楚辞》。想到的是你看书时右眉毛在光里变淡的样子。"

他什么都没有写。

他把稿纸翻到第三页。第三页只有四行字。四行字的下面是整页空白。他把笔放下了。

看了一遍。从头到尾。一字不漏。嘴唇跟着无声地动——不是在发声，是字的形状在口型上过了一遍。

然后折好。沿着折痕——先对折，把第一页叠到中间。再对折——从长边折下去。然后从中间再折一次。变成手掌大小的一块。纸边对齐——用手掌压了压。折痕很深——指甲在折痕上划过去的时候能听到纸纤维被压实的细微声响。

夹进那本黑皮笔记本。笔记本的封皮上有一道凹痕——是长期被夹在抽屉里的东西压出来的。

笔记本放回抽屉。抽屉推上。推的时候卡了一下——木头在冬天膨胀了一点。锁扣摁下去——咔嚓一声。那声"咔"在空荡荡的宿舍里响了一下。

钥匙放回衣领里面。贴着胸口。凉的。金属的凉从锁骨那个位置慢慢蔓延到皮肤周围。他用手指把钥匙推到衣服更深的位置。

窗外天已经黑了。路灯亮起来。黄澄澄的一团。灯光照着楼下的法桐——光秃秃的枝干在光里变成了黑色。远处有汽车经过——灯光在墙上扫过去，亮了一秒又灭了一秒。

他坐了很久。盯着抽屉。抽屉关着。锁锁着。信在黑暗里。那个蓝黑色的圆点——墨水洇渍。大概还没干。一月份的温度低，墨水干得慢。要过好几个小时——或者一整夜——那个圆点才会从潮湿的亮色彻底变成干燥的哑色。

外面有脚步声。钥匙在锁孔里响。李博推门进来——带进来一股冷空气和食堂的油烟味。"老顾——你还没吃饭——"

"来了。"

他站起来。膝盖碰到桌腿——桌上的墨水瓶子晃了一下。他伸手扶住了。背上书包。

抽屉里的信在黑暗里。那个蓝黑色的圆点——墨水洇渍。大概还没干。
