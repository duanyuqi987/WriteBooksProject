# 第023章 最后一门

时间：2011年1月14日，期末考试最后一天

地点：江城大学教学楼、宿舍区

卷次：第一卷：九月的新校门

最后一门是《中国古代文学史》。

上午九点开考，十一点结束。林玉起八点半就到了考场——教学楼三楼的阶梯教室，暖气烧得太猛，窗户上全是水雾。他把围巾解下来搭在椅背上，把准考证、学生证、两支黑色中性笔在桌面上摆整齐。

等发卷的时候，考场里有人趴着补觉，有人嘴里念念有词地在背最后几个名词解释，有人往嘴里塞巧克力。监考老师背着手在讲台上踱来踱去，皮鞋底敲在木地板上，节奏很慢。

卷子发下来的那一刻，整个教室忽然安静了。

林玉起翻了一遍试卷。论述题第三题果然考了第三章第二节的内容——苏安晴的便签上记的就是这个。他愣了一下，然后开始答题。

两个小时后，他把卷子从头到尾检查了一遍，把笔帽盖上，第一个交卷。

走出教学楼。天空是灰白色的，像一张用得发旧的棉被。积了一周的雪化了大半，路面湿漉漉的，踩上去会溅起很细的水珠。

他站在教学楼门口，不知道该往哪走。

考完了。大一的第一学期结束了。

他掏出手机。郭向阳还有两门没考——今天下午的数据结构和明天早上的离散数学。付小川剩一门电磁学，正在宿舍里进行最后的挣扎。

QQ上有一个好友动态提醒。他点开看，是苏安晴两小时前发的：

"最后一门。加油。回家倒计时。"

动态下面已经有几条回复。宋乔评论了一个飞机图标。陈嘉敏评论说"一起走一起走！"

林玉起没有回复。他把手机放回口袋。

往宿舍走的路上，他经过了教育学院的宿舍楼。

五号楼的晾衣绳上挂满了被子和床单，趁着放假前的最后一天出太阳，能晒的都拿出来晒了。一件藏青色的棉服挂在二楼阳台的晾衣架上，袖子被风吹得一晃一晃。

他认出了那件棉服。

他没有停下来。继续走。经过五号楼门口的时候，本能地偏了一下头。

门厅里有人在拖行李箱。粉色的、银色的、黑色的大箱子从楼道里滚出来，轮子在瓷砖上发出嘎啦嘎啦的声响。

苏安晴不在门厅里。

林玉起走过了五号楼，走过食堂，走过操场，回到429。

他打开衣柜，把要带回家的衣服叠好——两件毛衣、两条秋裤、几双袜子，还有他妈让他带回去洗的被套。东西不多，塞进书包里刚好。

收拾到一半的时候，他在书包侧兜里摸到一张票。

火车票。

2011年1月15日。K1238次。江城——封丘。开车时间：07:52。硬座。13车厢。

他把票摊在桌上，看了一会儿。

然后他打开电脑，登录了12306网站。

键盘声很轻，像是怕吵醒什么。

页面加载的进度条一格一格地往前走。他输入"封丘"，点了"查询"。2011年1月15日从江城出发的所有车次跳出来——一共四趟，上午一趟，下午一趟，晚上两趟。

他在"余票查询"里输入K1238。系统转了几秒钟，跳出了各车厢的余票信息。

他把每一节车厢的余票都查了一遍。硬座、硬卧、软卧。13车厢还剩3张，14车厢还剩5张，15车厢还剩1张。

他没有买票。

他关掉浏览器，把电脑盖子合上，在椅子上坐了很久。

下午四点，郭向阳回来了，把鞋踢掉，直接倒在床上。

"数据结构挂了，真的挂了。"

"成绩没出来你怎么知道挂了。"

"最后一题我连题目都没看懂。"

林玉起想笑，但脸上的肌肉不太配合。

他打开笔记本，写了一行诗，划掉了，又写了新的一行，又划掉了，最后什么都没写成。

窗外有人在打电话，笑声一波一波地传上来。

他站起来，走到窗边。楼下是通往校门口的主干道，已经不断有人拖着行李箱往外走了。轮子滚过路面的声音绵延不断，像一种有具体节奏的时间计量单位——响过一声，就少一个人。

> 行李箱的轮子碾过寒假前最后一段路面
> 我在四楼的窗口听着
> 每一次滚过都是一个人离开
> 我还在原地数着轮声
> 不知道该不该下楼

他把这首短诗写在手机备忘录里，存进了草稿箱。

然后他拿起火车票，放进棉袄内侧的口袋里——和那张折了四折的纸放在一起。

明天，1月15日。

K1238次。7点52分。13车厢。

他对着电脑黑掉的屏幕，看见了自己的脸。

脸上的表情，他自己也说不清楚。
