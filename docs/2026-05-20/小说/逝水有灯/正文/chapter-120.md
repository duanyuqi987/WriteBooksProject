# 第120章 楼下等她多年后

所属部卷：第三部：寒风里的花
时间：多年后
地点：女生公寓楼下圆石凳

## 诗歌插页 P120：原创仿唐绝句

石凳多年受雨风，
旧时等意已成空。
若非人许休言候，
留得清光照路中。

多年后陆允川回到女生公寓楼下，圆石凳还在，只是花坛换了新砖。楼门装了门禁，外来人员不能靠近，他站在路边，看见学生刷卡进出，门很快合上。

他没有坐到石凳上。那不是他的地方，以前也不是。这个认识来得太晚，却终于清楚。

手机相册里还保存着当年校刊终审的照片，文件夹名称很普通：大二五月。照片里没有苏晚枝，只有活动室桌上的红笔、样张和一张盖章回执。陆允川看了几秒，把其中一张重复照片删掉。

他想起自己当年的委屈：等了那么久，她为什么不温柔一点。现在再回头，他更愿意承认另一个事实：她已经说过不用等。边界不是要等到别人发火才算成立，轻声说出来时也成立。

公寓楼下有人取外卖，塑料袋被风吹得响。陆允川站在树影里，把旧日记里那句“我等了她很久”改成“我没有听见她说不用等”。

改完以后，句子短了，责任却更清楚。不是为了苛责自己，而是为了让那一小时不再继续压着别人。

离开时，他没有回头。五月的风从花坛边吹过，石凳仍在原处，安静地给真正需要坐下的人留着位置。
