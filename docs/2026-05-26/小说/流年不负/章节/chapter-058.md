# 第058章 春天来了

时间：2013年3月

地点：教室

卷次：卷三：此心藏久

三月开学。最后一个学期。

教室还是那间——五楼。座位没有重新排——老刘说最后几个月了，大家自己坐习惯的位置就好。顾临风坐第四排靠过道。苏晚晴在第三排靠走廊。

黑板右上角的数字变成了两位数。87。

距高考还有87天。

课表变了。全科复习——每周语文六节、数学八节、英语七节、物理五节、化学四节、生物四节。每周一次模拟考。周六不休息。周日只休息半天。

教室后面的墙上贴了一张红色横幅。白字印刷体——"拼搏改变命运"。横幅挂上去才两天，右下角的图钉就掉了一个。布歪了。没人去扶正。后来歪着歪着就习惯了。

窗外的银杏发芽了。三月的银杏，先是枝条上冒出了一些绿色的小点。比米粒还小。然后小点慢慢变大。展开。变成一小片一小片嫩绿的叶子。银杏发芽的第三天。顾临风注意到了。他在草稿纸上记了一笔——3月12日，银杏发芽。不知道为什么记。

三月中旬的一个周一。早上六点五十。

苏晚晴走进教室的时候穿了一件浅绿色的外套。

不是校服。是便装。高三年级可以穿便装——老刘去年九月说的。但大部分人都还穿着校服——方便。省事。不想在早上多花一分钟挑选要穿什么。她的座位附近——许若琳今天也穿的便装。一件粉色的卫衣。但只有苏晚晴的浅绿色外套让顾临风停住了笔。

她这件浅绿色外套他以前没见过。新的。布料看起来是棉的——有点厚度。袖口微微卷了一小截。露出一圈浅灰色的里衬。拉链是金属的——银色。领口竖起来了一点点。遮住了半个后颈。

不是那种亮的绿。是浅的。带一点点灰调。像是春天快来的那种颜色。不是夏天浓烈的绿。是早春刚开始的那种——犹豫的、试探的绿。领子是圆的。袖口有一圈窄窄的白色滚边。

她穿着这件外套走到座位上。马尾搭在浅绿色的肩膀上。和她书包上的小鱼挂件同色系——小鱼是淡青色的陶釉。

顾临风在第四排。他看到了。

他的笔在草稿纸上停住了。正在写的是一道物理公式。写到一半。

他看着她的背影——浅绿色外套的背影。不是校服的蓝色了。是另一种颜色。三年了，他看她穿校服看了三年。蓝色春秋季校服。白色夏季校服。深灰色冬季校服——她有时候会在里面套一件高领毛衣。他习惯了她的校服。就像习惯了黑板上的倒计时。习惯了食堂的炸酱面。习惯了手电筒的光。

今天她穿了一件自己的衣服。一件她没有穿过的衣服。一件他不知道的衣服。

这个发现让他感到了一种奇怪的不安。

不是因为衣服不好看。不是因为颜色不适合她。是因为——他从来没有想过，她穿便装的时候是什么样子。因为这意味着——她是一个有自己的衣柜、自己的喜好、自己的选择的女生。不是只活在教室里的"坐在窗边的女生"。她是一个有课外生活的、周末会去逛街的、会给自己挑衣服的真实的女生。

而他不知道她周末去了哪里。不知道她喜欢什么颜色。不知道她除了物理不好、除了喜欢吃包子先吃素的再吃肉的、除了书包上挂着奶奶给的陶瓷小鱼——之外的其他任何事。

他只知道教室里的那个苏晚晴。

他忽然有了一种感觉——再过87天。高考结束。她走出这间教室。她脱掉校服。穿回她衣柜里那些他没见过的衣服。去一个他没去过的城市。然后他再也没有机会在教室第四排的位置看她的背影了。

他把那道物理公式写完了。继续做题。但脑子里那个浅绿色外套的画面一直在。

下午。数学课。老师发了一张卷子。限时练习——40分钟。选择填空加一道大题。

他做了十分钟。做了五道选择。然后忽然停了笔。往窗外看了一眼。银杏的嫩叶在风里抖。他的脑海里浮现出一个画面——很久以后——他走在路上。看到一个女生穿着一件浅绿色外套。然后愣了一下。发现不是她。

他把这个画面晃了晃脑袋晃掉了。继续做题。

晚自习。黑板上的倒计时被值日生改成了86。86天。

课间顾临风去接水。走到走廊尽头——楼下的老位置。高三配了楼层饮水机，但这个偏僻的角落很少人来。路灯的光从楼下照上来，把树影投在教学楼的墙上。银杏的枝条还没长多少叶子，影子的形状很利落——就是那些枯枝的线条。他端着水杯，喝了一口热水。

走廊里另一个人走了过来。

苏晚晴。

她手里端着那个透明水杯。大概也是来接水的。两个人在走廊里面对面站住。和很久以前多少次一样——走廊。他。她。相隔几步。但这次她没有走他面前。她在离他三步远的地方停住了。靠在墙上。

"你也来透气。"

陈述句。她对他说。

"嗯。"

他回答了一个字。

"里面闷。"她说。

"嗯。"

两个人沉默了一会儿。走廊里的风不大。路灯的光在他们脚边铺了一片。她能闻到教学楼下面的玉兰花开了——三月的玉兰，白色的花瓣大得有点夸张。香气在晚上更浓。

"你排名又进了。"她说。

他看了她一眼。这是他记忆中她第一次跟他说与教室无关的话。不是"让一下"、"谢谢"、"水是常温的"。是在跟他说话。说的是他的名次。

"第七。"他说。

"看到了。"

她说她看到了。看到他进步的名次了。

他又想起了成绩单上那道指甲划痕。从第六名往下走。到第十五名停住。后来变成了从第四名往下。到第七名停住。

"你物理也进步了。"他说。

话一出口他就后悔了。他不应该告诉她他在意她的成绩。

但她没有追问。只是回答。

"还是拖后腿。"

"不拖。"

"你物理好。你说不拖就不拖。"

说完她自己也笑了一下——嘴角弯了一点点。然后拧开水杯的盖子喝了一口水。热水。白气在路灯下面飘了一小团。

她转身往教室走。浅绿色外套在走廊的暗处看起来是灰的。走到教室门口的时候，她侧了一下身——好像想回头说一句什么。但没说出来。然后推门进去了。

顾临风在走廊里多站了一会儿。把杯子里剩下的水喝完了。水已经不烫了——不知道什么时候凉掉的。他把杯子倒过来抖了抖。然后回到教室。

坐到座位上。他把错题本翻到最后一页。在密密麻麻的算式和箭头下面，找了一小块空白。写了一行字。

"她穿了浅绿色外套。"

写完。把本子合上。继续做题。

今天晚上手电筒没有打开。他熄灯就睡了。躺在床上的时候，他看着天花板——那两张荧光星星已经半点光都不发了。就是两个绿色的塑料片。贴着天花板上。黑暗里看不见。但他知道它们在那里。

他脑子里反复出现那个画面——走廊的暗处。路灯的光。她靠在墙上说"你也来透气"。她说"你排名又进了"。她说"看到了"。这三个字他翻来覆去地听了很多遍。她在成绩单上找过他的名字。不是随便看看。是找了。

她找了他的名字。

从第三十一到第十五。从第十五到第十一。从第十一到第九。从第九到第七。她一步一步看着他走到她旁边。

他翻了个身。上铺的板子没响。窗外的玉兰花香在这个季节很浓。隔着玻璃都能闻到。白花的香气。不是甜的。是那种干燥的、带点绿意的花香。他把被子拉到下巴。闭上眼睛。

86天后。他要站在她面前。告诉她自己用了两年半从三十一到第七。不是为了排名。是为了离她近一点。近到可以说出那句话。
