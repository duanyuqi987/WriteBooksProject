# 第066章 墙角的试卷之前

所属部卷：第二部：六月以后
时间：高考后第二天
地点：陆家小院的东屋

## 诗歌插页 P066：原创现代诗

我把凉亭旁的清晨擦亮，
不问它从哪一年开始发凉。
有人经过我的青春，
像风经过一排没有关紧的窗。
我终于学会不把名字攥疼，
只让灯亮一会儿，再让天亮。

这章写的是事情抵达之前的坡度。陆家小院的东屋并没有预告什么，它只是照常开门、落灰、响起脚步声。陆允川却已经把试卷边缘干掉的修正液看得过分清楚，像一个人提前认出了将来会刺痛自己的细节。

这一段的时间感很慢，慢到不像情节，倒像生活自己在铺床、关灯、洗杯子。陆允川在其中反复停顿，也是在反复学习：世界不会因为他的心事改变速度。所以这一章不只写一件事，也写那件事在岁月里怎样慢慢改变形状。那些形状有时锋利，有时柔软，最后都被他放进更宽的生活里。

宿舍上铺有人翻身，床板响了一下，提醒他再大的心事也要小声一点。陆允川站在那样的声音里，忽然觉得自己并不是故事中心，只是众多走过校园的人之一。这个发现当时没有让他轻松，反而让他更想抓住一点能够证明自己的东西。

陆允川把课本和卷子堆在墙角，三年变成一摞纸，他看了很久。真正发生之前，陆允川已经在相似的日常里练习过很多次沉默。那些练习不够戏剧化，却把后来的选择悄悄垫高。他后来反复想起这一段，不是因为它多么宏大，而是因为试卷边缘干掉的修正液。具体的物件比誓言诚实，它不替人辩解，也不替人作证，只安静地留在光里。

他沿着湖边慢慢走，走到第三棵柳树下才发现自己已经绕了一圈。他想说的话有很多，可真正到了嘴边，只剩下几句平常的话。年轻时他总觉得平常太轻，装不下喜欢；后来才知道，太重的喜欢常常先把自己压弯。

小卖部门口的冰柜嗡嗡响，玻璃上结着一层白雾。他伸手擦开一小块，看见里面整齐摆着汽水，像许多被冻结的夏天，标签鲜亮，味道却隔着一层冷。这些细枝末节在当时没有被他认真看见。他忙着判断自己有没有被在意，忙着从别人的一句话里找温度，反而忽略了日子本身怎样一格一格向前挪。

“吃饭，睡觉，去上课。你先把人过成个人，再谈喜欢别人。”祁野这样说时，语气并不郑重，甚至带着一点不耐烦。陆允川当时听不进去，只觉得朋友站得太远。后来他才知道，太近的人往往只能看见火，站远一点的人才看得见整间屋子正在冒烟。

关系在这里往前挪了一点，也往旁边偏了一点。这种变化先露出来：他第一次感到空，空得连喜欢都找不到落脚处。只是他还不肯承认。陆允川那时没有能力准确命名这种变化，只能把它写进纸页，写得越满，越像害怕空白。

朋友们的劝告当时都像噪音，过了很多年，反而成了他记得最清楚的路标。程砚催他去网吧，他说再等一会儿。朋友的声音并不总是温柔，有时甚至粗糙，可它们让他看见世界并没有随着一段关系变窄。

那时候大家都没有多少处理感情的经验，只能从电影、歌词和朋友的胡乱建议里学。学来的姿势常常不合身，所以他们说话用力，沉默也用力，连转身都像在赌气。这些周边生活把爱情从高处拉回地面。它提醒陆允川，心动不是唯一的课程，失落也不是唯一的作业；人总要在很多不浪漫的事情里，慢慢学会怎样对待一件浪漫的事。

她不讨厌被记得，却害怕被一个人用记得围住。那种围住没有恶意，却仍让人喘不过气。夏知遥在家帮父亲整理行李，未来对她来说同样陌生。这部分事实后来才被陆允川补进小说。补进去以后，故事没有变得轻松，却变得诚实了一些。

教室后墙的钟走得很响，每一下都像把某个年轻人的借口往前推一点。他把这一幕和诗歌放在同一页，并不是为了显得文雅。诗对那时的他来说，是一块可以暂时放下重物的石头；重物还在，只是手终于能松开片刻。

原创现代诗写完以后，他在下面空了一行。那一行空白比诗句更安静，像把解释留给未来，而不是逼当时的任何人马上回答。

这首现代诗没有押韵，也不急着把意思说满。它像陆允川当时的呼吸，长短不齐，却正因为不齐，才让人相信它不是从模板里抄出来的。因此这些插页不是装饰，也不是为了让青春显得更漂亮。它们更像一排小灯，照着他从自我感动走向自我审查的路；灯光不强，但足够让人看清脚下有水。

他开始相信，不被回应的喜欢也可以有尊严，前提是它不再向对方索取超出承受的重量。他越长大，越不愿把这段青春说成谁辜负谁。辜负当然存在，可它不是全部；全部里还有胆怯、误会、虚荣、善意和没学会好好告别的年纪。

周末的公交车晃得厉害，窗玻璃贴着一层灰。车经过河桥时，他看见水面一闪，像某个念头从心里掠过，还没来得及抓住就已经过去。这一类日常在小说里反复出现，是因为陆允川后来相信，真正改变一个人的不是某个爆炸瞬间，而是爆炸之后仍要买饭、交作业、赶公交、给手机充电的无数普通时刻。

他把结尾从一句总结改成一个动作。动作更慢，也更诚实。它不宣布成长，只让成长在读者眼前发生一点。他没有因此变得完全宽容，也没有把所有疼痛都解释成成长礼物。他只是终于愿意承认，自己和别人都曾经笨拙，而笨拙不必永远被写成罪。

她把杯子往桌角挪了挪，动作很轻，却让两个人之间多出一小块空地。这个动作很小，小到放进当年的日记里可能会被嫌不够痛。可真正能留下来的，常常正是这些不够痛的小动作，它们让人相信那一年确实活过。

在这一章的末尾，陆允川没有得到答案。屋外有人晾麦子，尘土和阳光一起浮在窗前。他只是比上一章多看见了一点自己，多承认了一点别人，多放松了一点攥紧的手。

很多年以后他回看这页，最刺眼的不是她的冷淡，而是自己把全部生活押在一个回应上的样子。那天最后留下来的，不是结论，而是一件很小的东西和很长的沉默。故事继续往前走，像水流过桥洞，没有替任何人停下，却把所有经过都带成了更低的声音。
