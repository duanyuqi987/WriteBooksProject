# 第128章 出门前

时间：2018年3月周六

地点：楚州，顾临风公寓

卷次：卷七：流年未断

周六。

他八点就醒了。比平时早了两个小时。

先在床上躺了一会儿。天花板上的裂缝在晨光里淡了很多——几乎看不见。窗外的光线是灰白的。楼下有人在遛狗——狗的爪子哒哒哒地从路面上跑过去。

他起来。刷牙。洗脸。刮胡子。

刮胡刀是电动的，超市买的，用了两年。刀网上有个小缺口。他照着镜子慢慢推——下巴、上唇、鬓角。刮完以后用手摸了摸，有点扎，但不明显。

然后对着镜子看了自己大概一分钟。

穿上浅灰衬衫。

第一颗扣子系了两次——手还在抖。扣子孔很紧，扣子又小，手指试了好几次才穿过去。

袖口缺了一颗扣子。右边那只。少的是最下面那个——不太看得出来，只要他不抬手。他把袖口卷了两圈，刚好盖住缺口。

对着镜子整了整领子。领口磨毛的地方他昨晚用手按了很久——没什么用，毛绒绒的轮廓还在。

就这样了。

穿长裤。黑色的。膝盖处有点发亮——熨斗熨过太多次了。他用湿毛巾擦了一下——暗了一点。

皮鞋。鞋油干了没法用，他用湿布擦了擦。鞋面上有些细小的划痕——擦不掉。他挨个擦了擦，又用干布蹭了几下。有点亮，但不多。

出门前又换了一件T恤——打算衬衫外面再套一件外套。衣柜里的外套只有两件。一件深蓝的风衣——太大了，穿上像偷了别人的衣服。一件深灰的夹克——袖口也磨毛了，但磨得比衬衫均匀一些。

他穿上深灰夹克。拉链拉到一半。

站在门口。回头看了一下房间。

被子没叠。枕头歪着。床头柜上放着水杯——喝了一半。床底下有一只袜子在灰里。窗帘没拉严——一条光从缝隙里打进来。

算了。回来再收拾。

他把门关上。用钥匙拧了两圈——锁芯有点涩。拔出来又插进去，重新拧了一次。

楼道里飘着邻居家烧鱼的味道——葱姜蒜。还有谁家的电视机在响。

他下楼。楼梯间声控灯亮了。二楼拐角处有个老太太在扫地。她回头看了他一眼。他点了一下头。她继续扫。

楼下的街上阳光很好。

三月的太阳不烈。风有一点凉。路边的樱花树还在落最后几瓣花——落得很慢。有一瓣粘在他肩膀上。他用手弹掉了。

他看了一眼手机。两点二十。走路到一茶一坐大概二十分钟。时间够。

他没有直接走。

先在楼下站了一会儿。花坛边的流浪猫在舔爪子——黑色的，很瘦，背上有一块秃毛。舌头是粉红色的。一下一下舔得很慢。他看了猫一眼。猫抬头看了他一眼——黄眼睛。竖瞳。然后又低头继续舔爪子。尾巴在地上扫了一下。

然后他往东门方向走。步子不快。两只手揣在夹克口袋里。右手在口袋里捏着手机。左手在口袋里攥着一包纸巾。

走过一家面馆。门上贴着红纸——"本店转让"。面馆已经关了一个月。门口堆着的啤酒箱积了一层灰。走过一家水果店。老板娘在门口削菠萝——菠萝皮在刀下转着圈往下掉。黄色的果肉露出来了。走过一家理发店——门口的地上全是碎头发。碎头发被踩过——和灰尘混在一起。变成了一绺一绺的灰黑色。有个理发师蹲在门口抽烟。看了他一眼。又低头看手机。

他停了一下。看着理发店玻璃门上自己的倒影。

浅灰衬衫。深灰夹克。头发理过了——比昨天短，但没有很短。左边鬓角理得比右边短一点。那个理发师大概手不稳。胡茬刮得也还算干净——只有左边下巴附近漏了一小块。大概两个毫米长。不仔细看看不出来。

看起来像一个正常去相亲的人。

但他知道不是。相亲是两个陌生人坐下来——聊工作、聊家庭、聊兴趣、聊未来。他们不是陌生人。他们之间有八年的距离——三年的同窗和五年的空白。这八年他从教室后排——走到高考——走到大三那晚电话亭——走到现在。她也是。

不是一个正常去相亲的人。

在十字路口等红灯的时候，他把手伸进口袋——摸到了手机。手机背面热热的。他按亮屏幕。相册里有个叫"一"的文件夹。他没有打开。

绿灯亮了。他过了马路。

一茶一坐就在前面了。

茶餐厅的招牌是绿色的，用发光管弯的字——"一茶一坐"。白天没亮。门口有两棵不大的桂花树。玻璃窗很大。里面看得见——暖黄色的灯光，木制的桌椅，吧台前坐着一个人在看书。

他看了看玻璃窗里的自己。

然后低头看了看鞋。鞋面上刚蹭了一块灰——在路牙子上蹭的。他弯腰用纸巾擦了擦。纸巾破了。灰还在。他又用手指抹了抹——灰少了一点。

直起腰来。

深吸了一口气。空气里飘着茶餐厅里飘出来的茶香——茉莉花味。

他看了一眼手机。两点四十。

提前了二十分钟。

他没进去。

在路边的长椅上坐下来。长椅是木条的，漆掉了很多。他坐在最左边。把手搭在膝盖上。

旁边的花坛里有几朵不知名的小花——黄色的，很小。

他看着茶餐厅的玻璃门。

有人推门进去——男的，戴眼镜，穿蓝衬衫。有人推门出来——女的，挎着一个大袋子。

门上的风铃每次都响——叮铃铃。声音很清脆。

两点四十五。

他站起来。拍了拍裤子上的灰。走了两步。又停下。又看了手机——两点四十五。

然后他又坐下了。

一个小孩从面前跑过去——手里拿着一个气球。气球是红色的。小孩跑远了——气球在树梢间一颠一颠的。

两点五十。

他站起来。这次没有坐下。把夹克的拉链拉好。又拉开。又拉好。

然后他走向了茶餐厅的门。

风铃响了。

叮铃铃。

他站起来。这次没有坐下。把夹克的拉链拉好。又拉开——手指在拉链头上停了一下。又拉好。拉到头了。然后拍了拍裤子上沾的灰——其实没有灰。然后看了一眼手机。两点五十。还有十分钟。

十分钟。六百秒。六百分之一天。他站在茶餐厅门外的樱花树下。树上的花瓣已经落完了。全是绿叶。嫩绿的。在午后的阳光里亮亮的。他伸手摸了一下树干——粗糙的。树皮上有裂纹。摸上去能感觉到一棵树的体温——凉的。但活着。然后他转身。走向那扇门。

走过面馆门口。那个"本店转让"的红纸掉了一个角——只剩一角还粘在玻璃上。胶带老化了。风吹过来的时候红纸哗啦响了一下。一个老头站在门口往里看——大概想看看里面还有没有卖吃的。看了两眼。走了。老头走的方向跟顾临风一样。

路过水果店。菠萝已经削好了——切成四块。装在一个透明的塑料盒里。盒子上贴了一个标签——"海南菠萝 特价8元"。老板娘把菠萝装进一个塑料袋里递给一个阿姨。阿姨接过袋子掂了掂。说了句"水分应该不少"。

经过理发店。门口那个剃头的老头已经剃完了。光着脖子站在门口照镜子——用手摸了摸后脑勺。大概剃得还行。理发师在里面扫地——把头发扫成一堆。碎头发在扫帚下翻滚。跟灰尘缠在一起。

他没有停下来看这些。但他都看到了。三月的楚州在他身边慢慢地过着——跟平时一样。面馆。水果店。理发店。菠萝。碎头发。红纸。每一个东西都在原位。只有他不在原位——他在去相亲的路上。去见苏晚晴。
