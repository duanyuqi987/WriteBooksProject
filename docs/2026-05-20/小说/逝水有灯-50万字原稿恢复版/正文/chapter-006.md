# 第006章 新课本的气味之前

所属部卷：第一部：初秋三楼
时间：高一开学周
地点：教室后排靠窗处

## 诗歌插页 P006：原创现代诗

我把早晨放低，
让一阵风从课桌边慢慢经过。
你的名字没有被我喊出，
只在透明书皮里压住的气泡旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写新课本的气味抵达之前的坡度。开学周的教室总有一股新课本的油墨味，像一场还没开始的考试提前把纸页摊开。后排靠窗处风大，书角一不留神就被吹起，哗啦啦响成一排小旗。

陆允川把透明书皮沿着封面压平，指甲顺着边缘刮过去，气泡被挤到书脊旁边，怎么也不肯完全消失。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。

那天真正的事还没有发生，透明书皮里压住的气泡却先被他记住了。它太小，小到放在任何人的日子里都不显眼，偏偏在陆允川这里有了重量。

“包这么认真，准备把数学书供起来？”程砚说。“书皮不平容易翘。”陆允川说。“你是书皮不平，还是心里不平？”程砚说。

程砚说完就被同桌叫去搬粉笔，临走前还把陆允川手边那张成绩单翻了个面，像替他挡住一次不够体面的偷看。

叶澄在发英语练习册，念到夏知遥名字时顿了一下，因为前排同时伸出两只手，她只好笑着说，一个一个来。

课间十分钟被铃声切得很碎。有人挤去小卖部买面包，有人趴在桌上补觉，有人拿着水杯一路跑向水房。陆允川站在这些声音中间，心里那点紧张没有消失，却被生活稀释了一些。

夏知遥这时还只是人群里的一个名字。夏知遥坐在前两排，拿到书后先在扉页写名字。她的字不花哨，横平竖直，最后一笔收得很稳。陆允川看见了，却没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子，又一行行划掉。留下来的，是教室里真实的光、真实的灰、真实的不知道该把手放在哪里。

后排有人借剪刀，有人把书皮裁歪了，急得拿胶带补。班主任在讲台上说新学期要有新气象，底下却全是尺子划过塑料的声音，细碎、认真，又有点好笑。

如果说喜欢在这里有了开端，也不是因为命运突然发声，而是因为一个少年开始过分留意普通事物。他还不懂，留意可以是温柔，也可能慢慢变成对自己的消耗。

包书皮的时候，程砚把透明胶带剪得太短，粘到手指上撕不下来，疼得龇牙。夏知遥听见动静回头笑了一下，又很快低头继续写名字。陆允川把那一下笑记得太清楚，清楚得有点不公平。

班长抱着剩下的练习册从讲台下来，一本一本往后传。传到陆允川这里时封面还带着油墨味，他把书放进抽屉，抽屉里原先留下的一枚订书针扎到手指，疼得很轻。

成绩单被贴在黑板旁边，大家围过去看。陆允川没有立刻挤进去，只装作削铅笔，铅笔屑在桌面绕成一小圈，像他心里那个不肯承认的念头。

放学后，教室里还剩几个人补作业。夏知遥走时把窗帘拉平，透明书皮反了一下光，陆允川眯了眯眼，才发现自己盯着同一个方向太久。

这一章要把脚步放慢，慢到能听见课桌被拖动时的短促摩擦声。事情还没到来，可人物已经在日常里摆好了位置：谁靠窗，谁靠门，谁总替别人递一张纸。

陆允川后来才知道，所谓开始并不是一个明确的按钮，而是很多个微小选择叠在一起。他没有说话、没有靠近、没有确认，可他已经在心里给那个人留出了一块地方。

事情还没有真正发生时，陆允川已经在心里排练了好几种反应。可排练没有用，现实总会把人推到更笨的位置，让他先拿错新课本，再说错一句无关紧要的话。

那天放学，他没有立刻回宿舍，而是在后排靠窗处外多站了一会儿。风把透明胶带吹得轻轻晃动，他忽然意识到，自己等待的并不是答案，只是一个能让明天继续期待的理由。

陆允川把透明胶带拿在手里，先把新课本包完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把书皮包完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她在扉页写下名字并把书角压平，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

后排靠窗处里最明显的不是心跳，而是尺子划过塑料书皮的沙沙声。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，小卖部门口的热豆浆把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。透明胶带、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情还没到来，透明书皮已经先进入他的视线。他当时不懂这种提前的留意有多危险，只觉得只要自己足够安静，就不会打扰任何人。

放学后他在后排靠窗处外多站了一会儿，听见楼下有人喊集合。那一声很普通，却把他从想象里拉出来，让他记起自己还要回宿舍、洗杯子、准备明天的早读。

这一段的力量藏在书皮和油墨味里。陆允川会走神，也会被程砚拉回到新课本前；夏知遥写完名字，生活就继续往下翻页。

如果只写疼，这章会很薄；加上透明胶带、尺子划过塑料书皮的沙沙声和小卖部门口的热豆浆，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在后排靠窗处发呆。”陆允川把透明胶带收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

本章最后，事情仍悬在半空。陆允川收好书，跟着人群往外走。楼道里风从窗缝挤进来，把他的校服袖口吹起一点，他下意识按住，像按住一句还没成形的话。
