# 第029章 借笔

时间：2011年6月下旬

地点：教室

卷次：卷二：窗边人影

自习课。教室里只有翻书和笔在纸上沙沙的声音。

吊扇在头顶转。转得很慢。六月的下午很闷。窗外有蝉叫了一声又短了。

顾临风在做数学题。笔是黑色中性笔。笔杆磨得发亮。

第二排。

一声叹气。

苏晚晴低头翻笔袋。翻了两遍。拉上拉链。

然后她回头。

"谁有多余的笔。"

声音不大。刚好能穿过教室的安静。

她目光扫过周围几排。扫过第三排——他的位置。

他的笔袋里有三支笔。一支黑的。一支蓝的。一支红芯圆珠笔。

他几乎没有想。

手伸进笔袋。抽出那支黑色中性笔。笔是新的。刚用了两天。换上之后还没写满一页。

他递过去。

笔头朝自己。

她接住。

"谢谢。"

声音落在空气中。两秒。然后散了。

他摇了摇头。

她把笔转过来。拔开笔帽。开始写。

马尾垂在椅背上。发绳是黑色的——上次的深蓝色换了。

顾临风看着她用那支笔。笔尖触在纸上。一行一行的字。他的手放在桌上。没动。手指在桌面上轻轻敲了一下。然后又停了。

他从桌洞里拿出那支红芯圆珠笔。在草稿纸上试了一下——断墨。划了好几下才出水。

赵一鸣从旁边凑过来。

"你那支黑笔呢。"

"借人了。"

赵一鸣往前面看了一眼。没说话。

自习课还剩下四十分钟。顾临风用那支断墨的红笔写了一道数学题。写了擦。擦了写。纸擦破了两个洞。赵一鸣看不下去，递了自己的笔过来。

"用我的。"

"你用什么。"

"我还有一支。"

顾临风接过来。继续做题。

下课铃响。

苏晚晴站起来。转身。手里握着那支黑色中性笔。朝他的座位走过来。

两步。一步。

她站在他桌前。

"还给你。谢谢。"

她把笔放在他桌上。笔是横着放的。笔帽朝左。笔尖朝右。

"不客气。"

她转身走了。

顾临风拿起笔。

笔帽上有一个指印。很淡。拇指的。她右手握笔时留下的。有点湿——大概是手汗。

他把笔帽拧开又拧上。没有擦那个指印。

从那以后他一直用这支笔。

用到笔芯干了。出水断断续续的。他还是用。赵一鸣问"你干嘛不换一支"。他说"还没用完"。

赵一鸣看了看那支笔。又看了看他。没再问。

六月末他在笔芯彻底没水的那天，在日记里写字。

"她今天借了我的笔。黑色的那支。还回来的时候笔帽上有个指印。我一直没有擦。我把那支笔用到了最后一滴墨。写出来的字越来越淡。像冬天的雾。但笔帽上那个指印还是能看到。"

然后翻了一页。

"她借笔的时候回头看着我说的。虽然只是在看我这里有没有笔。但是她看着我的。眼睛是深色的。跟她说话时的声音一样——沙哑的。"

写完合上本子。

窗外有人在打篮球。球弹在地上嘭嘭嘭。

他把干了的那支笔放进铅笔盒的最里层。铁皮铅笔盒的角落里。虎队贴纸旁边。

那个位置以前放着一支早就写完的圆珠笔芯。他挪了挪。给黑笔腾了地方。

晚自习。

教室里日光灯亮着。风扇嗡嗡响。

苏晚晴在做物理题。笔换了一支——不是借过的那支。她自己的新笔。粉色笔杆。

顾临风用的笔是赵一鸣那支。课后他买了一支新的还给赵一鸣。赵一鸣说"不用还——我有好多"。他还是塞到他桌上了。

他翻开物理练习册。

开始做题。

题做不出来的时候他就看看那支笔——铅笔盒最里层的那支。笔帽上还有她的指印。

然后低头。继续做题。
