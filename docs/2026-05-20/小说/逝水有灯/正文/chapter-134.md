# 第134章 她和别人说笑旁观

所属部卷：第三部：寒风里的花
时间：六月初
地点：校园超市门口

## 诗歌插页 P134：原创现代诗

她只是笑了一下，
因为同学把纸杯说成纸盆。
夏天的门帘被风掀起，
她看见他的脸色，
也看见自己那一点疲惫。

苏晚枝那天去超市，是替诗社买活动物资。宣传部的男生负责搬东西，路上把“纸杯”说成“纸盆”，两个人都笑了。

笑声很短，短到转身就会忘。可陆允川从超市里出来时，脸色明显沉了一下。苏晚枝不是看不见，她只是不想立刻解释。

她已经解释过很多次：开会就是开会，忙就是忙，不接电话是因为在处理事。每解释一次，她都觉得自己像在交作业，证明自己没有做错。

陆允川问“你今天不是没空吗”时，她心里那点疲惫终于冒出来。

她没有发火，只说：“我没空去听讲座，不代表我不能做别的事。”

这句话听起来有点硬。她知道。但如果不硬一点，陆允川可能永远听不见。礼貌的边界太轻，容易被他当成可以继续靠近的空隙。

后来她把物资放到活动室，宣传部同学问：“你朋友？”

“诗社同学。”她说。

对方点点头，没有再问。苏晚枝把纸杯拆开，数了两遍。数到第三十个时，她忽然觉得委屈。她没有答应谁，也没有欺骗谁。她只是和同学一起买了几袋物资，为什么要在心里准备一场辩解？
