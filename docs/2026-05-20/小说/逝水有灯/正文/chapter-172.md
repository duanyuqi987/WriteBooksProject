# 第172章 最重的话

所属部卷：第四部：失去的美丽
时间：大二冬末
地点：男生宿舍书桌前

## 诗歌插页 P172：原创仿唐绝句

夜半屏光照旧嗔，
一句出门便伤人。
若将疼痛还相掷，
先碎当年未稳心。

消息发出去后，聊天框安静了很久。陆允川盯着屏幕，心跳快得不正常。过了几分钟，苏晚枝回了三个字：“你别这样。”

他看着那三个字，反而更急，继续打：“我怎么了？我只是说实话。”

苏晚枝回得很慢：“这不是实话，这是责怪。”

陆允川手指停在键盘上。宿舍里有人翻身，床板响了一下。祁野从上铺探头，“你是不是发了？”

陆允川没说话。

祁野下床，把手机拿过去看了一眼，脸色变了，“你这话太难听了。”

“我难听？”陆允川声音压着，“她就不难听吗？”

“她拒绝你，不等于她羞辱你。”祁野把手机放回桌上，“你现在是在拿喜欢当刀。”

屏幕又亮了一下。苏晚枝发来一段话：“我没有晾着你。我已经说过很多次边界。如果你觉得我的拒绝是伤害，那我们暂时不要联系了。”

陆允川看着“暂时不要联系”，忽然像被抽空。他想撤回前面的消息，已经超过时间。那句话留在那里，黑字白底，清楚得没有退路。

最重的话不是吼出来的。它只是被他在夜里敲进屏幕，然后准确地到了对方那里。
