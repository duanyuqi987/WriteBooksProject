# 第71章 最大的恨



第二天清晨，男生宿舍天台门口总有一种慢下来的声音。许亦安后来很难准确说出那天的天气，只记得日记本里反复出现的删除线，还有人从他身边经过时带起的一点风。许亦安写下很重的话，写她冷漠、写自己心死。写完后，他删掉一半，只留下最不像控诉的几句。

那时学校还没有完全从白天的喧闹里退出来。食堂门口有人端着打包盒，宿舍方向有人喊同伴名字，教学楼的玻璃被夕光照得一块明一块暗。许亦安站在那里，觉得自己只是普通地等了一会儿，可心里已经把这会儿放大成了很久。

食堂的灯管白得发冷，塑料凳子被人拖来拖去，声音贴着地面跑。

林青禾出现的时候，事情并没有戏剧化。她没有说什么特别的话，也没有给他一个足够让人误会的眼神。她只是像平常那样把书抱在胸前，头发被风吹到脸侧，又很快用手指别到耳后。许亦安却记住了这个动作，像记住一枚后来会反复拿出来看的小扣子。

“你怎么在这儿？”她问。语气不重，像问一个刚好遇见的人。许亦安说：“路过。”说完自己也觉得不像。他明明在这里站了十几分钟，连花坛边第几块砖缺了角都看清了，却只能把漫长说成路过。

她笑了一下，没有拆穿。年轻时最容易让人误会的，常常不是承诺，而是这种没有拆穿。许亦安跟着她往前走了几步，听她说今天课上老师点名，听她抱怨水房排队，听她说舍友买了不好吃的饼干。那些话都很小，小到换个人听了就会忘，他却一路记到了晚上。

她和舍友说起许亦安时，并不是厌恶，更多是困扰。她说他人很好，可好不该变成一条看不见的绳子，把两个人都勒紧。

这并不能抹掉许亦安的疼，却让这份疼不再只有一个方向。

那时候他们都还年轻，年轻到以为情绪足够大，事情就会自动变得重要。

周一鸣说，真正放下的人不会急着证明自己放下了。许亦安听完把手机翻过去，屏幕贴着桌面，像把一张难看的脸先藏起来。

那天他们说到一件很无聊的小事。无聊到后来谁也不会承认它重要。可许亦安记得林青禾说话时低头看了一眼手机，屏幕亮了又灭，像一扇不肯完全打开的门。他没有问是谁的消息，只把这个动作记下来，放在心里一个很靠前的位置。

楼道里有人提着热水壶经过，水壶盖子碰着壶身，发出很轻的响。

他在路边买了一瓶水，拧开以后又忘了喝。林青禾伸手要看瓶盖上的中奖字样，他递过去，手指碰到她的手背，又很快缩回来。那一瞬间没有任何意义，至少对她来说大概没有。可是许亦安回到宿舍后，还是把它写进日记本里，写得比事情本身长得多。

天台门锁着，门缝里灌进来的风带着一点灰。许亦安蹲在门口，手机屏幕亮得刺眼。他写一个字，删两个字，删到最后手指发酸。楼下有人打篮球，球砸在地上的声音一下比一下远，他忽然觉得自己的愤怒也像那颗球，弹得很高，落下来还是在原地。

那时他还分不清关心和占有，只觉得自己越疼，就越有资格问为什么。

夜里宿舍熄灯后，曹原还在床上翻身，周一鸣戴着耳机听英语听力，走廊里有脚步声来来回回。许亦安把手机藏在被子里，屏幕光照着他的脸。他删掉“我今天很开心”，改成“今天风不大”。删完以后又觉得太冷，补了一句“但是她笑了”。

他写下很重的话，写到一半手指停住。那些句子看起来像骂林青禾，其实每一句都在暴露自己多想被回应。他盯着屏幕看了很久，最后按住删除键，让一整段话慢慢退回空白。

他点头的时候太用力，连自己都觉得可笑，只好低头看鞋尖。

他开始意识到恨只是放不下的另一种说法。恨看起来硬，里面却是软的，是没等到、没问清、没被选择。

删到最后，他只剩下一句“不想再这样了”。这句话没有对象，不指责谁，也不求谁回来。许亦安看着它，第一次觉得自己像从一间很闷的屋子里推开了窗。

第二天醒来时，宿舍窗帘只拉了一半，光从缝里斜斜进来。曹原趴在枕头上骂谁把闹钟设得这么早，周一鸣已经洗漱完，端着脸盆站在门口催他去吃早饭。许亦安把手机摸出来，先看有没有新消息，再假装自己只是看时间。

食堂早上的队伍总是很长，铁勺碰到不锈钢盆，声音一阵一阵。周一鸣买了两个馒头，塞给他一个，说人不能只靠情绪活着。许亦安笑了一下，把馒头掰开，里面的热气散出来。他忽然觉得这句话难听，却比很多安慰都实在。

上午的课讲得很慢，老师在黑板上写了半面公式，又用袖口擦掉一个写错的符号。许亦安看着那个被擦掉的位置，想到昨晚日记本里删掉的句子。粉笔灰落下来，细细的一层，他在本子边上写下一个“风”字，又用笔尖把它涂黑。

课间有人从后排传来一张纸条，问晚上要不要一起去网吧。曹原回了一个大大的“去”，周一鸣在旁边补了一句“别把他带坏”。许亦安没有参与，手机扣在桌面上，像一只安静的小盒子。他知道自己在等它亮，也知道这种等待没什么道理。

下午经过男生宿舍天台门口时，他又想起日记本里反复出现的删除线。有些物件本来只是物件，被人反复想起之后，就会变成一枚钉子，轻轻钉在记忆里。拔不出来时不见得多痛，可每次碰到，总会提醒你那里曾经受过力。

曹原半夜回来，看见他还坐着，骂了一句神经病，又把桌上的泡面推给他。那碗面泡得太软，却让他从一堆重话里回到胃和热气。

回宿舍的路上，他经过公告栏，看见上面贴着旧海报。海报边角卷起来，露出后面更早的一张纸。许亦安盯着那层纸看了一会儿，想到人心也差不多，新话盖着旧话，新热闹盖着旧难过，可只要风一吹，最下面的东西还是会露出来。

他没有把这些想法告诉林青禾。那时候他还以为，真正深的感情应该自己消化，不能给对方添麻烦。后来他才知道，不能说的感情很容易在心里变形，变成委屈，变成审问，变成某个夜晚突然冒出来的重话。

他把一句话删掉，又把它换成天气。天气比情绪体面，风、雨、晴天都不会追问原因，也不会笑他把一件小事记得太久。

那一行字不漂亮，也不锋利：别把疼写成刀。后来他觉得，这可能是那天最接近成长的一句。

他没有把所有答案都写死。年轻时他最怕含糊，后来才懂，很多温柔正是因为没有把别人逼到无路可退。

修到这一章时，他终于不急着给青春下结论。很多事只要放在那里，风吹一吹，读的人自然会知道它曾经重过。

很多年后他再想起这一页，最先想起的不是难过，而是那天具体的光线和风。人一旦开始记得细节，而不是反复追问谁对谁错，伤口就已经在悄悄收口。

这一章真正留下来的，不是他当时以为最重要的那句话，而是天台门锁着，他没有上去，只在门口站了一会儿。

他把那件事记住，不是因为它多惊天动地，而是因为它太像青春本来的样子。许亦安合上手机，听见外面有人笑着从楼下经过。那笑声很快远了，宿舍重新安静下来。他躺在床上，觉得这一天好像没有结束，又好像早就结束了。
