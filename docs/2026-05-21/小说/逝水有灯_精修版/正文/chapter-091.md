# 第091章 夏天的聚餐之前

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P091：原创现代诗

我把星放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
小饭馆圆桌上的玻璃转盘没有替谁回答，
只把那一年轻轻照亮。

八月中旬，各家的升学宴陆续开始了。

程砚妈妈最先张罗。她在县城最热闹那条街上的一家川菜馆订了两桌——家里亲戚一桌，同学一桌。提前三天就给陆允川打了电话："小陆你一定要来啊，程砚说你不来他就不吃了。"

"阿姨你放心，我肯定来。"

宴席那天，陆允川穿了一件干净的T恤，头发剪过了——母亲催了三次。他到了川菜馆门口，看到程砚穿了一件蓝色衬衫站在门口迎客，领口扣到最上面一颗，整个人看起来比平时正经了十倍。

"你这衬衫谁给你买的？"陆允川走过去。

"我妈。她说升学宴要穿衬衫。我说我都考上省城工学院了穿什么衬衫，她说就是考上了才要穿。"程砚扯了扯领口，"太紧了。我呼吸有点困难。"

"你把第一颗扣子解开。"

"我妈不让。"

陆允川伸手把他第一颗扣子解开了。程砚深吸一口气。"舒服了。谢谢你救了我一命。"

同学那桌坐了十来个人：陆允川、叶澄、周长海、林曼、还有几个平时玩得好的男生。夏知遥没来——她家那天正好也有亲戚来，杜雨薇替她带了一个红包过来。红包上写了四个字：前程似锦。字迹很工整，横平竖直。

"她写字还是这么好看。"林曼把红包放在桌上。

叶澄看了一眼字迹。"她自己练的。高二的时候她买了一本钢笔字帖，每天晚自习抄一页。她说学建筑的，以后图纸上要写字，写不好看丢人。"

"这也要练？"程砚凑过来看。

"你以为。优秀的人什么都练。"

程砚他妈端着茶壶过来给同学桌加水，笑眯眯地问了一圈人考上了哪个学校。问到叶澄的时候，叶澄说省城师范，程砚妈妈立刻握住她的手："老师好啊！你以后要当老师，程砚要是当年多听你的话就好了。"叶澄难得地脸红了一下。

问到陆允川的时候，程砚妈妈说土木好，以后可以修桥。又问陆允川妈妈怎么没来，陆允川说她今天上班。程砚妈妈说那你回去告诉她，你这孩子是我们程砚最好的朋友。

陆允川低下头喝茶。茶很烫。

周长海考上了一个专科学校——汽修专业。他在桌上说了这件事之后，大家沉默了一秒。然后程砚第一个开口："汽修好啊。你知道现在修一辆宝马多少钱吗？"

"不知道。"

"我也不知道。但肯定比修我们那辆破自行车贵。"

周长海笑了一声。"你安慰人的方式还是这么奇怪。"

"我没有安慰你。我是真的觉得汽修好。你修车，我造机器。以后说不定能合作。"程砚把茶当酒，举起来敬了周长海一杯。

周长海也举起茶杯碰了一下。两个人都没有说太多，但玻璃杯碰在一起的声音很响。
