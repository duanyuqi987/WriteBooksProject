# 第044章 诗抄本旁观

所属部卷：第一部：初秋三楼
时间：高二秋天
地点：教室后排的课桌抽屉

## 诗歌插页 P044：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
诗抄本里被铅笔写在角落的名字还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，诗抄本作业在全班范围内是一场大规模的手工劳动——有人抄李白抬头望明月，有人抄冰心繁星点点，有人直接在诗抄本上画火柴人然后告诉老师这是看图写诗。对大多数承受这项作业的人来说，诗抄本只是一个学期末必须交的作业——页码够、字数够、封面写好名字就行了。但对陆允川来说，诗抄本是整个高二秋季的精神日记——每首诗底下附着的注释比诗本身还长。

语文老师最先发现了异常。她在办公室跟物理老师聊天时说："我们班有个男生，诗抄本抄得太认真了——每首诗的注释都像是现场笔记。而且注释里总有某个人的名字——不是真名，是'她'。"办公室里的饮水机正烧着水，咕嘟咕嘟地冒着泡，物理老师的办公桌上堆着三摞还没批完的试卷，红笔搁在试卷最上面。物理老师正在批卷子，隔了好几秒才抬起头说："'她'——那可能是人家喜欢的女同学吧。写作业的时候开小差很正常。""但这个开小差已经形成了一套完整的观测系统——他每周交上来的注解里包括时间、地理坐标、该女性的行动路线——我觉得他应该去学新闻系。"

程砚对诗抄本的理解更市场化。"你这本诗抄本毕业以后留着——十年后它会成为你初恋纪念馆的一号文物。到那个时候你在上面写上腰封印上来——这本诗抄本里记录了一个少年每天用诗掩饰自己偷看喜欢的人的犯罪事实。"陆允川笑一声。

叶澄借走他的诗抄本看了一遍，还本子的时候说："你这些注释有一个共同特点：你只记录她的动作、位置、她说的话，但从来不记录你自己的心理活动。你把所有自我评判的空间压缩到零——然后她用五分钟做的事你写两句话——这种写作方式的客观性跟我写班务日志差不多。""那不是客观,"陆允川说,"那是怕记不清了——心理活动随时有，但她的细节我不记的话第二天会忘。"

叶澄想了一下接过本子，翻开里面一页指着一行给他看——那是一首顾城的诗底下："今天她把窗帘拉歪了一下,我以为她在调整光线。拉完之后她又把窗帘理正了——原来她只是不小心碰了一下。"叶澄说："你这叫客观吗？你连'我以为'后面都帮你自己的动机做了让步解释——你已经主观到不知道自己是主观的人了。"

蒋雯可能察觉到了一点点。她有次在发卷子时看见陆允川的诗抄本翻开放在桌上——某一页的边上有一行小字，记载着夏知遥某一次在课堂上打了一个喷嚏然后找纸巾。喷嚏声音很小，用手捂住了，她低头从笔袋里取出纸巾擦了擦鼻子——用的是一包小包装的清风纸巾，包装上印着蓝色花朵。他把这些全记下来了。蒋雯把卷子放过去以后对何敏说："陆同学的诗抄本里有点奇怪的东西——不像是诗——像是观察笔记。""观察什么？""不太确定。"后来蒋雯硬追了一条线索——观察对象终于从一堆打码信息里被解码出来了:她。蒋雯后续没说什么。但她把这件事埋进了心里——像把一颗弹力球塞进装满回忆的罐子里——不滚动不亮出来，就放在那里。
