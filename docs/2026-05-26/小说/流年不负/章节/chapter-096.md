# 第096章 大二

时间：2014年9月

地点：楚州师院

卷次：卷五：各自南北

九月。大二了。

新生报到那天，顾临风在宿舍窗口往下看。楼下是同样的场景——拉着行李箱的家长和学生，从红色横幅下面走过去。横幅换了新的，布上的金边还没褪——"欢迎2014级新同学"。

去年的这个时候，他站在那个横幅下面。一个人。

今天他从五楼窗口往下看。

李博在背后整理书架。"看什么呢。"

"新生。"

"感慨了？"

"还好。"

时间过得快。大一一年——从九月的陌生到六月的习惯。教室、食堂、图书馆、操场——五百多天走过了同一个路线。操场的跑道被跑出了磨痕。电话亭的玻璃被换了一块新的——鸟粪的痕迹没了。

不联系苏晚晴——已经一年多了。

三百九十多天。他没有给她发过一条短信。没有打过一次电话。没有路过师大的北门。301路公交每天从师院南门发车——早上六点到晚上九点——每十二分钟一班。他从来没有坐过。

许若琳升大二以后更忙了。英语系的课多——专业课加二外日语。她来文学社的次数变少了。偶尔在QQ上发一条消息——"最近社团怎么样"——"还行"——"听说刘思齐卸任了，新社长是写诗的那个师兄"——"对"。

她不再频繁地提起苏晚晴。顾临风也不再问。两个人之间的话题从"晚晴最近在做什么"变成了"你们文学社换届了吗"和"日语比英语难学一百倍"。

暑假支教的事已经过去四个月了。李家沟小学在太行山深处——两排平房、一个土操场、一根旗杆。教室里没有灯——白天上课全靠窗户的自然光。他的宿舍是学校腾出来的一间教室——用木板隔成两间，男生一间女生一间。睡觉的床是课桌拼的，被子是学校老师从家里拿来的。晚上山里冷——七月的太行山晚上要盖两床被子。

他教了十四天语文。二十一个孩子。每天四节课。早上一节、上午两节、下午一节。孩子们坐在歪斜的木头课桌后面，写的字歪歪扭扭——但每一个都很用力。他给他们讲《静夜思》。讲"床前明月光"的时候有个孩子举手问——"老师你家离这里多远"。他说不远。但其实很远——三百多公里，要转三趟车。

走的那天一个小女孩拉着他的书包带说"老师你明年还来吗"——他说"来"。

大二的课比大一深。古代文学进入了唐宋——李白、杜甫、白居易。现代汉语进入了语法结构——"主谓宾"、"偏正结构"。书法课从颜体转到了行书——他每天在宿舍的水泥地上铺报纸，用毛笔蘸清水临《兰亭序》。

阿坤有一次踩到了他的报纸。"兄弟你这练习环境也太艰苦了。"

"省钱。"

"你不是省钱——你是不想让自己闲着。"

顾临风没反驳。

九月下旬的一个晚上。他收拾书桌。从抽屉最里面翻出了高中的那本暗恋日记。蓝色封面。边角卷了——因为被压在一堆笔记下面一年多。

他翻开第一页。

"2010年9月13号。她今天换了发绳。以前是黑的。今天是浅蓝色的。"

"2011年5月20号。她物理72分。卷子发下来的时候她咬了咬嘴唇。我觉得她有点难过。"

他接着往后翻。每一页都是类似的记录——她今天穿了什么颜色的衣服、她今天笑了几次、她今天考了多少分、她今天跟谁说了什么话。不是日记。是一个暗恋者的记账本——他把关于她的一切都记了下来。怕忘记。怕哪天醒来就不记得了。

最后一篇日记停在2013年8月29号——"她说了那句话。她说暂时不要联系了。我答应了。"

他合上日记。放回原来的位置。坐在椅子上。窗外有新生在军训——喊口号的声音很远，透过玻璃窗只听得见嗡嗡的共振。

他打开手机。通讯录。S开头。苏晚晴。电话号码还在。没有删。他这辈子不会删这个号码。但他也不会打。

他把手机放下。翻开专业书。继续看《古代文学》。
