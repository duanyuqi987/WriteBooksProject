# 第030章 第一次说话多年后

所属部卷：第一部：初秋三楼
时间：多年后
地点：出租屋书桌

## 诗歌插页 P030：原创仿唐绝句

楼灯旧影已难寻，
一册归来误到今。
寻常话里无深意，
深意原来在我心。

多年后，陆允川翻到那本黑色物理笔记。

封面已经软了，姓名栏被磨得更淡。最后一页夹着一张揉皱又展开的纸条，纸条上是程砚的字：文具失而复得，恭喜。

陆允川看见这行字，笑了很久。

他把笔记本摊在桌上，试着回想第一次和夏知遥说话。奇怪的是，他记得楼梯间坏掉的灯，记得人群里的热水味，记得自己掌心的汗，却几乎记不清她那天的表情。过去被他写得太重，重到人物的真实面孔反而模糊了。

他以前总觉得“你的本子”四个字有某种特殊意义。后来才知道，意义不在那句话里，而在他当时迫切想要意义的心里。夏知遥只是做了一件普通、善良、合乎分寸的小事。真正把它变成大事的人，是他。

这个认识没有让那段记忆变坏，反而让它变得干净。

陆允川拿出铅笔，在旧纸条背面写：

谢谢你把本子还给我。
也谢谢你没有翻开它。

写完以后，他又觉得这两句话像一封永远不会寄出的信。可不寄也没关系。有些话不是说给对方听的，是说给当年的自己听的。

窗外下着小雨，雨点敲在防盗窗上，声音细碎。桌上放着几个文件袋：高一英语、高一理科、高中杂物。陆允川想了想，把这本黑色笔记放进“高中杂物”里，而不是单独拿出来。

它当然重要。可它不该独占那一年。

那一年还有程砚的薄荷糖、公告栏的红纸、看台的余热、食堂里的土豆丝、夏知遥自己的课本和电话。所有这些东西放在一起，才是高一。单独拎出一句话，反复擦亮，反而会让日子失真。

陆允川合上文件袋，拉链顺滑地合拢。那一声很轻，像终于替很多年前的楼梯间关上了一扇门。
