# 第161章 寒风里的台阶之前

所属部卷：第四部：失去的美丽
时间：大二冬天
地点：女生公寓外台阶

## 诗歌插页 P161：原创现代诗

我把枯花茎放回夜色，
不让它替谁证明深情。
风从未说完的话里穿过，
有人低头，学着不再追问。
后来水声慢下来，
灯还亮着，人已经能往前行。

大二那年冬天，陆允川做了一件后来想起就会皱眉的事：他在苏晚枝的宿舍楼下等了近两个小时。

事情的起因是一本书。苏晚枝在课上把一本复习资料借给了他，他在里面发现她夹了一张便签，上面用铅笔写着几个诗人名字，其中有一个他最喜欢。他立刻觉得这是某种信号。

不是信号。后来他才知道。她忘记清理废纸而已。

但当时的他不这么认为。他用一个下午翻遍了自己的书架，找到所有他认为苏晚枝会喜欢的书，装在一个帆布袋里。一共七本，每一本都在扉页写了赠言。内容各不相同，但主题一致：我懂你。

他准备在晚上送到她楼下，不提前发消息——给她一个惊喜。

下午五点半，他在宿舍选衣服。把三件外套依次铺在床上，问程砚哪件合适。

"你有什么事吗？"程砚从书上抬起眼睛。

"没什么事。就是把几本书还给苏晚枝。"

"还书需要挑外套？"

"顺便。就顺便而已。"

"你选中间那件。"程砚把书合上，"黑色那件显瘦，蓝色那件洗了还没干，军绿色那件——那是你爸的衣服吧？你从来不穿。"

陆允川把手从军绿色外套上收回来。"对。是我爸的。"

"那你为什么要铺出来？"

"——不知道。"

程砚没再问，重新翻开书。过了几秒才说了一句："还书就还书。别把'顺便'写成计划书。"

祁野在床铺上探下头来："你又要去楼下等人？大冬天的？"

"又不是没等过。"

"上次你等了多久？"

"四十分钟。"

"这次呢？"

"不知道。看情况。"

"'看情况'。"祁野咀嚼了一下这个词，"你的'看情况'一般意味着两小时起步。"

陆允川不接话，把黑色外套穿上，拉链拉到下巴。帆布袋拎在手里有点沉，七本书的重量不轻，走在楼道里袋子碰着腿，一步一撞。

下楼的时候，他在楼梯间碰见夏知遥。她刚下班回来，羽绒服外面套着奶茶店的围裙，头发被风吹乱，鼻尖冻得通红。

"这么晚去哪？"

"去还书。"

"还苏晚枝？"

"嗯。"

夏知遥拎了拎他手里的帆布袋。"七本？"

"你掂一下就知道是七本？"

"奶茶店练的——一杯中杯三百克，大杯五百克。你这个袋子大概四斤半。"她把袋子放下，"你确定她是需要这七本书的人？"

"不确定。"

"那你送什么？"

"——想送。"

夏知遥看了他几秒。她的表情在楼道昏黄的灯光下看不清楚，但语气是清楚的："不要用送东西来换见面。女孩子会觉得很累。"

"我没有要换什么——"

"你有。"夏知遥说，"你每次说'我没有要换什么'的时候，其实就是在换。你不说出来是因为你自己都不想承认。"她从书包里拿出一杯奶茶往他怀里一塞，"拿着喝。热的。奶茶你请——不是，奶茶白送你的。别在楼下站太久，今晚有风。"

她说完就上楼了，脚步很快。那双在奶茶店站了六个小时的脚此刻踩在楼梯上，发出不均匀的声响。

陆允川看着手里的奶茶。杯身有点烫手，标签上写着"中杯珍珠奶茶——无糖"。他的口味她知道？还是她随便拿了一杯？

他想，可能只是随便拿的。

走到女生公寓楼外时，天已经全黑了。夜风从操场方向灌过来，把花坛里的枯茎吹得贴着地面抖。台阶被风吹了一整天，坐上去腿会发麻。门灯在风里微微发抖，光晕断断续续的。

他选了台阶最右侧的一级坐下。帆布袋搁在旁边，一边用手护着，好像护一支快要被风吹灭的蜡烛。他给苏晚枝发了一条消息："在楼下。有东西给你。不着急，你忙完再下来。"

等了几分钟，苏晚枝回了："什么东西？"

"几本书。"

"什么书？"

"你看看就知道了。"

苏晚枝没有马上回。十分钟后，她回了三个字："你还在？"

"还在。"

"我换个衣服。"

陆允川看着这四个字，心里上升了一度。她说"换个衣服"——说明她愿意下来。愿意下来意味着她至少不排斥见他。不排斥见他意味着——

他忽然掐断了自己的思维链条。因为他想起南湖那夜叶澄说的："你一直在用她来证明自己是一个什么样的人。她可能根本不想参与这个证明。"

但南湖是后来的事。此刻是大二冬天，他还没有经历过南湖，不知道叶澄日后会说出那样的话。他只知道手里这个帆布袋里装着七本书，七本书的扉页上写着七段赠言，七段赠言的中心思想都是"我懂你"。

有人拎着热水瓶从楼门里出来，壶盖碰在壶身上，清脆的声音在冷空气里传得很远。那个声音像一句提醒——但他还没学会接收。

他把手缩进袖口。指尖冻得发麻，帆布袋的提手在掌心压出一道红痕。

手机亮了。程砚的消息，只有一行：

"还在等？"

"嗯。"

"你最好先想清楚：等下见面说什么。不要说多于三句话。超过三句就成了表白。"

陆允川打了一个"好"，又补了一句："程砚。"

"嗯？"

"你是怕我说太多，还是怕她听太多？"

"都怕。"程砚回复，"你说太多她会负担，她负担之后你会后悔，你后悔之后会找我和祁野复盘，我和祁野没有新词。"

陆允川看着屏幕，嘴角扯了一下。不是微笑，是冬天里被风吹出来的本能反应。

楼门又响了。不是苏晚枝——是另一个女生，出来倒垃圾。她看了一眼坐在台阶上的陆允川，脸上没有表情，像是在冬天的校园里见过太多坐在楼下等人的男生。那种目光匆匆扫过他的帆布袋，好像早就知道里面装了什么，在她眼里全是徒劳。

风又大了一点。他把外套的帽子扣上，继续等。
