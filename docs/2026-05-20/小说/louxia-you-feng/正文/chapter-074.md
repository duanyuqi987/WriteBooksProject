# 第74章 第二天早晨



七月初的清晨，学校食堂一楼窗口总有一种慢下来的声音。许亦安后来很难准确说出那天的天气，只记得食堂窗口递出的温热豆浆，还有人从他身边经过时带起的一点风。林青禾像平常一样买豆浆，许亦安也像平常一样打招呼。两个人都绕过昨夜，像绕过地上的一滩水。

那时学校还没有完全从白天的喧闹里退出来。食堂门口有人端着打包盒，宿舍方向有人喊同伴名字，教学楼的玻璃被夕光照得一块明一块暗。许亦安站在那里，觉得自己只是普通地等了一会儿，可心里已经把这会儿放大成了很久。

远处操场有人吹哨，声音被教学楼挡了一下，传到这里时已经软了。

林青禾出现的时候，事情并没有戏剧化。她没有说什么特别的话，也没有给他一个足够让人误会的眼神。她只是像平常那样把书抱在胸前，头发被风吹到脸侧，又很快用手指别到耳后。许亦安却记住了这个动作，像记住一枚后来会反复拿出来看的小扣子。

“你怎么在这儿？”她问。语气不重，像问一个刚好遇见的人。许亦安说：“路过。”说完自己也觉得不像。他明明在这里站了十几分钟，连花坛边第几块砖缺了角都看清了，却只能把漫长说成路过。

她笑了一下，没有拆穿。年轻时最容易让人误会的，常常不是承诺，而是这种没有拆穿。许亦安跟着她往前走了几步，听她说今天课上老师点名，听她抱怨水房排队，听她说舍友买了不好吃的饼干。那些话都很小，小到换个人听了就会忘，他却一路记到了晚上。

她也会不安。不是因为不懂许亦安的好，而是因为那份好越来越像一个问题，仿佛她必须给出答案，才配得上别人等待的时间。

她能给出玩笑、帮忙和短暂陪伴，却给不出许亦安暗暗期待的确定。

林青禾不是故意残忍。很多年后他才懂，一个人的犹豫也可能只是她保护自己的方式。

曹原看不下去，说他笑得比哭还累。这话许亦安当时不爱听。他更愿意相信自己看见了别人看不见的东西：她笑时眼睛弯起来的弧度，她把吸管咬扁的习惯，她说“随便你”时其实并不是真的随便。喜欢一个人后，人的听力会变奇怪，能听见很远的暗示，却听不见近处的提醒。

那天他们说到一件很无聊的小事。无聊到后来谁也不会承认它重要。可许亦安记得林青禾说话时低头看了一眼手机，屏幕亮了又灭，像一扇不肯完全打开的门。他没有问是谁的消息，只把这个动作记下来，放在心里一个很靠前的位置。

他点头的时候太用力，连自己都觉得可笑，只好低头看鞋尖。

他在路边买了一瓶水，拧开以后又忘了喝。林青禾伸手要看瓶盖上的中奖字样，他递过去，手指碰到她的手背，又很快缩回来。那一瞬间没有任何意义，至少对她来说大概没有。可是许亦安回到宿舍后，还是把它写进便签里，写得比事情本身长得多。

他总以为自己在包容，其实很多时候只是害怕失去，所以不敢表达不满。

夜里宿舍熄灯后，曹原还在床上翻身，周一鸣戴着耳机听英语听力，走廊里有脚步声来来回回。许亦安把手机藏在被子里，屏幕光照着他的脸。他删掉“我今天很开心”，改成“今天风不大”。删完以后又觉得太冷，补了一句“但是她笑了”。

他并不觉得自己在写小说。那只是便签，是怕忘，是怕某一天她说一切都没有发生过时，他至少还能拿出一些证据。证据里有时间，有地点，有食堂窗口递出的温热豆浆，有他被自己感动过的心情。

食堂的灯管白得发冷，塑料凳子被人拖来拖去，声音贴着地面跑。

粉饰太平让裂缝更清楚。这变化不是突然发生的。它像楼道里的风，一开始只是从门缝里进来，吹动纸页一角。等人真正觉得冷时，纸已经翻过去好几页了。

林青禾后来发来一条消息，内容很短，只是问他明天有没有课。许亦安盯着那几个字看了很久，仿佛里面藏着另外一层意思。他回得很慢，字斟句酌，最后却只发了“有，下午没课”。发出去的一瞬间，他又后悔自己为什么不更自然一点。

年轻时的许亦安总是在这些小地方用力。他怕太热，也怕太冷；怕对方看出来，又怕对方看不出来。于是每一句话都像一枚硬币，被他在指间翻来覆去，直到边缘磨得发烫。

第二天醒来时，宿舍窗帘只拉了一半，光从缝里斜斜进来。曹原趴在枕头上骂谁把闹钟设得这么早，周一鸣已经洗漱完，端着脸盆站在门口催他去吃早饭。许亦安把手机摸出来，先看有没有新消息，再假装自己只是看时间。

食堂早上的队伍总是很长，铁勺碰到不锈钢盆，声音一阵一阵。周一鸣买了两个馒头，塞给他一个，说人不能只靠情绪活着。许亦安笑了一下，把馒头掰开，里面的热气散出来。他忽然觉得这句话难听，却比很多安慰都实在。

上午的课讲得很慢，老师在黑板上写了半面公式，又用袖口擦掉一个写错的符号。许亦安看着那个被擦掉的位置，想到昨晚便签里删掉的句子。粉笔灰落下来，细细的一层，他在本子边上写下一个“风”字，又用笔尖把它涂黑。

课间有人从后排传来一张纸条，问晚上要不要一起去网吧。曹原回了一个大大的“去”，周一鸣在旁边补了一句“别把他带坏”。许亦安没有参与，手机扣在桌面上，像一只安静的小盒子。他知道自己在等它亮，也知道这种等待没什么道理。

下午经过学校食堂一楼窗口时，他又想起食堂窗口递出的温热豆浆。有些物件本来只是物件，被人反复想起之后，就会变成一枚钉子，轻轻钉在记忆里。拔不出来时不见得多痛，可每次碰到，总会提醒你那里曾经受过力。

他也试过把这一天讲给别人听。可讲出口以后，事情立刻变小，像一张被折过的纸，再展开也有折痕。朋友听到的是一件普通小事，他心里却藏着一整场天气。于是他慢慢少讲，只写。写下来至少不用解释为什么这也值得难过。

回宿舍的路上，他经过公告栏，看见上面贴着旧海报。海报边角卷起来，露出后面更早的一张纸。许亦安盯着那层纸看了一会儿，想到人心也差不多，新话盖着旧话，新热闹盖着旧难过，可只要风一吹，最下面的东西还是会露出来。

他没有把这些想法告诉林青禾。那时候他还以为，真正深的感情应该自己消化，不能给对方添麻烦。后来他才知道，不能说的感情很容易在心里变形，变成委屈，变成审问，变成某个夜晚突然冒出来的重话。

宿舍的灯熄了以后，他仍能听见键盘声和翻身声。那些声音把他的心事衬得不那么孤单，也不那么特别，像每个年轻人都在黑暗里藏着一页纸。

那一行字写得并不漂亮，甚至有些歪，却比他当时说出口的所有玩笑都诚实。

他没有再把自己写成唯一受伤的人。写到这里时，他停下来喝了口水，像终于肯把另一半事实也放进故事里。

这一轮修订里，他尽量不把每次等待都写成牺牲。等待有时只是不会处理情绪，不必被装饰成高尚，也不必被骂得一无是处。

很多年后他再想起这一页，最先想起的不是难过，而是那天具体的光线和风。人一旦开始记得细节，而不是反复追问谁对谁错，伤口就已经在悄悄收口。

这一章真正留下来的，不是他当时以为最重要的那句话，而是豆浆袋在指间晃着，热气很快散在早晨里。

那天没有什么决定性的告别，可生活最常见的告别，本来就不响。许亦安合上手机，听见外面有人笑着从楼下经过。那笑声很快远了，宿舍重新安静下来。他躺在床上，觉得这一天好像没有结束，又好像早就结束了。
