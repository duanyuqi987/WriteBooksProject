# 第80章 他的日记本



真正翻完旧日记本是在傍晚。宿舍窗户开着，风把纸页往回吹，像不愿让他读到后面。许亦安一只手压着封面，一只手把几张夹在里面的车票、电影票和借书条捡出来。

楼下有人把旧书卖给收废品的师傅，称重的铁秤哐当一声落下去。许亦安听见那个声音，忽然觉得很多东西一旦被拿出来称，就会显得很轻。

旧车票从书页里滑出来，落在地上，没有声音，却让他低头很久。

林青禾出现的时候，事情并没有戏剧化。她没有说什么特别的话，也没有给他一个足够让人误会的眼神。她只是像平常那样把书抱在胸前，头发被风吹到脸侧，又很快用手指别到耳后。许亦安却记住了这个动作，像记住一枚后来会反复拿出来看的小扣子。

“你怎么在这儿？”她问。语气不重，像问一个刚好遇见的人。许亦安说：“路过。”说完自己也觉得不像。他明明在这里站了十几分钟，连花坛边第几块砖缺了角都看清了，却只能把漫长说成路过。

她笑了一下，没有拆穿。年轻时最容易让人误会的，常常不是承诺，而是这种没有拆穿。许亦安跟着她往前走了几步，听她说今天课上老师点名，听她抱怨水房排队，听她说舍友买了不好吃的饼干。那些话都很小，小到换个人听了就会忘，他却一路记到了晚上。

如果只从许亦安这边看，林青禾像是忽近忽远的风。可她那天也不是空白，她要赶作业、回宿舍、处理家里的电话，还要在一群同学的眼光里保留一点自己的体面。

她没有把这些话说出口，因为说出口就太像责怪，而她并不想伤害他。

后来他想，真正难忘的从来不是某一句话，而是那句话发生时周围很小的东西。

曹原帮他把书塞进行李箱，说别把人也打包带走。这话许亦安当时不爱听。他更愿意相信自己看见了别人看不见的东西：她笑时眼睛弯起来的弧度，她把吸管咬扁的习惯，她说“随便你”时其实并不是真的随便。喜欢一个人后，人的听力会变奇怪，能听见很远的暗示，却听不见近处的提醒。

那天他们说到一件很无聊的小事。无聊到后来谁也不会承认它重要。可许亦安记得林青禾说话时低头看了一眼手机，屏幕亮了又灭，像一扇不肯完全打开的门。他没有问是谁的消息，只把这个动作记下来，放在心里一个很靠前的位置。

纸箱上的胶带被撕开又贴回去，边缘粘住一点灰尘。

他在路边买了一瓶水，拧开以后又忘了喝。林青禾伸手要看瓶盖上的中奖字样，他递过去，手指碰到她的手背，又很快缩回来。那一瞬间没有任何意义，至少对她来说大概没有。可是许亦安回到宿舍后，还是把它写进日记本里，写得比事情本身长得多。

日记本里有一页只写了林青禾的名字，写了四遍，又划掉三遍。最后剩下的一遍歪在格子边上，像一个不肯坐好的学生。他看了很久，终于用便宜中性笔在旁边写下：她不欠我解释。

他总以为自己在包容，其实很多时候只是害怕失去，所以不敢表达不满。

夜里宿舍熄灯后，曹原还在床上翻身，周一鸣戴着耳机听英语听力，走廊里有脚步声来来回回。许亦安把手机藏在被子里，屏幕光照着他的脸。他删掉“我今天很开心”，改成“今天风不大”。删完以后又觉得太冷，补了一句“但是她笑了”。

他把那几页拍了照，不是为了保存伤口，而是怕自己以后忘记曾经这样用力。拍完以后他没有重看，屏幕暗下去，纸页仍摊在桌上，普通得像一份旧作业。

楼下有人喊收废书，声音拖得很长，穿过窗户时已经轻了。

他看见自己丢失的不是林青禾，而是没有她就无法安排的生活。这个认知没有让他立刻轻松，只是让他第一次把目光从她身上移回桌面、箱子和明天的车次。

林青禾后来发来一条消息，内容很短，只是问他明天有没有课。许亦安盯着那几个字看了很久，仿佛里面藏着另外一层意思。他回得很慢，字斟句酌，最后却只发了“有，下午没课”。发出去的一瞬间，他又后悔自己为什么不更自然一点。

曹原把一卷胶带扔给他，说别愣着。许亦安接住，撕开胶带封纸箱。胶带声很刺耳，却比那些盘旋在心里的句子清楚得多。

第二天醒来时，宿舍窗帘只拉了一半，光从缝里斜斜进来。曹原趴在枕头上骂谁把闹钟设得这么早，周一鸣已经洗漱完，端着脸盆站在门口催他去吃早饭。许亦安把手机摸出来，先看有没有新消息，再假装自己只是看时间。

食堂早上的队伍总是很长，铁勺碰到不锈钢盆，声音一阵一阵。周一鸣买了两个馒头，塞给他一个，说人不能只靠情绪活着。许亦安笑了一下，把馒头掰开，里面的热气散出来。他忽然觉得这句话难听，却比很多安慰都实在。

上午的课讲得很慢，老师在黑板上写了半面公式，又用袖口擦掉一个写错的符号。许亦安看着那个被擦掉的位置，想到昨晚日记本里删掉的句子。粉笔灰落下来，细细的一层，他在本子边上写下一个“风”字，又用笔尖把它涂黑。

课间有人从后排传来一张纸条，问晚上要不要一起去网吧。曹原回了一个大大的“去”，周一鸣在旁边补了一句“别把他带坏”。许亦安没有参与，手机扣在桌面上，像一只安静的小盒子。他知道自己在等它亮，也知道这种等待没什么道理。

后来再想起《他的日记本》，他先想起的是旧日记本封面贴着的透明胶。物件把情绪按回地面，让他不再只剩一团解释不清的疼。

那晚他没有把读日记这件事讲给任何人。不是因为它神圣，而是因为它其实很笨，笨到只适合自己一个人慢慢承认。

回宿舍的路上，他经过公告栏，看见上面贴着旧海报。海报边角卷起来，露出后面更早的一张纸。许亦安盯着那层纸看了一会儿，想到人心也差不多，新话盖着旧话，新热闹盖着旧难过，可只要风一吹，最下面的东西还是会露出来。

他没有把这些想法告诉林青禾。那时候他还以为，真正深的感情应该自己消化，不能给对方添麻烦。后来他才知道，不能说的感情很容易在心里变形，变成委屈，变成审问，变成某个夜晚突然冒出来的重话。

那晚他又翻开日记本，把白天发生的事重新写了一遍。写到一半，他停下来听楼下卖烤红薯的人吆喝，忽然觉得生活并没有因为他的难过停住。

那一行字很短：我也在里面。日记本合上时，他听见纸页轻轻响了一下，像有人终于退后半步。

多年后他重写这一段时，特意删掉了几个太用力的词。年轻时他靠重话证明疼痛，后来才相信轻一点也能说清楚。

重读这一页时，他把自己从句子的中心往旁边挪了挪。故事不是为了替谁判案，而是把当时所有人的无能为力放在同一盏灯下。

他后来记住的不是那些重话，而是胶带粘住纸箱的声音。一个人开始整理物件时，往往也在整理自己，只是当时还不知道。

这一章真正留下来的，不是他当时以为最重要的那句话，而是行李箱拉链合上时，一张旧车票掉了出来。

这一页日记后来被他保存了很多年，保存到里面的怨气都慢慢变轻。许亦安合上手机，听见外面有人笑着从楼下经过。那笑声很快远了，宿舍重新安静下来。他躺在床上，觉得这一天好像没有结束，又好像早就结束了。
