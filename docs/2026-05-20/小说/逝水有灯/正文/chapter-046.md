# 第046章 元旦晚会之前

所属部卷：第一部：初秋三楼
时间：高二元旦
地点：学校礼堂后排

## 诗歌插页 P046：原创现代诗

礼堂还没开灯，
椅背先冷下来。
节目单从前排传到后排，
每个名字都很正式。
少年把纸折好，
像把一场没有开始的热闹
提前收进掌心。

元旦晚会前一周，礼堂开始排练。

下午最后一节课后，广播里通知各班文艺委员到礼堂集合。走廊里立刻热闹起来，有人借校服外套，有人搬音箱，有人抱着彩带从楼梯上跑过。高二理科班没有太多节目，最后报了一个合唱和一个诗朗诵。

夏知遥被选去朗诵，是因为普通话清楚。她不太愿意出头，但林曼说班里女生就她声音稳，她只好答应。拿到稿子后，她把重音一处处标出来，红笔写得很细。

陆允川知道这件事，是在程砚从前排打听回来以后。

“你不用问了。”程砚把节目单拍到他桌上，“她有节目。”

“我没问。”

“你脸上问了。”

陆允川把节目单拿起来，看见夏知遥的名字印在第三个节目后面。那一行很小，却像被单独加重。节目单油墨还没干透，指腹轻轻一抹，会沾上一点黑。

晚会那天傍晚，礼堂后排先坐满了不参加节目的学生。椅子是木头的，坐久了腰疼。程砚占了两个位置，给陆允川留了靠边一把。

“你坐这儿视野好。”程砚说，“也方便你假装不是来看人的。”

陆允川没理他，把节目单折成整齐的四折，放进口袋，又拿出来，再折一次。礼堂前面有人调话筒，音响发出刺耳的电流声。舞台幕布半开，灯还没亮。

他心里也像那盏没亮的灯，明明知道只是晚会，却还是提前紧张起来。
