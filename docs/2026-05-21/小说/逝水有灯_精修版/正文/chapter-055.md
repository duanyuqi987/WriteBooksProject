# 第055章 高三倒计时多年后

所属部卷：第一部：初秋三楼
时间：高三九月
地点：教室黑板右上角

## 诗歌插页 P055：原创七言小诗

灯过长廊日影深，
一行旧字压衣襟。
少年未解离分意，
先把沉默写到今。

多年后，陆允川在大学宿舍里接到程砚的电话。

"我今天收拾家里，翻到高三的东西。"程砚那边传来翻箱倒柜的声音——纸箱被拖过地板、抽屉被拉开、旧书本哗啦啦往下掉——"你猜我找到了什么？"

"什么？"

"一张你写给我的纸条。上面写着：'明天早上帮我带鸡蛋饼，加辣椒。'下面是我回的：'钱呢？'你又回：'明天给。'纸条是从练习本上撕下来的，边角撕得歪歪扭扭的，上面还粘着一小块透明胶。"

陆允川在电话这边笑了。"我给了吗？"

"给了。第二天就给了。一块五。"程砚也笑，笑完安静了一拍，"那时候真他妈穷啊。一块五都要记账。"

"你觉得那时候穷？"

"不是钱穷。是别的穷。"

两个人沉默了一会儿。窗外的声音各自不同——陆允川这边是大学宿舍走廊里的说笑声，程砚那边是家里电视的背景音。

"我前几天翻到一个旧本子。"陆允川说，"高三的日记。"

"写了什么？"

"大部分都划掉了。有一页写的是'想去一个有江的城市'。"

"你现在不就去了吗。长江边上。"程砚的声音里带着一种很淡的得意，像在说：你看吧，我当年没白问。

"是。但我当时写这句话的时候，心里想的不是江。"

"那你心里想的是什么？"

陆允川没回答。他在日记本里翻到那页的后面，还有一页，没有划掉。上面写的是：今天她跟我说作文写得好。就这一句，没有下文。

"陆允川？"

"在。我刚才……走神了。"

程砚在那边哼了一声。"你还是老样子。"

挂了电话，陆允川继续收拾行李箱。他刚从老家回来，箱子里塞了一些旧东西。他妈让他把自己房间清一清，说书架上灰太厚了。

他从箱子底下翻出一本物理错题本。封面已经卷了角，里面用红笔蓝笔黑笔画了好几层。有一页的页脚写着一行很小的铅笔字：这道题程砚说辅助线画错了。

他把错题本合上，放在新宿舍的书架上，跟大学课本排在一起。

晚上他又接到一个电话，这次是叶澄。

"程砚说你在翻旧东西。"

"你俩还联系？"

"大学群里看到他发的消息。"叶澄的声音还是那么平，但背景里有人在喊她的名字，她应了一声"等一下"，然后继续说，"你在看什么？"

"高三的错题本。还有日记。"

"日记里写了什么？"

"大部分是废话。"

"废话也是记录。"叶澄那边安静了一些，好像她换了个地方，"我前阵子也翻到我高三的英语笔记本。上面有一页写的是'陆允川作文跑题'。"

"你专门记我？"

"不是我专门记你。是我当时要收作业，每次你跑题我都得等作文课之后单独找你，很烦。"

陆允川笑了。"对不住。"

"你现在作文还跑题吗？"

"大学的作文？基本是实验报告，跑不了。"

"那挺好的。"叶澄顿了一下，"其实你那时候也不是真的跑题。你是想说的太多，一个都舍不得删。"

陆允川握着手机，没有说话。窗外的路灯透过窗帘缝隙照进来，在地板上划了一道细长的光。

"我先挂了，室友叫我去吃饭。"叶澄说。

"好。多吃点。"

"你也是。"

那天晚上陆允川做了一个梦。梦里他回到那间教室，黑板右上角的倒计时数字清晰得像昨天刚写上去的——用红色粉笔写的，粉笔灰还没擦干净，数字旁边有一道被板擦拖出来的淡红尾巴。他坐在自己的座位上，错题本摊开着，旁边的程砚在吃鸡蛋饼，油滴到了练习册上，用袖子擦了擦，油印子反而被抹得更大了。前面夏知遥的背影和往常一样，低着头，手里的笔没停过。

然后有人敲门。是班主任，探进头来说："陆允川，你爸在校门口等你，说给你带了晚饭。"

梦里他站起来往外走，走到门口又回头看了一眼。黑板上的倒计时变成了零。粉笔灰还没有落完。

醒来的时候，他发现自己睡在大学宿舍的上铺，头顶是白色的天花板。手机屏幕亮着，是程砚凌晨发来的一条消息：兄弟，我突然想起来，你当年欠我一块五，算上通胀现在应该还我三块。

陆允川笑了，在黑暗里打了两个字：还你。

然后他把手机扣在枕头旁边，翻了个身，继续睡。
