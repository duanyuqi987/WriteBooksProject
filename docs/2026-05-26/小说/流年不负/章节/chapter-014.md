# 第014章 2011

时间：2011年1月1日

地点：顾临风家中（石桥镇）

卷次：卷一：初见秋风

元旦。放假三天。

顾临风在家。早晨他起得晚——平时六点多钟就被学校那盏日光灯轰醒了，家里的窗户是朝东的，太阳比日光灯晚，没有早读。他妈在楼下喊了两声——"饭凉了——""听见了没——"然后不等了。他蜷在被窝里，没应。又翻了个身。楼下传来他妈扫院子的声音——竹扫帚在水泥地上的节奏——刷——刷——刷——一下接一下。过了好一会儿才慢慢坐起来。被子撩到一边——冷空气立刻裹住了他的肩膀。他把秋衣往裤子里掖。踩着拖鞋——下楼。拖鞋是老式的——鞋底快踏穿了——脚底板能感觉出地砖的缝。

快十点才下楼。饭桌上摆着两碟菜——昨天的剩菜。腌萝卜——萝卜片是斜刀切的，酱油腌的——酱色已经滲进萝卜肉里。炒土豆丝——炒的时间不长——丝是脆的——放了点干辣椒——在盘底有点橘红色的辣油渍。粥在锅里——铝锅。锅盖掀开——里面的粥已经不怎么热了。他盛了一碗——是白米粥，米已经煮碎了——筷子挑不出整粒。站着喝了——粘稠——筷子把菜夹到粥面上——然后粥顺碗边流下。

他妈在院子里晾被子。被子泡了一个月了——趁着元旦天好洗。棉被面是红白格子的——旧了但干净。被子搭在竹竿上——竹竿两头架在两根三角木桩的凹槽里。他妈把被子铺平——手掌在棉絮上拍了两下——灰从布料间扬起——在冬日早晨的日光里跳舞——那些灰——细小——一圈一圈飘——飘得很慢。他爸在堂屋里捣鼓一个旧收音机。那把收音机是他爸年轻时买的——红梅牌——外壳是塑料——上面有许多温度留下的痕迹和擦伤。天线断了——他用一根铁丝插在原位——铁丝比原来短一截——然后电工黑胶带缠了一圈。黑胶带黏糊糊的——手指捏下来的时候会拉丝。旋钮转起来有点涩——转到短波台——先是嘶嘶嘶的电声——白噪——好大的噪音——继续拧——忽然有段京剧跳出来:"我正在城楼观山景——耳听得城外乱纷纷——"嘶——又跑了——再拧——声音又出来:"我诸葛——"他爸停下手，在"诸葛"后面听了一下。然后又拧——新闻:"二零一一年一月一日——今天是新年的第一天——"

他爸把收音机关了。旋钮一摁——咔。堂屋里突然没有声——院子里的扫帚也停了——他妈大概扫完了。

中午他姐打来电话。座机——在堂屋的墙角。电话铃是老式的那种"铃铃铃"——铁铃撞钟的声音。他爸在屋内——没去接。他妈在院子——喊了一声"临风——接电话——"他跑过去。拿起话筒——手指在电话线上绕着。"喂。"

"爸妈身体还好吧。"他姐的声音——有点吵——背景里有人声——大概是在宿舍走廊打的。

"好。"

"你读高中了要用功。别像你姐这样——上了大专。"

"嗯。"

"钱够不够用。"

"够。"

"不够跟我说。我打工挣了点钱。寒假我去县里看你。给你带件棉袄。"

他把电话线绕在手指上。绕了三圈——松开——拉开——线在指节之间。再绕。"够。"

挂了电话之后。听筒放在座机铁托上——咔嗒——接通声断。他站在座机边——手还放在话筒上。他妈进屋里——端着一盆揉好的面——看了他一眼。"你姐？""嗯。""她说啥呢。""没说什么。"他妈没再问——端着面盆进了厨房。他往外面走去——天仍是灰淡淡的白——但有光。

上了二楼。房间冷。窗户是老式木制的——单层玻璃——夜里室外的寒气透过窗框的缝隙渗进来了——玻璃角的霜花已经化了。下午的阳光照进来——窗台的木边上有水滴——霜化后留下来的——正慢慢地浸进木头纤维里。他站在书桌前——窗外的石桥。河干了——桥洞下面是冻白的河石。桌前——桌下的那个抽屉。

他拉开抽屉。从里面翻出一个本子——不是在学校用的那种封皮印着"作业本"的牛皮纸练习本。是他自己买的——在县新华书店四个月前的八月底买的——买的时候想用来记笔记——后来没舍得写。封面是牛皮色——粗面的——没印任何字。本子有点薄——边角很齐。

他坐到桌前——翻开第一页。新本子。纸是米黄色的——横格不明显——得对着光才能看到。把钢笔从桌面笔筒里拔出来——拧开——检查墨水——蓝墨水还剩一小半管。吸了一下——把橡皮管捏了捏。然后在纸的第一行落笔。

新年愿望:希望她能看到我。

写完——搁笔。看着那行字。钢笔字——正楷。写"她"的时候这个字比其他字重——往左下方用力——笔画的转折有一道小小的墨渍。"看她"旁边——纸张洇了点——薄纸的关系——"看到她"的墨水进去之后往边上渗了比旁边多大概半毫米。他伸手在纸上碰了碰——干了——指腹压在"她"字上——又抬起来。

然后重新拿起笔。在"希望她能看到我"上面划了第一条横线——从"希"字起笔划到"我"——横线穿过所有字的中心。划了第二条——在下面——和第一条平行——这一条划得轻一些——中间有些没连上——有一条缝隙。然后划了第三条——第三条也从左到右——压在第二条边上——连成一片。这三条线把原来的句子压在后面——但"她"那个字笔画粗——墨重——透过线层——还是能辨别。用力有些大了——纸破了——绿豆大的一点破口——翘起了纸毛。

在第二行。他重新写了一行字——

希望自己变好一点。

写完。把笔套上——钢笔搁在桌边。把本子合上——手掌压在本子上——感觉纸皮凉凉地贴着掌心。合上之后——又翻开。又看了一眼第一页。划掉的第一行——"她"和"我"这两个字——隔着划痕——还是在淡蓝线上面显现——笔划没法完全盖住。他用手掌压在那页上——按了一会儿——掌心温温的——纸页也是温的。然后合上。

把本子放回抽屉。抽屉——拉的时候有点涩——用了点力——嘎——卡住——再拉——推进去了。木头和木头之间没有滑轨——是木头底板在木框上摩擦——用的过程就是越磨越光滑——现在还没磨够。

下楼。

院子里的被面上结了一层薄薄的白霜——她妈瞅见他:"把被子端屋里烤烤——"他把被子抽下来——抱在怀里——抱着的时候被子遮住了自己的脸——只有眼睛以上的额头露出来。被子里有洗衣粉的苦味——白猫牌的——还有棉布本身的味道。放在堂屋——搁在火炉边的木椅背上面——被子在炉边烘的时侯发出窸窣的声音——潮湿的棉布和热气抢——水分一点点蒸发。被面上有个补丁——白色的——在红白格子之间。堂屋里开始暖和起来——柴火气——柴是从屋后松坡上捡来的松柴——有些松脂黏在树皮上——烧的时候冒的烟是香的——和洗衣粉的味道混着。

他爸坐在炉子另一边的竹椅上。把他儿子的解放鞋放在炉口边上烤——那双旧球鞋——鞋底是牛筋底——昨天踏过河边泥路——沾了厚厚的一圈泥。泥干了之后变成了白灰色的土块——裂成龟壳的花纹。他爸用粗手指在上面弹了弹——土块啪地裂了——掉在地上——碎成粉——地上一摊灰。顾临风拖了一把小椅坐到炉边——竹椅——腿上绑过铁丝——因为竹椅的一只脚曾经散了——是他爸去年冬天用铁丝缠回去的。炉火映在脸上——暖是间断的——一阵一阵——随风门进气量改变。松柴噼里啪啦——炸火星——一粒火星从炉口弹出来——掉在水泥地上——灭了——变成一点黑煤渣。他把手伸到炉口前——掌心对着火——搓了搓——手指暖和了一些——冻僵的骨节松开了。

他爸递过来一个橘子——手伸过来——手指间夹着两个橘子——给自己一个——递一个过去。橘子皮有点干——橘子底部有个凸出来的小茸结——是刚从屋后储藏间拿出来的——秋天晒干存的。

顾临风接过来。剥橘子——手指第一次插进橘皮里——那层油包被撕开——橘子的生冲味立刻溢出来——飘到炉火周围——一个酸甜的刺激性气味。把橘子掰成两半——给了他爸一半。他爸摇了摇头。他把两半都吃了。橘子有点酸——后味——腮帮子感觉得到。

炉子里的湿柴一烧——烟气从炉口往上走——贴在铁壶的壶底——白色蒸气从壶嘴冒——先细细一束——风一吹。白汽绕着坐在对面的他爸。他爸拿起火钳——夹进去一块新柴——合上炉门——炉火在门眼里红红地闪。

窗外。石板路。没有一个人——冬天街上是空的——时间很慢——石拱桥在雾里——变成了两个三角形——桥本身和水里倒影。但桥身被雾气轻遮着——反而是水下那个三角形的边——格外清晰。远处不知哪个村里放了一串鞭炮——啪噼啪噼——响声在干燥空气里传得老远——清脆地。一分钟不到。又安静了。

天快黑了。晚云——灰白——低。没有雪。今天是二〇一一年的第一天。炉边的温暖让人有些昏昏欲睡。他靠着椅子的靠背歪了——打了个盹。梦里没有人。灯光透过眼皮变成橙红色。然后他妈妈在厨房说了一句——"吃饭了。"他睁开眼——炉火暗了些——他爸已经站起来——拿着锅铲在炒菜——铁锅和铲碰在一起——叮。他把椅子搬回桌边——揉揉眼——站起身——往厨房走了两步。

晚饭吃的是中午剩下的——添了新炒的一盘青菜。青菜叶子脆。饭后的碗是他洗的。然后一家人坐在堂屋里——炉门没关死。他妈补袜子。他爸拿起收音机又拧——拧到音乐台——正在播一首老歌——《光阴的故事》。"春天的花开秋天的风以及冬天的落阳——"他走到门边——抬头看天——天全黑了。近处没灯。但远处——有人家很早就开了门灯——灯是黄灯泡——黄光照着户前的台阶一个方方的黄块——暖。天上有几颗星——都不亮——云层还没完全散。
