# 第026章 她的朋友

时间：2011年5月下旬

地点：教室、走廊

卷次：卷二：窗边人影

苏晚晴朋友不多。

顾临风观察了将近一年。她在班上走得近的只有一个人——许若琳。许若琳是她的同桌。圆脸，短发，话多。两个人的性格看起来完全不一样。

一个安静。一个爱笑。

但她们在一起的时候苏晚晴会笑多一些。

五月的下午。

课间。阳光从走廊窗户照进来。瓷砖地上有光斑。

苏晚晴和许若琳站在走廊里晒太阳。苏晚晴靠在墙上，许若琳站在她对面。

许若琳在讲什么。手比划着。语速很快。苏晚晴听着。嘴角弯了一下。然后弯了第二次。然后笑出声了。

很短。

但是真的笑。

顾临风坐在自己的座位上。从座位透过窗户能看到走廊。窗玻璃上有一层灰——但还能看到轮廓。他低头翻书。又抬头看了一眼。

许若琳伸手帮苏晚晴把领口的什么东西摘掉了。大概是一根头发。大概是一个线头。

苏晚晴说了声"谢谢"——他读的唇语。

然后两人一起走进来。

苏晚晴的脸上还带着一丝笑意。很淡。像水面上最后一圈波纹。

顾临风把书合上。

他发现一件事——苏晚晴在许若琳面前是不一样的。不是大的不同。是些很小的不一样：

说话的时候少了一些停顿。笑的时候嘴角多弯了一点。听完话会看许若琳的眼睛。而这些事她跟别人在一起时都不做。

顾临风跟赵一鸣在一起时也会不一样。

被别人看见的另一面——那是信任。

中午食堂。顾临风远远看到她们两个坐在一起吃饭。

苏晚晴在吃土豆丝。许若琳在吃肉末茄子。

许若琳说了什么。苏晚晴摇头。许若琳又说。苏晚晴又摇头。然后许若琳直接把她碗里的肉末分了一半到苏晚晴碗里。

苏晚晴没推。低头吃了。

顾临风低头吃自己的饭。

赵一鸣坐在对面在讲篮球赛的事。讲得眉飞色舞。汤从勺子上滴下来都没注意。

赵一鸣也是这种朋友。

后来顾临风在日记本上写了一段。

"她有一个朋友叫许若琳。她们在一起的时候她会笑得多。笑得很真。有一个这样的朋友挺好的。"

写完之后他想了想。在"朋友"两个字旁边画了一个加号。加号旁边写了"赵一鸣"。

然后合上本子。

六月的某一天。走廊。顾临风路过的时候听到了她们的对话。

许若琳说"你数学卷子第十题错了没有"。

苏晚晴说"错了"。

"我教你怎么做"。

苏晚晴说"好"。

声音里有一点软——对她信任的人才有的软。不是撒娇。不是求助。就是很自然的软。

顾临风走过去。端着水杯。

走到饮水机前。接了半杯水。

回来的时候她们还在说话。许若琳在草稿纸上画着什么。苏晚晴低头看着。时不时点头。

他从她们身边走过。

隔了不到两米。

苏晚晴抬头看了他一眼。

一秒钟。

然后又低头看许若琳的草稿纸。

顾临风端着水杯走回座位。水是满的。一滴没洒。

他坐下之后。把杯子放好。翻开英语书。翻到单词表。第一个词是"friend"。

他把这个词看了好一会儿。

很多人。但真正走得近的只有那么一两个——就够了。

她是这样。他也是。

下课铃响。

苏晚晴和许若琳一起走出教室。两个人并排走在走廊里。许若琳比苏晚晴矮一点。走在一起像姐姐和妹妹——虽然不知道谁是谁的姐姐。

顾临风看着她们走出走廊转角。

然后他低下头。

开始做物理题。
