# 第075章 志愿

时间：2013年7月初

地点：各自家里

卷次：卷四：夏夜湖光

七月初，青阳县已经热得不行。

房间里没有空调。顾临风在桌上摆了一台小电扇。扇叶呼呼地转着，吹过来的风是热的。桌上的纸偶尔会飞起来——他用手机压住纸角。

电脑开着。志愿填报系统的页面灰蓝底色，表格很密。第一志愿那一栏里他打了几个字——"楚州师院"。然后选专业——"汉语言文学"。系统自动跳出了院校代码和专业代码——长长的数字串。他对着笔记核对了一遍。笔迹是他自己的。数字写了四个。两个院校代码。两个专业代码。

打开QQ。

苏晚晴在线。

"在填志愿。"

"填好了吗。"

"第一志愿楚州师院。第二志愿——也是。第二专业也是中文相关的。"

"你呢。"

"楚州师大。教育学。"

"第一志愿。"

"嗯。"

"后面几个呢。"

她发了一张截图——她填了一半的志愿表。第一志愿：楚州师大。第二志愿：南城师范。第三志愿：东江学院。

"万一楚州录不上——南城很远。"

"不会录不上。612分楚州师大稳稳的。"

"那第二第三就不重要了。"

"嗯。"

他鼠标在屏幕上晃了一下。已经写完了第一志愿和第二志愿。第三那一栏他空着。懒得填。他对着她的截图把第三志愿填了——南城师范。跟她一样。

不是想去。是万一两个人都考不上楚州的学校——如果她在南城，至少他也在南城。

这个他没有发出去。

她先发了——

"你第三志愿填的是什么。"

"南城师范。"

"你也是？"

"嗯。"

"因为你填了。所以我也填。"

这句是他难得的坦白。发出去之后他把头往椅背靠了一下。电扇的风吹过来——刘海被吹起来又落下去。看着这个句子在对话框里。没有撤回。

她过了好一会儿才回。

"那我们就填得一模一样算了。"

花了二十几分钟。两人各自在电脑屏幕前填完了全部学校的志愿。每一行都互相截图确认。他填的和她建议的一样，她填的和他默认的一样。两个人在不同角度把同一个表格填到完全一致——除了第一志愿的学校名不一样。

楚州师大。楚州师院。

两个学校。同一个城市。12公里。301路。

确认完毕。页面提交。

"提交了。"她说。

"我也提交了。"

"你的手在出汗吗。"

他低头看了眼键盘。键盘的右下角有三个指套的印子——潮的。

"在出。"

"我也是。"

"你不是有空调吗。"

"我妈开的空调离我书桌太远了。吹不到。"

"那你开电扇。"

"开了。对着脸吹。脸凉了。手还是汗。"

他站起来走到厨房。从冰箱里拿了一瓶冰水。拧开盖子喝了一口。喉咙被冰得疼——他皱了下脸。呼出来的气是热的。七月的青阳——四面环山——热气从山里蒸出来散不掉。

"我觉得。"她说。

"什么。"

"填完志愿了——比高考成绩出来的时候还要紧张。"

"我也是。"

"怎么了。"

"成绩只是数字。志愿是下一站。"

他打了一句"下一站我们已经约好了"——然后删掉。改成了——

"我把公交站背下来了。301路每一站。从楚州师大坐到楚州师院——胜利街口、和平路、华联广场、凤台路、跃进桥——"

"你背这个干什么。"

"万一哪一站你换车。我接你。"

对话框静了静。

"你想得太远了。"

"嗯。"

"但是想得挺对的。"

她把志愿填报页面截了个屏。截图发给了他。他下载了。存进收藏夹——放在"楚州师大到楚州师院"地图旁边。

"我们什么时候去楚州。"

"九月。"

"还有三个月。"

"嗯。"

"不长了。"

"嗯。"

他关掉了填报系统。桌面露出来了——默认的蓝天白云。他很久没看过桌面了——开着电脑就是为了查资料和填志愿。桌面的右下角有一个名叫"高中"的文件夹。点了之后里面有三个文件夹——"物理""数学""诗歌"。

"诗歌"里有六个文档。按日期排。最早那个叫"2010年12月7日"。就是那年冬天下雪——她站在窗边接雪花——他回去写了第一首诗——"那年九月风初凉"。

他点进去看了一遍。窗口的标题栏显示着那首诗。很短的四个句子。然后他点了关闭。又点开最近那首——还没写完——在六月中旬开的头。他有四句半。还差两句。

他在屏幕上打了几个字。又看了一遍。不太满意。全选——删除。

然后关掉窗口。

"你在干什么。"她的消息来了。

"看了一首诗。"

"哪一首。"

"以前写的。写过很多首。有的没写完。"

"看看。"

他犹豫了一下。然后截了一个屏发过去——不是发全部——只发了最近那首的开头和一个没写完的半句。

她回得很快——

"这句是不是差几个字。"

"嗯。想了半个月了。想不出来。"

她发来一张她写的字。她躺在床上用手机拍了一张在她桌面上写的几个字——""最后一笔太重了。""她说。

"我字不好看。"

他回："你写字虽然不好看，但每个字都是认真写完的。"

她把这句话存了下来。她不知道他是不是看的出来。她存了。

后来他就躺到床上去了。闭上眼睛。脑海里那个301路的站名还在往后退——胜利街口。和平路。华联广场。闭上眼睛能听到报站的声音——甜的电子音。

四年。他们要在这个城市里各自读四年的书。她读教育。她要去讲台上对着小学生说话。他读中文。他要对着课文里的古诗词对着学生分析平仄。

四年很长。

但301的公交车在脑子里开着。一辆绿白相间的公交车。从楚州师大的站牌往楚州师院方向开。一圈一圈地绕。绕到脑袋里只剩公交车轮碾过柏油马路的声音。

嗡嗡嗡。

醒了。

电扇还在转。窗外天全黑了。楼下有小孩在拍篮球。篮球嘭嘭嘭的声和房间里电扇的嗡嗡声叠在一起。

手机亮了。她五分钟前发的一条消息。

"晚安。"

他打了一个字："晚安。"

关了台灯。电扇对着床吹。风拂过脚背。凉快了一些。
