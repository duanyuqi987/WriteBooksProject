# 第036章 缺席

时间：2011年4月初，春游同日

地点：江城大学

卷次：第二卷：冬天的图书馆

春游那天苏安晴确实不在翠湖。她在大学生活动中心三楼的一间小会议室里。

支教社团的春季面试定在这一天——周六下午两点，地点在大活304。苏安晴提前二十分钟到的。她在面试名单上排在第四位。

面试官一共三个人。正中坐着一个大三的学姐，叫蒋慧，是支教团的团长。左边是一个戴眼镜的男生，叫孟书恒，大二，上一届支教的老队员。右边是个短头发的女生，叫赵今越，大二，负责宣传工作。

"苏安晴，教育学院，大一。"蒋慧看着手里的报名表，念出声来。"你为什么要来支教？"

苏安晴把手放在膝盖上，坐得很正。

"因为我自己就是从县里出来的。"她说，"我知道县城的孩子和城里的孩子——他们差别的不是智力，是资源。一个合格的老师，可能改变的不只是一个学生的成绩。"

"你高中在哪个县？"孟书恒问。

"封丘。"

"封丘一中？"

"二中。"

"二中教学质量怎么样？"

"一般。"苏安晴想了想，补充了一句，"我们那年考上一本的不到四十个人。我有几个同学成绩很好，但是英语听力拉分——县城中学的听力设备太旧了，放出来的英语和收音机坏了一样，所有人都听不清。最后他们就读了个大专。不是他们不努力，是他们听不见。"

会议室里安静了一下。赵今越抬起头看了她一眼。

"你说的是哪一年的高考？"蒋慧问。

"去年。2010年。"

"你自己怎么样？英语听力有没有受影响？"

苏安晴微微笑了一下。"有。考的分数比我模拟考低了十分。但还好够了一本线。"

赵今越在本子上记了点什么。孟书恒推了推眼镜，问："你有教学经验吗？"

"没有正式经验。但是高中的时候帮邻居的小孩补过语文——那个小孩的爸妈在外面打工，爷爷奶奶不识字，学校老师一个人管五十多个学生。我每周六去他家，教他读课文、写作文。"

"补了多久？"

"一整个学期。期末那个小孩的语文从五十九分考到了七十五分，他奶奶杀了一只鸡要送我，我没要。"

蒋慧笑了。"那你觉得支教最难的是什么？"

苏安晴没有马上回答。她看着窗外——大学生活动中心三楼的窗户正对着操场，操场上有人在踢球，一群穿球衣的男生在草地上跑来跑去，叫着什么听不清的话。

"最难的不是教。"她转过头来，"是我们走了以后，他们又回到了原来的状态。一个暑假的支教，你教了他们二十六个英文字母，教了他们一元二次方程的解法。开学以后，他们的老师还是那一个老师，五十个学生还是那五十个学生。我们能做的很少。但少总好过没有。"

面试结束的时候，蒋慧让她回去等通知。走出会议室，苏安晴在楼梯口站了一会儿。走廊的尽头是一扇半开的窗户，窗外是四月的校园——玉兰花已经谢了，取而代之的是樱花。粉白色的花瓣落了一地，被风吹得沿着墙根跑。

她其实还有一个理由没有说出来。

她想去支教，不是因为支教这件事本身有多么美好，而是因为她需要一个夏天——一个不在封丘也不在江城的夏天。寒假回封丘，她在福万家超市买菜的时候，看见了林玉起。

他站在水果区，手里拿着一袋橘子，低着头假装没有看见她。

她也没有走过去打招呼。不是因为不想——她其实记得他的名字，记得他在诗社里念的那首诗，记得他在火车上问她"你怎么总是去四楼"时那种紧张得快要碎掉的声音。但她没有走过去。因为经过这么久了，她还是不确定一件事：他是真的想跟她说话，还是只是在观察她。

他看她的那种眼神——带着距离的、小心翼翼的、像在博物馆里隔着玻璃看一件展品——让她有一种说不出来的不舒服。不是厌恶，是一种想要打破却不知道从何下手的无力感。

支教也许是一个破局的机会。离开图书馆四楼那个固定的场景，离开那些座位、那些翻书声和那些不敢说出口的目光。在一个不一样的夏天里，也许事情会有不一样的形状。

她不知道。但她决定试一下。

下午四点，她从大学生活动中心出来，往宿舍走。经过图书馆的时候，她习惯性地抬头看了一眼四楼。

窗户开着，灯亮着。那扇窗背后坐着谁？也许空无一人，也许正坐着那个在笔记本上写诗的男生。

她低下头，继续走。
