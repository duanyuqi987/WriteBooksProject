# 第013章 报名表

时间：2010年11月上旬，下午

地点：江城大学文学院教学楼后门台阶

卷次：第一卷：九月的新校门

文学社的活动结束后，林玉起坐在文学院教学楼后门的台阶上，手里捏着一张从青协活动室门口撕下来的表格。

A4纸，打印的，抬头是：江城大学青年志愿者协会入会申请表。

他已经捏了三天了。

第一天，表格放在书包夹层里。第二天，表格装在口袋里压出了折痕。第三天——也就是今天——他坐在台阶上，把表格摊在膝盖上，盯着"申请理由"那一栏，写了两个字，划掉，又写了两个字，又划掉。

"你在这儿干嘛呢。"

付小川从物理系那边过来，手里端着一杯奶茶，吸管咬得扁扁的。

"没干嘛。"

"没干嘛你拿着张表。"付小川弯腰看了一眼，念了出来："江城大学青年志愿者协会入会申——"他没念完，"你要加青协？"

"还没想好。"

"为什么还没想好？"

"就是还没想好。"

付小川在他旁边坐下来。两个人在十一月凉得发硬的台阶上并肩坐着。迎面是文学院后墙上一排快掉完叶子的爬山虎。

"我帮你分析一下，"付小川把奶茶放在台阶上，"你一个中文系的，突然要加青年志愿者协会，不是你想做志愿者了——你想追苏安晴。"

林玉起没吭声。

"青协的招新九月中旬就结束了。你现在填表加入——人家会不会觉得你动机不纯？"

"会的。"

"那你还要填？"

"还在想。"

付小川看了他一分钟。"林玉起，你知道你最大的问题是什么吗？"

"想太多？"

"不只是想太多。"付小川把吸管拔出来又插进去，"是你每次想完之后都选择不动。你写诗的时候那么干脆，四个字能说清的事绝不用五个字。怎么到生活里就反过来了？"

林玉起看着申请表上的那两行划痕。第一行写的是"我想参与志愿活动"，划掉了。第二行写的是"我想——"甚至没写完。

"我怕她看出来。"林玉起说。

"看出来什么？"

"看出来我不是为了做志愿活动才加青协的。"

"那你觉得她看不出来吗？"付小川说，"你在图书馆四楼坐了一个多月了，每天六点二十到，坐她斜后方，你以为她看不出来？"

林玉起的笔停住了。

"她如果不想让你看出来她看出来了，"付小川继续说，"那说明她——算了，我不替她说话。你把表填了，交上去，想加了就加，不想加了再退。再退再差，也比你每天坐图书馆斜后方，念她的皂角味强。你说是不是。"

一阵风刮过来，把地上的爬山虎落叶掀起来半寸，又放下了。

林玉起把笔盖拔下来，在"申请理由"那一栏写：

"想做一些有意义的事。"

他写完之后自己愣了一下。这六个字不是假的。他想做的确实是一件有意义的事。只是意义的方向跟青协的宗旨稍微偏了一点。

付小川伸头看了一眼。"写得挺好的。至少这次你没划掉。"

林玉起把表格折好，装进书包。周三下午青协有例会，他可以那时候交过去。赵书妍上次说"下次可以直接报名"，这句话他可以当真。

"最后一个问题。"付小川站起来，拍了拍裤子上的灰。"你为什么不直接约她吃饭？"

"太直了。"

"那不叫太直。那叫正常。"

"对我来说太直了。"林玉起把书包背好，"我不是那样的人。"

"你是什么样的人？"

"我是一个写诗的人。"

付小川看了他好一会儿。奶茶喝完了，吸管发出空吸的声音。

"行，"他说，"那你继续写。"

林玉起往文学院的侧门走。走了几步，付小川在后面喊了一声："别忘了周三得交——青协活动室在大学生活动中心二楼，别走错了！"

"你怎么知道得这么清楚？"

"因为宋乔也在青协，她跟我说的。"付小川说到"宋乔"两个字的时候，声音轻了半拍。

林玉起没有回头，但他记住了那个半拍。

当天晚上，他翻出来一张新的稿纸，没写诗。

他写了一个计划：周三交表。周五等回复。下周六，如果有活动——穿那双新买的白球鞋。

写完之后他把稿纸折起来，夹在《现代文学三十年》的第一百二十三页。那一页他虽然四十多天一个字都没看进去，但翻开来，折痕和笔压都在。

他把书放回书包里。书包侧袋里有一块黄色肥皂，超市买的，最便宜那种。他没有用过。他只是想知道那是什么味道。
