# 第026章 第一次说话之前

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：教室前门和讲台之间

## 诗歌插页 P026：原创现代诗

我把早晨放低，
让一缕声从课桌边慢慢经过。
你的名字没有被我喊出，
只在借橡皮那句"谢谢"的尾音旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写第一次说话抵达之前的坡度。陆允川在脑子里排练过很多次和夏知遥说的第一句话。不是那种郑重的告白——他没那么大胆——只是一些最普通的日常对话："你的笔掉了"，或者"作业交了吗"，或者只是传卷子时说一句"给"。每一句都是公共场所允许的安全距离内的句子，不越界，不解释，不暴露。

他在草稿纸上列过各种场景：早上她在走廊尽头背课文，他假装去那边接水，擦肩而过时说一句"早"——但这个场景被他自己驳回了，因为接水的水龙头离她的方向太远，走过去太假了。他又想了食堂场景：她排在他后面，他打完了饭转身时说一句"今天菜还行"——但他在食堂里从没有撞上过她在自己后面，每次都是她在前他在后，隔三重人。

"你最近在草稿纸上画什么呢？"程砚课间探过头来，陆允川手快地把草稿纸翻了个面——翻完才发现另一面上正是今天列的第五种场景。"没什么。""没什么？你那纸上写了五句话，每句话开头都是'如果'——不是问句，是陈述句。你在做语文修改病句还是什么？""我在想怎么说话。"

程砚把草稿纸从他手指底下抽出来，看了一遍，又放回去。看完之后没有取笑，只说了句："你不用想那么多。真的不用。'你的笔掉了'已经是满分回答了。"他说这话的时候没有笑——这是少数几次程砚没有在这种事上开玩笑的时候。

叶澄在这件事上的态度更直接。她有一次发练习册时路过陆允川旁边，停下来问他："你是不是有什么话想对谁说？""没有。""你说'没有'的时候眼睛眨了两下——你说真话的时候只眨一下。""你在分析我？""我在做班长工作——帮助有困难的同学克服表达障碍。"她说的那么一本正经，陆允川笑也不是不笑也不是。

那天放学前，他最后一次在草稿纸上列了一遍想说的话。纸上列了三个可能性，每个后面都标了概率——不是他自己真的会去做这些，而是他在练习把"和她说话"这件事从"决不可能"变成"有一定概率"。哪怕这个概率只有百分之五，也比零好了。
