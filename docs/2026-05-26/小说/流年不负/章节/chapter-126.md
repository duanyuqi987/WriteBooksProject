# 第126章 手抖

时间：2018年3月中旬

地点：楚州，顾临风公寓

卷次：卷七：流年未断

周五晚上。离相亲还有十四个小时。

他坐在桌前批最后几本作文。红笔在手里握着，指节发白。手在抖。

他放下笔。把手摊开在桌面上。手指还在微微地颤。

六年没见了。忽然要以相亲这种方式——坐在她对面，点一壶茶，聊工作、聊天气、聊楚州的生活。好像是第一次认识。但又不是。

他站起来去倒了杯水。手握着杯子——水在杯里微微晃动。

喝了一口。把杯子放下。

走到镜子前面。

镜子里的人穿着旧T恤。胡茬长了两天。黑眼圈从眼底蔓延到颧骨。头发有点长——他妈说的对，该理了。

他没见过这个人。

工作两年了。站在讲台上可以眼不眨地讲一节课。批作业可以批到晚上十点不动窝。跟同事吃饭的时候也能正常聊天开玩笑了。表情管理做了两年——在学生面前是温和的顾老师，在同事面前是随和的临风。

但镜子里的这个人——好像不是顾老师，也不是临风。

就像一个失眠了两天的学生。嘴唇有点干。眼神不知道往哪放。

他对着镜子站了很久。

然后去洗手间洗了把脸。水很凉。他用毛巾擦脸——毛巾边缘起毛了，挂在水龙头上方的横杆上。

又回到镜子前。

没什么变化。

他开始翻衣柜。

衣柜在床的侧面。推拉门，右边的门有点卡。他用力拉了两下才拉开——拉门的时候手在抖。门框撞在槽上。咣一声。

里面的衣服不多。几件T恤——有一件的领口完全垮了，往下套拉。两件衬衫——一件白的一件浅灰的。两条长裤——一黑一深蓝。一件外套。都是洗了很多次的。颜色发白。衣架是铁丝弯的，外面涂了一层绿色的塑料皮，有些地方已经掉了——露出里面生锈的铁丝。衣架挂衣服时总会吱一声。

他拿出白衬衫。对着镜子比了比。镜子只照到胸口，看不到全身。他把衬衫放在床上。退了两步。看了一眼——太白。太新。穿上像刚毕业的大学生。像面试。不像去见一个等了八年的人。

白衬衫放回去。拿出浅灰的。就是那件——他妈说的领口都磨毛了的那件。领口确实起了一层绒。袖口少了一颗扣子——那颗扣子现在在书桌抽屉里。上次脱下来的时候掉在地上，他捡起来放在抽屉里。忘了缝。他把衬衫抖了抖——领口的毛绒在灯光下看更明显。像一条洗了一百遍的旧毛巾。

衬衫放回去。手在衣柜里翻。T恤被翻得乱成一团。没有新衣服。两年前刚工作的时候买过一件新衬衫——去面试穿的那次。后来不知道放哪了。可能在老家的衣柜里。可能捐了。可能洗坏了。

又把那件浅灰的拿出来。对着光看。领口不是一点点磨毛——而是沿着缝线的位置起了一圈绒毛。像被铅笔刀刮过的纸边。

在床头摊开。用手抚了抚领口——压下去一部分绒毛。手一松开。又弹起来。拽了一根——没拽掉。线是棉的。牢固得很。

又抚了两下。然后停住了。手掌压在领口上。使劲按。

把衬衫挂回衣柜。衣柜推拉门又咣一声。声音在空荡荡的房间里回了一下——然后被对面的墙吸掉。

坐到床边。床垫是房东配的，弹簧有问题——人坐上去会嘎吱响。他挪了一下位置，又嘎吱响了一声。再挪一下——又响了。他不挪了。

不知道该穿什么。衣柜里只有四件能穿的衣服。他盯着那扇半开的衣柜门。门里面黑漆漆的——灯泡照不到衣柜深处。

不知道该说什么。你好。好久不见。苏晚晴。还是——你还记得我吗。

不知道该用什么表情推开那扇门。是笑。是先点一下头。是叫她的名字。还是——推开门，站在那里。等她先抬头。

见面以后第一句话该说什么。是你好。还是好久不见。还是直接叫她的名字——苏晚晴。

她听到他的名字以后停了一下。她说"好"。那她推开茶餐厅的门看到他的时候——会笑。会愣住。还是会点一下头然后坐下。

他站起来。又坐下来。

拿起手机。打开她的照片。看了看。放下。

拿起水杯。喝了一口。杯底剩的一小口水不太多了，水在舌头上有淡淡的铁锈味。

然后他靠在床头。闭上眼睛。

手还在抖。

六年了。他以为自己已经不会这样了。大学的四年他把自己填得很满——上课、看书、写东西。工作以后也忙——备课、上课、改作业、开会。他没有时间想她。

或者说——他让自己没有时间想她。

但现在——周六下午三点。一茶一座。她坐在窗边。他把手机解锁——又看了一遍照片。

马尾。浅蓝色衬衫。平底黑帆布鞋。

他站起来。在屋里走了一圈。经过镜子的时候没敢看。

然后从鞋柜底下翻出了鞋油。已经干透了。盒盖用力撬了两下才打开。里面的油膏裂了口子。他用旧牙刷柄抹了一点上去试试——抹不动。太干了。

他把鞋油丢进垃圾桶里。

转身去厨房。洗了晚间吃完泡面没来得及洗的碗。洗洁精挤多了——泡沫起来一大堆。他冲了很久才冲干净。

把碗放进碗架里。用抹布擦了擦手。

打开手机。搜索：相亲第一次见面穿什么。

搜索结果第一条：干净整洁最重要。

他把手机放下。重新站到衣柜前。

把浅灰衬衫拿出来。对着窗户看了看。窗外是星期六上午才看得见的那棵樱花树，花瓣落得差不多了。树下的草地上铺了一薄层白。阳光把那些花瓣照得半透明。

他把衬衫抖了抖。灰尘在阳光里飞了一会儿。叠好。对折。再对折。袖口那缺扣子的地方叠在里面——看不见。把衬衫放在床头。蓝色的枕套旁边是灰色的衬衫。他退后一步。看了看。然后又伸手——把衬衫往枕头那边挪了一点点。靠枕头更紧一点。

然后去洗澡。浴室的门把手有点松。他拧开了——门把手在手里转了半圈才吃上力。热水器是老式的——烧气的。打开水龙头要等十几秒才出热水。冷水淋在肩膀上一瞬。他缩了一下。然后热水来了。蒸汽慢慢弥漫开来。镜子上糊了一层雾。他从镜子里只能看到一个模糊的、肉色的轮廓。看不清脸。

明天就是第三天的约期了。

洗完澡出来。浴室里全是蒸汽。镜子糊了。他用毛巾角擦了擦——露出一小块。刚好照到自己的脸。下巴刮得还算干净。头发理过了。鬓角有点短。但看起来整洁。他把毛巾挂回去。走到床边。床头那件浅灰衬衫还是叠得整整齐齐的。窗外路灯的光照在上面——灰色在橘色的光里看起来像是暖灰。不是白天那种灰扑扑的灰。

他坐回床边。床垫嘎吱了一声。屋子里很安静。只有冰箱的嗡嗡声。窗外的风停了。明天天气会说——多云转晴。气温九到十七度。他看了一眼手机。晚上十一点。还有十六个小时。十六个小时后。他会坐在一茶一坐三号桌。对面坐着苏晚晴。

他把衬衫从床头拿起来。抖了抖。穿上了。第一颗扣子。第二颗。到第三颗的时候手指不抖了。大概身体已经接受了——明天就是今天了。十四个小时。然后变十三个。变十二个。变零。
