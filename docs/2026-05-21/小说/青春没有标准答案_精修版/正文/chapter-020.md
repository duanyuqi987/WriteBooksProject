# 第020章 县城车站的分别



时间：2008年八月初，北京奥运开幕前几天



地点：封丘县城车站和候车棚外



卷次：第一卷：九年级的志愿表



2008年八月初，北京奥运开幕前几天，封丘县城车站和候车棚外。一张去郑州亲戚家的短途车票把这一天的开头压得很实。车站候车棚下热气很重，卖矿泉水的小摊把冰块敲碎，玻璃杯边缘起了一圈白雾。段璟站在旁边，手上还有铅笔芯蹭出的黑印，耳朵却一直留意着常俊凡那边的动静。那时候他们还没有后来那么多说不清的关系，只是同一场中考里两个被分数推着往前走的人。

暑假最后一段时间忽然变短，开学在前面等着，分别却先到了一步。车站里有人喊班车，有人拎着编织袋挤过去，候车棚的小电视声音断断续续。分别没有电影里那么安静，它夹在车票、矿泉水和发动机的轰响里，反而更让人不知道怎么开口。

常俊凡要去郑州姨家住几天，买些高中要用的东西，也给自己一点离开县城的预演。她做事不急，先把笔袋摆正，再把要问的问题写在草稿纸右上角。段璟后来才明白，那不是冷淡，也不是故意把人挡在外面。她只是很早就知道，别人可以热闹，可以起哄，可以替她猜测，但真正要承担结果的人只有她自己。

段璟想把九月见说得更像约定，可车站太吵，他的话总显得不够郑重。他不是不会说话，只是一碰到常俊凡，平常那些贫嘴和硬话就像被粉笔擦抹了一遍，剩下一点不成句子的灰。他怕自己显得太在意，又怕自己真的一点也不在意。初三的男生常常这样，明明心里已经绕了好几圈，开口却只剩一个嗯。

他们的对话没有谁故意说得漂亮，都是初三学生能抓住的那些实话。“奥运开幕那天你们看不看？我爸说要把电视天线重新调一下。”蒋一诺说。“看吧。等回来就快开学了。”常俊凡说。“你什么时候回来？”段璟说。“报到前。不会错过的。”常俊凡说。话说完以后，谁也没有立刻把目光放到对方脸上，仿佛只要盯着桌面、墙面或手里的纸，心里的动静就不会被看出来。

蒋一诺也在场。抱着一本诗选等车，书页里夹着新生报到须知。这种小动作比豪言壮语更像那一年：大家都说自己不怕中考，可水杯里的水少得很快，草稿纸一张接一张被揉皱，课桌边缘全是指甲抠出来的小白印。

郭兰给女儿塞了一个布袋，里面有换洗衣服和一包金银花，说外面热，嗓子不舒服就泡水。家庭没有在他们的青春里大声喊口号，它只是藏在饭盒、车票、洗好的校服和一遍遍叮嘱里。封丘县的日子也是这样，街道不宽，消息传得快，谁家孩子能不能上一中，很快就会成为亲戚饭桌上绕不开的话。

段璟点头，手里攥着一张车站小卖部找回的零钱，想买瓶水送她，车已经开始倒车。这个动作没有被任何人郑重记录下来，甚至连旁边的人都未必注意到。但段璟记住了，因为它让他第一次意识到，所谓喜欢不是突然出现的一句表白，而是很多个没说出口的瞬间慢慢叠起来。

县城里的夏天靠近得很快。午后窗台烫手，作业本边缘卷起一点毛边，水房里有人一边接水一边背英语单词。广播里偶尔提到北京奥运，老师提到中考，家长提到一中，学生提到放假。所有大事小事挤在同一张课桌上，谁也没有多余的地方安放自己的慌张。

客车慢慢驶出车站，常俊凡隔着车窗挥了一下手。段璟站在原地，听见候车棚的小电视里传来奥运倒计时新闻，整个夏天忽然有了清楚的边界。段璟那时还不会把这种场面写成故事，他只会把它塞进心里，等很多年以后再慢慢认出：原来青春最真实的部分，不是后来回忆时替它加上的光，而是当时那些尴尬、具体、说错话又来不及补救的细节。

车站的小卖部卖汽水、矿泉水和塑料扇子。段璟摸了摸口袋里的零钱，先买不起郑重的礼物，连买一瓶水都要算找零够不够回家。这种窘迫没有人笑他，因为那时许多学生都这样。

蒋一诺抱着诗选，偶尔抬头看车牌。她说诗里写远方总写得轻，真正出门才知道行李很沉。常俊凡笑了一下，说她只是去几天，不算远方。段璟却觉得，能先离开县城几天，也已经很远。

候车棚的小电视正在播奥运倒计时新闻，画面颜色有点偏蓝。旁边几个大人讨论开幕式，说今年一定热闹。学生们听见奥运，想到的却是开学，因为开幕式过完，再过不了多久，他们就要穿着新校服进一中。

蒋一诺在旁边留下的小动作，也让这一章不只围着段璟和常俊凡转。封丘县的学生多数早早学会看大人的脸色：家里宽裕不宽裕，老师看好不看好，自己能不能把一次机会抓住。这些判断没有写在卷面上，却时时刻刻挤在卷面旁边。

常俊凡很少把情绪说满。她说可以，就是可以；说不改，就是不改。这种短句有时会让段璟不知道怎么接，可也正因为短，才不像临时编出来的温柔。她给人的距离感，不是拒绝所有人，而是提醒别人别替她安排人生。

段璟的笨拙则更容易被看见。他会在该说话时沉默，在该沉默时又憋出一句不合适的话。付小川他们偶尔笑他，他也会顶嘴，可常俊凡一看过来，他的顶嘴就像被人抽掉了半截，只剩下耳朵发热。

那时候大家还没有智能手机可以随手查消息，更多事情靠公告栏、电话、老师通知和同学传话。一个消息从办公室传到教室，往往会变出好几个版本。有人说分数线会涨，有人说一中要扩招，有人说住校床位紧，越传越像一场小小的风。

封丘县的夏天有自己的声音。三轮车从巷口过去，车斗里响着铁盆；早餐摊收摊时，塑料凳一只套一只；远一点的地方，麦场上有人拍打编织袋。这些声音落进少年人的心里，不会立刻变成故事，却会在多年后突然浮出来。

车站的水泥地被太阳烤得发白，候车棚下的人一动，热气就跟着晃。段璟站在小卖部门口，看见玻璃柜里摆着汽水和矿泉水，手指在口袋里摸了摸零钱，又悄悄松开。

常俊凡的布袋不大，里面却塞得很满：换洗衣服、一本词汇书、母亲塞进去的金银花，还有新生报到须知。郭兰替她把袋口系紧，说到了姨家别嫌麻烦，东西要放好。常俊凡嗯了一声，没有撒娇。

蒋一诺抱着诗选等车，书页被风掀起一角。她说诗里写远方总轻，真正出门才知道车票和行李都沉。常俊凡笑了笑，说她只是去郑州几天，不算远方。段璟却觉得，能先离开县城几天，也已经很远。

候车棚的小电视正在播奥运倒计时新闻，画面有些偏蓝。旁边几个大人讨论开幕式，说今年一定热闹。学生们听见奥运，想到的却是开学，因为开幕式过完，再过不了多久，他们就要穿着新校服进一中。

客车司机从车窗里探出头，喊了一遍目的地。人群往前挪，编织袋、凉席和纸箱互相磕碰。段璟忽然觉得自己的话太轻，轻得挤不过发动机声，只好把那句路上小心说得很快。

常俊凡上车前回头看了一眼。她不是那种会把分别演得很重的人，只把车票递给售票员，又把布袋往怀里抱了抱。段璟看见她这个动作，知道她连离开几天都要把自己安排得稳稳当当。

车站外有人卖塑料扇子，扇面上印着牡丹和山水。段璟站得久了，后背被汗浸湿，校服贴在身上。他想买一瓶水递上去，可车门已经关了一半，售票员催旁边的人别挡道。

客车倒车时扬起一层灰，常俊凡隔着车窗挥了一下手。那一下没有约定的样子，却比许多约定都清楚。段璟抬手回应，手抬到一半才发现自己掌心全是汗。

车走后，候车棚一下子空出一块。蒋一诺低头把诗选塞进书包，说九月就又见了。段璟点点头，却没有立刻走。他看着客车拐出路口，才慢慢把手里的零钱放回口袋。

回家路上，县城街边的电视店也在播奥运宣传片，屏幕一排排亮着。段璟骑车经过时，忽然觉得这个夏天被分成了两半：一半留在初中，一半正从车站、通知书和新校服那里慢慢开始。

这段日子里，封丘县像被中考轻轻拧紧了一圈。街上还是那些卖菜的三轮车、修车铺和文具店，可学生走过去时，总会下意识把书包带往肩上提一提，好像这样就能把未来背稳。

常俊凡不是不会累。她只是习惯先把事情排好，再允许自己短暂地松一口气。文件袋、笔袋、准考证、通知书，每一样都被她放在固定位置。段璟看见这些，才慢慢明白她的要强不是给别人看的。

段璟的心思则更乱。想到封丘一中时，他会想到校门；想到常俊凡时，他会想到她写字时压住纸角的手。两种念头挤在一起，让他有时分不清自己到底是在向学校靠近，还是向她靠近。

大人们的担心没有停止。生活费、住校、路费、孩子能不能适应，这些问题每天换一种说法出现。可他们也没有把孩子往后拉。很多时候，父母的支持只是沉默地把钱数好，把被褥晒干，把一句注意身体重复好几遍。

蒋一诺的存在让这段前史多了一点旁边的声音。有人贫嘴，有人算分，有人说自己无所谓，真正到了选择面前，又都把无所谓收了回去。群像不是热闹的装饰，它提醒段璟，每个人都在用自己的办法过关。

北京奥运的新闻越来越密，电视里常出现倒计时。县城大人谈起奥运时带着兴奋，学生们听见倒计时，却总会想到另一个倒计时：开学、报到、新班级，以及还不知道会不会同班的那些人。

如果只把这几章写成喜欢与被喜欢，就会太窄。真正让人信服的，是喜欢旁边还有考试、家境、选择和县城日常。青春不是悬在空中的情绪，它有尘土、有饭菜味，也有不得不填下去的表格。

段璟开始学着不把每一次沉默都理解成拒绝。常俊凡沉默时可能在算题，可能在想家里的话，也可能只是累了。一个人越是真实，就越不该只被另一人的心事解释。

常俊凡也不是冷冰冰的答案。她会提醒别人别吃太凉，会把借来的东西归整好，会在别人起哄时皱眉。这些小动作没有一句像表白，却比表白更能说明她是什么样的人。

到了夜里，县城慢慢安静下来，只有远处车声和院子里的虫鸣。段璟躺在床上，把当天的场面一遍遍翻出来，最后发现最难忘的不是某句漂亮话，而是常俊凡把自己的选择握稳的样子。

封丘一中这个名字在他们嘴里出现得越来越频繁。起初它只是宣传单上的校门，后来变成老师讲过的分数线、家长算过的生活费、同学拿来互相试探的一句你报不报。段璟每次听见这四个字，都会下意识去看常俊凡。

有人把离别说得很夸张，好像初中一结束，所有人马上就会被命运吹散。也有人故意说得很轻，说县城就这么大，早晚还能遇见。常俊凡听见这些话，通常不反驳，也不附和，因为她知道遇见容易，重新站到同一条路上并不容易。

段璟家离县城不算远，却也不是抬脚就到。早晨路上有卖菜的三轮车，有背书包赶车的学生，有刚从地里回来的大人。那些味道和声音混在一起，让封丘县不像地图上的名字，而像每天都要经过的生活。

常俊凡的学习方法有一种笨拙的可靠。她不喜欢把笔记涂得花里胡哨，只在最容易错的地方画小方框。如果一道题错了三遍，她会在旁边写明错因：看漏条件、公式混用、心急。

段璟偷偷学过这种写法，写到第三天就乱了。页边全是自己补给自己的借口。他看着那些借口，忽然有点佩服常俊凡，也有点不好意思。喜欢有时不是被一个人的光照到，而是被她的方法照出自己的毛糙。

同学们说话没轻没重，常把未来挂在嘴边。可真正落到自己身上时，谁都不轻松。有人怕考不上，有人怕上了也跟不上，有人怕住校，有人怕家里花钱。段璟第一次发现，青春不是只有热闹，也有很多不能轻易说出口的计算。

这种计算并不丢人。县城少年长大得早，早在他们懂得浪漫以前，就先懂了路费、饭费、资料费和家长眼里的期待。常俊凡清醒，段璟敏感，他们正是在这些现实里慢慢靠近。

所以九月还没到，九月已经在很多细节里提前出现了。新买的笔芯，晒过的被褥，压在字典里的宣传单，电视里反复出现的奥运倒计时，都在提醒他们：初中快结束了，下一段路不会等谁想明白以后才开始。

家长的爱在这一段里很少变成拥抱，更多时候变成钱夹里数了又数的零钱、塑料袋里装好的衣服、临走前塞过来的水。段璟曾经嫌这些叮嘱琐碎，后来才知道，县城家庭面对升学时能给出的郑重，往往就是这些琐碎。

街上的招牌把夏夜照得一块亮一块暗。有人讨论北京奥运开幕式，有人讨论一中分数线，有人说孩子考出去就好了。常俊凡听见这些话时没有接，她把布袋往肩上提了提，像先把自己的重量背稳。

段璟有时会在这种热闹里突然沉默。不是因为伤感，而是因为他发现自己没资格替常俊凡高兴，也没资格替她担心。他能做的只是站近一点，在需要递资料、按表角、排队或找零钱的时候，把手伸出去。

有些场面如果只写一句他们都很紧张，就会轻得像没发生过。真正让人相信它发生过的，是铅笔芯断掉以后谁递了一把小刀，是水房排队时谁往旁边挪了半步，是车铃响起时谁下意识护住了文件袋。段璟和常俊凡的靠近，也只能从这些动作里慢慢长出来。

饭盒、脸盆、准考证套和透明文件袋，都是那一年反复出现的东西。它们不抒情，却比抒情可靠。段璟记得常俊凡把文件袋抱在胸前走过楼梯口，也记得自己把饭盒塞进书包时，里面的勺子撞出一声闷响。生活先把他们推到这些物件面前，再让他们慢慢明白选择的重量。

大人们谈未来时，常常把声音压低，可压低并不等于学生听不见。段璟听见过父亲说要不就让孩子拼一回，也听见过母亲说住校可别冻着。常俊凡听见过母亲问学费，也听见父亲说姑娘想去就让她去。那些话没有直接出现在课堂上，却一直跟着他们进考场。

同学之间的关系也在这种时候变得微妙。平时一起打闹的人，到了报志愿时会突然认真；平时不怎么说话的人，会在排队时递过一支笔。蒋一诺有自己的紧张，也有自己的玩笑。群像不是背景，它让段璟知道，自己的心事只是青春里许多心事中的一个。

晚上电视里播新闻，奥运的画面和彩色广告交替出现。段璟一边看，一边给铅笔削尖；常俊凡可能也在另一个屋子里整理书包。他们并不知道彼此正在做什么，却都被同一个夏天推着往九月走。
