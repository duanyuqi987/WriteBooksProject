# 第219章 旧日记本旁观

所属部卷：第五部：逝水有灯
时间：工作第三年
地点：祁野所在城市

## 诗歌插页 P219：原创现代诗

我把牛皮封面放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

祁野帮陆允川搬完家以后，回到自己的城市，把行李箱里的脏衣服全倒进洗衣机，然后坐在阳台的折叠椅上抽了根烟。

他跟陆允川认识了八年。大一开学第一天，陆允川穿反了军训服的裤子，是他提醒的。从那以后，他就有了一种义务感——不是要监督陆允川，而是要在关键节点上出现一下。搬宿舍、散伙饭、搬家、翻旧日记。他不觉得这是负担。他觉得这是他在这部没有导演的人生剧里给自己安排的戏份。

翻旧日记那天，他是故意走开的。他看见陆允川把日记本从纸箱里抽出来的时候，脸上的表情经过了三个阶段：第一秒是惊讶——忘了这个本子的存在；第二秒是犹豫——要不要翻；第三秒是平静——翻就翻。

他走开的时候心里想：这个人的第三秒比第一秒短了很多。以前是反过来的。

洗衣机停了。他把衣服拿出来晾。衣架不够了，他从厨房拿了一个夹子，把最后一件衬衫夹在晾衣绳上。衬衫袖子垂下来，在他的视线里遮住了对面楼的半扇窗户。

他的手机震了一下。是程砚。

"听说你帮老陆搬家了？"

"对。周六搬的。"

"他还好吗？"

"什么叫还好？"

"就是——精神状态。翻旧日记有没有崩？"

"没崩。"祁野把烟灭了，弹到烟灰缸里，"他翻了几页，笑了一下。不是哈哈大笑那种，是那种——看了很久的账单终于对上了账的——笑。"

程砚那边停了一会儿。"所以他没删那本日记？"

"没删。放在书架上了。和毕业照放在一起。"

"放书架？不是压箱底？"

"不是。"祁野又点了一根烟，然后立刻又灭了，"我觉得这个细节比你写的任何评论都准。"

"什么细节？"

"他把日记本放在书架上。最上面一格。和毕业照并排。"

"那是他能看见的位置。"

"对。而且不是藏着的位置——是别人进来也能看见的位置。对他来说，放在别人能看见的地方，就等于承认这件事不再是秘密。"

"你观察得也太细了。"

"旁观者嘛。"祁野把湿衣服又拉了拉，让衬衫不要挡住窗。"对了，他还跟我说了一句话。"

"什么？"

"'旧东西拖太久就容易变成那种——你想翻又不敢翻的东西。'这是我告诉他的。他回了一句：'那现在不怕了。'"

电话两端都沉默了几秒。

"这句话比小说里的任何句子都像小说。"程砚说。

"那你跟他讲。让他写进去。"

晾完衣服以后，祁野打开电脑。他没有写小说的任务，但他有一份月度报告要交。写在文档里的时候，他发现自己也在做一件类似的事——把过去一个月的项目整理成条例和结论。和陆允川整理日记，本质上是同一个动作：把原始数据变成叙述。

只是陆允川整理的跨度是八年，他整理的跨度是一个月。

他在报告的最后加了一句非必要的备注：本月项目验收顺利，感谢配合。

然后删掉了"非必要"的判断。备注就是备注，没有必不必要。

阳台上衬衫的袖子还在风里晃。对面的窗户亮了一盏灯，有人回家了。

他把报告存了盘。把明天的闹钟调好。躺下来的时候，他在想一件事：多年后陆允川的小说里，不知道会不会有今天这个帮他搬家的人。

如果有，他不希望自己被写成一个符号。他希望自己就是那个说"旧东西别全扔"的人——原话，一字不改。
