# 第198章 回乡的暑假之后

所属部卷：第四部：失去的美丽
时间：大三暑假
地点：县城 / 陆允川家中

## 诗歌插页 P198：原创仿宋小令

桥痕边，夜色平，
半窗旧影入疏灯。
问当年、几回执念，几回无凭。
不把归人留梦里，
水声明日照常生。

从桥上回来以后，陆允川在家待了三天。

第一天睡到中午。他妈推门进来看了三次——第一次端了碗粥，第二次收走空碗，第三次站在门口问："你昨天去桥上了？"

"去了。跟程砚一起。"

"又去站着了？"

"没站。走了走。吃了碗凉粉。"

"就这些？"

"就这些。"

他妈在门口站了一会儿。然后说："碗拿出来。洗衣机开了。你的脏衣服也拿出来。"

"好。"

第二天下雨。县城夏天的雨来得很快——上午还是大太阳，中午突然天阴下来，然后豆大的雨点砸在瓦片上。陆允川坐在客厅的沙发上翻手机。高中群已经解散了——他退群之后又过了两年，群主终于受不了那些拼多多链接和投票，直接解散了。他看不到群消息，也不想看。但是张敏那天在超市说的话还在脑子里转——"她说'希望他退得轻松一点'"。

他在沙发上换了个姿势。他妈在厨房切菜——刀碰到砧板的声音均匀而规律。电视开着但没看——中央台在重播昨天的新闻。主持人说某地高温预警，某地暴雨预警。雨打在窗玻璃上，淌成一道一道的水痕。

"妈。"

"嗯。"

"你还记得我高中毕业那天吗。"

"记得。你回来以后不怎么说话。我冲了杯蜂蜜水给你。"

"你还记得蜂蜜水。"

"记得。因为那是你唯一一次喝完以后没评价。以前我冲什么东西你都会说——'太甜了''不够浓''今天的水温不对'。那天你什么都没说。喝完就回房间了。"

"那你当时觉得我怎么了。"

"觉得你失恋了。"

菜刀停了一下。然后又继续切。

"你怎么知道是失恋？"

"因为你从河边回来。裤腿湿了半截。鞋上全是泥。你高中的时候只有一个人能让你下雨天去河边站着。"

"你知道是谁？"

"不知道名字。只知道是个女生。姓什么——你从来不跟我说。"

"姓夏。夏天的夏。"

"夏知遥？"

"你怎么知道全名。"

"你高二有一次发高烧，说梦话。一直在喊'夏知遥''夏知遥'。我当时以为你烧糊涂了在背课文——后来想想，哪篇课文里有个女生叫夏知遥。"

陆允川从沙发上坐起来。"我烧到说梦话叫她的名字——你从来没告诉过我。"

"告诉你干嘛。你那会儿脸皮薄。说了你会不好意�。"

"那你后来问过吗。"

"没有。你自己的事自己处理。你要是想说早就说了。不说就是不想说。"

厨房里飘出炒菜的味道——蒜蓉下油锅。滋啦一声。

"那如果我现在想说呢。"

"现在说可以，但要等到晚饭。因为鱼已经下锅了。翻面的时候不能跟你聊天——会糊。"

陆允川走到厨房门口。他妈站在灶台前，一手拿锅铲，一手扶着锅柄。鱼在热油里两面煎得金黄——蒜蓉在油里变成了焦黄色。

"她叫什么来着——夏知遥。"他妈没有回头。"怎么——你还喜欢她？"

"不了。就是——高三毕业那天约了她。她没来。"

"我知道。"

"你怎么又知道。"

"因为那天晚上你撕了一堆纸。第二天垃圾桶里有碎纸屑。我换垃圾袋的时候看了一眼——最大的一片上面有'夏知遥'三个字。剩下的看不清了。但看得出是信的格式——顶格写的名字。"

锅铲在锅里翻了几下。他妈的语气很平。

"后来呢。"

"后来就没了。再也没联系过。"

"那今天怎么想起来说了。"

"因为前天去桥上走了一遍。跟程砚。买了两瓶汽水。吃了一碗凉粉。走完了高三时候设想过的路线——当时想的是跟她一起走。走了四年没走成。前天走完了。"

"走完就放下了？"

"嗯。而且发现一件事——如果真的跟她一起走，我大概会很紧张。要一直找话题。怕冷场。走完以后脚底板疼。现在跟程砚走——不用找话题。脚底板还是疼。但是不用装了。"

他妈把鱼出锅，盛进白瓷盘里。锅底的油还在噼里啪啦地响。

"你高中时候太懂事了。懂事到连喜欢一个人都不敢说。憋了三年。然后在毕业那天突然爆发——约人家去河边。换成我我也不去。"

"你为什么不去。"

"太突然了。你平时跟她说过几句话？"

"不超过二十句。"

"那你觉得一个女生——十七岁——她会在什么情况下答应一个'几乎没说过话'的男生去河边单独见面？"

"不会。"

"那不就得了。你没给她任何铺垫。直接跳到最后一步。她当然不答应。这不是她的问题。是你把步骤搞错了。"

陆允川靠在厨房门框上。他看着锅里的热气往上冒。

"那你觉得步骤应该是什么样的。"

"先跟她聊天。聊一阵子。让她习惯你的存在。然后约去人多的地方——比如奶茶店。让她不觉得有压力。再然后——如果她愿意跟你单独见面——说明她不反感你。那时候你再说你想说的话。但毕业那天你直接省略了前面所有步骤。你等于是在跟一个陌生人表白。"

"我以为我们不是陌生人。"

"对。你们是同学。同学和可以一起去河边的人之间有很长的距离。你把那段距离跳过去了。"

他妈把炒好的青菜也盛出来。两个菜摆在桌上。她拉开椅子坐下来。"不过你现在想通了就好。人一辈子要喜欢很多人。不是每个人都要有结果的。"

"那你有过没有结果的人吗。"

"有。但我不告诉你。"

"为什么。"

"因为我是你妈。保持一点神秘感。"

陆允川笑了一声。他坐到饭桌前。筷子刚拿起来——他妈又说了一句。

"不过我倒是想问你——你这次回来，跟苏晚枝联系过吗？"

"怎么突然提苏晚枝。"

"因为你大二回来的时候嘴里全是'苏晚枝'。'苏晚枝很忙''苏晚枝不理人''苏晚枝的实验室没空调'。那段时间我想说——你从高中到现在，喜欢的女生怎么都姓苏——后来才听清是'苏'不是'夏'。"

"不同的人。妈。"

"我知道。我虽然做饭，但我耳朵没坏。"他妈夹了一块鱼肉放进他碗里。"那这个苏同学——现在怎么样了。"

"还在诗社。当社长。我在大三上学期退社了。退的时候跟她说清楚了。"

"她说清楚了你也说清楚了。"

"对。"

"那现在后悔吗。"

"不后悔。"

"为什么不后悔——大二那会儿你不是挺喜欢她的吗。"

"喜欢过。但她喜欢的是诗社。不是诗社里的陆允川。我留在诗社是为了靠近她。她留在诗社是因为诗社本身——没有我她也照样办活动、招新、搞朗诵会。对我来说诗社是她的延伸。对她来说诗社是她的主场。我在她的主场里跑了一圈。发现跑道不是给我的。我就退了。"

他妈把鱼刺挑出来放到盘子边上。"你这话说得比大二那会儿清楚了。大二你说话绕圈。说了一百个字绕不出一个结论。现在三句就讲完了。"

"可能因为退社之后多出了很多时间。用来想清楚了一些事。"

"想清楚以后呢。"

"想清楚以后——就不需要诗社了。也不需要高中群了。也不是不需要。是——那些地方本来就不是我该待的地方。我在那里待着是因为想去见某个人。后来发现见不到了。或者见到了也没用了。那就退出来。退出来没那么难受。反而——松快。"

"那你接下来想干什么。"

"考研。下学期开始准备。跨专业考——中文系。以后想写东西。"

"写什么。"

"不一定。诗也好。小说也好。不一定出版。写给自己看也行。"

"写东西——"他妈把碗放下。"需要素材吧。你这些人——夏知遥、苏晚枝——是不是都会写进去。"

"可能。但不是控诉。就是——记下来。把她们写下来的时候，她们就不再是我脑子里的幻影了。她们变成能被翻过去的字。"

"那你记得写一句——'我妈会冲蜂蜜水'。"

"记住了。"

晚上洗完碗，陆允川坐在客厅里。外边的雨已经停了——空气里有泥土味和树叶味。他打开手机，看了一下程砚发来的消息。

"在姑妈家，吃得好饱。你呢。"

"跟我妈坦白了一部分。"

"坦白了什么。"

"高中暗恋史。省略版。"

"你妈什么反应。"

"'你太懂事了。把人吓跑了。'"

"阿姨厉害。"

"比你妈呢。"

"我妈只会说——'你别学陆允川，追不到女生就退诗社'。"

"你跟你妈说诗社的事了？"

"说了。她觉得退社很可惜。但我说——'他自己退的，又不是被赶走的。你在担心什么'。"

陆允川没回这条。他看着窗外。雨后的天空露出一小片深蓝色——边缘还有没散干净的黑云。

祁野也发了条消息过来。"毛概考完了。我猜能过。你桥上站了吗。"

"站了。还走了。吃了碗凉粉。"

"好。还有最后一个问题。"

"什么。"

"蚊子多吗。"

"不多。因为没站太久。走到桥尾就下去了。"

"那你真正想跟我说的是——你没站太久。"

"对。没站太久。"

"恭喜。"

陆允川放下手机。客厅的钟敲了九下。窗外的虫鸣一阵一阵——雨后蜗牛爬过院子的水迹，被路灯照成了银色的虚线。
