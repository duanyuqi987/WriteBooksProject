# 第071章 网吧通宵之前

所属部卷：第二部：六月以后
时间：高考后的夜晚
地点：县城老网吧

## 诗歌插页 P071：原创现代诗

网吧门口的灯管发白，
楼梯窄，
墙上贴着旧海报。
他们说终于自由了，
可自由先递来的，
是一张要填身份证号的登记表。

晚上七点半，陆允川到县城老网吧门口时，程砚已经等着了。

网吧在二楼，楼梯很窄，墙面被鞋印蹭得发黑。门口挂着一块旧招牌，灯管忽明忽暗。几个刚考完的男生围在下面，有人买了汽水，有人嚷着今晚必须通宵。

程砚看见陆允川，先把一瓶冰红茶塞给他：“你还真来了。”

“不是你一直催？”

“我以为你会在家和试卷深情告别到天亮。”

陆允川拧开瓶盖，没有反驳。冰红茶很甜，甜得有点发腻。他跟着大家上楼，网吧里机器声、键盘声和人声混在一起，空气闷热。老板坐在柜台后面，让他们拿身份证登记。

有人开玩笑说从今天起再也不用早睡，老板抬头看了一眼：“别吵，后半夜困了自己趴桌上。”

他们找了一排连坐的位置。程砚熟练地开机，登录游戏。陆允川坐下后，先看见键盘缝里积着灰，鼠标垫边缘卷起。自由没有想象中干净，反而有点黏手。

电脑启动很慢，屏幕一闪一闪。陆允川看着登录框，忽然想起手机里那个一直没有点开的聊天窗口。

他说不上自己想发什么。毕业快乐？考完了？以后常联系？

每一句都像太轻，也像太重。
