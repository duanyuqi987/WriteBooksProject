# 第051章 高三倒计时之前

所属部卷：第一部：初秋三楼
时间：高三九月
地点：教室黑板右上角

## 诗歌插页 P051：原创现代诗

我把星放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
粉笔写出的倒计时数字没有替谁回答，
只把那一年轻轻照亮。

高三倒计时真正到来之前，日子已经先变了颜色。高三九月的早读比以前更硬，窗还没完全亮，黑板右上角已经多了一行倒计时。白粉笔写出的数字像一枚钉子，钉在所有人的余光里。

陆允川把错题本摊开，先抄日期，再把昨天没订正完的函数题补上。笔尖划过纸面时，他总忍不住抬头看那三个数字，像确认自己还有多少天可以沉默。

他还没有遇到那个必须面对的瞬间，却已经被粉笔写出的倒计时数字提醒：有些事不能无限期地放在心里，它会跟着时间一起变形。

程砚把早餐袋塞进抽屉，压低声音说：“你要真想高考后再说，就先把今天的题做完，别让她替你的分心背锅。”

叶澄收作业时看了他一眼，说：“倒计时不是用来吓人的，是用来提醒你别把所有事都拖到以后。”

夏知遥把数学卷压在语文书下面，早读时背得很轻。她没有往后看，只在老师经过时把错题本往桌角挪了挪，动作利落，像早已知道该把日子切成一小块一小块。

早读声和粉笔擦过黑板的声音混在一起，把他的心事压得没有那么夸张。人在这样的声音里，很难一直把自己当成故事中心。

班主任把倒计时擦掉又重写了一遍，说数字变小不代表路变窄。陆允川听见这句话，手下那道题却还是算错了一个符号。

课间有人围着黑板讨论目标大学，也有人趴在桌上补觉。陆允川夹在这些状态里，第一次觉得未来不是远处的词，而是每天早晨都会被粉笔写出来的东西。

那时陆允川还不知道，所谓成长不是突然明白，而是先在粉笔头、错题本、早读铃这些小东西旁边学会停顿。

他在日记本里写了两行，又划掉一行。留下来的不是誓言，只是一个具体下午：早餐袋里半凉的鸡蛋饼、窗外的光，以及他没能说出口的话。

陆允川后来愿意留下这一笔，是因为它不漂亮。粉笔头不漂亮，错题本也不漂亮，可它们让人相信，这段青春不是从空中落下来的。

他没有再给夏知遥安排额外台词。她已经在教室里完成了自己的动作，剩下的沉默，应当由陆允川自己消化。

程砚把话说得粗，叶澄把话说得稳，两种声音都把他往现实里拉。陆允川当时不领情，很多年后却知道自己被救过几次。

那天的早餐袋里半凉的鸡蛋饼没有特别好吃，却因为太具体，抵住了那些过大的形容词。人有时候正是靠这些具体的味道，才没有被回忆吞没。

教室后排有人背单词，前排有人小声问题。这些声音把高三倒计时从一个人的心事里拉出来，变成许多人共同挤过的一天。陆允川不再只听见自己胸口的动静，也听见椅子、脚步、塑料袋和远处的喊声。

夏知遥把草稿纸折成四折夹进书里。这个动作很小，却让她从陆允川的想象里退回到真实的人：她会忙，会累，会被家里和未来推着走，也会在必要的时候保持沉默。

程砚用笔帽敲了敲他的错题本，说话还是老样子，不好听，却有效。陆允川皱了皱眉，还是照着他说的做了一点，因为朋友有时不是负责安慰，而是负责把他从自我感动里拽出来。

叶澄把一摞作业抱到讲台边。她没有把话说得太满，只把边界摆到桌面上。陆允川那时不一定立刻懂，却会在后来某个晚上想起这句话，知道自己当年被提醒过。

父亲晚饭后问他最近睡得好不好。家里的声音一进来，青春就不再只是校园里的风。它变成饭点、车费、睡眠、行李和一句不太会表达的关心。

他低头看着倒计时数字，忽然意识到它不是命运给他的标记，只是那天确实存在的东西。正因为确实存在，它比任何夸张的形容都更能留下来。

粉笔灰和早餐袋里的油香混在一起，让记忆有了可以落脚的地方。陆允川后来写到这里，没有再急着解释自己多难过，而是先把气味、光线和手上的动作写清楚。

他把错题本收好，又拿出来检查了一遍。这个来回不像英雄，也不像受害者，只像一个年轻人不知道怎样安放自己，于是先安放手边的东西。

真正的变化还没到来，教室黑板右上角已经先显出紧绷。陆允川站在里面，既想让日子快一点，又害怕快起来之后自己没有准备好。

他在日记本上写下“等以后”，写完又觉得“以后”太轻。很多拖延都披着克制的外衣，其实只是害怕被现实拒绝。

这一笔补进去以后，章节没有变得更响，反而更稳。高三倒计时不再只是一个标题，而是有倒计时数字、有错题本、有人说错话也有人及时停住的真实一天。

晚自习后值日生擦黑板，倒计时旁边留下几道浅浅的白痕。陆允川站在门口看了一会儿，才发现自己不是舍不得数字，而是舍不得那个还能把话往后推的借口。

夏知遥问老师题时把声音压得很低，问完后没有回头，直接把答案补进错题本。陆允川看见她那种专注，心里忽然有点羞愧，因为他曾把自己的犹豫包装成深情。

程砚晚自习后把早餐袋从抽屉里掏出来，发现鸡蛋饼已经冷硬，皱着眉说你看，拖到以后就是这个下场。陆允川想说这比喻太难听，最后还是笑了。

教室里的光线一点点变暗，粉笔头和早餐袋都还在原处。陆允川把错题本收好，动作慢得像怕惊动什么。可真正需要被惊动的不是别人，是他自己。

他后来重写这一章时，特意把倒计时数字留在中间。不是为了让它变成象征，而是因为人只有摸到具体东西，才知道自己曾经怎样站在具体生活里。

事情尚未发生，陆允川已经开始给自己找退路。他把“以后再说”想得很体面，可教室里的每一种声音都在提醒他，退路拖久了也会变成墙。

章节落下时，倒计时牌换成三位数，陆允川决定把喜欢放到高考之后，却发现越压越清晰。还没有真正展开。陆允川只是把东西收好，跟着人群往前走，心里那点执拗被现实轻轻推了一下。
