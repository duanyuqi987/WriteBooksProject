# 第186章 女儿

时间：2021年5月
地点：楚州妇幼保健院
卷次：卷十：不负少年

走廊里的椅子是铁的，扶手冰凉。

顾临风坐在第四张椅子上。旁边是一个饮水机。每隔几分钟就有人来接水，纸杯撕下来的声音，水桶里冒泡的声音。

他从上午十点坐到了下午一点。

中间站起来了两次。一次去厕所——洗手的时候看了眼镜子。一次去窗户边——外面在下雨，五月的雨不大不小。他看了大概五分钟的雨。然后回去坐着。

旁边来了一个中年男人。手里也攥着一顶帽子。帽檐被手指掐出了一个凹。两个人互相看了一眼。没有说话。

走廊尽头的手术室。门关着。上面的灯是红色的。

赵一鸣打来电话。他接了。

"怎么样。"

"还在等。"

"你别紧张。"

"我不紧张。"

"你声音不像不紧张。"

"信号不好。"

他把电话挂了。攥着手机。屏幕亮了。又灭了。又亮了——他妈发来一条消息:"生了没有"。他打了三个字:"还在等"。发送。

然后把手机放进口袋。

又拿了出来。看了一眼。放回去。

一点二十分。手术室的门开了。一个护士出来——口罩拉到了下巴。手里拿着一个夹子。

"苏晚晴家属。"

他站起来。膝盖有一点软。像蹲久了猛地站起来那种——大腿前面没力。

"在。"

"母女平安。顺产。六斤四两。"

"她——"

"在里面。等一会儿转移病房。你先去办手续。"

"好。"

他站在原地。手里握着手机。手机硌着手心。他低头看了一眼——手在抖。微信的输入框里打了一串乱码。他把乱码删了。打了四个字:"母女平安"。发给了所有联系人。

然后去办手续。在窗口排队。前面还排着三个人。他站在原地等了七分钟。他把队排完了。交钱。签字。签字的时候笔从手里滑了一下——手心里全是汗。他把笔捡起来。签完了。工作人员看了看他的名字:"顾临风——是'临风'不是'林峰'？""是。"

手续办完。回到病房。

门虚掩着。

他推开。

房间里有两个人。

苏晚晴靠在病床上。枕头叠了两个，垫在腰后面。脸上全是汗。头发贴在额头上和脸颊上。有些头发从马尾里散出来了——贴在脖子上。脸色不是白的。是薄——跟孕吐时一样，皮肤底下透出青青的血管。

但她眼睛是亮的。

她怀里抱着一个白色的小包裹。包裹里露出一个小脑袋——头发不多，几根，黑的，贴着头皮。脸皱皱的。眼睛没睁开。嘴巴动了一下——很小很小的幅度，像在梦里吃东西。

她抬头看了他一眼。

满脸是汗。

然后弯了弯嘴角。

笑的样子和高中时一模一样。先弯眼角。嘴角慢了半拍。

他想说点什么的。

但他说不出来。嗓子像被什么东西堵住了。

他走到床边。没有坐下。站着。低头看着那个小包裹。小脸上有一层淡白色——胎脂。小拳头攥着。五根手指，每根都只有他大拇指指甲盖那么大。

指甲是透明的。

他伸出手。用食指摸了一下她的拳头。

小拳头张开了一下。然后又攥紧了。把他的食指尖攥在了手心里。

他愣了很久。

抬头看苏晚晴。

嘴唇动了动。

"辛苦了。"

她说:"不辛苦。"

她说的时候，声音有一点哑。大概是刚才太用力了。她清了清嗓子。然后把怀里的小包裹往他这边斜了斜。

"你抱抱。"

他伸出手。护士在旁边教他——"这个手托着头，这个手托着腰"。

他托着。

小包裹比他想象的重。但不太重——六斤四两。

他弯腰。把脸凑过去。凑得很近。闻到了一股味道。不是香。是一种淡淡的、带一点奶味和血的混合味道。

他轻轻吸了一下。

然后把包裹还给了她。不是不想抱。是他的手抖得太厉害了。

"叫什么名字。"

"And you already have one picked out."

她说的英文——大概是产房里跟医生说的，还没切换回来。

他愣了一拍。

然后她说:"名字不是你想好了吗。"

"我想了几个。等你选。"

"先叫她——"她低头看着那个小脸。"先叫她女儿吧。"

"好。"

他坐在床边那张硬椅子上。椅子离床大概半米。

窗外还在下雨。雨点敲在空调外机上——嗒嗒嗒。窗台上的一个小盆栽里积了一点水。

病房里很安静。隔壁床的产妇在睡觉。轻微的鼾声。

她在给女儿喂奶。第一次。姿势不太熟练。护士在旁边指导。她低着头。把女儿的小嘴凑过去。女儿找了一会儿——嘴里发出吧嗒吧嗒的声音。然后找到了。开始吃。

很慢。很轻。

他看着她们。

窗外的雨声小了一些。大概是云飘走了。

他把手放进上衣内袋里。摸到了一个小本子——巴掌大小。封面是牛皮纸。他本来想今天写的。但发现写不出来。

有些东西不该写成诗。

有些东西，就是活着的样子。
