# 第121章 教室的灯之前

所属部卷：第三部：寒风里的花
时间：同一天夜里
地点：二号教学楼空教室

## 诗歌插页 P121：原创现代诗

灯亮着，
不代表有人为谁留下。
拖把从走廊经过，
水痕慢慢变浅，
他却把一间空教室，
听成了自己的回声。

楼下等待那天之后，陆允川和苏晚枝在诗社群里只谈校刊。她发文件，他回“收到”；他改错字，她回“辛苦”。所有字都很正常，正常得让人无处安放委屈。

周四晚上，叶澄说终审稿还差一页签名，苏晚枝在二号教学楼找老师。陆允川刚从自习室出来，看见群消息，背着书包往二号楼走。

他原本可以把文件明天再交，也可以让叶澄转交。可他还是去了。去的时候，他对自己说只是顺路。学校很大，“顺路”这个词却被他说得很窄。

二号楼夜里人少，楼梯间灯坏了一盏。三楼走廊有清洁车经过，车轮压过水痕，留下一条亮线。陆允川走到 307 教室门口，里面灯开着，讲台上有半截粉笔，黑板右下角写着“门窗已关”。

苏晚枝不在。

他站在门口，给叶澄发消息：“她还在吗？”

叶澄很快回：“老师临时换办公室，她和两个同学过去了。你不用找。”

陆允川盯着“不用找”三个字。它和“不用等”长得很像，他却仍然在门口站了几分钟。
