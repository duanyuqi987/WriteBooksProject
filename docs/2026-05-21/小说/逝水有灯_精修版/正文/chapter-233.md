# 第233章 诗歌索引之后

所属部卷：第五部：逝水有灯
时间：诗歌索引完成后的日子
地点：陆允川书房→公司→老家

## 诗歌插页 P233：原创仿宋小令

校徽边，晓风清，
半窗纸页动微灯。
问旧年、几番执念，几番无凭。
人各有程休再问，
水声仍向远天生。

余波没有立刻散开。

诗歌索引完成的第二天是个周三。陆允川照常起床上班。公司里没有人在意他的书完成了附录——他们甚至不知道他在写书。他也没说。

午休的时候他打开手机，发现苏晚枝发了一条消息："你的那本诗集，总共多少首？"

"四十五。"

"你大二那年写了多少首？"

"十五首。差不多一周一首。"

"一周一首。"她停了一会儿。"那你那时候的生活就是——上课、写信、写诗、等回信。"

"对。很单调。"

"不算单调。算专注。"

"专注和偏执之间的差别是什么？"

她想了半分钟。"专注是你做的时候意识不到时间过去。偏执是你做的时候意识不到除了这件事之外还有别的东西。"

"那我是兼有之。"

"谁说的。你现在回头看觉得是偏执。但我从另一侧看，那就是专注——你对我付出了极大的专注。不是每个人都承受得了那种专注。但那不是你的错。"

陆允川看着屏幕上的这几行字。

他回想了一下：这是他们之间第一次出现"不是你的错"这样的句式。以前都是"我那时候接不住""我以为你不理解"。

他回了一个"谢谢"。

然后补了一句："我刚才说的'偏执'是在讲笑话。不是很成功的笑话。"

她回："我知道。你不用解释。"

午休结束。他下午开了两个会，写了一封回邮，改了一份标书。做事的时候脑子里偶尔跳出诗索引表格里某个日期对应的场景。但那些场景不会再拉着他往过去钻。它们只是浮一下，然后沉回去。

下班以后他没直接回家。他绕道去了菜市场。不是要买什么——是想听一听不是键盘的声音。菜市场五点以后开始打折，摊贩在喊"菠菜便宜了""最后一条鲫鱼"。他买了条鲫鱼，打算晚上煮汤。

煮汤的时候才发现没有姜。冰箱里只有一根葱和半头蒜。他用蒜凑合了。汤煮出来有点腥，但能喝。

喝汤的时候程砚打电话来："你昨晚睡了几个小时？"

"三个。"

"在干嘛？"

"做诗歌索引。"

"做完什么感觉？"

"像把自己十年的歌词一字排开。"

"有什么发现？"

"发现最早的几首诗里没有'灯'。从大三开始才有。"

"为什么？"

"因为大一大二的时候我在找光。大三开始我发现光不是别人给的——是自己留的。"

程砚沉默了一会儿。"你把这句话写进书里没有？"

"没有。这是对你说的。"

"为什么不对读者说？"

"因为读者不需要知道我在找什么。读者只需要看到那些诗在哪里。剩下的让他们自己体会。"

"你这叫信任读者。"

"对。"

"那我能把'光是自己留的'转述给读者吗？"

"不能。你是我的朋友。不是作者。"

"行。"程砚的语气带着一点笑意。"那你煮的鲫鱼汤好喝吗？"

"没放姜。有点腥。"

"活该。你总省步骤。"

挂了电话。鲫鱼汤喝完了。碗底有几根蒜末的残渣——蒜不是姜，蒜融不进汤里，只会浮在表面。

他把碗洗了。在洗碗的时候突然想起了诗索引表格里第36首的那一句"水声仍向远天生"。这句本来是仿宋小令里的固定末句，他每首仿宋小令里都会出现。但此刻在洗碗槽的水声里重新听到这几个字，他突然觉得不对——不是这句不对。是"仍向"不对。"仍"字假设了方向是连续的。但河流改道，方向变了，水还是水。

他把这个念头记在手机备忘录里。没有改诗。因为那首诗锁的是当年的自己。当年的自己的"仍"是真的认为方向不会变。

把碗洗完以后，他把手机的备忘录合上。明天的闹钟设为7:15。床头的灯照在昨晚打印出来的诗歌索引表上。最后一行是备注："保留原貌。"

他关了灯。原貌还在暗处。不需要再看。
