# 第050章 操场边的单词



时间：2008年十二月下旬，早操后



地点：操场跑道边和十班教室门口



卷次：第三卷：雪落在两栋楼之间



2008年十二月下旬，早操后，操场跑道边和十班教室门口，李慕白贴在掌心的小单词条先把这一天按进了十班的日常里。早操后的冷风里，十班一边跑步一边背单词，月考后的松散被重新拧紧。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

段璟发现自己总把注意力分给常俊凡，英语默写却不会因为喜欢而放过他。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看李慕白贴在掌心的小单词条，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

段璟也在这一天被推到前面。段璟的特点是敏感、自尊、偏理科，喜欢常俊凡却还在学边界。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“这个词我早上背了五遍，还是忘。”李慕白说。

“忘了就第六遍。”马立群说。

“跑步背单词，嘴里全是冷风。”梁一帆说。

“冷风也比空白卷面好。”常俊凡说。

“我现在知道了。”段璟说。

操场边的树叶已经掉得差不多，跑道旁剩下几块没化干净的雪，踩上去会发出轻微的碎响。

早操结束后，大家喘着气背单词，声音断断续续，像每个人都在和自己的懒劲拔河。

李慕白把单词卡贴在掌心，一抬手就能看见，可真正默写时还是会漏掉一个字母。

段璟背到一半忘了词义，脑子里却浮出常俊凡刚才说话的语气，他立刻有点懊恼。

常俊凡不看他，只把自己的单词按词性分成几列，一列一列往下过。

马立群说背书不是喊给风听。她让大家回教室后立刻默写，错一个抄三遍。

梁一帆说冷风都从嘴里灌进去了，英语应该自己冻牢，结果第一个被罚抄。

这一早晨没有浪漫，只有冻红的手指、写错的单词和一排被冷风吹得发抖的人。

操场边的单词没有悬在空处。操场边还没化尽的雪和冻红的手指，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，操场边的单词带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

马立群进门时不一定立刻说话，鞋底在水泥地上一停，操场边的单词带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在段璟快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。操场边的单词越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。李慕白贴在掌心的小单词条让他想说话，又逼他先低头看题。

段璟被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，操场边的单词留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。段璟路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为操场边的单词翻身，谁还在被窝里背单词，旁边人都听得见。

操场外的风掠过树梢，背单词的声音断断续续，像一群人各自给自己打气。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理操场边的单词里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：李慕白贴在掌心的小单词条、一只冻红的手、一句老师点名、一块还没扫干净的雪。

段璟沿着跑道边背单词，冷风一灌进嘴里就漏掉一个音节。这样的细节没人会写进成绩单，却会留在同学之间。

常俊凡把单词条贴在掌心，跑完操时纸边已经被汗水浸软。窗外风一过，纸页轻轻动了一下，像替谁把话咽回去。

宋乔站在操场边看英语书，操场哨声一响又赶紧合上。这个小动作很快过去，却让教室更像一个真正在运转的班。

周明澈把默写错的词抄三遍，第三遍才终于写顺。旁边的人没有特意看，可那一瞬间还是被段璟记住了。

付小川沿着跑道边背单词，冷风一灌进嘴里就漏掉一个音节。常俊凡没有接话，只把手里的事做完。

段璟没有因此变成特别醒目的人，可在段璟眼里，段璟从这天起不再只是旁边一张脸。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，操场边的单词把青春拉回现实：会不会就是会不会，不能靠心事补分。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
