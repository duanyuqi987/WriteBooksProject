# 小说类型-技法映射表

不同小说类型对应不同的技法组合和写作策略。本表由 novel-outline 在规划阶段使用，novel-draft 在写作时参照执行。

---

## 一、系统爽文（升级流 / 系统流 / 都市逆袭）

### 核心技法组合

| 优先级 | 技法 | 作用 |
|--------|------|------|
| 主1 | 繁章波折法 | 爽文核心：每段交锋制造多重转折，情绪过山车 |
| 主2 | 反衬法 | 打脸专用：表面写对手嚣张，实际写主角即将翻盘 |
| 主3 | 白描 | 战斗可视化：用具体动作写战斗，不用抽象形容 |

### 辅助技法

| 技法 | 使用场景 |
|------|---------|
| 一击两鸣 | 系统提示同时推进实力展示和剧情推进 |
| 避繁文法 | 重复性修炼过程用侧面交代，不每次详写 |
| 草蛇灰线 | 系统隐藏功能 / 身世之谜 的长期伏线 |

### 章节节奏模板

```
压抑期 (30%)：反派嚣张 → 主角沉默 → 读者憋屈
转折点 (10%)：系统提示 / 实力曝光 / 底牌亮出
释放期 (40%)：碾压反派 → 众人反应 → 读者爽到
余味期 (20%)：奖励 / 新目标 / 下一层悬念
```

### 禁用技法组合
- 禁用不写之写：爽文读者要的是直接，留白会削弱爽感
- 禁用暗透法过度：偶尔用，但不能替代直接心理描写
- 禁用春秋笔法：爽文不需要含而不露的评判

## 二、怪诞悬疑（规则怪谈 / 无限流 / 克苏鲁）

### 核心技法组合

| 优先级 | 技法 | 作用 |
|--------|------|------|
| 主1 | 草蛇灰线 | 伏线密度最高：每个反转都需要前置伏笔 |
| 主2 | 不写之写 | 恐怖核心：不说出全貌比说出更可怕 |
| 主3 | 春秋笔法 | 规则表达：用角色语言呈现规则，不直接告诉读者 |

### 辅助技法

| 技法 | 使用场景 |
|------|---------|
| 暗透法 | 让配角说出主角发现的异常（增加可信度） |
| 繁章波折法 | 推理交锋场景，每次对话都可能颠覆认知 |
| 一击两鸣 | 日常场景暗含规则线索 |

### 信息控制节奏

```
恐惧 (20%)：发现异常 → 认知被入侵
推理 (40%)：分析规则 → 验证假设 → 发现更多异常
震撼 (20%)：真相揭示 → 之前的线索全部串联
余悸 (20%)：新的异常出现 → 更大的谜团
```

### 禁用技法组合
- 禁用白描过度：过于具体的描写会消除恐怖感
- 禁用避繁文法过度：悬疑需要细节，重复性内容也可能是线索

## 三、冷白描（现实主义 / 历史 / 战争）

### 核心技法组合

| 优先级 | 技法 | 作用 |
|--------|------|------|
| 主1 | 白描 | 最核心技法：只写看到听到摸到的，不写感觉到的 |
| 主2 | 不写之写 | 情绪留白：不写情绪让读者感受到更深的情绪 |
| 主3 | 春秋笔法 | 社会批判：用事实呈现代替直接评判 |

### 辅助技法

| 技法 | 使用场景 |
|------|---------|
| 层峦迭翠 | 相似场景的前后对照（两次离别/两次重逢） |
| 暗透法 | 通过他人之口说出角色不愿说的心里话 |
| 反衬法 | 喜景写悲：在平静场景中暗藏悲剧 |

### 情绪控制原则

```
平静 → 压抑 → 爆发 → 平静（回归克制）
```

核心情绪永远用行动和细节呈现，不直接描写。

### 禁用技法组合
- 禁用繁章波折法过度：刻意转折会破坏朴素感
- 禁用一击两鸣过度：每句都有深层含义会显得匠气
- 禁用草蛇灰线过度：过于精巧的伏线会破坏"生活感"

## 四、言情/古言（古代言情 / 宫斗 / 宅斗）

### 核心技法组合

| 优先级 | 技法 | 作用 |
|--------|------|------|
| 主1 | 暗透法 | 感情线核心：通过他人之口揭示角色真实情感 |
| 主2 | 繁章波折法 | 情感交锋：每次对话都暗含态度转变 |
| 主3 | 反衬法 | 口是心非：表面说的和心里想的不一样 |

### 辅助技法

| 技法 | 使用场景 |
|------|---------|
| 一击两鸣 | 日常场景暗含权力关系变化 |
| 白描 | 情感场景的动作化表达 |
| 层峦迭翠 | 男女主相似情境的不同反应（展示关系变化） |

### 禁用技法组合
- 禁用春秋笔法过度：言情读者要情感冲击，不含不露会显得冷
- 慎用不写之写：关键情感场景需要充分展开

### 辅助技法

|------|---------|

| 一击两鸣 | 修炼场景同时推进修为和心性成长 |

## 起点热门小说技法对照

基于起点中文网头部作品的分析归纳：

| 作品类型 | 代表技法 | 读者吸引力要素 |
|---------|---------|--------------|
| 系统爽文 | 情绪过山车（压抑→释放循环）+ 具体数字可视化 | 解气、代入感、可预期的回报 |
| 诡秘流 | 渐进式信息披露 + 认知颠覆 | 好奇心、解谜欲、第二次阅读发现新线索 |
| 重生文 | 信息差 + 预知优势的兑现 | 代入感、先知爽感、改变命运的满足 |
| 无限流 | 规则设计 + 层层反转 | 解谜欲、生存紧张感、智慧对抗 |
| 赘婿/逆袭 | 身份反差 + 打脸循环 | 身份认同、公平世界假设、翻身快感 |
