# 第001章 开学日

时间：2010年9月1日

地点：青阳县一中高一三班教室

卷次：卷一：初见秋风

九月一号。青阳县一中。

校门口挂了一条红横幅，上面写着"热烈欢迎2010级新生"。布是新的，红得扎眼。家长和学生从横幅下面挤过去，扛着行李、拎着暖壶。门口两边的砖墙上贴满了各类培训班的广告。红色的、蓝色的、黄色的纸。有的翘了角，在风里啪啦啪啦响。

一个中年女人拉着她儿子的手站在公告栏前面看分班表。人太多，她踮着脚看了一会儿，又退出来。"三班。"她说。儿子嗯了一声。两人往教学楼走。行李箱的轮子在水泥地上格格响。

顾临风是一个人来的。

他爸把他送到镇上汽车站，塞了两百块钱。两张红票子，对折了一下，塞进他裤兜里。他爸的手粗糙，指甲缝里还有泥。"省着花。"他爸说。然后就转身走了——步子不快，旧解放鞋踩在水泥地上没什么声音。走到拐角的时候咳了一声。然后消失在巷子口。

顾临风把头转回来。汽车站很小，候车室里只有三排塑料椅。检票口没人检票——司机站在车门旁边抽烟。蓝色中巴车，车身上溅了很多干泥巴。他背着书包上了车。靠窗坐下。座位上的海绵垫子坐塌了半边，弹簧顶着屁股。车窗开了一条缝，漏进来柴油味。

车开了。窗外的树一棵一棵往后退。白杨树。树干上刷了半截白灰。树后面是稻田。稻子已经开始泛黄了。有人在田里弯腰割草。

车上坐了十几个人。前排有个老太太抱着一只母鸡，母鸡被绳子绑了脚，蹲在她膝盖上不动。后排有个男人在听收音机，沙沙响，播的是天气预报。"明天晴转多云，最低气温二十二度。"

到县城是上午九点。车站门口有个卖烤红薯的老头。推车上挂了个锈铁皮烟囱。烟囱口冒着白气。红薯的甜味混在柴油味里。他买了一小个，五毛钱。老头从炉膛里夹出来，拿旧报纸包了一半。报纸上印着"青阳日报"四个字。他接过红薯。剥皮的时候烫了手指，换了两回手才吃完。

他问了两次路。第一次问了一个卖水果的大姐。"一中怎么走？""往前，过了桥右拐。"第二次问了一个穿校服的学生。"高一三班在哪栋楼？""教学楼。三楼。"

路上经过了一条河。河水是绿的，面上漂了一层绿藻。两头鸭子在绿藻里刨食，屁股朝天。河边的石栏杆上坐着几个老头在钓鱼，鱼线垂在水里一动不动。对岸有家理发店，门口的旋转灯在转。红白蓝三色。

一中大门是铁栅栏，漆成了绿色，有些地方锈了。门卫室里一个老头在看报纸。他走进去的时候老头没抬头。

校园里人很多。操场是水泥的，篮球架上有两个男生在投篮。球砸在地上嘭嘭响。没进。球滚到场边，一个女生捡起来扔回去。

高一三班在教学楼三楼。楼梯扶手是铁管的，上面坑坑洼洼的，漆早磨没了。台阶中间有道凹槽，不知道多少人踩出来的。走廊很窄，两边是教室门。门上贴着班级号——高一（一）、高一（二）、高一（三）。字是打印的宋体，贴在门楣上方的玻璃框里。顾临风在第三个门前停了一下。走廊里不断有人从他身边挤过去。然后他走了进去。

教室不大。四十来张课桌排成七八列。已经坐了二十多人。有人在聊天，有人在擦桌子，有人趴在桌上。日光灯亮着——灯管发出嗡嗡的低响。天花板上有两个吊扇在转，扇页上积了一层灰。

教室里有一股粉笔灰和旧木头混在一起的气味。黑板是新擦过的，还有水渍。靠窗那面墙的下半截刷了绿漆，上半截是白墙，交界的那条线不太直。

他选了倒数第三排靠过道的位子坐下。

桌面上有上届留下的圆珠笔印子——横竖都有。有些是数字，有些是字母，有些看不出来是什么。他用手指抹了抹，没抹掉。桌肚里有半截断掉的铅笔和一张揉皱的草稿纸。纸上的字看不清了。他把铅笔捡出来放到走廊的垃圾桶里。

椅子腿有点晃。他蹲下去看——有一颗螺丝松了。拧了两下，拧不紧。算了。

书包是灰色的，拉链有点卡。他拔了两下才拉开。里面一本语文书、一本数学书、一个铁皮铅笔盒。铅笔盒是他姐用过的，面上印着小虎队的贴纸，边角磨白了。打开里面——一支圆珠笔、一支铅笔、一块橡皮、一把透明直尺。还有一支钢笔，没水了，墨囊是空的。

他把书拿出来摞在桌角。语文书封面上有上一届学生的名字——张磊。用蓝水笔写在右上角。他拿橡皮擦了擦。没擦干净。又用指腹蹭了两下。蓝印子变淡了一些。

又过了一会儿。

一个女生从前门走了进来。

浅蓝色T恤。深蓝色牛仔裤。扎马尾。

九月的阳光从窗户斜打进来，刚好落在她侧脸上。她的头发在光里是褐色的。她往教室里扫了一眼——目光很快，像教室是个早就知道长什么样子的地方。然后走向第二排靠窗的位置。

顾临风的手放在铅笔盒上，没动。

她坐下。把书包挂到椅背上。书包是浅灰色帆布的，洗过很多次的样子。拉链上有个小东西晃了一下——看不清是什么。然后她拿出一本书翻开了。翻书的手势很轻。封面朝下扣在桌面上。书名看不见。

教室里又进来了几个人。一个男生扛着个大蛇皮袋，撞到了门框，咚的一声。几个女生从前门进来，边走边说话，声音压得很低但一直在笑。有个矮个子男生在第三排坐下，从书包里掏出一包饼干撕开了。

班主任姓王，四十多岁，男的，戴眼镜。眼镜是金属框的，镜片上有点灰尘。他穿了一件灰衬衫，袖子挽到手肘。他走进来敲了两下讲桌。粉笔灰扬起来一些。安静了一些。

"我姓王。你们的班主任。"声音不高，语速不快。手指在讲桌上点了两下。"今年我带你们这届。高中三年很关键。"他扫了一圈教室。有人在底下窸窸窣窣地翻书。

他念名单。名单是打在一张A4纸上的，他用手指顺着往下滑。念到一个，就抬头看一下站起来的那个人。

"顾临风。"

"到。"他站起来。椅子腿蹭了地面一下。

王老师从眼镜上面看了他一眼。在名单上打了个勾。继续往下念。

"苏晚晴。"

"到。"

声音从第二排靠窗的位置传来。不响。刚好能听见。像从窗帘后面透过来的一小块光。

顾临风在"苏晚晴"三个字下面用指甲划了一道印。指印很浅。然后他把纸翻到下一页。

下午发了课本。十几本。语文、数学、英语、物理、化学、生物、政治、历史、地理。本子好几摞。发书的时候从第一排往后传。课本封面又滑又凉，带着新书特有的油墨味和胶水的酸味。他把书垒在课桌左边，本子放右边。书架起来比铅笔盒还高了一点。

数学课本的封面是一个几何图形。他翻开看了一眼。第一页写着"集合"。合上了。

晚自习前回了一趟宿舍。宿舍在教学楼后面的那排平房里，红砖墙，水泥地。门上的绿漆掉了一大片。六人间，上下铺。他的床位靠门，下铺。铁架床，床板上铺了一层薄褥子。被子从家里带来的，蓝格子，洗得发白。枕套上有他妈补的一块补丁——白色的布补在蓝色的枕套上，针脚很细。

室友陆陆续续来齐了。对面下铺的是个戴眼镜的胖男生，抱了一箱方便面放床底下。红色包装。康师傅红烧牛肉面。他自我介绍叫孙磊——"孙悟空"的"孙"，"三个石头"的"磊"。说完推了推眼镜，笑了。眼镜片后面有一双小眼睛。

上铺那个叫刘洋，瘦瘦的，话不多，进门打了个招呼就躺上去看手机了。手机是诺基亚的，按键式的。屏幕是蓝光。

另外两个人住靠近窗口的两张床。一个叫赵一鸣，脸圆圆的，说话带点口音。"我家就在县城。"他说。另一个叫陈磊——跟孙磊同名不同姓。他进来的时候搬了一个大箱子，里面装的全是书。教辅书。

孙磊从床底下掏出一包方便面，干吃。捏碎了包装袋，撒上调料粉，摇了几下。倒在手心里抓着吃。问顾临风要不要。他摇摇头。孙磊又问了上铺的刘洋。刘洋从上面伸手下来接了一把。

宿管老师姓周，五十来岁。头发花白，穿一件蓝色工作服。他挨个敲门——"电灯十点关，水房十点停水。"重复了三遍。门牌上贴了值日表，蓝色圆珠笔写的。星期一是孙磊，星期二是刘洋，星期三排到顾临风。星期四和星期五是那两个人——赵一鸣和陈磊。字写得有点歪。纸边上还有一道蓝色指印。

水房在走廊尽头。里面一排水泥洗手池，水龙头好几个不出水的。有一面大镜子挂在墙上，镜子右下角缺了一块。晚自习前去刷牙。牙膏中华牌的，皮管子的，要从后面往上卷。另一个水龙头下有个男生在洗头，弯着腰，头发滴着水。泡沫流进下水道。

晚饭在食堂吃的。食堂在教学楼后面，铁皮顶，里面摆了上百张白桌子和蓝塑料凳。他打了三号窗口的土豆丝和米饭。坐在角落里吃完的。吃完了去水龙头下冲了冲饭盆。走回教室的时候天已经暗了。路灯亮了——橘黄色的。

晚自习九点四十下课。回到宿舍，灯十点整灭。灭之前周老师在走廊里喊了一声——"关灯了。"然后开关啪的一声。灯管暗下去。

窗帘没拉严实。月光漏进来一条。细的。打在水泥地上。

顾临风没睡着。上铺刘洋翻了个身，床晃了一下。对面孙磊打呼，不是很大声的那种，是吹气一样的声音。

他从书包侧兜里摸出一支笔。翻开语文课本的封底内页。在右下角写了两个字——苏晚晴。

写完看了几秒钟。然后拿笔在名字上划了两道横线。横线上宽下窄。

纸破了。笔画之间的纸碎了，透出下一页的铅字。他把碎纸屑捏在手指间，团成一个小粒，放在枕头边。

他把书合上，放到枕头底下。

走廊里有人在唱歌。唱的是《星空》。"命运偷走如果，只留下结果——"后面一句没听清。声音越来越远。然后没了。

窗外起了风。风吹进来的时候帘子动了一下。月光也跟着动了一下。

宿舍安静下来。外面偶尔有脚步声——大概是值班老师查夜。水泥地上沙沙的声响，走得不快。到了尽头。停了。

顾临风闭上了眼睛。又睁开。天花板上的脱漆点还在——白天看到的那几处。黑暗里看不清。但他知道它们在哪。

然后他又闭上了。
