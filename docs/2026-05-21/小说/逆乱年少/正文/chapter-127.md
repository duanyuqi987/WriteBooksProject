# 第127章 火车

时间：2012年7月5日，上午至下午

地点：江城开往太行山方向的绿皮火车

卷次：第六卷：雪落无声

绿皮火车。硬座。

八个人的座位不在一起，分了三排。陈维中和两个女生坐在前面，林玉起和苏安晴坐在中间靠窗的位置，孙亦舟、方远和赵一琳在后面。

车窗只能开一半。七月的风灌进来，带着柴油味和田野里割过的青草味。

火车开出去一个小时后，苏安晴开始犯困。

她先是低头看了一会儿手机——山里信号不好，屏幕上的QQ头像灰了大半——然后合上眼，靠在硬邦邦的座椅背上。头往左边偏了一下，又直起来。过了几分钟，又往左边偏。

这次偏过去没再直起来。

她的头慢慢滑到了窗玻璃上，靠着玻璃停住了。窗外的电线杆一根一根地往后退，她的头也跟着玻璃的震动一点一点地动。

林玉起坐在她右边，靠着过道。

他不敢动。

不是夸张——是真的不敢动。他的右手放在两人共用的那个扶手上，她的左手也放在上面，两根手指之间隔了不到一厘米。他能感觉到她手指的温度，隔着空气传过来，像一道很细很细的电流。

他没有把手拿开，也没有把手靠近。

就这么放着。

火车钻进了隧道。车厢暗下来，窗玻璃上映出了苏安晴的脸——闭着眼，嘴微微张着，眉间有一小片窗玻璃反光留下的光斑。

林玉起在玻璃上看了一会儿那张脸。

隧道过去了。光重新涌进来。苏安晴皱了皱眉，没醒。

坐在后面的方远探过头来，小声说："林玉起，你手僵不僵？"

林玉起回头看了他一眼。

方远笑了笑，缩回去。

火车在一个小站停了三分钟。站台上没有人，只有一块掉了漆的站牌和一株歪脖子的槐树。苏安晴醒了。

"到哪了？"

"才过武安。"

她揉了揉眼睛，坐直了，看了一眼两个人之间的距离——没变。

"你没睡会儿？"她问。

"不困。"

"昨晚没睡好吧。"

林玉起没说话。

她从包里翻出一瓶水，拧开喝了一口，又把水瓶递给他。"喝吗？"

林玉起接过来，瓶口还带着一点温热。他喝了一小口。

窗外的山慢慢多了起来。不是南方那种圆润的丘陵，是北方那种有棱有角的石头山，裸露的岩壁上长着矮松和酸枣棵子。

苏安晴看着窗外，说："快到了。"

"嗯。"

她的手指在窗玻璃上无意识地画了一道线。那条线跟着山脊的轮廓走，画得很慢。

>
铁轨三千里，青山一路同。
少年多不语，心事在窗中。
>
