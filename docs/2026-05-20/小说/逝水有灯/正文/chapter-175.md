# 第175章 最重的话多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：男生宿舍书桌前

## 诗歌插页 P175：原创仿唐绝句

旧屏无光字有痕，
当年重语不堪温。
若将拒绝称为罪，
先把深情误作刃。

多年后陆允川没有保留那段聊天截图。他只在旧日记里写了一句：“我说过一句很重的话。”

后来整理小说时，他把那句话补进正文，没有美化，也没有删轻。因为如果不写出来，读者只会看见他的疼，看不见他怎样把疼递成了别人的负担。

他曾经想为那句话找很多理由：年少、冲动、太喜欢、太害怕。现在他知道，理由可以解释来源，不能取消后果。

在旧宿舍楼下，他站了一会儿。那栋楼翻修过，窗框换成新的，书桌当然早就不在。他却还能想起屏幕的蓝光，想起祁野说“你这话太难听了”时的表情。

那是朋友很少露出的严肃。也是他后来最感谢的一种严肃。

陆允川在日记旁边补：拒绝不是伤害，责怪才是。写完后，他没有觉得自己高尚，只觉得终于把那句话放回了它应有的位置。
