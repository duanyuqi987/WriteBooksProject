# 第094章 夏天的聚餐旁观

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P094：原创现代诗

另一张桌也有风扇，
也有长辈问城市、分数和将来。
她把杯口转了半圈，
没有人知道，
被反复提起的名字，
也会在自己的热闹里沉默。

陆允川在小饭馆里听见那句“不一路”时，夏知遥正在城北亲戚家吃饭。她的录取通知书被姨妈传给一圈人看，红色封皮在玻璃转盘上转了一圈，最后落回她手边，边角沾了一点汤。

“以后去那么远，谈朋友没？”有人笑着问。

她低头夹青菜，“没有。”

“高中同学里没有合适的？”

夏知遥把筷子放下，又拿起来，“大家都忙着上学。”

这句话让饭桌安静了半秒，又被别的玩笑盖过去。她松了一口气，却也有些疲惫。她知道陆允川喜欢过她，也知道自己没有办法回给他同样的东西。可这种知道不适合拿到饭桌上解释，更不适合被亲戚当成轻松话题。

饭后，她站在阳台给母亲打电话。楼下有人搬啤酒箱，瓶子碰在一起。手机里母亲问她行李箱够不够大，别忘了带薄外套。她一一答应。

同学群里陆允川的升学宴合照跳出来。她点开看了一眼，看见他坐在边上，笑得不太自然。她没有私聊，只在群里跟着大家发了“顺利”。

这不是冷淡。她只是清楚，任何多余的话都可能被他保存成另一种期待。她能给的祝福是真的，她不能给的，也同样是真的。
