# 第030章 操场另一边

时间：2011年6月

地点：学校操场

卷次：卷二：窗边人影

六月。体育课。下午第三节课后——太阳偏西——不那么毒了——操场——土操场——扬尘——跑道上的煤渣——干燥——被踩碎——细粉末在脚底下——白帆布鞋踏上——鞋尖发灰。煤渣跑道边上的草已经被踩秃了一半——秃处——翻出干土——上面还有篮球鞋底的菱形印。

体育老师吹哨子——"吡吡——吡——"——集合——"男生——篮球——女生——排球——各去各的场地——器材室——找球——"男生们往南边的篮球场走——捡球——那个瘪了气的旧篮球——拍在水泥地上——通——通——拍不高——换了新的——"这个太轻——"再换——"那个，那个——去年打过的那颗——还不错。"女生们抱着排球——往北——排球场——需要先在泥地上架网——铁杆——尼龙网——许若琳在拉网绳——"你帮我这边——""够到吗——"

顾临风拿着篮球——跟在男生队伍的后面——右手——球托在手腕——转动——先左再右——他在看球的边线——皮面的接缝。赵一鸣走在他左边——拍了一下他手中的球——球从的手里弹出去——弹在地上——他接住——运了两下——赵一鸣:"今天打半场还是全场——""随便——"

"你今天不在状态——"

"还行。"他把球夹在腋下——然后——从他腋窝——出来——他和赵一鸣并排走着——地上的影子——一肥一瘦。

篮球场上——高一的男生们分两队——三对三——半场——篮筐的网兜烂了一半——剩下的两缕绳子——垂着——风来——摇——阴凉处的管子上的锈——褐斑。顾临风分到了穿白色背心那一队——他把校服外套脱了——挂在旁边单杠上——身上的深灰色圆领T恤——领口——有些旧了——边口卷。赵一鸣在对面队——挥手——"你们——开球——"

跳球。球被拍开——又高——落下——白色背心队的后卫接住——传——球——到了顾临风手里——他带球——左手护——右手运——往左边一拉——变向——过了第一人——来到罚球线——抬头。

然后——目光——往左边看了。

操场另一边。排球场。女生们在打排球——白色的排球在空中划弧线——来回——过网——落——脚——砂——有人喊——"接——"声音拉高——在空气里往上——断——球——"砰"——回到了另一边——网——抖了——一会儿——又回。苏晚晴在场上——灰色T恤——校服裤子——裤管往上——卷到脚踝——露出——一小截小腿——白。她在场上移动——不快——但是在动——在网前——半蹲——两手掌向外——接球——球飞过来——抬起手腕——拍——"砰"——球跑得直——过网。她打排球——动作有点笨——不太会——手合十时——两手掌之间总会留一个缝——做不到手指完全叠——她旁边的女生对她说"手腕——手腕——要定——"她点头——试了下——还是笨——球从她手上弹出去的弧度——是微微偏——然后许若琳在场上补位——挡——"没事——"排球从她脚前——落下——她弯腰——手去捞——捡起来了——把球扔回给发球的人。

顾临风看了一秒。赵一鸣在喊——"传球——你在看什么——"然后——一个篮球——从他右侧——朝他脸飞过来——硬——快——扑面——皮面和汗水——瞬间——遮住——视野——"啪——"

闷的一声。

顾临风往后退了一步——捂着右脸——手指——按着颧骨——脸侧的皮肉和骨——在震——篮球从脸上弹下去——落在地上——弹——弹了两下——滚到三分线外——停了。赵一鸣跑过来——"卧槽——你没事吧——"

"没事——"他把手放下来——脸上——颧骨位置——红了一块——皮肤——开始——热——发着——微胀——那块——麻——然后——开始跳痛。

"你怎么不接——我说传球——你愣着干嘛——"

"没注意——"

赵一鸣看了他一眼——嘴角——没说话。然后——顺着刚才他眼光的方向——看了一眼——操场对面——排球场那边——女生们的白色排球——还在空中来回飞。他什么都没说——看了两秒——收回——拍拍顾临风的肩膀——"行了——去水龙头冲一下——冷水——别肿——"

顾临风走到操场边的水龙头——下面那个——就是跑道北侧的——矮了一些——拧开——水是凉的——地下水——夏天也冷——手拢着——接住——往脸上泼——右颧骨——被冷水——激了一下——缩——然后——适应——皮肤——开始觉得那股冷——压在跳动的血管上——慢下来了——冲了大概两分钟——直起腰。水珠——从脸滑下——下巴——滴——。从水龙头的金属面板上——亚克力——反光——能看排球场——模糊——在反光面上——灰色T恤——白色排球——在金属罩上——像一幅扭曲的画——她的手臂——接球的——那个角度——一个轮廓——他认得的。然后他用袖子擦了一把脸——袖子——吸干了——脸——返回场。

下半场。他打得很专注——跑——抢——位置——补防——拿球——投篮——每一个——接到赵一鸣的传球——球路——稳——每一个——进了——传回来落地——抢到篮板——再投——对面三个——赵一鸣在旁边挡拆——"走——"他进去——球——投——弧线——进了四个球——投四进三——罚球线——进了一分——共得八分——没有再看左边一眼——没有——不再看——脚——跟地——粘着——用力——蹬——防守——进球——结束——他们队赢了。

赵一鸣在收篮球——往网兜放——弯腰捡那颗把人脸弹肿了的——看了一眼——扔——兜里。然后直起腰——"今天那个砸你的球——是李伟扔的——回头让他请你喝水——赔罪——""不用——""必须赔——你脸上现在还红着呢——"

顾临风揉了揉右颧骨——现在开始觉得——酸——有点肿——指头按下去——骨头——有点闷——"嗯——"

"你刚才在看什么——"

他把球放进球筐里——球筐是铁框——上面盖着防雨的布——"没什么。"

赵一鸣没再问。两个男生往操场外面走。阳光——从背后照——地上的影子——在前面拖——斜——长。经过排球场——女生们也散场了——苏晚晴在那边——弯腰——收网——把网绳解开——许若琳在旁边帮着扇网——还有几个女生——一起在卷——卷成圆柱——夹在铁杆和支柱之间——往器材室送。她回过身——顾临风从她身后——离了—十米——走过去——他右手摸着自己脸上那块红肿——左手——拎着校服外套。她没看到他——在许若琳说——"哎——天——这么热——粘住了——"然后用手扇着。

赵一鸣旁边说:"你知道你刚才站在场上发呆那次——对面三个人——两个防我——你空位——我差点骂人了——"

"下次不会了。"

"最好不会——"赵一鸣想了一下——走了两步——又说——"你下次——要看——就正大光明地看——躲着看——容易崴脚——也容易挨球——"说完——没停脚步——往前走。

晚上——回教室——顾临风坐在座位上——脸上涂了药膏——赵一鸣从宿舍给他带的——他进教室的时候——药膏放在桌上——笔压着——"擦——不擦明天肿——"

苏晚晴从前门走进来——书包——甩在肩上——他抬头——她——目光——落在他脸上——颧骨——停了一下——可能——看到了那块红印——眼皮——眨了一秒——但——没停——然后走到自己座位——坐下来——拉开书包——拿出物理书——和笔。

顾临风低下头——开始做题——那道力的题做了好几次——又划——药膏的薄荷味——凉凉的——在皮肤上——血管跳动——药膏下——感觉在抽——外面的蝉——从银杏——吱——一声——长——刺——停了。

晚上宿舍——赵一鸣还在翻漫画——翻完——说——"你以后打球能专心点不——""能——""那就好——你不专心——我助攻——少了——"顾临风嘴角动了一下——赵一鸣翻了个身——床——咯——然后说——"那个女生——打排球——打得挺好的——比你打球专心——"然后——没再说话。顾临风的眼睛睁开了一会儿——盯着天花板——水印——那道光斑——比上学期——更靠近墙角——然后——闭上——手——放在枕头边——枕头下——日记本——今天还没写——但今天的事——不需要写——记住了。
