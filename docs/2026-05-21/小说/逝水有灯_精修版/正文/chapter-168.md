# 第168章 牵手的误会之后

所属部卷：第四部：失去的美丽
时间：大二冬夜之后几天
地点：宿舍 / 操场

## 诗歌插页 P168：原创仿宋小令

炉火边，夜色平，
半窗旧影入疏灯。
问当年、几回执念，几回无凭。
不把归人留梦里，
水声明日照常生。

后来的几天，陆允川和苏晚枝没有联系。

不是冷战，只是两个人都需要消化那顿食堂里的话。苏晚枝说的那些，陆允川在跑步的时候会反刍；他问的那些，苏晚枝在复习累了的时候也会想起。

周五晚上，陆允川在操场跑步。跑到第七圈，夏知遥的身影出现在看台边——她刚从奶茶店下班，围裙还没摘，手里拎着一杯热奶茶。看到他在跑道上喘气，她放慢了脚步。

"又跑？"

陆允川停下来，弯着腰撑住膝盖。"叶澄说情绪激动就跑步。"

"叶澄的医嘱看来你终于听了。"夏知遥在看台第一排坐下。

陆允川擦擦汗，也在看台上坐下。两个人之间隔了半米的距离。操场上还有人在夜跑，鞋底在塑胶跑道上发出有节奏的闷响。

"那天的学长——你知道吗？"夏知遥问。

"不知道。苏晚枝解释过了——是通讯社的学长。扶她是因为她踩进下水道缝里。"

"对。"夏知遥说，"那个学长有女朋友。他女朋友是我们奶茶店的常客，每次来都点抹茶拿铁。上周他还带她来买了两杯，两个人用两根吸管喝同一杯，腻得不行。"

陆允川没说话。

"你想过没有，"夏知遥继续说，"如果那天你看到之后没去找苏晚枝求证，而是来问我，或者问叶澄——你可能五分钟就知道真相。但你没有。你自己先难熬着，再把你的难熬变成质问抛给她。她被问了，就必须摆出'受审者'的姿态来解释——好像你们之间有一个需要用解释维持的约定。"

"我确实没有想过先问你们。"

"对啊。你的本能是自己消化不了就去找当事人求证，而不是先找旁人核实。这本身不是什么大问题，但对你来说是致命的——因为你的求证方式总是长在负情绪上。你一开口，语气就沉了。"

陆允川靠回看台座椅，看着夜空。天上没有星星，但操场的大灯把整个天空照成淡橘色。

"你分析得跟叶澄一样。"

"因为叶澄培训过我。"夏知遥把奶茶插了吸管喝了一口，"她说——如果哪天陆允川来找你八卦苏晚枝的事，不要让他自己去求证。先帮他核实。他不是一个擅长单打独斗的人。"

"叶澄连这个都安排了？"

"对。她在预防领域是绝对专业的。"

陆允川笑了。不是苦笑——是终于被朋友织成一张网的感觉。他笑完之后沉默了很久，然后问："你觉得苏晚枝现在还会生我的气吗？"

"不至于生气。但可能——更多是失望。因为你在台阶那次之后说你不追问了，结果一周没到又犯了一次。"

"那她现在会怎么想我？"

"不是你的问题。"夏知遥把空奶茶杯放到一边，"她不会想'他又犯错了'。她会在心里告诉你——好吧，他又犯了一次，他能改，但需要时间。她给过你机会，就说明她还是觉得你能变。"

"你这么确定？"

"不是确定。是——"夏知遥站起来，把围裙往后腰推了推，"苏晚枝那类人不记仇。她记的是关系舒不舒服。如果有一天她觉得跟你相处不累，她就会靠近。如果一直很累，就会远离。所以你要做的就是让她不累。就这么简单。"

"就这么简单。"

"对。但对你来说——"她裹紧外卖工服，冬天的夜风把她围裙上的奶茶渍衬得格外分明，"简单的东西最不容易做到。因为你不信简单。你总觉得不写三千字、不面谈五十分钟就无法让事情过去。"

陆允川站起来。"那我试着信一次。"

"加油。"

夏知遥走了几步，又回头："对了。苏晚枝今天跟我说了一件事——她说，如果你不问她那次食堂的事，她其实会自己说的。"

"什么意思？"

"就是你那个误会——如果你没去找她，过两天她自己在宿舍聊到那天采访的事，可能顺便就说'差点踩空磕到下水道缝'。但你抢先一步。她就没有'自然说出来'的机会。你等于是抢走了她的主动性。"

陆允川把这句话反复咀嚼了几秒。

"所以我需要的不仅仅是等——是信任会自然发生的事不需要我来提。"

"你终于开窍了。"

夏知遥走了。她的脚步在跑道上踏出均匀的节奏——跟她在奶茶店站了六个小时后还能踩出这种节奏，是一种天赋。

陆允川沿着操场又跑了两圈。他想起程砚说的"吞下去"，祁野说的"吞到不想说为止"，叶澄让他去跑步，现在夏知遥又补了最后一块拼图——等她自己说。

四个人，四种表达方式，说的都是同一件事。

他回到宿舍，打开"Siri。帮我记个备忘录"——祁野和程砚同时转头看着他。

"内容是什么？"程砚问。

"'不要抢在她前面求证。等生活自然摊开。'"

"谁教你的？"

"夏知遥。"

"她是在奶茶店进修过心理学吗？"祁野一脸惊讶。

"不是。是叶澄培训的。"

程砚和祁野对视了一眼。

"叶澄这个人，"祁野说，"应该在学校开一门选修课。题目就叫《陆允川的情感预防与控制》。学分三分，必修。"

"你敢不敢当着她的面说？"

"不敢。"

三个人笑了。冬夜宿舍的灯管发出微微的嗡声，窗外操场的大灯还在亮着。

陆允川把备忘录关上，然后把手机放到一边。他想了想，打开微信，给苏晚枝发了一条消息。很短：

"那天食堂的事，对不起。我不应该用那种语气问你。下次我会先等，先问旁人，问过之后再决定要不要开口。"

发过去之后，等了半个小时，苏晚枝回了：

"其实你不用跟我道歉。我也反省了一下——我可能对'被审问'这件事反应过激了。从小就不喜欢被人怀疑。你刚好碰到了那个点。"

"那以后——再有这种事，我怎么做你会比较舒服？"

"你可以问我：'昨晚那个学长——他是谁啊？'语气轻一点。我能听出来的。"

"像这样？"

"对。就像讨论食堂的菜价一样。"

"食堂菜价最近涨了吗？"

"涨了。鸡蛋灌饼从三块涨到三块五。"

"什么？什么时候的事？"

"上周。你果然没注意。"

然后就顺着这个话题聊了下去。关于鸡蛋灌饼的涨价，关于食堂哪个窗口的大妈最凶，关于某个专业课老师今天讲错了一个公式。聊了大概十来条，没有任何沉重的内容。轻得像棉花，但比棉花有营养。

陆允川把聊天的最后一句放在那里——苏晚枝说："不过话说回来，你跟夏知遥的关系其实挺让我羡慕的。你们能这样——不费力气地帮对方。"

他没有回得太快。想了很久，打了一行字：

"因为也不是一开始就这样的。她也骂过我。"

"她还会骂人？"

"会。上周她在我面前骂了十五分钟，说我买笔还不给祁野带饭，导致祁野在宿舍里饿到泡面都没泡——听起来是骂我，但其实是骂我没照顾自己。"

"哈哈哈哈。"

四条"哈"，不多不少。这是这段时间以来最长的一次笑声。

陆允川看着这四个"哈"，觉得这比所有长篇解释加起来都管用。有些关系缝了针之后会肿，但最后会好的。不是不疼了——是不影响正常走路了。
