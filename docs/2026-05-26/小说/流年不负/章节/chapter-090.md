# 第090章 诗

时间：2014年3月

地点：楚州师院图书馆

卷次：卷五：各自南北

三月。图书馆三楼靠墙第四个座位。

他每天下午都在那里。上完课以后——有时四点，有时五点——走到图书馆，爬三层楼，坐到那个位置。楼梯是大理石的——被无数人踩过的台阶中部微微下凹——扶手是木头的，被无数只手掌摸得发亮，木头纹路都摸出来了。三楼走廊里贴着"静"字告示牌——红底白字塑料牌，左上角缺了一小块。三楼走廊里总有几把没人收的折叠雨伞——靠在墙根。

窗外槐树还没长叶子——光秃秃的枝干伸向灰色的天空。三月的阳光不太暖——从窗户斜进来，在桌面铺成一道白色长方形。他把笔记本放在那道白光里面——墨水字在光下有那种很深很透的蓝。桌上放着黑色封面的笔记本——没有横线的——封皮是布纹的，四个角磨白了，露出下面的灰纸板。左边摊着《现代诗歌选读》——从图书馆借的，书脊上贴着"I226/12"的索书号标签。封皮是深绿色的——跟高中时那本诗集一样——都是深绿色。右边是笔记本。看几页，写几句。

图书馆里有固定的声音。左边第三个座位是考研的学姐——翻书声很大，哗啦啦的，偶尔停下来用笔在笔记上画。对面是个看杂志的男生——低声笑出来，大概是看了笑话——笑了一声赶紧捂住嘴。斜后方是个戴耳机听英语的女生——耳机漏音明显，"listen to the conversation"，一直重复这句。管理员推着小车经过——车轮嗡嗡响着，停在新书架前面，开始一本本往架上摆——书脊滑进书架时发出沙的一声。

风格在变。

高一写古体诗。七律。模仿唐诗三百首。押韵、对仗、每一联都工整。意象离不开"月""柳""雁""泪"。"那年九月风初凉""三载同窗未敢言"。那些诗规矩得像他当时看她的目光——隔着三排课桌，不敢越界。但现在不一样了。

韵脚没了。对仗没了。句子变短——有时只有三四个字。分行也变得随性——断在让人不舒服的位置。意象也在变——从古体的"月""柳"变成了身边的东西。

"灯"——图书馆的绿台灯，照亮半张桌面。拉绳开关垂下来，末端有个黄色塑料坠子。他偶尔拨一下——坠子晃着，影子在桌面上摆动。

"站台"——师院南门公交站的绿铁牌，301路停靠的那一站。站牌油漆剥落了一块——露出里面银色的铁皮。站台上地砖裂了一道——裂缝里有个烟蒂——叼过的烟蒂已经浸在泥水里变灰了。

"走廊"——教学楼那条长长的走廊——尽头是男厕——门口挂着蓝色门帘，门帘被漂白水洗出白渍。走廊中间的窗台上偶尔会坐着看书的女生——两只脚搭在窗台外面空中。

"抽屉"——宿舍书桌抽屉——里面放着他从高中带来的笔记本。拉开的时候木头和木头摩擦发出涩涩的声音——抽屉边缘有被钥匙刮出的白色划痕。

"操场"——晚上走圈的塑胶跑道——红色，踩上去有回弹。跑道边上排水沟盖板有一块松了——踩上去咯噔响。

"窗户"——窗外远处那些宿舍楼——灰蒙蒙的，某扇窗贴着红色窗花——大概是过年贴的，一直没撕。被太阳晒褪了色——红色变成了淡粉色。

他写了一句——"走廊的尽头是另一条走廊"。写完以后自己看了很久——不知道自己在写什么。走廊的尽头为什么不能是一个房间、一扇门、一片操场。为什么偏偏是另一条走廊。他把笔停住——看着那几个字——图书馆暖气管嗡嗡响——窗外槐树枝在风里晃——影子在桌面移了一寸。

但他没有划掉。他把笔尖悬在纸上，又写了下一句——笔尖在纸上停了一下，墨水洇出个小蓝点——然后手动了——然后写到一半笔芯快没水了——字迹变淡。从笔袋里换了一支新笔芯。旧的放进去——笔袋里被染了一小块蓝色。

他把诗从头到尾看了一遍。字迹有深有浅——因为中途换过笔芯。有些地方改过——划掉的句子旁边补了新的——改了以后又划掉，换了第三种写法。纸面上有几处涂改——有的横线划掉——有的涂成黑方块——有个词被圈起来，打了三个问号——"走廊尽头"旁边的"镜子"被加了一个括号，括号里写着"需要吗"。然后放开笔。甩了甩手腕——咔嗒响。

翻开桌上那本《现代诗歌选读》。海子——"从明天起，做一个幸福的人——喂马，劈柴，周游世界"。他看了几分钟——阳光在书页上移了一小截。合上书。海子的诗太亮了——麦田、阳光、幸福——他的太暗了——走廊、站台、没有拨出去的电话。海子的世界是打开的——他的是锁着的抽屉。

合上笔记本。夹着笔的那一页有点鼓。窗外天快黑了——槐树剪影在深蓝的天上。图书馆的灯越来越亮——绿台灯沿着长条桌一个个亮过去。对面男生收起杂志——斜后方的女生摘下耳机塞进书包。他起身——椅面坐久了有点硬——把《现代诗歌选读》塞回书架I226区第四层——笔记本夹在胳膊下——背上书包——走出图书馆。

外面风冷。他把校服拉链拉到下巴。路灯亮了——橘光洒在台阶上。地砖有一块松了——踩上去晃一下。天是深蓝的——只有一颗星星——很淡。门口有学生结伴出来——讨论着什么——两个人的影子在路灯下一长一短——往不同方向走。他看了几秒。然后转身回了宿舍。
