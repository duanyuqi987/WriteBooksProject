# 第019章 前排和后排旁观

所属部卷：第一部：初秋三楼
时间：高一春天
地点：物理课后的教室

## 诗歌插页 P019：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
作业本边缘的蓝色墨点还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，传作业本这件事本来没有任何心跳可讲。它只是一个每天重复的物理程序：老师说传，全班开始动，前排往后排传或后排往前排传，本子在手和手之间走一条最短路线，最后抵达讲台。有人传本子的时候还在回头看后桌的答案，有人拿到本子先对一下自己的答案再往下传，有人把自己的本子和隔壁组的互换——因为自己没写完。这个过程对全班三十八个人来说都一样，除了一个人——他把它演算成了自己日常中最不可忽视的一个变量。

程砚坐在陆允川旁边，但他不是通过陆允川的表情观察的。程砚有另一套方法——他通过统计。物理课传作业本的那几秒，陆允川写字的速度会突然慢下来。本来每分钟写二十个字的笔记速度会降到零，因为他的手还在动但眼睛没在看纸。程砚甚至记录过具体数据：某次物理课传作业，陆允川的笔尖在纸上停了四秒半，期间写了半个字——那个"解"字的最后一横没写完。这个降速持续的时间和三样东西有关：前前排那个人的座位空了没有、传过来的本子是哪一摞、以及物理老师站的位置会不会挡住他的视线。程砚管这叫"陆氏三变量方程"，说如果能列出来大概可以拿个物理竞赛三等奖。

叶澄曾问过程砚，你怎么就这么肯定他喜欢的是谁。程砚把筷子搁在餐盘边上，想了想才回答："他在食堂看她的次数比看黑板看老师的次数少——少得多。但他看她的那种看法和看别的不一样。看黑板是在看公式，看老师是在看下一步要讲什么，看——那个人是在看'她今天好像心情不太好'。"他顿了顿，"你觉得这是同一个看吗？他看物理公式的时候眉头是皱的，看人的时候眉头不皱，但嘴唇会抿一下。他自己可能都不知道自己抿了嘴。"叶澄回想了一下，觉得程砚说的那个抿嘴动作她似乎也见过，但从来没留意。

叶澄没回答，但她知道程砚是对的。因为她也观察过陆允川，虽然观察距离比程砚远。她发现陆允川在夏知遥回答提问时，手中的笔会自动停下来——不是那种听完答案后核对笔记的停，而是一种完全停滞的静止，像录音机被按了暂停键。等她坐下后他才继续写，而且通常会写错——因为他忘了自己刚才抄到哪个步骤。

何敏第一次注意到陆允川是在一次传英语卷子的时候。那天英语卷子从前往后传，她转身把一叠卷子递给后排时，余光扫到了陆允川的动作。"我把英语卷子传到后排——传到他手里时他看了一眼名字，卷子边角有一个折角，他先把折角按平了，然后把卷子放到了旁边空座上。那个动作很轻，不像我们一般人放卷子随手一丢——我们传卷子都是捏着边角嗖地甩过去，不管正反。他放的时候是把卷子摆正了的，而且还把卷子上的订书钉那侧对齐了桌边。我注意到不是因为我在意他，而是因为他做那个动作的时候我刚好回头催他往下传——我以为他在发呆，结果他在帮别人的卷子整容。"

蒋雯后来补了一句："那你觉得他是温柔还是强迫症？"何敏想了想说："都有可能。但强迫症不会只对一个人的卷子强迫。"两人对视了一眼，然后同时看向夏知遥——她正在改错题，完全没有听她们的对话。她把错题抄在本子上，用红笔在题目旁边画了一个小小的五角星做标记。何敏说："算了，别乱猜。"蒋雯说："你每次都先乱猜然后说算了。"何敏把英语书竖起来挡住脸，从书后面露出半只眼睛看蒋雯："那你觉得我猜对还是猜错？""我觉得你猜对了。""那你刚才还让我别乱猜？""让你别乱猜是因为万一猜错了，我们两个就变成了在背后说人闲话的那种人。"

但她们没有猜错。陆允川确实只把夏知遥的卷子摆正，别人的没有——不是因为他故意区别对待，而是因为他根本没注意到别人的卷子歪了。在他眼里，班上三十八个人，有三十七个人的卷子传到他手里时角度是一样的歪，只有一个人的卷子像是被人不小心弄皱了一下，需要他摆正。他自己都不知道这是偏心，他以为这叫公平。

程砚在班会上被陈老师叫起来回答"你觉得自己在班级里扮演什么角色"，他说："后排监督员。专门监督靠窗那个同学有没有认真抄板书。"全班笑了，陆允川也笑了。

叶澄后来问他为什么总是用笑话来挡。程砚回答："你直接对他说'我知道你在暗恋'，他说不定会把日记本锁进衣柜。但你用笑话挡一下，他至少知道你没觉得他奇怪。他不是怕被知道，他是怕被当成怪人。"叶澄听完没有马上接话，她把桌上的英语单词卡收拢起来，用橡皮筋捆好，才说了一句："所以你那些笑话全是设计好的。""也不全是——有些是真的好笑。但你得分清哪些是给他铺台阶的，哪些是我纯嘴欠。"叶澄难得笑了一下，点了点头，把单词卡放回鞋盒里。
