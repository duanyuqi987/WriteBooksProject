# 第034章 玉兰

时间：2011年3月下旬

地点：江城大学校园

卷次：第二卷：冬天的图书馆

三月中旬，江城大学的玉兰花开了。

第一朵开在图书馆东侧的花圃里。林玉起周三下午从图书馆出来的时候注意到的——白色的大花瓣，肥厚饱满，在没有叶子的枝干上独自开着，像被谁用白纸剪出来然后用胶水粘上去的。

第二天，整排玉兰树都开了。白色的、粉白的、浅紫的，花朵挤在枝头，远远看去像一树鸟——不是麻雀，是白鸽子，停在枯枝上，随时要飞起来的样子。

何南星在周五下午的诗社活动上宣布了一件事。

"下周六，春游。地点是城郊翠湖。早上八点在校门口集合，下午四点回。"

诗社活动在人文楼301。开学以后人数比上学期多了一些，教室里坐了大概三十个人。何南星站在讲台上，拿着一个记事本，上面写着春游的安排——车费每人十五，自带午饭，有相机的人带上相机。

"有谁要去？举手。"

林玉起举手了。他旁边的郭向阳也举手了——郭向阳上学期被他拉来听了一次诗社活动以后，不知道为什么就开始固定来参加，虽然一首诗都没写过。"我来听你们念诗的，"他说，"管饭不？"

"你以为是旅游团啊。"后排有个女生笑了一声。

郭向阳回头看看那个女生——她叫徐雨晴，物理系的，也是被室友拉来的。"物理系就你一个，珍惜。"郭向阳说。

"你也物理系的？"

"计算机。我旁边这位才是我的文科生保护对象。"郭向阳拍了拍林玉起的肩膀。

教室里笑成一片。何南星用笔敲了敲讲台，大家安静下来。"统计完了。二十五个人。下周六早上八点，不要迟到。迟到的自己坐公交车追。"

散会以后，林玉起走在校道上。玉兰花在路灯下颜色发黄，像旧照片里的花。空气里有一种淡淡的甜，分不清是玉兰的味道还是别的东西——可能是草坪刚浇过水，湿泥巴的味道混着青草味。

他在图书馆门口停下来，透过玻璃门往四楼看了一眼。灯亮着。她应该在那里。

他没有上去。不知为什么，今天他不想坐在远处看她。他想在更近的地方——不是在图书馆，是在一个她不知道他在看她、而他能完整地看到她的地方。

这个想法在脑子里闪过以后，他觉得自己有点可笑。他又想起付小川那句话——"这不是暗恋，这是跟自己谈恋爱。"

他承认了。他就是跟自己谈恋爱。但跟自己谈恋爱的人有一样自由——可以随便想象她的生活、她的喜怒、她的下一步。她明天会不会早起、她后天会不会来图书馆、她喜不喜欢春天、她看张爱玲的时候会不会也心里不舒服好几天——这些问题都不用去求证，只要想象就够了。

> 拾起见春暮

回到宿舍，郭向阳还在兴高采烈地说春游的事。"翠湖那边有个农家乐，听说可以钓鱼。我准备带两根钓竿。"

"你是去春游还是去钓鱼？"林玉起问。

"都去。徐雨晴说她喜欢吃烤鱼。"

"你已经打听清楚了？"

"那必须的。"郭向阳把耳机往脖子上一挂，"你以为我是你啊？做什么事都偷偷摸摸的。喜欢就去说，想吃就去钓，想去就去车。有什么好犹豫的。"

林玉起没有说话。他在床边坐下来，从书包里拿出那本软皮笔记本。窗外路灯的光透过窗帘照进来，在纸页上投下一个暗黄的方块。

郭向阳忽然说："对了，你那个女生——叫什么来着——苏安晴——她去不去？"

"她不是诗社的。"

"你怎么不叫她？这是一个绝佳的机会。春游这种事，最容易拉近距离。你们在湖边散个步，念两句诗，她心一软，你就有戏了。"

"她不会去的。"

"为什么？"

因为她不认识诗社的人。因为她是教育学院的。因为她周三下午去图书馆不是为了参加诗社活动。因为他根本不敢开口邀请她——连在图书馆互相点头都要花一整个寒假来铺垫的人，怎么可能突然开口说"下周六我们有个春游你要不要来"？

但林玉起说出口的只有五个字：

"不为什么。"

郭向阳看了他一眼，没有再追问。他拿起耳机重新戴上，屏幕上出现一个射击游戏——枪声从耳机里漏出来，哒哒哒地响。

林玉起躺在床上，盯着上铺的床板。床板上贴着一张海报——一个NBA球星在扣篮，海报的边角已经翘起来了，用透明胶带加固过。

他在想苏安晴下周六会干嘛。她不会去春游——不是因为他不邀请她，而是因为她可能有别的事。她这种人做事很有计划——每件事情都有一个确定的时间和一个确定的地点。临时起意不是她的风格。

但他不知道自己想错了。
