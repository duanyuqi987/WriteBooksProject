# 第021章 操场边的单词之前

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P021：原创现代诗

单词书翻到 distance，
风从跑道那边过来。
他把页角折下，
不是为了记住英文，
是怕自己忘了
那一刻其实什么也没发生。

晚自习前，陆允川常去操场西侧看台背单词。

那里离教学楼不远，铃声一响，跑回教室还来得及。看台水泥被白天晒得发热，傍晚坐上去，热意会隔着校服裤子慢慢传上来。操场上有人跑圈，有人踢球，塑胶跑道被鞋底磨出一种轻微的涩声。

程砚第一次看见他坐在那里，拿着单词书，表情很认真，以为他终于开始用功。结果第二天、第三天，程砚发现他总挑同一个位置，脸朝跑道入口，便明白了。

“你背单词还讲风水？”程砚坐到旁边，手里拿着一根冰棍。

陆允川翻页：“这里安静。”

“安静个头。”程砚咬了一口冰棍，“那边高二打球能把地震出来。”

陆允川没回。他正在背 distance。距离。这个词放在单词书里很短，放在生活里却长得要命。他把音标读了两遍，舌头有点打结。

夏知遥常和林曼从跑道东侧经过。她们不是每天都来，有时要值日，有时被老师留堂，有时直接去食堂。陆允川并不知道规律，却还是提前坐在那里，好像只要自己足够耐心，生活就会赏他一次偶然。

那天她还没有出现。程砚吃完冰棍，把棍子夹在指间：“你这么坐着，不累吗？”

“背单词有什么累的。”

“我是说你这个人。”程砚看着跑道，“你想见她就想见，别非给自己找个学习的名义。名义找多了，人容易骗过自己。”

陆允川把单词书合上又打开。夕阳落在书页上，distance 那一行被照得发亮。他想反驳，却找不到合适的话。

预备铃还有二十分钟，跑道入口终于出现两个女生的身影。陆允川立刻低头，假装认真看书。程砚在旁边叹气，声音不大：“你现在像把整张脸都夹进书缝里。”

“别说话。”陆允川低声说。

他听见脚步声靠近，又从看台前方经过。林曼在说周五听写，夏知遥笑了一下，说自己还没背完。她的声音被风吹得很薄，落到陆允川耳朵里，却像一整页课文。

她们走远后，陆允川才发现自己把 distance 那一页折出了一个很深的三角。那个词他还是没背下来。
