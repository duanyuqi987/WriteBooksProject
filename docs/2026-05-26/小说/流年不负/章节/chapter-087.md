# 第087章 寒假回家

时间：2014年1月

地点：青阳县

卷次：卷五：各自南北

一月中旬。寒假。青阳县。

从楚州坐大巴回青阳——三个半小时。楚州长途汽车站在城西。候车厅不大——十几排蓝色塑料椅，椅子上坐了一半人。有人吃泡面——老坛酸菜的，那股酸味在空气里飘着。有人打盹——头一点一点地往旁边歪。有人抱着蛇皮袋——袋子鼓鼓囊囊的，用麻绳扎了口。他买了票——四十五块。过检票口的时候检票员咬着一根牙签，把票撕了一半还给他。

他上了车。坐在倒数第三排靠窗的位置。座位套是灰蓝色的，上面有几道圆珠笔划的印子。靠背的调节杆坏了，椅背一直往后倒。他用膝盖顶住前面的座位。大巴慢慢开出城区。经过师院南门的时候——那个公交站牌在晨光里是绿色的。301路的站牌。他看了两秒。车没有停。经过市体育馆。经过新华书店。经过人民医院。路线有一段和301路重叠——他认出来了。窗外的中山路口、公园西门——每个站名他都在手机备忘录里记过。他把头靠在玻璃窗上，看着那些站牌一个一个往后退。玻璃是凉的。贴在太阳穴上有点舒服。

高速公路风景单调——灰色的护栏、枯掉的灌木、偶尔的蓝色路牌。路牌上写着"距青阳 85km""距青阳 40km""距青阳 12km"。每看到一块路牌他的身体都会坐直一点。从十二公里到零——他在楚州也用同样的数字丈量过另一段距离。但那段距离他一次都没有走过。

到青阳的时候是中午十二点。大巴拐进汽车站。站外的路还是那条路。路边多了一家奶茶店——"茶の道"，粉色的招牌。高中时候没有奶茶店——那时候只有卖绿豆沙的推车。

从汽车站出来。走回家的路上经过了县城的主街。街上的铺子大部分还是老样子——修鞋摊的大爷还在那里，缝纫铺的大姐换了新招牌，新华书店的门脸没变。冬天的县城灰扑扑的。街面上有鞭炮放过后的红纸屑——大概是前几天有人结婚。红纸屑被风吹到路边的水沟里。

到家的时候是下午三点。他家在机械厂家属院——五层楼房，外墙是灰白色的，有的地方墙皮翘起来了。院子里的晾衣绳上挂着腊肉和腊鱼。他妈在厨房里炖鸡，整个院子都是鸡肉和姜片的味道。他爸在院子里晾腊肉——一块一块的红白相间的肉挂在竹竿上，在冬阳下面有点发亮。竹竿两端的绳子绑在水管和防盗窗之间。腊肉上面盖了一层纱布——防苍蝇。

"回来了。"他妈从厨房探头看了一眼。围裙上沾了面粉。手里还拿着一把菜刀。"瘦了。"

"没瘦。"

"还嘴硬。晚上多吃点。"

他回到自己房间。门上的锁还是那把——铜黄色的小挂锁。房间和四个月前走的时候一模一样——床上的被褥收起来了，只剩一个床垫。桌子上的课本还在——都是高三的教辅，《五年高考三年模拟》、《黄冈密卷》、一本翻烂了的《语文基础知识手册》。窗台上落了一层薄灰。他用手指抹了一下，一条干净的印。灰是灰白色的——大概是水泥粉。窗外能看到院子里的那棵柿子树——冬天叶子掉光了，光秃秃的枝干伸着。树上还挂着两颗干柿子——黑瘪瘪的，没有被鸟吃完。

晚上吃饭——四菜一汤。红烧鸡块、蒜蓉油麦菜、西红柿炒蛋、凉拌黄瓜。汤是紫菜蛋花汤。他妈给他夹了三回菜。"多吃点多吃点，大学食堂能吃到什么。"他爸不怎么说话，只是偶尔夹一筷子，喝一口酒。酒是散装白酒——用塑料壶打回来的。他爸把酒倒在玻璃杯里，一口一口地抿。

"在学校怎么样。"

"还行。"

"成绩呢。"

"班里第三。"

他爸举杯的手停了一下。"第三？不赖。"说完继续喝酒。但嘴角动了一下。

三天以后——腊月二十三，小年。

县城里的年味很浓。主街两边挂了红灯笼——塑料的，里面装着小灯泡。有些灯笼歪了，被风吹得在铁丝上打转。店铺门口贴着"春节大促销"的海报。卖鞭炮的摊子摆在路边——红纸盒堆成小山。有人当街放了一挂小鞭——噼噼啪啪的，硝烟呛得路人侧目。

他去县城街上买东西。他妈让他买对联和福字。老城区有一条主街，两边的店铺挂满了大红灯笼和年货对联。到处是"福"字和"恭喜发财"。街上的人很多——有骑电动车的，有牵着小孩走路的，有三两成群站在路边聊天的。电动车的喇叭声滴滴答答的。小孩手里举着氢气球——喜羊羊、灰太狼、红色的爱心。

他走得很慢。棉袄的拉链拉到下巴。手揣在兜里。经过了新华书店——门口贴了"寒假特惠"的横幅。经过了修鞋摊——老头今天没出摊，马扎空着。经过了那条有老槐树的巷口——他往里看了一眼。巷子里停着一辆旧三轮车。枇杷树的叶子是冬天那种暗绿色。

他继续往前走。

他在一家卖对联的摊子前面停下来。摊子摆在路边——一块塑料布铺在地上，对联一摞一摞地码在上面。摊主是个白头发的老头，坐在一个小马扎上。老头穿了件军绿色的棉袄，领口露出灰色的毛衣。"随便看随便挑——两块钱一副——"他把一摞一摞的对联摊在塑料布上。金粉字在红纸上闪着光，一股油墨的味道。他翻了翻——"春回大地""万事如意""家和万事兴"。翻了几张，没有买。红纸金粉沾了一点在手指上。他用拇指搓了搓。

他正要走。抬头往街对面看了一眼。

然后他不动了。

街对面是一家糕点铺。老招牌——"老李糕点"，木质匾额，字是金漆的。门口摆着炒货摊——瓜子、花生、炒栗子。铁皮炉子里烤着红薯——焦甜的味道飘了半条街。一个穿白色羽绒服的女人站在摊子前面，正在买东西。背对着他。但旁边还有一个人——一个中年女人。中年女人穿着深紫色的棉袄，头发烫了小卷。正在跟摊主讨价还价——"栗子怎么又贵了五毛"——"过年嘛——什么都贵一点嘛。"

白色羽绒服。马尾。侧脸转过来的一瞬间——鼻梁的弧度、下巴的线条。他认出了她的鼻子——挺的，但鼻尖有点翘。也认出了她的下巴——不太尖，是有点圆的。她侧头跟身边的妈妈说了一句话，嘴动了动。他没听见说的是什么。太远了。

是苏晚晴。

她妈在旁边。两人站在糕点铺门口，大概是也在办年货。她穿一件白色羽绒服——短款，领口有一圈褐色的毛。头发扎起来——马尾还是高中那个马尾。皮筋是黑色的。碎头发搭在耳朵前面——大概有两三根。她从摊主手里接过一袋栗子，低头跟身边的人说了句什么。然后母女俩转身，往巷子那边走。

那条巷子——有老槐树的巷子。

她拐进去了。

马尾在拐弯的时候甩了一下。幅度不大——就是她走路时惯常的摆动。跟她高一在走廊里端着水杯走过的样子一模一样。

全程大概十几秒。

他站在街对面。没有动。手不知道什么时候从对联摊上拿起了一张纸——红底金字的"福"字。他攥着那张纸。纸张的边缘把他的掌心硌出一道印子。他的脚钉在地上。街上的声音——电动车的喇叭、小孩的笑、鞭炮的响——都变成了一层嗡嗡的背景音。风声也是。他唯一能接收的信号是街对面——那个穿白色羽绒服的身影在巷子口拐弯，然后消失了。

摊主抬头看了他一眼。"小伙子你要不要——"

他低头看了看手里那张纸。金粉字有点被捏花了。纸上有他手指的汗印——金粉粘在指腹上。他把纸放回去。"不好意思。"他转身走了。

走的不是她那个方向。他绕了另外一条路——从菜市场后面拐过去，多走了大概十分钟。菜市场下午还开着——卖鱼的摊子在往地上泼水，水带着鱼腥味流到路面上。他跨过去。继续走。

回到家。他妈在厨房里炸春卷。油锅里的春卷滋滋冒泡——豆腐干和韭菜碎在油里面翻滚着，香味充满整个院子。春卷皮炸得金黄——他妈的筷子在油锅里翻动春卷，翻得很利落。他经过厨房的时候他爸在院子里劈柴。斧头落下去——木头裂开的声音很脆。劈好的柴码在墙根——整整齐齐的。

"买到对联了吗。"

"忘了。"

"忘了？"他妈从厨房探头。"你这记性。"

他走进房间。把门关上了。坐在床垫上。床垫的弹簧硌着屁股——中间那一块弹簧早就塌了。手心里还有刚才捏对联时沾上的金粉。他用拇指搓了搓指腹——金粉簌簌地掉下来。金粉在手指上像磨碎的星星。

窗外的天空是灰色的。屋檐上蹲着一只麻雀——歪着头，跳了两下，然后飞走了。麻雀飞走的时候蹬了一下瓦——一块碎瓦屑掉在窗台上。

她的白色羽绒服。马尾甩的那一下。拐进巷子的那个侧影。隔着一条街的距离——大概二十米。他隔着二十米，看了她十几秒。没有喊她的名字。她也没有看到他。她拎着一袋栗子，跟着她妈，拐进了那条有老槐树的巷子。好像什么都没有发生。好像那条街上根本没有一个叫顾临风的人在对面对着她发呆。好像这半年来的一切都只是他一个人脑子里的事。

他没有追上去。

也没有想太多。

他把手心里的金粉拍了。站起来，去厨房帮他妈洗菜。水龙头的水很冷——手指尖冻得发红。他把白菜叶子一片一片掰下来。白菜帮子上的泥土要搓好几下才能干净。他妈在旁边切姜——菜刀落在砧板上，当当当。厨房里蒸汽和油烟混在一起。砧板旁边放着那盘春卷皮——还剩半摞。他妈说"再洗一棵"。"嗯"。他又拿了一棵。掰开。水继续流。
