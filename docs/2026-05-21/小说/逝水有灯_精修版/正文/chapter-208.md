# 第208章 毕业照里的空位之后

所属部卷：第四部：失去的美丽
时间：大四六月
地点：北门蜀味轩

## 诗歌插页 P208：原创仿宋小令

松线边，晓风清，
半窗纸页动微灯。
问旧年、几番执念，几番无凭。
人各有程休再问，
水声仍向远天生。

余波没有立刻散开。

毕业聚餐六点开始，到八点半的时候菜已经上完了，酒喝到第三轮。蜀味轩二楼三张大圆桌，坐满了整个专业的人。辅导员在第一桌讲了几句场面话就提前走了，他一走，现场的纪律就松了下来。

陆允川坐在靠窗那桌，和程砚、祁野同一个桌。他面前的啤酒还剩半杯，从七点就剩这么多。

"你不喝了？"祁野问。

"够了。再喝就趴了。"

"你大二那次趴了以后，好像就没见你多喝过。"

"就是那次之后才控制的。"

程砚在对面剥花生，剥了一堆壳，花生米攒在小碟子里。他把碟子推到桌子中间："谁要？"

没有人要。他自己吃了。

"说起来。"程砚嚼着花生，"今天那个合照，你站苏晚枝班上去，后面有人问了几句。"

"问什么？"

"问你是不是苏晚枝男朋友。"

陆允川拿起啤酒杯又放下。"你怎么答的？"

"我说——曾经是，现在不是，以后也不会是。但合个影没什么大不了。"

"你这个回答，把三种可能性都堵死了。"

程砚很认真地看了他一眼。"你需要我留哪种可能性吗？"

"不需要。"

"那不就完了。"

祁野在旁边加了一句："我觉得你今天挺正常的。"

"什么叫正常？"

"就是，合照归合照，没有额外的眼神，没有额外的沉默。拍完就走了。像一个普通人。"

陆允川想了想。"所以你的意思是我以前不正常？"

"以前你太用力了。"祁野用筷子指了指他，"不管是看人还是不看人，都特别用力。今天就不一样——今天你是真的在看镜头。"

"因为那个镜头很贵。摄影师说的。不笑对不起租金。"

祁野笑了。程砚把花生碟推给他。

九点多的时候，有人提议去唱歌。响应的人不少，十几个人站起来穿外套。祁野问陆允川去不去。

"不去了。回去收拾东西。明天家里来车搬。"

"那行。到家了群里说一声。"

"好。"

程砚已经走到楼梯口了，又折回来。"对了，苏晚枝说你给了她一沓稿子。"

"面馆那章。"

"她看了吗？"

"不知道。一个小时前给的。应该还没看。"

程砚停了一下。"如果她看了以后找你，你打算怎么回？"

"不回。等她先说话。"

"有长进。"

"四年下来总得学会点什么。"

程砚拍了拍他肩膀，下楼了。

陆允川一个人走出蜀味轩。北门外面的街道已经安静了，白天的热闹散在路灯底下。有个卖水果的三轮车还没收，车上的草莓剩了小半筐，老板正用保鲜膜一个一个裹好。

他买了一盒，二十块钱。

拎着草莓经过校门口的时候，保安室里的收音机放着晚间新闻，男主播的声音从纱窗透出来，说着和这座校园完全无关的事。

他走到宿舍楼下，发现自己的宿舍亮着灯。他以为是室友没走，上去一看，是隔壁寝室的李哥在收旧书。

"你怎么在这儿？"

"你们宿舍不要的书，辅导员让我整理一下送给下一届。"李哥从床底下拖出两摞教材，"这些你还要不要？"

陆允川看了看。《概率论》《工程力学》《物业管理实务》。"都不要了。"

"这本呢？"李哥从书堆里抽出一本笔记本。不是教材——是手写的，封面贴着一张便利贴，写了"大二下 礼记"。

陆允川接过来翻开。前几页是课堂笔记，字迹潦草。翻到中间，突然出现了一整页的草稿——是一封信的草稿。收信人名字被涂掉了，但信的内容还在：

"...今天在图书馆看见你，坐在靠窗的位置，阳光把你的头发照成一种很淡的棕色。我想了很久要不要过去打招呼，最后还是没去。不是怂，是不知道说什么你不会觉得烦..."

他没有再往下看。合上笔记本，放进自己包里。

"这本我要。"

李哥没追问。"行。还有这些，你看看——"

他们花了半个小时把宿舍的旧物分类。能送人的送人，能卖的卖，不能送不能卖的装进垃圾袋。陆允川的床位下面翻出了很多东西：大一的课程表、过期的学生证、一把已经打不开任何锁的钥匙、一个空了的墨水瓶。

草莓放在桌上，忘了吃。

十点半，李哥走了，拉走了两袋子教材和一堆杂物。陆允川一个人的宿舍突然变大了。他把草莓洗了，坐在空床板上吃。床板没铺褥子，硬邦邦的，坐上去能感觉到木条的间距。

手机上来了消息。不是苏晚枝。是老妈："明天上午十点到。你东西收好了没有？"

"差不多了。"

"别忘了被子。顺便把你床单也带回来，我给你洗。"

"好。"

"还有你那些信——你爸说帮你整理了放在你书柜最上面一层。别乱扔。"

他愣了一下。"什么信？"

"你自己写的那些。高中到现在。你爸没看内容，就按日期排好了。"

"知道了。"

他把手机放下，草莓吃到第六颗就吃不下了。

然后他检查了班级群。下午拍的班级合照已经发出来了，原图，大得能把每个人的毛孔放大。他下载了原图，放大到最大，在像素变成色块之前，找到了苏晚枝的脸。

她站在中间那排，帽穗拨到了右边，笑得正常、得体、不热络也不冷落。是一个知道自己要去哪里的女生的表情。

他把照片缩小回正常大小。没有保存放大后的截图。

窗外有人在收晾晒的被单。风很大，被单鼓起来像一张帆。收被单的人骂了句脏话，把被单夹在腋下，匆匆下了楼。

陆允川把垃圾袋提到楼道尽头的垃圾桶扔掉。回来的时候经过走廊的窗户，能看见图书馆的轮廓。台阶上已经没有人了。彩纸被扫掉，台阶恢复了白色。

他关上宿舍门，把钥匙从钥匙串上取下来。明天要交还给宿管。

那个旧笔记本还在包里。他拿出来，又翻到那页信稿。被涂掉的收信人名字下面，隐隐约约能看见一个"苏"字的左半边偏旁。

他没有再涂。把便利贴也撕掉，在封面写上：大二下 旧稿 勿弃。

然后放回包里，拉好拉链。

宿舍的灯是最后一晚的灯。他关了以后，走廊里的声控灯亮了一下——有人走过，又灭了。

明天家里来车。明天开始，这个地址就不在他的住址栏里了。
