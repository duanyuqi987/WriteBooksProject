# 第194章 月湖边

时间：2022年10月
地点：青阳县月湖
卷次：卷十：不负少年

从一中出来。往南开了二十分钟。月湖到了。

湖还是这个湖。水比往年浅了一些——大概是今年雨水少。湖岸的石头露出来，白白的。上面有水干了之后留下的浅色印子——一圈一圈的。

柳树粗了一圈。树干的皮裂成了老样子——老的裂纹更深了，新的裂纹又加了上来。柳条垂到水面上。风一吹，条子在水面扫来扫去。弄出一圈一圈的小涟漪。

长椅换过了。以前是木头的——刷绿漆的那种，油漆年年剥落，坐上去屁股上会沾一层绿粉末。现在是铁艺的——黑色的，有靠背，能坐三个人。椅面上的铁条在太阳下有一点温。

苏晚晴先坐下去。铁椅子凉了一下她的腿，她并了并腿。

他在她旁边坐下。

女儿被放在地上。脚一落地就跑——往湖边跑。苏晚晴喊"慢点"。没喊住。女儿已经跑到湖边了。蹲下来。用手去摸水。水很凉。手缩了一下。又摸了一下。这次没缩。

湖边有石子。大大小小。圆的、扁的、白的、花的。她挑了一颗——光滑的鹅卵石，有鸡蛋那么大。站起来。把石子往水里扔。

扑通。水花溅起来。溅到了自己的脸上。

她用袖子擦了一下脸。弯腰又捡了一颗。又扔。这次扔得远了一些——大概半米。

"念念。别靠水太近。"苏晚晴说。

女儿往后退了一步。继续捡石子。

顾临风坐在铁椅上。看着女儿。马尾。一晃一晃的。她的马尾还没扎得很高——头发不够长，只能扎在脖子后面。但晃的样子很像——从左到右。从右到左。跟当年他在操场上看到的另一个马尾一样。

湖心亭还在。柱子上的红漆褪成了淡粉色。漆面上有裂纹——干裂的那种纹，一道道密密的像瓷器的开片。亭子顶上有几只鸟。飞走了。

"亭子没变。"

她说。手放在膝盖上。手指无意识地搓着膝盖上的裤子褶皱。

"柱子褪色了。"

"那是红的褪成了粉的。以前是大红的。"

他嗯了一声。

湖对面。能看到那条小石板路。那年高考后他走的那条路。他在路上截住她。说"苏晚晴，我有话跟你说"。她说"啊"。然后他看着自己的脚。看了一会儿。说了三个字。

现在那条路的路面上长了些青苔。边角被草遮住了一些。

"高考后那天——"

她开口了。没有往下说。湖风把柳条吹起来——柳梢擦过水面。水面上的光影碎了一下。又合上。跟十二年前一样。风还是从同一个方向来。柳条还是同一棵柳树的。只是在她的记忆里——那个晚上月亮更亮。人更年轻。话说出口之后还没有任何答案。

但他知道她指的是哪一天。

"2012年6月8号。"

"记得这么清楚。"

"那天晚上我走回家的路上，把那天所有的细节都记住了。走了大概二十分钟的路——走了快一个小时。因为每走到一盏路灯下就停下来——把刚才发生的事情在脑子里回放一遍。再走下一盏。你说'啊'的那种语气——那个'啊'是第三声还是第四声——我想了很久。大概是第三声。带着问号的那种。你接过去那首诗的时候手指碰了一下我的手指——你的手指是凉的。我的手指是热汗。你转身的时候马尾甩了半圈——往左甩的。你走的时候回头看了一眼。"

"我没回头。"

"你回头了。在石板路尽头拐弯的时候。路灯刚好在你站的石板路拐角。光在你身后。你的影子往前倒——倒在我的脚前面。然后你回头了。看了一眼。然后转回去。走进了暗处。"

她愣了一下。手指在铁椅的扶手上停住不划了。

"我真的回头了。"

"回头看了一下。很快。然后就走了。我站在那里看你走的——不是忘了回家。是故意等你影子也走远。直到你的马尾看不见了。然后路灯自动灭了——它是有定时的。然后是下一个路灯也灭了。然后整个月湖只剩水塔的灯。红色的。绿色的。红的。绿的。"

她沉默了。手放在铁椅的扶手上。手指在铁条上一下一下地划着。

"那首诗——你后来怎么处理了。"

"那首？表白的那首。"

"嗯。《月湖》那首。"

"收在词典里。后来抄进了那本蓝色笔记本里面。第二十二首——《月湖》。"

"你写的最后一句是什么。"

"不管去了哪里，以后都要回来看这个湖。"

"这不是诗。这是你说的。"

"对。不是诗。是约定。"

"回来了。"她说。

"嗯。回来了。"

女儿又往水里扔了一颗石子。这颗石子比较小——扁的。贴着水面跳了两下。沉下去了。

她回头。看着他们。脸上全是水珠——分不清是湖水还是汗水。然后跑过来。手里又攥着一颗新的石子。

"爸爸。这个石子是心形的。"

"哪里有心形的石子。"

"你看。这里。这里弯弯的——像心。"

她把手摊开。手心里一个深灰色的鹅卵石。确实有点歪歪的——左边比右边大。但轮廓勉强算是一个心形。

"好看。"

他把石子接过来。放在掌心里看了看。石子被水冲得很光滑。有一些很细的纹理。握在手心里——凉的。慢慢地变温了。

"拿回去。"

"嗯。"女儿把石子要回来。攥在自己手里。然后又跑回湖边继续捡。

苏晚晴靠在椅背上。头仰着。看着天。十月的天空很高。有一些卷云——薄薄的，被风吹成了丝状。一只鸟飞过去了。看不清是燕子还是别的。

"如果有下辈子——"她说。话只说了一半。

他等了十秒。她没往下说。十秒的湖风从柳树那边推过来。吹动了她的碎发。她把碎发别到耳后。这个动作——他不知道还能看多久。也许还有四十年。也许还有五十年。也许下辈子还会看到。

"如果有下辈子什么。"

"没什么。就是想到了——随口说的。"

"你说半句。你以前从来不说话说一半——高中的时候你回答问题都是完整的句子。主谓宾齐全。"

"如果有下辈子——我可能还会选你。"

"可能。"

"嗯。可能。不一定。也许会选一个更会说话的。第一天就走到我面前说'你好我叫xxx'。不是等了三年——站在后排。看了三年。让赵一鸣给你传话。自己一句话不说。"

他沉默了一下。低头看着自己的鞋尖。鞋尖前面有一颗小石子——念念扔的。圆圆的。白色。

"那下辈子我好好练说话。从开学第一天就开始练。早上起来对着镜子说——'你好我叫顾临风'——然后到了学校看到你又说不出来了。又开始了三年的循环。"

"你练了十几年了。还是不太行。"

"再练一辈子。下辈子总行了吧。如果下辈子还不够——那再下辈子。"

她笑了。弯了弯嘴角。这次弯得比平时快了一些。比他这些天见到的任何一次都快。

女儿跑回来了。手里又多了三颗石子。裤腿湿了半截。凉鞋里面灌了水——走路的时候咯吱咯吱响。

苏晚晴弯腰。把女儿的凉鞋脱下来。倒水。倒出来一摊水和小石子。女儿咯咯笑。她自己也笑了。

他把那颗心形的石子拿在手里。看了一会儿。然后放进了口袋里。
