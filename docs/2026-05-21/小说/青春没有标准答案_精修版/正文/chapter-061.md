# 第061章 校门外的公交



时间：2009年一月中旬，雪后周末



地点：封丘一中校门外公交站



卷次：第三卷：雪落在两栋楼之间



2009年一月中旬，雪后周末，封丘一中校门外公交站，车门边化成泥水的雪先把这一天按进了十班的日常里。雪后公交晚点，学生们挤在校门外等车，鞋底都沾着泥水。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

张若楠因为没考好不想回家，常俊凡没有劝她想开，只陪她等下一班车。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看车门边化成泥水的雪，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

张若楠也在这一天被推到前面。张若楠的特点是安静普通，成绩不冒尖，很多时候代表沉默的大多数。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“我不想现在回去。”张若楠说。

“那就等下一班。”常俊凡说。

“车来了也可以不上。”段璟说。

“你们三个说话怎么像在演政治题？”宋乔说。

“路滑，别抢着上。”胡向南说。

校门外的公交站到了冬天会显得更窄。学生、铺盖卷、饭盒和家长的电动车挤在一起，谁都想快一点。

路边小店烤红薯的香气很重，热气从铁桶里冒出来，把排队的人脸烘得忽明忽暗。

常俊凡把书包背带往肩上提了提，没有把手里的袋子递给谁。

段璟站在她旁边半步远的位置，不靠太近，也不故意退开。

宋乔嫌车慢，付小川嫌风大，周明澈则一直看路口，像他看久了公交就会出现。

车来时，人群往前涌，常俊凡先扶住旁边一个差点滑倒的女生，再自己上车。

段璟跟在后面，看见车门边的泥水溅到鞋面上，忽然觉得冬天的现实感很重。

公交开走后，校门口很快空了一半，剩下的人继续等，像每个人都有不同的回家速度。

校门外的公交没有悬在空处。公交站泥水、烤红薯热气和校服袖口，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，校门外的公交带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

胡向南进门时不一定立刻说话，鞋底在水泥地上一停，校门外的公交带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在张若楠快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。校门外的公交越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。车门边化成泥水的雪让他想说话，又逼他先低头看题。

张若楠被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，校门外的公交留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。张若楠路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为校门外的公交翻身，谁还在被窝里背单词，旁边人都听得见。

公交从县城路口慢慢拐过来，车窗上蒙着白气，里面的人影一晃一晃。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理校门外的公交里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：车门边化成泥水的雪、一只冻红的手、一句老师点名、一块还没扫干净的雪。

周五下午放学后，校门外的公交站比平时挤了一倍。住校生回家、通学生赶末班车、家长骑电动车来接，三种人流搅在一起，把窄窄的人行道挤成了漏斗。雪化了三天，路边还有残雪和泥水，人踩上去啪嗒啪嗒地响。

张若楠站在公交站牌旁边，两手插在校服兜里，下巴缩进围巾。她的数学卷子折了两折，塞在书包最外层，考得不好的人习惯把卷子放在最容易忘的位置。

段璟也在等车。他的东西不多——一个书包、一袋换洗的衣服、一个搪瓷缸用毛巾裹了塞在书包侧兜里。他本来想坐第一班车回家的，但看见常俊凡站在站牌另一边，他犹豫了一下。

这一犹豫，第一班车就过去了。

第一班公交来了。排队的人往前涌，常俊凡没有动。张若楠也没有动。她看着车门开了又关上，排在前面的同学一个接一个上车，最后车门关了，车屁股喷出一阵白烟，慢慢拐过路口。

"你不上？"常俊凡问。

"我不想现在回去。"张若楠的声音很小，像在跟自己商量，"我爸问成绩，我不知道怎么说——不是怕骂，是怕他叹那口气。"

常俊凡没有说"下次努力"或者"没关系"。她只是把书包换了个肩膀，站到了张若楠旁边——不是正对面，而是并肩，两个人都面对马路，等下一班车。

"我爸也叹气。"常俊凡说，"但我后来发现，他叹气不是对我失望，是觉得他自己没帮上忙。"

张若楠转头看她。常俊凡没有继续这个话题，但张若楠觉得这句话比十句"别难过"都有用。

烤红薯的香气从路边铁桶里一阵一阵地漫过来。铁桶旁边站着一个穿蓝布围裙的女人，用火钳翻着桶里的红薯，翻到一个裂皮的，就夹出来放在旁边的报纸上晾着。

段璟站在离站牌三步远的地方。他本来要走，但看见常俊凡没上车，他把已经攥在手里的月票又塞回了口袋。

"你也等下一班？"付小川从他旁边经过，手里提着两个塑料袋，一个装着苹果，一个装着饼。

"嗯。"

付小川顺着他的视线往公交站牌那边扫了一眼，看到了常俊凡和张若楠站在一起。他把塑料袋换了个手，拍拍段璟的肩膀："你等的人不一定在等你，但你等的车迟早会来。"

"这话是你自己编的？"

"我妈说的，每天早上催我出门的时候。"付小川笑了一声，拎着塑料袋上了刚来的第二班公交。

宋乔本来已经挤到车门边了，又退下来了。她退下来的姿势很显眼——两手扒着车门框，身子往后一仰，像坐过山车最后一段。司机看了她一眼，她冲司机摆了摆手。

"你怎么也下了？"常俊凡问。

"看见你们俩都不走，我觉得应该有点什么事。"宋乔把手套重新戴上，"什么事？"

"张若楠不想回家。"

"那就不回。"宋乔说，像是在宣布一项班级决议，"咱们去喝碗羊汤，那边巷子口有一家，三块钱一大碗。"

张若楠愣了一下："我——"

"你不用请客，AA。"宋乔打断她，"常俊凡今天早上是不是又少买了一个包子？我替你把那个包子吃回来了——不对，喝回来。"

常俊凡嘴角弯了一下。宋乔的逻辑永远是歪的，但方向永远是对的。

三个女生往巷子口走的时候，段璟还站在站牌旁边。常俊凡走过他旁边，偏了一下头："你不上车？"

"等下一班。"

她没有追问。但她在转身之前多看了他一眼，那个眼神短到只有半秒，但段璟觉得那半秒里写着：我知道你不是在等车。

羊汤店的门脸很小，门楣上挂着棉帘子，掀开帘子热气扑一脸。宋乔要了三碗羊汤，加了一份烧饼。张若楠端着碗喝了一口，热汤从嗓子眼一直暖到胃里。她忽然觉得刚才在站牌下跺脚的那二十分钟没那么长了——不是因为汤热，是因为旁边有人在用自己的方式告诉她不急。

"考试这种东西，"宋乔掰了一块烧饼泡进汤里，"以前我爸跟我说，考得不好就像踩水坑——你越低头看，水越深。不低头的时候，才知道只到脚踝。"

"你爸是老师？"张若楠问。

"我爸是泥瓦匠。"宋乔嚼着烧饼，"但他懂的道理不比当老师的少。"

常俊凡没有参与这个话题。她坐在桌子对面，慢慢喝着汤，偶尔看一下窗外。窗外又走了一班公交，车上下来几个穿校服的人，踩在雪水滩上溅起一片泥。

汤喝到一半时，张若楠从书包里把那张数学卷子掏了出来，摊在桌角。红色的分数印在卷头，旁边是陶建国用红笔写的两个字："加油。"

"他每次都在我卷子上写这两个字。"张若楠说，"我有一次考了五十九，他也写'加油'。后来我考了六十三，他写的还是'加油'。我发现他写的不是分数，是知道你没放弃。"

常俊凡低头看了一眼那张卷子。卷子上的错题旁边订正了好几遍，辅助线画了又擦，擦完再画，纸都快磨破了。

"你没放弃。"常俊凡说，"写'加油'是对的。"

张若楠把卷子折好，这次没有塞进书包最外层，而是夹进了英语书中间。

三个人从羊汤店出来时，天已经暗了大半。公交站牌下又排了一队人，比刚才少了一半。段璟还在站牌旁边，手里的月票已经被汗水浸软了一个角。

"你等了四班车了。"常俊凡走到他旁边。

"我想清楚了一个问题。"段璟说。

"什么？"

"你刚才陪张若楠等车，不是因为她需要有人陪。是因为你觉得——"他停了一下，"一个人的难过如果只放在自己心里，会发霉。放在两个人中间，就能晒到太阳。"

常俊凡看着他，那个眼神比刚才那半秒长了一倍。她没有说"你说得对"，也没有说"你想多了"。她只是说："车来了，你上不上？"

"上。"

他们上了同一班公交。张若楠和宋乔坐在前排，常俊凡和段璟坐在中间偏后，隔着走道。公交车开了，车窗玻璃上蒙着一层白雾，段璟用手指在玻璃上写了一个字，写完发现是"常"的第一个笔画。他赶紧用手掌擦掉了。

常俊凡好像没看见。但她的嘴角在车窗反光里微微弯了一下——那个角度刚好被玻璃上的雾气挡了一半，只露出一小截弧线。段璟不确定那是不是笑，但他决定不擦车窗了，让雾就这么糊着。

公交在封丘县城的路上慢慢晃。经过城关镇时，路边炸油条的摊子已经收了，只剩一口油锅和一个空板凳。经过黄河大堤路口时，风突然变大了，把车窗吹得嗡嗡响。段璟往窗外看了一眼，只看到一排光秃秃的杨树，树梢上还挂着一点没化完的雪。

他把头转回来时，发现常俊凡也正在看窗外。两个人目光在车窗反光里对了一下，然后各自把头转开。

段璟想，坐公交和喜欢一个人，大概共用同一种节奏：该停的站要停，该等的人要等，该拐的弯要拐。急不得，但也不能因为怕坐过站就不上车。

车到城关镇那一站时，常俊凡站起来准备下车。她走过段璟旁边时，书包带不小心蹭到了他的肩膀。她伸手把书包带扶正，低头看了他一眼。

"你这一路没写一个字。"

"在想题。"

"哪道题？"

段璟张了张嘴，没有说出话来。她等了两秒，见他没有回答，嘴角动了一下——那个表情段璟后来想了很久，觉得那是"我知道你在想什么但我不说破"的另一种写法。

常俊凡下车时，段璟从车窗里看到她在站牌旁边停了片刻，抬头看了一眼天。天已经全黑了，但西边还留着一道很窄的暗红色，像灯熄了以后铁丝的余光。她拉了拉围巾，转身走进了城关镇的巷子口。巷口有家文具店还开着灯，灯光照在她校服背面，把两肩之间的褶皱照得很清楚。

段璟把视线收回来。他发现自己又在草稿纸边缘写了半个字，这次是"常"字的上半部分——"小"。他把笔放下，不写了，但也没有涂掉。

坐在他后排的沈曼青翻了一页英语书。她不是刻意在看他，但两个人的视线在车窗反光里碰了一下。沈曼青没说话，只是把英语书往旁边挪了半寸，给他留出了车窗玻璃上更大的反光面积——那里面刚好映着常俊凡消失的方向。

"你在校门口等了四班车，在公交车上写了一笔字，然后只看窗外。"沈曼青说。

"你想说什么？"

"什么都没想说。"沈曼青把英语书翻回去，"我只是在陈述你今天下午的全部行动。"

段璟把草稿纸折好，塞进书包最里面。他忽然想到，沈曼青的"陈述"和常俊凡的"判断"有一个共同点：她们都不替你下结论，但她们让你知道了自己在做什么。这种能力大约是女生宿舍熄灯后交换出来的——男生宿舍熄灯后交换的是段子和埋怨，女生交换的可能是看人的方式。

公交在进县城之前经过了一座桥。桥下是冬天的河，水量少，河床上露出灰白的石头。桥上有人在卖气球，一捆气球绑在自行车后座上，车灯照过来的时候气球亮了一下又暗了。段璟想起小时候来封丘过寒假，外婆带他在这种桥头买过一只红气球，没到晚上就撞在院墙的砖角上炸了。外婆说："破了就破了，反正你也只想要它往上飘。"

他一直不太理解这句话，直到今天，直到他看见常俊凡消失在城关镇巷子口的那一刻，他才隐约摸到一点意思——有些东西你得不到，但你看见它往上飘的那几秒，就是它的全部价值。

八点半，公交到了终点站。段璟提着书包下车时，鞋踩在干燥的水泥路面上——县城这边的雪化得更晚，但路上没有泥，只有一层细细的白灰。三轮车在路灯下排成一排，有个大叔蹲在车轮旁边抽烟，烟头在夜里忽明忽暗。

段璟往家走的路上，经过了封丘一中旁边的文具店。文具店还开着，玻璃柜里摆着笔记本、圆珠笔芯、橡皮和透明文件袋。他站在柜子前看了很久，最后买了一个透明文件袋——和常俊凡桌上那个一模一样。

"换新的？"文具店阿姨认识他，"你上次那个还没用坏吧？"

"这个不是用来装卷子的。"

"那装什么？"

段璟没有回答。他把文件袋夹在胳膊底下走出了店门。冬夜的封丘县城很安静，路灯下偶尔经过一辆电动车，车灯晃过去，把一个细长的影子投在砖墙上。

段璟看着自己的影子，忽然笑了——不是因为高兴，而是因为他终于承认了一件事：他从四班车前等到四班车后，看见她消失在巷子口的那一刻就明白了。但他不急。这个冬天还很长，而且他的立体几何还没做完。

校门外的公交之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

张若楠没有因此变成特别醒目的人，可在段璟眼里，张若楠从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，雪后的公交站让段璟看见许多人安静的难处，不只属于他自己。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
