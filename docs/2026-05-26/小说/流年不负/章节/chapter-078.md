# 第078章 夏天的尾声

时间：2013年8月下旬

地点：青阳县

卷次：卷四：夏夜湖光

八月十四号。赵一鸣走了。

他站在长途汽车站的候车室门口。一个行李箱。一个蛇皮袋。蛇皮袋里是被褥——他妈妈塞进去的。他说太丢人了。他妈说他到广东就知道了——那边不冷但宿舍的床板硬得可以在上面摔盘。

赵一鸣穿了新买的白T恤。领口还没洗过就穿上了——后颈上有一条很小的商家标签。许是太兴奋了，没注意到。顾临风伸手帮他撕了。

"干嘛。"

"标签。"

"帮我扔了。"

顾临风把标签攥在手里。

"广东那边你去多久。"

"半个月。开工半个月，三十号回来。九月一号赶去东江学院。"

"注意安全。"

赵一鸣笑了。"我爸叫我别在外面打架。我说我不是去广东打架的。'你学机械。安分守己的。别跟着人家去混。'这些话我都快背下来了。"他顿了顿。"不过顾临风。我跟你说——"他压低声音。"我在网上搜了——广东那边的厂里。晚上不干活的时候可以去海边玩。深圳的海——大海。不是月湖这种。"

"月湖不是海。"

"对。月湖跟大海比就是鱼缸。"

顾临风没接这句话。他把手放在赵一鸣肩膀上拍了拍。"小心点。"

"哎。你什么时候变得这么啰嗦。"

赵一鸣把蛇皮袋从地上拎起来——拎的时候袋子撞了一下自己的腿。"嗖"地吸了口气——可能沉。然后他站直了。看着顾临风。

"你跟她。怎样了。"

"还行。"

"表白之后她怎么说的。"

"还在——"

"还在想一想？"

"嗯。"

"你们两个。真急死人。"赵一鸣把蛇皮袋换了个手。"一个表白了还要想一想。一个想了八年了还在等。我是不是坐飞机得等你们出结果。我——"

"你不是坐的大巴吗。"

"对。大巴。不是飞机。差点吹大了。"赵一鸣笑了一下。然后笑容慢慢收了。

"顾临风。"

"嗯。"

"我走了。你一个人——没事多出门走走。别老在家写诗。你写的诗我看不懂。但能看出来你很痛苦。"

最后那句话赵一鸣不是用平时的声调说出来的。低了两度。嗓音从嘴边的"吹牛调"掉回到了两个人初次认识时食堂后门抽烟时候的那种真声——高一冬天顾临风站在走廊上看雪。赵一鸣走过来。递了根棒棒糖。

两个人都没再说话。候车室的广播响了。去广东的班车开始检票。赵一鸣踩着广播声往前走。玻璃门推一半他回了头。

"楚州见。"

"九月楚州见。"

"别迟到。"

然后他走进了检票口。顾临风站在候车室的大玻璃墙前。看着那辆蓝白色的大巴往收费站方向开。大巴拐了个弯。排气管响了声。沉沉的。然后消失在南环路的枫杨树影之间。

他在玻璃墙前站了一小会儿。然后走出去。出门的时候把赵一鸣后颈上撕下来的那个标签丢在了门口的垃圾桶里。标签是圆形的。上面打着四个字——"品质保证"。

八月十七号。下午。他和苏晚晴去了新华书店。书店暑假在打折——"教辅买二送一"。但他们不买教辅了。苏晚晴在现当代文学的书架前站了很久。拿了一本张爱玲的《半生缘》。翻了两页。合上了。又拿了一本迟子建的《额尔古纳河右岸》。翻了三页——合上。

"大学读中文系是不是要看很多书。"

"应该是。"

"你大概会喜欢的。"

他把那本《额尔古纳河右岸》从她手里接过来。翻到她刚才翻到的那一页——有个折角。他抚平了折角。把书放回架上。

"我们以后可以一起看。一人看一本。看完交换。"

"那得买两本不一样的。"

"你拿那本——"他往书架左边指了指——《平凡的世界》。三大块砖头并排躺在那儿。

"这太厚了。"

"大学四年够看了。"

她把那本书抽出来——第一部——书的脊背被无数人翻过——书脊上有一条条白色的疲劳痕。她捧着第一部。他什么也没拿。走到收银台的时候他把钱付了——二十一块六。她把钱从钱包里往外拿。他按住了。

"我付。"

她看着他。

"你经常付。奶茶也是你付。冰棍也是你付。你妈给你多少零花钱。"

"不多。但够。"

她把那块六毛塞回钱包。把书装进绿灰色布袋里。布袋口子上的木珠碰着布面——轻轻的——像两个豆子碰在一起。

他们走出书店。书店门口贴了一张通知——"本店将于八月三十一日起维修。预计停业一个月。""一个月。"她读出来。"等你走了它关了。我也走了。走之前我们还可以来一次。"

"嗯。"他说。

两个人往前走。这条街走了很多遍了。走一遍四十分钟。十遍七个钟头。这个夏天他们在这条街上浪费的时间——用任何量词都量不准——小时不够用、日不够用、圈不够用。只能说"夏天"。一个夏天——约等于他们在青阳县街上走过的每一步的总和。

"那个奶茶店的老板换了。"她忽然说。

他转头看奶茶店——还是之前那个招牌。"泡泡奶茶"。玻璃门。门把上挂着"OPEN"的木牌子。透过玻璃可以看到柜台后面的服务员——不是之前那个姑娘了。新的。年纪大一点。

"你怎么知道的。"

"上午路过看到的。之前的老板是个小姑娘——今天是个大姐。"

"小姑娘大概去楚州念书了。"

"也可能是嫁人了。"

他没回答。

走了几步——"原来的老板冲的珍珠奶茶比较甜。新老板冲的——不知道。"

"要不要试试。"

"不试了。"她说。"上次的味道记着就行了。"

他们往回走。经过县城的十字路口——移动营业厅门口充值送的桶装油还没有被取走。一桶塑料壶装的油就在玻璃门前放着——落了些灰。路边卖红薯的推车上插着价格牌——"红薯二元一个""三元两个"——其实是一样的价格。

她说："县城太小了。"

"嗯。"

"走一圈只要一个小时。走多了就没地方去了。"

"你暑假有哪里想去没去成的吗。"

她想了想。"我本来想去看一次青阳河。出了县城沿着公路往西走大概三公里。青阳河上有片滩。我小时候去过。水很浅。但是能看到小鱼。"

"明天。去看。"

"明天？"

"上午去。还不热。骑车去。"

"你会骑车载人吗。"

"会。"他其实不太会。车后座很少带人。平时他自己骑。车是山地变速车——后面有个座位——锈了。

但他不想说不太会。他想带她去看她小时候看过的那条河。

八月十八号。

他们去了青阳河。顾临风骑车载她。他的技术确实不太好。上公路的时候龙头晃了三下。她坐在后面抓住坐垫的下面——没抓他的衣服。后来前轮压过一块碎石——车身剧烈晃了一下。她的手在这时才抓住了他的腰侧。捏着他灰色T恤的一小片。大概两个手指头——拇指和食指——揪着那一小片衣服。T恤的布料很薄。这个细绒的感觉透过那一点点布料传到他的皮肤上。他握稳了。

青阳河果然很浅。滩上的水只没过脚背。小石子在水中一粒一粒圆滚滚的。她把鞋脱了。他坐在一块大石头上。看着她赤脚在河水里走。小鱼在她脚边窜来窜去。水光在她的腿上晃。

"嗯。鱼还在。"

"跟你小时候一样？"

"小了一些。记得以前有很多一寸的——现在好像只有半寸不到了。"

她慢慢走。河滩上只有水的响声。轻的。柔的。河对岸是庄稼地。还没收割。绿色的、矮的——大概是大豆。远处的天际线上有淡蓝的远山。

她走回来。坐在他对面的石头上。她的脚湿淋淋的滴着水珠。她用手把脚上的水珠擦了一下。抬头看他。她眼睛里有河水的亮光。

"谢谢。"

"谢我什么。"

"带我来。还有——"她把毛巾拿起来——从布袋里抽出来半条——擦了脚。

她不说了。

"还有什么。"

"还有这个夏天。"

她把脚伸进鞋里。站起来。在河滩上走了两步。脚后跟还没完全塞进去——踩着后鞋帮。

"走吧。该回去了。"

他们骑车回县城的时候——顾临风的方向比上午稳了一些。过碎石的时候没再晃。后座上的人把脸轻轻靠在他背上。不重。他把背挺直了。骑得比去的时候慢。

头顶的梧桐叶掉下来一片。落在他的车篮子里。叶子背面——黄色的。路边所有的梧桐都开始落了。

夏天快要过去。
