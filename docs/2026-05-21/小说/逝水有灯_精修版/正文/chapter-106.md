# 第106章 封校的周末之前

所属部卷：第二部：六月以后
时间：大一秋天
地点：大学南门和围墙边

## 诗歌插页 P106：原创现代诗

我把围墙草放在黄昏里，
不让它替任何人回答。
风从校园的一侧经过，
带走一句没有说稳的话。
后来我学会把手松开，
让灯照着路，也照着空出来的地方。

封校的消息是周四下午出来的。

陆允川那会儿在图书馆二楼看书。人文社科区，靠窗的桌子，对面坐着一个女生在背英语单词，手指在桌面上一遍一遍划着拼写。手机震了一下——班级群。

"通知：根据疾控要求，本周末校园实行封闭管理。南门、北门、西门全部关闭，只保留东门应急通道。所有学生不得离校，已出校门的请在今晚八点前返回。"

陆允川看完，把手机翻过去扣在桌上。背单词的女生也收到了消息，她看完以后把耳机摘了，靠在椅背上叹了一口气。然后继续背单词。

他之前打算周六去市区一趟。程砚寄了一本书给他——《里尔克诗选》，商务印书馆的旧版，邮局说包裹到了校门口的代收点。封校的消息意味着周六出不去，书得在快递柜里多待两天。

图书馆外面的天色开始变暗。陆允川把正在看的那页折了一个角——一本关于宋词的论文汇编，他借来给诗社的活动做准备。周三苏晚枝在群里发了一条消息："本周五的读诗会改为讨论'词与物的距离'。每个人带一首自己喜欢的词来，讲讲为什么喜欢。别带《水调歌头》，太烂了。"

群里安静了一整天。后来有大二的学长回了一句："那我能带《声声慢》吗？"

苏晚枝回："如果你真的喜欢，不是因为它有名。"

"我是真的喜欢。寻寻觅觅冷冷清清，太像考试周。"

苏晚枝发了一个省略号。

陆允川从图书馆出来时天已经全黑了。路灯下有人跑步，有人推着自行车边走边接电话，有人在宣传栏上贴新的社团海报，浆糊的味道被夜风吹散。南门口保安亭外已经贴了临时通知，白纸黑字，用的仿宋体，大概是从行政楼的打印机直接拿过来的。几个学生围在前面看，有人念出声："本周末实行封闭管理——"

"这算什么，"旁边一个男生说，"我们高中封了两个月。"

"你哪个城市的？"

"武汉边上。"

没人再问了。

陆允川转身往宿舍走。路上遇到叶澄。她刚从实验楼回来，抱着一叠厚厚的讲义，下巴压在讲义顶上走路。

"看到通知了？"她问。

"看到了。周末出不去。"

"我本来约了高中同学在市区吃饭。她非要请我吃新开的那家酸菜鱼。"叶澄调整了一下讲义的抱法，"现在吃不成了。"

"酸菜鱼外卖能送进来吗？"

"不知道。你去东门问问。"

东门是应急通道，门卫拦着一辆面包车在检查通行证。铁门外面站着一排外卖骑手，头盔没摘，手机举在耳朵边，用各种方言对着电话道歉："对不起啊同学，保安不让进——你到门这边来拿行不行？"

陆允川和叶澄站在门内侧看了两分钟。骑手们把外卖举过门缝递进来，学生们伸手接。一袋麻辣烫的汤汁洒了，骑手连声说对不起，那个接外卖的女生说"没事没事，反正汤我也不喝"。

"这个周末应该挺无聊的。"叶澄说。

"是啊。"

"诗社可以搞活动。反正大家都困在校园里。"

"苏学姐可能会有安排。"

"她肯定会。"叶澄笑了一下，"一个周末不出门对别人来说是约束，对她来说是'正好可以多看两本书'。"

陆允川的手机震了。祁野发来一条语音消息。他点开，祁野的声音混着走廊里的回音："你在哪里？宿舍楼下现在摆摊了都。食堂搞了个'封校特供窗口'，说是卖宵夜到十二点。你要不要一起下去？"

陆允川回语音："我刚从图书馆回来。什么特供窗口？"

祁野秒回："就是平时不卖的那些——烤冷面、铁板豆腐、炸鸡柳。平时只在西门小吃街有的东西，现在封校了，食堂自己搞。你快点下来，鸡柳要没了。"

叶澄听见了。"我也要去。铁板豆腐我有兴趣。"

两个人往食堂走。食堂门口果然多了一个临时窗口，是用两张折叠桌拼起来的，桌上铺了一次性塑料桌布，电磁炉放在塑料凳子上。排队的人不少，大多数穿着拖鞋和睡衣，看起来是临时从宿舍里被室友喊下来的。

祁野在队伍中间，一手举着手机，一手护着两盒鸡柳。"慢了。这批最后一份我抢到了。"

"你一个人买了三份？"陆允川问。

"一份我自己，一份给你，一份给徐远。徐远在宿舍打游戏，说腿断了也要吃鸡柳。"

"他没断腿。"

"他说下楼等于断腿。"

叶澄走到铁板豆腐那边排队。那边人少，只有一个女生在前面，正在跟师傅讨价还价："多放点辣椒行不行？"

师傅说："辣椒贵。"

"那我加五毛。"

"五毛不够。一块。"

"成交。"

陆允川站在食堂外面吃鸡柳。油炸的脆皮在牙齿间碎开，里面的鸡肉还烫着。祁野站在他旁边，两根手指捏着一根鸡柳，看着远处南门的方向。

"这周末你打算怎么过？"祁野问。

"看书。诗社可能有活动。"

"苏晚枝组织的？"

"嗯。"

"那我就不意外了。"祁野把鸡柳签子扔进垃圾桶，"我是打算——反正出不去，不如把体育馆利用起来。篮球场周末不关，我约了几个人打全场。"

"打多久？"

"半天。上午九点到中午。"

"你上次不是扭了脚？"

"那是一个月前。好了。"祁野原地跳了两下证明，"你要不要来看？就当放风。"

"可以考虑。"

叶澄端着铁板豆腐过来了，豆腐还在锡纸盒里滋滋响。"你们在聊什么？"

"封校的周末怎么过。"祁野说。

"我打算组织一个宿舍读书会。"叶澄用筷子把豆腐切了一块，"我们宿舍四个人，每个人推荐一本书，互相看。到目前为止只有我一个人推荐了。"

"她们推荐的什么？"陆允川问。

"一个说《霸道总裁的小娇妻》，一个说《从零开始学Python》，还有一个说'我的课本算不算书'。"

祁野笑出声。

陆允川抬头看了看围墙的方向。南门的铁栅栏在夜色里只剩轮廓，围墙根有野草从水泥缝里长出来，秋天了还绿着。保安亭的灯光是暖黄色的，里面有人在喝茶看手机。

他想，这个周末不会太难过。有书、有人、有油炸食物，还有操场可以走。

叶澄忽然说："其实封校也挺好的。平时周末大家各走各的，现在至少都在围墙里面。"

"你这话说得像苏晚枝。"祁野说。

"真的假的？"

"真的。她也说过类似的话。她说'限制有时候是帮助，帮人把选择范围缩小到能决定的尺度'。"

叶澄想了一下。"她真的这么说的？"

"她写在诗社公众号上的。"

"你连诗社公众号都看了？"陆允川转头看祁野。

"我无聊。"祁野把脸别过去看围墙，"晚上躺床上刷公众号。刷完篮球教学刷诗歌鉴赏。每个人都应该全面发展。"

叶澄哈哈大笑。豆腐烫了她的舌头，她用手扇了两下风。

陆允川把鸡柳盒折起来扔进垃圾桶。夜风从宿舍楼之间穿过，带着洗衣房传出的洗衣液味道和远处广播台的晚间音乐。封校前的最后一晚，校园看起来和平时没有什么区别。但从明天起，所有人都只能在围墙里面活动了。

他拿手机给程砚发了一条消息："书可能要下周才能拿到了。周末封校。"

程砚回："不急。书跑不了。你倒是可以趁这个周末写点东西。"

"写什么？"

"写围墙。"
