# 第005章 初秋三楼多年后

所属部卷：第一部：初秋三楼
时间：2010年九月
地点：县一中三楼东侧教室

## 诗歌插页 P005：原创七言小诗

桥落三楼课后深，
一声铃响动衣襟。
少年不解寻常意，
错把微光作远音。

多年后陆允川重写初秋三楼，先把自己写得很委屈，写完又全部删掉。删到最后，页面上只剩下一片法桐的黄叶，以及一个年轻人不太体面的执拗。

很多年后他回想这一天，最先浮上来的不是她的脸，而是那个下午教室里的光影——阳光从窗棱斜切进来，在课桌上切出一道明一道暗的线条，他的座位刚好在暗的那一边。粉笔灰在光柱里飘着，像水里的浮游生物，缓慢而无目的。

他坐在出租屋的小桌前，窗外有车灯慢慢扫过墙面。旧日记本摊开，纸页发脆，里面没有惊天动地的证词，只有几行用力过猛的句子。其中一页用圆珠笔反复描了三遍同一句话：今天她穿的是灰色外套。灰色外套四个字下面画了两道横线，力透纸背。他看了很久，在旁边补了一句：她可能早就不记得自己有一件灰色外套了。

手机相册里还留着几张学校旧照片，是程砚两年前拍的。照片里三楼、操场、礼堂、公告栏，全都被像素磨得发虚。他放大看了看，发现自己当年坐的那个位置在照片里只占了一个像素点，和旁边同学的座位长得一模一样。他忽然笑了一下——原来在上帝的镜头里，他的心事连像素都算不上。

程砚后来在同学群里发过一张聚会照片，七八个人围着火锅，热气模糊了每个人的表情。有人问"当年谁暗恋谁还记得吗"，底下炸出一堆起哄。陆允川没有接话，回了一句"看着都没变"。后来他私下把那张照片放大又缩小，在每个人的脸上找当年的影子，找到了又觉得找得没有意义——变了就是变了，照片只是替大家保留一点旧样子，不是替大家撒谎。

叶澄在微信里私下问过他一次："你还真的要写那段？"

陆允川说："要写。但不是写成控诉。"

"那就让她也像个人，"叶澄回得很快，像这句话已经等了他很久，"不要只像你青春里的答案。"

这句话让他停了很久。隔天他翻出旧日记重读，发现他把夏知遥写得太薄了——她在他笔下只是一个名字、一个背影、一个让他心跳加速的符号。他从来没有写过她翻错课本页码时皱眉头的样子，没有写过她体育课跑八百米到第三圈时双手撑着膝盖喘气的狼狈，没有写过她某节课上忘带笔、找前后桌借了一圈没借到的窘迫。她在他十七岁的叙事里被压成了一页纸，她不值得被这样对待。

于是他补回了她的日常。补回她的课表、她的朋友、她的沉默和她的选择。也补回了她没有回应的权利——回应从来不是一个被喜欢的人的义务，而是她可以选择说"不"的自由。

他终于承认，喜欢不是法庭，青春也不是判决书。那些没得到回应的部分，可以难过，可以遗憾，但不能被写成对方的罪。

修改稿写完后，他下楼走了一圈。便利店门口的灯照着湿漉漉的地面，刚下过雨，空气里有泥土和沥青混在一起的气味。他买了一瓶矿泉水，站在店门口喝了两口，看见一个初中生模样的男孩在马路对面一边骑自行车一边回头看什么人，差点撞上电线杆。

陆允川忍不住笑了一声。笑完又觉得这个笑的位置很对——不是在嘲笑，而是在"我也干过这种事"。

合上日记本是之后的事。那天晚上他把日记本放回抽屉最里面，压在大学课本下面。日记本还在，但他不再每天翻它了。它在抽屉里慢慢发黄，像一片被夹进书里的叶子，从秋天变成标本。

他在修改稿旁边写下四个字：不要审判。写完又补了两个字：也别粉饰。温柔现实不是把疼痛擦干净，而是让疼痛和饭菜味、雨水声、普通人的疲惫放在同一个世界里。

多年后最难写的不是遗憾，而是分寸。写少了像逃避，写多了像控诉。他试着把句子放轻一些，不是为了显得大度，而是因为重要的东西确实经不起太用力的抓——就像那片黄叶，你捏紧了它就碎了，你只能让它停在原来的位置。

他没有再问如果当初怎样。因为问题问得太久，人会忘记继续生活。那些被反复回想的几秒钟，在真实的十年跨度里只是一粒沙子——不是因为它不重要，而是因为十年太长了，长到一个人必须学会把沙子放回河床。

今年过年回家，他路过县一中。大门换了，教学楼重新刷了外墙，三楼东侧教室的窗台上多了一排绿萝。他站在门口往上看，阳光照在三楼的窗玻璃上，刺得他眯起眼。他忽然想起自己当年在那个窗口看了很久的那片叶子——它应该早就不在了，落下去，烂在泥土里，变成了别的什么。

他把手插进大衣口袋，转身往车站走。路过校门口的小卖部时买了一包饼干，老板娘找零时多找了一块钱，他退回去了，老板娘说谢谢，他说没事。这个对话很普通，比他青春里任何一段对话都普通，但它让他想起叶澄说的那句话——让她像个人，不要只像你青春里的答案。

是啊。她像个人，他也像个人。两个普通人，在同一间教室里坐过一个学期，各自过着各自的生活。其中一个曾经心动得要命，后来把心动写进了日记本，再后来把日记本放进了抽屉最里面，压在一叠旧课本下面。

风吹过来，法桐的叶子在枝头摇晃。他看了一眼，没有数它什么时候落。
