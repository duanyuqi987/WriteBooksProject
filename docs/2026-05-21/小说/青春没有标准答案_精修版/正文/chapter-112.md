# 第112章 饭卡余额



时间：2010年六月底，期末复习周



地点：食堂充值窗口、教室后墙



卷次：第五卷：未来在黑板上倒计时



2010年六月底，期末复习周，食堂充值窗口、教室后墙，陈柏言记在本子上的饭卡支出表把2010年的春夏推到十班面前。陈柏言把一周饭卡支出列成表，发现期末复习越紧，大家越容易乱花钱买水和零食。

王子越笑他小题大做，陈柏言却说家里每周给多少钱都是有数的。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

陈柏言也在这段日子里显出自己的位置。笔记整齐，日期天气都记，擅长把混乱事理出顺序。群像不是名单，群像是每个人都有自己的难处。

“这周我多花了六块五。”陈柏言说。

“你连饭卡都做统计？”王子越说。

“不统计就不知道钱去哪儿了。”陈柏言说。

“这和错题一样，先知道漏在哪儿。”常俊凡说。

“会管钱也是学习，别只会管分。”陶建国说。

食堂窗口前总有人把饭卡拍得很响，像那样余额就会多出来一点。

充值处的队伍贴着墙排，前面是打菜的热气，后面是学生翻找零钱的声音。

饭卡上的数字很小，却比成绩还具体。少一块钱，窗口就会停一下，后面的人也会跟着看过来。

常俊凡不喜欢在这种时候追问。她知道有些窘迫越解释越难堪。

段璟端着餐盘站在旁边，第一次觉得自己平时说不出口的自尊，其实别人也有。

食堂师傅把勺子在盆边敲了敲，提醒后面的人快一点，队伍又慢慢往前挪。

午饭并不浪漫，白菜、豆腐、米饭和饭卡余额，却能把人的日子照得很清楚。

吃完饭回教室时，后墙的错题区还在那里，像另一种账本。

陶建国这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

陈柏言被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

六月底的期末复习周，食堂里排队的气温比外面还高。窗口上面吊着的三叶电风扇转了一整个中午，吹出来的只有热风。陈柏言端着餐盘从队伍里挤出来，没有立刻找座位，而是先在一张空桌上放下盘子，从裤兜里掏出一个巴掌大的记事本。

本子的封面是牛皮色的，扉页上工工整整地写着日期和天气。他翻到最新一页，上面是一个手绘的表格，分六列：日期、早饭、午饭、晚饭、零食/水、余额。每一行都用铅笔写得很小，数字后面都带到了"角"——比如"午饭 白菜豆腐1.0+米饭0.5+西红柿炒蛋1.2=2.7"。他站在食堂过道里，趁着排队打汤的间隙把这顿饭的账补上。

王子越端着餐盘从后面过来，从他肩膀后面瞄了一眼。"陈柏言你连饭卡都做统计？"

"这周我多花了六块五。"陈柏言头也不抬，写完最后一个数字才合上本子，"不统计就不知道钱去哪儿了。"

"六块五，就是多了几瓶水吧。有什么好记的。"

"不是水。"陈柏言翻开前一页给他看，"周一买了一块五的笔芯。周三又买了一块钱的饮料。周四帮人垫了一块钱的公交费，还没收回来。周五下午觉得嘴馋，在校门口树莓摊买了两盒树莓。六块五就是这么花掉的。"

王子越从头看到尾，沉默了一下，"你不是在记饭卡账，你是在记账本。"

"一样的。饭卡里的钱也是家里的钱。"陈柏言端起盘子找座位，"我家每周给我定量的生活费，花到多少、怎么花的，回去我妈会问。如果不记账，到时候说不清楚。"

两个人坐下来以后，王子越没再笑他。他低头吃了一会儿饭，忽然问了一句："那你觉得记账有用吗？"

"有。"陈柏言夹了一口菜，"记了以后才知道哪些是该花的，哪些是不该花的。该花的——比如笔芯、资料费、给朋友垫的钱出了急。不该花的——比如树莓、零食、买水的时候懒了一下没去水房。"

"可是树莓也就一块钱。"

"一块钱加一块钱就多了。"陈柏言的筷子停了，"我家不是穷到吃不起树莓，但每一块钱都有出处。花掉一块就是少一块。"

这顿饭吃完，陈柏言把饭卡余额记在最后一行：本周余额二十三块八。他在数字旁边画了一个小括号，里面写"够吃到周五，周六充五十"。

这件事本来不会引起任何关注——直到常俊凡在周三中午看到了陈柏言的记事本。

她当时正在帮陈柏言看一道化学题的错因，笔尖点在错题本上的时候，不小心碰倒了陈柏言压在课本底下的记账本。本子翻开的那一页正好是本周的每日汇总：周一花了八块三，周二花了七块六，周三预计控制在六块以内。数字旁边画了不同颜色的记号——绿色圈表示合理消费，红色圈表示不该花的，蓝色圈表示帮别人垫的。

常俊凡看了几秒。"你连帮别人垫钱都记？"

"记了才清楚。"陈柏言把本子扶正，"王子越上周借了我一块五还没还，这周又问我借了两块。我要是没记，就得一直心里算。"

常俊凡没有评价，只是把他的账本又翻了一页，看见了更早几周的记录。每一周都有汇总和分析——哪个时间段花钱快，哪种消费最容易超预算，哪个人的借款最常拖延。她看完以后说了一句："这和错题一样，先知道漏在哪儿。"

陈柏言愣了一下，然后点头，"对。先知道漏在哪儿。"

大课间，陶建国巡逻到教室后面的时候，无意中看到了陈柏言桌上摊开的记账本。他拿起来翻了翻，从头看到尾，没有批评，也没有表扬。他只是把本子放回去，说了一句："会管钱也是学习，别只会管分。"

陈柏言没有回应。但旁边耳朵尖的付小川已经听到了，他等陶建国走远以后凑过来，"陈柏言你这下牛了，班主任夸你。"

"他没夸。"陈柏言很清醒，"他只是说别只顾管钱忘了学习。意思是两件事都要做。"

"那不是更牛？班主任觉得你会管理。"

"别说了。再说就假了。"

下午的复习课上，后墙的错题区又被更新了一遍。常俊凡把自己和陈柏言的错题分类法并排贴在了一起——她的错题按类型分，陈柏言的消费按用途分。两个表格放在一起看，有一种奇特的对称感：左边是知识的漏洞，右边是钱的漏洞。虽然不能直接对比，但它们都是高二下半场最诚实的两本账。

放学以后，陈柏言去食堂充值窗口又充了五十块钱。充值的阿姨看了一眼他手里的小本子，说了一句"你每回来都记"。陈柏言笑了一下，在账本上写下新余额，合上本子，背起书包走出食堂。

六月的晚风把校门口树莓摊上的泡沫箱吹得轻轻动了动。摊主老婶正在看手机上玉树灾后重建的新闻——屏幕很小，字更小，她眯着眼一行一行往下翻。陈柏言路过的时候停了一下，想了想，没有买树莓。他在账本上写下今天最后一笔：路过树莓摊，控制住了，省一块五。写完以后他抬头看了看树莓摊的灯——一串五颜六色的LED小灯泡闪了一下，灭了一颗。

晚自习前，陈柏言翻出了自己最近两个月的账本。从五月到现在，每周的消费记录摊在桌面上，像一份个人财务年报。他用铅笔在每一周的末尾画了小箭头，箭头指向当周的分类汇总。五月份最大的一笔支出是补课费——一次交了四次课的，八十块。六月份最大的一笔是竞赛资料——花了十二块在新华书店买了三本物理专题练习册。除此之外，最大的窟窿是"帮人垫钱"那一栏：王子越借了一块五没还，付小川借了两块还了一块五，赵远航欠了八毛的公交费。三个人加起来欠了四块三。

他把"欠款"这一行用红笔圈出来，在旁边写了一行字：四块三，够充一次饭卡了。

段璟从旁边看到了这行字，放下笔，"你记这个干嘛？记了又不一定收得回来。"

"不是一定要收回来。"陈柏言说，"是要记住谁什么时候替我垫过钱，我也得还。不是只记别人欠我，也得记我欠别人。"他把账本翻到前面一页，指着一行："五月二十号，宋乔替我刷了一顿饭，三块。第二天还了。"再翻一页："六月五号，段璟公交费不够，我垫了五毛。第二天他还了。"

段璟看了一眼，发现自己的名字和还款日期确实被工整地记在上面。"你记性好到这种程度？"

"不是记性好。是写下来了。写下来就不用记了。"

段璟沉默了一会儿，从自己笔袋里拿出一个小笔记本——原来是一本物理草稿本，后面还有十几页空白。他翻到最后一页，用铅笔在页眉上写了四个字：支出记录。然后在第一行写了今天的日期和一句话：饭卡充五十。下面空了一行，又写：明天开始记。

常俊凡从后排走过来借橡皮，看到段璟也在写字，扫了一眼。她没说话，只是把自己的饭卡从笔袋里抽出来给他看——卡面上用铅笔写了四个数字：余额预估。底下是一道很细的横线，横线下面写着：六月结余够。

段璟看了看那行字，"你不是数学好吗，为什么还要预估？"

"数学好不代表不算。"常俊凡把饭卡收回笔袋，"算清楚了心里稳。不算的时候觉得够，月底一对账就发现不够。"

"那你够了吗？"

"够。"常俊凡说，"够到考试结束。等暑假补课的时候再充。"

段璟在自己的新账本第一行写下：饭卡余额，够到七月。他看着那几个字，忽然觉得把数字写下来，比在心里默算要踏实得多。数字在心里会变，会变成"差不多够""可能不够""管他呢不够再说"——但写在纸上，就只有一个确切的数字。那个数字不会骗人，也不会安慰你，它只是告诉你真实的情况。

周四中午，陈柏言的账本在班里引起了一场微型辩论。起因是王子越看到了陈柏言记的"欠款"专栏——上面写着自己欠了一块五没还。王子越觉得被公开处刑了，脸上挂不住，说你记这个干嘛又不是不还。陈柏言说不是针对你，是对所有借出去的钱都一样记。付小川在旁边火上浇油，说那你记不记你借别人的？陈柏言翻到前一页，指着自己欠付小川一块二的记录——日期、金额、用途（公交费）、还款日期（第二天已还），一笔一笔清清楚楚。

王子越看完以后，沉默了几秒，从裤兜里掏出两块钱，拍在陈柏言桌上。"一块五是本金，五毛是利息。"

"不收利息。"陈柏言只拿了一块五，把另外五毛推回去。"借钱就是借钱，还本金就行。"

"那你这账本以后准备记多久？"付小川问。

"记到不用记为止。"陈柏言合上本子，"或者记到我每一分钱都知道花在哪了为止。"

"那会不会太累了？"

"记清楚了才不累。"陈柏言说，"记清楚以后花钱不用犹豫。对就对，不对就不对。"

周五下午的班会课上，陶建国用了五分钟讲"财务管理"。他说你们现在花的每一分钱都是家里的，学会记账不是为了省钱，是为了知道钱的价值。"将来你们自己挣钱的时候，回过头看现在的账本，会对父母有不一样的理解。"讲完以后他看了一眼陈柏言，没有点名，但全班都知道他在说谁。

陈柏言低着头，铅笔在账本上继续写。六月的最后一行：本周总支出三十一块五，其中必需支出二十七块二，非必需支出四块三（树莓两盒、饮料一瓶）。下周目标：非必需支出控制在三块以内。他写完以后抬头看了看窗外——六月底的太阳正毒，操场上迎世博的横幅被晒得褪了色，但"Better City, Better Life"的英文还看得清。他想，Better Budget, Better Life。然后在账本扉页用很小的字把这句自己编的话写了下来。

周六中午在食堂，余小满端着餐盘在陈柏言对面坐下。她的餐盘里只有一份白菜豆腐和一碗免费汤——菜是一块钱的，米饭五毛，总价一块五。陈柏言扫了一眼她的餐盘，没有说什么，但铅笔在账本的空白处下意识地算了一个数字。

"你看我盘子干嘛？"余小满注意到了他的目光。

"没干嘛。"陈柏言把账本合上，"我在算你的午饭多少钱。"

"一块五。"余小满说得很干脆，"白菜豆腐一块，米饭五毛，汤免费。"

"够吃吗？"

"够。"余小满夹了一口菜，"我家给的生活费不多，但我算过——一天三顿控制在四块五以内，一周三十一块五，刚好够。剩下的钱还能攒起来买笔芯和练习册。"她看了一眼陈柏言的账本，"你也记账？"

"记。但我是记花到哪了，不是提前算。"

"那不一样。"余小满摇摇头，"提前算更省钱。先把一周的钱分成七份，每份多少心里有数。这样到了周五不会没钱。"

陈柏言把她的话在账本空白处记了下来：余小满——预算制，每周先分钱。他写完以后抬头说："你这方法比我的好。我是事后记，你是事前算。"

"两个都要。"余小满说，"事前算大的，事后记细的。"她把汤喝完，站起来端着空碗去洗了。陈柏言看着她的背影，在账本上又加了一行：提前预算+事后记账=钱永远够用。

走出食堂的时候，付小川从后面拍了陈柏言一下。"你那个账本能不能借我看看？我妈总说我零花钱不知道花哪了，我想学一下。"

陈柏言把账本递给他。付小川翻了翻，看到"非必需支出"那一栏的时候停住了。"你管树莓叫非必需？"

"树莓是嘴馋。嘴馋不是必需。"

"那什么是必需？"

"饭、菜、笔芯、资料费、公交费。"陈柏言一样一样数过来，"能让你活着和学习的东西。其他的都是非必需。"

付小川想了想，从自己口袋里掏出三块钱——他本来打算下午去买冰棍和树莓。"那我把这三块钱记上，今天开始试。"他在陈柏言的账本最后一页写了自己的名字和日期：付小川，六月最后一周，第三天。

陈柏言把账本要回来的时候，看到付小川歪歪扭扭的字，忍不住笑了。他自己当初写的第一行字也这么歪。但歪没关系，开始了才是最重要的。

饭卡余额之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

饭卡余额和错题墙一样，都是高二下半场最诚实的账。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，陈柏言记在本子上的饭卡支出表还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
