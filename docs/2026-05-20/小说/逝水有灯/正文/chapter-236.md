# 第236章 同学群的婚讯之前

所属部卷：第五部：逝水有灯
时间：工作日午后
地点：公司茶水间

## 诗歌插页 P236：原创现代诗

咖啡粉浮在水面
群消息浮在屏幕
成年人的午后很短
短到来不及
把旧名字想得太长

公司下午三点有评审会。陆允川提前十分钟去茶水间冲咖啡，纸杯薄，热水一倒进去就有些烫手。咖啡粉没完全化开，浮在水面，像一层细小的暗色。

旧同学群就在这时亮了。这个群平时很安静，只有节日祝福和偶尔的投票链接。今天消息一条接一条往上跳，有人发红包，有人说恭喜，有人喊着改天回县城聚。

陆允川把纸杯放在台面上，点开群。最上面是一张婚礼请柬截图，新郎是高中同学董向南，新娘不是他们班的人。请柬背景很红，地点在县城一家酒店。群里有人打趣，说当年坐最后一排的人居然最先结婚。

他看着那些名字，有几秒没反应过来。高中同学已经分散到很多地方，有人读研，有人考编，有人换了好几份工作。婚讯把他们临时拉回同一个屏幕，却也只是一小会儿。

会议提醒弹出来，挡住半个聊天窗口。陆允川按掉提醒，咖啡已经凉了一点。他还没有回复，只是把纸杯端起来，发现底部仍有没化开的粉。
