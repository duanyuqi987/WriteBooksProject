# 第030章 第一次说话多年后

所属部卷：第一部：初秋三楼
时间：高一末
地点：楼梯转角

## 诗歌插页 P030：原创七言小诗

水落三楼课后深，
一声铃响动衣襟。
少年不解寻常意，
错把微光作远音。

多年后陆允川重写第一次说话，先把自己写得很委屈，写完又全部删掉。删到最后，页面上只剩笔记本封面上一道浅浅的折痕，以及一个年轻人不太体面的执拗。

很多年后他写到第一次说话，删掉了所有夸张的心跳。真正留下来的只有她没有翻开本子这一点。那是一种边界，也是一种很普通的善意。

他坐在出租屋的小桌前，窗外有车灯慢慢扫过墙面。旧日记本摊开，纸页发脆，里面没有惊天动地的证词，只有几行用力过猛的句子。

他在旁边补写：夏知遥捡到陆允川掉下的笔记本，递给他时只说了一句你的本子。这句话写出来以后，比记忆本身更平静。平静不是不疼，而是终于不再要求那一年替自己负责。

手机相册里还留着几张学校旧照片。三楼、操场、礼堂、公告栏，全都被像素磨得发虚。陆允川放大看了看，又缩回去，觉得自己不必再从模糊里找答案。

楼梯间的灯坏了一盏，光从上一层斜下来，人影被拉得很长。有人抱着篮球从旁边挤过去，橡胶味和汗味混在一起，把那句谢谢衬得更加短。

程砚后来在同学群里发过一张聚会照片，笑说谁还记得当年谁暗恋谁。陆允川没有接这个话，只回了一个“看着都没变”，其实每个人都变了，只是照片替大家保留了一点旧样子。

叶澄私下问他，真的还要写这段吗？陆允川说要写，但不会再写成控诉。她回了句，那就让她也像个人，不要只像你青春里的答案。

这句话让他停了很久。于是他给夏知遥补回了日常：补回她的课本、朋友、沉默和选择，也补回她没有回应的权利。

他终于承认，喜欢不是法庭，青春也不是判决书。那些没得到回应的部分，可以难过，可以遗憾，却不能被写成对方的罪。

章节末尾，他合上日记本，去楼下买了一瓶水。便利店门口的灯照着湿漉漉的地面，风很轻。他忽然觉得，过去没有被洗白，也没有被毁掉，只是安静地退回了过去的位置。

那一摞书其实不重，重的是他下楼时总想快一点。前面有人堵住楼梯，他停得太急，最上面的笔记本便从臂弯里滑出去，像替他做了一件他一直不敢做的事。

夏知遥递本子时，指尖没有碰到他的手。陆允川却仍然僵了一下，僵得程砚在背后差点笑出声。她说完“下楼慢一点”，就侧身让过一群往上跑的男生。

回到座位后，陆允川把“谢谢”两个字在草稿纸上写了一遍，又觉得自己傻，揉成团塞进桌斗。纸团滚到最里面，和几张旧小测挤在一起。

叶澄晚自习后问他，本子没摔坏吧。她问得自然，陆允川却答得很慢。程砚在旁边替他说，没坏，人差点坏了。三个人都笑了，笑声很轻。

他在修改稿旁边写下四个字：不要审判。写完又补了两个字：也别粉饰。温柔现实不是把疼痛擦干净，而是让疼痛和饭菜味、雨水声、普通人的疲惫放在同一个世界里。

多年后最难写的不是遗憾，而是分寸。写少了像逃避，写多了像控诉。陆允川试着把句子放轻一点，不是因为事情不重要，而是因为重要的东西经不起太用力的抓。

多年后他把这一段重写时，特意保留了楼梯扶手和笔记本。这些东西没有抒情能力，却能证明那一年不是一团雾，而是一间有桌椅、有灰尘、有铃声的地方。

他没有再问如果当时怎样，因为问题问得太久，人会忘记继续生活。于是他把句子收住，让笔记本留在原处，也让夏知遥从他的解释里走出去。

如果只写疼，这一章会很薄；加上楼梯扶手、笔记本、练习册、坏掉的灯，它才重新有了重量。那重量不是压人，而是让读者相信他们确实在这里生活过。

陆允川把笔记本拿在手里，先把笔记本夹回书堆，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把书抱稳，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把本子递回来时没有翻开里面的字，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

楼梯转角里最明显的不是心跳，而是脚步撞在水泥楼梯上的回声。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

晚饭后，塑料袋里晃动的酸奶把人从心事里拽出来。程砚说再大的事也得先回教室，陆允川拎着袋子，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。笔记本、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

多年后他重写这一段，先删掉审判，再删掉自怜。留下笔记本，留下楼梯转角，也留下那个还不会把喜欢放轻的自己。

他终于知道，温柔不是替过去辩解，而是承认每个人都只拥有当时的能力。能写清这一点，比把谁写成坏人困难得多。

这一段真正留下的不是一句谢谢有多重，而是本子被捡起、书重新抱稳、楼梯重新通行。生活没有为他停住，却给了他一次学会分寸的机会。

如果只写疼，这章会很薄；加上笔记本、脚步撞在水泥楼梯上的回声和晚饭后塑料袋里晃动的酸奶，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在楼梯转角发呆。”陆允川把笔记本收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

所以第一次说话多年后留下来的，不是“如果当初”，而是“原来如此”。原来人会把小事看得很大，也会在很久以后学会把它放回小事里。
