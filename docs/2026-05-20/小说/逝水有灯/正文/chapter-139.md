# 第139章 叶澄的劝告旁观

所属部卷：第三部：寒风里的花
时间：六月午后
地点：南湖凉亭

## 诗歌插页 P139：原创现代诗

她不是站在高处的人，
她也有没交的表格，
有改不完的主持词，
有一下午快要溢出的烦。
清醒不是天生，
是她也被误会过。

叶澄不喜欢劝人。劝人很费力，还容易被误会成站队。她更愿意把活动流程排清楚，把社刊错字改完，把每个人的分工写在表格里。

可苏晚枝那天说“我不知道怎么说才不伤人”时，叶澄知道自己不能装没听见。

她见过很多类似的关系：一个人很认真，另一个人很为难；认真者觉得自己委屈，为难者觉得自己被推着走。再拖下去，两个人都会变得不好看。

叶澄找陆允川，不是为了替苏晚枝出头。她也不想把陆允川说成坏人。他帮忙时很可靠，写稿也能听意见。可正因为他不是坏人，才更需要有人早点提醒他，善意越界时也会压人。

谈话前，她在凉亭外站了一会儿，先把自己的火气压下去。她不想用一句漂亮话解决问题，也不想把他逼到防御里。

所以她说得慢，说得平。每一句都尽量落在事实上：连续电话，突然出现，脸色变化。她没有说“你太可怕”，也没有说“你没救了”。她只是告诉他，这些动作会让人累。

说完后，她抱着文件夹离开，手腕被纸边勒出一道红痕。清醒也要付代价。只是有些话如果没人说，后来会更疼。
