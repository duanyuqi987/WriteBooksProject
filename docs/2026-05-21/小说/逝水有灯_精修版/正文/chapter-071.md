# 第071章 网吧通宵之前

所属部卷：第二部：六月以后
时间：高考后的夜晚
地点：县城老网吧

## 诗歌插页 P071：原创现代诗

我把窗放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
键盘缝里的烟灰没有替谁回答，
只把那一年轻轻照亮。

傍晚的时候，程砚打来电话。

"晚上通宵，去不去？"

"去哪？"

"老地方。二楼那个网吧。周长海也去，还有隔壁班两个。"

陆允川正蹲在东屋地上整理试卷。纸箱已经快满了，胶带还没封。他看了看窗外——天光正在变暗，院子里那棵柿子树的影子拉得很长。

"几点？"他问。

"九点。你吃完饭过来。"程砚那边传来他妈喊他吃饭的声音，程砚捂着话筒回了一句"知道了"，然后又对陆允川说，"记得带会员卡。你上次那张还有钱。"

"多少？"

"好像剩了八块。"

"那不够通宵。"

"我有。我卡里还有三十。"

陆允川把自己的网吧会员卡从书包夹层里翻出来。卡面已经磨花了，上面的字——"极速网吧"——褪了一半。他想起高一第一次跟程砚去网吧，那时候还要偷偷摸摸的，怕被班主任抓到。现在高考完了，没有人抓了，反而觉得少了点什么。

晚饭是母亲做的炒面。她把昨晚剩的饺子馅和面条一起炒了，放了辣椒，油放得比较多——陆允川说过好几次她炒菜油太少，今天她大概特意多放了一点。

"晚上去同学家？"母亲问。

"嗯。程砚叫通宵。"

"又要通宵。"母亲皱了皱眉，但语气没有反对的意思，"明天不是还要回学校领材料吗？"

"下午才领。"

"那你自己注意时间。"母亲把炒面往他碗里拨了一些，"包里装件外套。后半夜冷。"

陆允川吃完饭去洗澡。换衣服的时候在衣柜里翻了件薄外套塞进书包。然后想起母亲说过的"带外套"，突然觉得有点奇怪——她每次都能猜到他要去干什么，而且从来不拦，只是加一句提醒。

出门的时候，母亲在客厅看电视。"几点回来？"

"明天早上。"

"那我不等你了。关好门。"

"知道了。"

县城老网吧在农贸市场旁边那栋楼的二楼。楼梯口贴着一张褪色的游戏海报，上面的人物已经看不出是谁了，只剩一把发光的剑还依稀可辨。楼梯扶手是铁管的，没有刷油漆——锈得发红，手握上去有种粗糙的磨砂感。推门进去，先闻到泡面、烟味和旧空调吹出的潮气混在一起的味道——这个味道让他无比熟悉，高一到高三，每个周末的下午和每次大考后的晚上，都是这个味道。

网管还是那个戴眼镜的光头男生，看见他们就点了点头，手上还在拆一台旧机箱的侧板。"通宵？"

"嗯。"程砚把自己的会员卡拍在柜台上，柜台是合成板贴了一层假木纹皮，边缘翘了，"三台机子，连座。"

"角落那三台。旁边那排的键盘坏了几个键——别按W和空格，按不动。"

他们往里走。陆允川选了靠窗那台——窗外是菜市场的顶棚，白天卖鱼卖菜后留下的积水在棚顶上反射出路灯的黄光，再远一点能看到路灯和一些零零星星的广告牌。他把外套搭在椅背上，按下开机键。主机嗡了一声，风扇转起来，屏幕慢慢亮起——显示器是那种老式的大屁股CRT，开机时屏幕表面会过一层静电，他用手指碰了一下，啪的一声，指尖一麻。

登录聊天软件的时候，他的手指习惯性地停了一拍。好友列表里，夏知遥的头像亮着——绿色的小圆点，状态是"在线"。

他没有点进去。先打开了游戏。
