# 第093章 夏天的聚餐之后

所属部卷：第二部：六月以后
时间：升学宴后夜
地点：县城小饭馆

## 诗歌插页 P093：原创仿宋小令

风过小楼西。
酒散人归，灯影压尘衣。
一句寻常留不住，
只剩鞋跟微微疼。

回家时，父亲拎着剩菜，母亲抱着礼金本。陆允川跟在后面，手里提一箱没喝完的橙汁。县城夜市还没散，烤串摊的烟贴着路灯往上走。

母亲问他：“同学都考得怎么样？”

“都还行。”

“那个夏同学呢？”

陆允川的手紧了一下，“也挺好。”

母亲没有再问。她把礼金本塞进布袋，叮嘱他回去泡脚，别把新鞋穿坏了。到家后，父亲在客厅算账，报一个名字，母亲就在本子旁边打勾。陆允川坐在院子里，脚放进红色塑料盆，热水一浸，磨破的地方立刻刺疼。

手机震了一下，是同学群发来的合照。饭桌上每个人都举杯，陆允川坐在边上，笑得很淡。有人在下面说：“都顺利。”夏知遥也跟了一句：“顺利。”

他点开输入框，写：“你也顺利。”写完又删掉。群里已经有很多人回了类似的话，他再发一次，并不会多出什么意思。

程砚发来私聊：“鞋跟贴创可贴。别问我为什么知道。”

陆允川低头看着脚后跟，破皮的地方泡得发白。他回：“嗯。”

那一夜他没有写长诗，只在草稿纸上写了四个字：别太用力。写完他把纸折起来，夹进录取通知书的信封里。后来他才知道，这四个字不是给夏知遥的，是给当时那个还不肯松手的自己。
