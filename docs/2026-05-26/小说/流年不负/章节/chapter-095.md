# 第095章 她没有接受

时间：2014年6月中旬

地点：楚州师院

卷次：卷五：各自南北

两周后。支教出发前两天。顾临风在图书馆看《中国古代文学》——《史记》"刺客列传"——豫让漆身吞炭——聂政自毁面容——荆轲刺秦王。手机亮了。许若琳QQ——卡通兔子头像在屏幕上跳。他打了两个字："什么。"

"那个人没追到。"

手指停在键盘上——没按下去。没有回。

许若琳继续发——"苏晚晴没有接受——她就说不想谈——大三太忙了。"对话框上方"正在输入——"闪了几秒——停了——又闪——"你别想多——我就是随便跟你说一声。"

他打了两个字——"知道了。"

关掉对话框。手机屏幕朝下——盖在桌上。下午的阳光从窗户斜射进来——照在《中国古代文学》页码上——"豫让——士为知己者死"。翻到下一页——又翻回上一页——刚才的内容没看进去。手指在书页上来回拨了两下。"她就说不想谈"——五个字——读了三遍。

她就说不想谈。

为什么不想谈。跟他有关系吗。因为那个约定吗。还是只是不喜欢那个人。还是她也在等——等时间——等毕业——等他。他不能问——也不能猜。

把手机放口袋。同一页看了大概十分钟——一个字没看进去——那页讲豫让——"漆身为厉——吞炭为哑"——眼睛在"厉"字上来回看了五六遍。抬头看窗外——四月的槐树长满叶子——绿油油——密得看不到对面楼。阳光从叶缝漏下来——桌面上细碎的光斑。推开窗户几寸——风里有香樟树花香——淡淡的——跟操场那边一样。

那晚他去了操场。下午下了阵雨——跑道上有几处积水——透明——能看到水底红色塑胶颗粒——积水反射路灯的光——晃眼。绕开积水走——低头看——水里有片泡烂的梧桐叶。一圈——两圈——三圈——四圈——五圈——六圈——七圈。比平时多三圈。第五圈呼吸不均匀——放慢步伐——手心有点汗——在裤缝蹭了蹭。

第七圈停下来——操场最外道。抬头——云层很厚——没有星星——月亮被遮住——只透一圈模糊毛边。大灯还亮着——白光打在云层底部。从兜里掏出三颗跑道上的小白石子——丢进排水沟缝隙——咚咚咚——声音被水泥壁吸收。走出操场——走的时候灯熄了——啪——整个跑道暗掉——没有回头——黑暗从背后压过来。

回宿舍——电视在播世界杯集锦——调了静音。阿坤在团战——"奶妈加血奶妈加血"比平时响。小伍写信——"展信佳"——"展"字第一笔有点歪。李博抬头看他的鞋——"今天多走了。""嗯。""今天有什么想说的吗？"声音放得很平——不是问问题——是陈述。"没有。""那就行。"

去洗脸——碰到隔壁宿舍的——光膀子刷牙——"你天天晚上出去——谈恋爱呢。""走圈。""有区别吗。"拧开水龙头——冷水泼在脸上——脑子里闪过——她没有接受那个人——不是因为那个人不够好——也许是因为——但他不能想。关水。擦脸。走回512。

手机亮了——许若琳——"出发那天记得带晕车药——山路很颠。""好——谢谢。""不用谢。"

躺回床上——手机烫的——碰了碰——热的。打了"晚安"——删掉——拇指按了五下。发的"好——谢谢"。按灭屏幕。天花板裂缝还在——数分叉——主裂缝分三条——又一条更细的——跟一年前一样。四条裂缝。十二公里。十一站。一个人蹲黑暗里数裂缝。没什么变化——也没什么不好。

翻了个身——脸冲墙——墙上小孔还在——食指伸进去摸了摸——水泥粉簌簌往下掉——缩回——搓搓食指上的灰。闭上眼睛。脑子里反复放——她没有接受那个人。不是高兴——不是难过——说不清楚——胸口一小块东西被捏了一下——不太重——但是酸。

想起高三那年——操场走圈——以为有男朋友——后来发现是表哥——走一个多小时走到脚底起泡——做完决定后浑身轻松——因为目标清晰。现在她也说不想谈——但他没有觉得轻松——他不知道她为什么不想谈——她的"不想"不一定是跟他的"想"有关系。他唯一知道的——她还在那个校园里——一个人——上课——吃饭——睡觉——骑车时摔了膝盖。

窗外摩托车又响了——由近到远——消失到巷子深处。

他睁开眼睛。黑暗里天花板上的裂缝被他看了无数遍——闭着眼睛都知道它从灯座往墙角走，分了三叉，最小的那条如果不仔细看会忽略。手机还热着——压在枕头下面，隔着枕套能感觉到那一小片温度。他把手伸进去——碰了碰屏幕。屏幕是暗的。许若琳的消息还停在对话框里——"出发那天记得带晕车药，山路很颠。"他翻了个身。脑子里是盘山公路——灰色的、弯弯曲曲的。前几个弯能看到后面的路，后面的弯看不清前面的路。

她没有接受那个人。

这句话在他脑子里循环播放——不是以句子的形式——是以一种感觉的形式——像胸口有两根橡皮筋被拉开以后慢慢弹回去。不疼。就是很慢。高三那年他说"等"——等高考，等毕业，等上大学。现在他知道自己在等另一个"等"——等时间，等她说的"暂时"变成"可以"。但这份等没有截止日期。没有指示牌告诉他"前方还有十二公里"。他只能一个人在操场上走圈。一圈。两圈。三圈。四圈。五圈。六圈。七圈。走到不想走了——就停下来。走出操场。回到宿舍。躺下。闭上眼睛。

明天收拾行李。后天出发去太行山。他要去李家沟教二十一个孩子语文。她要去另一个镇上教三年级的语文。两个人之间隔了三十多公里的盘山公路——比301路长。但方向是一样的——都是往山里走。都是去教同样的孩子。都在同一张支教联盟的海报上签过名。

也许在太行山里的某个晚上——他会在土操场上抬头看星星——山里没有光污染，星星应该很多。如果那时候他手里有手机，有信号——他会发一条短信：我在这里很好，你呢。

但他不会发。

因为"暂时"还没有变成"可以"。因为那个约定还在。因为他会遵守。

他把被子往上拉了拉。六月的夜晚已经不需要盖那么多了——被子是薄毯，灰色格子的。毯子的边角磨出了一些线头。他用手指绕了一根线头——绕了几圈，轻轻拉断了。闭上眼睛。呼吸慢慢变匀。窗外偶尔有夜鸟叫了一声——大概是夜鹭。叫声很轻。然后安静了。然后睡着了。
