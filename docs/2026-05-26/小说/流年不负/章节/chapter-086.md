# 第086章 第一个学期

时间：2013年12月

地点：楚州师院

卷次：卷五：各自南北

十二月下旬。大一上学期的最后两周。

期末考试前一周，顾临风在图书馆待到闭馆。

图书馆的桌子是长条的——深棕色木纹桌面，上面铺了一层透明塑胶垫。台灯是绿色的老式台灯。他的座位靠窗，窗外是一排光秃秃的槐树。他每天在同一个位子——三楼靠墙第四个座位。从下午两点坐到晚上九点四十五——闭馆铃响。

古代文学的笔记他整理了八十多页——全是手写的，字迹从潦草到工整再到潦草。现代汉语的重点用红色圆珠笔画了圈。哲学导论的名词解释写在活页纸上。

他不是刻意要拼。

就是花了很多时间看书。看累了就趴一会儿。趴醒了继续看。不去想别的。把脑子塞满，就没有空间想那些不该想的。

考试成绩出来了。

他在班里排第三。

李博在宿舍看了成绩单。"第三。可以啊临风。"

阿坤凑过来。"你们这种人就是变态。我六十分万岁，多一分都浪费。"

小伍排第二十一。他说"还行，没挂科就行"。

顾临风把成绩单折起来夹进笔记本。李博递过来一瓶汽水——橙色的芬达。"庆祝一下。你请我们吃饭。"

"我没钱。"

"那算了。"

期末最后一门考完。下午四点。顾临风从考场走出来。天已经快黑了。教学楼门口的老槐树在风里簌簌地响。他把手揣进棉袄的口袋里。棉袄是新买的——深蓝色，八十块钱，在师院门口那条街上。

室友们商量去聚餐——学校后门的"老刘烧烤"。

"临风去不去。"阿坤问。

"去。"

老刘烧烤在一条窄巷子里。店面不大，摆了七八张折叠桌。炭烟顺着排风扇往外飘。阿坤叫了一箱啤酒——十二瓶，绿色的瓶子在桌脚下排了一排。李博点了一桌子烤串——羊肉、鸡翅、烤韭菜、烤茄子。

"就四个人，吃得完吗。"小伍说。

"吃不完打包。"

老板把烤好的串放在铁盘里端上来。羊肉串上滋滋冒着油。阿坤用牙咬开了一瓶啤酒。"来。咱宿舍第一学期——干一个。"

四瓶啤酒举起来，碰了一下。啤酒很苦。顾临风喝了一口。然后又喝了一口。泡沫在舌尖上炸开，凉凉的。

阿坤喝得最猛。一口气半瓶下去了。"我跟你们说——大学四年就这第一学期最他妈重要。混熟了以后就不一样了。"

"有什么不一样的。"李博问。

"熟了以后你看每个人——都有故事。"

"你有什么故事。"

"我的故事很长。得喝三瓶才能讲。"

"吹牛。"

阿坤笑着又灌了一口。小伍喝了半瓶脸就红了。李博喝了一瓶，脸不红气不喘——"我从小就跟我爸喝白酒，这点啤酒不算什么。"

顾临风喝了大概两瓶。话比平时多了一点。阿坤问他什么他说什么。关于青阳县——"是个小县城，有一条河，叫青阳河"。关于高中同学——"很多人去了不同的城市"。关于大学——"挺好的"。

"你就没有别的词了。'挺好的'——什么都是挺好的。"阿坤说。

"就是挺好的。"

李博说。"你们知不知道临风暗恋过谁——"

"不知道。"阿坤往椅背上一靠。"他自己从来不说。"

顾临风又喝了一口。没接话。

十点多的时候从烧烤店出来。四个人沿着巷子往回走。路灯隔一盏亮一盏。路面上有积水——大概是下午下了点雨。他一脚踩进了水坑里，鞋底湿了，没管。

经过那条有电话亭的路。黄色的电话亭里面灯亮着。顾临风的脚步慢了一下。阿坤他们在前面走——"快点临风"——"来了"。

十二月的风很冷。他把棉袄的帽子拉上。帽子有点小。风吹到耳朵上——像刀子割。他把手揣得更深了。

到宿舍楼下的时候楼门还没关。五楼楼道里灯坏了，摸着黑上的楼。李博在黑暗中撞到了墙。"这破灯什么时候修。"阿坤在后面笑。

洗漱完。熄灯以后。

阿坤喝多了，躺床上三秒开始打呼噜。小伍照例翻了几次身——然后安静了。

顾临风没睡着。

两瓶啤酒的酒劲在太阳穴后面一跳一跳的。脑子转得很慢——但还是在转。他没有想成绩的事。他在想今晚经过的那条路——那条有电话亭的路。

如果那个电话能打通——如果他能打过去。如果电话那头能听见她的声音。如果他能说"我考了第三，你呢"。

他翻了个身。枕头下面的手机硌着他。

手机很安静。屏幕应该是黑的。

他把手机拿出来——按亮了屏幕。通讯录——苏晚晴。看了几秒钟。按灭了。

明天收拾行李。后天回家。寒假一个月。

睡吧。
