# 第098章 三个月

时间：2014年10月至2015年1月

地点：楚州各处

卷次：卷五：各自南北

和周雨在一起的三个月。从深秋到冬天。

十一月。楚州的梧桐叶子落得差不多了。地上铺了一层焦黄的卷边叶子，踩上去脆脆的。他们一起上自习——在图书馆三楼靠墙的位子。他坐第四个座，她坐第三个——就在他旁边。她带了薄荷糖，自习到一半会递过来一颗。他有时候吃了，有时候说"不用"。

自习结束以后他送她回宿舍。女生宿舍楼前面有一排灯——白色的灯柱，上面绕着枯掉的牵牛花藤。她站在灯光下面会说"今天自习效率很高"或者"明天中午食堂有红烧排骨"。他听着。偶尔"嗯"一声。

她爱说话。话题散——从文学社的事情到她们宿舍的女生到家里爸妈的"你怎么还不打电话回家"。有时候她会把他写进她的散文——她会把草稿拿给他看。有一篇叫《沉默的人》。里面有一段——"他坐在图书馆靠窗的位置，阳光打在他的笔记本上。他在写东西——我看不清写的是什么。他的笔停顿的时候，会抬起头看一眼窗外。窗外只有一棵槐树。"他看完了。没有说话。周雨问"像不像你"。他说"不怎么看窗外"——她笑了。

"你话好少。"她说过好几次。

"习惯。"

"你以前谈过恋爱吗。"

"没有。"

"真的假的——那你初恋是我。"她笑。眼睛眯起来。雀斑跟着笑出来的纹路往上翘了一点点。

"嗯。"

她相信了。她把头靠过来的时候，头发蹭到他的脖子——有点痒。她没有看到他的眼睛——他的眼睛那时候正看着窗外。窗外是梧桐树，叶子落了一半。

十二月初她过生日。他送了一本诗集——不是他写的，是他从学校图书馆附近那家旧书店买的。海子的《活在珍贵的人间》。书页有点泛黄，扉页上有人用钢笔写过一行字——"2008年购于楚州"。他用橡皮擦了擦，没擦干净。他在那行字下面写了一行新的——"生日快乐。顾临风。2014.12"。

她不嫌书旧。"旧书比新书好看。"把书放在枕头边。拍照发了QQ空间。

十二月中的一天。周六。他们去逛操场的跑道。他走在第五道。她走在第四道。两道之间隔着一根白线的宽度。走了两圈。她忽然把手伸过白线——碰了碰他的手背。"你走太快了。"他放慢了步子。她握住他的手指。他的手心里有汗。她的还是凉的。

操场远角的灯光打在她身上。让她头发边缘有很淡的金黄色。

三个月的相处里——一起上自习三十多次。一起吃饭二十多顿。一起逛操场十几次。一起看了三场电影——在师院北门那家小电影院，屏幕不大，座椅有点旧。第一场是《星际穿越》，她看哭了。第二场是《匆匆那年》，她看完说"青春片都是一个套路"。第三场是《霍比特人》，她睡了一小半。

最后一次一起出去是一月十号——楚河边上。冬天的河面很平。水是绿的，有点发灰。河岸上有几个人在钓鱼——支着马扎，一动不动。他们坐在岸边的长椅上。她靠着他肩膀。羽绒服摩擦着羽绒服——发出轻轻的沙沙声。远处有公交车开过——大概是301路，往东边去的。

她忽然说了一句话。"你听人说话的时候很认真——但总觉得你在看别的地方。"

他顿了一下。

"我在看你。"

他说的时候没有看她。
