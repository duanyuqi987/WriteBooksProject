# 第179章 朋友的烟旁观

所属部卷：第四部：失去的美丽
时间：大二冬末
地点：宿舍楼后门

## 诗歌插页 P179：原创现代诗

他站在朋友旁边，
并不比谁更清醒。
只是那晚风太冷，
总要有人把话说硬，
免得另一个人，
继续往错处走。

祁野其实很少抽烟。那包烟是室友聚餐时别人塞给他的，放在口袋里忘了拿出来。那天晚上他摸到烟盒，只是忽然不知道还能用什么动作填住沉默。

他也烦陆允川。烦他反复，烦他听不进去，烦他每次终于安静几天，又被一条消息打回原样。

可烦归烦，他还是把陆允川叫到后门。朋友不是永远温柔的人，有时只是知道再晚一点，这个人可能会做更糟的事。

祁野点烟时手有点抖，不是因为冷。他怕自己话说重，也怕话不够重。

“你不算坏，但你那句话很坏。”这是他想了很久才说出来的。把人和事分开，听起来简单，说出口却很难。说轻了没用，说重了伤人。

烟抽到一半，他就后悔了，嗓子难受，衣服也沾了味。可他没有立刻走。他知道陆允川那晚需要的不是劝完整套大道理，而是有人在冷处站到他不再继续发消息。

后来回宿舍，祁野把剩下的烟丢进垃圾桶。室友问怎么了，他说没事。

有些陪伴也需要被放下。不能总靠一支烟、一句狠话，把朋友从同一个地方拖出来。
