# 第036章 成绩单下滑之前

所属部卷：第一部：初秋三楼
时间：高一期末考试后
地点：公布成绩的公告栏前

## 诗歌插页 P036：原创现代诗

我把早晨放低，
让一份名单从课桌边慢慢经过。
你的名字没有被我喊出，
只在排名表上被铅笔划过的那行数字旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写成绩单下滑抵达之前的坡度。期末考试的成绩贴出来那天是憋闷的桑拿天，公告栏玻璃上蒙着白天的暑气雾——玻璃前面的铁框被太阳晒得烫手，有学生伸手按在框上看名单时被烫得缩回了手指。考得好的已经在提前讨论暑假补习班的安排，考得不好的一边看排名一边默默往回走，嘴上说"没事""还有下次"，但连声音都被热气压扁了。公告栏旁边的墙上贴着一张被晒褪色的防溺水安全教育海报，边角翘了，被偶尔经过的穿堂风掀起来又啪地贴回去。

陆允川站在人群外排着队。他不太着急看自己的成绩——他数学和物理估算了一下应该在年级前二十左右——但他在找另一个人的名字。他先找到自己的名字——十九名，然后顺着拼音往下翻，翻过前面几个X开头的名字之后，视线最终在一行数字上停了：夏知遥，年级排名：七十三。比上学期的排名掉了将近三十个位次。

她站在旁边，距离公告栏大概五步远，手里攥着成绩条。她的成绩条被手指攥皱了一角——她没注意到。她旁边的蒋雯伸手抱了她一下："没事，这次题难，大家都掉。"夏知遥没有说话，只是把成绩条折了三折塞进笔袋。折的时候动作很慢，像在叠一张不必让别人看到的地图。

陆允川站在原地想了好几秒钟——想说什么但什么都没说出来。他知道这时候"没考好没关系的""下次努力"这种公式化安慰对她来说只是多一道噪音。但他同时不知道怎么说一句她愿意听的话——她需要的是被当成一个不需要安慰的普通人，而不是一个刚被贴了排名标签的被喜欢对象。

晚自习结束，他回到宿舍。走廊尽头的饮水机坏了，有人在排队等热水泡面，一股红烧牛肉面的味道飘满了整层楼。程砚已经看过成绩了——这次掉得比谁都多。"我物理才六十几，公式用错了好几道。"程砚躺在床上双手枕着后脑勺，声音不太低落，但像是在用一种"我先认栽"的方式跟自己和解。他的床单皱成一团，枕头歪到一边，蚊帐有一只角脱落了垂在床沿晃来晃去。陆允川说了他的排名，程砚吹了声口哨："你前二十——那你稳了。我明年暑假得报补习班。"说完翻了个身，把脸埋进枕头里，闷闷地补了一句："暑假补习班的空调要是坏了我就退钱。"

陆允川在日记本上写了几个字，之后好几分钟只是坐着，一个字没写。他在想她折成绩条的动作——那个折三折的动作比他见过的任何哭或者质问都让他难受。不是因为他觉得成绩下滑有什么罪名，而是因为他知道她会在乎，他也会替她在乎——但她不知道他在替她在乎。这种在乎没有任何传递渠道，只是他一个人在夜里发酵。

明天开始放暑假。暑假里有整整两个月见不到她。他在日记本上写道：暑假有两个月，希望她休息好，好好看书，吃好饭。写完以后觉得这句话太像家长嘱咐了，又在下面加了一句：下学期继续一起值日。然后合上本子，关掉了台灯。这本日记里的主语永远是"她"，关于她希望她好的时候像一个无人接听的留言机——每一条关心都存储进去了，但对面永远不会听到提示音。
