# 第011章 黑板报

时间：2010年12月中旬

地点：高一三班教室

卷次：卷一：初见秋风

十二月。教室后面的黑板报轮到苏晚晴出。

黑板报是一个苦差事。别人放学走了，你还在教室后面站着。粉笔灰落在袖子上，洗不掉。黑板反光，字写歪了没人知道，等发现了已经写了大半板，擦也不是，继续也不是。

这期主题是"新年畅想"。班主任定的。他给了几个关键词:学习、理想、青春、未来。

周一到周三。每天放学后一小时。

头两天是画花边和排版。她和另外两个女生一起，用尺子和粉笔在黑板上打好格子。横线、竖线。花边是波浪形的，用彩色粉笔画。那个矮个子女生负责画花边，她画得很快，手很稳。另一个负责写标题字——大红粉笔，写了"新年畅想"四个字，每个字有巴掌大，楷体。

苏晚晴负责正文。

周三下午。放学铃响了。教室里的人一个接一个地走了。椅子推进桌底的声音此起彼伏。有人在门口喊了一声"晚饭吃啥"——"食堂。还能吃啥。"——然后声音远了。

教室里剩下的就三个人。前面两个女生，一个在擦黑板顶部，一个在收拾粉笔。苏晚晴站在黑板左边，手里拿着白色粉笔。

她在写第二节。

顾临风没走。

他的座位在后门旁边。他留得很自然——作业没写完。其实作业早就写完了。他把作业本合上，又打开，又合上。笔拿在手上转圈。不掉，一直转。赵一鸣说他转笔很厉害，能转一分钟不掉的。他没注意过这事。

黑板前面。苏晚晴的手在写字。

白色粉笔。一行一行往下。

她的字是工整的楷体。每个字的大小差不多，行间距很均匀。写到某个字的时候，她停了一下——应该是写错了。她拿抹布蘸了水，在那个字上擦了擦，等水干，然后重新写。

她握粉笔的姿势和握圆珠笔一样。小指翘起来的。

白色的粉笔灰落在她的袖子上。袖子是深蓝色的。灰很明显。她没管。

写完一段。她后退一步，侧头看了看效果。然后上前继续写。

另外两个女生在说话。声音不大，大概是商量明早吃什么早点。苏晚晴没参与。她一直在写。偶尔停下来看看底稿——底稿是一张草稿纸，上面记了一些文段的草稿。纸边撕得不太齐。

顾临风坐在后面。他看着那些字在黑板上一个一个地多起来。白色的。横平竖直。最后一个笔画收尾的时候，她会把粉笔往上提一下。很轻，不仔细看看不出来。

"你走不走——"前面的一个女生回头朝苏晚晴喊了一声。

"写完这段。"

那个女生点点头，然后和另一个一起走了。她们出门的时候，带上门，门没关严，又弹开了一条缝。

教室里只剩两个人。

苏晚晴还在写。顾临风还在看。

她写到最后一段的时候，粉笔断了，掉在地上，碎成两截。她弯腰捡起来。把断的那截丢进粉笔盒里，拿了一根新的继续写。

最后一个字写完。她退后一步。看了看整面黑板，然后走到桌子边拿抹布擦了手。把粉笔盒盖上，底稿折好放进书包。书包拉链拉开、合上。

陶瓷小鱼晃了两下。

她背起书包。从前门走了。门这次关严了，锁簧咔嗒一声。

顾临风还坐在座位上。面前摊开的那道数学题只写了三步。第一步写对了，第二步错了——他拿笔划了一道横线，底下重写了一行。第三步还没写。

他低下头，看那道题。

教室里另一个人的呼吸声没了。日光灯的电流声嗡嗡的。粉笔灰在光柱里飘着，很慢。还有些落在了地板上。

他合上了作业本。站起来，把椅子推进桌底。走到后门，手搭在门把上。回头看了一眼黑板报——

白色的正文字。红色的大标题。绿色的波浪花边。

她的袖口应该沾了不少粉笔灰。不知道她洗掉了没有。

拉开门，走了。
