# 第133章 她和别人说笑之后

所属部卷：第三部：寒风里的花
时间：六月初
地点：校园超市门口

## 诗歌插页 P133：原创仿宋小令

冷柜映人低。
瓶上清珠，落进掌纹里。
一句没空原有限，
不是关门向我时。

陆允川回宿舍时只拿了一瓶矿泉水。祁野坐起来，看着他空着的另一只手。

“我的冰红茶呢？”

陆允川这才想起，“忘了。”

祁野盯着他几秒，“你买水买出人生问题了？”

陆允川把矿泉水放到桌上，没有拧开。祁野下床，拿起他的手机看了一眼，又放回去，“遇见她了？”

“嗯。”

“她和别人一起？”

陆允川没有答。

祁野叹气，“你看，你自己都知道这话说出来没理。”

陆允川坐在椅子上，指腹按着瓶身的水珠。那些水很快把桌面弄湿。他用纸巾擦，越擦越乱。

晚上社团群里，苏晚枝发了活动物资清单，那个男生在下面补了一句“纸杯已放活动室”。陆允川看着名字，终于知道他是宣传部的同学。

他没有点进对方头像，也没有再问苏晚枝。只是把清单下载下来，照着自己的任务做了三项校对。

做到最后，他在草稿纸上写：她没空陪我，不等于她没空生活。

这句话让他不舒服。可不舒服有时正说明话是对的。
