# 第106章 点赞

时间：2015年12月

地点：楚州师院宿舍

卷次：卷六：平行轨道

他把照片放大。

又看了一遍。

苏晚晴的脸占据了整个屏幕。分辨率一般——手机是两年前买的，屏幕像素不太高，放大到最大倍率的时候边缘有点糊。白衬衫在强光里过曝了，领口那一块白得发亮。马尾。阳光在她的鼻梁上有一小块亮斑——不是白色的，是淡黄色的，边缘模糊。她的嘴张着——应该是正在喊口令。"一二三四"或者"伸展运动"。嘴角往上翘。那种翘法不是刻意的。是呼吸跟动作之间自然带出来的弧度——吸气的时候嘴巴张开准备喊口令，最后一个字出口之后嘴巴开始合拢但还没完全合上，停在中间那个半开半合的位置。

他的拇指在屏幕上来回动。指甲很短——前两天刚剪过。指肚的纹路在玻璃屏上印出一圈浅浅的油痕。

放大——她的眉毛。右边那道比左边稍微淡一点。以前他在教室里偷看她的时候，就发现过。那天她在座位上看书。光从窗外斜进来。照在她右半边脸上。右眉毛在光里几乎看不见——被光冲淡了。左边的眉尾则有一点微微往下弯的弧度。这个弧度他记得很清楚——好几次下课后他从她的座位旁边经过都能看到。

缩小——整个操场。红色的塑胶跑道——褪色了，边缘发白。绿色的假草坪——在太阳底下反射着塑料的光泽。一群灰白色校服的孩子——校服是运动款的，拉链开襟，袖口有松紧带。她站在正前方。周围的人——比她矮一半。高矮差使得画面构图出现一个奇怪的焦点：所有的视线线都汇聚在这个白色人影上。

放大——她的手。手指伸直。指尖往上翘。指甲盖是干净的——没涂指甲油。他记得她写字的时候小指也会翘起来。高中的时候他偷偷观察过——她的左手放在桌上，小指悬在纸面上面一厘米。悬在半空中，不碰纸。但那根手指同时又在跟着笔在动——写"捺"的时候微微往下沉，写"横"的时候往外侧飘。

缩小——黑板。"祝苏老师生日快乐"——粉笔字。二三十个字大小不一。"祝"字最大，占了两行粉笔的宽度。"乐"字被黑板槽挡住了一半。画了一只蛋糕——说是蛋糕，其实就是三个椭圆叠在一起。最上面的椭圆歪了一边。上面插了几根蜡烛——几条竖线顶着不规则的圆圈。蜡烛的火焰画得像一把歪了尖的叉子。蛋糕旁边还有一个小人——两根辫子、圆脸、两个黑点作眼睛。小人旁边写了几个字——"苏老师你最漂亮"。"漂亮"两个字写错了——"票"字被改成了"亮"，改得很潦草。

放大——她的脸旁边。泪痕还在。光线侧打过来，泪痕反射出一小条银色的线。嘴角也在翘——因为笑而把脸颊往上推了一点。她大概是被那些花和歪歪扭扭的字弄哭了。然后又被自己的"哭"弄笑了——旁边有个小女孩大概说了什么笑话——"苏老师你哭起来也好看"。她哭不下去了。然后在笑里发觉自己真的喜欢上了这份工作。不是"可以接受"——是喜欢。

他把照片放在这里。没有动。手机壳在手指间微微发热——处理器在运算图片，机身温度慢慢升高。

拇指悬在"赞"上。

那颗灰色的小爱心在拇指肚下面。离屏幕大概一毫米。那个位置——屏幕玻璃上有一个细小的指纹印。他刚喝了水——拇指上还有一点水温的余温。

窗外有人吹口哨。断断续续的。是一首流行歌——副歌部分翻来覆去地吹了三四遍。调子不准——那几个高音上不去，在每个高音的地方都降了半调。吹口哨的人大概正在倒垃圾——楼下有垃圾桶盖掀开的声音。金属盖子磕在水泥地上——哐当一声。

他的拇指悬着。

大概二十秒。二十秒里——宿舍里的声音只有暖气管偶尔发出的吱嘎声。还有李博在上铺的呼吸——比睡着时浅，大概在做梦。李博的脚动了一下——床板轻微地晃了晃。

他把手机锁屏了。

屏幕黑了。苏晚晴的脸消失了。锁屏壁纸是出厂自带的——一片蓝色的几何色块。跟他的其他设置一样——没有定制过任何东西。通讯录是默认排序，桌面图标是默认布局。

他把手机放在桌上。翻了个身。仰面躺着。后脑勺压着枕芯里的棉花。看着天花板。裂缝还在。从灯座往墙角延伸。三年了。裂缝没变。盯着看久了会觉得它在动——但并不是真的在动。是视神经疲劳造成的错觉。眼球在微颤，大脑在补充那些看不清的部分。

脑子里是那张照片。她的脸。她的眉毛。她翘起的指尖。孩子们歪歪扭扭的粉笔字。那朵像叉子的蜡烛火焰。

解锁。

屏幕亮了。照片还在。拇指划过屏幕的动作被手机记录为一个小小的触控弧线。

他看了几秒。锁屏。

把手机放回枕头底下。枕头底下是他自己手抄的考研笔记——几页纸翻得起了毛。手机的重量压在纸上面。

窗外有摩托车发动的声音。轰轰轰的。在夜里的街道上传得很远。然后拐了个弯。声音小了。没了。夜恢复了安静——那种"不完全安静"。远处有空调外机嗡嗡地转。一楼有人在听收音机——声音开到最小，透过地板传上来是闷闷的说话声。

他闭上眼睛。

点赞是什么意思。点一下——什么都不写。她没有他的任何消息已经两年多了。然后忽然一条通知——"顾临风赞了你的照片"。点了一下——没有文字，没有解释，没有"好久不见"。就是一个图标——一颗红色的小爱心。然后她看到这个屏幕弹出——可能会愣一下。可能会笑。可能会关掉通知继续批作业。也可能不会。

他想的是：如果点了赞，她会看到。她会知道他在看她的朋友圈。她会知道他没有消失——他还在用微信，他还在这个世界上，他还在看她。然后呢。然后可能会有一种尴尬——两年多的沉默被一颗小红心打破了。两个人都知道对方在看——但不知道说什么。

他想不到然后。然后太多了。两年多的空白后面——一句话必须特别精准才不会尴尬。但没人能在二十秒内找到这句话。

两年多了。两年多没有联系。没有电话。没有短信。没有QQ留言。没有微信消息。他唯一做的就是在她的朋友圈里——潜水。看她说"期末考完了""今天下雪了""食堂的饺子好难吃""教案写到凌晨"——每一次都看完。每一次都没有点赞。每一次都没有评论。

像一个站在窗外的人。不敲门。不说话。只是站着。看着窗里面的灯亮着、人影晃动、偶尔传来玻璃杯碰瓷盘的声音。他在外面看着这一切——然后天黑了。灯灭了。他走了。第二天又来。

如果现在点赞——那扇窗就破了。玻璃碎了不是坏——玻璃碎了说明有人进来了。问题是他还没准备好进门。

不是不想联系。是不知道该说什么。该用什么语气。"好久不见"太像在责怪她的沉默。"最近怎么样"太像没话找话。什么都不说只是点一个赞——又像在说"我一直在看你——但不想理你"。这句话他不想让她感受到。即使——即使这可能是真的。

他不知道。

所以他没有点赞。

深呼吸。一次。呼气从鼻子里走。气管里有一点刺痛——冬天的干燥空气。然后是喉咙深处的一点点咸味——大概是晚饭里放多了酱油。

他翻了个身。脸冲着墙。墙皮上有几处剥落的痕迹——刷的白灰起了一层小泡。在黑暗里看不清楚。但手能摸到——粗糙的、有点硬的。像砂纸。他的手指沿着那些剥落的地方慢慢滑过去。

那天晚上他醒了一次。凌晨三点四十。窗外没什么特别的——路灯还亮着。黄色的光投进来在地上铺了一片长方形。光里面有树枝的影子——窗外的法国梧桐秃了，但细小的枝丫还在。影子动了一下——有风。然后不动了。

他把手机从枕头底下摸出来。屏幕的光刺眼——蓝色调的，把脸照得有点白。

解锁。朋友圈。往下翻。翻过了夜里发的几张代购广告。翻过了别人半夜失眠发的文字。许若琳发的那组照片。苏晚晴。第四张。白衬衫。马尾。笑。

他把那照片放大到只看得清眼睛。她的瞳孔是黑的。高光点是一颗白色小圆点。他看了大概一分钟。

然后把手机放下。屏幕朝下。光被压在枕头和手机之间——周围又黑了。

窗帘缝里漏进来的光在墙上画了一道线。那道线很细。像一根头发。发尾散开——那不是缝隙本身，是光穿过窗帘纤维时被散射了。他看着那根线看了很久。久到眼睛发酸——眼球表面干燥，眨眼的时候眼皮像被砂纸擦了一下。然后闭上眼。黑暗里还有那张照片的余像——白衬衫、马尾、笑。视网膜上的化学物质还没完全代谢掉——图像的影子还在。白色衬衫那块区域最亮。

翻了个身。腿曲起来。膝盖碰到墙——凉意从皮肤透进去。墙是砖的。冬天的砖吸了冷空气之后会一直保持低温，直到第二天太阳把它晒暖。

睡了。
