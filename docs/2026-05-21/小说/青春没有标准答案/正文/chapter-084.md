# 第084章 国庆六十周年黑板报



时间：2009年十月一日前后



地点：高二十班教室后墙



卷次：第四卷：分科以前的夏天



2009年十月一日前后，高二十班教室后墙，黑板报右下角的一颗红色五角星把这段日子推到十班面前。十班做国庆六十周年黑板报，任思齐负责把主题字描稳。

有人急着回家看电视转播，有人担心作业写不完，任思齐却坚持把边框补齐。这不是突然降下来的大事，而是从早读、课间、饭卡、家长电话和后墙红榜里一点点长出来的。

段璟坐在教室里，常常先看见常俊凡的背影，再意识到自己其实该看卷子。从这个学期开始，这种走神不再只是喜欢，也会影响一道题、一次排名和一个选择。

常俊凡没有变成谁的答案。她会犹豫，会烦，会被家里问得沉默，也会在别人替她下结论时把话说得很硬。

任思齐也在这件事里显出自己的位置。字写得好，做事慢，喜欢把每一笔描准，成绩中等却很能坚持。十班从来不是只由两个人组成，它是四十多个人一起挤出来的青春。

“星角歪了，等我补一下。”任思齐说。

“你再补，国庆都过去了。”付小川说。

“让他补完，歪着也难看。”常俊凡说。

“我扶凳子。”段璟说。

“主题要正，字也要正，人先别从凳子上摔下来。”陶建国说。

黑板报的粉笔灰比平时厚，后墙下面铺着旧报纸，红纸剪下来的边角落了一地。

国庆六十周年几个字很大，谁写都会手抖，怕写歪了全班都要看一整周。

任思齐站在凳子上描边，常俊凡在下面扶着凳脚，段璟拿着透明胶等她叫。

有人赶着回宿舍洗衣服，有人想去小卖部买水，可看到后墙还空着，又不好意思先走。

教室前面还有没写完的作业，后墙却像临时变成另一个战场，大家用粉笔和纸胶带争取一点班级体面。

广播里偶尔放国庆相关的新闻，同学们听一耳朵，又低头继续剪星星。

陶建国没有把话说得很大，只提醒他们别站太高，别把红纸贴皱，别忘了明天还要交物理卷。

主题字终于描完时，晚自习灯管闪了一下，十班后墙亮得像被许多人一起撑住。

陶建国这段时间管得比高一更细。不是因为学生忽然变坏，而是因为分科、晨检、国庆活动和期中考试挤在一起，谁一松就会被推着往后退。

任思齐被叫到名字时，先看了一眼周围同学，再看老师。这个小动作让段璟发现，班里很多看起来不在乎的人，其实都怕被公开放到台面上。

封丘县的校园生活没有电影里那么整齐。校门外有卖烤红薯的，有修自行车的，有家长骑电动车来送饭，也有学生在路边背单词。

黄河故道那边的风有时会吹进县城，吹动校服袖口，也吹得人想起更远的地方。可更远的地方暂时还只是试卷最后一道大题旁边的空白。

国庆六十周年的新闻、甲流晨检的表格、分科确认的回执，都没有替他们解决任何一道题，却让他们知道，学校外面的时间也在往前走。

宋乔会把女生那边的情绪说出来，付小川会用玩笑把男生那边的沉默撬开，周明澈则常常在最乱的时候把表格、名单和班费重新理一遍。

段璟有时羡慕这种确定。他自己更像一张写了太多草稿的纸，明明知道下一步该写什么，却还是忍不住看常俊凡会不会也停在同一个地方。

常俊凡并不总是回头。她更多时候只是把自己的事情做好：改题、接水、记账、扶凳子、把别人的话挡回去，然后继续低头写下一行字。

白露用尺子压住红纸边，透明胶一贴还是起了一个小泡，桌洞里的试卷露出一角，上面还压着没改完的题。旁边的人没有追问，教室里的翻书声把尴尬遮住。

石磊站在凳子上描主题字，腿酸了也没有下来，校服袖口蹭过桌沿，留下一道浅浅的粉笔印。常俊凡看见了，也没有立刻说什么，只把自己的题继续写完。

田野把剪坏的星星揉成一团，又重新拿了一张红纸，操场上的哨声隔着窗户传进来，显得很远。段璟后来想起这一幕，才知道青春里很多变化一开始都很安静。

林舟帮忙扶凳子，手心全是粉笔灰和透明胶的黏，窗外的树影晃了一下，像把话题轻轻推远。窗外风一吹，纸角轻轻动了一下，像有人把话又咽回去。

闻佳把后墙旧报纸收起来，纸上沾满红色碎屑，有人把椅子往里收，木腿擦过地面发出短短一声。十班的热闹还在，只是每个人都比高一时多了一点分寸。

陆晨用尺子压住红纸边，透明胶一贴还是起了一个小泡，同桌抬头看了一眼，又像什么都没看见一样低头。老师从门口经过，所有人的声音立刻低了半截。

梅朵站在凳子上描主题字，腿酸了也没有下来，粉笔灰落在袖口上，拍了两下也没有拍干净。这样的细节不会出现在成绩单上，却会留在很多年后的记忆里。

宋乔把剪坏的星星揉成一团，又重新拿了一张红纸，小卖部那边的铃声远远传来，很快被上课铃盖住。这个动作很小，却比一句大道理更能说明当时的慌。

国庆六十周年黑板报之后，十班没有立刻发生巨变。真正的变化通常不是一声响，而是某个同学开始少和旧同桌一起吃饭，某个名字开始出现在另一张名单上。

国庆六十周年没有被写成口号，而是落在粉笔灰、红纸边角和晚自习后的灯下。段璟把这件事记得很清楚，不是因为它多戏剧，而是因为它让他第一次看见选择有后果。

他也慢慢懂得，喜欢不能替常俊凡做决定，也不能把她的每一次沉默都解释成和自己有关。

常俊凡要走的路，首先属于她自己。段璟若真想靠近，就得先学会站在不挡路的位置。

高二就是这样来的：没有高一开学那样热闹，也没有后来倒计时那样压迫，却把许多关系悄悄推到更真实的位置。

晚自习结束时，黑板报右下角的一颗红色五角星还留在教室里，像一枚小小的钉子，把这一天钉在十班的时间线上。

灯一盏一盏暗下去，楼道里传来脚步和水瓶碰撞声。段璟走到门口时回头看了一眼，发现常俊凡还在收书。

他没有催，也没有等得太明显。只是把书包带往肩上提了提，跟着人群往楼下走。
