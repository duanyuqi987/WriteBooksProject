# 第087章 没有发出的短信



时间：2009年春到秋，初春

地点：封丘一中校门口

卷次：第四卷：分科以前的夏天



2009年春到秋，初春，没有发出的短信这件事先从城关镇文具店露出头。女生宿舍的灯熄得晚，窗帘后面有人背单词，有人压低声音笑。段璟把树莓汽水放在桌角，眼睛却落在常俊凡的背影上。秦越在旁边插了一句闲话，把原本绷着的气氛扯松半寸。魏守诚从门口经过，提醒他们别把十班的名声丢在走廊里。2011年的高考越来越近，县城书店门口摆满冲刺卷，红色封皮一摞一摞，像把夏天提前搬来了。这些大事落到学生身上，最后还是变成作业、早读、饭卡和回家路上的一句叮嘱。



“你报封丘一中，是因为想来，还是因为大家都说好？”常俊凡问。
段璟捏着志愿表的边角，半天才说：“都有。”
“都有不算答案。”她说。
秦越从后排探头：“那你给个标准答案呗。”
常俊凡把笔放下：“没有标准答案，所以才要自己选。”



常俊凡记得没有发出的短信，但她记得的角度和段璟不一样。常俊凡不讨厌被人喜欢，可她害怕被喜欢以后，自己连普通说话都要小心。她不是冷，也不是故意把人推远。她只是太早明白，一个女生如果不替自己守住边界，很容易被别人的期待挤到没有位置。她把笔记整理得很齐，把错题分成几类，把难过压在课本下面。等晚自习铃声响起来，她仍然坐直身体，像什么都没有发生。



十班从来不是只有段璟和常俊凡。石磊按自己的办法活着：壮实，手巧，修桌椅一流，学习上是实操型，理化实验强。郭向阳坐在另一组，班级松弛感，一句话常把气氛拽到别处。段璟更安静，成绩不一定最亮，却让人相信普通人也有自己的路。陶建国常说，一个班最怕只剩几个名字，其余人都变成背景。所以后来段璟写十班时，总要把这些人一个个放回座位上：有人趴着补觉，有人偷吃面包，有人把橡皮借出去又忘了要回来。



金银花开的时候，很多院子里都有淡苦的香味。常俊凡说这种味道醒脑，段璟却觉得它像一张写不完的卷子。



这一天的重点不是没有发出的短信本身，而是它让段璟和常俊凡之间多出一个必须承认的事实。他们都还年轻，年轻到会把一次沉默想得很大，也年轻到第二天仍要照常早读、排队、交作业。封丘一中的生活不会因为谁心里翻江倒海就暂停。铃声照响，试卷照发，食堂窗口照旧有人催后面的同学快一点。



晚自习灯灭了一排，十班的人从楼梯往下走。雪水、粉笔灰和饭菜味混在一起，没人知道明天会先落在哪个人身上。

女生宿舍的灯熄得晚，窗帘后面有人背单词，有人压低声音笑。江雨珊把树莓汽水递过来时，没有问段璟为什么发呆。十班同学大多懂这种沉默：不一定是恋爱，也可能是分数、家里、志愿表，或者老师刚刚一句不轻不重的评价。周明澈在后排喊人借胶带，声音穿过半个教室，把刚要变重的气氛又拉回日常。

陶建国的脚步声一到门口，班里立刻安静半截。段璟低头写题，常俊凡也低头写题。两个人隔着一排桌椅，各自把心事压进不同的草稿里。窗外有人跑过，鞋底擦过水泥地，声音短促，像青春里很多来不及解释的事。

晚饭后，学校广播放了几分钟新闻。有人嫌无聊，有人趁机补作业。胡向南说时代不是挂在墙上的大字，它会钻进你们的饭卡、车票、志愿和将来去不去远方的选择里。段璟当时没听懂，只觉得这句话太像政治题。后来他才知道，那些大词最后都会变成很小的生活。

常俊凡把错题本合上，站起来去接水。段璟想叫住她，话到嘴边又咽了回去。他不是每一次都胆小，有时只是明白她已经很累。田野从走廊回来，带进一身冷风，说楼下又有人因为排名吵起来。十班的灯亮着，像一只被作业撑住的船。

午休时，沈曼青趴在桌上睡着，胳膊下面压着半张没写完的卷子。陶建国从窗外看了一眼，没有立刻敲玻璃。高三以前，老师偶尔还会允许学生短暂松一口气。段璟把笔帽咬出一圈牙印，常俊凡在前排翻书，纸页声音很轻，却足够让他知道她还在。

城关镇的街灯亮起来时，补课班楼下开始卖烤红薯。李慕白抱着练习册站在摊边等人，热气扑在脸上，眼镜片白了一瞬。常俊凡从旁边经过，问段璟要不要一起回学校。她问得自然，段璟却要花两秒才让自己也自然起来。

那天的作业不难，难的是所有人都心浮气躁。陆晨把英语默写本摊开，嘴里念着单词，手却伸进抽屉摸饼干。常俊凡回头提醒段璟别把草稿写到桌面上，他低头一看，才发现自己把一个公式抄了三遍。

黄河大堤那边的风到傍晚才停。段璟回家路上看见麦田一层一层暗下去，突然想起十班白天的吵闹。学校里的每个人在卷面上都像一个分数，出了校门又各自变回儿子、女儿、哥哥、姐姐。他第一次隐约觉得，常俊凡也一定有他看不见的家和疲惫。

晚自习第二节，教室后排传来很小的笑声。陈柏言在草稿纸上画了个歪歪扭扭的老师头像，被魏然用胳膊肘撞了一下。常俊凡没有回头，只把自己的卷子往桌子里面收了收。段璟看见她这个动作，知道她在努力不被这些热闹带跑。

周末放学前，陶建国让每个人把下周计划写在纸上。有人写提高二十分，有人写不迟到，有人写少去小卖部。段璟写到第三行时停住。他想写离常俊凡近一点，又觉得这句话放在计划表里很不像话，最后改成物理不低于八十五。

常俊凡的学习方法和他不一样。她不喜欢把本子写得很花，只在关键步骤旁边画小方框。段璟的错题本则总是挤满补充，像怕留下任何空白。有一次叶书兰看见两人的本子，笑着说，一个像在修路，一个像在筑墙。两个人都没接话。

邓桥家里送来的咸菜在宿舍传了一圈，最后只剩空玻璃瓶。男生们说话没轻没重，聊分数，聊球赛，也聊谁和谁最近走得近。段璟听见常俊凡的名字时皱了下眉，付小川立刻拿筷子敲碗，说别把女生名字当下饭菜。

女生宿舍那边也有自己的烦恼。宋乔嫌水房排队太久，刘青岚担心英语听力，孙佳怡被母亲电话催得眼圈发红。常俊凡听她们说完，才拿出自己的卷子。她不觉得自己比谁轻松，只是习惯把困难放到最后处理。

县城的夜不像大城市那样亮。封丘一中门口几家小店关门很早，卷闸门拉下去，街上就剩路灯和偶尔过去的三轮车。段璟站在校门里，看见常俊凡和几个女生一起走远。他没有追上去，只把手里的公交月票攥紧了一点。

王启民在年级会上说，十班最近太活，活到快不像毕业班。陶建国回来以后没有骂人，只把黑板擦拿起来，把右上角的倒计时数字重新写了一遍。那个数字比任何批评都管用，教室里一下子只剩翻书声。

很多事在当时看都只是小事：折角单词书放错了位置，一句玩笑说重了，方晴借出去的橡皮忘了还，常俊凡在试卷上多停了一秒。可长大以后段璟再想起十班，记住的偏偏是这些小事。它们比宏大的口号更像青春本来的样子。

叶书兰布置作文时没有给题目，只说写一个你不敢承认的愿望。班里一下子安静，连付小川都没立刻开玩笑。段璟写了半页又划掉，最后写成想把物理学明白。常俊凡交卷时没有看他，她的作文题目很短，叫《往前》。

水房排队的人很多，暖水瓶一只挨着一只，红的、蓝的、掉漆的，都站得很规矩。马晓彤抱着瓶子背英语单词，背到一半忘了中文意思，只好问旁边人。常俊凡把自己的瓶子往里挪了挪，给段璟空出半步。那半步没人议论，却足够他记很久。

县城文具店的老板认识不少一中学生，看见校服就知道该推哪一种笔芯。段璟买了两支最普通的黑笔，常俊凡买了一个透明文件袋。老板随口说又快考试了吧，两个人同时点头，又同时笑了一下。考试把他们变得像同一种人。

操场边的广播坏过一次，声音忽高忽低。体育老师让男生绕圈跑，女生在看台下压腿。梁一帆从段璟身边冲过去，喊他别像背着一麻袋心事。段璟想反驳，却发现这句话难听得很准确，只好加快脚步。

家长会前一天，班里开始集体收拾卫生。石磊负责修歪掉的桌腿，宋乔擦黑板报边框，常俊凡把窗台上的粉笔头一颗颗扫进纸盒。段璟被分去拖后门地面。他们隔着半间教室忙，各自都知道明天父母会在这里看见自己的座位。

有时真正让人难受的不是大事，而是午饭时少了一个座位。段璟端着餐盘站了几秒，常俊凡往旁边挪了一点，宋乔也挪了一点，长凳终于空出一截。付小川说挤吧，十班连未来都挤，何况一张凳子。大家笑起来，汤差点洒出来。

夜里查寝时，走廊灯一盏一盏亮过去。宿管阿姨的钥匙串响得很有规律。男生们把还没吃完的方便面藏到床底，女生那边则把小台灯关掉。段璟躺在上铺，听见远处火车声，突然想到封丘之外到底有多远。

有时真正让人难受的不是大事，而是午饭时少了一个座位。段璟端着餐盘站了几秒，常俊凡往旁边挪了一点，宋乔也挪了一点，长凳终于空出一截。付小川说挤吧，十班连未来都挤，何况一张凳子。大家笑起来，汤差点洒出来。

夜里查寝时，走廊灯一盏一盏亮过去。宿管阿姨的钥匙串响得很有规律。男生们把还没吃完的方便面藏到床底，女生那边则把小台灯关掉。段璟躺在上铺，听见远处火车声，突然想到封丘之外到底有多远。

很多事在当时看都只是小事：黑板擦放错了位置，一句玩笑说重了，宋乔借出去的橡皮忘了还，常俊凡在试卷上多停了一秒。可长大以后段璟再想起十班，记住的偏偏是这些小事。它们比宏大的口号更像青春本来的样子。

黄河大堤上的风比学校里硬，吹得人说话都要稍微提高一点声音。任思齐把饭卡递过来时，没有问段璟为什么发呆。十班同学大多懂这种沉默：不一定是恋爱，也可能是分数、家里、志愿表，或者老师刚刚一句不轻不重的评价。余小满在后排喊人借胶带，声音穿过半个教室，把刚要变重的气氛又拉回日常。
