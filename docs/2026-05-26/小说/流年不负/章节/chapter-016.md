# 第016章 期末

时间：2011年1月中旬

地点：青阳县一中教室

卷次：卷一：初见秋风

一月中旬，期末考试。

考前三天，各科老师都在发卷子。物理三张——李老师从办公室抱过来一摞卷子，卷子是手刻油印的，纸上还有没干透的油墨味。他拍了拍卷子:"回去做完——最后那道提高题，不强制——有能力的做。"数学两张——正反八道大题。英语两张半——半张是听力训练，班主任拿了一台录音机在讲台上放——磁带咔咔转——"听力考试现在开始——"底下有人悄悄说"听不清"。化学老师推门进来——最狠——四张，还带附加题。"附加题最后一问——配平，写完全方程式——做不出来不扣分——做出来加分——"教室里一片哀嚎——哀完了还是闷头做。课代表发卷子的时候，有人把卷子叠成纸飞机——从窗口往走廊扔——纸飞机飘了半圈——歪歪扭扭——掉在走廊的地砖上。物理老师李老师正好从走廊经过——弯腰捡起来——看了一眼飞机的叠法——把机翼打开——看了上面写的那道力学题——"受力分析画错了。"声音平。然后把纸飞机夹在腋下——继续走。

晚自习延长到十点。宿管周老师来通知的时候——"这几天考试——自习可以上到十点——"底下有人竖大拇指——然后又立刻低下头继续看书。快十点的时候走廊里泡面的味道越来越浓——有人拿热水壶去水房——杯子灌开水——茶叶泡了第三遍没味了。教室里的灯十点半之前不关。所有人都低着头——四十二个脑袋埋在书和卷子之间——日光灯的嗡嗡声比平时更明显——因为没人说话了。翻书声——纸张哗啦——笔尖在草稿纸上画辅助线的声音——椅子偶尔嘎一声——是有人屁股坐麻了换姿势。有人在吃饼干——掰成两半——一半塞嘴里——一半递同桌——同桌摇头——他又偷偷塞回饼干袋——把袋口捏了一下。

顾临风这次复习比以往都要拼。

拼的理由——他告诉自己是因为想进步——上高中第一学期总得有个态度——期末不能考太差——回去要给她妈看看成绩单——好交代。但其实他知道不是。期中他三十一名。她第八名。中间差了二十三个人名。那些人都是名字——在白纸上排成一列——从第八到三十一。他想跨到离她近一些的地方。不需要跟她平齐——他知道现在不可能。他在心里算过——光物理这一科——他从八十二可能上到八十五——可能上到八十八——她把物理从六十八往上——他比她多了十几分物理——但英语、语文——她比他好——英语至少能拉十多分——加起来还是超他许多。所以三十一进到二十几——大概可以——不太离谱可以——但够近。

这个计算在他复习的时候反复出现——不是有意识的——是做物理题的时候——力学——算摩擦力方向——F=μmgcosθ——写完。想到那个数字——然后他继续往下做题。他告诉自己不是为了她——是为了提高——然后心里那个声音说"是"——他说好——那就算是——继续。就这个。

他每天五点四十五起床。宿舍楼的走廊灯是声控的——他踩着拖鞋啪嗒走过——灯"啵"地亮了——再走几步——到了水房门——后面的灯灭了。他得跺一下脚——灯再亮。刷牙——冷水——牙髓刺痛——牙龈又出点血。洗脸——水房的镜子上沾了牙膏点——镜子照人不太清楚——他把脸凑近——看到自己下巴右边那颗痘——红的——不是白色的——还没熟。下巴上那张脸在镜子照着——白——瘦了些——眼底下有青——是连续睡六小时不到的痕迹。

六点到教室。教学楼走廊空荡荡——只有楼道口有灯——到三层的时候还有七八米。教室里——前面一半灯亮着——后面一半暗——他走到自己的位子——倒数第三排——把头顶上后一排的灯打开——还是暗——因为灯管只有两盏在工作。坐下来。新开的第一件事是把昨天看的错题重翻一遍。物理。力学——斜面加滑轮——那题错了两次——最后把静摩擦系数和动摩擦系数的条件搞混了。重新做——第一遍——列受力分析——重力的正交分解——斜面支持力——拉力——解方程组——做完再验——对了。物理笔记本上是三色——蓝色是原笔记——黑色是补充——红色是重点标记——红笔只画关键公式的地方——牛顿第二定律——F=ma——框起来——红框。这本笔记本到现在已经写了快八十页——边角卷——封底的硬纸板开裂——用透明胶带补过一回。

六点半之后教室里的人开始越来越多。从一个人——变成五六个——变成十几个——大多数是住宿生。后来走读生也来了——推门的声音——"吱——"带进一阵冷风——门后的拖把杆晃了一下。

苏晚晴大概是六点五十到。

她进门的时候书包还没摘——一条书包带挂在左肩——右手拎着一个透明塑料袋——里面的包子还冒热气。走到自己座位——把包子放在桌上——取下书包挂椅背——坐下。然后双手去剥塑料袋——袋里有水珠——内壁白花花一层。包子——素的一样还是韭菜鸡蛋——咬一口——嚼——拿英语笔记本翻开。笔记本的封皮是深蓝的——四角都已磨损——白色的纸芯从底下露出来。她一边吃——一边用左手压笔记本的页——右手换着夹包子——换第三口的时候包子里的馅漏了一点——她用纸巾接住。把最后一口包子吃了——抽纸巾擦手指——五根手指一起擦——擦完又擦了嘴上。然后翻开物理笔记本。

顾临风每次都注意到这个。

她也在补物理。那个前面写了擦、擦了写的封面本。上一个期中考试——物理五十九——整本笔记都乱了——力学没建立起完整框架。现在她从头开始——重新列知识结构——牛顿三大定律——第一定律画了惯性圆——圆里用虚线做了表达。她的字比原来更大了——之前写的很小——现在是显目的正楷。

而她翻开笔记本的时候——没有人看——顾临风从四排之外的路偏了一眼——又回到刚才在做的化学配平。

考前一天。全年级停课复习。

教室里的气氛像拉紧的弦。没老师但比有老师安静。靠墙那排的人在低声念——像念经——嘴巴翕动不出声——身体微微前后摇。中间的那个女生——手上拿着政治书——手指在课本目录上——"第一节经济制度——第二节——"一边念一边拿指甲在字上划道。连赵一鸣都不怎么说话了——他趴在桌上做数学——偶尔冒出来一句——"这道题是不是选C——"然后自己在本子上验算——算了一会儿——"还真是选C。"翻一页——又安静了。

顾临风没有说话。他在算——在写——在背。英语单词——January、February——他早就会了——别的——accumulate——她旁边笔记本上写了这个——不知道是不是。化学方程式——氧化还原——配平。政治简答——改革开放的意义——四条一定要全写出来——少一条扣分——他妈的政治最难背——背了三遍——第一遍刚合上书就忘。

中间去上了个厕所——男厕所小便池的水槽坏了漏水——地下湿了一片——往旁边绕——墙上一块瓷砖掉了没补——洗手。回来——物理笔记本摊开——继续。下午自习的时候。苏晚晴的物理笔记本从桌面上滑下来了——笔记本边角擦到了过道的椅子——滑——啪——面朝下——封底的黑纸板对着灯。桌子边的男生帮她捡起来——手从过道上拾起——递过去。她接过去——侧头——嘴动了动——嘴唇形状——"谢谢。"或"好的。"然后翻开——找到刚才做的那页——压压平整——继续。

顾临风从四排外看到了整个过程——手下的笔只停了不到一秒。然后低头——在草稿纸上画第二个受力图。

考试三天。每天两门——上午一门下午一门——共六门——语文、数学、英语、物理、化学、政治。化学和政治合在"综合"。考场是按年级名次排的——第一天的语文考试——他被分在第五考场——就是原来高一三班——在熟悉的教室。她是在第二考场——高一四班。

语文作文——《桥》。给了四五行材料——记不全了——大概是关于流逝和时间——一座桥什么的。他看着题目——眼神在纸上停了挺久。然后落笔——写了他家门前那座石拱桥——桥下夏天有水——冬天是干的。写了桥面上那块被踩凹的条石——凹槽的位置在桥的正中间——日积月累——数不清多少人。写了他小时候和姐在桥下抓鱼——鱼很小——放进玻璃瓶——水草在瓶里飘——晚上鱼死了。写了桥头的早点铺——油条豆浆——炸油条的锅——黑铁锅——火是柴火——火烧得旺旺时锅里的油泡欢跳。没写别的。没提人。没提县城的河。收尾的时候——做了个处理——留白——跟桥下的冬天一样。停了笔。看了看字数——够了。

物理卷子最后一道大题是力学综合——斜面上有木块——木块连着绳——绳绕过滑轮——下面吊砝码——求加速度和绳子拉力。他列了三个方程——m₁gsinθ-T=m₁a——T-m₂g=m₂a——联立——解得T和a。写到第二步的时候橡皮找不到了——手在桌面上摸——左边——没有——右边——只摸到尺子——皱了眉——手往下探——在桌子底下摸了摸——手指在地板上——碰到了橡皮——捡起来。这中间可能过了几十秒——心跳快了两拍——然后继续——擦掉了一个错误的符号——改了方向。第三问——求砝码下落的位移——用匀加速公式——s=½at²——代入——算出来了。查了一遍——草稿纸用满了正反——反面用力的分解图。然后把答案抄上去——收尾——笔搁下。

考完最后一门——监考老师说"停笔——"所有人往后一靠——长出一口气——像整个考场同时松了气。有人把笔往桌上一拍——黑色水笔在桌上弹了一下——掉到地上——没捡——滚到前排的椅子腿下面。有个女生趴在桌上——把脸埋在袖子里面——不知道是不是在哭。有人在翻卷子对答案——"第八题你选什么——""C。""靠——我选了B。"顾临风没趴。他把笔放进铅笔盒——自动铅笔按了一下——笔芯收回去——圆珠笔盖上笔帽——橡皮擦干净。卷子翻过来压在铅笔盒底下——站起来——走出教室。出门的时候监考老师在门旁收卷——看了他一眼。他点了点头。老师点回去。走廊里全是人——有人抱在一起——议论——"化学最后那道附加我会做——写了一半——""英语作文题目是啥来着——""我都忘了——"也有人趴在窗台上往外看——不说话——眼睛空着。他从三楼走廊往下看。操场上有一滩一滩的积水——考政治那天下过雨。积水的边缘反着白光——太阳刚出来——光线打在水的表面上——刺——看不清楚水里有东西——水面晃了晃——是风吹的。

出成绩是三天后。那是最后一天——拿成绩单——然后放寒假。

公告栏上——红纸换了——"2010—2011学年第一学期期末考试"。纸与期中那回是同一批红宣纸——但字换了。人比当时少——有些人知道排名后直接走——有些人还没来。顾临风挤进人群——右肩碰了旁边人一下——"对不住——"然后站在栏前。这次他没从中间找。直接从此头的最上面——第一名往下——名字——名字——名字。

苏晚晴。第六名。物理71。英语96。语文93。数学90。总分的划线——在高中部和别的数——她进了前十——前进了两名。物理从68到71——涨了三分——对别人来说少——对她来说——不一样。因为期中就物理这一科拉总分的。

他往下找。有些名字很陌生——高一的竞争比初中激烈——他们班那次考第一名的姓刘的人原来是中考状元。前二十多排了几分。

顾临风。第二十四名。物理87。高了她十六分。英语79——退了——因为英语主要靠背——他背不是不行——而是不够——下学期需要翻倍。数学89——跟期中一样。其他几门差不多。

她第六。他二十四。距离短了——期中差二十三名——期末差十八名。在榜上的距离——大概隔了半张纸。他抬头看了一眼她的名字——第六——"苏晚晴"三个字——不是第一——不是第八——第六。然后低头看看自己的——第二十四——他的名字。钢笔字——圆珠笔——也是正楷——没她写的好——但也端正。

从三十一到二十四。进了七名。

他把手从红纸上拿开。指尖在红纸上留了个湿指印——纸面有汗。指印刚按的时候是圆的——然后纸吸收了水分——印迹在"顾临风"的名字中间变淡了——洇成一小块模糊。他看了一两秒——转身走了。

学校里已经有人拉着行李箱往外走了——轮子在操场水泥地上咕噜滚——水泥缝的间隙让轮子弹——"阁——阁——阁——"一个女生的箱子轮子歪了——拉着拉着摔在地上——然后又立起来。家长在校门口——有的推着自行车——后座绑着蛇皮袋——学生把学校里的棉被和鞋架往回扛。下午的太阳——冬天的太阳——黄黄的——照着空教室的桌椅。宿舍的楼道里有人在大喊——"我把你的MP3落在书架上了——你自己拿——"声音从四楼传到楼下——回响了几步。寒假开始了。
