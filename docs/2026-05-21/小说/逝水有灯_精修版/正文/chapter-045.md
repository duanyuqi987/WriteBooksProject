# 第045章 诗抄本多年后

所属部卷：第一部：初秋三楼
时间：高二秋天
地点：教室后排的课桌抽屉

## 诗歌插页 P045：原创七言小诗

诗本旧痕已泛黄，
一行碎语各收藏。
年少不知注脚重，
错将抄写作断章。

多年后陆允川重写诗抄本，先把自己写得很委屈——写到那本黑色硬皮本从高二到现在一直存在家里的抽屉里，每页诗底下的"她"都像是一道无法愈合的伤口。写完以后翻开旧诗抄本实物放在桌面上——黄色的纸已经开始泛黄，注释的字迹因为氧气变成了深褐色。他一页一页翻。翻到某一页时发现有一行自己写的是："今天她笑了一下——在听程砚讲笑话的时候。那个笑的方向是程砚——不是朝我的。但我听见了。"

他读到这句时停住了——不是愤慨，是释然。因为他在看到那行字的时候，旁边他已经在修改稿中补了一行后来的话："今天跟程砚发语音——他说那个笑话是什么他早忘了。他说当时他们刚认识一两个月,她笑是因为那笑点很低——不是你想象的那种有所取舍的'只对谁笑'。你想太多了。"他把那页旧注释翻过去。

多年后他的太太——不是夏知遥——在收拾东西时无意中翻了那本诗抄本。她看完了前言和第一页，然后翻到后面一点发现了那些注释。她没有问问题。她说了一句让陆允川这辈子最后才准备的回答——"这些注释不是失恋，是当时的年轻人把注意力集中在别人身上的能力——是很好的能力。你现在还有这种能力吗？"陆允川想了想说我现在的观察点比较分散——再也不会集中在一个人身上。太太说："所以当时你能集中——也是很难得的。不要觉得集中是错误。"

叶澄在他的终稿里批了一个字——一个只在他这一章诗抄本终稿里批的字："成了。"她说不是成了爱情，而是成了自我和解。之前一直写诗抄本的故事像是在构建一个单人纪念馆——每首诗的注释是碑文。而现在他终于把这座单人纪念馆开放成了一个普通历史展览——允许别人进来看、允许自己走出去。诗抄本不是用来纪念她的——是用来纪念他自己年轻时的观察耐心的。那些注释里提到的所有"她"都不是她——她只是一个载体。真正被记录的是他自己的专注——那种十七岁才有的专注，把世界上所有的细节都分配给一个人。

他合上诗抄本实物，把这本旧本装进抽屉，然后在修改稿最后一页写上致辞：这本诗抄本是献给十六到十七岁的高中生陆允川——献给那个坐在教室后排、手里握着诗抄本硬皮、把每首古诗翻译进自己暗恋里去了的年轻人。他不是在抄诗，他是在发明自己情感历程的第三种语言——这种语法很绕，但对他来说足够有效。谢谢那些诗，谢谢那个时候的他自己。
