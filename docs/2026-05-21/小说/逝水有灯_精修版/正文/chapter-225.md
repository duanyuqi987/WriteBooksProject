# 第225章 重回县一中多年后

所属部卷：第五部：逝水有灯
时间：多年后
地点：陆允川书房

## 诗歌插页 P225：原创仿唐绝句

旧孔微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

多年后陆允川重写重回县一中那一章的时候，发现自己留了一个不该留的词。

原稿写着："他终于不再把学校写成只属于他的遗址。"

他删掉了"终于"。

然后删掉了"遗址"。

改成："他把学校写成一个还在被使用的地方。走廊里有新的学生走，饮水机换成温档的，扫把柄是红色的。"

又读了一遍。删掉了"还在被使用的"——因为"地方"本身不需要被界定。地方就是地方，有人的地方自然会有人。

最终定稿是："他把学校写成一个地方。走廊里有学生，饮水机里有温水。"

多一个字都不要了。

程砚来他家吃火锅的时候，聊到了高中同学聚会的事。是班级群发的通知，说今年毕业十周年，想组织一次聚会。地点定在学校附近的一家农家乐。

"你去不去？"

"去。"陆允川往锅里下了一片毛肚，"你呢？"

"我也去。我已经在群里接了龙。"

"多少人了？"

"到现在是十八个。班上一共五十五个。十八个不算少。"

"也是十年了。能来三分之一已经够多了。"

"你心态真好。"

"有人不来，可能是因为真的有事。也可能是觉得来了不知道说什么。"

"你觉得'不知道说什么'也是理由？"

"是。而且是很多人最重要的理由。"

火锅咕嘟咕嘟地滚着。程砚下了两片年糕，筷子搅了几圈。

"你有没有想过一个问题。"他说。

"什么？"

"大家不来同学聚会，可能不是因为不想见老同学。是因为不想面对一个问题——'你现在过得怎么样。'"

"'你现在过得怎么样'怎么就成了问题了？"

"因为大部分人的'怎么样'都不够好。不够好到能拿来当十年的交代。但也不差。就这么熬着。不成也不败。这种状态最怕被问。"

"所以你怕被问吗？"

"不怕。"程砚把年糕捞出来，"因为我知道你肯定会帮我回答。'程砚啊，他挺好，单位稳定，体重长了十斤。'这种回答他们不会追问。"

陆允川笑了。"那我的回答谁来帮？"

"你自己。"程砚认真地看着他，"你不是有这本书吗。他们问你现在在干嘛——你说我在写书。然后话题就拐到你身上了。完美避过。"

"所以我的书的功能之一是帮你避过同学聚会？"

"对。我在享受你写作的红利。不用谢。"

火锅吃完以后，两个人坐在阳台上。外面下起了小雨，雨不大，打在遮阳棚上声音很碎。

"你最后决定怎么写结尾了？"程砚掏出手机看明天的天气预报。

"还没有。但他心里有一个轮廓。"陆允川看着雨打在棚上又滑下去，"大概就是——没有总结。没有'他终于'。只有生活继续的声音。"

"比如什么？"

"比如自己说的话。水开了。电梯到了。群消息跳出来。"

"螺丝孔呢？"

"螺丝孔留在中间。不是结尾。是一路上偶然发现的——原来旧的还在，新的也没把旧的毁掉。"

程砚收起手机。"你这句说得不错。不要改了。"

"不改。但这种句子不能放在结尾。放在结尾就太像答案了。"

雨下大了。遮阳棚上的雨声从碎变成了整——一片一片地往下倾。阳台扶手上积了一小滩水，然后漫出来，滴到楼下的花圃里。

"这部小说，你觉得算成功还是失败？"程砚突然问了一个他从没问过的话。

陆允川没有直接回答。他想了一会儿，看着雨。

"不知道。写的时候没想过成败。只想过：对不对——是不是事实。是不是每一个人的事实。不是只有我一个人的。"

"那结果呢？"

"有些地方对。有些地方可能还是偏了。但偏了也没法再改——因为我已经离开那个年纪了。现在的我改不了过去的我。只能把过去的我写清楚。写清楚就走了。"

雨停了。阳台上的积水不再滴。遮阳棚边缘挂着一排水珠，在灯光下像一条不完整的省略线。

程砚站起来伸了个懒腰。"走了。火锅锅你洗。我明天早班。"

"你打车回去？"

"骑车。共享单车。雨停了。"

"小心路滑。"

"知道了。"程砚走到门口，换上鞋，拿起门口伞架上的雨伞——没用上。

开门的时候他回了一次头。"你觉得十年以后我们还会坐在这里吗？"

"会。但火锅可能换铜锅了。"

"铜锅好。铜锅涮肉好吃。"门关上了。

陆允川回到阳台上，雨已经完全停了。远处的云散了，露出几颗星星。他把火锅锅泡在水槽里，加了洗洁精，开热水。水龙头流出来的声音在安静的厨房里格外清楚。

他一边洗碗一边想：四十四个章节改了三年。还差最后一章。明天写完它。

水停了。锅还在滴水。他把锅扣在沥水架上。

明天见。
