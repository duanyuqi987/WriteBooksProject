# 第089章 大一下

时间：2014年2月

地点：楚州师院

卷次：卷五：各自南北

二月十七号。大一下学期开学。

从青阳回到楚州。他坐的上午十点的大巴。车上人不多——春节的返乡潮已经过了。他靠在窗边，看着高速公路边的田野——青灰色的天空下面，麦田是黄绿色的，很矮。

楚州的冬天比青阳干。回了宿舍，他第一个到。打开门的时候屋里闷了一口气——窗帘拉着的，地上有灰。他把窗户推开。冷风灌进来，吹得窗帘往里面鼓。

室友们陆续到齐。李博带了一大包洛阳特产——牡丹糕和花生糕。阿坤带了一台新的机械键盘——青轴的，说这学期要认真打游戏，"上钻石"。小伍带的行李箱最大——里面是被子、棉袄，还有他妈塞的十包榨菜。

"你妈把你当兵送了。"阿坤说。

"她说食堂菜不够咸。"

新学期课程比上学期多。古代文学进入魏晋南北朝。大学英语加了听力课。选修课还在上——书法课交了一次作业，他写了颜体《多宝塔碑》的六十个字，老师评了一个"A——"。

他开始参加社团。

第一个是文学社。师院的文学社名字叫"流萤"——社长是个女生，大三，叫刘思齐。她在社团招新会上讲了一番话："文学不是宏大叙事，是用细节留住时间。"顾临风听完填了入社表。

第二次文学社活动在一个小教室里。二十来个人。主题是"现代诗创作"。刘思齐在白板上写了三个词——"呼吸""骨头""距离"。让大家自由写。给了半个小时。

他在本子上写了几句。然后划掉了。重新写了四行。第一行是"走廊是一条横着的骨头"。写完以后自己看了几秒钟。没有交。

轮到分享环节。一个女生走到讲台上朗读自己的诗——读得很认真，声音有点发抖。读完大家鼓掌。顾临风跟着拍了两下手。

散场的时候他收拾东西。那个读诗的女生走过来。"你刚才没分享。"

"没写完。"

"没关系。下次可以分享。"

"嗯。"

她笑了一下。扎着低马尾，穿一件咖啡色毛衣。"我叫许若琳——英语系的。"

"顾临风。中文系的。"

"中文系来文学社——专业对口。"

"差不多。"

许若琳往门口走了两步，忽然回过头来。"对了——你认识苏晚晴吗。"

顾临风的手放在书包拉链上。停了半秒。

"认识。"

"你怎么认识她？"

"高中同学。"

"'哦——青阳一中的对吧。我跟她是大学室友。她跟我提起过你们学校。她说你们学校有不少人考到楚州来了。"

"嗯。"

"你们还有联系吗。"

顾临风把书包拉链拉上。声音很平。

"很久没联系了。"

"哦。"许若琳点了点头，没有追问。"那再见。"

"再见。"

他走出教室。走廊里的灯是白的。地砖是白灰相间的方格——每一格的边缝都嵌着黑色的灰。他在走廊尽头停了一下。窗外是师院的北操场。有几个人在跑步。跑道是红色的。

许若琳刚才说的那句——"她是我的大学室友"。也就是说——如果他想知道苏晚晴的消息——可以从这里知道。

但他没有问。

他把书包背带往上拽了拽。然后往食堂方向走。

下午去了书法社。交了一年的社费——二十块钱。社长是个戴圆框眼镜的男生，清瘦，讲起颜体和柳体的区别滔滔不绝——"颜体筋健、柳体骨劲"。顾临风听得仔细。毛笔蘸墨的时候手还是有点抖。

晚上回到宿舍。李博在看书，阿坤在打游戏，小伍在给家里写信——这次的信封换了一个印着长颈鹿的。

"今天见了一个人。"顾临风突然说。

三个人都停下了。

"女的？"阿坤把耳机摘下来。

"苏晚晴的室友。"

"谁？"

"许若琳。她说她是苏晚晴的大学室友。"

阿坤眨了眨眼睛。"所以呢。"

"没所以。"

"你说'没所以'——那就是有所以。"

顾临风把鞋脱了，放在床底下。"就是碰巧遇见的。"

李博没有参与话题。但他看了一眼顾临风的脸色。然后看了一眼阿坤。阿坤领会了班长那个眼神的意思——别问了。阿坤把耳机重新戴上，继续打他的副本。

顾临风躺在床上。从枕头下面拿出手机。翻到通讯录——许若琳——新加的。电话号码是今天社团活动结束前她主动给的——"以后社团有什么活动互相通知一下。"

他看了几秒钟。把屏幕按灭了。
