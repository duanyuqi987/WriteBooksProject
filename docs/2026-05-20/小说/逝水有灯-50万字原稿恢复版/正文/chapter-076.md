# 第076章 毕业照边缘之前

所属部卷：第二部：六月以后
时间：六月中旬
地点：学校操场主席台前

## 诗歌插页 P076：原创现代诗

我把水放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
照片边缘被风吹起的校服袖口没有替谁回答，
只把那一年轻轻照亮。

毕业照边缘真正到来之前，日子已经先变了颜色。六月中旬的操场晒得发白，主席台前临时摆了几排凳子。摄影师拿着喇叭喊高个往后，矮个往前，别挡脸。

陆允川被挤到最边上，脚尖踩着白线。他整理校服拉链时，风把袖口吹起来，像照片还没拍，就已经先开始告别。

他还没有遇到那个必须面对的瞬间，却已经被照片边缘被风吹起的校服袖口提醒：有些事不能无限期地放在心里，它会跟着时间一起变形。

程砚站在陆允川后面，弯腰小声说：“边上也挺好，至少以后翻照片不用找半天。”

叶澄帮前排女生扶了一下歪掉的凳子，又提醒大家别踩到横幅，语气像这场告别也需要有人维持秩序。

夏知遥站在第二排中间，旁边是林曼和叶澄。她听摄影师喊看镜头，便把散下来的头发别到耳后，笑得很浅，却很认真。

摄影师喇叭里的杂音、同学起哄声和快门声，把他的心事压得没有那么夸张。人在这样的声音里，很难一直把自己当成故事中心。

快门按下前，有人突然笑场，摄影师叹气说再来一张。陆允川也跟着笑，笑的时候没有看夏知遥，只看见阳光在镜头边缘亮了一下。

照片拍完，大家一哄而散。有人找老师合影，有人抢着和朋友勾肩，陆允川站在边上，忽然不知道该往哪个小团体里走。

那时陆允川还不知道，所谓成长不是突然明白，而是先在蓝色凳子、班级横幅、相机三脚架这些小东西旁边学会停顿。

他在日记本里写了两行，又划掉一行。留下来的不是誓言，只是一个具体下午：操场边小摊的冰棍甜味、窗外的光，以及他没能说出口的话。

陆允川后来愿意留下这一笔，是因为它不漂亮。蓝色凳子不漂亮，班级横幅也不漂亮，可它们让人相信，这段青春不是从空中落下来的。

他没有再给夏知遥安排额外台词。她已经在操场上完成了自己的动作，剩下的沉默，应当由陆允川自己消化。

程砚把话说得粗，叶澄把话说得稳，两种声音都把他往现实里拉。陆允川当时不领情，很多年后却知道自己被救过几次。

那天的操场边小摊的冰棍甜味没有特别好吃，却因为太具体，抵住了那些过大的形容词。人有时候正是靠这些具体的味道，才没有被回忆吞没。

大家一会儿喊老师合影，一会儿喊朋友别跑。这些声音把毕业照边缘从一个人的心事里拉出来，变成许多人共同挤过的一天。陆允川不再只听见自己胸口的动静，也听见椅子、脚步、塑料袋和远处的喊声。

夏知遥站在第二排，听见快门前把头发别到耳后。这个动作很小，却让她从陆允川的想象里退回到真实的人：她会忙，会累，会被家里和未来推着走，也会在必要的时候保持沉默。

程砚站在他后面小声吐槽太阳太毒，说话还是老样子，不好听，却有效。陆允川皱了皱眉，还是照着他说的做了一点，因为朋友有时不是负责安慰，而是负责把他从自我感动里拽出来。

叶澄帮前排女生扶正凳子。她没有把话说得太满，只把边界摆到桌面上。陆允川那时不一定立刻懂，却会在后来某个晚上想起这句话，知道自己当年被提醒过。

照片发到家里后，母亲把手机举远了看他说你站得太边。家里的声音一进来，青春就不再只是校园里的风。它变成饭点、车费、睡眠、行李和一句不太会表达的关心。

他低头看着毕业照边缘，忽然意识到它不是命运给他的标记，只是那天确实存在的东西。正因为确实存在，它比任何夸张的形容都更能留下来。

操场热塑胶和冰棍纸上的甜味混在一起，让记忆有了可以落脚的地方。陆允川后来写到这里，没有再急着解释自己多难过，而是先把气味、光线和手上的动作写清楚。

他把校服拉链收好，又拿出来检查了一遍。这个来回不像英雄，也不像受害者，只像一个年轻人不知道怎样安放自己，于是先安放手边的东西。

真正的变化还没到来，操场主席台前已经先显出紧绷。陆允川站在里面，既想让日子快一点，又害怕快起来之后自己没有准备好。

他在日记本上写下“等以后”，写完又觉得“以后”太轻。很多拖延都披着克制的外衣，其实只是害怕被现实拒绝。

这一笔补进去以后，章节没有变得更响，反而更稳。毕业照边缘不再只是一个标题，而是有毕业照边缘、有校服拉链、有人说错话也有人及时停住的真实一天。

摄影师喊三二一时，有人眨眼，有人没看镜头。陆允川后来反而喜欢这一版，因为它不像纪念册样张，更像他们真实站在六月里。

夏知遥从凳子上下来时差点踩到横幅边，叶澄伸手扶了她一下。两个女生低声笑了笑，陆允川在远处看见，忽然觉得她们之间的亲近比他想象的故事更真实。

程砚抢过他的手机，说我给你拍一张单人的，省得你以后只会放大集体照。陆允川骂他多事，却还是站到主席台边，让阳光照了满脸。

操场里的光线一点点变暗，班级横幅和冰棍纸都还在原处。陆允川把校服拉链收好，动作慢得像怕惊动什么。可真正需要被惊动的不是别人，是他自己。

他后来重写这一章时，特意把毕业照边缘留在中间。不是为了让它变成象征，而是因为人只有摸到具体东西，才知道自己曾经怎样站在具体生活里。

事情尚未发生，陆允川已经开始给自己找退路。他把“以后再说”想得很体面，可操场里的每一种声音都在提醒他，退路拖久了也会变成墙。

章节落下时，拍毕业照时陆允川站在最边上，夏知遥站在第二排，照片里两个人同框却离得很远。还没有真正展开。陆允川只是把东西收好，跟着人群往前走，心里那点执拗被现实轻轻推了一下。
