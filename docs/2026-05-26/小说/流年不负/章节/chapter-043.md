# 第043章 进步

时间：2011年11月

地点：教室

卷次：卷三：此心藏久

十一月中旬。期中考试。

三天。六门考试。每天上午一门、下午一门。考完最后一门生物的时候，顾临风从考场出来，靠在走廊墙上。站了一会儿。

赵一鸣在后面拍他肩膀。

"考得怎么样。"

"还行。"

"你上次说还行的时候考了三十一。"

顾临风没接话。把笔塞进铅笔盒里。铅笔盒的铁皮角磕了一个凹槽，不知道什么时候磕的。

等成绩的那三天，他该干嘛干嘛。上课。做题。晚自习。打手电筒。每天睡五个小时。有时候四个半。早上起来洗脸的时候，凉水打在脸上。眼睛睁不开。眼皮像粘在一起了。他用手指把眼睛撑开。毛巾擦了一把脸。眼眶下面青了一小块——像被人用指腹按了一下。浅青色。

赵一鸣说他眼眶青了。

"没睡好。"

"你再这样。我真跟你妈说。"

顾临风看了他一眼。眼神不凶。但就是那种——你试试看——的眼神。

赵一鸣闭嘴了。把快到嘴边的话咽了回去。然后把一根棒棒糖塞到他手里。"橘子味的。补充维C。""橘子味的是维A。""都一样。"

成绩公布的早晨。

第二节课下课。班主任老刘拿着一沓成绩单走进教室。白色打印纸。排名表格。从第一到最后一名的名字打印得整整齐齐。

老刘把成绩单贴到黑板旁边的墙上。用胶带粘住四个角。

"期中考试成绩出来了。自己看。"

呼啦一群人就围上去了。前面的同学挤在成绩单前，后面的踮着脚往前面看。有人在找自己的名次，有人在看别人的分数。

顾临风没有马上过去。

他在座位上多坐了一会儿。把桌上的橡皮屑用指尖拢到一起，拢成一小堆。然后吹掉。

站起来。走到黑板旁边。从人群的外围往里看。

成绩单上的字很小。名字后面跟着六门课的成绩——语文、数学、英语、物理、化学、生物——然后总分、排名。

他从上往下找。

第一名：李明泽。总分六百二十三。

第二名。第三名。第四名。第五名。第六名——苏晚晴。

总分五百九十七。

第六名。

顾临风的目光在她的名字上停了一下。五百九十七。班级第六，年级前八十。跟期中差不多，稳住了。

然后视线继续往下。

第十五名。

第十五名——顾临风。

总分五百一十二。

他看了两遍。

不是看名字——名字不会变。是看数字。语文九十三。数学九十八。英语八十九。物理七十八。化学七十六。生物七十八。总分五百一十二。班级第十五。

从三十一到十五。

跳了十六名。

他站在人群外面。右边有人在说"我掉了五名"、"这次数学好难"、"英语超纲了"。左边有人在笑。

他没笑。

但他的眼角动了一下。很细微。右边眼角往上提了一点点。只一瞬。

第三节课是数学课。数学老师姓孙，是个女的，短头发，戴着方框眼镜。说话很快。卷子发下来后，她把平均分、最高分、最低分念了一遍，然后说了一句。

"有几个同学这次进步比较大。"

她翻了翻成绩单。

"顾临风。数学从七十一到九十八。进步了二十七分。"

全班安静了一下。

有人回头看了他一眼。后排有男生说了一声"卧槽"。前面的女生也在嘀咕。

顾临风坐在自己的座位上。手放在桌面上。右手的食指在桌面上按了一下。

老孙继续说。"这次卷子比期中难度还高一些。能进步二十七分，说明平时用了功。"

"继续保持。"

他没抬头。但他知道——赵一鸣在旁边用胳膊肘撞了他一下。撞得很轻。他没反应。

老孙开始讲题。

顾临风翻开卷子。九十八分的卷子上，有几道题用红笔打了叉。他盯着那些叉看了很久。然后拿起笔，在草稿纸上重新算了一遍。

下课了。

大家都在讨论成绩。有人高兴，有人沮丧。后排的赵强把卷子揉成一团扔进垃圾桶，然后骂了一句"艹"。

顾临风站起来。走到教室后面。从后门出去。站在走廊上。

走廊里有风。十一月的风。凉。他靠着墙。手揣在校服口袋里。

他在想一件事。

成绩单上——苏晚晴旁边写着第六名。总分五百九十七。他十五名。五百一十二。差距八十五分。从三十一到十五，他跳了十六名。但第六名到第十五名——中间还有八个人。

还很远。

但他看到了。

从三十一到十五，是第一次。她说的那句话到现在，四十天左右。四十天，十六名。他在脑子里算了一道没什么意义的算术题——如果每四十天跳十六名，要追上第六名需要多久。

他没算出结果。

但他也没觉得灰心。

下午。成绩单旁边又围了一群人。课间十分钟，总有人在前面看。有人拿手机拍照。有人在用手指划着找自己的位置。

苏晚晴也在那儿。

她去接水回来的路上，在成绩单前停了几秒。水杯端在手里，透明的杯子里是半杯热水。热气从杯口冒出来。

她的眼睛在成绩单上移动。

她在找什么，不知道。

她的手指在成绩单上划了一下——从第六名那个位置往下。经过第七、第八、第九……第十一、第十二……第十三十四——

第十五。

手指停了。

大概停了半秒。

然后她收回手指。端着水杯走回座位。

坐下。翻开物理课本。她的马尾没有什么表情。但她喝水的时候，好像嘴角动了一下。

顾临风没有看到这个动作。

他在最后一排，被人挡住了。他只看到了她的马尾歪了一下——好像在往成绩单方向偏了偏。然后就不动了。

赵一鸣倒是看到了。

他什么也没说。但下课后，他趁顾临风不注意，在成绩单上用指甲在"第十五名"旁边划了一道浅浅的印子。跟顾临风在名单上划苏晚晴名字时一模一样。

晚上熄灯后。

手电筒亮着。赵一鸣给的电池已经用完了。现在是第三组电池——他自己买的，还是一块钱的杂牌。光有点弱。练习册上的字得凑近一点才看得清。

顾临风做的不是新课。是把期中考试的所有错题全部抄在一个新本子上。用红笔标了知识点。用蓝笔写了解题步骤。在最下面用黑笔写了一句话。

"八十五分。"

然后翻过这一页。继续做新题。

一道力学题。一道电磁感应题。一道生物遗传题。一道化学摩尔计算题。

手电筒的光越来越弱。快没电了。他没有停。做一道，对一道。错一道，抄下来，再做一遍。凌晨一点。手电筒灭了。他把练习册合上，放到枕头底下。闭上眼。

第二天早晨起床。洗脸的时候发现嘴唇干裂了。拿水冲了冲。没管。

刷牙。背单词。吃馒头。到教室。

苏晚晴已经在背英语了。她的声音不大。但整个空荡的教室——她是第一个到的。她的声音在空椅子之间回响了一下。然后散了。

顾临风坐到座位上。把昨晚做的错题本翻开。第一页。第一道错题——摩擦力方向判断。

他看了一遍。然后拿起笔，在题目旁边写了一个更简短的解法。字很小。笔画很用力。
