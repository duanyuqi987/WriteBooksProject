# 第063章 高考后的水之后

所属部卷：第二部：六月以后
时间：2012年六月八日
地点：回镇上的班车

## 诗歌插页 P063：原创仿宋小令

水半瓶，日微明，
车窗摇过县城声。
问少年、考场已远，心事仍轻。
母亲低问饿不饿，
一路尘风不忍听。

回镇上的班车很挤。

高考结束的人、陪考的家长、去县城办事的老人，都挤在同一辆车里。车窗半开，热风卷着灰尘往里扑。陆允川坐在靠窗的位置，手里还握着那瓶没喝完的水。

母亲坐在旁边，腿上放着装准考证和文具的塑料袋。她问他想吃什么，他说都行。母亲说家里炖了排骨，又说你爸上午就把东屋收拾出来了，回去可以睡一觉。

陆允川嗯了一声。

手机震了一下，是程砚发来的消息：晚上七点，老地方，网吧门口。

陆允川回：再看。

程砚立刻回：你这个“再看”我已经看透了。

陆允川笑了一下。母亲看见，问：“同学？”

“程砚。”

母亲点点头：“玩可以，别太晚。”

车经过河桥时，陆允川把水瓶放到窗边。阳光透过瓶身，水面晃了一下。他想起校门口夏知遥点头的样子。那点头很轻，轻到不能被解释成任何东西。可正因为轻，他才反复想。

班车晃得厉害，母亲靠着椅背闭上眼。陆允川看着她眼角的细纹，忽然有些愧疚。三年里他把太多情绪放在一个名字上，却很少认真看见母亲在校门外等他的样子。

他把水喝完，空瓶捏在手里，塑料发出轻响。那个声音不美，却很像高考结束后的真实：一点凉意，一点空，一点不知道接下来该做什么的响动。
