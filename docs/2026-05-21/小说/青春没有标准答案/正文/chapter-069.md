# 第069章 一张小纸条



时间：2009年一月底，成绩单发放前一天



地点：十班教室、后墙红榜旁



卷次：第三卷：雪落在两栋楼之间



2009年一月底，成绩单发放前一天，十班教室、后墙红榜旁，马晓彤清单本里夹着的纸条先把这一天按进了十班的日常里。马晓彤把寒假计划写成密密麻麻一页，常俊凡悄悄塞给她一张少一点也可以的纸条。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

段璟看见纸条一角，终于没有追问纸条内容，也没有把它变成自己的情绪。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看马晓彤清单本里夹着的纸条，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

马晓彤也在这一天被推到前面。马晓彤的特点是清单型学生，凡事写条目，期末前尤其紧绷。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“我寒假每天排到晚上十一点。”马晓彤说。

“你这是放假还是服役？”宋乔说。

“计划太满，坏一天就全乱。”常俊凡说。

“少一点也算计划。”段璟说。

“会休息的人，才有力气继续写。”叶书兰说。

小纸条在十班不是稀罕事。有人传题号，有人借橡皮，有人写下课去不去水房。

真正麻烦的是，纸条一旦被起哄，就会从一件小事变成谁也不好收拾的热闹。

段璟看见纸条角露出来时，第一反应不是好奇，而是上一次被误会后常俊凡皱起的眉。

他把纸条递回去，手指没有多停。这个动作太普通，普通到旁边人找不到起哄的缝。

宋乔看了他一眼，像在确认他这次终于学聪明了。

付小川张了张嘴，又自己闭上，把玩笑咽回去，低头装作研究橡皮屑。

常俊凡接过纸条时只说了声谢谢，声音不大，却把边界放得很清楚。

十班从这些小事里慢慢学会，热闹不是想要就要，别人的难处也不是拿来传的。

一张小纸条没有悬在空处。小纸条折痕旁边沾着的一点粉笔灰，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，一张小纸条带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

叶书兰进门时不一定立刻说话，鞋底在水泥地上一停，一张小纸条带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在马晓彤快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。一张小纸条越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。马晓彤清单本里夹着的纸条让他想说话，又逼他先低头看题。

马晓彤被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，一张小纸条留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。马晓彤路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为一张小纸条翻身，谁还在被窝里背单词，旁边人都听得见。

县城晚上暗得早，小店门口的灯一亮，纸条上的玩笑就显得没有白天那么轻。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理一张小纸条里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：马晓彤清单本里夹着的纸条、一只冻红的手、一句老师点名、一块还没扫干净的雪。

韩泽把纸条展开又折回去，折痕正好压住那一行字。十班的热闹没有停，只是换成了更低的声音。

何南星看见旁边有人起哄，先把卷子往自己这边拉了拉。这样的细节没人会写进成绩单，却会留在同学之间。

邓桥在草稿纸上补完公式，才把话压低解释给同桌听。窗外风一过，纸页轻轻动了一下，像替谁把话咽回去。

余小满把小纸片塞进错题本，像给这场误会找了个安静位置。这个小动作很快过去，却让教室更像一个真正在运转的班。

马晓彤没有因此变成特别醒目的人，可在段璟眼里，马晓彤从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，这张纸条把第三卷从热闹拉回温柔，提醒每个人都不必把自己逼到极限。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
