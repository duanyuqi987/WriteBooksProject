# 第105章 诗社传单多年后

所属部卷：第二部：六月以后
时间：多年后
地点：图书馆前广场

## 诗歌插页 P105：原创仿唐绝句

纸上红痕淡欲无，
秋风旧字隔前途。
若知相识皆寻常，
何必多年重叹书。

多年后整理旧书，陆允川在《现代汉语》里翻出那张诗社传单。纸已经发黄，折痕处快要断开，社团印章只剩一团浅红。

他把传单摊在桌上，第一眼看见的不是苏晚枝，而是那句“把生活写小，把心事写实”。旁边还有他当年用铅笔写的活动时间，字很硬，像怕自己忘记。

旧稿里，他曾经把这张传单写成命运递来的门票。现在再看，它只是社团周里一张很薄的纸。发传单的人负责发传单，报名的人负责报名，风负责把纸吹皱。没有哪一部分天然要替后来承担意义。

他没有把传单丢掉，也没有重新夹回书里。他找了一个透明文件袋，把它和第一首《入学第一周》放在一起。这样放更准确：那不是爱情的证据，而是一个年轻人开始练习把日子写小的证据。

窗外有学生经过，笑声从楼下传上来。陆允川看着那张旧纸，忽然觉得苏晚枝当年的礼貌终于回到它原来的大小。它很好，但它不等于承诺。
