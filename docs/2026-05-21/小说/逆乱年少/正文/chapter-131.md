# 第131章 星光

时间：2012年7月10日，夜晚

地点：李家沟村小操场

卷次：第六卷：雪落无声

山里的夜晚和城市的夜晚是两种东西。

城市里的夜晚只是白天关了一半的灯。山里的夜晚是真的全黑了——方圆几十里没有一盏路灯，唯一的光源是天上的星星和教室里那盏煤油灯。

山风从沟底翻上来，带着泥土和艾草的气味。操场边上的那棵老槐树被风吹得沙沙响，叶子摩擦的声音在夜里显得格外清楚。林玉起能分辨出至少三种虫子的叫声——蝈蝈那种拉长音，蛐蛐那种短促的啾啾声，还有一种他叫不上名字的虫子，叫声像一根绷紧的琴弦被手指轻轻拨了一下。

林玉起在操场上写诗。

他坐在地上，背靠着乒乓球台——水泥砌的，台面裂了两道缝，中间横着一排缺了角的红砖当球网。晚上不用上课，他把小本子摊在膝盖上，借着星光写字。

写到一半，听见身后有脚步声。

是苏安晴。她穿了一件薄外套，头发散着，刚洗过，还没有完全干。

"你每天坐这儿？"她问。

"也不是每天。"

她在乒乓球台另一端坐下。两个人之间隔着一张球台的距离和一个歪歪扭扭的砖头球网。

"在写什么？"

"乱写。"

"给我看看。"

林玉起犹豫了一下。苏安晴没催他，只是把手伸过来，手心朝上，搁在乒乓球台上。

他就把本子递过去了。

苏安晴接过去，凑近了看。星光不够亮，她得眯着眼。

看了一会儿。没说话。

"写得不好，"林玉起说。

"我没说不好。"

她把本子翻了一页，又看了一会儿。

林玉起看着她的手指在纸面上划过，指尖在星光下显得很白。他忽然有点后悔把本子递给她了——不是因为写得不好，是因为那些字里有一半都和她有关。

"你写诗的时候在想什么？"

"什么都不想。就是看到什么写什么。山、星星、风、虫叫。"

"还有呢？"

林玉起看了看她。星光下她的脸有一半在暗处，有一半被照得很白。

"还有人。"

苏安晴没接话。她把本子合上，还给他。

"写得挺好的。"她站起来，拍了拍裤子上的土。她拍土的动作很用力，手掌在裤子上啪啪响了两声——好像不只是在拍土，也是在拍走某种一闪而过的情绪。"但你可以写得更直接一点。"

"什么叫更直接？"

她想了想，说："就是——不要老是躲在一堆意象后面。你想说什么就说。"

她说完这句话就走了。脚步不紧不慢的，薄外套被夜风吹起来一个角。

林玉起看着她的背影消失在教室拐角处。她走路的姿势他已经很熟悉了——微微低着头，步子不大，两只手习惯插在口袋里。这个姿势在多少个课后的瞬间见过——她从图书馆出来，从教学楼出来，从食堂出来——他的目光总会不自觉地跟上去几秒钟，然后在她拐弯的时候收回来。

他把本子翻到新的一页。

对着星光想了很久。

最后写了两行。没有用任何意象。

>
山高月近人，夜静风先语。
纸上三千字，不如一句真。
>
