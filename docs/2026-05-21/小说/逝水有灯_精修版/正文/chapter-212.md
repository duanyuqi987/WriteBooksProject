# 第212章 南方出租屋

所属部卷：第五部：逝水有灯
时间：毕业第一年
地点：南方城市出租屋

## 诗歌插页 P212：原创仿唐绝句

白痕微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

事情落到当天，没有被命运单独照亮。

陆允川在出租屋住了两个月之后，终于给墙上的那块剥落白漆拍了张照片。

不是特意拍的。是有一天他在整理电脑，发现桌面上有两个文件夹并排躺着：一个是"大学照片"，一个是"小说稿件"。两个文件夹的名字都普通得像工作资料，没有任何情绪标记。他点开"大学照片"，翻了几张，翻到毕业照——学士服，图书馆台阶，几十个人的脸挤在六寸的像素里。

他关上文件夹，用手机给墙上那块白漆拍了张照。没有构图，没有滤镜。拍完以后放进了一个新文件夹，命名为"出租屋"。

隔壁搬来了一对情侣。男的在对面工业园上班，女的是超市收银员。隔音不好，能听见他们讨论今天吃什么、煤气费谁去交、电视剧下一集什么时候播。

有一次男的过来敲他的门，问能不能借用一下改锥。陆允川递给他一把，男人站在门口看了一眼他的房间。

"你一个人住？"

"对。"

"你这墙上那块漆怎么不补？有那种小罐装的补墙膏，十块钱一瓶。"

"懒得弄。"

"不嫌难看？"

"习惯了。"

男人拿着改锥走了。五分钟后还回来，顺便给了他一瓶没开过的补墙膏。"超市打折买多了，你用吧。"

陆允川接过瓶子，说了谢谢。那瓶补墙膏在他书桌上放了三天，最后被他塞进了抽屉最里面。

他没有补那面墙。

不是因为懒。是因为有一天晚上，他写到凌晨两点，抬头看见那块剥落的白漆在台灯底下投出一道很细的影子，形状像一个不完整的括号。他突然觉得，补上就少了点什么。不是少了那块漆，是少了那两个月——那些住进来以后才发现距离只能让人少碰见、不会自动清空的夜晚。

楼下的早餐摊发生了一件事。卖豆浆的老板娘记住了他要少糖。不是嘴上说"好的好的"然后照常加糖，而是真的只加半勺。他喝到第一口不甜的豆浆那天，在早餐摊前站了好几秒。

"今天的够不够少？"老板娘问。

"够。正好。"

"那就行。上次你说太甜，我后来试了一下，正常配方确实太甜了。"

陆允川端着豆浆上楼的时候，忽然想起程砚说过的话：理直气壮地退回去。他没有退。但对方主动改了。他不知道这算不算进步，但至少这杯豆浆比之前的任何一杯都好喝。

公司里的日子不好不坏。他的工位在办公室最里面，旁边是个爱养多肉的中年女同事。她在他入职第一天给了他一片多肉叶子，说放在土上就能生根。他把叶子放在办公桌角落，三个月过去了，叶子没有生根，但也没有枯萎。就那么瘪着。

"你这个叶子需要水。"同事经过时说。

"我喷了的。"

"喷不够。要浇透。"

"浇透？"

"对。水从盆底流出来才算透。"

他当天晚上回去把多肉盆放在水龙头下面冲了三十秒，水从盆底的孔渗出来，在窗台上洇了一片深色。

他开始思考"浇透"这个词。他给苏晚枝写的那些信，像喷水——每次一点点，以为够了，其实从来没有浇透过。而作为被浇的那一方，可能连喷水都算不上，只是偶尔的雾气。

他不是在心疼自己。他是在整理。把当年的行为分解成比喻，然后看这些比喻是否准确。不够准确就换一个。

这是他在出租屋里养成的习惯：把情绪拆成比喻，把比喻拆成事实。

事实就是：他喜欢过一个人。喜欢的方式不对。那个人跑了。多年后他们和解了。现在他们各自在不同的城市上班、交房租、喝豆浆。

他把这些事写进小说里。以前他会写"他用了三年才消化这个事实"。现在他删掉这一句，改成：他在出租屋住到第三个月的时候，给墙上那块白漆拍了张照片，放进了一个叫"出租屋"的文件夹。

楼下有人喊"豆浆少糖"。电动车从巷口拐过去。隔壁情侣讨论完煤气费，又讨论起了明天谁洗碗。

空调外机在滴水。节奏稳定，两三秒一滴。他已经不需要消耗注意力去忽略它了——它成了一个不需要被注意到，但确实存在着的基底声。

就像过去本身。

他打开文档，继续写。写到一半手机响了一声。苏晚枝发来的消息：稿子看完了。面馆那章，泡脚的时候看的。差点把泡脚水打翻。

他回：哪个部分？

她回：香菜叶那部分。你把香菜叶写了六遍。每次都不一样。

他回：是不是写太多了。

她回：不多。每一遍都不一样。你以前写信只会写同一遍。

他回：人总得进步。

她回：这句不要写进书里。太像道理了。

他回：好。

他把手机扣在桌上。屏幕上还亮着，苏晚枝的头像是一个模糊的背影，看不出是谁帮她拍的，在哪拍的。

然后他接着打字。不是回消息，是回小说的正文。

光标停在上一段的结尾。他打了：三个月后，他把补墙膏塞进了抽屉最里面。
