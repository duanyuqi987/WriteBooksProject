# 第208章 毕业照里的空位之后

所属部卷：第四部：失去的美丽
时间：毕业照当天午后
地点：大学图书馆前台阶

## 诗歌插页 P208：原创仿宋小令

帽穗斜，日光明。
快门落台阶。
半格无人也入影。
散后各前行。

拍完照后，台阶上全是彩纸和掉下来的别针。陆允川弯腰捡起一枚，递给旁边不认识的同学。对方说了声谢谢，又急着去拍寝室合照。

祁野嚷着要去校门口拍一张“以后混不下去还能拿来怀念”的照片。程砚说他嘴欠，三个人笑着往外走。经过图书馆侧门时，陆允川看见苏晚枝和叶澄站在树下，正在给一位老师送花。苏晚枝笑得很自然，脸被阳光照得有些亮。

陆允川没有过去。他不是冷淡，而是知道那一刻不属于他。过去他常把“不打扰”想成委屈，好像自己退一步就是输。现在他才发现，不走过去也可以是一种祝福，而且比许多话更干净。

下午他们在操场边拍完合照，祁野非要把帽子扔起来。帽子落下时砸到程砚肩膀，几个人笑得弯腰。陆允川也笑，笑到眼角发酸。毕业季把每个人都推向分岔路口，悲伤不只属于爱情，也属于宿舍门牌、借书证、食堂窗口和再也不用上的早课。

晚上回寝室，班长把精修前的合照发到群里。陆允川点开，看见自己旁边那半级空台阶。它很小，不仔细看几乎看不出。可他知道那里没有故事需要补上。空着就空着，也是一种完整。
