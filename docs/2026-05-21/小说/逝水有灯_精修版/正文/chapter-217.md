# 第217章 旧日记本

所属部卷：第五部：逝水有灯
时间：工作第三年
地点：新住处书桌前

## 诗歌插页 P217：原创仿唐绝句

旧封微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

事情落到当天，没有被命运单独照亮。

搬完家第三天，陆允川终于在书桌前坐了下来。房间已经收拾了大半，书架上的书按高低排列，地上的纸箱还剩最后一个没拆。那本牛皮封面的日记本放在桌角，已经在搬家车的颠簸中翻了页——封底朝上，封面上"2015.9-2017.6"的字样被磨得更白了。

他吃了碗泡面，擦了手，翻开第一页。

第一页是2015年9月3日，大二开学第二天。

"今天在三楼食堂看见她了。她换了发型。我端着餐盘经过她旁边，她没看见我。我在犹豫要不要打招呼的时候，已经走过去了。她的室友先注意到了我，推了她一下。她转过头来，我冲她笑了笑。她说'哦，是你'。我说嗯。然后我们就各自吃饭了。今天没什么大事。就是想记一下。"

陆允川看完这一页，用指腹摸了摸纸上的笔迹。圆珠笔写的，笔画很重，很多字的撇捺都戳出了凹痕。

他记得那个学期。大二的课多了，去食堂的时间不固定。他在日记里记录了每一次碰见苏晚枝的确切时间：周三下午五点四十三分、周五中午十二点零九分、周一下午六点十七分。精确到分钟，像列车时刻表。

他翻到其中一页，上面记录了一次错过的碰面：

"今天没在三楼看见她。她可能去了二楼。我想过去二楼看看，但去了就太像跟踪了。所以没去。在日记里告诉自己——你做得对。"

他当时确实是这么写的。现在重读，他发现那个年轻的自己在做一件非常艰难的事：一边让自己保持正常，一边诚实记录每个不正常的念头。

翻到中间，出现了大量的信件草稿。有一封信的打稿涂改了十几遍，同一个位置重复写同一句话：

"我想你可能不知道我对你的感觉——"

涂掉。

"我想你可能知道我对你的感觉——"

涂掉。

"我不知道你是否知道我对你的感觉——"

涂掉。

最终寄出的版本他没有记在日记里。但他记得是第三句，因为犹豫本身就是答案。

窗外的天色暗了。他开了台灯，光把日记本上的字照得更清楚——那些笔画深的地方在台灯下凹得更明显。

祁野发来消息："东西都拆完了没？"

他回："差不多了。在看旧日记。"

"笑得出来吗？"

"笑不太出来。但也没以前那么怕看了。"

"这是进步。你以前是连封面都不敢翻的。"

"你怎么知道？"

"我有一次去你出租屋，看见那本日记压在抽屉最底下，上面盖着三本教材和一摞账单。那种压法不是收纳，是害怕。"

陆允川想了想，发现祁野说的是对的。他以前确实在压。不是怕别人看见，是怕自己翻到某一页。现在不怕了——不是不疼，是疼的地方已经结了疤。

他又翻了几页，到了大四那部分。2017年6月，面馆争论之后第三天写的：

"她说'接不住'。我今天洗碗的时候想起这句话。水太烫，我缩了一下手。缩手的时候我想——她的'接不住'大概跟我缩手是同一个反应。本能，不经过大脑。我不该评判本能的。"

他把这段话读了两遍。不是因为写得好，而是因为他确认了一件事：那天他确实理解了她的意思。不是在信里，不是在争吵里，是在洗碗的时候。水太烫，他缩手——突然就懂了。

手机又响了。这次是程砚。

"听说你在看旧日记？"

"祁野告诉你的？"

"对。他说你在跟二十岁的自己对话。"

"没那么玄。就是翻了几页。"

"发现什么了？"

"发现——"他看着日记本上那些戳出凹痕的笔画，"——我那时候写字太用力了。每一笔都像在刻。" 

"那现在呢？"

"现在打字。手指头轻多了。"

"不光是笔迹吧？"

陆允川翻到最后一页。日记的终点是2017年6月22日，毕业那天：

"今天拍了毕业照。她站在另一侧，中间隔了一排人。快门响的时候我们都看着镜头。不是不看她，是不需要通过看她来确认什么了。写到这里，这个日记本正式结束。下一个本子，写新的东西。"

他合上日记本。牛皮封面被合上的时候发出了一声很轻的闷响——纸页被压迫后松开的声音，像一个人叹了口气。

"喂？"程砚说，"还在吗？"

"在。我刚才读到最后一页。"

"写了什么？"

"毕业那天的事。最后一句是'写新的东西'。"

"那还挺合适的。"

"什么合适？"

"你一直在写新的东西。这本小说不就是吗。"

陆允川没有回答。他把日记本放在书架上，摆在"大学旧物"那一格，和毕业照放在一起。不是抽屉最底下。是最上面。可以看见，不需要翻。

程砚等了五秒。"你又沉默了。"

"在想事情。"

"想什么？"

"想你说'一直在写新的东西'。其实这本书也不算新的。它还是旧的东西。只是整理过了。"

"整理过了就是新的。"

"为什么？"

"因为旧的是乱堆着的。新的是站起来可以走路的。"

陆允川看着书架上的日记本。牛皮封面在台灯下面反了一点光。封面上的年份——2015到2017——像一道他已经迈过去的门槛。

"行。你说的有道理。"

"那当然。"

"但我还是要提醒你，你刚才那句'站起来可以走路的'——太文绉绉了。收进小说的话一定被删。"

"随便删。反正我也不是写东西的人。"

挂了电话。台灯还亮着。书架上的旧物们沉默地站在各自的位置上。裂了缝的碗、烧焦线头的学士服照片、被翻旧了的牛皮日记本。

泡面的盒子还在桌上。他把空盒扔了，筷子洗了。

水龙头关了以后，厨房里没有水滴声。新公寓的水管是好的。
