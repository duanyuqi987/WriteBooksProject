# 第151章 城东城西

时间：2013年9月，大三下学期开学

地点：江城大学→城东中学

卷次：第七卷：实习与远方

九月的第一个周一，苏安晴六点二十就起了床。

天还没亮透。她把昨晚收拾好的帆布包又检查了一遍——一本《语文》八年级上册、一本教案本、一支红笔、两支黑笔、一个保温杯。杯子是去年她生日时室友宋乔送的，粉色的，她一直没用过。

今天第一次用。

教育学院安排她去城东的第十九中学实习，教初二语文。她站在宿舍楼下的台阶上等公交车时，把保温杯拧开又拧上，拧上又拧开。

清晨的风带着秋天特有的凉意。她把围巾往领口里拢了拢。

公交车来了。她上车，投币，选了个靠窗的位子坐下。车子启动时，她看见梧桐叶开始转黄。

从学校到十九中，一共十一站。她数过了。

到学校时七点二十。教导主任姓何，四十多岁，戴眼镜，说话很快。他给了苏安晴一张课表、一把办公室的钥匙、一本教师手册。

"初二二班，三十二个学生。"何主任边走边说，"语文老师休产假了，你跟着原来的进度走就行。"

苏安晴点头。

办公室在二楼，她和另外三个老师共用。靠窗的位子是她的，窗户外面是一排银杏树。银杏叶还没黄透，但已经开始掉。她坐了十分钟，拿出教案本，翻开第一页，在上面写了一个日期。

2013年9月2日。

第一节课是九点的。她提前五分钟走进初二二班的教室。

三十二张脸同时抬起来。

苏安晴站在讲台上，把自己的名字写在黑板上。粉笔有些涩，笔画不太顺。她写完，转过身来。

"同学们好，我姓苏，是这个学期来实习的语文老师。"

下面有人小声说"好年轻"。她没有回应，只是翻开课本，开始讲第一课。

四十分钟很快就过去了。下课铃响时，有学生围上来问问题。她一个一个回答，声音不高，但很稳。

下午四点半，她坐公交车回学校。车窗外，城东的街道在夕阳下一格一格地向后退。她想起今天是周一，林玉起也应该开学了。

她拿出手机，打开QQ，打了一行字。

"实习第一天，教初二语文。"

对方在十分钟后回了消息。

"怎么样？"

"还行。学生不闹，比想象的好。"

"那就好。"

聊天记录就停在这里。

她想说点什么别的，但想了想，又把手机收了起来。公交车上摇摇晃晃，她靠着车窗，闭上眼睛。耳朵里是引擎的轰鸣和时断时续的广播声。

十一站的距离，不长不短。

她在心里把今天讲的课文又过了一遍。这个习惯她从高中就有了，讲完课在脑子里复盘。哪里讲得快了，哪里学生没听懂，哪里可以换个更好的例子。

车子经过一座桥时，她睁开眼睛。

桥下的江水在傍晚的光线里泛着灰色的波纹。九月的江水不急，像摊开的布。

她忽然想到林玉起。他是中文系的，写东西好，如果让他来教语文的话，不知道他会怎么讲那篇课文。

这个念头来得莫名其妙，她把它按了下去。

到学校时天还没黑。她回宿舍，洗了把脸，开始写明天的教案。宋乔从外面回来，带了一份食堂的包子，放在她桌上。

"第一天怎么样？"

"比想的好。"

"那就行。"宋乔坐在床上，"我那个小学更累，一年级，一整天都在喊'安静'。"

苏安晴笑了一下。

她写完教案已经快十点了。窗外的路灯亮着，光透过窗帘洒进来。

她拿起手机看了一眼。林玉起在三分钟前发了一条动态。

"报社第一天。编辑部的窗户朝西，晒。"

她点了个赞。

然后关了手机，翻了个身。

天花板上有光斑在晃动，是窗外车灯路过时留下的。她看了一会儿，闭上眼睛。

明天还有课。
