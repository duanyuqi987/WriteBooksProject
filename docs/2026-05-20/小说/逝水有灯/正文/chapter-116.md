# 第116章 楼下等她之前

所属部卷：第三部：寒风里的花
时间：大二五月
地点：女生公寓楼下圆石凳

## 诗歌插页 P116：原创现代诗

她说也许晚点，
他听成一定会来。
五月的风吹过圆石凳，
把一个词吹得发亮，
也把另一个词吹丢了：
不用等。

大二五月，诗社准备校刊终审。苏晚枝负责联系老师签字，陆允川负责把终稿送到文印室。那天下午，三教走廊闷热，窗户开着也没有风。

苏晚枝抱着一摞文件从办公室出来，额前的碎发贴着汗。她把终稿递给陆允川，“我晚点还要去教室打扫，老师临时检查。校刊你先送过去吧。”

“你几点结束？我可以把回执给你。”

“不用特意等。”她低头看手机，“如果我出来早，就给你电话；晚了明天再说。”

陆允川听见了“给你电话”，却把“不用特意等”放得很轻。他送完终稿，拿着文印室盖章的回执站在路口，天色还亮。祁野打电话问他吃不吃饭，他说有事。

“什么事？”

“等人。”

祁野在电话那头停了一下，“人让你等了吗？”

陆允川没有回答，只说先挂了。

女生公寓楼下有几个圆石凳，靠近花坛，旁边停着几辆自行车。他坐下时，石凳还有白天晒过的温度。手里的回执纸被他折了一道，又摊开。

他知道苏晚枝说过不用等。可他也知道，自己已经把这件小事放大了。放大以后，它不再是交回执，而像一次证明：证明他可靠，证明他在场，证明她会不会记得一个电话。
