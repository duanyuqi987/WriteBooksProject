# 第083章 室友

时间：2013年9月中旬

地点：楚州师院宿舍512室

卷次：卷五：各自南北

宿舍四个人。李博、阿坤、小伍。顾临风。

李博是第一个到的。他每次进门都敲门——虽然他自己有钥匙。他管宿舍的卫生排班，管水电费的分摊，管谁用了谁的热水。开学第三天他贴了一张纸在门后面——"512宿舍卫生值日表"。打印的。字体是宋体。下面还画了一条线。

"你们谁要是值日那天没倒垃圾——"李博指了指那张纸。"上面有名字啊。"

阿坤看了一眼。"班长。你放过我们吧。"

"我不是班长。"

"你马上就是了。"

阿坤叫陈坤。但他让所有人叫他阿坤。他带来的行李最少——一个背包、一个手提袋。背包里装了三件东西：电脑、充电器、一条耳机。第一条晚上就通宵打游戏——魔兽争霸。键盘敲得啪啪响。屏幕的蓝光照着他的脸。凌晨三点他还在戴着耳机说"奶妈加血奶妈加血"。

他的桌子最乱。泡面盒摞了三层，可乐罐排了一排，瓜子壳掉在键盘缝里。李博每天路过他的桌子都皱眉头。"阿坤你那个泡面盒子都长毛了。"阿坤头也不抬。"明天倒。"明天又说后天。后天变成了下星期。

小伍叫伍思远。家里是崇阳县的。第一次住校。第一晚翻来覆去睡不着。凌晨一点多从上铺探头下来。

"顾临风——你睡了吗。"

"没。"

"我有点想家。"

"嗯。"

"你家住得近吗。"

"青阳县。三个小时。"

"三个小时也不近。"

沉默了一会儿。小伍把脑袋缩回去了。过了大概十分钟，上铺传来吸鼻子的声音。很轻。然后是翻身的动静。然后没声音了。

小伍胆子最小。第一次去食堂打饭，站在那里不知道点哪个窗口。顾临风走过去指了中间那个。"番茄鸡蛋面还行。"小伍点头。打好面以后坐在他对面，吃了两口，说"比我妈做的淡"。然后继续吃。吃完了。

小伍每周给家里写一封信。信封上画着熊猫——他爸是邮局的，家里攒了一摞熊猫信封。他写得很慢。一个字一个字地描。字迹工整——像小学生抄课文。有时候写到一半会抬头问——"临风，'适应'的'适'怎么写"。顾临风在空气里给他比划了一下。

开学第二周。晚上十点熄灯以后，宿舍进入卧谈时间。

阿坤先开口的。"咱们班那个女生——三班的——叫什么来着，刘什么的——好看。"

"刘思琪。"李博说。

"对对对。眼睛挺大的。"

"你怎么什么人都认识。"

"咱班花名册我看了一遍。"

"你看那玩意儿干什么。"

"提前做功课。"

小伍从上铺翻了个身。"你们高中谈过恋爱吗。"

阿坤说。"谈过。高二。谈了三个月就分了。她嫌我打游戏太多。"

李博说。"没。高中忙。没顾上。"

阿坤说。"班长说谎。你一看就是没谈过。"

"我真没谈过。"

"所以才说'没顾上'。谈过的人会说'没有'。"

沉默了两秒。

"临风你呢。"阿坤把声音往他那边转了。

"没有。"

"暗恋过吗。"

顾临风没有接话。窗外的路灯光透过窗帘的缝隙——在地面上切出一条细细的白色。他侧着身，脸朝着墙。墙上有那个小洞——他不知道前一个学长钉钉子挂过什么。钟？海报？照片？洞里是黑的。

"临风？"

"没有。"

"你们都不诚实。"阿坤说。"我看人很准的。"

李博说。"你先把你的魔兽打好吧。"

阿坤笑了。

又聊了一会儿——关于军训、关于食堂、关于哪个班的女生好看。阿坤说师院有个校花——大二的，叫什么他忘了，"反正是学音乐的"。小伍说音乐系的女生都好看。李博说"你们一天到晚就是女生女生"。阿坤说"不然呢，难道聊食堂的饭"。

声音渐渐低了。阿坤最后一个说完，然后宿舍安静下来。

顾临风听到小伍又吸了一下鼻子。

然后一切沉默了。

他翻了个身。枕头底下的手机硌着脑袋。他没有把手机拿出来。只是把手伸到枕头下面，碰了碰手机壳的金属边。凉的。

外面有人在走廊里趿着拖鞋走过去。脚步声由近到远。声控灯亮一下，灭了。

他闭上眼睛。

那些他没有说的话——还在脑子里转。关于高一的教室。关于二排靠窗的位置。关于一个扎马尾的女生。关于一封没有递出去的信。关于月湖边的那个晚上。关于她说"暂时不要联系"。关于那条301路公交路线。

他什么都不会说。

他把被子往上拉了拉。蒙住半张脸。呼吸落在被子里面，有洗衣粉的味道。

第二天早上。李博第一个起床。六点整。他叠被子的声音很有规律——啪啪啪。然后他去洗漱。回来的时候拎着四个包子。

"谁的包子。"

"我的。"阿坤从上铺跳下来。

"那是临风的。你的在塑料袋里。"

"班长。你过分了。"

顾临风接过包子。韭菜鸡蛋的。他突然想起高中食堂的包子——每次她排在他前面四五个人的位置。她买两个包子、一碗小米粥。包子先吃素的那个。吃的时候会站在食堂左边的窗前——可以看到操场上晨跑的人。她侧脸的轮廓在晨光里有很淡的金边。有一次她吃完包子，用纸巾擦了擦嘴角，然后把纸巾折了两下扔进垃圾桶。他发现她折纸巾的动作用的是左手。

他咬了一口包子。咀嚼。咽下去。

李博递过来一杯豆浆。"老顾——你刚才走神了。"

"在想事。"

"'在想着韭菜鸡蛋的事'——你这个借口太烂了。"

阿坤在旁边笑。牙膏沫还挂在嘴角。

上午的课是中国古代文学。老师姓孙，四十多岁，戴金丝边眼镜。讲话的时候尾音拖得很长。"屈原——的代表作——《离骚》——"顾临风坐在第三排。笔记本摊开。但他写的东西跟《离骚》没什么关系。他在本子边角写了几个字——然后划掉了。墨水把一个字的偏旁洇成了墨团。他把那个墨团又涂了两笔——变成了一棵树。

下课后李博凑过来看了一眼他的笔记本。"你画的是什么。"

"树。"

"不是树。是个字。你把字涂成树了。"

顾临风合上笔记本。放进书包里。

那个中午他没有去食堂。在操场走了一圈。跑道上有上体育课的学生在测五十米。体育老师拿着秒表喊"预备——跑"。脚步声啪啪啪地落在跑道上。他走过的时候一阵风把沙坑里的沙子吹起来。沙子打在裤腿上。他拍了拍裤腿。继续走。

走完以后回了宿舍。512里三个人都在。阿坤在打游戏。李博在看书。小伍在写信。他在自己的铺位上躺下来。枕头下面的手机硌着后脑勺。他没有挪动。只是枕着。手机壳慢慢地从凉变温。
