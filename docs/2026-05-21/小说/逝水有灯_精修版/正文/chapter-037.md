# 第037章 成绩单下滑

所属部卷：第一部：初秋三楼
时间：高一期末考试后
地点：公布成绩的公告栏前

## 诗歌插页 P037：原创仿唐绝句

一纸排名各西东，
少年心事上眉峰。
当时不解寻常意，
却将分数比秋风。

事情真正发生时，并没有陆允川想象得那么郑重。期末考试成绩贴出来以后，夏知遥的成绩排名从上学期的四十七掉到了七十三。那天教室里的气氛像闷在热锅底的粥——黏糊糊地透着焦。天花板上的吊扇开到了最大档，但吹出来的风是热的，把试卷边角吹得哗啦啦响却吹不走任何人脸上的汗。大部分人看完自己的分数就往教室里走，表情被公告栏玻璃上的反光遮住了。

陆允川站的位置在她后面三个人。轮到她看成绩时，她走到公告栏前，抬起下颌从左往右看，视线在第十三名左右扫过——大概是我她自己——然后往下移，在七十多名处停住了。停的时间不长——大概五秒——但那五秒里她的肩膀往上耸了一下，然后放下来。那个耸肩的动作很小，在整条走廊里只有陆允川一个人看见。

她看完没说话，从人群里退出去，走到公告栏旁边靠墙站着，在拧自己的水杯盖——拧了好几圈，拧不开，是因为没用力。蒋雯走过去把自己的成绩条翻过来给她看："我也掉了，掉到五十多了。这次数学最后两道题太变态了。"夏知遥说"嗯"，声音不高，也没有继续说"我没事"——她没有说自己有事还是没事。她只是说了"嗯"，然后把水杯盖子拧开了，喝了一小口水。

陆允川在公告栏前装作还在看自己旁边同学的排名——其实他已经看了第三遍了，每个数字差不多都能背出来。他在找一个机会，想靠近她，说一句什么——不是标准安慰，是一种让她知道有人在替她感到不安但不会说出来的那种。这个机会没有出现——因为蒋雯一直在她旁边。而他不敢在第三个人面前对她开口。她后来自己走回了教室，成绩条攥在手心没放进笔袋。

午饭时食堂里分数的事还在发酵。食堂的吊扇比教室的转得还慢，打饭窗口那边飘过来一股炒青椒的油烟味，黏在每个人的校服上。有人大声笑着跟后排说"我这次居然考前一百了"，有人端着碗不说话。程砚从窗口打完饭回来坐下——不锈钢餐盘里堆着米饭、炒青菜、红烧排骨和一碗紫菜蛋花汤——夹了块排骨放进嘴里嚼了半天才说："她掉到七十三了。你看了吗？""看了。""那你打算怎么办？""我打算什么？"程砚放下筷子，表情忽然严肃了一秒——是那种"你不打算我也替你想办法"的严肃。"你想帮她得先知道她想不想被帮。有些人高兴的时候被夸会高兴，不高兴的时候被夸反而更难受。她——我看不出来。她不是那种会把需要写在脸上的人。"

晚自习时，夏知遥坐在自己座位上复习错题——她拿红笔在错题旁边重新写了一遍步骤，每一道错题的订正都写了一整行，用力很大，纸背面都有红笔的印子。陆允川坐在后排看着她的背影，看不出她是在咬牙还是在憋泪——她的肩膀没有抖，她的背挺得很直，她在动手解决问题而不是坐在那里怨考题。这个背影让陆允川的心疼从"她怎么掉了"变成了"她得怎么办呢"。但这两者之间什么也做不了——他只能看着她把所有错的题重新写一遍，直到纸页被红笔和蓝笔填满了。

放暑假前的最后一节晚自习结束，陈老师在讲台上说祝大家暑假愉快，注意安全。同学们陆续起身收拾书包——有人约好明天一起去打台球，有人在黑板上写"高二见"。夏知遥站起来的时候，陆允川冲动了大概一秒钟——想去跟她说句暑假快乐——但他走到她座位旁边时她已经被人叫走了：蒋雯在前门喊她"知遥你家里来接你在校门口"。她背上书包快步走出去，经过他身边时没有停。他只来得及说了一句"拜——"，她已经走到门框外了。这个"拜"只有前半拍，后半拍被走廊的脚步声吞掉了。他站在原地，书包带子在手里攥成一道折痕，想是不是每一次他就要说话的时机都会被某种不可抗力抢先一步截走——截他的人不是敌人，而是她有她自己的安排。

明天就开始暑假。两个月零一周后重新回来上课。她会是同一个教室里的同一张脸——但那个从四十七掉到七十三的成绩会在她十六岁的暑假里发酵成什么，他无法参与，只能远远地想。
