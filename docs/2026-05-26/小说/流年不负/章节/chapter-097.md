# 第097章 一个女生

时间：2014年10月

地点：楚州师院图书馆门口

卷次：卷五：各自南北

十月中旬。文学社招新结束以后搞了一次迎新茶话会。在一间贴着诗词条幅的小教室里。桌子上摆着橘子、瓜子、一壶泡了很久的菊花茶。新社员有十几个人——大一的，眼睛里带着还没褪完的新鲜感。老社员坐在靠窗的那一排——包括顾临风。

他坐在最边上。手里剥着一颗橘子。橘子皮有点厚。指甲缝塞满了黄色的皮屑。

茶话会结束后，大家在门口三三两两地散了。他正要往图书馆方向走。

"顾临风。"

他回过头。

一个女生站在教室门口。穿一件米色针织衫。头发是直的——到肩膀。手里抱着一本笔记本——黑色封面，跟他那本有点像。

"我是周雨。文学社新社员——大一。你刚才茶话会上说你的诗主要在写日常——我挺感兴趣的。"

"嗯。随便写的。"

"我可以看一下吗。"

他想了一下。"没有带。"

"下次可以带来吗。"

"嗯。"

她笑了一下。牙齿很整齐。眼睛下面的皮肤有一点雀斑——浅浅的，大概四五颗。"那说定了。下次社团活动带过来。"

然后她转身走了。头发在肩膀上甩了半圈。步子不快。

那天以后，文学社的活动她每次都在。坐在第三排——离他不远不近。每次茶歇的时候她会主动走过来——"今天这首诗你觉得怎么样"——"还行"——"我觉得第三段有点太直了"——"是有点。"

她的散文写得很好。不像很多人写的那么空——不是那种堆满形容词的所谓美文。她的散文有细节——写了她们宿舍楼下的一只流浪猫、写了食堂阿姨打饭时总会在菜上面多放一片肉、写了她奶奶织的围巾有一针松了一直没补。她会观察——跟他有点相似。

阿坤有一回在社团室外面瞄了一眼。"那个叫周雨的——是不是对你有点意思。"

"不知道。"

"'不知道'——你心里清楚得很。"

顾临风没接话。

一周以后。十月底的一个下午。他从图书馆出来——抱着五六本书。周雨站在图书馆门口的台阶上——大概是在等他。

"顾临风——我有话跟你说。"

他停下来。

她把笔记本抱在胸前。抿了一下嘴唇。然后说。

"我喜欢你。"

很大方。没有结巴。没有低头。就是直接说出来了。阳光从梧桐树叶子中间漏下来，打到她侧脸上。雀斑在光里很淡。

他想了几秒钟。不是那种惊喜的几秒钟。是那种"要不要试一下"的几秒钟。然后他点了点头。

"好。"

"好——是什么意思。"

"就是——我们试试。"

她又笑了一下。这次笑的时候眼睛眯了起来。牙齿又露出来了。很干净的一个笑。

"那以后图书馆你帮我占座。"

"嗯。"

她伸出手。他把自己的手伸出去。握了一下。她的手比苏晚晴的凉。他的手掌还是那个毛病——汗多。

他心里有一句话没有说出来。他不是因为喜欢她。是因为觉得应该试一试。

也许能忘记呢。

也许呢。

那天晚上他躺床上。看着天花板。想了很久——关于自己做的这个决定。然后不想了。
