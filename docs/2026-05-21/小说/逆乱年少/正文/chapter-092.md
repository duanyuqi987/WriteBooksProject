# 第092章 越来越具体

时间：2011年十一月下旬，诗会结束后

地点：人文楼走廊和操场边的长椅

卷次：第四卷：暑假的信

诗会结束是下午四点半，走廊里夕阳从西边的窗户斜进来，把打磨过的水磨石地面照得发亮。何南星在门口收签到表，看见林玉起出来，拍了拍他肩膀。

"你念的诗是今天最好的。"

"你不是每个都说不做评价吗。"

"那是客套。对你的才可以不说客套。"

林玉起没说话。他手里拽着笔记本，但拽得太紧了，笔记本的封面被他攥出了一个汗手印。何南星看了他一眼，又看了看走廊尽头那道慢慢走着的浅蓝色短袖的背影——苏安晴一个人往楼下走，手里捏着一张签到表，边走边折。

"她听懂了。"何南星说。

"你怎么知道？"

"因为她刚才听完没有笑。没有笑就是听懂了。"

这个逻辑很奇怪，但林玉起信了。何南星在诗的事情上几乎没错过。

>"秋雨已停多日
>伞骨被你扶正
>豆浆凉了
>纸杯还在
>——你说越来越具体
>我说越来越少
>佛没接后半句
>你皱了皱眉"

诗会散场以后，付小川也来了——他一向对这些不感兴趣，但没有说到做到，大概是郭向阳通风报信的。他靠在人文楼门口的柱子上。"听说你当众读诗了？还读了给某人的诗？"

"不是给某人的。"

"好，不是给某人的。是给豆浆的。"

林玉起无奈地笑了一下。付小川递过一瓶矿泉水："你们这诗社活动能持续多久？我听说办完这场就放寒假前不办了？"

"何南星说十二月还会有一次，但规模会小很多——大概只对内。"

"那正好。"付小川拧开自己的水喝了一口，"我虽然不懂诗，但我刚才在后面听了一会儿。你念诗的时候声音比以前稳了，不像以前那样发抖。林玉起，你变了一点点。就一点点。但变的方向是对的。"

两个人在操场边的长椅上坐下来。梧桐树光秃秃的，夕阳从树干之间漏进来，在他们脚边打出橙色的光斑。操场上有人在慢跑，鞋子在塑胶跑道上发出闷闷的声音。

"付小川，她听完说了一句话——'越来越具体了'。这句话是什么意思？是在夸我吗？还是只是在陈述一个事实？"

"物理系的人不分析语义。"付小川耸耸肩，"但我可以告诉你——如果一个女生听完你的诗只说了一句话，不管那是什么话，都说明她认真听了。认真听过，才会有评价。不然她可以像其他人一样说'挺好的'就走了。她没说'挺好的'。她说的是你诗里的变化。"

付小川把矿泉水瓶往椅子上一放，双手交叉枕在脑后，看着天空眯起了眼。天还很亮，有几朵零星的云，被夕阳染成了浅橙色。

"玉起，我收回之前那句话。"

"哪句？"

"'如果你不发球就不会输'。你今天发球了。她接到了。接住了，没有扣回来，也没有扔到场外。你懂我的意思吗？"

林玉起懂了。她是——接住了。不是拒绝，不是明确接受，而是把它放在手里——不知道接下来要怎么办。就像他一样。

他把笔记本翻开，在刚才那首诗的末尾加了一行字：她皱了皱眉。
