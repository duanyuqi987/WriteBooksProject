# 第222章 重回县一中

所属部卷：第五部：逝水有灯
时间：十年后秋天
地点：县一中翻新的校门

## 诗歌插页 P222：原创仿唐绝句

旧孔微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

事情落到当天，没有被命运单独照亮。

冰红茶喝到一半的时候，陆允川决定去教学楼里面看看。

程砚留在小卖部门口，说他不想爬楼梯。

"你当年也不喜欢爬楼梯。"陆允川说。

"我现在更不喜欢。跟过去无关。跟体重有关。"

教学楼还是六层。楼梯的扶手换了不锈钢，原来的木扶手大概是蛀掉了。他摸着不锈钢扶手往上走，手掌挨着金属表面，凉凉的，很滑。三楼走廊里没有人，教室门都关着，门上的小窗透出日光灯管的光。他把目光从第一扇窗移到最后，停在最左边一间——那是他当年高三的教室。

门牌换过了。原来木制的白底红字换成了亚克力材料的蓝底白字，标注的是"高二(3)班"。里面坐着的全是陌生面孔——有人在转笔，有人趴在桌上补觉，有人在课本下面压着一本课外书。老师站在讲台上写板书，粉笔砸在黑板上发出那种他熟悉但很多年没听过的声。

他没有敲门。只是站在走廊上透过门上的小窗往里看。看了大概一分多钟。

坐在窗边的一个女生注意到了他，偏头看了看，然后推了推旁边的同桌。同桌也看过来。陆允川冲她们点了点头，转身离开了。

他走到走廊尽头。那里有一个饮水间，饮水机是新的，旁边贴着一张值日表。值日表上的名字都是不认识的。

一个男老师从办公室里出来，看见他在看值日表，问："你找人？"

"不是。校友。回来看看。"

"哪一届的？"

"一四届。"

"一四届。"男老师推了推眼镜，"那年高考理综挺难的。你们还好吗？"

"还好。"

"你现在在做什么？"

"做文职工作。顺便写点东西。"

"写什么？"

"小说。"

男老师哦了一声，没有露出你写小说啊的表情，而是问："文科类的？"

"算是。"

"那挺好。我们学校这几年语文教研组也在推文学社团。你要是有兴趣可以跟胡老师聊聊。他在四楼办公室。"

"谢谢。下次吧。"

男老师走了。陆允川靠在饮水机旁边，把冰红茶的盖子拧开了又拧紧。

"旧螺丝孔"在哪—他记得是在校门门牌下面。但三楼的楼梯扶手旁边也有一组——固定旧公告栏用的。公告栏早就拆了，换成了电子显示屏，但那几个螺丝孔还在。两个，对称的，在黑板上方的白墙上，颜色比周围的墙略深。

他伸手摸了摸。墙是凉的。螺丝孔的边缘光滑了。

有人从楼下走上来。脚步声很轻。是程砚。

"不是说不想爬楼梯吗？"

"坐了一会儿觉得不爬太不像话了。你看了教室？"

"看了。变了很多。门牌、黑板、饮水机。"

"学生呢？"

"还是转笔。还是趴着睡觉。还是往课本下面压课外书。"

"那核心没变。"程砚靠在扶手上。"有没有看到当年你的座位？"

"看了。大致的位——进门第三列倒数第二排。现在那里坐着一个穿深蓝校服的男生。他在做数学题。"

"你怎么知道是做数学？"

"因为他桌上摊着三角板。"

"三角板不一定是数学。物理也用。"

"好吧。那他在做数学或者物理。"

程砚笑了。笑完他说："你有没有觉得奇怪。"

"什么？"

"我们站在这栋楼里，说的是十年前的事。但是对这栋楼来说，十年前的事等于没发生过。因为学生换了，门牌换了，扶手换了，公告栏换了。只有几个螺丝孔没换。"

"螺丝孔也不是刻意留的。"陆允川说，"是工人懒得填。"

"对。但就是这几个懒得填的孔，证明我们真的来过。"

陆允川从饮水机接了一杯水、喝了一口。水是温的，不是凉的。新饮水机默认温档。

他们又在三楼站了一会儿。上课铃响了。走廊里的学生陆续走进教室。那个穿深蓝校服的男生抬起头看了眼走廊，然后重新低头做他的数学或物理。

陆允川把矿泉水瓶扔进走廊尽头的垃圾桶。垃圾桶旁边放着一个扫把，扫把柄是新的红塑料的——旧的木柄扫把他用了三年。

"走吧。"他说。

"不多看看？"

"够了。看再多也是别人的教室。"

下楼梯的时候，程砚在后面说："你刚才那句话可以写进小说。"

"哪句？"

"'看再多也是别人的教室。'"

"太直白了吧。"

"你总嫌直白。小说需要直白。"

"知道了。回去改。"

走出教学楼，太阳已经偏西。操场上体育课还在继续，哨声此起彼伏。校门口的那棵桂花树在斜阳下投出一整条林荫道的影子。

他们在校门口站了一会儿。电子屏还在滚动通知。旧螺丝孔还在门牌下面。新漆绕开的地方，两个小小的黑点，像两个不完整的省略号。
