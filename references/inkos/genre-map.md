# InkOS 题材规则映射

InkOS 内置中文题材规则已复制到 `references/inkos/genres/`：

| InkOS 题材 | 文件 | 本项目建议映射 |
|---|---|---|
| `urban` | `references/inkos/genres/urban.md` | 都市、现实、商战、人情关系；常配 `冷白描.md` 或 `通用中文小说.md` |
| `xuanhuan` | `references/inkos/genres/xuanhuan.md` | 玄幻、升级、战力体系；常配 `系统爽文.md`，但需防套路化 |
| `xianxia` | `references/inkos/genres/xianxia.md` | 仙侠、修行、宗门、因果；需强化世界规则和代价 |
| `horror` | `references/inkos/genres/horror.md` | 恐怖、怪谈、悬疑；常配 `怪诞悬疑.md` |
| `other` | `references/inkos/genres/other.md` | 不适合套固定题材时使用，回到项目自定义规则 |

## 使用原则

- 题材规则只提供边界和节奏提示，不替代 `project.md`。
- 如果题材规则与用户作品目标冲突，以 `project.md` 和当前对话要求为准。
- 不把 InkOS 网文题材规则强行套到文学向、现实向、短篇或实验小说。
- 长篇连载可读取题材规则；短篇和局部精修通常不需要。
