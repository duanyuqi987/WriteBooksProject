# 第190章 诗社告别多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：城市公寓

## 诗歌插页 P190：原创仿唐绝句

书脊微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

多年后陆允川在一家旧书网的店铺里看到了一本《中国诗歌选》。

那天他在书房里随便逛网页——周末下午，手边一杯已经凉了的红茶，窗外有小雨打在雨棚上的声音。旧书网推送了一个"你可能感兴趣"的链接，封面跳出来的时候他愣了一下——墨绿色底，烫金的行书字体，跟他大学时用过的那本一模一样。

封面跟他大一还在诗社时用过的那本一样。他下单买下来——纯粹是因为怀旧。书到的那天是周三，快递放在门卫室，他下班回来顺路取的。拆开快递袋的时候，牛皮纸和旧书混合的气味让他恍惚了一秒——那是活动室书架的气味。

打开扉页，没有他的铅笔批注。不是同一本。只是同一个版本。纸页比大学那本干净，连一个折角都没有——原主人大概没怎么翻过。

他把书放在桌上，先给自己的茶杯续了热水，再回来看它。封面角落隐约有一小块褪色——可能是被阳光晒过，跟他那本被窗台晒褪色的位置一模一样。

但他把书放上书架的时候，还是把它塞进了那个编号还没改过来的位置——是他跟苏晚枝学编目时贴的第一批标签——歪歪扭扭，但他从来没有纠正。书脊的歪角从书架里探出来一点点，像一个人站在队列边缘，没完全对齐但也没退出队伍。

多年后重看那次退出，他不觉得是失败。觉得是自己终于做了一件不属于"求爱"范围的事——没有目的，没有计算，没有用付出兑换回应的汇率表。就是还书，签字，出门。

做那件事的时候他没有期待苏晚枝挽留。没有暗示自己要被感激。他就是把借来的书还了，把登记表签了，然后出了门。门合页还是涩的，但他没有再回头去听那声响。

那是大三他做的最好的决定之一。

在他的"不再分析的日常"文件夹里，他后来加了一行：

"离开不一定是失望。有时候是——'谢谢你在这里，让我知道我该去别的地方了。'"

在多年后的某一天，他在朋友圈看到当年的诗社群炸了一次——是十周年纪念。有人发了老照片：搬活动室那天的合影——他在纸箱后面露出半张脸。苏晚枝拿了旧明信片在看。叶澄在角落插U盘。夏知遥手里举着什么东西被画面切掉了——可能是奶茶杯。

他在评论区回了一个"这照片好神"。苏晚枝回了他一个表情——是个双手合十的 emoji。没有多余的文字。他看了几秒，然后退出群聊，继续回去改方案。

那本买错了的旧诗集还是歪着放在书架上。每次经过时他都会被那个歪角挡一下袖子。但从来没去修正过。

有一次妻子问他为什么不把那本书摆正。他说："因为那是大三时的我。歪着，但不倒。"

妻子看了他一眼，那种眼神他很熟悉——既包容又带着一点"你又来了"的无奈。她认识他的时候他已经在写这本小说了，所以她从来没有见过那个二十三岁、会在日记里反复划删除线的陆允川。她见的是一本已经写到尾声的书和一个人到中年的丈夫。有时候她会问一些关于过去的问题——那些名字、那些地点、那些他写进书里的细节——但她从来不追问。她的聪明之处在于：她不需要了解全部，也能跟你过好剩下的日子。

有一次她翻到他书架上那本《中国诗歌选》——歪着的那本——问："这本也是你大学时候的？"

"不是。是我后来买的。买错了。"

"买错了为什么不退？"

"因为歪的那页——打开以后刚好是大一我自己写过的第一首诗。不是同一本，但是同一页。"

妻子把书抽出来翻了翻。她没有找到那首诗——因为他买的是新版本，没有他的铅笔批注。但她还是把书放回了原处，还是歪着。她说："你书架上的书，每本都有一个故事。以后孩子问起来，你可以讲很久。"

"那你可以帮我讲。有些故事你已经听腻了。"

"没腻。你反复讲的那些我可能会腻。但你没讲的——我反而好奇。"

陆允川想了想。他没讲过的故事还有很多。比如那个凌晨在后门陪他站到天亮的祁野，比如程砚在他的打印稿上留下的七处或八处铅笔批注，比如苏晚枝最近发了一条朋友圈——配图是她窗台上的银杏和一棵刚发芽的香菜。他没有点赞。不是因为不想，是因为那些事已经不需要再被标注成"互动"。

他现在的生活里，最频繁的互动不是朋友圈点赞，而是每天早上跟妻子在厨房里抢灶台——她要热牛奶，他要煮面条，两个人挤在两平方米不到的小厨房里，肩膀碰肩膀，谁也不让步。"你今天先让我烧水。""你每天都先烧水。""因为我的面条需要开水。""我的牛奶也需要。""牛奶可以隔水热。"妻子每次到最后都会妥协，但每次都会说一句："你就是个死脑筋。"

他在备忘录上记了一笔：吵架的理由从"你不回复我的信"变成了"你抢灶台"。进步。
