# 第015章 理科名单多年后

所属部卷：第一部：初秋三楼
时间：多年后
地点：同学群与旧校门照片前

## 诗歌插页 P015：原创仿唐绝句

旧榜无声纸已黄，
少年曾在字中忙。
同名一室非同路，
风过校门各自长。

同学群里有人发了一张旧校门照片。

照片像素很低，门口的蓝色校牌糊成一片，保安亭旁边还停着几辆旧自行车。有人在群里笑，说那时候校服真难看；有人认出自己当年站过的位置，说再也回不去了。

陆允川把照片点开放大，看了很久。照片里没有公告栏，可他想起那张红色分班名单。想起自己站在人群后面，先找自己，再找夏知遥。也想起程砚把他从人群里拽出来，说别把名单盯穿。

那时他把同班看得太重，像命运终于给了自己一个机会。多年后再想，同班其实只是学校排课、老师分配、学生成绩和许多偶然因素凑在一起的结果。它没有承诺，也没有暗示，更没有替谁写好结局。

群里程砚忽然艾特他：“陆允川，你当年是不是差点把公告栏玻璃看裂？”

陆允川回了一个笑脸，又打字：你当年化学书还包歪了。

程砚发来语音，声音比少年时低了一点，却还是那个调子：“歪怎么了，最后不也毕业了？”

陆允川听完，笑出了声。他把手机放在桌上，打开硬皮本。旧本子的纸页已经有些松，夹着几张当年的草稿。那张分班名单当然不在里面，他没有胆子去撕，也没有能力保存所有东西。

他只在本子里找到一行被划掉的话：同班不是答案，只是把问题放近了一点。

现在看，这句话仍然不算错，只是太急着聪明。十六岁的他不知道，一个人不是靠提前说出道理长大的。人要在很多次尴尬、误会、沉默和自以为是里，慢慢把道理活成身体的一部分。

他没有把那行划掉的字擦干净，只在旁边补了一句：同班也可以只是同班。

写完以后，他合上硬皮本。窗外晚风从楼缝里吹过，带着一点城市灰尘。旧校门在手机屏幕里亮着，很小，很远，却没有再把他拽回去。
