# 第070章 寒假前的成绩单



时间：2009年一月底，寒假前最后一天



地点：高一十班教室、校门口和回家路



卷次：第三卷：雪落在两栋楼之间



2009年一月底，寒假前最后一天，高一十班教室、校门口和回家路，被韩泽叠成四折的成绩单先把这一天按进了十班的日常里。寒假前成绩单发下来，第三卷的雪、月考、座位和处分都落到一张纸上。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

有人急着回家，有人不敢把成绩单拿出来，段璟和常俊凡都带着各自的结果往校门走。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看被韩泽叠成四折的成绩单，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

韩泽也在这一天被推到前面。韩泽的特点是沉稳，成绩中游，喜欢把成绩单叠得很小再放进口袋。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“我先不看，回家再看。”韩泽说。

“你都叠好了，还装不看。”付小川说。

“别在校门口起哄。”常俊凡说。

“你寒假还来补课吗？”段璟说。

“看家里怎么安排，也看我自己怎么安排。”常俊凡说。

成绩单发下来时，教室没有想象中那么吵。每个人都先看自己的，再悄悄看别人表情。

韩泽把成绩单叠成四折，边角压得很平，像这样就能把分数暂时关起来。

段璟看见自己的物理比上次高，语文仍然拖着。他没有立刻高兴，也没有立刻失望。

常俊凡把成绩单放进透明文件袋，手指在总分那一栏停了一下，很快又移开。

陶建国讲寒假作业，讲安全，讲不要把雪仗那点热闹带回家当英雄事迹。

付小川小声说我可以当反面教材，宋乔回头说你已经当过了，不用争取。

校门口人很多，铺盖卷和书包又挤在一起，像开学那天的场面被冬天重新翻出来。

段璟问常俊凡寒假还来不来补课。她没有给标准答案，只说要看家里，也看自己。

这句话让他想起整个冬天：月考、座位、纸条、雪仗、名单、成绩单，没有一件事只靠一个答案就能过去。

公交开走时，十班的人被不同方向带回县城和村庄。雪已经化了，可很多共同经历过的声音还留在他们身上。

寒假前的成绩单没有悬在空处。成绩单纸面上被手指按出的折痕，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，寒假前的成绩单带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

陶建国进门时不一定立刻说话，鞋底在水泥地上一停，寒假前的成绩单带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在韩泽快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。寒假前的成绩单越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。被韩泽叠成四折的成绩单让他想说话，又逼他先低头看题。

韩泽被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，寒假前的成绩单留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。韩泽路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为寒假前的成绩单翻身，谁还在被窝里背单词，旁边人都听得见。

寒假前的风从校门外吹进来，把成绩单、铺盖卷和回家的路线一起吹到每个人眼前。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理寒假前的成绩单里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：被韩泽叠成四折的成绩单、一只冻红的手、一句老师点名、一块还没扫干净的雪。

何南星把成绩单叠好夹进书里，书页却怎么也合不平。窗外风一过，纸页轻轻动了一下，像替谁把话咽回去。

邓桥检查寒假作业清单，铅笔尖在纸上点了好几下。这个小动作很快过去，却让教室更像一个真正在运转的班。

余小满把铺盖卷从宿舍拖出来，走到楼梯口才发现少拿一本书。旁边的人没有特意看，可那一瞬间还是被段璟记住了。

曹蔚站在校门口等家里人来接，眼睛却一直往十班那边看。常俊凡没有接话，只把手里的事做完。

段璟检查寒假作业清单，铅笔尖在纸上点了好几下。冬天把这些小动作放大，让每个人的慌张都有了形状。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，十班带着不完美的成绩和共同的雪仗记忆进入寒假，青春没有给他们标准答案。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
