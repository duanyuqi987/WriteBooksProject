# 第095章 夏天的聚餐多年后

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P095：原创七言小诗

尘过长廊日影深，
一行旧字压衣襟。
少年未解离分意，
先把沉默写到今。

工作第二年夏天，陆允川出差路过县城，待了一晚。

项目在隔壁市的一座桥——双塔斜拉桥，他在设计院负责桥墩基础的图纸校对。出差是跟着总工来看现场施工进度。回程的火车在县城停了一站，他就下来了。身上还穿着工地的防晒服，背后印着设计院的名字。

程砚正好也回来——厂里年中检修，放了三天假。他在群里看到陆允川发的火车站定位，立刻打了电话过来。"你在县城？！怎么不提前说一声？""临时决定的。明天还得赶回沿江。"那行。晚上七点，老地方。我去订位。"

"那家店还在不在？"陆允川问。

"在。换了个老板——原来的老板娘回老家带孙子了，把店盘给了她侄子。菜的味道没变，就是辣椒比以前更舍得放了。"程砚停了一下，"你穿什么？"

"防晒服。"

"……换一件。"

陆允川回了一趟家。母亲对他的突然出现毫无准备——正在院子里晒豆角，看到他推门进来，手里的筛子差点掉了。"怎么回来了？""出差路过。"他翻出旧衣柜里一件干净T恤换上。母亲在旁边站了一会儿，说"胖了一点"，又说"晒黑了"，然后转身去厨房给他切了一盘西瓜。

晚上七点，两个人坐到了当年升学宴那张桌子旁边的卡座。玻璃转盘还在，但比以前小了一圈——可能换了新的，或者是他自己长高了。菜单压了一层塑料膜，破了一个角，上面用手写加了一道"老板推荐：水煮牛肉"。旁边还贴着二维码，可以扫码点餐。程砚没扫——他说来老店必须看纸菜单。

程砚没有穿衬衫——穿了一件厂里发的文化衫，上面印着"XX机械厂安全生产100天"，领口洗得有点松了。他比以前胖了一点，主要是肩膀那一块——车间干活练出来的。陆允川也胖了一点，也是肩膀——还有小臂，晒黑了不少。

"你后来跟哪些人还有联系？"程砚一边翻菜单一边问，手指在过塑的纸面上滑过去。

"叶澄。你。没了。"陆允川从筷子纸套里把筷子抽出来，习惯性地互相刮了刮毛刺——这是他爸教他的动作。

"林曼呢？"

"群里有她的朋友圈转发。她现在在省城一个文化馆做编辑。没聊过私信。"

"祁野呢？"

"去年在省城碰到过一次。他来我们设计院送图纸——他转行做施工了。晒得跟我差不多黑。我们站在大厅里聊了十分钟，他说他马上要当爸爸了。"陆允川摇了摇头，"祁野。当爸爸。你能想象吗？"

"不能。他在我脑子里永远是那个做数学卷子做到摔笔的形象。"程砚笑了一下，"夏知遥呢？"

"没有。"陆允川把筷子从纸套里完全抽出来，"你上次问我这个问题是什么时候？"

"六年前。大二寒假。你那时候说没有。现在还是说没有。"

"因为真的没有。群里有她的消息——她参与的那个博物馆项目竣工了，叶澄转发了新闻链接。设计团队名单里有她的名字。我点进去看过照片。"陆允川的声音很平，"然后我就退出来了。没有点赞，没有评论。"

菜上来了。水煮鱼、回锅肉、麻婆豆腐、一盆酸菜粉丝汤。跟六年前一模一样的四道菜。新老板端菜上来的时候围裙上全是辣椒油，他说"你们两个是老客吧？我姑妈说以前有个升学宴的男生经常来，后来去了外地上大学。"

程砚看了他一眼。"你姑妈是——？"

"原来的老板娘。"

"那男生就是我。"

老板愣了一秒，然后哈哈大笑。"等一下，我给你们送个菜。姑妈交代过，以前升学宴的客人回来了送一份红糖糍粑。"

程砚等老板走了，压低声音对陆允川说："你听到没。我六年前吃了一顿饭，六年后的红糖糍粑是免费的。"

"你的面子保质期有六年。"

"不止。起码十年。"

程砚把鱼转到陆允川面前。"你先吃。你上次帮我挑了刺，今天我帮你挑。"

"你会挑鱼刺？"

"不太会。但可以学。"程砚拿起筷子，夹了一块肚子上肉最厚的鱼块，开始低头挑刺。他的动作很笨——在车间拧螺丝的手指太粗，鱼刺又细，挑一根要花十秒。有几次戳破了鱼肉，鱼肉碎成几块，他把碎的也夹到自己嘴里吃了，完整的留给陆允川。陆允川看着他挑了一会儿，笑了。"你好像一只熊在拆炸弹。"

"熊能把炸弹拆了那是天才。"程砚头也不抬，继续跟鱼刺较劲。额头上居然冒了一点汗——饭馆空调温度不低，加上他全神贯注。

鱼刺挑了一半，程砚忽然说："其实我一直觉得，那次升学宴是你高中时代的一个重要节点。那个节点如果写成论文，关键词应该有三个：我妈说你是我最好的朋友、你帮我应酬我三叔、还有那张桌子上的所有人。"

"为什么？"

"因为你那天晚上第一次在很多人面前没有低头。我三叔问你土木工程是干什么的，你说'就是造桥修路的'。"程砚把挑好刺的鱼肉夹到陆允川碗里——那块鱼肉已经凉了一半，"那是我第一次看到你用一句话回答一个人，而不是用三句话解释自己。你说完之后，三叔点点头就懂了。那个瞬间我就觉得——你以后能行。"

"你后来在大学里做presentation、答辩、面试的时候，是不是都是用的那个语气？就那种'一句话说清一件事'的方式。"

陆允川想了想。好像是。大一第一次在专业课上做报告——讲的是桥梁基础的类型和选择原则——他站在讲台上手有点冷，投影仪的遥控器按了两次才亮。然后他想到了程砚妈妈那天的笑脸，想到了她说"你回去告诉你妈妈，你这孩子是我们程砚最好的朋友"。然后他的紧张就消了一半。不是完全消了，但够用了。

"你妈那天说的话，我记到现在。"陆允川夹起那块挑好刺的鱼肉放进嘴里。鱼肉嫩，微辣，带着豆瓣酱的香味。跟他记忆里一样。

"她说什么了？"

"她说我是你最好的朋友。她说这句话的时候手里还握着叶澄的手。她把两个夸奖连在一起了——'你以后要当老师'和'你这孩子是我们程砚最好的朋友'。我觉得她不是随便说的。她是想让我知道，在她眼里我不只是你同学。"陆允川放下筷子，"我回去告诉我妈了。我妈说，程砚妈妈人真好。我说对。"

程砚放下筷子看了他一眼。那双眼睛在饭馆的黄色灯光下有一点亮——是那种水光，但没落下来。然后他又把筷子拿起来，夹了一块回锅肉塞进嘴里，嚼了几口。"肉还是那个味。够咸。也够辣。"

红糖糍粑上来了。是完整的一大块，还没切开，表面撒了黄豆粉和红糖浆。老板亲自端上来的，说"这盘不收钱，姑妈交代的"。程砚拿起筷子把糍粑切成两半，夹了一半给陆允川。

"为她——前程似锦。"程砚举了一下糍粑。

"为谁？"陆允川愣了一下。

"为你。六年前我为夏知遥举了一次杯。今天为你。前程似锦。"

陆允川夹起糍粑碰了一下。红糖浆黏黏的，拉了一条长长的丝。

吃完饭，两人在店门口站了一会儿。街对面的水果店还在——但换成了连锁品牌，门头亮着LED灯，二十四小时营业。水果店隔壁的便利店也换了。以前修自行车的周师傅的摊位不见了，那块地方现在停了一排共享电动车。只有那家川菜馆还在，招牌换了"老张川菜"，但辣椒还是一样辣，豆瓣酱还是一样的牌子。门口的玻璃门上还贴着六年前那张"冷气开放"的贴纸——被太阳晒褪了色。

"你觉得下次我们回来，这家店还会在吗？"程砚问。他站在店门口的路灯下，那件文化衫被风吹得贴在身上。

"会。只要老板还炒得动菜。只要吃的人还来。"陆允川把打包的酸菜粉丝汤拎在手里——打包盒已经不是泡沫的了，是塑料的。"走吧。你回去帮我跟你妈说，她当年那句话我记了十年。不，不止十年。从十八岁到现在。"

"从十八岁到现在是六年。你说十年。"

"我提前预支了四年。因为四年后我肯定还记得。"

"你数学变好了。"

"是你语文退步了。连'预支'都要我解释。"

程砚笑了——那种笑他把嘴角扯得很开，跟六年前一样毫无保留。"行。我回去跟她说。她会笑死。她最得意的就是那次升学宴。每次过年有亲戚来她都把升学宴的照片翻出来给人看。"

两人在街口分开。一个往南，一个往北——跟六年前一样的路线。路灯把他们的影子拉成跟六年前一模一样的角度。只不过这次影子比那时候宽了一点——两个人都胖了。但是肩膀都更直了。

陆允川走了几步，回头看了一眼。程砚正低头看手机——大概是在给他妈发消息说"陆允川回来了，我们一起吃了饭"。路灯把他的轮廓勾了一道黄色的边。

陆允川转身继续走。手里拎着的酸菜粉丝汤还是热的，透过打包盒的塑料壁，暖着他的指关节。
