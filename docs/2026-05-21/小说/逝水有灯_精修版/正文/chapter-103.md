# 第103章 诗社传单之后

所属部卷：第二部：六月以后
时间：大一社团周
地点：图书馆前广场

## 诗歌插页 P103：原创仿宋小令

红章边，晚声轻，
半窗灯影落无凭。
问旧年、几回执念，几回分明。
人去不须追远路，
水流仍向晓光生。

回宿舍时天已经黑透了。陆允川推开寝室门，屋里只有徐远的台灯亮着。他在打游戏，屏幕上花花绿绿的技能特效把他的脸映得一明一暗。

"报了什么社团？"徐远头也没回。

"诗社。"

"诗社？"徐远扭头看了他一眼，"咱们学校还有诗社？"

"有。人文楼401，每周三晚上活动。"

"那有什么意思。"徐远把视线转回屏幕，"你不如跟我报电竞社。社长说今年要组织校际联赛，奖金不少。"

"我不打游戏。"

"你不打游戏，不打球，不谈恋爱——那你上大学干什么？"徐远问得随意，但陆允川听进去了。

"写诗。"陆允川把鞋踢进床底。

"写诗能当饭吃？"

"不能。"

"那不就完了。"

宿舍安静了一会儿。键盘声噼里啪啦。

"但你写了，也不妨碍什么。"徐远忽然补了一句，"我就是随便问问，你别往心里去。"

陆允川把背包放在桌上，从里面抽出那张传单。诗社的报名表回执夹在中间，纸质偏薄，已经有点皱了。他把它夹进诗抄本的第47页——正好是空白页。

诗抄本是高中时候买的，硬壳，棕色封面，书脊已经磨出了白边。前半本抄满了高中写的诗，后半本还空着。他用拇指快速翻了一遍，翻到三年前写的一首《风铃》，停了一下。

那时候他还在喜欢夏知遥。这首诗是坐在操场看台上写的，旁边坐着程砚。程砚看了一眼诗稿，只说了一句："太文了。你写的不是风铃，是你自己在响。"

陆允川把诗抄本合上，打开笔记本电脑。QQ上有两条新消息。

一条是叶澄发来的："晚枝学姐说你的字写得好看。她说'现在写字好看的男生不多了'。附注：我只是转达，不代表本人观点，也不代表任何暗示。"

陆允川盯着屏幕看了几秒钟，打字："你原话转达的？"

叶澄秒回："差不多。她还问你高中那些诗能不能带来给她看。"

"下次活动吧。"

"周三。别忘了。人文楼401。"

一条是程砚发来的。只有一句话："大学还好？"

陆允川回："还行。今天报了诗社。"

程砚隔了一会儿才回，发来一张照片——他宿舍的书桌，桌面上摊着一本《机械原理》，旁边搁着一杯速溶咖啡。"我在啃这个。你能写诗，挺好。"

"你那边有文学社吗？"

"有。去了一次。社长是个戴帽子的女生，说社团宗旨是'用文字记录时代'。我说我只想写日记。她说日记也是文字。我说那行。"

陆允川发了一串省略号。

"你那个诗社的社长是什么人？"程砚问。

"文学院的学姐。话不多，看东西很准。"

"看东西很准——这个评价很高。"

"她就看了我写的几个字，说我应该多写。"

"那她确实看东西很准。"

陆允川没有马上回复。他去饮水机接了一杯水，站在窗边喝了两口。窗外是学校的北操场，操场灯已经亮了，有几个人在跑圈。跑道旁边的单杠上挂着一件校服外套，不知道是谁忘了收。

"夏知遥的大学离你多远？"程砚忽然问。

陆允川端着杯子的手停了一下。"火车四个小时。"

"你去过吗？"

"没有。高考完以后就没联系过了。"

"那她现在知道你在诗社吗？"

"应该不知道。"

"挺好的。"程砚打完这三个字，隔了一会儿又发来一条："我不是说'她不知道'挺好。我是说——你现在报诗社、写诗，不是因为想让她知道——这挺好的。"

陆允川把这段话读了两遍。

"那你呢？"他回过去。"大学有没有遇到什么人？"

"有。遇到一个室友，每天晚上打呼噜。"程砚发来一张室友睡觉的照片，被子裹得像个蚕蛹。

陆允川笑出声。徐远回头看了他一眼。

"跟谁聊天？这么开心。"

"高中同学。"

"男的？"

"男的。"

"那还行。我还以为你在跟诗社那个学姐网聊。"

"没有她联系方式。"

"那你打算什么时候要有？"

"下次活动。"

徐远在屏幕后面笑了一声，没有接话，继续打游戏。

陆允川又发了几条消息，然后合上电脑。他把传单从诗抄本里拿出来，对着台灯看了一会儿。传单的排版很朴素，标题是"诗酒趁年华"，正文用宋体，字距稍宽，看得出是手工排的。最底下有一行联系方式，苏晚枝的手机号以"138"开头，后面八位数他用手指遮住，一个一个念了一遍。没有存。

十点半，祁野推门进来。他刚洗完澡，头发还在滴水，毛巾搭在肩上。他先走到自己的床位把脏衣服扔进洗衣盆，然后走到陆允川旁边，看见他桌上的传单。

"还在看？你今天看了几遍了？"

"第二遍。"

"第二遍？"祁野把湿毛巾折起来挂好，"我数着至少第五遍了。晚饭的时候拿出来看了一遍，回宿舍路上又拿出来看了一遍，刚才跟程砚聊天之前又看了一遍。"

"你怎么知道我跟程砚聊天？"

"你打字的时候在笑。你只有跟高中的人聊天才会笑。跟大学的人你都是那种'礼貌微笑'。"

"什么礼貌微笑？"

祁野学了一下：嘴角微微上扬，眼睛不动。"就这种。看着很客气，但其实什么都没透露。"

陆允川把传单收进书里。

"我不是批评你。"祁野坐在自己床上，开始擦头发，"我只是觉得奇怪。你今天明明挺高兴的——报了诗社，跟那个学姐聊了十几分钟——但你回来以后一个字都没跟我们说。"

"说了。我跟徐远说了。"

"你说的是'我在诗社填了个表'。这是陈述事实，不是分享高兴。"

陆允川转过头，发现祁野说这话时没有看他，在专心擦头发，像只是在说一件很平常的事。

"我怕说多了显得——"

"显得什么？显得你在乎？"祁野把毛巾扔到床头，"陆允川，你有没有想过：在乎一件事跟别人说出来，这很正常。不正常的是你每次都假装自己不在乎。"

宿舍里沉默了一会儿。徐远戴着耳机打游戏，键盘声噼里啪啦。窗外操场上的灯灭了一盏。

"那个学姐，苏晚枝。"陆允川开口，"她说我的名字'允'字是诚实的意思。"

"你今天说过这个了。"

"我反复想这件事，是因为——"他停了一下。"因为以前没有人这样看过我的名字。夏知遥没有，程砚没有，连我妈都没有。他们只是叫我'小川'或者'允川'。她是第一个停下来，想了一下这两个字是什么意思的人。"

"所以你很高兴。"祁野说，这次不是问句。

"嗯。"

"那你直接说'我因为学姐解读了我的名字所以很高兴'就行了。不用绕那么远。"

陆允川看着祁野。祁野也看着他。

"你这个人有一个最大的问题：你总觉得自己的感受需要理由。但其实不需要。高兴了就是高兴了，在意了就是在意了。你不需要先证明'学姐的态度不是暗示'才有资格高兴。你高兴是因为有人认真看了你的名字——这跟暗示不暗示没关系。"

外面的灯又灭了一盏。

"你说得对。"陆允川说。

"我当然对。我是谁。"祁野躺下去，把被子拉到下巴。"关灯了。你如果要写诗，小点声打字。"

"我不写。"

"那你关灯。"

陆允川伸手把灯关了。黑暗中，徐远的屏幕还在闪，机械键盘的声音像一阵阵急促的雨。

他把传单塞回诗抄本第47页，想了想，又在书脊空白处用铅笔写了几个字："允，信也。"
