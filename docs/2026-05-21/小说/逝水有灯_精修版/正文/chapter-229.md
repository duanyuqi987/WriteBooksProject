# 第229章 南湖再见旁观

所属部卷：第五部：逝水有灯
时间：十年后冬天
地点：苏晚枝所在城市

## 诗歌插页 P229：原创现代诗

我把新绿漆放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

苏晚枝知道南湖的凉亭刷了新漆。

不是陆允川告诉她的。是她同一天下午看到的——一个当年也在诗社待过的学妹发了朋友圈："回母校，南湖亭子刷了浅绿的漆。没有当年深绿的好看。"

她点了个赞。然后把手机放在一边，继续做手头的事。

手头的事是报销单。她的工位在办公室靠窗的位置，窗户下面是停车场。三张发票，两张收据，一张手写的说明——因为有一张发票的开票日期写错了。打印纸的边角被手指捏出了一个小折痕，她用指甲把它抚平了，然后又折了起来。她把错误的那张拍照存证，然后在OA系统里发起了一次报销申诉。这种事她做过不下五十次。每一次都要附上说明，每一次都要等财务部门的二次审核。她做这件事的时候不生气，但是会有一种慢吞吞的厌烦——不是讨厌工作，是讨厌被一件简单的错误拖着。

发完申诉以后，她把手机拿回来，重新点开了学妹那条朋友圈。办公室里的空调嗡嗡响着，同事在隔壁工位打电话——声音压得很低，但偶尔飘过来一两个词：预算、截止日期、下周一。

照片里的凉亭确实比十年前浅了。浅绿色在阴天显得有点发白，像一件新买的衬衫洗了太多次。她记得以前的深绿——那种颜色在阳光下偏蓝，在阴影里偏黑。她倚着那根柱子站过很多次。手指抠掉过一块漆。那块漆被她的指甲弹进了湖里。

她不知道那块漆的碎屑后来去了哪里。大概沉在湖底的某个地方。或者被水冲走了。

她给陆允川发了条消息："听说南湖亭子刷了新漆。"

他回得很快："你怎么知道？我今天刚去过。"

她回："学妹发的朋友圈。"

他回："新漆把旧的那层盖了。摸上去很滑。"

她回："那你还找得到你当年做记号的柱子吗？"

他回："找得到。最左边那根，木纹是直的。其他柱子纹路是弧形的。"

她回："你又数了。"

他回："职业病。"

她把手机放下，靠着椅背。窗外的银杏已经落了大半，剩下的叶子像一小片一小片的黄纸片，挂在枝尖上。她记得南湖边也有银杏。那些银杏现在应该也落了——南方的冬天来得晚，但该来的总会来。

鲁萱发消息来："你们怎么最近都在发母校朋友圈？"

"谁发了？"

"学妹。程砚。叶澄。还有陆允川——他没发朋友圈，但他给所有人的母校照片都点了赞。"

"可能校友卡开始办了。大家都要回校一趟。"

"那你怎么不回去？"

"没什么需要办的。我的校友卡毕业那年就办好了——我那时候就预感到以后会用到。"

"你真有远见。"

"不是远见。是懒得以后跑一趟。"

她把报销单放进文件夹。文件夹的左边贴着一张标签：已办。右边贴着一张：待处理。

手机又震了。这次是工作邮件推送到手机上——一个项目验收的时间改到了下周。她回了个"收到"。

然后打开备忘录。备忘录里有一条很久以前写的备注："南湖灯泡——画得很丑——像洋葱。"

她把这条备注重新看了一遍。然后加了一个新备注："南湖凉亭。新漆。浅绿色。柱子最左边一根的木纹是直的。"

关了手机。银杏叶子又落了一片。明天再有一阵风，大概就剩枝了。

她站起来走到窗边。停车场的银杏树下铺了一层薄薄的黄叶，清洁工还没来扫。有一片叶子卡在车窗雨刮器上，被风吹得左右摇晃但就是不掉。她盯着那片叶子看了很久，忽然想起大学时候叶澄跟她说的一句话——"人跟树叶一样，不是被风吹走的，是时候到了自己松手。"

当时她觉得这句话太文艺了。现在她理解了它的另一层意思：松手不是放弃。松手是承认重力是真实存在的——不管你怎么用力抓住，到了秋天叶子就是会落。不是它不够坚强——是它的生物程序到了。

她坐回工位，继续处理下午没弄完的报销单。报销单上的数字很精确——交通费二百三十块，餐费九十六块五，办公用品四十五块。她逐项核对，用红笔画了一个圈标注其中一项发票号码写错了。生活中的精确有一种奇怪的安抚作用——当情感世界充满了无法量化的模糊和未竟之事，至少报销单上的数字可以对得上。

手机最后一次亮了。是陆允川发来的——他在南湖拍的那张夜景照片。照片里什么都看不清——一盏路灯变成了一条斜长的光痕，湖是黑的，亭子的影子也是黑的。下面附了一行字："拍糊了。不够清楚。但保留了。"

她看着这行字。她懂"保留了"是什么意思。不是每件不清楚的事都需要被弄清楚。有些东西模糊本身就是它的完整形态——就像她抠掉的那块漆，现在在湖底的某个地方，被水草缠着，被泥沙盖着，永远不可能被找回来。但这件事不需要被解决。只需要被知道。

她回了一个"+1"。

他回："程序员用语？"

她回："不。是'同上'。"

然后她把手机调成免打扰。报销单的最后一项处理完毕，放进"已办"文件夹。明天要交季度报告，后天要出差，生活不会因为一个人的感怀而暂停运转。但她在备忘录里多记了一条新备注——"今天他说'保留了'。这个词可以用。"
