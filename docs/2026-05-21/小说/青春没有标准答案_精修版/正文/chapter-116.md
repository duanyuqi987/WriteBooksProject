# 第116章 同学录第一页



时间：2010年七月中旬，暑假补课名单下发



地点：高二十班教室、女生宿舍



卷次：第五卷：未来在黑板上倒计时



2010年七月中旬，暑假补课名单下发，高二十班教室、女生宿舍，孙佳怡同学录第一页写下的补课名单把2010年的春夏推到十班面前。孙佳怡把暑假补课名单抄到同学录第一页，方便母亲安排接送。

她讨厌被家里盯得太紧，却又不得不承认自己一个人回家确实不方便。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

孙佳怡也在这段日子里显出自己的位置。家里管得紧，电话一来就紧张，不愿让同学看见自己的不自由。群像不是名单，群像是每个人都有自己的难处。

“我妈要我把每个补课时间都写清楚。”孙佳怡说。

“你妈这是排课表还是排兵布阵？”宋乔说。

“她担心也是真的。”常俊凡说。

“可我也想自己走一次。”孙佳怡说。

“和家里谈，不要只用赌气谈。”叶书兰说。

同学录第一页原本应该写名字和生日，到了这年夏天，却被写成了补课时间、路线和电话号码。

有人笑这不像同学录，像作战地图。笑完以后，又认真把自己的安排补上去。

新华书店、补课班、校门口、县城路口，这些地方被写在一页纸上，像十班临时散开的几个支点。

常俊凡写字很小，时间和地点分得很清楚。段璟看着她那一行，忽然觉得她把未来也写得这么有秩序。

赵远航讲路线时最有精神，哪个路口左拐，哪条巷子近，他都能说得像背课文。

宋乔嫌麻烦，却还是把女生宿舍电话也写了上去，说万一有人找不到人。

同学录被传到最后，纸角已经软了，第一页却比任何复习计划都像真的。

那一页没有写离别，可每个人都看得出来，时间正在把他们往不同地方分开。

叶书兰这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

孙佳怡被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

陈柏言把补课时间写在同学录第一页，竞赛名单的纸角已经微微卷起。老师从门口经过，原本散开的声音立刻往回收。

张若楠在电话号码旁边加了一个括号，新华书店塑料袋在手里沙沙响。这样的细节不会被写进红榜，却会留在很多人的夏天里。

刘青岚把新华书店路线画成细线，桌洞里的练习册露出红色封皮。这点动作很小，却把那天的压力落到了实处。

王子越把传到手里的本子往后递，树莓摊旁边的纸盒被太阳晒软。旁边的人没有追问，翻书声把难堪遮住。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，孙佳怡同学录第一页写下的补课名单还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
