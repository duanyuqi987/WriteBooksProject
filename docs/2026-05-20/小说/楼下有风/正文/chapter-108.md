# 第108章 回到校园之后



回到校园之后，他没有立刻离开。校庆人群往礼堂方向走，他一个人绕到旧小吃街，看见摊位换了招牌，油烟却还是从同一个角度飘出来。

校庆让校园显得既熟悉又陌生。新楼玻璃很亮，旧树还在，路牌换了字体，风却仍从操场那边慢慢吹过来。

校门口的路牌换了字体，底下还残着旧螺丝孔。

他走到东门外，发现那个曾经通宵的网吧已经不见了。水果店老板正在给橘子喷水，水珠落在青皮上，像一场很小的雨。

周一鸣没能来，在电话里让他替自己看一眼操场。许亦安说好，声音很平，心里却像有人轻轻推开一扇旧门。

他从校门往里走，没有刻意绕开任何地方。经过老花坛时，他停了一下，又继续往前。能继续走，已经比当年好很多。

后来他写到她时，常常故意少写几句。少写不是冷淡，而是尊重：有些人生长在他看不见的地方，不该被他用想象占满。

从她那里看，许亦安的等待既让人感激，也让人害怕。

许亦安并不擅长承认自己受伤，他更习惯把受伤说成玩笑，把玩笑说得像没事。

周一鸣没能来，在电话里让他替自己看一眼操场。这句话在当时没有用，后来却常常从记忆里浮出来。这话许亦安当时不爱听。他更愿意相信自己看见了别人看不见的东西：她笑时眼睛弯起来的弧度，她把吸管咬扁的习惯，她说“随便你”时其实并不是真的随便。喜欢一个人后，人的听力会变奇怪，能听见很远的暗示，却听不见近处的提醒。

同学群里有人说学校变化太大，有人说食堂还是难吃。许亦安看着这些抱怨，觉得它们亲切，因为抱怨也说明大家真的回来过。

水果店老板往青橘上喷水，水雾在阳光里亮了一下。

他买了两只青橘，剥开一瓣尝了尝，很酸。酸味让他想起很多年前没有说出口的下午，却不再让他皱眉。

他不是不知道自己这样不好，只是知道和做到之间隔着一整条晚自习后的路。

校庆晚会结束后，他在校门口等车。路灯照着青橘摊，老板把剩下的水果搬回店里。许亦安忽然觉得，过去没有消失，只是换了招牌。

校门口的青橘不解释青春，只提供一种酸而清亮的味道。许亦安把它尝完，就把果皮丢进路边垃圾桶。

操场广播试音，喇叭里传出短促的电流声。

回到校园不再等于回到当年。旧网吧换了招牌，许亦安也换了一种走路方式：经过，停顿，然后继续走。

他没有把校门照片发给林青禾。校园属于很多人，不再只属于他那段没结果的等待。

他剥开青橘，酸得皱了一下眉。皱完以后，他笑了，发现自己终于能让一个味道只是味道。

第二天离开前，他又从东门走了一遍。水果店还没开，卷帘门上贴着促销纸，纸角被风吹得卷起来。

楼下早餐店的豆浆装在纸杯里，没有塑料袋，也没有谁故意让他吃辣。他站在队伍里等了一会儿，忽然笑了笑。原来有些味道不是被忘记，只是终于从某个人身上退回到食物本身。

上班路上，他看见地铁广告牌换了一轮又一轮，像许多来不及说出口的话被新的生活覆盖。可覆盖不是抹掉，底下的痕迹还在，只是不再要求被所有人看见。

校庆的人群从身边经过，胸牌晃来晃去，每个人都像带着自己的旧版本回来短暂停留。

青橘的味道酸得清亮。许亦安后来记住这味道，比记住哪栋楼翻新了更久。

这次回校不再像寻找答案，更像确认自己可以经过旧地方而不被旧地方带走。

回去的路上，他看见写字楼门口换了新的海报。旧海报被卷起来放在墙角，露出背面泛白的纸边。许亦安看着它，忽然觉得记忆也可以这样被收起，不必撕碎，也不必展示。

他没有把这些想法告诉林青禾。那时候他还以为，真正深的感情应该自己消化，不能给对方添麻烦。后来他才知道，不能说的感情很容易在心里变形，变成委屈，变成审问，变成某个夜晚突然冒出来的重话。

他没有把这些段落发给任何人看。保存不再是证明，而是整理，像把旧衣服洗净叠好，放进不常打开的柜子。

那一行字写着：老网吧变成水果店。过去没有消失，只是不再按原来的名字等我。

修订时他发现，很多当年以为惊天动地的瞬间，其实只需要一个具体物件就够了。物件安静，安静才耐看。

写到这里，他删掉了一句很漂亮却太满的话。漂亮的句子若抢走了人的真实，就应该让开，让普通的动作自己说话。

他把校园从私人伤口里放出来，让它重新成为很多人的校园。这样一来，风也不再只吹向林青禾。

这一章真正留下来的，不是他当时以为最重要的那句话，而是青橘的味道酸得清亮，像某个没有说出口的下午。那画面后来被他想起很多次，每一次都比第一次轻一点。

这一页日记后来被他保存了很多年，保存到里面的怨气都慢慢变轻。许亦安合上电脑，听见窗外有人笑着从楼下经过。那笑声很快远了，房间重新安静下来。他坐在书桌前，觉得这段旧事好像没有结束，又好像早就结束了。
