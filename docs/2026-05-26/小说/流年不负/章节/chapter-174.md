# 第174章 伴娘
时间：2019年5月1日
地点：青阳县月湖边酒店
卷次：卷九：执手之约

许若琳从苏晚晴化妆开始就在擦眼泪。

其实那会儿还早。化妆师刚给苏晚晴画了眉毛——眉头到眉尾，一笔下来没断。许若琳说"好看"。声音很正常，没有哭腔。但过了大概十秒钟——化妆师开始画眼线的时候——她忽然转身去了卫生间。

苏晚晴从镜子里看到她了。没说话。

许若琳在卫生间里待了两分钟。出来的时候眼睛下面有一点红。她用纸巾按了按——纸巾上留下一点水渍。

"你哭了？"

"没有。睫毛掉进去了。"

苏晚晴没拆穿。化妆师在她眼皮上打了一层淡金色眼影。粉扑的触感和海绵一样轻。她闭着眼。嘴角没弯但也没压下去——是平的。

许若琳坐在旁边的凳子上。把手里的纸巾揉成一小团。又展开。又揉。

她想起大一开学那天。宿舍门口。四个新生。苏晚晴最后一个来——拖了一个蓝色行李箱，轮子掉了一个。她妈说"去修"，她说"不用——拖着走就行"。那天她穿着一件白色T恤，扎马尾，面无表情地把箱子拖进宿舍。许若琳当时想——这个人看起来不太好相处。

后来熟了。大二的时候——有一天晚上。苏晚晴忽然在宿舍里说了一句话。不是对着任何人说的——是对着天花板说的。她说"我高中有个人。他在看我的时候我其实都知道。"许若琳问她是谁。她没说。翻了个身被子盖住了头。

大三。支教。在贵州一个小山村。她跟苏晚晴住一间——墙壁是土墙，窗户糊着报纸。有一天晚上下雨。雨打在报纸上叭叭响。苏晚晴忽然说——"他也在支教。"许若琳说"谁？"苏晚晴没回答。隔了两分钟说——"写诗的那个人。"

大四毕业。她帮苏晚晴收拾行李。在书架上翻到一本旧笔记本。不是教材——是手写本。封面上写着"流年"两个字。她还没来得及翻开，苏晚晴就抽走了——"这个不是书。"然后藏进了一个不透明的袋子里。

后来许若琳知道了那个本子里写了什么。是在相亲之后——苏晚晴回宿舍说的。她坐在床沿上。两只手抱着膝盖。说"若琳。我见到他了。"许若琳问"谁？"苏晚晴说"写诗的那个人。"

六年的秘密。六年的等待。许若琳全知道。不全知道——苏晚晴说的很少。但每个细节她都听得出来里面有很重的东西。

今天那些很重的东西终于落下来了。

化妆到一半。苏晚晴开始戴耳环。一对白色珍珠耳钉——不是真的珍珠，是淡水贝珠。顾临风买的。她说不要太大的。他说他知道——买的这对只有绿豆大小。

许若琳凑过去帮忙。苏晚晴的耳洞很小——大一跟许若琳一起打的。在楚州大学城的一个小店——没有麻醉药，银针一下扎过去。许若琳当时被扎哭了。苏晚晴说"不疼"。但许若琳注意到她把椅子扶手攥出了四个指头印。

"耳环——"

"我自己来。你手抖比我厉害。"

许若琳把手收回去。手指确实在抖。不是因为紧张——是因为她压了太多年。从大一到今天——她压了八年。苏晚晴从来没哭过。但她坐在苏晚晴旁边的时候，能感觉到那些没有流出来的眼泪去了哪里——它们沉淀了，在心里结了霜，但没化。

她替苏晚晴哭。

苏晚晴把耳环戴好。站起来。走到镜子前。许若琳跟过去。站在她身后。镜子里的两个人——一个穿婚纱的新娘，一个穿浅粉色礼服的伴娘。新娘的眼角很干净。伴娘的眼角又湿了。

"你又哭了。"

"我没哭——你看，笑——"

许若琳把嘴咧开。但她笑得比哭还难看。

苏晚晴看了她一眼。从化妆台上拿了一张干净的纸巾。递过去。许若琳接过去，没擦——纸巾揉在手里。

下楼的时候。许若琳在后面帮苏晚晴提着裙摆。楼梯不陡。但苏晚晴的婚纱下摆太长——缎面面料拖在楼梯台阶上，每一级台阶都要小心。许若琳一级一级地往后拉裙摆。每一步都像在把自己的好朋友往岸上拉——拉了整整八年。

婚礼开始。从主持人讲话到顾临风发言、到赵一鸣发言。许若琳一直站在苏晚晴侧后方——伴娘的标准站位。她的位置正好可以看到苏晚晴的侧脸。那个侧脸跟大一时宿舍里对着天花板说话的侧脸一模一样——下巴的弧度、鼻梁的高度、马尾的方向。

但不一样的是眼睛。

大二说"我高中有个人"的时候，苏晚晴的眼睛是暗的。像关了灯的窗户。现在那盏灯亮了。

许若琳拿出纸巾。

赵一鸣发言到一半的时候。许若琳的纸巾已经湿透了。她换了第二张。苏晚晴用手肘轻轻碰了她一下——"别哭了，轮到你的时候我可不会哭。"

许若琳把纸巾收起来。看着苏晚晴。

"你说的。"

"嗯。"

"我记住了。到时候你不哭——"

她想说"到时候你不哭就不是人"。但没说出口。因为赵一鸣发言完了。赵一鸣说"兄弟，这十年的结果，是你该得的。"然后掌声响了。

轮到她了。

许若琳走到麦克风前之前。她的左脚在右脚上绊了一下——但她没停。直接走到麦克风前。粉色礼服的裙摆被她的快走带起来了一个角。

她站在麦克风前。看着台下的人。很多人她认识——苏晚晴的亲戚、顾临风的亲戚、高中同学、大学同学、同事。她张了张嘴。没发出声音。

"我——"

清了清嗓子。重来。

"我叫许若琳。苏晚晴的大学室友。八年了。"

她的声音很轻。但大厅里很安静——轻的声音也能被听到。

"大一开学那天。苏晚晴最后一个来宿舍。拖着一个坏轮子的箱子。我说'我帮你'。她说了两个字——'不用。'"

台下有人轻声笑了一下。是苏母。苏母知道她女儿——从小说"不用"说到现在。

"后来熟了。我发现这个人不太会说'好'。说'好'之前要在心里想很久。大三支教的时候——我说咱俩挤一张床你不介意吧——她想了两分钟。说了'好'。"

许若琳的声音开始抖。

"所以我知道。她今天说'好'的时候——她想了很久。从大一到现在。她一直都在想。想着什么时候能见到。想着还能不能见到。想着见到之后还会不会分开。"

她停下来。把麦克风往嘴边拉近了一点。麦克风杆子上系了一朵白玫瑰——是婚庆公司绑的，花已经开始蔫了，花瓣边缘有一点枯黄。

"你们看到的是今天的她——婚纱、戒指、笑。但你们不知道她等这一天等了多久。"

她的话越来越轻。每句之间换了两次呼吸。

"你们不知道——大二那年冬天。她在图书馆看到一个男生的背影。不是他。但她盯着看了很久。回宿舍之后说'真的很像。'我说'谁？'她翻了个身。被子蒙住脸。说'没什么。'"

"你们不知道——毕业后。她在楚州一小教书。离他工作的十八中只有三公里。三年。一次都没偶遇过。她会在放学后磨蹭很久——改作业、备课、画黑板报。我打电话问她怎么还不回家——她没说。但我知道。她在等——"

她停住了。呼吸了一口。

"她在等某个骑车经过的人。那个人三年骑车经过一小门口。每次她都不在。每次都错过了。"

苏晚晴低着头。手里捧着的雏菊轻轻地晃了一下。

"其实这些事她都没说过。她从来不说。什么都憋在心里。连她的名字都是——'晚晴'。下午放晴——雨下完了最安静的那一刻。"

许若琳转过头。看着苏晚晴。眼泪流出来了——她来不及擦，就让泪在脸上挂着。

"苏晚晴。八年了。你今天终于是——敢说的那个人了。"

这句话说得很轻很轻。麦克风没收到足够的声音——前两排没听清。但苏晚晴听到了。

她朝许若琳伸出手。手上戴着碎钻戒指。

许若琳握住那只手。两只手叠在一起——一只戴着戒指，一只空着。一只在手心很稳，一只抖得厉害。

苏晚晴把手里的捧花递过去。白色小雏菊。茎上缠着淡绿色丝带。丝带末端写了两个字——"给你"。

"别哭了。轮到你的时候我可不会哭。"

苏晚晴把捧花塞进许若琳手里。动作很轻。但花茎碰到许若琳手指的时候她愣了一下——花茎是湿的。

不是水。是汗。苏晚晴的手心全是汗。

许若琳接过捧花。白色小雏菊在风里晃了一下。她一边抹眼泪一边笑。一边笑一边扯纸巾擦妆——眼线花了，在下眼睑上晕了一小片灰。但她不在乎。

"你手上这么多年。从来没汗过。"

苏晚晴看着自己的手心。没说话。

"今天是第一次。"

许若琳把湿纸巾揉在手里。然后伸手碰了碰苏晚晴的脸。指尖在她眼角点了两下——那里是干的。

"苏晚晴。你今天没哭。但你手心出汗了。"

"这花你抱着。是给你的。"

"给我当伴娘费？"

"给你的嫁妆。"

许若琳愣了一下。然后笑得很响——跟平时那个话多嘴快的许若琳完全一样了。她把捧花举高了一些。丝带在两朵花之间晃了晃。上面"给你"两个字翻了个面——对着窗外月湖的方向。

"行。我先替你预备着。到时候你们得送回来——双倍。"

苏晚晴没接这个话。她看着许若琳的妆花了——眼线晕到外眼角，睫毛膏沾在下眼睑上。她从化妆台上的湿纸巾盒里抽了一张递过去。

"擦擦。"

许若琳接过来。对着镜子擦了一下——画花的眼线擦掉了一半。另一半没擦到。

"算了。反正今天的主角不是我。"

她说完又笑了。然后去后台洗了把脸。水龙头开得很小——洗得很轻。怕把脸上的伴娘妆洗掉但又想洗掉眼泪。最后洗了半张脸——右脸妆还在、左脸素颜。镜子里一边精致一边干净。

婚礼结束后。许若琳在洗手间补口红——补到一半放下口红管。看着镜子里的自己。她给苏晚晴递了很多年纸巾。今天苏晚晴给了她一束花。花茎上有汗。花心里没有泪。

她把那束雏菊抱好。没有再哭。

然后走出去——在酒店大厅门口。月湖的风吹进来。花茎上的丝带飘起来——"给你"两个字对着湖的方向，在水波的倒影里若隐若现。
