# 第165章 寒风里的台阶多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：城市 / 回忆中

## 诗歌插页 P165：原创仿唐绝句

枯枝微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

多年后，陆允川已经不记得那七本书赠言的具体内容了。

有一次搬家，他在一个旧纸箱里看到那本《海子的诗》——夹在苏晚枝后来托夏知遥退还的几册旧书中。扉页上的赠言还在，字迹有些褪色。赠言的下面多了一个铅笔字："好。"字很小，写在右下角，像是怕占太多位置。

他不记得苏晚枝什么时候写的这个字。可能是当时拿到书的第一天，也可能是后来整理旧书时翻到才补的。一笔一画很轻，但每一笔都很完整。她写什么都这样——不重，但很稳。

他把书合上，放回了纸箱。

此时的他已经有了一份稳定的工作，住在一个不大不小的公寓里。书架上有两百多本诗集——比大二多了不少，但每一本的扉页都空白。他不再在赠书上写东西了。不是怕重蹈覆辙——是学会了：有些话适合当面说，有些话适合留在心里，有些话不适合以书为媒介单向输送给别人。

那晚在女生公寓台阶上的冷风，多年后的某个冬夜还会偶尔回来。不是疼痛，是提醒。

他曾经以为那一夜的意义是"被拒了"。后来发现不是。意义是——他在选择离开的那一刻，学会了把伸出去的手收回来。

收回来比伸出去难多了。伸出去只需要热情，收回来需要尊重。

程砚有一次在聚餐时提起那个台阶的事。彼时他们都已经毕业多年，程砚的头发少了一些，祁野还是每天打游戏——只是换了更好的显卡。

"你知道吗，"程砚喝着酒说，"大二那年你去楼下等她，我其实想下楼把你拽回来。"

"为什么没拽？"

"因为叶澄说不要。"程砚说，"叶澄说——让他去。他需要一次'台阶'。不是物理上的台阶，是人生里的台阶。踩一次，崴了脚，以后就知道这种台阶不能站。"

"你们还专门讨论过？"

"不叫讨论。叫战略会议。"

祁野在旁边一边吃毛豆一边笑。笑完之后难得认真地说："不过说真的——你踩过那次台阶之后，确实不一样了。"

"哪里不一样？"

"以前你是进攻型的。追人、送书、等楼下、写赠言——全在输出。后来你开始会问别人了——问'你方便吗'，问'这样会让你不舒服吗'。问完之后真的听答案，不是假装听。"祁野剥开一颗毛豆，"这就是台阶那晚教的。"

"祁野。"陆允川说。

"嗯？"

"你说话怎么越来越像人话了。"

"因为戒了泡面。现在改吃沙拉了。"

"沙拉让你变聪明？"

"不。沙拉让我痛苦，痛苦让我思考。"

三个人笑作一团。窗外城市的灯光密密麻麻，没有当年的虫声和操场跑道，但这顿饭的味道比任何一顿都香。

苏晚枝后来在另一个城市工作，偶尔在朋友圈发一些日常：花、咖啡、某个周末的徒步记录。有一次她发了一张诗歌分享会的海报。陆允川在下面评论："里尔克的那首你后来读完没有？"

她私聊回他："读完了，用了三个月，刚好一周一首。你说得对，慢读才行。"

"现在还在写诗吗？"

"偶尔。不发表，只是自己写给自己。"

"那就挺好。"

"嗯。"

就这样。对话很短，但每条消息之间不再隔着任何东西。没有试探，没有解释，没有欲言又止。只是两个曾经在台阶上说过话的人，多年后在手机屏幕两端各自过着各自的生活。

这才是叶澄当年说的"不想被追问的自由"。也是程砚那条短信的核心——"不是所有等待都高贵"。他很多年后才完全听懂：等待只有在双方都愿意的时候才是浪漫，否则就只是单方面的自我消费。

那七本书的赠言，他后来在"不再分析的日常"里做了最后一次记录：

"大二送出去的书，多年后在苏晚枝的旧物里看到。她铅笔回了一个字：好。这个字比我当年那些长篇大论更有重量。她说'不重，但很稳'——那也是她自己写字的方式。"

他敲完这行字，按了保存，把文件夹关上。窗外的冬天很冷，但他不需要坐在台阶上了。他可以坐在自己的沙发上，盖一条毯子，看一本没有赠言的书。

有一年冬天他回老家，给母亲看了那本《海子的诗》。

"这本书的赠言是你写的？"母亲翻着扉页。

"嗯。大二的时候写的。"

"写了很多。"

"是太多了。后来就知道不应该写这么多。"

"为什么？"

"因为送书就是送书。送自己就是送压力。"

母亲把书翻了几页，又合上。她的眼神在灯光下有些柔和的光晕。

"你爸以前也给我送过书。扉页上一个字都没写。"

"那你怎么知道是他送的？"

"因为他夹了一张借书卡。上面是他的借阅记录，每一条都刚好比我的晚一天。"

"什么意思？"

"就是他借的书，永远是你爸前一天还的那本。"

陆允川看着母亲。她很少主动说起父亲，但每次说起，用词都很少。跟苏晚枝一样——不多，但每一笔都完整。

"所以他是在追你的读书轨迹。"

"对。但直到最后他也没在书里写赠言。后来我问他为什么，他说——赠言会把书变成信。书应该是书。"

陆允川想到这里，忽然明白了自己大二那年错在哪里。他一直在把书变成信，把台阶变成舞台，把等待变成证明。父亲的做法刚好反过来：他把爱藏进借书卡的日期里，不给对方任何回复的压力。

他靠回椅背，呼出一口气。

"妈，下次复查什么时候？"

"下个月。"

"我请假陪你。"

"好。"

这几年，"好"这个字出现的频率越来越高。从苏晚枝铅笔下的"好"，到母亲电话里的"好"，到他自己对叶澄那句"周末请你火锅"的"好"——这个字正在取代他以前爱用的所有长句。

短不一定浅。长不一定深。有时一个字就够了。

那晚他失眠了一会儿，起来翻了翻手机。祁野在群里发了一张照片——是他新养的一只猫，蹲在键盘上，眼睛半眯着。配文："终于有人陪我打游戏了。"

程砚回复："那是猫。不是人。"

祁野："猫更好。猫不打字说话。猫没有'消息免打扰'按钮。猫爱你它就踩你键盘。不爱你它就离开房间。简单。"

陆允川看了很久，然后回复："你这个描述很像某些人。"

祁野秒回："某些人指我自己吗？"

"不。指之前所有在该聊天的时候选择消失、在不该出现的时候忽然出现的人。"

"你这句话已经没有怨气了。就像在描述天气。"

陆允川看着祁野的回复，没有再接。把手机放回床头，关了灯。

黑暗里他想起大二冬天台阶上那阵风——很冷，冷到指尖发麻，帆布袋提手在掌心压出的红痕第二天才消。但多年后回头去看，那个晚上是一道必要的边界。风把花茎吹得轻轻发抖——那截枯花茎，顶端空着，像一支快写完的笔。他没有用那支笔写完他原本计划好的话——那些话他自己都忘了——但他用剩下的空白写了别的东西，更短，没有赠言。

写给自己，不是写给苏晚枝。

夜还很深，明天要上班。他翻了个身，闭上眼睛。被窝里的膝盖微微发酸——可能是变天了。大二那年坐在冷台阶上的膝盖骨，现在还会在冬天变天时提醒他：那里曾经有一级很冷的台阶，他曾经坐在上面，坐了很久。

后来他走了。不是被人拉走的，是自己走的。

这大概就是叶澄当年说的"方向要对"——走不是放弃，是开始走对的路。
