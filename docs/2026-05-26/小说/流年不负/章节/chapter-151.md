# 第151章 牵手
时间：2018年5月20日傍晚
地点：楚州街头
卷次：卷八：旧诗新读

街灯亮了一排。从月湖路口一直亮到淮海路。橘黄色的光连成一条线，把整条街笼在里面。

两个人的手握着。

她的手指凉凉的。指尖有点湿——辣椒粉的余味。他的手心全是汗。汗从掌心渗出来，沾在她手背上。她感觉到了，但没说话。

从月湖路口走到淮海路，大概一公里路。他们走得很慢。比平时散步的速度还要慢一倍。经过一家便利店。便利店的招牌灯是绿色的，照在地上——地砖变成了一半绿一半黄。玻璃门上贴着促销告示。可乐买一送一。

"等一下。"他松开手。走进便利店。出来的时候手里拿了两瓶可乐。瓶身冰凉，外面结了一层水珠。

他把一瓶递给她。

"谢谢。"

她又把手伸过去。不是接可乐——是把手放回他手心里。他愣了一下，然后握住。可乐在另一只手上冰着，手上的温度一冷一热。

继续往前走。路边有家花店。门口摆着成桶的红玫瑰。五月二十号——花店的生意很好。老板娘正往桶里加水，水管上的水花溅到玫瑰花瓣上。花瓣上挂着水珠。一个男生在挑花——拿起来一支，看了看，又放下去。拿起另一支。

他们从花店门口走过。他看了一眼那些玫瑰。

"你要吗。"他问。

她看了一眼花桶里的花。"不用。"

"真的不用。"

"嗯。花会谢。"

他想了想。"那你想要什么。"

"这个。"她举了举握在一起的手。"这样就行。"

他没说话。握得更紧了一点。力道变了——她的手骨感很清晰。第二指节能摸到骨头。他心里在想——太瘦了。但没有说出来。

路过一家奶茶店。门口排了五六个人。都是情侣。店员在窗口里喊——珍珠没有了，要等十五分钟。队伍里有个人说——等。旁边的人说——不等，换椰果。

"想喝吗。"他问。

"不想排队。"

"那我们去前面看看。"

前面有一家老式冰室。开了十几年了。门口的红招牌掉了一半颜色。卖红豆冰、绿豆沙。老板是个六十多岁的老头，坐在门口的竹椅上扇扇子。扇子是蒲葵叶做的，扇面上印着"楚州制药"四个字。

"红豆冰。"她指着菜单。

"两杯。"

老头放下扇子，站起来。冰柜里拿出冰沙，浇两勺红豆。红豆颗颗饱满，甜味在晚风里飘。他把两杯都递给她端。她手里端着冰沙——只剩右手空着。他伸手握住。

老头接过钱的时候看了他们一眼。又看了一眼。然后继续扇扇子。扇面上"楚州制药"四个字跟着扇子一上一下。扇子在拍——啪啪啪。

他们端着冰沙，右手握左手，慢慢往前走。走到淮海路的时候已经快八点了。街上的人比白天多。来来往往。有人看了他们两眼——大多是先看到冰沙，再看牵在一起的手，然后目光移开。

"你在看什么。"她问。他发现他一直在看前面一对老夫妻。那对夫妻大概七十多岁，走得也很慢。老头走在外面，靠近车流的那一侧。老太太走里面。两个人没牵手。但老头的左手一直虚扶着老太太的右肘。不碰。只隔大概两厘米。遇到坑的时候就往前伸一点。

"看那个。"他说。

她顺着他的目光看过去。看到了那对老夫妻。看了大概十秒。

他们同时转回头。什么都没说。

走到一个十字路口。红灯。等的人不少。她停下来——左手端着冰沙。右手在他手里。他的汗已经没那么多了——大概吹了半小时的风，手心里凉快了很多。

红灯变绿灯。人群往前拥。有个人从后面挤过来——差点撞到她。他往右拉了她一把。她往他身上靠了一瞬——肩膀碰肩膀——然后站稳了。

"挤吗。"

"还好。"

他换到她左边。把她换到里面——靠人行道里面那一侧。他的右手握她的左手。这样有车来的时候他在外边。

她注意到了。看了一眼他被换到外面的肩膀。没说话。

过了十字路口。街的右边是楚州的护城河。河水在夜色里是暗的。看不到底。河边的栏杆上靠着几个人——有的在打电话，有的在抽烟。烟头的红光一闪一闪。烧完了，掉进水里——滋一声。

"你听过楚州护城河的传说吗。"他问。

"没有。"

"说是明朝。在这条河边有个书生和一个姑娘约好了私奔。约好的那天晚上下大雨。河水涨了。书生过不了河。姑娘在河对岸等了整夜。第二天被人发现淹死在对岸的渡口。"

"因为什么。"

"她以为书生没来。其实是河涨了。书生在对岸——也等了整夜。两个人隔了一条河。谁都看不见谁。"

她沉默了。过了好一会儿。

"后来呢。"

"后来书生每年这个时候来河边坐着。什么都不做。就坐着。"

"这故事是真的吗。"

"不知道。小时候听运河边上的老人讲的。讲完就说——不要想在背后偷偷做事。你要让那个人知道。"

她转过头来看他。他的手还在她手里。汗又出来了——滚热的，潮潮的。这次不是紧张。

"你这几年做的事。"她说。顿了顿。"我现在都知道了。"

"嗯。"

"不用再瞒了。"

"嗯。"

护城河对岸有人在唱歌。不是街头艺人——就是一个人在哼。声音很低，听不清是什么歌。好像是《成都》。又好像不是。

走到街的尽头。该分开了。她的公交车站在东边，他的目的地在西边。两个方向。

"我送你上公交。"

"好。"

走到公交站牌下。末班的九路车还有七分钟。站牌上贴了一张撕了一半的广告——"招聘"。另一半广告已经不见了。

"周一早上。"他说。

"你不用每天来。"

"我想来。"

她看了他一眼。没再说什么。

公交车来了。车门打开。她松开他的手。走进车里。刷卡。滴一声。走到靠窗的位置坐下。

车窗开着。她把手肘撑在车窗上。

"周一见。"

"好。"

车开走了。尾灯越来越小。他的手还保持着握的姿势。空空的。风从手心穿过去——比刚才凉了一点。

他把手插进口袋里。往西边走了大概三步。手机响了——微信。

苏晚晴：你的手太烫了。

他看着屏幕。回了一条。

顾临风：你手凉。

苏晚晴：我知道。你可以给我暖手。

他站在护城河边。对着手机屏幕。手指停在键盘上——打了四个字，删了三个。又打了三个字，删了两个。最后只发了一句话。

顾临风：好。周一见。

然后他关掉屏幕。把手机放进口袋。护城河的水声在夜色里响着。河对岸的烟头亮了一下——熄了。柳条在河面上扫来扫去。

他在河边站了一会儿。手心里还有她指尖的凉意。没有汗了。凉凉的，干干的。像月湖的水面——被月亮照过之后，一点波纹都没有了。
