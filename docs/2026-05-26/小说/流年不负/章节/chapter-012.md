# 第012章 食堂

时间：2010年12月下旬

地点：青阳县一中食堂

卷次：卷一：初见秋风

食堂星期四的菜最差。

冬瓜炖得稀烂，汤多菜少。红烧肉的肥肉比瘦肉多，白花花的。米饭硬了，像是隔夜的。打饭的大叔勺子抖了好几下——每抖一次，菜就少一点。

顾临风和赵一鸣坐在靠近垃圾桶的角落。赵一鸣说这个位置好——打完饭不用端着找座，走过来就是。其实真正的原因是:这个位置没什么人坐，清静。

赵一鸣扒了两口饭。把筷子往盘子上一搁。

"我跟你说个事。"

"嗯。"

"你别生气。"

"说。"

赵一鸣往左右看了看。旁边没人——垃圾桶附近确实没竞争者。然后他往前凑了一点。

"你是不是喜欢苏晚晴。"

顾临风的筷子停了一下——很短的一下。然后继续夹菜。夹了一块冬瓜。放到嘴里。嚼了。

"不是。"

"真的？"

顾临风的筷子拨了一下盘里的冬瓜。冬瓜在盘子里转了小半圈。然后他说:"真的。"

赵一鸣歪了歪头。没说话。继续吃饭。吃着吃着他用筷子敲了一下顾临风的盘子边。"你知道猪是怎么死的吗。"

顾临风没搭话。

"跟你一样。闷死的。"赵一鸣把一块肥肉从嘴里夹出来扔回盘子里。"这食堂的肥肉越炖越白了。"

食堂里闹哄哄的。有人在对面喊——"等等我，带个馒头——"然后有人跑过去。碗筷碰撞的声音、吸面条的声音。有个男生端着一碗热汤挤过旁边，汤洒了一点在地上。

顾临风低头吃饭。米饭很硬。他喝了一口水。水是带的自来水——早上起来灌的，现在还是凉的，杯壁上凝了一层水珠。

赵一鸣把饭扒完了，抹了抹嘴。然后他扭头看着顾临风。顾临风还在吃。吃得很慢。筷子扒了一下，放到嘴里，嚼了，再扒。盘子里的饭只少了三分之一。

"顾临风。"

"嗯。"

"你耳朵又红了。"

顾临风伸手碰了一下耳朵。耳朵是烫的。他把手放下来。没说话。

赵一鸣笑了一下——不是嘲笑的那种。是知道了什么的笑，很轻。他把筷子搁在空盘子上。

"你这耳朵比嘴诚实。"

顾临风把最后一口饭吃了。然后把盘子端起来。站起来。走向垃圾桶。他把剩的菜汤倒进垃圾桶里。盘子放在回收台上。铁盘子磕在铁架子上面，当的一声。

赵一鸣跟在后面。也放了盘子。

回来路上。两个人并排走。赵一鸣走在右边，他的话匣子关了。走了大概五十米，他才开了一下。

"放心。我不会跟别人说的。"

走了三步。又说:"包括她。"

然后又走了几步。

"不过你那耳朵，是个有眼睛的人都看得出来。"

顾临风没说话。

走到教学楼底下。风很大。赵一鸣把校服拉链拉到下巴。脖子缩进领子里。突然又说。

"我觉得她还行。"

"什么还行。"

"苏晚晴。人不张扬。话不多。成绩好。"

顾临风沉默了一段。然后说:"你又不认识她。"

"这需要认识？看就能看出来。"

顾临风没再说话。

进了教学楼。楼梯上的防滑条已经磨平了。每一级上面都有一道浅浅的凹痕。他一步步往上走。到三楼。进了教室。

教室后面黑板报还在。白色的粉笔字没有擦。她写的那几段，在日光灯下面颜色更白了。

顾临风走回座位。赵一鸣坐到他旁边，翻开数学书。翻了两页，偏过头看了一眼顾临风的脸。"你脸也红了。"

顾临风把他脑袋推回去。"看书。"

赵一鸣笑了一声。然后真的看书了。翻了一页。又翻了一页。然后他嘴皮子动了一下——"第三题你会不会。"——他小声说话的本事还是一如既往的好。

顾临风把习题册拉过去。看第三题。二次函数顶点式。他列了一个方程。写了一半卡住了。然后往回推了两步。把第一行改了。

改完之后他把习题册推回去。赵一鸣看了一眼。

"怎么就写一半。"

"剩下你自己算。"

"你这人——"

顾临风低下头继续看自己的书。英语书翻到的那页是单词表——January、February、March——十二个月份。他的手指指在"February"旁边。二月。

"二月"就二月份。寒假。过年。明年。开学。

开学。

他在"February"旁边画了一个小圆圈。然后把书翻过去了。
