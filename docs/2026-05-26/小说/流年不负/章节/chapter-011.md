# 第011章 黑板报

时间：2010年12月中旬

地点：高一三班教室

卷次：卷一：初见秋风

十二月。教室后面的黑板报轮到苏晚晴出。

黑板报是一个苦差事。别人放学走了，你还在教室后面站着——站久了腰还酸——写完字退一步看——再上前——来来回回多了脚底的韧带都疼。粉笔灰落在衣服上。落在袖子上。袖子在黑板前面写一行字沾一层粉。深蓝袖子染上一层薄白霜。手指指甲缝里的粉灰也洗不掉。用指甲刮——刮下来是蓝色的粉迹。黑板反光。反光的时候站在正中间看自己的字——全是反的——看不清到底写歪了还是没歪。得退到后门那边。跟黑板成四十五度。才能看出字是否直。等发现已经写了大半板——擦也不是，继续也不是。

这期主题是"新年畅想"。班主任定的——王老师来教室后面指了一下——"这期的主题——新年畅想。再过一个星期就是元旦了——"给了几个关键词:学习、理想、青春、未来。用黄色粉笔写在黑板右上角，做个提示。"周三之前弄好——美术老师周三早上来检查——"然后就走了。粉笔头丢进槽里。

出黑板报的人一共三个。苏晚晴。还有一个矮个子女生——王曼，画花边特别快。还有一个负责标题的——李茵，平时收数学作业的课代表——字写得好。三人分工。苏晚晴负责正文。

周一到周三，每天放学后一小时。

头两天是画花边和排版。王曼拿尺子和粉笔在黑板上打格子。长尺是木制的一米长——尺子边有些缺损——画线的时候用尺子的边缘比着。横线敲一次——竖线敲一次。格子打得整整齐齐。花边是波浪形的——用绿粉笔和黄粉笔在黑板四周画双层花边。王曼手里捏两支粉笔——一支绿一支黄。一起画——手稳得很。她画画的时候嘴里还哼歌——哼的是《七里香》。哼一阶走了调——自己笑了一声——继续画。

李茵在黑板顶上写标题字——大红粉笔。写了"新年"两个字——第三个字写歪了——"畅"的右半部分向右侧偏了一点。她擦掉。抹布湿了之后把黑板擦成了深灰色——然后在湿的地方等水干——水干了重新写。"年"字右下角的折笔——蘸了一下粉笔的前面——侧着笔锋——露出一个锋。四个字写完——"新年畅想"——每个差不多有巴掌大。楷体。标题下面画了一条细线——分隔栏。

苏晚晴负责正文。正文是周一先在本子上拟好的。草稿纸——白色的横格纸。用圆珠笔分三节写的。第一节写了十几行。第二节十九行。第三节最短——七行。草稿纸上改了三处——一个错字用涂改带改的——涂改带在纸面上留了白色的胶条。

周三下午。放学铃响了。下午四点半——冬天的天已经开始有些灰。教室里的人一个接一个地走了。椅子推进桌底的声音一片——金属和水泥地摩擦——吱——咣当。前排收书包——后排收凳子——有人把扫帚碰倒在地上。有人在门口喊了一声:"晚饭吃啥——""食堂。还能吃啥。"——"食堂今天周三是炖土豆——不好吃。"——"算了算了——走——"声音远了。走廊里的脚步声渐渐稀了。

教室里还剩下三个人。黑板前——王曼站在黑板右侧。粉笔夹在耳后——手掌托着颜料盒——正在收。李茵蹲在讲台下面——把抹布拧干。苏晚晴站在黑板左侧——手里拿着白色粉笔——脸朝向黑板——在写正文的第二段——从中间往左移。

顾临风没走。

他的座位在后门旁边——第六排第三列，靠后门。从这里能看到黑板的全景。他留得很自然——桌上摊着数学作业本——做了三道题——横格作业本。他写了一半——笔放在本子上——然后转圈。笔在食指和中指之间绕。不掉。一直转。转了二十多秒——然后掉了——笔在桌面上滚了两圈——滚到数学书的旁边。他没捡。手放在下巴上撑着。

赵一鸣前两天说过他转笔很厉害——"转一百圈不带掉的。"当时是在课间——赵一鸣自己也在转——转了四圈就掉了。然后他就看顾临风转——"你这个怎么练的——"顾临风没理他。是后来发现的。不是故意练的。就是手上没劲的时候转了转。

黑板前面。苏晚晴的手在写字。白色粉笔。粉笔在滑面黑板上的声音——沙沙沙。一行往下——行距是刚刚她用铅笔打好的——淡铅笔线——写完会擦掉。她的字是楷体。每个字的大小差不多——同一行里面统一高度。写长句的时候匀速——先写左边的字——往右推——写到中线的时候往右移一步。那个绿色的波浪花边——在下一层分界处。她写到了花边上方两行。

写到某个字的时候，她停了一下。应该是写错了——笔已经成词了——最后一个字不是原先造句里那个。她把那个字擦掉——先用手指心抹了一下——白色粉笔的印色在手指上擦出一道白灰色。再拿湿抹布蘸一点水在那个字上滴一下——等待——黑板上的水迹慢慢干——水迹颜色从深灰变回浅灰——痕迹没了。重新写——这次对了。

她握粉笔的姿势和握圆珠笔一样。拇指和食指夹着粉笔——小指翘起来的。翘得不高——离开手心的那层皮肤大概一毫米。自己大概不知道。

白色的粉笔灰落在她的袖子上。袖子是深蓝色——那一层白粉——有些颗粒粘在布纹上。她的袖口本来就有道白边——洗太多次了。她没管。接着写。写完一段。后退一步。侧头看效果——黑板中间的排版——文字数——高度——跟前边的标题距离。看了一眼。然后往前走。捡起地上的草稿——翻了一页。继续。

另外两个女生在小声说话。王曼把颜料盒盖好——"明天早上你想吃什么。""食堂没什么好买的——超市那边的豆浆还行——就是烫手。""烫就烫呗——反正天冷——"两个人聊着聊着——王曼干咳了一声。苏晚晴没参与。她一直在写。草稿上有的地方——纸的边缘被撕得不是很齐——她用手压住纸角。手指按在纸面上往外捋了一下——从中间往侧边——捋平。

顾临风坐在后面。隔了两排的空凳子和空桌子。他看着那些字在黑板上一个一个地多起来。白色的。横平竖直——每一横的起笔都有一个小小的落笔——每一竖收尾的时候都会有把粉笔往上一提的习惯——就是很小的一个动作——外人看不出来。那些字排在一起——好看。连在一起了。

"王曼——你那支黄粉笔还在吗——我这边的黄色断没了——"李茵冲着旁边。

"在——在颜料盒底下——"

"哪——拿到——"

"你往前走一步——就在你右手那一格——"

两个人伸出手来递粉笔。苏晚晴没有加入。她朝左边走了一步——换个位置——写了新的一行。写了三个字——再往后退。把整句的语调排满。

"苏晚晴——你走不走——"王曼回头朝苏晚晴这边喊了一声。手里已经拿着书包的肩带——把空书包背上了。

"写完这段。"

王曼点点头。然后跟李茵一起走了。她们出门的时候顺手带门——门转回来——没关严——又弹开了一条缝。那条缝里有灯光和走廊的人声透过来——后来关门声隔壁也关了——安静下来了。

教室里只剩两个人。

教室里日光灯是开着的。风扇没转。吊着的扇叶上有层白灰——是粉笔飞上来的余灰。四十二张空桌椅全都是暗蓝色和原木色。地板上散了些纸片和粉笔断头——很小一点点。前面的挂钟——挂钟秒一格一格缓缓走着——那个声音。咔。咔。咔。

苏晚晴还在写。她写到了最后一段——第三节——草稿最短的那一章——七行正文。一字一字写在黑板上——到第四行的时候——粉笔掰断了一截——粉笔在空中断掉——"嘎巴"一声——掉在地上——碎成两截——脆响——在安静教室里特别明显。她弯腰捡起来——一小截从地砖的缝里拿出来——断口是整齐的正六角形——沾了点地板灰。把断的那截粉笔丢进粉笔盒里——盒子里的粉笔已经用剩大半了——白色的最多。然后从盒子里面挑了一下——摸到一根新粉笔——拔出来——继续写。

最后一个字写完。是文章的收尾——写了个"好"字。最后那笔——提钩——往左上方收的。退后一步。看了看整面黑板——正文三节。白色粉笔。字的大小、间距、高度——不错。左右也没有偏。她看了一会儿——手里的粉笔头放到槽子里——手指在指节间蹭了蹭——白灰顺着关节掉落。

然后走到桌子边——拿抹布擦手。抹布是湿的——在手腕上按了按——把粘在皮肤上的粉笔灰擦成湿的灰泥。然后手在裤子上抹——干了。把粉笔盒的塑料盖盖上。底稿——草稿纸——两面写过——从地上捡起来——折了两折——放进书包外层的侧袋里。书包拉链拉开——拉开的时候陶瓷小鱼轻晃一下——白色的小点儿在暗灯下也看得很清楚。拉链合上了。书包背带一股脑往右肩扛上。

她背起书包。安静地从过道往前走——从前门出去了。

门这次关严了。锁簧咔嗒——门和门框的缝隙消失。光线切断。

顾临风还坐在座位上。刚才她经过的时候——一路走过他的旁边——隔了一条过道和一个桌子的侧面——不到半米。她走过去的时候书包擦过前排椅背——陶瓷鱼碰在铁管上——叮——极轻。人声没了。现在的教室只有日光灯管的电流声。嗡嗡嗡。那种很低频的耳朵快听习惯了之后忽略的声音。粉笔灰还在光柱里飘着——在日光灯管的映照下慢悠悠地——在空中浮着——慢慢落地。有些已经落在地板上；有些还浮着——像十分细的碎雪。

他面前摊开的那道数学题——只写了三步。第一步解设。第二步画图。第三步列式。第二步写错了——向量方向折反了——他取过笔——在那道错误的式子上面画了一道横线。长横线——从式子的左边一直划到右边。然后在底下重写一行。第三步还是没写。笔停在空中——手指夹着圆珠笔——未写。

他低下头。看着那道题。又抬起头。黑板上的字——白色字的边缘在日光灯下有点反光——看不全文章——但能看到字的大致形状。是工整的。笔直地朝左看——她的袖口应该沾了不少粉。刚才她擦手时就注意到了——深蓝色袖子上一块白灰。

不知道她回宿舍洗掉了没有。水房冷水的话——冬天洗衣服手指碰到冰水——会搓不干净。她回去的时候可能食堂已经没饭了——刚才那两个女生没等她大概去食堂买包子了。那两个一直排队——晚了包子就没了。

他合上了作业本。站起来——椅子推进桌底——木椅腿在地上滑出轻微的响声。走到后门。手搭在把手上一把——铁把手在冬天捏着是冰的。回头看了一眼黑板报——白色的正文字。红色的大标题。绿色的波浪花边。花边上有个地方粉没填好——淡淡的空了一小点。正文最后一段"好"字旁边有半截断粉笔——静静地躺在粉笔槽的黑槽里。深蓝色袖口。日光灯还没关——走廊里其他教室灯已经黑了——这会再亮着就只剩这一盏了。

拉开门——手在开关上停了一下——按下——灯暗下来了——教室里变成了傍晚的颜色——窗外还有些淡淡的灰光——课桌椅的剪影整齐地排着——最前面第二排靠窗那张桌子——这会儿空了——比别的桌子略略地更暗了一些。

关上门——锁舌陷进锁孔——咔嗒。在走廊的空荡里听得很清楚。

他走到楼梯口——然后往回走——脚步。三十九级——一级一级踩——心里没数——直接走——不想东西。

一楼传达室的老头在听着收音机——正在播戏曲——京胡声悠悠地传到楼外的空地里。
