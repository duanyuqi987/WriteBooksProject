# 第127章 睡不着

时间：2018年3月

地点：楚州，顾临风公寓

卷次：卷七：流年未断

相亲前一晚。

他躺在床上。被子拉到胸口。枕头是荞麦皮的，枕了两年，中间凹下去一块。他的后脑勺刚好卡在那个凹槽里。

窗帘拉上了。但窗帘布的遮光性不好，对面的路灯光透进来，在天花板上铺了一层暗橘色。

他睁着眼睛。

天花板上的裂缝还是那条。从灯座往墙角延伸。在暗橘色的光里像一条很细的河。

看过几百次了。大一开学报到那天看了一次——陌生的城市、陌生的宿舍、陌生的天花板。考研成绩出来那天又看了一次——分数线过了，他的手在发抖。代课第一天上班前又看了一次——穿好衬衫打好领带去讲台之前，抬头看了一眼。

现在他又在看。

像一个高考后等成绩的高中生。六月九号晚——高考结束，答案对完了，分数大概估出来了，但还没出。躺在床上。不知道明天会来什么，但明天一定会来。那种感觉。胸膛里有什么东西在轻轻地跳——不快，但停不下来。

像一个大一新生报到前一晚。九月一号的晚上——青阳老家的床上。第二天要去楚州。一个从来没去过的地方。一整个新生活。被子还是蓝格子的。枕套上有他妈补的那块补丁。那时候也睡不着。

像一个——暗恋者。

他翻了个身。面朝墙。墙上的涂料还在起皮。他用手指摸了摸那个凸起的边缘——粗糙的，有一点冰凉。

高一那年他也是这样。每天晚上躺在床上，脑子里是她座位后面的阳光，是她手上的铅笔，是她走过走廊时马尾的晃动。那些画面在脑子里循环播放，从九月播到来年六月，再从来年九月播到来年六月。

那时候他是坐在后排、不敢看她的男生。现在是站在讲台上的顾老师。但他躺在这个房间里时，又变回了十六岁。

六年。从青阳到楚州。从一中考到师大。从学生变成老师。从少年变成男人。他的身份变了很多次。但在这个夜晚——黑暗里，他只是一个即将见到苏晚晴的人。

明天她会坐在窗边。他会推开门。他们会对视。已经想了八年的画面。

他以为第一次对视会是在高中教室里。他在走廊上悄悄等她经过。等了三次——每次她走过他面前的时候他都假装在看窗外。第四次她抬头看了他一眼——很快，像无意中扫到的。他假装没有发现。

那时候他就想——如果有一天，不是假装没发现。是真的看着她。她也看着他。目光不躲不闪。

明天就是那一天。

他翻了个身。床垫又嘎吱响了一声。

凌晨一点。

他起来上厕所。没开灯。脚趾踢到了床脚。很疼。他弯腰揉了一会儿。

回到床上。闭上眼睛。

还是睡不着。

凌晨两点。他把手机按亮。屏幕的光刺得眼睛疼。他眯着眼看了看时间。离约好的三点还有十三个小时。

他打开相册。点进那个叫"一"的文件夹。她的照片亮起来。

校门口。阳光。银杏叶黄了一半。马尾。浅蓝色衬衫。平底鞋。笑容比以前从容。

他看着照片，好像在问——你还记得我吗。

凌晨三点。他把手机放在枕头底下。屏幕朝下。压住了。好像压在枕头底下就不会再看。不会再打开相册。不会再对着那张照片发呆。

闭上眼睛。

外面的夜很静。偶尔有一辆车开过——轮胎压过路面的声音，由远及近，又由近及远。车灯的光从窗帘缝隙扫进来——在天花板上从左往右划了一道。然后消失了。过了一会儿又有一辆——这次是从右往左。车灯是白的。不是黄的。

凌晨四点。他大概睡着了一会儿。但梦里也是茶餐厅——推开门，风铃响了。窗边坐着一个人。马尾。浅蓝色上衣。脸看不清——像隔了一层磨砂玻璃。他往前走。但脚下的地板在往后移。桌子越来越远。她的背影越来越远。他想开口说话——嘴唇动了。但喉咙里发不出声音。像有什么东西卡在声带那里。一团棉花。或者一整个高中的沉默。

醒了。眼睛睁开。满额头的汗。枕头湿了一小块——荞麦皮枕头上印了一个深色的印子。心跳有点快——不是太快。但跳得重。能感觉到心脏在胸腔里撞——嗵。嗵。嗵。窗帘的缝隙里天色是灰的——三月的凌晨，天亮得很慢。太阳还没出来。路灯还没灭。

他翻了个身。被子裹得很紧。像蚕茧。他松了松。凉风灌进来。又裹紧了。

五点。六点。天亮了。窗外的光线从深灰色变成了灰白色。然后又变成了浅金色。太阳出来了。三月早晨的太阳——不刺眼。温温的。像被一层薄纱滤过。

他起来。走到窗前。拉开窗帘。樱花树还在。树上的叶子比昨天又多了几片。花瓣只剩最顶上那几朵了，在早晨的光里微微发白。有一只麻雀在树枝上站着——叽叽叫了两声。然后飞到对面楼的窗台上去了。

楼下有清洁工在扫地。扫帚划过地面的声音——沙。沙。沙。扫得很慢。从这边扫到那边。然后铲起来倒进垃圾桶。

他在窗边站了一会儿。手指在窗台上轻轻敲着——嗒。嗒。嗒。没有节奏。就是随便敲。

然后去洗脸。水龙头的水很凉。他用凉水拍脸，拍了三四下。抬头看镜子——眼睛有点红，但不是太明显。眼白里有一根红血丝。从左边眼角往外延伸。他凑近镜子看了看。然后退回来。不管了。

他用毛巾擦干脸。毛巾的边缘摩擦过颧骨——有点糙。该换了。但一直没换。

走到床边。床头放着那件浅灰衬衫——叠得整整齐齐。他对它点了点头。没有出声。然后深呼吸了一下——把肺里的气慢慢吐出来。吐了很久。胸口的紧绷感松了一点点。

他看了衬衫一眼。

穿上了。第一颗扣子扣了两遍——手指还是有点僵。但他不急。解开了。重新扣。这次扣上了。

穿好了。领口磨毛的地方还是看得见——一圈细细的白绒。但不算太明显。他把袖口卷了卷——刚好遮住缺扣子的位置。然后走到鞋柜前。把那双黑皮鞋拿起来。鞋头上的划痕还在。他用手指抹了抹——抹不掉。算了。穿上了鞋。系鞋带的时候手指比平时灵活了——大概是身体醒了。

站到门口的小镜子前。从头到脚看了一眼。浅灰衬衫。深灰夹克。黑色长裤。黑皮鞋。头发理过了。胡子刮了。看起来很整齐。像一个去赴约的人。像一个等了很久终于要赴约的人。他点了一下头——对着镜子里的自己。然后伸手去开门。
