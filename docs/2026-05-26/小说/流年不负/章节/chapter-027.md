# 第027章 图书馆

时间：2011年6月初

地点：校图书馆

卷次：卷二：窗边人影

六月初。周六下午。

青阳一中的图书馆在教学楼后面一排平房里。平房是红砖墙——砖缝间抹的白灰脱落了不少——露出来的缝隙里有蜂在筑巢——小黑蜂——比蜜蜂小——在五月飞进飞出。窗户朝南——铝合金窗——开一半——下午有阳光洒进来——从窗格中溜进——照在深褐色的木质阅览桌上——长方形光块——桌面上——还照出少许细木纹和被人划过的原子笔痕。馆里三间教室那么大——书架是铁架子——一排一排——靠墙密——中间矮——整体闻起来是老纸和阳光干燥的混合味——还有一种樟脑——放在书架上驱虫。

顾临风周六没回家——他爸说这周工地有活——让他自己待在学校——"工地上乱——你没地待——"电话里——他爸的声还是那两个字——"省着花——"他没多说——放下电话——听筒搁在铁座上。

上午洗了衣服——把作业写了一半——下午——去了图书馆。进门的时候管理员阿姨看了一眼他的学生证——"高一三班——"念了一声——"理三的啊——"他点头——她挥了挥手——放他进去。图书馆里人不多——周六住校的学生——不到二十个——分散在三排大阅览桌——各自看书。

他往里走——走到第三排书架的位置——然后停了。

苏晚晴在。她坐在靠窗的第三个位置——面前摊着一本很厚的物理参考书——《高中物理完全解读》——封面是蓝色——上面印了各个力学的箭头——红色——黄色。书打开——压在一角——破旧的透明书套包着——大概是图书馆里的——很久没有人借过——书页微黄。她低着头——右手拿笔——是那支蓝圆珠笔——小指翘起来的——还是那个习惯。左手按在书页左边——拇指放在行字上——辅助着眼睛——那些复杂公式旁的注释——一些小的便签——可能来自之前借书的人——黄——边卷。阳光从窗外照进来——照在她的右半张脸上——光线把脸上的小绒毛——镀成很淡的白色——额角的——颊边的——不细看不像有——眼睫毛在眼睑下投了一排很浅的阴影——眼睫毛不算长——密度刚好。她没有看到他。

顾临风往后退了两步——退到书架后面——后背靠在了架子上——铁架子微震了一下——手臂碰到书脊——一排书——词典、中国古典名著选读——他抽了一本——不知道是什么书——封面不是他看的——拿在手里——让手不那么空。然后他从书架后面绕到第三排阅览桌的另一端——最左边靠墙的位置——离她大概三张桌子——间隔三把空椅。他把书放在桌上——翻开——《中国古代史》——翻开——第三章——"秦汉时期的政治制度。"——印刷的文字——秦始皇——郡县制——三公九卿——编——分。他一个字都没看进去。

从阅览桌的间隙能看到她。她还在做物理题——那道题——看她手动作——应该在算向心力相关的——笔在草稿纸上画圆——逆时针——画了约三圈——然后停——画一条切线方向的箭头——停——回到圆——标上力的方向——tension——拉力。然后停下来——抬头——往窗外看了一眼——不是看什么——就是——脑子里在运算——眼睛——停留在窗外的银杏枝条上面——然后又低头——继续算。

她抬手把右耳边的碎头发拢到耳后——右手放笔——左手取代——理发——拢过——耳廓露出来——耳后——脖子——线条干净——然后右手重新拿起笔。这个动作她做了三四次——每次一样——笔放——手拢发——笔拿——动作重复——但每次做完头发都没完全拢好——是因为有碎发——被风吹散的——从马尾里溜出来的——又掉了——第四次拢完之后——不太再管。

管理员阿姨在门边打瞌睡——手撑在脸侧——眼睛闭着——每隔一会儿深吸一口气——然后恢复。中午的阳光——从深黄往淡——慢慢往西——落在桌子上——光块斜了一两厘米。

她做那道物理题。大概做了四十分钟——中间翻了两次书——换了一页——喝了一口水——杯子放在左侧——拿起——杯底碰在桌面——闷声——放下。然后站起来——走到书架那边——找一本别的参考书——手指在书脊上划——《物理疑难解析》——抽出来——翻了两页——看目录——不太满意——推回去——又找——《高考物理题型大全》——抽——翻中间——第十六章——看了片刻——夹在胳膊底下——带回座位。

她经过的时候——往顾临风这个方向看了一眼。他立刻低头——眼睛盯着《中国古代史》第七十三页——不眨——全身不动——只有眼珠盯在纸上——手指——按在书页的一角——指腹压着纸面——感受着纸张干纹。她走过去了——他没抬头。然后是椅子刮木地板的声音——她坐下——翻书——纸页的声音——很轻——"哗——"

他把头抬起来。书还在——"秦汉时期。"——七十三页。他翻了一页——七十四——是关于两汉的官制——不知道——没往心里。从余光里——那边——她继续翻那本《高考物理题型大全》——右手握着蓝圆珠笔——左手——在题旁边的空白上写数字——直式——计算——大概对了一道题——然后翻下一题。下午四点半——太阳已经从阅览桌彻底挪到书架底下了——光弱——黄色变灰。她开始收拾东西——把参考书放回书架——本《物理完全解读》先还——再放那本题型——把蓝笔放进笔袋——笔袋是蓝色的——抽绳——扎紧。草稿纸——上面写了大半页——对折——夹进自己的笔记本——放回书包。站起来——背上书包——小鱼挂件在书包拉链上——晃了两下——日光的最后一小块被那个白瓷反射了一下——细微的光点——又灭了。她走出图书馆——门推开时——门外一道更亮的光缝——然后门关上——光没了。

顾临风放下《中国古代史》。走到她刚才坐的座位前。桌面上——有一小块橡皮擦的碎屑——白色的——笔灰——还有一小滴干了的汗渍——可能是上午接水时滴的——淡淡的圆。他看了两秒——然后走回去——把《中国古代史》放回书架——书脊对齐——手指在书脊上按了一下——然后出馆。

出门时管理员阿姨叫住他——"那个同学——你借的书呢——""没借——""那你来干嘛——""看看书——"阿姨看了他一眼——目光在他脸上停了一两秒——然后说"哦"——没再问。他推开玻璃门——外面的阳光还是很亮——水泥地上有热气往上升——地上能看到光的热浪——空气往上走的——歪了地上的影子。

晚上——他坐在床上——翻出日记本——手电筒——光——橙——在牛皮纸上照出一个小黄圆。翻开新页——写："她在图书馆做物理题——做了将近两个小时——很认真——那道题——好像是求向心力。她发绳今天是淡绿色的——和上次那条蓝的不一样。"顿了一下——接着写——"秦汉时期的政治制度——一个字都没记住——秦——郡县——中央集权——只在第一页——后面都是她的笔在纸上的影子。"写完——合上本子——手电筒关掉。枕头下——日记——语文课本——黑色水笔——四样——叠在一起——不多——够一个书包。窗外——有人在打篮球——球弹在水泥地上——嘭——嘭——嘭——节奏不快——好像在等什么——然后接住——安静——两三秒——又弹——越来越远——听不见了。
