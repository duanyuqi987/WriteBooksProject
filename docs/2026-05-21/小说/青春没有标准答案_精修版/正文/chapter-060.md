# 第060章 雪化以后



时间：2009年一月中旬，雪停后的第三天



地点：楼前空地、水房和十班教室



卷次：第三卷：雪落在两栋楼之间



2009年一月中旬，雪停后的第三天，楼前空地、水房和十班教室，扫帚上结成硬块的脏雪先把这一天按进了十班的日常里。雪慢慢化掉，被点名的同学下楼扫雪，十班在沉默劳动里重新靠近。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

付小川想用玩笑把难堪糊过去，陈柏言却认真记下每个人负责的区域。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看扫帚上结成硬块的脏雪，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

陈柏言也在这一天被推到前面。陈柏言的特点是笔记整齐，日期天气都记，擅长把混乱事理出顺序。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“一组扫东边，二组扫水房门口。”陈柏言说。

“你连扫雪都能列计划。”付小川说。

“列了就少漏。”常俊凡说。

“这边还有一块冰。”段璟说。

“把路扫干净，比检讨写漂亮更有用。”陶建国说。

雪化得比下的时候慢。楼前空地上到处是灰白色的水，扫帚一扫就带起泥点。

陈柏言把负责区域写在本子上，连水房门口那一小块冰都单独标出来。

付小川说你以后当班长吧，陈柏言说我只是怕你们又漏。

常俊凡弯腰把一块硬雪推到墙边，手指冻红了，也没有把扫帚丢给别人。

段璟和梁一帆一起铲台阶，梁一帆铲得快，溅了段璟一裤脚泥水。两个人看着彼此，又忍不住笑。

扫到最后，楼前路面露出来，食堂师傅推着桶经过，说这才像学生做的事。

雪仗的热闹在这几小时里慢慢降温，留下的不是谁赢谁输，而是谁愿意把残局收完。

傍晚回教室时，常俊凡把湿袖口拧了一下，段璟没有说辛苦，只把窗户关严。

雪化以后没有悬在空处。扫帚毛上结住的脏雪，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，雪化以后带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

陶建国进门时不一定立刻说话，鞋底在水泥地上一停，雪化以后带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在陈柏言快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。雪化以后越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。扫帚上结成硬块的脏雪让他想说话，又逼他先低头看题。

陈柏言被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，雪化以后留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。陈柏言路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为雪化以后翻身，谁还在被窝里背单词，旁边人都听得见。

雪化以后，县城路边露出泥色，校园里那些喊声也慢慢沉到各人的作业本里。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理雪化以后里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：扫帚上结成硬块的脏雪、一只冻红的手、一句老师点名、一块还没扫干净的雪。

张若楠把扫帚往墙根一靠，蹲下去看自己刚才扫过的地方。雪化了以后，水泥地上露出一道一道的水痕，顺着地势往低处流，像有人在灰布上画了歪歪扭扭的线。

"你画的是地图吧？"付小川从旁边路过，指着她脚边那条弯弯曲曲的水迹说。

"扫帚又不是画笔。"

"那你蹲着看什么？"

"看有没有漏掉冰。"张若楠站起来，指给他看，"这块冰卡在砖缝里，扫帚够不着，得用脚踢。"

付小川蹲下去看了看，用脚尖把冰碴踢出来。冰碴飞进了旁边的水坑，溅起一小圈涟漪。

"你眼挺尖。"他说。

"我奶奶家在黄河大堤边上，冬天院子里全是这种砖缝冰。"张若楠把扫帚拿起来，"我从小就会踢。"

陈柏言站在楼前空地中间，手里拿着一个蓝皮笔记本。本子上画了一个粗糙的平面图，东区一组、水房门口二组、台阶三组，每个区域旁边写了负责人名字。名字后面空了一栏，是"完成情况"。

"你这跟排兵布阵似的。"付小川凑过去看。

"排了就不漏。"陈柏言把笔从本子缝里抽出来，"上次值日牌被砸的时候，就是因为没人管，谁都不知道该谁扶。"

"所以你现在什么都管？"

"我只管我能管的。"

周明澈从陈柏言手里把本子拿过去看了两眼："你这个图比例不对。东区比水房门口大了一倍，但两组人数一样。"

陈柏言愣了一下，又把本子拿回来："那你调。"

周明澈也不客气，拿起笔在图上画了两道线，把东区划成三块，多加了一个人。划完他看了一眼陈柏言："画线只是建议，你是负责人。"

"我同意。"陈柏言说，"你字写得比我好，把调完的人名也写上。"

段璟在旁边看着这两个人对着一个笔记本上的草图认真讨论，忽然觉得十班的"班长储备"不止常俊凡一个。陈柏言用计划兜底，周明澈用效率补齐，这两个人一前一后，像两扇门，一扇挡住混乱，一扇让事情走得更快。

沈曼青走过来，把陈柏言本子上的分配图看了一遍，问了一句很沈曼青的话："你画了东区、水房和台阶，可没有标老师办公室门口。"

陈柏言低头看本子上的图，愣了一下。确实没标。

"老师办公室门口不是咱们的卫生区。"他说。

"雪仗那天王启民从办公室出来，差点踩到雪泥滑倒，你忘了？"沈曼青说，"咱们扫雪不是只扫卫生区，是扫大家会走到的地方。"

陈柏言把笔给她："你说怎么加？"

沈曼青在本子右下角画了一个小方框，写了"办公室门口"四个字，然后在旁边加了一个人名："孙佳怡。"

"为什么是她？"

"她走路最小心，扫那个区域不会带泥水进办公室。"

段璟听着，想起沈曼青在雪仗那天也是第一个看窗户、楼梯和老师会从哪边过来的人。她和陈柏言的仔细不一样——陈柏言是列计划不漏，沈曼青是看风险不漏。两个人合在一起，刚好把"做对的事"和"别做错的事"都包住了。

扫雪进行到一半，食堂师傅推着一桶热水从旁边经过。桶底有小轮子，轮子在湿水泥地上发出咕噜咕噜的声音。师傅姓刘，五十多岁，棉袄外面套着白围裙，围裙边上沾了面粉。

"十班的？"他停下来，扫了一圈。

"十班的。"周明澈说。

"你们班今年挺能干的。"刘师傅把桶推到一边，从围裙口袋里掏出几块花生糖，放在花坛边上，"暖手不够，嚼两口甜的。"

付小川第一个伸手去拿。花生糖很硬，咬下去咔的一声，甜味从牙根慢慢渗出来。

"师傅，你这糖放了多久了？"付小川含糊地问。

"去年来这食堂之前就买了。"刘师傅咧嘴笑，"糖又不会过期。"

"这逻辑听着耳熟。"付小川嚼着糖，往段璟那边看了一眼，"感觉我们班有人也会说这种话。"

段璟没理他，但也拿了一块花生糖。

刘师傅推着热水桶继续走了。花坛边上剩下几张花生糖的玻璃纸，被风吹起来，粘在湿砖面上。沈曼青弯腰一张张捡了，团成小团塞进口袋。

常俊凡把搪瓷缸里最后一点热水浇完，冰面终于完全松开了。她用扫帚把碎冰推到墙边一起归拢，站起来时膝盖上多了一道灰印。段璟看见了，从口袋里掏出那块灰手帕递过去。

"擦一下膝盖。"

常俊凡低头看了看膝盖上的灰，又看了看手帕，犹豫了半秒才接过来。她擦得很轻，像在擦书页上的一小块水印。擦完她把灰手帕叠好递回来——叠得比他给她时还要整齐。

"谢谢。"她说。

"不用谢。"

两个人同时安静了一拍。段璟想说"花生糖你吃了吗"，说出口却变成了："你膝盖没湿吧？"

"没有。"

"那就好。"

旁边的付小川把整段对话都听见了，但他这次没有出声调侃。他只是把手里剩下半块花生糖塞进嘴里，咬得特别响，像给这段没有说出口的对话配了一小节只有他能听见的伴奏。

水房门口的冰最难扫。冰被踩实了，贴着地砖，扫帚推上去纹丝不动。常俊凡从锅炉房借了半壶热水，端着搪瓷缸慢慢浇在冰面上。热水碰到冰，滋的一声，冒起一缕白烟。冰面裂开一道细纹，然后慢慢松了。

"这招谁教的？"段璟问。

"我姥姥。"常俊凡把搪瓷缸放在地上，"以前在城关镇，冬天院子里水管冻住了，她就用热水浇。"

"你姥姥是封丘人？"

"嗯。她说封丘的冬天跟别处不一样——冷是硬冷，风是干风，但中午出了太阳以后，温度会从土里往回返。"常俊凡看着冰面慢慢软下去，"她说人也是这样，不能一直冷着。"

段璟觉得这句话不像是在说水管和冰。但他没有追问，只是蹲下去帮她把化开的冰铲到一边。

雪化以后，楼前空地上的声音变了。雪仗那天的喊声和笑声已经散干净了，取而代之的是扫帚拖地的摩擦声、铁锹碰砖缝的叮当声、热水浇冰的滋滋声。这些声音不整齐，但每个声音都连着一个人，每个人都在做一件具体的事。

梁一帆和赵远航在台阶那头铲冰。赵远航负责撬，梁一帆负责搬。两个人配合了没几下就吵起来了——梁一帆嫌赵远航撬得太慢，赵远航嫌梁一帆搬得太远。

"你搬那么远干嘛？放在花坛边上不就行了？"

"花坛边已经堆满了。"

"那就放台阶底下。"

"台阶底下是路，你让人怎么走？"

两个人都停下来。最后梁一帆把冰块搬到墙根放下，赵远航看着他的路线，说了一个"行"字。

段璟远远地听见，心想，"行"这个东西在十班大概分两种：一种是嘴上的行，说完了该干嘛干嘛；另一种是梁一帆刚才那个行——搬完了冰，回头看着你，意思是"这样总行了吧"。

付小川负责的区域是教室门口那一小块。他扫得最慢，不是偷懒，而是每隔几分钟就有人从里面出来，踩一脚湿鞋印。他每踩一次就要重扫一次，终于在被踩了第四次以后，他站在门口把扫帚一横。

"能不能等人扫完了再进出？"

"不能。"宋乔端着饭缸从他旁边侧身过去，理直气壮地说，"食堂的土豆烧鸡块，去晚了就没有了。"

"那你踩脏了谁扫？"

"你扫。"宋乔已经走远了，"你不是正在扫吗？"

付小川把扫帚在地上拍了两下，又继续扫。他发现自己虽然嘴上不服气，但身体已经习惯了一种东西——十班没人会真的把一件事撂给别人。宋乔说"你扫"，但她下次路过的时候会踮着脚走，尽量少踩两个鞋印。

段璟在下午的课间去了水房接水。水房里的湿脚印比早上更密集了，暖水瓶沿墙排着，瓶身上满是水珠。他找到自己那个灰色暖水瓶时，发现瓶口多了一个红色塑料盖——他原来那个盖子前天碎了，还没找到替换的。

他把瓶塞拔出来，看见红盖内侧用圆珠笔写了三个小字：你的。

不是"段璟的"，是"你的"。

这两个字是常俊凡写的。段璟认得她的字——笔画很细，收笔的时候从来不带钩。她说"你的"，而不是"段璟的"，大概因为后一种写法太像班级财产登记，前一种写法更像人在说话。

他把红盖套回瓶口，接满热水，端着搪瓷缸往教室走。路上碰到沈曼青抱着一摞作文本从办公室出来，她看了一眼他手里的搪瓷缸。

"常俊凡也给你找了一个？"

"也？"

"我的杯盖前天也碎了，今天早上桌上多了个盖子——蓝的。"

段璟没有接话。他把搪瓷缸端平，热水在杯里轻轻晃了一下。他想，常俊凡的清醒不是只对她自己负责，而是对整间教室里碎掉的东西都有数。

傍晚时分，雪基本化完了。楼前空地上的水泥地面露了出来，颜色比没下雪之前深了一点。花坛里的冬青被压弯了好几根枝条，还没有弹回来。陈柏言在本子上把每个区域的"完成情况"一栏都打了勾，然后把本子合上。

"齐了。"他说。

"你那个本子借我看一下。"常俊凡说。

陈柏言递过去。她翻开看了两页，然后拿起笔在其中一行旁边加了一个注：下次扫雪，水房门口先浇热水再铲，省力。

"你加的这个，下次还不知道谁扫雪呢。"陈柏言说。

"不管谁扫，方法留着。"常俊凡把本子还给他。

段璟在旁边听见，觉得这就是常俊凡最让人放心的地方——她做的事情永远不只是为了今天，也为了下一次她不一定会参与的那一天。

晚自习时，陶建国在讲台上说了两句关于积雪清扫的话。他说得很快，像在念一个必须念完但不打算展开的通知。但最后一句话他放慢了速度："地扫干净了，你们自己心里踩着也舒服。"

他说完就走了，皮鞋在走廊上一下一下地敲远。

段璟低头翻开英语书。第三单元的单词他已经背熟了，"sincerely"不会拼错了。但他把草稿纸翻到背面，没有写单词，而是画了一个小的搪瓷缸，里面冒着三条波浪线代表热气。他画得不好，杯子口比底还窄，但他觉得这个样子挺好看的。

付小川从后排探过来："你画的是——茶缸？"

"嗯。"

"画它干嘛？"

"练练线。"段璟把草稿纸翻回去。

付小川没有追问。他看见了纸背透过来的那三条波浪线，也看见了段璟翻纸时手指微微避开了一小块地方——那一小块地方刚好在波浪线正下方，大概写着什么他没来得及看的东西。

熄灯以后，宿舍里有人小声说话。今天的话题不是谁扔了雪球、谁被点到了名字，而是明天会不会出太阳。查了天气预报的人说"多云转晴"，没查的人说明天看窗户就知道了——窗户玻璃上的霜如果退得快，就是晴天。

段璟闭上眼睛听着。他觉得雪化以后，十班确实变了一点。不是说每个人都突然好了，而是坏的地方被雪水洗掉了一些，剩下的部分——那些会出错、会怕、会嘴硬也会递扫帚的部分——反而看得很清楚。

雪化以后之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

陈柏言没有因此变成特别醒目的人，可在段璟眼里，陈柏言从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，雪仗从笑话变成班级记忆，段璟和常俊凡都看见了热闹背后的责任。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
