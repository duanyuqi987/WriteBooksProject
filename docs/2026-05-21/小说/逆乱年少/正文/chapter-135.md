# 第135章 山路

时间：2012年7月20日，中午

地点：核桃沟回李家沟的山路

卷次：第六卷：雪落无声

铁蛋没来上课的原因是奶奶摔了一跤。

老太太在院子里晾衣服，踩到一块青苔滑倒了，摔伤了腰。不算严重，但下不了床。铁蛋在家里照顾奶奶和弟弟，还要喂鸡、做饭、晒核桃。

苏安晴在铁蛋家坐了一个小时。她帮铁蛋奶奶换了腰上的膏药——她从学校多带了张膏药来，说是"正好书包里有"。然后她坐在铁蛋家院子里的石墩上，把这两天的课文从头给他讲了一遍。

铁蛋听得很认真。他坐在苏安晴对面的小凳子上，课本摊在膝盖上，嘴唇无声地跟着念。他奶奶在床上说："这娃，天天念叨苏老师。"

苏安晴走的时候，铁蛋送到村口。苏安晴回头说："明天来上课。"

"奶奶咋办？"

"我明天给你布置一篇作文，题目叫'我帮奶奶做家务'。你周四周五写了，下周一交。"

铁蛋愣了一下，然后咧嘴笑了。"好。"

回程的路上苏安晴的脚肿了。

她没说什么，但走路的速度比去的时候慢了不止一半。每一步落下去都先停一停，像在试水。

林玉起走在她左边。

"休息一下。"

"不用。再走一阵就到了。"

又走了五分钟。过一个弯的时候，苏安晴踩到了一块凸起的石头，身体晃了一下。

林玉起伸手扶住了她的肩膀。

这一次扶的力度比来的时候大。他的手从她的手臂上滑到肩膀的位置，掌心贴着她的肩胛骨，感觉到了薄薄一层汗和肩胛骨下面骨头的形状。

苏安晴没有挣开。

两个人就着这个姿势走过了一段最陡的坡。

"铁蛋的奶奶说，铁蛋的娘三年前走的。"苏安晴说。她的声音听起来很平静，但手抓林玉起小臂的力气比刚才大了。"丢下两个孩子走了。一个九岁，一个五岁。"

"去哪了？"

"不知道。铁蛋奶奶不肯说。就是说'走了'。"

山路上只有两个人的脚步声和远处的鸟叫。一只松鼠从路上穿过，在石头上停了一下，又窜进草丛里不见了。

"铁蛋从来没在我面前提过他妈妈。"苏安晴说。"他写的作文都是关于奶奶——奶奶做的饭、奶奶洗的衣服、奶奶说'好好念书'。"

"他在学校不哭吗？"

"不哭。他是我见过的最爱举手的小孩。"

苏安晴说完这句话之后沉默了大概半分钟。然后她停下来，不是因为脚疼——她站住了，看着远处的山。

"林玉起。"

"嗯。"

"我忽然觉得——我以前觉得自己挺不容易的。考大学、做兼职、想着以后留校。但跟铁蛋比——"

她没有把这句话说完。

但林玉起听懂了。

回到学校的时候天快黑了。陈维中看见苏安晴的脚，让她赶紧用凉水冲一下。赵一琳跑去找村长要了半瓶红花油。

苏安晴坐在教室门口的台阶上，把鞋脱了，脚腕肿得老高。

孟小荷拿了条毛巾蘸了凉水给她敷。沈佳慧在旁边说："你傻啊，三里地你也走。"

苏安晴笑了笑，没说话。

林玉起站在远处，没有围过去。他看见苏安晴坐在台阶上，在一群女生的包围圈里，低头看着自己肿起来的脚腕，脸上还带着从铁蛋家带回来的那种表情——不是难过，是一种很安静的、很久都不会消散的东西。

他回到教室，打开小本子，写了几句。

>
脚伤三里路，心疼九岁童。
师者非为教，原是共悲欢。
>
