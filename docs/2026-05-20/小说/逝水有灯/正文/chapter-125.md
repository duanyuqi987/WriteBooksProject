# 第125章 教室的灯多年后

所属部卷：第三部：寒风里的花
时间：多年后
地点：二号教学楼空教室

## 诗歌插页 P125：原创仿唐绝句

旧楼灯换照新尘，
粉字无踪桌色新。
当日空明非暗许，
一窗夜色还行人。

多年后陆允川回学校，二号教学楼翻修过。307 教室门牌换成了电子屏，黑板变成白板，走廊尽头的水房也改成了饮水机区。

他站在门口看了一会儿。里面有学生在讨论比赛，桌上摊着笔记本电脑和外卖杯。没人知道这里曾经有一块没擦干的水痕，也没人需要知道。

陆允川打开旧日记，找到那天写下的一句：“她没有等我。”他看了几秒，在旁边补了一行：她没有让我去。

这行字短得多，也准确得多。

他不再责怪当年的自己太笨。十八九岁的人常常把行动当成真心的证据，却很少先问对方需不需要那份行动。那不是罪，只是需要被看清。

离开二号楼时，走廊感应灯一盏盏亮起，又在他身后熄灭。灯很正常，照每一个经过的人，不替谁保存委屈，也不替谁证明深情。
