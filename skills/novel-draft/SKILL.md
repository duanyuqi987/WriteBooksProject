# 正文起草

写当前章节正文。采用三层写作模型，集成写作技法和记忆系统。

## HARD-GATE

**每一章都必须完整执行以下 10 步读取，不允许因为"之前读过"而跳过：**

1. 读取 `outline.md` → 当前章任务说明 + 精细场景拆分 + 推荐技法
2. 读取 `project.md` → 风格/世界观禁止事项/技法偏好/体验规格
3. 读取 `memory-snapshot.md` → 最近 5 章摘要（恢复上下文）
4. 读取 `skills/novel-writing/styles/[风格名].md` → 句式/对话/情绪规则
5. 读取 `skills/novel-writing/references/ai-flavor-banlist.md` → 禁用词表 + 6 类 AI 检测
6. 读取 `skills/novel-writing/references/writing-techniques.md` → 当前章推荐技法详解
7. 读取 `人物/` 所有角色卡 → 身份张力 + 声纹档案 + 当前状态
8. 读取 `伏笔/` 所有未兑现伏笔 → 当前需埋入/提示/兑现的伏笔
9. 读取前一章末尾 500 字 → 衔接点 + 避免重复结尾模式
10. 读取 `skills/novel-writing/references/reader-experience-spec.md` → 对照读者感受目标

**未完成以上 10 步之前，绝对不允许写任何正文。每读取一个文件输出 `[✓] 文件名: 一句话摘要`。**

## 三层写作模型

不一次性写出最终稿，而是分三层逐步打磨：

### 第一层：粗写骨架（快速铺陈）

- 按照精细规划的场景顺序，逐个场景快速写出事件序列
- 此时不追求文采，只追求"发生了什么"
- 每个场景写：谁 + 在哪里 + 做了什么 + 结果是什么
- 对话只写核心内容，不修饰
- 这一层达到目标字数的 60-70%
- **目标**：确保情节逻辑通顺、事件序列完整

### 第二层：细写填充（血肉附着）

- 在第一层骨架上，为每个场景添加：
  - 五感细节（看到什么、听到什么、闻到什么）
  - 环境氛围渲染
  - 对话的口语化打磨
  - 角色的身体语言和微表情
  - 过渡和节奏调整
- 执行生成期自检（每 300-500 字）
- 这一层达到目标字数的 100%

### 第三层：技法注入（精神点睛）

- 根据本章推荐技法，对关键段落做技法强化：
  - 标记为「不写之写」的段落：确认留白是否到位
  - 标记为「一击两鸣」的段落：确认过渡段是否含深层信息
  - 标记为「草蛇灰线」的段落：确认伏笔埋入是否自然
  - 标记为「白描」的段落：确认是否用具体动作代替了抽象形容
  - 标记为「繁章波折法」的段落：确认情绪转折是否够 3 次
  - 标记为「暗透法」的段落：确认他人说出的心里话是否微妙
  - 标记为「春秋笔法」的段落：确认评判是否含而不露
  - 标记为「反衬法」的段落：确认表面和内里是否有张力
  - 标记为「层峦迭翠」的段落：确认对照是否自然不刻意
  - 标记为「避繁文法」的段落：确认重复内容是否做了侧面交代
- 完成后执行技法自查

## 场景写作流程（每个场景）

```
1. 读取当前场景的规划（任务 + 推荐技法 + 预估字数）
2. 对照体验规格：这个场景应该让读者感受到什么？
3. 写第一层：快速铺陈事件序列（约场景目标字数的 60%）
4. 写第二层：添加细节、对话打磨、氛围渲染
   - 对话检查：对照角色声纹档案，确保说话方式正确
5. 执行生成期自检（禁用词/AI腔/风格禁止项）
6. 写第三层：注入推荐技法
7. 执行技法自查
8. 声纹检查：遮住说话人标注，能判断谁在说话吗？
9. 体验检查：读者这一刻应该感受到什么？达到了吗？
10. 写衔接句 → 下一个场景
```

## 生成期自检（每 300-500 字执行一次）

1. **禁用词扫描**：检查 ai-flavor-banlist.md 禁用词
2. **排版检测**：碎片化分行？省略号过多？连续单词行？
3. **AI 句式检测**：连续 3+ 相同句式？段落长度连续相近？
4. **风格禁止项**：违反当前风格禁止事项？
5. **字数节奏**：当前场景字数是否在预估范围内？

违反任一条 → 重写最近 300-500 字段落后再继续。

## 技法自查清单（第三层完成后逐项打勾）

- [ ] 白描是否出现在人物出场/群体场景/情绪传达处？
- [ ] 任何过渡段是否只用了一次信息（如果有，尝试一击两鸣）
- [ ] 本章是否埋入或提示了规划的伏笔？
- [ ] 高潮场景是否使用了繁章波折法（至少 3 次情绪转折）？
- [ ] 是否有可以改用暗透法的心理描写？
- [ ] 重复性内容是否做了避繁文法处理？
- [ ] 是否有可以改用不写之写的直接描写？
- [ ] 是否有相似的场景可以形成层峦迭翠（与之前章节对比）？
- [ ] 是否有需要春秋笔法处理的评判性内容？
- [ ] 是否使用了反衬法制造表面和内里的张力？
- [ ] 声纹检查：每个角色的对话符合其声纹档案吗？
- [ ] 体验检查：对照体验规格，读者的感受匹配目标吗？
- [ ] 第一性原理检查：这个场景的存在是因为"应该这样写"还是因为"它真正服务于核心价值"？

以上 10+3 项中，至少要有 2-3 项得到"是"的回答（根据本章推荐技法确定）。

## 章末检查

- 最后一段是抒情/总结/感慨/预告？→ 删掉，用动作或钩子替换
- 最后一段包含"开始了""结束了""转动了""新的"等词？→ 删掉
- 最后一段让读者想知道"然后呢"？→ 否则加钩子
- 连续 2 章不出现相同结尾模式
- 钩子类型轮换使用：行动钩 / 发现钩 / 选择钩 / 危机钩

## 核心原则

- **三层递进**：粗写→细写→技法，步步为营
- **记忆驱动**：每次先恢复上下文，再动笔
- **技法嵌入**：不是在写完后再想"应该用什么技法"，而是在规划阶段就已标注，写作时参照执行
- **写中自纠**：每 300-500 字自检一次
- **技法自查**：第三层完成后逐项打勾
- **宁多勿少**：字数不低于目标的 90%

## 反模式

| 错误 | 正确 |
|------|------|
| 跳过记忆加载直接写 | 每章必须执行 9 步读取 |
| 一稿到底不回头打磨 | 三层写作：粗→细→技法 |
| 不读技法库就写 | 每章根据推荐技法重新读对应技法详解 |
| 一次性写完整章不思考 | 逐场景写作，每场景后思考 |
| 章末用抒情/总结收尾 | 用动作/发现/选择/危机钩子 |
| 技法堆砌（一章用 5+ 种技法） | 每章聚焦 2-3 种核心技法 |
