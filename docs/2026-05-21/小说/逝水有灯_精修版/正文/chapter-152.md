# 第152章 南湖问答

所属部卷：第三部：寒风里的花
时间：六月末夜里
地点：南湖河边

## 诗歌插页 P152：原创仿唐绝句

湖灯微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

事情落到当天，没有掌声，也没有命运感。

陆允川到的时候，叶澄已经在了。她坐在那张掉漆的长椅上，腿翘着，正用手机拍湖面的灯。闪光灯闪了一下，她皱眉，关掉闪光又拍了一张。

"拍什么？"陆允川走过去。

"拍灯。"她把手机转给他看。屏幕上湖灯是糊的，前景的椅子扶手倒是很清楚。"手抖了。"

"要不要我帮你拍？"

"不要，糊的更好看。"叶澄收起手机，"坐。"

他坐下。湖水在黑暗里泛着细碎的亮，远处有人背书，声音忽高忽低，像收音机找不到频道。

叶澄拧开她带来的水，喝了一口，然后转过来看他。她不急着说话，这是她一贯的方式——先等对方把沉默用完。

"你说有事想聊。"陆允川开口。

"是。但你先说你的。"

"我的什么？"

"你约我出来，总不是因为闷得慌想散步。"

陆允川的手指碰到了扶手掉漆的那块木头。指腹摸过去有一点刺。他的整个大学时代好像都有这样一块掉漆的地方，别人看不见，他自己一直摸。

"我想问你，"他说，"如果一个人过了好几年还在想同一件事，是不是说明这件事永远过不去？"

"不是。"叶澄回答得很快，快到他还没来得及准备好被否定。

"那是什么？"

"说明你一直用同一种方式在想它。"叶澄把水瓶放到一边，"你有没有换过姿势？"

"什么姿势？"

"比如，不再问她为什么走，而是问自己为什么一直站在原地。"

湖边的英语听力又响了。那句"What matters most"一遍一遍重复，像某种没预约的背景音乐。陆允川以前觉得这种巧合很玄，现在只觉得厌烦。

"我是不是一直把喜欢弄得太重？"他说。

这句话落地之后，没有人接。

叶澄等他等了很久。她知道陆允川一旦被刺痛，就会先用沉默把自己包起来。所以她没催，只是陪他听了一段英语听力和一段虫声。

"你刚才问我有没有换过姿势，"他重新开口，"但我觉得我不是不想换。我是不知道该怎么换。"

"那我来帮你。"叶澄侧过身子，面对他，"不过话说在前面，我不负责给你标准答案。"

"嗯。"

"第一件事：苏晚枝不是辩题。"

"我没有把她当辩题。"

"你有。你一直在用她来证明自己是一个什么样的人——被伤害的人、执着的人、放不下的人。你从来没想过，她可能根本不想参与这个证明。"

陆允川的手指停住了。

"第二件事，"叶澄说，"真心不是交换券。不是你放了真心进去，对方就必须找零给你。边界不是冷漠，是别人对她的生活有决定权。"

湖边有人起身走了，英语听力也跟着走远了。那片安静忽然变得很沉。

"我是不是对很多人都这样？"陆允川的声音低了一些。

"对谁都这样。"叶澄说，"对苏晚枝是这样，对你妈也是这样。你总觉得只要你付出得够多，事情就会变成你想的样子。但很多事情不是这样的。"

"你怎么知道我妈的事？"

"程砚说的。你别怪他，他是替你着急。"

陆允川没吭声。他妈的事是另一条暗线，他平时不怎么提，也不让别人提。但叶澄提了，而且说得对。

"第三件事，"叶澄举起三根手指，像在数证据，"你不是忽然成熟。你只是在一次次难看之后，终于开始承认——你的强烈不能替别人做决定。"

这句话让他想起很多年前的一个下午。他忘了是哪一年，只记得他站在某个门口，说了很多话，门里没有回应。那时候他以为是自己说得不够多，后来才明白是说得太多了。

"我说完了。"叶澄把手放下，"轮到你了。"

"我……还没消化完。"

"那慢慢消化，我不急。"

"程砚是不是什么都跟你说了？"

"也不是什么都说了。主要说你最近又翻旧日记，翻到最后给自己翻进死胡同了。"

陆允川苦笑了一下。"他观察能力什么时候这么强了。"

"他一直挺强的，只是不爱说。跟你不一样，你是太爱说，说太多了。"

"那你觉得我应该怎么办？少说话？"

"不是少说话。"叶澄转回去看湖，"是想清楚了再说。你现在的问题是话说得很多，但没有一句是给自己说的。你总是在对别人解释，对别人表白，对别人道歉。你什么时候对自己说过一句实话？"

他没有马上回答。湖风把他脚边的塑料袋吹动了一下，又停住。

"我对自己说过——"他停了很久，"——说过我不甘心。"

"不甘心什么？"

"不甘心我努力了这么久，结果还是这样。"

"什么样的'这样'？"

"就是你努力了很久，以为自己在变好，以为快走出来了，然后某天晚上忽然发现你其实哪里都没去。你还站在原来的地方，只是换了一套好听的说法。"

叶澄沉默了几秒。

"你知道你这段话里最大的问题是什么吗？"她说。

"什么？"

"你把'走出来'当成一个目标。但你从来没有定义过什么叫'走出来'。是忘了她？是不再想她？还是找到下一段感情？你没有定义，就一直觉得自己在原地。"

陆允川张了张嘴，没有出声。

"其实你早就往前走了。"叶澄的声音忽然轻了很多，"你写了那么多日记，换了那么多说法，认识那么多人，做了那么多事——这些都是在走。只是你走的方式跟你想象的不一样，你就以为自己没动。"

虫声忽然变密了。夏天夜晚的蝉鸣和蟋蟀声搅在一起，像许多细小的针脚缝着夜的缺口。

"那我接下来该做什么？"

"把日记本合上。"

"啊？"

"合上一段时间。别再重读，别再重修，别再分析。去吃饭，去上课，去打球，去跟祁野吵架。日子长这样，不是长在日记里。"

陆允川愣住了。他从来没想过可以不写。写日记是他消化情绪的方式，但叶澄说的没错——有时候消化本身就是一种慢性咀嚼，嚼了几万遍还在嚼同一口。

"你是不是觉得我这人特别麻烦？"他问。

"还好。"叶澄说，语气正常得就像在评价今天的天气，"麻烦是有上限的，你还没到上限。"

他笑了，这一次是真的。

"要不要再走一圈？"叶澄站起来。

"好。"

两个人沿着湖边走。英语听力声彻底消失了，背书的学生也走了大半。剩下的几个坐在远处，说的话模糊成嗡嗡的背景音。

"如果真心不能兑换真心，我还打算拿什么继续证明？"陆允川把这句话问出来的时候，其实已经不期待答案了。

叶澄却答了："你不用证明。真心不是拿来证明的，是拿来用的。"

"怎么用？"

"用在值得的地方。别把它当投资，当礼物。送了就送了，别记账。"

他想起很多年前给苏晚枝写过一封信。信里写了很多话，现在回想起来，每一句都在记账——你看我为你做了这些，你看我多难受，你看我多认真。那封信后来没有寄出去，但它的逻辑一直留在他身上，跟着他长大，跟着他喜欢别人，跟着他被拒绝，跟着他坐在这张长椅上听叶澄说"别记账"。

"叶澄。"

"嗯？"

"谢谢你约我。"

"不客气。下次轮到你约我。"

"约你干嘛？"

"请我吃火锅。今晚这话的值一顿火锅。"

陆允川又笑了。"好。"

走到大路时，叶澄停下来。

"我往这边走了。"她指了指图书馆方向，"你回去别翻日记。回去洗澡睡觉。"

"好。"

"说话算话。"

"算话。"

叶澄走了几步，又回头："对了，你刚才说的那句话——'不甘心努力了这么久结果还是这样'——你确定你努力的方向是对的？"

"什么意思？"

"如果你努力的方向是'成为一个不再想她的人'，那方向就是错的。"她说完这句话，没有再解释，转身走了。

陆允川站在路灯下，看着她的背影拐进图书馆那条路。她走路很快，步子不大但频率高，像赶时间又不像真的在赶什么。

他往宿舍方向走了几步，又停住。掏出手机，打开微信。苏晚枝的对话框里，上一次的消息还是三天前。他打了一行字：

"考研加油。需要什么资料跟我说，我帮你找。"

发出去。然后把聊天记录往上翻了几十页，一直翻到两年前。那时候他们还会通宵聊天，聊一些不相关的事：电影、食堂的菜、某个老师的口音。每一条消息都很短，但频率很高，像两个人在同一个房间里说话。

翻着翻着，他停了下来。

不是因为难过。是因为他发现，翻到后面他已经记不起当初为什么要把某句话说得那么重了。

他把微信关掉。

祁野的消息弹进来："回来带瓶水，你欠我的。"

他回了一个"好"，把手机塞进口袋。

湖灯还在远处亮着。明天还会有，后天还会有。不管他有没有想通，不管他有没有合上日记本。湖水只管流它的，路灯只管亮它们的，食堂只管开它的门，祁野只管等着那瓶水。

他加快脚步走向小卖部。
