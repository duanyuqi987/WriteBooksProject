# 第092章 夏天的聚餐

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P092：原创仿唐绝句

小店灯黄照酒痕，
一盘凉菜隔前尘。
少年未敢言离别，
先把空杯递故人。

饭馆里开了三桌。亲戚坐靠门的两桌，同学被安排在里间，电风扇吊在头顶，转一下响一下。陆允川穿着那双新鞋，脚后跟已经磨破，坐下时悄悄把脚往椅子底下缩。

程砚来得晚，进门就把书包扔到他旁边，“你这鞋像借来的。”

“新买的。”

“那就是脚不配合。”

同学们笑起来。桌上有人问录取学校，有人把汽水倒进一次性杯子里，泡沫往外冒。说到夏知遥时，气氛短了一拍。一个男生用筷子点着花生米说：“她去的城市离你挺远吧？你俩本来也不是一路。”

这句话其实不重，甚至像玩笑。陆允川却没有接。他低头夹菜，筷尖碰到盘子边缘，响得突兀。

程砚把鸡腿夹到那男生碗里，“吃你的，升学宴不是相亲会。”

那男生讪讪笑了笑，话题被拉到别处。陆允川松了一口气，又觉得自己这口气松得难看。他想反驳，又知道反驳之后没有下文。夏知遥没有答应过他什么，别人说“不一路”，他甚至找不到一句能站稳的话。

饭吃到一半，父亲端着酒杯过来，拍他的肩，说以后去了外面要争气。亲戚们跟着点头。他看见母亲站在门边，把没动几筷子的菜倒进打包盒，动作很快，像怕浪费，也像怕被人看见。

散席时，程砚把他拉到饭馆后门。后门堆着空啤酒箱，油烟从厨房窗户往外冒。

“别因为一句话把自己憋坏。”程砚说。

陆允川靠着墙，脚后跟疼得发热，“我知道。”

“你不知道。你现在只是不知道该怎么生气。”

陆允川没再说话。他把鞋跟往下踩了一点，忽然觉得这一天最难忍的不是那句玩笑，而是他终于听见别人把事实说出来。
