# 第097章 省城南湖

所属部卷：第二部：六月以后
时间：大一九月
地点：大学南湖凉亭

## 诗歌插页 P097：原创仿唐绝句

南湖初照客衣新，
柳下行人各问津。
不是故乡风色浅，
少年还带旧时尘。

军训结束后的傍晚，祁野拉着陆允川去南湖。两个人从食堂后面绕过去，路边桂花刚开，甜味被汗味和洗衣粉味搅在一起。

南湖比地图上大。湖边有凉亭，柱子上的绿漆掉了一块，露出里面暗色的木头。几个女生在亭子里背英语，声音很轻。远处篮球场传来拍球声，水面把那些声音拆得碎碎的。

祁野买了两瓶冰红茶，递给他一瓶，“你高中喜欢的人也在省城？”

“不在。”

“那你看湖看得这么深情干什么？”

陆允川拧开瓶盖，气泡轻轻冒出来，“第一次来。”

祁野没再问。他坐到栏杆上，鞋尖晃来晃去，说大学就是这样，第一周谁都像换了一个人，第二周就开始找不到教室，第三周就发现饭卡不够用。

陆允川笑了。湖边风吹过来，他第一次觉得旧事没有跟着火车完全抵达这里。它在身上，却不是这里全部的天气。

回宿舍前，他在凉亭柱子旁站了一会儿。手机里高中群还在响，有人说军训黑了，有人发食堂照片。夏知遥没有再说话。

他把群消息设成免打扰。动作很小，却让他心里空出一点位置。那一点位置后来会住进诗社、苏晚枝、叶澄和许多新的误会。可在这一天，它还只是南湖边一阵能让人喘气的风。
