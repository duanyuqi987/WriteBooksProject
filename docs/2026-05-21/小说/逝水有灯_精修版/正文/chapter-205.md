# 第205章 最后一次争论多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：陆允川书房

## 诗歌插页 P205：原创仿唐绝句

香叶微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

多年后重写最后一次争论的那个下午，陆允川的外卖迟了四十分钟。送餐员打电话说电梯坏了，需要爬十二楼。他说没关系，自己下楼拿。

他在楼道里碰到了邻居。一个四十多岁的男人，抱着一箱矿泉水，满头汗。

"你几楼？"

"十二。"

"我也是。一起爬。"

两个人喘着气上楼，中间在七楼歇了一次。邻居问他在写什么，他说写小说。邻居说写的什么，他说写以前的事。邻居说哦，以前的事好写还是不好写，他说不好说。

话题在十楼就断了。各回各家。

陆允川把外卖放在桌上，是一份盖浇饭。他打开电脑，翻到几年前写的初稿。那个版本的"最后一次争论"里，他把苏晚枝写成了一个沉默的符号——她说的话不超过三句，剩下的都是他替她解释的。

程砚那时候看完初稿说："你这写的是苏晚枝吗？这写的是你对苏晚枝的想象。"

"那她是什么样的？"

"我不知道。我又没跟她谈过恋爱。"程砚往沙发上靠了靠，"但我知道她不是你想的那样。你把她写得太抽象了。什么'沉默如深潭'，什么'眼里藏着整个青春的秘密'——大哥，人家可能只是在想晚上吃什么。"

陆允川当时不太服气。现在重读这几行，他把"沉默如深潭"圈出来，在旁边批注：删。

初稿里还有一句："她走后的每一天，都像没有光的隧道。"

他重新批注：也删。她走后他有吃早饭、上课、考驾照、写论文、找工作。隧道是自己想象出来的。

他吃了几口盖浇饭，饭已经温了。他把蒜苔挑到一边。

手机上有消息。妹妹发来的："哥，妈问你今年过年回不回来。你要是回来的话她提前腌酸菜。"

他回："回。腌吧。"

"几号回？"

"还没定。买了票告诉你。"

"那你定了告诉我。对了，你小说写完了没有？妈说你写了好几年了。"

"快了。"

"'快了'是多久？从你大三开始就'快了'。"

陆允川没有回复这一条。他把手机扣在桌面上，继续翻初稿。

翻到面馆那一段，他注意到初稿里有一个细节是后来修改稿里没有的：那天结账的时候，收银台上放着一盆假的塑料花，花瓣上积了一层灰。苏晚枝站在收银台前等找零，用手指在灰上画了一条线。

这个细节他在重修的时候删掉了，觉得"俗"。现在再看，那条线比他的很多句子都真实。

他在修改稿里重新加上了这盆塑料花。

窗外的天色暗下来了。他把大灯打开，又觉得太亮，换了台灯。光罩在键盘上，把F键和J键上的小凸起照得很清楚。

他打字的速度比以前慢了。以前他是想到什么打什么，一行接一行，写完才看。现在他打一行停一下，把句子在心里过上两遍，有时候整段删掉重写。

那些年他给苏晚枝写信也是这个节奏。写信的时候觉得每句都是真理，寄出去以后才后悔——这句太重了，那句太多了，这整页都不该寄。

但寄都寄了。

多年后他能做的不是撤回那些信，而是在书里给苏晚枝应有的台词。

他打了：她说"我不是讨厌你，我只是那时候真的接不住。"

又打了：她说"你一直在'以为'。"

又打了：她说"十九岁的女生不知道怎么拒绝一个对她特别好的人。"

他把这些台词一字不动地放进去。不是替她补，是还给她。

九点多的时候，程砚打电话来。

"进度如何？"

"改到面馆那章了。"

"第几遍了？"

"第三遍。"

"老陆，"程砚说，"你有没有想过，你改这么多遍不是因为它不够好，而是因为你不想写完。"

陆允川没有说话。

"写完就真的结束了。"程砚替他回答了。"你不舍得。"

"也许。"

"那就别舍不舍得的。"程砚说，"该舍就舍。又不是死了，写完了你也可以继续改别的。"

"你说得轻巧。"

"旁观者就是轻巧。"

电话挂断了。陆允川喝了口水，继续往下翻。翻到面馆那一章的最后一段，原稿写着："他终于明白，体面不是赢。"

他在这句话上停了一会儿。然后删掉了"他终于明白"五个字。留下"体面不是赢"，想了想又不对——这话太像道理了。

最后他改成了：她走以后，锅盖还在响。

窗外有人在遛狗。狗链拖地的声音从十二楼也能听见，细细的金属摩擦声，一路从楼下滑过去。

盖浇饭已经凉透了。他把盒子封好，扔进垃圾桶，把筷子洗了搁在沥水架上。

电脑屏幕上光标还在闪。他看了很久那段文字，然后关了文档。不是累了，是今天写的已经够了。再多写就是硬凑。

睡前他刷了一下朋友圈。苏晚枝发了一条：新工作第一天，工位对着窗户，能看见一棵银杏。

他点了个赞。没有评论。

这是多年后他能做的最好的事：看见，但不打扰。点赞，但不要求回复。

关灯的时候，床头柜上亮着一盏小夜灯。不是特意留的，是早上出门忘了关。暖黄色的光在墙上投出一个很小的圆形，像面碗口的投影。

他翻了个身。

明天接着改。
