# 第226章 南湖再见之前

所属部卷：第五部：逝水有灯
时间：工作后一次培训
地点：大学南湖凉亭

## 诗歌插页 P226：原创现代诗

我绕过会议楼
看见南湖还在
它没有等我
只是把傍晚的灯
一盏一盏接住

那次培训安排在大学旁边的酒店。陆允川下午听完讲座，胸前还挂着参会证。同行的人约着去商场吃饭，他推说还有点事，沿着熟悉又陌生的路往校园走。

大学校门改了闸机，外来人员要扫码登记。他站在保安亭旁填信息，来访事由写“培训间隙参观”。这几个字比“重回南湖”稳妥，也更接近现实。门禁开时发出一声短响，他把参会证塞进口袋，往南湖方向走。

路边的店铺换了许多，原来的打印店变成了奶茶店，二号教学楼前多了共享屏。学生们抱着电脑经过，谈论简历、实验、社团招新。陆允川混在人群里，忽然觉得自己不像回到过去，更像误入了别人的今天。

南湖还在。凉亭柱子重新刷过绿漆，栏杆旁贴着禁止攀坐的提示。湖面有风，水纹把傍晚的灯拉得很细。他站在离凉亭几步远的地方，没有急着坐下。

过去他以为再见南湖，会再见苏晚枝、叶澄、那个失控的自己。可真正到了这里，最先看见的是新刷的油漆和提示牌。现实总比回忆更会安排轻重。
