# 第191章 夏知遥的消息之前

所属部卷：第四部：失去的美丽
时间：期末复习周
地点：旧同学群

## 诗歌插页 P191：原创现代诗

一个名字浮上来
屏幕亮了一下
我没有伸手打捞
水面很快
又回到水面

期末复习周，宿舍里到处都是打印资料。陆允川坐在下铺，把法学选修课的重点划到第三遍，手机忽然连震几下。旧同学群久违地热闹起来，有人发婚礼请柬，有人问暑假回不回县城，还有人提到夏知遥，说她在外省准备考研，最近回家办档案。

那个名字出现在屏幕上时，陆允川的笔尖停住。高中那条河、迟到的表白、没说完的话，一起从很远的地方泛起一点光。过去他很容易把这样的光当成召唤，仿佛一个旧名字出现，就能证明自己并不是一直被拒绝的人。

他没有立刻点开群头像，也没有翻她的朋友圈。手机扣在试卷旁边，他继续背重点。背到“合同解除”那一页时，他忽然觉得这个词有点好笑。青春里的许多关系其实从没签过合同，只是有人在心里单方面盖了章。

晚上熄灯前，群消息已经刷到九十九加。室友问他是不是也回县城，陆允川说：“看实习安排。”语气很普通。他把手机放到枕头下面，屏幕还残着一点温度。

夏知遥的消息还没有真正到来，它先试探了陆允川一下。试探他会不会把当下的狼狈，推回一个更早、更安全的梦里。
