# 第003章 初秋三楼之后

所属部卷：第一部：初秋三楼
时间：2010年九月
地点：县一中三楼东侧教室

## 诗歌插页 P003：原创仿宋小令

风过三楼，
纸页小窗明。
欲把旧年藏一笔，
却听铃后有人行。
灯下水声轻。

初秋三楼之后，陆允川表面上没有变化。早读照旧，跑操照旧，作业本照旧从前排传到后排，可他自己知道，有一处地方被轻轻碰过。

第一天的自我介绍结束后，教室里开始有了味道：粉笔灰、书皮塑料、汗水、拖把没拧干的潮气，还有刚认识的人努力装出的从容。

他把窗外法桐落下的第一片黄叶反复想了几遍，每想一遍都添一点不存在的意思。到第三遍时，他自己也觉得过分，便打开练习册，硬把目光压到题目上。

“你这人真有意思，刚来就盯着窗外。”程砚说。“我只是看看座位。”陆允川说。“座位在屋里，窗外又不发课本。”程砚说。

程砚把书包往桌上一扔，拉链没拉好，几本练习册哗啦滑出来。他笑得很轻，倒没有恶意，只是把陆允川那点紧张照得无处可放。

晚饭时食堂排队很长，铁勺敲在不锈钢盆上，声音一阵一阵。陆允川端着饭找位置，程砚抢先坐下，把一半桌面让给他。

“你别一有事就不吃饭。”程砚把筷子递过来，“感情没进展，胃先退学，不划算。”陆允川接过筷子，笑了一下，笑得很短。

叶澄在前排帮老师分表格，经过他们这排时敲了敲桌角，提醒新同学先写家庭住址，别把县城和乡镇填反。

夏知遥并不知道这些后续。夏知遥进门时没有抬头，她抱着两本书，发梢被风吹得贴在脸颊边。她先问班长空座在哪里，声音不高，像怕惊动还没安顿好的早晨。她有自己的课表、错题、家里打来的电话和不想解释的疲惫，不可能替陆允川安置所有心情。

这一点，是陆允川很晚才懂的。当时他只懂把一个小动作翻来覆去，像把一颗糖含到没有味道，还舍不得吐掉。

窗框底下积着一线灰，手指摸过去会留下白印。班主任在门口叫名字，叫错一次，教室里就有人低低笑一声。陆允川跟着笑，又很快收住，怕自己笑得不像这个地方的人。

夜里回到宿舍，他在日记本上记了几行，又怕别人看见，把本子压到枕头下面。窗外的风刮过晾衣绳，衣架轻轻碰撞，像有人替他把没说完的话敲碎。

班主任拿着花名册站在门口，念到陆允川的名字时停了半秒，问他是哪个乡镇来的。他答得很快，答完才发现自己的普通话带着一点方言尾音，耳根立刻热了。

夏知遥坐下后把书放到桌角，又从笔袋里取出一支黑色水笔。她没有加入旁边女生的闲聊，只在表格上慢慢写家长电话，写错一个数字，又安静地涂掉。

午饭时程砚带他去食堂，指给他看哪个窗口米饭少、哪个窗口阿姨手稳。陆允川端着餐盘跟在后面，第一次觉得县城学校也不是传说里的光鲜，它一样有排队、油烟和找不到座位的狼狈。

晚自习前，叶澄把多余的登记表交给老师，回座位时顺手关了半扇窗。风小下来，法桐叶贴着玻璃滑过去，陆允川低头写名字，最后一笔压得很重。

他把这一天拆成很多小段：哪一句话是真的，哪一个动作是自己添上的，哪一点失落其实来自期待太高。拆到最后，他发现能留下的并不多，可那一点点留下来的东西反而更干净。

寝室灯灭以后，走廊还有脚步声。陆允川闭上眼，脑子里反复出现白天的画面，出现到后来也疲惫了，只剩一道模糊的光。人总会在疲惫里比在激动里诚实一点。

事情过去以后，三楼东侧教室又恢复原样。报到单还在，旧课桌也还在，只有陆允川看它们的眼神变了一点，像有人在同一页纸上多压了一道折痕。

晚自习结束，程砚拍着他的肩说：“别复盘了，再复盘明天早读都开始了。”陆允川把书合上，终于跟着人群往外走。

如果只写疼，这一章会很薄；加上报到单、旧课桌、校服外套、粉笔灰，它才重新有了重量。那重量不是压人，而是让读者相信他们确实在这里生活过。

陆允川把报到单拿在手里，先把报到单填完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把报到单填完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把家庭电话写错一格后安静涂掉，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

三楼东侧教室里最明显的不是心跳，而是粉笔灰被风吹起的轻响。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，食堂铁盘里的番茄炒蛋把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。报到单、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情过去以后，三楼东侧教室又恢复原样。人群散开，桌椅归位，只有陆允川自己还把法桐叶想得太久，像把一页纸反复折到发白。

夜里他把这件事写进日记本，写到一半停住。原本想写“命运”，最后改成了“今天”，因为今天已经足够具体。

这一段不靠告白推进，而靠报到单、食堂队伍和晚自习把人稳稳托住。陆允川会尴尬，会饿，也会在夏知遥转身以后继续学着适应这个班。

如果只写疼，这章会很薄；加上报到单、粉笔灰被风吹起的轻响和食堂铁盘里的番茄炒蛋，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在三楼东侧教室发呆。”陆允川把报到单收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

这一章收在平常处：灯灭了，楼道静下来，陆允川睁着眼，终于承认今天不是命运给他的暗号，只是生活递来的一个片段。可片段也会发光，只要他不把光攥得太疼。
