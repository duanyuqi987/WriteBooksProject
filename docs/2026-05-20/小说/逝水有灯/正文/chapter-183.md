# 第183章 道歉未发送之后

所属部卷：第四部：失去的美丽
时间：第二天清晨
地点：图书馆四楼

## 诗歌插页 P183：原创仿宋小令

灯未暖，纸微凉。
昨夜一行藏。
不寄春风到旧窗。
留我自思量。

第二天早上，陆允川比平时早到图书馆。四楼还没坐满，清洁阿姨推着拖把从走廊那头过来，桶里的水轻轻晃。昨夜坐过的位置空着，桌面上的铅笔划痕仍在，好像什么都没发生。

他打开邮箱，草稿还躺在里面。标题栏空着，正文短短几行。没有发送，也没有消失。陆允川盯着它看了一会儿，忽然觉得这种“不动”比昨天的写作更难。写的时候还有动作可以躲，停下来以后，人只能面对自己。

手机震了一下，是班群通知下午交实习材料。他点进去，下载附件，去打印店排队。队伍前面有人忘带零钱，老板找不开，一群人便在门口等。生活就是这样，不会因为他终于克制了一次，就给他发一张奖状。

中午他在食堂买了最普通的青菜豆腐，端盘子时汤洒出来一点，烫到虎口。他下意识想拍照发给谁，又把手机扣在桌上。对面座位很快来了两个陌生女生，低声讨论选修课。他们的声音从他身边经过，没有为他的心事停留。

下午回到宿舍，陆允川把草稿复制进日记本，夹在那一页的不是“深情”，而是一条提醒：没有发出去的道歉，也要真正算数。不能因为她不知道，就偷偷把自己放回原来的样子。
