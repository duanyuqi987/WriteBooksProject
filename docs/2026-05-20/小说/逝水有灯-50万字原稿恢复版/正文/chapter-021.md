# 第021章 操场边的单词之前

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P021：原创现代诗

我把早晨放低，
让一粒尘从课桌边慢慢经过。
你的名字没有被我喊出，
只在单词书页角折起的三角旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写操场边的单词抵达之前的坡度。晚自习前的操场还有白天剩下的热。看台水泥摸上去发烫，塑胶跑道被夕阳照出暗红色，像一条慢慢冷下来的河。

陆允川摊开单词书，嘴里念着 abandon，眼睛却跟着跑道边的影子移动。念到第三遍，他才发现自己连中文意思都忘了。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。

那天真正的事还没有发生，单词书页角折起的三角却先被他记住了。它太小，小到放在任何人的日子里都不显眼，偏偏在陆允川这里有了重量。

“你这单词背得挺远，眼神都跑两百米了。”程砚说。“我看她们是不是要上楼。”陆允川说。“你管得比值周老师还宽。”程砚说。

程砚把自己的单词本拍到他膝盖上，叫他随机抽查。陆允川抽到的第一个词是 patient，他愣了一下，程砚说，正好，你先学这个。

叶澄从主席台旁边经过，抱着广播站的稿子。她看见他们坐在看台上，远远喊了一句快上自习，别等铃响才跑。

课间十分钟被铃声切得很碎。有人挤去小卖部买面包，有人趴在桌上补觉，有人拿着水杯一路跑向水房。陆允川站在这些声音中间，心里那点紧张没有消失，却被生活稀释了一些。

夏知遥这时还只是人群里的一个名字。夏知遥和同伴从看台下经过，手里拿着水杯。她笑的时候没有看这边，声音被风吹散，只留下几个轻快的尾音。陆允川看见了，却没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子，又一行行划掉。留下来的，是教室里真实的光、真实的灰、真实的不知道该把手放在哪里。

跑道边有卖烤肠的小摊，油烟越过围栏飘进来。有人边跑边喊借饭卡，有人在看台最上面弹吉他，音不准，倒也热闹。陆允川坐在这堆声音里，心事显得没那么孤单。

如果说喜欢在这里有了开端，也不是因为命运突然发声，而是因为一个少年开始过分留意普通事物。他还不懂，留意可以是温柔，也可能慢慢变成对自己的消耗。

看台最下一级积着一层灰，陆允川坐下前用书拍了拍，灰尘扬起来，被夕阳照成细小的金色。程砚嫌他讲究，直接坐下，裤子上立刻多了一道白印。

夏知遥从跑道边经过时，鞋带松了。她停下来弯腰系鞋带，同伴站在旁边等她。陆允川把视线挪回单词书，下一秒又忍不住看过去，像做错题的人偷看答案。

广播里放着晚自习前的通知，声音有点失真。叶澄抱着稿子从主席台下来，经过他们时说，背单词可以，别把操场坐成你们家客厅。程砚立刻举手保证三分钟后撤退。

回教室的路上，人群从操场往楼梯口挤。陆允川把单词书夹在胳膊下，页角一直硌着皮肤，他没有抚平它，像舍不得抚平那一点荒唐的标记。

这一章要把脚步放慢，慢到能听见课桌被拖动时的短促摩擦声。事情还没到来，可人物已经在日常里摆好了位置：谁靠窗，谁靠门，谁总替别人递一张纸。

陆允川后来才知道，所谓开始并不是一个明确的按钮，而是很多个微小选择叠在一起。他没有说话、没有靠近、没有确认，可他已经在心里给那个人留出了一块地方。

事情还没有真正发生时，陆允川已经在心里排练了好几种反应。可排练没有用，现实总会把人推到更笨的位置，让他先拿错单词书，再说错一句无关紧要的话。

那天放学，他没有立刻回宿舍，而是在操场西侧看台外多站了一会儿。风把晚自习铃吹得轻轻晃动，他忽然意识到，自己等待的并不是答案，只是一个能让明天继续期待的理由。

陆允川把单词书拿在手里，先把这一页单词背完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把单词背完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她弯腰系好鞋带又跟同伴继续往前走，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

操场西侧看台里最明显的不是心跳，而是晚自习铃从教学楼里散出来。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，围墙外烤肠摊飘来的油烟把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。单词书、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情还没到来，单词书已经先进入他的视线。他当时不懂这种提前的留意有多危险，只觉得只要自己足够安静，就不会打扰任何人。

放学后他在操场西侧看台外多站了一会儿，听见楼下有人喊集合。那一声很普通，却把他从想象里拉出来，让他记起自己还要回宿舍、洗杯子、准备明天的早读。

这一段把心事放回操场。单词要背，铃声会响，夏知遥会继续走向自己的同伴，陆允川也要跟着人群回到晚自习。

如果只写疼，这章会很薄；加上单词书、晚自习铃从教学楼里散出来和围墙外烤肠摊飘来的油烟，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在操场西侧看台发呆。”陆允川把单词书收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

本章最后，事情仍悬在半空。陆允川收好书，跟着人群往外走。楼道里风从窗缝挤进来，把他的校服袖口吹起一点，他下意识按住，像按住一句还没成形的话。
