# 第123章 媒人的话

时间：2018年3月

地点：楚州，电话

卷次：卷七：流年未断

他在通讯录里找到了张阿姨的号码。

存了很久了——从高中时候就有。但从来没打过。手指悬在那个号码上方停了两秒。然后按了下去。

拨号中。他把手机贴在耳边。听筒里传来等待音——嘟。嘟。嘟。他的心跳也跟着这个节奏跳——一下。两下。三下。

张阿姨是他初中同学周磊的妈，也是青阳人，在楚州住，平时爱给人做媒。去年过年回青阳，他妈跟张阿姨在菜市场碰上了，大概就是那时候说起的。周磊跟他坐过一学期同桌——人挺好的，就是话多。他不知道周磊知不知道他妈在给他高中同学做媒。大概不知道。

他拨了电话。

响了三声。接了。

"喂。"对方的声音很响，带着青阳口音。

"张阿姨。我是顾临风。"

"噢——临风啊。"声音立刻热了。"你妈跟我说了。周六下午去相亲对吧。姑娘叫苏晚晴。"

他说"嗯"——尽量让声音听起来像只是确认一下信息。

"我跟你说，这姑娘真的不错。"张阿姨的声音很大。他在电话这头能听见她那边有炒菜的声音——铲子碰锅沿，叮叮当当的。

"她也是青阳的。青阳一中毕业的吧——对，一中的。跟你一个学校。"

他知道。

"然后在楚州师大读的书。学教育的。"

他也知道。

"现在分到一小了。有编制，正式的老师。"

他也知道。

"人很文静。我见过一次——去年暑假，她回青阳，在超市里碰上的。白白净净的，扎个马尾，笑起来很舒服。一看就是个好姑娘。"

他没有说话。

"你妈说你也在楚州当老师。那多好。两个人都是老师，工作稳定，还有共同语言。"张阿姨越说越快。"而且都是青阳的。过年回家也方便。你说是不是。"

"是。"

"我跟她说的你——我说你在楚州师大读研，还在中学代课，人踏实，老实本分。她觉得挺好。"

他握着手机的那只手紧了紧。

"她答应的时候问了什么吗。"他说。

"问了什么？"

"她有没有问——我的名字。"

"问了啊。我说叫顾临风。她听完停了一下——然后说'好'。"

张阿姨语气里没有觉得这有什么特别的。

但他的心停跳了半拍。

她说"好"。

她听到他的名字。停了一下。然后说"好"。

"她有没有说别的。"

"没说什么啊。就说好，周六下午见。怎么了——你们认识啊。"

"不认识。"

张阿姨没有问下去。她大概不觉得这有什么奇怪——高中同校不认识很正常。一中两千多个学生。谁会记得六年前隔壁班的人。他也不能说你认识她。说出来就太长了——要讲三年同窗、手电筒、成绩单、水塔。太长。张阿姨正在炒菜。锅里的油在冒烟。

他说出这两个字的时候，喉咙里有点干。舌根有点苦——像含了一小片没化开的茶叶。说了个谎。但他也不知道"不认识"是不是谎。六年没见了。从电话号码删掉到现在——他的毕业、考研、代课——她也毕业、考编、工作。这六年里他们是两个陌生人。只是以前认识。现在认不认识——要等周六才能知道。

"那就周六见。我跟你说临风啊，这姑娘真的不错。你可得认真点。穿整齐点。别迟到。"

"知道了。"

"那就这样。我先炒菜了。锅里油都冒烟了。"

电话挂了。

他站在原地。手机还贴在耳朵上。听筒里是嘟嘟嘟的忙音。忙音在耳朵里响了很久。他把手机从耳边拿下来。屏幕暗了——通话结束。

炒菜的油冒烟了。铲子碰锅沿的声音在挂断的一瞬间就没了。

张阿姨没注意到——她说到这个名字的时候停了一下。

他注意到了。那个停顿很短——张阿姨大概以为她只是在回想。但他知道那个停顿是什么。是记忆被忽然敲了一下——有一个名字从六年没听到过。忽然出现在耳朵里。需要用一秒钟把这个名字从遥远的仓库里搬出来。确认一下。是她认识的那个顾临风。还是同名同姓的陌生人。确认了。是他。然后她说——好。

她把手机从耳边拿下来。屏幕暗了。他看着黑色屏幕上自己的脸。

她说"好"。

停了一下。然后说"好"。

他把手机放在桌上。走到窗前。窗外还是那棵樱花树。花瓣快落完了。树枝上冒出了嫩绿的叶子。

她听到他的名字。没有拒绝。

他不知道这是好事还是坏事。

如果她忘了——那他就是去相一个陌生姑娘。六年没见，她大概连他的脸都记不清了。

如果她没忘——那她为什么答应。

如果她记得的是高中时候那个坐在后排不说话的他——那个连正眼看她都不敢的他——她会失望的。

他不知道答案。

窗外有人骑着自行车过去。车铃响了。叮铃铃。

他站在窗边。

直到天黑。

窗外那棵樱花树还剩最后几朵花——在暮色里几乎是白色的。路灯亮了以后，花瓣的影子投在他的窗台上。他把手指放在那片影子上——凉的。花瓣在他指尖轻轻晃了一下——不是他碰的，是被楼下的风吹起来的。

周六。一茶一坐。三号桌。靠窗。

她会坐在那里。他会走过去。他会叫她的名字——或者她会先抬头。

他不想等了。他已经等了八年。

他把窗帘拉上。屋里暗了。他躺回床上——躺了一会儿。又起来。拿起手机，又给张阿姨发了一条消息。

"阿姨，周六我一定准时到。辛苦了。"

发完之后他把手机握在手里。屏幕亮了很久才暗。

外面的路灯把窗帘照成了一块长方形的橘色。他盯着那片橘色看了很久。

然后闭上眼睛。嘴角有一条很细的弧度——不是笑。是那种——像是终于把一笔拖了很久的债还上了前半段的弧度。

窗外有人骑着自行车过去。车铃响了。叮铃铃。骑车的是个中学生——校服是蓝色的。跟青阳一中的不一样。但也是蓝色的。书包在背上晃来晃去。

他看着那个学生的背影消失在街角。想起自己高中时也是这样——骑车上学。骑车放学。书包在背上晃。车筐里放着课本。有时候在路上碰到苏晚晴——她在前面走。他在后面骑着。不敢按铃。就慢慢骑。跟在她后面大概十米的位置。一直到她拐进她家的巷子。他才加速骑走。她大概从来没注意到后面有个人。六年了。她不知道这件事。

他把手机放在桌上。走到窗前。窗外那棵樱花树在风里晃。树枝上新叶已经成形了——嫩绿的。花全落了。树枝上只有叶子。一只麻雀停在树枝上——歪着头。低头啄了一下——大概啄到了一个虫子。然后飞走了。

他又想起六年前。高三那年——她从他课桌边走过去。大概是去交作业。经过的时候她的衣角碰了一下他桌角上堆的那摞书。最上面那本化学课本晃了晃——没掉下来。她伸手想扶——他已经先伸手了。两个人的手指尖在空中碰到了。不到一秒。她缩回去了——说了句"不好意思"。他摇头说"没关系"。那是他们高中三年——唯一一次碰到手。
窗外的樱花树还在晃。树枝上的新叶在风里沙沙响。他拉上窗帘。躺回床上。
