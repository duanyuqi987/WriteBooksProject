# 第167章 牵手的误会

所属部卷：第四部：失去的美丽
时间：大二冬末
地点：学校东门外小路

## 诗歌插页 P167：原创仿唐绝句

东门灯缺影成双，
一瞬扶人误作伤。
若肯近前听半句，
不须独自判寒霜。

苏晚枝抬头看见陆允川，先招手，“你来得正好，横幅好像拿错了。”

陆允川走过去，目光落在她手腕上。那里没有什么痕迹，只有冬天冻出的微红。

男生解释：“刚才她踩到砖，我扶了一下。”

苏晚枝也说：“这边路太暗了。”

陆允川点头，却没有接横幅。他心里明明听见了解释，却像不肯让解释进来。那一瞬间，他觉得自己所有努力都被推翻了：不打电话，不追问，不多说话，好像都挡不住她和别人自然亲近。

“横幅我看看。”叶澄从后面赶来，打破了沉默。

她摊开横幅，发现尺寸确实错了。几个人站在路边商量要不要换。苏晚枝说她回打印店，男生说他去，叶澄让大家别在小路口堵着。

陆允川全程说话很少。苏晚枝看了他几次，最后问：“你是不是又误会了？”

这个“又”字让他脸上热起来。

“没有。”他说。

苏晚枝没有再追问，只把横幅递给叶澄。那一晚真正让误会变坏的，不是男生扶她那一下，而是陆允川听见解释后，仍然选择把自己关在那个画面里。
