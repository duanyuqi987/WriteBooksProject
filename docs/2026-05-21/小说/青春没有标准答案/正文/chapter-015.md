# 第015章 早读铃声



时间：返校领档案的清晨



地点：空了一半的初三教室



卷次：第一卷：九年级的志愿表



返校领档案的清晨，空了一半的初三教室。黑板右上角还没擦掉的早读安排把这一天的开头压得很实。校门外早餐摊的油条刚下锅，热油声和旧铃声一起挤进校园。段璟站在旁边，手上还有铅笔芯蹭出的黑印，耳朵却一直留意着常俊凡那边的动静。那时候他们还没有后来那么多说不清的关系，只是同一场中考里两个被分数推着往前走的人。

不用早读的早晨反而让人不习惯，像忽然被允许离开，却不知道该往哪走。刘海峰在讲台前把粉笔放下，提醒大家先把手里的题写完。窗户开着，外面有人跑过楼梯，鞋底擦过水泥地，声音短促得像一句没说完的话。

常俊凡想把借过的书和该还的人情都理清，不带着糊涂离开。她做事不急，先把笔袋摆正，再把要问的问题写在草稿纸右上角。段璟后来才明白，那不是冷淡，也不是故意把人挡在外面。她只是很早就知道，别人可以热闹，可以起哄，可以替她猜测，但真正要承担结果的人只有她自己。

段璟想把她留下来说几句话，但教室里每个人都在告别，单独的告别反而最难。他不是不会说话，只是一碰到常俊凡，平常那些贫嘴和硬话就像被粉笔擦抹了一遍，剩下一点不成句子的灰。他怕自己显得太在意，又怕自己真的一点也不在意。初三的男生常常这样，明明心里已经绕了好几圈，开口却只剩一个嗯。

他们的对话没有谁故意说得漂亮，都是初三学生能抓住的那些实话。“你的照片掉了，别夹错袋。”许念安说。“谢谢。这个本子是段璟的，也麻烦你递一下。”常俊凡说。“不用麻烦，我自己拿。”段璟说。“领完档案就回家，别在楼道里装舍不得。真舍不得，暑假也要看书。”刘海峰说。话说完以后，谁也没有立刻把目光放到对方脸上，仿佛只要盯着桌面、墙面或手里的纸，心里的动静就不会被看出来。

许念安也在场。帮同学把散落的照片夹进档案袋，动作温柔得像怕弄皱别人的三年。这种小动作比豪言壮语更像那一年：大家都说自己不怕中考，可水杯里的水少得很快，草稿纸一张接一张被揉皱，课桌边缘全是指甲抠出来的小白印。

李桂芹一早给段璟装了两个菜角，说档案别折，路上别疯跑。家庭没有在他们的青春里大声喊口号，它只是藏在饭盒、车票、洗好的校服和一遍遍叮嘱里。封丘县的日子也是这样，街道不宽，消息传得快，谁家孩子能不能上一中，很快就会成为亲戚饭桌上绕不开的话。

常俊凡把一本薄薄的错题本递给段璟，说你的资料我都还清了，只剩这本，是我自己抄错了封皮。这个动作没有被任何人郑重记录下来，甚至连旁边的人都未必注意到。但段璟记住了，因为它让他第一次意识到，所谓喜欢不是突然出现的一句表白，而是很多个没说出口的瞬间慢慢叠起来。

县城里的夏天靠近得很快。午后窗台烫手，作业本边缘卷起一点毛边，水房里有人一边接水一边背英语单词。广播里偶尔提到北京奥运，老师提到中考，家长提到一中，学生提到放假。所有大事小事挤在同一张课桌上，谁也没有多余的地方安放自己的慌张。

早读铃声按时响了一遍，可教室里没人开口读书，所有人都在那阵铃声里低头找自己的档案袋。段璟那时还不会把这种场面写成故事，他只会把它塞进心里，等很多年以后再慢慢认出：原来青春最真实的部分，不是后来回忆时替它加上的光，而是当时那些尴尬、具体、说错话又来不及补救的细节。

教室后排的墙皮翘起一小块，谁靠在上面都会蹭下一点白灰。段璟坐下时先拍桌面，再把书摞齐；常俊凡则先看黑板右上角的日期，把当天要做的事写在草稿纸背面。两个人的动作完全不同，却都像在给自己找一个稳住的办法。

课间十分钟其实很短。有人冲去水房，有人趴在桌上睡，有人围着红榜装作只是路过。常俊凡不太参与这种围观，她会等人散得差不多再走过去，看一眼就离开。段璟有时跟在后面，连自己都嫌自己没出息。

晚自习最后一排总有人小声传橡皮和尺子。刘海峰一回头，那些手立刻缩回去，只剩卷子还在桌面上摊得很无辜。常俊凡偶尔会被逗笑，但笑意很快收住，像她知道自己不能在这个时候完全松下来。

许念安在旁边留下的小动作，也让这一章不只围着段璟和常俊凡转。封丘县的学生多数早早学会看大人的脸色：家里宽裕不宽裕，老师看好不看好，自己能不能把一次机会抓住。这些判断没有写在卷面上，却时时刻刻挤在卷面旁边。

常俊凡很少把情绪说满。她说可以，就是可以；说不改，就是不改。这种短句有时会让段璟不知道怎么接，可也正因为短，才不像临时编出来的温柔。她给人的距离感，不是拒绝所有人，而是提醒别人别替她安排人生。

段璟的笨拙则更容易被看见。他会在该说话时沉默，在该沉默时又憋出一句不合适的话。付小川他们偶尔笑他，他也会顶嘴，可常俊凡一看过来，他的顶嘴就像被人抽掉了半截，只剩下耳朵发热。

那时候大家还没有智能手机可以随手查消息，更多事情靠公告栏、电话、老师通知和同学传话。一个消息从办公室传到教室，往往会变出好几个版本。有人说分数线会涨，有人说一中要扩招，有人说住校床位紧，越传越像一场小小的风。

封丘县的夏天有自己的声音。三轮车从巷口过去，车斗里响着铁盆；早餐摊收摊时，塑料凳一只套一只；远一点的地方，麦场上有人拍打编织袋。这些声音落进少年人的心里，不会立刻变成故事，却会在多年后突然浮出来。

早晨校门口的早点摊还没完全收，油条筐上盖着一块发硬的白布。段璟经过时，总能听见老板娘喊谁的豆浆没拿。这些声音和黑板右上角还没擦掉的早读安排一起留在他记忆里，让早读铃声不像一句章节名，而像一个能闻到油烟味的上午。

课间十分钟没有想象中轻松。有人冲去水房，有人围着老师问题，有人站在走廊里用手扇风。许念安从人群里挤过去时说了一句让让，常俊凡把资料袋往怀里收，段璟则下意识侧身，给她留出半步。

午饭时，食堂的白菜豆腐热得烫嘴，大家还是吃得很快。段璟把饭盒放到桌上，听见旁边有人讨论分数线，筷子停了一下。常俊凡没有加入，只把自己碗里的葱花挑到一边，动作慢得很稳。

下午最容易犯困，窗台被太阳晒得烫手，试卷边角卷起来一点。刘海峰让他们把错题重新抄一遍，班里顿时响起一片很小的叹气声。常俊凡没有叹，她只是换了一支笔，像换一种力气继续往下写。

傍晚放学时，县城街灯一盏盏亮起来，小摊开始把塑料凳搬到路边。段璟有时会远远看见常俊凡和几个女生并排走，她们说话声音不大，经过文具店时会停一下，看看新到的笔芯和文件袋。

晚自习里的时间比白天慢。吊扇转得吱呀响，黑板上的粉笔字被灯管照得发白。段璟低头写题，偶尔听见常俊凡翻页，声音很轻，却足够让他知道她还在前面某个具体的位置。

回家路上，自行车铃声从身后过来，又很快被风推远。封丘县的路灯不算密，明暗交替落在路面上，像一格一格还没写完的选择题。段璟骑得不快，脑子里却一直在想一中、成绩和常俊凡。

夜里院子里有蛙声，远处偶尔传来货车经过的低响。段璟躺下以后，会把白天没说好的话重新想一遍。想得再顺，第二天见到常俊凡时，还是会卡在喉咙里。

文具店老板认识不少初三学生，看见他们进门，不用问就把黑色笔芯和透明文件袋往外推。段璟买东西时会先看价签，常俊凡也一样。两个人都没把这种节省说出口，却都知道高中还没开始，钱已经先变成了一道题。

小卖部门口的冰柜嗡嗡响，最便宜的雪糕总是先卖完。许念安说考完一定要吃两根，常俊凡提醒他先别把嗓子吃坏。段璟听着他们说话，忽然觉得这种普通的玩笑比任何保证都让人踏实。

水房排队的时候最能看出一个人的性子。有人插队，有人抱怨，有人默默把瓶子往里挪。常俊凡通常不抢，也不让自己吃亏。轮到她时，她把瓶口对准热水龙头，手腕很稳，像对待一道必须小心计算的题。

作业清单贴在黑板左侧，每天都被人用粉笔补上一行。段璟有时看着那几行字发愣，觉得它们像一道矮墙，把所有人围在中考前的日子里。常俊凡则会把清单抄到本子上，完成一项划掉一项。

隔壁班偶尔传来一阵哄笑，刘海峰就会走到门口看一眼。本班立刻安静，连付小川也把嘴闭上。常俊凡低头写题，段璟看见她的耳朵被窗外的光照得有点红，很快又觉得自己不该看这么细。

家长打来的电话常常很短。吃饭了吗，别熬太晚，表填了吗，钱够不够。这些句子听上去都不新鲜，却能把一个学生从自以为很大的心事里拉回现实。段璟和常俊凡都在这样的电话里，学着不把烦躁说给家里听。

有一次下雨，走廊地面被踩出一串泥点。常俊凡从书包侧袋里拿出一张旧纸，垫在文件袋外面，怕水溅上去。段璟看见以后，也把自己的资料往怀里收了收。他不是忽然懂事，只是忽然知道有些东西应该被好好护着。

班里有人把封丘一中说得像一座很远的城，也有人说它就在县城里，没什么稀奇。常俊凡听到这两种说法都不争。对她来说，远不远不是地图决定的，是自己有没有真的走进去。

段璟偶尔会把草稿纸写满，再从边角撕下一小条算题。他不舍得浪费纸，也不舍得承认自己在意常俊凡。于是很多心事都跟公式挤在一起，写得乱七八糟，最后被他揉进桌洞。

常俊凡的沉默并不空。她沉默时会做事：削铅笔、抄题、整理文件袋、把借来的资料按原来的顺序夹好。段璟慢慢看懂，真正清醒的人不一定说很多道理，她们只是把每一步都走得不含糊。

操场边的杨树叶在热风里翻动，叶背一闪一闪发白。体育课集合时，大家站得歪歪扭扭，心思却还留在教室里的卷子上。段璟听见有人喊常俊凡的名字，回头太快，被付小川看见，笑了半天。

那种笑没有恶意，却让段璟明白，喜欢一旦露出一点边，很快就会被同龄人看见。他开始学着把眼神收慢，把话说少，也把自己真正想问的问题压到更深的地方。

可越是压着，细节越会自己冒出来。一支笔放在桌沿，一页资料被风翻起，一句平平常常的提醒，都能让段璟在很多年后想起那年夏天的温度。

封丘县的傍晚来得不急。太阳落下去以后，屋檐和树影还热着，路边摊的煤炉慢慢红起来。学生们背着书包从街上走过，校服后背被汗洇出浅浅一片，谁也不觉得狼狈，因为每个人都差不多。

常俊凡走路时很少东张西望，段璟却会把周围的动静都看一遍。他们像两种不同的学习方法：一个先定方向，一个不断确认旁边有没有路。这样的不同，后来也一直影响着他们。

所以这一段不能只写成暗恋。暗恋只是表层，底下还有县城学生对升学的谨慎、普通家庭对花销的计算，以及两个少年第一次认真面对选择时的笨拙。这些东西合在一起，才让他们的靠近有了真实重量。

纸张在这一卷里出现得太频繁：成绩条、志愿表、作文纸、捐款名单、招生说明。它们看上去都很薄，却能让一个家庭晚饭吃得沉默，让一个学生整夜睡不好。段璟以前觉得纸就是纸，直到常俊凡把每一张都收得平平整整，他才知道有人是在用这种方式把人生扶正。

刘海峰不喜欢学生把话说得太满。他批评人时常先看一眼窗外，好像要把火气压下去再开口。可他也不是不懂学生的紧张，有一次经过常俊凡座位，看见她把一道题改了三遍，只留下一句：别只改答案，改原因。常俊凡把这句话抄进本子，段璟也偷偷记住了。

教室后面的黑板报还留着上个月的标语，红粉笔写的字已经淡了。宋乔说该换新的，付小川说等中考完再换，谁还有心思画花边。常俊凡没有参与争论，只把掉在地上的图钉捡起来，放进粉笔盒旁边。这个动作让段璟安静了一会儿。

有些场面如果只写一句他们都很紧张，就会轻得像没发生过。真正让人相信它发生过的，是铅笔芯断掉以后谁递了一把小刀，是水房排队时谁往旁边挪了半步，是车铃响起时谁下意识护住了文件袋。段璟和常俊凡的靠近，也只能从这些动作里慢慢长出来。

饭盒、脸盆、准考证套和透明文件袋，都是那一年反复出现的东西。它们不抒情，却比抒情可靠。段璟记得常俊凡把文件袋抱在胸前走过楼梯口，也记得自己把饭盒塞进书包时，里面的勺子撞出一声闷响。生活先把他们推到这些物件面前，再让他们慢慢明白选择的重量。

大人们谈未来时，常常把声音压低，可压低并不等于学生听不见。段璟听见过父亲说要不就让孩子拼一回，也听见过母亲说住校可别冻着。常俊凡听见过母亲问学费，也听见父亲说姑娘想去就让她去。那些话没有直接出现在课堂上，却一直跟着他们进考场。

同学之间的关系也在这种时候变得微妙。平时一起打闹的人，到了报志愿时会突然认真；平时不怎么说话的人，会在排队时递过一支笔。许念安有自己的紧张，也有自己的玩笑。群像不是背景，它让段璟知道，自己的心事只是青春里许多心事中的一个。

晚上电视里播新闻，奥运的画面和彩色广告交替出现。段璟一边看，一边给铅笔削尖；常俊凡可能也在另一个屋子里整理书包。他们并不知道彼此正在做什么，却都被同一个夏天推着往九月走。

如果多年以后再删掉这些细节，只留下他们喜欢过彼此，故事会一下子轻很多。真正让这段青春站起来的，是他们还没懂爱情时先懂了考试、家境、县城和选择。这些东西不浪漫，却给浪漫留出了可信的地面。

常俊凡不喜欢别人把她写成遥远的光。她会皱眉，会算错，会因为母亲一句别太累而突然红眼圈，也会在第二天照常把错题整理完。段璟喜欢的不是一个被想象出来的完美女孩，而是这个会把文件袋抱得很紧、却仍然往前走的人。

段璟也不应该只站在暗恋的位置上。他要面对的同样有成绩、家里、钱和选择。他如果只是看着常俊凡发呆，这个故事就会变窄；他真正开始长大，是从意识到她也有自己的难处，而不是只在他的青春里发亮开始。
