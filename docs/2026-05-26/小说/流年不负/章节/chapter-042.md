# 第042章 深夜的灯

时间：2011年10月中旬

地点：男生宿舍

卷次：卷三：此心藏久

手电筒换了一节新电池。

两块钱一节。南孚的。学校小卖部最贵的电池。顾临风在货架前站了大概十秒。拿了最便宜的——一块钱的杂牌。转身走了两步。又回来。把杂牌的放回去，拿了南孚。

晚上十点。熄灯。

月光还是从窗帘缝里漏进来。青阳十月的晚上已经凉了。窗户外面有风，吹得窗帘那条缝一会儿宽一会儿窄。地上的月光也跟着变。

顾临风把被子蒙过头。手电筒打开。光比前几天亮多了——新电池就是不一样。光圈打在练习册上，边缘清晰。

今晚做的是数学。

函数。

他数学不算差，但也不算好。高一期末考，数学考了七十八分——一百五十分制。选择题错了五道，填空题空了四道，大题第一问做了、第二问不会。

他把那张卷子找出来。折了两折，塞在枕头套里。每天晚上拿出来看一遍。错题重新做。

被子里的空气越来越闷。脖子后面出汗了。

冬天的手电筒，塑料壳一开始是凉的，过了一会儿就热起来。热度从手心里传开。他的左手托着下巴，右手写字。课本压在膝盖上，被子是天然的隔音罩——外面听不到翻书的声音。

十一点二十。上铺的刘伟翻了个身。床板吱了一声。

顾临风关了手电筒。等。

呼吸声。刘伟又翻了两下。然后呼吸均匀了。

重新打开手电筒。把被子往下拉了拉，露出半个脑袋透气。月光照在地板上，也照在他的眼睛上。他眯了眯眼。

继续做题。

从函数转到物理。从物理转到英语。英语单词抄了三页练习本。他写字本来就小，在昏暗的手电筒光里更小了。一个字一个字挨得很紧。一排排往下走。

零点过一刻。

他听见了——有人从床上坐起来。

脚步声。赵一鸣的。他认得赵一鸣走路的节奏——慢，有点拖，右脚落地比左脚重。

顾临风把被子的缝隙捏紧。光从指缝里漏出去了一点点。他赶紧关手电筒。

赵一鸣去了一趟厕所。然后走回来。

在顾临风的床前停住了。

没声音。

顾临风屏着呼吸。

然后赵一鸣伸手——拉开了他被子的一角。

手电筒光漏了出来。打在赵一鸣脸上。赵一鸣的半张脸是亮的，半张脸在暗处。眉毛是皱的。

顾临风没动。右手还拿着笔。

赵一鸣看着他，他看着赵一鸣。

大概五秒。

赵一鸣的声音压得很低。

"你疯了？"

顾临风没有说话。手电筒的光圈在天花板上晃了一下。

然后他说了一个字。

"嗯。"

赵一鸣站在那儿。月光从他身后照过来。他的脸变成了完全的剪影。

"几点睡的？"

"一点左右。"

"昨晚。"

"一点左右。"

"昨晚昨天晚上。"

"一点左右。"

赵一鸣把手收了回去。被角落下来，重新盖住了光。

"明天还要上课。你考十五名够了好吗。"

顾临风在被子里没有回答。手电筒亮了太久，塑料壳烫得有点拿不住了。他换了一只手。

赵一鸣的脚步声回到了自己的床位。被子窸窣了几下。然后安静了。

手电筒重新打开。

他做了一道物理大题。然后第二道。

凌晨一点。手电筒的光黄得很厉害。新电池也用得差不多了。

他把练习册合上。笔夹在中间。手电筒关了。塞回枕头底下。

翻了个身。面朝墙。

墙上那张课程表和去年一样。边角翘起来了。字在黑暗里看不见。

他闭上眼。

第二天中午。食堂。

赵一鸣端着餐盘坐到他对面。一盘西红柿炒蛋，一碗米饭。他用筷子把菜拨到饭上面，然后大口吃。

"你这样不行的。"

顾临风低头吃饭。

"人会垮。垮了考个屁。"

"还有两百多天高考。你现在就拼，拼到明年你拿什么拼。"

顾临风嚼完嘴里的饭。

"有数。"

"你有个屁的底。高中三年又不是跑百米。是马拉松。"

顾临风没说话。

赵一鸣看了他一会儿。然后把自己的西红柿炒蛋拨了两勺到他碗里。

"多吃点。至少饭要吃饱。"

顾临风看着碗里多出来的菜。西红柿切得很碎。蛋花炒得有点散。食堂的菜。

"谢了。"

赵一鸣没应。低头继续扒饭。

下午课间。赵一鸣从书包里掏出一个东西放在他桌上。

一节新电池。南孚的。

"昨天晚自习去买的。"

顾临风看着那节电池。绿色的包装。南孚两个字是金色的。

"多少钱。"

"两块钱你别给了。"

"不收。"

赵一鸣把电池往他那边推了推。

"不是给你的。是投资。等你考上好大学以后请我吃饭。"

顾临风沉默了几秒。

然后他把电池拿起来。放进了铅笔盒里。

铅笔盒的小虎队贴纸又磨白了一块。

那天晚上。熄灯后。

两节电池——一节赵一鸣给的、一节自己买的。他把赵一鸣给的那节先装上了。

手电筒亮了。比昨晚暗一点。但够用。

他翻开练习册。从昨晚停下的那道物理题开始继续做。

做了二十道。然后英语。英语做完是生物。

凌晨一点。

关了手电筒。裹紧被子。青阳十月的晚上真的很凉了。脚底是冰的。

他蜷了一下腿。

闭上眼。

脑子里是那道明天要问数学老师的题。还有苏晚晴说的那句话——知道自己在要什么。

他知道自己在要什么。
