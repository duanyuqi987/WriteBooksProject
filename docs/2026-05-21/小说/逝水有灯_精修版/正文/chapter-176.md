# 第176章 朋友的烟之前

所属部卷：第四部：失去的美丽
时间：日记事件后凌晨
地点：宿舍

## 诗歌插页 P176：原创现代诗

我把烟头放回夜色，
不让它替谁证明深情。
风从未说完的话里穿过，
有人低头，学着不再追问。
后来水声慢下来，
灯还亮着，人已经能往前行。

日记事件之后的第三天凌晨，陆允川醒了。

不是自然醒——是梦见苏晚枝站在食堂那个位置看着他，眼睛里有话但不说。他猛地睁开眼，心跳很快。程砚的闹钟还有两小时。祁野的鼾声很轻。窗外天还没亮，远处食堂开始卸菜，三轮车倒车时响了两声。

他轻手轻脚下了床，坐在书桌前。没有开台灯，只是在黑暗里用手指摸着日记本封面磨毛的边角。昨晚他把删除线那两页翻过去后就再也没打开。但在梦里又把那些话全回想起来了——划掉的、没划掉的、写得最重的、写得最长的——全部。

他穿上外套，想出门走一圈。走到门边时，祁野忽然醒了。不知道是真醒了还是根本没睡熟——但坐起来的那一下，他的眼睛不是迷糊的，是清醒的，像一直在等他动身。

"去哪。"

"出去走走。"

祁野看了他几秒。然后套上自己的外套。

"我跟你去。"

"不用，我一个人——"

"你上次说'一个人'的时候，在楼下冻了将近两个小时。上上次说'一个人'的时候，写了十行删掉十行。"祁野从抽屉里摸出一包皱巴巴的烟——他不怎么抽，这包还是上个月程砚过生日时别人送的。"走。"

陆允川没再推辞。两个人在走廊里轻手轻脚地走，下楼，推开宿舍楼后门。

后门的水泥地有点潮。昨天下午可能下过一点小雨。垃圾桶旁边堆着几个压扁的纸箱，被雨水泡得软软的。空气里有洗衣粉和冷风的混合味——洗衣粉味来自旁边洗衣房的排风。天还是很黑，远处食堂的灯已经亮了，三轮车倒车的提示音闷闷的。

祁野靠墙站着。把烟抽出来一根，夹在指间——然后拿出打火机，没打火，只在掌心里转了一圈，又塞回口袋。

"你以前凌晨出来过几次？"祁野问。

"好几次。台阶那次——还有还书那次——都是在晚上。但凌晨是第一次。"

"为什么出来？"

"在宿舍憋着更难受。"

"对。憋着更难受。"祁野终于把打火机重新掏出来，点了烟。他不常抽，咳嗽了一下，但没丢——只是用手夹着，没吸，让它在冷空气里自燃。

"我凌晨出来过很多次，"祁野说，"但不是因为感情。是因为游戏。打排位打到凌晨，饿了，下来吃夜宵。吃完觉得今天又浪费了。站在这个垃圾桶旁边抽根烟，回去继续打。听起来很废吧。"

"不废。"陆允川说，"至少你知道自己在浪费时间。"

"你不也是？你知道自己在浪费情绪。你知道放不下苏晚枝已经是浪费了。但你还在浪费。"

陆允川没回答。后门的风从北面灌进来，把纸箱吹得轻轻挪动了一下。

"你觉得我是不是应该——跟苏晚枝道歉？"

"道什么歉？"

"为所有让她觉得不自在的——"

"你道过歉。"祁野打断他，"台阶那晚道了。还书道了。食堂误会道了。你应该问你自己——你道歉是为了什么？是为了让她原谅你，还是为了让自己心里舒服。"

陆允川沉默了很久。三轮车又倒了一次，声音在冷空气里传得很远。

"可能——是为了让自己舒服。"

"那就不要再去道歉了。如果你的道歉都是为了自己——那就跟送书一样——你在借道歉的名义索取回应。"祁野把打火机在手掌里转了几下，然后揣回口袋，带点不耐烦地戳了戳自己的腕，"我说话很直，你扛一下。"

"扛得住。"陆允川说，顿了一下，又说，"你现在说的话跟叶澄差不多了。你什么时候跟她一个水平线了。"

"因为我还欠她一顿火锅——上个月她帮我修了一篇选修课的论文。代价就是以后陆允川出事我要用叶澄的方式说话。"

"她真的在搞培训。"

"对。她的培训班全名叫'学会用人的方式对待陆允川'。"

陆允川忍不住笑了一声。不是因为好笑——是因为没办法反驳。那包烟在祁野指间慢慢烧向滤嘴，烟灰被风吹落，掉在潮湿的水泥地上，火星在灰之间一亮又灭了。

"烟快烧完了。"陆允川说。

"嗯。"

祁野没有猛吸，只是让烟自己烧。燃到滤嘴还剩一小截红点。他把烟头按在对面的墙上——不是扔到垃圾桶里，而是直接按在砖缝里。动作不重。
