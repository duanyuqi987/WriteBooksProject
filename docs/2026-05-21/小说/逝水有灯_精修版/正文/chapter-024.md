# 第024章 操场边的单词旁观

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P024：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
单词书页角折起的三角还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，操场上的晚自习前时间对每个人来说都是不同的。有人绕着跑道跑圈——跑得快的人为了增强体能，跑得慢的人为了打发时间。有人坐在草坪上吃一包干脆面，有人靠在单杠边对英语小测的答案，有人只是从教学楼到宿舍楼之间经过操场抄近路。对绝大多数人来说，操场是一段过渡空间——从白天过渡到晚上的间隙。对陆允川来说不是——操场是有功能性的：它提供从看台到五米外台阶之间的一条单向通道。

程砚不止一次坐在陆允川旁边，他发现陆允川背单词的效率完全取决于操场上有没有人散步。"你知道吗，你背单词的速度分成两档——操场有人的时候大概每小时背五个词，没人的时候大概每小时背二十个。""那是因为有人的时候我容易走神。""走神？你那不叫走神，你那叫定向观测。"

程砚有一次走下看台去买水，刚好经过蒋雯和夏知遥旁边。他听见蒋雯说"明天晚上我们别来了——太冷了，风大"；夏知遥没说话，只是点了点头，把手里的水杯攥紧了。程砚回来后把这个情报转告给了陆允川："明天她们不来操场——可以全速背单词，不用减速了。"他说得很随意，像是汇报天气预报。

叶澄的观察角度更有趣。文科生的教室仍然在老楼，离操场比较远。但有一天她因为帮语文老师拿东西路过操场，正好看到陆允川在看台上背单词——隔着整个操场，她还是认出了他的坐姿。"他坐在看台上背单词的姿势很特别——不是低头看着书，是把书摊在膝盖上往跑道方向看。那不像是在背书，像在守什么。"叶澄后来这样描述。

她走过去，在他旁边坐了两分钟，没有讨论英语，而是问："你每天来这里背，操场上的风那么大，书页不会一直吹翻吗？""用夹子夹着。""那你夹的功夫还挺好。"她站起来拍拍裤子走了，没再多说——但她注意到程砚的书页上夹的夹子是从陆允川那里借的，而陆允川的手边放着两个备用的。

蒋雯是最晚意识到的人。她在高二的时候翻旧照片——班群里有人发了一张高一时期的偷拍，画面里有看台、夕阳和一个人坐在台阶上看书的背影。蒋雯忽然在下面评论说："这是不是陆允川？他那时候天天在操场背单词，我以为他是学霸——后来发现他是我们班的没错，但他去操场不是为了学习吧？"她发完以后等了一会儿，没人接——程砚看到了，但没回复，因为他不确定陆允川看不看群。

夏知遥始终没有参与这场讨论。她不知道这个话题被提起过——因为她不在那个群里。就算在，她大概也不会多想。因为对她来说，看台上坐着背单词的人多得是，她不会单挑其中一个去分析他的内心活动。她的日常生活里充斥着更紧迫的事：期末排名、化学选修课的考核、物理题做不出来时用修正带涂了一遍又一遍的草稿纸。那些才是占据她注意力的核心对象，而不是一个坐在五米外的男生的视线。她不可能去注意，因为注意一个人需要理由，而她没有理由。

很多年以后程砚说了一句话："她在操场走那三圈的时候，老陆看她的每一个眼神都是单向的。单向得干干净净——没有回应，没有交流，没有任何一个信号是双向的。但这不是悲剧。因为单向也可以是平静的——只要他不把单向当成一种过错。"
