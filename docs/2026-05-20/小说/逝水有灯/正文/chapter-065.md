# 第065章 高考后的水多年后

所属部卷：第二部：六月以后
时间：多年后
地点：旧照片文件夹

## 诗歌插页 P065：原创仿唐绝句

旧照门前日色明，
一瓶凉水认曾经。
少年回望方知晓，
等在树荫也是情。

多年后，陆允川在电脑里找到一张高考结束那天的照片。

照片是母亲拍的，角度有点歪。他站在县一中门口，手里拿着半瓶水，眼睛被太阳照得眯起来。身后全是人，谁也看不清。那时他以为自己记住的是夏知遥在人潮里点头，后来才发现，照片里最清楚的是母亲站在树荫下等他的影子。

他把照片放大，看见自己手里的水瓶。瓶身上的水珠当然早已不在，可他仍能想起那股凉意。那不是爱情给他的回答，是生活给他的照看。

年轻时，他总急着从某个名字那里确认三年的价值。现在他知道，价值也藏在别处：母亲递来的水，程砚发来的消息，班车上的尘风，父亲收拾好的东屋。

他给照片重新命名：六月八日，校门口。

没有写“她”。也没有写“结束”。因为那天并不是只关于她，也不是结束得那么干净。它是很多人共同松了一口气的下午，是一瓶水把他从考场带回人间。
