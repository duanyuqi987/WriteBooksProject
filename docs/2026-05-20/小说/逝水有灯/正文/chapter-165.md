# 第165章 寒风里的台阶多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：女生公寓外台阶

## 诗歌插页 P165：原创仿唐绝句

旧阶风过不留痕，
门禁新声换旧门。
一句以后曾伤夜，
今知平稳怕追问。

多年后陆允川再走到那栋公寓楼外，门禁系统换成了人脸识别。台阶也修过，边角没有当年那么缺。

他站在路边，没有靠近。女学生们抱着快递进出，门开合很快，里面的灯光闪一下又收回去。

旧日记里写：“我只是想知道以后。”现在他在旁边补了一句：她只是想保住当下。

这句补注让他安静了很久。

他曾经觉得苏晚枝不愿给答案，是因为她不够温柔。后来才看见，自己把“以后”问得太勤，每一次都像把一张刚铺平的纸重新揉皱。

离开时，风从楼角吹过来，仍然很冷。他把手放进口袋，摸到一张旧纸的边角，是那年南湖问答留下的三个词。纸已经软了，他没有拿出来。

有些提醒放在口袋里就够了，不必每次都展开给别人看。
