# 第032章 她看到了吗

时间：2011年7月中旬

地点：教室、宿舍

卷次：卷二：窗边人影

暑假快到了。

期末考试——六门考了三天。考场设在教学楼二三层。座位按成绩排——上次考得好的在前面，考得差的在后面。

每场考试顾临风先扫前面——找她的马尾。

她总是坐在前三个考场。他在中间靠后的。隔了大概两个教室的距离。看不到人。只能看考试安排表上那两个字。

但每次考完发卷子的时候他能看到她。卷子从前往后传。她的卷子从前面几排传过来——有人碰了一下，有人翻了一下。传到他手里的时候纸是凉的。

期末考试结束后两天。成绩还没公布。

教室里没什么事。老师懒得讲新课。自习。

顾临风在草稿纸上算分。每一科估了分。数学120左右。语文110左右。英语95。物理76。化学71。生物64。总分估出来在530左右。

他放下笔。

这次的题比期中难。所以绝对分数可能降了。但名次不知道。

他在意的是她的名次——她这次第几名。

他站起来去接水。

接完水回来路过公告栏。成绩单还没贴——白纸空在那里。只有上次期末那张还在。她的名字在第六行。

他看了一秒。然后走回座位。

下午。

班主任进来。贴成绩单。

这次顾临风没有等。直接走过去了。

苏晚晴——第六名。没变。

他自己——第二十三名。退了一名。

他站在成绩单前。没有马上走。把各个科目的分数看了一遍。物理67——比上次跌了5分。化学68——跌了3分。总分472——总体稳定。

看完走回座位。

赵一鸣过去看了。回来坐在顾临风旁边。

"你第二十三。还行。你进步那么多名本来就不是一次性能稳定的。"

"嗯。"

"她第六名。又没动。"

"她挺稳的。"

赵一鸣没再说别的。从桌洞里拿出望远镜——那种迷你的小望远镜。对着窗外看了会儿。

"银杏叶里有鸟窝。"

"嗯。"

晚自习。

顾临风做英语阅读理解。做了三篇。错了五道题。拿出一张新的草稿纸。把错题抄了一遍。

苏晚晴在前面做作业。她没被成绩影响——或者说看不出来。做作业的姿势跟平时一样。后背挺直。马尾安静。

下课铃响。

她收拾书包的速度比平时快了一点。可能是想早点回去。经过后排的时候看了他一眼——目光从脸上扫过去。

一秒钟。

然后走了。

顾临风低下头。翻开日记本。在上面写——不是在宿舍、在晚自习的座位借着日光灯写。

"她这次第六名。我第二十三名。她非常稳定。我进步后回调了一名——但整体还是在往上走。"

翻页。

"成绩单前我每次先看的是她的位置。她好像也注意到了。但我不知道她在意的是我的进步还是别的。不知道。"

"有进步。但还不够好。"

合上本子。放进书包底层——日记本不能被人看到。尤其是赵一鸣。

晚上回到宿舍。

赵一鸣躺在床上看漫画。窗外有人在放周杰伦。

顾临风躺在床上。手枕在脑后。天花板上有一条裂纹。从灯座裂到墙角。寝室里光线暗。裂纹像树的枝条。

他想起上学期第一次考试——她第八名。他三十一。差了二十三名。

现在她第六名。他二十三。差了十七名。

差距缩小了吗。说不清楚。

但他在往前走。

他闭上眼睛。

天花板上的裂纹在眼皮上印的形状——还是那种像树枝、像银杏的。

旁边上铺赵一鸣翻完了一本漫画。

"你睡没。"

"没。"

"想什么呢。"

顾临风睁开眼。

"在想英语阅读理解那道错题。"

赵一鸣没说话。过了一会儿发出一声闷笑。

"鬼才信。"
