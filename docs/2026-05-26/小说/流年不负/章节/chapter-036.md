# 第036章 她的背影

时间：2011年8月中旬

地点：学校

卷次：卷二：窗边人影

八月中旬。补课进入第二周。

顾临风发现自己认得她的背影。

不是刻意去记的。就是看了一年了。每天看。每个课间看。每次放学看。

自然就认得了。

她的背影有几个特征：

马尾的高度——在耳垂的延长线上。不高不低。刚好在脖子和后脑之间。走路的时候马尾会轻微摆动。幅度很小。像钟摆。

右肩比左肩略低——大概半厘米。因为右手写字、右手拿东西、右手翻书。一年的消耗让右侧比左侧稍微沉一些。

书包背在正中间。不像有些人歪一边。小鱼挂件在拉链上。走路时有节奏地晃。步子不大。但频率快。从教室到楼梯口通常走十五步——他数了很多遍。

走姿——脚尖微微内收。从小养成的习惯。踩在地上几乎没有声音。帆布鞋——白色那双，红色那双。换着穿。

顾临风能在一百米外认出她的背影。能在人群里认出来——食堂、操场、走廊。不管她穿什么衣服。不管那天光线怎样。

这让他觉得有点奇怪。但又不是那么奇怪。一年了。三百多天。

每天放学。她走前面。他走后面。隔着大概二十米。不远不近。

出了教学楼走右边的路。经过一块空地。经过图书馆。经过篮球场。然后左拐。到女生宿舍楼下。这条路走了无数遍。每一步都清楚。

她从来没发现。

有时候她跟许若琳一起走——两个人并排。但大多数时候她一个人。一个人走的时候她走路的速度会快一点。不东张西望。盯着前面的路。

顾临风跟在她后面。二十米。

这个距离是刻意保持的。太近了会被发现。太远了过转角会跟丢。二十米——刚好能看到，刚好不会被注意到。

有一次她在路口停下来系鞋带。

他停住了。站在一棵槐树旁边——不是巷子里那棵。是校园路边的。小槐树。树干只有手腕粗。

她系好鞋带站起来。继续走。他等她走了十步才迈步。

赵一鸣有一天问。

"你每天放学怎么都往女生宿舍那边走。男生宿舍在南边。"

"绕路散个步。"

"绕得够远的。"

顾临风没回答。

赵一鸣停了停。

"那个方向——是那谁的方向吧。"

"谁。"

"你说谁。"

赵一鸣没再说。拿着水壶去接水了。

八月中旬的某一天。放学。

顾临风跟在她后面。跟往常一样。二十米。她走着走着忽然在篮球场边停下来。抬头看了看天。

天上没什么。几片薄云。太阳快落到山后面了。

她看了大概十秒钟。然后继续走。

顾临风不知道她在看什么。也许只是累了——补课一下午。也许只是抬头看了一眼天空。也许什么都没想。

但他把那十秒记住了。

她的背影。站在篮球场边。抬头看天。夕阳在她头发上镀了一层金边。

这个画面后来在他的日记里写了两次。第一遍写完觉得不够好。撕了。第二遍只写了四个字——"金色马尾"。

然后他在诗的草稿纸上写了几行。

"暮色从她的肩头落下来/我离她二十米/这二十米我走了三百天/还是没走完。"

写完看了一遍。把"暮色"改成"夕阳"。又把"夕阳"改回"暮色"。

然后把纸夹进日记本里。

晚上回到宿舍。赵一鸣躺在床上翻漫画。忽然冒出一句。

"你是不是怕跟她说话。"

顾临风躺在床上。没说话。

赵一鸣翻了一页漫画。

"不问你了。"

过了一会儿又开口。

"不过跟你说个事——我今天听到许若琳跟人聊天。说苏晚晴物理又没考好——选择题错了一片。她可能想找人辅导。"

顾临风把手从脑后抽出来。

"你在试探我。"

"没有。就是告诉你。"

赵一鸣翻了一页漫画。

"不过吧——你要是想帮人家。先得敢跟人家说话。"
