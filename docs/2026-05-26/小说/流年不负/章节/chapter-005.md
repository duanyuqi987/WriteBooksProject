# 第005章 挂件

时间：2010年10月

地点：高一三班教室

卷次：卷一：初见秋风

十月。天凉了一些。

校门口的梧桐树开始掉叶子。值日生每天早上扫一堆，中午又掉一地。

课间的时候，大家都去走廊晒太阳。秋天的太阳不晒人，暖融融的。有人靠着栏杆聊天，有人趴在窗台上睡觉。

顾临风坐在座位上翻数学课本。

教室前后两个门都开着，穿堂风从前往后吹。他桌上的一张试卷被风掀起来，飘到地上。他弯腰去捡。

就在这时。

浅蓝色从眼角晃过去了。

是苏晚晴。去后门那边扔废纸。

她经过他座位的时候，书包从椅背上垂下来。书包拉链上挂着一个东西——

陶瓷小鱼。黄豆大小。白色的底，上面画了两道蓝线。

书包蹭到了他课桌的桌角。小鱼晃了两下。撞在桌边上，发出一声很轻很脆的响。像两个手指尖轻轻碰了一下。

然后她走过去了，把废纸扔进垃圾篓。转身往回走。

小鱼还在晃。从左晃到右，又从右晃到左。幅度越来越小。最后不动了。

顾临风捡起来的试卷还捏在手里。

他盯着那个鱼看。直到她走回自己的座位。小鱼在她身后晃。

那天下午的课他没怎么注意听。

那个陶瓷小鱼——黄豆大小。小到一般人不注意就看不清。鱼身上的蓝线可能是手工画的，不太直。看起来有些年头了，底部的白色有点泛黄。

不是新东西。

后来几天他特意留意了那个鱼。课间她走动的时候——去讲台交作业、去黑板前面看通知、去找后排的同学借笔——小鱼都会晃。不同的晃法。走路快的时候鱼撞在书包拉链上来回弹，像敲小鼓。走路慢的时候鱼只是轻摇，像水面上浮着的一片叶子。

有一次她在教室门口跟人说话。站在门槛上。风吹过来，书包碰到门框上。小鱼撞出一声脆响——叮。很小的一声。她自己好像没听到。

顾临风听到了。

周六下午没有课。宿舍里只有两个人。另外四个人回家了。赵一鸣也回了——他家在县城，骑自行车十五分钟就到。

顾临风没回去。石桥镇太远，来回车费十几块。

宿舍很安静。对面床铺上，留下来的那个男生戴着耳机听歌，偶尔哼一两句，不成调。

他坐在床上，膝盖撑着下巴。

那个陶瓷小鱼。

小到一般人不会挂。黄豆大小。

挂在书包拉链上。

是路边摊买的的塑料件早就断了、磨坏了、变色了。瓷的不一样。瓷的好好地挂在那里。

他一直想这件事。

想到后来，他翻开数学课本。看了两页。什么都没看进去。

上铺的人翻了个身，床咯吱响了一下。

窗外的天暗下来了。傍晚的云是灰蓝色的。有几只鸟飞过去。

顾临风把书合上。侧身躺下来。脸对着墙。

墙上贴的那张课程表还在。数学、语文、英语、物理。星期五最后一节是班会。

他闭上了眼睛。

然后睁开。

从枕头底下摸出语文课本。翻到封底内页。那两个划了横线的字还在——苏晚晴。

他把书合上，放回枕头底下。这次放得很平。没有折角。

外面的风变大了，吹得窗户框在响。那个留在宿舍的男生摘了耳机，站起来把窗户关上了。扣上窗扣之后回头看了一眼。"要下雨了。"

顾临风嗯了一声。

雨大概八点钟开始下的。打在铁皮屋顶上，声音很大。先是噼里啪啦的一阵，然后慢下来了。雨丝斜打在窗户上，顺着玻璃往下滑。窗外的路灯在雨线里变成了一个模糊的橙色点——像隔着毛玻璃看的烛火。

他躺着听雨。

脑子里是那个黄豆大小的鱼，在晃。

后来雨小了。路灯的光重新变得清楚了一些。铁皮屋顶上还有零星的滴水声。一颗。两颗。然后停了。

顾临风翻了个身。被子拉到肩膀上面。蓝格子被套在暗处看不清颜色。他把脸埋进枕头里。枕头上昨天留的牙印还在——不知道什么时候咬的，大概睡着了咬的。

外面的树在滴水。一滴一滴打在水泥地上。周而复始。
