# 第155章 许若琳
时间：2018年6月
地点：楚州
卷次：卷八：旧诗新读

六月中旬。楚州入梅。雨连着下了好几天。街上的香樟树叶被打得又绿又亮。

许若琳到楚州出差。发微信给苏晚晴——"在楚州。出来喝咖啡。"

苏晚晴回——"好。哪里。"

"你们学校附近有一家。叫'此处停留'。"

"你怎么知道。"

"我搜的。你们学校方圆一公里就这一家咖啡店评分过四星。"

"你连这个都查。"

"必须的。"

周六下午三点。苏晚晴到的时候"此处停留"里面只有两桌人。空调开得很足——推开门一股冷气。跟外面湿热的黄梅天形成鲜明对比。许若琳坐在靠窗的位置。面前放了一杯拿铁——还没喝，奶泡已经塌了一半。

"你来晚了三分钟。"

"下雨。骑不动。"

苏晚晴坐下来。把雨伞靠在椅子边。伞上滴下来的水在木地板上印出几个深色的圆点。服务员过来问喝什么——"美式。热的。"

许若琳抬头看她。看了三秒。没说话。

"你看什么。"

"我看你有没有变。"

"变了没。"

"没变。"许若琳停顿了一下。"但又有点不一样。说不上来。"

苏晚晴拿过桌上的糖包——撕开。往美式里倒了半包。用勺子搅——顺时针搅了三圈。逆时针搅了一圈。然后把勺子拿出来，放在碟子上。碟子上印着咖啡店的logo——一个小孩在读书。线描的那种，很简笔画。

"你和他在一起了。"许若琳说。不是问句。是陈述句。

"嗯。"

"什么时候。"

"五月二十号。"

"五月二十号。"许若琳重复了一遍。然后把咖啡杯端起来——喝了一大口。奶泡沾在上唇上。她没擦。

"你们两个终于。"

终于。这两个字她说得很轻。但她的眼睛不轻——眼眶有一点发潮——不是哭了。就是眼睛变得特别亮。

"许若琳。"

"你先别说话。让我说。"她把杯子放下来。瓷杯碰在瓷碟上，磕了一声脆响。

"你知道我等这一天等了多久吗——从大一开始。每次我告诉你她在图书馆几楼——在三食堂——在礼堂彩排——你那个假装不在意的表情。你装得一点都不像——你知道吗。"

苏晚晴没回答。也没低头。就那样平静地看着许若琳。她的美式还冒着热气。热气碰到空调的冷风——在半空中弯了一下。像一条透明的丝巾被风吹歪了。

"大二那年冬天。你在操场上跑步。顾临风也在跑。你在内圈，他在外圈。他比你多跑了两圈。你以为他没看到你——他当然看到你了。他就是没敢跟你并排跑。"

苏晚晴的手指在杯壁上轻轻转了一圈。瓷杯很烫。指尖碰到杯壁的时候被烫了一下——她缩回来。甩了一下手。

"那次我知道。我看到他了。"

"你看到他你也不打个招呼。"

"我跑得比他慢。他绕到我身后的时候——回头不方便。"

许若琳叹了口气。不是真的叹气——就是把呼吸放重了一点。咖啡勺在杯子里转了两下。然后她把咖啡勺拿出来——放在桌子上。咖啡勺在木桌面上磕了一下。很轻。

"我每次给你透露她的消息。你那个假装不在意的表情。我都看出来了。"

"你看出什么了。"

"看出你耳朵红。跟现在一样。耳尖红的——耳垂也是。我每次说苏晚晴三个字——你耳朵的温度就升高至少一度。"

苏晚晴看着许若琳。嘴角微微动了一下——往上弯了大概两毫米。

"你每次透露他的消息——是不是故意的。"

"当然啊。不然你以为我为什么每次都忽然提他。'哦对了今天在食堂看到顾临风''哦对了顾临风今天上台做了个演讲''哦对了顾临风签了楚州的学校'——你以为这么多'对了'都是不小心想起来的？"

"你每次都是先说别的——然后绕到他身上。"

"是。我会先聊天气、食堂、上课——再慢慢绕。因为他自己不问。我就帮你们中间的墙捅一个小洞。让你从那个洞里看到他在干什么。"

苏晚晴沉默了片刻。热美式已经不烫了。她端起来喝了一口。苦味在舌根上停留了两秒——然后化开。

"你大一的时候。有一次在电话里跟我说——你们系有个男生追你。"

"对。我说我不要。我说我心里有人。"

"我以为你说的是——"

"说的是顾临风。不是他追我——是我一直觉得。我跟顾临风中间隔的不过是一层纸。但这层纸要你们自己去捅。我不帮你们捅——我只负责让你们知道对方还在。"许若琳说这些的时候没有看苏晚晴。她看着窗外的雨。雨帘在玻璃上画出一道道水痕。水痕从左上角歪歪扭扭地流到右下角。

"大一那年——你告诉我的第一件关于苏晚晴的事是什么。"许若琳收回目光。

"高中时候坐我前排的那个女生。头发特别多。扎马尾的时候要用两圈皮筋——因为一圈绑不住。"

许若琳嘴角一歪。然后笑了一下。

"你们俩。一个在教室这边看马尾巴。一个在教室那边看蓝色笔记本。一个故意绕远路经过走廊。一个故意在窗户上反光看走廊。三年的时间就这么被你们看来看去——看到毕业谁也没开口。然后又是六年。两个青阳跑到楚州。一个去了楚州一小当小学老师。一个去了楚州师范附中教物理。两条线同时往楚州跑——你说这是巧合？楚州有什么好的——除了月湖——除了你们两个人。"

苏晚晴把杯子放在碟子上。杯底在碟面上磕出一小圈水渍。

"不是巧合。"她说。"我知道他在楚州。他签了合同之后许若琳你就告诉我了。"

"对——我当时说的是'啊对了——顾临风签了你们楚州的那个师范附中'。我说得特别自然。"许若琳顿了顿。嘴角弯起来的弧度大了一点。

外面雨小了。玻璃上的水痕流到一半——不再往下淌。

许若琳收住笑。看着苏晚晴的眼睛。沉默了一会儿。

"苏晚晴。"

"嗯。"

"我在你们两个人的世界里——一直扮演了一个传话的角色。但我一点都不介意。因为我知道。你们两个人之间的墙——不用别人拆。自己会倒。我就是站在墙外边——往墙这边递话。往墙那边递话。递了八年。"

苏晚晴把手伸过桌子。在许若琳的手背上轻轻拍了一下。动作很轻——跟那天在月湖亭子外她向顾临风挥手一样。手心碰到手背——一秒半。然后收回去。

"许若琳。"

"嗯。"

"他跟我说的第一句关于你的事。是大二那年——赵一鸣告诉顾临风——许若琳今天在图书馆跟苏晚晴坐同桌。然后他回了两个字——'好的'。赵一鸣说——就'好的'？你知道顾临风回什么吗——'她旁边的位置。我想坐。但她的旁边有你。够了。有人陪她就可以了。'

许若琳不说话。指尖在咖啡杯边上停了很久。然后她把勺子拿起来——又放下去。勺子在桌子上滚了半圈。

"你们两个人。"她看着苏晚晴。"一样的傻。"

窗外的雨停了。玻璃上的水痕开始干。边缘变成浅浅的白。六月的太阳从云缝里挤出一道窄窄的光。光落在许若琳的咖啡杯上。杯里的奶泡已经全塌了。

"我买单。"许若琳站起来。

"我来吧——"

"不许动。你们欠我的。欠了我八年。这杯咖啡——我来。"然后她拿起手机扫码。手机壳是浅蓝色的——跟苏晚晴高中时候绑马尾的发圈颜色一模一样。

店门推开。外面的空气又湿又热。苏晚晴撑开伞——虽然雨停了，伞还是习惯性地撑开了。她忽然想到了什么——从包里掏出手机。发了一条微信。

给许若琳的：那个蓝色的发圈。你是不是也有一个。

许若琳秒回：我买的。跟你同款。你高二那年我买了两个。一个自己戴。一个放在包里。准备万一你的丢了——我给你。

苏晚晴抬头。许若琳已经走到街对面了。她没有回头看。马尾在六月的风里晃了一下。

苏晚晴把手机放回口袋。伞收了——用手臂夹着。水滴在裤脚上——不在意了。

她站在咖啡店门口。站了一会儿。然后沿着街道慢慢走回去。路面上还湿着。水洼里映着云。云在水里——白白的。往东飘。
