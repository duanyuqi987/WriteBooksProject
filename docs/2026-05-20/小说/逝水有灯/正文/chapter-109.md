# 第109章 封校的周末旁观

所属部卷：第二部：六月以后
时间：大一秋天
地点：大学南门和围墙边

## 诗歌插页 P109：原创现代诗

她把社刊拆成页码，
把错字圈进红框。
有人在旁边认真递纸，
认真得像递出自己。
她感谢这份认真，
也悄悄给它留了边界。

苏晚枝对那次取样张的印象，主要是累。封门让所有事绕远，打印店又催她快点确认。她在活动室里坐下时，手臂酸得抬不起来，叶澄递给她一杯温水。

“新来的那个男生挺帮忙。”叶澄说。

“嗯，陆允川。”苏晚枝低头看名单，“校对也认真。”

“你记名字挺快。”

“登记册我写的。”

她说完才笑。陆允川确实细心，捡纸时会按页码顺序放好，递红笔时笔尖朝自己，怕戳到别人。这样的细节让人舒服，却也让苏晚枝有一点警觉。太细的照顾如果没有停顿，容易让接受的人不知该怎样回礼。

她不是不喜欢和他相处。社团里愿意踏实做事的人不多，陆允川算一个。可她希望这份踏实留在事情本身，而不是很快被拉到别的地方。

晚上群里收到他的表格，她回“收到，谢谢”。打完后，她看着输入框停了一下，没有再加表情。不是故意冷淡，只是她知道，有些关系从第一步就要把距离放清楚。

叶澄在旁边翻样张，忽然说：“你又开始谨慎了。”

苏晚枝把红笔盖上，“谨慎省事。”

她说得轻，但那是经验。温柔不是随便开门。她愿意感谢陆允川的帮忙，却不愿让感谢变成一条他可以一直走近的路。
