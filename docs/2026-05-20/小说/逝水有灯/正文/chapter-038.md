# 第038章 成绩单下滑之后

所属部卷：第一部：初秋三楼
时间：高二期中后
地点：男生宿舍和自习室

## 诗歌插页 P038：原创仿宋小令

榜上名，纸上惊，
十二阶低到夜深。
问少年、几番心动，几番荒情。
灯下错题翻又改，
才知喜欢也须清。

成绩下滑后，陆允川有几天没去操场看台。

晚饭后，他直接回教室。别人还在食堂排队，他已经把物理错题本摊开。题目看得很慢，公式也总是套错。以前他遇到不会的题，会忍不住往前排看，好像看一眼夏知遥就能缓一口气。现在他强迫自己低头。

程砚坐在旁边陪他补题。说是陪，其实自己也有一堆错题要改。两个人一人占半张桌子，中间放着热水杯和一块快用完的橡皮。

“你这里方向反了。”程砚指着他的推导。

陆允川看了看，叹气：“又反了。”

“你最近什么都反。”程砚说，“看人看反，学习也看反。”

陆允川抬头瞪他，最后还是低下头改。程砚的话不温柔，却能把人从自怜里拽出来。自怜最舒服，因为它让人觉得自己很深情；改错题最不舒服，因为它只问你会不会。

周末母亲打电话来，问成绩。陆允川握着手机，沉默了几秒，如实说掉了。电话那边安静下来。母亲没有骂，只说：“那就补，别慌。”

这三个字比责备更让他难受。他嗯了一声，挂断电话后在走廊站了很久。

那天晚上，他把硬皮本拿出来，翻到写雨伞的那一页。看了几行，又合上。他不是不想写，只是忽然明白，写不能代替解决。诗可以安放情绪，却不能替他把物理题做对，不能替他把落下的十二名追回来。

于是他重新打开错题本，把第一道题从头写了一遍。写到最后，窗外教学楼的灯一盏盏暗下去。陆允川没有觉得自己变好了，只觉得终于没有继续往下滑。
