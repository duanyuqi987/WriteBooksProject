# 第188章 新手父母

时间：2021年5月-2022年5月
地点：楚州
卷次：卷十：不负少年

第一个月。

凌晨两点。女儿哭了。

哭声像猫叫——尖尖的，细细的。在安静的夜里格外扎耳朵。

顾临风从床上弹起来。脚踩到地上的拖鞋——左脚找到了，右脚的光脚板踩在了地板上的积木上面。积木硌了一下。他没管。

奶粉罐在厨房台面上。半夜的光线很差。他眯着眼睛看罐子上的说明——"一平勺配三十毫升水"。他用手机照着，把勺子里的奶粉刮了两遍——第一遍太少了，第二遍觉得多了又倒回去一点。

开水。倒了半杯。又加了一点凉水——热水瓶旁边放着一个量杯，他用量杯兑了三次。太烫。加凉水。太凉了。又加热水。反复了三回。

手腕上试了一下温度——不烫了。

他把奶瓶拿进卧室。

苏晚晴已经把女儿抱起来了。女儿靠在她的肩膀上。哭得小脸通红。嘴巴张着——下嘴唇翻出来。

"好了。好了。"

她把奶嘴塞进去。女儿吸了两口，不吸了。还是哭。苏晚晴把奶瓶拿出来，看了看奶嘴的孔——没堵。又把奶嘴放回去。女儿吸了一口，又吐出来。

"是不是尿了。"

她把手探进尿不湿里。是湿的。

他找来一片新尿不湿。拆开。展开。尿不湿很大，女儿屁股很小。他把女儿放在床上，把尿不湿垫在屁股下面。贴胶带的时候——手忙脚乱。左边先贴上。右边拉过去——短了。

"贴反了。"

她靠过来。把尿不湿转了一下——前后反了。她把尿不湿拆下来重新垫——这次对了。胶带贴上。

"你看这里——"她指着尿不湿的前面。"有个图案。有小花在前面。"

"知道了。"

第二天晚上。他一个人换。

又反了。

她把小花图案那面往前转过来。"小花在前面。记住了。"

他记住了。小花在前面。

三个月。夜里哭的次数少了。偶尔还是会哭。

女儿哭的时候两个人同时从床上坐起来。他往左边翻身——她去右边找奶瓶。他抱起女儿在客厅里走圈——一圈、两圈、三圈。女儿趴在他肩膀上，小脑袋一颠一颠的。有时候走着走着哭停了。有时候不停。

他又开始走了。经过电视柜。经过书架。经过茶几。又经过书架。他低头看了一眼书架上那本深绿色书脊的词典。词典旁边是两本浅蓝色封面的笔记本。

女儿趴在他肩膀上。小手抓着他的T恤领子。口水弄湿了他的锁骨。

他在书架前停了停。低头。嘴唇碰到女儿的脑门——很轻。一嘴的奶香。

六个月。女儿会翻身了。放在床上，一秒钟的功夫就能从仰卧翻成俯卧，然后把头抬起来，像一个侦察兵。她抬着头眼睛到处看。看到妈妈——笑了。嘴张着，口水顺着下巴滴到床单上。

苏晚晴拿纸巾给她擦。擦完口水。又流出来了。再擦。女儿咯咯笑——她觉得擦口水是个游戏。

他开始一只手就能换尿不湿了。

把女儿放在床上。一只手按着她的脚踝，一只手把尿不湿抽出来。新的尿不湿垫进去——小花在前面。胶带一左一右贴上。整个过程不到一分钟。

有一天晚上。换完尿不湿。女儿还醒着。躺在床上看着他。眼睛很亮——黑葡萄似的。他看着她。然后伸手——把自己的食指放进她的小手里。

她攥住了。

攥得很紧。

他看着自己的手指和她的小手。看了一会儿。

然后低头。用额头碰了一下她的额头。

"念念。"

她没回他。眼睛眨了眨。然后笑了。

一周岁。女儿可以扶着沙发走路了。走一步，腿软一下——膝盖弯了——他就伸手给她扶住。她再站起来。再走一步。又软了。但嘴角一直翘着——不气馁。

她第一个会叫的词不是"爸爸"也不是"妈妈"。

是"灯"。

她指着天花板上的灯。"灯。"

然后指着台灯。"灯。"

然后指着窗外路灯。"灯。"

苏晚晴说这孩子以后眼睛好。专门看亮的东西。

有一天晚上。苏晚晴在改作业。红笔在学生的本子上打着一个个对勾。女儿在旁边的小垫子上爬。爬到书架前面——停下来。仰头看着书架。

伸手指着那本深绿色书脊的词典。

"书。"

她说了第二个词。

苏晚晴抬起头。看着女儿。看着书架。

"那是爸爸的书。"

"书。"

"对。书。爸爸写了很多字的书。"

"书。爸爸。"

顾临风从厨房里探出头。手里还在擦碗。

苏晚晴说:"你女儿会叫爸爸了。"

他站在厨房门口。手里攥着洗碗布。水从布角滴到地上——嗒、嗒、嗒。

女儿又指着灯:"灯。"

然后指着书:"爸爸。"

他走过去。蹲在女儿面前。用没有拿洗碗布的那只手摸了摸女儿的头发——头发长了很多。软软的。有点卷。

然后他站起来。去厨房把碗擦完。

水龙头开着。他低着头。洗了一个碗。又洗了一个。然后抬手抹了一下脸。分不清是水花溅的，还是别的。
