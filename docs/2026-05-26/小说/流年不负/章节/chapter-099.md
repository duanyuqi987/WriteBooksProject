# 第099章 分手

时间：2015年1月

地点：楚州师院

卷次：卷五：各自南北

一月十五号。期末考试完的第一天。楚州下了一场雪——不大，薄薄一层白，盖在梧桐树光秃秃的枝干上。

周雨发来消息——"今天下午有空吗。老地方。"

老地方是图书馆门口的台阶上。他们第一次说话的地方。

他到了。她已经在台阶上站着。穿了一件深灰色的大衣。围巾是红色的。她手里拿着那本诗集——海子的《活在珍贵的人间》。旧旧的，扉页上被他写过"生日快乐"的那本。

他走过去。站在她面前。

她抬起眼睛。没有哭。眼睛下面那几颗雀斑在冬日的冷光里比平时深了一点。

"顾临风——我们分手吧。"

他看着她。没有接话。等她说理由。

"你心里有别人。"

她把诗集递给他。他没有接。

"跟你在一起的时候你像在出神。我不知道你在想什么——但我知道你想的不是我。"

她的声音很稳。没有拔高。没有颤。她说的每一个字都像是准备了很久的话——在脑子里练习了很多遍。

"我一开始以为你只是慢热。三个月了——你还是很慢。慢到我都能猜到你出神的时候在想谁。"

他没有反驳。

"对不起。"他说。

"你不用对不起。你没有做错什么。你对我挺好的。生日礼物、陪我自习、陪我逛操场。"她把诗集放在他面前的台阶上。"但你的心不在。"

他站在原地。周雨退了一步。把围巾往上拽了拽——盖住了下巴。她被冻得有点发红的皮肤上，有很小的白色呼息。

"我想过跟你说——'你改一改'——但这不是'改'的事。你改不了。你根本就没打算改。"

"对不起。"他又说了一次。

"我走了。"

她转过身。走下台阶。靴子踩在薄雪上发出轻的咯吱声。走了一半——停了一下。她背后说他心里有别人——那个人大概也在这个城市。她没有回头看。

他捡起那本诗集。书壳被雪水打湿了一小块。扉页上他写的那行字被雪水洇开了一点——"生日快乐"四个字被墨水洇成模糊的蓝色。

他把诗集夹进书包里。

周雨的背影在图书馆前面的梧桐大道上越走越小。红色的围巾在灰白色背景里一点一点地变淡。最后她拐进了教学楼的走廊那头。不见了。

他站在台阶上。雪落在肩上。又落在诗集封面。他手里握着那本书——很凉。

然后他走回了宿舍。

512里面只有他一个人。其他人都还没回来。阿坤的电脑屏幕黑着。李博的书架码得很整齐——书脊朝外。小伍的床头贴了一张家里寄来的照片——四个人站在一栋砖楼前面，他妈、他爸、他妹、他家的狗。他的床铺还是跟以前一样——蓝色格子被、灰色的床单、枕头下面压着一部手机。

他坐在椅子上。坐了大概半个小时。没有开灯。窗外的天慢慢变暗——从灰色变成深灰色。

他坐了很久。久到楼道的声控灯自己亮了起来——有人从楼梯口跑过来，脚步声啪啪地响。然后是开门声——隔壁宿舍的。门又关上了。安静了。

他把诗集拿出来。翻了翻。周雨在最后一页用铅笔写了什么东西——他之前没有看到。字很小。

"我知道你写了很多诗。但不给我看的那些——才是真的。再见。"

铅笔字很轻。好像写的时候怕用力太大把纸划破了。

他把书放在桌上。起身去洗脸。走廊尽头的水池还有牙膏泡沫——跟去年一样。镜子里的自己没有表情。眼皮微微有点肿——可能是冻的。水龙头打开。冷水。往脸上泼了几把。

擦脸的时候碰到了刚才周雨放诗集的那个位置——左胸口。棉袄的布是凉的。

他回了宿舍。把灯打开。阿坤回来了。"今天怎么这么早。"

"嗯。"

阿坤看了一眼他的脸。然后没有问。
