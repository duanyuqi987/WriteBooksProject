# 第126章 启程

时间：2012年7月5日，清晨六点

地点：江城火车站候车室

卷次：第六卷：雪落无声

七月五号，天还没亮透。

林玉起到得最早。候车室里人不多，头顶的吊扇转得很慢，把灯光搅成一圈一圈的。他把书包放在长椅上，拿出手机看了一眼——六点零三分。火车七点十分开。他来得太早了。

带队的是教育学院的陈维中老师，四十出头，戴一副银框眼镜，拎着一个深蓝色旅行袋，第二个到。他看见林玉起，点了点头："中文系的林玉起？"

"陈老师好。"

"你到得真早。"

林玉起笑了笑，没说他是四点半就醒了。

陆续有人来。教育学院的赵一琳、孟小荷、沈佳慧；文学院的孙亦舟；数学系的方远。加上林玉起和苏安晴，一共八个人，去太行山里的李家沟小学，为期四周。

苏安晴是六点四十到的。

她背着一个很大的登山包，鼓鼓囊囊的，包带把肩膀勒出两道印子。头发扎起来了，穿一件浅灰色T恤和深蓝色运动裤，脚上是双半旧的白色运动鞋。

她在候车室门口站了片刻，扫了一圈，看到林玉起，朝他走过来。

"你到得早。"

"嗯。"

她把大包卸下来，放在林玉起旁边的椅子上。"里面装了书。给孩子们带的。"

林玉起看了一眼那个包，至少二十斤。

"我来拿吧。"

"不用，"苏安晴说，"自己的东西自己拿。"

她的语气很平常，但林玉起注意到了——她在说这句话的时候没有看他，而是在看墙上的列车时刻表。

陈维中开始点名。七点整，八个人排队检票进站。

站台上晨风很凉。苏安晴走在林玉起前面，登山包比她的肩膀宽出一截，走起路来一晃一晃的。林玉起看着她后脑勺上被包带压散的那一缕碎发，什么都没说。

火车进站，汽笛声很长。

>
山路不知远，
少年各自忙。
此去非为名，
但求不负光。
>
