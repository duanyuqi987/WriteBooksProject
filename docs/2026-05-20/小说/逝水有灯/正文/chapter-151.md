# 第151章 南湖问答之前

所属部卷：第三部：寒风里的花
时间：七月初
地点：南湖河边

## 诗歌插页 P151：原创现代诗

长椅掉了一块漆，
露出里面暗色的木头。
他把问题换成别人的名字，
湖水没有拆穿，
只是把路灯拖得很长。

食堂争执后的第二天，陆允川去南湖找叶澄。期末周的湖边人少，长椅上晒着几张社团海报，边角被风卷起。

叶澄正在背英语单词，水杯压着活动报名表。她看见他过来，把书合上。

“你找我？”她问。

陆允川坐到旁边，隔着半个人的距离。他没有立刻说苏晚枝，只问：“如果一个人做错了事，后来想改，别人是不是应该给机会？”

叶澄低头看了一眼长椅掉漆的扶手，“你说的是你，还是谁？”

陆允川沉默。

“你要问就直接问。”叶澄说，“别把自己藏进题目里。”

湖边有学生骑车经过，车铃响了一声。陆允川把手放在膝盖上，指节捏得发白。他想问苏晚枝是不是太狠，想问自己是不是已经没有机会，又怕叶澄给出他不想听的答案。

叶澄没有催他。她把报名表翻过来，用背面空白处写下三个词：道歉、改变、等待。

“你先想清楚，”她说，“你到底要哪一个。”
