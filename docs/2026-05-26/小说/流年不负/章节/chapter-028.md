# 第028章 雨天

时间：2011年6月中旬

地点：教学楼门口

卷次：卷二：窗边人影

下午最后一节自习课。窗外开始打雷。

先是闷的——远处——在天那边滚——声音——"轰——"像大石板在铁轨上推——然后是近的——近雷——在头顶裂开——"咔——"脆——把走廊上声控灯劈闪了一下——亮了——停了。风把银杏叶吹得翻过来——叶背是灰绿的——树冠往一边倒——往北——风很猛——树枝——弯到极限——抖着。天空从灰白变成深灰——像把黄昏提前了一个多钟头——然后豆大的雨点砸下来——打在窗玻璃上啪啪响——不是先小雨再大——是直接就大的那种——噼里啪啦——砸在玻璃上——水花——一滴接一滴——盖住了窗外的银杏。

教室里有人"哇"了一声——后排的那个——坐在靠暖气片那边——喊——"下暴雨了——"然后站起来——去关窗——手伸到窗外——往回拉——窗上的铝合金滑槽被雨打湿——滑——关不上——又推了一把——"啪"——关上了。苏晚晴站起来——面前的窗还开着——她倾身——手指——推窗框——雨水已经糊了半扇玻璃——她的手收回时手上带着些水——甩——几滴——落在地上——窗关上——外面的雨声从"砰——"变成更沉的"啵——啵——啵——"像隔着水听。窗户玻璃上——水——往下流——一道道的——把外面那个银杏变成了水下的影子——模糊的绿。

放学铃响——雨没有停的意思。走廊里挤满了没带伞的人——从教室推门出去——站在门廊下——走廊——手挡着额头——看天——雨还是那么大——像天上有人把洗衣盆推翻了一样。有人在打电话——"你把我伞带下来——在教学楼——对——三楼——"那边说了什么——"我说了在教学楼——不是——宿舍——没有——"声音在雨里飘。有人等不及——把校服外套蒙在头上——往外冲——狠狠踩在水洼里——"啪——"——水溅到裤腿上——"靠——"——继续跑——跑向宿舍楼——经过操场——人影在雨幕里变得又小又模糊——脚步声——被雨吞掉了。

顾临风带了一把伞——灰色折叠伞——三折——旧了——伞面是最外层的那层面料——雨水打久了有点漏水——针脚处——有细缝。伞骨有一根弯的——是被他爸坐上去弄的——那次下雨——他爸从镇上回来——坐在石凳上——伞在屁股下——忘了——起来——"嘎"——弯了——还能用——撑起来——歪了一点——向右偏——伞面往右侧倾——下雨的时候——那边空间多一点——左边空间少一点。他把伞拿在手里——手指——捏着伞柄——伞柄是塑料的——发黄——上面有凹凸的人字形纹——捏着——有点滑——手心汗——因为——。走到教学楼门口。门廊不大——水泥地上已经湿透了——被人鞋底踩得到处是泥水——黑色的脚印——重叠着——没规律的。门口站了五六个人——都是没带伞——在等雨停——认识的不认识的——挤在一起——有人说"这雨——半小时内不会停——"另一个人说"我手机快没电了——"

苏晚晴站在最边上——背靠着门廊右侧的墙——瓷砖——白色的——有条裂缝——她没靠着裂缝——书包抱在怀里——两只手——搂着——手指——抓在书包肩带上——那处布——洗久了——起了一点毛。陶瓷小鱼被书包挡住了——看不到。她的目光——落在雨里——看着雨丝——从天到地——白茫茫的——从屋檐下来的水——连成一道水帘——喷在地上——水花有一拃高——水花外面——是更密的雨。头发——马尾——有几根碎发——被潮气打湿——贴在颈后——她没有去拢——可能忘了——可能在等谁。

她没有伞。

顾临风站在她身后——三步远。从教室出来——经过走廊——和那些人一起——然后停在这里。伞——握在手里。手心——汗——指关节——在伞柄上——能看到骨节的轮廓——一张一合的肌肉——绷紧——松开——绷紧——松开——循环着。她在看雨——没有注意到身后。

他心里——想——把伞给她——走过去——递给她——说"你用吧"——然后——转身跑。这件事——从逻辑上讲——极简单——往前走两步——抬起手——把伞递出去。但脚——好像被钉在地上——脚尖——在鞋里——缩了一下——脚趾——弯——前掌——地面——腿部的肌肉——收——但它不动。

一个男生经过——看了看门口——看了苏晚晴一眼——不是要递伞——是他也挤在门口——看了一眼——然后决定不等——拿外套——包在头顶——"嘭——"冲出去了——水花踩了——一个——两个——溅起的水——喷在旁边人的裤腿上——"你看着点——"没人理。

时间——一秒一秒地过。屋檐的水——滴——滴——一滴打在地面——碎。顾临风的指关节——发白——握伞握得太紧了。雨从屋檐上砸下来——砸在地上——砸在台阶上——门前的水泥地——冲刷——原本的沙尘被雨水洗得干干净净——只有新落上去的溅水印和一个一个的水坑反光的涟漪。他身上没淋到雨——站在门廊顶下——雨水离他肩膀就差半米——但他额头上有汗——不是热——是——闷——从皮肤底下渗出的——微潮——用手背擦——擦完又有。

五点半。

一个身影穿过雨幕跑过来——打着红伞——红色——是折叠伞——能看到伞下面——那个——短直发——戴银色椭圆眼镜——许若琳——跑得有点急——伞往一边歪——衣服袖子湿了一块——鞋——白色运动鞋——已经被水浸灰了——鞋头——踩出水——她腋下夹着另一把伞——蓝色格子——长柄——不是折叠——撑开——到脚——柄是弯头的——木制的——应该是苏晚晴的——可能是借给她——或是帮她拿着的。许若琳在门口收了红伞——甩了甩伞上的水——呼——"你怎么没带——我就知道——给你——我刚去你宿舍找——在床下的篮子里找到了——钥匙都给你——"她说着从口袋里摸出一个小钥匙串——银色圈——递过去。

苏晚晴接过伞。"谢谢——"声音——不大——但比平时——略高了一点——"谢"字的尾音有点急——像是感谢——也像是松了一口气——肩膀——微微下去了一些——从靠墙——改为站直。许若琳甩了甩头上的雨水——"走吧——快——食堂今天周三——炖鸡腿——晚了就没了——剩下全是鸡骨头——"她把小红伞抖开——撑起——"快点——"苏晚晴撑起那把蓝格子伞——长柄——手往下——一拉——伞面——张满——伞骨——完好——没有弯。然后两个人走进雨里——许若琳的红伞——苏晚晴的蓝格子伞——并排——在灰茫茫的雨中——两个彩色的圆——越走越小——在小路转弯处——被雨幕吞了——看不见了——脚步声被雨声盖掉——只有伞——最后的两个颜色——在远处——模糊——淡——消失。

顾临风站在原地。手里那把歪伞——还没撑。他终于——把伞撑开——"砰"——伞面——嘭——布面的细孔——雨声打在上面——闷——不乱——但伞歪了——水从弯的那根伞骨滴下来——沿伞面斜线——聚在那根弯骨最弯处——一滴——一滴——滴在地砖上——拍——拍——水花——比外面的小——小——小——然后滴在自己左肩上——又一颗——两颗——把他的左袖子洇成深灰色——棉布——湿了之后贴着皮肤——凉。他自己撑着——一步一步——走向宿舍楼方向——踩在水洼——鞋底——水——咕啾——每一步——都在吸水——鞋里——袜子——湿了——脚趾发凉。

到宿舍的时候——左半边衣服全湿了——肩膀——袖子——裤腿——从上往下——左侧贴着身体——水在鞋底——在走廊留下湿的鞋印。

赵一鸣在床上看漫画——阿衰——看到顾临风进来——抬头——漫画放下——手指夹在书页里压着。"你不是带伞了吗——"

"嗯。"

"那衣服怎么湿成这样——"

"伞有点歪——"顾临风脱下湿衣服——前胸——后背是干的——左边是湿的——他单手——从头上脱下——拉链——把衣服——拎在手里——挂在床边的铁管上——管上已经晾了昨天洗的秋衣——还在——半干不干。

赵一鸣看了看那把伞——靠在门边——没合——伞面还是张开着的——那根弯的伞骨——很明显——向右歪了大概六七度——其他几根是直的——就它——弯——像受了伤的什么——歪在一边。赵一鸣说:"那根伞骨弯了多久了——你买个新的不行吗——"

"还能用。"

赵一鸣从床头抽了一条干毛巾——甩过去——毛巾罩在顾临风头上——"擦擦头发——"顾临风接住毛巾——按在头上——搓——发梢的水——吸走。

晚上。日记本——手电筒——那束橙黄的光——照在纸上——在床头——他趴着——写字。"她没带伞——我带了——有一把伞——但是歪的——在门口站着——我在她身后——三步——很近——可以——但是我没有——许若琳去了——红伞——蓝格子——她带了两把——给她——她们一起走了——我就——还站在那里——然后回来——左边衣服湿——因为伞歪——但不是伞的关系。"

停笔。窗外的雨小了些——沙沙——沙沙——比刚才细——比刚才柔——打在水洼——涟漪——玻璃上——水痕——几道——弯着——往下滑——细细地。他继续写——"我应该把伞给她的——下次——"写到"下次"——停顿——把"下次"涂掉了——圆珠笔——在"下次"两个字上来回——黑墨迹——盖住了字本身——然后写——"下一次——我还是——"写完划掉——又写——"不——没有那么多次——这样的小事——也许——"这句话没写完——也划掉了——最后——只有一句——"下回。不管怎样。递过去就行。"合上本子——关了手电筒——雨声灌进来——上铺赵一鸣翻了个身——床——咯——咯——两下——停了。天花板上——今晚没有那道月光——全被雨云盖住了——只剩路灯昏昏的光影透过窗帘——在地砖上模糊着。想:下次她没带伞的话——一定递过去——不说"你用吧"——就说"伞给你——用不用随便——"——或者——直接放到她旁边的墙上——然后走——什么也不说。然后翻了个身——被窝——暖——枕头——湿的那点——太阳穴那里——凉——还没干透。雨还在下。
