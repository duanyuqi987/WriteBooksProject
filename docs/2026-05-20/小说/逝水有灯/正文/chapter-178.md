# 第178章 朋友的烟之后

所属部卷：第四部：失去的美丽
时间：大二冬末
地点：宿舍楼后门

## 诗歌插页 P178：原创仿宋小令

烟灭后门寒。
一句闭嘴，重过万千言。
错处无人能代领，
天明方可递清单。

第二天上午，陆允川把昨夜那条草稿改了十几遍。最后留下的只有两行：

“昨晚那句话是责怪，不是事实。对不起。你不用回复。”

他发出去后，把手机放进抽屉。祁野坐在床边盯着他，像看一个危险物品终于被放进柜子。

“别拿出来。”祁野说。

“我知道。”

“你不知道，所以我看着你。”

陆允川没有反驳。整整一上午，手机没有响。中午去食堂时，他下意识想回宿舍看，被祁野拎着书包带往前拽。

“吃饭。”祁野说。

食堂里人很多，窗口排队很长。陆允川端着餐盘，第一次没有把没等到回复这件事立刻放大成判决。他只是觉得难受。难受在胸口，不体面，也不浪漫，却可以暂时由他自己拿着。

下午，苏晚枝回了两个字：“收到。”

没有没关系，没有继续谈，也没有安慰。陆允川看着那两个字，心里空了一下。然后他把手机扣回桌上。

祁野瞥了一眼，“这就够了。”

陆允川点头。是的，这就够了。道歉能被看见，已经是对方愿意给出的最大回应。
