# 第037章 成绩单下滑

所属部卷：第一部：初秋三楼
时间：高二期中后
地点：班级后墙成绩栏

## 诗歌插页 P037：原创仿唐绝句

风影微明照旧尘，
少年心事隔新春。
回看不见当时路，
一角书声认故人。

事情真正发生时，并没有陆允川想象得那么郑重。陆允川因为分心掉了十几名，他盯着夏知遥稳定的名字，第一次觉得喜欢也会让人羞愧。

期中成绩贴出来那天，后墙前比公告栏还挤。红笔圈出的排名像一个个小伤口，大家嘴上说随便看看，脚却都不肯走。

陆允川从上往下找自己的名字，找到时喉咙发紧。他又往上看，夏知遥的名字稳稳停在那里，像一根没有被风吹动的线。他越想装得平常，动作越显得多余。手指碰到纸边、桌角、书脊，哪一样都比他的表情诚实。

夏知遥只看了一眼成绩单，就回座位整理错题。她没有炫耀，也没有安慰谁，手里的红笔一下一下，把错误改得很清楚。

“掉了就补，别站这儿给成绩单守灵。”程砚说。“我知道。”陆允川说。“你不知道，你现在脸上写着天塌了。”程砚说。

程砚把他拉到座位上，翻出物理卷子，说从这道题开始，别把喜欢当借口，也别把借口写得像喜欢。

他当然知道程砚是在提醒他。可人在心事最热的时候，很难分清提醒和打断，只觉得旁人都站在一条更凉的岸上。

叶澄路过后墙，看到陆允川还站着，就把一张草稿纸递给他。纸上写着几道同类型题，她说，先会一道是一道。

班级仍按自己的速度往前走。老师催交作业，值日生擦黑板，后门口有人借校牌，有人因为饭卡找不到翻遍整个书包。没有哪一件事为陆允川停下来。

也正因为没有停，这一刻才显得可信。成绩栏上被红笔圈出的名次被他看见，被他放大，又被日常一点点压回原来的大小。

晚自习的灯很亮，亮得让人无法躲。陆允川把错题本摊开，第一道题抄了三遍，笔尖在纸上戳出一点毛边。

陆允川那时还不会把喜欢说成一句完整的话。他只会绕路、停顿、假装翻书，像很多笨拙的少年一样，把靠近做成一串别人未必能读懂的小动作。

后墙下的垃圾桶里塞满废纸，几张揉皱的卷子露在外面。有人假装开玩笑说完了完了，声音却发虚。陆允川听着那些声音，忽然不想再把失控说得很浪漫。

成绩单贴出来后，班主任没有立刻批评，只让大家把卷子拿出来订正。可越是这样，陆允川越坐不住，仿佛那张纸还贴在后墙上，隔着一整个教室看他。

夏知遥在前排问老师一道化学题，问完后把步骤重新写了一遍。她的稳定不是天生的，陆允川看见她草稿纸上密密麻麻的计算，才知道自己曾经把别人的努力看得太轻。

程砚把他的错题本翻到空白页，说先别谈心，谈心不能提分。陆允川想反驳，程砚已经把第一道题抄了上去，笔画很重，像把他从发呆里敲醒。

晚自习下课后，陆允川没有立刻走。他把后墙那张成绩单看了一遍，又把视线收回来。名字仍在那里，名次也仍在那里，可他终于愿意从第一道错题开始，而不是从一场情绪开始。

这一刻真正有力量的，不是心跳有多响，而是它嵌在一堆普通事情里。有人催他让路，有人问他借笔，有人把椅子碰到他膝盖；世界越不配合，他越意识到自己不能把心事演得太满。

夏知遥没有给他特殊回应，也没有刻意冷淡。她只是按照自己的节奏把事情做完。这个事实在当时让陆允川失落，许多年后却让他松了一口气，因为真实的人本来就不该只负责别人的想象。

真正撞上这一刻时，陆允川反而没有准备好的台词。他只听见自己衣料摩擦书包的声音，还有程砚在旁边压低的一声笑。

夏知遥的反应很平常，平常到几乎让他失望。后来他才懂，这种平常不是冷淡，而是她没有被他的想象绑架。

陆允川把错题本拿在手里，先把物理卷订正完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把卷子订正完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把一道化学题重新列了三遍步骤，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

班级后墙里最明显的不是心跳，而是红笔盖被按上的咔哒声。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，晚饭里凉掉的米饭把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。错题本、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

真正发生的那一刻，陆允川没有准备好的台词。他只来得及把错题本攥紧，又很快松开，怕自己连手上的动作都显得用力。

夏知遥没有给他特殊回应，也没有故意回避。这个平常的事实让他短暂失落，却也让这一章不必靠夸张成立。

这一段把喜欢从成绩栏前拉回错题本。名次不会因为难过自动升上去，陆允川必须把羞愧折成一道道能订正的题。

如果只写疼，这章会很薄；加上错题本、红笔盖被按上的咔哒声和晚饭里凉掉的米饭，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在班级后墙发呆。”陆允川把错题本收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

章节落下去时，他没有得到答案，却比早晨多明白一点：一个人可以被打动，但不能因为被打动，就把世界看成只围着自己转。
