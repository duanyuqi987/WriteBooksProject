# 第182章 道歉未发送

所属部卷：第四部：失去的美丽
时间：当天
地点：图书馆四楼

## 诗歌插页 P182：原创仿唐绝句

微光微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

陆允川没有立刻关电脑。那个"不发了"的结论他需要自己复核一遍。

他重新打开那三张草稿纸——第三张他没有扔进垃圾桶，只是揉皱了，又展开了。纸面上皱褶铺平后有几道很明显的褶痕，像某种地图。他沿着其中一条最直的痕——横跨了大半个纸面的——看下去。

第三张纸上写着：

"我一直以为我对你的好是'无条件'的。但昨晚写完那些重话之后我才发现——我所有的好都在等你签字确认。送你的书在等你回消息。楼下等你是在等你下来。那次扶肩膀的一秒我在等你的解释。那些删除线底下的话在等某天如果你看到它们会感到内疚。我的每个动作都在等着某一种回应——哪怕只是回头看我一眼。"

他看到这里停住了。他的道歉也在这个位置——等一个回应。

他在文档里把这段话重新打了一遍，然后接下去：

"如果我今天把这封信发给你——你会怎么读？你会读到我真的知道错了？还是会读到——他又来了——他又在用新的方式等我的回应。"

他停下来。光标闪了好几次。

最后他写："我选择不发。不是因为我不敢。是因为这一次——我终于不想再用道歉来交换你的任何东西。"

他把后一句话加粗——不是为了给任何人看。只是自己需要看清楚。

旁边那个查论文的同学又打印了一页。打印机吐纸的声音在这一刻像是某种认可。

他把文档关掉。系统弹出"是否保存更改？"——他点了是。

文件名：2021.12.15.docx。

没有苏晚枝的名字。

他把电脑塞进书包，站起来。经过叶澄的座位——她就在对面不远的桌子，他没有主动叫她。叶澄也没抬头——只是把手机屏幕压了一下，放平在桌面上。

回去的路上他给叶澄发了后半段："刚复核了一遍。确实是不该发。那封信的核心还是——'你看，我变了'。但变的证明不应该放在她桌上。"

叶澄回："你这个自检水平快赶上我了。"

"还差很多。"

"但速度已经快了。从台阶到今晚——你用了大概三周。一开始是一次一次撞墙。后来是撞了一下顿一下。今天是撞之前自己叫停。"

"你记得好清楚。"

"我是你的记录员。跟程砚分工——他管时间线和数据，我管因果链。"

陆允川笑了一下把手机放进口袋。图书馆门外的冷风从东山的方向吹过来。冬天的夜空没有星星。

他忽然想起来，三天前祁野在后门陪他站到天亮的时候说的最后一句话——不是"你可以丢脸"那句话，是更后面的一句："如果你真的想让她看到你变了——最好的方式不是道歉。是变到她不需要任何道歉。"当时他太困了没记完整。现在他完整地记起来了。

他在回宿舍的石子路上停下脚步。掏出手机在备忘录上打了一行：

"最好的道歉不是'对不起，我错了'。是——你现在可以不用再提心吊胆了。"

他看了几秒，保存。然后继续往宿舍走。
