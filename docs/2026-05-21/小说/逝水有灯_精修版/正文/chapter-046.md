# 第046章 元旦晚会之前

所属部卷：第一部：初秋三楼
时间：高二元旦前
地点：布置中的教室

## 诗歌插页 P046：原创现代诗

我把早晨放低，
让一盏灯从课桌边慢慢经过。
你的名字没有被我喊出，
只在晚会彩排时被风铃碰响的伴奏音旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写元旦晚会抵达之前的坡度。高二的十二月最后一个周一，陈老师在讲台上宣布："今年的元旦晚会，每班出一个节目。咱们班报的是合唱——全班参与。值日班委会负责采购彩带和气球，有才艺的同学请到蒋雯那里报名参加晚会策划小组。"放学后，教室里开始贴红纸窗花——有人沾了水把窗花贴反了，红纸背面朝外，白底朝里，被路过的同学提醒后红着脸撕下来重贴；有人踩在桌子上挥舞彩带被赶下来，彩带垂下来拖在地上，后面的同学踩了一脚差点绊倒。陆允川负责挂黑板上面的横幅——"恭贺新禧"四个美术字，用钉子钉在墙上的软木板上，钉子偏了，他踩着椅子调整时顶上的日光灯刺得眼睛发酸。

排练时间定在每天下午最后一节自习课。全班要求都到场——因为是合唱。夏知遥和几个女生站成一排在教室前面练声——"哆来咪发嗦"。音乐课代表把电子琴从音乐教室搬过来，推到了讲台边，琴键上有一层薄灰，她用袖子擦了两下才开始弹。她的声音在众人之中不突出——不是领唱，是中音部。她唱"咪"时声音悄悄地往上飘了一点，被程砚听出来了。程砚嘀咕说"有人唱错调了"——但全班没人当真。陆允川站在后排唱"嗦"——声音低到只敢震动自己的胸腔骨，旁边站着的男生唱得大声把他盖住了。他唱每一个音时都能听见她的部分混在全班背景音里，像风里的一丝草味——能感觉到但不能单独捏出。

程砚负责晚会当天的舞台布置。他用粉笔在地板上画站位——男生站后排，女生站前两排。画到前排靠右的位置时他故意多画了一道弧线——刚好对应夏知遥站的位置。陆允川看见了，走过去把他画的线擦掉："你画错了。""没画错——多画了一道，可以让人站得舒服点。""你不用替我设计站位。"程砚把粉笔揣进裤兜："那你自己画。""我不画——她站哪儿都一样。""你刚才说'她站哪儿都一样'的时候，眼睛抿了一下——你又说谎了。"

晚会前一天放学，陆允川帮叶澄搬音响——文科班也在准备节目，叶澄来借麦克风。借完东西后她在走廊里问他："明天演出，你会特别看谁吗？""全班都在看，没什么特别的。"她说："我建议你明天演出的时候把她当成乐队里的一把乐器——她是一只中音号。不要把目光收窄到一个人的脸上——把整班人放到视界里，她就在里面——作为整体的一部分。这样你应该不会出错。"她说完就走了。陆允川站在原地想了几秒——叶澄这人的本事就是能把所有抽象的问题都能转化成物理操作手册：位置、音域、视界宽度。他决定明天试试。
