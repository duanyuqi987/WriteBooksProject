# 第096章 省城南湖之前

所属部卷：第二部：六月以后
时间：大一九月
地点：大学南湖凉亭

## 诗歌插页 P096：原创现代诗

我把灯放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
南湖凉亭边晃动的柳影没有替谁回答，
只把那一年轻轻照亮。

九月一号，陆允川坐火车到了省城。他本来应该在省城转车去沿江那边的大学，但程砚说他九月二号才报到，让陆允川提前一天到省城，两人可以逛一逛。

火车晚点了四十分钟。陆允川拖着行李箱出站的时候，天已经黑了。站前广场上全是新生和接站的学长学姐，举着各色牌子，上面写着学校名字。他在人群中一眼就看到了程砚——不是因为他举了牌子，而是因为他站在出站口最显眼的位置，手里举着一瓶没开的矿泉水。

"你怕我渴死？"陆允川走过去。

"对。你上次在考场出来脸色白得像答题卡。"程砚把水塞给他，"走吧，先去住的地方。我订了个青旅，六人间。"

"还有别人住？"

"还有两个也是新生。一个考上了省大，一个跟你一样是土木的——不过他是省城大学的土木。"

青旅在老城区一栋旧楼的四楼，楼梯很窄，行李箱要侧着才能过。房间里已经坐了两个人。一个戴眼镜的男生正趴在床上看书，书名是《材料力学》——显然是土木那个。另一个靠在窗边抽烟，剃了个板寸，脸型很瘦。

"你们也是新生？"板寸头男生先开口，语气直来直去。

"嗯。"程砚把背包甩到床上，"你哪个学校？"

"省大。计算机。"他弹了一下烟灰，"祁野。你呢。"

"程砚。省城工学院，机械。"程砚往陆允川的方向抬了抬下巴，"他陆允川。沿江那边的大学，土木。明天走。"

祁野看了陆允川一眼。"土木好。以后修桥。"

"你也学会这句话了。"陆允川说。他想起来——他曾经用"就是造汽车发动机的"帮程砚跟三叔解释机械工程。

"不是学。是常识。"祁野把烟掐了，从窗台上跳下来，"你们吃饭了吗？楼下有家面馆，开到十二点。"

四个人一起去吃面。在面馆里，那个看《材料力学》的男生终于放下了书——他叫许知恒，省城大学土木系。戴一副很厚的眼镜，说话的时候会先把眼镜往鼻梁上推一推。

"你暑假就开始看材料力学了？"程砚盯着他手里那本书，封面上印着密密麻麻的公式。

"随便翻翻。怕开学跟不上。"

"兄弟，你还没开学呢。开学之后再怕也不迟。"

许知恒推了推眼镜，没有反驳，但也没有把书收起来。

祁野吃面的速度很快——三两牛肉面，他大概五分钟就吃完了。吃完以后把筷子一放，靠在椅子上看着程砚和陆允川。"你们俩高中同班？"

"嗯。三年。"

"那挺难得的。我跟高中同学一个都没联系了。群里没人说话。"

"正常。"陆允川说，"我们班群也没人说话了。除了逢年过节有人发红包。"

"那不一样。你们俩是私下还聚。大多数人毕业之后连红包都懒得发。"祁野把腿翘起来搭在旁边的凳子上，"大学重新洗牌。能交到新朋友最好，交不到就自己待着。"

程砚看了祁野一眼。"你说话真直接。"

"对。我从小就这样。得罪过不少人。但我懒得改。"祁野站起来，"我先回去。明早你们走的时候叫我一声，我去火车站送。"

他往外走的时候手里已经又点了一根烟。程砚低声对陆允川说："这人挺有意思的。"

"嗯。说话像刀。但心不坏。"

"你怎么知道？"

"因为他主动问了明早几点走。"

程砚想了想，点了点头。
