# 第089章 录取通知书旁观

所属部卷：第二部：六月以后
时间：七月下旬
地点：夏知遥家中

## 诗歌插页 P089：原创现代诗

她的通知书也到了，
红封放在桌上。
母亲问路远不远，
她说还好。
其实每个远方
都先从一句还好开始。

夏知遥收到通知书那天，家里刚下过雨。

邮递员把快件送到门口，母亲签收。红色信封递到她手上时，外面的地面还湿着。她拆得很小心，怕撕坏里面的纸。学校名字露出来的一瞬间，母亲先笑了，随后又问：“离家远不远？”

“还好。”夏知遥说。

她其实查过地图。坐车要转一次，第一次去可能要很早出门。可她不想让母亲担心。她把入学须知摊开，逐项看要带什么。母亲在旁边说家里给她买个新箱子，再买两套衣服。

夏知遥说不用买太多。

母亲说：“上大学了，总要像个样子。”

她低头笑了一下，眼睛有点热。

班级群里陆续有人晒通知书。夏知遥拍了一张自己的，发出去。很快有人祝贺，陆允川也发了一句：恭喜，顺利。

她看见后，回：谢谢，你也顺利。

这句话没有别的意思。她是真心希望他顺利。河边那场谈话之后，她一直有点担心自己是不是说得太重。现在看见他能在群里自然祝贺，她心里也松了一点。

晚上，夏知遥把通知书放进文件袋。文件袋里还有身份证复印件、照片和一张要买物品的清单。她没有把高中带走太多，只留了几张照片和一本英语错题本。

她要去新的城市。不是逃离谁，也不是奔向谁，只是她自己的路终于也写到了纸上。
