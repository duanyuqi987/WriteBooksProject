# 第107章 饭卡余额



时间：2010年五月底，午饭后



地点：食堂一楼、女生宿舍电话旁



卷次：第五卷：未来在黑板上倒计时



2010年五月底，午饭后，食堂一楼、女生宿舍电话旁，宋乔被母亲问起的饭卡消费记录把2010年的春夏推到十班面前。宋乔接到母亲电话，被问为什么饭卡花得比上月多。

她嘴上说食堂涨价，挂了电话却承认自己最近总替别人买水。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

宋乔也在这段日子里显出自己的位置。爽快、敢说话，愿意替人出头，但不喜欢让家里知道自己也会委屈。群像不是名单，群像是每个人都有自己的难处。

“饭卡怎么花这么快？”母亲说。

“天热，买水多。”宋乔说。

“你上次还替我买了两瓶。”刘青岚说。

“别记那么清。”宋乔说。

“热心也要量力，不然你先被家里骂。”常俊凡说。

食堂窗口前总有人把饭卡拍得很响，像那样余额就会多出来一点。

充值处的队伍贴着墙排，前面是打菜的热气，后面是学生翻找零钱的声音。

饭卡上的数字很小，却比成绩还具体。少一块钱，窗口就会停一下，后面的人也会跟着看过来。

常俊凡不喜欢在这种时候追问。她知道有些窘迫越解释越难堪。

段璟端着餐盘站在旁边，第一次觉得自己平时说不出口的自尊，其实别人也有。

食堂师傅把勺子在盆边敲了敲，提醒后面的人快一点，队伍又慢慢往前挪。

午饭并不浪漫，白菜、豆腐、米饭和饭卡余额，却能把人的日子照得很清楚。

吃完饭回教室时，后墙的错题区还在那里，像另一种账本。

陶建国这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

宋乔被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

曹蔚低头看饭卡余额，桌洞里的练习册露出红色封皮。窗外风吹了一下，纸角轻轻动了动，像有人把话收回去。

周明澈把餐盘往旁边挪了半寸，树莓摊旁边的纸盒被太阳晒软。十班的热闹还在，只是每个人都比去年更会藏心事。

付小川从口袋里摸出几枚硬币，黑板报边角的透明胶又翘起来。老师从门口经过，原本散开的声音立刻往回收。

何嘉树在充值窗口前数了一遍零钱，家长的电动车铃声从校门外传来。这样的细节不会被写进红榜，却会留在很多人的夏天里。

宋乔低头看饭卡余额，饭卡机短短响了一声。这点动作很小，却把那天的压力落到了实处。

沈曼青把餐盘往旁边挪了半寸，水房方向传来暖水瓶碰撞声。旁边的人没有追问，翻书声把难堪遮住。

梁一帆从口袋里摸出几枚硬币，补课班楼道里有人喊让一让。常俊凡看见了，没有替谁下结论，只把题号重新写清楚。

罗雨桐在充值窗口前数了一遍零钱，窗台上的金银花茶凉了一半。段璟后来才知道，未来最早就是这样从小事里露头的。

沈曼青低头看饭卡余额，讲台粉笔盒里只剩半截白粉笔。窗外风吹了一下，纸角轻轻动了动，像有人把话收回去。

梁一帆把餐盘往旁边挪了半寸，后墙的错题纸被风掀起一角。十班的热闹还在，只是每个人都比去年更会藏心事。

罗雨桐从口袋里摸出几枚硬币，竞赛名单的纸角已经微微卷起。老师从门口经过，原本散开的声音立刻往回收。

饭卡余额之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

饭卡余额把热心也照得很具体，谁替别人多做一点，家里账本上都会留下痕迹。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，宋乔被母亲问起的饭卡消费记录还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
