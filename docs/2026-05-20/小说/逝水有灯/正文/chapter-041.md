# 第041章 诗抄本之前

所属部卷：第一部：初秋三楼
时间：高二冬天
地点：男生宿舍下铺

## 诗歌插页 P041：原创现代诗

冬天把窗缝吹窄，
宿舍灯光发白。
一本空本子摊开，
等他把话写进去。
话太多，
纸太薄，
少年只好先写风。

高二冬天，陆允川买了一本新的诗抄本。

本子封面是深蓝色，纸质不算好，翻页时会有一点毛边。小卖部老板把它和两支黑色中性笔一起装进塑料袋，问他是不是要写作文素材。陆允川说是，声音有点虚。

他确实抄作文素材，也抄课本里的古诗词。但更多时候，他是在写自己不敢说的话。成绩下滑后，他把硬皮本收起来一段时间，后来又忍不住换了这本深蓝色本子，好像换一个封面，心事就能更规矩。

程砚看见新本子，第一反应是翻白眼：“你又买本子？”

“抄诗。”

“抄题不行吗？”

陆允川把本子塞进枕头下面：“题也抄。”

冬天的宿舍冷，窗户关不严，风从缝里钻进来。晚上熄灯前，大家都缩在被子里背书。陆允川坐在下铺，膝盖上垫着一本物理练习册，诗抄本压在练习册上。

他先抄了一首自己仿写的绝句，又写了几行散句。写到一半，笔尖停住。那些句子都绕着一个名字走，却不敢碰到名字本身。

窗外有人在楼下喊热水瓶别忘拿。宿舍里有人打喷嚏。程砚从上铺探头：“你再不睡，明天早读站着背。”

陆允川把本子合上。

他那时还不懂，诗不是心事的保险箱。写进去的东西不会因此变得安全，只是换一种方式继续发热。
