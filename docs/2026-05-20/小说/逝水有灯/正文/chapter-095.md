# 第095章 夏天的聚餐多年后

所属部卷：第二部：六月以后
时间：多年后
地点：县城小饭馆

## 诗歌插页 P095：原创仿唐绝句

旧店门前暑气深，
菜单红字半浮沉。
当年一句寻常话，
不必重翻作利针。

多年后陆允川回县城，小饭馆还在，只是招牌换成了发光字。门口原来洗青菜的地方摆了外卖柜，老板娘不认识他，问他几位。

“一位。”他说。

他坐在当年里间的位置，要了一盘青椒肉丝和一瓶汽水。墙上的菜单换过，红烧鱼的价格比当年高出很多。风扇没了，空调出风口往下滴水，服务员拿抹布在地上擦了两次。

菜上来后，他拍了一张照片，却没有发给任何人。照片里看不出什么旧事，只有桌布、筷子、半杯汽水和窗外停着的电动车。

他想起那句“你俩本来也不是一路”。年轻时他把这句话听成羞辱，后来才知道它更像一个粗糙的事实。事实不一定体面，却不必总被写成伤害。那天说话的人也只是十八九岁的同学，不懂分寸，不懂暗恋里藏着多少自尊。

吃完饭，他在柜台结账。老板娘把找零压在小票上，小票被风吹得翻了一下。他没有留它，转身丢进门口垃圾桶。

出门时阳光很亮，照得他眯起眼。陆允川站了一会儿，脚后跟当然早就不疼了。他忽然明白，有些疼不是消失，而是终于不再需要拿出来证明当年自己有多认真。
