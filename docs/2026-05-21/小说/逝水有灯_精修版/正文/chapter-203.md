# 第203章 最后一次争论之后

所属部卷：第四部：失去的美丽
时间：大四前夜
地点：陆允川住处

## 诗歌插页 P203：原创仿宋小令

香叶边，晓风清，
半窗纸页动微灯。
问旧年、几番执念，几番无凭。
人各有程休再问，
水声仍向远天生。

余波没有立刻散开。

陆允川回到住处已经快九点了。他换了拖鞋，把外套扔在床上，从冰箱里拿出一瓶水喝了半瓶，然后坐到电脑前面。

屏幕亮起来，光标在昨天写到一半的段落上闪烁。他看了一遍那几行，又看了一遍，觉得那些句子太安静了，安静到不像是刚经历了一场争论的人写的。

手机响了。程砚。

"吃了吗？"

"吃了。牛肉面。"

"一个人？"

"不是。"

电话那头停了一下。"苏晚枝？"

"嗯。"

程砚没有追问。他说："我在楼下买水果，要不要给你带点？"

"不用。你上来吧。"

程砚来的时候真的没带水果，但带了两罐啤酒。他把一罐搁在陆允川的桌角，自己开了另一罐。

"说说。"他靠着窗台。

陆允川也开了啤酒。气泡冲上来的声音很响。

"她说她那时候接不住。她说她只是十九岁，不知道怎么拒绝一个特别好的人。"他喝了一口，"然后她说了对不起。"

"你接受了吗？"

"接受了。"

"真接受了还是嘴上接受了？"

陆允川转着手里的啤酒罐。"不知道。也许是五五分。"

程砚点了点头，没有说"交给时间"之类的话。他的优点是知道什么时候闭嘴。

"你书写到哪里了？"他换了个话题。

"写到面馆那一章。"

"那就是今天的事。"

"对。"

"很难写吧？把刚发生的事写成文字。"

"相反。"陆允川把脚搁在桌腿横挡上，"以前觉得难写的，今天反而好写了。因为我听到她亲口说了。她亲口说的那些话，比我脑子里编的要准确一百倍。"

程砚喝了大半罐啤酒，把空罐往垃圾桶里一扔，没进，在地上滚了一圈才停住。

"允川，你有没有想过一个问题。"

"什么？"

"你写了这么多年，到底是想理解她，还是想理解你自己？"

陆允川没有回答。他把脚从桌挡上放下来，弯腰捡起那个啤酒罐，扔进垃圾桶。

"都有。"他说。"以前是前者多一些，最近几年是后者多一些。"

"那今天之后呢？"

"今天之后，"他把剩下的啤酒喝完了，"也许两者都不重要了。"

电脑屏幕自动变暗了。节能模式。陆允川碰了一下鼠标，屏幕重新亮起来。

"你那一章打算怎么写？"程砚问。

"怎么写..."

"对。把她说的话照搬上去？"

"照搬。一字不改。"

"不怕她看了不舒服？"

"她让我写完了给她看。"陆允川看着屏幕上那段话，"她自己说的话，她应该有勇气再读一遍。如果她没有，那我替她写下来的意义就是另一回事了。"

程砚又待了一会儿，把窗台上几本乱放的书摞整齐了，然后说他该回去了。

"明天早课？"

"对，八点半。代课老师要点名。"

程砚走到门口又回头。"允川。"

"嗯？"

"你没喝多吧？"

"一罐啤酒而已。"

"不是。"程砚指了指他的样子，"我是说今天这件事。你回来以后还没哭过。"

"为什么非要哭？"

"以前你会。"

陆允川想了想。"以前我还会写十五页日记分析她的每一个表情。现在不会了。"

"那你现在会做什么？"

"把那一章写完，然后去洗个澡，然后睡觉。明天起来改稿子。"

程砚看了他几秒，点了头。"行。我走了。"

门关上。陆允川在电脑前坐了一会儿，开始打字。

他写苏晚枝说"不知道怎么回"的时候，面汤上浮着一层薄薄的油。他写叶澄在旁边吃馄饨，勺子碰碗沿的声音比他们的对话还响。他写苏晚枝把香菜叶排成一排，又一片一片夹回碗里。

他写她最后说"对不起"的样子。

写到这里，他停下来，去厨房烧了一壶水。等水开的时候他靠在灶台上，看窗外的路灯把对面的墙照成橘黄色。隔壁有人在放音乐，鼓点很轻，穿过墙壁就变成了一种有节奏的振动。

水开了。开关跳起来的声音干脆利落。他泡了杯茶，端回电脑前。

屏幕上最后一行是：她的道歉，我接受了。

他把光标移到下一行，打了一个句号，又删掉。打了一个逗号，也删掉。最后他什么都没加，直接存了档。

关了电脑以后，房间里只剩下冰箱的嗡鸣声。他坐在床边，把外套从床上拿起来挂到门后，关了大灯，开了床头的小夜灯。

手机又亮了。这次不是程砚。是一个旧群聊——高中同学群——有人在发毕业聚会的时间。

他看了一遍那条消息，回了一个"收到"。

然后他把手机屏幕朝下扣在床头柜上。小夜灯的光罩在他的手上，一道很淡的暖黄色，像面碗边那片被热汤泡软的香菜叶。

他闭上眼睛。今天的余波还在胃里，但没有翻上来。就那么沉着。
