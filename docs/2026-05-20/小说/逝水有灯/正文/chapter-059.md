# 第059章 最后一场模拟考旁观

所属部卷：第一部：初秋三楼
时间：高三春天
地点：县一中考场

## 诗歌插页 P059：原创现代诗

她走出考场，
手心也有汗。
别人看见她稳定，
看不见她在草稿纸上
划掉三次辅助线。
青春不是谁望着谁，
更多时候，
是各自和一道题坐到天黑。

夏知遥最后一场模拟考也考得不轻松。

语文作文她写得顺，数学最后一题却卡住了。她在草稿纸上画了三次辅助线，都觉得不对。铃声前五分钟，她放弃了最后一步，把能写的过程补完整。交卷时，心里还是堵。

走出考场后，林曼问她：“怎么样？”

“数学难。”

林曼松了口气：“你也觉得难我就放心了。”

夏知遥笑了一下，却没有真的轻松。她知道别人习惯把她当作稳定的参照，可她自己并不稳定。每次考试前，她也会失眠，也会担心发挥失常，也会在电话里对母亲说“还可以”，挂断后坐着发呆。

楼梯间遇见陆允川时，她有点意外。

他说数学挺难的。她点头，说最后一题没做完。这是很自然的考后对话。说完以后，她发现陆允川没有像以前那样紧张，也没有把一句话拖得很重。他只是站在那里，像一个同样被数学折腾过的同学。

这让夏知遥觉得舒服。

人和人之间，有时候并不是越靠近越好。能自然地说一句题难，能自然地点头，能在后面有人催路时各自让开，也是一种合适的距离。

考后走廊里，程砚说自己和最后一题和平分手。大家都笑。夏知遥也笑了。笑完，她把卷子卷起来，和林曼一起往食堂走。

她没有把这件事记成特别。可多年后如果有人问她高三春天是什么样，她也许会想起那条走廊：很多人，很吵，手心有汗，大家都在试着从一场考试里走出来。
