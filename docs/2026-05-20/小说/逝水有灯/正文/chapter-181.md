# 第181章 道歉未发送之前

所属部卷：第四部：失去的美丽
时间：大三下学期，晚自习前
地点：图书馆四楼

## 诗歌插页 P181：原创现代诗

我把一句对不起
放进空白文档
它没有变轻
只是学会了不敲你的门

图书馆四楼靠西的窗户坏了一小角，胶带贴了三层，风还是能从缝里钻进来。陆允川抱着电脑上楼时，楼梯间的声控灯亮了又灭，他站在平台上等了几秒，像等自己心里那股急火也跟着暗下去。

下午叶澄的话还在耳边：“你可以道歉，但不要拿道歉去换她回复。”这句话不重，却像一枚钉子，把他从连夜解释的冲动里钉住。祁野也说过差不多的意思，只是更直白：“你现在发，还是在要答案。”

他选了最里面的位置。桌面有一道铅笔划痕，像谁在背单词时无意识留下的线。陆允川打开文档，标题敲成《给苏晚枝》，又删掉，只留下一个没名字的文件。第一句写得很快：对不起，我之前太急了。第二句却停了很久。他发现自己每写一句抱歉，后面都会忍不住跟上一句解释，像把错递出去之前还要替自己垫一层纸。

窗外的树影贴在玻璃上，晚课铃远远响起。有人在书架后面找书，抽出来一本，又塞回去。陆允川把光标移到第二行，删掉“可是我也很难过”。屏幕空了一块，他的胸口也跟着空了一下。

那天道歉还没有开始，它先变成一场删减。删掉委屈，删掉求证，删掉“你能不能理解我”。剩下的句子短得可怜，却终于不再向她伸手。
