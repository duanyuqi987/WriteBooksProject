# 第045章 诗抄本多年后

所属部卷：第一部：初秋三楼
时间：多年后
地点：书桌灯下

## 诗歌插页 P045：原创仿唐绝句

蓝封旧册压灯尘，
半是诗行半是真。
年少不知风月窄，
写来写去写孤身。

多年后，陆允川重新翻开那本深蓝色诗抄本。

纸页已经松了，最前面的几页写得端正，越往后越乱。有些诗句旁边夹着物理公式，有些日期下面写着听写错词。那本子不像一本纯粹的诗集，更像高二冬天混乱生活的截面。

他读到“我把名字藏进风里”那句时，停了一会儿。

年轻时他觉得这句很克制，现在看，克制里仍有期待。他把名字藏起来，并不是不想被知道，而是希望某一天有人能从风里听见。这种心情不算罪，也不算高尚，只是少年人常有的拧巴。

他没有删掉，也没有嘲笑。

诗抄本救过他。那些句子让他在无处可说的时候有地方落笔，让冬夜不至于只剩错题和排名。但诗抄本也骗过他。它让他误以为写得深，就等于爱得深；写得美，就等于做得对。

陆允川把本子翻到最后一页，那里夹着程砚当年写的复习表，字迹潦草：物理先做，诗后写。

他笑了一下，把那张纸拍了照片发给程砚。程砚很快回：我那时候竟然这么有文化？

陆允川回：主要是字丑。

手机屏幕亮着，书桌灯也亮着。深蓝色本子躺在光里，不再像秘密。它只是一本旧本子，证明一个人曾经用很笨的方式，试着把心里的风写下来。
