# 第006章 新课本的气味之前

所属部卷：第一部：初秋三楼
时间：高一开学周
地点：教室后排靠窗处

## 诗歌插页 P006：原创现代诗

我把早晨放低，
让一阵风从课桌边慢慢经过。
你的名字没有被我喊出，
只在透明书皮里压住的气泡旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写新课本的气味抵达之前的坡度。开学周的教室总有一股新课本的油墨味，像一场还没开始的考试提前把纸页摊开。后排靠窗处风大，书角一不留神就被吹起，哗啦啦响成一排小旗。

陆允川把透明书皮沿着封面压平，指甲顺着边缘刮过去，气泡被挤到书脊旁边，怎么也不肯完全消失。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。看着书皮上一排排小气泡，他忽然想起镇上初中发的书从来不包皮，因为用不着——课桌缝里的灰足够把书页弄脏好几遍了。

九月的日光从窗户斜进来，在课桌上切出一条亮带。阳光照到的地方桌面微微发烫，照不到的地方凉凉的，他把手背轮流贴在两个区域上试了试温差。前排有人撕新本子的塑料封膜，嘶啦一声，接着是几个人同时翻书的哗啦响。这些声音和气味混在一起，让他觉得县城中学确实不一样——书是新书，桌是新桌，连空气里都少了镇上那种煤灰味。头顶的吊扇转得很慢，扇叶上积了一层灰，每次转到一定角度就吱呀一声，像一只老鸟在房梁上叫。

那天真正的事还没有发生，透明书皮里压住的气泡却先被他记住了。手指刮过书皮时那些气泡在塑料膜下往旁边跑，像困在冰层下面的小鱼。

"包这么认真，准备把数学书供起来？"程砚从旁边探过头来，手里拿着自己的书皮——已经裁歪了，边缘多出一截，像荷叶边。"书皮不平容易翘。"陆允川说。"你这话听起来像我奶奶说的。"程砚把裁坏的书皮揉成一团，从抽屉里翻出一个新的，"来来来，帮我弄，我手笨。"

陆允川接过他的课本和书皮，替他把封面压平、刮掉气泡，动作比给自己包还仔细。程砚在旁边看着，忽然冒了一句："你这手艺以后能开个店。""书店还是包书皮店？""你开个专门帮女生包书皮的店，生意肯定好。"陆允川手顿了一下，没搭话。程砚见他不吭声，自己又补了一句："我帮你宣传，就说高一三班陆允川专业包书，包完连一个气泡都找不到。"他说着拿圆珠笔在课桌上比划了一个写招牌的动作。陆允川终于没忍住笑了一声："你先把你那荷叶边书皮换了再说。"程砚低头看了看自己裁歪的书皮，也笑了："这叫个性，你不懂。"

方老师抱着一摞卷子走进来，说上学期摸底的成绩贴在后黑板上，大家自己去看。话音还没落前排的人就冲过去了，椅子刮地的声音响成一片。陆允川没有立刻站起来，他从笔袋里抽出一支铅笔，开始削——其实铅笔不需要削，他只是让自己手上有事可做。

成绩单贴在黑板的右上角，总共三栏，密密麻麻的名字。他远远扫了一眼，没能看清。靠近黑板的几个人已经在互相拍肩了："你考这么好还装！""我这叫发挥稳定。""哥们你这英语怎么考的，比我多二十。"这些声音钻进他的耳朵，他没有参与，只是把削尖的铅笔在草稿纸上画了一条线。那条线画得很直，没用尺子。他又在旁边画了第二条、第三条，三条线平行地躺在纸上，像三条没有目的地的铁轨。前排传来一阵起哄声，有人被拍了后脑勺，回头骂了一句，声音里带着笑。

叶澄抱着一摞英语练习册从办公室回来，从前往后传。传到夏知遥那一排时，前面同时举起两只手，她愣了一下，然后把练习册放到了先举手的人桌上，对另一个说："你的在后面，等一下。"处理这种小冲突对叶澄来说已经是日常了，她连眉毛都没皱。

夏知遥这时还只是人群里的一个名字。她坐在前两排靠走廊的位置，拿到书后先在扉页写名字。她的字不花哨，横平竖直，其中"遥"字的最后一捺收得很稳，没有拖尾。陆允川看见了，没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子——什么"她的字是那年九月唯一干净的线条"——然后又一行行划掉。留下来的，是教室里的油墨味、书皮上的气泡、以及铅笔屑在桌面上绕成的那个小圈。还有方老师抱卷子进来时胳膊上沾的粉笔灰，还有窗台上那盆没人浇水的绿萝——叶子已经有些发黄了，土干得裂了口。

后排有人借剪刀，有人说自己把书皮裁反了，胶带贴了一层又一层，书变得像从水里捞出来的。坐在中间一排的男生干脆不包了，把书皮往抽屉里一塞，说反正过两天就皱了。同桌的女生白了他一眼，把自己多出的一截书皮裁下来递给他。方老师在讲台上说新学期要有新气象，底下没人抬头，所有人都忙着和塑料书皮搏斗。尺子划过塑料的声音此起彼伏，细碎、认真，又有点好笑。偶尔有人撕胶带撕歪了，扯出一声刺耳的尖响，全班就哄一下。

放学后他没有立刻走，把桌上的铅笔屑扫进手心，倒进桌边的垃圾袋里。窗外的法桐叶子被风掀起来，绿中带着一点点焦边。他看了一会儿，想起自己今天削铅笔削了三次，其实一个字都没写。教室里的风扇已经被值日生关了，空气一下子安静下来，只剩下走廊里断断续续的脚步声。

如果说喜欢在这里有了开端，也不是因为命运突然发声。只是他忽然觉得，把一本书包好需要一个理由——这本书值得被包好，这个人值得在写名字时不被打扰。他不确定这两个"值得"有没有因果关系，但他没有再往下想。
