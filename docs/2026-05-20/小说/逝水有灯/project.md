# 《逝水有灯》项目底座

## 基本信息

- 作者：段锦佑
- 类型：温柔现实青春成长长篇
- 目标体量：约50万字
- 章节规模：245 章
- 素材来源：本地2010-2013青春情感素材（公开版不记录来源细节和姓名）
- 公开原则：只保留情绪结构、事件种子和时代质感，不保留来源姓名、路径、文件名或可识别细节。

## 一句话

一个从乡镇中学考入县城高中的男生，先用三年暗恋学会仰望，又在大学里把喜欢变成执拗，最后靠诗歌、旧日记本和漫长生活，学会把旧日子里的微光还给时间。

## 核心主题

- 喜欢不是占有，也不是要求回应。
- 旧日子里的美不必被追回，它可以成为照亮后来道路的一盏灯。
- 青春里的错误不只属于某一个人，成熟是愿意看见自己和别人各自的局限。
- 诗歌不是装饰，而是人物处理情绪的方式。

## 叙事风格

- 温柔现实：生活细节驱动，不写爽文，不写狗血虐恋，不写控诉文。
- 场景优先：每章至少有一个具体场景和一个可触摸物件。
- 诗歌嵌入：每章一首诗歌插页，包含原创现代诗、原创仿唐绝句、原创仿宋小令。
- 减法原则：尽量让人物通过动作、对话、物件和场景变化完成情绪。

## 人物写法

- 陆允川不能只写成受伤者。他也有自尊、虚荣、控制欲、逃避和迟来的反省。
- 夏知遥不能只写成白月光。她有学习压力、家庭处境、怕生和自己的未来路线。
- 苏晚枝不能只写成拒绝者。她有温柔、边界、社团责任，也有被他人期待压住时的疲惫。
- 朋友不能只承担劝导功能。程砚、祁野、叶澄都要有自己的生活压力和说话方式。

## 诗歌策略

- 现代诗负责直面当下情绪，不要求整齐押韵。
- 仿唐绝句负责克制和收束，让情绪转入景物。
- 仿宋小令负责迟疑、回望和余音。
- 诗歌必须服务人物，不作为炫技装饰，也不直接引用大段古诗词原文。

## 禁用项

- 不出现来源路径、来源文件名、真实姓名和高识别度关系细节。
- 不使用“便签”作为叙事载体，统一使用旧日记本、诗抄本、草稿纸、照片文件夹等。
- 不把任一女性角色写成单纯的伤害者或奖品。
- 不写复合幻想、报复爽点、跪求式挽回或单向控诉。
