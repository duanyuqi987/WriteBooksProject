# 第016章 期末

时间：2011年1月中旬

地点：青阳县一中教室

卷次：卷一：初见秋风

一月中旬，期末考试。

考前三天，各科老师都在发卷子。物理三张，数学两张，英语两张半。化学老师最狠——发了四张，还带附加题。课代表发卷子的时候，教室里一片哀嚎。有人把卷子叠成纸飞机扔出窗外——物理老师从走廊经过，捡起来看了一眼，说了句"受力分析画错了"，然后继续走了。

晚自习延长到十点。宿管老师默许——他十点半才来锁门。

教室里的灯十点半之前不关。所有人都低着头。翻书声、笔尖在纸上刮的声音、椅子偶尔被屁股挪动时的嘎吱声。有人在吃饼干——掰成两半，一半塞进嘴里，一半递给同桌。同桌没接。他又偷偷塞回饼干袋里。

顾临风这次复习比以往都要拼。

拼的理由他告诉自己是因为想进步。上高中第一个人学期总得有个态度。但他其实知道不是。期中他三十一名。她第八名。

他想让期末的成绩单上她的名字旁边多一个可以跟她比较的名字。

是这个。

就这个。

他每天五点四十五起床——比晨读提前一个多小时。天还没亮。宿舍楼的走廊灯是声控的，他踩着拖鞋走过去，灯亮了，走到水房门口，灯又灭了。他得跺一下脚。然后刷牙洗脸。对着水房的镜子看了自己一眼。镜子上面有牙膏点子。他的脸在牙膏点子的间隙里。下巴上长了一颗痘——右边。

六点到教室。教室灯还没全开——只有前面那一半亮着。他坐在倒数第三排。面前一摞书。先从物理看起。力学、运动学、电学基础。每一章的例题重新做一遍。错题再算一遍。物理笔记本上写得密密麻麻——蓝色的是原来的笔记，黑色的是补充的，红色的是用蓝黑红三色圆珠笔标的重点。

六点半之后教室里的人开始多起来。

苏晚晴大概是六点五十到。

她每天进门的时候书包还没摘——一只手拎着书包带，另一只手把吃了一半的包子往嘴里送。包子是食堂的，塑料袋捂着，热气把袋子撑鼓了。她走到自己座位。坐下。把书包挂到椅背上。然后把最后一口包子吃完。掏出一张纸巾擦手指。

然后翻开物理笔记本。

顾临风每次都注意到这个。

她也在补物理。

考前一天。全年级停课复习。

教室里的气氛像是绷紧的弦。连赵一鸣都不怎么说话了。他偶尔冒出一句"这道题是不是选C"然后自己在本子上验算，算完来一句"还真是C"然后又安静了。

顾临风没有说话。他在算。在写。在背。英语单词、化学方程式、政治简答。一本接一本。中间去上了个厕所。回来继续。

下午自习的时候。苏晚晴的物理笔记本从桌面上掉了下来。掉在过道上。啪的一声。旁边一个男生帮她捡起来递过去。她接过去说了句什么——嘴动了一下。

顾临风从四排外看到了整个过程。然后低头继续写题。

考试三天。

每天两门。上午一门下午一门。一共六门——语文、数学、英语、物理、化学、政治。政治和化学合在一场考试里，算"综合"。

语文作文题目是《桥》。他写了他家门前那座石拱桥——桥下夏天有水，冬天干了。写了桥面上的石板被踩凹了一个坑。写了他小时候跟姐在桥下抓鱼的事。写了桥头的早点铺——炸油条的铁锅下面火烧得很旺。没有写别的。字数够了。他停了笔。

物理卷子最后一道大题是力学综合——斜面、滑轮和拉力。他列了三个方程。解到第二步的时候橡皮找不到了。手下意识往旁边摸——没有。他急了大概三秒钟。然后低头在桌子底下摸了摸——橡皮在地板上。捡起来。继续算。第三问做出来了。

考完最后一门。监考老师说"停笔"。有人往后一靠，长吐一口气。有人趴在桌上。有人把笔往桌上一拍——黑色水笔从桌上弹起来掉到地上，滚出去半米，没捡。

顾临风没趴。他把笔放进铅笔盒。卷子翻过来压在铅笔盒下面。然后站起来。走出教室。

走廊里很多人。有人在哭——说数学最后一道题没做出来。有人在笑——说只要及格就行。有人抱在一起。有人往外跑。

他从三楼走廊往下看。操场上积了一滩一滩的水——考政治那天下的雨。太阳刚出来，水潭上面的光反光刺眼。

出成绩是三天后的事。

公告栏上的红纸换了一张新的——"2010—2011学年第一学期期末考试"。顾临风挤进人群。这次他没从中间找。

直接从最上面往下看。

苏晚晴。第六名。物理71——进步了。

他往下找。

顾临风。第二十四名。物理87。

她第六。他二十四。距离短了。

他把这个名字看了一会儿。苏晚晴。第六名。然后又看了看自己的。第二十四名。

从三十一到二十四。进了七名。

他把手从红纸上拿开。红纸上沾了一点汗。字模糊了一个角。抹了抹，字还在。然后转身走了。

学校里已经有人拖着行李箱往外走了——轮子在水泥地上咕噜噜地响。寒假开始了。
