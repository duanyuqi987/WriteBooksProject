# 第075章 网吧通宵多年后

所属部卷：第二部：六月以后
时间：多年后
地点：旧聊天记录前

## 诗歌插页 P075：原创仿唐绝句

旧号重登夜色空，
未发之言不再红。
当年幸有旁边友，
拦住凌晨一阵风。

多年后，陆允川重新登录旧账号，聊天列表已经陌生。

很多头像变成灰色，很多备注看不出是谁。夏知遥的名字也在其中，聊天记录停在很早以前，干干净净，没有那个高考后的凌晨。陆允川看着空白窗口，忽然感谢自己当年没有按下发送。

不发并不代表没有感情。相反，那时正因为太有感情，才容易把一句普通问候发成索要回应的绳子。

他想起程砚坐在旁边，说不知道就先别发。那句话当年听着平常，现在看，很珍贵。朋友没有替他做决定，只是在他最容易越界的时候，把灯稍微调亮了一点。

陆允川把旧账号退出，电脑屏幕暗下来。他没有截图，也没有把聊天窗口保存进文件袋。因为真正值得保存的不是空白窗口，而是那次没有打扰。

他在硬皮本里写：有些克制，当时像失败，后来才知道是善意。

写完以后，他给程砚发了条消息：你还记得高考后通宵吗？

程砚回得很快：记得，你坑了我一整晚。

陆允川笑了。那一夜终于从苦情里退出来，变成朋友之间可以轻轻提起的一段旧事。
