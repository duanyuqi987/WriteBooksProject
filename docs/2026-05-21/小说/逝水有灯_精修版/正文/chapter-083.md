# 第083章 迟到的表白之后

所属部卷：第二部：六月以后
时间：六月末
地点：河边护栏旁

## 诗歌插页 P083：原创仿宋小令

风入旧窗轻，
纸上有余声。
欲问前程何处去，
水边灯影自分明。
回首不须惊。

从河边回来那天晚上，陆允川在家吃了一整碗红烧肉。

母亲看他食欲好，又给他碗里夹了两块。"你爸说你今天去学校填志愿了。"

"嗯。"

"填了哪里？"

"第一志愿是沿江那边的大学。第二是省城的。"

"沿江那边离家多远？"

"火车六个小时。"

母亲的手在筷子上停了一下。筷子夹着一块肉，悬在半空中，然后慢慢放进了自己碗里，不是他的碗。"一个人去那么远，行不行？"

"程砚也报了那边的学校。"

"他也去？"

"不一定能录上。他分比我低。"陆允川把肉咽下去，"但他第二志愿也是省城。我第二志愿也是省城。如果都没录上第一志愿，就一起去省城。"

"省城近。火车一个半小时就到了。"母亲说这话的时候没看他，盯着碗里的饭。

"嗯。"

父亲在客厅看电视，声音开得很小，是新闻联播后的天气预报。播报员在报沿江城市的气温——三十四度，比他们这边热。母亲显然也听见了，但她没说什么，只是站起来去厨房给陆允川盛了一碗汤。

母亲似乎放心了一点。她低头扒了一口饭，又问："那个女生呢？"

陆允川的筷子顿了一下。"什么女生？"

"你之前模拟考那次，考完出来你跟我说，你们班有个女生数学选择题选错了——"母亲难得地露出了一个不易察觉的笑意，"你说的时候，语气跟说别人不一样。"

陆允川没说话。他不知道母亲是从什么时候注意到的——可能是那次模拟考后他在厨房里无意中提了一句，也可能更早。母亲的观察力在很多事情上并不敏锐，但涉及到他的时候，总是意外地准。

"她报了别的学校。以后应该不会在一个城市。"陆允川说得很平，像在陈述明天的天气。

母亲看了他一眼，然后又夹了一块肉放进他碗里。"多吃点。"

她没有问更多。陆允川低头吃饭，肉很香。

饭后他去院子里打水洗脸。井水还是冰凉的，顺着下巴滴进领口。他抬头看天，北斗七星挂在老位置上。北斗的勺柄指向南方——他忽然想起地理课上老师讲过，夏天北斗的勺柄指南。南方，就是夏知遥要去的方向。

高考之后第二天晚上他也站在这里看星星。那时候他不知道分数，不知道志愿，不知道夏知遥会考去哪个城市。现在全都知道了。世界反而安静了下来。不是空虚的那种安静，是答案落定之后的那种安静。好比考场上最后一遍铃，你知道交卷了，不管答案对不对，都不能再改了。

手机响了。是叶澄。

"程砚跟我说了。"她开门见山。

"说什么？"

"说你在走廊里遇到夏知遥了。你没说什么。"

陆允川把毛巾搭在肩上。"他觉得我应该说什么？"

"他觉得你至少应该让她知道。"叶澄的声音从电话里传过来，带着她一贯的平稳，"但我倒不觉得。你今天不说的理由，是不是因为看懂了她的反应？"

"……算是。"

"她什么反应？"

"我说'祝你大学顺利'，她说'你也是'。然后转身走了。全程连一秒多的停顿都没有。"

叶澄在电话那头沉默了一会儿。"你知道这种反应意味着什么吗？"

"知道。她早就知道我想说什么，也早就准备好了答案。她只是等我主动开口，或者等我自己放弃。"

"差不多。"叶澄的声音放轻了一点，"夏知遥不是不懂你的意思。她是给了你足够的时间和空间，让你自己想清楚要不要说出来。她不会替你拒绝你自己。但她也不会给你任何鼓励。"

"你什么时候知道的？"

"一直都知道。"叶澄顿了一下，"有一次在楼梯口，我帮她拉书包拉链，看到她的草稿纸上写了一行字——'想考去一个有江的城市'。跟你写的完全一样。我当时以为你们之间有什么事是我不知道的。后来我才明白，那句话她不是写给你的。她真的想去一个有江的城市。跟你没关系。"

这话说出来不太好听。但它是对的。陆允川握着手机，看着井口里的水面。水很黑，倒映着天上的北斗星，晃一晃就碎了。

"陆允川。"

"嗯？"

"你今天做的决定是对的。不是因为表白会失败——那个没关系。是因为你终于学会了在看懂别人之后再行动。"叶澄的语气像在批改一篇作文，"这个比表白本身重要。"

"你又在给我写评语。"

"对。习惯了。"她停了一下，"但说真的，你今天在走廊里做的那个判断，比我给你的任何评语都准确。'

"你这是在夸我？"

"是在陈述事实。"叶澄的声音里带了一点笑意，"好了，早点睡。明天程砚肯定又要找你。"

"你怎么知道？"

"因为他刚在群里发了三条消息，问明天有没有人去网吧。@了你三次。"

陆允川把手机切到群聊页面，果然看到程砚发了三条消息："明天网吧有人吗""在线等""陆允川你别装死"。他回了一个字："去。"

然后他切回来跟叶澄说："你猜对了。"

"当然。"叶澄挂了电话。

陆允川笑了一声。他把毛巾从肩上拿下来挂在井边的晾衣绳上。晾衣绳上还挂着白天晒的被单，被晚风吹得鼓起来，像一面软软的帆。院子里的鸡已经回窝了，安静的翅膀偶尔扑腾一下。他推开东屋的门，倒在床上。窗外有虫鸣，一声接一声，密密地织在夜风里。手机屏幕亮了一下——是程砚的第四条消息："行，明天下午两点老地方。"

他把手机翻过去扣在枕头边，闭上了眼睛。
