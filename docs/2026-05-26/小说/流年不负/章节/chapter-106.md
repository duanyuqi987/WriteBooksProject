# 第106章 点赞

时间：2015年12月

地点：楚州师院宿舍

卷次：卷六：平行轨道

他把照片放大。

又看了一遍。

苏晚晴的脸占据了整个屏幕。白衬衫。马尾。阳光在她的鼻梁上有一小块亮斑。她的嘴张着——应该是正在喊口令。"一二三四"或者"伸展运动"。嘴角往上翘。那种翘法不是刻意的。是呼吸跟动作之间自然带出来的弧度。

他的拇指在屏幕上来回动。

放大——她的眉毛。右边那道比左边稍微淡一点。以前他在教室里偷看她的时候，就发现过。那天她在座位上看书。光从窗外斜进来。照在她右半边脸上。右眉毛在光里几乎看不见。

缩小——整个操场。红色的塑胶跑道。绿色的假草坪。一群灰白色校服的孩子。她站在正前方。像一颗棋子站在棋盘上。周围都是比她矮一半的人。

放大——她的手。手指伸直。指尖微微往上翘。他记得她写字的时候小指也会翘起来。那是个改不掉的习惯。

缩小——黑板。"祝苏老师生日快乐"——粉笔字。二三十个字大小不一。"祝"字最大。"乐"字被黑板槽挡住了一半。画了一只蛋糕——说是蛋糕，其实就是三个椭圆叠在一起。上面插了几根蜡烛。蜡烛的火焰画得像一把叉子。

放大——她的脸旁边。泪痕还在。嘴角也在翘。她大概是被那些花和歪歪扭扭的字弄哭了。然后又被自己的"哭"弄笑了。然后在笑里发觉自己真的喜欢上了这份工作。

他把照片放在这里。没有动。

拇指悬在"赞"上。

那颗灰色的小爱心在拇指肚下面。离屏幕大概一毫米。

窗外有人吹口哨。断断续续的。是一首他听过的歌。忘了叫什么名字。

他的拇指悬着。

大概二十秒。

他把手机锁屏了。

屏幕黑了。苏晚晴的脸消失了。

他把手机放在桌上。翻了个身。仰面躺着。看着天花板。裂缝还在。从灯座往墙角延伸。三年了。裂缝没变。盯着看久了会觉得它在动。

脑子里是那张照片。她的脸。她的眉毛。她翘起的指尖。孩子们歪歪扭扭的粉笔字。那朵像叉子的蜡烛火焰。

解锁。

屏幕亮了。照片还在。

他看了几秒。

锁屏。

把手机放回枕头底下。

窗外有摩托车发动的声音。轰轰轰的。在夜里的街道上传得很远。然后拐了个弯。声音小了。没了。

他闭上眼睛。

他在想——点赞是什么意思。

他想的是：如果点了赞，她会看到。她会知道他在看她的朋友圈。她会知道他的消息。然后呢。

他想不到然后。

两年多了。两年多没有联系。没有电话。没有短信。他唯一做的就是在她的朋友圈里——潜水。看她说"期末考完了""今天下雪了""食堂的饺子好难吃"。每一次都看完。每一次都没有点赞。没有评论。

像一个站在窗外的人。不敲门。不说话。只是站着。

如果现在点赞——那扇窗就破了。

他还没准备好。

不是不想联系。是不知道该说什么。该用什么语气。该发"好久不见"还是什么都不说只是点一个赞。

他不知道。

所以他没有点赞。

深呼吸。一次。呼出的气从鼻子里走。稳的。

他翻了个身。脸冲着墙。墙皮上有几处剥落的痕迹。在黑暗里看不清楚。但手能摸到——粗糙的、有点硬的。

那天晚上他醒了一次。凌晨三点四十。窗外路灯投进来的光在地上铺了一片长方形。

他把手机从枕头底下摸出来。

解锁。朋友圈。许若琳发的那组照片。苏晚晴。第四张。白衬衫。马尾。笑。

他看了大概一分钟。

然后把手机放下。翻了个身。

睡了。
