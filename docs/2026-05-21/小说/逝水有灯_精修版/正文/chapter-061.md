# 第061章 高考后的水之前

所属部卷：第二部：六月以后
时间：2012年六月八日
地点：县一中校门口

## 诗歌插页 P061：原创现代诗

我把星放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
母亲递来的矿泉水瓶没有替谁回答，
只把那一年轻轻照亮。

六月八日下午，陆允川已经考完了语文和数学。还剩明天一天——英语和文综。

中午他回家吃饭。母亲做了四个菜，比平时多了两个，还有一个是他上次随口说好吃的红烧茄子。

"你多吃点。"母亲把菜往他面前推了推，"下午的理综，时间够不够？"

"够了。两个半小时。"

"那就好。上次模拟考你不是说没做完吗。"

"那次是题偏。"

母亲没有再问。她坐在对面，自己饭碗里的饭几乎没动。陆允川吃了几口茄子，抬头看她。"你怎么不吃？"

"我不饿。早上吃多了。"

"你又没吃早饭。"

母亲笑了一下。那是说谎被拆穿的笑，但也不解释，只是拿起筷子夹了一点青菜放进碗里。

吃完饭母亲从冰箱里拿出一个面包，塞进他的书包侧袋。"下午考完可能会饿。这个带上。"

"考场不让带吃的。"

"放在外面，考完出来吃。"

陆允川把面包塞进侧袋，拉上拉链。

到校门口的时候，警戒线已经拉起来了。下午考理综，理科生在同一个时段进考场。校门口比早上安静一点——早上的时候家长挤得很密，现在外头的人少了一些，大概有些家长回去做饭了，打算等考试快结束再来。

母亲站在警戒线外面，没有往里挤。她从包里掏出一瓶矿泉水，拧了拧瓶盖，没拧开，又试了一次。

"我来。"陆允川接过去拧开，递回去。

"你自己喝。天热。"

陆允川喝了一口。水是常温的，带着一股塑料瓶的味道——和平时小卖部买的没什么区别。

"那我进去了。"他说。

"嗯。别紧张。考完我来接你。"

"你不用来，我自己回去就行。"

"我来接你。"母亲重复了一遍，语气不变，但这次没有商量的意思。

陆允川没有再推。他拎着透明笔袋穿过警戒线，在进教学楼之前回头看了一眼。母亲站在校门口那群家长中间，个子不高，被前面的人挡了一半，只露出肩膀和一截矿泉水瓶。她正在用手背擦额头的汗，动作很轻，像怕被谁看见。

下午的理综考完，陆允川走出校门时天还没黑。人群中他一眼找到了母亲——她还站在中午那个位置，手里多了一把遮阳伞。伞打得很低，挡了半张脸，但陆允川认出她的凉鞋和脚边那个布包。

"考完了。"他走过去。

母亲把伞举起来罩住他。"累不累？"

"还行。"

"回家吃饭。"

两个人并肩往回走，母亲收了伞——太阳已经落到楼后面去了，不晒了。她把矿泉水瓶递过来，瓶身被她握得微微变形，标签纸的边缘翘了起来。

陆允川接过去喝了一口。水已经被太阳晒温了。他拧上瓶盖，拎着笔袋，跟在母亲旁边走。

路上碰到程砚和他妈。程砚走在前面，他妈在后面打电话，声音很大。程砚远远看到陆允川，就快步走过来。

"你明天考完有什么安排？"

"什么安排？"

"我们几个人约了去吃烧烤。"程砚压低声音，往自己他妈的方向瞥了一眼，"叶澄去，周长海也去。你也来。"

"好。"

"那就说定了。"程砚说完就走了，没给他反悔的机会。

母亲在旁边听见了，说："去呗，考完了让你放松一下。"

"我还没考完呢。"

"明天就考完了。"

陆允川看了母亲一眼。她说这话的时候语气很平淡，像是在说一件不需要悬念的事实。好像不管明天考得怎么样，他都应该去吃那顿烧烤。

家里的晚饭还是四菜。母亲把他的饭碗压得很实，又在他碗里多夹了一块肉。父亲下班回来，换鞋的时候往他碗里看了一眼，说："明天考英语，你作文注意别跑题。"然后又补了一句，"上次你们英语老师打电话来，说你作文论点不清。"

"知道了。"

"什么叫知道了。"父亲坐下来，拿起筷子，"论点要一句话就说清楚。不要绕。"

"我知道怎么写。"陆允川声音有点大，说完自己也愣了一下。

父亲没再说话。母亲从厨房端汤出来，看了父子俩一眼，把汤放在桌子中间。"先吃饭。"

晚上陆允川在房间里看英语作文模板。他翻到叶澄之前借给他的那本英语错题本，里面有一页写着"作文论点不要超过一行"。他想了想，在草稿纸上写下三个论点模板，每个不超过一行。

明天考英语。如果作文不跑题，他就有希望在英语上拿个好看的分数。至于别的——他看了一眼桌上那个透明笔袋，里面准考证照片上的自己对着镜头面无表情。

他把笔袋拉链拉好。明天考完，就不用再打开了。
