# 楼下有风

## 作品形态

- 篇幅类型：长篇
- 目标字数：10 万字
- 创作取向：温柔现实、青春成长、校园情感
- 目标读者或阅读场景：愿意回看青春里笨拙喜欢、误会和释怀的读者
- 素材来源：`C:\Users\usv\.agents\yjunbook\手机便签\张青`，仅作为情绪和事件种子，正文虚构化处理

## 核心 Premise

> 许亦安在大学里把对林青禾的喜欢写进无数便签，误把等待、嫉妒和被需要当成爱情的证明；多年后，他重返校园，终于明白那段青春不是谁辜负谁，而是自己曾经太不懂得如何温柔地喜欢一个人。

## 主题与核心问题

- 主题：爱不是把对方留在自己的期待里，而是在记住之后仍允许彼此继续生活。
- 核心问题：一个敏感、执拗的男生，怎样从“我这么喜欢你”走到“我祝你自由”？
- 不落入的惯例：不写复合爽感，不写报复，不把女主写成单一伤害者，不把男主写成纯受害者。

## 读者体验

- 温柔：伤感但不怨毒，遗憾但不撕裂。
- 真实：校园细节、手机便签、网吧、小吃街、宿舍楼下等生活场景承载情绪。
- 释怀：结尾不是得到某个人，而是看懂曾经的自己。
- 失败体验：读者不应觉得这是控诉文、狗血虐恋或原始日记照抄。

## 世界、时代与硬规则

- 背景：2010 年代初的普通大学校园，智能手机、QQ、网吧和短信仍是重要交流方式。
- 规则 1：所有情绪必须落到校园生活细节里，不做空泛抒情。
- 规则 2：林青禾不是反派，她有自己的边界、犹豫和生活。
- 规则 3：许亦安的成长必须通过行动体现，而不是最后突然顿悟。

## 核心冲突

- 外部冲突：许亦安想靠近林青禾，却不断遇到未接电话、约定落空、旁人介入。
- 内部冲突：他把被需要当作爱情，把等待当作付出，把嫉妒当作深情。
- 关系冲突：林青禾的退让和回避，让许亦安越来越不安；沈安晴的温柔让他照见自己的含糊。
- 冲突升级方式：靠近 → 误读 → 等待 → 嫉妒 → 裂缝 → 离校 → 重返 → 释怀。

## 叙事策略

- 人称与视角：第三人称有限视角，贴近许亦安。
- 时间结构：主体顺叙，末部跳到毕业后和多年后。
- 信息披露方式：便签、照片、电话记录和校园场景逐步显影。
- 节奏偏好：温柔现实，慢热推进，每章有生活场景和心理变化。

## 写作风格

- 默认风格：温柔现实
- 语言要求：自然、克制、细节驱动，少用大词和控诉。
- 诗歌要求：正文中嵌入至少 30 首便签诗，模仿素材中“数学意象 + 青春记忆 + 花风窗电话 + 自问式抒情”的写法，但不照抄原文。
- 对话要求：口语化、轻微笨拙，保留大学生玩笑和回避。
- 禁止事项：禁止爽文腔、狗血虐恋、直接照搬素材、女主妖魔化、男主自我美化。

## 技法偏好

- 白描：用日常动作和物件承载情绪。
- 不写之写：关键告别不说破。
- 层峦迭翠：楼下等待、网吧照片、吸管、风等意象前后回响。
