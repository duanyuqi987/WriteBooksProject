# 第069章 墙角的试卷旁观

所属部卷：第二部：六月以后
时间：高考后第二天
地点：陆家小院的东屋

## 诗歌插页 P069：原创现代诗

我把校徽旁的清晨擦亮，
不问它从哪一年开始发凉。
有人经过我的青春，
像风经过一排没有关紧的窗。
我终于学会不把名字攥疼，
只让灯亮一会儿，再让天亮。

如果把镜头从陆允川身上稍稍移开，陆家小院的东屋里还有别人的呼吸。夏知遥在家帮父亲整理行李，未来对她来说同样陌生。这并不能抹掉他的疼，却能让疼不再只有一个方向。

时间在这里不是背景，而是一层慢慢压下来的纸。2010年的风、2012年的阳光、2013年的寒夜，后来都被他叠在同一本日记里。每一层都薄，叠多了却有重量。所以这一章不只写一件事，也写那件事在岁月里怎样慢慢改变形状。那些形状有时锋利，有时柔软，最后都被他放进更宽的生活里。

教室后墙的钟走得很响，每一下都像把某个年轻人的借口往前推一点。陆允川站在那样的声音里，忽然觉得自己并不是故事中心，只是众多走过校园的人之一。这个发现当时没有让他轻松，反而让他更想抓住一点能够证明自己的东西。

从朋友和她的角度看，陆允川把课本和卷子堆在墙角，三年变成一摞纸，他看了很久。并不是单线条的伤心故事，而是几个人各自笨拙地保护自己。他后来反复想起这一段，不是因为它多么宏大，而是因为试卷边缘干掉的修正液。具体的物件比誓言诚实，它不替人辩解，也不替人作证，只安静地留在光里。

他把纸页抹平，指腹在折痕上停了一会儿，像想把那条细线按回从前。他想说的话有很多，可真正到了嘴边，只剩下几句平常的话。年轻时他总觉得平常太轻，装不下喜欢；后来才知道，太重的喜欢常常先把自己压弯。

雨后的操场有一种塑胶和泥土混在一起的味道。鞋底踩上去会带起一点水，他低头看着那些短暂的脚印，心里想，原来痕迹也会很快干掉。这些细枝末节在当时没有被他认真看见。他忙着判断自己有没有被在意，忙着从别人的一句话里找温度，反而忽略了日子本身怎样一格一格向前挪。

“你以为你在等她，其实你在等一个能证明你没白疼的答案。”祁野这样说时，语气并不郑重，甚至带着一点不耐烦。陆允川当时听不进去，只觉得朋友站得太远。后来他才知道，太近的人往往只能看见火，站远一点的人才看得见整间屋子正在冒烟。

关系在这里往前挪了一点，也往旁边偏了一点。他第一次感到空，空得连喜欢都找不到落脚处，旁观者看得更清楚，也更无能为力。陆允川那时没有能力准确命名这种变化，只能把它写进纸页，写得越满，越像害怕空白。

祁野把话说得很糙，却句句落在地上，不像安慰，更像把人从水里拎出来。程砚催他去网吧，他说再等一会儿。朋友的声音并不总是温柔，有时甚至粗糙，可它们让他看见世界并没有随着一段关系变窄。

朋友们也各有难处。程砚要准备考试，祁野担心家里生意，叶澄给社团收拾烂摊子。可他们仍然在某个晚上分出一点时间听他说话，这一点后来被他记了很久。这些周边生活把爱情从高处拉回地面。它提醒陆允川，心动不是唯一的课程，失落也不是唯一的作业；人总要在很多不浪漫的事情里，慢慢学会怎样对待一件浪漫的事。

当他把她从怨气里放出来，她才重新成为一个普通人，一个会犯难、会犹豫、也会选择自保的人。夏知遥在家帮父亲整理行李，未来对她来说同样陌生。这部分事实后来才被陆允川补进小说。补进去以后，故事没有变得轻松，却变得诚实了一些。

食堂的塑料凳被拖来拖去，贴着地面发出闷响，所有人的青春都显得很忙。他把这一幕和诗歌放在同一页，并不是为了显得文雅。诗对那时的他来说，是一块可以暂时放下重物的石头；重物还在，只是手终于能松开片刻。

原创现代诗写完以后，他在下面空了一行。那一行空白比诗句更安静，像把解释留给未来，而不是逼当时的任何人马上回答。

这首现代诗没有押韵，也不急着把意思说满。它像陆允川当时的呼吸，长短不齐，却正因为不齐，才让人相信它不是从模板里抄出来的。因此这些插页不是装饰，也不是为了让青春显得更漂亮。它们更像一排小灯，照着他从自我感动走向自我审查的路；灯光不强，但足够让人看清脚下有水。

那时他还不懂，一个人的沉默不一定是审判，也可能只是她不知道怎样在不伤人的情况下保住自己。他越长大，越不愿把这段青春说成谁辜负谁。辜负当然存在，可它不是全部；全部里还有胆怯、误会、虚荣、善意和没学会好好告别的年纪。

宿舍里有人洗头，有人背单词，有人对着游戏输赢骂得很响。陆允川坐在这些声音中间，忽然觉得自己的难过没有想象中那么特殊，却也没有因此减轻多少。这一类日常在小说里反复出现，是因为陆允川后来相信，真正改变一个人的不是某个爆炸瞬间，而是爆炸之后仍要买饭、交作业、赶公交、给手机充电的无数普通时刻。

旧稿里他总急着解释自己为什么会那样，现在他愿意让读者自己看见。看见一个年轻人怎样坐立不安，怎样说错话，怎样把手伸出去又收回来。他没有因此变得完全宽容，也没有把所有疼痛都解释成成长礼物。他只是终于愿意承认，自己和别人都曾经笨拙，而笨拙不必永远被写成罪。

他把消息写好又删掉，最后只剩一个句号，孤零零地停在输入框里。这个动作很小，小到放进当年的日记里可能会被嫌不够痛。可真正能留下来的，常常正是这些不够痛的小动作，它们让人相信那一年确实活过。

在这一章的末尾，陆允川没有得到答案。屋外有人晾麦子，尘土和阳光一起浮在窗前。他只是比上一章多看见了一点自己，多承认了一点别人，多放松了一点攥紧的手。

时间并不负责把一切变好，它只把事情放远一点，让人终于能看见当年被情绪挡住的另一半事实。灯光落在地面上，薄薄一层，像谁把话说到一半就收住了。故事继续往前走，像水流过桥洞，没有替任何人停下，却把所有经过都带成了更低的声音。
