# 第073章 网吧通宵之后

所属部卷：第二部：六月以后
时间：高考后的夜晚
地点：县城老网吧

## 诗歌插页 P073：原创仿宋小令

风入旧窗轻，
纸上有余声。
欲问前程何处去，
水边灯影自分明。
回首不须惊。

天亮的时候，网管把他们叫醒了。

"六点了。通宵时间到了。"

程砚从桌上弹起来，脸上压出一道红印——是键盘的轮廓。他眨了几下眼睛，看着周围空了大半的机位。"几点了？"

"六点。"

"靠。"程砚抹了抹嘴角的口水印，"我昨晚几点睡的？"

"你四点就趴下了。"周长海摘下耳机，眼睛里全是血丝，"我打到五点半。最后一把终于进了决赛圈，然后被一枪爆头。"

"那你不是白熬了。"

"怎么白熬，我打败了四十七个人。"

"打败四十七个没用，活到最后一个才有用。"

"你管我。"

陆允川活动了一下脖子。在椅子上靠了一夜，颈椎发僵。他把外套从肩膀上拿下来穿上——清晨的温度比夜里还低。窗外菜市场的顶棚上落了几只鸟，灰扑扑的，应该是麻雀。

三个人收拾好各自的东西——泡面桶、矿泉水瓶、揉成团的纸巾——通通塞进网吧的垃圾桶里。程砚把会员卡揣进兜里，说下次还来。

"大学开学前再来一次。"他说。

"你上次通宵完说再也不想通宵了。"陆允川提醒他。

"那是上次。现在不一样了。"

下楼的时候，楼梯间的声控灯没亮——大概坏了。三个人摸着黑往下走，程砚走在最前面，到了门口一把推开铁门，外面已经很亮了。

清晨六点的县城街道和傍晚完全不同。早点铺子刚开门，蒸笼冒着白汽。环卫工人在扫街，竹扫帚划过水泥地面发出沙沙的声音。几个老大爷在菜市场门口聊天，手里提着刚买的油条和豆浆。

"饿了。"程砚揉着胃，"谁有钱？"

"你卡里不是还有三十吗。"周长海打了个哈欠。

"那是网吧的钱。现在我需要真实的人民币。"

陆允川从兜里摸出十块钱。"一人三个包子。"

"你哪来的钱？"

"昨天出门前我妈塞的。"

"你妈真好。"程砚接过钱，"我妈只会问我要不要带外套。"

"我妈也问了。"

"天下的妈是不是统一个出厂设置。"

包子铺在街角，老板是个胖大姐，看到三个脸色发青的男生就知道是通宵出来的。她麻利地给他们一人两个肉包一个菜包，又送了三杯免费豆浆。"高考完了吧？"

"嗯。"程砚咬了一口包子，"您怎么知道？"

"每年这个时候网吧通宵的都是毕业生。我在这里卖了十几年包子，看惯了。"

陆允川端着豆浆站在路边喝。豆浆很烫，杯口糊了一圈白色的蒸汽。清晨的阳光从街对面的楼后边漫过来，把他的影子投在身后。

"你说，大学新生报到那天会不会也是这样？"程砚忽然问。

"什么这样？"

"就是天刚亮，路上没人，我们背着包往外走。"

"那你得早起。大学生一般起不来。"周长海说。

"我在说感觉。不是在说几点起床。"

陆允川喝完豆浆，把杯子扔进路边的垃圾桶。他想了想程砚的问题。"应该不一样。但至少街上也有早点铺子。"

"那就够了。"程砚把最后一个包子塞进嘴里，含糊不清地说。

三个人在岔路口分开。周长海往东走，他家在菜市场后面那条巷子。程砚往南走，他妈已经在打电话催他回去吃早饭了。陆允川往北走，那个方向有个小公园，穿过公园就是他家。

他经过公园的时候，看到几个老人在打太极。音乐放得很小声，是一首他叫不出名字的古筝曲。他想起昨晚在网吧打开的聊天软件——夏知遥的头像灰着。他打开又关掉了，没有发消息。

这个动作在很多年以后可能不会有人理解。但此刻他站在公园边上，太阳刚刚升过屋顶，老人们还在打太极，古筝曲轻轻响着，他忽然觉得，没有发那条消息是对的。

不是因为骄傲，也不是因为胆怯。只是因为时间不对。凌晨四点不该给一个正在睡觉的人发消息。

他穿过公园往家走。到门口的时候，院门虚掩着——母亲给他留的门。推门进去，鸡在院子里踱步，看到他进来就围过来讨食。厨房里飘出煮粥的味道。

"回来了？"母亲的声音从厨房传来。

"嗯。"

"桌上给你留了饼。趁热吃。"

陆允川走进厨房。母亲正在切咸菜，刀起刀落很均匀。他拿起桌上那个饼咬了一口——是早上刚烙的，里面卷了土豆丝。热乎的。

"我昨晚在网吧没发消息。"他忽然说了一句，说完自己也觉得莫名其妙。

母亲切菜的手停了一下。"发什么消息？"

"没事。就是……没发。"

"没发就没发。先吃东西。"母亲继续切菜，语气里没有任何追问的意思。

陆允川又咬了一口饼。饼皮很脆，土豆丝还有点烫。他站在厨房里吃完了一整个，然后回东屋把门关上，倒在床上。衣服没脱，鞋也没脱。

窗外的光越来越亮。他闭上眼睛，很快就睡着了。
