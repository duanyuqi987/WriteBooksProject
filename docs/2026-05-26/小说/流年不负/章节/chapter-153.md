# 第153章 她妈妈
时间：2018年5月下旬
地点：电话
卷次：卷八：旧诗新读

隔了一天。周三晚。

苏晚晴又给妈妈打了个电话。上次通话之后她心里有个疙瘩——不是难受。是好奇。她妈知道得太多了。多到不像是碰巧的。

"妈。你老实跟我说。"

"说什么。"

"相亲的事。是不是不是偶然的。"

电话那头安静了一会儿。电视没开。没有天气预报的背景音乐。只有窗户外面隐隐约约的风声——她妈在阳台上接的电话。

"不是。"

苏晚晴拿着电话的手紧了一下。

"是你。还是他妈妈。"

"是王婶。"她妈顿了顿。"王婶就是那个媒人——她跟顾家的妈是小学同学。你高三毕业那年夏天。顾家妈妈回青阳——她虽然搬到运河镇了，但老房子还在青阳。那年夏天在街上碰到王婶——说到孩子的婚事。"

苏晚晴静静听着。

"顾家妈妈说——她儿子心里有人。王婶问是谁。她说了你。"

窗外有鸟飞过。翅膀的影子从窗帘上掠过去。大概是一只夜鸟。体型不大，飞得很快。

"那年你才大一。大一上学期。顾家妈妈就记住了你。"

"然后呢。"

"然后她每年过年回来。都会找王婶打听你。你有没有对象。你过得怎么样。"她妈用很平的语气把这些事一件一件摊开。"但她不让王婶来找我。她怕你还在读书——不合适。"

"不合适。"

"她是老师。知道学生的心思。她说你还在上学——如果他儿子配不上你。不能拿这件事去耽误你。"

苏晚晴把手机换到左手。右手握住自己左手腕——手指在上面摩挲着腕骨。腕骨突出，皮很薄。能摸到脉搏——咚。咚。咚。

"所以你们就一直等着。等到我毕业。"

"也不是等。就是放在那里。后来你毕业了——到楚州工作了。顾家妈妈去年也搬到了楚州——不是巧合。"

"她搬来是为了——"

"有一部分是。顾家妈妈在楚州买了房子。离你学校不远。她说——如果她儿子能跟你走在一起。以后方便。"

苏晚晴把手从腕上拿开。放在书桌上。桌面的塑料垫板是凉的。她手指在垫板上写了两个字——写完了才意识到是"临风"。

"妈。你什么时候知道这些的。"

"去年。王婶来找我的时候。她把这些全告诉我了。我说——那得看两个孩子自己怎么想。"

"你们就安排了相亲。"

"不是安排——就是。你刚好没有对象。他也没有。王婶说有一个人想介绍给你——我不说你也知道是谁。"

"你知道我一听姓顾就会想起来。"

她妈在电话那头没有回答。但苏晚晴能听到一声极轻的呼吸——是她妈在微笑。

"那场相亲。你以为我只是劝你去见一面——其实我知道。你一直记得他。"

苏晚晴没说话。台灯的光照在她手背上。手背上有写粉笔磨出来的薄茧。她用另一只手的拇指去搓那颗茧子——搓了一遍又一遍。

"妈。你怎么知道我记得他。"

"你有一本笔记本。高中用的。蓝色封面。你夹在那个本子里的一张成绩单——那是高一期中考试。你用红笔圈了一个人的名字。名字是——顾临风。物理年级排名第八十六。"

她说不出话。那个笔记本是她整理书架的时候翻出来的。成绩单夹在中间那一页。纸上已经泛黄了。红笔圈的那个名字，墨水褪成了浅红。她十年没碰那个笔记本——她妈翻到过。

"你什么时候看到的。"

"去年。帮你收拾旧书的时候。那个本子掉下来。成绩单从里面滑出来——飘到地上。我捡起来看了一下。看到红圈。"

"你从来没问过我。"

"你的事我不全问。但我会记住。"

苏晚晴把手从腕上拿下来。放在被子面上。被面是棉的——有点粗糙。手指在上面来回走了两圈。她想起七年前——高二那年。有个物理年级排名第八十六的男生。一年后变成了年级前二十。再过一年——年级前三。物理满分。那个在成绩单上画圈的人——圈的是他起跑的地方。不是他后来的位置。

"妈。那场相亲。你们在上面花了多少功夫。"

"不多。就是等了八年。"

等了八年。跟她等了八年。跟他等了八年。原来这个世界上——在运河边、在青阳街、在老槐树底下——还有另外两个人也在等。等得最安静。

"妈。"

"嗯。"

"谢谢。"

她妈在电话那头轻轻应了一声。没有说"不客气"。只是说"早点睡。明天还要上课"。

"好。晚安。"

挂电话之前。她妈妈又补了一句。

"晚晴。"

"嗯。"

"那个小鱼。还在包里吧。"

苏晚晴愣了一下。手不自觉地摸到了枕边——小鱼刚从包里取出来，放在枕头边上。鱼身上的裂纹在指尖下细细划过。

"你怎么知道小鱼——"

"那次在书店门口。你背的书包上挂着一个小鱼挂件。大概是陶瓷的。后来那个挂件不见了——你换了个书包。"

"你看到过。"

"看到了。挂了三年。高二就没了。我以为你弄丢了。但你没有。"

"你怎么知道我没有——"

"你换书包的时候。把小鱼从外面挪到里面了。妈妈不用开你的包——妈妈只要看你每天打开书包时的表情就知道了。每次打开包的时候。你的手先伸进内侧口袋。摸一下。确认它还在。然后才拿书。"

苏晚晴说不出话。过了好一会儿——

"妈。你观察得太细了。"

"做妈妈的。看女儿。跟你看你的学生一个道理。"

她妈妈在电话那头轻声笑了一下。然后挂了。

嘟——嘟——嘟。

忙音。窗外又有一只鸟飞过。这次是蝙蝠——在路灯下扑了三四下，飞到月湖的方向去了。影子在地上拉了一道黑的横线——然后消失了。

她躺在床上。天花板是白的。灯的影子从天花板一角往下淌。她想起很多年前——运河镇的下午。顾临风在运河边上捡到一颗小鱼。他把它洗干净，揣在左边口袋里。骑车来的路上手里全是汗。他把鱼放在青阳中学对面——那棵老槐树下的长椅上。跟她肩头的距离——隔了一个书包带。

她闭上眼睛。左手还攥着小鱼。右手指尖轻轻划过鱼肚子上的裂纹。裂纹很细。往上是鱼眼——那两个小黑点。黑点不褪。这么多年了。油墨已经渗进了陶瓷里面。

手机亮了。微信。

顾临风：睡了？

苏晚晴：没。

顾临风：我刚跟我妈打完电话。

苏晚晴：我也是。

过了三秒。

顾临风：我妈说。你妈什么都知道。

苏晚晴：我妈也是这么说的。

又过了三秒。

顾临风：所以那两个老太太背地里都干了什么。

苏晚晴：等。像我们一样。等了八年。

屏幕那一端安静了大概十秒。然后跳出两个字。

顾临风：值了。

苏晚晴把手机放在胸口上。屏幕的光透过被子——一小片蓝白色。她能听见自己的心跳——不是急促。是稳。一下。一下。一下。像月湖水面上的波纹——散开，散远。碰到柳树根。再荡回来。
