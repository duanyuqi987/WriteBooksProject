# Skill 查找系统

根据用户的 prompt 意图，快速匹配最相关的写作技巧、第三方技能和方法论。

---

## 查找流程

```
用户输入 prompt
  ↓
1. 提取核心意图关键词
  ↓
2. 匹配写作技法（10 种核心技法 × 场景类型表）
  ↓
3. 匹配第三方技能（29 个小说相关 × 技能描述）
  ↓
4. 匹配风格文件（3 种预设 × 小说类型）
  ↓
5. 返回 top 3-5 推荐结果
```

## 五、查找算法（供 AI 内部使用）

当接收到用户 prompt 时，执行以下步骤：

```
1. 提取用户 prompt 中的关键词
2. 在"用户说了什么"列中做模糊匹配
3. 在"当前在写什么场景"列中做场景识别
4. 在"小说类型"中做类型判断
5. 综合所有匹配结果，返回 top 5 推荐（含技法+第三方技能+风格文件）

推荐格式：
🔧 推荐技法：技法名（匹配原因）
📦 推荐技能：第三方技能名（匹配原因）
🎨 推荐风格：风格文件名（匹配原因）
📋 下一步：建议的具体操作
```
