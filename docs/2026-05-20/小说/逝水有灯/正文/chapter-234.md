# 第234章 诗歌索引旁观

所属部卷：第五部：逝水有灯
时间：同一周
地点：自己的书房

## 诗歌插页 P234：原创现代诗

朋友说
别把诗写成遮羞布
我把这句话
夹进索引
让它比许多诗
更像诗

叶澄后来也看了几页诗歌索引。她已经很久不参与陆允川的写作，只是偶尔听他说起这本小说。那天她在电话里沉默了一会儿，说：“你这些诗有几首很美，但我更在意它们会不会把人写偏。”

陆允川问：“哪里偏？”

“比如你写苏晚枝，不要总写她像风、像灯、像你够不着的地方。她不是意象。”叶澄的声音还是那样稳，“她是会赶论文、会累、会烦、会怕麻烦的人。夏知遥也一样。”

陆允川握着手机，看着索引表格里那一列“人物关联”。许多章写的是物件，背后却仍藏着他的偏心。他以为自己已经把人物放出来了，可朋友一眼就看见，诗有时又把她们推回了象征。

“我改。”他说。

“不是为了显得你多尊重，”叶澄说，“是为了让她们在书里能站住。”

电话挂断后，陆允川把索引里所有“她像”开头的备注删掉，改成动作：她收文件夹，她按灭手机，她把伞靠在门后，她保存照片后继续整理实习材料。那些动作没有意象漂亮，却更像一个人。
