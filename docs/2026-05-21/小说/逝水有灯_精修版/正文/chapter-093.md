# 第093章 夏天的聚餐之后

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P093：原创仿宋小令

风入旧窗轻，
纸上有余声。
欲问前程何处去，
水边灯影自分明。
回首不须惊。

宴席散场的时候，程砚站在饭馆门口送人。他那件蓝衬衫第一颗扣子又被扣回去了——他妈妈趁他不注意扣上的。

"你在门口站得像迎宾。"陆允川最后一个走出来。

"我就是在迎宾。我妈说要有始有终。"程砚的声音有点哑——刚才说了太多话，喝了太多茶。

两个人站在饭馆门口的人行道上。八月的夜晚还是闷热，地面蒸了一天的热气还没散完。街对面的水果店还开着，老板娘在收摊，把一箱一箱的西瓜搬进店里。

"今天谢谢你帮忙应酬。"程砚说，"我妈那边的亲戚我都不太会打交道。你在旁边帮我说了几句，我妈回去肯定会夸你。"

"我没帮什么。"

"帮了。我那三叔问我机械工程是学什么的，我说了半天他也没懂。你在旁边说了一句'就是造汽车发动机的'，他立刻就懂了。"程砚笑了一声，"你怎么什么都能一句话说清楚？"

"跟你学的。你当年跟我解释物理题的时候也是这么干的。"

"我那叫把复杂问题简单化。你那叫把抽象概念具象化。还是你厉害一点。"程砚拍了拍他的肩膀，"我先进去了。还有几个亲戚没走。"

"好。"

陆允川往家走。路口碰到了叶澄和林曼，她们俩一起往东边走——林曼要去叶澄家住一晚。叶澄说她妈今天炖了银耳汤，多了一碗。

"陆允川。"林曼叫住他。

"嗯？"

"今天程砚他妈说你最好的朋友。你低头喝茶的样子，我看到了。"

"……你观察力真好。"

"不是我观察力好。是你们这些人表情太透明了。"林曼笑了，"程砚他妈说那句话的时候，你低头喝茶，程砚在旁边看你的眼神也很复杂。"

叶澄在旁边没说话，但她的眼神表达了同一种意思。

"你们想说什么？"

"我们想说——"林曼看了一眼叶澄，叶澄微微点头，林曼接着说，"你以后去了大学，多交点朋友。你这个人容易把所有的情感投资都放在一两个人身上。这样不好。"

"谁说的。"

"我说的。叶澄也说过。"

叶澄开口了："上次你在走廊里跟夏知遥说'祝你大学顺利'之后，我给你的评语是——你终于学会了在看懂别人之后再行动。但我没说的是——你看懂之后，要学会往前走。不要停在走廊里。"

陆允川站在路灯下，光线在他脸上投了一半明一半暗。"我没有停在走廊里。"

"你有。你只是把走廊从学校的走廊换成了自己心里的走廊。"叶澄的语气像在批改一篇作文的最后一段，"但没关系。大学还有四年。你可以慢慢走。"

叶澄说完这句话，林曼在旁边拉了拉她的袖子。"走吧，银耳汤凉了就不好喝了。"

两人转身往东走。陆允川往北走。路灯把三个人的影子往不同的方向拉。

回到家，母亲还没睡。她在缝一个布袋子，说给陆允川带去学校装零钱。布袋子用的是一块旧衬衣的布料——他初中时候穿的一件格子衬衫。陆允川看着那块布，想起了那件衬衫的袖口，磨破过，被母亲补过一次。

"这个布袋子能不能不要用这件衬衫。"他说。

"为什么？"

"这件衬衣我挺喜欢的。留个完整的。"

母亲看了他一眼，然后把布料折起来。"那我明天重新找一块。"

"谢谢妈。"

"去洗澡。你身上一股火锅味。"

陆允川去洗澡。热水冲下来的时候，他想起今晚桌上那些人说的话。程砚的妈妈说他最好的朋友。林曼说他表情太透明。叶澄说不要停在走廊里。夏知遥的红包上写"前程似锦"。

他把水开大了一点。蒸汽把镜子糊住了。他用手指在镜子上画了一道弧线——是一个像桥一样的形状。土木工程。要学五年。
