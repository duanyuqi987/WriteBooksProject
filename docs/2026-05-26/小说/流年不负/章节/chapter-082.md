# 第082章 十二公里

时间：2013年9月

地点：楚州师院

卷次：卷五：各自南北

开学第一周，他做了一件事。开学第二天晚上，宿舍还在闹哄哄地认识彼此——阿坤在讲他的魔兽段位，李博在整理书桌，小伍在擦他那双新买的军训胶鞋。顾临风搬了凳子坐在阿坤的桌子前面，说"用一下你的电脑"。

阿坤的笔记本是联想牌的，屏幕有点小——十四寸，分辨率不高，字看起来有点糊。键盘上的字母"WASD"四个键磨得发亮——这是打游戏最常用的几个键。笔记本的散热器嗡嗡地转——出风口吹出来的风是热的。他打开浏览器。浏览器的主页被阿坤改成了游戏论坛——上面有一排帖子，标题都带着"副本""装备""加点"这些词。他新建了一个标签页。搜索框里打了一行字，用的是拼音输入法——"楚州师院到楚州师大"。

回车。硬盘转了一下——咔嗒。

页面跳出来一条结果。301路。11站。大约四十分钟。搜索结果底下还有一个小字标注——"全程票价2元，刷卡1.6元"。

起始站：师院南门。

终点站：师大北门。

中间九站——市图书馆、市体育馆、中山路口、公园西门、工商银行、人民医院、新华书店、广场东路、师范大学。

每一站的站名都是蓝色字体——可点击的链接。他把鼠标移到第一个站名上——光标变成了手指。没有点。移到第二个。又没有点。依次移过九个站名。移到最后一个——"师大北门"——的时候鼠标在那个蓝色的字上面停了几秒钟。食指在鼠标左键上轻轻搭着。没有按下去。

然后他把鼠标移到别处。打开了手机备忘录。九键拼音输入法。一个一个地打。市——图——书——馆。按了七八下键。手机键盘很小，他的拇指有点大，按错了三次，删了重新打。删除键要按两下才能删一个字。手指在数字键和方向键之间来回跳。打完以后他又看了一遍。一个站名一个站名地对——电脑屏幕上的和手机屏幕上的。有一种在做数学习题最后检查答案的感觉。没有漏。没有错。十一个站名——包括起点和终点。

301路。师院南门到师大北门。全程大约12公里。

他盯着屏幕上的数字看了十几秒钟。十二公里——换算成他熟悉的东西。青阳县城从南到北大概三公里——十二公里是四个青阳县城的长度。青阳一中到月湖公园大概两公里——十二公里是六个来回。操场跑道一圈四百米——十二公里是三十圈。他在操场上最多走过五圈。三十圈他没有试过。

然后他又搜了一个页面——楚州公交线路图。地图加载得有点慢——图片一格一格地出来。是绿色的底色，公交路线用不同颜色的线标出来。301路是蓝色的——一条蓝色的线从地图左下角弯弯曲曲地往右上角延伸。他滚动鼠标滚轮把地图放大。那条蓝线穿过楚河上的中山桥——桥标注了名字，字体是斜的。拐进老城区——路网密集，像一张蜘蛛网。最后停在标注"楚州师大"的红点旁边。他把地图放大了最大倍率。师院南门是一个蓝色的小圆圈——旁边标了公交图标。师大北门是另一个蓝色的小圆圈。两个圆圈之间——是那条蓝色的线。线的每一个弯折他都看了一遍。在中山路口拐了一个大弯——大概是因为那条路是单行道。在人民医院门口有一个小小的折角——可能是绕过了医院门口的环岛。在新华书店那一段是直的——沿着主干道走了大概两公里。他用手指在屏幕上量了一下——拇指和食指打开，从师院南门拉开，拖到师大北门。手指松开——地图弹回去了。

他看了大概五分钟。

阿坤从上铺伸头过来。脖子探得很长——能听到脊椎骨咔嗒一声。他看了一眼屏幕——没看仔细，只看到一张地图。"找什么路子呢？想出去逛？"

"随便看看。"

他把浏览器关了。鼠标移动到右上角的叉——点了。点击声很轻。硬盘又转了一下。把电脑还给阿坤——合上屏幕的时候铰链有点紧。站起来，走到窗边。用手撑着窗台。

窗外那栋远处的宿舍楼还是灰色的。天空有点阴——云层很低，颜色像被水泡过的报纸。楼下有人在晒被子，用一根竹竿拍打着被芯。啪啪的声音隔着玻璃传进来，闷闷的。被芯上腾起细小的灰尘——在灰色的天空里几乎看不见。晒被子的人嘴里哼着一段曲子——跑调了。

他拿出手机。翻到备忘录——301路、11站、12公里。

然后把手机合上，放在桌上。合上手机的时候屏幕闪了一下——显示时间11:23，然后黑了。

下午他去上了第一节专业课——现代汉语。教室在三楼，能坐一百多人。桌椅是那种阶梯式的——从门口往里往下走，一排比一排低。桌面是米黄色的，边角贴了一层塑料封边——有几处封边翘起来了。椅子是翻座的，坐垫上有一块污渍——墨蓝色的。他坐在靠窗的第三排。把书包挂在椅背上——椅背的挂钩缺了一个螺丝，书包总往下滑。老师姓刘，五十多岁，头发是灰白的。说话很慢——慢到你可以把他说的每一个字都记下来。他穿了一件灰色夹克，袖口磨毛了。阳光从窗户斜进来，在课桌上切出一道明暗分界。他把笔记本放在明的那半边——墨水的字在太阳光下是深蓝色的。

"现代汉语——研究的是我们每天都在用的语言。你所听到的每一句话——从母亲叫孩子起床，到电视里播新闻，再到你们现在在心里对自己说的那些话——都是现代汉语研究的范围。"

他在笔记本上写下"现代汉语"。然后在本子右下角画了一个圈。圈画得不圆——像一只压扁的鸡蛋。又在圈外面画了一条线——线画出去以后停了。他不知道线要往哪去。线就那样悬着——从圈的边缘往外延伸了两三厘米。

下课的时候教室里的日光灯自动亮了——窗外天色转了。他收拾书包。把笔放进笔袋，把笔记本放进书包，把椅背上滑下去的书包重新挂上。经过学校南门的时候是下午五点出头。南门有两扇铁栅栏。铁栅栏上的漆是暗绿色的，被太阳晒得发灰。其中一扇的下角有个瘪痕——大概是被人撞过。门口对面就是公交站。

他把脚步放慢了。步子从快步变成了走步，又从走步变成了——几乎是停下来。鞋底蹭着水泥地面。

站牌是绿色的铁牌子，大概一米八高。上面印着线路图——白底黑字，外面罩了一层透明塑料板。塑料板上有几道划痕。三条线路——18路、25路、301路。301路排在第三行。站牌的杆子底下锈了一截——褐色的铁锈从根部往上蔓延了大概二十厘米。锈斑的边沿是不规则的，像一张地图。大概是冬天撒了融雪剂。站牌顶端落了鸟粪——白色的，干了，龟裂成了几块。

他站在站牌前面。逐个看那个站名——市图书馆、市体育馆、中山路口、公园西门、工商银行、人民医院、新华书店、广场东路、师范大学。每一个名字对应一个他没去过的地方。市体育馆——他在网上看过图片，是圆形的建筑，顶上有个大钟。能把整点报时的钟声传到方圆两公里。人民医院——他在楚州的第一天就看到过，白色的大楼，楼顶有个霓虹十字架。晚上会亮。新华书店——门面是红色的，不知道跟青阳那家是不是连锁的。师范大学——倒数第二站，离终点站只差一站——大概就是隔了一条马路。

师大北门。终点站。站牌上的字比别的站名粗了一点——大概是加过粗。

他伸出手摸了一下站牌的边沿。铁皮是热的——被太阳晒了一下午。热从指尖传过来——不烫，就是有一种在太阳下烤了半天的铁器的干热。他手指上沾了一点锈屑——褐色的粉末，蹭了两下没蹭干净。

一辆公交车开过来了。他先听到了引擎声——从远处传过来的低频轰鸣。然后是车头拐进车道时轮胎与地面摩擦的声音。蓝色的车身，侧面刷了一条白色的腰线。车头贴着红字——"301"。红字是贴纸——"3"的边角有点脱胶，翘起来一块。前门打开——气刹的放气声。车身的引擎嗡嗡地响着。排气管冒出一股黑烟——黑烟在空气里翻滚了两下就散掉了。车上下来三四个人——一个提菜的阿姨，塑料袋里装着芹菜和土豆。两个背书包的学生——一男一女，大概是同一个学校的，但没说话。一个夹着公文包的中年人，下车的时候用手捂了一下领带——怕被车门的夹缝卡住。又上去五六个人。前门关上——橡胶密封条挤在一起的闷响。车发动。尾灯闪了一下红色。往东边开走了。

往师大方向开走了。尾灯在视线里越来越小。红色的两点——在灰色路面上慢慢模糊。

他没有上车。

他站在站牌旁边看了大概两分钟。等第二辆301路开过去。这一辆人少——车窗里只有两三个乘客。一个穿灰色外套的男生靠着车窗在看手机。后面坐了一个老太太——菜篮子放在旁边的座位上。靠窗的位置空了好几个——第三个窗户、第四个窗户、第五个窗户——都空着。那个靠窗的位置——如果坐上去，可以看到的风景会依次是：师院南门的站牌、市图书馆的红色砖墙、市体育馆的圆顶、中山桥下的楚河水、人民医院的白色大楼。然后到师大北门——终点站。然后就可以下车了。

但他没有上车。那天没有。这学期没有。这一年没有。

他站在站牌旁边又看了几十秒。第二辆301路完全消失在马路尽头了。然后转身，走进了校门。校门口的保安坐在值班室里——窗户开着。他路过的时候保安抬头看了他一眼。没说话。他也看了保安一眼。也没说话。这种沉默里有一种彼此知道的名堂——他们以后会经常这样互相对看但不说一句话。

回了宿舍。走廊里有人在踢毽子——彩色羽毛毽子，在五楼的高度确实有点危险。他从旁边绕了过去。

李博在自己的书桌前整理选课表。桌上的台灯亮着——灯罩是绿色的，拉出了很长的影子。他面前摊了三张打印纸——课程总表、选修课列表、体育课列表。每张纸上都用红笔画了圈。他拿着铅笔在纸上勾画——动作像在研究作战地图。"你选修课选了吗？"

"还没。"

"抓紧。有的课名额满了。计算机基础只剩下六个名额。你要是想选周二下午的那个书法课——那个还剩十四个位置。"李博把笔倒过来，用笔帽那头点了点选修课列表。"对了——你想选什么方向的？"

"都行。"

李博看了他一眼——眼镜后面的眼睛眯了眯。"都行是什么意思。"

"什么有意思就选什么。"

他选了最多的选修课。中国古代文学、西方哲学导论、书法基础、中国历史地理。在电脑上勾选的时候手指在鼠标上点得很快——左键、左键、左键、左键。四门课的名称前面的方框里依次出现了对号。系统提示总学分23分——超出建议学分5分。他还是点了提交。周一到周五每天都有课，周六上午也有两节。选修课比必修课多了一倍。课表打印出来——一张A4纸，上面排满了黑色的字。周一早上八点开始，到周五下午五点四十结束。

李博看了他的选课表——从顾临风手里把那张纸抽过去，看了三秒钟。"你疯了吧。选这么多。"

"闲着也是闲着。"

李博把课表还给他。"你是不是那种不把自己排满就受不了的人。"

"大概吧。"

小伍的课表只有他的一半。小伍看着自己的课表——上面还有很多空白——又看看他的——说"你的课表像一份菜单"。阿坤说"菜单都比你的课表有空白"。顾临风没解释。

晚上熄灯以后。走廊里的声控灯灭了。宿舍里的日光灯也灭了。只有窗外透进来的路灯光——橘色的，在天花板上画了一条窄窄的带子。室友们在聊高中的事。阿坤的床在上铺，他的声音从上面传下来。"我高中翻墙出去上网被逮过三次。第一次被教导主任抓到——写了三千字检讨。第二次被门卫抓到——请了家长。第三次我从厕所窗户翻出去的——你们猜怎么着——教导主任在网吧门口等我。那天是通宵。他比我晚到两分钟。"

小伍在上铺翻了个身——床架咯吱响了。"你们高中那个教导主任怎么知道你在哪个网吧的。"

"全县就三家网吧。他一家一家找的。"

李博说。"我高中是班长兼学生会主席。三年没翻过墙。但我帮同学翻过——给他们打掩护。假装去老师办公室送材料。"

小伍说他的高中有个数学老师外号叫"光头强"——不是因为秃头，是因为他会砍掉所有不及格的卷子。砍——是撕掉重做的意思。他说他高一下学期被砍过五次。五次重做——做到最后连题目都能背下来。

话题转到班花。阿坤说他高中班花长得像刘亦菲，但后来发现她发质不好——"远看是刘亦菲，近看是刘亦菲的发型师"。小伍说他高中班花是化学课代表——"每次发卷子的时候人都很好"。李博说"你们一天到晚就是班花班花的"。

阿坤把话题抛到了顾临风这张下铺上。"临风你呢？你们高中什么样？"

"普通的。"

"有没有什么特别的事？特别的人？"阿坤的声音里带了一种很轻的调侃——但也不太用力。

"没有。"

他把枕头翻了一面。枕套凉凉的。荞麦皮枕头翻过来的时候能闻到一股干草的味——很像秋天晒过太阳的谷子。手机压在枕头下面。屏幕应该是黑的。他用手指隔着枕头碰了碰手机的轮廓——长方形的，硬硬的，边角硌手。

天花板上那条裂缝还在。路灯光把裂缝的影子加深了。他看了几分钟。闭上了眼睛。

301路车的路线在他脑子里过了一遍——师院南门、市图书馆、市体育馆、中山路口、公园西门、工商银行、人民医院、新华书店、广场东路、师范大学、师大北门。

每站之间大概三四分钟。离开师院南门后——车的速度大概在四十码左右。第一个站是市图书馆。图书馆门口有个喷泉——他在网上搜过图片，喷泉是圆形的，中间立着一个不锈钢雕塑——大概是本书或者什么。水喷出来的时候在太阳下有彩虹。公交车会停在喷泉正对面。车门打开的时候会有图书馆里的冷气飘出来——带一股旧书的味道。上车的人里有学生模样的——大概是去师大的。到市体育馆的时候——车会绕着体育馆的转盘转半圈。转盘上的草坪修剪得不整齐——有一块黄了一块绿的。到中山路口的时候会经过一座桥——中山桥。桥栏杆上锁了一些同心锁——生锈了。桥下的水是绿的，绿得发暗。到公园西门的时候——会在一个十字路口等红灯。红灯大概四十秒。四十秒里——能听到车里的广播在报站，能听到有人在咳嗽，能听到司机的方向盘皮套在摩擦。到工商银行的时候——路边有一排小吃摊：煎饼果子、烤面筋、炸串、烤冷面。煎饼的铁板上冒着白烟——鸡蛋打在铁板上嗞嗞响。到人民医院的时候——有人推着轮椅上下车。司机会把车身降下来——气动悬挂放气的嘶声——翻出踏板。推轮椅的人会先把轮子卡在踏板的槽里，上去以后找一个轮椅专座——蓝色的，靠门最近的。到新华书店的时候——门面是红色的，跟青阳不一样——青阳的是蓝色。书店的橱窗里摆着最新一期的《读者》和《青年文摘》。到广场东路的时候——路两边种着银杏树。银杏叶子到秋天会变黄。现在是九月——叶子还是绿的，但叶缘已经开始有一点黄边了。到师范大学的时候——那是倒数第二站。离师大北门只差一站。中间隔了一条马路——大概是解放路。师大的侧门就在这一站。下了车能看到师大的操场——绿色的草坪、红色的跑道。师大北门是终点站。下了车往右拐——就是校门。校门是灰白色的。门柱上贴着学校的名字——"楚州师范大学"——六个字，是铜字，被太阳晒得有点发绿。

他把路线背下来了。闭着眼睛都能数。不是刻意背的——只是看了太多次，每一个转弯、每一个地标、每一个红绿灯都刻在了该刻的地方。

十一站。十二公里。四十分钟。两块钱。

有时候他会在脑子里坐一遍这趟车。买票——投币两块钱，硬币从投币口滚下去，落在钱箱里叮当一声。坐在靠窗的位置——第三个窗户。右手边的窗外会依次掠过那些站名对应的建筑。每过一站，车上的人少几个。到师大北门的时候——车停了。下车。站在师大校门口。然后呢。

然后没有然后。

他睁开眼睛。天花板上的裂缝还是那条裂缝。

第二天下午他又经过了南门的公交站。他是从自习室出来的——书包里装着刚写完的书法作业，宣纸上写着颜真卿体的三十个字。墨汁的味道还在袖口上。经过那条路的时候步子自动慢了下来。天是阴的。站牌下面站着三个等车的人——一个拎着菜的老太太，菜篮子里装着莴笋和白萝卜。一个戴着耳机的中年人，耳机的线在胸口一晃一晃的。一个抱着书的女生。女生的马尾扎得有点松，几根碎头发落在脖子后面——大概是扎了一天的头发，橡皮筋松了。碎头发在风里微微动着。他看了那个马尾一眼。然后移开了视线。移到了站牌的绿铁皮上。铁皮上今天没有被太阳晒热的温度——因为是阴天。

301路来了。引擎声先到的——然后是车身。前门打开的放气声。三个人都上去了——老太太先上，中年人跟上，那个女生最后。她上车的时候马尾在车门框的阴影里闪了一下——从亮处进入暗处。车开走。尾灯闪了闪。车身拐过十字路口——不见了。

他又站了几分钟。第二辆301路开过去的时候，他数了车上的人数——大概十二三个。有人拉着扶手站着。有人靠窗坐。靠窗的那个位置——第三个窗户——空着。第四个窗户也空着。整排靠窗的位子只坐了一个人——一个戴鸭舌帽的男生。不是她。他把视线移回来。右脚鞋底蹭了一下地面——水泥地上有一颗小石子，被蹭得滚了一下。

他转身进了校门。门口的保安看了他一眼——大概是认出他了。他在门口站太多次了——开学这几天，每天下午经过南门他都会停一停。保安的眼神不是那种警惕的眼神——是好奇的。保安的茶杯放在窗台上——玻璃杯，里面的茶水已经泡得没有颜色了。

保安没说话。他也没说话。

走回宿舍的路上，他在心里把那十一个站名又默了一遍。每一站都出现在该出现的位置。市图书馆是绿的。市体育馆是圆的。中山桥下的水是绿的。人民医院楼顶的十字架是红的。广场东路两边的银杏是绿的——快变黄了。师大北门是两块钱就能到的地方。没有漏。没有错。还是那十一个站名。

十二公里。他没有去过一次。

他遵守了约定。
