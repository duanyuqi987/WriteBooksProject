# 第093章 错题本的折痕



时间：2009年十二月初，分科确认表二次核对



地点：高二十班教室后墙、办公室门口



卷次：第四卷：分科以前的夏天



2009年十二月初，分科确认表二次核对，高二十班教室后墙、办公室门口，闻佳夹在错题本里的分科确认回执把这段日子推到十班面前。分科确认表二次核对，闻佳把回执夹进错题本，却迟迟不交。

她怕去了文科班就被说逃避，怕留在理科班又一直追不上；常俊凡帮她把害怕一条条写出来。这不是突然降下来的大事，而是从早读、课间、饭卡、家长电话和后墙红榜里一点点长出来的。

段璟坐在教室里，常常先看见常俊凡的背影，再意识到自己其实该看卷子。从这个学期开始，这种走神不再只是喜欢，也会影响一道题、一次排名和一个选择。

常俊凡没有变成谁的答案。她会犹豫，会烦，会被家里问得沉默，也会在别人替她下结论时把话说得很硬。

闻佳也在这件事里显出自己的位置。爱照镜子，笔记好看，文科感受力强，怕别人把她的选择说轻。十班从来不是只由两个人组成，它是四十多个人一起挤出来的青春。

“他们肯定说我选文是因为理科学不会。”闻佳说。

“别人怎么说，不能替你填表。”常俊凡说。

“你要是真想去，就别让他们占你的位置。”段璟说。

“你这话不像你。”闻佳说。

“签名是小事，承认自己想要什么才难。”叶书兰说。

错题本摊开时，纸页中间有一道旧折痕，像被反复翻到同一个地方留下的印子。

高二以后，错题不再只是错题。它们像一个个小钉子，把每个人钉在自己真实的水平上。

常俊凡改题时很少抱怨，先标题号，再写原因，最后才写正确步骤。

段璟的错题本更乱，箭头、补充、红笔和黑笔挤在一起，像他不肯承认的慌。

后墙红榜没有新的排名，却已经有人在心里重新排位置。分科之后，比较换了方式继续存在。

老师从办公室出来，手里拿着一沓卷子，大家立刻把桌面收得像什么都没发生。

常俊凡把本子递给别人时，会先把和自己私人情绪有关的纸条抽掉，只留下题目。

段璟看见这个动作，知道她不是没有秘密，她只是比他更懂得把秘密放好。

叶书兰这段时间管得比高一更细。不是因为学生忽然变坏，而是因为分科、晨检、国庆活动和期中考试挤在一起，谁一松就会被推着往后退。

闻佳被叫到名字时，先看了一眼周围同学，再看老师。这个小动作让段璟发现，班里很多看起来不在乎的人，其实都怕被公开放到台面上。

封丘县的校园生活没有电影里那么整齐。校门外有卖烤红薯的，有修自行车的，有家长骑电动车来送饭，也有学生在路边背单词。

黄河故道那边的风有时会吹进县城，吹动校服袖口，也吹得人想起更远的地方。可更远的地方暂时还只是试卷最后一道大题旁边的空白。

国庆六十周年的新闻、甲流晨检的表格、分科确认的回执，都没有替他们解决任何一道题，却让他们知道，学校外面的时间也在往前走。

宋乔会把女生那边的情绪说出来，付小川会用玩笑把男生那边的沉默撬开，周明澈则常常在最乱的时候把表格、名单和班费重新理一遍。

段璟有时羡慕这种确定。他自己更像一张写了太多草稿的纸，明明知道下一步该写什么，却还是忍不住看常俊凡会不会也停在同一个地方。

常俊凡并不总是回头。她更多时候只是把自己的事情做好：改题、接水、记账、扶凳子、把别人的话挡回去，然后继续低头写下一行字。

周明澈把错题本翻到折角那页，先看错因，再看分数，有人把椅子往里收，木腿擦过地面发出短短一声。老师从门口经过，所有人的声音立刻低了半截。

刘青岚在受力图旁边补了一条虚线，线画歪了又重新描，同桌抬头看了一眼，又像什么都没看见一样低头。这样的细节不会出现在成绩单上，却会留在很多年后的记忆里。

孙佳怡把排名纸条从本子里抽出来，夹到看不见的后一页，粉笔灰落在袖口上，拍了两下也没有拍干净。这个动作很小，却比一句大道理更能说明当时的慌。

段璟用红笔圈住题号，手腕上沾了一点墨水，小卖部那边的铃声远远传来，很快被上课铃盖住。旁边的人没有追问，教室里的翻书声把尴尬遮住。

常俊凡问同桌借橡皮，擦掉的不是答案，是自己刚才太急的字，红笔帽没有扣紧，轻轻碰到桌面滚了一圈。常俊凡看见了，也没有立刻说什么，只把自己的题继续写完。

郑天宇把错题本翻到折角那页，先看错因，再看分数，走廊里有人喊了一声报告，很快又安静下去。段璟后来想起这一幕，才知道青春里很多变化一开始都很安静。

蒋一诺在受力图旁边补了一条虚线，线画歪了又重新描，黑板右上角的日期被擦得只剩一点白影。窗外风一吹，纸角轻轻动了一下，像有人把话又咽回去。

薛北辰把排名纸条从本子里抽出来，夹到看不见的后一页，门口冷风进来一小股，很快又被人把门带上。十班的热闹还在，只是每个人都比高一时多了一点分寸。

唐小满用红笔圈住题号，手腕上沾了一点墨水，后墙那张旧座位表在风里轻轻翘起。老师从门口经过，所有人的声音立刻低了半截。

错题本的折痕之后，十班没有立刻发生巨变。真正的变化通常不是一声响，而是某个同学开始少和旧同桌一起吃饭，某个名字开始出现在另一张名单上。

回执最终交上去，闻佳没有立刻变勇敢，只是终于替自己签了一次名。段璟把这件事记得很清楚，不是因为它多戏剧，而是因为它让他第一次看见选择有后果。

他也慢慢懂得，喜欢不能替常俊凡做决定，也不能把她的每一次沉默都解释成和自己有关。

常俊凡要走的路，首先属于她自己。段璟若真想靠近，就得先学会站在不挡路的位置。

高二就是这样来的：没有高一开学那样热闹，也没有后来倒计时那样压迫，却把许多关系悄悄推到更真实的位置。

晚自习结束时，闻佳夹在错题本里的分科确认回执还留在教室里，像一枚小小的钉子，把这一天钉在十班的时间线上。

灯一盏一盏暗下去，楼道里传来脚步和水瓶碰撞声。段璟走到门口时回头看了一眼，发现常俊凡还在收书。

他没有催，也没有等得太明显。只是把书包带往肩上提了提，跟着人群往楼下走。
