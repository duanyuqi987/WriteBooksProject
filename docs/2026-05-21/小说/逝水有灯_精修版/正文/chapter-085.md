# 第085章 迟到的表白多年后

所属部卷：第二部：六月以后
时间：六月末
地点：河边护栏旁

## 诗歌插页 P085：原创七言小诗

尘过长廊日影深，
一行旧字压衣襟。
少年未解离分意，
先把沉默写到今。

大四毕业那年，陆允川回过县城一次——办户口迁移。

从派出所出来，他顺路去那条河边走了走。护栏还是蓝色的，但漆补过了，新的蓝漆比旧的深，像打了一块块补丁。河水还是慢慢地流，对岸钓鱼的老大爷换成了两个小孩，拿着自制的竹竿在玩，鱼钩上连饵都没有。

他靠在护栏上，想起那个傍晚。程砚站在他旁边，把瓜子分成两份，说"确定了就好。确定了比一直悬着强"。

那是很久以前的事了。现在程砚在厂里当车间副主任，上个月刚谈了一个女朋友——是厂里的会计。程砚在群里发了两人的合照，配文"我不是单身了，请各位周知"。周长海回了一条"你女朋友眼睛是不是近视才看上你"，被程砚追着骂了三页表情包。

叶澄在市里的初中教书，带的班级英语平均分全区第二。她偶尔会在群里发学生的搞笑作文，每次陆允川看到都会想——这大概是最好的结局。一个当年最会写英语作文的人，现在在教别人怎么写英语作文。

夏知遥的消息是最少的。群里偶尔有人转发她的朋友圈——她在南方一个建筑事务所工作，参与了一个小型博物馆的设计。有一张照片是她戴着安全帽站在工地前面，对着镜头笑，牙齿很白，晒得有点黑。那是陆允川最后一次在群里看到跟她有关的消息。

他靠在护栏上，拿起手机拍了一张河面的照片。夕阳还在，河水还在，护栏还在。他把照片发到他们三个的小群里。

程砚秒回：你又去河边了？

陆允川：路过。

叶澄：你有什么感想？

陆允川想了想，打字：感想是护栏补过了。新漆比旧漆深。

程砚：你能不能有点浪漫的感想。

陆允川：很浪漫了。护栏补过了说明有人在维护。说明很多东西即使旧了也不会被放弃。

叶澄：你这句可以当作文结尾。

陆允川：你又在改作文。

叶澄：职业病。

程砚：我问你们一个事。你们有没有想过，如果当年陆允川在走廊里说了那句话，现在会怎样？

陆允川看着屏幕上这个问题。河水在下面流，和很多年前一样的声音。

陆允川：可能连同学都做不成。也可能什么都发生了然后又分开了。但大概率跟现在一样——各自在不同的城市，干不同的事，偶尔在群里看到对方的消息。

程砚：所以不说跟说，结果差不多？

陆允川：不一样。不说的话，留在记忆里的是一条没有被打断的走廊。说了的话，记忆里大概率是一句尴尬的拒绝。

叶澄：你现在觉得当时的选择是对的吗？

陆允川靠在护栏上想了很久。然后打字：

陆允川：对错不重要。重要的是那是当时那个我唯一能做出的选择。现在的我回到那个走廊，可能真的会说出口。因为现在的我不怕被拒绝。但那时候的我怕。那时候的我需要那个怕。那个怕保护了我。

程砚：行。你这段话我截屏了。写得真好。

陆允川：是不是太矫情了。

叶澄：不矫情。比你的高中作文强多了。至少论点清晰。

陆允川笑了。他把手机收进口袋，双手撑在护栏上，看着河水。护栏上新补的蓝漆在夕阳下泛着不一样的光泽——比旧漆深，但也不是不好看。河水一直流，从他十八岁流到二十二岁，还会继续流。

他转身往回走。经过桥头那棵香樟树的时候，看到树干上有人刻了一行字，被时间磨得只剩下几个模糊的笔画。认不出是什么了。

他没有停下来认。继续往前走，走进县城傍晚的街道里。夕阳在他身后沉下去，把整条路的影子拉得很长。
