# 第162章 封丘确实需要人

时间：2013年9月下旬

地点：江城大学食堂

卷次：第七卷：实习与远方

林玉起沉默了很久。

"封丘的教育确实需要人。"

他说这句话的时候，两个人正坐在食堂二楼的角落里。午饭时间已经过了，周围没什么人，只有一个食堂阿姨在远处收碗，瓷碗碰撞的声音空洞地传过来。

苏安晴正在吃一盘炒青菜，筷子停在嘴边。

"你说什么？"

林玉起没有重复。他低头扒了一口饭，嚼了又嚼，嚼了很久。

苏安晴把筷子放下。

"林玉起。"

"嗯？"

"你是不是觉得我应该回去？"

林玉起抬起头来。苏安晴看着他，眼睛里面不是疑问，是一种他在答题的神色。她是在给学生改作文时也会露出这种表情——她在等正确答案。

"我不是觉得你应该回去。"他说，"我是觉得，你选择回去是对的。"

他把筷子放在碗沿上。

"你的性格我多少知道一点。实习的时候你每天都六点起床坐一个小时公交去城东，回来改作文改到九点多，从来没有抱怨过一个字。你做事不是为了别人说好，是因为你真心觉得这个事值得做。"

他停了停。

"封丘的教育值得做。"

苏安晴没有说话。

她低下了头，把炒青菜里的一块生姜挑出来放在碟子边上。动作很慢，慢到像在故意拖时间。

过了一会儿，她开口了。

"吕老师你知道吗？"

林玉起摇头。

"我初中的语文老师。个子不高，戴眼镜。教室窗户玻璃破了，冬天上课的时候她站在窗户下面，给我们挡风，一站就是一节课。后来她哮喘了，冬天一冷就犯。有一年冬天下大雪，她怕耽误我们期末复习，裹了一床被子坐在讲台上给我们上课。"

苏安晴说着说着声音就小了。

"后来我考上大学去跟她报喜。她已经调到另外一个学校去了，我没找到她。听说后来身体不行，办了提前退休。"

她把生姜片又翻了个个儿。

"我就是觉得，封丘还需要这样的老师。如果没有人愿意回去，那以后就没有吕老师了。"

林玉起听着。

他想起自己初中的语文老师。姓赵，黑黑瘦瘦的，板书写得非常漂亮，横平竖直，每一个标点符号都仔佃细地打在黑板上。赵老师教了他三年，从初一到初三。后来他去县一中读了高中，才知道赵老师在他离家那天站在校门口看了他半天。

"那你就考回去吧。"他说。

语气很平，像是在说一件已经决定了的事。

苏安晴看着他。

"那你自己留城里？"

林玉起没有直接回答。他把碗端起来大口扒拉着，吃完最后一口饭，又用筷子刮了刮碗底。

"我考研的事还没定。"

他说完把碗放下来，站起身。

"走吧。下午图书馆。"

苏安晴也站起来。她端着自己的餐盘跟在他后面，走了几步，忽然说："我不是在给你压力。"

林玉起回头看了她一眼。

"我知道。"

"真的不是在给你压力。"

林玉起站在食堂门口，回过头，嘴角动了动。

"苏安晴，你做什么决定我都会支持的。你回封丘也好，留江城也好，还是去别的什么地方。我们四年同学，这个话我说到做到。"

他说完推开门，食堂外面的光照进来，亮得有些刺眼。

苏安晴跟了出去。秋日的下午风很大，从操场上吹过来的，把她的头发吹得贴在脸上。

她伸手把头发撩到耳朵后面，看了他的背影一眼。

然后快步跟上去。
