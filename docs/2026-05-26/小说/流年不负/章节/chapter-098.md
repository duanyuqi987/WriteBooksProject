# 第098章 三个月

时间：2014年10月至2015年1月

地点：楚州各处

卷次：卷五：各自南北

和周雨在一起的三个月。深秋到冬天。

十一月。楚州梧桐叶落得差不多了——地上一层焦黄卷边叶子——踩上去脆脆的。一起上自习——图书馆三楼靠墙——他坐第四个座——她坐第三个——旁边。她带薄荷糖——自习半路递过来一颗——他有时吃了——有时说"不用"。她偶尔会在自己的笔记本空白处画他的侧脸——画得不像——下巴画尖了——她自己也笑了。

自习结束送她回宿舍。女生楼前一排灯柱——绕着枯掉的牵牛花藤。她站在灯光下说"今天自习效率很高"或"明天中午食堂有红烧排骨"。他听着——偶尔"嗯"一声。

她爱说话。话题散——文学社的事——她们宿舍——家里——"我妈说我怎么还不打电话回家"。有时把他写进散文——拿草稿给他看——《沉默的人》——"他坐在图书馆靠窗位置——阳光打在他笔记本上——他在写东西——看不清写什么——笔停顿时会抬头看一眼窗外——窗外只有一棵槐树。"他看完没说话。周雨问"像不像你"。他说"不怎么看窗外"——她笑了。

"你话好少。""习惯。""你以前谈过恋爱吗。""没有。""真的假的——那你初恋是我。"她笑——眯眼——雀斑往上翘。"嗯。"她相信了。她把头靠过来——头发蹭他脖子——痒。没看到他的眼睛——那时候正看窗外——梧桐叶子落了一半。

十二月初她生日——他送了海子的《活在珍贵的人间》——旧书店买的——书页泛黄——扉页上有人写过"2008年购于楚州"——用橡皮擦了擦没擦干净——下面写了一行新字——"生日快乐——顾临风——2014.12"。她不嫌旧——"旧书比新书好看"——把书放枕头边——拍照发了QQ空间。

十二月中——周六——逛操场跑道。他走第五道——她走第四道——隔一根白线。走两圈——她忽然把手伸过白线碰他手背——"你走太快了。"他放慢——她握住他手指——他手心有汗——她还是凉的。操场远角灯光打她身上——头发边缘有很淡的金黄色。

三个月——自习三十多次——吃饭二十多顿——逛操场十几次——看三场电影——师院北门小电影院——屏幕不大——座椅旧。第一场《星际穿越》——她看哭了——他从书包掏纸巾——没用过的——折得整齐——她接过去擦眼泪——"你也太冷静了"——没解释。第二场《匆匆那年》——看完说"青春片都是套路"——在门口风里站了一会儿——她问"你的青春有没有值得拍电影的故事"——他说"没有"。第三场《霍比特人》——她睡了一小半——他看完全部——中土世界的山和峡谷在屏幕上展开又收拢。

跟她在一起像认真做一件事——认真程度不亚于他的笔记、毛笔字、支教。配合她的节奏——自习陪着——冷脱外套——她笑嘴角也跟着动。这些动作都真实——不是演的。但有些东西不是动作能带出来的——看他时眼里的光——她说到某个词时他脑里跳出的另一个画面——她靠他肩膀时他在想十二公里外另一个人会不会也在靠某个人肩膀。这些周雨不知道——但他自己知道。

一月十号——楚河边上——最后一次一起出去。冬天河面平——水绿得发灰。岸上有人钓鱼——支着马扎——一动不动。坐在岸边长椅上——她靠他肩膀——羽绒服摩擦羽绒服——沙沙声。远处有公交车开过——大概301路——往东边去。他的视线跟了那辆车大概两秒。

她忽然说："你听人说话时很认真——但总觉得你在看别的地方。"

他顿了一下。

"我在看你。"

说的时候没有看她。

周雨笑了一下。她的笑不是惯常的那种——不眯眼。是嘴角动了一下——然后收回去。她把头从他肩膀上抬起来。羽绒服摩擦羽绒服——沙沙声停了。

"你说谎的时候眼睛会往右偏。"

"错了。往左偏。"

"你看——你刚才说的就是谎。"她说这话的时候语气不重——像在讨论一个已经确认的事实。她站起来——拍了拍羽绒服上粘的碎叶子。楚河边上的柳树还没完全掉光叶子——几片枯叶落在长椅上。"走——回去。冷。"

回去的路是他们常走的那条——从楚河边沿绕着走——经过市体育馆——再沿着校后门那条巷子回师院。在体育馆门口——她停在铁栅栏前。体育馆是一座圆形的建筑——晚上没人，只有几个路灯亮着，把圆顶的外墙照得半亮半暗。她说"这个体育馆我去过一次——高中时候来楚州参加全省英语演讲比赛——初赛第一轮就被淘汰了。"他说"你口语不是很好吗。"她说"那天紧张——站在台上看到台下评委的眼镜片反光——一句话都说不出来。"他们继续走。经过一家水果店——店门口摆着一摞柚子。柚子皮的清香味飘过来。她鼻子动了动——"我奶奶以前过年总会买十斤柚子。"她说起奶奶时声音跟平时不太一样——少了一点活泼——多了一层很淡的东西。他听着。他们走到了校门口。

她转身进了女生宿舍。过道前的白灯柱还是那一排——枯掉的牵牛花藤绕在柱子上，藤条被风一吹就轻轻地晃。她站在灯下——"今天挺开心的。"他"嗯"了一声。她往他这边走了半步——在他左边的脸碰了一下。嘴唇很轻——像一片冷叶子贴了上来——然后就离开了。不是吻——是道别。

"明天见。"

"明天见。"

她转身进了楼道。铁门在她身后关上——一声闷响。灯灭了。他站在女生楼前——手还揣在口袋里。左边脸上那个位置凉的。他用手指碰了碰。然后转身往12号楼走。

那之后又过了一周。期末考试开始了。古代文学、现代汉语、外国文学史、英语、哲学导论——一门接一门。他们自习的次数减少了——各自在考场上忙碌。考完最后一科的那天下午——她在QQ上发了两个字："老地方。"

第二天。一月十五号。楚州下雪。老地方——图书馆门口的台阶——他们第一次说话的地方。
