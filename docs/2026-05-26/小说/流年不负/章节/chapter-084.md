# 第084章 遵守约定

时间：2013年9月至10月

地点：楚州师院

卷次：卷五：各自南北

大学生活在九月下旬进入了轨道。

每天早上六点半起床。洗脸。刷牙。去食堂。包子和豆浆——每天都一样。七点四十到教室。坐第三排靠窗的位置。听课。记笔记。下课。食堂。下午的课。下课。食堂。晚自习。十点回宿舍。十一点睡觉。

日子是重复的。

他让自己忙起来。

周一的课排到下午五点四十。周二晚上有书法选修课——毛笔字，从颜体练起。周三晚上中国古代文学——讲《诗经》和《楚辞》。周四晚上西方哲学导论——讲柏拉图。周五晚上历史地理——讲黄河改道。周六上午两节书法。

他的课表比宿舍里任何一个人都满。

小伍有一回看了一眼他的课表。"临风——你是不想让自己闲着吗。"

"只是想多学点。"

"学这么多不累吗。"

"不累。"

李博在宿舍复习近代史纲要。他翻了翻顾临风的笔记本——几本都写了半满。字是行书，写得很整齐。

"你这笔记记得比上课讲的还多。"

"怕忘了。"

"忘不了。考试又不考这些。"

顾临风没答。他把笔记本合上，放进书包里。

十月的一个周二。书法课。教书法的是个退休返聘的老先生——姓钱。头发全白了，但手很稳。执笔的时候指节像树根——有力而且有纹理。他站在顾临风桌子旁边看了一会儿，说"腕力不够"。顾临风把腕往下压了半寸。钱先生说"好"。

下课以后他还在练。教室里的灯关了——只剩他那一排。毛笔在纸上走——横、竖、撇、捺。墨汁的味道散开来。窗外有风吹进来，吹得宣纸的边角一掀一掀的。他用镇纸压住。继续写。写到手酸了。把毛笔搁在笔架上。看了看写的字——多是多、像也不像。

十月初。天气开始转凉。楚州的秋天比青阳干。路上的梧桐叶子卷着边，有风的时候在地上打转。他每次从教学楼走到食堂的路上，会踩到脆的叶子——咔嚓一声。

晚饭以后他偶尔去操场走圈。

操场的跑道是塑胶的。红色。八条跑道。中间是足球场，草是假的——绿色的塑胶草皮，踩上去有点硬。操场边上有十几盏大灯，晚上九点半准时打开。灯光是白色的，把跑道照得跟他脚下的路一样清楚。

他开始走路。

不是跑步。是走。

从第一道开始——最内圈。走完一圈回到起点。然后往外走到第二道。再走一圈。走到第三道。再一圈。

走到第五道的时候他停了一下。

第五道。高中那年在操场走圈也是第五道——那次是他以为她有男朋友那次。那次他决定高考完就表白。那次之后他就一直在走第五道。

现在也在走第五道。

操场里有跑步的人超过他——有男生有女生。有人戴着耳机跑，有人两个人一起跑，有人手牵手在散步。足球场上有人在踢球——喊声和哨声混在一起，在夜风里散开。

他继续走。

第五道一圈四百多米。他走三圈。走完第三圈，走出操场。

操场门口的饮水机。不锈钢的。三个水龙头——一个坏了，用胶带缠着。他弯腰在没坏的龙头上接了一口水。水是热的——含着一股铁锈味。他喝了两口，把剩下的倒掉了。水洒在地上，很快就干了。

回宿舍的路上经过一个电话亭。师院的电话亭是黄色的，像一个倒扣的碗。里面亮着日光灯。电话挂在机身上——银色的听筒。

他停下来。

他在电话亭前站了大概十几秒钟。手插在裤兜里。裤兜里是手机。手机里存着一条路线——301路，11站，12公里。手机里还存着一个号码。

十月晚上有点冷。他能看到自己呼出来的白气。很薄。呼出来马上就散了。

他没有走进电话亭。继续往前走。

李博在宿舍看《新闻联播》。电视是他从家里带来的——一台十四寸的旧电视。画面有点糊。女主播的声音很标准。阿坤在打副本——今晚好像是什么团本，键盘敲得特别响。小伍在写信——这次的信纸是带花边的。他说是他妈寄来的新信纸。

"临风——你跑步去了？"阿坤头也没抬。

"走圈。"

"'走圈'——你这种说法老年人用的。"

"老年人也挺好。"

顾临风坐在床边。脱掉运动鞋。鞋底沾了塑胶跑道的红色颗粒。他用手指拨掉了两颗。颗粒掉在地上——红红的，像碎掉的什么东西。

阿坤的游戏打完了。他把耳机摘下来。"咱们宿舍也该集体活动一下。老是各干各的。"

李博说。"你想干什么。"

"出去吃一顿。师院后门新开了一家烤鱼。听说是活鱼现杀的。"

"贵不贵。"

"AA。"

小伍说"好"。李博说"周五晚上"。顾临风说"行"。

周五晚上四个人去了那家烤鱼店。店不大。墙上贴着红纸菜单。烤鱼装在铁盘子里端上来——鱼是草鱼，上面盖了一层辣椒和花椒。阿坤用筷子拨开辣椒，露出鱼肉——白白的冒着热气。李博叫了四瓶啤酒。小伍喝了一瓶脸就红了。阿坤喝了三瓶开始讲他的网恋故事——"那个女生打奶妈特别厉害"。李博说"打游戏认识的能算恋爱吗"。阿坤说"怎么不算，她给我加过一百多次血"。

大家都笑了。顾临风也笑了。笑得很轻。

吃完饭他们沿着学校外面的街走回去。路灯间隔不均匀——有的路段很亮，有的路段很暗。经过师院南门的时候，顾临风看了一眼公交站牌。301路。站牌在路灯下面。没有人等车。公交站空旷地立在路边。

阿坤他们走前面。"快点临风——"

"来了。"

他快步跟上。

回到宿舍。洗漱。熄灯。

窗外的风带进来一股食堂做饭的气味——炒洋葱、酱油、大锅饭的蒸汽。又到了晚饭的点了。但他已经习惯了这种味道。

他把鞋整齐地放在床底下。躺在床上。拿出手机。屏幕亮了一下——没有什么新消息。他把手机翻过去。屏幕光灭了。

闭上眼睛。

师大的操场——跑道是什么颜色的。她会不会也在走圈。她走第几道。

他想了一会儿。然后不想了。

遵守约定——就是不联系。不找她。不打电话。不发短信。不去她的学校。

他做到了。

一天一天地做。

到今天为止——已经四十二天了。
