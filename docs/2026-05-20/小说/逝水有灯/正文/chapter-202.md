# 第202章 最后一次争论

所属部卷：第四部：失去的美丽
时间：大四开学后
地点：校外面馆

## 诗歌插页 P202：原创仿唐绝句

面汤微沸话微凉，
旧伞收声靠短墙。
不问春风归哪处，
愿将执念各归箱。

争论是从一句很轻的话开始的。苏晚枝说：“毕业以后，很多事就自然散了。”陆允川听见“自然散了”四个字，手里的筷子停住。

“所以你一直等它自然散？”他问完就后悔。那语气太像从前，带着审问，带着委屈，也带着不肯认输的刺。

苏晚枝抬头看他，眼神里没有愤怒，只有累。“陆允川，我不是在等什么散。我只是没有办法按照你的节奏处理。”

“可你从来不说清楚。”他声音低下去，“我很多时候不知道自己到底算什么。”

苏晚枝把筷子放下。面汤的热气隔在他们中间，慢慢变薄。“你想要的说清楚，常常不是听我的答案，是要我给你的期待盖章。我要是说没有，你会觉得我否认了你全部的认真；我要是说有一点，你又会觉得还有希望。”

陆允川张了张口，没有立刻接上。店门被人推开，雨声涌进来，又被门挡住。老板端着一碗牛肉面经过，提醒他们小心烫。

苏晚枝继续说：“我不讨厌你。也不是没有被你感动过。但感动不能替我答应。我害怕的是，只要我承认一点柔软，你就会把它当成我应该继续靠近的理由。”

这句话让陆允川脸上发热。他想辩解，想说自己已经改了很多，可最终只是把筷子放平。“对不起。”他说，“这次不是要你回复。只是这句话我应该当面说。”

苏晚枝看着他，很久后点了一下头。“我听见了。”

面没有吃完。争论也没有胜负。只是那天之后，他们终于不再假装问题只是误会。
