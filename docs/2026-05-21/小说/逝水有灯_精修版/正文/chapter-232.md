# 第232章 诗歌索引

所属部卷：第五部：逝水有灯
时间：深夜书桌前
地点：自己的书房

## 诗歌插页 P232：原创仿唐绝句

校徽微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

事情落到当天，没有被命运单独照亮。

凌晨两点，陆允川的诗表格填到了第三十首。

他把四十五首诗歌按诗体分成三组：现代诗十九首，仿唐绝句十三首，仿宋小令十三首。不是平均分布——现代诗多了六首，因为中间有几组他打破过自己的格式，把仿宋改成了现代。他当时说"身体不舒服，写不动格律"。现在回看那个理由，觉得"身体不舒服"只是个说法——真正的原因是他不想被困在一个固定的节奏里。某个场景需要松一点，就需要自由体的诗。

标注日期的时候他碰到了麻烦。前十五首的日期从日记里可以查到——他写诗的节奏和写信的节奏同步，几乎每周一首。但从第十六首开始，日期出现了断层——有时候间隔两三个月，有时候隔着半年。不是因为没写，是因为写完就夹在日记里，没标日期。

他翻开那些诗，试图通过内容来推断时间。

第21首，意象是"松线"。这首诗只能出现在毕业那年。因为只有那件学士服的袖口才有一截松线。

第26首，意象是"白漆"。这首诗明显对应出租屋。但具体是哪天——他翻了出租屋时期的日记，找到了线索：日记里记了一笔，"今天给白漆拍了张照"。日期是三年前的10月14日。那么这首诗大概率写于10月14日前后。

第34首，意象是"牛皮封面"。这是搬家发现旧日记本那天写的。他在日记本第一页的空白处写下了草稿。日期可以查到搬家那天的日历。

第41首，意象是"旧螺丝孔"。他回一中那天。朋友圈那条"旧螺丝孔还在"底下，他标记了自己当时是在从县城回来以后写的。所以日期可推定。

第45首——最后一首——意象是"灯慢慢亮了"。他不知道具体日期。因为这首诗改了很多年。从第一版到最后定稿，跨越了至少四个版本。第一版写于大二，意象是"灯还没有灭"；第二版改于大四，意象是"灯又亮了一次"；第三版改于出租屋时期，意象是"灯在滴水的空调下"；第四版就是最终版，"灯慢慢亮了"。

他把这四个版本的日期都填进了表格。一首诗占四行。

程砚有一次问他："你写诗的时候知道自己是在为多年后的索引做准备吗？"

他当时说："不知道。写的时候只觉得——写完这一首，这一整个人的状态就算是归档了。"

现在逐首整理索引，他验证了自己当时的直觉：诗歌从来不是装饰。它是每一章的呼吸口。叙述写到最紧的时候，诗歌松一下；对话最密的时候，诗歌停一拍。像一个乐队里的钢琴——不抢戏，但缺了它就不对。

旧校徽在键盘旁边。屏幕光打在金属面上，氧化的那块斑在光里变成了浅灰色。

他把最后一首的日期填好。表格完成。

四十五首诗。三个诗体。跨越十年。

他另起一行，在表格最下面加了一行备注：

"所有诗歌均系作者原创。仿唐绝句与仿宋小令参照古典格律，但内容皆为当下一刻的真实情感——未经时间粉饰，未经再次修改。保留原貌。"

打完这行字，他站起来去了趟洗手间。镜子里的自己眼睛有点红。不是因为熬夜。是因为刚才整理到第21首的时候，重新读了一遍毕业照的"松线"——那首诗的最后一句是"人也终于学会往前行"。他写这句的时候二十三岁。现在十年过去，他确实往前走了。但是是慢慢走的。不是终于。

回来以后他关掉表格，重新打开正文。还有几章的修改没做完。但今晚做够了。他存了盘，把电脑合上。

旧校徽还亮着。屏幕的光已经灭了，但校徽本身不发光——它不是灯。它只是一个被别在无数书包上又被扔进无数文具盒里的普通物件。

他把校徽放进抽屉。不是扔掉。是收起来了。和诗歌索引一起。
