# 第200章 回乡的暑假多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：镇上老桥

## 诗歌插页 P200：原创仿唐绝句

桥边新柳换成行，
旧日波声不必藏。
回首方知灯火近，
家门一盏胜斜阳。

多年以后，陆允川再回县城，老桥已经拓宽。原来的白栏杆拆了一半，临时围挡上贴着施工公告。河边多了步道和路灯，夜里有人散步，有人直播唱歌，旧日的安静被新的生活覆盖得很自然。

他站在围挡外，没有惋惜太久。很多地方都不会按记忆停着等人，能留下大致方向已经不错。桥还在，河还在，只是它们不再负责证明他的少年心事。

母亲的头发白了不少，走路比从前慢。那次回家，他陪她去医院复查，排队、缴费、拿药，半天过去，两个人都有些累。回来的路上，母亲突然说：“你小时候不爱说话，长大了还是不爱说。”陆允川笑了笑：“现在好多了。”

晚上他整理抽屉，找到那本高中日记。关于老桥的那页还在，字迹用力得有点陌生。他没有撕掉，只在旁边夹了一张新纸，写：这座桥后来修过，家里的灯也一直亮着。

这句话很普通，却比当年许多激烈的话更靠近真实。人长大以后才知道，青春里的远方固然重要，可真正把人接住的，往往是回家后桌上一碗温着的汤。
