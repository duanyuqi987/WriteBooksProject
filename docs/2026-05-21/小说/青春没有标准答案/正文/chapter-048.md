# 第048章 家长会以后



时间：2008年十二月中旬，家长会后的傍晚



地点：十班教室、校门口和城关镇小街



卷次：第三卷：雪落在两栋楼之间



2008年十二月中旬，家长会后的傍晚，十班教室、校门口和城关镇小街，桌洞里没带走的家长会通知单先把这一天按进了十班的日常里。家长会把月考压力从教室带到每个学生家里。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

有人被夸，有人被沉默盯住，有人第一次觉得父母坐在自己座位上很陌生。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看桌洞里没带走的家长会通知单，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

余小满也在这一天被推到前面。余小满的特点是住校后总想家，学习靠清单推进，家长会最怕父母沉默。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“我妈一句话都没说，比骂我还吓人。”余小满说。

“我妈问我食堂饭贵不贵，我差点哭。”宋乔说。

“你爸妈来了？”段璟说。

“来了，问的问题都很实际。”常俊凡说。

“家长会不是审判会，是让你们知道下半学期该怎么走。”陶建国说。

家长会后的教室总有一种奇怪的安静，像每张课桌都刚被成年人坐过，留下不属于学生的重量。

桌面上有人忘了带走矿泉水瓶，有人把成绩分析表压在书下，像压住一封不想看的信。

余小满把母亲签过字的表格折了两次，折痕正好压在总分那一栏。

段璟回到座位时，发现桌洞里的草稿纸被父亲整理过，乱糟糟的一叠忽然变得很规矩，他反而不适应。

常俊凡的家长没有在教室多停，问完老师问题就走。她站在后门边看了几秒，脸上没有明显表情。

宋乔说我妈坐我座位上比我还像学生，大家笑了一下，很快又各自安静。

宿舍电话旁排队时，许多人把同一句话改了好几遍：考得还行、还可以、下次会更好。

这类话听起来像模板，可每个人说出口时，喉咙里都有自己的卡顿。

家长会以后没有悬在空处。家长会后桌面上留下的矿泉水瓶，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，家长会以后带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

陶建国进门时不一定立刻说话，鞋底在水泥地上一停，家长会以后带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在余小满快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。家长会以后越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。桌洞里没带走的家长会通知单让他想说话，又逼他先低头看题。

余小满被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，家长会以后留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。余小满路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为家长会以后翻身，谁还在被窝里背单词，旁边人都听得见。

家长们从城关镇的小街散开，棉鞋踩过路边薄冰，把学校里的话带回各自家里。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理家长会以后里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：桌洞里没带走的家长会通知单、一只冻红的手、一句老师点名、一块还没扫干净的雪。

曹蔚把家长签过字的分析表压进书里，露出的边角还是很显眼。十班的热闹没有停，只是换成了更低的声音。

何嘉树听见校门口有人喊自己名字，肩膀先僵了一下。这样的细节没人会写进成绩单，却会留在同学之间。

段璟回宿舍后把家里带来的棉衣摊开，袖口还带着厨房油烟味。窗外风一过，纸页轻轻动了一下，像替谁把话咽回去。

常俊凡把电话卡夹进英语书，像把家里的声音暂时收起来。这个小动作很快过去，却让教室更像一个真正在运转的班。

宋乔把家长签过字的分析表压进书里，露出的边角还是很显眼。旁边的人没有特意看，可那一瞬间还是被段璟记住了。

周明澈听见校门口有人喊自己名字，肩膀先僵了一下。常俊凡没有接话，只把手里的事做完。

家长会以后之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

余小满没有因此变成特别醒目的人，可在段璟眼里，余小满从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，段璟和常俊凡在校门口短短说了几句，都没有替对方的家庭压力下结论。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
