# 第090章 录取通知书多年后

所属部卷：第二部：六月以后
时间：七月下旬
地点：镇上邮政所

## 诗歌插页 P090：原创七言小诗

尘过长廊日影深，
一行旧字压衣襟。
少年未解离分意，
先把沉默写到今。

大学毕业两年后，陆允川回家过年。镇上邮政所已经翻新了——原来的小门面换成了标准化的邮政网点，玻璃门、电子叫号机、墙上贴着"寄递到户 便民服务"的绿底白字标语。门口多了一个快递柜，铁灰色的，分成大大小小的格子，有人在扫码取件。

织毛衣的周阿姨退休了，新来的营业员是个年轻小伙子，戴着一副黑框眼镜，低头玩手机。陆允川在门口站了一会儿，小伙子没抬头。

他走进去。"请问原来在这里工作的周阿姨——？"

"退休了。"小伙子头也不抬，"去年退的。您有什么事？"

"没事。就是问问。"

小伙子终于抬起头看了他一眼，大概觉得这个专门来问退休营业员的人有点奇怪。然后低下头继续玩手机。

陆允川站在门口往里看了一眼。柜台还是那个位置，但木头柜台换成了不锈钢的。原来放挂号信的那个抽屉还在，但上了锁。墙上原先贴着的泛黄业务海报被撕掉了，换成了新的绿色宣传画——"EMS次日达"。

他想起那年夏天，他每天早上走到这里，周阿姨头也不抬地说"还没有"。想起她放下毛衣针递给他红色信封的那一刻——"到了。"想起母亲听到消息时肩膀轻轻动了一下，想起父亲在电话里说"土木工程好"。想起周阿姨递给他那颗大白兔奶糖——"我孙子考上初中我都给他糖吃。你考上大学，更该吃。"

他拿起手机拍了一张新邮政所的照片，发到群里。程砚很快回了：我家楼下那个邮政所去年也翻新了。卖起了土特产。

叶澄：我们学校门口的邮局被快递柜取代了。EMS跟别的快递没区别了。

陆允川：周阿姨退休了。

程砚：你怎么知道？

陆允川：刚问的。新来的小伙子说她去年退的。在邮政所干了二十三年。

程砚：你还记得她？

陆允川：记得。她每次都说"还没有"，然后就继续织毛衣。

叶澄：她给过我一颗糖。说我考了六百多分，值得吃一颗。

程砚：她怎么没给我？！

叶澄：因为她认识你三年，知道你不是吃糖的性格。

程砚：歧视。

那天晚上，陆允川在老家房间里翻出了一个旧文件袋，里面装着他带到大学又带回来的各种证件。身份证复印件、团员证、报到时发的校园卡、还有那封录取通知书——红色信封已经褪色了，烫金的字不再发亮。

他把通知书抽出来重新看了一遍。纸张是普通的复印纸，学校的红色公章已经褪成淡粉色。报到须知背面有两行铅笔字，是他自己当时写的：学费5800，住宿费1200，总计7000。这个数字对当时的他们家不是小数目，但母亲没有犹豫过——她只是问了句够不够。父亲说够，然后那几天他下班之后多接了几个零活。

陆允川把通知书翻到正面。自己的名字，十八岁，土木工程系。那个夏天的那些字，现在已经变成了他在设计院画的一张张图纸。

他把通知书放回文件袋。又拍了一张照片，这次只发给了程砚。

陆允川：通知书还在。

程砚：我的也在。在我大学宿舍的书架上放了四年。毕业的时候收起来放家里。我妈把它和我小学的三好学生奖状放在一起。

陆允川：三好学生？

程砚：小学五年级的。全班第四十名之外的人拿三好学生——靠的不是成绩是靠关系。靠的是我跟班主任关系好。

陆允川笑了。他发现程砚有一种特殊的能力——不管聊多正经的话题，最后都能被程砚拐到一个完全出人意料的方向。

程砚：其实每次看到那个信封，我都会想起一件事。我去邮政所拿通知书的时候，周阿姨问我考了多少分。我说五百五十九。她说"不低了，小伙子。够用了。"她那个语气，好像不是在评价分数，是在评价我这个人。

陆允川：她确实是在评价你这个人。

程砚：你怎么知道？

陆允川：因为她从来没评价过我的分数。她只是每次都告诉我"还没有"，然后继续织毛衣。

程砚：这说明什么。

陆允川：说明她看得出来你缺评价，我不缺。我缺的是"还没有"之后的那份耐心。

程砚：你想她了？

陆允川：想周阿姨织毛衣的样子。还有你第一次拿到通知书时说"老子有学上了"——修车摊大叔跟我说的时候笑了一下午。

程砚：什么？！她告诉了修车大叔？！

程砚：什么？！她告诉了修车大叔？！完了。我的形象。

陆允川：你在修车大叔面前有什么形象？

程砚：我每天早上修车的时候路过他摊子，他都跟我打招呼。我以为他是友善，原来是在看笑话。

陆允川：他不是在看笑话。他跟我说的时候是笑着的，但是是那种"看你们这群孩子长大"的笑。

程砚：你怎么区分"看笑话"和"看孩子"？

陆允川：看笑话的人眼睛是冷的。看他笑的时候眼睛是弯的。

陆允川把这个消息发出去之后关掉手机。窗外有小孩子在放烟花——年年如此，不过年的晚上也爱放。烟花声很脆，炸开以后带着很长的回声，在巷子里撞来撞去。他听着烟花声，想起那年夏天，从邮政所跑回家，母鸡被吓到墙角，母亲择菜的手停下来，父亲在电话里说了"不错"。

那是他的十八岁。那些声音现在还在——院子的井水声、厨房的切菜声、邮政所毛衣针的咔嗒声——只是隔了很远。隔了六年，隔了好几个城市，隔了大学四年和工作的两年。但还听得见。

他把文件袋放回抽屉。抽屉关上时发出木头摩擦的钝响。窗外又炸了一声烟花，然后是小孩的欢呼。他站起来走到窗边，看见巷子里几个小孩蹲在地上点引线，旁边站着的那个大人大概是他们的父亲，口袋里露出一截烟花棒。

他忽然想，当年自己拿着红色信封从邮政所跑回来的时候，父亲接到电话说"知道了"之后，是不是也这样站起来，把消息告诉了旁边的同事，然后继续干活。那时候他不觉得父亲激动。现在他有点懂了——父亲不是不激动。他只是需要一个更安静的方式，把所有激动都压在"知道了"和"不错"之间。
