# 第213章 南方出租屋之后

所属部卷：第五部：逝水有灯
时间：毕业第一年
地点：南方城市出租屋

## 诗歌插页 P213：原创仿宋小令

灯花轻，旧梦醒，
小窗深处有余灯。
问少年、几番执念，几番无凭。
人去不须追远影，
水流仍向明朝生。

余波总比事件慢。陆允川在出租屋整理电脑，发现高中毕业照和大学诗社照片，两个名字都安静地躺在文件夹里。之后，陆允川把细节一遍遍放回心里比较，像检查一张做错的卷子，越检查越不肯交卷。他回到宿舍以后还在想出租屋墙上剥落的一块白漆，想得越具体，心里越乱。

时间在这里不是背景，而是一层慢慢压下来的纸。2010年的风、2012年的阳光、2013年的寒夜，后来都被他叠在同一本日记里。每一层都薄，叠多了却有重量。所以这一章不只写一件事，也写那件事在岁月里怎样慢慢改变形状。那些形状有时锋利，有时柔软，最后都被他放进更宽的生活里。

风从窗缝钻进来，把试卷角吹起，又很快压回去，像人心里反复犹豫的念头。陆允川站在那样的声音里，忽然觉得自己并不是故事中心，只是众多走过校园的人之一。这个发现当时没有让他轻松，反而让他更想抓住一点能够证明自己的东西。

陆允川在出租屋整理电脑，发现高中毕业照和大学诗社照片，两个名字都安静地躺在文件夹里。之后，陆允川把细节一遍遍放回心里比较，像检查一张做错的卷子，越检查越不肯交卷。他后来反复想起这一段，不是因为它多么宏大，而是因为出租屋墙上剥落的一块白漆。具体的物件比誓言诚实，它不替人辩解，也不替人作证，只安静地留在光里。

他把消息写好又删掉，最后只剩一个句号，孤零零地停在输入框里。他想说的话有很多，可真正到了嘴边，只剩下几句平常的话。年轻时他总觉得平常太轻，装不下喜欢；后来才知道，太重的喜欢常常先把自己压弯。

雨后的操场有一种塑胶和泥土混在一起的味道。鞋底踩上去会带起一点水，他低头看着那些短暂的脚印，心里想，原来痕迹也会很快干掉。这些细枝末节在当时没有被他认真看见。他忙着判断自己有没有被在意，忙着从别人的一句话里找温度，反而忽略了日子本身怎样一格一格向前挪。

“你以为你在等她，其实你在等一个能证明你没白疼的答案。”祁野这样说时，语气并不郑重，甚至带着一点不耐烦。陆允川当时听不进去，只觉得朋友站得太远。后来他才知道，太近的人往往只能看见火，站远一点的人才看得见整间屋子正在冒烟。

关系在这里往前挪了一点，也往旁边偏了一点。他学会让记忆停在自己这里，不再拿它敲别人的门，但余波比事件本身更慢，也更难收拾。陆允川那时没有能力准确命名这种变化，只能把它写进纸页，写得越满，越像害怕空白。

祁野把话说得很糙，却句句落在地上，不像安慰，更像把人从水里拎出来。程砚在电话里问他还会不会想，他说会，但少了。朋友的声音并不总是温柔，有时甚至粗糙，可它们让他看见世界并没有随着一段关系变窄。

朋友们也各有难处。程砚要准备考试，祁野担心家里生意，叶澄给社团收拾烂摊子。可他们仍然在某个晚上分出一点时间听他说话，这一点后来被他记了很久。这些周边生活把爱情从高处拉回地面。它提醒陆允川，心动不是唯一的课程，失落也不是唯一的作业；人总要在很多不浪漫的事情里，慢慢学会怎样对待一件浪漫的事。

当他把她从怨气里放出来，她才重新成为一个普通人，一个会犯难、会犹豫、也会选择自保的人。夏知遥和苏晚枝都已经不在他的日常里，这反而让她们更像真实的人。这部分事实后来才被陆允川补进小说。补进去以后，故事没有变得轻松，却变得诚实了一些。

宿舍上铺有人翻身，床板响了一下，提醒他再大的心事也要小声一点。他把这一幕和诗歌放在同一页，并不是为了显得文雅。诗对那时的他来说，是一块可以暂时放下重物的石头；重物还在，只是手终于能松开片刻。

原创仿宋小令写完以后，他在下面空了一行。那一行空白比诗句更安静，像把解释留给未来，而不是逼当时的任何人马上回答。

仿宋小令写到第三行时，他总会自然放慢。词牌似乎天然适合迟疑，适合把话说到一半，适合让一个年轻人承认，余音比答案更长。因此这些插页不是装饰，也不是为了让青春显得更漂亮。它们更像一排小灯，照着他从自我感动走向自我审查的路；灯光不强，但足够让人看清脚下有水。

时间并不负责把一切变好，它只把事情放远一点，让人终于能看见当年被情绪挡住的另一半事实。他越长大，越不愿把这段青春说成谁辜负谁。辜负当然存在，可它不是全部；全部里还有胆怯、误会、虚荣、善意和没学会好好告别的年纪。

宿舍里有人洗头，有人背单词，有人对着游戏输赢骂得很响。陆允川坐在这些声音中间，忽然觉得自己的难过没有想象中那么特殊，却也没有因此减轻多少。这一类日常在小说里反复出现，是因为陆允川后来相信，真正改变一个人的不是某个爆炸瞬间，而是爆炸之后仍要买饭、交作业、赶公交、给手机充电的无数普通时刻。

旧稿里他总急着解释自己为什么会那样，现在他愿意让读者自己看见。看见一个年轻人怎样坐立不安，怎样说错话，怎样把手伸出去又收回来。他没有因此变得完全宽容，也没有把所有疼痛都解释成成长礼物。他只是终于愿意承认，自己和别人都曾经笨拙，而笨拙不必永远被写成罪。

朋友把饮料瓶推到他手边，没有劝，只用瓶底在桌上轻轻磕了一下。这个动作很小，小到放进当年的日记里可能会被嫌不够痛。可真正能留下来的，常常正是这些不够痛的小动作，它们让人相信那一年确实活过。

在这一章的末尾，陆允川没有得到答案。空调风很轻，吹得窗帘边角一下一下贴着墙。他只是比上一章多看见了一点自己，多承认了一点别人，多放松了一点攥紧的手。

有些路走错了，也不能只骂路。人要承认，是自己曾经太想从一个方向走到所有答案里。灯光落在地面上，薄薄一层，像谁把话说到一半就收住了。故事继续往前走，像水流过桥洞，没有替任何人停下，却把所有经过都带成了更低的声音。
