# 第066章 墙角的试卷之前

所属部卷：第二部：六月以后
时间：高考后第二天
地点：陆家小院的东屋

## 诗歌插页 P066：原创现代诗

东屋的窗开着，
灰尘在光里慢慢落。
三年的书摊在床上，
每一本都像刚从战场下来，
却没人再命令它们
明天早读前翻开。

高考后第二天，陆允川睡到快中午。

醒来时，东屋窗外的树影已经晒到墙上。母亲在院子里洗菜，水声一阵一阵传进来。父亲把门推开一条缝，说：“起来把书收一下，床上都没法坐人。”

陆允川应了一声，过了好一会儿才坐起来。

床边堆着三年的课本和卷子。语文、数学、英语、理综，按科目分过，又被高三最后几个月弄乱。卷子边角卷起，修正液干成白疤，草稿纸上还留着密密麻麻的推导。

以前这些东西每天压着他。现在考试结束，它们忽然失去威力，变成一堆没人急着看的纸。

他从床头拿起一本物理错题本，翻到最后一页，看见自己写过“从这里开始补”。那行字旁边还有程砚画的一个歪笑脸。陆允川笑了一下，又把本子合上。

母亲在外面喊：“不要的就卖废品，留着也占地方。”

陆允川看着满床试卷，第一次发现“不要的”三个字很难判断。有些题他确实再也不想看，有些纸却像还连着某段日子，一扔就像把那段日子也扔了。

他找来一个纸箱，准备先把它们堆到墙角。
