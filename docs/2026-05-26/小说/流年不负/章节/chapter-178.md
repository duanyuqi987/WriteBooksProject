# 第178章 新婚
时间：2019年5月
地点：楚州小家
卷次：卷九：执手之约

婚后第一周。两个人回到楚州的那套一室一厅。

房子不大。四十多平方米。从门口到阳台走十三步。客厅和卧室之间没有门——用一面书架隔开。书架是顾临风用旧木板钉的，有一格钉歪了——左边比右边高了大概三毫米。苏晚晴把几本比较薄的书塞在低的那头垫平。书不厚——一本《现代汉语词典》、一本《素年锦时》、一本《怎样写毛笔字》。全都不是他买的。她从学校带过来的。

搬家那天顾临风一共搬了四个箱子。三箱书，一箱衣服。他的东西少得可怜——衣柜里挂满了她挑衣服之前的空杆子。现在杆子上多了几件她的衬衫和裙子。他的三件衬衫挤在角落——挪到了左半边。

"你的衣服怎么这么少。"

"够穿。"

"三件衬衫。一件是相亲那件。一件是订婚那件。一件是昨天穿的。"

"够了。"

"怎么不说不够。"

"够。"

她把他的三件衬衫从衣架上拿下来。一件一件地摊在沙发上。浅蓝色那件袖口磨了——磨得发白。白色那件领子脱了一个线头——线头拉出来大概两厘米，他没剪也没缝。还有一件灰色条纹衬衫——那是他妈买的，尺码偏大一号，他的肩膀撑不起来，穿了三年。

"浅蓝色这件别扔。"

"磨了。"

"磨了也知道。是你相亲穿的。"

她把衬衫叠好。四四方方的——比领口上那个线头整齐。然后从抽屉里拿出一根针和一团线——是她妈塞在嫁妆里的。针线盒外面贴着红纸，纸上用毛笔写着"持家有方"。她把针穿上。白线。对着光——在台灯底下穿了三下才穿进去。

领口的线头被她缝回去了。针脚很密——不是省钱，是怕再跑线。缝完最后一个结的时候线断了——她没打结，线自己从针眼滑出去了。她把针插回针线盒里。领口的缝线跟她三根手指的指纹一样整齐。

"好了。"

他摸了摸领口。手指摸到新的针脚——凸凸的，像一行盲文。

"谢谢。"

"谢什么。"她站起来把衬衫挂回去。"你把糊了的青菜吃完也没谢过。"

阳台上多了两盆花。

一盆茉莉。一盆文竹。

茉莉是顾临风从花鸟市场搬回来的。挑的时候老板娘说这盆好——已经打了花苞，很快就要开。他问怎么养——老板娘说多晒太阳少浇水。他说记住了。搬回家的时候茉莉的花苞在塑料袋里蹭了蹭——没掉。花苞很小，只有绿豆大，颜色是淡绿带着一点白——像没完全刷完的墙。

文竹是苏晚晴从她家的文竹上分出来的。她妈帮她装进一个白瓷盆——盆底垫了两块碎瓦片。她说文竹要通风，还要偶尔给它喷水。他说阳台通风好。她就把文竹放在阳台栏杆靠左边——跟茉莉隔着大概三十厘米，中间留了一个人的肩膀那么宽。

每天早上她给两盆花浇水。茉莉浇半瓢——老板娘说茉莉怕涝，水多会烂根。文竹浇几滴——文竹怕旱也怕涝，水多了一片叶子就会黄。她把握得越来越准——从第三天开始，再也没多浇过一滴。

水壶是她从学校带回来的。一个塑料小水壶，本来是给教室里的绿萝浇水的。现在退休了——装在家里阳台的水龙头上。每天早上的浇水声——从水龙头接水、提着水壶走到阳台、水从壶嘴里洒下来淋在泥土上。三声响。

第一声。水接满了。第二声。壶放在地上。第三声。泥土吸水的时候发出轻微的咕嘟声。

浇完水。她看向楼下——楼下有一排桂花树。五月桂花还没开，叶子是深绿色的。她站在厨房的窗户前看了大概几秒。然后走到玄关换鞋——帆布鞋。他也换鞋——皮鞋。两双鞋在门口的鞋架上并排摆着。

他每天早上先送她去一小。再赶去自己的学校。

她教三年级语文。早自习七点四十开始。他说第一节课八点半——不急。但他把她送到一小门口之后要骑二十分钟才能到十八中。如果路上堵了就是二十五分钟。他每天都踩在铃响前大概三分钟进教室——不是故意的，是路线太熟了，每一分钟做什么心里都有数。

电动车是他买的那辆二手。后座本来没有脚蹬——他去车行焊了一个。焊得不漂亮——铁条有点歪，但他试过了——撑得住她。她坐上去。两只手原本扶后座扶手。后来不扶了——手放在他腰上。不是抱——是放。手掌轻轻搭在腰两侧。

他骑车的时候背挺得很直。不是因为姿势好——是因为后面坐着一个人。

从家到一小骑十分钟。路经过月湖路——她不让他绕远，但每天他还是会绕一小段月湖路。他说是"试试新路线。"她没说破。每天早上十分钟——从小家到一小，经过月湖的晨光。湖面上早上有雾。五月的雾气不重——淡淡的一层浮在水面上，像水刚烧开时壶口冒出的那种白气。

校门口。她下车。帆布鞋落在水泥地上——"嗒"。她整理了一下斜挎包的带子——包里有语文课本、教案、一盒彩色粉笔。

"中午过来吗。"

"不一定。看上午有没有课。"

"那你骑车慢点。"

"嗯。"

她走进校门。他还在门口站了大概七八秒。看着她走进教学楼——教学楼的门是玻璃的，她推开玻璃门，门弹回来的时候晃了两下。每次都是这两下。不多不少，两下。

然后她回头。站在玻璃门里面，朝着他的方向。隔着校门口的铁栅栏和一段水泥路。距离大概五十米。她回头看了他一眼——不挥手。只是看了一眼。确认他还在。

然后转身消失在楼道里。

他拧了车把。电动车"嗡嗡"响——电机的声音不大，但在早晨安静的街道上很清晰。他骑出去十几米又回头看了一眼校门口——她已经不在了。他把车速加到最快——往十八中方向骑。风从耳朵两边刮过去。他的衬衫背后——她刚才手放的位置——还残留一点点暖意。

傍晚。他去接她。校门口等她的人不多——只有两个家长和一个送快递的。他把电动车停在门口梧桐树下。自己靠在车座上。校门开了——小学生们从教室里涌出来，很多颜色的书包在阳光下闪。他在人群中找她。扎马尾，白色帆布鞋。每次都是在人群中第一个看到——不是因为眼力好，是因为他看了十年，每个角度都认识了。

她从人群中走出来。斜挎包里的粉笔盒把包撑出一个方形鼓包。脸上没有疲惫——但能看到粉笔灰。右手手指上有一层白色的粉末，从拇指到食指。她在他面前拍了拍手，粉尘在夕阳里飘了一下。

"今天讲了《春晓》。"

"小朋友听懂了吗。"

"有人说'处处闻啼鸟'——不知道什么叫'啼'。我说就是叫。他又问为什么不用'叫'要用'啼'。"

"你怎么回答。"

"我说——啼比叫好听。叫听起来是喊——啼听起来是说话。"

他笑了一下。不是大声笑——嘴角动了动。他的学生从没看过他这种笑。在十八中——他是比较严肃的语文老师。学生怕他——也敬他。但从没看他对着任何人这样弯过嘴角。

回家路上。她坐在后座。经过月湖路的时候夕阳刚好照在湖面上。五月的夕阳不浓不淡——橙黄色，像煎蛋的蛋黄打散之后铺在平底锅上那一瞬间的颜色。

她说今天教室里飞进来一只蝴蝶——把学生们全吸引了，没人听课。她让每个人都画那只蝴蝶。然后选了一张画得最好的贴在黑板边上。

"画得最好的那个孩子——平时成绩不好，语文考试经常不及格。但是蝴蝶画得真好。"

"你让他贴黑板边上了。"

"嗯。"

"你以前也贴过黑板报。"

她愣了一下。然后说:"你怎么知道。"

"高一那年冬天。你负责黑板报。我每天放学都最后走——看你画的。"

"你从来没说过。"

"现在说了。"

傍晚的风吹过月湖。湖边的柳树在夕阳下变成金色的剪影。水塔上的灯还没亮。但野鸭子已经回家了——三四只排成一排，靠岸边的芦苇丛里挤在一起。

他们骑车经过的时候惊起了一只白鹭。白鹭从湖面上飞起来——翅膀很宽，影子掠过水面。

她仰头看着白鹭。他把车速放慢了一些。电动车在石板路上缓缓滑行。这一刻——月湖、晚霞、白鹭、两个人、电动车、两个装着教案和语文课本的包——一切都刚刚好。不是浪漫。是刚好。从早上七点浇花到傍晚七点回家——每一天都很普通，但每一天都不一样。

到家。她换鞋。他换鞋。她把斜挎包挂在门后——包上的粉笔灰沾在了门把手上，留下几个白指印。她没有马上擦——让他先洗手。他在水龙头下冲了三分钟手——粉笔灰被水冲走，在水槽里沉下一层白色粉末。

晚饭简单。西红柿炒蛋。青椒肉丝。米饭。她炒菜，他切菜。切菜的砧板是竹子的——他妈给的，正面刻着"百年好合"。他切青椒的时候往砧板上的刻字看了好几次——刀切下去的时候会避开那四个字。

吃饭的时候。她坐在桌子左边。他坐在右边。隔着大概半米——那张桌子是两个人一起去旧货市场买的。桌面有一道裂缝——老板说降价三十块。她说不用降——拿来。

饭后他去洗碗。她坐在沙发上备课。水流的声音从厨房传过来——哗哗哗哗，然后是碗碟碰到不锈钢水槽的"叮"声。她低头在教案上写——写着写着停了一下。是在听厨房的水声。

水声停了。他擦干碗，放回碗架。三个碗、两双筷子、一个炒锅——全部归位。他在围裙上擦了手。走到客厅。她还在教案上写字。他在她对面坐下——批改学生作文。

客厅很安静。台灯照在两个人的手上一双在写教案，一双在改作文。钢笔写在纸上的声音——沙沙沙。翻纸的声音——哗。偶尔她抬头，他也刚好看她——目光在台灯的暖光里碰在一起。碰完了各自低头继续写。

窗外的天完全黑了。阳台上茉莉的花苞在夜色里微微舒展——花瓣还没有打开，但花苞尖上已经能看到一丝白色。

离花开还有几天。
