# 第204章 最后一次争论旁观

所属部卷：第四部：失去的美丽
时间：同一天晚上
地点：校外面馆

## 诗歌插页 P204：原创现代诗

她说我听见了
不是开门
也不是关门
只是把一封迟到的信
放到桌面中央

苏晚枝回到寝室时，裤脚湿了一截。室友正在吹头发，见她脸色不太好，问是不是论文导师又改题。她摇摇头，把伞靠在门后，去阳台拧干裤脚。

她没有把面馆里的对话讲给别人听。不是因为它多秘密，而是讲出来太容易被简化。别人也许会问“那你到底喜不喜欢他”，或者说“他都道歉了，你就别想太多”。可只有她自己知道，问题从来不只是喜欢或不喜欢。

她曾经欣赏过陆允川。欣赏他写诗时的认真，欣赏他在活动室一待就是一下午，也欣赏他有时候笨拙却真诚的关心。可欣赏一旦被期待追赶，就会变得紧张。她越来越怕自己一句普通的谢谢，被他读成可以继续靠近的证据。

晚上她打开论文文档，盯着标题看了很久。手机亮起，是叶澄问她读书会照片要不要修。苏晚枝回：“明天我看。”她没有提陆允川，只在聊天框停了一会儿，最后又补一句：“今天有点累。”

叶澄很快回：“那早点睡。”

苏晚枝把台灯调暗。窗外雨还在下，宿舍楼对面的灯一格一格亮着。她想起陆允川说“这次不是要你回复”时低下的眼睛，心里也不是完全轻松。可她终于不用再把所有柔软都藏起来，以免被误认为承诺。
