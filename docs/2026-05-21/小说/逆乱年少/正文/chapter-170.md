# 第170章 各自的倒计时

时间：2014年12月中旬，考前最后一周

地点：江城大学考研自习室

卷次：第七卷：实习与远方

考前一周。自习室的灯从早上七点亮到晚上十点半，有时候十点半以后还有人不走，管理员老头就站在门口咳嗽，咳到最后一个学生离开为止。

倒计时的便签本撕得只剩最后几页了。法学院姓纪的男生每天撕一张，撕下来的纸用来记政治选择题的易错点——正面是"距考研还有X天"的黑体字，背面是密密麻麻的小字。

苏安晴的教师编制考试也在同一个月。她和林玉起的倒计时不太同步——编制考试比考研早一周。林玉起还有十天时，苏安晴只剩三天了。

最后一周她几乎天天来自习室。

十二月十九日。苏安晴考前最后一天。她下午来的时候带了一袋橘子，给了林玉起两个，放在他桌角上。

"补充维C。你最近嘴唇起皮了。"

林玉起摸了摸自己的嘴唇。确实起皮了，自习室里暖气管太热，空气干燥，他一到冬天就这样。他没注意到苏安晴注意到了。

他剥了一个橘子，橘皮的气息在周围的空气中弥漫着，这是十二月自习室里最鲜活的颜色。她把一瓣橘子塞到嘴里，含糊地说了句"明天我不来了"。

"考试几点？"

"上午九点。在十九中旁边那个考点——市教育学院。"

"东西准备好了没？"

"身份证、准考证、2B铅笔、橡皮、签字笔、手表。"她一样一样地数，像在报菜名。然后从包里拿出一个文件袋，给他看了看。透明的，里面整整齐齐地放着所有东西。准考证被她夹在一本硬皮笔记本里，生怕折角。

"跟你说个事。"她又说。

"嗯？"

"其实我有点紧张。"

林玉起看着她。苏安晴说"紧张"这两个字的时候没有看他，手指拨弄着手下试卷的折角，卷子上的字在她指尖下来回歪曲。

他认识她三年多，从没听过她说紧张。期末考试不紧张，实习第一天不紧张，站在讲台上面对三十二个学生不紧张。现在说紧张了，大概是把全部退路都押在这次考试上了。

"你紧张才正常。不紧张的要么是吹牛，要么是没上心。"他说。

苏安晴看向他。

"你跟别人说话也这么直接吗？"

"只跟好朋友。"

她愣了一下，然后笑了一声。

"好吧。那我去了。"

她把剩下的橘子塞进嘴里，站起来，把帆布袋挂在肩膀上。走到门口时回头说："林玉起，你也好好考。考完了我跟你说件事。"

"什么事？"

"考完再说。"

她走了。

脚步声从楼道的尽头传来，好像持续了很久才彻底消失在拐角处。

她走后，林玉起在座位上坐了一会儿。他把桌上的书横过来又竖过去地摞了两遍。然后把笔放下，站起来走到窗边，拉开窗帘往外看。

窗外是十二月灰沉沉的天空。操场边上最后一批梧桐已经落尽了叶子，光秃的枝丫伸向天空，像一个瘦峭的人撑开双臂。地面上飘满了落叶，清洁工早上扫过了，下午风一吹，又落了一层。

他把窗帘拉上，回到座位上。

翻开她的笔记本，在最后一页空白处写了一行字。

"第一志愿：河南师范大学。"

他合上本子。

这是他写给自己的备注。不是日记，不是给她的信，就是一个事实。

然后继续看《中国古代文学史》。
