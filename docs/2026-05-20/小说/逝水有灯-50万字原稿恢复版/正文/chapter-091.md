# 第091章 夏天的聚餐之前

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P091：原创现代诗

我把星放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
小饭馆圆桌上的玻璃转盘没有替谁回答，
只把那一年轻轻照亮。

夏天的聚餐真正到来之前，日子已经先变了颜色。升学宴前后，县城小饭馆总是坐满毕业生。圆桌上铺着一次性塑料桌布，玻璃转盘转起来时，油光和灯光一起晃。

陆允川坐在靠墙的位置，杯子里倒着橙汁。大家轮流说去哪里读书，说到夏知遥时，有人顺嘴开了句玩笑，说你俩本来也不像一路人。

他还没有遇到那个必须面对的瞬间，却已经被小饭馆圆桌上的玻璃转盘提醒：有些事不能无限期地放在心里，它会跟着时间一起变形。

程砚立刻把话接过去，说：“吃饭就吃饭，别拿别人当下酒菜。”桌上安静了一瞬，又有人赶紧转话题。

叶澄把玻璃转盘轻轻停住，看着陆允川说：“别人随口说的话，不一定要变成你心里的判决。”

夏知遥没有参加这顿聚餐。她的缺席反而让那句玩笑更空，像一颗被扔到桌上的小石子，声音不大，却让陆允川听见了。

空调声、碗筷碰撞和邻桌劝酒的声音，把他的心事压得没有那么夸张。人在这样的声音里，很难一直把自己当成故事中心。

那顿饭菜很多，大家却吃得很快。每个人都急着说未来，好像声音一停，就会意识到这个夏天真的快散了。

陆允川没有发火，只把杯子往自己面前拉了一点。杯底在桌布上拖出一道水痕，他盯着那道水痕，才把顶到喉咙口的话咽回去。

那时陆允川还不知道，所谓成长不是突然明白，而是先在玻璃转盘、橙汁杯、一次性桌布这些小东西旁边学会停顿。

他在日记本里写了两行，又划掉一行。留下来的不是誓言，只是一个具体下午：盘子里慢慢凉掉的糖醋里脊、窗外的光，以及他没能说出口的话。

陆允川后来愿意留下这一笔，是因为它不漂亮。玻璃转盘不漂亮，橙汁杯也不漂亮，可它们让人相信，这段青春不是从空中落下来的。

他没有再给夏知遥安排额外台词。她已经在小饭馆里完成了自己的动作，剩下的沉默，应当由陆允川自己消化。

程砚把话说得粗，叶澄把话说得稳，两种声音都把他往现实里拉。陆允川当时不领情，很多年后却知道自己被救过几次。

那天的盘子里慢慢凉掉的糖醋里脊没有特别好吃，却因为太具体，抵住了那些过大的形容词。人有时候正是靠这些具体的味道，才没有被回忆吞没。

邻桌有人劝酒，服务员端着热汤从他们背后挤过。这些声音把夏天的聚餐从一个人的心事里拉出来，变成许多人共同挤过的一天。陆允川不再只听见自己胸口的动静，也听见椅子、脚步、塑料袋和远处的喊声。

夏知遥没有来，名字却被玩笑带到桌上。这个动作很小，却让她从陆允川的想象里退回到真实的人：她会忙，会累，会被家里和未来推着走，也会在必要的时候保持沉默。

程砚把话截住，说吃饭别拿别人当下酒菜，说话还是老样子，不好听，却有效。陆允川皱了皱眉，还是照着他说的做了一点，因为朋友有时不是负责安慰，而是负责把他从自我感动里拽出来。

叶澄把转盘停住，提醒陆允川别把玩笑记成判决。她没有把话说得太满，只把边界摆到桌面上。陆允川那时不一定立刻懂，却会在后来某个晚上想起这句话，知道自己当年被提醒过。

家里提前交代他少喝酒，他端着橙汁没有解释。家里的声音一进来，青春就不再只是校园里的风。它变成饭点、车费、睡眠、行李和一句不太会表达的关心。

他低头看着玻璃转盘，忽然意识到它不是命运给他的标记，只是那天确实存在的东西。正因为确实存在，它比任何夸张的形容都更能留下来。

糖醋里脊、空调水和一次性桌布的塑料味混在一起，让记忆有了可以落脚的地方。陆允川后来写到这里，没有再急着解释自己多难过，而是先把气味、光线和手上的动作写清楚。

他把橙汁杯收好，又拿出来检查了一遍。这个来回不像英雄，也不像受害者，只像一个年轻人不知道怎样安放自己，于是先安放手边的东西。

真正的变化还没到来，县城小饭馆已经先显出紧绷。陆允川站在里面，既想让日子快一点，又害怕快起来之后自己没有准备好。

他在日记本上写下“等以后”，写完又觉得“以后”太轻。很多拖延都披着克制的外衣，其实只是害怕被现实拒绝。

这一笔补进去以后，章节没有变得更响，反而更稳。夏天的聚餐不再只是一个标题，而是有玻璃转盘、有橙汁杯、有人说错话也有人及时停住的真实一天。

那句玩笑落下以后，玻璃转盘还在慢慢转，一盘凉菜从陆允川面前滑过去。没人立刻接话，空气里只剩空调水滴进桶里的声音。

开玩笑的人很快意识到不对，端起杯子说自己嘴欠。陆允川没有借机发作，只说没事。说完他才发现，没事不是原谅别人，而是先放过自己。

饭后结账时，程砚抢着把零钱凑齐，叶澄在旁边核对菜单。大家突然都变得具体起来，具体到没人再谈谁和谁合不合适。

小饭馆里的光线一点点变暗，一次性桌布和糖醋里脊都还在原处。陆允川把橙汁杯收好，动作慢得像怕惊动什么。可真正需要被惊动的不是别人，是他自己。

他后来重写这一章时，特意把玻璃转盘留在中间。不是为了让它变成象征，而是因为人只有摸到具体东西，才知道自己曾经怎样站在具体生活里。

事情尚未发生，陆允川已经开始给自己找退路。他把“以后再说”想得很体面，可小饭馆里的每一种声音都在提醒他，退路拖久了也会变成墙。

章节落下时，朋友们聚餐，有人打趣他和夏知遥不合适，陆允川第一次因为这句话和人沉默相对。还没有真正展开。陆允川只是把东西收好，跟着人群往前走，心里那点执拗被现实轻轻推了一下。
