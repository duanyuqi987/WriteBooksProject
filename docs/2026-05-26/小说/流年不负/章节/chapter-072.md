# 第072章 第一次单独见面

时间：2013年6月中旬

地点：青阳县城奶茶店

卷次：卷四：夏夜湖光

六月十八号。下午两点。

苏晚晴在QQ上说："明天下午有空吗。"

"有。"

"县城那条新华书店旁边的街上，有一家奶茶店。"

"我知道那家。"

"你去过？"

"经过。"

"明天下午三点。去坐坐。"

"好。"

顾临风把手机放回桌上。然后拿起来又看了一遍。确认时间是三点。确认地点是奶茶店。确认她没有说"还有别人一起来"——是"去坐坐"。没有主语说明只有两个人。

他把手机放下来。

走到衣柜前。拉开柜门。里面挂了六件T恤。灰的。蓝的。白的。黑的。条纹的。灰色的。

他把灰的拿出来。在身前比了比。挂回去。把白的拿出来——领口有点松了。穿不出门。蓝的——这件还行——但是胸前的印花洗了太多次有点龟裂。他把蓝的放回去。拿出黑的。

黑的。不显脏。

他对着镜子把黑的穿上。

然后他脱了。

又穿回灰的。

镜子里的那个人穿着灰色的T恤。头发有点长——遮了眉毛——但是笑起来的时候眼睛还是能看到的。

他没有笑。只是在镜子里练习了一下——嘴角动了一下。嘴角动了。眼睛没跟上。很丑。算了。

第二天下午。两点半。

他提前到了。

奶茶店不大。门口贴着绿色的招牌。四个字——"泡泡奶茶"。玻璃门。推开门的时候门顶上的风铃响了。轻脆的声音。他往里面看了一眼——靠窗的位置空着。角落的位置也空着。只有两个初中生在靠里面的墙那边坐着。一人一杯奶茶，头碰头地看手机。

他选了靠窗的位置。不是角落。角落太像约会的选择了。靠窗——光明正大。

坐下来的第一件事是把椅子往后退了退。腿伸直。然后又把膝盖收回去。把手放在桌上。

桌面是木纹的贴皮。右侧有一小块被烫过的印子——白色的圈。上一个坐这里的人把热饮放在这儿了。

服务员是个姑娘。大概二十岁。

"先生点什么。"

"柠檬水。"

"就一杯柠檬水？"

"嗯。"

姑娘愣了愣。然后在单子上写了"柠檬水"。走了。回头又看了他一眼。

他把目光从姑娘身上移开。移回窗外。

三点整。

奶茶店门被推开。风铃响了。

苏晚晴站在门口。

白色的短袖。浅蓝色的牛仔裙。马尾——白天不出意外还是马尾。书包上——今天没背书包。她只拿了一个小布袋——灰绿色的。

她在门口往里面扫了一眼。看到靠窗的他。点了一下头。走过来。

她在对面坐下的时候椅子腿在地上磨了一下。吱——声音不大。但是店里太安静了。两个人都听到了。她脸偏了一小下。把椅子往前拉了拉。坐得规矩。

她把小布袋放在桌角。

"到了。"

"嗯。"

"等了多久。"

"大概二十分钟。"

"来这么早。"

"没事做。"

"你手机呢。带了吗。"

"带了。"他把手机从口袋里摸出来。屏幕是黑的。他平时不玩手机——手机里除了QQ和短信什么都没有。"怎么。"

"没怎么。就是看你干坐着——一个人。也不用手机打发时间。"

"等的时候看窗外就够了。"

服务员走过来。她对着菜单看了一会儿菜单。"珍珠奶茶。热的。"

"热的？"服务员说。"今天二十五度。"

"那就常温。"

服务员走了。她把桌上的餐巾纸盒转了一下。从横着改成竖着。然后从竖着改回横着。

珍珠奶茶端上来。

柠檬水和奶茶放在一起。柠檬水里冰块在慢慢融化。奶茶杯里有珍珠沉在底下——黑黑的一圈。

她拿起小勺搅了搅奶茶。珍珠在杯底慢慢滚。

"你喝过这家的吗。"

"没有。"

"好喝吗。"

他喝了一口柠檬水。太酸了。眼角皱了一下。

"还行。"

她喝了一口奶茶。杯子放在桌上，手指圈着杯身。她的手指甲剪得很短——每个指头的指甲都剪到指甲肉下面去了。他看到的时候想到了她说的奶奶——"小指就是那时翘起来的"——大概她奶奶也教过她很多事。

"这家的珍珠比学校小卖部的好喝。"

他点头。

她看着他。

他看着柠檬水。杯沿上有个唇印——不是他的。他刚才喝之前用纸巾擦了的。可能是上一个人留下的。他盯着看了几秒。

然后抬头。正好她也刚低头要喝奶茶。两个人的目光在杯子上方碰了一下。他先移开。她后移开。

"你喝过这家的吗。"她问。

"没有。"

她点头。搅了搅奶茶。珍珠滚了两圈。又搅。又滚。

"你刚才问过了。"

她愣了一下。然后低头看了看自己的奶茶。然后把手从杯子上移开。放在桌面上。手指收拢。

"我忘了。"

"没关系。"

过了几秒。

"你喝过这家的吗。"

第三次了。

他说："没有。"

两个人的目光这次碰上了。

她忽然笑了一下。这次的笑不是那种浅的——是没绷住的笑。耳朵到了——从脸颊往耳根的方向——有一点点红。

"我有点紧张。"

"我也是。"

她把奶茶杯推到桌子中间。又拉回来。

"明明已经聊过很多次QQ了。"

"线上和线下不一样。"他说。

"怎么不一样。"

"线上——你可以想很久。打出来。删掉。重新打。线下——"

"线下怎么了。"

"线下你说的每一句我都听到了。"

她看着手里的奶茶杯。

"所以刚才有一句话你听到了三次。"

"嗯。"

"我其实记得我问过了。"

"我知道。"

"我就是找不到话说了。"

"不用说。坐着也行。"

她把奶茶杯放了。两只手放在膝盖上。他看着她的手指在膝盖上轻轻刮着——刮一下。停一下。再刮一下。

店里的背景音乐忽然换了一首——从《后来》换成了《简单爱》。

"说不上为什么，我变得很主动——"

服务员在柜台后面擦杯子。玻璃杯在水龙头底下冲——水声断断续续地。

"那个。"他先开口了。

"什么。"

"你喝过这家的吗。"

半秒的空白。

然后她笑了——这次笑出了声音。很轻的。从嗓子眼里漏出来的一点。

"第四次了。"

"这次是我问的。"

"没有。你也没有。"

"嗯。"

她又拿起了奶茶杯。低头喝了一口。过了很久。

"你这个人。"她说。

"嗯。"

"不说话的时候让人着急。说的时候让人——"

她没说完。杯沿碰着下唇。奶茶的热气在杯口上飘。

"让人怎样。"

"让人不知道说什么好。"

店里的歌又换了一首——《遇见》。"我遇见谁会有怎样的对白"——前两句。然后她的视线从他的衣领上移到了他的眼睛上。

"以后。不用紧张。"

"说给我听的还是说给你自己听的。"

她顿了顿。

"都是。"

然后两个人都看着窗外。窗外有人走过去。卖气球的。红色心形的气球窜上来又沉下去。他看着她看气球。她看着气球。他看着她的侧脸。

玻璃窗上映着两个人的影子。桌上两杯饮料。一杯奶茶。一杯柠檬水。珍珠在杯底。

风铃响了。那天的下午有了形状。
