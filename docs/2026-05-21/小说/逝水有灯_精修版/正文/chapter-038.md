# 第038章 成绩单下滑之后

所属部卷：第一部：初秋三楼
时间：高一期末考试后
地点：公布成绩的公告栏前

## 诗歌插页 P038：原创仿宋小令

榜上新痕，
数字各浮沉。
欲把关心藏半句，
却因暑假闭窗门。
蝉噪旧黄昏。

成绩单下滑之后，陆允川表面上没有变化。暑假开始，他回了乡镇的家。每天的生活规律得像个退休老人：早起跑步，中午帮家里看店，晚上做暑假作业。家里的店是个小杂货铺，夏天最热的时候卷帘门拉到一半，外面的阳光从门底下切进来像一条金色刀片。他坐在柜台后面看书，偶尔有邻居来买酱油或者冰棍，他就起来收钱找零——抽屉里的硬币分门别类装在三个塑料格里，一毛的、五毛的、一块的。妈妈问他这回考得怎样，他说还行——还行是真话，但也因为还行后面还藏着"但看见她掉了三十名整个暑假心情都不好"。这个后面半句没说出来——不是不想说，是没法跟妈妈解释谁是她为何会掉名次这件事。

七月下旬的一天，程砚打来电话说班里要办个暑假补习小组——几个理科成绩比较好的人每周聚一次，讨论一下高一给高二过渡的知识点。时间是暑假某个闷热的热风天，地点是小县城的图书馆借阅室。陆允川在那天早上去之前，在镜子前照了一分钟——不是因为要见程砚，而是因为他隐约希望补课小组里可能会有夏知遥。实际上夏知遥没有来——她据蒋雯说她回了外婆家，不在县城。图书馆借阅室那天只有四个人到场，空调坏了，窗户全开着，热风从纱窗外灌进来把桌上的草稿纸吹得到处飞，大家一边按着纸一边算题，汗滴在纸面上洇开了一小块墨迹。

那天补完了函数图像和受力分析，几个人一起出去吃饭。程砚在小炒店吃饭时忽然说："成绩掉了你们都是怎么消化掉的？""不理它。""心理上是不可能不理的——你数学考差的时候你会不理吗？""你是说——那个人是你自己还是？"另一个补课的男生问。"我说的是大家——不是都说了全班平均分偏低了嘛。"

饭后程砚单独跟陆允川走到公交站时说："她不在补课小组——你失望不？""我没想那么多。""你没想那么多为什么出门之前洗了头吹了发型？""天热，洗头舒服。""天热——三十八度的下午你去图书馆借了本高数入门，你洗头是为了向图书馆报到的？"陆允川没回答。程砚拍拍他的肩："暑假还有一个半月，你急什么——开学以后又是新学期。一个暑假她在家把错题全做完了，你再看她的时候她会带着一本被红笔涂爆了的笔记本。你想想那个画面——你还有什么好担心的？"

暑假的后半程，陆允川开始做高三的物理题——不是因为自己需要提前，而是因为他想万一她以后问物理题时自己能替她讲解清楚。他在草稿纸上写下题干时会附带一个假想学生对象：假设看了这道题之后不会画受力图——那么第一步该怎么引导？画这里。第二步？标箭头。第三步？代入公式。他每解出一道题就在旁边写一份"教辅版解释"。这个文件夹叫做"备用教案"，存在书桌右手边抽屉里，厚度到了暑假结束时已经超过了他自己的物理作业。

开学的前一晚，他在日记本上翻到上次期末考试那页，在"希望她休息好，好好看书，吃好饭"下面又加了一行字：明天开学。高二开始。她成绩应该稳了。然后他合上本子，闭眼。外面的蝉还在叫——叫到后来变成了背景音。他把手放在枕头下面按着日记本。明天他会回到那间教室，坐回那个靠窗的位置。她也会坐在同一张课桌前，笔袋里揣着那支写了无数遍错题的钢笔。有些担心放到她身上是多余——她显然不需要任何人在她背后帮她整理步骤。但她不需要不等于他不担心——担心没法开根号，但它一直都在。
