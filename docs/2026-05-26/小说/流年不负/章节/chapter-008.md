# 第008章 第一次考试

时间：2010年11月初

地点：高一三班教室

卷次：卷一：初见秋风

期中考试成绩贴出来了。

学校门口公告栏上贴了好几张大红纸。从高一到高三，每个年级一张。纸是整张的大红纸，墨是浓墨，毛笔字。写的是小楷。最上面一行用略大一号的字体写着——"2010-2011学年第一学期期中考试成绩表"。纸贴得不算正——右上角翘了一点。在微风中轻轻动着。公告栏前摆了木制的公告屏，玻璃门被推开。底下积了去年没清干净的薄尘。

公告栏前人挤人。有个高个子男生站在最前面，拿手指顺着排名往下划。从第一名一直划到二十几名，然后往回——"靠这呢。找到了。"他旁边的人伸着脖子看。后面又挤过来一个人——"让让让让——"挤到前方。退出来的人摇摇头——"要完蛋了。"退出来后走了几步，不自觉地回头看那页倒字的大红纸一眼。有人伸出手去——用指尖按着某一个名字。按住了——然后收手。转身走开。

顾临风在人群外围站了一会儿。空气燥燥的——公告栏前的木制屏风被人群遮住了三分之二。十月底天已经不热了，但人挤在一起还是会出汗。等了三五分钟，散了一些人——下课铃响，有的人赶去上课了。他靠近。

他没有从最上面开始看。

他从中间开始。手指按在红纸上面——纸有点粗糙。宣纸的手感。往下滑。滑过快几十个人的名字。很多不认识的名字——那些考在最前面的学霸的名字、还有一些成绩中等的普通学生的名字。他瞥了一眼。没停。

往下继续找。往上七行——再往下五行。然后停住了。

手指下面是一个名字。

不是自己的名字。

苏晚晴。年级排名：第八名。语文91。数学89。英语94。物理68。化学79。生物85。政治88。历史90。地理82。

他看了一会儿那行数字。物理只有68——总分就是被这一科拖下来的。英语94最高——如果物理能到80以上，她的排名就不是第八了。语文91——作文大概没扣太多分。英语94——前面选择题至少90%是对的。历史90——也许她喜欢历史。

九门总分在他的脑子里算了算——757分。还有一个数字——英语94和物理68差了二十六分。她差的地方不是数学，是物理。

然后他继续往下找。

找到了自己。

顾临风。年级排名：第三十一名。语文85。数学88。英语83。物理82。化学76。生物80。政治81。历史84。地理78。

他在物理那一栏逗留了一下——82。还行。英语83——他没想到能过八十，以前他以为自己英语过不了八十分。赵一鸣说过"英语好难"——不是空话。

然后他又往回翻。回到前面。

手指重新放在"苏晚晴"三个字上面。这三个字写得算是正的——毛笔的笔画不算粗。排行第八。全年级四百多人——第八。

然后他离开了公告栏。

走回去的时候踩了一脚草——操场上没有草路。他脚底下没感觉。

回到教室。已经有不少人知道成绩了。有人翻着成绩单大声念——"化学太水了——这老师改的有问题吧。"同桌顶回去——"你自己不会做你怪老师。"还有人在凳子上蹦着——坐下去的那一下故意加了一点。椅腿木头顶在水泥地上——邦。

赵一鸣跑过来。手搭在他肩上。赵一鸣的手是暖的。手指间的皮肤有一点粗糙——是平常抱篮球蹭出来的。汗没干。

"你考了三十一。不错啊。"

"你多少。"

"六十二。差不多吧。我妈说了，能进前二百就行。"赵一鸣笑了笑。"晚上可以回去不看书了。"

然后他顿了一下。"刚才路过办公室——物理李老师在跟你班王老师说话。好像说到咱们班里谁的物理不太行——"没往下说。顾临风没接话。赵一鸣看了看他。"哎——你物理多少。"

"八十二。"

"靠。比我高二十——"

赵一鸣转身就想往回走。然后停住了——回过头。"我不是说你。"

顾临风把手上的铅笔放下——笔离开纸上。没说什么。

晚自习正常上。

班主任巡视完之后让大家自已复习——自修课的安静不像是白天上课那种安静。晚自习的安静是成片的。天花板上的日光灯管，一根有点闪——在角落里每隔半秒就要忽暗一下。黄白交替。有些人的台灯也开了——桌灯是那种塑料夹子式的，LED灯是蓝白光。一桌子都照得清楚。灯光打在书上反射在脸上——大家都挺白。

教室里很静。翻书声——偶尔。有人在翻数学书。有人翻英语——旁边有人在用塑料尺划线。笔掉在地上的声音——清脆一响。笔滚了一段。有人咳嗽——忍了半截。有人喝水——塑料杯底磕在桌面上——声音挺响。他后面那个人在转笔——圆珠笔在手指间转了一圈掉了。在地上滚了两下——他弯腰捡。

苏晚晴没趴。

晚自习刚开始的时候她趴在桌上——大概两分多钟。第三分的时候直起身。取出物理笔记本放在桌面上。那本本子的封皮是深蓝的——边角翘了一点——是经常翻导致的。然后又摊开物理练习册。

她咬着笔帽——圆珠笔，蓝杆。笔帽是蓝色的——咬的次数不少——塑料帽顶上有小小的牙印。眼睛看着题目，一动不动。题目好像很难——她看一遍。又看一遍。放下笔翻笔记本。翻到某一页——手指停下——指尖指着某一行字。她笔记本上的字比较圆——行距也一致——她大概昨晚预习的时候好好整理了。对着那一行字看了六七秒。然后翻回去——继续算那道题。

日光灯管闪了一下——那个坏灯管在角落里重新暗了一下。满屋子的人都习惯了——没人抬头。只有靠墙坐的那个女生在灯管闪的时候正好抬头——看了一眼灯管——然后低头继续做。

苏晚晴也抬头了。看了一眼闪动的灯管。然后低头继续做。右手握笔——左手拿三角尺在作业本上比着。尺子划的地方画出一条辅助线——尺子收起来的时候纸张发出细碎的一道摩擦声。

顾临风把这些都看到了。四排+两个过道之外。他在自己的座位上坐着。面前的数学练习册摊开着。笔在纸上停了好一阵。前面那几道题——他做了两道，后面六道还没动。看了一会儿前方。低头。翻了一页书。看了三行字。合上。拿出物理练习册。翻到倒数几页——那后面是提高题——题少但难度比前面的基础题高一档。拿起笔。在草稿纸上画受力分析。箭头——画了四五个箭头。方向没画对。

晚上十点。宿舍熄灯。

走廊里有宿管老师的脚步声。沙沙沙——周老师的鞋底永远是这一种橡胶底磨水泥地的声音。"关灯了。"门缝底下透进来的光突然一收。铁门被拉上的声音——另外一头是厕所方向的。上锁——钥匙在锁孔里转。卡——卡——两圈。

顾临风没睡。宿舍里灯都黑了。窗户外面路灯的光漏进来——现在路灯换上了一颗新的。光线比上周亮一些。他用手背碰了下枕头下面的手电筒——塑料手电筒。外壳是蓝色的——是学校门口地摊上买的。三块钱一个。他摸出来。推开关——嘎——光打在墙壁上。一个小小的黄色光圈。最中间有两个亮斑——是因为手电筒的塑料壳歪着——灯泡没对齐中心。他把另一只手伸到被子里——挡住漏光。

物理练习册摊开。用手肘固定住页角。手电筒的光圈沿着题目一行一行走。字体在手电筒下是暗黄色——纸面上的钢笔字被黄光照得勉强可读。

一道题。受力分析——重力。弹力。摩擦力。匀变速运动——位移公式。两道题——速度合成。三道题——能量守恒。动能转换。

他做到第三道的时候。手电筒灭了。光圈突然消失——眼睛还没适应黑暗。视觉里的残像是手电筒光圈——黄色的残影在黑暗的墙壁上漂着。他把手电简摇了摇——电池在里面滚动——又亮了一下——弱了一点。然后又灭了。电池的金属底已经锈了——可能是上次不小心压到水弄湿了。

他放下手电筒。眼睛闭着。脑海里是刚才做的那道题的受力图。

上铺的人说了句梦话——"妈——我不吃——"后面没说完。沉寂了。然后翻了个身——床板吱哑响。又沉寂了。

他把物理练习册合上。放回枕头底下。合上的时候书页压出轻微的纸张摩擦声。练习册的边角折了——上次翻的时候不小心的。他在黑暗里用手指把折角抹平——从折角里往外捋。捋了两下。捋不平。折痕不会因为你捋了就消失——它还在那——只是被压平了。

然后侧过身。面对着墙。墙上贴的那张课程表在月光里是一个模糊的灰方块。月光从窗帘缝隙漏进来——月亮是半个——不是满月。星期五最后一格——班会。旁边有一行小字:"交期中总结"。字是铅笔写的。他看清楚那个字是因为上次重新描过了。

窗外。月亮在暗处。宿舍里的其他五个人都在睡。有人翻身的时候被子滑下来——拖到了地面。

第二天早晨。他第一个到教室。那时候刚过六点十分。天还没全亮。教学楼门口的铁栅栏刚打开——保安师傅正弯腰往传达室里走。三班教室的灯还没开。走廊里只有这一个教室的黑板能看到一点点光亮——窗外天光从铝合金框的方向打进来。

他把书包放下。从抽屉里掏出物理练习册。翻开昨晚做的那三道题。在日光下面看了一会儿。光线是凉的——早上六点半的光。教室还暗着。但能看清字了。

第一道——错了。方向搞反了。手电筒那会儿画方向的时候可能看错了。斜面上的摩擦力方向——他画成了沿斜面向上。而题目给的力是平行于斜面向下推。那样的话摩擦力的方向就是沿斜面向上了。就是说——手电筒那会儿他画对了？不对。等等。重看。他拿橡皮把受力图擦掉。橡皮在纸上磨——橡皮屑落在练习册的页缝和桌子之间。三角块后面有一道白印子。他重新画。新答案写在旁边——这次用的是蓝色圆珠笔——不是铅笔。箭头画直了——用尺子比的。摩擦力箭头沿斜面向下——因为外力沿斜面向上推——推的方向是外力方向——摩擦力就沿斜面向下。

第二道——他昨晚居然漏了一个已知条件。v₀=0——初速为零。少代了一个零。重新算——加速度a=2m/s²。位移s=at²/2=2×9/2=9米。对上了。旁边打了个星号——提醒自己下次仔细读题。

第三道——对了。手电筒光很弱的那一题。用能量守恒做的。对了。

六点五十。门被推开。冷风从走廊灌进来——外面下了霜。教学楼前面的草地上是白的。

苏晚晴进来。手里捏着一个包子——塑料袋是白色的，里面凝了水珠。肉包子还是素包子——透过塑料袋看不清。包子还在冒少许热气。走到第二排靠窗。坐下。把书包拉链拉开——拉链上的鱼轻轻晃了一下——把今天第一节课的英语课本拿出来。然后拿起包子咬了一口。素包子——韭菜鸡蛋。她低下头——包子放在了左边的桌面上。先吃素的。

顾临风低下了头。手里的笔继续动。受力分析图上的箭头画得很端正——笔直的一条——每根箭头长短差不多。他往左下角看了一眼——四排和两个过道之外。隔了六米。隔了一张张排满座位的教室过道。然后收回目光。擦掉牛顿第二定律中多画的一个符号。把练习册翻到下一章。

窗外的银杏叶早黄透了——已经掉了一大半。还剩下几片，贴在树枝上。太阳出来了一点。银杏叶的反光打在窗户上边角的位置。

然后他把铅笔放下。翻开了英语课本。今天是周一。早上第一节是英语课。
