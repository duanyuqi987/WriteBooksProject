# 第135章 六年

时间：2018年3月周六下午

地点：楚州"一茶一坐"茶餐厅

卷次：卷七：流年未断

后来话题慢慢松了。

不再是一问一答。不再有那种"轮到谁开口"的默契。而是——一句话还没说完，下一句就从对方的眼睛里接上去了。然后顺着往下说。像两辆车从不同方向开上同一条路。开始是各开各的。后来发现——是同一条路。

话题从楚州的天气开始。三月。樱花开了又落了。她说一小操场旁边的樱花树今年开得特别早——二月底就开了。花期短。一个星期就落完了。他说二中的樱花现在还开着——晚樱品种。花瓣是重瓣的。粉色的。一团一团。她听着。点了点头。把茉莉花茶倒了一点在新杯子里。推到他面前。

"尝尝。还是热的。"

他端起来喝了一口。热的茉莉花茶比凉的好喝。甜味在舌尖上先来。然后才是茉莉的香。他说了一句"好喝"——然后把茶壶拿起来，也给她倒了半杯绿茶。他的茶已经凉了。他招了手——服务员续了水。绿茶在热水里重新舒展开来。

说起工作——她说她毕业后考了一小的编制。第一年笔试差了三分。她记得那个分数——七十九分。差三分进面试。她把成绩单贴在冰箱上看了好几个月。第二年重新看了一遍教育学教材——把高数的题也做了——考过了。面试的时候讲了一节课——讲的是《荷花》。评委给了高分。她说那节课她准备了三天——对着镜子练了很多遍。把话说得自然。

"考了两次才考上。挺丢人的。"她说。然后用手指敲了敲杯沿。

"不丢人。"

"你考了几次。"

"一次。"

她点了点头。好像早料到了。嘴角的弧度多弯了一点点——替他在心里划算。好像又看到了高三教室后墙成绩单上那个慢慢往上挪的名字。

说起研究生——他说考研的时候英语拖了后腿。政治靠背的——每天早晨五点半起来背。英语找了一个考友帮忙补——考友后来考上了北师大，去了北京。他考了两次。第一次英语没过线。第二次过了。上岸的时候赵一鸣给他打电话——说"你他妈终于考上了"。他说赵一鸣在电话里骂了他五分钟。但她有没有人给她打电话说这种事——他没问。她也只是笑了笑。

"你以前成绩就挺好。"她说。语气很淡。

他愣了一下。

这句话不是一个问句。也不是一个感叹。就是一个很淡的陈述——像在说今天的天气，或者路口那家面馆的面比之前咸了。

"你怎么知道。"

"我看过成绩单。"

她的声音没有变。端杯子的手也没有停。

空气静了一下。

他看着她的眼睛。想在她的眼睛里找到这句话的分量。但没有找到。她好像真的只是记得一张成绩单——贴在教室后墙上的、用A3纸打印的、几十个人的名字排成一列的那张纸。上面有他的名次——不太好，但她记得。

他端起杯子喝了一口。茶已经凉了。凉茶有点涩。他含在嘴里多停了一秒——然后咽下去。

"你记得好多东西。"他说。声音比前一句话低了一些。

她没有回答。只是低头看杯子里沉在底部的几片茉莉花。花瓣已经泡得透明了，经脉隐约可见。她用杯子的余温暖着手。杯壁上凝了一层雾。

过了一会儿。她开口。

"你大学也是在楚州读的。"

"嗯。楚州师大。"

"中文系。"

"对。"

"我猜也是。"

"你怎么猜的。"

"高中你作文写得好。而且你读诗。"她看了他一眼。"我记得有一年元旦晚会——你在黑板上写了一首诗。应该是你的。"

他僵住了。

高一那年的元旦晚会——每人要出一个节目。不会唱歌不会跳舞——他在黑板上写了一首诗。写完之后回到座位上。没人在意那首诗是谁写的。他以为没人知道。

但她知道。

"你看到了。"

"我在教室里——收拾东西。你写完就走了。我走过去看了一眼。"

"记得是什么吗。"

"前面忘了。最后一句——'待到来年花落时，不知道你在哪里'。大概是这样。"

他低头。茶杯里的茶水已经凉透了——凉得像井水。他喝了一口——把杯子里的剩茶一口喝完。

写那首诗的时候他十六岁。不知道诗好不好——只是心里的句子憋太久了，不写出来的话喉咙里会堵着。写完之后他不敢署名。写完了就快步走回座位，低头看桌上的课本——假装从头到尾都是别人写的。

他不知道她看见了。更不知道她记得。

过了很多秒。他抬起头。

"那首诗——不太好。"

"挺好的。我背了很久。"她说。

然后她好像意识到自己说了什么——低下头。马尾的发梢从肩上滑下来——遮住了半边脸。

他把空茶杯放在桌上。手在杯口边停了一下。然后把茶壶端起来——壶里剩了一点底。他给自己倒了最后一点——茶水浑浊，带着茶叶末。他一口气喝了。

"你在师大读的什么。"

"教育学。"

"也在中文楼上课。"

"对——中文系在二楼，教育系在三楼。"她顿了顿。手指在杯沿上停住了。不转了。"我有时候会从你们楼下过。走楼梯的时候经过二楼——有时候会往走廊里看一眼。看了一会儿。没碰到过你。"

他在脑子里迅速地翻——他在二楼走廊上走了两年。中文系的课都是在这栋楼上的。他在教室里上了几百节课。在走廊上走过几百次——去厕所、去接水、去办公室交作业。每次在走廊上走的时候，头顶就是三楼的地板。天花板上有脚步声——嗒嗒嗒的——有时候轻有时候重。他从来没想过抬头。三楼的教育系。教育系有几百个学生。其中有一个人。是她。

她在他头顶走过了两年。

不知道的时候，一切都没发生。知道之后，每一个已经过去的瞬间都像是错过了。

她大概在想一样的事——因为在说完"没碰到过你"之后，她的眼睛暗了一点点。只是一点点，零点几秒。然后她端起杯子——才发现杯子里已经空了。

他把自己的茶壶推过去。壶里剩了一点绿茶——浑浊的，凉的。她看了一眼。笑了笑。倒了一点在杯子里。尝了一口。眉头皱了一下。

"苦的。"

"你的茉莉花茶是甜的。"

"放糖了。"

他愣了一下。然后笑了。苏晚晴——喝茉莉花茶放糖。这个细节他从来不知道。过去八年来他对她所有的想象里——从没有出现过她往茶里放糖的画面。

她看了他一眼。"怎么了。"

"没什么。"

"我高中就放糖了。"

"我不知道。"

"你当然不知道。"她笑了笑。"你又没跟我喝过茶。"

这句话让两个人都静了。

窗外有个环卫工在扫人行道。扫帚刷过地砖——沙，沙，沙。把前一天落下来的樱花花瓣扫成一小堆。然后铲进垃圾桶里。

他看着她，喉咙动了一下。"以后可以。"

她低头。耳朵尖红了一点。不是害羞的红——是被看穿了的红。是被人认出了什么不愿意说破的东西的红。

"嗯。"她的声音压得很轻。"以后可以。"

然后她站起来。说去一下洗手间。

他从桌面上拿起她的杯子——杯沿上那个淡色的口红印还在。他看着那个印子。然后把杯子轻轻放回去。

窗外太阳又移了一格。阳光现在只能照到桌角了。他的左手还在光里——右手已经暗了。

她回来了。坐下来。把茶壶拿起来晃了晃——空的。她招了一下手——服务员过来续了水。开水壶往壶里倒水的时候，他注意到她的眼睛有一点红——鼻尖也是。大概是洗了脸。

他假装没看见。

她坐下。把茶杯推到他面前。里面是新续的茉莉花茶——没放糖。

"尝尝不放糖的。"

他端起来喝了一口。不甜。有一点淡淡的茉莉花香。咽下去以后——唇齿间有一点涩。

"不放糖也好喝。"他说。

她笑了。也给自己倒了一杯。绿茶——不放糖。尝了一口。皱眉。然后从包里拿出一个小纸包——白砂糖。打开了往杯子里倒了一点。用勺子搅了搅。勺子和杯壁碰撞——叮叮响了两声。然后放下勺子。

两个人继续喝茶。窗外的路灯亮了一盏。

天快黑了。
