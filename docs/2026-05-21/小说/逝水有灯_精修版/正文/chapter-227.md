# 第227章 南湖再见

所属部卷：第五部：逝水有灯
时间：十年后冬天
地点：大学南湖凉亭

## 诗歌插页 P227：原创仿唐绝句

新绿微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

事情落到当天，没有被命运单独照亮。

诗社帐篷早就不在了。缓坡上那块刻着"明德"的石头还在。石头上多了些青苔，绿色的，深一块浅一块，像是有人用湿毛笔在灰色石面上点了不规则的斑点。

陆允川在凉亭里坐下来。石凳很凉，隔着裤子也能感觉到那种沁人的冷。他把外套的下摆拉了拉，垫在底下，好一点。

有两个男生从湖对岸走过来。一个戴着眼镜，一个抱着滑板。他们走到亭子的时候大概没想到有人坐着，脚步顿了一下，然后绕到亭子另一边，压低声音继续讨论刚才的话题——好像是关于某个比赛的报名截止日期。

诗社解散的那天，也是在这座亭子里。当时来了七个人：他、程砚、叶澄、祁野，还有三个低一届的社员。没有人说解散。只是程砚说"下学期我不来了，要考研"。然后叶澄说"我也不来了，要实习"。然后一个接一个。最后就散了。

没有悲伤的仪式。就是人少了。然后就没了。

现在坐在这张新的石凳上，看着换了漆的柱子，他突然觉得解散是对的。不是因为诗社不好，是因为它应该在那个时候结束。就像某些关系应该在某个节点停止——不是因为不好，是因为再延续下去就会变形。

他用鞋尖踢了踢地面。地面的瓷砖有一块是松的——以前也是松的。松了这么多年，居然没换。可能是每次换的时候都恰好漏了那块。

手机震了。工作群。有人在问下周的汇报材料什么时候交。

他回了个"周一前"，然后把手机静音了。

他用眼睛重新数了一遍亭子的柱子——六根。还是六根。但最左边那根大概是换过，因为它的颜色和另外五根有微弱的色差。不刻意看看不出来。他现在在数——数这些没变和变了的东西。不是要给它们赋予意义，只是单纯地想确认：十年的时间能改变哪些东西，又奈何不了哪些。

石头。青苔。松动的瓷砖。六根柱子换了中间一根——不对，不是中间的，是最左边那根。他能确定，因为那根上面的木纹是直的，而另外五根的纹路是弧形的。

一个保安骑着电动车经过湖边，用喇叭喊了一句"湖边禁止垂钓"。没有人在垂钓。保安可能是例行公事。

湖面的灯开始亮了。一圈暖黄色的光浮在水面上，被风吹成断断续续的碎线。他站起来，走到亭子最外沿，把手机对准湖面拍了一张。照片里什么都看不清——一盏路灯变成了一条斜长的光痕，湖是黑的，亭子的影子也是黑的。

不够清楚。他保留了。

走到凉亭柱子旁边的时候，他又摸了一次那块新漆。很光滑。找不到当年剥落的地方。他知道那个地方还在——在漆下面，在木纤维的深处，只是被覆盖了。

覆盖不是消除。

程砚说得对——有些事情不需要"然后"。

他走出凉亭，没有绕湖走第二圈。草坪刚修剪过，鞋底沾了几截草屑。他跺了跺脚，草屑落在地上。很快就和别的草屑混在一起，分不清哪一截是从他鞋底掉下来的。

校友卡在口袋里。他放进钱包的第三个卡槽。那个槽空了很久，之前装的是大学时那版校园卡——卡面褪色了，他在上个搬家的时候扔了。

旧卡的槽，装了新卡。

他把钱包合上。去公交站的路上，风又吹了几次。湖在后视镜里越来越小，路灯在黑色水面上还在亮着。
