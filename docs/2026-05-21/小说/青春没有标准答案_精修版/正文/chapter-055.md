# 第055章 楼上楼下的雪球



时间：2009年一月上旬，大雪后的午饭时间



地点：高一楼、高二楼之间的空地



卷次：第三卷：雪落在两栋楼之间



2009年一月上旬，大雪后的午饭时间，高一楼、高二楼之间的空地，落在高一楼台阶上的第一个雪球先把这一天按进了十班的日常里。高二楼有人先扔雪球，砸在高一楼门口，两个年级的喊声一下子炸开。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

十班原本只想看热闹，梁一帆和付小川却先把雪团捏在手里。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看落在高一楼台阶上的第一个雪球，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

宋乔也在这一天被推到前面。宋乔的特点是热闹爽快，女生群像核心，敢替朋友把话说出来。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“高一的，接住！”高二楼有人说。

“这能忍？”梁一帆说。

“别砸玻璃，砸人肩膀！”宋乔说。

“你这提醒也不算安全。”常俊凡说。

“十班只反击，不挑事。”付小川说。

午饭后雪越下越密，高一楼门前先铺了一层薄白，脚一踩就露出下面的水泥色。

第一个雪球从高二楼方向飞过来，不大，砸在台阶上散成一摊碎雪，却像一声开场哨。

高一楼走廊里立刻有人起哄，喊声从一楼传到三楼，连正在喝汤的人都端着碗探头。

梁一帆反应最快，弯腰抓雪时差点滑倒，付小川一边笑他一边也蹲下去。

宋乔站在走廊边喊别砸玻璃，话音刚落，又补了一句砸肩膀也轻点。

常俊凡被她逗笑，笑完又看了看楼上窗户，提醒大家别往教室里面扔。

段璟捏了一个很松的雪团，举起来又放下，最后砸中了高宇的书包。

那一刻十班像突然有了共同的心跳，很多还没熟透的人，借着一场雪同时笑了出来。

楼上楼下的雪球没有悬在空处。楼前空地的白雪和从二楼飞来的喊声，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，楼上楼下的雪球带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

王启民进门时不一定立刻说话，鞋底在水泥地上一停，楼上楼下的雪球带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在宋乔快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。楼上楼下的雪球越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。落在高一楼台阶上的第一个雪球让他想说话，又逼他先低头看题。

宋乔被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，楼上楼下的雪球留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。宋乔路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为楼上楼下的雪球翻身，谁还在被窝里背单词，旁边人都听得见。

两栋楼之间的空地忽然成了整个校园最亮的地方，连远处食堂门口都有人探头看。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理楼上楼下的雪球里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：落在高一楼台阶上的第一个雪球、一只冻红的手、一句老师点名、一块还没扫干净的雪。

第一颗雪球从高二楼三楼的走廊飞下来时，高一十班刚吃完午饭正在往回走。那颗雪球不大，大约一个搪瓷杯盖的大小，捏得很紧实，从三楼划了一道抛物线，啪一声砸在高一楼门前的台阶上，碎成一蓬白雾。台阶上刚被值日生扫过，新雪还没积起来，那一蓬碎雪落在灰色的水泥面上，像有人在上面打了一个白色的惊叹号。

"高一的，接住！"楼上传来一声喊。喊的人站在三楼走廊最外侧，校服袖子卷到胳膊肘，整条手臂冻得通红——这在零下五度的天气里是一种赤裸裸的挑衅。

梁一帆是十班第一个反应过来的。他本来走在队伍最前面，手里还捏着从食堂带回来的一截玉米——吃了一半，剩下一半本来打算回教室当下午的加餐。听到楼上那声喊，他站在原地，把那半截玉米往段璟手里一塞，蹲下去，两只手在台阶旁边的花坛边缘扫了一把雪。那雪是上午刚下的，松得像棉絮，握在手里还没来得及捏紧就从指缝里簌簌往下掉。

"你行不行？"付小川站在他后面，笑得弯了腰，"你捏雪球的手法跟捏豆包似的——"

话没说完，第二颗雪球就从二楼飞了过来，直接砸在付小川肩膀后面的墙上。碎雪弹进了他的领口。他的笑声在半空中掐断，整个人像被烫到了一样原地跳了一下。"凉的！凉死我了！"他一边喊一边把校服领子扯开往外抖雪，抖出来的雪渣沿着后脊梁往下滑，让他又跳了两下。

十班一下子就沸腾了。

走廊里吃饭回来的人听到外面轰的一声——不是雪球砸墙的声音，是高一楼整栋楼的起哄声。从一楼到三楼，每个班的门口都有人探出半个身子。有人手里还端着饭盒，有人把筷子夹在手指缝里就往走廊边上挤。高二楼那边也不甘示弱，窗口一个接一个地冒出了人头。两栋楼之间那块不到二十米的空地，忽然变成了一片被喊声灌满的竞技场。

段璟把梁一帆的玉米放在窗台上——那个窗台是整个高一楼最安全的角落，被两片暖气片夹在中间，上面已经放了好几个搪瓷杯和水壶。他回头找常俊凡，看见她站在一楼走廊的柱子后面，没有加入战斗，但也没有回教室。她的位置很巧妙——刚好在门框内侧，可以看清整个战场，但雪球从任何角度都打不到她。宋乔站在她旁边，半个身子已经探出了走廊。

"别砸玻璃！"宋乔冲外面喊，声音盖过了至少五个人同时发出的起哄，"砸肩膀可以——肩膀也轻点！"

常俊凡在旁边被她逗笑了。不是平时那种嘴角微微动一下的笑，是真的笑了——肩膀在抖，拿杯子的手晃了一下，热水从杯沿溢出来洒了一点在地上。她把杯子换到左手，用右手手背挡住嘴。"你这提醒，"她说，声音还带着笑的余震，"到底是劝架还是教人怎么瞄准？"

"我提供战术指导。"宋乔理直气壮。

两栋楼之间的雪球大战在接下来五分钟里升级到了空前的规模。高一楼这边的战术是以走廊立柱为掩体，三人一组——一个人蹲在花坛边负责造雪球，一个人站在柱子后面负责投掷，还有一个人在走廊里专门往造雪组运新雪。高二楼占据地理优势——他们高一层，抛物线的落点更好控制，而且他们楼前的台阶背阴，雪没化，雪源比高一丰富得多。

付小川负责投掷。他投雪球的姿势像投铅球——先把雪球举过肩膀，然后整个上半身往后仰，再猛地往前甩。但这个姿势的命中率很低，他连投了三个，全部打在了高二楼二楼和三楼之间的墙面上，连人家的窗台都没碰到。第四个雪球他调整了角度——结果用力过猛，雪球在出手的那一刻就散了，变成一蓬碎雪浇在他自己头上。

梁一帆蹲在花坛边造雪球，笑得蹲不住了，一屁股坐在地上。"付小川，你到底是在打雪仗还是在给自己洗头？"

何南星是造雪组的核心。他造雪球的速度极快——抓雪、压紧、揉圆，三个动作一气呵成。他面前的花坛边沿上排了整整齐齐一排雪球，大的有拳头大，小的跟一颗鸭蛋差不多。每一个都压得很实，表面光滑，几乎没有裂纹。段璟从他手里接过一个的时候，能感到雪球的密度比肉眼判断的要高——这颗雪球飞到高二楼三楼应该是不会散架的。

"你以前是不是经常打雪仗？"段璟问。

何南星头也没抬，继续造下一颗。"我家在城关镇开药材铺，冬天没有顾客的时候，门口那条街上的小孩就互相扔雪。我从小学开始就是那一块的雪球工厂。"他把新造好的雪球放在花坛边，排进队伍里，"记住捏的时候要用力，往手心压三次，压到雪的表面发亮为止。不发亮的雪球飞不到对面楼。"

段璟照着何南星的方法捏了一颗。他握在手心里用力压了三下，雪在手掌的温度下开始融化，水渗进雪球内部重新结冰，表面的确浮起了一层极薄的亮光——像上了釉。他把这颗雪球递给付小川。"用这颗试试。"

付小川接过雪球掂了掂分量。"这颗比你刚才给我的那颗重了至少一半。"他重新站好位置，瞄准——这次没有瞄高二楼的墙面，而是瞄了三楼走廊里一个正探着身子往下看的男生。他深吸一口气，把雪球往后拉到肩膀后面，然后猛地向前甩出。

雪球划过一道又直又快的弧线，飞过高一楼和高二楼之间那块空地，从正在互相喊话的两群人头之间穿过，精准地砸在了那个高二男生面前的栏杆上。碎雪弹起来，溅了那个男生一脸。他愣了一下，然后冲楼下竖了一个大拇指——不是挑衅的那种，是"可以啊"的那种。

高一楼这边的欢呼声炸开了。

常俊凡从柱子后面走出来了一步。她把搪瓷杯放在窗台上——和其他人的杯子并排放在一起——然后走到花坛边蹲下来，从何南星的"生产线"上拿起了一颗雪球。她捏了捏，把雪球在两手之间倒了一下，然后递给了刚从后面跑过来的宋乔。

"你不是说不参与吗？"宋乔接过雪球。

"我不扔。但有人要多练英语——"常俊凡站起来，拍了拍手上的雪沫，"扔雪球不耽误第八单元的单词抽查。"

宋乔假装没听见后半句，转过身就把那颗雪球往高二楼方向扔了出去。她的投掷姿势比付小川好——侧身，用腰劲，手臂甩出的弧度很漂亮。雪球飞过空地，擦着高二楼二楼的窗框飞了出去，差一点就飞进了开着的窗户里。高二楼那边有一个人吓得往后跳了一步，撞到了身后的同学，两个人一起跌进了走廊——但笑声很快传了下来。

段璟也加入了造雪组。他蹲在何南星右边，学着何南星的手法——抓一团雪，放在手心，用两根大拇指压一遍，再用整个手掌裹住压实。他的第一颗雪球表面有裂纹，放在花坛边没几秒就碎了。第二颗好了一点——没有裂纹，但没有何南星要求的"亮光"。第三颗他终于压出了那一层薄薄的亮面。他把这颗雪球放在手心里看了几秒，忽然想到一件事——这种从松到实、从裂纹到光面的过程，跟他最近做英语改错有点像。不是一下子就能对的，是反复压、反复改。

"段璟！别发呆！"付小川从后面拍了他肩膀一下，从他手心里抢走了那颗雪球，"高二楼又派人了——你看，他们造了一条流水线，比我们还专业。"

段璟抬头往高二楼看。确实。高二楼一楼走廊里，五个人排成一排，每个人负责一道工序：铲雪、揉团、压实、码放、投掷。他们的"工厂"规模比高一楼这边大了不止一倍，花坛边上已经堆起了一座小雪球垒成的小山。

"我们需要加快速度。"何南星说。他把眼镜推到额头上——镜片上沾了雪水，视线已经不太清楚了。"段璟、梁一帆负责铲雪，我负责压球，付小川、周明澈负责投掷。宋乔你站在走廊这头当观察员，如果高二那边有人直接从楼梯冲下来，你立刻喊。他们的地面部队也需要防范。"

这个临时起意的战术分工竟然被所有人严格执行了。梁一帆用搪瓷杯的盖子当铲子，从花坛里往外挖新雪，段璟把雪运到何南星面前。何南星的眼镜推到额头上以后就一直在往下滑，每做三颗雪球就要用袖子蹭一下鼻梁，但他造雪球的速度不但没慢，反而更快了——他的手下意识地记住了那个"压三次"的节奏，不需要再用眼睛看了。

常俊凡不知道什么时候从窗台上拿回了她的搪瓷杯。她站在走廊内侧，没有加入生产线，但她的位置刚好可以看到整条战线——从段璟铲雪的花坛到付小川投掷的柱子，再到高二楼那边的窗口。她没有指挥——以她的性格也不会主动站出来当指挥——但每当有人忘了回头看，她会用一个很轻的声音提醒一句："何南星，你眼镜要掉了。""段璟，那块雪底下有泥。"

这些提醒轻到只有被提醒的人能听见，但每次都很准。

十分钟后，高二楼那边有人开始往下跑。不是来打近战的——是他们的班主任出现在了三楼走廊尽头。那是一个穿深蓝色棉袄的中年男老师，站在走廊尽头没有走过来，只是把手里的文件夹举高了一点，喊了一声："都回去上课！"那个声音不大，但穿透力极强，两栋楼之间所有正在喊的人都听见了。

高二楼那边的"工厂"在三秒之内就解散了。五个人同时把手里的雪往花坛里一撒，转身往走廊里跑，最后一个人还因为地面太滑差点摔了一跤，被后面的人拽住帽子拉了回来。他们的流水线原地留下的遗迹——花坛边沿上排着一排没来得及投出去的雪球，整齐得像是要在那里办一个小型雪球展览。

高一楼这边，何南星也把手里的雪球放回了花坛。他站起来拍了拍膝盖上的雪，把一直滑到鼻尖的眼镜推回原位。

"高二的班主任发现得快。"梁一帆说，"我们这边——"他回头往走廊深处看了一眼，高一楼这边暂时还没有老师出来的迹象。王启民（教导主任）的办公室在一楼最东头，离他们的战场隔了一整个年级走廊和两扇关着的铁门。

"收不收？"付小川问，手里还握着一颗没投出去的雪球。那颗雪球已经被他捏得发亮了，表面紧实光滑，是何南星"工厂"生产的最后一颗。

"收。"周明澈拍了拍手上的雪水，"打够了。再打下去不是雪仗，是给教导主任送检讨素材。"

但宋乔没有收。她站在走廊最外侧，双手捧着一颗最大的雪球——足有两颗拳头合在一起那么大——冲着高二楼方向高高举起来。她没有投，只是举着，对着已经空荡荡的高二走廊，喊了一句："下次别先从楼上扔——这样不公平！"

楼上没有回应。但二楼最边上的那扇窗户里，有一个人隔着玻璃冲她比了一个OK的手势。

全班人都笑了。

段璟把搪瓷杯从窗台上拿回来时，发现他的杯子被旁边的人换了位置——他原本放在最外面，现在被挪到了里面一排，和常俊凡的杯子并排放在一起。两个搪瓷杯挨得很近，杯身上都印着"封丘一中"四个字，字体不一样——他的杯子上是楷体，她的杯子上是宋体。

他抬头找常俊凡，看到她蹲在花坛边，把刚才战斗中散落在台阶上的碎雪归拢在一起，捧起来放回花坛里。她做这件事的动作和她在课间收拾讲台上的粉笔头一模一样——不是值日生的义务，但看到了就会做。她的手指在雪里按了一下，按出一个浅浅的指印，然后站起来把手插回校服口袋里。

"你不冷吗？"段璟站在她旁边，犹豫了半秒后问了这句。

"冷。"常俊凡把围巾往上拉了拉，盖住了半张脸。"但刚才宋乔举那个雪球的时候，我笑得太大声了——笑完就不觉得冷了。"

段璟想了一秒才知道她在说什么。不是雪球本身，是宋乔冲着高二楼喊那句"不公平"的时候，她站在走廊里面笑了一声——刚才在混战中段璟也听到了那声笑，但没来得及转头去看。那声笑的音量确实比平时大了不少。不是被逗笑的那种节奏，是从胸口直接迸出来的，没有过滤也没有收束。

"所以你也觉得不公平？"段璟问。

"从一开始就不公平。他们站得高，雪源多，先动手——"她把围巾从脸上拉下来，露出嘴唇和下巴，"但如果我们比他们团结，不公平就能被拉平。"

段璟把这句话在脑子里记了下来。她现在说的是雪仗，但段璟知道她可能也同时在说另外一些东西——关于她坐在第一排被所有人看着的压力，关于十班在新座位安排下需要重新磨合的关系，关于她自己在一大群各怀心事的人中间，坚持把每一件小事都做对。

傍晚时分，雪停了。两栋楼之间那片空地上落了一层新的薄白，盖住了上午被踩乱的脚印和碎雪的痕迹。走廊里的雪水被值日生拖了一遍，但鞋底踩过的地方还是能看到浅浅的水痕。

马立群在经过走廊时看见了窗台上那几个搪瓷杯——大约七八个，都印着"封丘一中"的字样，排成一排靠墙放着，没有一个掉在地上。她停下来看了一眼杯子排列的方向，对旁边的陶建国说了一句："你们班这杯子排得挺整齐。"

陶建国看了一眼，只说："那不是杯子的问题。"

晚自习前，段璟回到教室时发现桌上多了一样东西——不是纸条，不是单词本，而是一块用旧报纸包着的烤红薯。他打开报纸——报纸还是那张印了奥运新闻的，铅字模糊——红薯已经有点凉了，但掰开来里面的瓤还是软糯的。他不知道是谁放的，但红薯旁边被撕掉的报纸角上，有三个他认得的字迹组成的残笔：买给——

后面一个字被撕掉了。但段璟认出前面两个字的笔迹风格——和常俊凡在单词本封面上写自己名字时的笔触一模一样。买给谁？他没有看到完整的句子，但他把那张撕剩的报纸角折好，夹进了文具盒最下面那层。那里已经有了两张纸——英语改错答案和数学公式纸条。现在又多了一张烤红薯的包装纸角。

付小川从后排探过头来，看见了红薯。"谁给你的？"

"不知道。"

"你又不知道。"付小川笑了一声，但这次他没有说更多。他只是从自己的桌洞里拿出一张数学卷子，在段璟面前摊开。"来，帮我看看这道题。梁一帆讲了一遍我没听懂。他说要用三角函数的性质——"

段璟接过卷子。教室里暖气片又哐当响了一声，窗外的路灯准时亮了起来，在雪地上铺了一层橘色的光。前排常俊凡坐在第一排靠窗的位置上，从段璟的角度可以看到她的背影——她的围巾搭在椅背上，校服领口折得很整齐，正低头在改今天的英语默写本。

他想，今天大概是他进入这所学校以来过得最开心的一天。不是因为打了雪仗，而是因为他第一次同时看到了十班的二十多个人站在同一条战线上的样子。造雪球的、投掷的、侦查的、在走廊里喊规则的、站在柱子后面笑的——每个人都在做一件没有分数、不计排名、不会被写进成绩单的事。

而常俊凡在混战中说的那句"如果我们比他们团结，不公平就能被拉平"，大概就是这一天最准确的注脚。

楼上楼下的雪球之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

宋乔没有因此变成特别醒目的人，可在段璟眼里，宋乔从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，宋乔一边笑一边提醒别砸玻璃，十班被卷进全校最响的一场冬雪。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
