# 第217章 旧日记本

所属部卷：第五部：逝水有灯
时间：第二次搬家前
地点：搬家纸箱旁

## 诗歌插页 P217：原创仿唐绝句

纸页翻开旧墨深，
少年用力到如今。
不将怨字重誊写，
只认灯前未稳心。

陆允川最终还是翻开了。第一页写着县一中的日期，字很小，行距挤得紧。再往后，有分班红榜，有第一次借笔，有高考后的水，也有河边那场迟到表白。每一页都像把当年的自己推回面前。

他读得很慢。读到大学部分时，纸面上出现许多划掉的句子。有一页写着“她是不是故意”，后来又被蓝笔重重划掉。另一页写着“我再也不找她”，后面却隔了三天又写“今天在楼下等了很久”。陆允川看着那些反复，脸上发热。

日记本没有骗人。它记录的不只是深情，也记录了自怜、占有欲、误读和求证。过去他总想从这些页里挑出能证明自己受伤的证据，后来才发现，证据太完整，完整到连自己的不体面也一起保存了。

搬家纸箱旁的地板有一道划痕，是上次拖书柜留下的。他把日记本放在膝盖上，翻到一页空白，拿出黑色中性笔，写了一行：这些都发生过，但不能只按我当时的理解保存。

写完，他没有撕掉旧页，也没有往后补长篇辩解。纸张很薄，旧墨透到背面，新字也会透过去。人能做的不是把过去改干净，而是承认它曾经那样乱。
