# 第049章 元旦晚会旁观

所属部卷：第一部：初秋三楼
时间：高二元旦前
地点：布置中的教室

## 诗歌插页 P049：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
晚会上被风铃碰响的伴奏音还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，元旦晚会是全校一起看表演的一个热闹节点。台上表演相声的忘词、跳舞的跳到两边塑料屏风撞倒了——台上台下笑成一团，笑声把冷冬的干冷气挤出了会场。理科一班上台演唱时台下有同学们自己班的叫好声——"理一最棒"——有人举着一块用修正布写成的简易灯牌。

对程砚来说，这场晚会是他的义务工——他布置了舞台并负责唱后排看大家有没有走音。他后来在跟叶澄交流观察时说:"老陆唱歌的水平明显不如他物理考试的水平——他唱到'友谊地久天长'的时候节奏对但音准差。不过你知道吗——我发现他不是因为不会唱——他在唱副歌那段的时候有回头往右边瞄了一下——你知道他在瞄谁吗？"叶澄说"他没瞄我。"程砚说"废话——他会瞄你除非是物理考试时你站在答题卡旁边。"

对叶澄来说，这个晚会的重点是——陆允川终于完成了他对她的第一次自我主动表达。她说:"他之前对她说的那些话都是被动触发——'你捡''谢谢''你这边拖过了'——全部是因为有外部事件触发了他才说话。元旦晚会他在散场之后对她说'你的那一段很稳'——这句话没有外部事件触发——是她问他答。他跨过了从'被动反应'到'主动表达'的边界——虽然只有七个字——但这七个字的分量超过前一年加起来的那所有零碎字。"程砚说"你这个分析很学术——应该说他的'被动反应态'到'应答态'的转变"。叶澄说"不止——是应答加个人判断——他不是一个只在回应的人——他还在对她的表现做了评价"。

蒋雯全程在帮老师发节目单、帮后排的同学摆椅子、数座位号——她非常忙，几乎没有时间注意到后排男生们的持续观测动态。但她走到侧台时撞见了一幕——夏知遥的胳膊被挤到他后面压在他肩上大概好一会儿——两个人都没动。她心里想:"这个姿势站得这么挤连我都被挤成这样了——他们两个大概根本没注意到。"然后她绕过侧台继续去做自己的工作。

对夏知遥来说，这场晚会的重点是四班的街舞、三班的小品、理一的合唱、最后那个班主任们即兴跳舞。她给蒋雯拍了元旦合影——用蒋雯的数码相机，拍完之后两人凑在相机屏幕前看，蒋雯说"这张我闭眼了重拍一张"。她给大家写了一张卡片放在讲台上——卡片是从校门口文具店买的，折起来是一棵圣诞树的形状，打开里面是空白，她用黑色水笔一笔一划地写："新年快乐——给理一同班所有人。愿2012年大家身体健康。"——签名只是一个字"夏"。陆允川在晚上清理教室时看到了那张卡片——他停了几秒钟——念到"所有人"这三个字——其中包括他自己。他没有把它带走，把它放回讲台上让明天值日生交给方老师换成晨报。他在卡片旁边站着，双手揣在衣兜里，心里被"所有人"的定义圈住了——他是所有人的一员——他不需要单独被提到，这也很好。因为被提到就等于被记得——被他人存在的目录记下了——虽然只有在检索班级名册时才翻得到——但还是记到了。

很多年后程砚跟他喝酒时提到:"你觉得为什么元旦快乐四个字那么平常，你反复回忆？因为那是你们之间互相说的第一句无外部事件触发的完整对话。不是作业本，不是伞，不是拖把，不是考场——是对话本身。对话的主题不是物体，是喜欢或希望——那个主题叫'新年'。你们是在新年祝福的语境下、面对面、说出彼此的名字——那才是你们之间真正意义上的第一句交流——那以前所有的话都是被动触发的环境音。"
