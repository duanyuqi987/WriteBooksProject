# 第233章 诗歌索引之后

所属部卷：第五部：逝水有灯
时间：索引完成后
地点：自己的书房

## 诗歌插页 P233：原创仿宋小令

红笔尽，纸声微。
灯下索春归。
删去浓词见旧扉。
不许怨成碑。

索引完成后，陆允川把电子表格备份到硬盘，又把纸质稿夹进文件夹。文件夹封面写着“诗歌索引”，下面标注版本日期。这个动作很像工作，甚至有点枯燥，却让他心里安稳。

第二天早上，他重新读前几章。删过诗之后，正文反而透气了。以前某些章节像急着告诉读者“我已经懂了”，现在只留下新课本书皮、楼梯转角、校门口水瓶。懂不懂，不必总由作者站出来说明。

中午他给程砚发了几页索引截图。程砚回得很快：“你现在写小说像做审计。”陆允川回：“也许需要审计一下自己。”程砚发来一句：“审自己可以，别审别人。”

这句话让他停了一会儿。他在索引备注栏加了一条规则：涉及他人选择的诗，不使用判词。写完后，又把几处“冷”“狠”“辜负”替换成更具体的词：沉默、转身、未回复、没有停留。

下午阳光照到书桌边缘，纸页上的红线显得很细。陆允川忽然觉得，诗歌索引不是为了让全书更文艺，而是为了防止情绪重新泛滥。它像书桌上的尺子，量不出心疼，却能提醒手别写过界。
