# 第008章 接力棒

时间：2010年10月上旬，下午

地点：江城大学田径场

卷次：第一卷：九月的新校门

十月八号，院运会。中文系和教育学院共用一个操场。

各班的方阵走过去之后，广播里开始放《运动员进行曲》，翻来覆去那一首。林玉起坐在看台第五排，旁边是郭向阳。

郭向阳是来给计算机系当啦啦队的，但他从坐下来开始就在用手机看NBA文字直播。"火箭打湖人，"他头也不抬地说，"今天麦迪手感好。"

"嗯。"

"你在看什么？"

"看比赛。"

"还没开始比呢。"郭向阳抬头扫了一眼操场，"你骗谁呢。"

林玉起在看女子4×100米接力的检录处。

苏安晴站在第三棒的位置。她穿一件白色T恤，胸前印着"教育学院"四个红字。下面是黑色运动短裤，露出一截很直的小腿。她把号码布别在腰侧，别了两下才别正。旁边一个扎马尾的女生教她怎么别——林玉起后来知道那女生叫沈悦，练过田径。

苏安晴做热身的时候动作很认真，不像旁边几个女生边压腿边聊天。她从脚踝开始活动，膝盖、腰、肩，每一个关节都转到。像她做任何事一样——有条理，不敷衍。

发令枪响了。

第一棒跑出去。第二棒交接顺利。

第三棒。

苏安晴接过棒的那一刻，身体前倾的角度刚刚好。她不是跑得最快的——沈悦比她快，物理系有个短头发的女生更快——但她跑得最稳。步频均匀，手臂摆动不慌，呼吸节奏不乱。

林玉起站起来。

郭向阳从手机屏幕上抬起头，看了他一眼，又看了看跑道，又看了他一眼。"原来你在看这个啊。"

"看比赛。"

"行，看比赛。"

苏安晴交棒之后，弯着腰喘了几口气，然后直起身，和第四棒的女生击了一下掌。

教育学院最后拿了第三。

颁奖的时候林玉起没有看。他在笔记本上写：

>
> 接力棒递出去的那一刻
> 跑道像一条没有署名的信
> 你用脚写下去
> 我用眼睛读

"不是我说，"郭向阳凑过来，"你这个诗写得也太含蓄了。人家跑个接力赛你就写成这样，以后人家要是请你吃饭你还不得出诗集。"

"她不会请我吃饭。"

"你怎么知道。"

林玉起没有回答。

操场的风把他笔记本的边角吹起来。他用手按住，掌心里有一点汗。

十月的江城已经开始凉了。看台下，苏安晴和沈悦并肩往教育学院的看台走。她手里拿着刚发的奖状，卷成一个筒，握在掌心里。

经过中文系看台的时候，她往上看了一眼。

林玉起不确定她看的是不是自己。

但他把手从笔记本上挪开了。风把纸页翻到了前一页，上面什么都没有。
