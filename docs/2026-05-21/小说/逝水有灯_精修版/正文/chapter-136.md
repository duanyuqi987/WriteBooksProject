# 第136章 叶澄的劝告之前

所属部卷：第三部：寒风里的花
时间：六月午后
地点：南湖凉亭

## 诗歌插页 P136：原创现代诗

我把南湖水放在黄昏里，
不让它替任何人回答。
风从校园的一侧经过，
带走一句没有说稳的话。
后来我学会把手松开，
让灯照着路，也照着空出来的地方。

叶澄约陆允川在南湖凉亭见面的时候发了一条特别的消息："下午三点，南湖凉亭。不要跟任何人说。包括苏晚枝。"

陆允川到了。叶澄已经坐在凉亭的长椅上，面前摊着一袋薯片和一罐雪碧。她拍了拍旁边的位置，示意他坐下。

"什么事这么神秘？"

"你等一下。"叶澄撕开薯片袋，吃了几片，喝了口雪碧，然后面向湖面说了一句开场白："我是你们这段关系的旁观者。从诗社传单那天开始——我把传单递给你——到今天，我看着你们两个从讨论现代诗到讨论两棵树到讨论瓶盖门帘。"

"瓶盖门帘你也知道？"

"我在301教室看到过。那串三色瓶盖——橙子是你，柠檬是苏晚枝，青苹果是何屿白。她把它挂在门上，用瓶盖挡住了冷光。"叶澄转头看他，"你觉得她做这个门帘是为了什么？"

"她说——实物比道理更有效。"

"对。她在给你做心理辅导。而且是实物版。"叶澄又吃了一片薯片。"你觉得一个女生花时间把瓶盖一个一个捡起来、穿孔、串成门帘——是为了谁？"

"为了诗社。"

"你觉得诗社需要门帘吗？"

陆允川没回答。

"诗社不需要门帘。"叶澄自己答了。"苏晚枝需要。她需要你知道——不同颜色的盖子可以一起挂在同一根绳子上。橙子不会被柠檬挤掉。青苹果也不会。"

"她已经说了。用比喻。"

"对。她一直在说。用树、用瓶盖、用信、用建筑结构的章节标题。但陆允川——你听懂了没有？她说的是中文。不是密码。"

南湖水面被午后的风吹出了一层细涟漪。远处有人划着学校的那种老式铁皮船，船桨敲在水面上，发出闷闷的咚咚声。

"我听到她在说。"陆允川说。"但我在等一个不含比喻的版本。"

"什么版本？"

"比如'我喜欢你'或者'我们可以在一起了'或者'我想清楚了'。任何一个不需要解读的、直接的句子。"

叶澄放下薯片。她的表情从不耐烦变成了认真。

"你是在等她自己想清楚。这很对。但你知道她在想清楚什么吗？"

"什么？"

"她在想清楚：她能不能在同时做好两件事的前提下，再做好第三件事。第一件事：双学位的建筑课。第二件事：诗社的经营和301教室那棵树。第三件事——是你。"

"我不需要她'做好'我。"

"你是不需要。但她需要。她对所有重要的事情都有标准——不是别人的标准，是自己的标准。她的标准是：对一个人认真，就不能只认真一半。但她现在的时间——双学位加诗社加自己生活——已经满了。她觉得如果跟你在一起，一定会亏欠你时间。她不愿意亏欠任何人。"

陆允川看着湖面。湖水映着天上的云，一片白云被风吹散了边缘，慢慢往东南方向滑。

"你怎么知道的这些？"他问叶澄。

"她上周约我也约在了这个亭子里。那次是她找我谈。她说她不知道该怎么办。"

"她说了什么？"

"她说——"叶澄学苏晚枝的语气，"'叶澄，如果我现在对陆允川说好，但接下来一整年我都在画图和考试，两个人最多一周见一次面、吃一顿饭——你觉得对他来说够吗？'"

"你怎么回答的？"

"我说：'你问错人了。你应该问他。'她说：'我不敢问。因为如果我问了，他说够，我怕他是在委屈。他说不够，我不知道该怎么办。'"

风吹过湖面，把铁皮船的划水声推到了凉亭这边。陆允川一直看着那片被吹散边缘的白云。

"所以你找我来——是想跟我说她的难处。"

"不是。我想跟你说她已经在想。不是不想——是在想了，只是还没想通。"叶澄站起来把薯片袋揉成团放进垃圾桶。"另外我想跟你说一句我自己的想法：你等得太小心了。你怕推动任何东西。但有时候不需要等她自己把所有问题都想通——有些问题需要两个人一起想。"

"比如？"

"比如'一年里一周见一次面够不够'——这个不是她一个人能回答的。也要你回答。"

叶澄背上包。"我去写实验报告了。你们两个自己商量吧。"

她走了几步又回头。"对了。祁野说你在模仿她的笑。他说你在超市门口发呆的时候笑了一下——眼角往下弯。他说他分不清那是苏晚枝感染你了，还是你自己长出来的。"

"我自己也分不清。"

"分清了告诉我。"叶澄挥了挥手。

陆允川在凉亭里又坐了一会儿。湖面的涟漪慢慢变平了。那个划铁皮船的人把船靠了岸，把船桨拎起来搁在码头边上，船桨上的水滴在木板上，留下了一道深色的水渍。

他想：是时候直接跟她说了。不是等她给出结论。而是告诉她他也在想——在想同样的问题，在得一些还不成熟的结论，在等一个机会把这些结论放到桌面上。

他拿起手机。没有发消息——直接拨了电话。

"喂。"苏晚枝的声音。

"你在哪？"

"建筑系馆。在做模型。"

"做完以后有空吗？我想跟你说一件事。就今天。"

电话里安静了两秒。"关于'那件事'？"

"对。'那件事'。"

"好。七点。301教室见。"

"好。"

挂掉电话后，他在凉亭里站起来，把坐皱的裤子拍了拍。湖面上又起了一阵小风，吹皱了白云的倒影。
