# 第031章 雨天值日之前

所属部卷：第一部：初秋三楼
时间：高一下学期雨季
地点：放学后的教室

## 诗歌插页 P031：原创现代诗

我把早晨放低，
让一阵雨从课桌边慢慢经过。
你的名字没有被我喊出，
只在值日表上被雨溅湿的毛笔字迹旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写雨天值日抵达之前的坡度。雨季在四月下旬突然抵达——不是润物细无声的那种雨，是从早到晚毫不停歇的大雨，砸在窗玻璃上像有人一把一把地撒碎石子。教学楼后面的排水沟满了，水从沟沿溢出来，在水泥地上铺成一片一片的浅塘。走廊里到处靠着湿伞——长柄的、折叠的、从家里带来的格子伞、小卖部买的透明伞——散出塑料布和雨水混在一起的霉味。地砖被雨伞滴下来的水和鞋泥踩成了深灰色，干了以后留下一条一条的泥渍，像褪色的斑马线。

值日表贴在黑板右侧，用的是大红纸，边角用透明胶粘着，贴了一个多月已经翘了角。毛笔字是开学时陈老师写的——"星期一：夏知遥、陆允川、程砚"。陈老师的毛笔字写得很正，横平竖直，每个名字之间的间距几乎相等。这三个名字排在同一组是因为按座位依序分组，每周轮换一次，名字之间没有任何人为调整。陆允川在开学第一周就发现了这个排列——他在值日表前面站了大概十几秒，假装看值日分工（扫地、拖地、擦黑板、倒垃圾、整理讲台），实际上在看名字的排列。他和夏知遥的名字之间隔了三堵看不见的墙——她在前排，他在后排，中间有个程砚。值日表是距离最近的一次：整张表上，他和她之间只隔了一个程砚的名字，物理距离大概两厘米。

"你知道你和夏知遥一个值日组吗？"程砚在课间问他。
"知道。"
"你那天打算什么态度？精神饱满还是装肚子疼请假？"
"为什么要请假？"陆允川捏着笔，在书上划了一道横线。
"因为你和心仪的他排到同一天扫地的概率——你知道这个算术题怎么做吗？"程砚把椅子转过来，一条腿搭在课桌横杠上说，"全班三十八个人，每周五人一组轮。你和她同一组的概率大约是八分之一。八分之一的概率你中了——你是不是该考虑多打扫一下？"

陆允川没有回答。但他后来在草稿纸上真的算了一下——三十八个人分七组，同组概率确实是八分之一左右。他把这个算式写在纸上，又划掉。因为他觉得自己在算一件不需要算的事——不管概率是多少，只要她的名字在值日表上距离他的不到两厘米，他就已经在提前紧张了。

雨季的值日比平时多了擦地——湿伞带进来的水会在过道上积成浅水洼。陆允川在宿舍里跟程砚说他要申请负责拖地。程砚笑出声："你拖地？你平时拖宿舍的地都拖不干净——但你拖教室大概会非常认真。""因为教室地砖比宿舍大。""因为教室里有需要被你拖到的那个方向。"

值日那天是星期三。天气预报说大到暴雨——从早上开始窗外的雨就没停过，梧桐叶被打得翻过来，露出灰白的叶背，像被人从下面掀开的裙子。教室里的日光灯全天开着，因为自然光不够，灯管发出的嗡嗡声和雨声混在一起，分不清哪个是哪个。陆允川把一把从家里带来的旧伞放在书包旁边——伞是深蓝色的，边角已经翻毛了，伞柄上有一处生锈的地方，每次打开都会发出刺耳的金属摩擦声。他心想今天她的伞不知道带没带。如果没带，她可能会在教室多待一会儿等雨小——而值日就是放学后做的事。这两件事如果重叠，他等于多了一个多小时的时间和她在同一间教室里。他不确定这是好事——因为他可能会一句话都说不出来，干扫地扫到同一块地方扫了三遍——还是坏事——因为他可能会紧张到手心出汗，拖把从手里滑出去。但他知道这是他整个雨季里最不想错过的一次值日。他甚至在早上出门前多看了一眼镜子——不是臭美，就是确认一下校服拉链拉好了没有。
