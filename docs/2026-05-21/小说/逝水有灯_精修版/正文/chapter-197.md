# 第197章 回乡的暑假

所属部卷：第四部：失去的美丽
时间：大三暑假
地点：镇上老桥

## 诗歌插页 P197：原创仿唐绝句

桥痕微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

到县城的第三天傍晚，陆允川和程砚去了老桥。

程砚是从姑妈家骑电动车过来的。车子的后轮有点缺气——骑的时候一颠一颠的。他把车停在桥头的小卖部门口，拔了钥匙，跟小卖部老板娘说："两瓶汽水。冰的。"

老板娘从冰柜里翻出两瓶玻璃瓶装的老式汽水——瓶子外壁结了一层薄霜。"四块。"

"涨价了？"

"没涨。你去年买的也是两瓶四块。"

"你居然记得。"

"你们这几个大学生每次回来都买汽水。从来不还瓶子。"

陆允川接过汽水。瓶盖撬开的时候，白色的汽雾从瓶口冒了出来。他仰头喝了一口——甜的，气泡在舌头上扎了一下。

"走。上桥。"

老桥还是老样子。桥身是石头砌的，桥面铺了水泥。栏杆上的白漆剥落了不少，露出底下灰色的底子——像是皮肤脱了一层痂。桥下河水流动得很慢，傍晚的光碎在水面上，被揉成一片一片的金色碎片。桥那头有人在洗拖把——水声一下一下打在石阶上。远处的菜摊开始收伞。

陆允川把汽水瓶子搁在栏杆上。手指摸过那些剥落的白漆——指尖触到粗糙的粉末。他把手指在裤子上蹭了一下，指腹发灰。

"你还记得高三毕业那天吗。"他问。

程砚靠在栏杆另一边。"你是指河边那场'表白'？"

"对。就在这条河。往上走大概五百米。有个小码头。台阶直接伸到水里。"

"然后你约了夏知遥在那里见面。"

"约了。毕业典礼结束以后。她答应了说会来。"

"然后呢。"

"然后我从下午三点等到五点。没来。两点半下了场小雨——台阶很滑。我踩了一下差点摔进河里。后来她发了一条消息——'不好意思今天家里有事来不了'。我回了两个字：没事。然后把手机塞进裤兜。又在台阶上坐了半小时。然后回家了。"

"那晚你哭了吗。"

"没有。晚上吃饭的时候我妈问我毕业典礼怎么样。我说还行。她说你怎么不说话。我说嗓子不舒服。我妈去厨房冲了一杯蜂蜜水。我喝完。然后回房间。把写给她的信撕了。"

程砚把汽水喝了一口。气泡的声音很脆。

"你现在站在这里——想起那天下午——什么感觉？"

"感觉——那个在台阶上等了两小时的男生，不是蠢。是年轻。年轻人觉得'等'是一种表达。等得越久，越能证明自己的真心。但实际上不是。等就是等。不等就是不值得等。把等包装成深情——是年轻人才做的事。"

"那你觉得那两小时算什么？"

"算——一个必要的结束。如果没等那两小时，我可能会后悔好几年——觉得'要是当时我等了呢'。等了。知道结果了。后悔反而少了。"

"所以你不后悔等了？"

"不后悔。但我也不觉得那两小时很浪漫。就是很冷。很滑。蚊子很多。"

程砚笑了一声。"跟上次你描述的一样——可乐没拧开。这次是蚊子很多。你回忆青春的时候能不能稍微加点滤镜。"

"不加了。加了滤镜就不是真的了。真的就是：台阶很滑、蚊子很多、她没来、我渴得要死。然后回家喝了杯蜂蜜水。我妈冲的。"

桥下有人喊了一声——是洗拖把的那个人在跟岸上的熟人打招呼。"收摊啦——""晚点再收——"。声音在水面上弹了一下，散成回音。

陆允川沿着桥栏杆往前走。手指划过那些剥落的白漆。每一步，指尖都沾上一层灰色的粉末。他从桥头走到桥中央。停下来。然后又从桥中央走到桥尾。

"你干嘛。"

"走一遍。"

"走一遍什么。"

"高三毕业那天——如果她来了——我们应该会从桥头走过来的。不是应该——是我当时设想过。我设想了整整一条路线：先在小卖部买两瓶汽水。然后走上桥。走到桥中间停下来看河水。然后走到桥尾的小吃摊吃一碗凉粉。然后送她到公交站。"

"这路线你演练过多少遍？"

"在脑子里——至少一百遍吧。但从没走完过。因为第一步——买两瓶汽水——就停住了。那天我只买了自己的。她的那瓶我放在小卖部冰柜里——跟老板娘说'等会儿有人来拿'。然后自己去河边等了。"

"那瓶汽水后来谁拿了？"

"不知道。可能老板娘自己喝了。可能浪费了。也可能有个路人买了去。"

程砚没有笑。他看着陆允川从桥尾走回来的方向——桥的对面是一片老居民区。灰色的楼、晾在阳台上的被单、二楼窗户里透出来的暖黄色灯光。

"你跟我说了这么多——有一件事你一直没说她为什么没来。"

"她说家里有事。但我知道不是。"

"你怎么知道。"

"因为后来——大二那年——我在QQ空间里看到她当天下午发了一条动态。时间是三点半。就是她说'家里有事'的同时。动态的内容是——跟朋友在KTV唱歌。配了一张自拍。"

陆允川把最后一口汽水喝完。瓶子空了。他把瓶子放在栏杆上，看着它在晚风里微微晃动。

"所以你等了两个小时，她在唱歌。"

"对。但我不怪她。她那时候十七岁。一个不熟的男生约她去河边——换成我我也不一定去。KTV热闹。河边冷。选热闹不是很正常吗。"

"你什么时候想通的。"

"今年。大三。之前一直在怪她——觉得她不够真诚。今年突然明白了：她没有义务对我真诚。我也没给她真诚的理由。我对她说的话加起来不超过二十句。然后毕业那天突然来一句'我有话想跟你说——在河边码头——下午三点'。换成谁都会觉得不对劲。"

程砚把手里的汽水瓶子也喝完了。他把两个空瓶子并排放在栏杆上。玻璃瓶在夕阳下闪着琥珀色的光。

"那你今天走完了这条路线——从桥头到桥尾。最后一步是什么。"

"最后一步——"陆允川指着桥尾那家小吃摊。"去吃一碗凉粉。"

"那走。"

两个人走到桥尾。小吃摊是个矮矮的棚子——蓝色遮雨布、塑料桌椅、一个煤炉。老板是个六十多岁的老头，正在往凉粉上浇醋。

"两碗凉粉。"

"加辣不。"
"一个加。一个不加。不加的是我的。"

凉粉端上来了。透明、弹滑、醋的酸味混合着蒜香。陆允川用筷子夹起一块——凉粉在筷子尖上颤了一下。

"你知道我刚才走完那段路的时候在想什么吗。"

"什么。"

"在想——当年设想的路线终于走完了。但不是跟她。是跟你。"

"遗憾吗。"

"不遗憾。跟你走不用想话题。跟她走——我大概每走三步就要想一个话题。然后怕冷场。然后更紧张。最后连话都不会说了。"

"那我等于帮你走完了一条你演练了一百遍的路线。"

"对。谢了。"

"别谢。凉粉你请。"

陆允川往碗里又浇了一勺醋。河面上吹来一阵风——带着水草的腥味和远处菜市场收摊后的空荡感。头顶的遮雨布在风里打了几个颤。桥头的那两瓶空汽水还在栏杆上——远远看去像两个小小的琥珀色光斑。

他吃完最后一口凉粉，把一次性筷子折成两截放进碗里。"走吧。回家。"

"这就走了？不多站一会儿？"

"不站了。走完就行了。站太久蚊子又来了。"

"那以前站那么久——蚊子没把你抬走？"

"抬走过。真的。大二暑假那次腿上二十几个包。我妈以为我得了皮肤病。涂了一整瓶花露水。"

程砚站起来，拍了拍裤子上的塑料屑。"那你这次回家怎么说。"

"就说——去桥上吃凉粉了。吃到一半醋不够又加了一勺。"

"不说别的。"

"不说。描述食物永远比描述情绪安全。"

他们沿着桥走回去。经过小卖部的时候，老板娘正在收冰柜外面的遮阳伞。"玻璃瓶还你不？"程砚指了指桥上那两个空瓶子。

"不要了不要了。拿走吧。"

"每次都不收——你冰箱里的瓶子会越来越少。"

"那我就再进货。反正你们大学生隔几个月就要回来一趟。"

程砚把两个空瓶子捡起来递给老板娘。"那这次给你。下次再买新的一轮。"

老板娘接过瓶子，在围裙上擦了一下瓶口的灰。"行。下次你们来——还是四块。不准赊账。"

"好。"

程砚骑上电动车——后轮还是有点瘪。"送你一段？"

"不用。走过去十分钟。"

"行。那开学见。"

"开学见。"

电动车突突突地开走了。陆允川一个人往家的方向走。经过老桥正下方的时候，他抬头看了一眼——桥拱的阴影投在河面上。河水在阴影里看起来是深绿色的——跟高三毕业那天下午的颜色一样。那天他从码头走上来的时候也经过了这个位置。当时他在哭。不是流泪那种哭。是眼睛发酸但是忍住了——只滚了一颗出来。被风吹干了。

现在他站在同一个位置。眼睛是干的。

他把手插在裤兜里，沿着河岸往前走。夕阳正在沉入远处的楼房后面。老桥的轮廓被衬成了一个黑色的剪影。桥上有两个人——看不清脸——也在走。从桥头走到桥尾。

不是他和夏知遥。

只是两个人。
