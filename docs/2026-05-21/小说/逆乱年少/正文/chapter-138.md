# 第138章 回程

时间：2012年8月4日，上午至下午

地点：绿皮火车，从太行山回江城

卷次：第六卷：雪落无声

回去的火车是同一班——太行山方向开往江城，下午两点二十发车。

村长李德厚又开着那辆三轮车把他们送到火车站。他往每个人手里塞了一袋核桃——自己家晒的，壳上还带着没擦干净的青皮渣。铁蛋没来送，但托村长带了一封信给苏安晴——信封是用作业本纸糊的，正面写着"苏老师收"，背面画了一面红旗。

苏安晴把信放进包里，没有立刻拆。

火车上大家都没有来的时候那么兴奋了。赵一琳靠着椅背打瞌睡。孟小荷在翻照片——她用手机拍了两百多张，内存都快满了。方远和陈维中在低声讨论李家沟小学缺什么设施，说回去可以写个报告。

孙亦舟忽然说了一句："我回去第一件事是洗个热水澡。"

"我想吃火锅。"沈佳慧说。

"我想我妈做的红烧肉。"赵一琳闭着眼睛说。

苏安晴没有说话。她坐在靠窗的位置，看着窗外——山在往后退，一座接一座。从太行山退出去，退了很久。山下是层层叠叠的梯田，有的地里种着玉米，有的刚收了麦子，只剩一排排整齐的麦茬。

林玉起坐在她对面——回去的票座位和来时不一样了，这次他们面对面。

"在想什么？"他问。

苏安晴把目光从窗外收回来，看着他。

"在想这一个月比我想的要长。"

"长是好还是不好？"

"不是好不好的问题。"她说。"就是——这一个月比我大学两年都长。"

她说这句话的时候语气很平常。但林玉起听出了分量。

"大学两年过得很快。上课、自习、考试、放假。一年一年地重复，回头一看，说不清到底经历了什么。"苏安晴低头看着自己的手指——指甲剪得很短，边上还有批改作业时蹭的红笔印，洗了两次没洗干净。

"但这一个月不一样。每一天都不一样。第一天上课、铁蛋举手、做家访、陈小娟学会查字典、刘小云背完了二十首古诗。每一天都有具体的东西。时间就变慢了。"

林玉起没说话。他在想她说的话。

苏安晴又转头去看窗外。窗外的山终于退完了，变成了平原——大片大片的玉米地，偶尔有村庄和工厂。天色从蓝变成灰，天空变宽了，视野变大了。

快到江城的时候苏安晴忽然说了一句话。

"林玉起。我希望以后的日子都是这样的——过完了还记得过了什么。"

"会是的。"他说。

火车在江城站缓缓停下。站台上人流如织，跟李家沟是两个世界。

八个人背着大包小包下了火车。陈维中在出站口点了一次名，笑着说："任务完成。大家辛苦了。开学见。"

苏安晴背上那个鼓鼓囊囊的大包——比来时轻了很多，书都留给了孩子们，但包里多了一堆核桃、一张画、一颗糖纸、一封信。

她走了一步，回头看了一眼火车站的方向。

不是看站台。

是看太行山的方向。

>
山退平原阔，路长人未还。
一月短如瞬，却比两年长。
>
