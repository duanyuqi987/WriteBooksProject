# 第127章 睡不着

时间：2018年3月

地点：楚州，顾临风公寓

卷次：卷七：流年未断

相亲前一晚。

他躺在床上。被子拉到胸口。枕头是荞麦皮的，枕了两年，中间凹下去一块。他的后脑勺刚好卡在那个凹槽里。

窗帘拉上了。但窗帘布的遮光性不好，对面的路灯光透进来，在天花板上铺了一层暗橘色。

他睁着眼睛。

天花板上的裂缝还是那条。从灯座往墙角延伸。在暗橘色的光里像一条很细的河。

看过几百次了。大一开学报到那天看了一次——陌生的城市、陌生的宿舍、陌生的天花板。考研成绩出来那天又看了一次——分数线过了，他的手在发抖。代课第一天上班前又看了一次——穿好衬衫打好领带去讲台之前，抬头看了一眼。

现在他又在看。

像一个高考后等成绩的高中生。六月九号晚——高考结束，答案对完了，分数大概估出来了，但还没出。躺在床上。不知道明天会来什么，但明天一定会来。那种感觉。胸膛里有什么东西在轻轻地跳——不快，但停不下来。

像一个大一新生报到前一晚。九月一号的晚上——青阳老家的床上。第二天要去楚州。一个从来没去过的地方。一整个新生活。被子还是蓝格子的。枕套上有他妈补的那块补丁。那时候也睡不着。

像一个——暗恋者。

他翻了个身。面朝墙。墙上的涂料还在起皮。他用手指摸了摸那个凸起的边缘——粗糙的，有一点冰凉。

高一那年他也是这样。每天晚上躺在床上，脑子里是她座位后面的阳光，是她手上的铅笔，是她走过走廊时马尾的晃动。那些画面在脑子里循环播放，从九月播到来年六月，再从来年九月播到来年六月。

那时候他是坐在后排、不敢看她的男生。现在是站在讲台上的顾老师。但他躺在这个房间里时，又变回了十六岁。

六年。从青阳到楚州。从一中考到师大。从学生变成老师。从少年变成男人。他的身份变了很多次。但在这个夜晚——黑暗里，他只是一个即将见到苏晚晴的人。

明天她会坐在窗边。他会推开门。他们会对视。已经想了八年的画面。

他以为第一次对视会是在高中教室里。他在走廊上悄悄等她经过。等了三次——每次她走过他面前的时候他都假装在看窗外。第四次她抬头看了他一眼——很快，像无意中扫到的。他假装没有发现。

那时候他就想——如果有一天，不是假装没发现。是真的看着她。她也看着他。目光不躲不闪。

明天就是那一天。

他翻了个身。床垫又嘎吱响了一声。

凌晨一点。

他起来上厕所。没开灯。脚趾踢到了床脚。很疼。他弯腰揉了一会儿。

回到床上。闭上眼睛。

还是睡不着。

凌晨两点。他把手机按亮。屏幕的光刺得眼睛疼。他眯着眼看了看时间。离约好的三点还有十三个小时。

他打开相册。点进那个叫"一"的文件夹。她的照片亮起来。

校门口。阳光。银杏叶黄了一半。马尾。浅蓝色衬衫。平底鞋。笑容比以前从容。

他看着照片，好像在问——你还记得我吗。

凌晨三点。他把手机放在枕头底下。

闭上眼睛。

外面的夜很静。偶尔有一辆车开过——轮胎压过路面的声音，由远及近，又由近及远。

凌晨四点。他大概睡着了一会儿。但梦里也是茶餐厅——推开门，窗边坐着一个人，脸看不清。他往前走，但桌子越来越远。他想开口说话，喉咙里发不出声音。

醒了。窗帘的缝隙里天色是灰的——四月初的凌晨，天亮得很慢。

他翻了个身。被子裹得很紧。他松了松。又裹紧了。

五点。六点。天亮了。

他起来。走到窗前。拉开窗帘。樱花树还在。树上的叶子比昨天又多了几片。花瓣只剩最顶上那几朵了，在早晨的光里微微发白。

楼下有清洁工在扫地。扫帚划过地面的声音——沙。沙。沙。

他在窗边站了一会儿。

然后去洗脸。水龙头的水很凉。他用凉水拍脸，拍了三四下。抬头看镜子——眼睛有点红，但不是太明显。

他用毛巾擦干脸。

走到床边。床头放着那件浅灰衬衫——叠得整整齐齐。

他看了衬衫一眼。

穿上了。
