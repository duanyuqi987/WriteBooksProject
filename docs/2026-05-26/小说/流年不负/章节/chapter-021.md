# 第021章 新座位

时间：2011年3月

地点：理科三班教室

卷次：卷二：窗边人影

三月。高一下学期开学。

理三班的教室在三楼东边——走廊最尽头——窗户比原来三班的多——前后两排窗——窗外是梧桐树——三月梧桐还是秃的——枝条干瘦——交叉着。窗框是新漆的绿——味道还有点刺鼻——坐第一排的人说闻着头晕——开了窗——冷风灌进来——又关了。

顾临风到教室的时候门口的座位表换了。贴在门边的白墙上——陈老师用一张A3纸打印的——表格——按姓名、按名次排好。新座位的规则是按上学期期末成绩重新排位——前排给薄弱学科的人——后排给理综好的。他从最下面那行往上找自己的名字——倒数第三排——靠窗。和上学期一样——倒数第三——只是从过道换到了窗边。手指在纸上按着"顾临风"三个字——确认了一下——然后走进去。

他把书包放好。拉链拉开——拉链还是卡——使了点劲——嘣——开了。桌面上有粉笔灰的残印——是上午装修磨水泥留下来的——他用手掌抹了两把——灰飞起来——细粉在从窗户进来的日光里飘——很慢。桌腿下面压了一张课表——是这学期的——周一第一节物理——陈老师。他低头看了看课表——然后抬头——扫了一圈教室。

教室里人还不多。日光灯亮着——新装的白灯管——比原来那个一直闪的灯管新得多——光更白——照得水泥地反着光。吊扇没开——三月天还凉——扇叶上还有装修的灰——没人擦。暖气片已经停了——但教室角落还有一点余温——靠近暖气片的地方坐着两个女生——在低声聊天——"你听说了吗——李老师不教咱班物理了——换了陈——""知道——点名表上都写了。"

赵一鸣还没到——他的新座位在第三排——离顾临风远了。赵一鸣期末考了第十五名——往前挪了——从倒数第三跳到正数第三——进步了十二名——他说"回家过年多吃了几个饺子——脑子突然好用了——"然后笑——笑完又说"其实是看了你记的物理笔记——那个受力分析图我终于看懂了——"

门被推开。吱——门轴是新的——不涩——声音短。

苏晚晴走了进来。灰色外套——比以前那件墨绿色棉服薄了一些——三月了。深蓝色牛仔裤——马尾扎得很紧——额头干净——她抬手的时候能看到袖口——灰色的袖口卷了一道。书包还在右肩上——拉链上的那个小鱼挂件——隔了三排也看到了——白色底——蓝色线——晃了一下——她的脚步停了——在门口环顾了一下教室——找自己的座位——然后往前排走。

第二排靠窗。

她坐下了。椅子和上次的教室一样——椅子腿刮了一下地面——"吱"——很轻——她扶了一下桌面。然后她拉开书包——翻出课本——语文书——封面是白的——封面左角上有她写的名字——他用指甲轻轻划过的那两个字——现在看不见了——隔着教室和大半个班。

顾临风忽然意识到一件事——从这个位置——他能看到的不再是她的侧脸。不再是四排加两个过道之外的那半边脸——马尾、耳朵、小指翘起。现在他坐在倒数第三排靠窗——她在第二排靠窗——方向对准的是她的后脑勺。马尾——脖子——耳后那一点向下斜的轮廓。

这个距离比上学期近了。但看到的内容少了。

她的后背挺直——写字的时候肩膀微动——头会轻轻左右晃一下——根据笔画往外推。他盯着那个方向看了几秒——然后低下头——翻开数学书——第一页——第二页——第三页——三页翻完——什么都没看进去——又翻回第一页。数学书上印着"第一章——三角函数"——sin、cos——那个圆——角度——r。他把笔拿起来——在草稿纸上画了一个坐标系——然后停在x轴——笔不动了。

窗外有一棵银杏树——教室外。上学期三班教室外面是梧桐——这学期换了——变成银杏。三月的银杏还没长叶子——枝条更细——更密——光秃秃地伸在窗框里。她现在坐在窗与银杏之间——背影——马尾——后颈——在窗户透进的光和窗外干枯的枝条之间——像一张静止的画。

阳光从枝条间漏进来——落在她桌上。她低头——光打在她的右肩——灰色外套上——肩线那一块被照得比别处亮。她往旁边挪了挪课本——光移到了桌面的左上角——她没动——重新开始写。

顾临风盯着那个方向看了几秒。然后低下头——翻了翻数学书——在草稿纸上画了几个圈——乱画——然后停笔。

赵一鸣从后门进来——脚步急促——书包还没放稳就拍了他肩膀一下。"换座位了——你在看什么呢——"

"没看什么。"

赵一鸣看了看他——又顺着他的视线方向看了一眼前面。停顿了一秒——嘴角动了动。"哦。"

他没再说别的。把书包放在第三排自己的新座位上——从座位里转回身——举着手里的一根橘子味棒棒糖——"给你——新学期第一个——"

顾临风接过来——放在桌角——没剥。

上午四节课。语文、数学、英语、物理。语文课——陈维中老师——头发花白——语速慢——讲《荷塘月色》的前半段——他在黑板上抄了几句——"这几天心里颇不宁静——"字写得很端正——像他这个人——不快——不慢——刚刚好。底下有人在悄悄翻后面的课文——没出声。老师没叫人朗读——这堂课不讲朗读。

数学课她上黑板做了一道题。三角函数的恒等式——sin²θ+cos²θ=1——证明——她写了两行——然后停下来——手里拿着粉笔——擦了一行——重新写——做完之后从黑板下来——马尾在从黑板往回走的时候摆了两下——很轻——从颈后落到右肩。坐下后——轻轻收脚——椅子微微往后拖了一下。

英语课她被点起来背单词。背了五个——contemporary、melancholy、symphony、breeze、horizon——全对。发音——melancholy那个"lan"音节她把元音发得有点长——但整体算对。老师点了点头——"Good——"她坐下来——把英语书翻到下一页。

物理课她皱眉了。陈老师在黑板上讲圆周运动——向心加速度——a=v²/r——她在笔记本上写——写到一半——眉间皱起来——右手拿笔——左手的尺子在那道题上来回比——画了一条向心的箭头——擦了——重新画——又擦了。他把这个也记在了脑子里——不是故意——就是看见了——然后留在了那里。

下午的阳光换了个角度——从银杏枝条间移到了另一个窗格。那个位置的光照在她右肩上——右肩比左肩亮了大概一个色阶。她把物理笔记本往前翻了一页——在页角画了一条线——然后停了笔——手指放在那根线上——不动——看着窗外的银杏枝。

下课铃响了——她站起来——绕到讲台那边——保温杯放在桌边——盖子拧开——接了水——回来的时候从后排过道经过——离他不到一米。书包在椅背上，小鱼挂件摇了摇——白色的瓷——蓝色线——从暗处晃到明处。她走过去——脚步不快——鞋是白色帆布鞋——鞋底在过道水泥地面上没发出什么声音。那小鱼在她经过后——还在书包上晃——从左晃到右——小弧渐停。

顾临风把刚才在本子上画的线涂掉了——铅笔横着在纸上推了两下——灰色的石墨涂成一道长方形的暗块。

晚自习前——他去走廊尽头站了站。饮水机在旁边——灯没开——开水器亮着红灯——桶里还剩小半桶水——桶壁上有气泡——一串——往上冒。走廊很长——从这一头能看到尽头那个教室——理科六班——那边的灯亮了——有几个人影——远远的。从走廊栏杆往右看——能看到理科三班的窗户——从这个角度——教室里的日光灯最亮——课桌椅剪影——数过去——一——二——三——四——看到了——第二排靠窗——马尾——她还在教室里——头低着——应该在做题。

天还没完全黑。窗外的银杏枝——春天天黑得比冬天晚——天空还是灰蓝色的——树枝在灰蓝底上像一张极细的网——细枝交叉——张着。从走廊尽头能看到银杏枝——也能看到她——窗框中一个小小侧影——固定不动。

赵一鸣走过来——手里拎着两瓶水——走廊的鞋底啪啪响——"你最近老往走廊跑——"

"透透气。"

"前几天还说肚子不舒服——"

"好了。"

赵一鸣没追问——递给他另一瓶水——瓶盖上带着水珠——拧开了——递过去。顾临风接过来——喝了一口——水是温的——开水机灌的——已经放凉了。

晚自习——她一直在做物理题——向心力——角速度——线速度——周期。做到九点还没停——教室里人走得差不多了——只剩下前排两个——后排一个——还有他。他故意收拾书包收拾得很慢——把笔一支一支放回铅笔盒——按下去——每支笔放进槽里——圆珠笔放在最左边——自动铅笔放在中间——橡皮——三角尺——放好。把课本一本一本摞好——物理——数学——英语——语文——摞整齐——书角对齐。她终于站起来——手里拿着空杯子——关上台灯——椅子推近桌底——转身——马尾一晃——从前门出去。

他比她晚十秒出门。关灯——倒数第三排挨着的那个开关。走廊里的灯有点暗——只有安全出口的绿色标志亮着——那个男人的小人——往哪个方向跑。她的影子在前面拖得很长——上课换下来的教室里暗处——走廊灯又不够亮——穿着平底鞋——影子流流地往前。到楼梯口——她往左拐——往女生宿舍那边去了。他往右拐——往男生宿舍——走了几步——在拐角回头。走廊空——尽头黑。她走进黑暗里——脚步声远了。窗外的银杏树还在——枝条——三月光秃——在二楼走廊的路灯边——剪影。

教室的灯灭了。亮了一天——在黑暗的时候才最安静。他转身——往下走——手心攥着刚才那张涂掉的铅笔线迹。
