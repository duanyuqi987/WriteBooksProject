# 第150章 食堂争执多年后

所属部卷：第三部：寒风里的花
时间：六月末
地点：校外小饭馆

## 诗歌插页 P150：原创仿唐绝句

残油微凉近晚风，
少年心事隔帘中。
不知旧梦归何处，
一盏疏灯照水东。

多年后陆允川偶尔还会去那种红色招牌的小饭馆——校门口、巷子里、开了很久还没倒的那种。不是为了吃饭——是为了闻那个味道：茉莉花茶、酸菜鱼的油辣、干煸四季豆的焦香混在一起。这些味道会把他拉回那张时间表放在桌上的时刻。她当时从布袋里拿出来展开，用手指沿着周三晚上的红色圆圈慢慢画过去，画得很慢。她接下来说要试，然后真的试了。

女儿上小学那年，妻子单位发了一张附近新开酸菜鱼馆的优惠券。他带着全家去吃。女儿说太辣了，喝了好几杯水。妻子吃了两口说还不错但不如你做的好。他尝了一下说鱼的鲜味被辣盖住了——然后想起苏晚枝当年在老马饭馆挑花椒的动作。她用筷子把花椒一颗一颗挑到盘子边上，排成一排。那个动作和当年她在食堂挑葱花一模一样——只不过对象从葱花变成了花椒，汤底从粥变成了鱼。

她对食物的挑剔从来不是口味问题。是秩序。

那天晚上吃完饭回来，女儿在客厅写作业。他坐在书桌前，翻开了诗抄本。不是看——是在第60页新空页上写了几行字。新页上没有诗歌。只有一段记录：

"关于争执：
那年在老马饭馆我说她在跟工具死扛。她承认了。说'因为是你给我的表'。那一刻我意识到——我给她排的时间表在她那里的权重比我想象的重。她不是把它当成工具——她把它当成我。所以当工具不适合的时候，她没有办法调整——因为调整工具等于调整我。我后来理解了这个逻辑。但当时花了三周。她在这三周里自己给自己做了减法——交钥匙、让树叶给别人挂、学会开后排灯。她把减法完成了以后才告诉我。这是她惯常的模式：先做到，再告诉别人。

而我只做了一件事——没有替她做减法。我给了建议，然后等她。等的过程中没有催。因为我知道催等于再给她加一件事——'回应催'又会变成额外的待办事项。

后来我想：也许当初争执的意义不在于说了什么。在于争执之后她知道了——就算我们有不同的节奏，关系不会断。节奏可以调。树不会因为少浇一周水就死。"

写完以后他把这一页拍了照片发给了苏晚枝。他很久没有给她发消息了——最近的记录是去年过年互相祝了新年快乐。

她当天晚上回了消息："谢谢你把这段记下来。我在工作室的书架上给那段争执专门留了一个位置——放的是一张时间表复印件。已经黄了。上面铅笔字还看得清。每次有人问我做项目为什么总比工期快，我给她们看这张时间表。说：这是当年一个人帮我在三件事之间排的空间。他排的不是时间——是呼吸的间隔。他让我把'证明自己'从那三件事里拿走了。"

陆允川看着这条消息。她的回复方式和大学时一样——具体、有条理、给感谢留白。只是这次的"谢谢"后面不用再藏着任何未说的话。过去的都已经说过了。

女儿写完作业跑过来趴在他的椅背上。"爸爸，你看什么？"

"看老朋友的短信。"

"苏阿姨？"

"对。"

"她还在南京？"

"在。"

"你什么时候再去看她？"

"不一定去。有时候不需要见面。知道就够了。"

"知道什么？"

"知道当年那些争执没有白费。知道那棵树还在别人脑海里活着。知道她现在帮别人排时间表的时候，用的也是同一套逻辑。"

"什么逻辑？"

"呼吸的间隔。"

女儿不懂"呼吸的间隔"——她觉得是一个大人的怪说法。她转身跑回客厅继续看动画片了。

陆允川关掉手机，把诗抄本合上。第60页的新字墨迹还没完全干——他用的是一管买了很久但一直没用的钢笔。墨水是深蓝色，在台灯下有点映光。他把抄本放回书架中层。旁边隔两本词典的距离，妻子那本《说文解字》书页折了一个角——折在"允"字那一页。

窗外的城市灯光在窗帘上打出一片暖黄的模糊光斑。楼下有小孩在滑滑板，滑轮碾过地面，发出一串密集的震动，又消失在转弯处。

他想起很多年前的一个画面——不是任何重要时刻。只是普通的一个晚上。301教室。前排灯亮着。苏晚枝站在树墙前面，手举着一片新叶子在比划挂在哪根枝上。他坐在讲台下面看着。两个人都没有说话。瓶盖门帘被开门的气流推动，轻轻响了一声。老郑在走廊里推着拖把车，从门口经过没说话。

那一个画面——不是争执，不是叶澄的劝告，不是任何转折性的事件。只是一个普通的晚上。但多年后他还记得。因为所有的理解都发生在那个时候——不在对话中，不在争执里，不在方案里。只在安静里。

他关了书房的灯，走到客厅。女儿在沙发上睡着了，动画片还在播。他把她抱起来送回房间，在她额头上亲了一下。她身上有儿童沐浴露的味道——柠檬味的。

"柠檬——喝柠檬的人喜欢光。"他默念了一句。

然后关了走廊的最后一盏灯。

（第三部完）

## 第三部结语

第三部《寒风里的花》覆盖了陆允川大学时期从大一到暑假结束的全部重要节点。这段时期的核心是关于"学会表达"——从诗社传单那天的观察开始，到"不够但试"的表达，到争执中学会说"你在死扛"，到多年后为争执写注。

这一部里有三种关系：
1. 和过去的关系——以夏知遥的邮件为结束标志，从"不敢说"变成了"不需要说"
2. 和现在的关系——以苏晚枝为中心，从"观察和验证"到"在并排长成的两棵树"到"争执后能调整"
3. 和空间的关系——以301教室为核心，从"秘密基地"到"诗社的物理地基"到"交钥匙给别人"

所有的主要人物都在这一部里完成了各自的成长弧线：
- 陆允川学会了直接表达感受，不再过度包装
- 苏晚枝学会了把控制权转成信任，不再一个人扛所有责任
- 祁野从旁观者变成了深入参与者，学会了用精准语言代替随意的嘲弄
- 叶澄持续做观察者和推动者，在关键时刻给予推动
- 沈让退社后仍保持联系，以另一种方式为社团提供支持
- 小孟接过了钥匙，代表社团的传承

301教室的树墙上46片叶子代表46次认真。这些认真不一定会永远留在墙上——但会留在记得这棵树的人的脑海里。
