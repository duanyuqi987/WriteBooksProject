# 第108章 封校的周末之后

所属部卷：第二部：六月以后
时间：大一秋天
地点：大学南门和围墙边

## 诗歌插页 P108：原创仿宋小令

墙影压秋声。
纸重人轻，同路到灯明。
一句“乱也可以改”，
被他藏到夜三更。

晚上回宿舍，陆允川把样张上的错字整理成表格发到诗社群。苏晚枝回了一个“收到，谢谢”，后面没有多余表情。

他盯着那四个字看了一会儿，又点开私聊，想补一句“今天辛苦了”。打完觉得太熟，删掉。又想发“封面其实还不错”，也删掉。最后只把手机放到一边。

祁野坐在床下吃泡面，瞥见他反复拿起手机，“你又开始了？”

“什么？”

“把别人一句正常回复当成作文题。”

陆允川把手机扣在桌上，“没有。”

“没有就去洗澡。十一点以后没热水。”

这句话比劝告有用。陆允川拿着脸盆下楼，澡堂门口排队的人很多，大家抱着拖鞋和洗发水，烦躁地往前挪。他站在人群里，忽然觉得自己那点心事也得排队，不能总插到生活前面。

洗完澡回来，苏晚枝在群里发了新版修改计划，语气清楚，分工明确。陆允川被安排校对第二版第三页到第六页。

他回：“好。”

这一次他没有再加别的话。共同的小事可以让人靠近，但靠近不等于别人把心交出来。他当晚还不完全懂，只是第一次按住了想把每个细节都写成暗号的冲动。
