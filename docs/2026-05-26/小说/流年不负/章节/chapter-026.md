# 第026章 她的朋友

时间：2011年5月下旬

地点：教室、走廊

卷次：卷二：窗边人影

苏晚晴朋友不多。

顾临风观察了将近一年——从去年九月到现在——快两个学期。她在班上走得近的只有一个——许若琳。她的同桌——圆脸、短发、戴银色椭圆眼镜——话多——下课会说"我跟你说——那个——"然后自己先笑——嘴里的声音在笑之前往下降——没说出来。笑起来——眼睛下面的苹果肌往上推——眯成两道月牙——不是那种刻意的笑——是真的——看见什么了——在想——然后就笑了。两个人的性格——看起来完全不一样——苏晚晴是那种——沉默——一个人坐在窗边——笔在纸上写——写完——划掉——许若琳是那种——"哎——今天那个老师穿的——"语速快——讲到一半——自己打断自己——"不对——我刚才想说啥来着——"然后忘了——笑——苏晚晴在旁边——嘴角轻轻弯一下。

但她们在一起的时候，苏晚晴会笑多一些。

五月的下午。课间——下午三点——太阳从走廊窗户照进来——光斑落在瓷砖地上——长方形的——光里有时有灰尘——缓缓飘——逆着看——一粒一粒。苏晚晴和许若琳站在走廊里晒太阳。苏晚晴靠着墙——后背贴着——墙上的瓷砖——凉——她大概没在意。右手垂在身侧——手指贴着大腿外侧——微微一握——松——左手拿着英语书——卷着的——侧面被手汗浸得有点发软。许若琳站在她对面——离一臂远——手里不知道什么时候拿起了自己的笔——圆珠笔——在用笔的尾端在空中比画——"他是——他是——那个——对——就是那个——"。她语速快——嘴动得跟不上脑子里场景——手比划着——圆圈——往下——"然后他就——"苏晚晴听——嘴角弯了一下。然后弯了第二次——然后笑出声了——很短——像水底往上冒的一个气泡——从喉咙出来——出声——嘴微微张了一点——气从齿间过——"呵——"——然后收——手在嘴唇前掩了一下——指节内侧遮着嘴——眼睛——还在许若琳脸上——在等她讲后续。

是真的笑。

顾临风坐在自己的座位上——从座位透过教室的窗户能看到走廊。窗玻璃上有一层灰——薄——可能是粉笔灰混上灰尘——透过去看——人的轮廓——肩膀——头发——浅一点的衣服——能看到——但细部——手——表情——得靠猜和——熟悉——靠着天天看——他知道她如果微微侧头——是在说什么。他低头翻书——眼光——往上一抬——继续——又抬了一下——耳尖——感觉在动——不是——然后收回来——看着课本——看着那道数学题。许若琳伸手帮苏晚晴把领口的什么东西摘掉了——手指——从她领口拿下一小根——大概是头发——也许是自己衣上的毛线头——不在意——随手往旁边一甩——像甩掉一个没有重量的小东西。苏晚晴说了声"谢谢"——唇语——从教室看不到——他读的是唇——从玻璃外面——隔着灰——大概四米的距离——两个字的嘴唇形——"谢"——嘴唇扁——小——"谢谢"——"谢"字尾音咬合时——牙齿轻轻碰在下唇——松了。然后两人一起走进来——苏晚晴的脸上——还带一丝笑意——很淡——像水面上的最后一圈波纹——散着散着——就收了——回到平常的样子。

顾临风把书合上。

他发现一件事——苏晚晴在许若琳面前是不一样的。不是大的变化——是些很小——说话的时候少了一些停顿——平时——说话前有个提前的——微微吸半口气——那个从她犹豫的时候就有的——在许若琳面前——没有。笑的时候嘴角多弯了一点——那种——自己可能都不知道——笑完会看许若琳的眼睛——不是看一两下——是她听完整段——然后抬眼看她——判断她到底说完没。而这些事——她跟别人在一起——都不做——跟老师——跟班上——体育课——活动——合唱——不做的。顾临风跟赵一鸣在一起时也会不一样。他说的话会多——比跟别人多——不只是"嗯""好""还行"——在赵一鸣面前他可以沉默——也可以开口——"你刚才说的那个——"然后想——赵一鸣等着——不催。被别人看见的那一面——那是信任。

中午食堂。顾临风远远看到她们坐在一起吃饭——靠墙角落——和之前的角落一样——只是现在——她不是一个人——旁边多了一个人。苏晚晴吃土豆丝——筷子在盘边把土豆丝拨到米饭一边——不夹——然后挑一根——放进嘴里——嚼——许若琳在吃肉末茄子——勺子——不锈钢——碰撞盘子的声音——勺子碰到了盘子边——"叮"——然后她从苏晚晴桌子上拿了一叠面纸——抽一张——分了一半给她。许若琳说了句什么——苏晚晴摇头——继续吃——许若琳又说——苏晚晴又摇头——然后许若琳用勺子——把自己碗里的肉末分了半勺到苏晚晴碗里——不问她——直接——"啪"。苏晚晴低头吃了——没推——放嘴里含着——嚼——然后抬头看了许若琳一眼——眼里不是"不好意思"——是"不用这么多——"但没说出来。顾临风低头吃自己的饭——赵一鸣在对面——讲篮球赛——讲得眉飞色舞——"那个六班中锋——他那个转身——你看——那天——"嘴里的饭没咽——嚼着说——声音含含糊糊——手比划着——向前推——汤从勺子上滴下来——一滴——落在餐盘边——没发觉——继续——"我觉得我们下次——再遇到——压——压他左边——"赵一鸣也是这种朋友——不是因为他话多——是因为他不需要顾临风回话。

后来顾临风在日记本上写了一段。

"她有一个朋友叫许若琳。她们在一起的时候她会笑得多。笑得很真。有一个这样的朋友——挺好的。"

写完——想了想——在"朋友"两个字旁边画了一个加号——加号旁边写了"赵一鸣"。然后合上本子。

六月的某一天。走廊。顾临风路过——端着杯子——去接水。听到她们说话——"你数学卷子第十题错了没有——"许若琳——声音方向——在栏杆那边。"错了——""我教你怎么做——看她——那——这个公式——分母——"然后许若琳在找纸——没带——用自己的手比划——拍在苏晚晴右手边的墙上——手指——像笔——画了一道——"这是切线方程——"苏晚晴说"好。"声音里有一点软——对他信任的人才有的软——不是为了满足需求——是那种——"我知道你懂——"自然的。顾临风走过去——端着杯子——从她们身后经过——隔了不到两米——走向饮水机——接水——杯子接满——旋上盖——回来——苏晚晴抬头看了他一眼——一秒钟——目光碰了一下——平——静——像在路上隔着车窗和不认识的人交错的一瞬间——然后低头看许若琳的"空气解法"。顾临风走回座位——水杯是满的——放置——一滴没洒——翻开英语书——单词表——第一个词——"friend"——f-r-i-e-n-d——他看了好一会儿——然后用铅笔在旁边画了颗很小的星——五角——角有点歪——底边那条——往左倾了倾。

很多人来来去去。像教学楼里上下课的人——一批批——但真正走得近的——只有那么一两个——就够了——够把一天过成好的一天。她是这样。他也是。

下课铃响。苏晚晴和许若琳一起走出教室——两人并排——在走廊里——影子——整齐的——一高一低——低个子的那个话多——高的安静——在外人眼里——她们走在一起像姐姐和妹妹——虽然不知道谁是姐姐。许若琳把手搭在苏晚晴的肩膀上——搭了一下——说完那半句——然后拿开。顾临风看着她们走到走廊转角——然后低下头——物理练习册翻到新的一页——滑轮组——牛顿第二定律联立——开始做。
