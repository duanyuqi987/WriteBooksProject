# 第192章 夏知遥的消息

所属部卷：第四部：失去的美丽
时间：期末复习周夜里
地点：旧同学群

## 诗歌插页 P192：原创仿唐绝句

旧群忽报故人归，
一页灯前字已微。
不向春风追往事，
只将凉水洗尘衣。

真正的消息是晚上十一点多来的。夏知遥在群里发了一张县城车站的照片，配了一句：“回家办点事，谁在县城有空吃粉。”底下很快有人接话，问她什么时候走，问她现在在哪个学校，问她还记不记得高三班主任。

陆允川点开照片。车站外墙翻新过，蓝白两色，和他记忆里灰扑扑的样子不同。照片边缘有一只行李箱的轮子，旁边露出半张广告牌。他看了很久，看的其实不是夏知遥，而是自己曾经站在车站外等录取通知书的夏天。

他在输入框里打了“好久不见”，又删掉。打了“我可能也回”，又删掉。最后只跟着大家发了一句：“欢迎回来，办事顺利。”这句话平淡到几乎没有存在感，很快被表情和玩笑顶上去。

可陆允川松了一口气。他没有把旧同学群当成另一个出口，没有让夏知遥承担自己对苏晚枝的失落。他知道那样对谁都不公平。一个人不能因为在此处受伤，就去彼处索取补偿。

群里继续热闹，有人发语音，有人发错字又撤回。陆允川把手机放到桌上，倒了一杯凉水。水杯外壁很快起了薄雾。他拿指尖抹了一下，留下短短一道透明的痕。
