# 第038章 高二分班

时间：2011年9月1日

地点：教学楼公告栏

卷次：卷二：窗边人影

九月一日。开学。

校门口横幅换了——"热烈欢迎2011级新生"。布是新的，红得扎眼。跟去年九月份一模一样。家长和学生从横幅下面挤过去——高一的新生。顾临风从旁边走过。他现在是高二了。不需要再问"教学楼在哪"。不用新生横幅下面站。

但今天有一件事让他紧张。

分班名单。

高二文理分科。所有班级重新划分。理科班从四个变成三个。文科班两个。高一三是理科——所以他在理科班。但具体哪个班要等分班名单公布。

名单贴在公告栏。公告栏在教学楼一楼大厅。一块大黑板。上面贴了四张白纸。三张理科。一张文科。

上午九点半。公告栏前面挤满了人。比期末看成绩还多。每个人都伸长了脖子找自己的名字。有人蹲着——从下往上看。有人踮着脚从上往下看。有人挤不进去在后面跳着看。

顾临风站在人群外面。

他的手心全是汗。

手指握在书包带上——带子捏出了印子。手背上的筋鼓着。他没有挤。就站在那里等。呼吸比平时快了半拍。

赵一鸣从人群里挤出来。

"我在理科一班。你呢。"

"还没看。"

"你去看看。名单在最右边那张——理科三班。"

顾临风往前走了两步。然后停下来。没有继续。

"你帮我看。"

赵一鸣看着他。呆了半秒。

"你干嘛。自己去看不就行了。"

"帮我看。"

赵一鸣看了看他的手——还攥着书包带。指节发白。他大概明白了什么。

"行吧。名字。"

"两个人都看。"

赵一鸣看了他一眼。没问第二个人是谁。转身挤回人群。

过了大概两分钟。赵一鸣从人群里出来了。脸上带着一种很微妙的表情——像是在忍着笑。又像是在装严肃。

"理科三班。"

"谁。"

"都在。"

"什么。"

"苏晚晴——上面。你——下面。"

顾临风的手松开了书包带。

"两个都在三班？"

"你自己去看。名单上就是这么写的。"

顾临风没有去看。他站在原地。呼吸从快到慢。手里还有汗——但攥着的书包带松了。

又分到同一个班。

理科六个班——不对、高一六个班。高二理三文二重新分班。概率大概是三分之一。三分之一。

他觉得自己好像跑完了一个四百米。腿不酸——但是心跳快。

赵一鸣在旁边观察他。

"你耳朵红了。"

顾临风用手摸了一下耳朵。热的。

"没有。"

赵一鸣没再说什么。把手搭在他肩上。

"不过我在一班。以后不能上课给你传纸条了。"

"你以前也没传。"

"现在想传也传不了了。"

"嗯。"

赵一鸣松开手。

"不过你在三班也挺好的。"

顿了顿。

"至少窗边应该还有位置。"

赵一鸣走了。去一班教室报到。

顾临风走到公告栏前面。人群已经散了不少。还有稀稀拉拉几个人在看名单。他走到理科三班那张白纸前面。

从上往下。

第一个名字不是她。第二个也不是。第三个——苏晚晴。

她的名字端端正正印在那里。跟开学第一天被念到的时候一样。三个字。每一笔都很清楚。

他的手指点在她的名字下面。没有碰——隔了大概一毫米。

然后往下找。

中间有很多不认识的名字。有些是原来三班的。有些是别的班转过来的。十一个名字。十二个。十五个。二十个。

第二十四个。

顾临风。

他的手指停在自己名字上面。跟她隔了二十行。

理科三班。又在一起了。

他站在公告栏前面站了很久。久到后面的人说"麻烦让一下"。他才往旁边挪了两步。然后继续站着。

阳光从大厅的窗户照进来。地上的瓷砖有光影。学生的汗味、洗发水味、新课本的纸味混在一起。广播里在放集合通知。

又可以在一个班了。又可以看到她了。

第二个高一。

他转身。朝教学楼三楼走。理科三班的新教室——还是原来那间。黑板上写着"欢迎理科三班新同学"。日光灯亮着。吊扇在转。窗外那棵银杏还在。叶子还是绿的。秋天还没正式开始。

他找了一个位置坐下。不是原来的倒数第三排——新座位表还没贴。暂时随便坐。他选了一个中间偏后的位置。

几分钟后。

苏晚晴从前门走了进来。

浅绿色T恤。深蓝色牛仔裤。扎马尾。去年九月她穿的是浅蓝色T恤。今年是浅绿色。她往教室里扫了一眼。然后走向一个位置——第二排靠窗。

窗外的银杏枝在风里摆了一下。

顾临风翻开了一本书。

书上的字没有在看。

# 同一群人中。同样的窗边。又一个九月。
