# 第091章 夏天的聚餐之前

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P091：原创现代诗

我把星放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
小饭馆圆桌上的玻璃转盘没有替谁回答，
只把那一年轻轻照亮。

八月中旬，各家的升学宴陆续开始了。

程砚妈妈最先张罗。她在县城最热闹那条街上的一家川菜馆订了两桌——家里亲戚一桌，同学一桌。那家川菜馆叫"蜀味轩"，门面不大，但老板娘手艺好，回锅肉和水煮鱼是招牌。程砚妈妈说这家店她吃了十几年，知道分量足。

程砚妈妈提前三天就给陆允川打了电话。电话拨过来的时候陆允川正在院子里晾衣服——母亲去上班了，他负责把洗好的被单挂上晾衣绳。"小陆你一定要来啊，程砚说你不来他就不吃了。他说你在沿江读书，以后放假才能见到。我问他那我呢，他说你不一样。我就知道——你在他心里排第一。"

"阿姨你放心，我肯定来。"陆允川把被单抖开，夹上夹子，"需要我带什么吗？"

"什么都不用带。人来就行。你要是能帮我劝程砚穿衬衫，我额外给你红包。"

陆允川笑了。挂了电话以后他回屋里翻了翻衣柜，把几件T恤摊在床上比较了一下。最后选了一件深蓝色的——不是因为好看，是因为这件领口没变形。母亲下班回来以后看到床上摊开的衣服，说了句"蓝色好，显精神"，然后催他去剪头发。催了三次。第三次的时候直接骑电动车载他去的理发店。

宴席那天，陆允川穿了那件深蓝色T恤，头发剪过——理发师问他想剪什么发型，他说"随便，短就行"。理发师端详了他的头型三秒，说"你这头型适合寸头"，然后就推了寸头。效果意外地不错，至少母亲说"精神多了"。

他到了川菜馆门口，看到程砚穿了一件蓝色衬衫站在门口迎客，领口扣到最上面一颗，整个人看起来比平时正经了十倍。

"你这衬衫谁给你买的？"陆允川走过去。

"我妈。她说升学宴要穿衬衫。我说我都考上省城工学院了穿什么衬衫，她说就是考上了才要穿。"程砚扯了扯领口，"太紧了。我呼吸有点困难。"

"你把第一颗扣子解开。"

"我妈不让。"

陆允川伸手把他第一颗扣子解开了。程砚深吸一口气。"舒服了。谢谢你救了我一命。"

同学那桌坐了十来个人：陆允川、叶澄、周长海、林曼、祁野、杜雨薇、还有几个平时玩得好的男生。祁野来得最晚——他说是因为在家找红包找了二十分钟，最后发现红包被他家猫叼到沙发底下去了。他穿了一件有领子的Polo衫，一看也是被家长逼着穿的。

夏知遥没来——她家那天正好也有亲戚从深圳过来，杜雨薇替她带了一个红包过来。红包是暗红色的，不花哨，正面写了四个字：前程似锦。字迹很工整，横平竖直，每一个捺脚都收得很干净。

"她写字还是这么好看。"林曼把红包放在桌上，用手指摸了摸那四个字——指腹能感受到笔锋的凹痕。

叶澄看了一眼字迹。"她自己练的。高二的时候她买了一本钢笔字帖，每天晚自习抄一页。她说学建筑的，以后图纸上要写字，写不好看丢人。"

"这也要练？"程砚凑过来看，衬衫领子又勒到他脖子了。

"你以为。优秀的人什么都练。"叶澄把红包转过来看了一眼背面——背面什么都没写，就是一面干净的红色。

祁野凑过来看了一眼红包上的字。"她写字比我好看一百倍。我写的'前程似锦'看起来像'前程似绵羊'。"

"那你就别写。直接给红包就行。"杜雨薇说。

程砚他妈端着茶壶过来给同学桌加水——茶壶是不锈钢的，壶嘴有点歪，水流出来的时候要偏一点才能对准杯口。她笑眯眯地问了一圈人考上了哪个学校。问到叶澄的时候，叶澄说省城师范，程砚妈妈立刻眼睛一亮，放下茶壶握住她的手："老师好啊！你以后要当老师，程砚要是当年多听你的话就好了。"叶澄难得地脸红了一下——不是害羞，是被一个长辈当众握手有点措手不及。

问到祁野的时候，祁野说省城理工土木工程。程砚妈妈愣了一下："你跟陆允川一个专业？"祁野说对，但不同学校。程砚妈妈说那以后你们可以一起修桥。祁野看了陆允川一眼，两人同时笑了——因为他们都想起了刚才陆允川说过的"我们是甲方和乙方的关系"。

问到陆允川的时候，程砚妈妈说土木好，以后可以修桥。又问陆允川妈妈怎么没来，陆允川说她今天在厂里上班，请不了假。程砚妈妈沉默了一秒，然后很郑重地说："那你回去告诉她，她培养了一个好儿子。你这孩子是我们程砚最好的朋友。我跟你妈没见过几次面，但我知道她肯定是个好人。"

陆允川低下头喝茶。茶很烫——是新泡的菊花茶，程砚妈妈特意给同学桌泡的。菊花在杯底慢慢绽开，他把茶杯转了半圈，看着那朵菊花转到了另一边。

周长海考上了一个专科学校——汽修专业。他在桌上说了这件事之后，大家沉默了一秒。圆桌转盘上那道水煮鱼正好转到他面前，他没动筷子。然后程砚第一个开口："汽修好啊。你知道现在修一辆宝马多少钱吗？"

"不知道。"

"我也不知道。但肯定比修我们那辆破自行车贵。"程砚一本正经地说，"而且你想想，以后我们这些学机械的、学土木的，造出来的东西坏了，不都得找你修吗？你是产业链的终端。"

周长海笑了一声。"你安慰人的方式还是这么奇怪。"

"我没有安慰你。我是真的觉得汽修好。你修车，我造机器。以后说不定能合作——你负责修我造的机器的外壳。"程砚把茶当酒，举起来敬了周长海一杯。

祁野也举起杯子。"我做甲方，项目坏了喊你来。"

"你们能不能不要把我当工具人。"周长海也举起茶杯碰了一下。三个杯子碰在一起的声音很响，旁边几桌的亲戚都转头看了一眼。然后周长海喝完茶，放下杯子，又夹了一块回锅肉。这次夹得很自然，像刚才那个沉默从来没存在过。
