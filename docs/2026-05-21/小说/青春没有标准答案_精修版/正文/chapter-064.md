# 第064章 一张小纸条



时间：2009年一月下旬，期末前最后一周



地点：十班教室前排和水房门口



卷次：第三卷：雪落在两栋楼之间



2009年一月下旬，期末前最后一周，十班教室前排和水房门口，写着别怕二字的纸条先把这一天按进了十班的日常里。孙佳怡接完母亲电话后眼圈发红，宋乔写了张纸条塞给她。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

纸条被风吹到段璟脚边，他没有打开，只递回给常俊凡。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看写着别怕二字的纸条，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

孙佳怡也在这一天被推到前面。孙佳怡的特点是家里管得紧，电话一来就紧张，但不愿让同学看见。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“你别看，就是两个字。”宋乔说。

“我没看。”段璟说。

“谢谢。”常俊凡说。

“我就是有点烦。”孙佳怡说。

“烦也可以写题，写题不解决全部，但能先解决一道。”叶书兰说。

小纸条在十班不是稀罕事。有人传题号，有人借橡皮，有人写下课去不去水房。

真正麻烦的是，纸条一旦被起哄，就会从一件小事变成谁也不好收拾的热闹。

段璟看见纸条角露出来时，第一反应不是好奇，而是上一次被误会后常俊凡皱起的眉。

他把纸条递回去，手指没有多停。这个动作太普通，普通到旁边人找不到起哄的缝。

宋乔看了他一眼，像在确认他这次终于学聪明了。

付小川张了张嘴，又自己闭上，把玩笑咽回去，低头装作研究橡皮屑。

常俊凡接过纸条时只说了声谢谢，声音不大，却把边界放得很清楚。

十班从这些小事里慢慢学会，热闹不是想要就要，别人的难处也不是拿来传的。

一张小纸条没有悬在空处。小纸条折痕旁边沾着的一点粉笔灰，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，一张小纸条带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

叶书兰进门时不一定立刻说话，鞋底在水泥地上一停，一张小纸条带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在孙佳怡快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。一张小纸条越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。写着别怕二字的纸条让他想说话，又逼他先低头看题。

孙佳怡被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，一张小纸条留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。孙佳怡路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为一张小纸条翻身，谁还在被窝里背单词，旁边人都听得见。

县城晚上暗得早，小店门口的灯一亮，纸条上的玩笑就显得没有白天那么轻。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理一张小纸条里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：写着别怕二字的纸条、一只冻红的手、一句老师点名、一块还没扫干净的雪。

期末前最后一周，十班教室里弥漫着一种比平时更重的安静。不是纪律好，是每个人都在算——算模考分、算弱势科、算期末还能往前挪几名。笔尖在纸上的摩擦声比之前密了，翻卷子的声音也勤了。黑板右上角用红粉笔写着倒计时数字，值是"7"。

孙佳怡从宿舍电话旁回来时，眼圈是红的。她不是那种会当众哭出来的人，眼泪只在眼眶里转了一圈，就被她低着头快速眨回去了。她坐回座位后，把英语书翻开，目光落在单词表上，但段璟注意到她的眼睛一直没有移动——她看的不是书，是书页边缘那道被手指磨出来的毛边。她坐回座位后，把英语书翻开，目光落在单词表上，但段璟注意到她的眼睛一直没有移动——她看的不是书，是书页边缘那道被手指磨出来的毛边。

宋乔看见了。宋乔不用问原因，因为她自己也被家里电话问哭过——女生宿舍的电话，在某些夜晚能变成全封丘最沉的听筒。孙佳怡的妈每次都问同样的问题：成绩够不够考新乡的学校？生活费够不够？棉袄够不够厚？三个问号像三根钉子，钉在孙佳怡的"还行"上。她的"还行"其实不行——上次月考她进步了六名，但她妈觉得进步六名太慢，因为"你表哥在郑州考了前三"。

宋乔从草稿纸角落撕下一张纸条。纸条只有两指宽，她拿起圆珠笔，犹豫了一下，写了两个字："别怕。"她没写"别担心"，也没写"会好的"。她选了最直接的那个——怕。因为怕不是成绩的问题，是人的问题。

纸条从宋乔手里传出去，经过两个人的桌面，第三个人递到孙佳怡手上。孙佳怡展开纸条时手指尖在抖——不是冷，是刚才那通电话的余震。她看了一眼纸条上的字，没有抬头找宋乔，只是把纸条折好塞进校服口袋，然后翻开了下一课的预习内容。

但纸条在传递过程中被窗缝里灌进来的风吹了一下。那阵风很小，但刚好把纸条从孙佳怡的桌面上掀起来，翻了两个滚，停在段璟的脚边。纸条没有全展开，只露出"别"字的上半部分。

段璟低头看见纸条，然后又看见了周围三四个人的目光正往他这边聚。在十班，"男生捡到女生的纸条"这种事自带放大效果。付小川的嘴已经张开了一个"哦"字的预备口型——他的舌头都顶到上颚了，只差一个爆破音。

段璟弯腰把纸条捡起来。他没有展开，甚至没有低头看那个"别"字。他把纸条捏在手里，站起来，往前走了三步，递到常俊凡桌上。

"你的？"

常俊凡看了一眼纸条的折痕——那是宋乔特有的折法，从宽边往窄边折两下，像折人民币的那种。她抬头看了段璟一眼，那个眼神里有一层很薄的"谢谢"和一层更薄的"你这次做对了"。

"谢谢。"她说，声音不大。

段璟转身回了座位。付小川的嘴合上了，把那个还没出生的"哦"字吞了回去。他低头研究自己桌上的橡皮屑，像刚才什么都没发生过。

常俊凡把纸条夹进错题本最后一页，然后继续做她的题。整个过程不到二十秒，旁边的人连起哄的缝都没找到。

宋乔从这件事里看出了两个东西。第一，段璟这次真的学聪明了——他没有打开。第二，常俊凡配合他的方式不是帮他解释，而是让他自己用动作证明。她只是说了那声"谢谢"，但那声"谢谢"刚好挡住了所有可能往段璟身上涌的猜测。

下午的课间，常俊凡在水房接水时碰到了宋乔。宋乔把搪瓷杯放在水龙头底下，没拧水，先转头看她。

"你那个谢谢，说得真好。"宋乔说。

"不是说的好，是他做的好。"常俊凡说，"他不捡，或者捡了打开，我的谢谢就没用。"

"你是在夸他？"

"我在说事实。"常俊凡拧开水龙头，热水哗地冲进搪瓷杯。杯子里的金银花茶被冲得翻了个身，那两瓣花瓣从底浮到面上，像两条突然翻白的鱼。

宋乔看着花瓣，忽然觉得常俊凡对段璟的评价方式有一个固定格式：不说好坏，只说事实。而"他没打开"这个事实，在常俊凡的语言体系里，就是最高的评价。

晚自习前，段璟在走廊上碰到了孙佳怡。她正把一张纸塞进废纸篓——是数学草稿纸，背面画满了圈，中间写着"六名"两个字，圈外面是大大的叉。她把废纸团成团扔进去，然后抬头看见段璟。

"谢谢你没看。"她说。

"没什么。"

"不是没什么。"孙佳怡把校服袖口往上拉了一下，"上次因为雪仗那张画，罗雨桐被误会的时候，我什么都没做。这次我自己差点变成被误会的那个人。"

段璟没有说话。他知道孙佳怡不需要安慰，她只是在跟自己做一笔账——上次别人被起哄，她沉默了。这次轮到她差点被起哄，有人没让事情变大。这中间的差距就是她眼眶发红以后从纸条上得到的最大的教训。

纸条上那两个字被风吹到段璟脚边时，吹它的那阵风正好也被暖气管的热气顶了回去。十班的冬天总在这些小小的气流对冲里，把一些本来会烧起来的东西慢慢扑灭。

段璟后来在草稿纸上写了一道数学题，证完两条直线垂直之后，在纸的右下角写了三个很小很小的字："没打开。"他用尺子压住那行字，继续做下一题。

晚自习第二节，叶书兰从后排走到前排，在孙佳怡桌上放了一张卷子——是她自己整理的英语易错题合集，手写的，每道题旁边用不同颜色的笔标注了陷阱类型。叶书兰没有说话，放下就走了。她走路的方式跟常俊凡有点像：不做多余动作，但每一步都会落在需要的地方。

"她怎么知道我英语不好？"孙佳怡小声问宋乔。

"因为上次英语听写你错了五个她看了十遍卷子。"宋乔说，"叶书兰连老师的圆珠笔什么颜色都记得。你错哪几个她肯定知道。"

宋乔说这话时把自己的纸条也夹进了错题本——那张写了"别怕"的纸条被她用一个废弃的回形针固定在错题本封二上，回形针已经有点锈了，在纸上留下两个很小的锈点。宋乔觉得锈比新东西好看，因为锈说明放得久，而有些话就是应该放得久一点。

马立群走过走廊时往十班教室看了一眼。他看见孙佳怡正在看叶书兰的手写卷子，在门口停了半拍。他没有进来表扬谁，只是继续走到办公室，把一沓新的英语卷子搁在桌上。马立群不擅长当众表扬人，但他会在第二天发卷子时把孙佳怡的那份放在第一页——这对他来说就是表扬。

段璟在晚饭时发现付小川给了他半个馒头。他没要馒头，但付小川说"你捡纸条那个动作值半个馒头"。段璟把馒头吃完了，馒头很干，但他没喝水。他觉得付小川的支付方式很怪异但很公平——你帮我省一个"哦"，我给你半个馒头。

"如果我看了一眼纸条呢？"段璟问。

"那你要给我整个馒头。"付小川说，"现在这样各吃半个，正好。"

十班的经济学大约就是这样：用没说的话、没打开的字、没发生的起哄来相互结算。没有账本，但每个人都记得自己的账。

孙佳怡把那张手写卷子翻开。第一道题就是她上次做错的——"make"后面跟动词的什么形式。叶书兰在旁边标注了一个小括号，里面写着："此坑全班踩了十四次，你是第十四个。"孙佳怡看着那个括号，觉得这比"加油"有用一百倍。因为它不是在安慰你，是在告诉你：这是一个客观存在的坑，踩进去不丢人，不爬出来才丢人。

教室的窗玻璃上又结了一层雾气。这次不是圆的——雾很均匀，把窗外操场上的路灯变成一团模糊的橙黄色光晕。后排有人用手指在雾上画了一道线，又画了一道，两道线交叉成一个十字。画完那人没有继续画，只是在十字旁边用指腹点了一个小点，像点在坐标纸上。

段璟看着那个十字，想到自己刚才证的两条直线垂直。他觉得人跟人之间的关系也像几何证明——不是靠直觉，是靠一步一步推出来的线段和角度。每一步都要有理由，每一步都不能跳。

水房门口今天晚上特别忙。天气越冷，打热水的人越多。暖水瓶排满了整条墙根，瓶身上画着各种标记——有的是名字，有的是班级号，有的只是一道和别的瓶子撞色才划上去的红油漆杠。段璟去水房接水时，看见孙佳怡的暖水瓶旁边多了一个蓝盖子——跟她之前用的那个不一样，更紧，盖上去的时候不会往外冒热气。

"谁给你的？"段璟问。

"宋乔昨天晚上给我的。她说我那个盖子松了，每次接完水回到宿舍就凉了一半。"

段璟把搪瓷缸放在出水口下面接水。热水哗哗地响，把他今天发生的事从头冲了一遍：孙佳怡的眼眶、宋乔的纸条、常俊凡的"谢谢"、付小川吞回去的"哦"、叶书兰的手写卷子、那个窗玻璃上的十字。

他想，这些东西加在一起，大概就是十班给他的全部教育：不是每件事都要说出来，不是每个发现都要宣告。有时候不做的力量比做更大——没打开的纸条、没喊出来的起哄、没画完的十字。

这些动作凑不成一个完整的理论，但在封丘一中的冬天，它们已经够一个人慢慢变好了。

熄灯前，段璟在宿舍里听见付小川跟赵远航聊那天雪仗的后续。付小川说起宋乔塞纸条这件事，说宋乔的方法跟常俊凡不一样但效果相同。赵远航说怎么个相同法。付小川想了想，说："常俊凡是把别人漏的东西补上。宋乔是把别人藏的东西翻出来，但翻的方式让人不疼。"

段璟在被窝里听着，觉得付小川其实很敏锐。他只是用玩笑把这些敏锐压住了——就像他用玩笑把今天差点说出口的那个"哦"压住一样。

他把手从被窝里伸出来，在空气中画了一道线。然后又画了一道。两道线交叉成十字，消失在黑暗里。

一张小纸条之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

孙佳怡没有因此变成特别醒目的人，可在段璟眼里，孙佳怡从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，这一次没人起哄，十班从上次误会里学会了把别人的难处放回别人手里。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
