# 第021章 操场边的单词之前

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P021：原创现代诗

我把早晨放低，
让一粒尘从课桌边慢慢经过。
你的名字没有被我喊出，
只在单词书页角折起的三角旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写操场边的单词抵达之前的坡度。晚自习前的操场还有白天剩下的热。看台水泥摸上去发烫，塑胶跑道被夕阳照出暗红色，像一条慢慢冷下来的河。陆允川摊开单词书，嘴里念着"abandon"，眼睛却跟着跑道边的影子移动。

念到第三遍"abandon"时他发现自己连中文意思都忘了。他又看了眼书角那个被折起的三角——那是程砚折的，上周两人在看台上背单词，程砚说折个角好翻页，折得歪歪扭扭。这个角现在支棱着，像一个不肯被书本压平的念头。

"你是来背单词的还是来看夕阳的？"程砚从旁边的台阶上转过来，手里的单词书翻到了第七页——明显比陆允川的进度快。程砚背单词的办法很粗暴：先读三遍中文，再读三遍英文，然后闭上眼睛在脑子里把十个单词编成一个离谱的故事。

"我背到第四个了。"陆允川说。
"你背到第四个，手里拿着的是第三页。第三页第一个单词是abandon，中文意思放弃。你在放弃这个单词上停了五分钟——你想说明什么？"
"说明我背得很认真。"
"说明你在看跑道。"

操场上有人在跑步。不是校队的，是几个因为体育课被占而自己来补跑的高二学生。跑道上除了他们还有一圈慢走的人——夏知遥和蒋雯。蒋雯说今天的物理作业太多，需要走几圈放空脑子。夏知遥没有跑步，只是慢慢地走，手里拿着一瓶水，偶尔喝一口。她的影子被夕阳拉得很长，从跑道里面斜着伸到草坪上。

陆允川的目光在单词书和跑道之间来回切换，频率越来越不规律。他开始在心里给自己找借口：背单词需要远眺保护视力，看书太近会加深近视，塑胶跑道的颜色刚好是暗红色，看着能放松眼球。每一个借口都在骗自己，但他需要骗——因为不骗的话他没法继续坐在这看台上。

晚自习前的时间到了，操场边的路灯亮起来，跑道上的影子从一道变成了两道——人和灯各给了一道、一深一浅。蒋雯和夏知遥走完了三圈，从西看台下面经过时蒋雯抬头看见了他，挥了一下手："嘿，背单词啊？明天英语小测加油！"陆允川举起手里的单词书回了一句"你们也加油"。蒋雯笑了笑继续走。夏知遥没有抬头，她低着头在用脚踢跑道边沿的一块碎石子——那石子滚了两圈停住，她收脚绕过去，继续走了。

陆允川把目光收回单词书。他翻到第四页，逼自己从"abandon"隔壁开始重新背。新的单词是"absorb"——吸收。他把这个词抄在手上，中文意思也抄了一遍，手心写得满满的。他告诉自己：下一次来操场背单词，不抄满一页不许走。

但他也知道，下一次还是会在同一个角度看同一个方向，手中的单词书翻到同一页，在同一个"abandon"上停下来。
