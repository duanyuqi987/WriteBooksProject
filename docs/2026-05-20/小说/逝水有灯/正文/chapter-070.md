# 第070章 墙角的试卷多年后

所属部卷：第二部：六月以后
时间：多年后
地点：旧纸箱前

## 诗歌插页 P070：原创仿唐绝句

墙角旧卷落微尘，
红字犹存不问人。
三年重看非风月，
半是题痕半是身。

多年后，陆允川从老家东屋翻出一个旧纸箱。

箱子已经软了，胶带发黄。里面还有几本高三课本和一沓没卖掉的卷子。母亲说早该扔了，他却蹲在地上翻了很久。卷子上的字迹有些淡，修正液边缘开裂，像干掉的白色旧伤。

他看见那张写着“回头补”的数学卷，忽然笑了。

后来当然没有回头补。人生里很多句“回头再说”最后都没有回头。但这并不全是遗憾。有些事情的意义不在于补完，而在于曾经逼着一个人坐下来，低头算，算到夜里灯暗。

他把几张有纪念意义的卷子留下，其余重新装好。不是舍不得全部，而是终于能分清：保存过去，不等于把过去整箱搬走。

文件袋上，他写：高考后东屋。

里面有一张数学卷、一页作文模板、一小片脱落的透明胶带。没有谁的名字，却有那段日子的重量。陆允川把文件袋放回书架，心里很静。

那堆墙角试卷终于不再像一个结束。它更像一段路边留下的纸屑，被风吹过很多年后，仍能证明有人从那里经过。
