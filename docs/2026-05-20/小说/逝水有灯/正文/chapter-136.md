# 第136章 叶澄的劝告之前

所属部卷：第三部：寒风里的花
时间：六月午后
地点：南湖凉亭

## 诗歌插页 P136：原创现代诗

南湖的风很慢，
慢到能把话吹清。
她把活动表压在水杯下，
看见他一页页翻着自己，
却总不肯读别人。

叶澄找陆允川谈话前，先在活动室和苏晚枝一起核对讲座名单。报名表少了三个人，音响借用还没批，指导老师临时要求改主持词。

苏晚枝坐在桌边，低头改稿，眼下有一点青。叶澄递给她一杯温水，“你今天先回去吧，剩下的我来。”

“没事。”

“你说没事的时候，通常就有事。”

苏晚枝笑了一下，没反驳。过了会儿，她说：“陆允川最近有点紧。我不知道怎么说才不伤人。”

叶澄没有立刻接。她把报名表理齐，装进文件夹。她认识陆允川，也知道他不是坏人。他帮忙认真，写诗也认真，可认真一旦总要别人回应，就会变成另一种沉。

午后，叶澄在南湖凉亭看见陆允川。他坐在栏杆旁，手里拿着一本旧诗抄本，纸页被风吹得翻动。

“有空吗？”叶澄问。

陆允川抬头，“有。”

她把水杯放到石桌上，用报名表压住一角，“那我说点不好听的。”
