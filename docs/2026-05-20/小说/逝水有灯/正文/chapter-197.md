# 第197章 回乡的暑假

所属部卷：第四部：失去的美丽
时间：暑假
地点：镇上老桥

## 诗歌插页 P197：原创仿唐绝句

老桥白漆半成灰，
暑气沿河缓缓回。
年少一声沉水底，
如今过处不相催。

县城汽车站比记忆里亮了许多。陆允川下车时，母亲已经等在出口，手里提着一袋桃子，见他出来先问：“瘦了没有？”他笑着接过袋子，说学校食堂还行。母亲不信，拉着他去路边吃面。

下午办完材料，他一个人绕到镇上老桥。桥栏杆重新刷过白漆，但靠近拐角的地方又被雨水泡出灰斑。河水比高中那年浅，岸边多了一排健身器材，有老人慢慢压腿，小孩把矿泉水瓶踢来踢去。

陆允川在桥中间停下。高考后的那个傍晚，他曾在这里把喜欢说得太急，声音被风吹散一半。那时他以为没被回应就是世界塌了一角，回家后把自己关在东屋，连晚饭都没好好吃。现在站在同一个地方，他最先想到的却是母亲刚才把桃子塞进冰箱，说晚上冰一冰更甜。

桥下水流很慢，漂着几片叶子。他没有给任何人发消息，也没有拍照证明自己来过。只是扶着栏杆站了一会儿，手心沾到一点未干透的油漆味。

离开时，路口新开了一家文具店。陆允川进去买了一支黑色中性笔，老板问他是不是老师。他说不是，还在读书。老板把找零推过来：“读书好，趁年轻多读。”这句话很旧，却让他心里安稳了一点。
