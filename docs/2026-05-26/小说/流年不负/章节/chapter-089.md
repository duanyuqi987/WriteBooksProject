# 第089章 大一下

时间：2014年2月

地点：楚州师院

卷次：卷五：各自南北

二月十七号。大一下学期开学。元宵节刚过三天——楚州的街上还能看到零星的鞭炮碎屑，红红白白的铺在路边水沟里。

从青阳回到楚州。他坐的上午十点的大巴。车上人不多——节后返校潮过去了，车厢里只坐了十来人。他靠在窗边——看着高速公路旁的田野——枯萎的青灰色天空下麦田是黄绿色的，很矮。麦田间有大棚——白色塑料膜一垄一垄的，里面的温度高，棚膜上凝了一层雾气。车窗玻璃冰凉——他把头靠上去，渐渐从车窗传来困意——但没睡着。

楚州的冬天比青阳干。回到宿舍——他是宿舍第一个到的。整个楼里很安静——只有一楼的热水器嗡嗡响着。开门的时候屋里闷了一口气——窗帘拉着，地上有一层灰。桌上放着的泡面盒还没倒——阿坤走时忘扔了，盒子里剩的面汤早就干成了一层皱皱的膜，膜的边角翘起来，下面露出了黄色的面屑。他把窗户推开——冷风灌进来，窗帘往里面鼓，鼓成了一个弧形。空气中飘着干的灰尘味和旧木头的味道。

他把泡面盒扔了。桌面上用湿布擦了擦——布上蹭了一层灰。然后铺床——被子从蛇皮袋里拿出来，蓝格子的，叠得整整齐齐的。他闻到了被子上家里的樟脑味——淡淡的、有点冲——是他妈放了防虫的樟脑球的味道。枕套是新换的——白色布料，他妈缝的——靠边的一处针脚扎歪了，有一个短一截的针距。铺好床后他打开窗往下看——校园里陆续有人拖着箱子回来了，轮子擦着水泥地——咕噜咕噜——声音在一楼就听得见。

室友们陆续到齐。李博带了一大袋洛阳特产——牡丹糕和花生糕，包装盒红底金字。他取出来一块递过来——"尝一下——洛阳的——。"阿坤带来一台新机械键盘——清轴的，说要"上钻石"。拆开放在桌上——黑色底壳，RGB灯光一圈圈地闪——他敲了两下键——啪啪——非常清脆，像踩碎干树叶的声音。"换了这个键盘——你们晚上别嫌吵——我尽量轻点敲。"小伍的行李箱最大——里面是被子、棉袄，还有他妈塞的十包涪陵榨菜——红色包装透明塑料袋。他掏出来一包包往每个人手里发——"这个给你、这个给你、这个给你"——数到最后自己只剩四包。阿坤说"你妈把你当兵送了。"小伍说"她说食堂菜不够咸。"

顾临风把李博递来的牡丹糕掰了一半——咬了一口，花生味的，有点甜，里面的花生碎嚼起来有脆的颗粒感。

新学期课程比上学期多。古代文学进入魏晋南北朝——老师在讲台上讲陶渊明——"采菊东篱下，悠然见南山"。他在笔记本上写下这两行。然后在旁边画了个小圈——圈里面写着"南山"两个字。写完以后在旁边打了三个问号——不知道为什么写这个。大学英语加了听力课——语音教室里每人戴一个耳机，耳机很旧，海绵耳罩脱了半块，露出里面的黑色泡沫。放的BBC新闻——女播音员的英音一长一短，偶尔穿插一段音乐。他在答题卡上勾选项——ABCD。书法课交了一次作业——颜体《多宝塔碑》的六十个字，老师评了一个"A——"——批在宣纸边上——"结构尚稳，笔力不足。多练"。

他开始参加社团。

第一个是文学社。师院的文学社叫"流萤"——名字取自"轻罗小扇扑流萤"。社长是个大三女生，叫刘思齐。她在社团招新会上穿了一件盘扣的棉布外套，站在讲台上讲话："文学不是宏大叙事，是用细节留住时间。你闻到食堂炒洋葱的味道了吗——那就是文学。"台下有几个人笑了。顾临风填了入社表——粉色纸上印着"姓名""院系""最喜欢的作家""想说的话"。他在最后一项写了两个字——"日常"。

第二次活动在一个小教室里。桌椅围成了一圈。二十来个人。主题是"现代诗创作"。刘思齐在白板上写了三个词——"呼吸""骨头""距离"——字迹好看，一笔一划都像是临过字帖的。"自由写——半个小时。"教室里安静了。只听到翻纸声和笔尖划过纸面的沙沙声。窗外的阳光照在黑板上——投在白板边缘有道金边。他在本子上写了——划掉了——又写了几行——第一行是"走廊是一条横着的骨头"。停笔。看了几秒。没有交。

分享环节——上台朗读的女生声音轻，有点发颤——她写着关于母亲的诗，读到一半声音稳了下来。读完大家鼓掌。顾临风跟着拍了两下手。散场时那个女生走过来——她扎低马尾，穿咖啡色毛衣，毛衣袖子长遮住半个手背。"你刚才没分享。""没写完。""没关系——下次可以分享。"她笑了笑——牙齿很白。"我叫许若琳——英语系的。""顾临风——中文系。""专业对口。"她笑着说。

许若琳往门口走了两步，忽然回过头——马尾甩了半圈碰到桌角。"对了——你认识苏晚晴吗。"顾临风的手放在书包拉链上——停了半秒——书包从高中背到现在，拉链头涩了。"认识。""你怎么认识她？""高中同学。""哦——青阳一中的对吧。我跟她是大学室友。她说你们学校有不少人考到楚州来了。""嗯。""你们还有联系吗。"顾临风把拉链拉上——在那个涩的角度卡了一下——再用另一只手按住布料拉过去。"很久没联系了。""哦。"许若琳点头——没追问。

走出教室。走廊的日光灯很白，地砖是白灰相间的方格。他停了一下——窗外是北操场，有人在跑步，有人在放风筝，一只红色金鱼风筝在天上摇摇摆摆的。许若琳刚才那句——"我跟她是大学室友"——也就是说，如果他想知道苏晚晴的消息，可以从这里知道。但他没有问。

下午去了书法社。书法社活动室在大学生活动中心三楼——一间不大的教室，四面墙上贴满社员的作品——楷书、行书、隶书。社长是个戴圆框眼镜的男生，清瘦，穿一件灰色中式对襟衫——讲起颜体和柳体的区别滔滔不绝："颜体筋健、柳体骨劲。颜像蹲马步，柳像挺胸收腹。"顾临风毛笔蘸墨的时候手还是有点抖——在宣纸上写了第一个"永"字——横不够平，捺拖太长。社长走过来说"起笔太重——再写一个"。他又写了一个——这次横平了，捺还是长了。

晚上回到宿舍。李博在看书——《中国历代政治得失》。阿坤在打游戏——新键盘的RGB光闪得热闹，键盘啪啪响着。小伍在写信——这次信封印着长颈鹿。顾临风把鞋脱了。手指上墨汁的味道还在——肥皂搓了两遍没完全洗掉，指甲缝里还有些墨黑。

"今天见了一个人。"他突然说。

三个人都停住了。阿坤把耳机摘下来挂在脖子上——里面有游戏技能音效在响。"女的？""苏晚晴的室友。""谁？""许若琳。英语系的。跟她同宿舍。"阿坤眨了眨眼睛："所以呢。""没所以。""你说'没所以'——那就是有所以。"顾临风把鞋放床底下——鞋底又沾了红色跑道颗粒。"碰巧遇见的。"

李博看着顾临风的脸色——然后看了阿坤一眼。阿坤领会到班长的眼神——用力把耳机重新戴上，继续打他的副本。键盘啪啪响着。

顾临风躺到床上。从枕头下摸出手机——翻到通讯录——许若琳——新存的号码。号码是她活动结束前主动给的——"社团有活动互相通知"——存成"许若琳文学社"。他看了几秒——按灭了屏幕。

窗外有人在放周杰伦的《晴天》——吉他声从某个宿舍窗户飘出来，声音很轻，被夜风揉碎了——"为你翘课的那一天——花落的那一天——"。他闭上眼睛。脑子里是许若琳那句话——"我跟她是大学室友"。苏晚晴——十二公里外——每天跟许若琳住在同一间宿舍，用同一张桌子，在同一盏日光灯下看书。而那个校园——301路的终点站——他从来没有去过。
