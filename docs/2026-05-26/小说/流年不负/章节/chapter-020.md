# 第020章 同一个班

时间：2011年3月初

地点：青阳县一中理科三班新教室

卷次：卷一：初见秋风

分班名单贴在公告栏上。

公告栏在教学楼一楼大厅。玻璃罩子包着的那块黑板上贴了三张红纸——理科一班、理科二班、理科三班。纸比上学期成绩单短一点，但名字印得密。一张纸上大概有四十几行。

上午八点。公告栏前面已经围了人。有人在往里面钻，有人递手指，有人踮脚。高一的校服和高二的不一样——高三的没有——所以看身高就能分辨年级。人群大多是高一的脸。

顾临风到的时候前面有七八个人。他站在后面等着。阳光从大厅门口照进来，落在地板上是一块长方形的白光。有人踩在光里，走过去的时候影子在上面晃了一下。

前面的人走了一个。他又往前了一步。

理科三班。

他在红纸上找到了那个名字。苏晚晴。在列表中间偏上的位置。她的名字写得不大。黑色的圆珠笔。字是正楷的——和黑板报上粉笔字的写法很像。

他的手指在"苏"字上停了一下。然后继续往下找。找到了自己的名字——顾临风。他在她的名字下面某一行。隔了大概十几行。但在一张纸上。

他看了一会儿这两个名字。一上一下。隔十几个空格。在一张红纸上面。

然后退出了人群。

理科三班的教室在三楼东边。

走廊最尽头。门牌换了——"高一（三）班理科班"。字是新贴的。纸还是白色的，裁得不直。门半开着。能听见里面有搬桌子的声音。椅子腿刮地面的声音。

顾临风背着书包走到门口。

站了一下。然后走进去。

教室比原来的高一三班大一圈。窗户更多——教室前后两排窗。窗框是绿色的漆，新漆的，味道还没散干净。玻璃擦过，外面梧桐树的影子投进去，在课桌上动——风在吹树枝。

日光灯全开着。讲台上没人，但黑板擦干净了。粉笔槽里有半根白的和半根蓝的。

课桌排成八列。已经坐了不少人。有些是老面孔——从高一三班来的、从别的班来的。有个男生在最后一排把桌子搬正，桌面磕在墙上"咚"一声响。

第二排靠窗。

一个身影坐在那里。

墨绿色棉服搭在椅背上，露出里面浅蓝色T恤的领子。马尾扎着。头低头在翻一本刚发的新教材。右手拿着笔——蓝色圆珠笔。小指翘起来一点点。

书包挂在椅背上。拉链上的陶瓷小鱼在光里闪了一下——白色的，上面那两道蓝色线。黄豆大小。

苏晚晴。

她已经坐在窗边了。

像上学期一样。第二排。靠窗。

顾临风看了她一个呼吸的时间。然后把视线移开了。

他往教室后面走。脚步不快。经过她那一排的时候，隔了两列走道。她没抬头。

他走到倒数第三排。把书包放到桌上。坐下。

新教室。新座位。比上学期位置偏左了一列。往左前方看——第二排靠窗。距离比原来进了一点。她不用隔四排两个过道——现在是隔两个过道。上课的时候不用太刻意就能看到她的方向的粉笔灰——粉笔灰在日光灯的光柱里飘。

赵一鸣从门口进来了。他一眼就看到了顾临风。走过来把书包摔在旁边座位上。

"又在一个班。"

"嗯。"

"而且又是同桌。"

"嗯。"

赵一鸣往前面看了一眼。然后嘴角一提。没说别的话。然后开始往桌斗里塞书。塞完往前拱了拱，胳膊肘压在桌边上，环顾四周。

"这个班的人好像不行。好多是新面孔。看来考得好的不认识。"

上课铃在八点十五分响了。

新班主任走进来。一个戴眼镜的中年男人。个子不高。微胖。头发从左边往右边梳，遮住头顶。——"我姓陈。教物理。"

顾临风低了一下头。然后翻开新发的物理课本。第一页——曲线运动——平抛物体。正上方那行铅字很新。纸是白色的——还没被手摸过。

他翻开书的时候。从课本中间掉出了一张纸片。

是他从分班名单上抄的名字。钢笔写的——蓝色——苏晚晴。理科三班。顾临风。理科三班。

他拿起纸片。折了一下。夹进物理课本的封底内页。

然后抬起头。

往左前方。第二排靠窗。

她在听课。右手握笔。蓝色圆珠笔在笔记本上写着什么。陈老师在黑板上画了一个曲线运动的示意图——带箭头，标了初速度v₀。她在笔记本上把那个图也画了。曲线画得不太圆——弧度抖了一点。然后她拿橡皮擦掉了，重新画了一道。

马尾在她脖子上晃了一下。

窗外。三月的梧桐树还是光秃秃的。但树枝的关节处冒出了一些绿色的小疙瘩——芽。

春天还没到。但不远了。

顾临风低下头。翻开物理笔记本。在第一页上写了日期——三月二日。

新本子。新教室。新同学。同一个她。

流年还在等。
