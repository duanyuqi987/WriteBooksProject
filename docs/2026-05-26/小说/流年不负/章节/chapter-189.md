# 第189章 抓周

时间：2022年5月
地点：楚州
卷次：卷十：不负少年

五月。女儿一岁了。

客厅地板擦了两遍。茶几挪到了墙角。地上的爬行垫铺开——蓝色的，上面印着字母和动物。

苏晚晴把一圈东西摆在地上。书——一本《唐诗三百首》。笔——一支黑色的中性笔。算盘——她爸从青阳带过来的，木头框子有点松了，珠子拨起来噼里啪啦响。一张百元钞票——新的，纸硬硬的，折痕分明。还有一个小电子琴——赵一鸣送的，一按就响。

顾临风放了一本他的诗集上去——浅蓝色封面的那本。苏晚晴看了一眼。"你放这个干什么。"

"万一她抓了呢。"

"她才一岁。看不懂诗。"

"她妈也看不懂。看了五年。"

苏晚晴拍了他一下。然后笑了。

她把女儿抱过来。放在爬行垫中央。女儿坐着。腿伸在前面。嘴里含着一根手指。看着地上的东西。又看看妈妈。又看看手指。把手指从嘴里拿出来——手指头上沾了口水，亮晶晶的。

"念念。选一个。"

女儿看看妈妈。

低头看看地上的东西。伸手——摸了一下电子琴。琴响了——叮。女儿缩回手。等了一会儿，又伸手按了一下。又叮。她笑了。但她没有拿琴。

她转头。又看看地上的东西。手伸向钱——碰到了。拿起来。捏了捏。然后把钱举起来晃了晃——钱纸哗哗响。她笑了两声。然后把钱扔了。

最后她往前面爬。趴下来——小屁股撅着。手撑着地。爬向那支笔。

黑色的中性笔。

她把它拿起来。握在手里。握笔的姿势不太对——整只手攥着。但她把笔举起来，对着灯看了看——像在辨认一样。然后握着笔往爬行垫上戳。戳了一个小点。然后又戳了一个。

蓝黑色的墨油在垫子上留下了几个小点。

苏晚晴看着女儿的手。然后看顾临风。

"你看她握笔。"

顾临风蹲下来。凑近了看。女儿握着笔的手——小指微微翘着。她写字小指翘起来——和苏晚晴一模一样。

不是遗传。

是模仿。孩子模仿妈妈。

他看了好一会儿。然后笑了。笑得很轻——鼻孔张了一下，嘴唇抿着往两边拉。

"跟你妈一样。写字小指翘起来。"

苏晚晴低头看着自己的手——她握着红笔的手。她批作业的时候。小指果然翘着。她从来不知道自己写字是这个姿势。

她把红笔放下。手指伸直。然后又弯起来。看着自己的小指——指尖有一层薄薄的茧。是十几年握笔磨出来的。

"以前同学笑我。说像古代小姐。我就老想改。"

"没改。"

"改不了。后来就放弃了。"

"不用改。"

女儿还在戳。笔头戳在垫子上——笃笃笃。墨油已经不出了，但她在练——一下一下。很专注。

他走过去把女儿从垫子上抱起来。女儿手里还攥着笔。他把笔从她手上取下来——她很配合地张开了手。手指上沾了一点墨油。蓝黑色的。很小的一个点。

"以后当作家。"

"别。当老师吧。老师稳定。"

"你刚才说当作家。"

"我改主意了。"

"这么快。"

"作家太穷。你写了十几年诗——赚了多少钱。"

他想了一下。"零块钱。"

"所以说。"

她把女儿接过来。往孩子脸上亲了一下。女儿咯咯笑。用手拍她的脸。嘴里的口水滴到了她衣服上。

赵一鸣在旁边拍了张照片——他今天也来了，带了一箱酸奶和一兜子水果。照片里女儿握着笔，小指在发光。

他发了朋友圈。配文:"抓周抓了支笔。以后不是作家就是老师。也可能是诗人——随她爸。"

三分钟后顾临风在底下回了一条:"诗人没钱。"

赵一鸣回:"那你写了十几年图什么。"

顾临风没有回。他把手机放在茶几上。看着女儿在苏晚晴怀里拱来拱去。阳光从阳台照过来。照在地板上那本浅蓝色笔记本上。封皮上反光。

他弯腰把笔记本捡起来。拍了拍灰。放在了书架第二层靠左。

那本旧词典旁边。

以前是一本。后来是两本。现在是三本。

晚上。苏晚晴给女儿洗澡。浴盆里水汽蒙蒙。女儿的头发贴在头皮上——像一层黑绒。她用手捧了些水淋在女儿肩膀上。女儿嘻嘻笑。用小手拍水——水花溅了她一脸。

她抹了一下脸。也跟着笑。

客厅里。顾临风在地上收拾抓周的东西。他把那支笔捡起来——笔头上还留着女儿的牙印。他拿在手里看了看。然后把笔插进了自己的笔筒里。笔筒里有好几支笔——红笔、蓝笔、铅笔、一支笔帽上有个牙印的旧中性笔。

他看了那支旧笔一眼。高中那支。笔芯早就干了。但一直没扔。

他把女儿的牙印笔插进去。和那些旧笔放在一起。
