# 第140章 第三次

时间：2018年4月初

地点：楚州一小门口

卷次：卷七：流年未断

第三次是她约的。

周三晚上发的消息。他改作业改到一半，手机震了。

"周五下午我没有最后一节课。你要是有空——可以来学校接我。"

他放下红笔。屏幕亮了。他看着这段话，打了一行字"几点"。删掉。又打"我在门口等你"。删掉。最后只回了两个字。

"几点。"

"四点半。"

周五下午四点钟。楚州一小。顾临风提前到了四十分钟。本来打算踩点到的——换了衣服，出了门，沿着街走，不小心走快了。到的时候四点还没到。

他就站在校门口等着。

一小的校门跟照片上一样。浅红色的砖墙。铁栅栏。门柱上挂着学校的牌子——铜字，"楚州市第一小学"。校门两侧的银杏树已经绿了——嫩叶长了一层，密密地铺在枝头。四月的阳光已经不刺眼了——温温地照在门卫室的白墙上。门口的地面用灰色方砖铺的——有几块碎角的，中间夹着一道一道很细的草。传达室的大爷戴着老花镜在看报纸。偶尔抬头扫他一眼——看他不像坏人，继续看报。

他靠在门口的银杏树上。树干冰凉。背靠着。手插在口袋里。银灰色的夹克——新买了一件——不是最便宜的但差不远。领口没磨毛。袖口扣子都尚在。鞋上刷了食用油代替鞋油——肉眼不太看得出来。

他看着校门上的铜字。楚州市第一小学。苏晚晴在这里教书——每天从这个门进进出出。从这个门进去，她会穿过操场，上二楼，进三年级办公室，拿粉笔，走进教室——四十五个学生站起来说"老师好"——她点了点头，翻开课本——然后上今天上午的第一节课。那是他站在这儿等她的等待中发生的一切。同一座城市。同一天。同一个午后。不同教室里的两个大人。

他在马路边——她在楼那边。

下课铃响了。很响——电铃，跟青阳一中的是一个型号。他条件反射地站直了——像一个学生。然后意识到自己不是了。笑了笑。

学校开始往外吐学生。都是一年级的，排着队，穿着校服。打打闹闹，嘴巴不停。门卫把门打开——铁栅栏推开发出吱一声。接孩子的家长围了过来——电动车、自行车、还有几个老人。有人喊名字。有人挥手。有人把一个书包挂在车把上。有个小女孩跑到他脚边停下，抬头看了看他——发现不是自己的家长，又跑掉了。

他站在人群旁边。看着这些背着书包的小学生一个一个被接走。人群慢慢散去——校门口又安静了。

传达室的大爷合上报纸，看了他一眼。

"接人。"

"嗯。"

"哪个班的。"

"三年级的老师。"

大爷扶了扶老花镜。"哦——苏老师。"

"您认识。"

"苏老师嘛。教三（1）班和三（2）班的。"大爷点了个头。"挺好的老师。学生喜欢她。"

顾临风点了一下头。大爷没再说话，继续看报纸了。

然后他看见她了。

她从教学楼里出来。背着那个米白色的包——还是那个包。马尾在肩后轻轻晃。手里拿了一个文件夹。先跟几个同事挥手道别——有两个女老师，还有一个男老师。她说了几句什么，然后手指了一下校门口的方向。

她看到他了。等在门口的银杏树下。

停了大概两秒。然后跟同事又说了句什么——同事们都往他这边看了一眼。有人在笑。有人凑到旁边的人耳边说了句什么。然后大家又笑了起来——声音不大，但气氛很轻松。

她往前走。脚步快了一些——但没跑。走到他面前的时候脸上有一个不好意思的笑。

"等很久了。"

"刚到。"

"走吧。"

两个人并着肩往街角走去。

身后的同事还在看。他能感觉到那些目光——有好奇的，有善意的。然后他听见一个女老师说"是相亲认识的"——声音压着，但压不住——压在手指下面的话反而字字清楚。另一个女老师说了句"挺好的"。然后就听不太清楚了——大概是拐进走廊了，谈话变得低沉起来。

她低头看着脚下的路。耳朵尖又红了——跟以前一样。

"她们一直在问。"她说。更像自言自语——让他知道这件事的存在——不过并没有抱怨。

"问什么。"

"'相亲相的怎么样'。一回来就问。"

"你怎么说的。"

她走了两步才回答。"我说——还在互相了解。"她的头低了一点。然后抬头看着他。"你觉得这个回答怎么样。"

"挺好。"

又走了几步。街角的红绿灯。红灯。两个人停在斑马线前面。并排站着。旁边有个老人牵着一条小狗——白色的，毛很长。小狗在他的左脚旁边闻了一下，然后摇了摇尾巴走开了。

"你今天怎么来这么早。"她问。

"下课早。"他顿了一下。"其实是——走快了。"

她嘴角往上弯了一点。没有点破。

绿灯亮了。两个人过了马路。她的脚步刻意慢了半拍——跟他的步幅对齐。他注意到了。也慢了半拍。

然后两个人的步伐就一致了。

街的对面是一排小店。有文具店——门口挂着一串一串的彩色卡纸。有小饭店——炒菜的油烟味飘出来。她对这条路很熟——毕竟天天走。跟他介绍——那家文具店的红笔好写——那家小饭店的鱼香肉丝特别辣——那个水果摊的苹果甜而且便宜——就是在路边站着的时候灰尘比较大。他听着。偶尔应一声。她说话的时候手指比划，马尾一晃一晃。路上的地砖有几块松动的——踩上去水溅在裤脚上——她没有在意，还在讲那个卖苹果的大叔的胡子有一个尖特别长。

经过面包店的时候她停了一下。玻璃柜里的蛋挞刚出炉。她看了蛋挞一眼。又准备走。他说"想吃"——她还没说话，他就进去买了两个。买二送一。手里攥着三个蛋挞出来了。她接过一个。很烫——用两个手指捏着纸托。轮流换手。咬了一口。蛋液很嫩。层酥很脆。碎屑落在她的白色卫衣上——这次不是浅蓝色——大概是换过洗了。她用手指捏起来——碎屑粘在指尖——她没有丢掉——而是放在纸巾上——大概准备带回去喂操场后面的鸟。

两个人捧着蛋挞在路边吃完。嘴角都有酥皮的碎屑。他先擦了。然后从口袋里拿了一张纸巾给她。

她擦了。然后把纸巾捏在手心。

两个人继续往前走。

前面是一个小广场。广场中间有个喷泉——没开。池子里的水很浅，映着天上的云。周围有几条木条凳。她指了一下喷泉旁边的长椅。

"坐一会儿。"

"好。"

坐下。长椅的木头有点旧——木板之间有缝。木条晒了一天太阳——坐上去温温的。旁边的砖地里长了几棵绿色的草。她把手里的纸巾——包着蛋挞碎屑的——轻轻放在脚边一块石头上。等风来，或者等下会有鸟。

他们面前是那个干涸的喷泉。池底的瓷砖有点发黄了。水面的倒影里能看到天上的云——一朵一朵，慢慢往东移。远处有放学的小孩在跑——踢一只空的易拉罐。罐子在地上滚动——咣咣咣，越来越远。

"前两次都是你约的。这次我约的——算扯平了。"

"不扯平也行。"他说。

她看着水池里云的倒影。手指在膝上轻轻敲着——大概在数什么。嘴唇动了一下。好像想说一件很早就想好的事情。

"以后——你约吧。我不擅长这个。"

他没说话。但点了一下头。手不自觉地伸到口袋里——摸到了那张对折了四次的纸巾。上面有她的名字。还有一个她扎出来的小洞。纸已经不白了。折痕处快磨穿了。

她把头靠在长椅的靠背上。眼睛眯起来看着天空。云在动。她的睫毛在下午的阳光下投下很浅的影子。嘴角的弧度——已经不再是礼貌的笑了。是一种很松弛的弧度。是那种在一个人面前不再需要紧张的弧度。

他说"好"。然后两人安静地坐了一会儿。

喷泉的水面映着两个并肩坐着的人影——一个白卫衣的，一个银灰夹克的。影子有点模糊——被水波揉皱了，又展开，又揉皱。风停的时候——水面平得像一块浅蓝玻璃。两个人的倒影清清楚楚地靠在同一条长椅上。
