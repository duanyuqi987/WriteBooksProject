# 第010章 新课本的气味多年后

所属部卷：第一部：初秋三楼
时间：多年后
地点：出租屋书桌

## 诗歌插页 P010：原创仿唐绝句

旧书翻处有晴尘，
一角残膜认早春。
莫把微光夸作命，
人间只是少年人。

多年后，陆允川在出租屋里整理书箱，翻出一张发黄的透明书皮边角。

那东西已经硬了，折痕处有一道白线，摸上去像快裂开的冰。他最初没想起来它从哪儿来，直到看见旁边那本硬皮本，才慢慢记起高一开学周，记起三楼教室，记起程砚那句“黑板今天穿了白裙子”。

他坐在地板上笑了一下。笑不是因为那段日子可笑，而是因为自己当年太郑重。一个人若把所有细节都当成命运，就会活得很累。透明书皮里的一粒灰，别人桌角的一只水杯，自己姓名栏里写歪的一笔，都能被他反复解释，解释到最后，反而看不见那天真正的样子。

真正的样子是：天气很热，班主任很严，程砚的化学书包得歪七扭八，夏知遥也在为新环境发愁。他们每个人都有自己的小麻烦，没有谁专门为了谁发光。

陆允川把那张书皮边角夹回硬皮本，后来又取出来，放进普通文件袋。文件袋上他写了四个字：高中杂物。

写完这四个字，他心里轻了一点。以前他总觉得普通二字委屈了青春，仿佛不把它写得盛大，就说明自己白白疼过。现在他反而感谢普通。普通让人活下来，也让人终于有机会诚实地回头。

窗外有人推电动车经过，刹车声轻轻响了一下。屋里没有新课本的气味，只有纸箱、旧书和夏天快要下雨前的潮气。陆允川给自己倒了一杯水，把硬皮本翻到空白页，写下几行：

我曾把一粒灰，
看成一场雪。
后来雪化了，
桌面还在，
书还在，
人也终于肯从故事里走出来。

他没有把这首小诗写进正文，只留在草稿里。因为有些话写出来，是为了发表；有些话写出来，只是为了让手从旧物上松开。
