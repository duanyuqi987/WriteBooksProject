# 第091章 操场

时间：2014年4月

地点：楚州师院操场

卷次：卷五：各自南北

四月的晚上。操场。

天气暖和了一些。跑道上的跑步人数比冬天多了很多。顾临风还是在走圈。每天晚上九点左右——自习完或是从图书馆出来——走到操场，从第一道开始走。

师院的操场比一中大。一中是煤渣跑道，直道一百米，弯道是碎石子垫的，跑起来硌脚。师院是标准的四百米塑胶跑道，八条红道，白色的分界线画得很直。跑道里有足球场——人工草皮，晚上总有几组人在踢球。看台上偶尔坐着几对情侣，靠得很近，在说着什么。

他走第五道。

第一学期他只走了三圈。这学期走四圈。每圈四百多米。四圈大概十三四分钟。走完不做任何事——拉伸、拉伸腿、深呼吸——都不做。就是停下来，站几秒。然后走出操场。

四月的操场上飘着各种味道。人工草皮有一股胶味。跑道两边的香樟树在开花——一种淡淡的白花香，若有若无。偶尔有夜风把食堂的油烟吹过来——炸鸡排和辣椒炒肉的味道。

他走在第五道上。步子不快。手插在运动裤的口袋里。运动裤是深灰色的，洗得膝盖那块有点发白。

前面有一个女生在跑步——马尾，浅绿色运动服。马尾在跑动时左右晃动。他看着那个马尾——是橘色的灯光下面有节奏地左右摆动。跟中那个马尾很像。但不是。

他低头看脚下的白线。继续走。

走着走着想起高三那年。也是操场。也是走圈。那次是因为看到了她跟一个男生并肩走——后来赵一鸣打听到是她表哥。那次他在操场上走了一个多小时——走到脚底起泡。那次他做了一个决定——高考后立刻表白。

那次他做了一个决定——高考后立刻表白。

那次的决定做得很快。因为目标很清晰——考完、表白、让她知道。就像跑四百米——有起跑线、有终点线、有明确的跑道。但现在不一样了。

现在他不知道自己还能决定什么。

她说"暂时不要联系"。没有说"永远"。但也没有说"什么时候可以"。他等了一年。不知道还要等多久。不知道还能不能等。但他没有别的选项——等是唯一的选项。不等就等于放弃。放弃——他不会。

他停下脚步。站在第五道的弯道上。弯道的时候身体会不自觉地往内侧倾斜——大约是十五度。跑道上的白色分界线在弯道处是弧形的。他站在弧线的中点上。前后都是弧。看不到起点。也看不到终点。

他走到第四圈。速度慢了——第四圈他从来都走得比前三圈慢。操场里的踢球声开始散了——有人在收球，有人在互相道别。看台上的情侣少了一对。还剩一对——两个人坐在路灯照得到的地方，在看手机，肩并着肩。

他完成了第四圈。站在跑道最外道的起点线上。这次没有马上走。深吸了一口气。然后呼出去。

操场旁边有个饮水台。他走过去接了一口水。自来水的水——有股漂白粉的味道。他咕咚咕咚喝了两口。剩下的浇在旁边的草上。

看台上的最后一对情侣也站起来走了。男生拉着女生的手上台阶。女生的头发是直的——到肩膀。她不知道说了一句什么——男生笑了。声音很轻，在夜风里散了。

他看了看表。十点十分。今天多走了一圈。回到宿舍之前他在操场门口站了一会儿。操场的灯还没关——整个跑道在白色灯光下很亮——像一跑道是另一个世界。他转过身，朝宿舍方向走。

经过那个黄色的电话亭。电话亭的灯灭了。大概是灯泡坏了。没人修。亭子在路灯光边缘，玻璃脏了一块——上面有一道白色的鸟粪，已经干了。他在电话亭前停了三秒钟。然后继续走了。
