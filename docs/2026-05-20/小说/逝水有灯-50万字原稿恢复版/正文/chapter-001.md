# 第001章 初秋三楼之前

所属部卷：第一部：初秋三楼
时间：2010年九月
地点：县一中三楼东侧教室

## 诗歌插页 P001：原创现代诗

我把早晨放低，
让一线水从课桌边慢慢经过。
你的名字没有被我喊出，
只在窗外法桐落下的第一片黄叶旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写初秋三楼抵达之前的坡度。报到那天，三楼东侧教室的门一直敞着，风从走廊穿过去，把黑板上残留的粉笔灰吹成淡淡一层。陆允川把行李箱靠在墙边，先低头看鞋尖上的泥，再看讲台上摞着的登记表。

他把新发的校服外套搭在椅背上，又把桌斗里的碎纸一点点捡出来，像这样做就能让自己快些属于这个班。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。

那天真正的事还没有发生，窗外法桐落下的第一片黄叶却先被他记住了。它太小，小到放在任何人的日子里都不显眼，偏偏在陆允川这里有了重量。

“你这人真有意思，刚来就盯着窗外。”程砚说。“我只是看看座位。”陆允川说。“座位在屋里，窗外又不发课本。”程砚说。

程砚把书包往桌上一扔，拉链没拉好，几本练习册哗啦滑出来。他笑得很轻，倒没有恶意，只是把陆允川那点紧张照得无处可放。

叶澄在前排帮老师分表格，经过他们这排时敲了敲桌角，提醒新同学先写家庭住址，别把县城和乡镇填反。

课间十分钟被铃声切得很碎。有人挤去小卖部买面包，有人趴在桌上补觉，有人拿着水杯一路跑向水房。陆允川站在这些声音中间，心里那点紧张没有消失，却被生活稀释了一些。

夏知遥这时还只是人群里的一个名字。夏知遥进门时没有抬头，她抱着两本书，发梢被风吹得贴在脸颊边。她先问班长空座在哪里，声音不高，像怕惊动还没安顿好的早晨。陆允川看见了，却没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子，又一行行划掉。留下来的，是教室里真实的光、真实的灰、真实的不知道该把手放在哪里。

窗框底下积着一线灰，手指摸过去会留下白印。班主任在门口叫名字，叫错一次，教室里就有人低低笑一声。陆允川跟着笑，又很快收住，怕自己笑得不像这个地方的人。

如果说喜欢在这里有了开端，也不是因为命运突然发声，而是因为一个少年开始过分留意普通事物。他还不懂，留意可以是温柔，也可能慢慢变成对自己的消耗。

班主任拿着花名册站在门口，念到陆允川的名字时停了半秒，问他是哪个乡镇来的。他答得很快，答完才发现自己的普通话带着一点方言尾音，耳根立刻热了。

夏知遥坐下后把书放到桌角，又从笔袋里取出一支黑色水笔。她没有加入旁边女生的闲聊，只在表格上慢慢写家长电话，写错一个数字，又安静地涂掉。

午饭时程砚带他去食堂，指给他看哪个窗口米饭少、哪个窗口阿姨手稳。陆允川端着餐盘跟在后面，第一次觉得县城学校也不是传说里的光鲜，它一样有排队、油烟和找不到座位的狼狈。

晚自习前，叶澄把多余的登记表交给老师，回座位时顺手关了半扇窗。风小下来，法桐叶贴着玻璃滑过去，陆允川低头写名字，最后一笔压得很重。

这一章要把脚步放慢，慢到能听见课桌被拖动时的短促摩擦声。事情还没到来，可人物已经在日常里摆好了位置：谁靠窗，谁靠门，谁总替别人递一张纸。

陆允川后来才知道，所谓开始并不是一个明确的按钮，而是很多个微小选择叠在一起。他没有说话、没有靠近、没有确认，可他已经在心里给那个人留出了一块地方。

事情还没有真正发生时，陆允川已经在心里排练了好几种反应。可排练没有用，现实总会把人推到更笨的位置，让他先拿错报到单，再说错一句无关紧要的话。

那天放学，他没有立刻回宿舍，而是在三楼东侧教室外多站了一会儿。风把粉笔灰吹得轻轻晃动，他忽然意识到，自己等待的并不是答案，只是一个能让明天继续期待的理由。

陆允川把报到单拿在手里，先把报到单填完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把报到单填完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她把家庭电话写错一格后安静涂掉，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

三楼东侧教室里最明显的不是心跳，而是粉笔灰被风吹起的轻响。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，食堂铁盘里的番茄炒蛋把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。报到单、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情还没到来，法桐叶已经先进入他的视线。他当时不懂这种提前的留意有多危险，只觉得只要自己足够安静，就不会打扰任何人。

放学后他在三楼东侧教室外多站了一会儿，听见楼下有人喊集合。那一声很普通，却把他从想象里拉出来，让他记起自己还要回宿舍、洗杯子、准备明天的早读。

这一段不靠告白推进，而靠报到单、食堂队伍和晚自习把人稳稳托住。陆允川会尴尬，会饿，也会在夏知遥转身以后继续学着适应这个班。

如果只写疼，这章会很薄；加上报到单、粉笔灰被风吹起的轻响和食堂铁盘里的番茄炒蛋，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

本章最后，事情仍悬在半空。陆允川收好书，跟着人群往外走。楼道里风从窗缝挤进来，把他的校服袖口吹起一点，他下意识按住，像按住一句还没成形的话。
