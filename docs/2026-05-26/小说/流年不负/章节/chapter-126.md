# 第126章 手抖

时间：2018年3月中旬

地点：楚州，顾临风公寓

卷次：卷七：流年未断

周五晚上。离相亲还有十四个小时。

他坐在桌前批最后几本作文。红笔在手里握着，指节发白。手在抖。

他放下笔。把手摊开在桌面上。手指还在微微地颤。

六年没见了。忽然要以相亲这种方式——坐在她对面，点一壶茶，聊工作、聊天气、聊楚州的生活。好像是第一次认识。但又不是。

他站起来去倒了杯水。手握着杯子——水在杯里微微晃动。

喝了一口。把杯子放下。

走到镜子前面。

镜子里的人穿着旧T恤。胡茬长了两天。黑眼圈从眼底蔓延到颧骨。头发有点长——他妈说的对，该理了。

他没见过这个人。

工作两年了。站在讲台上可以眼不眨地讲一节课。批作业可以批到晚上十点不动窝。跟同事吃饭的时候也能正常聊天开玩笑了。表情管理做了两年——在学生面前是温和的顾老师，在同事面前是随和的临风。

但镜子里的这个人——好像不是顾老师，也不是临风。

就像一个失眠了两天的学生。嘴唇有点干。眼神不知道往哪放。

他对着镜子站了很久。

然后去洗手间洗了把脸。水很凉。他用毛巾擦脸——毛巾边缘起毛了，挂在水龙头上方的横杆上。

又回到镜子前。

没什么变化。

他开始翻衣柜。

衣柜在床的侧面。推拉门，右边的门有点卡。他用力拉了两下才拉开。

里面的衣服不多。几件T恤。两件衬衫——一件白的一件浅灰的。两条长裤。一件外套。都是洗了很多次的。衣架是铁丝弯的，外面涂了一层绿色的塑料皮，有些地方已经掉了。

他把白衬衫拿出来。举在身前——镜子只照到胸口，看不到全身。他把衬衫放在床上，对着床头那面小镜子又看了一遍。

太正式了。白的。像是去面试。

他把白衬衫放回去。把浅灰的拿出来。就是那件——他妈说领口都磨毛了的那件。领口确实起了一些绒。袖口的扣子有一颗松了，用白线重新缝过，缝得不太整齐——线头翘在外面。

他把衬衫放回去。手在衣柜里翻了一下。没有别的新衣服。

又把那件浅灰的拿出来。

在床头摊开。用手抚了抚领口。毛绒绒的。他拽了一根——没拽掉。

又抚了两下。

然后把衬衫挂回衣柜。

坐到床边。床垫是房东配的，弹簧有问题——人坐上去会嘎吱响。他挪了一下位置，又嘎吱响了一声。

不知道该穿什么。

不知道该说什么。

不知道该用什么表情推开那扇门。

见面以后第一句话该说什么。是你好。还是好久不见。还是直接叫她的名字——苏晚晴。

她听到他的名字以后停了一下。她说"好"。那她推开茶餐厅的门看到他的时候——会笑。会愣住。还是会点一下头然后坐下。

他站起来。又坐下来。

拿起手机。打开她的照片。看了看。放下。

拿起水杯。喝了一口。杯底剩的一小口水不太多了，水在舌头上有淡淡的铁锈味。

然后他靠在床头。闭上眼睛。

手还在抖。

六年了。他以为自己已经不会这样了。大学的四年他把自己填得很满——上课、看书、写东西。工作以后也忙——备课、上课、改作业、开会。他没有时间想她。

或者说——他让自己没有时间想她。

但现在——周六下午三点。一茶一座。她坐在窗边。他把手机解锁——又看了一遍照片。

马尾。浅蓝色衬衫。平底黑帆布鞋。

他站起来。在屋里走了一圈。经过镜子的时候没敢看。

然后从鞋柜底下翻出了鞋油。已经干透了。盒盖用力撬了两下才打开。里面的油膏裂了口子。他用旧牙刷柄抹了一点上去试试——抹不动。太干了。

他把鞋油丢进垃圾桶里。

转身去厨房。洗了晚间吃完泡面没来得及洗的碗。洗洁精挤多了——泡沫起来一大堆。他冲了很久才冲干净。

把碗放进碗架里。用抹布擦了擦手。

打开手机。搜索：相亲第一次见面穿什么。

搜索结果第一条：干净整洁最重要。

他把手机放下。重新站到衣柜前。

把浅灰衬衫拿出来。对着窗户看了看。窗外是星期六上午才看得见的那棵樱花树，花瓣落得差不多了。

他把衬衫抖了抖。叠好。放在床头。

然后去洗澡。
