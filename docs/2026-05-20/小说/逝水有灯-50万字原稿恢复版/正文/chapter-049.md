# 第049章 元旦晚会旁观

所属部卷：第一部：初秋三楼
时间：高二元旦
地点：学校礼堂后排

## 诗歌插页 P049：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
礼堂后排一张折皱的节目单还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，元旦晚会会显得没有那么沉重。教室、走廊、操场里站着许多人，每个人都有自己的迟到、作业、饭卡和回家路。

叶澄后来记得的不是陆允川的沉默，而是他那天总在看礼堂后排一张折皱的节目单。她觉得那样的眼神很熟，像很多人刚开始喜欢谁时都会有的样子：认真，又有点不讲道理。

叶澄坐在前几排，跟着主持人鼓掌。她回头看了一眼后排，看见陆允川坐得笔直，像被一束他自己想象出来的光照住。

程砚看得更直接。程砚嘴上打趣，灯光暗下去时却没有再说话。他知道有些时刻不能起哄，起哄会把真正的东西弄脏。他不喜欢把话说得太柔，因为太柔的话陆允川听了会继续躲。

“你别把节目单揉碎了，等会儿还要抽奖。”程砚说。“我没用力。”陆允川说。“纸都快替你紧张了。”程砚说。

夏知遥在这一章里也不该只是一个被看的背影。夏知遥站在舞台中央，校服外面套着白色毛衣。灯光照得她眼睛很亮，她握着话筒，开口第一句比陆允川想象中平稳。她有时会因为题做不出皱眉，有时会在朋友讲笑话时笑出声，有时也会因为家里电话而沉默很久。

她没有义务知道陆允川每一次停顿。就像陆允川也不知道她书包里那张没交的申请表，不知道她冬天手指容易冻红，不知道她回宿舍前常在水房多站一会儿。

元旦晚会那晚，礼堂门口挂着红色横幅，胶带贴得不太牢，边角被暖气吹得轻轻掀起。后排座位冷，塑料椅一坐下去就响。

在旁观者眼里，元旦晚会上夏知遥朗诵一首诗，陆允川坐在后排，第一次觉得她的声音比他想象中更稳。这件事只是一天里很短的一折。可短并不等于无意义，它只是提醒陆允川：意义不能只从自己这边生长。

后台有人搬道具，木板磕在地上发闷响。音响偶尔啸叫，主持人笑着圆场。礼堂外卖烤红薯的小车还没走，甜味从门缝飘进来，和舞台上的亮片味混在一起。

叶澄晚自习后对他说，喜欢谁不是错，但别让别人替你承担你没说出口的戏。她说完就去交作业，留下陆允川站在门口，手里捏着一本卷了角的练习册。

礼堂里暖气开得太足，后排同学把校服拉链拉开，椅子腿在地面上磨出刺耳声。主持人念错一个班级，台下笑成一片，紧张才被冲淡了一点。

夏知遥上台前站在幕布边，低头看稿。灯光还没打到她身上，她只是一个准备节目的学生，袖口有一点皱，发卡也没有完全别正。正是这些小地方，让陆允川多年后仍觉得可信。

朗诵到中段，话筒轻轻啸叫了一声。夏知遥停了不到一秒，继续往下念，声音没有抖。陆允川那时第一次明白，他喜欢的也许不是想象出来的柔弱，而是她面对台下几百双眼睛时的稳。

晚会散场后，地上到处是瓜子皮和彩带。程砚弯腰捡到一张节目单，塞给陆允川，说留着吧，反正你刚才已经把自己的揉烂了。陆允川接过来，没有否认。

旁观不是冷漠，而是把故事从一个人的胸口挪开一点。挪开以后，夏知遥有了自己的呼吸，程砚有了自己的烦躁，叶澄也有了不愿明说的担心。

陆允川不喜欢被旁观，因为旁观会让他的深情显得没有那么独一无二。但正是这种不独一无二，让后来的人生有了出路：大家都年轻过，也都误会过，不必把一次笨拙写成终身审判。

从叶澄的角度看，陆允川的心事并不神秘，只是太年轻，所以总以为别人都该看懂。他越沉默，周围人越不知道该不该靠近。

夏知遥也许注意到了一点异常，也许没有。她那天要处理的不是谁的暗恋，而是自己的作业、座位、朋友和回宿舍前必须打完的一通电话。

陆允川把节目单拿在手里，先把节目单摊平，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先好好看节目，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她站在幕布边低头确认稿纸，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

学校礼堂后排里最明显的不是心跳，而是话筒轻轻啸叫后又恢复平稳。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

晚会散场时，礼堂门口烤红薯的甜味把人从心事里拽出来。程砚说再大的事也得先回队伍，陆允川把节目单折好，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。节目单、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

从旁边看，陆允川的心事并不神秘。它只是少年人常有的笨拙：怕被看见，又怕完全不被看见，于是每个动作都带着一点别扭。

夏知遥也许注意到了一点，也许没有。无论哪一种，她都不该被写成答案。她有自己的等朗诵节目上场，也有不想向任何人解释的沉默。

这一段把晚会从光里放回人群。节目会结束，椅子要归位，地上的彩带要被扫走，陆允川也要把她从自己的比喻里还出来。

如果只写疼，这章会很薄；加上节目单、话筒轻轻啸叫后又恢复平稳和礼堂门口烤红薯的甜味，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

这一章的火被旁人的目光压低了。压低以后，才看得见灰里的温度，也看得见每个人不被爱情叙事吞掉的完整生活。
