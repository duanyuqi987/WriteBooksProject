# 第206章 毕业照里的空位之前

所属部卷：第四部：失去的美丽
时间：大四六月
地点：大学图书馆前台阶

## 诗歌插页 P206：原创现代诗

我把松线放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

毕业照九点开拍，八点半图书馆台阶上已经站满了穿学士服的人。六月的太阳升得早，光照在白色台阶上反得人眼睛疼。

陆允川从宿舍走过来花了十分钟。学士服是昨晚从辅导员办公室领的，袖口有点脱线，他出门前找了针线盒，没找着，就算了。

"老陆！这边！"祁野站在第二排台阶上冲他挥手。

陆允川挤过去。周围全是同班同学，有人拿着自拍杆在拍小视频，有人在帮女生别胸针，有人把学士帽反扣在头上扮飞行员。

"你帽子呢？"祁野问。

"手里。"

"戴上啊。"

"急什么。热。"

祁野的帽子已经戴好了，帽穗垂在左眉上方。"辅导员说帽穗在左边，拨到右边才算毕业。你先拨了，省得到时候手忙脚乱。"

"你又提前做功课了。"

"这叫什么功课？这叫常识。"祁野帮旁边一个女生正了正帽子，回头继续说，"对了，学校发了条通知，今天校长只在大合影出现，班级合照是辅导员带队。"

"无所谓。反正也看不清脸。"

"你心态真好。"

"这不叫心态好。这叫四年下来已经不指望学校有什么仪式感了。"

旁边一个同学插话："你们听说没有，隔壁专业今天请了个航拍。"他指了指头顶，"无人机。到时候会从上面洒花瓣。"

"真的假的？"

"真的。他们班干部众筹了三百块。"

祁野仰头看了看天。"我们班怎么没有这种天才？"

"因为我们班长是程砚。"陆允川说。

"对哦。程砚呢？"

"在那儿。"

程砚站在台阶最下面，正跟摄影师沟通站位。他手里拿着一张班级名单，风把纸吹得啪啪响。他按住了，继续比划。

"他比辅导员还忙。"祁野说。

"他喜欢。"

程砚走了上来，额头上已经出汗了。"摄影师说三排。女生坐第一排，男生后面两排，最高的站最后。老陆你站第二排左数第八个。"

"你安排得也太具体了吧。"

"不然呢？上次班级合照拍出来像春运现场，辅导员说这次再不整齐就取消聚餐补贴。"

陆允川只好服从调度。他站到第二排左数第八个位置，把手里的学士帽戴上。帽檐压得有点低，他往上推了推。

袖口那根松线又垂下来了。他捏住想扯断，线没断，指尖反而勒出一道浅印。

"别扯了。"背后有人说，"越扯越长。"

是夏知遥。她站在第三排，比他高一个台阶。

"你什么时候站这儿的？"

"刚才。程砚安排的。第三排右数第四个。"

"那你离我挺近。"

"因为身高差不多。"夏知遥替他看了看那根线，"回去用打火机烧一下就好了。不过现在别烧，把学士服点着了就上新闻了。"

陆允川笑了一声。

摄影师举起喇叭："前排同学蹲低一点！后排的再往中间靠靠！左边第四个男生，别看手机了！"

大家一阵手忙脚乱地调整。前排女生蹲下去的时候，有人裙摆踩到了，有人帽子掉了，有人不知道手该放哪里。

"都站好。"祁野在后面喊，"别一副遗照脸，毕业照也是照。"

"那你笑一个？"旁边人说。

"我笑了啊。你看不见是因为我脸僵。"

"你昨晚又打游戏到几点？"

"三点。"

"牛逼。"

摄影师的喇叭又响了："三——二——一——"

快门声连着响了七八下。有人喊"再来一张"，有人喊"毕业快乐"，有人在快门间隙中偷偷换了个表情。

彩纸从某个方向撒出来，落在台阶上，落在学士服上，落在学士帽上。不知道是谁准备的，但大家都默认了。

"结束了吗？"祁野问。

"结束了。"

"那我们班没有航拍、没有花瓣、没有彩纸——刚才是谁撒的？"

"隔壁班的。"程砚走过来，把班级名单折好放进裤袋，"风吹过来的。"

"牛逼。"祁野又说了这两个字。

大家开始散开，三三两两地下台阶，去找各自的亲朋友好友拍小合照。陆允川站了一会儿，把学士服脱下来叠好，搭在胳膊上。

那根松线还垂着。

夏知遥从后面走上来，递给他一个打火机。"烧一下。别真把衣服点着了。"

"你去哪？"

"找叶澄拍照。她说她在西草坪。"

"去吧。"

陆允川用打火机烧断线头，一股很轻的焦味散在空气里。他把打火机还给夏知遥，她接了就跑下台阶，学士服的下摆在身后扬起来。

他决定今天不给任何人发合照请求。来的人自然会来，不来的人也要毕业。

手机震了一下：班级群发了一条消息，"毕业聚餐晚上六点，北门蜀味轩。"

他回了一个"收到"。

台阶上的人渐渐散尽了。彩纸被风吹到最下面一级台阶，又吹起来，打着旋滚进灌木丛。保洁阿姨提着扫帚过来了，没急着扫，站在台阶下看了一眼满地的彩色碎片，摇了摇头，开始从上面往下扫。

图书馆的钟敲了十点。
