# 第117章 楼下等她

所属部卷：第三部：寒风里的花
时间：大二五月
地点：女生公寓楼下圆石凳

## 诗歌插页 P117：原创仿唐绝句

楼下风迟石凳凉，
一张回执握昏黄。
等来不是归人意，
先照少年自作忙。

陆允川等了一个小时。石凳从温热变凉，花坛里的土有一股潮味。公寓门口女生进进出出，有人拎外卖，有人抱着书，有人经过时看他一眼，很快又移开。

他给苏晚枝发了一条消息：“回执我拿到了，在你楼下。”

十分钟后，消息显示已读。没有回复。

陆允川把手机按灭，又点亮。回执纸被他折成四折，边角发软。他开始后悔自己说在楼下，又不愿意走。走了像赌气，留下又像逼人。

天快黑时，苏晚枝终于从路口跑过来，手里还拿着拖把杆的塑料头，像是从教室里匆忙带出来的。

“你怎么还在？”她喘着气，“我不是说不用等吗？”

这句话不重，却让陆允川脸上热了一下。他站起来，把回执递过去，“我以为你会打电话。”

苏晚枝接过纸，眉心轻轻皱着，“我说如果早。后来一直在打扫，手机放讲台上。”

“我知道。”

“你不知道。”她把声音放低，“你等这么久，我会有压力。”

陆允川握着手机，没有说话。风从楼道口吹出来，带着洗衣粉和晚饭的味道。他突然发现，自己所谓体谅，其实把选择留给了她：如果她感动，他就被证明；如果她不来，她就像亏欠。

“下次别这样了。”苏晚枝说。

他点头，“对不起。”

她没有再责备，只说谢谢他送回执，然后转身进楼。陆允川站在原地，手里空了，心里却没有轻下来。圆石凳就在旁边，冷得很明确。
