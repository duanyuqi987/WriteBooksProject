# 第043章 诗抄本之后

所属部卷：第一部：初秋三楼
时间：高二秋天
地点：教室后排的课桌抽屉

## 诗歌插页 P043：原创仿宋小令

诗过课堂，
墨迹渐已干。
欲把词心藏半页，
却因午后日偏黄。
照见旧铅香。

诗抄本之后，陆允川表面上没有变化。每周周日晚自习之前他把诗交上去，语文老师翻一遍写个"OK"字，然后还给每个人。他把每次批回来的本子放在抽屉里最左边——那不是文件，是他这学期的一揽子情感记录——每一首被抄进去的诗旁边都有注解，注解不光标了日期还标了当天她在教室做的某件小事：她今天用了一根新的圆珠笔、她的物理实验报告放在后窗台上晾。他把这些碎末记在诗底下，像往一本游记里贴便签——但游记的目的地并不在书外。

程砚偶然借他的诗抄本翻了一页——翻到那首余光中《乡愁》时，旁边有一行小字："今天教室后门有人把门摔了，她回头瞥了一眼。"程砚把本子放回去说："你这哪里是诗抄本——你这分明是观测日志。NASA要是把你送去观测外太空，你每天记的笔记上行星的名字旁边都会写着'旁边那颗小行星距她XX光年'。"陆允川说"不是观测日志——是备查资料"。程砚说"行——备查。"

语文老师到第五周检查的时候在陆允川的本上批了一段长的:"抄写认真,注释有所侧述。但诗是写的——注释可以简单一些,让诗自己说话。你的注释是另一首诗,但和原诗放在一起有时候会互相干扰。"那意思就是:你太用心了——用心的程度已经接近不健康了。

晚饭时陆允川碰到了夏知遥——她在食堂排队等汤,排到她自己前面只剩下两个人时他正好排在她后面。她回头看到他说:"呀,你——你也来打汤。""嗯,今天的汤是紫菜蛋花——""对,比较淡。蛋花没散但味道不太浓。"这又是一个新的六字以上的对话——一次随口交谈,持续大约十几秒。打完汤两人各自走回座位,她对他说了声"拜",他说了声"拜"。他端着汤坐下以后,程砚看看他问"你这碗汤到底好喝在哪里——你从头到尾嘴角都在翘",陆允川说"汤淡——但汤里加了蛋花"。

回宿舍熄灯后,陆允川躺在被窝里回想那十几秒的对话——她的声音比以前轻了一点——可能是周一课太多累了;她在排汤时站在地砖线上,她站着的那块铺地砖的灰色跟他们值日那天拖地时拖把的拖过的地砖是同一批——他记得那个光泽。这些细节在别人看来可能是无意义的,但对陆允川来说,对话只是一条线索,线索绑住了他之后一个人的全部解析。

他在诗抄本第五周的诗——陆放翁的诗——旁边又加了一行朱批:"今天在排队打汤的时候说了十几秒。她说汤的蛋花散——但有味道。这句话本身是句子也有味道。"然后他笑了——不是因为她说了什么特别的话,而是因为他抄诗抄了五周,第一次在诗旁边写的她的事是重复她的对话——而不只是记录她的动作。这是一种升级——他的日志本从单向观测进步到了有交互记录。
