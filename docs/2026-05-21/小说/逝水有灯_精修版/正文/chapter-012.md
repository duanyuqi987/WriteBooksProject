# 第012章 理科名单

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：新分班后的理科教室

## 诗歌插页 P012：原创仿唐绝句

名分新旧各一楼，
少年心事上双眸。
当时不解寻常意，
却把一行看作秋。

事情真正发生时，并没有陆允川想象得那么郑重。分班榜贴在公告栏上，陆允川挤在人群里找理科一班名单，从上往下数到第十七个名字时，手指停住了。她在名单上，和他同班。

那个午后，公告栏的玻璃反射着走廊尽头照进来的光。他的名字和她的名字被印在同一张纸的正反两面——他的在正面左下角，她的在正面右上角，中间隔了二十二个名字。这二十二个名字里他认识的不超过五个，但他把每一个都认真读了一遍，像在熟悉通往她名字要经过的路标。

他站的时间不长不短。旁边有人挤过来看名单，胳膊肘碰到他的肋骨，他往旁边让了一步。一个男生大声说"我靠，我跟周涛一个班"，另一个说"惨了惨了，班主任是陈铁面"。这些喜庆或哀嚎的声音和他没有关系，他的耳朵只接收了一个信息——同班。从今天开始，每天都能在同一间教室里看见她。

分完班以后的第一件事是搬课本。原来的课本有些可以沿用，有些因为分班要换新教材。程砚从旧教室抱着一摞搬过来的书跨进新教室的门时差点绊了一跤，书从手上滑出去三本，正好掉在讲台前面。陈老师——新班主任，一个戴银框眼镜的中年男人——正在写黑板上的座位表，头也没回地说："书掉了不要抢，等会儿一起发。"

"不是抢，"程砚弯腰把书捡起来，"是不小心掉的。您看着像抢吗？"
"看着像"陈老师终于回头看了一眼，"先放课桌上，别摞太高。"
程砚抱着书走到后排，一屁股坐下来，对陆允川说："这个班主任不好糊弄。"
"你怎么看出来的？"
"他从我的后脑勺就能判断我掉了三本书，你想想。"

新的座位表贴在讲台上。陆允川的位置在后排靠窗，夏知遥的位置在中前排靠走廊。两个人之间隔着四排课桌、一个过道和一扇半开的窗。这个距离既不是最近也谈不上最远——他不是坐在她后面可以直接传作业的位置，他得站起来走过去，或者等别人把东西传到他手上。

第一堂课是陈老师的数学课。陈老师在黑板上画了一个坐标系，开始讲函数图像。他写字的速度很快，粉笔在黑板上一路划过去，声音像枯枝踩在干燥的泥土上。陆允川抄了五分钟的笔记，发现自己抄错了一个函数表达式，f(x)写成了f(y)。他拉了一条横线重新写，心里想的却是——她刚才回答陈老师提问时说"定义域是全体实数"，语气很肯定，没有一丝拖泥带水的犹豫。陈老师点了点头说"好"，然后继续往下讲。那一个"好"字仿佛也落到了陆允川心里。

下课后他去水房接水。水房在三楼楼梯拐角，水龙头老旧，拧开时铁锈味混着水管里的气喷出来。他把杯子伸到水流下，接满了，关掉水龙头。转身时差点撞上程砚——程砚也来打水，手里的矿泉水瓶刚才被他捏扁了，现在正鼓着气放到嘴边吹圆。"你打水打了三分钟，锅都快烧干了？"程砚问。陆允川说"杯子大"。程砚看了一眼他手里那个普通大小的水杯，没戳破。

午饭时食堂今天供应的是红烧茄子和番茄炒蛋。程砚打了两个菜堆得满满的，陆允川打了一个荤菜，坐下以后把肉菜推给程砚。"你今天怎么又吃这么少？""不饿。""你不饿，你这种人就是靠光合作用活着的。可问题是——"程砚把一块红烧肉放到陆允川碗里，"你今天也没怎么见光啊，一直坐在后排。"陆允川把肉夹起来吃了，嚼着嚼着问了一个不相关的问题："你觉得陈老师严不严？""现在才问？你不是应该说'陈老师讲得真好'——哦不对，你刚才在看她吧，没听讲。"

旁边桌坐了两个女生，一个是夏知遥班里的新同学，另一个是原班的熟人。她们在讨论新班级的桌位安排，其中一个说："我坐第四排靠窗，旁边那个女生好安静，一节课没跟我说一句话。""谁啊？""姓夏的，个子跟我差不多，头发扎起来那个。""夏知遥？她本来就这样，熟了以后才会跟你讲话。"陆允川端着汤碗挡住自己的脸，一口一口地喝，听到这里汤已经凉了。

晚自习时新教室的日光灯管嗡嗡响，一排四个灯管有一个不停地在闪。前排有人站起来用扫帚柄捅了一下灯管，闪了几下终于亮了。陆允川翻开物理课本，公式在纸面上排得整整齐齐，牛顿第一定律的"匀速直线运动"几个字下面被他用铅笔划了一道横线。他告诉自己要认真复习，可目光还是会不自觉地往那个方向移动——移动的幅度很小，像指南针被旁边的磁铁轻微偏了一下。

放学前，陈老师把每个人叫到讲台前签分班确认表。陆允川站在门口排队，前面三个人的名字他在听到的瞬间就忘了，直到轮到他——他拿起笔，找到自己的名字，在表格最后一栏签上"陆允川"。签完以后他下意识往下扫，看到她的名字——她已经签过了，字迹和开学那天在扉页上写的一模一样。

他把笔帽盖好，还给前排的同学，回到座位上收拾书包。走之前最后看了一眼教室——明天在这里，她会坐在同一个位置，他也会坐在同一个位置。这比所有的心跳都更具体。
