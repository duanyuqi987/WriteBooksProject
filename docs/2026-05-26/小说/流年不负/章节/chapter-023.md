# 第023章 她的声音

时间：2011年4月中旬

地点：理科三班教室

卷次：卷二：窗边人影

语文老师姓陈，叫陈维中。五十来岁。头发花白，戴老花镜。他教了三十年书，说话慢。

他的课大家比较安静。不是怕他——是他声音有一种沉的感觉。让人觉得吵不太好意思。

今天学朱自清的《荷塘月色》。

陈老师先让学生们自己看一遍。教室里安静了大概十分钟。只有翻书的声音。

然后他抬起头。

"谁来读一下。"

没人举手。

陈老师扫了一圈。

"苏晚晴。你来读。"

顾临风的手放在桌上。停住了。

苏晚晴站起来。

她把书端起来。清了清嗓子。

"这几天心里颇不宁静——"

声音不大。但教室很安静。每个字都听得很清楚。

她咬字很准。"颇"字的声调往上扬，然后"不宁静"三个字落下来。不是那种拖长腔的朗读方式——就是普通的念出来。但每个字之间有节奏。

顾临风没看课本。他看着她的后背。

她站着。书端在胸前。后背挺直。马尾垂在后领口。

"今晚在院子里坐着乘凉——忽然想起日日走过的荷塘——在这满月的光里——总该另有一番样子吧——"

她的声音不尖。不细。带一点点沙。像秋天踩在干树叶上的声音。像雨滴落在石板上。

教室里只有她的声音。风扇嗡嗡响。窗外有鸟叫。

她读到"曲曲折折的荷塘上面"的时候停了一下。换了口气。然后继续。

"弥望的是田田的叶子——叶子出水很高——像——"

她读到这里的时候咬了一下舌头。很小的失误。自己笑了一下。然后继续读。

那个笑极短。声音来了一下就没了。

顾临风看到了她笑的时候肩膀动了一下。

她继续读。读得越来越稳。"月光如流水一般——静静地泻在这一片叶子和花上——"

顾临风第一次这么近地听她的声音。

他知道这不是对他说的。她只是在读课文。老师点到的。谁都可能被点到。

但他还是听着。

像在听一首歌。

她读完了。教室安静了两秒。陈老师点了点头。

"读得不错。请坐。"

苏晚晴坐下了。她把书放回桌上。同桌——一个圆脸女生——凑过去跟她说了什么。她摇了摇头。

顾临风把手放回桌上。

他发现自己的手指不知道什么时候在桌面上画了一横。用指甲。画了一道。不深。

他用手抹掉了。

下课之后赵一鸣过来找他去小卖部。

"你刚才走神了吧。"

"什么。"

"陈老师让你回答问题。叫了两遍你都没听见。"

顾临风愣了一下。

"后来呢。"

"后来他叫了别人。你是不是昨晚没睡好。"

"可能吧。"

赵一鸣没追问。

走到楼梯口的时候赵一鸣回头看他。

"对了——刚才苏晚晴读课文读得挺好的。你听到了吗。"

顾临风下了两级台阶。

"听到了。"

"声音挺好听的。"赵一鸣说。

"嗯。"

顾临风没再说别的。

下午自习课。他在草稿纸上写了几行字。朱自清原句后面跟着他自己的字——"声音如月影/洒在课桌一角/我在后排听着/阳光从窗外漏进来"。

写完了。

看了一遍。

把草稿纸揉成团。塞进桌洞。

晚上回到宿舍。赵一鸣已经趴在床上翻漫画书了。

顾临风把语文课本翻到《荷塘月色》那一页。在课文空白的地方写了四个字——那个沙哑的声音。

写完合上书。

月光从窗帘缝漏进来。跟课文里写的一样。
