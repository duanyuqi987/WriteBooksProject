# 第145章 第二本

时间：2012年11月15日，下午

地点：江城大学教育学院教学楼门口

卷次：第六卷：雪落无声

十一月十五号是苏安晴的生日。

林玉起提前两周就开始准备了。他买了一个新的本子——比第一本大一倍，封面是布面的，深灰色，摸上去有一点粗糙的质感。本子的扉页上他写了四个字："给苏安晴。"

然后他用那支英雄钢笔，把这两年来他写过的所有诗重新抄了一遍。不是全部——他筛选过。删掉了那些太幼稚的、太晦涩的、写的时候连自己都不知道在写什么的。留下了四十五首。

四十五首。每首之间隔了两页空白。他算过的——加上空白，本子刚好写完三分之二。剩下的三分之一，他在最后一首诗后面写了一行字：

"剩下的留到以后再写。"

这两周他每天晚上熄灯之后都在抄。郭向阳有一次半夜起来上厕所，看见他打着手电筒趴在桌上写字，说了一句"你疯了"，就又倒下去睡了。

十一月十五号。下午没课。他在教育学院教学楼门口等。

苏安晴出来的时候身边跟着宋乔。宋乔看见林玉起，用手肘捅了一下苏安晴。

"那个中文系的又来了。"

苏安晴拍了宋乔一下。宋乔笑着跑了。

"生日快乐。"林玉起把本子递过去。

苏安晴接过来。她看了一眼封面，然后翻开第一页。扉页上那四个字——"给苏安晴"——是钢笔写的，笔锋比圆珠笔好看，每一笔每一划都像用了力气的。

她抬头看了林玉起一眼。没有说谢谢。继续翻。

一页一页地翻。

在教学楼的门口，人来人往，有人打招呼说"苏安晴生日快乐"，她也只是应一声，眼睛没有离开本子。

翻到第十首的时候，她的手停住了。那一首的题目是《银河》——就是支教的时候写的那首。旁边有一行小字："7月10日，李家沟。她指给我看银河。"

翻到第二十首。她的速度慢了下来。不是跳着看，是在读。

翻到第三十首。她的手指放在纸页的右下角，迟迟没有翻页。

翻到第四十首。她把本子合上了。

"你是不是把这两年写的都抄进去了。"

"大部分。删了一些不好的。"

"抄了多久？"

"两周。晚上。"

苏安晴把手放在封面上。深灰色的布面，衬得她的手很白。

"比第一本厚。"

"嗯。"

她低头看着本子，沉默了几秒钟。教学楼里的下课铃响了——但在这一刻听起来很远，像是隔了一层什么。

"林玉起。"

"嗯。"

"你以后不要这样了。"

林玉起心里一沉。

"你抄了两周，"她说，"天天熬夜。你大三了。明年要毕业。你该把时间花在更重要的事情上。"

她说这些话的时候很认真。语气不像责备，像一个老师在劝一个学生。

"——但是，"她顿了一下，"这本诗集我很喜欢。非常喜欢。"

她把本子抱在胸前，像那天在图书馆里抱着书一样。

"谢谢。"她说。

这两个字说得很轻。但她是看着他的眼睛说的。

林玉起站在原地。十一月的风吹过来，凉得有点刺骨。但他的手是热的——刚刚递本子的时候她的手指碰了一下他的手背。

"明年你生日我还送。"

"再说。"

她朝他摆了一下手，转身走进了教学楼。

本子抱在胸前，没放进包里。

>
四十五首诗，两千多个夜。
抄尽少年意，不知已被阅。
>
