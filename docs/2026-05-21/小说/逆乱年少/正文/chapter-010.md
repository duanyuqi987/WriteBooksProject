# 第010章 敬老院

时间：2010年10月下旬，周六上午

地点：江城大学南门外，福寿康敬老院

卷次：第一卷：九月的新校门

青协的大巴车停在福寿康敬老院门口的时候，院墙上的爬山虎正红了一半。

林玉起不是青协的人。他跟来的理由是"文学社布置了一篇生活素材的采风作业"。他在大巴车上跟青协的带队学姐赵书妍说这句话的时候，赵书妍看了他一眼。"文学社的？何南星那个社？"

"嗯。"

"那你坐后排吧，别影响我们点名。"

事实上这趟行程跟何南星一点关系都没有。文学社这周没有作业。林玉起是周三路过青协活动室的时候，从门口贴的A4纸上看到"周六敬老院，8:00南门集合"这几个字的。他想起了高中时候爷爷住的那个养老院，走廊里永远是消毒水的气味。然后他想起了苏安晴。

两个关联其实没什么逻辑。但他还是来了。

敬老院不大。三层楼，白瓷砖外墙，窗户上贴着红色的窗花。院子中间有一棵槐树，树下一张石桌，四个石凳。苏安晴正在石桌旁蹲着，跟一个坐轮椅的老太太说话。

"李奶奶，您别动，我给您剪。"

她在给老人剪指甲。左手托着老人的手指，右手拿着指甲刀，剪一下，挫一下，动作慢而稳。老人絮絮叨叨说着什么，听不太清，苏安晴偶尔点头，偶尔应一句"嗯"，偶尔停下来看老人的表情，确认自己没有剪到肉。

林玉起站在槐树的另一边。他手里拿着一个本子和一支笔，站了三分钟，一个字也没写。

"你是来写素材的？"

声音从背后传来。是宋乔。她端着一盆热水从楼里出来，胳膊上搭了两条毛巾。

"嗯。"林玉起举了举手里的本子。

"那你得写写安晴。"宋乔朝石桌那边努了努嘴，"每次来都是她剪指甲。别人嫌老人指甲脏，她不嫌。"

"每——次？"

"青协两周来一趟。她一次没落过。"

林玉起没接话。他在想两周前的事了。院运会那天。她跑步的时候。再往前，图书馆，食堂——

"你把本子拿反了。"宋乔说完就走了。

林玉起低头看了一眼。没有拿反。宋乔在骗他。但这个骗法让他觉得自己被看穿了，整个人不太自在。

他把本子翻到空白页，开始写：

>
> 你把老人的手托在掌心里
> 像托一片秋天的叶子
> 剪掉的指甲落在地上
> 长成一排不说话的句号
> 每一句都在说我来了
> 每一句都在说我会走

写完他站了一会儿，把本子合上了。

十点半，赵书妍组织大家给老人表演节目。有个叫李子豪的男生唱了一首《月亮代表我的心》，跑了三个调。宋乔说谁教你这么唱的，李子豪说KTV教的。

苏安晴坐在最后一排，膝盖上趴着一只橘猫——敬老院里养的，据说叫"大黄"。大黄平时不亲人，专门挑远道来的志愿者挠。但今天它趴在苏安晴膝盖上，眯着眼睛打呼噜。

苏安晴用一根手指慢慢挠着大黄耳朵后面那块。

林玉起坐在离她三排远的地方。他想，如果自己是只猫就好了。

不对。猫不会写诗。

回程的大巴上，苏安晴坐在最前面睡着了。头靠在窗户上，玻璃上凝了一层薄薄的水雾。车内有人在小声聊天，有人在吃饼干，有人戴着耳机听歌。林玉起坐在最后一排刚才来时的位置，看窗外倒退的行道树。

赵书妍经过他旁边的时候停了一下。

"素材够了吗？"

"够了。"

"下次可以直接报名，不用编采风作业。"

林玉起没有否认。
