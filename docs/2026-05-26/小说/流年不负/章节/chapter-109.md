# 第109章 大四

时间：2016年9月

地点：楚州师院

卷次：卷六：平行轨道

二零一六年九月初。大四。

开学的第一天。顾临风走进校门。校门口的横幅换了新的——"热烈欢迎2016级新生"。红底白字。他低头看了一下自己的脚——四年前也是这双帆布鞋走过的路。鞋是新的、旧的一直换了四双。现在脚上穿的是第四双——鞋底磨平了，走路有点滑，鞋面也发白了。路还是那条路。水泥地上一道道的伸缩缝。缝里长了一小丛草。

512宿舍。四个床位三个满了。靠门的上铺空着——阿坤彻底搬走了。床板上光秃秃的，木板的纹理一道一道的，灰落在上面，积了薄薄一层。李博在床上贴了一张纸条："此铺待租，月租零元，包热水。"纸条的边角卷了。没人来。热水房的水经常不够热。

小伍在翻手机。"阿坤说他找到工作了——在老家一个公司做文员。一个月两千八。包吃住。"他把手机屏幕转向大家——是一张阿坤穿着白衬衫、坐在格子间工位上的照片。背后是一面浅灰色的隔板。

"挺好。"顾临风说。

李博从上面探出头来。头发乱着——刚醒。"兄弟——你想好了吗。考研还是找工作。"

"考研。"

"哪个学校。"

"楚州师大。"

李博愣了一下。从被窝里完全坐起来，被子滑到腰上。"楚州师大？不是——你成绩还可以。上学期古代文学期末考了前三。干嘛不报个更好的。省师院——华东师大——南师大——"

"离家近。"

李博看着。张了张嘴。然后嘴巴闭上了。又张开。又闭上。最后说——"行吧。你自己的路。"说完把被子拉回去，躺下了。过了一会儿又补了一句——"不过老顾——你知道楚州师大旁边就是楚州一小吧。"

"知道。"

"就上次许若琳发朋友圈、苏晚晴实习的那个小学。"

"知道。"

李博没再说话了。翻了个身。床板吱嘎了两下。

顾临风打开电脑。笔记本电脑是他去年暑假打工买的——戴尔的旧款，银灰色外壳。屏幕亮度已经不太好了——最亮的那一档相当于别人电脑的中档。键盘上的几个字母磨掉了——"A"键和"S"键只剩下隐约的凹陷。电脑的风扇在低速运转——嗡嗡嗡的，像远处有人在用吸尘器。

他打开浏览器。网速不快——无线网是宿舍楼共用的，下午这个时段上网人多。页面慢慢加载出来——先是标题栏的文字，然后是导航条，最后是主体内容。研究生招生信息网。

登录。注册信息。学信网跳转。验证码刷了三次——第二次输错了。

姓名。身份证号。本科院校——楚州师范学院。毕业时间——2017年7月。

报考院校。

他在搜索框里打入"楚州师范大学"。输入法自动联想——"楚州师范大学研究生院""楚州师范大学文学院""楚州师范大学分数线"。他选了第一条。

页面跳转。学校的招生简章在最上面。红色标题。下面是一行小字——"楚州师范大学研究生院欢迎您。"背景是一张图书馆的照片——红木书桌，落地窗。他没见过这个图书馆——从没进去过。

他往下翻。专业目录。文学院。中国古代文学。研究方向：先秦两汉文学、唐宋文学、元明清文学。招生人数：6人。推免占了两个名额。实际统考招4个人。

他把鼠标移到"填报"按钮上。光标从箭头变成了小手。手心有点出汗——指尖在鼠标左键上滑了一下。

窗外有人在打篮球。声音很响——球撞篮板、落地、弹起来。球鞋在水泥地上摩擦——吱——吱——。有人在喊"好球——"。还有人在喊"传球——传球——眼瞎——"

他没有马上填。

他在想一个问题。

为什么要考这个学校。

楚州师大在省内的确不是最好的。中文系的师资也不如省师院——省师院有一个教唐宋文学的老教授，博士生导师，上过百家讲坛。华东师大更不用说——中文系全国前五。考楚州师大——可以说是"求稳"。也可以说是"躺平"——选一个不出错、门槛不高的地方。还可以说是"离家近"——青阳到楚州大巴四个半小时，周末可以回家。都是合理理由。都是真话。都——不全是真的。

但他知道不是。他知道自己为什么要填楚州师大。因为她在楚州师大。因为她去年在楚州师大教育学院。因为她现在大概还在那里——或者已经毕业了，在楚州一小教书。因为她的名字跟这所学校是联系在一起的。他填下这六个字的时候——不是填了一所学校。是填了一个坐标。一个能让他靠近她的坐标。不是地理上的靠近——他已经在楚州了。是一个比地理更微妙的靠近——走她走过的路。去她去过食堂。在同一个校园里坐着同一片树荫。研究生的教室在文学院楼，教育学院的楼就在隔壁——中间隔着一排法国梧桐。

但他不会承认。连对自己都不会。他对自己说的是——楚州师大离家近。他对自己说的是——楚州师大考上的概率更大一些。他对自己说的是——楚州是个好城市，他想留在那里。这三句话不能说谎——它们都是真的。只是它们是前三个理由。第四个理由他一个人在夜晚躺在下铺的时候会承认——但他不会在任何其他地方承认。包括在这里。包括在键盘前面。包括在他手指悬在"填报"按钮上方的这两秒钟里。

他不需要看她离校的日期。他不需要查她宿舍的门牌号。他甚至不需要知道她现在还在不在学校里。他只需要把这个名字填上去——"楚州师范大学"——就好像把一条轨道接在另一条轨道的旁边，让两根铁轨在同一座城市里并行。

铁轨不会交汇。设计铁轨的人——本来就没打算让它们交汇。两条铁轨之间永远隔着一段固定的距离。太近了不安全。分叉了也不行。

他会读他的研究生。她会在她的小学教书。他们之间依然隔着那条街——二十分钟车程。依然没有任何联系。依然在朋友圈里互相潜水。他考研的时候她大概在批改三年级学生的日记。他写论文的时候她大概在备《燕子》那篇课文。两件事都不坏。只是不在一起。

但是至少——是同一所学校了。不是同一间教室。不是同一张课桌。不是同一个生活。但——至少是同一所学校。这个名字把他们连在了一起。他考进去的时候——他可以对自己说：这是她的母校。她在这个校园里走了四年。他会在她走过的路上再走三年。脚印叠在脚印上——只是隔着时间。

他点了提交。

屏幕弹出一个确认框。"确定报考楚州师范大学中国古代文学专业吗？"确认框的按钮两个——"确定"和"取消"。"确定"在右边，"取消"在左边。他的手指在鼠标上停了一下。外边的篮球声忽然变得很远。

然后他点了。

确认框消失。页面跳转——"报名信息已提交，请等待现场确认。"绿色的勾。页面底下有一个蓝色的链接——"生成报名信息表"。他点开了。表格在屏幕上展开——"报考单位：楚州师范大学。报考专业：中国古代文学。报考类别：非定向就业。"

他把胳膊放在桌上。手撑着下巴。看着屏幕。屏幕的光照在脸上——把他眼睛里的几根血丝照亮了。窗外又有鸟叫。不是麻雀了——是八哥。黑色的身子，黄的嘴。在窗台上跳了两下。歪着头往窗户里看了一眼。飞走了。

他低头看着键盘。手指放在键面上。没有打字。只是放着。指尖上沾了一点灰——键盘上积的灰。他用拇指把灰弹掉了。

很久。

李博在背后看到了。他从上铺翻下来——床板在他脚下发出一个短促的呻吟。"所以你真的报楚州师大了？"

"嗯。"

"为什么。"

"离家近。"

李博没有追。但他看顾临风的眼神里有几秒的停留——那种"我大概知道为什么，但我不会说出来"的停留。然后他把视线转回自己的电脑。"反正兄弟支持你。"屏幕上弹出来一个考研论坛的帖子——"楚州师大古代文学历年报录比"。李博点进去了。大概是在帮他查。

晚上他躺在床上。新被子。旧的。他妈把高一的被子重新缝了一遍——加了一层棉絮。比原来厚了。厚了但不太匀——有的地方厚，有的地方还是能摸到原来的薄层。他把被子盖到胸口。被子里的棉花味——是青阳家里的味道。手搭在肚子上面。呼吸的时候肚子起伏，手指也跟着动。

新床单。洗过一次。还有洗衣液的味道——薰衣草味的，超市促销时买的，十块钱一瓶。但不是她洗衣服的那个牌子。他高一住宿舍的时候——有一次课间从她背后经过，闻到一阵若有若无的干净棉布味。不是香水——是洗衣液。一种特别的洗衣液。从那之后他跑遍了青阳县城里的三四个超市——在洗衣液区一瓶一瓶地打开闻——从来没有找到过那种味道。后来他放弃了。大概是她妈从外地买的。或者是他自己记错了。但每次闻到跟那味道相近的洁净棉布味——他的鼻子还是会竖起来。

他翻了个身。膝盖撞到墙上。那面墙是承重墙。秋天储存的凉意还在里面。冬天储存的凉意也还在里面。凉凉的——隔着秋裤传到膝盖骨。

他想——现在报了她的学校。然后呢。

见到她怎么办。她在校园里忽然看到他——站在图书馆门口——背着书包——拿着借阅卡。她愣一下。然后呢。她说"你怎么在这儿"怎么办。她说"真巧"怎么办。她说"我快要毕业了——以后不在这里了"怎么办——她大四，他研一。她毕业那天他刚好开学。两个人隔着一个暑假的距离。

他没有这些问题的答案。但他还是报了。不是因为想见她。不是因为要追她。不是因为觉得还有机会。只是因为——他觉得在同一所学校里待着，呼吸同一座城市的空气，看着同一排法国梧桐叶子黄了又绿，已经是一件事了。一件不用见到她就能完成的事。

他已经想好了——他不会去找她。不会刻意去教育学院门口等她下课。不会打听她住哪栋宿舍。不会在她可能去的食堂蹲点。他会做他的研究生。在图书馆里查文献，在文学院写论文，在食堂吃同样的土豆丝。如果某天在校园里偶然碰到了——那就碰到了。如果碰不到——那就不碰到。

他只是想在楚州师大的校园里走一走。走她走过的路。她曾经在图书馆三楼靠窗的位置看过的日落——橘黄色的太阳从法桐树冠后面沉下去。她曾经在哪个食堂吃过午饭——大概是第一食堂，靠教育学院最近。她曾经在哪条林荫道上看书——长长的，两边的悬铃木交叉在一起，夏天长出绿色的叶子，秋天变成金黄色，冬天剩下光秃秃的枝干。

然后他过他的日子——上课、泡图书馆、写论文、代课。她过她的日子——教书、改作业、下班骑车回家。井水不犯河水。两条轨道平行延伸。不交汇。但朝着同一个方向。

他闭上眼睛。被子拉过了肩膀。窗外有风——吹得树梢沙沙响。宿舍里的电扇不转了——秋天了。隔壁有人在打呼噜——隔着一堵墙，呼噜声闷闷的。他睡着了。

梦里有一条很长的走廊。两边的门都关着。门是木头的——刷了浅黄色的漆。门上的号码牌是白底黑字。他在走廊里走，脚步声在墙上弹来弹去。走了很久——走廊还是那一条。门还是那些门。有一扇门虚掩着——里面透出来一线光。他想推门。手伸到一半——梦醒了。窗外是苍白的晨光。手机屏幕上显示六点二十。

新的一天。大四。三个月后考研。
