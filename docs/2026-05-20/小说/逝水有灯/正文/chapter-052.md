# 第052章 高三倒计时

所属部卷：第一部：初秋三楼
时间：高三九月
地点：教室黑板右上角

## 诗歌插页 P052：原创仿唐绝句

粉字高悬二百天，
书声压过旧缠绵。
少年抬眼知时短，
不敢多情误案前。

倒计时写上黑板以后，班里的节奏明显快了。

早读提前十分钟，晚自习延后半小时。黑板右上角的数字每天都会少一。值日生擦黑板时，别的地方可以随便擦，那个数字却总要最后重写，像一枚小小的钉子，把整间教室钉在高三里。

陆允川一开始不习惯。每次抬头看黑板，目光都会先撞上数字，再经过前排。以前他会在夏知遥低头写字时停一停，现在他逼自己继续看题。

夏知遥也变了。她课间很少离开座位，错题本越叠越厚。林曼有时候叫她去接水，她总说等会儿。她的“等会儿”常常变成下一节课。

有一次午休，陆允川去办公室交作业，路过前排，看见夏知遥趴在桌上睡着了。她手边摊着数学卷子，最后一道大题只写了一半，笔还夹在指间。

他放轻脚步走过去，没有停。

这一次，他没有把她睡着的样子写进诗里，也没有把它想成属于自己的秘密。他只是在回座位后，把自己的卷子翻到最后一题，继续算。

程砚看他难得安分，低声说：“高三真有用，治好了你的很多毛病。”

“闭嘴。”

“没全好。”程砚补充。

陆允川拿笔敲了他一下。两个人压低声音笑，笑完又各自低头。教室里只有笔尖摩擦纸面的声音，黑板右上角的数字像一盏冷灯，照着每个人的额头。

那天放学前，值日生把 279 改成 278。陆允川看着少掉的一天，心里没有热血，只有一种很实在的紧迫。时间并没有因为他终于收心而奖励他，它只是继续少下去。
