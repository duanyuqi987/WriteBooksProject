# 第176章 朋友的烟之前

所属部卷：第四部：失去的美丽
时间：大二冬末
地点：宿舍楼后门

## 诗歌插页 P176：原创现代诗

后门的灯坏了一盏，
冷空气贴着墙走。
朋友把烟盒拿出来，
不是为了显得成熟，
只是那一刻，
他也不知道还能怎样陪。

最重的话发出去后，陆允川和苏晚枝真的断了联系。诗社群还在，通知还发，文件还传，但他们不再私聊。

第一天，陆允川还能撑着去吃饭。第二天，他开始沉默。第三天晚上，他坐在宿舍书桌前，把同一页书看了半小时。

祁野合上电脑，“走。”

“去哪儿？”

“后门。”

宿舍楼后门平时少有人走，靠近水房和旧自行车棚。夜里风很冷，洗衣房的排水声断断续续。祁野从口袋里摸出一包烟，抽出一支，又停了停。

“你抽？”

陆允川摇头。

“那我抽一支。”祁野说，“别学。”

他点烟时手挡着风，火苗晃了几下才稳。陆允川站在旁边，看那点红在冷空气里亮起来，又暗下去。

他们一开始都没说话。沉默比劝告更长，也更适合那个晚上。
