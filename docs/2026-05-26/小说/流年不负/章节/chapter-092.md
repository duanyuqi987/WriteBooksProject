# 第092章 支教社团

时间：2014年5月

地点：楚州师院

卷次：卷五：各自南北

五月初。学校公告栏贴了一张海报。公告栏在食堂门口——玻璃罩着，里面有层层叠叠的海报：考研辅导班、英语四六级冲刺、二手自行车转让、失物招领。新贴的这张在最上面——浅蓝色底，黑色字。全省高校支教联盟暑期支教行动招募志愿者。右下角印着联盟标志——一个圆圈，一双手捧着一本书。

顾临风在公告栏前站了好几分钟。手里端着的豆浆已经凉了——塑料袋里的包子也没来得及吃。

海报上说——暑期支教，为期两周，地点在太行山区几个乡镇小学。联盟高校包括楚州师大、楚州师院、楚州理工学院等六校。报名方式——填表、面试、分配。左下角有张分组表格——六所学校的分队编号——楚州师大是第一分队——师院是第三分队——理工学院是第四分队。

苏晚晴在师大。许若琳说过她加了支教社团。如果她参加——他大概知道她在第几分队。他盯着"楚州师大"和"第一分队"看了很久——公告栏玻璃反光——他模糊的脸映在分组表格上方。玻璃上有几道手指印——大概是有人指着海报讨论时留下的。手伸进书包侧兜——摸到一支圆珠笔——拔出来——又插回去——笔帽被他咬过，有一道牙印。

其实他可以想办法调队。可以跟面试官说想去第一分队——编个理由——"有个高中同学在师大"。但他没有。因为见到她该说什么——她说了不要联系。如果他忽然出现在她支教队里——她会怎么想。他会打乱她的节奏——也打乱自己的。当天晚上他在宿舍填了报名表——用的是阿坤的电脑。阿坤去洗澡了——键盘前摆着半碗泡面。他把泡面推到一边——打开报名网页——浅绿色页面——姓名学号专业联系方式。"志愿"栏——"地点无所谓"。"理由"——"想去山里教孩子"。"特长"——"语文"。打得很慢——一个键一个键按——空格键有点涩。阿坤从背后经过——端着泡面，肩上搭着毛巾——"又填什么表呢。""支教。""暑假不回家了？""就两周。"

面试在第二教学楼阶梯教室。面试官是两个大三学长——一男一女。男生胡子刮不干净——下巴有青色印子。女生戴红框眼镜——头发很短。"为什么参加支教。""想做点有用的事。""有什么特长。""语文——会写毛笔字。""对山区孩子有什么了解。""不了解——想去了解。""条件很差——没有热水——没有网络——你怎么办。""不是去度假的。"面试官互看了一眼——女生在表格上写了几个字。

三天后。短信来了——"顾临风同学，你已被录取为暑期支教志愿者。分配分队：第三分队。支教地点：李家沟小学。"

他把短信看了两遍。不是第一分队。第三分队——理工学院的——太行山另一个镇上——离她那个支教点隔了三十多公里盘山公路。

他回复了短信——"收到。确认参加。"——四个字。比十一个站名短——比十二公里短。然后他把手机放在桌上——拿起来又看了一遍——第三分队——李家沟——太行山深处。

晚上阿坤问"支教队分到哪了。""第三分队。李家沟。""远不远。""太行山里面。大概五六个小时的车。""就你一个人去？没有认识的人吗。""没有。"

李博从对面床铺往这边看了一眼。眼镜片反着台灯光——他没有问——但他们都知道顾临风说的"没有认识的人"是跟谁比——第一分队是师大的队伍。

他去洗脸。镜子还是那道斜着的裂痕。毛巾擦脸的时候——水很凉。然后他坐在床边——弯腰系鞋带——系得紧——打好两个结。躺床上——手机压在枕头底下。天花板裂缝还在。第三分队。李家沟。太行山。十二公里。301路。两座山峰——同一个方向——不同的点。

当天晚上他在宿舍的电脑上填了报名表。用的是阿坤的电脑——阿坤去洗澡了，键盘前面摆着半碗没吃完的泡面。他把泡面推到一边。打开报名网页——页面是浅绿色的，表格很长，要填很多项。姓名。学号。专业。联系方式。志愿一栏——"地点无所谓"。理由——"想去山里教孩子"。特长——"语文"。他打得很慢。一个键一个键地按。空格键有点涩——要用力按才弹起来。阿坤从背后经过——端着一碗新泡好的泡面，肩膀上搭着毛巾——"又填什么表呢。"

"支教。"

"暑假不回家了？"

"就两周。"

"你还真是闲不住。"

面试在第二教学楼一间空教室里。教室的桌椅是阶梯式的——第一排最低，最后一排最高。面试官是两个大三的学长——一个男生一个女生。男生的胡子刮得很不干净，下巴有一道青色的印子。女生的头发很短，戴着一副红框眼镜。他们坐在第一排，面前摊着几张报名表。

"你为什么要参加支教。"

"想做点有用的事。"

"你有什么特长。"

"语文。会写毛笔字。"

"对山区孩子有什么了解。"

"不了解。想去了解。"

"如果分配到的学校条件很差——没有热水、没有网络——你怎么办。"

"不是去度假的。"

面试官互相看了一眼。女生在表格上写了几个字。男生的笔在纸上点了一下——大概是在打分。女生说"可以了。结果会发短信通知。"

三天后。短信来了。当时他在食堂吃午饭——西红柿炒蛋和米饭。手机振了一下，屏幕亮起来。他放下筷子，拿起手机。短信来自"楚州师院支教联盟"——"顾临风同学，你已被录取为暑期支教志愿者。分配分队：第三分队。支教地点：李家沟小学。"

第三分队。

不是第一分队。

他把短信看了两遍。然后把手机放在桌上。筷子夹了一颗西红柿——手有点凉。

第三分队——是楚州理工学院的队伍。跟楚州师大不在同一个分队。李家沟小学在太行山另一个镇上——离她那个支教点大概隔了三十多公里的盘山公路。

其实他是想去有她的那队的。

但他没有提。报名的时候他在志愿栏写了"地点无所谓"。面试的时候他没有问分队的事。接到分配通知后他也没有提出调队申请。他回复了短信——"收到。确认参加。"就四个字。

四个字。比十一个站名短。比十二公里短。他把手机放在桌上，然后拿起来又看了一遍短信。第三分队。李家沟。太行山深处。三十多公里的盘山公路——坐小车大概要四五十分钟。比301路还短一点。但301路他没有坐过。这趟盘山公路他大概会坐很多次——接下来的两年暑假，如果还去支教的话。

去不了有她的那队——也没关系。他还是会去。他不是为了她才报支教的。至少——不完全是。

晚上阿坤问了他一句——"支教队分到哪了。"

"第三分队。李家沟。"

"远不远。"

"太行山里面。大概五六个小时的车。"

"就你一个人去？没有认识的人吗。"

"没有。"

李博从对面床铺往这边看了一眼。他的眼镜片反射着台灯的光——两块白色的圆片。他没有说什么。他知道顾临风说的"没有认识的人"是跟谁在比——第一分队是师大的队伍。

他比谁都清楚。

顾临风坐在床边。弯腰系鞋带。鞋带是白色的——有些地方已经变成了灰色。他系好以后站起来。去走廊尽头的水池洗了把脸。水很凉。镜子里自己的脸上挂着水珠。他拿毛巾擦了一下。毛巾是蓝格子的——跟被子一个颜色。

回到宿舍。他从书包里拿出手机。翻到那条短信。第三分队。李家沟。太行山。他又看了一遍。然后按了锁屏键。屏幕黑了。
