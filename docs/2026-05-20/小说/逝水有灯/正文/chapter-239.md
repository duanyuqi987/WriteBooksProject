# 第239章 同学群的婚讯旁观

所属部卷：第五部：逝水有灯
时间：婚讯发出当天
地点：公司茶水间

## 诗歌插页 P239：原创现代诗

发婚讯的人
只是在核对桌数
她不知道
一个旧群的震动
会惊醒谁的下午

董向南的未婚妻沈曼并不是陆允川的同学。婚讯发到群里前，她正坐在婚庆店里核对名单，手机备忘录里写满桌数、烟酒、伴手礼和酒店尾款。董向南问她高中群要不要也发一份，她说发吧，别漏了人。

对她来说，那只是众多待办之一。她不认识夏知遥，也不认识苏晚枝，更不知道陆允川曾经在旧同学群里因为一个名字停顿。她关心的是请柬有没有错字，日期有没有写清，收礼台谁负责。

董向南把请柬发出去后，群里很热闹。沈曼坐在旁边看了一会儿，觉得这些人说话挺亲切，又很快低头改表格。有人提到谁在哪个城市，她没有特别记住。每个人的青春在别人那里，往往只是名单上一行名字。

晚上回家，她和董向南一起算预算。两个人为酒席规格争了几句，又商量怎么接外地来的亲戚。婚礼不是童话，是一连串具体安排。沈曼累得肩膀酸，最后把手机往沙发上一放，说：“明天再看。”

同学群里的热闹就这样退下去。旁观者并不冷漠，只是她有自己的生活要忙。陆允川曾经以为旧名字出现必然有意义，可对发消息的人来说，那只是婚礼流程里一项普通通知。
