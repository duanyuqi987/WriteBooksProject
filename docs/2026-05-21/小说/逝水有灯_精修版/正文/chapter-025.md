# 第025章 操场边的单词多年后

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P025：原创七言小诗

看台旧影各西东，
一页单词带晚风。
年少不知朝暮意，
错将背影作从容。

多年后陆允川重写操场边的单词，先把自己写得很委屈——写到他每天带着单词书去操场其实是为了一个人。他用了一整段写操场夕阳的颜色和跑道上那道被拉得很长的影子，写她手里捏着的那瓶农夫山泉，写自己翻到"abandon"那一页就不动了。写到一半把那一页撕了。撕的时候纸很脆，声音很干脆，像掰断一根干树枝。不是因为写错了，而是因为他终于分清楚了两件事：去操场背单词有用吗？有用，他的高考英语分数里至少十分是操场给的——操场上风大，背单词不容易犯困，这一点是真的。在操场看到她想什么？想很多。但这两件事之间没有因果。背单词不是因为她不出现就背不了，心动也不是因为背了单词就消失了。它们是两条偶尔在同一个位置交汇的平行线——操场给了它们交汇的空间，但他不能把它们拧成一条。

多年后他在手机里翻到一张高一拍的操场照片——是程砚拿他的诺基亚手机拍的，像素很低，画面边缘发糊，跑道颜色偏红，像是加了滤镜。照片里的跑道是暗红的，看台是灰色的，夕阳从西边斜着照过来，在看台台阶上留下阶梯形的光影，最下面那级台阶被光切成了一半亮一半暗。他放大看了看，发现自己当年坐的那个台阶上现在多了一排花盆——学校大概为了美观放的。他笑了一下——当年那个座位现在被一盆吊兰占了，吊兰的叶子垂下来，刚好盖住了他坐过的那个位置。他想，如果吊兰有记忆，大概会替他把那座篮球场后面的残阳重新照一遍——照出他十七岁时在台阶上，手捧一本皱巴巴的单词书，装作在背"permanent"，其实眼睛在追一道五米外慢走的人影。

程砚后来在聚会上又提起操场的事。那是一家中餐馆，包间里的圆桌上摆着转盘，转盘上搁着七八个菜，白酒瓶子已经空了半瓶。桌上的人已经喝了不少，话题也从工作跳到往事。有人说起当年谁暗恋谁的事，程砚忽然放下筷子，说："你们还记得咱们高一那会儿，老陆每天去操场看台上背单词吗？""记得，他不是学霸吗？每天夹着一本单词书风雨无阻。""他不是学霸，他是去那儿看人的。"众人大笑——笑声大得连隔壁包间都安静了一下。有人用筷子指着陆允川，说"你藏得可真深，我们那会儿都以为你想考清华"。陆允川端起酒杯碰了一下程砚的杯子，说"你终于说出去了"。"十年了，早过了保密期。"程砚一饮而尽，放下杯子的时候杯底磕在转盘玻璃上，发出清脆的一声。

叶澄那次没来。她还在外地出差，但在群里发了一段话。她说："操场上的晚自习之前，你们每个人都做过一些不会写在毕业纪念册上的事。老陆是在看台上背单词，你们其他人在干嘛？有人在天台上拿收音机听球赛，有人在水房里面偷偷泡面，有人在走廊镜子上偷偷练表情。那些事情没有任何人给你们点赞，但你们每天都在做。因为不做的话，那天就只剩下作业和考试了。"

这段话被转发给了陆允川。他截了图，存在了"旧档案"文件夹里，旁边另一张图是他在操场上拍的最后一张夕阳。那是高二的某一天——不是高一——他发现自己已经有一周没去操场了，于是在一个不是晚自习前的下午单独走去了一趟。操场上空无一人，看台冷冰冰的，跑道上刮着一层薄薄的灰。他坐在老位置上拍了张照，然后站起来走了。他记得那天没有背任何单词，只是坐在那里坐满了十分钟——像补一个告别仪式，告别的是那个高一的时候每天来这里、以为靠近五米就能靠近一个人的自己。

改完这章之后他下楼散步。路边有一群高中生在等公交车，其中一个男孩一边背单词一边偷瞄旁边站着的女孩——女孩也穿着一件淡蓝色的运动外套，像是校服连帽衫。男孩的目光来回切得很快：单词书看三秒，女孩看一秒，然后低头背，再抬头看。旁边其他人都没注意到，但陆允川注意到了——因为他自己做了太多年了。

他没有笑，也没有觉得亲切。他站在那里看了大概十秒，公交车来了，男孩把单词书往书包里一塞，跑了两步挤上车，女孩也上了同一辆车，两人隔着两个扶手站着。车门关了，尾灯亮起来，公交车慢慢驶出站台。陆允川转身继续往前走。他只是觉得——那个男孩以后也会写一本小说，写一章叫"操场边的单词"，写到一半会撕掉几页，然后在修改稿上写：其实我每天去操场不是为了背单词，我是为了发明一个不离开的理由。等操场不再是唯一的交汇点，那个理由就过期了，但他会把过期的理由收进抽屉里，和旧单词书、褪色的照片、以及一张写着"permanent——永久的"的草稿纸放在一起。
