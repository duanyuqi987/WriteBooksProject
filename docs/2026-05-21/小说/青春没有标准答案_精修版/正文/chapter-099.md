# 第099章 黑板报边角



时间：2010年四月上旬，世博主题黑板报检查前



地点：教室后墙、讲台旁边



卷次：第五卷：未来在黑板上倒计时



2010年四月上旬，世博主题黑板报检查前，教室后墙、讲台旁边，黑板报边角翘起来的一截蓝色卡纸把2010年的春夏推到十班面前。十班做世博主题黑板报，邓桥用从网吧看来的资料补了一小块展馆介绍。

有人嫌他资料太多、耽误复习，邓桥嘴上不服，心里却怕自己只能靠这些显眼。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

邓桥也在这段日子里显出自己的位置。喜欢电脑和资料，理科反应快，爱抬杠，成绩一掉就更想证明自己。群像不是名单，群像是每个人都有自己的难处。

“这个展馆资料我查了半天，不能删。”邓桥说。

“你查资料，不是写百科全书。”宋乔说。

“留三行，重点别散。”常俊凡说。

“我去拿透明胶。”段璟说。

“黑板报也讲取舍，作文更讲。”叶书兰说。

黑板报边角总容易翘，透明胶贴上去，过一会儿又被风掀起一点。

世博专题、英语标语、竞赛名单和暑假倒计时挤在后墙上，像十班这几个月的缩影。

有人觉得黑板报耽误复习，有人觉得后墙不能太难看。争到最后，还是要有人拿尺子压住纸边。

常俊凡负责把文字压缩，周明澈负责看版面，宋乔负责把起哄的人骂回座位。

段璟拿透明胶时，指尖被胶带黏了一下。很小的疼，却让他从走神里回到眼前。

后墙下面铺着旧报纸，粉笔灰、红纸屑和剪坏的边角混在一起。

老师路过时没有立刻评价，只说别贴太满，留白也是版面的一部分。

黑板报边角终于压平时，晚自习铃也响了，所有人又被卷子拉回去。

叶书兰这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

邓桥被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

邓桥从网吧出来的时候把U盘攥在手里。U盘外壳是透明的，里面绿色的电路板看得见。他在网吧待了一个半小时，没打游戏，就查了上海世博的展馆资料——英国馆的蒲公英种子、法国馆的感性城市、中国馆的东方之冠。他把文字复制到Word文档里，图片另存到一个叫"世博资料"的文件夹。

"你查这些干嘛，又不是地理作业。"网吧老板在收银台后面看了他一眼。邓桥没回话，把U盘拔下来，交了押金，走出网吧时外面的太阳刺得他眯了眯眼。

他回到教室时黑板报已经开工了。邱可可站在凳子上，手里拿着粉笔，黑板报的布局线已经用白色粉笔画好。世博专题占了后墙正中间最大的一块，左边是英语标语栏，右边是竞赛名单更新区，最右边是暑假倒计时——现在还是空白，因为还没到倒计时的日子。

"邓桥你查的资料呢？"邱可可从凳子上跳下来，拍了拍手上的粉笔灰。

邓桥把U盘插进教室电脑。电脑是老式的台式机，开机要一分多钟，风扇声音比小卖部的电视还响。Word文档打开的时候格式有点乱，字体变成了宋体——他在网吧用的是WPS，教室电脑上只有Word2003。

"展馆介绍写了这么多？"宋乔凑过来看了一眼屏幕，"黑板报一共就那么点地方，你们打算把中国馆的结构图画上去？"

"我查了半天，不能删。"邓桥说。

"你查了半天资料，不是为了证明你查了半天。是为了让别人看得懂。"宋乔把Word文档往下一拉，资料确实多——光英国馆的介绍就写了三页。

邓桥嘴上要强，但眼睛开始往下看自己的资料。他第一次发现，自己好像真的写太多了。他喜欢电脑，喜欢查资料，理科反应也快，但这些优势一到考试就变成跟不上的分数。黑板报是他少数能显眼的地方——别人做黑板报是完成任务，他是真的想让人看他的东西。

常俊凡走过来，没有看屏幕，先问了邓桥一个问题："你觉得十班同学路过黑板报最多停几秒？"

邓桥想了想。"十几秒吧。""那你的资料要有重点。一个展馆最多三句话。第一句是馆名和位置，第二句是特色，第三句是跟中国的关联。剩下的删掉。"

她没有说"你的资料太长"，也没有说"你写的没用"。她说的是结构——跟你讲题一样，把大问题拆成小步骤。

宋乔负责版面。她把后墙分成了五个区，每个区的边界用蓝色卡纸框起来。卡纸是她从新华书店买的，一毛钱一张，买了五张，花了五毛。她剪卡纸时手指很稳，剪刀沿着直尺走，边缘没有毛边。

周明澈负责整体版面效果。他站在教室中间偏后的位置，双手比了一个取景框："左边太高了，往下降三厘米。""你那是比相机还是比黑板报？"付小川从旁边经过时说。"你管我，有效就行。"周明澈没有放下手，继续在那里比。他看版面的方式跟看地图一样，讲究比例和重心。

段璟被派去拿透明胶。他从小卖部买了三卷——一卷一块五，三卷四块五。回来时看见邓桥正在按常俊凡说的改文字，Word文档从三页缩成了半页。邓桥删掉一行就抬头看一眼屏幕，那个动作让他想起自己在草稿纸上改物理题时的样子——每改一步都先看看对没对齐。

"透明胶。"段璟把三卷胶带放到讲台上。

"三卷太多了。"邱可可说。

"多出来的下次用。反正透明胶永远不够用。"段璟撕了一段透明胶，指尖被胶带边缘黏了一下——那种丝状的痛不重，但刺挠。他把透明胶按在黑板报边角上——就是上次翘起来被邱可可钉了图钉的那个位置。贴完用手掌按了三秒，确定不会翘了才松手。

邓桥把改好的文字打印出来。打印机是学校的旧喷墨，打一张A4纸要半分钟，纸出来时带着微微的热度。他把纸裁成合适的大小，蹲在后墙前面比位置。常俊凡在第三稿的基础上又帮他删了两句话。

"这两句是中国馆的建筑面积和高度。数字很准，但普通读者不需要记住。"常俊凡说。

"那我写室内的展区介绍？""展区太多，挑一个最有特点的写——动态清明上河图。"

邓桥改完第四稿的时候，叶书兰路过教室门口，进来看了看。她教语文，看黑板报的方式跟其他老师不一样——她看的是文字的信息密度和可读性。

"邓桥这几行字是你写的？"她问。邓桥点头。"信息量够了，但读者是你同学，不是来查资料的研究员。加一句个人的话——为什么你喜欢这个展馆。"

邓桥想了大概一分钟。他在中国馆介绍的最后加了一行小字：以后想亲眼去看一次，不是从网上搜的图片。

叶书兰看完了没有评论，只说了句"留白也是版面的一部分，别贴太满"。然后走了。

何南星用尺子压住新贴的透明胶边角。他现在在后排，黑板报就在他左手边，起身就能碰到。"你以前是不是怕别人觉得你不行？"何南星问邓桥。"是。""现在呢？"邓桥没有回答。他把透明胶在手指上绕了一圈，黏住又撕开，黏住又撕开，最后把胶带按在蓝色卡纸边上，按了五秒钟。

树莓摊旁边的小卖部在这个下午格外热闹。魏然和曹蔚去买了饮料，回来时在教室门口说外面有个玉树捐款箱被风吹倒了，有人捡起来重新放好。郭向阳听见了，跑出去看了一眼，回来时口袋里多了两块钱硬币——他要放进去。

补习班楼道里陆续传出下课的声音。高一的背着书包涌出来，空气里弥漫着汗味和圆珠笔油的味道。有个男生在楼道里喊"物理作业借我抄一下"，被他旁边的女生拍了脑袋。

段璟站在后墙前，看着黑板报边角——这次贴得特别平，透明胶没有翘起，蓝色卡纸的边跟白墙对齐。邓桥写的中国馆介绍在最下方，最后一行是那行小字。字不大，站着要低头才能看清。但段璟觉得那句话可能比上面所有的数据都重要。

"你把网吧的资料全用上了？"段璟问。

"差不多用了十分之一。剩下的都在U盘里。"邓桥把U盘从电脑上拔下来，透明的外壳在手指间转了一下。"那剩下十分之九怎么办？""留着。将来去上海的时候用得上。"

他说这句话时没有看任何人，只是把U盘塞回校服口袋。那个口袋的位置跟段璟装饭卡的位置一样——左边，靠近心脏但不在心脏上面。

教室里黑板报做完了，但后墙底下的旧报纸还没收。邱可可蹲在地上捡粉笔头——白的归白的，彩色的归彩色的，她分了两个粉笔盒。余小满拿扫帚扫纸屑，扫到一半被一股风从后窗吹回来，纸屑又散了一地。

"窗户谁开的？"余小满喊。

"我开的，太闷了。"何南星在后排举了下手。

"你开的你过来扫。"

何南星过来了。他没有拿扫帚——他把窗关了。风停了，纸屑落回地上，余小满扫完了。

邓桥收拾完电脑后没有立刻走，他从桌洞里摸出一个笔记本。笔记本是硬壳的，封面贴着一张小标签：电脑知识。他翻到中间一页，上面手抄着HTML标签——`<html>`、`<head>`、`<body>`、`<p>`——每个标签旁边用铅笔写了中文解释。这是他寒假在网吧抄的，网吧老板看他天天来查"学习资料"，最后告诉他其实可以查网页源代码。

"你以后想干什么？"付小川不知什么时候站到了他旁边。

"不知道。也许做跟电脑有关的。"

"那就去学。封丘的网吧教不了你太多，但你能在网上找到教程。"

付小川说话不绕。他的关心方式跟常俊凡不同——常俊凡是把问题拆开给你看结构，付小川是直接告诉你下一步该怎么做。两个人的共同点是都不说废话。

"你家里同意？"付小川又问。

"还没跟他们说。我说了他们也听不懂。"邓桥把笔记本合上。

"那你先做给他们看。U盘里存的资料、做的Word文档、查的网页源代码——这些都是证据。有了证据再开口。"

邓桥点了点头。他把笔记本塞回桌洞时，书堆从中间塌了一小块——练习册、化学卷、草稿纸，垮下来的时候扬起一股纸味。

宋乔用剩下的蓝色卡纸做了一个小标签，贴在后墙错题区旁边。标签上用白粉笔写着：本次黑板报资料提供——邓桥（网吧查询）。

"你这个是什么？"邓桥看见了。

"证据。"宋乔说，语气跟平时一样冲，但标签的字写得比平时工整。

付小川在旁边看了一眼那个标签，没说话。他的表情看不出赞成还是反对——但如果他反对，他会直接说。

邱可可把黑板报剩下的透明胶收进讲台抽屉，粉笔盒盖上盖子。她回头看了一眼后墙。黑板报的主题是"世博来了"，中国馆的简笔画占了右上角，邓桥写的展馆介绍占了左下角，中间是十班同学写的"想去的城市"小纸条——郭向阳写了上海，秦佳禾写了北京，段璟写了一个"还没想好"，常俊凡没有写。

"常俊凡你为什么没写？"邱可可问。

"想去的城市要根据分数定。"常俊凡说，语气很平常。她没有写纸条，但她在纸条旁边画了一个小箭头，箭头指向了竞赛名单那一栏——意思是答案在那里。

郭向阳把他的上海纸条贴在了邓桥的展馆介绍正上方，两张纸之间只隔了一厘米。郭向阳的地图知识全用在了地理课上，邓桥的电脑知识还没用上——但两个人都在后墙上留了一个位置。段璟看着这个画面，忽然觉得郭向阳的上海地图和邓桥的展馆介绍其实是同一件事：把远方拿到眼前，先看一遍，再想怎么去。

晚自习前，段璟去水房接水。水房排了三个人，常俊凡在最前面。她拧开水龙头，把暖水瓶灌满，铝制瓶盖放在水泥台上。段璟在她后面一个位置接凉水。

"你竞赛的事找辅导老师谈了吗？"段璟问。

"上星期天谈过了。老师说名额可以预留，但我要在期中考试前把最后一轮初选卷子做完。"

"能做完吗？"

"能做多少做多少。"常俊凡拧紧瓶盖，"辅导老师说了一句有意思的话——他说竞赛不是看谁全对，是看谁把会的全做对。"

段璟把这句话在脑子里转了两圈。"这不就是你跟邓桥说的吗——留三行，重点别散。"

"差不多。"常俊凡嘴角有一个很淡的弧度，不太像笑，更像一种确认——确认这句话被听进去了。

晚自习前，黑板报检查的老师来了。没有给分，只说"排版比上期好，展馆介绍有看点"。十班教室里没有欢呼，也没有人回头看邓桥。邓桥在自己的座位上低着头翻化学卷——他化学作业还有一道大题没做完。

但邱可可走到后墙前，把黑板报右下角空着的那一小块补上了三个字：制作组。她在三个字下面用铅笔写了参与人的名字缩写，邓桥的缩写排在第一个。

常俊凡没有在制作组里写自己的名字。她用湿抹布擦掉手上沾的蓝色卡纸屑时，段璟看见她手指侧面有一小片干了的胶水印。那个胶水印很淡，不仔细看看不出来。

"你的手。"段璟说。

常俊凡低头看了一眼。"没事，洗一下就好了。"

她走向水房，段璟没有跟。他回到自己的座位，打开物理错题本——今天是Day 5，选择填空稳定在十九分钟以内，错题率降到每套两道以下。

晚自习开始后，教室里安静下来。黑板报上的透明胶终于没有再翘起来。邓桥在化学卷上做完了最后一道大题。做的过程中他停了两次笔，不是因为不会，是在检查有没有跳步。

放学铃响时，段璟从后门走出去，路过黑板报。蓝色卡纸的边缘在日光灯下泛着微光，邓桥那行小字在最后面——不仔细看会被当成日期。

他走出教室时回头看了一眼。黑板报还在，透明胶还在，灯也还在。那些东西都不会上红榜，但会留在一个春天的晚上。

黑板报边角之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

黑板报边角被贴平后，段璟明白每个人都在用自己的办法争取被看见。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，黑板报边角翘起来的一截蓝色卡纸还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
