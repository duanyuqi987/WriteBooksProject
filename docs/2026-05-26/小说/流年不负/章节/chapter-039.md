# 第039章 新学期

时间：2011年9月初

地点：理科三班新教室

卷次：卷二：窗边人影

新学期。新教室。新老师。

教室还是原来那间——高一三的教室。但门牌换了。原来是"高一（三）"，现在换成"高二（三）"。字的颜色从蓝色变成了红色。别的没变。窗外的银杏还是那棵银杏。只是叶缘多了一圈淡淡的黄边。

班主任换了。原来王老师不带理科班，调去带文科班了。新班主任姓刘，三十岁出头，女的，短头发，说话快。她在黑板上写了"高二（三）"四个大字，然后转身。

"我是你们的班主任。我教英语。高二难了。不是高一那种难度。高考从高二开始。每一分都要算。"

然后发课本。高二的课本比高一厚了三分之一。数学多了立体几何。物理多了电场。英语多了两单元。每本都不薄。

课表也换了。一天九节课——比高一多了一节。晚自习延长了半小时。到十点十分。高二没有音乐课、没有美术课。体育一周只排一节。

赵一鸣在一班。教室在二楼。隔了一层。现在课间不去走廊尽头站着了——他直接去二楼。赵一鸣有时候在走廊等他。

"你们班主任凶不凶。"

"挺凶的。"

"我们班主任更凶。第一节课就说'你们已经高二了——高二不努力高三哭去吧'。"

"所有老师都这么说。"

"问题是他们说的可能是对的。"

顾临风没接话。靠在走廊栏杆上往下看。楼下的空地上有高一新生在排队。校服还没发——穿的都是自己的衣服。各种颜色。看起来很乱。

"想回高一。"赵一鸣说。

"嗯。"

"但你不想回吧。"

"为什么。"

"因为高一开学的时候你还不认识她。"

顾临风转过头看着赵一鸣。赵一鸣没看他，在看楼下新生。

"我在一班什么都知道。"赵一鸣说。"你这点秘密要是能瞒住我——我这三年就白活了。"

顾临风没承认。也没否认。

九月的第一周。

新座位表终于贴出来了。新班主任按成绩排——跟以前一样。苏晚晴第六名，坐第二排靠窗。顾临风第二十三名，倒数第三排。跟以前几乎一模一样。

新学期第一件事是选班干部。刘老师让大家自荐。有人举手当班长。有人举手当团支书。有人被推举为学习委员——就是苏晚晴。她推了一下。刘老师说"你成绩好，你就当吧"。她没有再说。

于是她的名字写在了黑板左上角——"学习委员：苏晚晴"。

顾临风坐在座位上。看着黑板上的字。粉笔字。白色的。

她的角色变了一点——收发作业、登记迟到、统计分数。都是些小活儿。但跟以前不一样了。以前她只是一个坐在窗边的女生。现在她有名字挂在黑板上。每个人都能看到。

他呢。还是什么都不是。一个倒数第三排的男生。暗恋她。

但他在写诗。他在写日记。他在进步。

九月八号。星期四。

她的马尾短了。

顾临风是在第二节课间发现的。她从座位上站起来去接水。经过过道的时候——他看到她的马尾从原来的及背长度变成及肩。剪掉了大概十厘米。

发尾是齐的。刚剪的。发绳还是浅绿色的。马尾在耳后的高度——但短了。短了的马尾晃动的幅度小了。更轻。更快。像一条小鱼——跟书包上那条一样。

她在接水。接满一杯。端回去。路过他座位的时候杯子里的水晃了一下。像她的马尾。

顾临风低下头。

翻开日记本。在最新一页写字——课间。赵一鸣不在。旁边的位子是空的。

"她剪头发了。从及背到及肩。大概剪了十厘米。还是很好看。"

写完合上。

物理课。刘老师是班主任但只教英语——物理老师姓周。男的。四十多岁。课讲得很细。讲到电场那一章时用了半节课讲受力分析。苏晚晴在记笔记。写了两个公式。然后盯着黑板想了想。继续写。

物理。还是让她头疼。

下午。放学后。

顾临风从教学楼出来。走右边那条路。经过空地。经过图书馆。经过篮球场。

他看到她走在前面。马尾短了。晃动的幅度小了。但背影还是那个背影——他认得。她拐进女生宿舍楼。

他继续往南走。去男生宿舍。

赵一鸣在宿舍等他。两个人打了二十分钟篮球。赵一鸣问他今天怎么笑笑的。

"没有。"

"你有。从开学到现在——你这几天心情都挺好。"

顾临风没说话。投了一个三分——没进。球弹筐出来。赵一鸣抢到篮板。自己投进了。

"你得练。你这三分从那次之后就没进过几个。"

"什么时候。"

"上次篮球赛。投进三分的那个。你忘了。"

顾临风接过球。

他记得。那场比赛他下半场跟疯了一样。因为她站在场边看。

那个球是夏天投的。现在是秋天。

但投球的感觉他还记得。
