# 第153章 南湖问答之后

所属部卷：第三部：寒风里的花
时间：六月末夜里
地点：南湖河边

## 诗歌插页 P153：原创仿宋小令

湖灯边，夜色平，
半窗旧影入疏灯。
问当年、几回执念，几回无凭。
不把归人留梦里，
水声明日照常生。

余波没有立刻散开。

回宿舍的路上，陆允川一直把手机攥得很紧。经过小卖部时，他买了两瓶水——一瓶给祁野，一瓶给自己。冰的，瓶身上立刻凝了一层水珠。

推开宿舍门，祁野正在打游戏。听到门响，头也没回。

"水呢？"

陆允川把水扔到他床上。祁野单手接住，一边继续敲键盘一边说："谢了。"

程砚不在——他今晚有实验课，要十点才回来。宿舍里只有祁野的游戏声和风扇转动的嗡鸣。

陆允川坐到自己的椅子上，拧开水喝了一口。水很凉，顺着喉咙滑下去，像把南湖边上那些发热的话都压回胃里。

"怎么样？"祁野把游戏暂停，转过来。

"什么怎么样？"

"叶澄啊。她找你什么事？"

"聊天。"

"聊什么？"

"聊我。"陆允川看着手里的水瓶，"聊我为什么一直想不通。"

祁野靠在椅背上，双手枕在脑后。"你是不是又在翻旧日记了？"

"你怎么知道？"

"程砚说的。你那个日记本放在桌上好几天了，每天回来翻几页，然后就沉默。"祁野的语气不带评判，像只是在报告观察结果，"叶澄也说你最近状态不对。"

"你们专门开会讨论我了？"

"不叫开会。"祁野说，"就是程砚提了一嘴，我说确实，叶澄说她去找你聊。"

陆允川不知道该说什么。他从小到大都不习惯被人操心，但最近几年，操心他的人越来越多。

"她说我应该把日记本合上一段时间。"

"我觉得她说得对。"

"你也觉得？"

"不止我觉得。"祁野把椅子转回去，重新拿起鼠标，"你每次翻旧日记都会翻进坑里。写的时候是一回事，隔几年再读又是一回事。但你现在读的不是当时的日记，是你用现在的眼光重新编了一遍的故事。"

陆允川沉默了一会儿。"你什么时候说话这么有道理了。"

"一直都有，只是你不爱听。"祁野的游戏角色在画面里跑了起来，"对了，晚饭吃了没？"

"吃了半碗面。"

"那你待会儿饿了自己解决。"祁野顿了顿，"我不借泡面了。你上周借我三包还没还。"

"我记得两包。"

"三包。有一包是你半夜爬起来偷偷泡的，以为我不知道。"

陆允川愣了一下，然后笑出声来。这种笑声是今晚第一次——不是为了缓解尴尬，不是为了表示"我没事了"，就是被祁野逗的。

宿舍安静了几分钟。只有游戏音效和风扇声。窗外的走廊偶尔有人走过，拖鞋啪嗒啪嗒地响。

"祁野。"

"嗯？"

"你失过恋吗？"

祁野没转过来。"失过。"

"怎么走出来的？"

"没走出来。"

"嗯？"

"我不觉得需要'走出来'。"祁野说，手指在键盘上敲得飞快，"她就是我的经历，不是我的疾病。你要我说忘记她，我不想忘。但你要我天天想她，我也没空。"

陆允川把这句话放在脑子里转了几圈。

"所以你的意思是——"

"我的意思是别老想着'处理'自己和别人的关系。"祁野暂停了游戏，终于转过来面对他，"你这个人有个毛病：什么事都想处理干净。但人和人的关系从来不干净。干净了就不叫关系了，叫收据。"

风扇又转了一圈。陆允川看着天花板，上面贴着一张已经褪色的周杰伦海报。

"那我之前在日记里写那些，算不算在处理收据？"

"你终于意识到了。"祁野重新转回去打游戏，"恭喜你，叶澄今晚没白约你。"

程砚回来的时候，已经十点二十了。他脱掉实验服挂在门后，身上的消毒水味慢慢在宿舍里散开。

"怎么样？"他问陆允川，一边洗着脸。

"你们一个个都问我怎么样。好像我今天去做了手术。"

"比手术复杂，"程砚用毛巾擦着脸，"手术有标准方案，你没有。"

"叶澄让我把日记本合上一段时间。"

"那就合上。"程砚把毛巾搭好，坐到自己的位置上，"日记本不会跑。但你如果一直翻，日子就会跑。"

陆允川想了想，从桌上拿起那本日记本。封面磨出了毛边，内页有些地方被汗浸过，字迹微微洇开。他翻到最后一页——那是三天前写的，满篇都是分析和判断，把苏晚枝的每一条消息都拆解成证据，把自己的每一点情绪都放大成结论。

他看了一会儿，然后把日记本合上，放进了抽屉最里面。上面压了两本教材。

"放好了？"程砚问。

"放好了。"

"能坚持几天？"

"不知道。"

"那就不知道。"程砚开始翻他的实验报告，"不知道也是一种进步。"

祁野从游戏里抬起头："你们俩今晚怎么都这么哲学。"

"是你的'收据论'先开始的。"陆允川说。

"那不算哲学。那是我玩卡牌游戏悟出来的。"祁野一本正经，"你抽卡没出货，你能一直抽吗？能。但你要分清楚你是想要这张卡，还是只是不甘心没抽到。"

陆允川和程砚同时沉默了。

"你这个比喻，"程砚说，"虽然很不恰当，但道理是对的。"

"什么不恰当？我说的是真理。"

"那你上次为了抽一张SSR充了多少钱？"

"那不一样。"祁野飞快转回去，"那是真的有概率。"

宿舍又笑了。这次不止陆允川一个人笑，程砚也在笑，祁野自己也笑。三个二十来岁的男生在深夜的宿舍里，为一张游戏卡争论数学问题，然后笑声慢慢低下去，各干各的事。

陆允川掏出手机。苏晚枝回了他那条"考研加油"的消息：

"谢谢。资料暂时够用，需要的时候找你。"

很短。但"需要的时候找你"那六个字是以前没有过的。她以前的消息都是"不用了谢谢"或者"好的我知道了"，像在关窗户。这一次，她留了一条缝。

他把手机放在桌上，没有再回。

走廊的灯灭了。空调压缩机低沉地哼着。程砚的台灯照着他的实验报告，祁野的显示器闪着蓝光，陆允川坐在他们中间，没有翻开任何东西。

窗外的虫声和南湖那边是一样的。同样的蝉鸣，同样的蟋蟀，同样的风从同样的方向吹进来。但这次听着没有那么扎人了。

不是因为他想通了什么。是因为他身边还有两个人，一个正在研究实验数据，一个正在抽他的第二十张SSR。

"陆允川。"祁野忽然喊他。

"嗯？"

"你欠我的泡面，还差一包。"

"你不是说三包吗？"

"我说的是三包。但你说的是两包。既然你这么在意数量，那就两包。明天还。"

"好。"

"加一颗卤蛋。"

"行。"

祁野满意了，继续抽卡。程砚的笔尖沙沙地响。陆允川靠着椅背，闭上眼睛。

南湖的灯光在他脑子里淡了，但叶澄那句话还没有——"你什么时候对自己说过一句实话？"。

他想了想，觉得今晚可能已经对自己说了一句。不是跟叶澄说的那些，是跟祁野承认的：我不甘心。

不甘心是实话，比所有漂亮的和解都真。他以前总想跳过不甘心直接到达"我放下了"那个阶段，结果每次都不甘心，每次又重新开始"我放下了"。

也许不用跳过。也许不甘心就是往前走的一部分。跟泡面和水瓶和SSR和实验报告一起，塞在某个深夜的宿舍里，不需要被解决，只需要被放好。

他睁开眼，窗外的树影在路灯里晃了一下。明天还有课，还有食堂的早饭，还有要给祁野还的那包泡面加一颗卤蛋。

他把这些事在心里列了一遍，然后拿起毛巾，去水房洗漱。
