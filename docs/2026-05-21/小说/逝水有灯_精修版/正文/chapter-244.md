# 第244章 写成小说旁观

所属部卷：第五部：逝水有灯
时间：书稿发出后一周
地点：程砚所在城市

## 诗歌插页 P244：原创现代诗

我把修订稿放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

程砚收到《逝水有灯》终稿的第二天，请了一天假。

不是因为书太厚需要读一整天。是因为他读完第一章的第一句，就知道这一遍需要慢。不是扫读，不是批评——是陪着这个人走完这十年的最后一程。

他坐在自家客厅的沙发上，把五十万字从头读到了尾。中间起身泡了两次茶，吃了两个包子，去了一趟洗手间，接了三次电话。其余时间都在读。

读到面馆那一章的时候，他没有喝茶。因为那是他陪着陆允川去过的。他当时坐在角落那桌——不是，是被陆允川写成了在旁边吃馄饨。但事实上他那天不在面馆。在场的是叶澄不是他。陆允川把他的台词写给了叶澄——"这算迟到的体面，别再拿它换更多答案"——那句话确是他程砚说的，但在另一个场景下。被陆允川移到了这里。

文学虚构。但台词是对的。

读到毕业照那边。他看到了"都站好，别一副遗照脸"——这是祁野说的。陆允川搬动的两年搬家之后，他认出了这两句是自己说的，搬给了祁野。但在读的时候完全看不出来被换过——太自然了。

这也是文学虚构。但人的气质是对的。

读到出租屋那几章的时候，他差点把茶洒在书稿上。因为那段电话对话——关于"直接退回去"、关于"理直气壮"——陆允川几乎一字不改地用了他的原话。那确实是他在出租屋时期说过的话。没有被编，没有被美化。就是当时的他。

他当时就想给陆允川打电话。但忍住了。不是因为打电话不好——是他觉得自己应该先把全书读完再打。否则每读几十页打一个电话，电话费都不够。

读到南湖那章的时候，他看到自己在柱子上的批注被写进了正文——"柱子上曾有粉笔字，被雨洗后留下一道淡印"。他知道这是真的。因为他路过那根粉笔字。字是他写的。淡印是自然留下的。陆允川本可以忘记它——但他没有。

最后读到第245章——终稿上的最后那句："修订稿摊在桌上，边角压着一支旧中性笔，笔帽已经松了。"

程砚把书稿合上。他知道那支笔。两块钱的图书馆笔。陆允川从大二用到现在。不是矫情的物件，是真的。

然后他端起茶杯。茶已经凉透了。他走到阳台上。他的阳台没有河——只有一条省道。但这条省道的尽头向北有一座城，城里有一个书房，书房里有一架子终稿。他给陆允川发了条消息：

"读完了。不是perfect。是honest。honest比perfect值钱十倍。"

陆允川回得很快："你什么时候学会了英文批评。"

"跟老外开了几年会。没办法。"

"你不是应该用粉笔在页边写批注？"

"时代变了。现在是键盘和电子批注的时代。"

"你当年在柱子上写的那五个字——'旧日子像灯'——还在。在新漆下面。"

程砚看着这句话。窗外的省道上，货车一辆接一辆。声音很大，路很旧。但路还在。

他回："那这盏灯烧了十年——现在还在亮。"

然后收起手机。明天还要上班。但他今晚已经决定了一件事：等陆允川厨房换了新橱柜，他要去新厨房里吃顿饭。不是庆祝书完成。是庆祝书里的人都没有被写成符号——包括他自己。
