# 第039章 成绩单下滑旁观

所属部卷：第一部：初秋三楼
时间：高一期末考试后
地点：公布成绩的公告栏前

## 诗歌插页 P039：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
成绩条上被铅笔划过的数字还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，期末考试排名是一张四十二列的Excel表格，被贴在公告栏同一面玻璃上。对绝大多数学生来说，成绩排名是半天以内的情绪事件：考得好的高兴半天，考得不好的沮丧半天——然后暑期补习班、游泳班、游戏或者小说把这些情绪冲走。暑假太长了，容得下一整场考试的结果被慢慢消化、被冲淡到开学基本不剩什么——除了一个人。他一个暑假都在担心另一个人的成绩波动——不是自己的成绩，是别人的。这个别人的成绩波动在物理意义上完全没有影响到他——年级排名是分开排的，不以任何方式干预他的选科或者分班。但心理意义上的影响是：他替她消化了这次成绩下滑的大部分后遗情绪，她可能自己早就消化完了，可他这里还在剩余消化任务。

程砚在暑假中旬打电话给陆允川，一提就提到成绩的事。"我问你一个抽象的问题——看到别人考试掉名次你为什么比自己考得不好还难受？""因为当你能帮到别人时你是主动的，当你帮不到只能看的时候你就被动了——被动的难受更深。"程砚沉默了两秒，然后说："你暑假有没有给自己做些运动或户外活动？不要老待在房间里想这些。你这些想法都是闲出来的——你听说过暗恋导致的功能性消化不良吗？""没。""那你现在正在经历。"

叶澄那年暑假见了一次陆允川。她的文科班也是贴了排名——她觉得自己的分数不错，心情不错。她到县城图书馆借了几本书，正好碰见在借阅室里整理书包的陆允川——他的书架上有一排他新买的教辅书，都是物理化学的。她翻了翻那堆书问他借这些是为了自己的学习还是为了以后当老师——当老师是叶澄的一种委婉的说法，她知道他是为了另一种目的。

"你暑假准备这些教辅，她自己买了吗？"叶澄问。"应该买了吧。""那你准备了这些，万一有的跟她的版本不一样——你不就是浪费了？""万一一样——"陆允川说到一半停了。叶澄不再往下追问。她只是轻声说了声"好好读书"，然后背着自己的书离开了，心里想着——他不是在准备教辅，他是在准备一种也许永远不会有人来拿的陪读套餐。

蒋雯在九月开学后才注意到一件事。学期开始测验，夏知遥的物理成绩突然拔高到年级前二十——上次期末那种断层全不见了。蒋雯震惊地问她怎么弄的，夏知遥轻描淡写说了句"暑假看了些错题"，蒋雯逼问她"看了多少"，夏知遥说有三百多道吧。三百多道物理错题。这就是暑假的大半——关在外婆家闷热的后房里，把上学期所有错题重新订正到纸边，每一题旁边都写着红笔批注。她没有等任何人的帮助——她自己就是自己的补习老师。蒋雯把这个消息告诉了程砚，程砚转告了陆允川。陆允川说了声"她真厉害"——这句话没有半点虚假，但他心里同时也有点微妙的失望：那些"备用教案"完全没用上。

叶澄在日记上写了一段话，后来被陆允川看到了。她说："你替她担心了一个暑假，你有没有想过她也许不需要你——她靠自己就行了。她不是拒绝你的帮忙，她是根本没走到需要别人帮忙的地方。那个你替她备好了救生圈的水域，她自己游过去了。你可以为她感到骄傲，但别为自己的备用教案感到可惜——她不需要不等于你没用，只是没用在她那里。"
