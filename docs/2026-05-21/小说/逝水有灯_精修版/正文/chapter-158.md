# 第158章 短暂和好之后

所属部卷：第三部：寒风里的花
时间：七月初傍晚
地点：回宿舍的路上 / 宿舍

## 诗歌插页 P158：原创仿宋小令

书章边，夜色平，
半窗旧影入疏灯。
问当年、几回执念，几回无凭。
不把归人留梦里，
水声明日照常生。

陆允川走出活动室时的步伐比进去时轻了三两。

但他立刻在心里给自己浇了一盆水：别把这种感觉当回事。正常活动，正常告别，正常"下周末见"。不要分析，不要过度解读，不要开始写三千字的日记论证"她是不是在暗示什么"。

他走到篮球场边的时候，祁野的消息又弹了进来。这次不是视频，是一张图片——一碗泡面，上面卧着一颗卤蛋。配文："等你回来指导我吃。"

陆允川发了一条语音："你吃泡面还需要指导？"

祁野秒回语音："需要。今天的面太烫了。我一个人对付不了。"

"那你先放着。等我回来，凉了再一起吃。"

"好的老板。"

陆允川把手机塞进口袋。祁野这个人说话永远没正经，但他其实很会照顾人——他用"需要你帮我吃泡面"这个理由来确认你吃没吃饭。他从来不直接问"你吃饭了没"，他觉得那样太婆婆妈妈。

回到宿舍的时候，程砚不在，祁野的泡面已经吃了大半。他用筷子指着旁边桌上另一碗泡面："那碗是你的。卤蛋已经加了。"

"你还真请我。"

"废话。我玩游戏的时候说的话不算数，但吃泡面的时候说的话永远算数。"祁野严肃地说，"这是我的底线。"

陆允川坐下来撕开泡面盖。热气扑上来，混着酱料的味道。他拿起筷子搅了两下，发现碗底还多放了一根火腿肠。

"卤蛋加火腿肠。你今天中彩票了？"

"没有。"祁野眼睛不离屏幕，"但今天有个女生加我微信。"

"然后呢？"

"然后我通过了。然后她说她加错人了。"

陆允川停住了筷子。"那你还请我吃泡面？"

"因为加错人的感觉很好。"祁野暂停游戏，转过来，"你想想，她在通讯录里搜某个人的名字，结果名字太像，加到了我。说明有人在找一个跟我名字很像的人。那个人可能很帅，可能很有才华，而且跟我的名字差不多——所以我间接分到一点光。"

陆允川盯着他看了三秒。"你的逻辑体系确实自成一套。"

"谢谢。"

吃完泡面，陆允川去水房洗碗。回来的时候祁野已经又开了一局。他坐到自己桌前，习惯性地伸手去拉抽屉——日记本放在那里。

手碰到了抽屉把手，又停住了。

少翻旧日记。叶澄说的。合上一段时间。

他把手收回来，转而去拿桌上的水杯。然后他注意到手机亮了一下。

苏晚枝的消息。

他深吸了一口气，点开。

是一张照片——《中国现代诗选》翻开到某一页，上面是她用铅笔在空白处写的批注。批注只有一句话："这首诗前面明明说的是田野，但读到最后，忽然觉得是写一个人。"

下面是苏晚枝加的："这就是你说的那种镜子。确实扎人。"

陆允川看了很久。然后他回："你批注的地方，正好是我当年跳过去的那一页。"

苏晚枝："那你现在要不要补？"

陆允川："现在手上没书。"

苏晚枝发了一张截图过来，是那首诗的全部内容。

陆允川把那首诗从头读到尾。读到最后一节的时候，他停下来。那句结尾写的是——

*"我转身把背留给风声/不是告别/是让自己走得更快一些。"*

他看了一会儿，然后打字："读完了。比五年前看得清楚。"

"哪里清楚？"

"以前觉得最后一句是潇洒。现在觉得最后一句其实是不舍。只是不舍得太重，只能转身，假装走得快。"

苏晚枝那边隔了一小会儿才回复。"你这个解读比我好。"

"你的批注也很好。'前面说的是田野，最后变成一个人。'——这个描述很准。"

"谢谢。"

停顿了几秒。苏晚枝又发了一条："今天的搬书——我想再说一句。谢谢你没有追问任何别的事。只是来了，搬了，走了。如果以后都能这样，我可以跟你做朋友。"

如果以后都能这样，我可以跟你做朋友。

这句话他以前想都不敢想。但现在他看到了，感觉不是狂喜，也不苦涩——就是一块石头从胸口往下降了几寸，降到更稳的地方。

他回："好。"

过了一会儿，他把手机放下。没有打开日记本，但他打开了电脑上的一个新建空白文档。

他打了一行字：

"今天下午搬书。活动室很热。阳光从窗台移到了第四排书架。苏晚枝说如果以后都能这样，她可以跟我做朋友。我没有追问，没有分析，没有写日记论证。我回了好。然后来水房洗碗。祁野今天加了一个加错人的女生，他说分到一点光。"

打完这些字，他停了很久。光标在句尾一闪一闪，像在等他接下去。

他又打了一行使：

"忽然想给我妈打电话。"

他没有犹豫，拿起手机，拨了妈妈的号码。

响了好几下才接。电话那头有些杂声，像是在厨房。

"喂？川川？这么晚怎么了？"

"没事。"他说，"就是刚搬完社团的旧书，想到你了。"

"想到我什么？"

"想到——"他停了半秒，"想到你以前的单位图书馆是不是也这样搬过。你跟我说过你们图书馆那年大清理，你还掉了一本不要的，被我爸捡回来当宝贝。他说那是你们两个第一次说话。"

电话那头也停了。然后妈妈的声音有点不一样——不是哭，但跟平时不太一样。

"你还记得这件事啊。"

"记得。你跟我说了好几次。"

"因为你爸走得太久了。久到我都怕你们把他也忘了。"

"不会的。"陆允川说。说完之后他没来由地觉得喉咙有点紧。"妈——你复查的事，下次是什么时候？"

"下个月。"

"我陪你去。"

"不用，我一个人去就行——"

"不是因为你一个人不行。"陆允川把叶澄和苏晚枝教他的话说出来，"是因为我想去。"

电话那头又停了。这次更长。然后妈妈说："好。"

就这一个字。陆允川发现，他以前说了那么多话来论证自己的关心，但都不如今天这两个"好"有效。一个给苏晚枝，一个给妈妈。这两个"好"都不需要解释，不需要去辩护，不需要用日记本记下来怕自己忘了。

他把电话挂了之后，在电脑的文档里又打了一行：

"今天说了两个字，加在一起就是'好好的'。"

祁野从游戏里抬头："你刚才跟你妈打电话？"

"嗯。"

"说复查的事？"

"嗯。"

"很好。"

"你也觉得很好？"

"当然。"祁野把游戏暂停，"你以前跟你妈通话不会超过三分钟，而且从来不说了之后的内容。这次你说了。四舍五入差不多算你进步了一个数量级。"

陆允川靠在椅背上看着天花板。"祁野。"

"嗯？"

"你说得对。但是'一个数量级'这个说法有点夸张。"

"好吧。半个数量级。"

程砚回来的时候，陆允川正在把今天写的文档存进一个新的文件夹。文件夹名字叫"不再分析的日常"。

"新项目？"程砚凑过来看了一眼。

"嗯。不是日记。"

"那是什么？"

"就是记一些事情。不分析，不总结，不写读后感。记下来就完了。"

程砚看了他几秒。"这个项目很好。"

"你觉得好？"

"很好。"程砚脱下实验服，"比旧日记强太多了。旧日记是你的自辩律师。这个新文档是你的证人——它只做证，不替你辩护。"

陆允川笑了。"今晚你们的比喻怎么都这么精准。"

"因为今晚的泡面比较好吃。"祁野插嘴。

三个人同时笑了。笑声从宿舍门缝里传出去，被走廊的风带到很远的地方。

陆允川关上电脑，去水房洗漱。刷牙的时候，他对着镜子里的自己看了几秒。镜子里的人没有变帅，没有变瘦，头发还是老样子，额头上一颗痘还在老位置。

但他觉得今天这张脸顺眼了一些。不是因为做了什么大事，就是因为搬了一个下午的书，说了两个"好"，没有追着任何人要答案。
