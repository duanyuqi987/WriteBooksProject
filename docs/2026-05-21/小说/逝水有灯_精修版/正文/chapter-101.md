# 第101章 诗社传单之前

所属部卷：第二部：六月以后
时间：大一社团周
地点：图书馆前广场

## 诗歌插页 P101：原创现代诗

我把红印章放在黄昏里，
不让它替任何人回答。
风从校园的一侧经过，
带走一句没有说稳的话。
后来我学会把手松开，
让灯照着路，也照着空出来的地方。

图书馆前广场的帐篷一排排撑开，白布顶被风掀起又落下，扩音器里反复喊着社团名字。有个帐篷的绳子松了，一个戴眼镜的男生蹲在地上重新系，嘴里嘟囔着"早上明明绑紧了"。

陆允川原本只是陪祁野去看篮球社。"你一米八几不去打篮球，窝在宿舍看书不觉得浪费？"祁野把手搭在他肩上，推着他往广场中间走。

"打篮球和看书又不冲突。"陆允川说。

"是是是，都不冲突。那你报一个，我报一个，互相监督。"

走到广场中间时，叶澄从侧边冒出来，手里捏着一叠传单。她先把一张塞到祁野手里，祁野看都没看就折成纸飞机往远处扔，被叶澄一掌拍在胳膊上。

"你神经病啊？"叶澄说。

"诗社，更适合你。"祁野指了指陆允川，又从叶澄手里抽了一张，直接递过去，"填不填再说，先看看。"

陆允川接过来。传单边角还湿着，刚才叶澄从印刷室一路小跑过来，手心的汗洇进纸边。社团章压得有点歪，红印在纸上洇出一圈细毛边。

叶澄说："晚枝学姐上午刚画的章，印泥还没干透就开始发了。她说今年想招几个真正会写的。"

"什么叫真正会写的？"陆允川问。

"她说会写现代诗的最好。你高中不是写过几首？"

"那种不算。"陆允川把传单翻过来，背面是往期社刊的摘录，一首四行的短诗印在左下角。

旁边摄影社在调三脚架，一个男生趴在取景器前喊"往左一点，再左一点"。动漫社有人穿着厚重外套发糖，两个女生为一支笔找了半天，其中一个蹲下去摸草地，另一个说"算了算了，用我的唇线笔也行吧"。

祁野凑过来看传单，念出声："诗酒趁年华——这标语谁想的？"

"我觉得还行。"陆允川说。

"你当然觉得还行，你什么都能觉得还行。"祁野把传单从他手里抽走，折成方块又塞回他口袋，"先说好，报诗社可以，别又像高中那样闷头写，写完谁也不给看。写了就拿出来。"

"我那时候写了什么你都不知道。"

"不知道也能猜。肯定又是什么'月光如水'之类的。"祁野笑起来，"走不走？篮球社那边在登记了。"

"你先去，我再看看。"

"别又对会写字的女生缴械。"祁野瞥了一眼诗社帐篷下正在摆弄印泥的苏晚枝，"你先看清楚报名表上写的是什么，再看人。"

"我什么都没想。"

"对，你什么都没想。我信了。"祁野说完拍了拍他肩膀，自己往篮球社帐篷走。

诗社帐篷下只有一张桌子，铺了米色桌布，四角用透明胶粘在铁架上。苏晚枝坐在折叠椅里，面前摊开一本登记簿。她戴一副细框眼镜，低头写字时额前碎发落下来，她用小指勾回耳后，指腹上还沾着红色印泥。

陆允川走过去。帐篷的阴影刚好落在他肩膀以上，脸在光里。

"你好，拿一张报名表。"

苏晚枝抬起头，把笔搁下。"好。你的名字是哪几个字？慢一点说，我怕写错。"

"陆地的陆，允许的允，山川的川。"

她低头写完，笔尖在"川"字收笔时稍微压了一下。"高中写过东西吗？"

"写过一点。"

"什么类型的？"

"新诗为主，偶尔写散文。"陆允川说得很轻，手不自觉地去摸后脑勺。

苏晚枝点点头，没有继续追问，只是把报名表推过来："填好以后交到人文楼102就行，周五之前。"

"谢谢。"

"不客气。"

他接过报名表，指尖在纸边停了一下，像怕被看出来似的立刻收回。

后来广场西边有人打翻了饮料，橙汁顺水泥地缝流了半米，几个志愿者跑过去收拾。苏晚枝站起来看了一眼，又坐下。陆允川捏着报名表走回人群中，太阳把纸背晒得微微发烫。

他把报名表折了两折，放进牛仔裤后袋。叶澄从后面追上来。

"填了没？"

"拿了表。"

"那就是没填。"叶澄和他并排走，"晚枝学姐人挺好的，就是话不多。她说诗社缺男生，去年就一个男的，还退社了。"

"为什么退？"

"说是'诗和远方不如编程和工资'。"

陆允川笑了一声。

叶澄又说："其实你完全够格。高中那首《风铃》，秦老师不是还让你在班上念过？"

"那是老师照顾我。"

"你这人就爱这样。明明在意又非要说不在意。"叶澄说完停下来，弯腰系鞋带。

陆允川站在旁边等，手里捏着一张没填名字的报名表，纸的边角在指腹下慢慢变软。

祁野从篮球社那边回来了，报名表格咬在嘴里，两只手在拧水瓶盖子。

"搞定了？"陆允川问。

祁野把表格从嘴上拿下来："嗯。你呢？"

"拿了表，还没填。"

"那你到底填不填？"祁野拧开瓶盖喝了一口，水顺着下巴滴在领口，"磨磨蹭蹭的。想填就填，不想填就不填。多大点事。"

"想填。"

"那不就行了。"祁野把瓶盖拧回去，"但我跟你说一件事——那个学姐对所有人都很礼貌。她不是只对你一个人说话轻。"

陆允川没接话。

祁野补了一句："我不是说你不行，我是说你别太早把别人正常的态度当成特别的意思。"

"我知道。"

"知道就好。走吧，食堂。"祁野把陆允川的报名表抽出来看了一眼，又替他压回书里，"怕你被风吹走了。"

那天的日记，陆允川最开始写了很多判断。"她也许只是客气""我是不是又想多了""这算不算又是那种老毛病"。写到一半，他把这些句子全部划掉，改从动作重新写起：谁站着，谁坐着，谁先移开目光。祁野说了哪句话，叶澄在哪个路口停下来系鞋带。帐篷的影子落在肩膀以上，脸在光里。红印在纸边洇开的细毛边，像一小片没有声音的边界。

原创现代诗写完以后，他在下面空了一行。

他一共想了三件事。第一件：苏晚枝是对所有人都认真，还是对字写得好看的人更认真一点？第二件：祁野说得对，他确实习惯把普通问话听成特别照顾。第三件：就算如此，报名本身不是什么大错。写诗的人找个诗社，天经地义。

校园里的声音一直没停过。铃声响，饭卡刷过，水房门被推开，谁的名字被喊了一声，又很快被新的声音盖过去。陆允川把日记本合上时，宿舍窗外刚好暗下来，路灯的光从窗帘缝隙里挤进来，落在桌上的红印章传单旁边。
