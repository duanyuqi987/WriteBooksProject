# 第057章 最后一场模拟考

所属部卷：第一部：初秋三楼
时间：高三春天
地点：县一中考场

## 诗歌插页 P057：原创仿唐绝句

春风不语过窗棂，
卷上姓名写得轻。
最后一场非诀别，
少年低首算前程。

第一科是语文。

陆允川拿到卷子后，先看作文题。题目不难，却很容易写空。他把材料读了两遍，在草稿纸上列提纲。考场里很安静，只有翻卷子的声音。监考老师坐在讲台上，保温杯旁边放着一支红笔。

写作文时，他忽然想起诗抄本。那些年他写过太多风、灯、雨和名字。现在真正坐在考场里，他反而不敢写得太飘。他尽量把句子落到具体的人和事上，写努力，写选择，写一个人在有限时间里怎样把手头的事做好。

铃声响起时，他刚好收尾。交卷后，学生们涌出考场，走廊一下热起来。有人问作文怎么写，有人抱怨阅读太绕。陆允川走到水房洗手，冷水冲过指尖，心里没有想象中轻松。

下午数学更难。他在最后一道题卡住十分钟，额头开始冒汗。以前遇到这种时候，他会忍不住想别的，把注意力从失败里移开。可这一次，他强迫自己把前两问重新检查，能拿的步骤分一分不丢。

考完数学，他在楼梯口遇见夏知遥。

她手里拿着透明笔袋，脸色也不轻松。两个人隔着几级台阶，都停了一下。

陆允川先开口：“数学挺难的。”

夏知遥点头：“最后一题我也没做完。”

“我也是。”

很普通的两句考后交流，却比许多被他反复想象过的对话更真实。没有谁高高在上，没有谁负责拯救谁。两个高三学生站在楼梯间，一起承认一道题很难。

后面有人催路，他们各自让开。陆允川下楼时，心里有一点平静。原来靠近不一定要靠心事，有时也可以靠同一张难卷子。
