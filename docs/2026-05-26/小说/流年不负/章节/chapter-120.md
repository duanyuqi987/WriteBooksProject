# 第120章 春天

时间：2018年3月

地点：楚州 公寓

卷次：卷六：平行轨道

三月初。楚州的樱花开了。

那条街——常青小区门口的那条建新路——两边种了一排樱花树。楚州很早就把这条路划作了"樱花大道"。粉白色的花开得很满——枝头上挤挤挨挨的。风一吹，花瓣往下落。不是一朵一朵地掉——是大片大片地从枝头松开来。往下坠的时候不是直线的——是左摇右晃的，像找不到着陆点。街上是春天特有的那种吵——遛狗的绳子在地面上拖着；买菜的大妈提着塑料袋边走边跟身边的人说话；放学的小孩子穿着校服，背着大书包边走边踢石子——石子飞起来撞到花坛的边缘，又弹回来。空气里飘着很淡的花香——樱花的香是很虚的——凑近了反而闻不到，隔了一段距离才若隐若现地飘过来。还有油炸烧饼的芝麻味——楼下那个炸烧饼的推车在冒着烟。

顾临风站在六楼公寓的窗前。窗子开了一条缝——窄窄的，刚好可以伸进两根手指。窗框上的漆——绿色的——也在春风里开始化了——软了一点点。樱花瓣从缝里飘进来一瓣——粉白色的，花瓣边缘有一点褐色的瑕疵。在空气里慢慢往下坠，落在书桌上——《古代文学论文选题指南》的封面上。他没去动。花瓣静静地躺在"古"和"代"两个字之间。

窗台上那盆文竹长出了一根新芽。嫩绿色——绿得发黄，娇嫩得有点不对劲——跟旁边深绿色的老叶对比起来——新芽的颜色亮了好几倍。嫩芽是从盆土侧边破出来的——头上还顶着一小粒土。他看着新芽——看了一小会儿。然后把手指伸过去——用指腹轻轻碰了一下芽尖。芽很软，一碰就弯。收回手。然后关上了窗子。窗子合上的时候发出一声很轻的"砰"。

转身。书桌上摊着古代文学论文的材料——打印出来的博尔赫斯谈杜诗的文章，A4纸，边角被翻得起毛了。旁边放着一本翻开的《杜诗详注》——扉页上有图书馆的章——"楚州师大图书馆藏"。一个笔记本——黑皮布纹的——翻到空白的一页。笔——英雄钢笔——笔帽没盖上，笔尖旁边有一小片墨迹——在纸上点了几个蓝色小圆点。

他拿起笔。又放下来——笔放在书边，笔尖离纸面两厘米。

转身去厨房烧水。煤气灶——火苗跳起来——蓝色的底焰，黄色的尖。壶底的水垢在烧水的时候发出细微的"嗞——"——那层白色沉淀在热胀时裂开细小的缝。水开了——壶盖被蒸汽顶得轻轻跳——他在墙上靠着等了一分钟——等水稍微凉一点。然后倒进杯子里。茶叶搁得不多——龙井——刚好够让水有点颜色——浅绿色的。茶叶在热水里慢慢舒展开——一片一片的。端回书桌前。茶杯底在桌面磕了一下——"笃"——留下一小圈水印。在椅子上坐下。椅子的木面硬而凉。

三月十四号。周三。下午两点四十五。

窗外有条狗在叫——汪汪汪的——大概是被路过的孩子逗急了。孩子的笑声很脆——"它追我——它追我——"。还有自行车铃铛——"叮铃——"。还有楼下那个老爷爷在拉二胡——拉《二泉映月》，拉到最悲伤的那一段——琴弦被左手手指压着往下滑。

手机响了。

振动先到的——"嗡嗡嗡——"——手机在桌面上挪了一下——往左边滑了半厘米。然后屏幕亮了——"妈"。来电头像是一张旧照片——他妈在青阳家里的自拍——不太会找角度，脸占了屏幕的一半，背景是电视和那个十字绣。

他把杯子放下。杯底磕在桌面上。手指在杯把上停了一秒。然后拿起手机——拇指在绿色接听键上滑了一下。屏幕闪了一下——接通计时开始——00:00——00:01——

"临风。吃饭了没有。"他妈的声音熟悉得不能再熟悉——那把嗓子带一点常年被油烟熏的沙。

"吃了。"

"今天下午张阿姨来电话了——她上次说那个姑娘答应见面了。姑娘说'最近学校比较忙，但周末可以抽空'——人家还专门把时间排出来了——你听着：这个周六下午三点——"

他的手指在杯子把手上收紧。杯身里茶水的液面——纹丝不动。但食指和中指的指腹——压在白瓷上——指尖开始有点发白。

"我给你安排好了。下周六下午三点。一茶一座——就在楚州师大北门旁边——你是不是去过——那个茶餐厅——琥珀色的——对。姑娘叫苏晚晴。你必须去。"

沉默。

客厅能听到秒针在走——或者不是秒针——是窗外二胡完了一首——换了一首——现在是《赛马》。琴弓在琴弦上飞快地跳着——那模拟马蹄的碎弓。然后是那只狗又叫了两声——这次是更远的。然后是楼下老奶奶在喊——"吃饭啦——"——拖着长长的尾音。

他没挂电话。把手机在耳边贴稳——手机的扬声器贴着他的耳朵——他能听到他妈在那头的呼吸声——跟他的呼吸——两个频率。另一只手从杯子把手上滑下来——指尖离开白瓷——按在桌面上。桌面是木贴皮的——能摸到木纹的微凸。手指在桌上展开——然后逐渐攥紧——掌心在桌面上摩擦——发出干涩的轻微的沙响。

窗外樱花在风里摇了摇。有一阵较大的风——从建新路拐角处灌过来——把整排樱花的枝丫吹得同时往下点头。花瓣在空气里被打散——那股风把几十片花瓣从枝头掳走。它们没有一起落——有的还在往上飘——被风底下的乱流托了一下。有的在旋转——像一个小陀螺。有的直线往下坠。他可以看到它们飘下来的轨迹——不是直的——不是弯的——是乱的。最后落在人行道边上——有些落在马路牙子上——被后面的一阵风又吹到了车道上。

"临风。"他妈的声音从听筒里又冒出来——比刚才高了一点——"你到底听见没有。我说你是不是个木头——我跟你在这里替你操碎了心——你那边连一个'嗯'都没有——"

他深吸了一口气。气息从鼻子进去——在肺里展开——胸廓往上提了一点——肋骨之间的空隙拉开。稳的。停留一瞬——那个停顿——那个逗号。慢的。然后——

呼出来的时候。声带动了一下。气息经过喉头时——声带被气流推着合拢——幅度很小——频率很准——发出了一个极简单的单音。

"好。"

就一个字。音节只有一个——"h-ao"——第三声——先降后升。从喉咙底部升到口腔——嘴唇收成一个小小的圆形——然后张开——声音从唇之间带着剩下的气息一起释放出去。

电话那端静了大半秒。他妈急促的——但中间停顿了一下的——然后是"好了就这样——你把人姑娘的微信记一下——等会儿我给你发电话号码——"

后面的话他听不太清楚。电话里是他妈翻记号码的声音——纸张沙沙地响——"姑娘叫苏晚晴——手机号13XXX——你记着——我跟她张阿姨说我儿子要去——你别反悔——周六下午三点——一茶一座——"然后是电视背景音——"各位观众——下面为您播放的是——"然后"嘟"——挂了。

他把手机从耳边拿下来——屏幕上的通话计时停在"02:37"。然后屏幕变暗——自动锁屏——桌面图片是蓝色的几何色块——没有任何特色——出厂默认的。手机放在桌上——屏幕朝上——然后又翻过去——屏幕朝下。

手指按在桌面上。中指压住了那个笔记本的角——布纹的封面在指腹下——粗糙的，熟悉的。

窗外有一阵风掠过来——比刚才那股更大——整排樱花都在晃——花瓣被搅成一团——空气里全是白的——粉的。窗外那一排花树里藏着春天的全部秘密——不是秘密——是自然地释放出来的——温暖、复苏、在地下藏了整个冬天终于破土的新根。

他忽然感觉到自己的手在发抖。不是因为冷——室温十五度。不是因为怕——他刚刚只是说了一个"好"字。是身体比脑子先知道了。身体知道"下周六下午三点"这几个字组合在一起意味着什么。身体知道"一茶一座"是哪家——他知道那家——每张桌上铺着格子桌布，靠窗的位置能看到楚州师大的北门。身体知道要见的那个名字——苏晚晴——三个字——八年的前缀。手在替心发颤。心还在撑着——告诉自己这只是一次见面——只是喝茶——不代表任何承诺。但手出卖了他。手指的抖是那种幅度很小的、高频率的颤——肉眼几乎看不见——但他自己能感觉到——指尖的神经在把每一个微小的痉挛传给大脑。

他站起来。在屋子里走了一圈。六步——从书桌走到床前——木板在脚下发出一声微响。又走回来——四步到椅子旁边。又走回去——三步就到了床头。他不知道自己想做什么。只是走。脚步没有方向——只是需要动。窗外的樱花还在飘——他把视线从窗口移开——又移过去。然后停下来。站在屋子中间。

那封信在抽屉里。三页稿纸。对折成四块。夹在黑皮笔记本里。那个被他划了三道线的句子——"如果有一天你回头看，我还在原地"——在黑暗的抽屉里安稳地躺着。

他蹲下来。膝盖在木地板上磕了一下——硬的——但没在意。拉开抽屉——抽屉像往常一样卡了一下——他用手指把抽屉角翘起来——推动了一点——然后顺利拉开。拿出黑皮笔记本。信从笔记本里滑出来——稿纸的边角擦着封面的布纹——发出极轻微的沙声。三页稿纸——对折成四块——纸边有些发黄——那个蓝色墨水的颜色从深蓝褪到了灰蓝——有些地方还留着原色——那个被他划了三道横线的"我还在原地"——横线还是深蓝的——墨水在那三道线上没有褪。墨水褪得有些斑驳——有的段落深，有的段落浅——深浅不一的色调在纸面上铺着——像一份手稿在时间里被慢慢冲刷的痕迹。

他把信展开。沿着原来的折痕——对折——反向对折——摊平。第一个句子还在。

"苏晚晴："

第二行。

"很久没跟你说话了。其实高考后就没说过。"

整封信读了一遍。读得很慢——慢到一个字一个字地。读到第五章第五段——"不知道你现在怎么样"——读到那个从笔尖滴下来洇成一个小点的句号——读到被划掉的那七个字——"我一直没有忘记你"——虽然被三条横线压在底下——但每个字都还在。那些空白处不敢写上的话——在每一行每一段中间。像窗外的樱花——风不吹时你不知道——风一吹漫天都是。

他把信叠好——沿着原来的折痕。放回笔记本。推上抽屉。站起来。膝盖上沾了一点灰——他用手指弹掉了。又走到窗前。窗外的樱花还在飘。他攥紧的拳头搁在窗台上——指节抵着窗台的边缘。松开——手指伸直——又攥紧——又松开。掌心里有四道指甲印——白色的——在慢慢变淡。文竹的新芽在风中动了一下——嫩绿色的芽尖在春天的风里晃。他看着那个新芽。看了很久。

三月。春天。楚州。下周六。一茶一座。

他站了很久。

久到楼下那条街上的路灯亮了。路灯亮的那一刻——不是突然亮的——是先闪了一下——然后全亮——橙黄色的光铺在柏油路上。晚风又吹过来一波樱花。花瓣在路灯的黄光里——不再是粉白色——变成了金色的碎点。远处电视塔的灯又开始一闪一闪——红的，灭，红的，灭。

他重新在书桌前坐下来。椅子还是那张木椅——硬而凉。翻开笔记本新的一页——空白的——米黄色横格线。拿起笔——英雄钢笔——笔尖在纸上点了几个蓝点。然后开始写字。字迹比平时更用力——每一笔都压得很深。笔尖在纸上推着——拉出弧线——起承转合。手指在笔身上调整着力道。纸面被墨水浸透——从蓝黑色到纸背的淡蓝。写了半个多小时——桌面上那一瓣樱花已经干了——边缘卷起来。最后一行写完了。最后一个字——不是横——是捺——从左上往右下——压笔——出锋——提起来。笔放下。他把笔记本举起来。看了一遍。字很密——有些句子被涂改了——有些句子旁边有箭头指向另一行——是改了顺序。

不是信。是一首诗。

他把笔放下。抬头看着窗外的樱花。花瓣还在飘。天色暗下来——那种暮春的晚蓝——不是全黑——还有一层深深的蓝紫色浮在天边。路灯的光穿透樱花树冠投在街面上一圈一圈的。

风。花。春天。书桌上的诗。抽屉里的信。窗台上发了新芽的文竹。远处电视塔一眨一眨的灯。

八年了。他终于说了一个"好"。

下周六下午三点。一茶一座。他会去。推开门——背上书包。春风吹进来——门上的铃铛响了一下——"叮铃——"。她会坐在靠窗的位置——马尾——白衬衫——跟八年前一样——也跟八年前不一样。那个从来不说话的少年终于要开口了。

然后他会坐下来。把书包放在旁边的椅子上。说——"好久不见。"她也许会笑。她也许还记得。她也许不记得。但此刻窗外的樱花开得不管不顾——春天的花从来不问该不该开。就像那些藏了很久的话——不问该不该说。它们只是等着一个春天。

它们等了八年。从树根到树冠，从泥土到花苞，从抽屉里的信到推开门的手。八年。一场春风吹过去——满天的花瓣都在替他开口。
