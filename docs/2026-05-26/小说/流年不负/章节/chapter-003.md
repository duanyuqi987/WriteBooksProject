# 第003章 目光

时间：2010年9月中旬

地点：教学楼走廊饮水机旁

卷次：卷一：初见秋风

走廊尽头有一台饮水机。

老式的那种。下半截是白色的铁壳，上半截是透明的储水桶。桶里的水剩半截的时候水里会有一串一串的小气泡从下往上冒。机器外壳上贴了张纸，写着"热水请勿触摸"。墨水晒褪了色。

课间的时候那里总排队。热的这边排得长——泡面的人多。凉的这边快一些。水桶空了就没人管。得等到有人去办公室喊班主任，班主任再打电话叫水站的人来送。水站的人在楼下喊了一声"水来了"。一个人扛两桶，扁担挑的。桶是蓝色的PC桶，磕得坑坑洼洼。

顾临风一般第三节课下课去接水。那时候人少一点。他端着杯子从教室后门出去，站在饮水机前等前面的两三个人打完。外面的光照在走廊里。窗户玻璃有点脏。上学期末擦窗的时候留下的水渍还在——一条一条的。

那天他不知道为什么拖到了第四节课下课。数学老师拖堂拖了五分钟——"再讲一道题。"粉笔在黑板上画了一个圆。讲完又讲了一道。"这道题中考一定会考——"底下有人拿笔敲铅笔盒，敲了两下。数学老师偏过头来，推了推眼镜。扫了那个方向一眼。笔停了。

他端着水杯往走廊尽头走。杯子是不锈钢的，磕过几次，杯底有道小凹痕。杯子里剩了半杯凉水。杯壁上贴着三年前她姐给他贴的一张卡通贴纸——一只蓝色的猫。脸上的颜色掉了一半。猫的左眼只剩下一个白圈。

走廊这时候人已经不多了。下节课不是主课（音乐课），有些人在教室里磨蹭没出来。有个男生从后面跑过来，边跑边叫前面一个人的名字。脚步声噼里啪啦。跑过去的时候带了一阵风，把他杯子里的水面吹皱了。

饮水机前排了三四个人。他排在后面。他前面是个矮个子女生，在打热水泡面。红色的方便面碗。撕调料的时候粉末洒在饮水机上——方便面调味料的黄粉末。她打完水就走了。

然后前面是两个男生，再前面是——

浅蓝色T恤。

马尾。

她也在打水。站在热水这边排队。顾临风排在凉水这边。两条队并排的。两个人离了大概一米。

她的杯身是透明的，能看到里面的水位线。还剩一点点。她拧开盖子，按热水。按一下停一下。水流断断续续的。热水从龙头里出来的时候容易溅——她用杯子接得很近。接得太近也会溅，溅到她手上了。她皱了皱眉，没有缩手。水接得很慢。热气从杯口往上冒。

前面两个男生打完了，走了。一个人的杯子盖掉地上了，捡起来在裤子上蹭了蹭。

她还在接水。

顾临风往前挪了一步。两步。停下来。凉水这边的队伍短，本来应该先排到的。但是他让后面的人先打了。那个人打完说了声谢谢。他没回话。

她拧上盖子。盖子拧紧的那一下有轻微的塑料摩擦声。然后她直起身。

转过身。

两个人面对面站住了。

大概隔了半步。半步的距离——能看清她衣服上清洗过的淡淡折痕。能闻到洗衣粉的味道。不是浓的那种。是有点苦的皂基味，可能是雕牌的。

她比他矮一个头。视线平着他的锁骨——锁骨下面第二颗扣子的位置。她抬起头。抬头的动作把马尾往上提了半厘米。

目光碰在一起。

一秒。

她的眼睛是黑色的，很深的那种黑。睫毛不算长。眼皮单层。眉毛没修过，尾部有一点杂——不是那种画出来的眉形。眉头离得很近。额角有一小点青春痘褪掉后留的红印。

她愣了一下。然后弯了弯嘴角。不是什么明显的笑。嘴角往上动了一下，很细微的一个动作。像夏天窗户开了一条缝。缝很小，但是风进来了。

然后侧身从他旁边走过去了。侧身的时候身子偏了偏，肩膀往后收了一下。像在窄巷子里让路——实际上走廊不窄，不用让。

马尾甩过去的时候，头发梢差一点碰到他的胳膊。大概差了一根手指头的宽度。没有碰到。空气动了一下。

她杯子里飘出来的茶叶味——茉莉花茶——在他鼻子前面晃了一下。然后散了。

顾临风在原地站了几秒钟。周围有人在走过去。有个男生经过的时候说了句"借过"。他没听到。那个男生又说了一遍，绕开他走了。

然后他走过去接水。按了一个按钮——凉的。水流出来。声音很清楚。水花溅在杯子里，水位往上走。杯子接满了，他没停。水溢出来了，流到虎口上，又流到手腕上，凉的。顺着胳膊往肘弯流。

有人在他后面说了一声"水满了"。

他松开按钮。水流停了。

他端着杯子往回走。杯子没盖盖子。水在杯沿上晃。

走廊不长，从这头到教室大概三十步。他走了多久，没数。中途有一扇窗户没关。外面的风吹进来，把他领口的第二颗扣子吹得轻抖了一下。

杯里的水晃出来了，洒了一路。水滴在地砖上，一个一个的圆形。他没注意。手湿了一片。虎口上的水已经干了，留下了凉凉的印子。

回到座位。把杯子放到桌角。杯底的铁皮磕在木桌面上，响了一声——笃。

同桌翻了一页书，看了他一眼。赵一鸣正在吃一袋小饼干。饼干夹在手指间送到嘴边。嘴张了一半。看了他一眼。饼干放下来了。

"你脸上怎么了。"

"没怎么。"

"你脸怎么红了——"

"热的。"

赵一鸣看了一眼天花板上的吊扇。吊扇在转。他又看了一眼窗外——九月天的太阳不算毒。他又看了顾临风一眼。然后收回了。把刚才拿起来的那块饼干塞进嘴里。"哦。"

他没说别的。

下午两节课，他听进去的不多。物理老师讲匀变速直线运动——"我们从静止开始，加速度a是恒定的——"在黑板上画了一个坐标系。横轴是时间，纵轴是速度。一条斜线往上走。粉笔头断了一截掉在地上，他踩了一脚。地上一片白印子。

他在笔记本上也画了一条斜线。本子上横格太密，画出来的斜线被格子挡着，有点抖。他用笔在上面加重了一下。然后又擦了。橡皮擦过的地方留下了灰色的橡皮屑。手指沾了一粒。捻了捻。

数学课他也没怎么听。几何证明题——已知三角形ABC中，AB=AC。求证角B等于角C。老师在黑板上画了辅助线——从顶点A到底边中点D。粉笔敲在黑板上的声音很脆。他在草稿纸上画了三角形。手指顺着图形的边走过来。然后又画了一个。画到第五个的时候下课了。草稿纸上的三角形连成了一座山的样子。山脚下有一个水字——大概是画到第四个三角形的时候手里的笔滑下去了写出来的。

晚饭他没去食堂。赵一鸣帮他带了一个馒头和一杯豆浆。赵一鸣把馒头放他桌上，袋子上有水珠——是食堂蒸汽凝的。袋子提着的那一头打了个死结。

"你怎么了。"

"没怎么。"

"你水杯空了一下午。"

顾临风看了一眼杯子。空的。杯壁上那只蓝色的猫对着他。猫的右眼还在。左眼是个白圈。像睁一只眼闭一只眼。

"忘了打。"

赵一鸣把豆浆往他面前推了推。"喝吧。"推的时候豆浆杯在桌上蹭了一下，杯身上也凝了水珠。水珠流到桌子边上。赵一鸣抽了张纸巾擦了擦。擦完纸巾扔到桌肚旁边的塑料袋里。

他咬了一口馒头。掰开，里面塞了点榨菜。榨菜有点咸。嚼了。嚼了七八下。咽下去的时候有点哽。脖子前面的喉结动了一下。

赵一鸣本来还想说什么。嘴巴张了张。然后又闭上了。从书包里翻出一本《读者》杂志看。封面上是一个小女孩的照片。

晚上，熄灯之后。十点整——走廊里周老师喊了一声"关灯了"。灯灭了。先是天花板上的日光灯暗下来，然后是走廊里的灯。然后是水房那边的灯。

他在床上翻了两次身。铁架床嘣地响了一下。上铺刘洋的床板也跟着动了动。

"顾临风。你睡不着？"

"嗯。"

"我也是。"刘洋翻了个身。"上个礼拜这时候也没睡着。"

两人没再说话。

床不响了。

他侧过身面对着墙。墙上贴了一张课程表。字太小，黑暗中看不清楚。只有靠触觉——他伸手摸了摸。纸面有点糙。星期五最后一节是班会。星期四下午是体育课。星期三上午最后一节是数学。他摸到星期三的时候手指停下来了。

外面有人在打电话。声音模模糊糊的。是走廊尽头的公用电话。应该是在跟家里说话。偶尔听到一个"好"字。"嗯——好——知道了。"鞋底在台阶上摩擦的声音。然后挂了。然后是长长的沉默。

顾临风的手搭在被子外面。手指蜷了一下。指尖触到了枕头边——缝线的那条棱。

天花板上有两个荧光星星——上届贴的，撕了一半，还剩一半。撕的那个留下了胶水的残迹，在黑暗里看不出来。剩的那个还在。在黑暗里发着绿光。很淡。比窗户漏进来的月光还要淡。淡到你要是不专心看，就看不见。

他看了很久。久到那颗星星在他眼睛里不再是星星，只是一个绿色的点。

后来不知道什么时候睡着了。

第二天早晨醒来，枕头掉在地上。枕芯是荞麦皮的，摔下去的时候没发出声音。枕套上他妈的补丁对着天花板。他从床沿探下身子，伸手捡起来。枕头上沾了一根头发。他的头发——短的。捻掉。然后把枕头放回去。被子叠好。叠得不算整齐。边角有点歪。

晨光从窗帘缝隙里漏进来。今天窗帘拉得比昨天松。

外面有人已经在刷牙了。泡沫砸在水池里的声音。啪嗒。啪嗒。
