# 第215章 南方出租屋多年后

所属部卷：第五部：逝水有灯
时间：毕业第一年
地点：南方城市出租屋

## 诗歌插页 P215：原创仿唐绝句

白云微明照旧尘，
少年心事隔江春。
回看不见当时路，
一树风声认故人。

多年后他重写南方出租屋，没有再把自己写成被命运专门为难的人。他把夸张的句子删掉，把过重的词往回收，只留下能让读者看见的动作和物件。

有些日子只有过完很久才显出轮廓。当时只是上课、下课、等人、错过、回宿舍，后来才发现这些动作排在一起，已经是一条完整的河。所以这一章不只写一件事，也写那件事在岁月里怎样慢慢改变形状。那些形状有时锋利，有时柔软，最后都被他放进更宽的生活里。

小卖部的灯管白得发冷，玻璃柜台里摆着廉价中性笔和没卖完的面包。陆允川站在那样的声音里，忽然觉得自己并不是故事中心，只是众多走过校园的人之一。这个发现当时没有让他轻松，反而让他更想抓住一点能够证明自己的东西。

多年后陆允川重写这一段，才发现陆允川在出租屋整理电脑，发现高中毕业照和大学诗社照片，两个名字都安静地躺在文件夹里。留下的不是判决，而是一盏很小的灯。他后来反复想起这一段，不是因为它多么宏大，而是因为出租屋墙上剥落的一块白漆。具体的物件比誓言诚实，它不替人辩解，也不替人作证，只安静地留在光里。

她把书合上，书页里夹着的票根露出一点边，像一个不愿彻底消失的日期。他想说的话有很多，可真正到了嘴边，只剩下几句平常的话。年轻时他总觉得平常太轻，装不下喜欢；后来才知道，太重的喜欢常常先把自己压弯。

那天最先被他记住的其实不是情绪，而是饭盒里凉掉的米粒、楼梯扶手上的灰、还有窗台一角晒得发白的塑料花。人年轻时总爱说永远，真正回头看，永远常常藏在这些不体面的边角里。这些细枝末节在当时没有被他认真看见。他忙着判断自己有没有被在意，忙着从别人的一句话里找温度，反而忽略了日子本身怎样一格一格向前挪。

“你现在最大的问题不是喜欢谁，是你一喜欢谁就把自己弄丢。”祁野这样说时，语气并不郑重，甚至带着一点不耐烦。陆允川当时听不进去，只觉得朋友站得太远。后来他才知道，太近的人往往只能看见火，站远一点的人才看得见整间屋子正在冒烟。

关系在这里往前挪了一点，也往旁边偏了一点。他学会让记忆停在自己这里，不再拿它敲别人的门，到了后来，它终于从伤口变成可以回看的旧路。陆允川那时没有能力准确命名这种变化，只能把它写进纸页，写得越满，越像害怕空白。

程砚说话一向不软，但那天他放低了声音，像怕一句真话把朋友推得更远。程砚在电话里问他还会不会想，他说会，但少了。朋友的声音并不总是温柔，有时甚至粗糙，可它们让他看见世界并没有随着一段关系变窄。

他后来很珍惜这种笨拙的具体。比如一张写错座号的考场表，一支漏墨的笔，一条总是发不出去的消息。它们不解释爱情，却能解释一个人怎样在爱情之外继续活着。这些周边生活把爱情从高处拉回地面。它提醒陆允川，心动不是唯一的课程，失落也不是唯一的作业；人总要在很多不浪漫的事情里，慢慢学会怎样对待一件浪漫的事。

她不是故事里负责制造遗憾的人。她只是恰好被一个男生长久地看着，于是很多普通动作都被放大成了信号。夏知遥和苏晚枝都已经不在他的日常里，这反而让她们更像真实的人。这部分事实后来才被陆允川补进小说。补进去以后，故事没有变得轻松，却变得诚实了一些。

街边摊的油烟慢慢升起来，把人的脸隔得有些不真切，像回忆自己加了一层雾。他把这一幕和诗歌放在同一页，并不是为了显得文雅。诗对那时的他来说，是一块可以暂时放下重物的石头；重物还在，只是手终于能松开片刻。

原创仿唐绝句写完以后，他在下面空了一行。那一行空白比诗句更安静，像把解释留给未来，而不是逼当时的任何人马上回答。

四句写完，余下的话只能留在纸外。这样的限制对年轻时的他反而有用，因为它让他第一次知道，不是所有情绪都必须说到尽头。因此这些插页不是装饰，也不是为了让青春显得更漂亮。它们更像一排小灯，照着他从自我感动走向自我审查的路；灯光不强，但足够让人看清脚下有水。

写作让他学会了删减。删掉怨气，删掉自怜，删掉把别人写成坏人的冲动，剩下的才更接近真实。他越长大，越不愿把这段青春说成谁辜负谁。辜负当然存在，可它不是全部；全部里还有胆怯、误会、虚荣、善意和没学会好好告别的年纪。

图书馆闭馆音乐响起时，大家把书合上，椅子腿在地上拖出参差不齐的声响。他站在人群里，第一次觉得离开某个地方并不需要仪式，只需要灯一盏盏暗下去。这一类日常在小说里反复出现，是因为陆允川后来相信，真正改变一个人的不是某个爆炸瞬间，而是爆炸之后仍要买饭、交作业、赶公交、给手机充电的无数普通时刻。

这一段保留了沉默。沉默不是偷懒，而是因为他们当时确实没有更好的语言。很多关系坏掉，并不是缺少情绪，而是缺少能容纳情绪的说法。他没有因此变得完全宽容，也没有把所有疼痛都解释成成长礼物。他只是终于愿意承认，自己和别人都曾经笨拙，而笨拙不必永远被写成罪。

他把纸页抹平，指腹在折痕上停了一会儿，像想把那条细线按回从前。这个动作很小，小到放进当年的日记里可能会被嫌不够痛。可真正能留下来的，常常正是这些不够痛的小动作，它们让人相信那一年确实活过。

在这一章的末尾，陆允川没有得到答案。空调风很轻，吹得窗帘边角一下一下贴着墙。他只是比上一章多看见了一点自己，多承认了一点别人，多放松了一点攥紧的手。

那时他还不懂，一个人的沉默不一定是审判，也可能只是她不知道怎样在不伤人的情况下保住自己。纸页在桌上翻过去，露出下一页空白，空白安静得像允许。故事继续往前走，像水流过桥洞，没有替任何人停下，却把所有经过都带成了更低的声音。
