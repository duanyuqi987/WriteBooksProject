# 第051章 高三倒计时之前

所属部卷：第一部：初秋三楼
时间：高三九月
地点：教室黑板右上角

## 诗歌插页 P051：原创现代诗

我把星放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
粉笔写出的倒计时数字没有替谁回答，
只把那一年轻轻照亮。

高三九月的早读比以前更硬，窗还没完全亮，黑板右上角已经多了一行倒计时。白粉笔写出的数字像一枚钉子，钉在所有人的余光里。

陆允川把错题本摊开，先抄日期，再把昨天没订正完的函数题补上。笔尖划过纸面时，他总忍不住抬头看那三个数字。

程砚从后门晃进来，书包只挂了一边肩膀，把早餐袋往陆允川桌上一搁。"你的，鸡蛋饼，加了辣椒。"他自己咬着一根油条坐下来，瞥了眼黑板，"哟，倒计时都挂上了。"

"嗯。"陆允川接过袋子，没打开。

"你这什么表情。"程砚凑过来看了眼他的错题本，拿油条指了指其中一行，"这题你昨天就错了，今天还错，符号反了吧？"

陆允川低头看，果然。他用橡皮擦掉重写，笔尖在纸上顿了顿。

"你是不是又在想考完之后的事？"程砚把油条咽下去，声音放低了一点，"我跟你说，你现在想那些有什么用。你先把这破函数搞明白行不行？"

"我知道。"

"你知道个屁。"程砚笑了一声，不是嘲笑，是真的觉得好笑，"每次你说'我知道'，后面准跟一个'但是'。"

陆允川也笑了，笑得很轻，像被戳穿又不太想承认。

早读铃响了。语文课代表站到讲台前领读，教室里渐渐填满了背诵声。

叶澄收作业走到他旁边时，把一摞本子码整齐了才开口："你英语卷子呢？"

陆允川从书堆里翻出来递给她。

叶澄接过去，没马上走。她看了看他，又看了看黑板上那行倒计时，说："你别把那行数字想太多了。它就是提醒你几号考试，不是催你干什么。"

"我没想太多。"

"行。"叶澄点了下头，语气里没有追问的意思，但也没有信，"你说了算。"

她抱着作业往前排走，马尾辫在肩头轻轻晃了一下。

夏知遥坐在他斜前方，早读时背得很轻。她的声音混在周围几十道声音里，不单独突出，但陆允川能分辨出来。他把目光拉回错题本上，用红笔在刚才那道函数题旁边打了个星号。

课间的时候，前排几个男生围着黑板讨论目标大学。一个说想去武汉，一个说想去南京，说到分数线时声音就大了，像在给自己壮胆。旁边有个女生趴在桌上补觉，头发遮住半边脸，同桌帮她盖了件校服外套。

程砚从抽屉里翻出一包饼干，拆开了分给陆允川几片。"你说大学食堂的饭会不会比现在好吃？"

"不知道。"

"你什么都不知道。我跟你讲，我听说南方的大学食堂有宵夜，晚上十点还能吃上热的面。"

"那你考南方去。"

"我就是这么想的。"程砚把饼干嚼得咔咔响，"你呢，你想去哪？"

陆允川想了想，说："还没想好。"

程砚看了他一眼，没追问。他换了个话题："昨天那道物理题你看了吗？就是老赵留的那道附加题。"

"看了，没做出来。"

"我也没。"程砚把最后一片饼干塞进嘴里，"等会儿一起去问他？"

"行。"

午休时陆允川趴在桌上没睡着，听见后排有人在小声背英语单词，翻书页的声音很规律。风扇在天花板上转，发出轻微的嗡嗡声。

下午自习课，班主任进来把倒计时擦掉又重写了一遍，粉笔在黑板上敲了两下。"数字变小了，但路没变窄。该干嘛干嘛。"

陆允川听见这句话，手下那道解析几何题还是列错了式子。他把草稿纸揉掉，从作业本上撕了一张新的。

程砚从旁边递过来一支笔芯。"你的笔快没水了。"

"谢了。"

晚自习前那半小时，教室里人不多。有人在走廊上吃泡面，味道飘进来。程砚坐在陆允川旁边刷物理题，忽然问："你觉得夏知遥会考哪个城市？"

陆允川笔尖顿了一下。"我怎么知道。"

"你不想知道？"

"……想。"

程砚笑了笑，没再说话，继续低头做题。

晚自习结束后，值日生上去擦黑板。倒计时旁边被粉笔擦掠过的地方留下几道白痕，数字还在，只是淡了一点。陆允川收拾书包的时候，程砚站在门口催他："走不走？再磨蹭食堂要关了。"

"来了。"

他拉上书包拉链，最后看了一眼那行数字。二百多天，看起来很多，但他知道会过得很快。

走出教室时走廊里的灯还没全灭。程砚边走边跟他讲今天食堂做了什么菜，又说应该是最后一份红烧肉了去晚了就没了。陆允川跟在他后面，手插在校服口袋里，摸到早上叶澄塞给他的一颗糖——她收作业时顺手放的，什么也没说。

他把糖纸剥开，糖是薄荷味的。
