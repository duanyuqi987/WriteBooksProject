# 第026章 第一次说话之前

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：教室前门和讲台之间

## 诗歌插页 P026：原创现代诗

我把早晨放低，
让一缕声从课桌边慢慢经过。
你的名字没有被我喊出，
只在借橡皮那句"谢谢"的尾音旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写第一次说话抵达之前的坡度。陆允川在脑子里排练过很多次和夏知遥说的第一句话。不是那种郑重的告白——他没那么大胆，他连在草稿纸上写"我有话想对你说"都会耳朵发热。他排练的只是一些最普通的日常对话："你的笔掉了"——这句最安全，因为笔真的会掉，教室里每天至少有三四支笔从课桌滚下去；"作业交了吗"——也还行，但容易暴露他平时根本不关注别人交没交作业；传卷子时说一句"给"——最简单，一个字，零风险，但同时也零内容。每一句都是公共场所允许的安全距离内的句子，不越界，不解释，不暴露。

他在草稿纸上列过各种场景，列了又划掉，划了又列。场景一：早上她在走廊尽头背课文，他假装去那边接水，擦肩而过时说一句"早"——但这个场景被他自己驳回了，因为接水的水龙头离她平时站的位置太远，走过去需要拐两个弯，太假了。而且那里的水压特别低，接一杯水要等半天，他没那么多时间站在那里假装。场景二：食堂场景——她排在他后面，他打完饭转身时说一句"今天菜还行"。但他认真观察了一个星期，发现自己在食堂里从没有撞上过她在自己后面——每次都是她在前他在后，隔三重人。有一次他特意放慢了打饭速度，结果她打完走了，他还站在那里拿勺子搅汤，被食堂阿姨催了一句"同学快点后面还排着呢"。场景三：传卷子时把卷子递到她手里说"给"。这个场景的问题在于，卷子从来不是从后排直接传到前排的——中间要经过两三个人，他根本没有亲手递卷子的机会，除非中间两个人同时请假。

"你最近在草稿纸上画什么呢？"程砚课间探过头来，嘴里叼着一根棒棒糖——橘子味的，糖棍从嘴角伸出来一晃一晃的。陆允川手快地把草稿纸翻了个面——翻完才发现另一面上正是今天列的第五种场景：如果英语课代表请假，我替她发卷子。程砚的眼睛很快。"没什么。""没什么？你那纸上写了五句话，每句话开头都是'如果'——不是问句，是陈述句。而且每句话都写了概率——'25%''10%''<1%'。你在做语文修改病句还是什么？""我在想怎么说话。"

程砚把草稿纸从他手指底下抽出来，看了一遍。陆允川想抢回来但又怕动作太大会引起更多人注意，只好僵在那里。程砚看完了，又放回去，用手指把那张纸按在桌面上，像按住一只随时会飞的蝴蝶。看完之后没有取笑——这很反常，因为程砚平时对什么事都能编出一个笑话来——只说了句："你不用想那么多。真的不用。'你的笔掉了'已经是满分回答了。"他说这话的时候没有笑——这是少数几次程砚没有在这种事上开玩笑的时候。陆允川不知道该说什么，最后说了个"哦"。程砚用手背敲了一下他的肩膀："你别哦。你现在就缺一次不计后果的冲动。不是让你去表白——就是让你随便说句话，说完就跑的那种。你跑得快吗？""不快。""那就说完走过去，别跑。跑的话太明显了。"

叶澄在这件事上的态度更直接。她有一次发练习册时路过陆允川旁边，把练习册往他桌上一放——不是丢，是放，书角对齐了桌角——停下来问他："你是不是有什么话想对谁说？"她的声音不大，刚好够他一个人听见。"没有。""你说'没有'的时候眼睛眨了两下——你说真话的时候只眨一下。我看你两学期了，这个规律没变过。""你在分析我？""我在做班长工作——帮助有困难的同学克服表达障碍。"她说的那么一本正经，手里还拿着剩下的半摞练习册，封面的油墨味冲进他的鼻子。陆允川笑也不是不笑也不是，最后选择把语文书翻开，假装找课文。

那天放学前，他最后一次在草稿纸上列了一遍想说的话。纸上列了三个可能性：第一，"帮老师发作业时叫到她的名字"——概率约百分之十七，因为发作业的顺序是随机的，他需要刚好拿到她的本子；第二，"水房遇见，提醒她水杯忘了拿"——概率约百分之五，需要她先忘水杯，他再恰好看见，这个概率他实在算不清楚；第三，"不做任何准备，等事情自然发生"——概率未知。每个后面都标了估算的百分比，不是他自己真的会去做这些，而是他在练习把"和她说话"这件事从"决不可能"变成"有一定概率"。哪怕这个概率只有百分之五，也比零好了。他把这张草稿纸折成一个小方块，塞进笔袋最里面，拉上拉链。如果明天还是没能说话，他后天会再写一张。
