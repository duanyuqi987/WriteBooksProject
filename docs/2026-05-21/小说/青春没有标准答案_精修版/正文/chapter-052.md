# 第052章 月考排名



时间：2009年一月初，第二次月考后



地点：十班教室后墙、办公室门口



卷次：第三卷：雪落在两栋楼之间



2009年一月初，第二次月考后，十班教室后墙、办公室门口，第二张月考排名表先把这一天按进了十班的日常里。第二次月考排名贴出，陶建国宣布要按学习互补调整座位。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

有人想离朋友近，有人怕被换到前排，段璟担心和常俊凡的距离被重新安排。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看第二张月考排名表，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

周明澈也在这一天被推到前面。周明澈的特点是临时班干部气质，稳、能记人名，管事时不爱摆架子。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“不是我想调，是老师要看各科互补。”周明澈说。

“你这话已经像班干部了。”付小川说。

“调座位不是调感情，先看学习。”常俊凡说。

“那要是调远呢？”段璟说。

“坐哪儿都得把题做出来。”陶建国说。

排名表贴上去的时候，教室先安静了两秒，随后才有人装作随便地往后墙走。

有人只看自己，有人先看同桌，有人嘴上说无所谓，眼睛却在前二十名里来回找。

段璟找到自己的名字后，手心里出了一点汗。他没立刻找常俊凡，怕自己找得太明显。

常俊凡看排名很快，看完就回座位，把试卷翻到错题页，像把自己从墙上那张纸里领回来。

邓桥在旁边一遍遍算单科差距，铅笔头敲着桌面，敲得前排都忍不住回头。

刘青岚把英语卷子折好，声音很轻地说，进步也得有人看见才行。

马立群把听力题重新放了一遍，磁带机卡了一下，班里没人笑，因为大家都知道笑完还得听。

这一天的分数把十班切开，又把他们拉回同一间教室。谁都不能只靠名次活，也不能假装名次不存在。

月考排名没有悬在空处。后墙红榜的纸角和同学们压低的呼吸，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，月考排名带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

陶建国进门时不一定立刻说话，鞋底在水泥地上一停，月考排名带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在周明澈快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。月考排名越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。第二张月考排名表让他想说话，又逼他先低头看题。

周明澈被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，月考排名留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。周明澈路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为月考排名翻身，谁还在被窝里背单词，旁边人都听得见。

有人说将来要去新乡、郑州，也有人只盯着红榜上那一行小字，先把下一张卷子改完。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理月考排名里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：第二张月考排名表、一只冻红的手、一句老师点名、一块还没扫干净的雪。

这是2009年1月初的第二次月考出分日。上一次的排名还贴在墙上没撕——纸角翘起来，被冷风一吹就嗒嗒嗒地拍着墙皮。新的一张直接叠在旧榜上面，用透明胶粘了两个角，底下那张的名字从纸背上隐隐透过来，像是上一轮的自己还在隔着纸张往上看。

陶建国在早自习之前就把新排名表贴好了。他贴表的方式很统一：先按左上角，再拉平右下角，最后在中间按一下确认没有鼓包。贴完后退两步看了一眼，把掉在地上的透明胶纸圈捡起来扔进垃圾桶。然后他转身在黑板上写了一行字：今天下午第三节课后进行座位调整，原则是学习互补，有特殊视力问题单独找我。

这行字比排名本身更让十班动荡。后排有人已经开始在底下传座位表草稿——不是真的表，是自己画的理想座次：A列谁靠窗、B列谁跟谁并排、C列后面能不能挨着后门好溜出去接水。付小川在草稿纸上画了一个自己名字的圈，然后从那里发射出几条线，一条连向"后门"，一条连向"暖气片"，一条连向"周明澈（可抄作业）"。周明澈从后面探过头来看到那张图，把他的名字划掉了，在旁边写了一行：抄作业不是互补。

常俊凡这次的名次和上次几乎没有变化——第三名，英语单科第一，数学第二，物理进了前十。"稳定"这两个字在老师嘴里是表扬，在同学嘴里是觉得理所当然，但在段璟眼里是另一种东西：她在没有人注意到的时候，把所有变数都消化掉了。

段璟自己这次考了第三十七名，比上次的四十二提高了五名。英语从八十九进步到了九十四，他自己在成绩条旁边用铅笔写了一行：volunteer没写错。然后觉得这行字太幼稚，又用橡皮擦掉了。但他注意到常俊凡走过他桌边的时候，目光在那行还没擦干净的铅笔印上停了一下。不是那种意味深长的停，而是像看到一道自己曾经画过圈的旧题。

邓桥这次退步了。从二十七掉到三十一。他把成绩条翻过来，背面画了一张他自己才能完全看懂的单科差距表——物理少三分，化学少一分，英语及格线低空飘过。画完以后他没有敲桌子，而是把成绩条折了四道，塞进物理书封面内页的塑料套里。那个动作很平静，平静到段璟觉得比上次的铅笔敲桌更让人在意。

午饭时，邓桥一个人坐在食堂靠墙角的位子，吃饭速度比平时慢了一倍。何南星端着自己的饭盒走过去，在他对面坐下，也没说什么，就是把菜里的土豆片往邓桥碗里拨了两片。邓桥抬头看了他一眼，说了句"我物理只比上次多考一分"。何南星嚼完嘴里的馒头才回答："多一分也是多。"

下午第三节课，陶建国把新的座位表贴在讲台旁边。全班安静下来，但不是那种老师要求的安静——是一种每个人都屏着呼吸往纸上找自己名字的安静。

段璟的名字在中后排靠走廊。常俊凡的名字在第一排靠窗——第一排。她上次第三排，这次直接被调到了第一排。不是惩罚性的前调，而是马立群跟陶建国提的建议：英语课代表坐第一排方便收发作业和带读。但段璟知道这不是全部原因——马立群想要一个能稳定在前排的英语高分，像定海神针一样压住全班早读的调子。

他离她远了三排和一整条过道。说近不近，说远——这次是真的远了。她的背影被前面两张桌子挡了一大半，他只能从两个肩膀之间的缝隙里看到她的马尾尖，偶尔在她侧头跟宋乔说话时能看到半张脸。

调座位的过程持续了二十分钟。桌椅腿刮过水泥地面的声音此起彼伏，有人往左挪，有人往后撤，有人因为桌面上的粉笔槽方向不对而跟新同桌争执了两句。何南星抱着自己的一摞书从第三组搬到第二组，路上掉了一本物理练习册，被邓桥弯腰捡起来，拍了拍封面上的鞋印递回去。余小满搬到了靠窗的位置，她坐下后做的第一件事是把窗缝里的报纸重新塞紧。

常俊凡搬到第一排时，先把桌面擦了一遍——用湿抹布从左上角擦到右下角，确认没有上一任主人留下的铅笔印和橡皮屑。然后把书一本一本放进桌洞，顺序和以前一样：课本在左，错题本在中，文具盒在右。透明文件袋放在课本上面，围巾叠好搭在椅背上。宋乔坐在她后面一排，俯身跟她说了一句什么，常俊凡没有回答，只是把英语书翻到第九单元，用笔在单词表第一行画了一道线。

段璟搬到新座位时发现桌面上有一块硬币大小的墨渍，怎么擦都擦不掉。他把卷子摊开盖住了那块墨渍，然后抬头往第一排看了一眼。常俊凡正在低头看书，没有回头。段璟忽然觉得，调座位这件事最残酷的地方不是距离本身，而是你发现她的日常仍然在正常运转，并不会因为距离变远而产生任何断裂。

下课后，周明澈被几个人围在走廊里。不是质问，但语气里都带着不满。"凭什么把我和同桌拆开""我们俩坐在一起互相背单词效率很高"。周明澈把陶建国给他的座位调整原则表拿出来，一条一条指给他们看——文科弱和文科强配对，理科强和理科弱互补，视力不好的往后调需要医生证明。他没有提高声音，但额头上出了一层薄汗。

段璟路过时，周明澈叫住了他。"你怎么样？"段璟愣了一下。"什么怎么样？""你对新座位有什么意见？趁现在说，我还能在晚自习之前跟陶老师反映一次。"

段璟想了想，说没有意见。

他走开以后，付小川从后面追上来。"你刚才应该说有意见。"

"我说不出口。我的意见只有一条——坐得离常俊凡太远。"

"对。这一条就够了。"

"这一条是什么学习互补？什么视力问题？都不是。"段璟把步子放慢，"如果我去反映，就等于在全班面前承认我调座位的原因跟她有关。她刚被调到第一排，正要稳住她的位置。我不能在这个节骨眼上添乱。"

付小川听完以后沉默了很久。然后他在走廊拐角处停下来，说了句让段璟意外的话："你这个人比我想的要懂事。但我还是觉得你应该告诉她——不是为了换座位，是为了让她知道你是为了她留在原位的。"

晚自习前，陶建国把周明澈叫到办公室讨论了调整后的几处微调。段璟不知道具体内容，但他看见周明澈从办公室出来时手里多了一张新表，表上用红笔圈了两个名字，旁边写了一个字：查。他后来听宋乔说，周明澈帮何南星争取了往前挪一排——因为何南星的视力确实不好，坐在第六排看黑板上的公式要眯眼。何南星开始不好意思说，是周明澈在收作业时注意到他的草稿纸上有一行字写得特别大，问他为什么，他才承认的。

段璟在晚自习时做了一件他自己后来觉得很蠢的事：他把草稿纸竖起来，用手指量了一下从自己座位到讲台的距离，然后再量了一下从自己座位到第一排的距离。两个数字记在草稿纸背面，用铅笔写的，写完后又用橡皮擦掉了。但草稿纸的背面留下了两个浅浅的凹痕——一个线段，两个端点。如果没人特意找，没人会发现。

第二节晚自习结束时，段璟去第一排收英语作业——本周轮到他当值日组长。收到常俊凡那里时，她把作业本递给他，手指在本子边缘停了一下。

"你这次英语九十四了。"她说。

"还差六分及格。"他说。

"不是差六分。是进步了五分。"她把作业本放进他怀里那摞本子最上面，"你上次八十九，这次九十四。不是你差及格差六分，是你比上次好了五分。你算分的顺序错了。"

段璟愣了一下，然后点了点头。他抱着那摞作业本走出教室时，在走廊里把这句话在脑子里重复了三遍。你算分的顺序错了。她纠正的不是他的分数，是他看自己的方式。

走到办公室门口时，他听见里面马立群和陶建国在说话。马立群说："常俊凡调到第一排是对的吧？"陶建国说："对的。但第一排靠窗，冬天冷，让后勤把那个位置的窗户密封条换一下。"马立群说："你连窗户密封条都管？"陶建国说："窗户不漏风，人才能坐得住。"

段璟在门口站了片刻，没有进去。他把作业本放在办公室门口的桌子上——值日组长不用进门，放在门口的本班作业箱里就行。然后他转身往回走，脚步比来的时候轻了一点。不是因为自己的分数，而是因为他在这个世界上发现了一件新的事：有人会在你以为自己只是在例行公事的时候，注意到一个窗户的密封条。

回教室的路上经过水房，段璟听见里面有人在小声说话。他放慢了脚步——不是有意偷听，是那个声音太熟悉了。是常俊凡。

她和一个女生在水房里提水。对方问了一句什么，段璟只听见常俊凡回答的后半句："——调到第一排不是因为我成绩好，是因为马老师要我在前排带读。带读的人必须发音准，不然一班的人都被带歪。这不是奖励，是任务。"

段璟停在水房外面的墙边，背靠着贴了白瓷砖的墙壁，瓷砖上有一层冬天特有的潮气，隔着校服渗透进来，凉丝丝的。

那个女生又说了一句什么，听不清。然后是常俊凡的声音："距离确实远了。但这不是现在要考虑的事。先把下学期选科的事定下来再说。"

段璟悄悄走开了。他没有让常俊凡知道自己听到了这段话。但他往回走的路上，心里有一小块地方被照亮了——因为她说的是"距离确实远了"和"这不是现在要考虑的事"。这两句话放在一起，说明她注意到了距离，也说明她有意不在现在处理它。不是不在乎，是有更优先的事排在前面。

回到教室时，晚自习第三节刚开始。段璟在自己的新座位上坐下，把英语书翻到第九单元。他低头看见桌面上那块墨渍还在，边缘被灯管照出一层薄薄的反光。他没有再用卷子盖住它，而是把手指按在上面，感受了一下桌面上那种微微凹下去的质感。然后他翻开错题本，在第一页空白处写了一行字：不在于你坐第几排，在于你往哪个方向看。

月考排名之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

周明澈没有因此变成特别醒目的人，可在段璟眼里，周明澈从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，周明澈帮老师统计意见，也第一次被同学埋怨管得太多。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
