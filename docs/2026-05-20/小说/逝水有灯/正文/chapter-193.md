# 第193章 夏知遥的消息之后

所属部卷：第四部：失去的美丽
时间：第二天上午
地点：旧同学群

## 诗歌插页 P193：原创仿宋小令

消息短，旧城长。
车站换新墙。
故人归处众声忙。
我自坐灯旁。

第二天上午，旧同学群已经冷下来。昨夜约饭的人最后只凑成三四个，地点定在车站附近的小店。有人艾特陆允川，问他要不要一起。他看见时正在教务处排队盖章，前面还有十来个人。

他回：“这周走不开，你们吃好。”发完就把手机收了起来。队伍慢慢往前挪，窗口老师一次只收五份材料，印章盖下去，声音闷而准。陆允川忽然很感激这些琐碎的程序，它们把他的注意力按回眼前。

中午他去打印店复印身份证，老板的孩子趴在柜台后面写作业，橡皮屑散了一桌。电视里播着天气预报，说县城这几天会下雨。陆允川抬头看了一眼，心里有一点被雨声提前打湿的旧感受，却没有继续往里走。

下午夏知遥在群里发了几张粉店照片，碗很大，辣椒油很红。大家笑她吃不了辣还逞强。陆允川点了个赞，没私聊。

这一点“不私聊”不是冷漠，而是迟来的分寸。过去他总觉得只要心里有波动，就必须找一个人承接。现在他开始学会让波动自己落下去。旧城可以被想起，故人也可以被祝福，但两者都不是他用来逃开现在的门。
