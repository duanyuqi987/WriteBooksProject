# 第094章 夏天的聚餐旁观

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P094：原创现代短诗

那时我们都以为，
结束会有清楚的门。
后来才知道，
门只是普通下午的一阵风。
有人走向远处，
有人学着把远处还给远处。

杜雨薇那天晚上回家以后，给夏知遥发了很长一条消息。

"程砚的升学宴。你让我带红包，我带到了。红包放在桌上，程砚看到你的字说'她写字还是这么好看'。叶澄说你自己练过。然后话题就绕到你了——绕了很久。"

夏知遥隔了几分钟才回复："你们聊我什么了？"

"聊你爸。林曼说了你爸打电话的事。一桌人听完都安静了。陆允川低头挑鱼刺，挑了很久。"

"挑鱼刺？"

"他夹了一块水煮鱼，把刺一根一根全挑出来，然后放进了程砚碗里。程砚说谢谢。但我看到他的表情——他挑鱼刺的时候眼神是往下的，不看任何人。他不是在挑刺。他是在消化情绪。"

夏知遥发了一个省略号。

杜雨薇继续打字："后来程砚举杯说'为她说一句——前程似锦'。全桌人都举了杯。陆允川也举了。碰杯的时候他的手指没有抖。"

"这有什么特别的？"

"有。因为高一下学期那次运动会之后，程砚提到你的名字，陆允川碰杯的时候手指是抖的。我坐他对面，看得很清楚。"

"你记性真好。"

"不是我记性好。是你们的变化太慢了。慢到旁人要花三年才能看出差别。"杜雨薇靠在床头，看着手机屏幕的亮光在黑暗里浮着。她想起自己笔记本上那个写了一半就被划掉的"陆"字。三年过去了，那张纸早就不知道去哪了。但那个字她还记得。

夏知遥：他后来有说什么吗？

杜雨薇：没有。散场的时候他在门口帮程砚应酬亲戚。林曼跟他说他表情太透明了。叶澄跟他说不要停在走廊里。他站路灯下面听着，听完往北走了。

夏知遥：他听懂了吗？

杜雨薇：听懂了应该。你没看到他的表情。

夏知遥过了很久才回复："其实我今天本来可以去的。我亲戚中午就走了。"

杜雨薇看到这条消息，整个人从床上坐起来。"那你为什么不去？"

"因为我不知道去了该坐在哪里。不是位置上的哪里——是心理上的哪里。同学桌？他不是也在吗。我坐同学桌，跟他隔两个座位，吃饭的时候说什么？'祝你土木工程学得顺利'？这句话我在走廊里已经说过了。"

杜雨薇看着这行字，忽然理解了夏知遥这些年为什么一直保持距离。不是因为讨厌陆允川——从来不讨厌。是因为不知道怎么靠近一个不设防的人，而不伤到他。

"你知道吗，"杜雨薇打字，"我今天观察到一件事。程砚妈说陆允川是程砚'最好的朋友'的时候，陆允川低头喝茶，脸都快埋到杯子里了。他是那种——别人对他好一点，他就感动得不行，但他又不想让人看出来。所以他选择低头。"

"我知道。"夏知遥回得很快，"所以我从来不敢对他太好。我怕他误会。"

杜雨薇盯着这三个字——"我怕他误会"。她明白了。夏知遥不是不懂得回应，她是太懂了。她知道自己的一举一动在陆允川眼里都会被放大，所以她选择最小幅度的动作：点头、说"你也是"、不在升学宴上出现。

这不是冷漠。这是一种很多十八岁的人不会的克制。

杜雨薇打了最后一行字："你也是另一种拧巴。"

夏知遥回了两个字："我知道。"
