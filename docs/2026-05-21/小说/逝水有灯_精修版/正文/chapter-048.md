# 第048章 元旦晚会之后

所属部卷：第一部：初秋三楼
时间：高二元旦前
地点：布置中的教室

## 诗歌插页 P048：原创仿宋小令

灯散台空，
余音尚绕梁。
欲把同歌藏半页，
却因散场剩空廊。
夜半未凉光。

元旦晚会之后，陆允川表面上没有变化。三天元旦假期他回了家——帮妈妈在店里收银，空闲时做物理题，晚上看跨年晚会。新年第一天早上他起床时给手机里的联系人发了祝福——他给程砚发了一条"新年快乐，最好把那个三角方程解清楚"，程砚回了他一个翻白眼的表情。给叶澄发了一条"新年快乐，希望你们的文科班新年出更多范文"，她回"你也是，再也不能把作业本摁成湿煤了——新年了给你换新本子"。给夏知遥没有发单独的消息——他不知道怎么开头，也不知道发出去了应该等还是不应该等。但他后面在某一个分组祝福的大众转发里，把能包括的人都加进去了——她大概不会发现里头有他。她的反应也理所当然是零。

开学回来第一周，教室里还是元旦的残留痕迹——彩带还没来得及拆，一些气球已经泄气挂在窗角落。蒋雯拆彩带的时候说笑了一下怎么还没拿掉气球，夏知遥踩上椅子把挂在窗角那个快瘪成橘皮的白气球扯下来——扯落下来时气球里漏出一片飞灰，是她没留神的——她伸手扇了扇。

陆允川在新课本的第一页写下新年的日期:2012年1月1日。这是个他很想保存的年份——高二下学期即将进入高三倒计时，再有一年半高考，再之后大家会散落各地。他在这段时间轴里意识到一件事——这部小说写的第一部叫初秋三楼。初秋和冬天快过完了，春天很快会来——"初秋"这个季节点在做第一章时是最合适的时间标志，但时间线推进后它必然是封存在篇章结构里的旧标签。新的章节需要春天——需要三月、需要跑操时的春风、需要她脱下羽绒服后穿着单衣走在走廊上。而他还没有想好接下来要写什么。

程砚在开学第一顿饭堂饭就直接提了他的新年后续话题。他说："我一直在想元旦那天晚上你对她说'你的那一段很稳'——那句话的长度是七个字。你原来六字记录被她自己打破了——现在纪录保持在七个。七个之后你想达到几个？"陆允川说"我打算达到一百"。程砚没信——"一百字那是一篇小作文。你打算什么时候到达一百——高三毕业？"陆允川笑了——"也许就是在高三毕业的晚会上。""好，进度表我帮你记着——现在是七个，目标一百，剩余九十三。九十三字分两年，平均每年四十六点五个——也就是每学期你只需要多说二十三个字左右。按目前进度——你一个学期涨了二十个字左右——你差不多快达标了。""那你接着说——她应该跟你说什么？""她应该说——你从什么时候开始跟我说话这么长的——那句话至少——十五个字——你最终总共一百。""你自己已经整个脚本都写好了。"程砚说"不是写好——是希望你这两年不要白费"。

当晚宿舍灯灭了之后，陆允川握住手环，笔芯在纸上写着:新的一年从元旦晚会那四个字"元旦快乐"开始。四个字之外，还留了很多需要慢慢说的话。但至少他已经能在她面前说出不靠任何意外辅助的完整句子——"你的那一段很稳"——像是从别人的曲子里终于过渡到了自己能写下的第一句正文。
