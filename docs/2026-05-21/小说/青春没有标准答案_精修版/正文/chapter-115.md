# 第115章 树莓摊旁边



时间：2010年七月中旬，期末考试前一周



地点：城关镇树莓摊、家长饭桌



卷次：第五卷：未来在黑板上倒计时



2010年七月中旬，期末考试前一周，城关镇树莓摊、家长饭桌，王子越带回家的一小袋树莓把2010年的春夏推到十班面前。王子越把树莓带回家，饭桌上却被父亲问起补课费和期末排名。

他在学校能起哄，在饭桌上却一句笑话也接不住。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

王子越也在这段日子里显出自己的位置。后排活跃，喜欢起哄，也最怕老师和父母真点名。群像不是名单，群像是每个人都有自己的难处。

“树莓多少钱？补课费又是多少？”父亲说。

“没多少钱。”王子越说。

“你别只会说没事，成绩单拿来看看。”母亲说。

“还没发。”王子越说。

“那就先说你自己估多少。”父亲说。

家长饭桌上，菜还没凉，话题已经绕到成绩、补课和以后能去哪里。

学校里能用玩笑挡住的问题，在家里常常挡不住。父母一句你自己说，就能让人握筷子的手停住。

树莓放在桌角，小小一袋，红得很好看，却没人先吃。

王子越平时话多，到了饭桌上反而说得少。不是不想回嘴，是知道每句话都会被接成成绩。

母亲把汤往他面前推，说别光低头。父亲问补课有没有效果，声音不重，却更难回答。

窗外有人骑车过去，铃声响了一下，很快又远了。屋里只剩碗筷碰到桌面的声音。

他突然想到十班食堂里的吵闹，才明白学校有时反而像一个可以喘气的地方。

那天树莓最后还是吃了，只是甜味来得很慢，像一句迟到的安慰。

陶建国这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

王子越被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

七月中旬的封丘，期末考试前一周。周六下午没有补课，王子越在校门口的树莓摊买了一袋树莓，三块钱两盒，是老婶给的收摊价。塑料袋底部被树莓的汁水洇湿了一小块，他拎着袋子走回家的时候，手指一直捏着袋口，不让汁水滴到裤子上。

他家在城关镇一条窄巷子里，巷口有个修自行车的摊子，补胎用的胶水味顺着热风飘了小半条街。他爸的电动车停在巷口，车座晒得滚烫，车筐里有几根钢筋头——他爸在工地上做钢筋工，每天早上六点出门，晚上七点回来，车筐里永远有东西。

王子越进门的时候，他妈正在厨房炒菜。油锅的声音很大，油烟从厨房门缝里钻出来，把客厅的白墙熏出了一小块黄印。他爸已经回家了，坐在饭桌前翻看今天的《大河报》，报纸头版上印着世博会河南活动周的照片——少林武僧在园区里表演，围观的人里三层外三层。

"树莓。"王子越把塑料袋放在饭桌角上，"校门口买的，收摊价。"

他爸从报纸上面看了他一眼，没说话，把报纸翻过去继续读。他妈端着菜从厨房出来——一盘炒豆角，一盘炒鸡蛋，一碗稀饭。三个人坐在方桌前，电风扇对着墙吹，热风反弹回来，吹得树莓袋子轻轻动了一下。

"树莓多少钱？"他爸放下报纸，拿起筷子。

"三块。"

"三块就这点？"

"两盒。收摊价便宜。"

"补课费又是多少？"

王子越的筷子在饭碗里停了半秒。"一节课十五，一周三节。这个月交了四次。"

他爸算了算，没说话。他妈往他碗里夹了一筷子鸡蛋，"你先吃。一会儿凉了。"

但话题已经开了。他爸没有接着问补课费——那个数字他每个月都知道，交钱的时候已经心疼过一次——他换了方向。"期末考试下周？"

"嗯。"

"估一下分。"

"没估。"王子越把头低了一点，"还没考完呢。"

"考完之前心里得有个数。"他爸的声音不重，但很稳，每个字都像钉子钉在木头上，一颗一颗很慢。"你上回月考总分掉了十几分。老师说理科可以，语文英语拖后腿。"

"我知道。"

"你知道。那你打算怎么办？"

王子越没有立刻回答。他在学校里是最能起哄的，后排活跃分子，谁的玩笑都能接住。但坐在家里这张方桌前，面对着他爸那双被钢筋磨出厚茧的手，他发现满肚子能说的话都没有用。玩笑不能当成绩交，起哄不能挡追问。他握筷子的手心出了一层细汗。

"语文要多背。英语要多读。"他终于开口了，声音比在学校里低了半截，"我找了刘青岚借了英语笔记。她英语好。"

"借了会看吗？"

"在看。"

他爸看着他的脸，看了几秒，然后夹了一口豆角。"看不是光翻。看要记。记了要背。"他把菜咽下去，"爸不识字多，也不懂英语。但爸知道一件事——自己说要看的东西，自己得看完。"

王子越把碗里的稀饭一口喝干净，碗底碰到桌面的时候，发出轻轻一声响。他站起来去盛第二碗的时候，看见树莓还放在桌角——红得很好看，上面的颗粒饱满，没有被筷子碰过。是他自己没吃。他觉得如果自己先吃了，就像在用甜品挡掉一件严肃的事。

"你说你在学校挺好的。"他妈忽然接了一句，"那考试也得一定好。不是光嘴皮子好。"

"我没光嘴皮子。"王子越说，声音有了点急，"我物理和数学确实不差。"

"那你语文英语得跟上。总分不是靠两科撑的。"

"我知道。我已经在补了。"他把第二碗稀饭放在桌上，坐下来。"刘青岚的英语笔记我看了一个多月了。每周抄三页。"

他爸放下筷子，看了一眼他妈。两个人交换了一个很微妙的眼神——不是不信任，而是听到了一个新信息。之前他们只知道儿子在学校活跃，不知道他也有固定抄笔记的习惯。他爸拿起筷子重新夹菜，说了一句："抄得整齐不整齐？"

"还行。"

"拿出来看看。"

王子越愣了一下，走进自己房间，把书包拎了出来。他从英语文件夹里抽出几张活页纸——上面是他用铅笔抄的英语笔记，字不算漂亮但很清楚，每页按日期排好，每周三页，从不间断。笔记的页边被翻得有点脏，纸角卷着——是反复翻看的痕迹。

他爸拿起来看了看。他不认识英文，但他认识数字——日期连续，没有断档。"七月三号，七月七号，七月十一号。一周三页。"他把笔记还给王子越，"那行。你爸看不懂，但写得认真。"

王子越接过笔记，低头把它重新夹进文件夹里。他忽然觉得鼻梁有点酸，但忍住了。

晚饭吃到最后，树莓还是没人动。王子越站起来把袋子打开，把两盒树莓倒进一个碗里，放在桌子正中间。他爸看了一眼，拿起一颗吃了。"酸的。但是后味甜。"

"树莓就是酸甜的。"王子越说。

"跟你成绩一样。"他爸说，"现在是酸的，后味得是甜的。"

王子越夹了一颗放进嘴里，咬破的第一口是酸——很酸，比平时在学校吃的时候酸得多。他把那种酸咽下去，等着后面的甜。

那天晚饭结束以后，他回到自己房间，打开英语笔记新的一页。上面是刘青岚写的英语作文范文——题目是"What I've learned from the World Expo"。他在空白处用自己的话写了一句翻译：世博让我学到了什么？学到了远方的城市很大，但我在封丘也可以用它来写作文。

写完以后，他把英语笔记翻到最后一页，在空白处画了一个很小的树莓——三颗果子，两根叶子。旁边写了一行字：期末考试后，成绩单和树莓一起放在饭桌上。希望是甜的那一口。

周一回到学校，王子越在早自习前把英语笔记的最新一页给刘青岚看。刘青岚从头到尾翻了一遍，在第七页停住了。"这一页你抄错了两个单词。一个是expo——你少写了一个o；另一个是volunteer——你把后面的eer写成了ere。"

王子越凑过去看，"这两个词重要吗？"

"世博这个词今年肯定考。志愿者的拼写也高频。"刘青岚用铅笔把他错的单词圈出来，"你抄笔记是认真的，但是抄完以后要自己拼写一遍。抄是眼睛的事，拼写是手的事。两回事。"

"那你当初怎么学会拼写的？"

"我记单词不是抄。是默写。先看一遍，合上书，自己写一遍，再打开对。不对的重新写三遍。"刘青岚把笔记还给他，"你这样每周抄三页，抄了一个多月，很不容易了。但效率可以更高一点。"她在王子越笔记的第一页上写了一行小字：抄→默→对→改→再默。

王子越看了看那五个步骤。"这跟做数学题一样。做一道，对答案，改错，再做。"

"对。学习方法都是一样的。"刘青岚说，"只是你拿英语当任务，拿数学当挑战。你对你喜欢的事更用心。"

"那怎么把英语也变成喜欢的事？"

"不用喜欢。"刘青岚说，"先用对的方法。用对的方法做上半年，慢慢就不讨厌了。"

王子越把那句话记在了心里。他翻开英语笔记，在扉页上写了刘青岚的五步流程：抄→默→对→改→再默。然后他试着做了第一个单词——expo。合上笔记本，闭眼，在草稿纸上写了一遍，翻回来看。对了。又做了第二个。也对了。

大课间的时候，付小川从树莓摊回来，带了三盒收摊特价——一块五一盒。他给王子越桌上放了一盒。"你周一早上怎么蔫蔫的？"

"没蔫。在默单词。"

"你什么时候开始默单词了？"

"从今天。"王子越把树莓盒子打开，拣了一颗大的放进嘴里。这次树莓是先甜后酸的——跟上周带回家那批不太一样。也许只是熟得更透了，也许是他的心情先变了。他把籽吐在纸巾上，低头继续在草稿纸上写单词。volunteer，v-o-l-u-n-t-e-e-r。写完以后他看了一眼自己写的——刚才错过的那个词，这次对了。

付小川看着王子越一笔一划写单词的样子，忽然说了句："你认真的样子跟你起哄的样子判若两人。"

"都是一个人。"王子越头也没抬，"起哄是放松，默单词是干活。干活的时候就得认真。"

"你爸那句话有用？"

"哪句？"

"'自己说要看的东西，自己得看完'。"

王子越停住笔，抬起头。"有用。"他说得很短，但语气很准。"我爸认字不多，但他说的这句话是对的。"

期末考试前最后一周的晚自习，教室里比平时安静得多。电风扇的声音是最大的背景音，然后是翻书声和铅笔写字的沙沙声。后墙的错题区已经被反复更新了不知道多少次，黑板报的世博专题褪了色。树莓摊的伞换了新的——青岛啤酒的伞面终于被换成了纯色深蓝，但挂灯泡的老钩子没换，还是那颗生锈的膨胀螺丝。

王子越在期末考试前一天晚上，把英语笔记从头到尾翻了一遍。一页一页，每周三页，从六月坚持到现在。每一页的页边都卷了，那是反复翻的痕迹。他把笔记放在书包最上面，拉上拉链。窗外路灯照着黄河大堤上的杨树，叶子哗啦啦响，像是远处有江河流过。

期末考试前的最后一个周末，王子越早上六点就起来了。他坐在房间里，把英语笔记按刘青岚教的五步流程做了一遍——先看一遍单词，合上书，默写，打开对，错的重新写三遍。他做完二十个单词之后，他爸推门进来送早饭——一碗热稀饭和半个馒头，放在书桌角上。他爸看了一眼桌上摊着的笔记，没说话，又看了一眼王子越手上握笔的姿势，说了一句："握笔握得比以前正了。"说完就出去了。

王子越低头看自己的手——他以前握笔很歪，大拇指压在食指上，写完字手会酸。他不知道什么时候改的，可能是抄笔记的时候，字写小了自然就握正了。他继续写单词，写完最后一个的时候，他爸又推门进来收碗。碗是空的，馒头也吃完了。

"爸，"王子越叫住他，"期末考试要是考不好，你还会让我继续补课吗？"

他爸在门口站住，手扶着门框。"你补课有没有效果？"

"有效果。英语比上学期多了十几分。"

"那你觉得有救吗？"

"有救。"

他爸点了头，很慢，像在测算一个承诺的重量。"有救就接着补。爸花这个钱不是要你一下子考第一。是要你知道学习这条路是能往前的。"

王子越把这句话记住以后，在英语笔记的扉页上又写了一行字——在刘青岚的五步流程下面："学习这条路是能往前的。"然后翻到新的一页，开始背今天的新单词。

期末考试的那一周，封丘下了两场雨。第一场是星期二的午后雷阵雨，操场上积了小半寸的水，世博阳光长跑的横幅被雨水打湿，红底洇成一片。第二场是星期四早上，淅淅沥沥的小雨，飘了一个上午。王子越穿了一件长袖校服外套去考场，口袋里装着笔和三颗树莓——早上路过树莓摊买的，收摊价两块三盒，他分了一盒给余小满，一盒自己带着，剩下一盒留在了教室后墙的错题区旁边，上面压了一张字条："考试前吃一颗，甜一下。王子越。"

常俊凡考完最后一场出来，站在教学楼下看雨停。段璟从旁边走出来，犹豫了一下，站在她旁边，中间隔了半步。雨停了以后，操场上积水的倒影里映出一截彩虹，很短，只跨了小半个天空。常俊凡抬头看了一会儿，说了一句："期末完了。暑假补课见。"段璟点了点头——他本来想说点什么，但想了想，觉得只是点头就足够了。因为他知道她现在不需要甜言蜜语，她需要的是暑假补课、数学竞赛和一步一步往前走的秩序。

树莓摊旁边之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

家长饭桌让他明白，很多同学在学校的热闹，只是为了遮住家里的沉默。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，王子越带回家的一小袋树莓还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
