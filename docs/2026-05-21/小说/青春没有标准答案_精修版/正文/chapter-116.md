# 第116章 同学录第一页



时间：2010年七月中旬，暑假补课名单下发



地点：高二十班教室、女生宿舍



卷次：第五卷：未来在黑板上倒计时



2010年七月中旬，暑假补课名单下发，高二十班教室、女生宿舍，孙佳怡同学录第一页写下的补课名单把2010年的春夏推到十班面前。孙佳怡把暑假补课名单抄到同学录第一页，方便母亲安排接送。

她讨厌被家里盯得太紧，却又不得不承认自己一个人回家确实不方便。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

孙佳怡也在这段日子里显出自己的位置。家里管得紧，电话一来就紧张，不愿让同学看见自己的不自由。群像不是名单，群像是每个人都有自己的难处。

"我妈要我把每个补课时间都写清楚。"孙佳怡说。

"你妈这是排课表还是排兵布阵？"宋乔说。

"她担心也是真的。"常俊凡说。

"可我也想自己走一次。"孙佳怡说。

"和家里谈，不要只用赌气谈。"叶书兰说。

同学录第一页原本应该写名字和生日，到了这年夏天，却被写成了补课时间、路线和电话号码。

有人笑这不像同学录，像作战地图。笑完以后，又认真把自己的安排补上去。

新华书店、补课班、校门口、县城路口，这些地方被写在一页纸上，像十班临时散开的几个支点。

常俊凡写字很小，时间和地点分得很清楚。段璟看着她那一行，忽然觉得她把未来也写得这么有秩序。

赵远航讲路线时最有精神，哪个路口左拐，哪条巷子近，他都能说得像背课文。

宋乔嫌麻烦，却还是把女生宿舍电话也写了上去，说万一有人找不到人。

同学录被传到最后，纸角已经软了，第一页却比任何复习计划都像真的。

那一页没有写离别，可每个人都看得出来，时间正在把他们往不同地方分开。

叶书兰这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

孙佳怡被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

孙佳怡把同学录摊在课桌上时，圆珠笔尖在"第一页"三个字上方停了好一会儿。

这本同学录是她初一买的，封面上印着粉色卡通兔子，贴纸已经被磨得只剩半张。扉页上她用蓝色圆珠笔写过"友谊地久天长"——那是初一时候的字，笔画圆圆的，每一横都往上翘，跟现在完全不一样。

现在她要在这六个字下面，用更小更整齐的字迹，把整个暑假的补课时间一行一行填进去。

"周一三五 数学 新华书店三楼"
"周二四 英语 校门口往东两百米"
"周六上午 物理 县教育局旁边巷子里"

写到最后一个时间时，笔尖在纸上顿了一下。星期六下午空着。

那个空白的下午不是漏掉的。是她故意不写的。

她妈在电话里问过三次"时间都写清楚了没有"，她爸在饭桌上说过两遍"别人都在补你是知道的"。她把所有能交代的时间都写上了——周一到周六上午，每个时段都明明白白。唯独那个星期六下午，她想留给自己。

不是为了偷懒。是想一个人去一次黄河大堤。

她从小在封丘长大，大堤离她家不到三公里。可她从来没一个人去过。每次都是学校组织春游，或者跟着邻居家小孩结伴跑一趟。一个人去，意味着她可以从大堤这头走到那头，想看多久看多久，不用跟任何人交代。

这个念头小得不能再小——一个下午，一条路，一个人——却让她在写同学录的时候心跳快了两下。

宋乔从旁边探过头来，下巴搁在她肩膀上，念出第一行字："周一三五数学新华书店三楼——你这真的不是作战地图吗？"

"我妈要的。"孙佳怡把笔帽盖上。

"你妈这是排课表还是调兵遣将？"宋乔拉开椅子坐下来，"不过说真的，你暑假不在学校住？"

"回家住。我妈说补课期间必须回家，她好看着。"

"看着什么？"

"什么都看着。"孙佳怡声音低了一点，"怕我跟男生走太近，怕我成绩往下掉，怕我一个人在外面不安全，怕我吃饭不规律——"

"怕你长大。"宋乔替她把话说完了。

孙佳怡没否认。

她妈是一个好人。每天五点起来做饭，晚上十一点还等她下晚自习。电话费大部分花在跟班主任沟通上，家里的日历上密密麻麻记满了她的考试时间和家长会日期。那些关心像一件很厚的棉袄——冬天穿着暖和，夏天穿着就透不过气来。

同学录第一页本来应该写心愿。她初一的时候写过"考上好大学""变瘦变好看""去北京看天安门"。现在这些都被补课时间覆盖了。

不是不想要心愿了。是暂时腾不出手来许愿。

常俊凡站在孙佳怡桌边，手里拿着一支红色圆珠笔，低头帮她核对补课路线。

"新华书店那边周一三五的数学老师讲得还行，就是题量太大，你做完要自己筛一遍。"常俊凡指给孙佳怡看，"英语那个补习班周三会发一套模拟卷，题型跟期末差不多，可以重点看阅读理解部分。"

"你连这个都知道？"孙佳怡抬头看她。

"我去年暑假上过那个班。"常俊凡说，又低下头继续画路线，"英语补习班旁边有个树莓摊，阿姨的树莓比别家甜。你周三下课早的话可以去买一盒，中午摘的最新鲜。"

她说话的方式跟别人不太一样。别人帮人是居高临下地给建议，她是蹲下来跟你一起看地图。给完建议又退回去，像什么都没发生过。

段璟在第三排靠窗的位置上，面前摊着物理练习册，笔停了快十分钟。

他在看常俊凡。

他看见她把孙佳怡的时间表从头到尾捋了一遍，用红笔标出了三条最佳路线。他看见她在讲解的时候偶尔会微微蹙一下眉头，那是她在认真思考时的习惯表情。他看见她帮完忙以后安静地把笔盖合上，没有多说一句话。

以前他觉得喜欢一个人就是想多看她几眼。可现在多看几眼以后，心里反而空了——他看到的不只是她的侧脸，还有她说话时把声音压得很低的习惯，她把别人路线画得比自己的还仔细的耐心，她帮完人以后忽然安静下来的那几秒钟。

那些东西不能只靠看。要靠近，要在她需要的时候正好站在旁边，要说出一句不是废话的话。可他现在能做什么？他只能把物理题做对，把月考排名往前提，让自己在将来的某一天变成一个有能力站在她身边的人。

他在草稿纸上写了两个字：先做题。写完自己都笑了一下。

付小川从后门撞进来，满头大汗，校服前襟湿了一大片。

"外面热死了！大堤的风都是热的，吹在身上跟站在电吹风底下一样。"他把书包往桌上一甩，从兜里掏出一根快化了的冰棍，"我跟你们说，我刚从树莓摊那边过来，阿姨说今年天太热，树莓熟得早，再过一星期就没了。"

"所以你只买了冰棍？"赵远航头也不抬，还在画他的路线图。

"树莓也买了。"付小川从书包里掏出一个塑料袋，里面是几盒被挤变形的树莓，"但是路上摔了一跤。树莓飞出去三盒，我就捡回来两个。"

赵远航终于抬起头看了他一眼，然后问了一句让付小川差点把冰棍吞下去的话："物理卷子呢？"

付小川的表情变化非常精彩——从炫耀变成心虚，中间经过大约零点五秒。

"在家里。"他说。

"在家里的哪个位置？"

"书桌抽屉里。"

"拿出来。"

"没——没做完。"

"做多少了？"

"选择题。"付小川声音越来越小，"前五道。"

周围几个人全笑了。何嘉树笑得最响，差点从椅子上翻下去，被宋乔拽住书包带才稳住。

赵远航叹了口气，从自己书包里掏出一张物理卷子，上面答案写得密密麻麻，每道题旁边还标了页码和解题关键步骤。

"拿去看——不是让你抄答案，是看解题思路。每道题旁边我都写了为什么要选这个选项，你看完再自己做一遍。错的地方用红笔标出来。"

付小川接过卷子，愣了好一会儿，说了一句不太像他的话："你怎么写这么多？"

"因为我暑假不踢球，只补课。"赵远航说，"等你下学期能把物理题做及格，我就能放心踢球了。"

这句话说得很平淡，每个人却都不笑了。

赵远航是十班成绩最好的几个人之一。他爸妈对他管得不算严，但他自己给自己定了一条规矩：作业不做完不踢球，月考不进前二十不加练。去年冬天他整整三个月没碰足球，就为了把物理从及格线拉到八十五分以上。三个月里他每天比别人多做一个小时物理题，足球鞋放在床底下，上面落了一层灰。

"那你现在能踢球了吗？"付小川问。

"能了。"赵远航把最后一条路线画完，"暑假补完课以后，每天下午四点，操场见。"

付小川把卷子小心折好放进书包侧面，那个动作跟他平时的毛手毛脚判若两人。折了三折，每折都对齐了。

余小满在女生宿舍收东西。暑假宿舍不关，但很多人选择回家住，补课的时候再回学校。

她把衣服叠好放进编织袋——三件T恤、两条裤子、一套校服。东西不多，叠得却很仔细，每件都抚平了边角，再把袖子往里折齐。

叶书兰从对面床铺探过头来："小满，暑假真回家？"

"嗯。"余小满把编织袋的拉链拉上一半，"我妈说麦子收完了，有空管我了。"

她顿了一下，又纠正道："其实她不是有空管我。她是想我了。"

叶书兰没有追问，但她听懂了言外之意。余小满家在封丘下面的村里，父母种地，一年到头最怕的不是收成不好，而是女儿在学校受了委屈他们不知道。

那种怕很具体。她妈每次打电话都要问："食堂的菜够不够吃？""被罩换洗勤不勤？""老师对你好不好？""跟同学处不处得来？"余小满每次都说"够了""换了""好""处得来"，用最短的字把担心挡回去。

可挡不完。

有一次她在食堂只打了一份青菜和一份米饭，因为那个月饭卡快没钱了。她妈不知道这件事，但三天后突然出现在学校门口，手里拎着一袋煮鸡蛋和一罐自己做的酱菜，说"怕她吃不好"。余小满站在校门口接过袋子的时候手都在抖——不是感动，是觉得亏欠。她在学校省吃俭用的那点心思，她妈隔着几十里地都感应到了。

现在她把编织袋拎起来试了试重量，忽然回头问了一句："书兰姐，你说高三以后还能经常回家吗？"

叶书兰想了想："能是能，但肯定没现在这么多了。"

"那就趁这个暑假多回几趟。"余小满说这句话的时候语气很平，像在做一道没什么难度的选择题。可她往外走的时候，在门框那儿停了一秒。

七月中旬的封丘县，热得不像话。

麦子收完以后，田里剩下一排排淡黄色的麦茬，在太阳底下干得发白。黄河大堤上的杨树叶子卷得紧紧的，只有知了还在一片一片地叫。大堤上的风从东南方向刮过来，带着黄河水的泥腥味和下游瓜地的甜味，吹在身上没有凉意，却让汗蒸发得更快。

树莓摊的阿姨把煤炉换成了电风扇。风扇叶子转得嗡嗡响，吹出来的风也是热的，但比炉子的热气好得多。摊子旁边的纸箱里堆着上午刚摘的树莓，红色的浆果在太阳底下亮晶晶的，每一盒都用保鲜膜封得严严实实。有学生路过时停下来买一盒，阿姨会多塞两颗进去，说"学生娃辛苦"。

新华书店三楼的补课班把窗户全打开了，六个电风扇同时开到最大档，还是不管用。学生们把校服袖子卷到胳膊肘以上，露出的手臂被晒成了两种颜色——手腕以上白，手腕以下黑。练习册的纸页被汗水洇潮了，钢笔字微微晕开。老师讲题的声音跟窗外的蝉鸣纠缠在一起，听久了分不清哪个是蝉叫哪个是板书声。

十班的教室也没闲着。学校把两间空教室借给了郑州来的一个培训机构，走廊里每天有陌生的面孔进进出出。两边的人偶尔在水房碰见，彼此看一眼对方校服，什么话也不说，各自打完水走开。黑板旁边的墙上除了期末倒计时，又多了补习班的手写广告，粉笔字写得歪歪扭扭，写着"暑假突破，高三领跑"八个字。

曹蔚在错题墙上贴了一张新的便利贴，上面写着："暑假不是用来休息的，是用来拉开差距的。"字很大，感叹号画了两次。

陈柏言路过时停下来看了一眼，在旁边加了一张更小的纸条："但也不要太焦虑。拉开差距的前提是别把自己拉垮了。"

郭向阳又在陈柏言的字下面贴了第三张："你们能不能别在墙上写议论文？有这功夫多做两道题。"

三张便利贴并排贴在那里，大小不一，颜色各异，像十班这个暑假的三句座右铭——在拼命和保命之间摇摇摆摆。

讲台上的玉树捐款箱已经放了两个多月。透明塑料箱的边角磨得发白，里面的钞票新旧不一——有一块的、五块的、十块的，还有一张二十块的。最上面压着两枚五毛的硬币，不知道是谁吃早饭时剩下来的零钱。

宋乔某天经过时停下来看了看，说里面好像比上周多了。

"有人又捐了？"何嘉树凑过来。

"应该是。不知道是谁放的。"宋乔说。

"食堂涨价以后省下来的。"何嘉树看了一眼箱子，"最近荤菜多了五毛，好多人中午只打两个素菜。素菜一份一块五，两份三块。省下来的五毛、一块，顺手就丢进去了。"

宋乔没接话。她把捐款箱扶正了一下，箱底发出硬币滚动的轻响。

上海世博会的新闻在教室后排的收音机里沙沙地响。刘青岚在晚自习前调到新闻频道，广播里说世博游客突破四千万，中国馆的红斗拱已经成了最热门的拍照点。教室里没什么人接话——四千公里外的事情对他们来说，远不如眼前一张补课时间表来得真实。

可也不是完全无关。王子越在午休时忽然冒了一句："世博会要开到十月底呢。到那时候咱们就该期中考试了吧。"

这句话把远方跟眼前串在了一起。几个人同时抬头看了她一眼，又同时低头继续做题。

远方和眼前本来就是一回事。世博会、玉树、暑假补课、月考排名、饭卡余额——它们在不同的形状里，压在同一个人的身上。

常俊凡把同学录传回孙佳怡手里时，第一页已经快写满了。

不止孙佳怡一个人的时间表——全班大部分人都在上面留了痕迹。不同颜色的笔迹挤在一起：黑色圆珠笔、蓝色钢笔、铅笔写了又擦掉的灰印子。乍一看很像一张手绘的公交线路图，从新华书店到校门口，从县教育局到某个巷子里的补习班，每一站都有标注，每条路线都通向一个不想让家长担心的孩子。

常俊凡在最下面写了一行很小的字：

"周六下午空着挺好的。树莓该熟了，去的时候带个塑料袋。一盒二十个，你跟阿姨说你天天从那儿路过，她给你算十八。"

孙佳怡读完这行字，笑了一下。

那是她今天第一次真心实意地笑——不是客气，不是敷衍，而是从心里涌上来的一种松快，像七月傍晚忽然从大堤方向刮过来的一阵风，不大，但足够让人深呼吸一次。

段璟在旁边看见了这一幕。他看见孙佳怡把同学录抱在怀里，拇指遮住了角上那行小字的大半。他看见常俊凡已经转过身去整理自己的书包，像什么都没做过一样。

他心里忽然冒出一个念头，很安静地落在某个地方：如果将来真的要分开，至少在这个夏天，他们已经学会了把彼此的路线记在心上。不是爱情，比爱情更早。不是承诺，比承诺更稳。就是在一页被补课时间填满的纸上，有人留了一行关于树莓的话。

这已经足够让一个封丘的夏天从此变得不一样。

同学录第一页之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

同学录第一页夹在自由和照看之间，像她整个夏天的心情。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，孙佳怡同学录第一页写下的补课名单还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
