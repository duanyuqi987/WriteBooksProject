# 第016章 前排和后排之前

所属部卷：第一部：初秋三楼
时间：高一春天
地点：物理课后的教室

## 诗歌插页 P016：原创现代诗

我把早晨放低，
让一场雨从课桌边慢慢经过。
你的名字没有被我喊出，
只在作业本边缘一处蓝色墨点旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写前排和后排抵达之前的坡度。春天的物理课总在下午最困的时候。窗外树叶刚冒新绿，阳光透过新芽在课桌上投下稀稀疏疏的光斑，教室里却全是圆珠笔划过纸面的沙沙声——物理老师让大家当堂完成一道受力分析题，限时五分钟。

陆允川做完了。他的作业本上画着一个物体和三个箭头——重力、弹力、摩擦力，每个力都标了大小和方向。他把本子合上，等着老师喊"从后往前传"。

"你做完了？"程砚从旁边探过头来，手里还捏着笔，纸上只有两个歪歪扭扭的力，"这摩擦力方向到底是往左还是往右？""看相对运动。""我知道看相对运动——但你这三个箭头画的比我的标准十倍，你是不是背了步骤？"陆允川把本子翻过来，盖在桌上。程砚笑了一声，没再偷看，低头继续和摩擦力掰扯。

物理老师说了一声"传"，整个教室开始动。从后排往前排传，每列一摞。陆允川把自己的本子递出去，前桌接了，摞在下面继续往前传。他的目光跟着自己的本子往前移动——它被一只手接过去，叠在下面，再被另一只手接过去。传到第三排时他看见自己的本子被搁在了另一本旁边——那一本的封面边角有一处蓝色钢笔水的墨点，圆圆的，像一滴没擦干净的雨。那是夏知遥的作业本。他的本子和她的被叠在了一起。

"你看题还是看本子？"程砚回头看了一眼他视线的方向。
"都看。"
"那你倒是把第三题看对——你刚才说摩擦力等于三牛，但题里给的是五牛。"
陆允川低头看了一眼自己的作业本——刚才他回答程砚的摩擦力值时确实说了三牛，而题目写的是五。
"你在想什么？"程砚压低声音。
"想错了。"
"你不是想错了，你是根本没想。"

叶澄从前排回头借橡皮。她已经把作业本交上去了，现在正在帮英语老师收拾一叠散开的单词卡。她拿着橡皮敲了敲陆允川的桌角，说了一句"别把别人的本子当成你的草稿纸看"。语气平平的，像在提醒他把水杯挪开别碰湿作业本一样。陆允川接过橡皮，低头在自己的一处错误上擦了擦，发现她说的"错误"可能不在纸上。

春天下雨，窗外开始飘细雨，雨点打在法桐新叶上，发出细碎的啪嗒声。教室里有人叹气——因为下午的体育课多半要改成室内自习了。后排两个男生开始讨论如果真的下雨就去体育馆占羽毛球场。陆允川听着这些声音，忽然发现自己的作业本还没有被传回来。其实已经传回来了——埋在前排那摞批完的本子里，而他在等的不是自己的分数，是另一个人翻开自己本子时的样子。

夏知遥这时还只是人群里的一个名字。她把本子交出去时没有回头，只低声提醒同桌："最后一道题单位别漏了，他上次扣了我三分。"她说得很轻，像把一枚图钉按进软木板。陆允川听见了她的语气、内容、以及她说话时手指压在作业本边角的小动作——所有这些都比他自己那道摩擦力题记得更牢。
