# 第024章 操场边的单词旁观

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场跑道

## 诗歌插页 P024：原创现代诗

她从跑道边经过，
不是为了进入谁的诗。
她在想听写，
想周末回家的车，
想母亲电话里没说完的累。
少年把风听重了，
风只是把校服吹起一角。

夏知遥那天去操场，是被林曼拉去的。

林曼说晚自习前走一圈，脑子会清醒。夏知遥本来想留在教室背单词，但坐太久肩膀酸，便拿着书跟她下楼。操场边有卖冰水的小摊，几个男生围在那里，塑料袋被风吹得哗啦响。

“你最近好拼。”林曼说。

“怕跟不上。”夏知遥翻着单词书，“老师讲太快了。”

“你都怕，我们怎么办？”

夏知遥笑了一下：“一起怕。”

她们沿着跑道慢慢走。林曼讲起班里一个男生把 chemistry 拼成了奇怪的样子，笑得差点把橙汁洒出来。夏知遥跟着笑，笑完又低头看书。她的书页里夹着一张小纸条，上面写着今天要背的二十个词。

经过西侧看台时，她感觉有人在那边，但没有认真看。看台上常有人背书，也有人偷吃东西，还有人借着背书的名义发呆。她不想把每一道目光都变成需要处理的事情，那样太累。

林曼忽然压低声音：“后排那个陆允川是不是坐那儿？”

夏知遥没有转头，只说：“可能吧。”

“他好像经常在。”

“操场又不是谁家的。”夏知遥把书合上，“走吧，预备铃快响了。”

她说得平常，不是冷淡。她只是本能地给事情划了边界。一个人的出现可以被看见，但不必立刻被解释；一个人的认真可以被尊重，但不必变成另一个人的负担。

回教室的路上，她在楼梯口停了一下，把小纸条重新夹进单词书。上面有一个词：silent。她默读了一遍，忽然觉得这个词很好。安静，有时候不是空白，而是一个人把自己的生活好好拿在手里。

晚自习铃响，她坐回前排，摊开本子，开始默写。后排有没有人看她，她并不知道。即使知道，她也只能继续写自己的单词。
