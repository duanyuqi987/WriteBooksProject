# 第028章 第一次说话之后

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：教室前门和讲台之间

## 诗歌插页 P028：原创仿宋小令

语落阶梯，
六字响如溪。
欲把初逢藏一页，
却因课后日偏西。
照见少年低。

第一次说话之后，陆允川表面上没有变化。他还是每天背单词、做物理题、跟程砚一起吃饭，物理作业本上的受力分析图还是画得很标准，早读课还是偶尔会读窜行。但他在心里把那六个字归档了——他给自己的内心开辟了一个"已确认联系人"的文件夹，里面至今只有一条对话记录：某年某月某日下午课间，物理课代表把作业本撒了一地，他帮她捡起来。她说"谢谢"，他说"没事"。他甚至给这条记录编了一个编号：CONV-001——"CONV"是conversation的缩写，"001"是因为他计划收集至少三位数的对话记录。当然他知道这不太可能，但编号野心还是要有的。

第二天物理课，课代表照常发作业本。陆允川接过自己的本子时留意了一下封面——本子边角有一处浅浅的指纹，可能是昨天捡本子时沾上的粉笔灰。当然也可能是印版时机器留下的印刷瑕疵。他在两种可能性之间摆荡了三秒钟，最终告诉自己那百分之百是粉笔灰——这种小小的谎言对他来说已经不是第一次了。而且它很有用：只要告诉自己那是指纹，今天整个下午的学习状态就会提高百分之二十。

程砚在课间观察他的状态。陆允川今天的橡皮没动过——平时他做物理题时会反复擦改，今天橡皮原封不动地待在笔盒里。"你今天状态不错？"程砚问。"还好。""昨天那六个字的效果能持续多久？我看看明天要不要帮你制造一个'第二次说话'的场景——比如传卷子的时候我故意把水杯打翻，然后你们俩一起拿抹布擦桌子——""不需要。"你刚才说不需要的时候居然在笑——你以前说不需要都是面无表情的。"

午饭时食堂今天吃回锅肉。不锈钢菜盆里堆着肉片、青椒和蒜苗，油光发亮，热气把打饭窗口的玻璃熏了一层雾。陆允川打了满满一盘——米饭铺底，回锅肉盖在上面，菜汁浸进米饭里，饭粒变成了浅褐色。他坐了程砚的对面，程砚正在对付一块特别肥的肉片，把肥肉拨到盘子边上。刚好夏知遥从窗口那边打过饭往回走时，经过他们这桌——她端着盘子，盘上搁着一碗白粥和两个包子，小心地绕过一把往外伸的椅子。陆允川抬起头，目光刚好和她的对上。她没有停步，但是——这次和以前不一样——她的嘴角动了一下。幅度很小，大概只有嘴唇最外角向上弯了大概两毫米，持续了大约零点五秒。那不是微笑——微笑需要至少嘴角上扬五毫米——那是"我看见你了"的默认信号，那种你在走廊里碰到半生不熟的同学时会给的信号：不认识你，但知道你在自己班上。但这也是陆允川从她那里收到的第一个双向信号。不是她发的信号大，而是他第一次确认自己不在她的认知盲区里。她把餐盘端到了另一边的桌子，和蒋雯坐在一起。蒋雯马上开始汇报化学课的作业截止日期——"星期三，实验报告要交到三楼办公室，别放错格子了"——夏知遥夹了块肉放进嘴里点了点头，嚼的时候腮帮子鼓了一小团。

陆允川低头把回锅肉翻了个面，筷子戳在肥肉上，没夹起来。程砚看见了他的筷子动作——筷子尖在肉上停了两秒以上——但他这次没有戳穿。

下午体育课，老师让两人一队练排球对垫。体育馆里回声很大，排球砸在手臂上的声音"砰""砰"地在空间里弹来弹去，空气里有地板蜡的味道。陆允川和程砚组队，程砚的垫球很烂，每次球飞过来都是斜的，陆允川要跑两步才能接到。隔壁训练区里夏知遥和蒋雯一组——蒋雯垫球比她稳，她接球时手臂会下意识弯曲，球有时候会歪。排球滚到场外时，陆允川跑过去捡——正好夏知遥也去捡同一个球。球在两个人之间滚了半圈停住。她先到一步，弯腰把球捞起来——球是黄蓝相间的，沾了地板的灰，她在裤腿上蹭了一下——抬头看见他，这次说的是"你的在那边的——是你的"。她的声音不变，还是不高不低，但她主动说了五个字，而且是完整的一句话，有指示性——"在那边的"——她在帮他定位。比昨天的"谢谢""没事"少了一个字，但因为是主动发起的，意义翻了倍。陆允川说了声"哦好"，然后去捡了自己的球——那个球确实在另一边，贴着墙根。他走回场地时，程砚从他手里接过球，说了句："五个字。今天的外交成果。明天争取再翻倍——十一个字就够你写一篇周记了。"陆允川把球往程砚怀里一塞，程砚笑着垫了一个歪球。

回宿舍的路上，路灯把他和程砚的影子拉成两根歪歪扭扭的柱子。程砚在聊明天化学实验的预习报告怎么写，陆允川嗯嗯地应着，实际上在脑子里计算另一组数据：到今天为止，他和夏知遥之间的对话总字数是六加五等于十一。加上今天的"你的在那边的——是你的"是她主动说的，权重应该乘以一点五——所以有效字数大概是十四点五。这十一个字（或十四点五个等效字）中的每一个停顿、每一个语调、每一个弯起嘴角的弧度，都已经被他用不同的速度、不同的角度回放了十几遍。回放到后来他自己都觉得累了——脑子里像有一个坏掉的录像机，不停地自动重播——但累和停是两种完全不同的事。他累了，但他没有停。就像操场上的跑圈，腿累了可以放慢速度，但不会停下来。因为他已经在跑道上了。
