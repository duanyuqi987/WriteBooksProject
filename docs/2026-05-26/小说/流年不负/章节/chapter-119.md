# 第119章 拒绝

时间：2018年2月（农历正月初五）

地点：青阳县 顾临风家

卷次：卷六：平行轨道

正月初五。破五。接财神。

县城整天鞭炮不断。有的近——就在楼下，炸开的声音脆而短。有的远——在城南那边，闷闷的，像远方的雷。窗户关着也挡不住——玻璃在每一次巨响中轻微抖动。硝烟味从窗缝钻进来——有点辣，又带着木炭的焦甜。

他妈没出门。在缝东西——他的旧毛衣，袖口脱线了。是一件深蓝色毛衣，高中时候就开始穿的那个颜色。袖口被磨出了一道口子——大概两厘米。他妈拿了针线在沙发上一针一针往回钩。线是蓝色的——跟毛衣原来的蓝色不太搭。旧线是深蓝——洗过很多次后变成了偏灰的深蓝。新线是藏蓝——刚从针线盒里拆的。他妈说"差不多——穿了反正看不出来——谁会盯着你袖子看"。针在她手里走了几十个来回。他坐在对面翻着一本旧书——《古文观止》。看到哪里了——他自己也不确定。

安静了一阵子。墙上的钟在整点报时——"叮——"了四声。下午四点了。外面的鞭炮忽然全停了。寂静张开。然后几只远处的狗叫。然后又有了零星的一点鞭炮——"啪——"——"啪啪——"

"初七你回楚州。张阿姨说她初八就给人姑娘打电话——安排见面。你下个月中随便找个周末——见一面就行。在楚州找个茶馆——一茶一座——或者那个什么——'悦来'——随便。见个面。喝杯茶。看一看。不喜欢就算了。见个面能少块肉吗。"

他妈没有抬头。注意力在毛衣的袖口上。一针。再一针。针从毛衣的正面穿到反面——又从反面穿回来。线在空中拉出一道细长的蓝色弧。

"不去。"

声音比他预想的更急。像打开水龙头时原本应该出热水，但那一瞬间喷出来的是冷水——刺人的凉——刺得他自己都打了个激灵。他自己都没防备。话就冲出去了。

他妈的手停了一下。针尖扎进毛衣——停在里面没拔出来。他的视线落在他妈的手上——那双手皱了，皮肤上有冬天干燥带来的几道白纹。然后她拔出了针。

"怎么了。"

"我去见什么。她初中还是高中——我都不知道。"

"你等见面了不就知道了——"

"初中还是高中。我自己查。"

"你查她干嘛——你认识她？"

"我查——不认识。不用查。反正不去。"

"为什么不去——"他妈把手里的针线放下——针插在膝盖上的那团线球上。"就是见个面。喝杯茶。又不是让你明天就结婚。人姑娘也是青阳人——在楚州教书——跟你一个城市——多好的条件。你这个人怎么这么别扭。说个话这么凶——你那语气——像谁欠了你八百块钱似的。"

窗户外面鞭炮响了一大片——噼噼啪啪的——夹杂着几声响炮——"砰——砰——砰——"。有人在街上大声喊——"财神来啦——"然后是几个人同时放鞭炮——噼里啪啦——天边那片白烟越来越浓。空气里的硝烟味从窗缝挤进来——钻进鼻腔——舌尖能尝到苦味。

屋里没人说话。

他妈把针拔出来——放在嘴里抿了一下线头——舌尖在嘴唇之间推了一点——把线头抿湿。然后对着光——把线穿过针眼。眼睛眯了眯——老花镜反着光。手很稳。穿过去了。

"你认识她？"

声音平淡。不是试探——是问题的平调。问得很轻。但不盼着答案是或不是。好像答案是什么都没关系。

"不认识。"

他回答得很快。太快了。快得像没听过那个名字一样。快到"苏晚晴"三个字从他舌尖弹出去又弹回来——被四个字（"不认识"）挡住了所有可能。但他说这四个字的时候——声音里有一点点不一样——"不——认——识"——中间有一个极其细微的卡顿。卡在"认"和"识"之间。那个停顿不是语言上的——是呼吸上的——像心脏漏跳了一拍然后被下一拍的收缩盖过去。

他妈又穿了一针。慢慢地说："不认识就不认识。你吼什么。"针从毛衣的反面穿回来——线在外面拉直。"我还以为你认识——不好意思说。"

他没接话。

低下头。穿着拖鞋的脚踩着沙发的边——沙发布是一块旧缎子——红色底，上面印着开得很繁的牡丹花和金鱼——金鱼眼睛是黑的。布料有些地方已经磨薄了，能看到底下的经纬线。小时候他坐在上面写作业——腿不够长，脚够不到地板——就缩在这块布上。一缩就是整个下午。现在他坐在上面——应对他妈说出来的"苏晚晴"三个字。十七年过去了——作业变成了相亲。沙发还是那张。缎子还是那块。

不知道过了多久。大概三分钟。或者五分钟。外面的鞭炮停了——又起了一波。节奏混乱的——东边的"啪啪"刚落下，西边的"噼里"紧接着起来。在那嘈杂的空白里——他妈又开口。

"你这孩子谈了也是你说的。不谈也是你说的。"他妈没多说下去。把针从膝盖的线团上抽起来——穿过了另一针——"你不去相亲当然算了。反正这辈子你的老婆你自己找。等到三十岁别来怨我不管你。我替你操心了——你不领情——我也没办法——"

顾临风放下了手里那本旧书。手在书封面上停了一下。然后站起来。往自己房间走。书放在茶几上——封面朝上——《古文观止》。翻到了第174页——是欧阳修的《祭石曼卿文》。那句话还看得见："呜呼曼卿。生而为天下重。死而无憾乎道。"

他关上了房间的门。门关上——锁舌弹进门框——"咔"。然后他靠在门后——背抵着门板。门板的木头传着他的体温。

在床边坐下。床垫往下陷了一点。弹簧发出一声细微的吱嘎。

屁股下面的床单是新的——蓝白格子。他妈几天前刚换的。棉布还没洗软，纤维有点硬。冬天干的皮肤碰在棉布上——有一点糙糙的触感。手指在床单上滑过——沙沙的。

窗外又有鞭炮响。不远——就在楼下那条街上。能听到小孩子在笑——"再放一个——再放一个——""我不敢——你来——""你胆小鬼——"——然后是打火机拨动滚轮的声音——嗤——嗤——。鞭炮点着了——那小火星沿着引线往下跑——然后"噼里啪啦"炸开。孩子们的笑声和"哇——"的惊讶同时炸开。鞭炮响完了有短暂的安静——安静里能听到硝烟飘散的细微声响，和远处另一台电视里播放的戏曲声。然后火药味从窗缝里透进来——那苦涩又带着一点碳甜的味道，在冬日的空气中沉在更低处。

他的呼吸不匀。他自己知道。不是喘。不是缺氧那种——身体不需要更多氧气。是呼吸之间那个停顿没了——平时吸进去，在肺里停那么一瞬（像两句话之间的逗号），然后呼出来。现在逗号被抽掉了。吸进来马上就要呼出去——吸——呼——吸——呼——节奏变得急。像一个人在跑步机上，但身体是坐着的。他自己不能控制。手指在床单上抓紧——松开——又抓紧。

他躺在床上。把枕头拿过来抱在胸口。枕头是荞麦皮的——里面的荞麦皮发出细细的沙沙声。那种声音——干燥的、植物的——在小时候陪过他整个童年。

蓝色的枕套——有点旧了，棉线磨薄了。枕套边缘有两处脱线的——又缝上了。是他妈几年前缝的——针脚走得好，线头藏得齐。跟他的毛衣一个命运——此处脱线彼处缝上。那些经年累月的磨损是遮不住的——即使缝上了，缝线的地方跟原来不一样——更凸，更硬。

他想着高中的时候——枕套跟现在的差不多。只是那时候没这么旧。高中时他经常半夜不睡觉——翻来覆去。辗转反侧。想同一个问题。要不要告诉她。要不要写一封信。要不要在走廊上碰到她的时候多看她一眼——如果她回看了——要不要抓住那一秒说点什么。

那一秒从来没有发生过。

现在他二十三了。当了老师。考上了研——381分——超了线29分。写了不止一首诗——好几首都被划掉了——只有零星的句子留在了那本黑皮笔记本里。但答案跟高中一样——"不知道怎么告诉。"而且现在更难。八年比三年更难说出口。因为时间越长——"从来不说"本身变成了一个问题。"你为什么从来没告诉过我"——比"你为什么不告诉我"多了"从来"两个字。这两个字重得搬不动。

八年。

八年没有说过一句话。

他妈要是打电话过去——"姑娘你好。我是顾临风的妈妈——你们张阿姨介绍的——下周六下午三点——一茶一座。你方便吗？"她接到电话——电话那头传来一个陌生中年女人的声音。她愣一下——"顾临风的妈妈？您说他叫——"——"顾临风——照顾的顾——来临的临——刮风的风。他也是青阳一中的——你们是同学——在楚州读书——"她握着手机的手会不会收紧一点。她会不会知道这个名字——她会不会记得。她会不会在通话结束后——挂了电话——看着手机屏幕上的通话记录——上面写着"顾临风的妈妈"——盯着这几个字发呆。然后打开通讯录——往下翻——翻到"S"那一栏。看到那个名字。"苏晚晴。手机号***。微信号***。"都没有未读消息。从来没有。

她会不会说——"顾临风——你说的是青阳一中2010级那个顾临风吗。个子差不多——不太爱说话——瘦瘦的。他以前坐倒数第三排——数学课经常被老师点名叫醒——但他语文特别好——作文每次都被贴在黑板旁边的通知栏上。"

或者她根本不会记得。她可能会说——"顾临风——这个名字没什么印象呢。"然后说——"好——我有空——到时候见一面。几点来着——周六下午三点。"语气很平常——像对待任何一次普通的相亲。

那他推开一茶一座的门——她坐在靠窗的位置——马尾——白衬衫——手里端着杯子——抬起头——看到他——表情先是一秒的迟疑——然后是——"——顾临风？"然后他站在门口——背着书包——手里不知道拿着什么。两个人的视线在空气里交汇。然后他说——"嗨。"

然后呢。

两样他都不能想。一样太感动——她记得他。八年了。她记得他坐倒数第三排。她记得他的语文作文。她记得他的存在。这太感动了——感动得他自己承受不住。另一样太伤人——她不记得了。她的记忆中已经不存在"顾临风"这三个字了。他是一串从她生活里滑出去的符号。没有任何意义。只有编号——跟张三李四王五挤在一起。她记不得他的脸——他的书包——他写在课本空白处的字。她记不得他高二那年运动会——给她递了一瓶水。她记不得毕业那天——他站在校门口偷偷看了她。她记不得每一次在朋友圈她发状态他都没点赞。

他把脸埋进枕头。眼眶有点酸——泪腺在往外面分泌液体——但他没有让它们掉下来。他翻了个身——压住那股从鼻梁传到眼睛后面的酸感。没掉眼泪。不会掉眼泪。

外面又有鞭炮。这次是远处——隔着三四个街口。闷闷的——"砰——砰——"——像很远的地方有人在用盲杖敲地面。然后安静了。完全的安静——持续了大概十几秒。然后又有狗叫，然后远处又有了鞭炮声。像楚河的波浪——一潮一潮的。

他妈的声音从客厅飘过来。她在讲电话。隔着门——隔着他的呼吸声。声音传到房间时已经变了——但大致的意思是在往这边飘。对方大概是张阿姨。他妈的声音有一种克制的无奈——"他不想见。他说不想见——不知道——他说忙——他说想自己找——他不想见我有什么办法。人家姑娘那边你先别提了——别让人家空等着——"语气里有妈对孩子的妥协——但那种妥协不是认输了——是一个母亲认识到孩子还没准备好——而她也知道不能用蛮力推——推也没用。她顿了顿又说："算了——随他吧。不过你先把姑娘的微信给我留一个——万一他哪天想通了——"

他坐起来。身子从床上弹起来——背直了一下。然后下床。脚底踩在地板上——凉的。

把枕头放回床头。拍了两下——调整好方向。站起来。走到书桌前。拉开抽屉——那个抽屉从来不锁——因为这是在自己家。但里面的东西跟锁在楚州宿舍里的东西是一模一样的——黑皮笔记本。那封信的复印件——不是复印件——是原件的副本——不——只是同一本笔记本。他一直带在身上。从楚州背回青阳，又从青阳背回楚州。

黑皮笔记本。打开。布纹封面在指尖摩擦留下轻微的质感。翻到最末页——从后面往前的。那里有九年前写的字。2010年9月1日——高一下午开学第一天的日期。那天下午阳光从窗户外斜照在他的胳膊上——胳膊上的汗毛在光里变成了金色。他在纸上写了一个字母——"S"。然后划掉了。横线很深——钢笔尖压进纸纤维——划的时候纸被拉出一道细微的撕裂——纸都快被划破了。但没破。横线在线条的末端轻微收了一下——大概是划到头的时候他后悔了——笔被提起来一毫米——划出去的力软了。

八年了。那行线还是那么深。蓝黑色的墨水，横穿过"S"的上下曲线。"S"在横线下面变成一个被关在栅栏里的字母——还能看到，但已经被封住了。

他把笔记本翻到新的一页。纸上还是空白的——米黄色的横格线。拿起笔——钢笔，英雄牌。笔尖在纸面上点了几下——好像要写，又收回来。墨点聚集在笔尖的缝里——然后他写了。字很小。笔画很用力。纸背面鼓起来——摸上去是一道道小的凸痕。写了半个多钟头。最后一行写完。把笔放下。看着自己写的字。下巴搁在手掌上。

然后合上。手指在封面划过布纹——像在告别一个熟悉的房间。放回抽屉。关上。抽屉推到底——那个跟楚州宿舍抽屉一样——有点卡的——用膝盖顶了一下。

回到床边坐下。床垫又是那声细微的吱嘎。门还关着。门缝底下透进来一道光——客厅的灯。光在地板上投了一根很细的线条。

深呼吸一次。吸——停——呼。两次。吸——停——呼。三次。吸——停——呼。呼吸之间那个停顿——那个逗号——慢慢找回来了。
