# 第086章 录取通知书之前

所属部卷：第二部：六月以后
时间：七月下旬
地点：镇上邮政所

## 诗歌插页 P086：原创现代诗

邮政所门口有一棵老树，
树荫下停着几辆车。
红色信封还没到手，
人已经先想起远方。
远方不响，
只在柜台后面
被人一封封翻找。

七月下旬，录取通知书开始陆续到镇上。

邮政所每天都有人来问。柜台后面堆着快件，工作人员戴着袖套，一边翻登记本，一边喊名字。陆允川第一次去时，通知书还没到。第二次去，工作人员让他明天再来。

第三天，父亲陪他一起去。

太阳很大，邮政所门口的树荫被晒得很薄。父亲把摩托车停好，说：“别急，该到会到。”

陆允川点头。其实他急的不只是通知书。他也听说夏知遥的分数不错，可能会去另一个城市。迟到表白之后，他们没有再单独联系，只在班级群里看见彼此偶尔发言。

柜台里的人翻了几封快件，问：“陆允川？”

他立刻抬头。

“身份证。”

陆允川把身份证递进去，手心有汗。工作人员核对后，从一摞红色信封里抽出一封。信封很硬，烫金字在日光下亮了一下。

父亲站在旁边，脸上没什么夸张表情，只说：“拿稳。”

陆允川接过通知书，忽然觉得它比想象中轻，也比想象中重。省城的名字印在上面，像一条路终于从纸里伸出来。
