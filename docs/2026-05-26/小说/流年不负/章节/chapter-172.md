# 第172章 婚礼准备
时间：2019年2-4月
地点：青阳县/楚州
卷次：卷九：执手之约

过完年。正月十五一过，顾母就开始翻黄历。

家里那本黄历是镇上供销社发的。封面印着"农历己亥年"，里面的纸很薄，翻起来哗哗响。顾母戴上老花镜——镜腿用胶布缠过一遍，缠得不平整，但她舍不得换。她坐在客厅日光灯下，把黄历放在膝盖上，一页一页地翻。

每一页上都印着宜忌。宜嫁娶、宜纳采、宜祭祀；忌破土、忌动土、忌出行。红字是宜，黑字是忌。她看得很仔细——红字多的日子在纸上画个圈。圆圈画歪了，又补了一个。

"五一劳动节——这个日子怎么样。"

顾临风在旁边备课。蓝色教案本摊在茶几上。他抬头看了一眼黄历上那个歪歪扭扭的圈。

"行。苏晚晴也放五一假。大家都有空。"

"关键是宜嫁娶——你看到没，五个红字。五个。这一天全年的好日子排名——"

"妈，黄历上排第几不重要。"

"怎么不重要。你结婚就一次——"

"我知道就一次。所以日子不重要。人重要。"

顾母愣了一下。放下黄历。看着他。看了几秒。

"你这孩子。从小到大不说人话。一说就说这种。"

顾临风低头继续备课。

日子定了。五一。五月的第一天。春天刚过，夏天还没到。月湖边的柳树是全绿的，湖面上能看到水鸟教小水鸟划水。

场地定在月湖边的那家酒店。

不是什么豪华酒店。就是湖边的一家中式餐厅——白墙黛瓦，两层楼。老板是青阳本地人，姓陈，五十来岁，做了一辈子餐饮。他说"我这家店没接过多少婚宴，但接过的每一对新人都没离过婚。"也不知道真的假的。

餐厅的二楼是个大厅，三面都有窗户。朝湖的那一面全是落地窗——窗外就是月湖的水面。湖对岸能看到水塔和柳树。晴天的时候湖面是蓝绿色的，阴天是灰蓝色的。他们看了场地之后都没再看别家。不是因为这家多好——是因为窗外的湖。

"这里。"

"嗯。"

"六年前在湖边。六年后在湖边。"

"还有六十年后。也在湖边。"

他说这个话的时候没笑。但她知道他不是在开玩笑。

请柬是手写的。

苏晚晴说"不用印刷的。手写的每一张都不一样，收到的人能感觉到。"顾临风说好。去书店买了五十张红色卡片——一面是描金喜字，一面是空白的宣纸。墨水买了黑色和金色两种。黑色写名字。金色描边上那个喜字。

周末下午。两个人坐在茶几两边。茶几上摊开了红卡片、墨水瓶、毛笔、钢笔、一方砚台。砚台是陈维中老师送的——顾临风读师范时一个老师听说他要结婚，寄过来一块端砚。不是新的，是老师年轻时用的。砚堂里还有旧墨渍——洗不掉的，但老师说不影响用。

"你先写。我看看。"

顾临风拿起钢笔。笔尖在手背上试了一下墨水——一个小黑点。然后在红纸的左上方落了笔——"谨定于2019年5月1日"。

他的钢笔字还是不好看。但比高中强多了。至少结构不散。横竖的力度也稳了很多——当老师的，板书练出来了。

写了三张。每张都差不多。不漂亮。但端正。每个字都很用力——纸背面的凸痕很深，摸着像盲文。

"你写吧。"

苏晚晴接过笔。她写的字比他好——她从小练过一点点毛笔字，没系统学过，但课间在黑板上出黑板报的时候，粉笔字写得很工整。钢笔落在宣纸上的力度比他轻——每笔都不深不浅。写在红纸上的字能看到细微的笔锋——起笔和收笔处有毛笔字的痕迹。

她写她的亲戚同学。

他写他那边的。

一人一半。

"赵一鸣"——他写到这个名字的时候笔停了。然后在那行字后面多写了一行字：

"你当年说的对。"

写完自己看了看。把卡片放好。继续写下一个。许若琳。苏晚晴的表姐。顾家的三叔。苏家的二姨。一个名字接着一个名字。五十张请柬写了整整两个下午。中间吃了两顿饭，喝了五杯茶。

写到手指酸的时候她就甩甩手。钢笔在指关节上印了一个小黑圈——墨漏了一点。她去洗了——水龙头冲了很久，冷水把墨迹洗淡了但没洗掉。手指上还留了一个淡蓝色的印子，像一块小小的胎记。

"洗不掉了。"

"过几天自己就褪了。"

"不褪也行。"

她在台灯下看了看手指。然后继续写。

三月底。他们去了一趟婚庆公司。县城里的婚庆公司——不大，但业务范围很全：婚礼策划、婚纱租赁、化妆、跟拍、司仪。店里有一面墙挂满了婚纱照——各种风格的，有的在草地上，有的在海边，有的在教堂里。每张照片上的新郎新娘都笑得很灿烂。

店主是个三十出头的女人，姓刘，戴着金色耳环。问他们要什么风格。

"最简单的。"顾临风说。

"简单也有简单的做法——田园风、森系、极简——"

"不要风格。就是把该做的事做一遍。"

刘店主看了他一眼。又看了苏晚晴一眼。苏晚晴在翻桌上的婚纱画册。

"你男朋友——哦不对，未婚夫——"

"他就这样。"苏晚晴替他说了。"不是省钱。是真的不喜欢做多余的事。"

刘店主懂了。翻出一个简洁的方案——白色背景、不铺花海、不摆拱门、不请乐队。司仪用最简单的话走完流程。背景音乐是古筝曲。

"主持词你们自己定吗。"

"不要主持词。"顾临风说。"我自己说。"

"你自己主持？"

"不是主持。是说几句话。"

刘店主记下来了。用笔在表格上勾了"新人发言"。然后抬头看了顾临风一眼——这个男人说话不多，但说出来的每个字都像是想好了的。

四月初。请柬全部写完。

顾临风去邮局寄了一批——给外地的同学和朋友。剩下的当面送。赵一鸣的那张他留着亲自给。约在青阳县一中门口——那是他们认识的地方。

赵一鸣骑车来的。还是那辆旧电动车，换了三次电瓶。车把手上挂了个塑料袋，里面是两瓶啤酒。

"喏。庆祝你脱离单身——哦不对，你马上脱离——也不对——你是自愿脱离。"

顾临风把请柬递过去。

赵一鸣接过去。翻开看了看。看到那行小字的时候停住了——"你当年说的对。"

他抬头看顾临风。

"我说的对。我当然对。高一第一天我就坐你旁边。你看了她多久我看了你多久。"

他把请柬合上。装进衬衫口袋里——贴胸口。

"你的婚礼。伴郎是谁。"

"你。"

"算你还有点良心。"

他们在校门口站了一会儿。隔着铁栅栏能看到里面教学楼——三楼的窗户，第三个教室，最后一排。他们的教室。那扇窗户现在关着——高二的学生在里面上物理课。物理老师在黑板上写公式，粉笔划过去发出很细的尖叫声。跟十年前一模一样。只是窗里的人换了。

赵一鸣把一瓶啤酒递给他。瓶盖用手拧开——他不嫌手疼。两个人靠在栏杆上喝了一会儿。风从月湖方向吹过来，把啤酒瓶口上的泡沫吹散了。

"你还记得她说那句话吗。"

"哪句。"

"不管去哪。以后都要回来看这个湖。"

"记得。"

"你们做到了。以后还要做到。"

顾临风没说话。把瓶身举起来——朝着月湖的方向。啤酒瓶底对着太阳，瓶口对着他的嘴。

"回来了。"

三个字。

赵一鸣在他的肩膀上拍了一下。比顾父在提亲时拍的轻一点，比苏父倒茶时拍的重一点。力道刚好。

四月下旬。苏晚晴去试了婚纱。

不是店里那些大蓬裙。她挑了一件简单的——白色及地长裙，没有大裙撑，没有亮片，没有长拖尾。面料是缎面的，在灯光下有一点很淡的光泽。领口是方形的。袖口是七分袖。腰身收得刚好——不太紧，但能看出她的身材。

她站在试衣镜前。转了一下。裙摆在脚踝上摆了半圈。

"就这件。"

她没问好不好看。因为镜子里她自己看到了——那件婚纱穿在她身上，不像婚纱。像夏天的长裙。像她高中时穿过的那种棉布白裙子——只是更长、更白、更有分量。

顾临风的礼服也是最简单的。黑色西装。白衬衫。没有燕尾、没有领结。衬衫的领子是方领——他坚持不要小方领，说"不舒服"。穿着试衣服的时候照了一次镜子——然后就脱了。说"行。"

老板问他"不再试试？"

他说:"衣服不重要。婚礼是穿给人看的。我是穿的。"

四月的最后一周。两家的妈妈开始张罗。顾母包了喜糖——一个红袋子里装六颗：大白兔奶糖、徐福记花生酥、玉米糖、巧克力、红枣、一块喜饼。装了六十份。手上沾满了糖粉——她舔了舔手指，甜的。

苏母在准备床品——大红的四件套。绣着龙凤呈祥。苏晚晴说"太红了"。苏母说"结婚不红什么时候红"。最后放在柜子里——说婚礼前一天铺。

一切准备好了——请柬寄了、礼服有了、场地定了、照片拍了、两本结婚证并排放在抽屉最里面、旁边是一叠诗。

只差那一天了。
