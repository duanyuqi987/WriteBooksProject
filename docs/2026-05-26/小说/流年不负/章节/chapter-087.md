# 第087章 寒假回家

时间：2014年1月

地点：青阳县

卷次：卷五：各自南北

一月中旬。寒假。青阳县。

从楚州坐大巴回青阳——三个半小时。楚州长途汽车站在城西。他早上六点起床，七点到车站。候车厅不大——十几排蓝色塑料椅，椅子上坐了一半人。有人吃泡面，有人打盹，有人抱着蛇皮袋。他买了票，过检票口，上了车。坐在倒数第三排靠窗的位置。

大巴慢慢开出城区。经过师院南门——那个公交站牌在晨光里是绿色的。经过市体育馆。经过新华书店。经过人民医院。路线有一段和301路重叠——他认出来了。窗外的中山路口、公园西门——每个站名他都在手机备忘录里记过。他把头靠在玻璃窗上，看着那些站牌一个一个往后退。

高速公路风景单调——灰色的护栏、枯掉的灌木、偶尔的蓝色路牌。

到家的时候是下午三点。他妈在厨房里炖鸡，整个院子都是鸡肉和姜片的味道。他爸在院子里晾腊肉——一块一块的红白相间的肉挂在竹竿上，在冬阳下面有点发亮。

"回来了。"他妈从厨房探头看了一眼。"瘦了。"

"没瘦。"

"还嘴硬。晚上多吃点。"

他回到自己房间。房间和四个月前走的时候一模一样——床上的被褥收起来了，只剩一个床垫。桌子上的课本还在——都是高三的教辅，《五年高考三年模拟》、《黄冈密卷》、一本翻烂了的《语文基础知识手册》。窗台上落了一层薄灰。他用手指抹了一下，一条干净的印。

晚上吃饭——四菜一汤。他妈给他夹了三回菜。"多吃点多吃点，大学食堂能吃到什么。"他爸不怎么说话，只是偶尔夹一筷子，喝一口酒。

"在学校怎么样。"

"还行。"

"成绩呢。"

"班里第三。"

他爸举杯的手停了一下。"第三？不赖。"说完继续喝酒。

三天以后——腊月二十三，小年。

他去县城街上买东西。老城区有一条主街，两边的店铺挂满了大红灯笼和年货对联。到处是"福"字和"恭喜发财"。街上的人很多——有骑电动车的，有牵着小孩走路的，有三两成群站在路边聊天的。

他在一家卖对联的摊子前面停下来。摊主是个白头发的老头，把一摞一摞的对联摊在塑料布上。金粉字在红纸上闪着光，一股油墨的味道。他翻了翻——"春回大地""万事如意""家和万事兴"。翻了几张，没有买。

他正要走。抬头往街对面看了一眼。

然后他不动了。

街对面是一家糕点铺。门口摆着炒货摊——瓜子、花生、炒栗子。一个穿白色羽绒服的女人站在摊子前面，正在买东西。背对着他。但旁边还有一个人——一个中年女人。

白色羽绒服。马尾。侧脸转过来的一瞬间——鼻梁的弧度、下巴的线条。

是苏晚晴。

她妈在旁边。两人站在糕点铺门口，大概是也在办年货。她穿一件白色羽绒服——短款，领口有一圈褐色的毛。头发扎起来——马尾还是高中那个马尾。她从摊主手里接过一袋栗子，低头跟身边的人说了句什么。然后母女俩转身，往巷子那边走。

那条巷子——有老槐树的巷子。

她拐进去了。

马尾在拐弯的时候甩了一下。

全程大概十几秒。

他站在街对面。没有动。手里还捏着一张对联纸——不知道什么时候从摊子上拿起来的。

摊主抬头看了他一眼。"小伙子你要不要——"

他低头看了看手里那张纸。金粉字有点被捏花了。他把纸放回去。"不好意思。"他转身走了。

走的不是她那个方向。他绕了另外一条路。

回到家。他妈在厨房里炸春卷。油锅里的春卷滋滋冒泡，香味充满整个院子。他经过厨房的时候他爸在院子里劈柴。

"买到对联了吗。"

"忘了。"

"忘了？"他妈从厨房探头。"你这记性。"

他走进房间。把门关上了。坐在床垫上。手心里还有刚才捏对联时沾上的金粉。他用拇指搓了搓指腹——金粉簌簌地掉下来。

窗外的天空是灰色的。屋檐上蹲着一只麻雀——歪着头，跳了两下，然后飞走了。

她的白色羽绒服。马尾甩的那一下。拐进巷子的那个侧影。隔着一条街的距离——大概二十米。他隔着二十米，看了她十几秒。没有喊她的名字。她也没有看到他。她拎着一袋栗子，跟着她妈，拐进了那条有老槐树的巷子。好像什么都没有发生。好像那条街上根本没有一个叫顾临风的人在对面对着她发呆。好像这半年来的一切都只是他一个人脑子里的事。

他没有追上去。

也没有想太多。

他把手心里的金粉拍了。站起来，去厨房帮他妈洗菜。水龙头的水很冷。他把白菜叶子一片一片掰下来。他妈在旁边切姜——菜刀落在砧板上，当当当。
