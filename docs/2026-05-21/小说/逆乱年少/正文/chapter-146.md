# 第146章 大雪

时间：2012年12月20日，清晨至上午

地点：江城大学校园

卷次：第六卷：雪落无声

十二月二十号。大雪。

林玉起是被郭向阳的声音吵醒的。"下雪了！卧槽——好大的雪！"

他从床上坐起来，往窗外看。不是那种飘飘洒洒的小雪，是铺天盖地的大雪——雪片密密地往下砸，把对面宿舍楼的屋顶、路边的自行车、操场上的篮球架全部盖住了。整个世界变成了白色，只剩下一些黑色的线条——树干、电线、围墙的棱角。

入冬以来江城下过两次小雪。都是飘了一上午就停了，地上只积了薄薄一层，踩上去就化。但这次不一样。看这雪的势头，昨晚就开始下了。地上至少积了三寸厚。

郭向阳套上羽绒服就往外跑。过了一会儿回来报告："路都看不见了！梧桐树上挂满了！操场上可以打滚！"

林玉起穿好衣服，走到宿舍楼下。

校园在雪里完全变了一个样子。图书馆的灰墙变成了白墙，梧桐树的枝干上压着厚厚的雪，偶尔有一枝承受不住重量，"哗"地一下把雪抖落下来，像树在甩头发。路上的学生都走得很慢——怕滑。女生们撑着伞，男生们直接走在雪里，头发和肩膀上落满了白。

有个大一的新生在雪里写字——"期末不挂科"。

林玉起在旁边看了两秒。然后他从地上抓了一把雪，在掌心捏成了一个小雪球。凉气从掌心传上来，刺得皮肤发麻。他没有扔，只是捏着，雪在手掌的温度下慢慢融化，水从指缝里滴下去。

吃完早饭他去图书馆。路上碰到付小川——高中同学，在机械学院，平时不太见面。付小川穿了一件军大衣，领子立得高高的，像个过冬的猫。"好久不见啊。你头发怎么长了。"林玉起说"懒得剪"。付小川说"你气质越来越像文艺青年了，不对，文艺中年"。林玉起踹了他一脚，两个人在雪地里互相踹了几脚，留下一地乱七八糟的脚印。

从图书馆的窗户看出去，整座校园都是白的。雪还在下，没有停的意思。图书馆里很安静——考试周快到了，大家都在复习。暖气片发出咕噜咕噜的水声。

林玉起在靠窗的位置坐下。他拿出小本子翻开，看到上次抄完诗集之后空了很多页。他拧开那支英雄牌钢笔的笔帽。

在纸上写了一行字：

"下雪了。整个学校都白了。"

然后他停了。他觉得这句话太普通了，但还是继续往下写。

>
雪落万物白，天地一时新。
皆说冬日冷，谁知雪中温。
>
