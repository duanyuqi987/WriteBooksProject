# 第116章 日记本归档



回到住处那晚，许亦安把旧日记本摊在书桌上。台灯压低，光只照亮桌面一小块，旁边放着手机、数据线和一只空杯子。

房间很普通。窗外有人关车门，楼上椅子拖过地板，烧水壶发出短促的咔哒声。没有哪一种声音像告别，却每一种都让告别落到生活里。

手机相机对准纸页时，屏幕自动调亮，纸上的旧字忽然变得清楚。

他拍到一页写着林青禾名字的地方，手指停住。那一页旁边有一小块咖啡渍，形状像一片缩小的云。

他对自己说，拍下来就好，不要重读。说完又觉得这话像给小孩定规矩，忍不住笑了一下。

他起身去倒水，回来时手机屏幕已经暗了。那一小段离开，让他没有被那页旧字拖回去。

林青禾在这些纸页里出现过很多次，但她不属于这些纸页。她真正的人生在别处，有许亦安不知道、也不该补写的部分。

承认不知道，是他能给她的最后一点尊重。

有些保存必须带着距离。离得太近，就会把整理重新变成凝视。

周一鸣发消息问他校庆有没有收获。他想了想，回：拍了几张照。这个回答很窄，却正好。

聊天框停在这里，没有继续。许亦安把手机放回支架，继续一页一页拍。

数据线垂在桌边，轻轻碰着抽屉把手。

文件自动生成一串数字。他没有改名，任由那些数字排成冷静的顺序。

旧日记本被拍下来以后，没有变得更神圣。它只是从纸页变成图片，从抽屉搬到文件夹。

电脑右下角弹出导入完成，提示音很短，很快就消失了。

他不再给文件夹取伤感的名字。普通，是这次整理里最重要的分寸。

他没有把任何一页发给别人看。能安静保存，就不要把保存变成展示。

拍到最后几页时，他删掉几张模糊照片，重新拍。不是为了庄重，只是为了以后真的需要时能看清。

凌晨一点，他把文件夹命名为“大学”。两个字很轻，轻得让他松了口气。

桌上的空杯子留下一个水圈。他后来记得那个水圈，比记得某一句旧话更久。

这场归档没有仪式。没有音乐，没有长信，只有一盏台灯和一个人反复确认图片是否清楚。

旧日记本合上时，封面没有完全贴平。许亦安没有再按它，任由那一角微微翘着。

那一行字写着：文件夹名，大学。

给文件夹起名时，他原本想写得郑重一点，最后还是删掉了。普通名字更合适，像把东西放回柜子，而不是摆到台面上。

保存完成以后，他没有变得轻松很多，只是终于知道这些旧页可以待在一个固定位置。

这一章真正留下来的，是导入完成的短提示音。

许亦安关掉台灯，只留电脑屏幕亮着。文件夹安静地躺在桌面上，没有催他打开，也没有求他忘记。
