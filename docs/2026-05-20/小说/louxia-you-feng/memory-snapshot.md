- chapter-117《便签迁移之后》：许亦安把旧手机便签导出来，一条条看完，再存进电脑里一个普通文件夹。文件夹名字叫“大学”。之后，许亦安反复回想当天的细枝末节，把能记住的都写进便签，也把不能说出口的留在空行里。 他不再删除，也不再反复打开。保存就是告别的一种，但他要到更晚才肯承认这件事已经改变了他们之间的位置。
