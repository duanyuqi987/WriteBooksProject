# 第118章 楼下等她之后

所属部卷：第三部：寒风里的花
时间：大二五月
地点：女生公寓楼下圆石凳

## 诗歌插页 P118：原创仿宋小令

石冷晚灯低。
一句别等，吹散旧心机。
回执已交人已去，
剩他原地学边陲。

回宿舍路上，陆允川没有立刻给任何人打电话。他沿着操场外圈走了一圈，文印室的回执复印件还在书包里，硬硬地硌着背。

祁野在宿舍门口等他，手里拿着两份炒饭，“你那事等完了？”

陆允川接过饭，“嗯。”

“人高兴吗？”

“没有。”

“那你图什么？”

陆允川坐到台阶上，打开饭盒，热气扑到脸上。他很想说自己只是负责，只是顺路，只是怕她明天麻烦。可这些理由说出来都站不稳。苏晚枝那句“我会有压力”还在耳边，像一根细线，把他的体面抽开了。

祁野蹲在旁边吃饭，“你不是等她，你是在等一个能证明你很重要的反应。”

陆允川夹了一筷子米饭，没有咽下去。

“难听，但差不多。”祁野说，“人家说不用等，你偏等。等完还希望别人谢谢你，这就不太行。”

夜风从宿舍楼后面绕过来，炒饭很快凉了一点。陆允川把饭盒盖上，低声说：“我知道了。”

“你最好是真知道。”

那晚他没有写诗给苏晚枝。他在草稿纸上把“等待”两个字写了三遍，最后在旁边补了一句：没有被请求的付出，要先问自己是不是在讨债。

这句话不好听，却比他以前写的许多漂亮句子有用。
