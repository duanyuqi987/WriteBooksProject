# 第094章 有人追她

时间：2014年6月初

地点：支教社团聚餐

卷次：卷五：各自南北

六月初。支教社团出发前最后一次聚餐。在学校后门的一家东北菜馆。老板是个东北人，菜的分量很大。锅包肉一盘摞得像小山。地三鲜的油汪了一层。桌子坐了三桌人——十来个人。男生喝了四扎啤酒。

顾临风坐在角落。啤酒杯在手里转了很久，没怎么喝。许若琳坐他对面，正夹着一块锅包肉。

旁边一个女生——第二分队的，跟许若琳很熟——随口问了一句："若琳，你们宿舍那个——那个叫苏什么的——是不是有人在追她。"

顾临风手里的筷子停了一下。没有抬头。

许若琳一边吃肉一边说。"你说晚晴？"

"对对对。苏晚晴。"

"嗯。有。师大的一个男生——大二的，叫什么来着——姓周。追了快一个学期了。"

"成了吗。"

"不知道。她没怎么提。"

"长得帅不帅。"

"我没见过。"许若琳把肉吞下去了。"不过听说条件挺好的。学体育的。打篮球的。"

旁边女生"哦"了一声，开始聊别的话题——关于支教点有没有蛇——关于能不能洗澡——关于要不要带花露水。

顾临风还是低着头。手里的筷子夹了一颗花生米。夹了很久没有夹起来。花生米掉在了盘子外面。他把它捡起来放在骨碟里。

他继续坐着。喝了半口啤酒。啤酒是温的。泡沫已经消掉了。液面平静得像一杯隔夜茶。他把酒杯放下。又坐了一会儿。然后去了一趟洗手间。

洗手间的灯是日光灯。白色的。很亮。水龙头有点漏水——一滴一滴地打在洗手池的陶瓷面上，声音在封闭的空间里被放大了。

他洗了手。用的凉水。然后抬头看镜子里的自己。镜子里的人脸色没什么变化。跟来吃饭时一样。但他的手在水龙头下面冲了很久——冲到手背发红了。

他关掉水龙头。拿纸巾擦了手。纸粘在手心里——有五处湿掉了。他把纸丢进垃圾桶。走出洗手间。

回到桌上。许若琳正在讲支教时候要带什么——"充电宝一定要带，那边不一定有插座。"有人问"山区有没有信号"，另一个人说"移动的有信号，联通的没有"。

"临风——你手机是什么卡。"许若琳忽然问他。

"移动。"

"那没事。有信号。"

"嗯。"

他又坐了一会儿。看着桌上的锅包肉、地三鲜、花生米、拍黄瓜。每个人都在说话。笑声。碰杯声。老板在厨房里喊了一嗓子——"锅包肉两份——"。

但他什么都听不太清楚。

十点多散席。大家往外走。许若琳她们往女生宿舍方向走。顾临风一个人往男生宿舍走。

回到512。阿坤在打副本——今晚大概是团战，他戴着耳机，键盘啪啪啪地响。李博在床上看书。小伍在写信——信封又换回去了，这次还是熊猫。

顾临风脱了鞋。换了睡衣。躺在床上。把手臂枕在头下面。天花板上的裂缝还在。他看了很久——大概两个小时。中间翻了一回身。李博把床头灯关了。阿坤的键盘声也停了。黑暗里面只剩下小伍偶尔翻身的声音。

他没睡着。

有人追苏晚晴。姓周。大二。学体育的。打篮球的。追了快一个学期了。

快一个学期。

也就是说——从他开始上大一下学期的那一天起，那个人就已经在追她了。他在操场走圈的时候，那个人可能在师大的操场上打着篮球。他在图书馆写诗的时候，那个人可能站在师大的教室走廊里等她下课。他查301路公交路线的时候，那个人可能骑着电动车从师大的南门骑到北门。

他没有生气的资格。没有难过的资格。是他说遵守约定的。是他一次都没有联系。

他翻了个身。把被子蒙住头。被子里闷着一股洗衣粉的香味。他闭上眼睛。脑子里是她被追的样子——模糊的。没有脸的。那个姓周的人也是模糊的。只有两个人影——在师大的梧桐树下并肩走。

他不知道自己什么时候睡着的。

第二天早上。闹钟响了。他按掉闹钟。洗脸。刷牙。去食堂——包子和豆浆。去教室——坐第三排靠窗。听课。记笔记。跟每天一样。什么都没说。
