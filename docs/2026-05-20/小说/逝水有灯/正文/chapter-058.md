# 第058章 最后一场模拟考之后

所属部卷：第一部：初秋三楼
时间：高三春天
地点：考场外走廊

## 诗歌插页 P058：原创仿宋小令

卷已收，日将斜，
走廊人语乱如麻。
问少年、几分松手，几分牵挂。
难题未解风吹过，
各自归班看落霞。

最后一科结束时，走廊像忽然松了一口气。

有人把笔袋往空中一抛，又赶紧接住；有人对答案对到一半崩溃；有人说不管了，先吃饭。陆允川从考场出来，肩膀酸得厉害，手指因为握笔太久有点僵。

程砚在楼梯口等他，第一句话是：“活着没？”

“半活。”

“那就行。”程砚把一瓶水递给他，“数学最后一题我放弃得很果断，尊严保住了。”

陆允川喝了一口水，笑了一下。笑完，他看见楼下夏知遥和林曼站在公告栏旁边说话。夏知遥手里拿着卷子，眉头轻轻皱着，像还在想某道题。

这一次，陆允川没有停在那里看很久。他和程砚一起下楼，经过时只是点了点头。夏知遥也点头，林曼问程砚数学怎么样，程砚说：“我和最后一题和平分手。”

几个人都笑了。

笑声很短，很快被别的学生声音盖住。可陆允川喜欢这种短暂。它没有负担，也没有设计，只是高三春天一个考后走廊里自然发生的几秒钟。

晚饭时，程砚说：“你今天正常多了。”

“以前不正常？”

“以前你每次看见她，都像准备写遗书。”

陆允川差点把汤呛出来。他想反驳，最后只说：“现在没空。”

这是真话。高考越来越近，他没有那么多力气给心事修枝剪叶。喜欢仍然在，只是被放到一个更小、更安静的位置。它不再占满整张桌子，桌上还要放试卷、饭卡、水杯和未来。
