# 第196章 回封丘的火车

时间：离校当天傍晚至晚八点

地点：江城→封丘的大巴+高速公路

卷次：第八卷：没有答案的答案

大巴从江城出发的时候，天还没有完全黑。西边还剩最后一抹橙红色的光，薄薄地摊在天际线上。高速公路两旁的杨树一排一排地往后倒，倒得很快，像是被什么无形的力量从地里拔出来甩到后面去。

林玉起和苏安晴坐在倒数第三排。大巴的座位是蓝灰色的绒布面，坐久了会在裤子上留一道深一条浅一条的印子。苏安晴坐在靠窗的位置，林玉起坐在过道旁边。和他们大一寒假回家的那趟大巴一模一样——她在前面几排，他在后面几排，隔着几颗陌生的脑袋，他不敢走过去。

那次他不敢。他怕走过去手里没拿任何东西，只有一个"想和你说两句话"的理由，而这个理由不够。

今天他不需要走过去。她就坐在他旁边。

车厢里人不多。毕业季已经过去了，大部分学生要么早就走了，要么还在等着最后一批清场。车上除了他们两个，还有一个中年男人和一个抱着小孩的年轻女人。年轻女人坐在前排，孩子在怀里睡着了，她把外套脱下来盖在孩子身上。中年男人坐在司机后面，低着头在看手机，手机屏幕的光把脸照成一小块蓝色的方块。

苏安晴把车窗的遮阳帘拉了一半。外面的光线透过帘子，在她膝盖上画出细长的条纹。她的膝盖上放着一个布袋——里面是她从学校带回来的最后一点东西。布袋是白色的，上面印着"江城大学教育学院"几个红字，已经洗得有些褪色。

"你还记得大一寒假那趟大巴吗？"林玉起问。

苏安晴转过头来，看了他一眼。

"哪次？"

"放假回封丘那次。你坐在前排靠窗，我坐后面。你一路上都在看窗外，中间睡了一会儿。到了封丘汽车站，你先下了车，我在车里等了一会儿才下去。因为——"

他停了。不是因为不知道该不该说，而是这个原因说出来有点可笑。

"因为你怕和我一起下车会被我看见？"

"不是。"

"那是什么？"

"怕和你一起下车的时候，不知道该说什么。然后你走了，我就想——又错过了一次。"

苏安晴没有说话。她把遮阳帘往下又拉了拉，让光更少一些。然后她转过来，看着林玉起。

"那天回家以后我跟我妈说——今天车上看到一个男生，也是从江城回来的，也是封丘人。我妈说那你为什么不跟他说话。我说：我和他不是一个学校的。"

她停了一下。

"我妈说：不是一个学校也可以说话。"

林玉起没有接话。他看着苏安晴在车厢昏暗光线里的侧脸。四年，一千多个日夜，他在心里攒了一百条关于"为什么不和她说话"的理由。每一条都是真的——怕打扰，怕不合适，怕她不想听，怕一开口就把什么珍贵的东西打碎了。但苏安晴给了第一百零一条理由，这条理由是他从来没有想到过的。

"不是一个学校也可以说话。"

这句话不是苏安晴说的，是她妈妈说的。但她记住了，四年后讲给了他听。

车窗外的天空彻底暗下来了。高速公路的路灯亮成了一串橘黄色的珠子，一颗一颗地往车后飞。
