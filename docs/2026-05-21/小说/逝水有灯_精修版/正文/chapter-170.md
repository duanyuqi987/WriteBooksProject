# 第170章 牵手的误会多年后

所属部卷：第四部：失去的美丽
时间：多年后的一个冬夜
地点：出差途中 / 回忆中

## 诗歌插页 P170：原创仿唐绝句

炉火微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

多年后的某个冬夜，陆允川在出差途中路过一条小吃街。

炉火一明一暗，烤串摊的油滴到炭上冒出呛人的白烟。走在人群中的妻子挽了一下他的手臂——不是怕摔倒，就是习惯性地靠近。

他忽然想起大二冬天东门外那条小路。同样是炉火，同样是冬夜，同样是人群——但那时的他看到一个男生扶了苏晚枝一秒钟，心里就像针扎了一样空。

他现在回想起来，觉得那时的自己既好笑又可怜。好笑的是那一秒根本不代表任何事——后来那位学长通过他女朋友约夏知遥一起喝过奶茶，全宿舍都知道了故事的真相。可怜的是他连那一秒的停顿都承受不住，因为他的心理架构太脆弱了——任何一帧的暧昧都等于他对整段关系的否定。

现在的他不会再为这种一秒而心慌。不是因为不在意伴侣，是因为他终于理解了三个基本事实：

第一，扶人的前一秒和后一秒不是感情，是重力。人的脚踩进下水道缝会歪，会需要人扶。这是宇宙规律，不是情感背叛。

第二，即使扶那一下真的有什么——那也是苏晚枝自己的事。她有权选择她跟谁近、跟谁远。近和远都不是针对他的。他那时把自己当成所有事的中心，才会觉得别人做的每一件事都在反射他自己。

第三，如果伴侣真的想走，追问挽留不回来。真正留得住的关系不需要安检系统。

他坐在小吃街的塑料凳上，妻子端着两串烤鱿鱼回来。油滴在她手背上，她皱着眉放到他碗边。

"你上次说的那个老同学——叫苏晚枝是吗——她最近怎么样？"

"挺好的。订婚了。"陆允川接过烤串。

"你什么感觉？"

"替她高兴。"

"真的？"

"真的。"他咬了一口烤串，鱿鱼有点烤老了，嚼起来费劲，"她是一个很好的人。好人不一定非要跟我在一起才有好结局。她自己找到好结局，就很好。我们不是属于彼此的那类关系——我们只是路过彼此。"

妻子看了他几秒，没有追问。她知道陆允川会自己接着往下说——而他确实会往下说。但她自己提了一个问题。

"你呢？你对'路过'的定义是什么？"

"想要但不强求。曾经很难受但后来接受。不是怀念，是承认——承认那段路确实走过，确实是她，确实有很多不该的情绪。路还在，人早就往前了。"

妻子点点头。"这句话可以写进你的'不再分析的日常'里。"

"你怎么知道那个文件夹？"

"你电脑没锁。"

"你偷看我私人物品。"

"不算偷看。我那天想找个文件，搜关键词，搜到了你的文件夹名——太显眼。"

陆允川笑了一声。"里面内容还行吗？"

"像一本很慢的折子戏。你以前的自己跟现在的自己对话——以前的你一直都在焦虑，现在的你一直在回答：没事，别怕。"

他把鱿鱼吃完，把剩下的竹签放在盘子边上。炉火溅起一粒红星，亮了一下又暗了。多年前他也会为这样一粒火星感到惆怅——他记起青涩时代那句被写过很多遍的比喻：路边摊的炉火溅起一粒红星，亮了一下又很快暗下去。人的执念也一样，燃得越旺，暗得越快。

但现在的他不惆怅了。红星亮就亮了，暗就暗了，那是炉子的事，跟他没关系。

第二天早上在酒店，他翻出"不再分析的日常"的文件夹，加了新一条：

"出差。小吃街。买了两串烤鱿鱼。妻子说我的文件夹像折子戏。她想得比我准。我的诗都没有她的形容好。"

打完保存。窗外的上班车流在高速上拉成一条条流线。

同一时间城市另一端的苏晚枝，在上班之前翻了翻手机里的老照片。有一张是大二诗社搬活动室那天拍的，照片里很多人在搬书，陆允川正搬着一只纸箱，脸被纸箱遮了一半。旁边是她自己——手上拿着旧明信片，嘴角有一丝真实的弧度。

她看了一会儿，把照片划到"大学回忆"的文件夹里。文件夹名字和陆允川没有关系。

她前一阵清理旧书的时候，又看到那本《海子的诗》。扉页上的铅笔字还在——自己的"好"和他的赠言在同一页上，两种笔迹，两个字，隔了多年。她没有重读整本——只读了自己当年批注过的那一页。批注写的是："前面说的是田野，最后变成一个人。"是她写的没错。

她把书放回书架。

未婚夫的脚步声从厨房传来，问早餐要吃什么。

"面包加煎蛋。蛋不要煎太熟。"她回答。

冬日的早晨，阳光不强，但足以照亮客厅的半边。她端着咖啡在窗前站着，想起来了少年时代那些男生像夜宵摊的炉火一样明亮而匆忙——燃得炽热，灭得也急。有些人就是那个性格——他对你的喜欢像炭烧得通明，但你一转身他就开始不安，你需要不断地喂新风进去。

后来她不需要那种感情了。不是怕烧疼自己——是学会了照顾火候。

关于那个冬夜的误会，她很多年后只剩一个印象：不是那个误解本身，而是陆允川后来发的那条很轻的道歉——他说以后会先问旁人之后再来找她。那意味着他有在改。

能改的人不多。能道歉的人比能改的人更少。陆允川两条都占了。

她喝完咖啡，把杯子放进水槽。未婚夫把煎好的蛋盛进盘子，蛋黄的形状不太圆，但口感正好。
