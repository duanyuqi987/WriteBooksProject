# 第133章 "是你？"

时间：2018年3月周六下午

地点：楚州"一茶一坐"茶餐厅

卷次：卷七：流年未断

"是你？"

两个人几乎同时说了这句话。然后同时笑了。

声音在半空中撞在一起。他的声音低，她的声音轻。两个"是你"重叠了半秒钟——然后各自落在木桌的两端。像两颗石子同时扔进水里，波纹从两端向中间荡，在桌心交汇。

苏晚晴先停了笑。用茶壶的壶嘴对着杯沿，慢慢地倒茶。茶水在杯里绕了两圈才满。

他说"我来吧"——伸手去接她手里的壶。他的手指碰到了她握着壶柄的手指。碰到了指尖——很凉。她的手指缩了一下。然后没有缩。壶还在她手里。他的手悬在半空。然后收了回来。

她把壶递给他。他用手指接的时候——两人的中指隔着瓷壶的壁，都是凉的。

他给她倒了茉莉花茶。水注不抖——虽然手在桌下攥了几次拳。

"谢谢。"

"不用。"

她又笑了。拿起茶杯抿了一口。然后放下。手指在杯沿上慢慢转了一个圈。那个习惯——高中就有。他用余光看过几百次，但从来没有离这么近看过。

服务员走过来问"两位要点什么吃的吗"。两人都说"不用"。服务员又问"茶续吗"——她说了句"续"，他也同时说了句"续"。服务员愣了一下："到底续不续"——三个人都笑了。

服务员走后，两人又沉默了。

不是无话可说的沉默。

是想说的太多，不知道从哪里开始的沉默。

他把左手从桌上拿下来，在膝盖上摊开。掌心里有三道很浅的指甲印——刚才攥拳掐出来的。他看了一眼，然后把左手放回桌上。手心朝下。

她也把目光移开了。看着窗外。窗外有个卖气球的老人走过去——手里牵着一大串气球，各种颜色都有。有个小孩拉着妈妈的手指着气球——要。妈妈弯下腰说了什么，小孩不闹了，跟着走了。

她转回头。看着他。

"你跟我记得的样子——不太一样了。"

他说。

她歪了歪头。"哪里不一样。"

"说不上来。"

其实是说不上来。瘦了一点。脸更清楚了。笑的时候眼角有细纹——不是老的，是总在笑才会有的纹路。说话的时候声音比以前稳了。看人的时候不再那么快移开目光了。但还是那个人。还是那个马尾。还是浅蓝色。还是那双亮晶晶的眼睛。

"你也没怎么变。"她说。

他低头看了看自己。看着自己放在桌上的手——指节分明，手背上有几道青筋，食指上粉笔磨的茧已经很厚了。"老了吧。"

"没有。"

她很快接了话。快到可能自己都没意识到。说完就低下头，端起茶杯。喝了一口。杯沿遮住了半张脸。

空气静了一下。吧台上的水壶又咕噜咕噜响了。服务员在往茶壶里续热水。水蒸气从壶嘴冒出来——白蒙蒙的一团。

"你在一小教几年级。"他开口了。

"三年级。"

"几个班。"

"两个。"

"学生皮吗。"

"还好。"她放下杯子。"有几个小男孩——挺皮的。但不坏。"

"嗯。"

"你在中学教什么。"她问。

"语文。初一。"

她点了点头。嘴角有一点点弧度。好像在想什么——然后没说。

"怎么了。"他问。

"没什么。"她摇了摇头。然后又停了一下。"高中的时候你语文就很好。作文被贴在走廊上过。我记得。"

他不知道这个。

他的作文被贴过——他知道。但她记得——他不知道。

她记得。他低头看着茶杯里的绿茶。茶叶沉在杯底，一片一片的，被水泡得舒展开了。他盯着那几片茶叶看了几秒。然后端起杯子。喝了一口。含在嘴里。慢慢咽下去。

"你怎么知道。"

"什么。"

"我作文被贴过。"

"我路过走廊的时候看到的。"她笑了一下。"好几篇。用红纸贴的。你的字挺好看的。"

"那时候的字——很丑。"

"现在呢。"

他把手边的纸巾推过去。从口袋里拿出笔。在纸巾上写了一个字——他的名字。顾。纸是软的，笔尖有点涩。字迹不大——很端正，略有一点向右斜。

她低头看了一会儿。然后伸出一根手指，在他写的那个字旁边轻轻放了一下。"没怎么变。"

她的指尖离他的名字不到一厘米。指尖很干净。指甲剪得很短。食指上还有那个红墨水印子了——在第一个指节的位置。指甲缝里有一点点白色的粉笔灰。

他把笔递给她。

她接过笔，在他写的字下面写了自己的名字。苏晚晴。字比他的小——方方正正，一横一撇都很工整。写完之后她顿了顿——在名字后面画了一个很小的小圆圈。然后用笔尖轻轻扎了一下那个圆圈——扎出一个小洞。

"这是句号。"

她说完，自己也笑了。把笔还给他。

纸巾上两个名字——一个在上面，一个在下面。中间隔了一行空白。

他把纸巾折起来。对折。再对折。放进口袋里。

她看着他把纸巾放进口袋——没有说。

窗外太阳又往西移了一点。
