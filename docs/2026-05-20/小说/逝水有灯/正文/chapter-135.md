# 第135章 她和别人说笑多年后

所属部卷：第三部：寒风里的花
时间：多年后
地点：校园超市门口

## 诗歌插页 P135：原创仿唐绝句

旧店门前暑色轻，
冰柜新声替旧声。
当年笑语非伤我，
是我心窄误分明。

多年后校园超市换了名字，门口装了自动感应门。陆允川进去买水，门一开，冷气扑出来，和当年很像。

冰柜旁边站着几个学生，正为买哪种汽水争论。有人笑得很大声，收银员低头扫码，没有抬头。

陆允川拿了一瓶矿泉水，站在门口喝了两口。他想起苏晚枝那天拎着袋子，想起袋子里汽水瓶碰在一起的声音，也想起自己忘记给祁野带冰红茶。

旧稿里，他曾经把那个男生写得很模糊，越模糊越像威胁。后来他把这一段改清楚：宣传部同学，纸杯，胶带，活动物资。清楚以后，嫉妒就失去了很多自以为有理的地方。

他不是不疼。他只是终于愿意承认，疼并不能让别人失去正常社交的权利。

离开超市前，他又买了一瓶冰红茶，放在书包侧袋里。祁野当然早已不在这个宿舍楼了，可这瓶水让他想起当年的朋友，想起那些难听却把他往回拉的话。
