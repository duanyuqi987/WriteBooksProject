# 第085章 她的消息

时间：2013年11月

地点：楚州师院

卷次：卷五：各自南北

十一月的一个周三晚上。赵一鸣在QQ上发来一条消息。

"哥们在不在。"

"在。"

"给你看个东西。"

赵一鸣发了一个QQ空间的链接。链接后面跟了一句话："苏晚晴的动态。"

顾临风盯着那句话看了几秒钟。然后点了链接。

页面跳转——是她的QQ空间。背景是淡蓝色的。日志不多，动态只有几条。最新的那条是一张照片。

照片里是一个社团招新的摊位。一张折叠桌、一张海报、两摞宣传单。海报上写着——楚州师范大学支教社团招新。桌子后面站着三个人。中间是她。

苏晚晴。

马尾。浅蓝色外套。牛仔裤。她在笑——不是大笑，是那种抿着嘴的笑。秋天的阳光从侧面照过来，把她半边脸照得有点亮。

她没什么变化。

三个月了——第一次看到她的样子。头发好像比高中时长了一点。外套袖子上好像沾了一点什么东西——可能是灰。

他把照片放大了。

马尾还是那个马尾。刘海还是斜分的。陶瓷小鱼——照片里看不到，但他知道应该还在书包拉链上。

他看了大概两三分钟。

然后把图片保存了。

"谢了。"他回复赵一鸣。

"你俩咋样了。"

"没什么。遵守规定。"

"真不联系啊。"

"她说的。暂时不要联系。"

"那你就真不联系？你之前那股劲儿呢。"

"不一样。"

"哪儿不一样。"

顾临风没有回复。他把QQ窗口最小化了。屏幕回到桌面——蓝天绿地的默认壁纸。

不一样的地方是——这次是她主动说的。她需要时间，需要空间，需要各自好好读书。他听懂了。

阿坤从后面走过来，一手端着泡面。"你看什么呢。一脸不对劲。"

"没看什么。"

"你刚才那个表情——像在看一份考砸了的成绩单。"

"你看错了。"

阿坤吸了一口泡面，没再追问。

那天晚上他没有去走圈。他坐在书桌前看《中国古代文学》。课本翻到《诗经·关雎》——"关关雎鸠，在河之洲"。他在课本空白处写了两行字。写完以后划掉了。课本的边角被圆珠笔印子划破了。

李博在背后说。"明天思修课小组讨论要交提纲。你们写了没。"

没人回答。

顾临风合上课本。去洗脸。走廊尽头的水池里，有人没倒的牙膏泡沫干在水池边上——白色的一块一块的。他把水龙头拧到最大，冲掉了泡沫。镜子里的自己眼睛有点红。可能是看屏幕太久了。也可能是别的。

回到宿舍。他把手机塞到枕头底下。

躺下。

天花板的裂缝还在。他看了一会儿。闭上眼睛。脑子里不是裂缝。是那张照片——马尾、浅蓝色外套、抿嘴笑。笑容没有变。跟高一那年从教室前门走进来时一样。

他没有联系她。

他遵守了约定。但是他把照片保存了。

照片存在手机里——在一个叫"misc"的文件夹下面，没有名字，只有日期：2013-11-13。

第二天他照常上课。现代汉语。老师讲了一节课的声母韵母。他在笔记本上记了很多——bpmf、dtnl。课间的时候他打开手机屏幕，手指在键盘上按了两个字母——"苏"——拼音输入法跳出了她的名字。他按了取消。把手机放回裤兜。

下午在食堂吃面。番茄鸡蛋面。李博坐他对面，跟他讨论选修课的作业。他听着，点头，吃面。面有点咸。

"你今天话特别少。"李博说。

"面太咸了。"

"是咸。"李博吃了一口。"明天让老板少放酱油。"

他点头。

面吃完。汤没喝完。他把碗端过去放到回收台上。筷子搁在碗沿。转身走出食堂。十一月的天黑得早。路灯已经亮了。橘黄色的光照在地面上，把他的影子拉得很长。

他在路灯下面站了一会儿。

然后往图书馆方向走。
