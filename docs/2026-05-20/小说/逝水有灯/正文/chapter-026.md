# 第026章 第一次说话之前

所属部卷：第一部：初秋三楼
时间：高一末
地点：楼梯转角

## 诗歌插页 P026：原创现代诗

楼梯间的灯慢半拍亮起，
人群从上往下流。
一本笔记滑出书包，
还没落地，
少年已经替它
摔出了很多种可能。

第一次真正说话之前，陆允川先丢了一本笔记本。

那是期末前的傍晚，晚自习还没开始，楼梯间挤满从食堂回来的学生。有人端着热水，有人抱着练习册，有人边走边啃面包。陆允川背包拉链没拉紧，物理笔记本从缝里慢慢滑出来，他自己毫无察觉。

程砚走在后面，正和人争一道选择题，没顾上提醒。笔记本掉到楼梯转角时，发出很轻的一声响，像有人用指节敲了敲木门。

陆允川还在往上走，听见身后有人说：“同学，你的本子。”

他回头，看见夏知遥站在下一阶楼梯上。她手里拿着那本黑色笔记本，封面边角已经磨白。楼梯间的灯坏了一半，亮一下暗一下，她的影子落在墙上，也跟着忽明忽暗。

这一秒还没有成为第一次说话。因为陆允川没有立刻开口。他只是愣在那里，像一个被老师点名却忘了题目的人。

夏知遥把本子往前递了递：“是你的吧？”

陆允川这才下了一阶，接过来。手指碰到封面时，他发现自己掌心全是汗。

“谢谢。”他说。

声音比他想象中小。

夏知遥点点头：“没事。”

人群从他们旁边继续往上走。有人催了一句快点，楼梯口堵住了。夏知遥侧身让开，陆允川也往旁边退。两个人之间很快又隔出一条流动的人路。

程砚终于从后面挤上来，看见陆允川抱着本子站着不动，问：“你干嘛？”

陆允川把本子塞回书包，拉链拉到最底：“没事。”

程砚看了看他，又看了看前面已经走远的夏知遥，明白了几分。他没有当场取笑，只伸手替陆允川把书包拉链又检查了一遍。

“以后拉好。”程砚说，“文具不会每次都有人救。”

陆允川嗯了一声。楼梯间热，空气里有饭菜和汗的味道。他抱紧书包，心里却像被那句“你的本子”轻轻推了一下，推向一个更窄、更亮、也更容易失控的地方。
