# 第093章 许若琳

时间：2014年5月

地点：支教社团活动

卷次：卷五：各自南北

支教社团出发前有三次培训。培训地点在师院大学生活动中心二楼——一间有落地镜的舞蹈教室。每次去都要脱鞋。顾临风每次都坐在最后一排——背靠着镜子。

第一次培训讲安全事项。第二次讲教案编写。第三次讲山区教育的现状。

第三次培训结束以后，大家在门口穿鞋。许若琳也在——她是第二次培训才出现的，坐得离顾临风不远。

"你也来支教社团了？"她系着鞋带，抬头看他。

"嗯。第三分队。"

"第三分队是跟理工学院的。你认识那边的人吗。"

"不认识。"

"那你挺勇的。"她站起来，拍了拍膝盖上的灰。"我是第一分队——师大的那队。"

"嗯。"

"苏晚晴也在第一分队。"许若琳说这句话的时候正在把鞋带塞进鞋子里。她低着头，看不到顾临风的表情。

"嗯。"

"她教三年级的语文。我教英语。"

"嗯。"

许若琳把脚塞进去了，抬起了头。"你怎么每次提到她都是'嗯'。"

"不知道说什么。"

"行吧。"她笑了一下。"不过我倒是可以偶尔给你传递一下情报——她在师大的近况。你要是想知道的话。"

"不用。"

"真的不用。"

"嗯。"

顾临风穿好了鞋。站起来。把旁边的墙当椅子——靠了一下。

但许若琳还是说了。每次社团活动或者培训碰到面，她总会在谈话的某一刻忽然把话题带到苏晚晴——"晚晴最近在准备教师资格证"——"她实习教三年级语文"——"她说小朋友很可爱"——"她上次在食堂点了一道糖醋排骨然后说太甜了"。

她说的时候像在闲聊。语气很轻。说完就看别处——不盯着他看。

顾临风每次都是一样的反应——不做声。不多问。不接话。

但他把每一句话都记住了。

她准备教师资格证。她实习教语文。糖醋排骨太甜了。小朋友很可爱。每一句都像是从另一个人的人生里截取下来的碎片——碎的，不完整的。他不能拼，也不敢拼。只能收着——把它们塞进那个叫记忆的抽屉，拉开的时候里面有光，关上的时候灯灭了。

有一次许若琳说——"晚晴学会了骑自行车，在师大里面骑，把膝盖摔破了皮。"

他那天晚上在操场多走了一圈。

不是因为膝盖破了皮——是因为他想象了她骑车的样子。马尾。浅蓝色外套。骑在师大的梧桐路下面。然后摔了。膝盖破了皮——她会自己上红药水吗。

他不知道。也不会问她。

他只知道——她在十二公里外的另一个校园里，学会了骑自行车。膝盖上多了一块擦伤。跟所有普通的大学生一样在上课、吃饭、睡觉。他住在十二公里外的另一个校园里，这些不是他应该知道的——但许若琳还是说了。

他没阻止。也没问更多。

五月中旬。最后一次培训前。许若琳忽然在QQ上说了一句话——"她好像瘦了一点。"

"为什么。"

"不知道。可能是实习太累了。"

"嗯。"

"你不是说不关心吗。"

"随口问的。"

"随口问——也是关心。"

顾临风把QQ对话框最小化了。窗外是五月傍晚的光——金黄色的，斜斜地照在对面楼房的墙上。玻璃反着光。

他没有告诉许若琳——那天晚上他查了师大的地址。在地图上找到了师大北门。然后退出了。

十二公里。十一站路。四十分钟公交。

他没有问许若琳——能不能帮我带句话——能不能告诉她我在师院挺好的——能不能告诉她我考了第三名——能不能告诉她我其实每天都——

他没有问。他关了电脑，去了操场。
