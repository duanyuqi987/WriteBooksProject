# 第245章 写成小说多年后

所属部卷：第五部：逝水有灯
时间：多年后
地点：书桌和窗边

## 诗歌插页 P245：原创仿唐绝句

逝水无言灯未休，
书成不为问来由。
窗前一页风翻过，
各自人间各自秋。

多年后，陆允川再打开《逝水有灯》，最先看的不是开头，也不是那些曾让他停顿的名字，而是修改记录。每一次删除、替换、重写，都像一层薄薄的时间，盖在旧伤和旧错上。

书已经不再只属于他。读者会记住不同的人，有人喜欢夏知遥的清醒，有人心疼苏晚枝的疲惫，有人觉得程砚说话难听却可靠，也有人只记得南湖边那道旧划痕。陆允川不再急着纠正。一本小说一旦写成，就该允许别人从别的门进去。

他偶尔还会想起真实的青春。想起县一中三楼，想起大学台阶，想起出租屋雨季，想起那封没有发送的道歉。想起时心里仍会有一点轻微的响动，但那响动不再要求他做什么。

窗户开着，风把桌上的纸页翻过去。旧日记本在书柜底层，诗歌索引在硬盘里，合订本安静地躺在文件夹中。它们互相挨着，却不再互相拉扯。

陆允川关掉文档，给书房灯换了低一档的亮度。夜色从窗外慢慢落下来。逝水仍往前，灯也还在，只是不再只照一个人。
