# 第204章 最后一次争论旁观

所属部卷：第四部：失去的美丽
时间：大四前夜
地点：校外面馆

## 诗歌插页 P204：原创现代诗

我把香菜叶放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

叶澄是来吃晚饭的，不是来当观众的。她住在面馆楼上那栋旧楼的三楼，窗台上养了一盆绿萝，每天下楼吃饭都能闻到面馆汤锅的白汽味。但那两个人挑的面馆刚好在她楼下，她推门进去的时候，陆允川正把醋瓶从左手边挪到右手边，苏晚枝还没到。

"你也在这儿？"她问了一句。

"约了人。"陆允川说。

"谁？"

"苏晚枝。"

叶澄把包放下来，坐到隔了两张桌子的位置。"那我坐这边。你们聊你们的。"

"你不一起？"

"你们的事，我掺和什么。"

但她没有走。她点了馄饨，加了辣椒，用勺子慢慢地搅。苏晚枝进来的时候，她看见苏晚枝的头发比上次短了，也看见陆允川把菜单推过去时手上的动作比平时慢。

她比他们两个都先认识陆允川。大一那年公共课，陆允川坐在她后面，整整一个学期没跟她说过话，后来期末考试帮她占了座，才交换了QQ号。那时候苏晚枝还不在这所学校里。

所以她现在旁观这两个人的重逢，心里有种说不清楚的感觉。不是嫉妒，也不是心酸，而是——时间确实过去了，但人站在原地说话的样子，和时间没过去之前几乎一模一样。

他们的对话她听得很清楚。面馆不大，安静的时候连筷子碰碗的声音都像在插话。

"不是我讨厌你。"苏晚枝说。

"我知道。"陆允川说。

"你不知道。"苏晚枝把声音放低了，"你要是知道，就不会在信里写那么多'我以为'。"

叶澄听到这里，勺子停在半空中。她在心里替陆允川疼了一下，但她没有出声。

"那你为什么不说？"陆允川问。

"因为我十九岁。"苏晚枝的声音没有抖，"十九岁的女生不知道怎么拒绝一个对她特别好的人。"

叶澄的馄饨凉了。她没有叫老板娘加热，只是把碗推开了一点。

她认识苏晚枝是后来的事。大三那年，苏晚枝来学校做交流，程砚组的局。叶澄当时就觉得这个女生身上有一种奇怪的矛盾：表情很柔，说话很直。后来她理解了——那是被陆允川的信裹挟过的人，学会了一套自己的防御方式。

现在苏晚枝在说"对不起"，在把香菜叶排成一排。叶澄看着那个动作，突然觉得香菜叶这种东西，要是没有人特意写下来，一辈子都不会成为什么象征。它只是今天面碗里确实有的东西。

"你们两个，说完了没有？我的馄饨都凉了。"她终于开口。

她说这句话不是为了打断，是为了让他们知道——旁边有人，但旁边的人不会拦住你们把话说清楚。

苏晚枝走了以后，面馆里就剩她和陆允川两个人。老板娘在柜台后面刷短视频，配乐一段一段地跳。

"她比大一的时候会说话了。"叶澄说。

"你大一就认识她？"

"不认识。但听你说过。"叶澄把筷子横在碗上，"你那时候说的苏晚枝，是个不说话的苏晚枝。今天这个，话很多。"

陆允川没有否认。

"你不觉得奇怪吗？"叶澄继续，"她花了这么多年，终于学会说话。而你刚好相反——花了这么多年，终于学会不说话。"

"你点评得很到位。"

"旁观者清。"叶澄站起来，把碗端到回收台，"但旁观者也有自己的事。我先上去了，明天要交中期报告。"

她走到楼梯口，回头看了一眼陆允川。他一个人坐在靠墙的位子上，手机亮着，屏幕上是备忘录的界面。

"你还要待多久？"

"写完这段就走。"

"不急。"她说，"面馆十点才关门。"

叶澄上楼了。她没有开电脑写中期报告，而是站在窗户旁边，看着路灯下面的面馆门口。

塑料帘又掀开了一次。陆允川走出来，往右边的路灯走。他没有抬头看她的窗户。

叶澄拉上窗帘，坐到桌前打开电脑。中期报告的文档已经开了，光标停在"实验方法"后面。她打了几个字，又删掉。

窗台上有一盆绿萝，是很久以前陆允川搬家时寄放在她这里的。她一直没有还。绿萝的叶子多了几片，有几片黄了，被她剪掉了。

她接着打字。这次不是中期报告。是一个新建的文档。

"旁观者也是一部小说的组成部分。因为有人在场，对视才不只是两个人的幻觉。"

打完这一行，她删了。重新打：

"今天他接受了她的道歉。他以为自己只接受了百分之五十。但他不知道的是，他接受的那一刻，肩膀往下沉了两公分。"

打完又删了。最后她什么都没写，关了电脑。

绿萝的盆底有一点渗水，在窗台上洇出一个深色的圈。叶澄用纸巾擦了，把盆栽挪到另一边。

楼下面馆的灯灭了。锅盖盖上，抹布挂好，一天的事情终于收进了抽屉里。
