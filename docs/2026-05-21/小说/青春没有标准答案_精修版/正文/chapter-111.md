# 第111章 同学录第一页



时间：2010年六月下旬，暑假补课安排公布后



地点：高二十班教室、校门口



卷次：第五卷：未来在黑板上倒计时



2010年六月下旬，暑假补课安排公布后，高二十班教室、校门口，赵远航写在第一页的几条县城路线把2010年的春夏推到十班面前。暑假补课安排公布，赵远航在同学录第一页写下从学校到各个补课班的路线。

大家笑他像公交站牌，他却说路线写清楚，谁晚了还能互相等。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

赵远航也在这段日子里显出自己的位置。熟悉县城路，爱说路线，做事有点莽，关键时刻很能跑腿。群像不是名单，群像是每个人都有自己的难处。

“新华书店那边拐进去，比走大路近。”赵远航说。

“你不写同学录，你写县城地图。”付小川说。

“路线有用，别笑。”常俊凡说。

“我周日下午也在那边补课。”段璟说。

“补课路上别骑太快，安全比近路重要。”王启民说。

同学录第一页原本应该写名字和生日，到了这年夏天，却被写成了补课时间、路线和电话号码。

有人笑这不像同学录，像作战地图。笑完以后，又认真把自己的安排补上去。

新华书店、补课班、校门口、县城路口，这些地方被写在一页纸上，像十班临时散开的几个支点。

常俊凡写字很小，时间和地点分得很清楚。段璟看着她那一行，忽然觉得她把未来也写得这么有秩序。

赵远航讲路线时最有精神，哪个路口左拐，哪条巷子近，他都能说得像背课文。

宋乔嫌麻烦，却还是把女生宿舍电话也写了上去，说万一有人找不到人。

同学录被传到最后，纸角已经软了，第一页却比任何复习计划都像真的。

那一页没有写离别，可每个人都看得出来，时间正在把他们往不同地方分开。

王启民这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

赵远航被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

暑假补课安排是在六月下旬的一个周三下午公布的。教务处把通知单发到各班，十班的那张被宋乔贴在了黑板旁边的通知栏上，用了一块吸铁石和一段透明胶。通知单上的字打印得很密：物理补课周一三五上午、数学补课周二四六上午、语文补课每周两次、英语补课每周三次。暑假一共六周，补课占了五周。底下的备注写着一行小字：家庭经济困难学生可申请减免补课费，需提供村委会（居委会）证明。

赵远航站在通知单前面从头到尾看了三遍——不是看补课时间，而是看各个补课班的地点。有的在学校的实验楼，有的在校外的人民路新华书店楼上，有的在粮管所旁边的居民楼里。地点散得很开，跑一趟要穿小半个县城。

"这得画一张地图。"赵远航自言自语。

"你说什么？"何嘉树刚从外面进来，手里拿着同学录笔记本。

"我说——"赵远航指着通知单，"这些补课班分布在至少五个不同地点。如果不规划路线，每天跑来跑去的时间比补课本身还长。"

何嘉树听完，把同学录翻到第一页，递给他。"你画。"

赵远航愣了一下，接过本子，从笔袋里抽出一支自动铅笔，开始画路线。他先在校门口画了一个小方块代表学校，然后往左画了一条线到新华书店，在旁边标注"物理补课、周六上午"，再往右画了一条线到粮管所方向的居民楼，标注"英语补课、周二四六上午"，最后在校门口往南画了一条线到人民路拐角，标注"数学补课、周一三五上午"。

每一条线上都标了估算的时间。新华书店方向的线路他写了"步行12分钟，骑车5分钟"，粮管所方向写了"步行15分钟，骑车7分钟，坐2路公交1站"。写到最后一条线的时候，他在路口画了一个小箭头，箭头尽头写了"树莓摊"——因为是地标，岔路口认路好用。

付小川凑过来看了一眼。"赵远航你不写同学录，你写县城地图。"

"路线有用，别笑。"常俊凡的声音从旁边传来。她正在做数学竞赛题，甚至没抬头看赵远航的本子，但声音很稳，"路线写清楚，谁晚了还能互相等。"

赵远航继续画。他把每一条路线的拐弯处都标注了地标——"拐弯处有个修鞋摊""过了红绿灯就能看见卖凉皮的""到粮管所门口就左拐"。这些地标在别人看来是路边摆摊的，在他看来是导航的节点。

段璟凑过来看了两眼，"我周日下午也在新华书店那边补课。物理和数学两门连着上，中间休息半小时。"

"那你那半小时怎么过？"赵远航问。

"在新华书店看会儿书。或者去楼下凉皮摊吃碗凉皮。"

"新华书店到凉皮摊之间的路线我也给你标出来。"赵远航在纸上加了一条短线，旁边写了"段璟周日下午休息路线"。

"这条线也太短了。"段璟说，"就斜对面三十米。"

"三十米也得知道往哪走。万一凉皮摊换位置呢？"

段璟看了看那条短得只有一厘米的线，没反驳。赵远航在某些方面有强迫症——路线必须完整，不能有缺漏，哪怕只是三十米的距离。但他也觉得这种较真不讨厌，反而让很多事情变得踏实。

周五下午放学后，赵远航拉着段璟和付小川去实地踩点。三个人骑了两辆自行车——赵远航骑自己的，段璟骑他爸的老二八，后座载着付小川。从校门口出发，沿着人民路往南，第一站是粮管所旁边的英语补课点。

六月下旬的封丘，下午五点的太阳还很毒。路边的梧桐树刚长出新叶，树荫还没有形成连片的绿廊，骑一段晒一段。赵远航在前面骑，每过一个路口就停下来回头看，像在确认路线。

"你骑得跟勘测队似的。"付小川在后面喊。

"就是要测。"赵远航回头说，"我看通知上写补课时间表排得很满，如果路线不对，中间要浪费不少时间。时间是什么？时间就是——"他想了想，没说出来。

"时间就是分数。"段璟把下半句说了。他说完以后自己愣了一下，觉得这句话太像高三标语了，但在这个夏天，在这个县城，在这个马上要被补课占满的暑假里，时间确实就是分数。

三个人骑到粮管所门口，看见对面确实有一栋六层的居民楼，外墙刷着浅黄色涂料，楼门口贴着一张打印纸：英语提高班，三楼三零二，电话：139xxxxxxxx。赵远航把路线在脑子里过了一遍：从学校骑出来，右拐上人民路，直行过一个红绿灯，到粮管所门口左拐，进居民楼，一共花了十二分钟。他在同学录上把"步行15分钟"改成"骑车12分钟"，然后把原来的数字涂掉。

"为什么把数字涂掉？"段璟问。

"因为不准。"赵远航说，把铅笔字写得端端正正的。"写路线跟做物理题一样，有一说一。测出来是十二分钟就是十二分钟。不能写十五。"

下一站是新华书店。三个人又骑了八分钟，在那棵掉皮的老法桐下面停好自行车。段璟抬头看了看新华书店二楼的窗户——那扇窗户后面就是他每周六上午坐的地方，物理补课班的课桌是深绿色的，桌面被人用圆珠笔画了无数道印子。其中有一道印子他觉得像电磁感应的线圈，每次上课走神的时候就看那道印子。

付小川在新华书店门口买了一根冰棍，两毛钱的绿豆糕，掰成三截分给赵远航和段璟。三个人蹲在书店门口的法桐树荫下吃冰棍，赵远航一边吃一边在本子上标注：新华书店门口有冰棍摊，两毛一根。

"这个为什么要记？"付小川问。

"夏天补课路上热。"赵远航说得很认真，"走这条路的时候能买一根。两毛钱，够凉快一路。"

最后一站是人民路拐角的数学补课点。这个地方最偏，在一栋老式办公楼的三楼，楼下的店铺是一家卖农资的，门口堆着几袋化肥。赵远航在楼道口停下，把铅笔举起来对着楼门口的牌子看了看，"这地方如果不是提前来踩点，第一次上课肯定找不到。"

段璟抬头看了看，"是挺偏的。只有一班2路公交经过。"

"那我把公交线路也标上。"赵远航在路线旁边画了一个公交车的简单符号——一个小长方形下面带两个圆圈——然后在旁边标注：2路公交，末班车晚上七点。

踩完三个点，天已经快黑了。三个人的自行车骑过黄河大堤下面的土路，大堤上的杨树已经长满了叶子，风吹过去哗啦啦响。县城的路灯亮起来，橘黄色的光在暮色里拉出一条一条的细丝。赵远航把同学录合上，夹在自行车的后座上，用橡皮筋绑好。这个本子里记着的每一条路线，他觉得自己不会忘——但他还是要记下来。因为记下来以后，就不仅仅是一条路，而是一件已经完成的事。

第二天一早，同学录又传到何嘉树手里。何嘉树翻开第一页，看到赵远航补上去的路线图以后，又仔细看了三分钟。他在每一段路线旁边加了一行小字——不是修改，而是补充信息。比如在"粮管所"旁边写"公交站名：粮管所站，2路、5路经过"，在"数学补课点"旁边写"楼下农资店早上七点开门，周六早到可以进去等"。

"你连农资店几点开门都知道？"何嘉树抬头问赵远航。

"我昨天问过店里的人了。"赵远航说，"她说是七点开门。如果周六早上我们到早了，可以在店里等，不用站在外面晒太阳。"

何嘉树愣了片刻，然后低头继续写。他把赵远航画的路线图又描了一遍，用尺子重新比了比线——虽然赵远航的线是徒手画的已经足够直了，但他还是忍不住用尺子。段璟在旁边看到了，说了一句：你们两个一个负责搞调查一个负责整理，做事风格刚好互补。赵远航没听懂"互补"是什么意思，付小川在旁边翻译了一句"就是说你俩适合搭伙干活"。赵远航想了想，对何嘉树说了一句那以后路线我踩点你记录。何嘉树没抬头，只嗯了一声，但铅笔在他手上转了一下——那是他高兴时才会做的动作。

课间的时候，常俊凡走过来翻了一下同学录。她只看了一页，然后翻回去，又看了一遍。她和别人不一样——别人看路线图看的是方向和地标，她看的是时间。每个补课点之间的距离对应的时间她都默算了一遍，然后用铅笔把赵远航写的几个数字圈了出来：十二分钟、八分钟、骑五分钟。

"这三个时间之间有冲突。"常俊凡说。

赵远航愣了，"什么冲突？"

"你写从学校骑到粮管所十二分钟，粮管所骑到新华书店要经过人民路红绿灯，你写八分钟——但你忘了计等红灯的时间。"她把铅笔移到他画的路线交叉点，"这个红绿灯是四十秒。骑八分钟要加四十秒，所以实际时间是八分四十秒，不是八分钟。"

赵远航低头看了看，沉默了两秒，然后拿起笔把"八分钟"改成"八分四十秒"。改完以后他抬头看常俊凡，"你怎么知道那个红绿灯是四十秒？"

"我每天骑车经过。"常俊凡说，"数过。"

"为什么要数？"

"因为想知道自己有没有时间在红绿灯那里喘一口气。"

赵远航把这句话记住了。后来他把这句话也写在了同学录的边角上，就在红绿灯标记旁边：此处可喘气四十秒。

下午，王启民在班上宣布了一个消息：暑假补课期间，学校食堂也开放，但只开一个窗口。意思是每天中午的选择变少了——以前可以选炒菜、面条、馒头包子，暑假只剩一个窗口，做什么就得吃什么。全班发出一阵低沉的哀嚎，最响的一嗓子是付小川嚎出来的——他说这意味着他吃不上食堂的红烧茄子了，因为暑假窗口的老师傅不做茄子，嫌费油。

赵远航在王启民讲完以后，又在同学录上补了一行：暑假食堂只开一个窗口，午饭在食堂解决的人统一行动，路线可简化为"教室→食堂→补课点"。后面还加了一个括号：想吃什么提前自己带。

何嘉树拿到更新版的同学录以后，在封面写了一行字：2010年暑假补课路线手册（原同学录第一页）。写完以后他对着封面看了很久，觉得自己把同学录搞成了工作手册，有点太过了。但他没有把字擦掉，因为赵远航说得对——路线也是同学录的一部分。知道一个人每天走哪条路、花多长时间、在哪里等红灯，比他生日是哪天、喜欢什么颜色，更能知道他过着什么样的生活。

晚自习之前，段璟在食堂碰见了常俊凡。她端着一碗面坐在角落，面前摊着数学竞赛的错题本，筷子插在面里忘了捞。段璟端着盘子走过去，在她对面坐下。常俊凡抬头看了他一眼，又把目光移回错题本上。

"还在看题？面都坨了。"段璟说。

常俊凡低头看了一眼碗里确实已经涨开的面条，拿起筷子搅了搅。"这道题我做了三遍，每遍错的步骤不一样。"

"哪道？"

常俊凡把错题本转过来给他看。是一道不等式证明，草稿上写了三种解法，每种都有被红笔划掉的部分。段璟看了两分钟——他对数学竞赛题的理解不如物理，但他能看到常俊凡的解题思路：三次尝试用了三个不同的切入点，每次都在最后两步出错。

"你每次都在收尾的时候出错。"段璟说。

"我知道。"

"是不是越到最后越急？"

常俊凡沉默了一秒。"我算到第八步的时候会开始想——这道题如果算不对怎么办。然后第九步就错了。"

段璟想了想，"跟我做物理大题一样。做到最后两个式子的时候手会快，想赶紧写完。"

"那你怎么解决的？"

"没解决。"段璟说，"但是我现在会在第八步停一下，把前面七步重新扫一遍，确认没问题了再往下走。"

常俊凡听完，没有回答。她把错题本翻到新的一页，在那道三次都做错了的题下面，用铅笔写了一行字：第八步停一下。写完以后她把面端起来，大口吃完了已经凉了的面。段璟看她吃了，才开始吃自己的饭。两个人都没再说话，但食堂窗口外面六月末的晚霞刚好红了一整片天，把他们的影子印在了食堂的瓷砖地上。

同学录第一页之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

同学录第一页从玩笑变成地图，十班开始为即将被暑假拆开的时间做准备。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，赵远航写在第一页的几条县城路线还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
