# 流年不负

作者：段锦佑

## 作品形态

- 篇幅类型：长篇
- 目标字数：50万字（约200章，每章约2500字）
- 创作取向：文学向校园青春小说
- 目标读者或阅读场景：喜爱含蓄情感、古典诗词、慢节奏叙事的读者

## 核心 Premise

> 顾临风从高一开始暗恋同班女生苏晚晴，三年未敢言说。高考后鼓起勇气表白，两人短暂靠近，却被命运推向不同城市。大学四年、工作初期，各自经历、各自成长，那份年少的心动却从未真正熄灭。多年后一场相亲，推门而入的竟是当年窗边那个扎马尾的姑娘。这一次，流年没有辜负他们。

## 主题与核心问题

- 主题：暗恋的时间力量——有些感情不说出口，却在岁月里悄然生长；有些人不在一起，却从未真正离开
- 这部小说真正追问的问题：年少的喜欢，能不能经受住时间的考验？当两个人各自成长后，还能不能认出当年的彼此？
- 不想落入的惯例或套路：拒绝霸道总裁、拒绝狗血三角恋、拒绝为了冲突而冲突、拒绝夸张的久别重逢戏剧化

## 读者体验

最多选择 3 个核心感受。

- 感受 1：心疼——为那个不敢表白的少年，为那段被时间拉长的暗恋
- 感受 2：温暖——流年虽长，终不负有心人
- 感受 3：回味——读完想起自己青春里那个从未说出口的名字
- 失败体验：读者最不该觉得"太慢了""太淡了""太套路了"

## 世界、时代与硬规则

只记录会影响情节选择的规则。

- 背景：2010年-2023年，从高一开学到婚后生活；主要地点：青阳县（县城）、楚州（大学城）、南城（工作城市）
- 规则 1：高考是中国式青春的分水岭——它决定了谁去哪座城市、谁和谁能继续见面
- 规则 2：相亲在中国县城是普遍的婚恋途径——父母的安排既是压力也是机会
- 规则 3：时间不可逆——错过的时机不会重来，但新的时机可以被创造

## 核心冲突

- 外部冲突：高考分流（不同城市）、家庭期望（相亲压力）、社会时钟（该结婚了）
- 内部冲突：顾临风的自卑与渴望、苏晚晴的独立与脆弱
- 关系冲突：彼此暗恋却都不说破、重逢后如何面对"已经变了"的对方
- 冲突升级方式：不是通过外部事件升级，而是通过时间推移和选择积累——每一个"没说出来"的选择都在为最后的"终于说了"蓄力

## 叙事策略

- 人称与视角：第三人称有限视角，以顾临风为主视角，部分章节切换苏晚晴视角
- 时间结构：顺叙为主（2010→2023），关键节点有回忆呼应，但不倒叙
- 信息披露方式：场景展示优先，拒绝大段心理独白；重要信息通过对话和细节泄露
- 节奏偏好：慢节奏、呼吸感、章节之间留白；不追求每章都有"爽点"

## 写作风格

- 默认风格：冷白描 + 通用中文小说
- 语言要求：短句为主（10-20字占60%以上），口语化，用动作代替形容词，用细节代替概括
- 对话要求：省略主语，靠语气区分说话人，潜台词大于表面台词
- 禁止事项：禁止心理描写（"他心想""他暗自决定"），禁止感叹号，禁止"不禁/缓缓/微微/淡淡/默默"等叠词副词，禁止"仿佛/宛如/好似"等比喻词，禁止抒情或评论性叙述

## 技法偏好

从 `skills/novel-writing/references/writing-techniques.md` 中选择 1-3 个常用技法。

- 核心技法 1：白描——用最少的笔墨勾勒人物和场景，让事实本身产生力量
- 核心技法 2：不写之写——最重要的情感不直接写出来，通过动作、细节、环境暗示
- 核心技法 3：草蛇灰线——诗歌作为伏笔贯穿全文，前后呼应；细节在多年后回收

## 诗歌策略

- 每卷插入2-3首原创诗歌（七律/五言/宋词/现代诗），出现在情感关键节点
- 诗歌由男主顾临风创作——写诗是他表达情感的唯一出口
- 诗歌内容随年龄增长而变化：高中时青涩模仿古体 → 大学时尝试现代自由诗 → 重逢后诗歌变得沉稳含蓄
- 全书最后一首诗回收第一卷第一首诗的主题/意象，形成闭环
