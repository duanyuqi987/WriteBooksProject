# 第108章 座位往后挪



时间：2010年六月初，竞赛培训开始后



地点：教室靠窗后排、补课班楼道



卷次：第五卷：未来在黑板上倒计时



2010年六月初，竞赛培训开始后，教室靠窗后排、补课班楼道，沈曼青从前排搬到后排的一摞竞赛资料把2010年的春夏推到十班面前。沈曼青因为参加化学竞赛培训，座位往后挪，方便晚自习后直接去办公室。

有人羡慕她有竞赛机会，她却说自己连基础题都不敢松。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

沈曼青也在这段日子里显出自己的位置。冷静，提问直接，化学好，越被期待越怕自己失误。群像不是名单，群像是每个人都有自己的难处。

“资料放后排吧，省得天天搬。”沈曼青说。

“竞赛生就是不一样。”梁一帆说。

“竞赛不替我写基础卷。”沈曼青说。

“你晚上还去办公室？”段璟说。

“她不是轻松，是时间被切得更碎。”常俊凡说。

搬桌椅时最乱，桌腿擦过地面，书从桌洞里滑出来，没人好意思承认自己舍不得原来的位置。

座位表贴在黑板旁边，几个人围上去又散开，像怕多看一眼就显得太在意。

偏科后的错题墙贴在后排，红笔圈出的题号密密麻麻，比红榜更不留情面。

常俊凡把错题按类型分好，计算、概念、审题和步骤，一项项写得清楚。

段璟发现自己最怕的不是坐远，而是别人已经开始做事，他还在心里计算距离。

窗边风大，试卷角总会翘起来。靠窗的人一边按纸，一边听老师讲下一道题。

老师没有把话说重，只说座位会变，题不会因为你坐哪儿就变简单。

座位往后挪以后，教室看起来还是那个教室，人的心却被轻轻挪动了一格。

胡向南这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

沈曼青被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

六月初的座位调整在课间操之后。班主任胡向南在讲台上念完新座位表就去了教务处，剩下的交给学生自己搬。桌腿拖过水磨石地面的声音此起彼伏，有人用脚推，有人两个人抬，有人把书包顶在头上从过道里挤过去。

沈曼青的新座位在靠窗倒数第二排。她原来的位置在前排第三排，离黑板近，看板书不用眯眼睛。搬到后排以后，黑板上的小字有点糊，她把化学竞赛的讲义举到眼前看了一遍，又从书包里翻出一副眼镜——镜片度数不深，平时不戴，只在看不清的时候才拿出来。

"你有眼镜？"梁一帆从旁边探过头来，"我在你前面坐了半个学期都没见你戴过。"

"配了两年了。"沈曼青用袖口擦了擦镜片，"平时不戴。戴眼镜显老。"

"你戴眼镜像化学老师。"

"闭嘴。"

梁一帆笑着缩回去。沈曼青戴上眼镜以后确实看起来严肃了不少，加上她本来就话少，坐在后排靠窗的位置，像一尊安静的佛。但熟悉她的人都知道，她不是不爱说话，是把说话的精力都留给了化学方程式。

搬到后排不到十分钟，胡向南又折回来，站在后门口喊了一声："沈曼青，化学竞赛的培训教室改了。从今晚开始改到实验楼三楼西头的实验室，原来的教室让给物理竞赛了。"

"知道了。"沈曼青应了一声，在笔记本上记了一笔。

段璟正好端着水杯路过，听见了这句话。"物理竞赛为什么占化学的地？"

"因为物理人多。"沈曼青头也没抬，"化学就五个人，哪个角落都能塞。"

"五个？"

"我、秦佳禾、邓桥、曹蔚，还有一个理科班的。"她把名单报出来，像在背诵元素周期表，"上周陶建国说如果再凑不到八个，化学竞赛组的经费可能会砍。"

段璟把水杯放在窗台上，"那你们不够八个人怎么办？"

"不够就不够了。"沈曼青把眼镜摘下来放回盒子里，"又不是人多就能拿奖。"

话虽这么说，但她重新翻开讲义的时候，拇指在纸张边缘反复搓了几下——那是她紧张时的小动作。段璟注意到了，没追问。他自己参加物理竞赛也是因为凑数，物理组人够多，不存在砍经费的问题，但另一个问题摆在那里：人多意味着淘汰率也高。

下午最后一节自习课，沈曼青收拾了一摞资料准备搬到后排。化学竞赛的辅导书比普通课本沉得多——一本《无机化学》就有三指厚，加上历年真题汇编和两本笔记，摞起来比她的水杯还高。

曹蔚从过道那头过来帮忙。"你这些全要搬到后排？"

"嗯。晚自习以后直接去实验室，不用回来拿。"

"那你以前前排不是更方便？"曹蔚搬起最上面的两本，看了一眼封面，"天哪你这书是从哪弄的，大学教材？"

"我表哥在郑大化学系，他寄给我的。"沈曼青把剩下的书拢了拢，"他说竞赛题很多直接从大学教材里出。"

"那你等于比别人多学一门。"

"多学一门的前提是基础题不能出错。"沈曼青说，"上周的化学单元卷我错了两道基础题，陶建国没说什么，就看了我一眼。"

"那一眼是什么意思？"

"意思是——'别飘'。"

两个人抱着一摞书从教室前面走到后面，像抬着一个小型图书馆。路过后墙的时候，错题区贴满了各科的卷子。化学区贴了十几道离子方程式的错题，红笔圈出的错误原因写得很小很密。有一张纸上只写了四个字："看错条件"。

"这是你写的？"曹蔚指着那张字条。

"常俊凡帮我分类的。"沈曼青把书放在新座位上，甩了甩手腕，"她说我化学错的题里有一半不是因为不会，是因为审题太快。她给我分了个类——审题失误、计算错误、概念混淆、步骤跳跃。每类贴不同颜色的纸条。"

曹蔚抬头看了一眼错题墙，果然不同颜色的纸按类型排好了。常俊凡的字迹很小，每个分类的标签都用铅笔写在纸角，不仔细看会以为是墙上的污渍。

"她给你做这些，是要你还的吗？"曹蔚问。

沈曼青想了想，"她没说过要还。但她自己的数学错题也按同样的方式分类，贴在后墙另一边。所以她应该只是觉得这样做有效率。"

曹蔚沉默了一秒，说了句："你们这帮人，做事方式越来越像对方了。"

新座位在靠窗后排最大的问题是风。六月初的窗户不敢关——电风扇吹的都是热风，不开窗就变成蒸笼。但窗户一开，过堂风把桌上的试卷吹得到处飞。坐在窗边的几个人想了个土办法：用铅笔盒压住左上角，用橡皮压住右下角，中间的手指跟着笔走，像在跟风抢时间。

沈曼青在这样的座位上坐了两天，发现后排有一个前排没有的福利：安静。没有人从背后经过，没有老师突然停在旁边看她做题，也没有前桌回头借橡皮。后排是一个小王国，她可以慢慢地把一道按部就班的题算到小数点后两位，然后发现自己是错的，再从头算。

"后排是不是比较自在？"付小川转过头问她。他坐在沈曼青前面一排，书包挂在椅背上，拉链开着，露出半包旺旺雪饼。

"没人看我。"沈曼青说。

"那你也别太自在。你化学那么好，小心被人说闲话。"

"什么闲话？"

"就那种——"付小川压低了声音，"竞赛生看不起普通生之类的。"

"谁说的？"

"没人说。但就怕有人这么想。"

沈曼青推了一下眼镜——她刚戴上，还没来得及摘，"我连基础题都不敢放松，谁会觉得我看不起人？"

付小川看了她两秒，忽然点点头，"你说得对。你这个态度没问题。"转回去继续吃雪饼。

晚自习前，沈曼青去实验楼三楼踩点。新换的化学竞赛培训教室在走廊尽头，隔壁是生物标本室，走廊里飘着一股福尔马林和樟脑混合的味道。实验室的门没锁，她推门进去，里面只有三张实验台，黑板上还残留着上午物理课的板书。她在靠窗的实验台前面坐下，把讲义翻开，拿出从常俊凡那儿借的错题分类法，开始重新整理上周的错题。

窗外能看见操场上的"迎世博阳光长跑"横幅，红底白字，布边被风掀起了一个角。校门口粮油站的广播正在放新闻，说玉树地震灾区学生已全部在板房里复课，播音员的声音很平，像在念一份报表。

沈曼青写错题的时候很慢。每道题先在草稿纸上做一遍，再对答案，再找错误原因，再用常俊凡的分类法标上颜色，最后抄到专门的错题本上。一道题的工期是十五分钟，像在实验室里做一个定量实验。

实验室门被推开，秦佳禾端着饭盒走进来，嘴里还在嚼。她看见沈曼青已经坐在这，愣了一下，"你吃饭了吗？"

"吃了。"

"什么时候吃的？我去食堂没看见你。"

"第四节下课就去吃了，吃了十分钟。"沈曼青把错题本的页角按平，"然后过来看了两章无机化学。"

秦佳禾在她对面坐下来，打开饭盒，里面是食堂的土豆炖肉和米饭，已经凉了。"你是不是压力太大了？"

"还行。"

"'还行'的意思是很大。"秦佳禾用筷子夹了一块土豆，"我跟你一起竞赛培训，你那根弦绷得多紧我看得出来。上次小测验你考第三，回来以后一整个晚自习没说过一句话。"

沈曼青停住笔，抬起头，"那天确实很烦。不是烦第三，是烦自己错的那些题明明都看过。"

"看过跟做对是两码事。"

"我知道是两码事。"沈曼青把笔放下，"但就是接受不了。花了那么多时间，还是一模一样的错误。"

秦佳禾放下筷子，"我跟你说的不是同一件事。我说的是——你觉得花了时间就必须有回报。这个逻辑在化学竞赛里不成立。"

"那什么成立？"

"时间花在哪里，只有回头看才能看出来。"秦佳禾把饭盒盖上，"你现在回头看，其实做对了很多事。只是你盯着错的看。"

沈曼青没有回答。她把笔重新拿起来，翻到错题本新的一页，在上面写了几个字，然后划掉，又写，又划掉。最后她停在一行标题上，写的是："本周错题归类——审题失误部分（共四题，已全部重做两遍）"。

四个字后面跟了一个括号，括号里的话比前面正文还长。秦佳禾看了一眼，没再说什么。她知道沈曼青需要的是把每一件事都做具体，具体到每一个括号都填满。这是她对抗焦虑的方式。

晚自习铃响过以后，段璟从物理竞赛培训回来，教室已经安静了。他轻手轻脚地推开后门，看见沈曼青还坐在后排靠窗的位置，面前摊着化学讲义，右手腕上贴了一张膏药——写字太久手腕酸，医务室给的，两毛钱一张。

他在自己的座位上坐下，擦了一把汗。物理竞赛培训今晚讲的是电磁感应，老师讲得太快，他从头抄到尾，笔记上画了一大堆线圈和箭头，中间漏了好几行没跟上。他把笔记翻开对着看，发现有一道题的推导过程断了——从第三步直接跳到了第六步，中间怎么变的他完全没印象。

他回头看了一眼沈曼青。她正在用常俊凡的分类法整理化学错题，手指在不同颜色的纸条之间来回移动，像在棋盘上下子。

"沈曼青，"段璟压着声音，"你竞赛讲义上有电磁感应那章吗？"

"物理竞赛？我化学的。"

"我知道。就问一下。你表哥不是郑大化学的吗？有没有物理系的同学？"

沈曼青终于抬起头，"你中间推导断了？"

"你怎么知道？"

"因为物理竞赛培训的人都这个表情。"她把笔放下，揉了揉手腕，"我帮不了你推导，但我可以告诉你一件事——我化学讲义上也有很多我表哥写的旁注，有些我也看不懂。看不懂的时候我就空着，先往下看，回来再看。"

"能看懂吗回来？"

"有时候能，有时候还是不能。但继续往下走比卡在原地强。"

段璟想了想，把笔记翻到下一页，继续抄后面的内容，把那道断了的题空出来。他忽然觉得沈曼青搬到后排以后，不只是座位变了，整个人也变了。她以前坐在前排的时候，做什么都小声小气的，怕被别人看见自己额外花的时间。坐到后排以后，她反而放开了——愿意把讲义摊满一整张桌子，愿意让曹蔚看她的错题本，愿意在付小川开玩笑的时候顶回去一句。

后排是一个更容易做自己的位置。段璟想。他看了看自己坐的靠窗后排——他和常俊凡隔了两排，但都在靠窗这一列。她的座位上现在空着，桌面上整齐地码着数学竞赛的讲义和常俊凡自己画的分类纸片。课桌侧面贴了一张手写标签，写的是"本周做不完的题移到下周"，下面画了一道箭头指向空白处。

段璟盯着那张标签看了几秒。他觉得常俊凡这个人最厉害的地方不是聪明，而是她能把每一件混乱的事都归类、命名、归档，然后一个一个地解决。像一台有序运转的机器，内部的齿轮从不卡壳。但他也知道那只是看起来——她在错题墙上的每一张分类纸条背后，都写着她没对任何人说过的烦躁。

窗外起了一阵风，把后排几个人的练习册吹得乱翻。段璟按住自己的草稿纸，看见沈曼青也在用手肘压住讲义的边角。两个人对视了一眼，谁也没说话，各自继续做题。

补课班放学的时候，楼道里涌出一股潮湿的热气。六月的封丘到了晚上也不凉快，白天晒了一整天的水泥地面往空气中返热，走在路上像踩在暖气管子上。段璟在楼道口碰见了付小川，后者刚从树莓摊过来，手里拎着两盒特价树莓——快收摊了，一块钱两盒。

"你每天都吃树莓，不怕拉肚子？"段璟说。

"怕什么，夏天不吃水果会得夜盲症。"

"你从哪听说的？"

"我自己总结的。"付小川吃了一颗，"而且树莓比苹果便宜。苹果一个就五毛，树莓一盒才一块。"

段璟看着他手里的树莓盒子，想到了余小满。她的饭卡余额这周不知道还剩多少，但下午在食堂他看见她打了两个素菜——比平时多了一个。也许麦子真的收得不错，也许只是今天心情好。

"你在想什么？"付小川问。

"在想你记不记得给别人留一点。"

付小川愣了一下，然后把两盒树莓中的一盒塞给段璟，"那这盒你帮我给余小满。别说是我给的。就说——"他想了想，"就说是树莓摊快收摊了送的。"

"你当她是傻子？"

"不是。是怕她不肯收。"

段璟接过树莓。盒子底部已经有点软了，紫红色的汁水从泡沫盒的边缝渗出来，染了一点在手上。六月的风从粮管所的方向吹过来，带着新收麦秸的干燥气味。这个夏天，十班每个人都在用自己的方式，把善意藏在不起眼的地方。

座位往后挪之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

座位往后挪成了另一种提醒：看起来更远的人，未必压力更小。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，沈曼青从前排搬到后排的一摞竞赛资料还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
