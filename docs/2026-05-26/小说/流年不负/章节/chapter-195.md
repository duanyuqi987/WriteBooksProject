# 第195章 女儿在湖边跑

时间：2022年10月
地点：青阳县月湖
卷次：卷十：不负少年

女儿沿着湖边跑。手里攥着那颗心形的鹅卵石。

她跑得不快——一岁半的腿还很短。每一步跨出去只有大人的半步。脚跟先落地，然后前掌。有时候绊到地上的石头或者树根——趔趄一下。快倒了。手撑地。爬起来。拍拍手上的土。继续跑。

马尾在后面颠。一颠一颠。

苏晚晴在几步之外跟着。手里拿着女儿的凉鞋——鞋面上沾了泥。她走得不快。保持着刚好能够到女儿的距离。女儿又一次趔趄的时候她伸手——但还没够到。女儿已经自己爬起来了。

"慢点。"

女儿没听。继续跑。

顾临风站在原地看着她们。

女儿的马尾从左边晃到右边。又从右边晃到左边。阳光在头发上有一层薄薄的光晕——不是金黄色的。是褐色的。头发细软，风吹起来的时候能看到底下的头皮。脖子后面有一个小窝——汗毛湿了一撮，粘在皮肤上。

他站的位置离她们大概二十米。柳树的树荫遮着他半边脸。

他看着。

2010年9月1号。高一开学日。一个扎马尾的女生从教室前门走进来。浅蓝色T恤——领口有一圈白色的边。微微低头——看着地面走路的。从来不看前方。阳光从窗户外面打在她侧脸上——阳光里浮着粉笔灰。他坐在倒数第三排靠窗。她坐在第二排靠窗。他看她的马尾从左晃到右。看了三年。从高一看到高三。从月湖边看到楚州。从301路的公交站牌看到结婚登记处。

2022年10月。月湖边上。另一个扎马尾的小女孩在湖边跑。

那个当年的女生现在是一个三十岁的女人了。她跟着她的女儿在湖边走——裤腿湿了半截，凉鞋拎在手里。她的马尾还在——黑亮的，扎得不高不低。碎发在耳朵边上飞。她伸手把碎发别到耳后——右手的食指和中指夹住碎发，往后拨，拨到耳后。然后指尖在耳垂上停一瞬。这个动作和十二年前完全一样。和2010年9月1号上午第一节课后她在饮水机前接水时做的动作一样。和2013年6月8号晚上她在月湖边看月亮时做的动作一样。

那个当年的男生现在是一个三十岁的男人了。他站在柳树下面。柳树也老了十二岁。树皮裂得更开了。但柳条还是软的——跟以前一样。他手里握着一颗心形的鹅卵石——女儿捡的。歪歪的——左边比右边大。但形状确实像一颗心。他的口袋里还有一支笔芯干了的旧中性笔。笔帽上有手指磨出的凹痕。笔帽上还有她的指纹——十二年前在讲台上捡起来的时候留下的——已经被一层又一层的灰尘覆盖了。但还在。

女儿跑到他面前。气喘吁吁的。脸上红扑扑。额头上有一层细密的汗。她把石子举起来:"爸爸。念念跑了好远好远。"

"跑多远了。"

"从这里——"她指着柳树。"到湖边那里。"

"很远。"

"嗯。"

她说完又跑了。这次往小石板路的方向跑。那条路通向上面的小山坡。山坡上有一片草地——十二年前她妈妈在那里坐过。接过去一首诗。手和手碰了一下。

顾临风看着女儿跑。

他看着她的马尾在风里晃。恍惚了一下。有一瞬间，他以为又看到了高一的她。

但不对。高一的时候她的马尾更高。更紧。走路的时候晃得快一点。跑操的时候马尾会甩起来——像马尾巴。

女儿的马尾还太短。软塌塌的。扎不紧。跑几步就散了。她从石板路上跑回来的时候，发绳已经滑到了马尾尖上。头发散开了一半。像一把小扫帚。

苏晚晴把她叫过来。蹲下。重新把发绳推上去。绕了两圈。又绕了一圈。

"好了。再跑。"

女儿又跑了。

苏晚晴站起来。拍了一下膝盖上的草屑。然后转头——看到他在看她。二十米外。柳树下。阳光穿过柳条洒在他身上是一条一条的。

她向他走回来。脚踩在草地上——软软的。凉鞋还没穿。光着脚。走到柳树下面。挨着他站。

"看什么呢。"

"看你们。"

"看了多少年了。"

"十三年。"

"看腻了没有。"

"没有。"

一阵风吹过来。柳条全荡起来。水面上起了鳞一样的波纹。远处的水塔上——天还没黑。但灯好像闪了一下。也可能是他看错了。

女儿在湖边喊:"妈妈——这块石子上有蚂蚁。"

苏晚晴迈步走过去。弯腰。把女儿手里的石子拿过来看了一下——石子底下果然有一只蚂蚁，慌乱地爬。她放在地上。蚂蚁跑了。

顾临风还站在柳树下面。

他从口袋里拿出了那颗心形的鹅卵石。石子在口袋里蹭了一阵子，沾了些裤子口袋里的绒毛。他用手指把绒毛抹掉。石子光滑。有一点温温的——被口袋捂热的。

他看着石子。又看着湖面。又看着那个弯腰在湖边找石头的女人。又看着那个蹲在地上看蚂蚁的小女孩。

中间十二年。像湖面上的波纹。风吹过来的时候一层一层地扩散——从湖中心到岸边。从高一到高三。从青阳到楚州。从结婚到念念出生。等风停了，波纹没有了。水面平了。但水面还是那片水面。月湖还是月湖。柳树还在。水塔还在——灯到了晚上还是会闪。

那年风初凉。现在风又凉了。还是同一阵风。从2010年吹到了2022年。从九月的教室窗户吹到了十月的月湖柳树下。风的温度没变。风里的人变了——从一个人变成了两个人。从两个人变成了三个人。从三个人——看柳树、捡石子、在太阳底下跑。风还在吹。一直吹。不知道会吹到什么时候。大概会吹很久。吹到念念长大。吹到她和他们一样——在某个湖边上看着自己的孩子在跑。然后她也会说"慢点"。然后她的孩子也不会听。

他低头。把鹅卵石放进另一只口袋——这边是放着那支旧笔的口袋。笔和石子碰了一下。发出很轻的声音。石子碰塑料笔管——笃。像敲门。没人应。也不需要应。

苏晚晴抱着女儿回来了。女儿累了。把头靠在妈妈肩膀上。手指还攥着一个小石子——这次是一个白色的圆形石子。

"回去。"

"好。"

他接过女儿。女儿换了个肩头——趴在他肩膀上。眼皮已经开始打架了。嘴里嘟囔了一句"蚂蚁"——然后睡着了。口水滴在他的衬衫领子上。

苏晚晴从地上捡起女儿的凉鞋。把鞋底磕了磕——飞出来一些土。然后穿上。

三个人往回走。往停车的地方走。

苏晚晴走在他旁边。光脚穿凉鞋。脚后跟有一点泥。她走了几步，泥干了——变成一层淡灰色的粉末。

他一手抱着女儿。一手捏着口袋里那颗心形石子。

路上一个人也没有。太阳开始往西边偏了。金色的光线从柳条之间漏下来。一道一道的。像当年高三教室里的百叶窗——光也是这么一道一道地漏进来。照在她的侧脸上。照在念念的小脸上。

月湖的水面在落日里变成了金色的。一张一张金色的鳞片。水波往岸边推——推了一层。又推一层。金色的鳞片碎了。又合起来。又碎了。又合起来。跟十二年前一样。也跟十三年前一样。也跟刚才一样。也跟明天一样。
