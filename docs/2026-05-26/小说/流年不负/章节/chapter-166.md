# 第166章 订婚
时间：2018年11月
地点：青阳县一家酒店
卷次：卷九：执手之约

十一月的第一个周六。青阳县入秋以来最晴的一天。阳光从早上就很足，照在酒店门口的红色拱门上——"顾临风先生与苏晚晴女士订婚宴"——气球扎成的，有一只在风里歪了一下，被人扶正了。

酒店比起提亲那家又高了一档。但也不算豪华——县城的婚宴酒店，格局都一样：大圆桌、红桌布、舞台上拉了一张喷绘背景板。背景板上印着两个人的名字，金色的字，在灯光下反光。

五六桌人。不多。都是亲戚和亲近的朋友。顾家那边坐了四桌——镇上的亲戚来了不少，有些人顾临风好几年没见了。苏家这边坐了两桌——县城里的亲戚，加上苏晚晴的几个同学。

苏晚晴在酒店二楼的包间里换衣服。

红色旗袍。是她妈陪她去挑的。挑了三次。第一次试了一件大红色的，她妈说"太艳了。"第二次试了一件暗红色的，她妈说"太暗了，不像订婚的。"第三次试了这件——枣红色，领口和袖口绣了一圈暗金色的边，不长不短，刚好到小腿。

她站在镜子前。许若琳在旁边帮她整理领口。领子有点紧——她自己扣了两次没扣上，许若琳替她扣了。扣子是一颗小小的盘扣，红色的线编成蝴蝶形状。

"好看。"许若琳说。

苏晚晴看着镜子里的自己。没说话。

"想什么呢。"

"没想什么。"

许若琳笑了一下。"你骗谁。手都在抖。"

苏晚晴低头看自己的手。确实有一点——手指在旗袍下摆上抓着，抓出了一道折痕。她松开了。折痕还在。

楼下传来声音。有人在试麦克风——"喂，喂——"然后是音响的反馈啸叫——"吱——"

苏晚晴从二楼窗户往下看了一眼。楼下大厅里，有人在往桌上摆瓜子和喜糖。赵一鸣搬了一箱饮料从门口走进来——他把箱子扛在肩上，走得不快但步子稳。

顾临风站在酒店门口。

西装。黑色的。白衬衫。领带是赵一鸣帮他打的——赵一鸣说"你这个领带怎么跟麻绳一样"，然后拆了重新打。打完后退了一步，看了看。"行了。勉强像个人。"

顾临风扯了扯领口。太紧了。脖子里有点不舒服。但他没解。

门口的红毯从台阶铺到大厅。红毯两边摆了两排花篮——百合和玫瑰。花篮上的水珠在太阳底下发亮。

客人都到得差不多了。亲戚们互相打招呼——"好久不见""你胖了""你家孩子多大了"。大厅里的声音嗡嗡的。

赵一鸣从人群里挤出来。走到顾临风旁边。看了一眼他的西装。然后吹了一声口哨。

"顾临风穿西装！活久见！"

顾临风没理他。

"你高中穿的什么——校服。大学穿的什么——T恤衫。上班穿的什么——还是T恤衫。"赵一鸣围着他转了一圈。"今天居然穿西装了。这不得拍个照纪念一下。"

"你拍。"

赵一鸣掏出手机。拍了一张。然后低头看。

"不行不行，太僵硬了。跟木头桩子一样。笑一个。"

顾临风没笑。但嘴角动了一下。赵一鸣赶紧按下快门。

"行。这张可以。虽然是假笑。"他把照片晃给顾临风看。"等会儿苏晚晴下来的时候你肯定笑得比这个真。"

大厅里的音乐换了。从背景音乐变成了一首古筝曲——《渔舟唱晚》。这是苏晚晴选的前奏。她说"不要婚礼进行曲——那是婚礼用的。订婚用这个就好。"

苏晚晴从楼梯上走下来。

枣红色旗袍。高跟鞋——她不太会穿，走得很小心，每步都踩得很实。手扶着楼梯扶手。头发盘起来了，几缕碎发掉在耳朵前面。许若琳在后面帮她提着裙摆。

大厅里安静了一下。然后是一阵低声的赞叹。

顾临风站在红毯那头。看着她一步一步走过来。她走得很慢——高跟鞋踩在红毯上没有声音，但她的旗袍下摆在摆动。枣红色的裙摆晃起来的时候，能看到暗金色绣线的一闪——

像水面上的波纹。

她走到他面前。抬头看他。

"领带歪了。"

他低头看领带——"赵一鸣打的。"

"我重新给你打。"

她伸手到他领口。解开那个赵一鸣说"勉强像个人"的结。然后重新绕、交叉、穿过去——手指在他胸前动了几下。拉紧。

"好了。"

她的手在他胸前停了大概两秒。然后收回去。

赵一鸣在旁边拿手机拍。拍完低头看——"可以。这张是真的笑了。"

订婚宴很简单。没有司仪——顾临风自己说的流程。他拿着麦克风说了几句话——声音比平时大，但麦克风让他不习惯，第一句"大家好"后面就卡住了。他停了大概两秒。看了一眼苏晚晴。她站在台下，朝他微微点了一下头。他继续说了。

"谢谢大家来。今天是我和苏晚晴订婚的日子。我们认识八年了。在一起的时间不长。但八年前我就知道——就是她了。就这个人。不会变。"

他把麦克风放下。没有多余的话。

顾母在台下擦眼泪。苏父坐在角落里。没有站起来。但他在鼓掌。两只手拍得很慢，每一下都很响。

然后是敬酒。一桌一桌地走。端着酒杯——他喝白的，她喝橙汁。亲戚们说"恭喜恭喜""白头到老""早生贵子"。她端着果汁杯一一谢过。他喝酒——每一桌都干了。喝到第三桌的时候脸就红了。

苏父那一桌。他们走到的时候，苏父端着茶杯站起来。

"叔叔。"

苏父没说话。把茶杯里的茶倒掉。从旁边桌上拿起白酒瓶子——倒了半杯。

"今天喝这个。"

两个人碰了杯。杯沿碰在一起的声音跟上次提亲时一模一样。苏父一口喝完。顾临风也一口喝完。

苏父拍了拍他的肩膀。第三次。这次没说话——但是拍在肩膀上的手停了一下。然后才收回去。

宴会结束时天已经黑了。客人都散了。大厅里只剩下服务员在收拾桌布和碗筷。喷绘背景板上的两个名字还亮着灯。

苏晚晴在大厅门口站着。脚上的高跟鞋已经脱了，提在手里。光脚站在门口的地毯上——酒店的红地毯凉凉的。

"脚疼。"

顾临风蹲下来。看了看她的脚。脚后跟磨红了一片。

"怎么不早说。"

"说了你还能替我穿。"

他站起来。把她手里的高跟鞋接过来。自己的皮鞋脱了。光脚站在她旁边。

"走。回去。"

"你的鞋——"

"我不疼。"

两个人光脚走出酒店大门。外面的地上有点凉。石板路上有水渍——不知道什么时候洒的。她的旗袍下摆拖在地上，沾了点水——暗金色的绣线湿了之后颜色变深了。

赵一鸣开车经过。摇下车窗。看到两个光脚的人站在酒店门口。

"你们俩——鞋呢？"

"车里。"顾临风说。

赵一鸣笑得很响。摇上窗户。车走了。

十一月的晚风吹过来。有点凉。她把手臂缩了一下。他没穿外套——西装脱了只剩一件白衬衫。他把衬衫袖子卷起来，没用——卷袖子也不挡风。

他把她的手握住了。

她的手还是凉的。跟高考后在月湖边一样。跟他每次牵她一样。

但这次她没有缩回去。她把手指张开。让他的手指穿过去。十指扣在一起。

两个人光着脚站在酒店门口的水泥地上。头顶是红色的拱门——那只被扶正的气球又歪了一下。这次没人去扶了。

星星出来了。十一月的天很晴，能看到很多颗。她抬头看星星。他看她的侧脸。

跟八年前在教室里看的是同一张侧脸。

但这次。她的手上戴着戒指。
