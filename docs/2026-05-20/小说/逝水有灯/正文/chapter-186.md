# 第186章 诗社告别之前

所属部卷：第四部：失去的美丽
时间：大三下学期，周五傍晚
地点：诗社活动室

## 诗歌插页 P186：原创现代诗

钥匙在掌心
硌出一个小小的圆
门还可以打开
只是我不该
每次都进去

诗社活动室在综合楼三层，门牌掉了一角，胶水痕迹留在木板上。陆允川到的时候，走廊尽头正在拖地，水迹从窗边铺到门口，他只好贴着墙走，手里攥着那把用了两年的钥匙。

钥匙扣上挂着一块蓝色塑料牌，写着“诗社三楼”。那几个字被磨得发白，尤其是“社”字，偏旁已经快看不清。陆允川把它放进掌心，反复翻过来，像在确认一个普通物件为什么会忽然变重。

活动室里没人。桌上堆着上周朗诵会剩下的报名表，角落有半箱矿泉水，窗台上摆着几本没人认领的杂志。他打开柜子，把自己借过的诗集一册册取出来，擦掉封面上的灰，又按编号塞回去。最上面那本夹着一张粉色便利纸，是叶澄写的值日提醒，纸边卷起，字迹很稳。

他原本想等苏晚枝来，再当面说“我以后不负责这边了”。可真正站在屋里，他又觉得这句话没必要说得太隆重。退出一个地方，不一定要让所有人见证。那样反而又像在等某种反应。

他把活动室的窗关好，检查插排，最后在登记本上写下归还日期。笔尖划过纸面，声音轻得几乎听不见。告别还没开始，他先学会了把自己的位置收拾干净。
