# 第110章 封校的周末多年后

所属部卷：第二部：六月以后
时间：大一秋天
地点：大学南门和围墙边

## 诗歌插页 P110：原创仿唐绝句

墙草微凉近晚风，
少年心事隔帘中。
不知旧梦归何处，
一盏疏灯照水东。

大学同学会在毕业十周年那年选在了学校门口的一家饭店。陆允川开车三个半小时到，把车停在北门外的新停车场——以前那里是一片平房，大学生常去租碟片。如今平房拆了，停车场铺了沥青地，画了白线，扫码收费。

祁野先到了，在饭店门口等他。他胖了十五公斤，头发剃得很短，穿着一件深蓝色的教练外套，胸前印着"市体校"三个字。远远看见陆允川就喊："你还这么瘦？你老婆不给你吃饭吗？"

"她天天做，我天天吃。"陆允川走过去，"吃了不长。"

"年纪到了就长了。我三十岁以前也瘦。"

饭店里到了十八个人。叶澄坐在靠窗那一桌，旁边是她丈夫——一个戴眼镜的中学历史老师，话不多，全程负责倒茶。叶澄看到陆允川先站了起来。

"坐这儿。"她拉开旁边的椅子，"我有东西给你看。"

她打开手机相册，翻到一张照片：一本棕色的硬壳笔记本，封面磨出白边。

"你的诗抄本？"陆允川问。

"你当年落在人文楼401的。后来被晚枝捡到了，她说先收着，等你找她要。结果你一直没要。"

"我忘了。她也没提过。"

"她提过。你在毕业聚餐上说你的诗抄本丢了，她说'没丢，在我这'。你没听见。"

"我那天喝了酒。"

叶澄把照片转发给他。"她后来把诗抄本还给了你——你毕业的时候她装箱子寄给你的。你忘记了吗？"

陆允川想了一下。模糊有印象。一个牛皮纸包裹寄到老家，里面是几本书和一个诗抄本。他当时刚入职新工作，忙到没时间拆，放在父母家的书柜里，一直没打开。

沈让也来了。他剃了光头，眼角的皱纹很深，但精神很好。他后来没有从事文学工作，开了家小公司做电商仓储，业余时间拍了十几年短片，得过几个市级奖项。

"你还写诗吗？"沈让问陆允川。

"偶尔。"

"我拍过一个短片，讲封校。去年投了一个独立影展，没入围。"

"什么内容？"

"只有两天，一个周末。用一镜跟拍的，一男一女从南门走到实验楼后面，再走到打印社，再回来。全程没有对话，只有环境音。最后入围的是另一部片子，拍的是一对老夫妻在菜市场吵架。"

"你的短片叫什么？"

"《墙草》。"

"我能看吗？"

沈让找出手机里的文件，把耳机递给他。陆允川戴上耳机，屏幕里是大学南门。铁栅栏关着，保安亭里的倒影映在玻璃上像一块褪色的绿。镜头慢慢移，移到围墙边——围墙根的野草被踩平了，泥土夹着烟头。不是他记忆中的样子，是沈让拍的样子。镜头晃了一下，有个穿白色衣服的女生走过，不是苏晚枝。有个男生跟在后面，也不是他自己。但走姿很像——肩膀微微往里收。

镜头到了打印社。老板刘阿姨已经不在店里了，换了新的门楣，印着"学子数码图文"。机器全换了。柜台没有人，只有电脑屏幕亮着，屏保是一行字："学子印务——1987-2015"。

短片里最后一段是围墙边的窄路。镜头推得很慢，到围墙根的草，到泥土，到一只被雨打蔫的烟头——沈让是专门找到的，他说那是个道具烟头，从便利店买的同款。

屏幕黑掉。结尾字幕：墙草微凉近晚风。

陆允川摘下耳机。

"你这个短片结尾的那句诗，是当年诗社绝句里的。"他说。

"嗯。我改了一个字。原句是'墙草微凉近晚风'——我没改。"

"你没拍人脸。"

"不敢拍。人脸一拍就成具体人了。我想拍的是那个感受——那个周末所有人都在围墙里面，但有些人在围墙里面也很自由。有些人在外面也不自由。"

饭局进行到中间，有人提议回校园走走。

北围墙那截矮墙还在。砖头换了新的，垒得更整齐了，底下的泥土被踩得光溜溜的。有个学生蹲在墙头看手机——可能是信号好。南门的保安亭还是老位置，但保安不是老周了。老周五年前退休，现在在老家养蜜蜂，朋友圈天天发蜂箱照片。

食堂门口的特供窗口已经撤销了。现在食堂旁边开了一家正式的夜宵店，铁板豆腐和炸鸡柳都在菜单上，买了就能扫码付款。

图书馆二楼靠窗的桌子还是原来的位置。陆允川在门口站了一会儿。刷卡进馆的闸机换成了人脸识别，外人不让进。一个女生从里面走出来——戴细框眼镜，扎低马尾，手背上沾着红色印泥。她匆匆走过去，不是苏晚枝。但她摘眼镜的动作、揉眼角的动作，和当年一样。

"如果是晚枝，"叶澄不知什么时候走到他旁边，"她肯定会说你站在门外干什么，进来。"

"你确定？"

"不确定。但我确定她会摘眼镜。"

回到饭店时天已经黑了。陆允川坐在靠门的位置，喝了一杯没有喝完的啤酒。旁边有人问他现在在做什么工作。他回答了。另一个人问他孩子几岁了。他也回答了。

散场时，祁野送他上车。

"你那首写围墙的诗后来怎么样了？"

"什么诗？"

"封校那个周末你写的那首。那天晚上宿管阿姨来敲门，我说你在写诗。你写到了凌晨一点。你不记得了？"

陆允川想了一下。"不记得了。可能写在诗抄本里了。"

"那你回家以后找找。"

"好。"

车开出校园。导航里女声说"前方三百米右转"——和搬家那天一模一样的句子。陆允川看了一眼后视镜里逐渐变小的大门弧线，想起沈让那个短片里的结尾字幕。墙草微凉近晚风。墙草是真的，不是道具。那年秋天封校的周末只有两天，但墙草一直都在。

回家以后他在父母家书柜里找到了那个牛皮纸包裹。拆开来，里面有几本书，和一本棕色硬壳的诗抄本。第47页夹着传单。第48页是封校周末写的诗。他当时写的不是围墙。他写的是那天的铁板豆腐和鸡柳，以及祁野说"每个人都应该全面发展"时嘴里的油。

他把那页读了三遍，然后把诗抄本带回自己家，放在书架的中间一层，没有编号，没有书名。
