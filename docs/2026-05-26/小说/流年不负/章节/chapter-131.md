# 第131章 推门

时间：2018年3月周六下午3:00

地点：楚州"一茶一坐"茶餐厅

卷次：卷七：流年未断

风铃响了。

她抬起头。

目光从手机屏幕移到他身上。

顾临风的手还放在门把上。门在他身后缓缓合上——弹簧很紧，门框撞在胶条上发出一声闷响。风铃又震了一下。

她的表情在他推门的这三秒钟里做了三次变化。

先是皱眉——像在辨认。眼睛微微眯起来。手机放在桌上。头歪了一点点。嘴唇动了一下——大概是想说什么但没说。那个表情像一个在火车站人群中忽然觉得看见了熟人——但不能确定的人。

他往前走了一步。阳光从窗子打在他身上——浅灰衬衫、深灰夹克、头发有点翘。

然后她的眉头松开了。

眉头展开的过程很慢。不是那种恍然大悟——而是那种确认。像在看一个记忆里的轮廓慢慢跟眼前这个人的轮廓重合。从眉骨的弧度、到鼻梁的高度、到下巴的线条。从无法确定——到好像可以确定——到真的可以确定。

她的嘴唇微微张开了。嘴角动了一下。

然后她笑了。

慢慢地。嘴角往上弯。很慢——好像不确定是不是该笑。然后又弯了一点。然后整张脸都松了。

不是那种久别重逢的激动。是一种——好像是这样的。好像本该是这样的。好像这么多年就是该在一个普通的周六下午的茶餐厅里，推开门的人是他。

他也笑了。

嘴角不自觉地上扬。左边先动——然后右边跟上。幅度不大。但眼睛里的笑意比嘴角明显。他感觉到自己的眼尾出现了细纹。

阳光从窗外照进来，照在两人之间的地板上。那是一块很大的方形光斑。他站在光斑的这一边。她坐在光斑的那一边。阳光在他们之间画了一道明亮的分界线。

他往前走。踩进了光斑里。鞋底的灰在地板上留下一个很淡的脚印。

她在看他。没有移开目光。

不像高中时候。高中她总是很快移开目光。走廊上那几次对视——她和他目光相遇不会超过一秒就会低下去。现在她看着他走进来，目光稳稳地——不移开。

他的喉咙动了一下。想说什么，但嘴唇动了动没有发出声音。

她先开口了。

声音很轻。比高中时厚了一点点——大概是当了六年班主任老师磨出来的，但尾音还是一样的。微微往上提。

她说的话混在茶餐厅的背景音乐里——听不太清。他只看到她的嘴唇在动。

然后他听清了。

"顾临风。"

她叫了他的全名。连名带姓。像高中班主任点名时念出的自己的名字。

不是"你是"。不是"好久不见"。就三个字——顾临风。声音很轻。尾音在空气里挂了零点几秒。然后落下。

他站在她面前大概三秒钟。然后点了点头。

"嗯。"

声音比想象的沙。今天第一天开口说话——跟谁都没说过。从早上到现在，他没跟任何人说过话。他清了清嗓子。手在裤兜里攥了一下。

"苏晚晴。"他说。

他也叫了她的全名。也是连名带姓——就像他在纸上写过的很多遍一样。但这次是说出来的。对着她说出来的。

她听了以后，眼睛亮了一下。

不是哭。是亮。像灯忽然调大了一格。

然后她低下头。看着桌上那杯喝了一半的水。杯子里的水面微微晃动——是她刚才端的时候手在抖——或者是他推门时候风铃震动传到了桌面上。

她抬起头的时候，嘴角还是弯的。

"坐。"她说。指了指对面的椅子。椅子的把手是藤编的。好像刚才一直在等人来。

他把夹克的拉链拉开。脱下。搭在椅背上。椅子被拉开——木腿在地板上发出一声轻响。

他坐下。

双手放在桌上。手指交叉。然后松开。然后放在膝盖上。然后又放回桌上。

她看着他坐下来。眼睛跟着他从门边走到桌边——从站着到坐下。这个过程中她的目光一直在——像在确认他是真的。

服务员走过来。

"先生喝什么。"

他看了她一眼。她面前还是那杯水。

"绿茶。"

"您呢——"服务员转向她。

"茉莉花茶。"

服务员走开了。水壶放下来——里面有烧水的咕噜声。

两人面对面坐着。木桌的宽度大概七十厘米。

近到他能看见她鼻尖上有一点很小的光——窗外光线反射的。近到她能看见他领口磨毛的纹路。

两个人同时张了张嘴。又同时闭上了。

窗外的樱花树枝在风里晃了一下。

一片嫩绿的叶子贴在了玻璃上。很快又被风带走了。
