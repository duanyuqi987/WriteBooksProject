# 第110章 封校的周末多年后

所属部卷：第二部：六月以后
时间：多年后
地点：大学南门和围墙边

## 诗歌插页 P110：原创仿唐绝句

旧墙斜照草痕平，
当日纸声隔岁听。
共走一程非暗许，
风来只认路分明。

多年后陆允川再走到南门，围墙边的小侧道已经改成了停车棚。原来递东西的栏杆口被焊死，旁边竖着快递柜，学生扫码取件，几秒钟就走。

他站在停车棚外，看见一个社团学生抱着宣传册跑过，纸页被风吹得乱翻。那一瞬间，他想起苏晚枝当年抱着社刊样张，手指上沾着红笔印，认真地说“乱也可以改”。

旧稿里，他曾经把这句话写得很重，像她给过某种私人许可。现在他愿意把它放回原处：那是对稿子说的话，也是对一个新社员的鼓励。它美好，但不隐含承诺。

他绕着停车棚走了一圈，没有找到当年的野草。找不到也好。很多场景不需要保存原样，保存原样反而容易让人误以为当年的误读也有了证据。

回去的路上，他在手机备忘录里写：共同完成一件事，会让人靠近；但靠近只是靠近，不是归属。

写完他把手机收起。南门外车声很密，校内铃声从远处传来，两个声音叠在一起，像过去和现在终于不再争谁更真实。
