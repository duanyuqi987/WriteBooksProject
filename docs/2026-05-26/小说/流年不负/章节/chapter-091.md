# 第091章 操场

时间：2014年4月

地点：楚州师院操场

卷次：卷五：各自南北

四月的晚上。操场。白天下了阵雨——傍晚停了，跑道上的水渍一块一块的，在路灯下面反着白光。空气里水汽很重——往鼻子里吸的时候带着一股雨后泥土的腥味。

天气暖和了一些。跑道上的人数比冬天多了很多。顾临风还是在走圈。每天晚上九点左右——自习完、图书馆出来——走到操场，从第一道开始走。

师院操场比一中大。一中是煤渣跑道，直道一百米，弯道是碎石子垫的——鞋底会被染成黑的。师院是四百米塑胶跑道，八条红道，白分界线画得很直——有些地方已经被磨淡了。跑道内有足球场——人工草皮，晚上总有人在踢球。足球在草皮上滚过的声音沙沙的——像鞋底轻轻在地板上擦。看台上偶尔坐着几对情侣——靠得很近，说话声音很低，偶尔有笑声被夜风吹过来。

他走第五道。第一学期三圈——这学期四圈。每圈四百多米，四圈大概十三四分钟。步频不快——手插在运动裤兜里。运动裤是深灰的——膝盖那洗得发白。

四月的操场上飘着各种味道。人工草皮的胶味——天热时更明显。跑道两边香樟树开花——白花香淡淡的，若有若无。夜风偶尔把食堂油烟吹过来——炸鸡排和辣椒炒肉的味道。塑胶跑道本身的橡胶味——被白天的太阳晒过以后更重。还有洗衣液的香味——从跑道边经过的女生身上飘来的。

第五道上。步子不快。前面有个女生在跑步——马尾，浅绿色运动服——马尾在跑动时左右晃动。他看着那个马尾——橘色灯光下有节奏地摆动——跟高中那个很像。但不是。浅绿色不是她的风格——她喜欢白色和浅蓝色。他低头看脚下的白线。继续走。

高三也是在这条道上——那次——看到她跟一个男生并肩走——后来赵一鸣打听到是她表哥。他走了一个多小时——走到脚底起泡——走到决定高考后表白。

那次的决定做得快——目标清晰——考完、表白、让她知道——像跑四百米——起跑线、终点线、明确的跑道。但现在不一样了——现在他不知道自己还能决定什么。她说"暂时不要联系"——没说"永远"——也没说"什么时候可以"。他等了一年——不知道还要等多久。但他没有别的选项——等是唯一的选项——不等就等于放弃——放弃，他不会。

他停了步——站在第五道的弯道上——身体不由自主往内侧倾了大概十五度。白色分界线在弯道处是弧形的——前后都是弧——看不到起点——也看不到终点。他站在弧线中点上——灯光下的影子弯成奇怪的形状。

第四圈——走得更慢。有个穿红色运动服的男生从他身边超过——跑得很快——后面跟了一个女生，保持大概五米的距离。男生跑到弯道尽头停下来等她，两个人一起慢慢走——男的说了句什么——女的拍了他手臂一下。操场踢球声开始散了——有人收球、有人道别。一个球滚到他脚边——他停下踢回去——踢球的人远远举了下手——他点头。

看台上一对情侣站起来走了——男的拉女的的手上台阶——女的说了一句——男的笑了——声音很轻，在夜风里散了。

四圈走完。站在最外道起点线上——这次没马上走——深呼吸一下。操场饮水台——不锈钢台面，三个水龙头，中间那个坏了——透明胶带缠着。左边龙头上贴着纸条——"出水量大，小心溅湿"。在没坏的龙头上接了口自来水——漂白粉味——喝了两口——剩下的浇在旁边草上——水渗进土里——草叶动了一下。

看表——十点十分——多走了一圈。回去前在操场门口站了一会儿——整个跑道在白色灯光下很亮——像一个独立的世界。经过黄色电话亭——电话亭的灯灭了——灯泡坏了——没人修。亭子的玻璃脏了一块——上面有一道干的白色鸟粪。玻璃里能看到银色听筒——挂着。电话亭旁的法国梧桐开始冒新叶子——嫩绿的、小小的、毛茸茸的。他在这停了三秒——然后继续走。

回到512。阿坤在打游戏——李博已经睡了。小伍床空着——去自习室通宵了。顾临风坐在床边脱鞋——鞋底又沾了红色跑道颗粒——用手指拨掉。弯腰捡起来——放进阿迪达斯旧鞋盒里——鞋盒里已经攒了一小层红色颗粒——没数过——但大概几十颗了。

李博翻了个身。床板咯吱响了一声。他闭着眼说了一句——"今晚走了几圈"。

"四圈。"

"比平时多。"

"嗯。"

"操场那边有人放风筝。"

"没注意。"

"红色的。下午放的。天黑了还在。"

"嗯。"

李博不说话了——大概又睡着了。阿坤的键盘还在响。窗外有风把梧桐树枝吹在窗户玻璃上——轻轻地刮了一下。

顾临风坐在床边。把鞋放好。躺在床上。枕头下面的手机硌着后脑勺。他伸手进去碰了碰手机壳——温的。没有拿出来。天花板的裂缝在路灯光里是一条细细的黑线。他盯着那条黑线。脑子里没有明确的画面——只有操场的跑道——红色的——白色的分界线——第五道——一圈一圈地转。转到最后停在一个电话亭前面——电话亭是黑的——灯灭了。

他翻了个身。脸冲着墙。墙上那个小洞还在——前一个学长钉钉子留下的。他把手指伸进去——洞的内壁是凉的。缩回手。闭上眼睛。外面的路灯在窗帘上投了一片橘色的光。那片光一动不动——像一块贴在墙上的透明纸。

301路的路线又在脑子里过了一遍。没有刻意去想。是自己走出来的。师院南门——市图书馆——市体育馆——中山路口——公园西门——工商银行——人民医院——新华书店——广场东路——师范大学——师大北门。每一站之间的距离大概是三到四分钟车程。从人民医院到新华书店这段路是直的——公交车会加速到五十码。从广场东路拐向师范大学的时候有个小弯——司机会打转向灯，右拐进一条窄街。窄街两边种着银杏树。然后到师范大学——倒数第二站。然后师大北门——终点站。下车往右拐——校门是灰白色的。

他把这些站名在脑子里过了一遍。没有顺序——这次是从终点站往回走的。师大北门——师范大学——广场东路——新华书店——人民医院——工商银行——公园西门——中山路口——市体育馆——市图书馆——师院南门。倒着数也一个不落。

然后他睡着了。
