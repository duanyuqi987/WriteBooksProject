# 第059章 王启民的处分名单



时间：2009年一月中旬，雪后第二天



地点：年级办公室、十班教室和楼前空地



卷次：第三卷：雪落在两栋楼之间



2009年一月中旬，雪后第二天，年级办公室、十班教室和楼前空地，王启民手里那张被折过的名单先把这一天按进了十班的日常里。处分名单最终没有变成正式处分，而是通报批评、写说明和扫楼前积雪。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

赵远航觉得自己没砸中不该被写名字，常俊凡问他有没有参与。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看王启民手里那张被折过的名单，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

赵远航也在这一天被推到前面。赵远航的特点是熟悉县城路和地名，爱说路线，做事有一点莽。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“我又没砸中人。”赵远航说。

“没砸中不等于没扔。”常俊凡说。

“名单不是看谁运气好，是看谁参与了。”王启民说。

“扫雪就扫雪，别再吵了。”曹蔚说。

“我也去。”段璟说。

王启民手里的名单被折过一道，边角已经软了。那张纸没有贴出来，却比贴出来更让人紧张。

赵远航盯着名单，说自己只是喊了几句，没砸中人。曹蔚让他小声点，别在办公室门口继续加分。

常俊凡问他有没有扔。赵远航愣了一下，说扔了，但没中。她说那就写扔了。

段璟听见这句话，低头看自己的鞋尖。鞋边还沾着昨天的泥雪，干了以后变成灰白色。

王启民没有给正式处分，改成通报批评、说明书和扫楼前积雪。有人松口气，有人觉得丢脸。

付小川说扫雪好，至少比在全校大会念检讨强。宋乔说你别把扫雪说得像旅游。

陶建国把名单拿回班里，没有念得很重，只说参与过的人自己站起来。

站起来的人比大家想象的多。那一排校服在冬天的教室里显得有点笨，也有点真实。

王启民的处分名单没有悬在空处。办公室门口那张折过的名单，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，王启民的处分名单带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

王启民进门时不一定立刻说话，鞋底在水泥地上一停，王启民的处分名单带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在赵远航快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。王启民的处分名单越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。王启民手里那张被折过的名单让他想说话，又逼他先低头看题。

赵远航被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，王启民的处分名单留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。赵远航路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为王启民的处分名单翻身，谁还在被窝里背单词，旁边人都听得见。

王启民手里的名单一折，热闹就被折成了后果，谁也没法再只说自己是看客。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理王启民的处分名单里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：王启民手里那张被折过的名单、一只冻红的手、一句老师点名、一块还没扫干净的雪。

陈柏言接过扫帚时先看了看名单，确认自己名字没有被漏掉。这个小动作很快过去，却让教室更像一个真正在运转的班。

张若楠低头把说明写完，最后一句没有再替自己找理由。旁边的人没有特意看，可那一瞬间还是被段璟记住了。

刘青岚扫到台阶边的脏雪，才知道热闹结束后还有这么多泥水。常俊凡没有接话，只把手里的事做完。

王子越听见通报批评四个字，脸上发热，手却还得继续握着扫帚。老师没有立刻点名，那一秒沉默反而让人更知道该收住。

赵远航没有因此变成特别醒目的人，可在段璟眼里，赵远航从这天起不再只是旁边一张脸。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，十班第一次明白热闹也有后果，群像不是只一起笑，也要一起收场。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
