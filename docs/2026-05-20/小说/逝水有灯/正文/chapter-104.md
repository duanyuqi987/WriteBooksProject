# 第104章 诗社传单旁观

所属部卷：第二部：六月以后
时间：大一社团周
地点：图书馆前广场

## 诗歌插页 P104：原创现代诗

她写下许多名字，
每一个都要对齐格线。
有人把这一笔当作开始，
她只是在风里按住纸，
不让报名表被吹走。

苏晚枝那天登记了四十多个新生。她记得陆允川，是因为他的名字笔画多，写到“川”时，上一阵风把传单吹到地上，他弯腰替她捡起来。

“谢谢。”她说。

“没事。”

除此之外，他们没有多余的话。她对每个来报名的人都差不多：问清学院，提醒活动时间，告诉对方可以先旁听。她并不觉得这是什么特别的温柔。社团招新忙了一整天，她只希望晚上能赶上食堂的热汤面。

叶澄坐在旁边盖章，手腕酸得厉害。趁人少时，她低声说：“刚才那个男生看你写字看得挺认真。”

苏晚枝把登记册往左挪，“可能怕我写错。”

“也可能是文学青年通病。”

苏晚枝笑了笑，没有接。她并不讨厌被人认真看见，但她更怕别人把礼貌误当成某种信号。高中以后，她已经学会在笑容里留一点距离。不是冷淡，是给自己省麻烦，也给别人省误会。

招新结束时，广场上满地传单。苏晚枝蹲下捡了几张，把还能用的理平，坏掉的扔进纸箱。她根本不知道，那张被陆允川折进书包夹层的传单，后来会在他的故事里停留很久。

她只记得那天手指上的印泥洗了两遍才洗掉。
