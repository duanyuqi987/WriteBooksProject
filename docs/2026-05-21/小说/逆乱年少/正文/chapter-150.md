# 第150章 三年

时间：2012年12月31日，夜晚

地点：江城大学宿舍区及楼顶天台

卷次：第六卷：雪落无声

十二月三十一号。元旦前夜。

学校没有组织跨年活动。大部分学生都回家了——元旦假期连着周末一共三天。校园很空。宿舍楼道里只有零星的几盏灯亮着。

林玉起没回家。他说"要复习"。郭向阳也没回家——他在准备一个公司的面试，一月中旬的。两个人守在宿舍里。郭向阳在刷题。林玉起在看书。外面偶尔传来一两声爆竹——市区禁放了，但总有零星的人偷偷放。

晚上九点多。郭向阳合上电脑说"去楼下买点吃的"。宿舍里只剩下林玉起一个人。

手机震了。

苏安晴的消息。

"你在学校？"

"在。"

"我也没回家。"

过了大概十秒。

"林玉起。三年了。"

他看着这四个字。三年了。二零一零年秋天到二零一二年的最后一天。大一。大二。大三。加起来两年多。但她说"三年"——把入学年算成了第一个年头。

他不知道怎么回。打字框里的光标一闪一闪的。

然后她又发了一条。

"谢谢你一直在。"

这六个字。

林玉起把手机放在桌上。盯着屏幕。这六个字太轻了——任何人跟任何人都可能这么说。但也太重了——"一直在"。不是"来过"。不是"在过"。是"一直在"。

他拿着手机看了很久。久到屏幕自动暗了，他又点亮。暗了。又点亮。

他打了几个字。删了。又打了几个字。又删了。

最后他打了这行字：

"不用谢。我习惯了。"

点了发送。

发完之后他把手机放在桌上。过了大概五秒钟——他后悔了。

"我习惯了"这三个字。

太轻了。

太轻——因为"习惯了"像是在说：我在这里不是因为选择了在这里，而是因为惯性。不是因为你想让我在，而是因为我懒得走。

太重了——因为"习惯了"也是在说：三年不短。我每一天都在。我习惯了每天早上想到可能会见到你。我习惯了你每次从我身边走过的时候我的心跳会快一拍。我习惯了你的声音、你写字的样子、你讲课的时候在黑板上写自己名字的动作。我习惯了你的一切。所以不用谢。我真的习惯了。

但苏安晴不会听到这些。

她只会看到屏幕上的四个字——"我习惯了"——并试图在这四个字里读出他真正想说的话。她是那种会读字里行间的人。但字里行间的距离太远了。隔着三年的沉默、四十五首诗、一支钢笔、一个雪球、三里山路、一颗化在嘴里的糖。

这些她都知道。但她能拼出正确答案吗？

林玉起不知道。

他把手机放到枕头底下。站起来。在宿舍里走了两步。又坐回去。把手机拿出来。苏安晴没有回复。

QQ聊天窗口上方那一行"对方正在输入…"没有出现。

他等了十分钟。

还是没有。

郭向阳回来了——拎了两瓶可乐和一包瓜子。"楼下小卖部快关门了。只剩这个。"他看见林玉起脸上的表情。"你怎么了？哪个科目挂了？"

"没有。"

"那你这副表情是什么？"

林玉起没说话。郭向阳也不再问——他把可乐打开，放了一瓶在林玉起桌上。"喝。糖分治百病。"

十一点半。

林玉起穿上外套。郭向阳问"你去哪"。他说"出去走走"。郭向阳叹了口气说"跨年夜一个人在校园里走——也就你了。"

楼道里空荡荡的。走廊尽头的应急灯亮着绿幽幽的光。他走下楼。操场上没有人。雪停了但地上还有残雪——灰白色的，硬邦邦的，踩上去不是咯吱声，是轻微的碎裂声。像踩在很薄很薄的玻璃上。

他走到宿舍楼的天台上。

天台的门平时是锁的——但锁坏了。去年就坏了。宿管大爷修过一次，又坏了，就不管了。林玉起推开铁门，走进天台。

天台上的风比地面的风大了不止一倍。迎面吹过来，把他的头发全部往后掀。耳朵瞬间冷了，然后开始发疼。他把衣领立起来，走到天台边缘的栏杆前面。

从这里能看到大半个校园。教学楼。图书馆。操场。梧桐树。全部被残雪盖着。路灯的光在雪上晕开，一圈一圈的，像在水面上投了石子。远处城市的灯火连成一片——橘色的。偶尔有几道光柱在楼宇之间穿过，是汽车的远光灯。

手机震了。

他掏出来看。苏安晴的消息。

"你在哪？"

"天台。宿舍楼的天台。"

"冷不冷。"

"还好。"

"我想了很久你说的那句话。"

林玉起屏住了呼吸。

"你说的'习惯了'——我想了很久。我觉得你其实是三个意思。"

"哪三个？"

"第一个：'你不用谢，我在这里不是因为帮你，而是因为我自己想在这里。'"

"第二个：'三年不短，但我没觉得长。'"

"第三个——我不知道第三个是什么。但我觉得是你没说的那个。"

林玉起看着这四行字。天台上的风吹得他手指发僵，打字很慢。

"你怎么猜的。"

"因为你写诗也是这样的。每首诗都藏了一层没写出来的东西。读你的诗读久了，我就知道你在说'习惯了'的时候，一定还有一句没发出来的话。"

远处传来零星的爆竹声。有人在家里提前放烟花了。几朵白色的花在城市的天空上绽开，又灭掉。声音传到天台上的时候已经很远了——轻轻的、闷闷的，像远处有人关了一扇门。

"苏安晴。"

"嗯。"

"新年快乐。"

隔了大概五秒。

"新年快乐。"

又隔了三秒。

"林玉起。那句话你没有发出的——你可以写成诗给我看。"

这一次林玉起没有犹豫。他站在天台的栏杆前面，用冻僵的手指按手机键盘。

"好。写好了发给你。"

等了一会儿。

"你不怕我读了猜错吗？"

这次是苏安晴先问的。

"不怕。你刚才已经把三句错了两句。"

"错在哪？"

"其实只有一句。三个意思其实只是一句话换了三种说法。你拆开了。"

对方正在输入。

又停了。

又输入。

然后停了很久。

然后发了两个字。

"傻子。"

林玉起看着这两个字。风吹得屏幕在眼前晃。他抓紧手机。然后笑了。

他靠在栏杆上。栏杆的铁管冰得刺骨——隔着羽绒服都能感觉到。但他的胸口是热的。

天台上。二零一二年最后一分钟。

远处的天空上有更多的烟花升起来——不是零星的，是连成一大片的。红的、绿的、金黄的。在寒冷的夜空里炸开，映在残雪上，给白色的雪补了一瞬间的颜色。

他低头看手机。苏安晴没有再发消息。但他知道她也在。在宿舍里。可能在床上。可能也在看烟花。可能也在看手机。

他把手机放进口袋里。口袋里很暖。他又掏出来。打开记事本——用手指打字太慢，但在天台上，在这个时间，他想先记下来。不用钢笔。用备忘录就行。写完了，明天用钢笔抄。

他打了三个字作为新诗的标题。然后停住了。把标题删了。重新打。

不再用标题了。从第一行开始。

>
天台风正冷，三年雪未融。
少年不敢说，字字在诗中。
>
