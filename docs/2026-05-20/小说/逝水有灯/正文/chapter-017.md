# 第017章 前排和后排

所属部卷：第一部：初秋三楼
时间：高一春天
地点：物理课后的教室

## 诗歌插页 P017：原创仿唐绝句

红笔轻划纸生寒，
前后书声隔一栏。
莫问少年何处近，
同窗也有万重山。

夏知遥的作业本没有传到陆允川手里。

这件事本来很正常。座位安排在那里，本子按组传，轮不到就是轮不到。可陆允川看着那本淡蓝色练习册从前排往后走，停在中间，再被另一个男生打开，心里还是像被什么轻轻碰了一下。

他拿到的是程砚的本子。

程砚第一题漏了单位，第二题把方向写反，第三题干脆只画了个图。陆允川用红笔给他改，越改越无语：“你上课到底听没听？”

“听了。”程砚趴在桌上，“听见老师说下课前要收。”

陆允川忍不住笑。笑完以后，他的目光又往前面飘。夏知遥坐得很直，正在给同桌的作业写批注。她写字不快，每写一行都会停一下，像在确认自己有没有说清楚。

物理老师从走道经过，敲了敲陆允川桌面：“别只看别人，自己的也要订正。”

附近几个同学笑起来。陆允川脸一下热了，低头去看自己的本子。老师说的是作业，他却像被戳中了另一件事。

下课后，程砚把自己的练习册抽回去，看见上面密密麻麻的红字，皱眉：“你对我是真狠。”

“你错得多。”

“我错得多也没见你这么认真过。”程砚收拾书包，“你是不是把没改到的那一本，全补偿到我这本上了？”

陆允川不说话，拿起水杯去水房。

水房里人很多，地面湿滑。陆允川排在队伍后面，听见前面两个男生讨论夏知遥的字好看。一个说她批注细，另一个说想让她改自己的本子。陆允川心里忽然有点不舒服，可他很快意识到，这种不舒服没有正当理由。

轮到他接水时，热水溅到手背，他才回过神。水烫得他缩了一下，杯子差点掉进池子。

回教室的路上，他看见夏知遥站在窗边背公式，手里还是那本淡蓝色练习册。她没有注意到他。她只是用红笔在某道题旁边补了一个箭头，然后把书合上。

陆允川停了半步，又继续往后排走。前排和后排之间只有几张课桌，可他第一次清楚地感到，距离不是用步子量的。
