# 第059章 最后一场模拟考旁观

所属部卷：第一部：初秋三楼
时间：高三春天
地点：县一中考场

## 诗歌插页 P059：原创现代短诗

那时我们都以为，
结束会有清楚的门。
后来才知道，
门只是普通下午的一阵风。
有人走向远处，
有人学着把远处还给远处。

从旁观者的角度看，最后一场模拟考没有陆允川心里那么沉重。它是很多人共同经历的一天，只是他把其中一角看得格外亮。

程砚看见的是他的迟疑。程砚从考场出来，第一句话不是问考得怎样，而是说：“走，先喝水。你脸色比答题卡还白。”这句话带着玩笑，却也带着不想看他继续自困的急。

叶澄看见的是边界。叶澄把用坏的中性笔芯扔进垃圾桶，提醒他们别在走廊大声对答案，后面还有班级没考完。她不是站在谁那边，只是不愿让一个人的心事压过另一个人的自由。

夏知遥也应当有自己的画面。夏知遥交卷后没有立刻走，她和同伴站在走廊窗边对答案，说到最后一道大题时眉头皱了一下，又很快松开。她不只是被喜欢的人，也是在同一段时间里被未来推着走的人。

模拟考那天，考场门口贴着座位号，胶带边缘沾了灰。桌子被拉开距离，书包统一堆到前面，整个教室像临时拆掉了往日的亲近。

旁观让答题卡边角的压痕变小了，也变准了。它不再是命运，只是当时确实存在的一个物件，放在考场里，有光，有声音，有人来往。

老师拆开密封袋时，教室里安静得只剩纸张摩擦。陆允川低头写姓名，忽然觉得这三个字不再只是自己，也包括家里人期待的目光。

那张答题卡后来没有留下，但边角被橡皮压出的痕迹一直留在他的记忆里。它提醒他，青春不只是喜欢谁，也是一场场必须独自交卷的考试。

陆允川不喜欢这种视角，因为它让他的深情显得没有那么唯一。但正是这种不唯一，给了后来的人一点余地。

陆允川后来愿意留下这一笔，是因为它不漂亮。准考证不漂亮，答题卡也不漂亮，可它们让人相信，这段青春不是从空中落下来的。

他没有再给夏知遥安排额外台词。她已经在考场里完成了自己的动作，剩下的沉默，应当由陆允川自己消化。

程砚把话说得粗，叶澄把话说得稳，两种声音都把他往现实里拉。陆允川当时不领情，很多年后却知道自己被救过几次。

那天的小卖部冰柜里的矿泉水没有特别好吃，却因为太具体，抵住了那些过大的形容词。人有时候正是靠这些具体的味道，才没有被回忆吞没。

走廊里有人笑着说完了，也有人低头算分。这些声音把最后一场模拟考从一个人的心事里拉出来，变成许多人共同挤过的一天。陆允川不再只听见自己胸口的动静，也听见椅子、脚步、塑料袋和远处的喊声。

夏知遥站在走廊窗边和同伴核对最后一道题。这个动作很小，却让她从陆允川的想象里退回到真实的人：她会忙，会累，会被家里和未来推着走，也会在必要的时候保持沉默。

程砚把矿泉水塞到他手里，说话还是老样子，不好听，却有效。陆允川皱了皱眉，还是照着他说的做了一点，因为朋友有时不是负责安慰，而是负责把他从自我感动里拽出来。

叶澄提醒他们小声一点，别影响隔壁考场。她没有把话说得太满，只把边界摆到桌面上。陆允川那时不一定立刻懂，却会在后来某个晚上想起这句话，知道自己当年被提醒过。

母亲晚上问他要不要加个鸡蛋，他说不用又改口说要。家里的声音一进来，青春就不再只是校园里的风。它变成饭点、车费、睡眠、行李和一句不太会表达的关心。

他低头看着答题卡，忽然意识到它不是命运给他的标记，只是那天确实存在的东西。正因为确实存在，它比任何夸张的形容都更能留下来。

橡皮屑、试卷油墨和走廊热风混在一起，让记忆有了可以落脚的地方。陆允川后来写到这里，没有再急着解释自己多难过，而是先把气味、光线和手上的动作写清楚。

他把透明笔袋收好，又拿出来检查了一遍。这个来回不像英雄，也不像受害者，只像一个年轻人不知道怎样安放自己，于是先安放手边的东西。

旁观者看见的陆允川没有那么悲壮。他只是别扭、敏感、想被理解，又不愿直接承认自己需要理解。

从夏知遥那边看，事情也许更简单。她正在处理自己的生活，不可能替一个没有说清楚的人完成所有情绪。

这一笔补进去以后，章节没有变得更响，反而更稳。最后一场模拟考不再只是一个标题，而是有答题卡、有透明笔袋、有人说错话也有人及时停住的真实一天。

监考老师把多余答题卡收回袋子里，红色密封条贴上去时发出很轻的响声。陆允川突然觉得很多事情也是这样，到了时间就会被封住，不等人补写。

夏知遥和同伴讨论错题时，没有一句话和陆允川有关。这个事实很普通，却像一盆冷水，让他看见自己这些年只是站在她生活外面。

程砚拉他去小卖部买水，路上一直骂最后一道题缺德。陆允川听着听着，才从夏知遥那边收回神，开始认真回想自己到底哪里算错。

考场里的光线一点点变暗，准考证和矿泉水都还在原处。陆允川把透明笔袋收好，动作慢得像怕惊动什么。可真正需要被惊动的不是别人，是他自己。

他后来重写这一章时，特意把答题卡留在中间。不是为了让它变成象征，而是因为人只有摸到具体东西，才知道自己曾经怎样站在具体生活里。

旁人看见的不是传奇，只是一个少年把心事攥得太紧。把镜头挪远以后，最后一场模拟考才显出真实比例：它重要，却不是世界的全部。

这一章把镜头挪开一点，不是为了削弱他，而是为了让每个人都重新站回自己的生活里。收卷铃、桌椅挪动声和走廊里压低的对答案声继续响着，谁都不能只活在他的解释里。
