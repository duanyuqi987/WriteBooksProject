# 第097章 省城南湖

所属部卷：第二部：六月以后
时间：大一九月
地点：大学南湖凉亭

## 诗歌插页 P097：原创仿唐绝句

灯色微明照旧尘，
少年别后识前因。
一声铃尽人潮散，
半页风光不问春。

第二天早上，祁野果然在六点就起来了。他叫醒程砚和陆允川的时候，自己已经洗漱完毕，烟都抽完了一根。

"起这么早？"程砚揉着眼睛从床上坐起来，发型爆炸。

"送人就要早。晚了火车站人多。"祁野把背包甩到肩上，"我下楼买包子。你们吃什么馅的？"

"肉。"程砚说。

"韭菜鸡蛋。"陆允川说。

"收到。"

许知恒也起来了——他戴着眼镜靠在枕头上，手里还抱着那本《材料力学》。程砚路过的时候拍了拍他的书皮。"你以后肯定考研究生。"

"你怎么知道？"

"因为开学前就提前学的人只有两种：一种是天才，一种是已经把大学当跳板的人。你明显是第二种。"

许知恒嘴角动了一下，没有否认。

祁野买了包子回来，一人一个塑料袋拎着。四人边走边吃，往火车站方向走。省城火车站的广场白天人很多，到处是拉客的三轮车和举牌接站的学生。祁野把烟掐在垃圾桶上，转头对陆允川说："以后来省城，找我。我在省大计算机系。宿舍号我等会儿发给你。"

"你昨天才认识我。"

"昨天够了。我交朋友不看时间。"

程砚在旁边笑。"你这台词怎么像港片里的老大。"

"港片老大会说'以后我罩你'。"祁野把包子最后一口塞进嘴里，"我不是老大。我就是觉得你们两个人不错。一个损得够狠，一个忍得够稳。"

陆允川低头吃包子。韭菜鸡蛋馅的，包子皮有点厚。但热乎的。

检票进站的时候，程砚陪陆允川走到站台。两个人站在黄线后面，旁边是去沿江方向的绿皮火车。

"到了给我打电话。"

"好。"

"你宿舍条件怎么样？"

"六人间。有空调。应该还行。"

"有空调就不错了。我那宿舍据说只有风扇。"程砚叹了口气，然后忽然换了个表情——不是嬉皮笑脸，是难得严肃的那种。"陆允川。"

"嗯？"

"你在大学，别老一个人待着。多跟人说话。你高中那种闷法，不适合大学。大学没有我在旁边损你，你得学会自己找人聊天。"

"我知道。"

"你说'我知道'的时候我反而不太放心。"程砚拍了拍他的肩膀，"算了。你自己看着办吧。有事打电话。"

火车开动了。陆允川坐在靠窗的位置，看着站台上程砚的身影越来越小。程砚站到最后一刻才走——火车拐弯之前，他还在朝这个方向挥手。

沿江的大学比他想象的大。从校门口到宿舍楼要走二十分钟，路两边是高大的悬铃木。他拖着行李箱走在林荫道上，身边不断有新生经过——有人被学长学姐领着办报到手续，有人跟家里人一起拍照留念，有人骑着共享单车在人群里穿梭。

宿舍在五楼。他到了门口，发现门开着，里面已经有人了——一个瘦高的男生正在铺床单，嘴里哼着歌。听到脚步声他回头看了一眼。"你就是陆允川？名单上那个？"

"对。你怎么知道？"

"门上贴了名单。四个人的名字都在上面。"瘦高男生把床单扯平了，从上铺跳下来，"我叫季朗。本地人。你呢？"

"外地的。沿江那边的一个县城。"

"县城好啊。县城考出来的学生学习都猛。"季朗说话速度很快，像嘴里一直含着一口气，"你要不要我帮你铺床？我铺床技术一流——当过两年兵。"

"你当过兵？"

"不是。我军训标兵。高中军训的时候学的。"

陆允川没忍住笑了。他把行李箱打开，拿出母亲帮他准备的床单和被子。床单是母亲新买的，蓝色格纹，边角缝了她的名字缩写——大概是怕跟别人搞混。

铺好床以后，他站在宿舍阳台往外看。夕阳正从远处那座江桥后面沉下去，把江面染成橙红色。有江的城市。他真的到了。
