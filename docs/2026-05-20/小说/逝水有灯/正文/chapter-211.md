# 第211章 南方出租屋之前

所属部卷：第五部：逝水有灯
时间：工作第一年，雨季
地点：南方城市出租屋

## 诗歌插页 P211：原创现代诗

出租屋的墙皮落下来
不是雪
只是成年以后
许多话
终于没有地方夸张

南方的雨下得很久。陆允川下班回来，裤脚湿到小腿，楼道里堆着快递纸箱，电梯停在十七楼不动。他住在六楼，便拎着电脑包一层一层往上爬。钥匙插进门锁时，屋里有一股潮味，像没晒干的衣服被关了一整天。

房间很小，一张床、一张折叠桌、一个衣柜，窗边墙皮鼓起一块，边缘已经翘白。他把外套挂到门后，先去接了半盆水，因为水龙头关不严，总是一滴一滴响。成年生活没有他想象中体面，账单、日报、房租、地铁末班车，把人的心事切得很碎。

那天他原本只想清电脑内存。硬盘快满了，系统提醒了三次。他打开文件夹，把工作资料、发票照片、培训课件拖到不同目录。鼠标划过一个旧文件夹时，他停住了。文件夹名字很普通：照片备份。

他没有立刻点开。窗外有车灯扫过玻璃，墙皮的影子在桌面晃了一下。第五部的故事不是从某个隆重回忆开始，而是从一个加班后的晚上开始。人累到没有力气煽情，旧事才有机会按真实大小出现。
