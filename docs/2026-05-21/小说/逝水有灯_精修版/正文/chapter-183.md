# 第183章 道歉未发送之后

所属部卷：第四部：失去的美丽
时间：当天傍晚
地点：宿舍

## 诗歌插页 P183：原创仿宋小令

微光边，夜色平，
半窗旧影入疏灯。
问当年、几回执念，几回无凭。
不把归人留梦里，
水声明日照常生。

没发出去的那封信，比发出去的难处理。

因为发了之后会有回应——好的坏的都有一端落定。但存在本地不发的道歉信，只能自己消化。就像自己给自己缝针，每一针都要自己找位置。

陆允川回到宿舍的时候，天已经快黑了。走廊里的灯还没全亮，只有尽头的应急灯亮着那点绿幽幽的光。推门时一股暖气和一股泡面味同时扑过来——暖气片终于恢复了，虽然还是不够暖，但至少不再是冰凉的铁片了。

祁野正在为抽卡愁眉苦脸。他盘腿坐在床上，手机屏幕的光打在他脸上，把他的表情照得一阵蓝一阵白。

"又没出货？"

"没。第一百二十抽了，保底还没出。我要写投诉信。"

"对一款免费游戏的抽卡系统写投诉信？"

"对。内容已经写好了——'我自开服以来累计氪金已超过三个648元，至今未获得限定角色，特此致函。'语气要义正词严但我不是真投诉。就是想写。"

"然后发吗？"

"不发。写完了我就舒服了。发出去人家也不看——但这种信不发比发了更让人心情好。因为不寄出去的抗议——是我赢了游戏公司，不是游戏公司无视我。"

陆允川看着祁野，忽然觉得这个逻辑他很熟悉。

"我刚刚在图书馆做了一件跟你打算做的事一模一样的事。"

"写投诉信？"

"写道歉信。想了半天——最终没发给苏晚枝。"

"那你写信的时候有没有用敬语——"

"正经一点。"

"好吧。"祁野放下手机，正色道，"那封信写了什么？"

"写我意识到——我在所有事里都在等回应。不是真心道歉——是想用道歉换她原谅。"

"那为什么不发？"

"因为发出去就是新一轮索取。"

祁野靠在椅背上，双手枕在脑后。他用这个姿势盯着天花板看了几秒。

"我跟你之间最大的区别——"

"什么？"

"你的不改是一种智慧。我的不改是一种懒。"

程砚在床上忽然出声——他一直戴着耳机翻书，但耳机里的英文听力刚好在章节末尾停住了，他把耳机摘下来，对话就断断续续飘进他耳朵里。

"你们两个在说什么——道歉信和投诉信？"

"对，"祁野说，"陆允川写了封道歉信没发。我写了封投诉信没发。他是因为不想再索取。我是因为懒得贴邮票。"

"你没有邮票。那是免费游戏，投诉入口在设置页面。"

"那就更懒了。连点开设置页面都不想。"

程砚把书反扣在床上看向陆允川。"道歉信——自己写的还是叶澄审核过的？"

"自己写的。全篇自己写，然后自己决定不发。"

"很好。那如果之后你真的有机会跟她重新说起这些——你想说什么？"

陆允川想了想。"不是道歉。是说——在你身上的那几年，让我学会了怎么对下一个人更好。不是'对不起'，是'谢谢'。"

"这句不需要道歉的对象。只需要一段足够长的停顿。"

"嗯。"

程砚把书翻回去。QPCR的数据又跑了一页——他看不懂英文论文时就爱用这种点头表示理解，但这次他完全看懂了。

那晚熄灯之后，陆允川把手机备忘录打开，看了眼白天打的那行字：

"最好的道歉不是'对不起，我错了'。是——你现在可以不用再提心吊胆了。"

他把这行字复制到"不再分析的日常"文件夹里。然后加了一行新的：

"今天写了人生第一封不发出去的道歉信。不是因为怕被拒绝——是因为这次真的没想要回应。那封信的收件人不是我需要的东西——是我曾经欠她的东西。还她不等于亲手交到她手里。还钱要亲手交——还不安不能亲手交。亲手交等于——'你看，我把你的心结还给你了，你看我多诚恳。'"

打完这些字，他关掉手机。宿舍里很安静。泡面桶已经扔掉了。
