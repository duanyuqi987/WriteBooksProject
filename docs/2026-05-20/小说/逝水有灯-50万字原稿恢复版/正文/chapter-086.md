# 第086章 录取通知书之前

所属部卷：第二部：六月以后
时间：七月下旬
地点：镇上邮政所

## 诗歌插页 P086：原创现代诗

我把水放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
红色录取通知书信封没有替谁回答，
只把那一年轻轻照亮。

录取通知书真正到来之前，日子已经先变了颜色。七月下旬，镇上邮政所的风扇转得很慢，柜台玻璃上贴着取件须知。陆允川拿着身份证排队，手心被汗浸得发黏。

邮递员从一摞信封里抽出红色通知书，让他签字。陆允川写下名字时，笔尖在最后一笔停了一下，像确认自己真的要去另一个地方。

他还没有遇到那个必须面对的瞬间，却已经被红色录取通知书信封提醒：有些事不能无限期地放在心里，它会跟着时间一起变形。

程砚拿到自己的通知书后在群里发照片，说谁先到大学谁请第一顿饭。陆允川回了一个好，手指却停在夏知遥那条消息上方。

叶澄私下发来一句恭喜，又补了一句：新的地方不是用来逃旧事的，是用来学会正常生活的。

夏知遥的消息是从同学群里传来的。她去了北边一所学校，离省城不算近，也不算远，刚好足够让人停止幻想偶然。

风扇转动、柜台叫号和邮递员翻信封的声音，把他的心事压得没有那么夸张。人在这样的声音里，很难一直把自己当成故事中心。

母亲把通知书拿过去看了很久，摸信封边缘时动作很轻。她说省城远，坐车要早点去。陆允川点头，第一次觉得远不是形容词，而是车票、行李和一段无法临时折返的路。

同学群里热闹了一整晚，学校名、专业名、城市名不断跳出来。陆允川看着那些名字，忽然明白大家真的开始被分往不同方向。

那时陆允川还不知道，所谓成长不是突然明白，而是先在身份证、红色信封、邮政所柜台这些小东西旁边学会停顿。

他在日记本里写了两行，又划掉一行。留下来的不是誓言，只是一个具体下午：邮政所门口冰柜里的绿豆雪糕、窗外的光，以及他没能说出口的话。

陆允川后来愿意留下这一笔，是因为它不漂亮。身份证不漂亮，红色信封也不漂亮，可它们让人相信，这段青春不是从空中落下来的。

他没有再给夏知遥安排额外台词。她已经在邮政所里完成了自己的动作，剩下的沉默，应当由陆允川自己消化。

程砚把话说得粗，叶澄把话说得稳，两种声音都把他往现实里拉。陆允川当时不领情，很多年后却知道自己被救过几次。

那天的邮政所门口冰柜里的绿豆雪糕没有特别好吃，却因为太具体，抵住了那些过大的形容词。人有时候正是靠这些具体的味道，才没有被回忆吞没。

邮政所里有人取包裹，有人给外地孩子寄被褥。这些声音把录取通知书从一个人的心事里拉出来，变成许多人共同挤过的一天。陆允川不再只听见自己胸口的动静，也听见椅子、脚步、塑料袋和远处的喊声。

夏知遥去北边学校的消息在同学群里跳出来。这个动作很小，却让她从陆允川的想象里退回到真实的人：她会忙，会累，会被家里和未来推着走，也会在必要的时候保持沉默。

程砚晒了自己的通知书，说开学谁先到谁请饭，说话还是老样子，不好听，却有效。陆允川皱了皱眉，还是照着他说的做了一点，因为朋友有时不是负责安慰，而是负责把他从自我感动里拽出来。

叶澄私下发来恭喜，又提醒他新地方不是逃避旧事的。她没有把话说得太满，只把边界摆到桌面上。陆允川那时不一定立刻懂，却会在后来某个晚上想起这句话，知道自己当年被提醒过。

母亲把通知书看了又看，开始念叨路费和行李。家里的声音一进来，青春就不再只是校园里的风。它变成饭点、车费、睡眠、行李和一句不太会表达的关心。

他低头看着红色信封，忽然意识到它不是命运给他的标记，只是那天确实存在的东西。正因为确实存在，它比任何夸张的形容都更能留下来。

柜台旧木头和门口绿豆雪糕的甜味混在一起，让记忆有了可以落脚的地方。陆允川后来写到这里，没有再急着解释自己多难过，而是先把气味、光线和手上的动作写清楚。

他把签字笔收好，又拿出来检查了一遍。这个来回不像英雄，也不像受害者，只像一个年轻人不知道怎样安放自己，于是先安放手边的东西。

真正的变化还没到来，镇上邮政所已经先显出紧绷。陆允川站在里面，既想让日子快一点，又害怕快起来之后自己没有准备好。

他在日记本上写下“等以后”，写完又觉得“以后”太轻。很多拖延都披着克制的外衣，其实只是害怕被现实拒绝。

这一笔补进去以后，章节没有变得更响，反而更稳。录取通知书不再只是一个标题，而是有红色信封、有签字笔、有人说错话也有人及时停住的真实一天。

邮递员核对身份证时念了一遍他的名字，声音很平常。陆允川却听得很认真，像这个名字从今天起要被贴到新的宿舍、新的课本和新的城市里。

同学群里有人问夏知遥具体哪个专业，她回得简短，后面加了一个笑脸。陆允川看着那个笑脸，终于没有把它翻译成和自己有关的意思。

母亲把通知书放进塑料文件袋，又找出一个旧行李箱。箱子拉链有点卡，她来回拉了几次，陆允川蹲下去帮忙，才真实感觉到离开正在发生。

邮政所里的光线一点点变暗，身份证和绿豆雪糕都还在原处。陆允川把签字笔收好，动作慢得像怕惊动什么。可真正需要被惊动的不是别人，是他自己。

他后来重写这一章时，特意把红色信封留在中间。不是为了让它变成象征，而是因为人只有摸到具体东西，才知道自己曾经怎样站在具体生活里。

事情尚未发生，陆允川已经开始给自己找退路。他把“以后再说”想得很体面，可邮政所里的每一种声音都在提醒他，退路拖久了也会变成墙。

章节落下时，录取通知书到了，陆允川去省城读大学，夏知遥去了另一个方向。还没有真正展开。陆允川只是把东西收好，跟着人群往前走，心里那点执拗被现实轻轻推了一下。
