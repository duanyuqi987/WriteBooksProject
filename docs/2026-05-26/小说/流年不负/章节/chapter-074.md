# 第074章 成绩

时间：2013年6月25日

地点：各自家里

卷次：卷四：夏夜湖光

六月二十五号。高考成绩出来的那天。

顾临风一大早就醒了。醒的时候床头的闹钟显示六点零二分。窗外的天刚刚亮透。淡蓝的。有朝霞。橙的一小条。

他从床上坐起来。站到窗边。靠在窗台上。看楼下的垃圾桶旁边有一只鸟在地上啄东西。看不清楚什么东西。大概是昨天谁丢的瓜子皮。

他把电脑打开。主机发出嗡嗡的声音——风扇里头大概积了灰，声音比以前响了。屏幕亮起来的时候他眯了一下眼。太亮了。从黑屏到亮屏——眼睛没适应。他把屏幕调暗了。按了两下显示器底下的按钮。亮度从80降到50。

他坐在椅子上。椅子是木头的——扶手的位置被他磨出了凹痕。三年。每天坐在这把椅子上写作业、复习、写诗。椅面的漆磨掉了——露出底下浅色的木头。

八点。查分系统还没开。他刷新了一下浏览器。空白。再刷新。还是空白。第三次刷新的时候网页加载得特别慢——可能全国都在刷。进度条走到一半。卡住了。又动了一格。然后又卡住了。最后跳出四个字——"系统未开放"。

QQ上赵一鸣在线。

"你进得去吗。"

"进不去。没开。"

"我有预感。我考得不错。"

"你每次都有预感。"

"这次真的有。"

"嗯。"

"你的呢。能上多少。"

"不知道。"

"不知道还是不敢想。"

顾临风没回。他去厨房倒了一杯水。温的。他妈早上烧的。放在暖壶里。暖壶嘴上的水垢有点厚。水倒出来的时候带了几片白渣。他看了一会儿。把水喝了。白渣滤在杯底。

回到电脑前。赵一鸣又发了一条。

"苏晚晴多少你知道吗。"

"不知道。没问。"

"你猜。"

"不猜。"

"你这个人真没意思。"

把浏览器刷了一遍。这次页面变了。查分入口的按钮亮了。顾临风把鼠标放在按钮上。停顿了两秒。点进去。

准考证号。

身份证号。

他的手抬起来。手指悬在键盘上面。然后开始打字。一个一个数字打。打错了一个——删掉重新打。

页面跳转。

网页刷了一下——先跳出表格的框架，然后一格一格地填充。最先出来的是理综。然后是语文。然后是数学。然后是英语。最后——总分那行从灰色变成了黑色。

理综：197
语文：112
数学：127
英语：113
总分：589

他看着屏幕上的589。看了一阵。没有动。鼠标的光标在屏幕的空白处闪。闪了大概十几下。他伸手把光标移到总分那一栏。选中——复制——然后没地方粘贴。又把光标移开了。

589。他对着那三个数字点了点头。他估的是589。一分不差。

他把成绩单从上到下又看了一遍。数学127——选择题全对。填空错了两道。解答题扣了步骤分。理综197——物理那道电磁感应果然没给全分。英语113——阅读大概错得不多。作文扣分应该正常。

他把分数截图。发给了赵一鸣。

赵一鸣秒回。

"靠。589。兄弟你可以啊。"

"你多少。"

"579。"

"那不是也行吗。"

"差你十分。"

"差不多。"

"差十分算差不多？"

"嗯。"

赵一鸣发了一串省略号。

"苏晚晴多少你知道了吗。"

"还不知道。"

"等她发吧。"

"嗯。"

他从椅子上站起来。走到阳台上。阳台上晾着被单。白色的。被风吹得鼓起来象一面帆。他站在被单后面。被单的影子落在他脸上——明一下暗一下。

隔壁的阳台上有收音机在放戏曲。梆子腔。字咬得很重。唱的是"人生如戏——聚散不由己——"。收音机里掌声响了。唱完了。然后主持人说了句什么。然后是另一段梆子腔。

手机响了。

QQ消息。

苏晚晴。

"你查了吗。"

"查了。"

"多少。"

"589。"

"你估的好准。"

"你的呢。"

停了大概四十秒。

"612。"

"你考得不错。"

他打这四个字的时候手指在屏幕上擦了一下。手机屏幕左边有个暗角——碎过——赵一鸣那次闹的时候把他手机碰掉地上了。

"你也是。"

他也考得不错。589。虽然比她低了23分。但在他们学校也算是好的了。他的成绩一直排在班里三十几名——589是他高中三年考得最好的一次。

高三那最后几个月——他不知道是因为怕跟不上她而开始用功了。还是为了能去楚州师院。

只知道每一页翻过去的题都和她有关。

手机响了。

"23分。"

她发这两个字的时候。QQ对话框里没有别的。就两个数字和一个句号。

他知道她在说什么。

"嗯。"

"差得不远。"

"23分算是隔了很多人。"

"不算多。"她回了一条。然后第二条。

"在高考排名里。23分可能是一千人。在全省排名表上——六千名和七千名的差距。在走廊里可能差两层楼。"

"但楚州很大。"

这句话。他想了很久才发的。手指在屏幕上打了又删。删了又打。第一次打的是"没关系"。删了——太轻。第二次打的是"我不在乎"。删了——好像在逞强。第三次打了"但楚州很大"。读了三遍。确认没有说错——他想说的是。楚州够大，大到那23分不会把两个人隔开。大到12公里和301路可以把两个人连起来。大到四年足够把23分缩短到零。

"嗯。楚州很大。"

她重复了他的话。聊天框上方又出现了"正在输入"。

停了一下。消失。又出现。

"你考的比我好。语文比我高。"

"语文多少。"

"112。"

"我114。"

"比我高两分。"

"其它呢。"

"理综你高还是我高。"

他往前翻了翻自己刚才发给她的成绩。"197。你多少。"

"201。"

"你高。"

"你数学高。127。我118。"

"我跟你比物理化学生物。"

"物理你一定比我高。"

"不一定。"

"最后那道电磁感应，你写了一半。我全部放弃了。"

他愣了愣。"你记这么清楚。"

"我的选择题做得很快。大答题每一道都在那里放着。哪一道没做——我记得很清楚。就是电磁感应那道。"

两个人把成绩一项一项比完。像是在做最后一道物理题。只是这一次他们坐在一起了。

最后她说。

"589挺好的。楚州师院够了。"

"你的612也够楚州师大了。"

"嗯。"

又停了停。

"七月初填志愿。你会填楚州师院吗。"

"第一志愿。"

"我也是。楚州师大。第一志愿。"

"同一个城市。"

"301路。11站。"

她记住了。上次他说的时候她记住了。没有问他"301路哪11站"——只是记住了301路11站。

"到时候去看你。"

她回了一个字。

"好。"

然后她又发了一句。

"你来楚州师大。我带你看——图书馆。操场。食堂。还有——"

他没等到下一句。正在输入消失了一分钟。又出现。

"什么。"

"没什么。"

后来他又刷了几遍查分页面。系统已经流畅了。考生们都查完了。页面上的数字不动。589还在那里。和两小时前一样。没有因为他心里的波涛而有任何改变。

他把页面关了。

然后他打开浏览器重新搜索。百度——"楚州师院中文系"。

收藏夹里保存了这个页面。他以前查过。再看一次。

专业介绍。培养方案。师资力量。

然后搜索"楚州师大到楚州师院"。百度地图跳出来。蓝线条12公里。预计公交45分钟。301路11站。他把这个页面也收藏了。

躺在床上。天花板上的水渍还是那个形状。但明年这个时候——水渍不在这个天花板上。在楚州某个宿舍的天花板上。

他想的是这个。
