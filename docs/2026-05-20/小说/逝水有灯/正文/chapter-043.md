# 第043章 诗抄本之后

所属部卷：第一部：初秋三楼
时间：高二冬天
地点：教室后排

## 诗歌插页 P043：原创仿宋小令

灯影薄，纸声轻，
一册蓝封压旧情。
问少年、几番欲写，几番难停。
窗外冬风吹不尽，
课间仍要背方程。

诗抄本被班主任差点收走，是在一个周三下午。

那节自习课，邵老师从后门进来，大家立刻低头。陆允川正在诗抄本上改一句话，没来得及收，只能把物理卷子盖上去。卷子太薄，深蓝色封面露出一角。

邵老师走到他旁边，敲了敲桌子：“下面是什么？”

陆允川把卷子掀开，耳朵发热：“素材本。”

邵老师拿起来翻了两页。她没有读太久，只看见上面有诗，有日期，也有几道物理公式。教室里很安静，陆允川觉得每个人都在听自己的心跳。

“高二了，可以有爱好。”邵老师把本子放回去，“但先把该做的题做完。”

这句话不重，却让他比被骂更难受。

下课后，程砚把椅子转过来：“吓傻了？”

陆允川把诗抄本塞进书包：“差点完了。”

“完什么？老师又没念出来。”程砚说，“你怕的不是被收，是怕别人知道你写得这么认真。”

陆允川低头不语。

认真本身没有错，可当认真全压在一个不能说出口的名字上，就会显得不稳。像一摞没码好的书，看起来高，轻轻一碰就倒。

那天晚上，他没有继续写诗。他把诗抄本翻到空白页，列了一张复习表：物理两题，数学一页，英语听写。写完以后，他在最下面补了一行小字：诗放到题后面。

这行字没有坚持太久，却是一个开始。至少那天，他先做完了两道物理题，才把深蓝色本子重新打开。
