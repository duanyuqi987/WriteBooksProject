# 第105章 树莓摊旁边



时间：2010年五月中旬，补课班下课后



地点：补课班楼下、树莓摊旁边



卷次：第五卷：未来在黑板上倒计时



2010年五月中旬，补课班下课后，补课班楼下、树莓摊旁边，付小川手里一张揉皱的补课收据把2010年的春夏推到十班面前。补课班楼道挤满学生，付小川拿着收据不敢回家，因为数学补课费又涨了。

他嘴上说自己以后一定考个贵的分，转身却问段璟值不值得继续补。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

付小川也在这段日子里显出自己的位置。贫嘴、讲义气，成绩忽上忽下，越害怕越会把话说得很轻。群像不是名单，群像是每个人都有自己的难处。

“这张收据比我数学卷子还刺激。”付小川说。

“你真不想补了？”段璟说。

“想补，又怕补了还这样。”付小川说。

“先看你上课有没有真听。”常俊凡说。

“补课不是买分，别把钱花成安慰。”马立群说。

补课班楼道窄，墙上贴着各科提分班的广告，纸边被学生书包蹭得发毛。

下课铃一响，楼道里全是汗味、墨水味和家长在楼下喊名字的声音。

树莓摊就在楼下拐角，甜味被热气一蒸，和烤红薯、尘土混在一起。

补课费、练习册、饭卡和公交钱，平时散在不同地方，这一刻像都堵在楼梯口。

常俊凡走得不快，手里抱着一摞题。她不是不累，只是很少让累先开口。

段璟在楼道里看见许多熟面孔，才明白大家所谓靠自己，其实都背着家里的钱和期待。

老师从楼梯口下来，提醒学生别挤。家长们往上看，像能从每个孩子脸上看出分数。

楼下树莓摊慢慢收摊时，补课班的灯还亮着，像县城夏夜里一块很硬的光。

马立群这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

付小川被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

邓桥抱着练习册挤下补课班楼道，水房方向传来暖水瓶碰撞声。十班的热闹还在，只是每个人都比去年更会藏心事。

余小满把补课收据折进书里，补课班楼道里有人喊让一让。老师从门口经过，原本散开的声音立刻往回收。

曹蔚在楼梯口等同学一起走，窗台上的金银花茶凉了一半。这样的细节不会被写进红榜，却会留在很多人的夏天里。

周明澈抬头看楼下树莓摊的灯，讲台粉笔盒里只剩半截白粉笔。这点动作很小，却把那天的压力落到了实处。

付小川抱着练习册挤下补课班楼道，后墙的错题纸被风掀起一角。旁边的人没有追问，翻书声把难堪遮住。

何嘉树把补课收据折进书里，竞赛名单的纸角已经微微卷起。常俊凡看见了，没有替谁下结论，只把题号重新写清楚。

宋乔在楼梯口等同学一起走，新华书店塑料袋在手里沙沙响。段璟后来才知道，未来最早就是这样从小事里露头的。

树莓摊旁边之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

树莓摊旁边的甜味压不住补课费的现实，大家第一次把学习和钱明明白白连在一起。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，付小川手里一张揉皱的补课收据还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
