# 第023章 操场边的单词之后

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P023：原创仿宋小令

夕落操场，
一页被风分。
欲把单词藏半句，
却因跑道人留痕。
看台旧日温。

操场边的单词之后，陆允川表面上没有变化。每天晚自习前还是去操场上背单词，还是坐在西看台第三层台阶最边上那个位置。但他在心里给那个位置做了标记——就是台阶上有一道裂缝刚好从右侧拐过去的那块水泥。她的身影曾经坐在那道裂缝的下方，只隔一层台阶。

英语小测成绩下来了，陆允川考了七十六分——不算好也不算差，阅读理解和完形填空都在中游。程砚考了八十九，翻着卷子在他旁边坐下来："你哪里错了？""阅读理解第三篇错了三道。""第三篇是考古类，生词多。你就别气馁了——反正你也没认真做吧。"陆允川没反驳，因为程砚说的是实话，他做那篇时确实在走神。

"我以后能不能不在操场上背单词？"陆允川课间突然问程砚。
"为什么？"
"操场上人少的时候容易发呆。"
"那你准备去哪里背？教室里又吵，宿舍里你不能带书进去。"
"走廊尽头。"
程砚看了他一眼："走廊尽头没风景，你能背进去最好。不过——你自己想去的操场，背不进去就不要去了。我又不押着你。"

陆允川没有改地方。第二天晚自习前他还是去了操场。他带了单词书——第八页，生词十个，例句五句。但他坐下以后，先看了三分钟跑道。跑道上没有人。连散步的都没有。操场空得像一口倒置的碗，晚风吹过来把红塑胶的味道往看台上推。他发现自己开始计算她出现的概率：如果是周一，她大概率会来——周一物理作业最重，需要走几圈；如果是周三，不太一定——她周三下午有选修课的出勤任务；今天是周三，概率大概百分之三十。

他在单词书第八页的角上把这个概率用铅笔写了下来：30%。写完之后马上擦掉了，因为他忽然意识到自己在做什么——用统计学的工具分析小概率心件事，就像用化学实验的吸量管喝汤。

夜里回宿舍，程砚问他："今天在操场背了几个？""十个。""真背了？""真背了。还记住了。""那你背一个给我听听——今天背的第九个。"陆允川想了一下："permanent——永久的。""例句？""Nothing is permanent.""嗯。用到自己身上来。"陆允川关他手电筒，室友们讨论明天的化学实验分组，陆允川听着听着忽然插了一句："明天分组是按座位分吗？""按学号。你想和谁在一组？"上铺的人问。"没人。随便问问。""随便问我可不信——你刚才问的时候声音往上翘了。"陆允川用枕头挡了一下脸，说"你们真能分析"。室友们笑得轻，各自翻了个身。

第二天下午，化学实验果然按学号分了组。陆允川的学号是L-23，分到了和另一个男生一组，在第三实验台。夏知遥的学号是X-31，分到了第五实验台，和他隔了两个桌子。陆允川站在自己的实验台前配硫酸铜溶液，手很稳——烧杯、量筒、玻璃棒，每个步骤都按标准操作。他做完自己的实验以后，余光扫到第五台那边——她的烧杯被玻璃棒碰翻了，硫酸铜蓝色溶液洒到桌面，实验老师走去帮她处理。她说了声"没关系我自己来"，然后拿抹布把溶液吸干净，重新配。陆允川收回目光，把自己配好的硫酸铜溶液倒进指定收集瓶，标签写了自己的学号和姓名。他写完之后发现她的名字还没出现在收集瓶架子上——她那一组的实验还没做完。他在架子前面站了大概两秒，然后转身回座位。

放学后他经过化学实验室，从门口往里看了一眼——里面的瓶子已经收走了，架子空了，窗户也被关了。他不知道她的实验结果最后怎样，也不知道后来有没有人替她把抹布洗干净挂回去。这些细节他永远也不会知道——因为他不敢问，她也不会主动说。在两个人的轨迹之间有一块无人看管的空白地带，而他所有无法安置的问题都堆在那个地带里。
