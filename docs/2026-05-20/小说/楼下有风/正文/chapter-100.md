# 第100章 没有寄出的相片之前



没有寄出的相片之前，是一次很普通的手机清理。系统提示存储空间不足，许亦安坐在出租屋床边，把旧相册从头往后翻。

出租屋里风扇转得很慢，床边纸箱还没拆完。清理照片原本只是为了腾空间，却把几年压缩成一排缩略图。

旧手机相册加载得很慢，缩略图一格一格亮起来。

他看见那张照片时，没有立刻难过。先出现的是网吧风扇的灰、鼠标垫翻起的蓝边和林青禾弯腰认照片时的头发。

他想把照片发给林青禾，附上一句你还记得吗。字打出来以后，像一个不合时宜的敲门声。

他起身去倒水，回来时手机屏幕已经暗了。那一小段离开救了他，让他没有把一张旧照片变成新的打扰。

后来他写到她时，常常故意少写几句。少写不是冷淡，而是尊重：有些人生长在他看不见的地方，不该被他用想象占满。

从她那里看，许亦安的等待既让人感激，也让人害怕。

后来他想，真正难忘的从来不是某一句话，而是那句话发生时周围很小的东西。

周一鸣在电话里问他是不是还会想起她，他说会，但少了。他说得很轻，像怕一句重话把朋友推得更远。这话许亦安当时不爱听。他更愿意相信自己看见了别人看不见的东西：她笑时眼睛弯起来的弧度，她把吸管咬扁的习惯，她说“随便你”时其实并不是真的随便。喜欢一个人后，人的听力会变奇怪，能听见很远的暗示，却听不见近处的提醒。

同事问他手机怎么老提示内存不足，他笑说以前乱拍太多。乱拍两个字很轻，把一段复杂心事放回普通技术问题里。

电脑导入窗口停在百分之九十九，像故意让人多等一秒。

他清理旧手机相册，删掉截图、糊照和重复的风景。那张照片留下来，不因为她，而因为那里面还有一群年轻人和一个旧网吧的光。

他后来才懂，喜欢一个人不该让自己变得越来越不像自己。

那晚他把相片导进电脑，文件夹名字仍叫照片。没有青春，没有林青禾，没有任何自我感动的标题。

那张旧照片不再被他拿来确认什么。它属于相册，属于旧网吧的蓝光，也属于一群彼时还不知道散场将至的人。

出租屋的风扇吹着纸箱边缘，纸板轻轻翘起。

他留下照片，却没有寄出照片。保存和打扰之间终于分清，旧相册也因此安静下来。

他没有把照片发出去。不是怕尴尬，而是终于承认，想让别人一起回忆，有时只是自己还舍不得独自收好。

他从发送框里删掉那句你还记得吗。删完以后，他去给手机充电，让旧照片留在相册，不再去敲别人的门。

第二天上班，他在地铁上继续删重复照片。删到那张时，他停了停，最后只点了收藏，没有点发送。

楼下早餐店的豆浆装在纸杯里，没有塑料袋，也没有谁故意让他吃辣。他站在队伍里等了一会儿，忽然笑了笑。原来有些味道不是被忘记，只是终于从某个人身上退回到食物本身。

上班路上，他看见地铁广告牌换了一轮又一轮，像许多来不及说出口的话被新的生活覆盖。可覆盖不是抹掉，底下的痕迹还在，只是不再要求被所有人看见。

同学群里偶尔有人发旧照，大家笑几句就散。许亦安也学会只笑几句，不再从每张照片里寻找旧关系的余温。

电脑导入完成时，进度条消失得很快。许亦安看着空出来的手机容量，忽然觉得有些空间确实需要腾出来。

这段回望不再写成深夜长叹。它就是一次手机清理，一张没寄出的照片，和一个终于按下取消的人。

回去的路上，他看见写字楼门口换了新的海报。旧海报被卷起来放在墙角，露出背面泛白的纸边。许亦安看着它，忽然觉得记忆也可以这样被收起，不必撕碎，也不必展示。

他没有把这些想法告诉林青禾。那时候他还以为，真正深的感情应该自己消化，不能给对方添麻烦。后来他才知道，不能说的感情很容易在心里变形，变成委屈，变成审问，变成某个夜晚突然冒出来的重话。

他没有把这些段落发给任何人看。保存不再是证明，而是整理，像把旧衣服洗净叠好，放进不常打开的柜子。

那一行字写着：没有发送。它短得像操作记录，却比很多抒情更稳。

修订时他发现，很多当年以为惊天动地的瞬间，其实只需要一个具体物件就够了。物件安静，安静才耐看。

写到这里，他删掉了一句很漂亮却太满的话。漂亮的句子若抢走了人的真实，就应该让开，让普通的动作自己说话。

他没有删除过去，也没有把过去重新送到林青禾面前。相片留在文件夹里，像一盏灯关小了，但没有熄灭。

这一章真正留下来的，不是他当时以为最重要的那句话，而是电脑屏幕合上后，屋里只剩空调很轻的风声。那画面还没有成为记忆，只是很安静地停在那里。

这一页日记后来被他保存了很多年，保存到里面的怨气都慢慢变轻。许亦安合上电脑，听见窗外有人笑着从楼下经过。那笑声很快远了，房间重新安静下来。他坐在书桌前，觉得这段旧事好像没有结束，又好像早就结束了。
