# 第057章 最后一次元旦

时间：2012年12月31日

地点：教室

卷次：卷三：此心藏久

高三的元旦没有礼堂。

没有舞台。没有红灯笼。没有化妆。没有西装和白裙子。没有合唱《明天会更好》。

只有一间教室。五楼。黑板上用彩色粉笔写着的"元旦快乐"四个字。没有花边。没有插画。粉笔字写得不太好看——文体委员写的。写完之后拍了拍手上的粉笔灰。"就这样了——大家将就看——"没人应。

下午最后一节课结束。老刘进来说了两句话。

"今天晚上元旦晚会。班里自己组织。可以晚一点回去。但不能太晚——明天还要上课。"然后走了。

大家把课桌推到两边。中间空了出来。有人拿了一个小音箱——还是去年那个，插上了手机放歌。第一首是《我的歌声里》。有人喊"换周杰伦的——"。切了一首《稻香》。然后是《安静》。然后是《简单爱》。没人管音响了，就让它随机播。

桌上摆着橘子和瓜子。没有苹果——去年的苹果没人吃，今年生活委员就没买。橘子是从学校门口的水果摊买来的。三斤十块钱。有几个橘子皮有点青——还没完全熟。有人掰开了，吃了一瓣，酸得直皱眉。

顾临风坐在教室后面。他坐的是一张推到边上的桌子。腿悬在空中。手里拿了一张英语单词卡——是来之前背的。现在已经没在背了。单词卡放在膝盖上，正面朝下。

赵一鸣在旁边吃瓜子。他吃瓜子的技术不好——嗑不开。吐出来的瓜子壳上还粘着瓜子肉。他把瓜子壳吐在地上——反正明天值日生要扫。

"你今天怎么不去角落里坐着了。"

"坐这儿也一样。"

赵一鸣看了他一眼。没有戳破。角落里——以前不管什么活动，顾临风总是坐在最不显眼的地方。别人闹别人的，他看着别人闹。今天他坐在了中间靠后排的位置。不是正中间。但也不是角落。

人到齐了。七点多。外面的天已经黑了。教室里的日光灯太亮，有人提议把灯关了，只开着讲台上那盏小台灯。于是教室里暗下来了，只剩讲台上那一小片光。

有人唱歌。文体委员先唱了一首《老男孩》。跑调了。跑到后来他自己笑了出来。有人接过去——另一个男生——唱得比他好一些。然后是女生——《遇见》。孙燕姿的。唱着唱着有人跟着哼。开始是两三个人，然后是七八个。最后半个班都在哼。"我遇见谁会有怎样的对白——我等的人他在多远的未来——"

苏晚晴坐在窗边——不是窗边，现在是教室中间靠窗的位置。窗玻璃上贴了一层报纸——冬天挡风用的。她的背靠着窗台。手里拿了一个橘子。没有剥。只是拿着。

许若琳在旁边戳了戳她的胳膊。"你唱一个。"

"唱什么。"

"随便。你去年合唱挺好的。"

苏晚晴想了想。然后说。"送别。"

许若琳帮她喊了一声。"苏晚晴唱《送别》——大家安静一下。"

教室静了下来。

她站起来。走到中间的圈子里。脚下的木地板很老。有些地方油漆磨没了露出了原木色。她站在那儿。教室里很暗。只有讲台上那盏小台灯的光从侧面打过来。照在她的左脸上。右边脸是暗的。她站在明暗交界的那条线上。

她深吸了一口气。然后开口。

"长亭外——古道边——芳草碧连天——"

她的声音比以前安静了。

高一的合唱——他在台下看她。她的声音在三十多个人的合唱里听不出来。但她的嘴在动。眼睛很亮。今天这间教室里只有她的声音。一个人。没有伴奏。没有麦克风。她的声音像流水——不快。不响。但很清。

"晚风拂柳笛声残——夕阳山外山——"

唱到"夕阳山外山"的时候，她的声音落了一点。她的眼角动了一下。不是哭。是某种比哭更安静的东西——被歌词触到了什么。只是一瞬。她继续往下唱。声音和刚才一样。稳住了。

"天之涯——地之角——知交半零落——"

"一壶浊酒尽余欢——今宵别梦寒——"

唱完了。教室里安静了一会儿。然后有人鼓掌——开始是一两个，然后所有人。

苏晚晴弯了一下嘴角。坐回了她的位置。拿起刚才那个橘子。剥开。白色的橘络被她一根一根撕掉。放了一瓣在嘴里。挺甜的——她嘴角弯了一下。不是刚才那种礼貌的弯。是真的——甜。

顾临风没有鼓掌。他把单词卡拿起来。翻到正面。又翻到反面。再翻到正面。然后把卡片放进了口袋里。

这是他高中最后一个元旦了。他知道。他听着《送别》的那些词——长亭外、古道边、芳草碧连天——脑子里想的是另外一些词。五楼。水塔。倒计时。高考。她唱的那些离别。还有半年就会变成真的。高中三年就像这首歌唱的那样——今宵别梦寒。

后面又有人唱了几首歌。然后有人提议玩游戏。击鼓传花——没有鼓，用手机闹铃代替。闹铃响了三分钟。橘子当"花"。橘子最后传到了一个男生手里。他站起来说了个笑话。大家笑得很不统一——有人笑了，有人在趁机聊天。笑完以后换了一个闹铃。继续传。"花"这次传到了苏晚晴。

她站起来。手里捧着那个被人摸过好几轮的橘子。橘皮都有点皱了。她站了几秒。没有说笑话。没有唱歌。说了几个字。

"希望大家。都能去想去的地方。"

然后坐下。

教室里又安静了一下。然后有人鼓掌。

"去想去的地方"——这几个字可以翻译成很多意思。想考上的大学。想去的城市。想在一起的人。她没有给任何人解释。但每个人都可以往里面加自己想要的。

顾临风把单词卡从口袋里掏出来。在卡片的背面写了一行字——2012年12月31日晚。苏晚晴唱《送别》。写完后把卡片放回口袋。

赵一鸣凑过来。"你在写什么。"

"单词。"

"单词你需要在元旦晚会上背。"

"明天要听写。"

赵一鸣看了他两秒。忽然笑了一下。

"好。背单词。你背你的单词。"

顾临风没有辩驳。他把单词卡往口袋里塞深了一点。

快十点的时候。老刘回来了。看到教室中间还有人围着，橙子皮和瓜子壳撒了一地。没说什么。只是让大家把灯打开，桌子搬回去。"差不多了。明天还要上课。赶紧回去睡觉。"

大家开始收拾。桌椅在地上拉得吱吱响。有人把没吃完的橘子揣进兜里。音响被拔了。手机被收回了口袋。黑板上的"元旦快乐"被值日生擦掉了一半——圆圌圙圐的"快乐"两个字变成了"快"加一个白印子。

顾临风帮忙搬了一张桌子。搬完之后站到了门边。让同学们先出去。他是最后一个离开教室的。

关灯。关门。在黑暗里站了大概三秒。

走廊里的脚步声渐渐往楼梯口的方向远去了。他靠在自己座位旁边的墙上。月光从对着水塔的窗户照进来。在木地板上铺了一个方形。方形里是窗框的十字阴影。

他想起去年元旦——她在台下，他在更远的地方看她的侧脸。她在合唱——三十多个人一起唱。她化了淡妆。眼睛很亮。他在心里下了一个决心——去年的事。

一年了。

那个决心还在。

高考后。告诉她。

他推开门，走出走廊。关上门的一刻——听到里面有一阵风从窗户灌进来。桌子上的报纸被吹动了一下。然后安静了。
