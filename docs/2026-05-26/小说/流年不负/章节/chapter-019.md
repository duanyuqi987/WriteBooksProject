# 第019章 分班

时间：2011年2月下旬

地点：青阳县一中

卷次：卷一：初见秋风

寒假过完了。石桥镇街上的炮仗屑被扫到路边堆成一堆——红的碎纸裹着泥水。一辆货车开过去——泥水溅上去——红纸变成了泥红色——车轮碾进碎纸堆——纸浆吞进轮胎纹。镇上的早点铺又重新开了门——炸油条的锅里——铁锅底下的柴火烧得旺旺的——油泡在锅里翻滚——老板拿长筷子翻面——油条在油面膨胀——金棕色。他买了两根油条——路上一边吃一边走——油条的热气和冬天早晨的白雾混着——嘴里嚼着油香。

顾临风坐了二十分钟班车回学校。车上人不多——年后——不是所有人都非要赶今天——有些人还想在家多待一天。他靠在最后一排靠窗的位置——头抵着玻璃窗——窗外还是冬天——田里光秃秃的——土是褐色——有些地方盖着残霜——像撒了层白盐。田埂上有人骑摩托车——车后座上趴着一只山羊——羊腿捆着绳子——羊的眼睛半闭——嘴在嚼什么——车过的时候按了喇叭——羊受惊——挣了一下——绳子勒紧了腿。摩托车开远了。

二月底。高一下学期开学。

头一天没上课。打扫卫生、交作业本、领新书、开班会。教室里——寒假堆了将近一个月没人擦的黑板——上面期末考试前的公式还没全擦掉——粉笔灰凝了一层——白雾——字迹隐约。有人在擦——湿抹布上去——干字迹顽固——擦几遍才透出绿色底。日光灯开不全——有一根灯管坏了——一直闪——叫了电工来换。电工扛着梯子——换了——新灯管更白——教室突然亮了一块。窗玻璃上寒假积的灰——在外面——风吹不掉的——擦窗的人在玻璃上哈一口气——看里面自己的脸——然后擦掉——玻璃清楚——脸从模糊变清——头发乱。讲台的粉笔盒空了——补了两盒新粉笔——白色和彩色各一盒。

新发的课本比上学期多——多了一本生物。每一科的下册——物理、化学——尺寸和上学期一样——封面色系也是蓝白——但"下册"两个字是偏右上。顾临风从课代表手里接书。书是新书的味道——油墨味——翻开——纸是白的——印刷的铅字边缘锐利——有点反光——第一单元曲线运动——平抛物体——圆周——匀速圆周——向心力F=mv²/r。往后翻——匀速圆周运动那节配了一张图——小方块在圆周上——绿色的箭头表示速度——红色表示向心力。

新书往书包里塞——书包塞得鼓鼓囊囊——拉链拉不拢——拔了几下——用拇指把拉链齿按在一起——硬拉上来——书包的拉链头是光秃秃的金属片——漆掉了一片——银白色的底。书包上没有挂件——物理拉链发出的声音是金属磨金属——嗤。和陶瓷的——不一样。

周三上午。早读铃刚过。班主任在讲台上站了站——拿了粉笔——在黑板上写了四个大字——文理分科。写得很慢——楷体——每个字有拳头大。底下起初还有小声说话——"文理——""分科了吗——"然后都停了。

他在讲台上停了一下——转身——往前跨了一步。"这个决定很重要。"语速不快——声调是平常的。"你们回去跟家长商量——不要只凭感觉——想清楚自己的方向。周五之前——分科表交上来。"

然后把一张表格用磁铁贴在黑板上——A4纸——油印的——纸面淡白。第一行"姓名"——第二行"选择"——理科/文科——中间用一条黑竖线隔开。表格最后有七八个空白行——待填。

然后他走了。门关上的时候锁舌卡嗒——停了片刻——第一节自习课没人看书。

讨论声涌出来了。整个教室从安静变成闹——嗡嗡——好像前两分钟没说过话的人都被按了开关。前排有人在问隔壁——"你选什么——""理——文科背书太多——"后排有人在争——"理科将来工作好找——""你物理能考几分——""化学更难——配平永远配不平——""政治那些题不是只要背就行——""但我英语还可以——文科占优势——"

赵一鸣用笔戳了一下顾临风的胳膊。戳的力道不大——笔头是圆珠笔的钝端。

"你选什么。"

"还没想。"

"骗人。你物理考那么好。"

"你呢。"

"我不知道。"赵一鸣把笔横放在鼻子下面——上嘴皮夹着——像胡子一样——笔滑下来了——他捡起来——在桌边上敲了两下——咚。"我文理都差不多——跟我爹水平差不多。"

"你爸什么水平？"

"我爸开出租的。数学——算账速度还行——零头从来不算错——但是代数他不懂。语文——骂堵车挺能说——出口成章。"他自己笑了——推开化学书——"你呢——你选什么——说实话——"他看着顾临风——眼神是那种已经猜到答案的——嘴却还在问。

顾临风的手在课本上顿了一下。"不知道。"

"苏晚晴选什么——你知道吗。"

"不知道。"

"你不去问问？"

"跟我有什么关系。"

赵一鸣笑了。那是一种"知道了"的笑——嘴角的上边缘轻轻往上——不大的表情——只有他们俩之间能看见。他没有再往下说。翻开新发的生物书——第一章——生命的物质基础——细胞——他看了三四行——对细胞膜不感兴趣——合上。

中午吃饭。食堂的人比上学期多——新入学的高一第二波——文理分科预备班。面孔很陌生——穿着新校服——有些校裤长了——拖在地上——裤腿边磨黑了——都是水泥地磨的——家长还没来得及往上缝。打饭窗口排的队从尽头弯到门口——三号窗口的阿姨今天换了一个——短头发——以前那个她——去年打菜抖勺子的——说是不干了。新阿姨勺不太抖——但土豆丝抖了一下——不是因为想抖——是因为手腕没力——勺子在手里打了个滑。

顾临风打了西红柿炒蛋和米饭。端着饭盘找位子的时候——他看到她了。苏晚晴。她也排在队伍中间——手里端了个空盘——头微微低着——看地面——在看那些人站的地方——大概在想今天有没有包子。头发——和早上扎的不太一样——早上扎的到了中午皮筋有点松——马尾形状往下坠了一些——头发根有点散开了。她还是那样——不跟谁说话——排在人堆里——旁边有人在讲什么——"文科好""理科工资高"——她目光只看着前面的人。

墨绿色的棉服左边口袋里鼓起来——钥匙串吧——口袋侧边有一道不太明显的线——大概口袋上补过。

顾临风端着盘子——从她身后经过——没抬眼——走到三排后面找了个靠墙的位子坐下。扒了两口饭——西红柿炒蛋糖放少了——酸。又吃了口米饭——饭粒还跟以前一样硬——不知道这次是隔夜还是隔了三天前——嚼的时候明显感觉米粒中心是粉的。

旁边。短头发女生坐下——跟另一个扎马尾的女生说话——"你选什么科——""文科——数学不行——""我也是——化学太难了——"声音远了——那俩人端盘子回教室去了。

吃完饭。把盘子放回收台。铁盘磕在铁架上——当——盘子在水龙头下冲两下——水是冷的——残渣冲进水池。擦了嘴。

接下来两天。他在等一个消息。不知道在等什么——是等谁开口说。他坐在倒数第三排座位——上课——老师在讲新课本第一课——曲线运动——"平抛——斜抛——圆周——"他在物理笔记本里画图——手指没停——但耳朵比平时多挂在后排、旁边——这些不说话的地方。很多人谈论苏晚晴——她的成绩————她的分科。——学生之间——"苏晚晴理科——理科——分科——"

周四上午——课间操回来。赵一鸣从走廊过道——小跑进来——差点撞到前排的女生——"对不住——"然后一下子坐到椅子上——肩还在轻微喘。他把头往顾临风那边一偏——小音量——典型的赵一鸣——音量刚好过线——"我刚听到——苏晚晴——选了理科。"

"确定？"

"第一批交上去的——她名字在上面——我刚去办公室——班主任和王老师在聊——名表放在桌上——我看到。"他把手上跑步跑的汗在裤子上蹭。"她就交的理科。"

消息还在一个很小的圈子里。名单没全——到周五还有人犹豫。顾临风没说话——把刚才数学课上的那道三角函数题写完一小题——然后那支笔——停在空中——在纸上顿了一个很深的蓝点。

下午。分科表发到每个人手里——薄薄一张纸——大小比A4纸半张——白纸——上面两条横线——文科/理科。二选一——打钩——签名——交。他没动——把纸放在桌上——纸很轻——风窗缝进来的风——吹得纸张翘了一下——他用手按住——五指在纸面上。两条灰色线。文科。理科。二选一。他拿起笔——取下笔帽——在"理科"右边——画了一个钩子。钩子画得很快——向右甩尾——蓝墨水——尾端拉出去比预想长了一点——笔尖拖出来——钩尾有一道纤细的墨迹渐淡直到消失——断了。把表放在桌角。

赵一鸣回头——"你也选理。"

"嗯。"

赵一鸣点了点头——在自己的纸上也打了个钩——理科。然后转回来——眼神往左侧瞟了瞟——"你知道吗——整个班就咱俩选理。"

"放屁。"

"真的。"赵一鸣用手指掰着算——他一排一排数——前几排的人——后排——手指从左边点过右边。自己笑——"不对——是咱仨——苏晚晴——也选理。"他把表拿到面前——吹了口气——"行。理就理——一起死。"

顾临风没说话。笔在草稿纸上画了一条横线——从纸的左边到右边——平的——稳稳的。然后把草稿纸翻过去。

他是最后一个交表的。别人都去课间跑操的时候——他拿着表——从教室后门出去——往办公室走。教学楼楼道里安静——高二的人在楼上上课——只有他一个人——脚步声——从三楼到二楼——推开办公室门——门上玻璃震了一下。班主任在办公室里看报纸——《中国青年报》——报头黑字。他在办公桌前站了一下。"王老师——交表——班主任抬头——眼镜从鼻梁上滑下来一点——看了他一眼——从手上接过表——"顾临风——选理科——""嗯。"他应了声。班主任把表反过来——看了一眼分数——"你理科成绩确实不错——好好学——"然后把表压在一摞已经交上来的表格最上面。

顾临风低头。表摞——最上面的是赵一鸣——"赵一鸣——理科"——笔迹潦草但用力——赵一鸣写字——胖子风格——圆——笔道粗。底下露出一角——"苏"字——第一笔横——是她的写字习惯——横起笔轻轻提个锋——下面——"晚晴"两个字被上面那页盖了半边。但能看到"理科"——钩子画了很小——和他画的钩不一样——她的钩比较短——弧线圆——尾端从瘦到无——像收笔。他看了一眼——转身走了。

周五下午。分科结果报上去了。物理李老师来通知的——手上拿了张总表——"理三班的教室在三楼东边——到时候全年级理科班的同学打散重新编班——各班人数大约四十——明天上午班牌就换了——周一来新教室——"

高一三班选理科的——三十二人。选文科的——十人。那十个选文的——有几个是语文课代表——短头发——以前老是叫顾临风抄黑板上的短语——"快抄""知道"——现在——这个班散了。

周六。宿舍楼比平时安静——走读生回家了——他们班的人搬走了一批。走廊里全是箱子和包袱——孙磊的泡面箱从床底下拖出来——搬到推车上——"走了——走了——下学期理六班——"他推推眼镜——别人在门口跟他说了句"六班见——"他点个头。箱子滚轮在水泥地上——咕咕。有人把被褥捆成一大包——肩一扛——下楼——碰着墙——不管。赵一鸣回家了——"下周理三见——"他背包装满薯片棒棒糖——然后又加一句——"帮我留着座——靠你旁边——"人走了。

顾临风一个人坐在床沿。宿舍里——上铺刘洋的床铺空了——他选文——床板上的报纸杂志也清走了——打扫干净。下铺赵一鸣的被子还在——人不在了——回去过周末。他从床底拖出自己的旧纸箱——物理笔记本——翻开——物理笔记本的最后一页——那道题解——折痕又平——笔迹轻微磨损。翻了几页——放回去——推进床底。窗外的煤渣跑道被雪水浸黑了——冬天的雪渗进去之后——煤渣颜色变深——踩上去是软的——水还没干——跑道上有几个脚印——早上有人跑步——大概是田径队的——寒假一直在训练——他不在的时候——跑道被人跑着。三楼的走廊上能看到县城房顶——灰色、红色——有一条巷子里的矮房冒着烟——有人烧煤炉——烟在冷空气里打了好几个弯——然后被风拉散——看不见了。

他把新课本——理三的新教材——码在床头。一本挨一本——物理、化学、数学、英语——书脊朝外——排齐。用手从一侧捋——书脊高——低——物理书的厚度比化学多一些。手指在书脊上弹了一下——化学书比物理轻——发出的是不同的响声。

周日下午。学校里重新热闹起来了。箱子轮子的声音——滑——接着轰——有人行李掉了。家长的喊声——"小心点——别踩到你弟弟的被子——"新生往宿舍楼里扛箱子——高一个年级的脸在找自己的新教室——都是理科班的。顾临风没有去公告里挤——明天早上早一点去——人少。他今天不去看名单——今晚先睡觉。窗帘拉上了——窗外的路灯透过旧布——变成模糊的橙色光斑——和上学期那个学期的每个周日晚一样——一样的灯——一样的窗帘——但明天开门——走廊尽头——往东数第三间——门口贴着白底黑字名单——上面写着名字。

他把眼睛闭上。理三班——赵一鸣——苏晚晴——赵一鸣——苏晚晴——数着——然后不想——呼吸慢慢匀了。外面又开始有人搬床板——脚步声——空心铁管碰在走廊——"咣——咣——咣咣——"——铛。然后又安静了。灰色水泥地——窗外的灯照着——今晚没有星。
