# 记忆快照

《青春没有标准答案》是一部发生在2008-2011年封丘县、封丘一中、十班的校园青春爱情群像小说。段璟和常俊凡从九年级认识，到中考报考封丘一中，再到高中十班前后桌、雪仗、分科、高考、志愿和大学初期现实相守。全书坚持第三人称双线和班级群像，不把常俊凡写成男主成长工具。

## 已固定 canon

- 男主：段璟。
- 女主：常俊凡，虚构人物。
- 学校：封丘一中。
- 班级：十班。
- 地点：封丘县，县城生活和黄河故道、金银花、树莓等地方细节可进入正文。
- 必写情节：九年级相识、中考报考、十班重逢、两栋楼打雪仗、现实相守。

## 001-020 精修后连续性

- 001-003：段璟在初三模拟考、借物理资料、红榜名字中真正注意到常俊凡；两人的关系从普通同学的工具性接触开始。
- 004：汶川地震后的默哀和捐款进入班级日常，让2008年的大事件通过操场、广播、捐款箱落地。
- 005-010：报考咨询、宣传单、黄河大堤、中考、志愿表共同完成“封丘一中第一志愿”的选择。
- 011-020：停电晚自习、作文重写、宿舍夜谈、成绩等待、录取通知书和县城车站分别，把中考后的空档写成具体生活，而不是回忆解释。

## 021-045 精修后连续性

- 021-023：封丘一中十班名单、临时座位、奥运余温进入教室，段璟和常俊凡从重逢变成前后桌。
- 024-030：黄河大堤、早读、停电晚自习、作文重写、宿舍夜谈和第一张正式座位表，把十班从名单压成真实群像。
- 031-038：月考前后，停电、作文、宿舍、班会旧新闻持续强化人物主体性；常俊凡也会害怕退步，不再只是稳定符号。
- 039-045：期中压力、早读整顿、宿舍排名表、冷空气和校服袖口，收束第二卷开篇质感，准备进入冬雪和雪仗段。

## 046-070 精修后连续性

- 046-054：冷空气、月考排名、家长会、纸条误会、早操背词、公交站和座位调整，把高一冬天的压力压实。
- 055-060：高二楼先扔雪球，高一楼反击，十班被点名，王启民给出通报批评、写说明和扫雪后果；雪仗从热闹变成群像高光。
- 061-070：雪后公交、期末模拟、宿舍电话、纸条边界、操场单词、准考证袋、估分和寒假成绩单，把第三卷收束到现实后果。

## 071-095 精修后连续性

- 071-075：高一下学期文理倾向摸底、未发短信、错题本、水房和小卖部，把分科压力从表格落到具体关系。
- 076-083：2009年秋高二开学，甲流晨检、理科月考、朋友分流和黑板报准备共同推进，段璟学会不把常俊凡的选择和自己绑定。
- 084-095：国庆六十周年黑板报、甲流加严、期中、家长会、分科确认和最后一晚，十班从完整群像走向分流后的新秩序。

## 096-120 精修后连续性

- 096-101：2010年春，上海世博新闻、饭卡余额、错题墙、世博黑板报、树莓摊和玉树捐款箱，把远方大事和县城日常并置。
- 102-109：饭卡、座位、竞赛名单、补课费、同学录第一页和黑板报边角，让偏科、补课和家长期待具体落地。
- 110-120：补课班楼道、新华书店练习册、家长饭桌、暑假补课倒计时和第一次认真谈未来，把喜欢推进到现实相守的前置问题。
