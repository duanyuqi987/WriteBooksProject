# 第156章 短暂和好之前

所属部卷：第三部：寒风里的花
时间：七月中旬
地点：诗社活动室

## 诗歌插页 P156：原创现代诗

旧书堆在窗下，
灰尘轻得像暂时的和平。
他把手伸向同一本书，
又及时停住，
让一本书先属于桌面。

期末结束后，诗社活动室要清理旧书。叶澄在群里发了名单，谁有空谁去。陆允川本来想避开苏晚枝，到了活动室门口才发现她也在。

窗户开着，风把窗帘吹到书架边。地上堆着几箱旧刊和旧书，封面发黄，扉页上盖着借阅章。苏晚枝蹲在地上，正把破损严重的书单独分出来。

“你来了。”她抬头说。

语气平常，没有躲，也没有热。陆允川点头，把书包放到角落。

叶澄分配任务：“苏晚枝查目录，陆允川贴编号，我去教务处借封箱胶。”

叶澄一走，活动室里只剩纸页声。陆允川拿起标签纸，手心有点汗。他想说点缓和的话，又怕任何缓和都像试探。

苏晚枝把一本旧诗集递过来，“这本编号掉了，你看看是不是 2009 年那批。”

“好。”

他接书时刻意没有碰到她的手。这个动作很小，苏晚枝却看见了。她低头继续查目录，肩膀比之前放松了一点。
