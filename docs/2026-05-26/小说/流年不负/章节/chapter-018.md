# 第018章 除夕

时间：2011年2月

地点：顾临风家中（石桥镇）

卷次：卷一：初见秋风

除夕那天从中午开始有人在放炮仗。小孩扔的摔炮，在地上响一声就没了。还有冲天炮——"啾——"一声上天，然后"砰"一声响了，落下来的是灰。

他妈在厨房里忙了一整天。他爸在堂屋里洗桌子、摆碗筷、贴窗花——窗花是他姐寄回来的，红色的剪纸。十二生肖的。兔子的那对贴在堂屋窗户上。左边一只兔子，右边一只兔子。

灶台上面蒸汽弥漫。猪肉炖粉条在铁锅里咕嘟咕嘟冒泡。醋溜白菜。红烧鱼——鱼是镇上菜市场买的，活的，他爸用桶提回来的，放在脸盆里游了半天。他妈杀鱼的时候他躲开了。过了一阵子回来，鱼已经躺在盘子里了，眼睛变白了。

饺子包好了。整整齐齐码在盖帘上——盖帘是高粱秆子编的，用了好多年。他妈包的饺子跟过年的饺子一样——每个褶子都掐得很紧。他在旁边按剂子——把面团搓成条，揪成小段，然后用手掌按扁。按得不太圆，但能用。

天黑了。

外面的爆竹声密了起来。不是一声一声的那种。是连成片的。整个石桥镇都在响。噼里啪啦的，分不清谁家放的。天空中不时亮一下——烟花炸开了，金色和红色的。小孩的尖叫声隔着窗玻璃传进来。

七点钟。他爸把电视机打开了。CCTV春晚——开场舞是一大群人在台上转圈，衣服各种颜色。主持人走出来说了几句"全国人民""新春快乐"。台下在鼓掌。

他妈把年夜饭端上桌了。七碟八碗的。中间的鱼热气最旺，葱丝和姜片盖在上面，酱油的颜色很深。旁边摆着一盘饺子、一盘猪头肉、一盘炸花生米。酒杯里有半杯白酒——他爸的。他妈喝橘子水——汇源牌的，橙色铁罐。顾临风跟前放了一杯茶。茉莉花茶，茶叶在杯底展开了。

一家三口坐下来。他爸举了一下酒杯，然后喝了一口。没说话。他妈给他夹了一个饺子。他咬了一口，是芹菜猪肉的——和初一前一天吃的馅一样。那天他妈包的破了好几个。今天的饺子一个都没破。

饭吃到一半，手机震动了一下。是放在桌上他爸那部老诺基亚。他爸接起来——"嗯。"——起——"新年好啊新年好。嗯嗯。好好好。"挂了。然后他妈也接了一个——"老婶子过年好——"她笑着说了好一阵子才挂。

顾临风没有电话。他有一部手机——他爸淘汰的。诺基亚1100，黑色的。能打电话、发短信。屏幕上有一道划痕。

通讯录里存了三个号码。他爸的，他妈的，他姐的。没有第四个人。

吃完饭。他妈收拾碗筷。他爸躺在沙发上看春晚。赵本山出来了——"公鸡中的战斗鸡——"——他爸笑了一声。他妈在厨房里喊——"声音小点——"他爸按了一下遥控器。

顾临风坐在旁边木椅子上。妈妈把瓜子端过来给他。他抓了一把。嗑了一个——空的。扔进炉子里。又嗑了一个——苦的。吐出来。第三个味道正常。他磕了半盘子。吃得嘴有点干了。

快零点的时候鞭炮声突然大了起来。

他爸出去放炮仗——五千响的，红皮的。挂在门口的竹竿上。他爸拿烟头去点引线——引线冒火了，他爸往后跑了几步。炮仗响了。红光一闪一闪的，青烟往上冒。空气中是火药味道。

零点。电视里倒计时——"五、四、三、二、一"。钟声响了。烟花炸满了天空。

顾临风站在院子里。烟花把整个院子的雪地照得红一阵绿一阵。他旁边的墙头上落了一小片炮仗屑——红色的碎纸，还有没炸开的小炮仗。雪地里有个脚印——他爸踩的，很深。

远处的鞭炮声汇成了一个巨大的轰隆声。分不出远近。村子那边有人在唱歌，唱的是《难忘今宵》。调子跑了好几个音。

顾临风把手伸进口袋里。口袋里什么都没有。手机没有——放在楼上充电。他把手拿出来，对着手心哈了口气——白的。

他抬头看着烟花。

然后嘴张开了。很小。没有发出声音。只是嘴动了一下——两个字。像是在念一个名字。

烟花又炸了一个。金色的，炸开之后变成了好多小星星。小星星掉下来。在半空中就灭了。

他又站了一会儿。然后转身进去了。

堂屋里电视机里的主持人还在说话——"新的一年，新的开始——"他妈靠在椅子上，眼皮在打架。他爸把炉子捅了捅，火旺了一些。

顾临风上了二楼。窗外的烟花还在放。他把手机拿起来——没有未接来电。屏幕上时间显示:00:12。日期:二零一零年正月初一。

他把手机放在桌上。躺到床上。窗户外的光一下一下地亮着。照在天花板的水印上。

外面鞭炮声还没有停。

他闭上了眼睛。
