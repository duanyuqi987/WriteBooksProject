# 第082章 十二公里

时间：2013年9月

地点：楚州师院

卷次：卷五：各自南北

开学第一周，他做了一件事。

用宿舍里阿坤的电脑查了公交路线。

阿坤的笔记本是联想牌的，屏幕有点小。他搬了凳子坐在阿坤的桌子前面，打开浏览器。输入框里打了八个字——楚州师院到楚州师大。

回车。

页面跳出来一条结果。301路。11站。大约四十分钟。

起始站：师院南门。

终点站：师大北门。

中间九站——市图书馆、市体育馆、中山路口、公园西门、工商银行、人民医院、新华书店、广场东路、师范大学。

他把十一站的站名一个一个记在手机备忘录里。按了七八下键。手机键盘很小，他的拇指有点大，按错了三次，删了重新打。

打完以后他又看了一遍。

301路。师院南门到师大北门。全程大约12公里。

他盯着屏幕上的数字看了十几秒钟。

阿坤从上铺伸头过来。"找什么路子呢？想出去逛？"

"随便看看。"

他把浏览器关了。把电脑还给阿坤。站起来，走到窗边。

窗外那栋远处的宿舍楼还是灰色的。天空有点阴。楼下有人在晒被子，用一根竹竿拍打着被芯。

他拿出手机。翻到备忘录——301路、11站、12公里。

然后把手机合上，放在桌上。

下午他去上了第一节专业课——现代汉语。教室在三楼，能坐一百多人。他坐在靠窗的第三排。老师姓刘，五十多岁，说话很慢。

"现代汉语——研究的是我们每天都在用的语言。"

他在笔记本上写下"现代汉语"。然后在本子右下角画了一个圈。又在圈外面画了一条线。

下课的时候经过学校南门。门口的公交站牌上写着301路。站牌是绿色的，字有点模糊。车来了——一辆蓝色的公交车。前门打开，下来三四个人，上去五六个人。

车关上门，往东边开走了。

往师大方向开走了。

他没有上车。

他站在站牌旁边看了大概两分钟。等第二辆301路开过去。然后转身，走进了校门。

回了宿舍。

李博在整理选课表。"你选修课选了吗？"

"还没。"

"抓紧。有的课名额满了。对了——你想选什么方向的？"

"都行。"

李博看了他一眼。"都行是什么意思。"

"什么有意思就选什么。"

他选了最多的选修课。中国古代文学、西方哲学导论、书法基础、中国历史地理。周一到周五每天都有课，周六上午也有两节。选修课比必修课多了一倍。

李博看了他的选课表。"你疯了吧。选这么多。"

"闲着也是闲着。"

晚上熄灯以后，室友们在聊高中的事。阿坤说他高中翻墙出去上网被逮过三次。小伍说他高中班花长得像刘亦菲。李博说他高中是班长兼学生会主席。

顾临风躺着没说话。

"临风你呢？你们高中什么样？"

"普通的。"

"有没有什么特别的事？"

"没有。"

他把枕头翻了一面。枕套凉凉的。手机压在枕头下面。屏幕应该还是黑的。

天花板上那条裂缝还在。他看了几分钟。闭上了眼睛。

301路车的路线在他脑子里过了一遍——师院南门、市图书馆、市体育馆、中山路口、公园西门、工商银行、人民医院、新华书店、广场东路、师范大学、师大北门。

每站之间大概三四分钟。到市图书馆的时候，窗外应该能看到一个大的广场。到人民医院的时候，可能会有人推着轮椅上下车。到新华书店的时候——那家新华书店的门面大概是红色的。到师范大学的时候——那是倒数第二站，离师大北门只差一站。

他把路线背下来了。闭着眼睛都能数。

十一站。十二公里。四十分钟。

有时候他会在脑子里坐一遍这趟车。买票——投币两块钱。坐在靠窗的位置。右手边的窗外会依次掠过那些站名对应的建筑。到师大北门的时候——下车。站在师大校门口。然后呢。

然后没有然后。

他睁开眼睛。天花板上的裂缝还是那条裂缝。

第二天下午他又经过了南门的公交站。站牌下面站着三个等车的人——一个拎着菜的老太太、一个戴着耳机的中年人、一个抱着书的女生。301路来了。三个人都上去了。车开走。

他又站了几分钟。第二辆301路开过去的时候，他数了车上的人数——大概十二三个。有人拉着扶手站着。有人靠窗坐。

他转身进了校门。门口的保安看了他一眼——大概是认出他了。他在门口站太多次了。

保安没说话。他也没说话。

走回宿舍的路上，他在心里把那十一个站名又默了一遍。没有漏。没有错。还是那十一个站名。

十二公里。他没有去过一次。

他遵守了约定。
