# 第228章 南湖再见之后

所属部卷：第五部：逝水有灯
时间：离开南湖后
地点：大学南湖凉亭

## 诗歌插页 P228：原创仿宋小令

湖灯起，晚风平。
旧径有新名。
饭卡无声过不了门。
扫码也成行。

离开南湖后，陆允川绕到食堂。旧饭卡早就不能用了，门口闸机只认学生码。他站在入口看了一会儿，正准备走，食堂阿姨指了指旁边的临时窗口：“校友可以手机付。”

他买了一份最普通的盖饭，味道和记忆里相似，又不完全相同。米饭偏硬，青菜偏咸，汤免费。陆允川端着餐盘坐到角落，旁边几个学生在讨论考试周。有人把书摊在桌上，一边吃一边划重点。

这顿饭把南湖带回了地面。湖边的风再温柔，也挡不住人会饿，会找座位，会担心第二天培训迟到。陆允川吃到一半，手机又响，是同事催他过去集合。他回了一个定位，说二十分钟到。

走出食堂时，他把餐盘放到回收处，听见金属边轻轻一碰。这个声音比许多回忆都实在。大学不是只有恋爱、等待和错过，还有吃饭、排队、刷码、找零钱、赶下一场安排。

他在校门口打车。司机问是不是老师，他说不是，以前在这里读过。司机说那变化挺大吧。陆允川看着窗外点头：“挺大，也挺好。”
