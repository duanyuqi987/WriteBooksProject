# 第093章 许若琳

时间：2014年5月

地点：支教社团活动

卷次：卷五：各自南北

支教社团出发前有三次培训。地点在大学生活动中心二楼——一间有落地镜的舞蹈教室。木地板有时咯吱响。镜子占了整面墙——镜面有刮花痕迹。每次进去要脱鞋——门口摆了一排鞋子——运动鞋、帆布鞋、凉鞋。鞋里塞着各种颜色的鞋垫。顾临风每次都坐在最后一排——背靠镜子——镜子里能看到自己后脑勺。

第一次讲安全——"山区有蛇""不要单独行动""随身带手电筒"。第二次讲教案——发打印模板。第三次讲山区教育现状——PPT放了照片——泥墙教室、破课桌、黑板上写着"好好学习"。

第三次培训结束——门口大家穿鞋。许若琳也在——第二次培训才出现的——坐得离他不远。她系着帆布鞋的鞋带——白色鞋带带花——鞋底踩了泥。"你也来支教社团了？""嗯。第三分队。""第三分队是跟理工学院的——你认识那边的人吗。""不认识。""那你挺勇的。"她拍拍膝盖上的灰——站起身。"我是第一分队——师大的那队。""嗯。""苏晚晴也在第一分队。"她低着头把鞋带塞进鞋子——声音不大——舞蹈教室回声重——尾音被放大。"嗯。""她教三年级语文——我教英语。""嗯。"

她抬起头——眼里有一丝笑意——是那种看穿什么的笑。"你怎么每次提到她都是'嗯'。""不知道说什么。""行吧。"她笑——眯起眼。"不过我倒是可以偶尔给你传递一下情报——她在师大的近况——你要是想知道的话。""不用。""真的不用。""嗯。"

顾临风穿好鞋——蹲下系紧——鞋带拉太紧又松了松。但他后背贴着墙。

可许若琳还是说了。每次社团活动碰到——谈话的某刻——她会忽然把话题转到苏晚晴——"晚晴最近在准备教师资格证""她实习教三年级语文""她说小朋友很可爱""她食堂点糖醋排骨说太甜了""她室友说梦话——'同学们坐好'——""她学骑自行车了，在师大梧桐路下面练"。说的语气很轻——说完就看别处——不盯着他。说完以后低头系鞋带或者翻书包——假装刚才的话只是随便提。顾临风每次反应一样——不做声——不多问——不接话。但他把每句话都记住了。

教师资格证考了两门——教育学和心理学。实习教语文——第一堂课紧张得声音发抖——但讲完了。糖醋排骨太甜。小朋友很可爱。室友说梦话"同学们坐好"。学了骑自行车。每句都像从另一个人的人生截下的碎片——碎的——不能拼——只能收着——塞进记忆抽屉。

有一次她说——"晚晴学骑自行车时在师大里面骑——摔破膝盖皮了。"她在整理书包——带子打了个结——低头解结。他那天晚上操场多走了一圈——第五圈——走得很慢。第五道白线在灯光下有微弱的光。不是因为担心——而是想象她骑车的样子——马尾、浅蓝外套——在梧桐路下——摔了——膝盖破皮。她会自己上红药水吗。他不知道。也不会问她。

五月中旬——最后一次培训前。许若琳忽然QQ上发了句——"她好像瘦了一点。""为什么。""不知道——可能实习太累。""嗯。""你不是说不关心吗。""随口问的。""随口问——也是关心。"

他把QQ最小化。窗外五月傍晚的光——金黄色——斜照在对楼墙上。他没有告诉她——那天晚上他搜了师大地址——在地图上放大缩小——看了很久——然后退出了。十二公里。十一站路。四十分钟公交。301路——从备忘录翻出来又看一遍。他没有问许若琳——能不能帮忙带句话——能不能告诉她我挺好——能不能告诉她我考了第三——能不能告诉她其实我每天——没问。

关了电脑。去操场。那天晚上人很少——空气湿度大——大概要下雨。云层压得很低——路灯的光在云底反射出一圈灰白色的光晕。第五道上走了五圈。走的时候步子比平时重——鞋底踩在塑胶跑道上，每一步都有一个闷闷的回响。跑道上只有他一个人。足球场上也没有人——假草坪在即将到来的雨前显得格外绿。看台上空着——那些情侣大概都躲回宿舍了。

走完第五圈。饮水台前弯腰接水。漂白粉的味道比平时更重——大概是今天加了氯。喝完以后用袖子擦了一下嘴。袖子在嘴角留下一道湿痕。

回宿舍的路上经过电话亭。电话亭还是黑的——灯没修好。玻璃上那行被哈气写下的"等"已经彻底消失了——大概被雨水冲掉了。他停了一下。手在裤兜里攥了攥手机。没有掏出来。继续走。

回到512。李博在看新闻——电视上在播一则关于暑假山区支教的报道。记者站在一个土操场上，背后是一排矮矮的平房，旗杆上国旗在风里飘。李博说"你们支教的地方大概也这个样子吧"。他看了一眼屏幕。镜头扫过一群孩子的脸——黑黑的、瘦瘦的，眼睛很亮。"差不多。"他说。坐到床边。阿坤的键盘今天出奇地安静——他在看一个游戏论坛的攻略，屏幕上全是文字。小伍又在写信——今晚的信封上画了一只长颈鹿，他说"之前的熊猫信封用完了，我妈又寄了新的"。他拆开新信封的包装——塑料袋撕开的声音在安静的宿舍里有点刺耳。

顾临风躺下。从枕头下摸出手机。屏幕按亮。通讯录。翻到S。苏晚晴——还是那个号码。手指在屏幕上停了一下。然后往下翻——许若琳文学社。点进她的号码。打了一行字——"她膝盖的伤好了吗。"拇指在发送键上停了几秒。然后又一个字一个字删掉。删完以后屏幕又回到空白的输入框。他盯着那个闪烁的光标。按灭屏幕。黑暗里天花板上的裂缝还在——依然是四条分叉。

他闭上眼睛。脑子里不是裂缝。是师大的梧桐路——那条想象中的梧桐路。路两边种满法国梧桐。九月的时候叶子是绿的。十月的叶子开始变黄。她在下面练自行车——车龙头歪歪扭扭的，轮子在地面上画着S形的线。她穿浅蓝色外套——马尾在后面甩。然后摔了——膝盖碰到水泥地，擦破了皮。她站起来拍了拍裤子。看了看膝盖。然后扶起车，又骑了上去。

他不知道这个画面是真的还是假的。但他的脑子把这个画面一遍一遍地重放——每次膝盖破皮的位置差了半寸，每次马尾甩的幅度不一样，每次梧桐叶的颜色在绿和黄之间偏了不同。

然后他翻了个身。明天最后一次培训。后天出发去太行山。大后天开始——他在李家沟小学的土操场上教二十一个孩子语文。她在另一个镇上的另一所小学里教三年级的语文。中间隔了三十多公里盘山公路——比十二公里远。但方向是一样的——都是往山里走。
