# 第186章 诗社告别之前

所属部卷：第四部：失去的美丽
时间：大三开学
地点：宿舍 / 诗社活动室

## 诗歌插页 P186：原创现代诗

我把书脊放回夜色，
不让它替谁证明深情。
风从未说完的话里穿过，
有人低头，学着不再追问。
后来水声慢下来，
灯还亮着，人已经能往前行。

大三开学第二周，陆允川做了一个决定：退出诗社。

不是一时冲动。整个暑假他都在想这件事。诗社是他和苏晚枝唯一还能自然碰面的地方——活动室搬书、群里通知、偶尔的朗诵会——但他发现每次去活动室之前都会心跳加速，每次结束后都在日记里分析见面时的每一句话。不是正常社团参与者的心态。

他在宿舍宣布这个决定的时候，祁野正在为新换的电脑调试分辨率，程砚正在预习大三的实验课表。

"我打算退诗社。"他说。

祁野的手离开键盘。"决定了？"

"决定了。"

"不是赌气？"

"不是。想了两个月。"

程砚从课表上抬起头。"这两个月你想出了什么结论？"

"结论是：我现在去诗社的动机，百分之七十不是为了诗。是为了能在活动室里看到她。就算什么都不说——只是在那里——也不是真的爱诗。是习惯性地需要一个能看到她的空间。这个空间对我有害。"

"那百分之三十呢？"

"百分之三十是真的喜欢诗。但百分之三十不够我在那里继续待。"

祁野靠在椅背上，双手交叉放在肚子上——这是他的思考姿势。

"那你退社之后要干嘛？"

"不知道。可能去旁听别的课。可能多去打球。"

"你上次打球还是大一下学期。"

"那是因为大二都在写日记。"

"那你去打球吧。"祁野把椅子转回去，"退社申请书需要我帮你写吗？"

"你会写申请书？"

"不会。但我会帮你写'退社可以，别退化成人形日记'——作为参考语录。你的申请书正文完全可以围绕这句话展开。"

陆允川笑了一下。不是苦笑——这次属于祁野专属的无厘头但正确的幽默。

程砚合上课本，看着他说了最后一句："你退出的是诗社，不是你的朋友圈。有什么活动我们还是会在群里叫你。"

"我知道。"

"那就去办。趁还没开始招新。"

陆允川当天下午去了诗社活动室。门还是那个涩涩的门——大三了还没修好。推开发出的短响让里面正在整理册子的苏晚枝抬了一下头。

"我来还书。"他把背包放下来，开始一本一本往外拿。旧版的《中国诗歌选》、用过的几册社刊合订本、一本记着他写过几次朗诵主持词的活页夹——所有带诗社标签的书和资料。

苏晚枝看着这些书摞在桌上，没有说话。窗帘下摆蹭着地面，扫出一道细细的灰线。墙角的展板比半年前褪色更多了。走廊里新生抱着海报跑过去，胶带在手腕上甩着，边跑边喊"借剪刀——谁有剪刀——"。

"这是全部了。"他把最后一本放在摞堆上。

"好。"苏晚枝说，"登记表上我给你勾'退社'。"

"谢谢。"

苏晚枝从抽屉里拿出那张活动成员登记表。找到他的名字——在一栏一栏的出席记录旁边。她用笔画了一个小圈，把这一行标注为"退社"。动作很轻，但每一笔都看得到。

她在圈后面写了三个字："老社员。"

"不是直接划掉？"

"不退就划掉。退社的——是老社员。"

她把表放回抽屉。诗社里一时没有别的声音。新生们在走廊里还在找剪刀，胶带刺啦撕开的声音很脆。陆允川把空背包拉链拉上。

"那你以后还会写诗吗？"苏晚枝问。

"会。不是诗社的事。"

"好。那以后在哪发表了别忘了发链接。"

"会的。现在还没发表过什么，但以后有了就发。"

苏晚枝点了点头。然后转身继续整理册子。他在活动室门口站了一下——门涩涩的，那个还没上油的合页能把人的所有犹豫都翻译成一声短响。
