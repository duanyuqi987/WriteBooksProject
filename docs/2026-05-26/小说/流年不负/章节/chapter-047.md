# 第047章 台上的灯光

时间：2011年12月31日深夜

地点：学校礼堂至操场

卷次：卷三：此心藏久

晚会结束的时候。礼堂的灯光亮了。

刚才台上的暖光都灭了。现在亮起来的是日光灯——白色的。照得每个人的脸都显得很疲倦。妆花了。腮红糊了。口红的颜色淡得只剩下一条线。

有人在拆舞台上的装饰。红色气球一颗一颗被解下来。有人拿针去扎。砰。砰。气球炸了。碎红色橡胶皮掉了一地。

主持人把手里的稿子折起来塞进口袋。白色的裙子在后台拖了一地——刚才在台上看着很亮，现在在日光灯下，能看见下摆有点脏了。

苏晚晴从后台走出来。她已经换回了校服。脸上的妆用湿巾擦了大半——额头还有一点没擦干净的粉。嘴唇上的颜色也没完全卸掉。远远看过去，像是嘴唇本身带着一点红。

马尾重新扎好了。扎得很紧。发绳是黑色的。跟平常一模一样。

她在礼堂后面站了一会儿。跟几个合唱的同学聊天。有人说了个笑话。她笑了。笑声不大。然后她们一起往外走。

零八年。新年快到了。

操场上有人在放烟火。

不是学校组织的。是几个高三的男生自己买来的——从校门口小卖部。两个人抬着一箱。"别让值周老师看到——""废话。跑快点。"

顾临风站在礼堂外面的台阶上。手揣在口袋里。围巾是蓝色格子的——他妈去年织的。毛线有点扎脖子。

他看到她了。

她正从礼堂出来。门推开的一瞬，冷风灌进去。她缩了一下脖子。把校服的拉链拉到最高。

然后她看到操场上空的烟火。

金色的烟花在她眼睛里炸开。

她的脸被照得一明一暗。烟花升上去的时候，她的脸是亮的。烟花落下来的时候，她的脸又暗了。一明一暗。一明一暗。

她站住了。

仰着头。嘴唇微张。一团白气从嘴里冒出来。然后被下一颗烟花的光打成金色的。

她身边还有一些同学。有人在说话。有人在拍照。有人在喊"哇——好漂亮"。她没有说话。她只是站在那里。仰着头。看着天上一朵朵炸开又熄灭的光。

顾临风离她大概十二步远。

他在人群的左边。她在人群的中间。中间隔着别人。他没有往前挤。就站在那个位置。

他也在看烟火。但每隔几秒钟，他的目光会从天上落下来——落到她的后脑勺上。落到她的马尾上。落到她被烟火照亮的侧脸轮廓上。

她的耳朵很小。耳垂在冷风里有点发红。

她伸手把碎发别到耳后。那个动作。他见过无数次。每次都是这样。食指和中指并拢，往外一拨。碎发就挂在耳朵后面了。过几秒钟，风一吹，又掉下来。她又拨一次。

烟火放了大概十五分钟。

最后是一颗最大的——红色的。升得很高。在最高点炸开，变出一个圆形的光。很大。然后那条光慢慢往下坠。一点一点碎。碎到一半就灭了。

天空黑了。

操场上的烟味还散不掉。灰烬从天上落下来。落在草皮上。落在跑道上。落在人们的头发上。

前面有人喊了一句"新年快乐——"

然后一个人接一个人地喊。声音此起彼伏。男生喊。女生喊。不同的嗓子。汇成一片嘈杂的声浪。在操场上空回响。

顾临风没有喊。

他还站在台阶上。手在口袋里攥了一下——手心里是凉的。

苏晚晴跟着人群往宿舍的方向走。走过操场第四道跑道的时候，她的步子慢了一下。好像踩到了一颗没点燃的小烟花——纸壳的。她踢了踢。继续走。

她的背影越来越远。最后融进了宿舍楼门口的人群里。

顾临风还在操场边。

赵一鸣从后面走过来。手里拿了一根还没点的小烟花——细长条。铁丝芯。银色的。

"给你一根。"

"不放了。"

"留着以后放。"

赵一鸣把烟花塞进他口袋里。然后拍了拍他的肩。拍了两下。手重了一点——不像拍肩。像在打气。

"新的一年了。"

"嗯。"

"今年说什么也得把话说了吧。"

顾临风没有回答。他低头看了看口袋里的那根烟花。铁丝芯。银色的外壳。在路灯下反光。

"高考后。"

"啥。"

"高考后。告诉她。"

他说这句话的时候声音不大。但很清楚。不是商量。不是犹豫。是一个结论。他在脑子里已经放了很久。现在只是说出来而已。

赵一鸣沉默了几秒。

"你确定。"

"确定。"

"不怕她拒绝。"

"怕。"

"那你还要说。"

顾临风看着操场上空已经没有烟花的黑夜。他口袋里那只烟花很轻。隔着一层布也能感觉到铁丝芯的硬度。

"不管结果。我就是把这句话说出来。三年了。这句话不说。我自己过不去。"

赵一鸣没有说话。他把手也揣进口袋里。两个人站在操场边。路灯在头顶嗡嗡响。

远处有人在唱《明天会更好》。不知道哪个班。合唱结束了。还在哼那个旋律。断断续续。声音被风吹散了。

两个人往回走。经过教学楼的时候，顾临风抬头看了一眼。三楼的窗户全黑了。高一三班的教室。她坐在第二排靠窗的位置。窗外是银杏。银杏已经秃了。明天就是新的一年了。

明天。银杏还是那棵银杏。她还是那个她。

但他下了个决心。

从现在到高考——还有一年半。一年半。他要做两件事。第一件：把成绩提到能跟她考同一所大学的水平。第二件：在高考结束的那天晚上。站在她面前。说出那句他三年都没说的话。

不管她听不听得见。

不管她听完以后是什么表情。

不管结果。

他说。

赵一鸣在旁边打了个哈欠。"走了。冷死了。明天还得背单词。"

"你什么时候也开始背单词了。"

"被你传染的。妈的。"

两个人走过操场。走过教学楼。走过食堂。走过那条从教学楼到宿舍的路。路灯还是一盏亮一盏暗。飞虫围着灯泡绕圈子。

到宿舍楼下。赵一鸣说了句"明早别叫我——""你叫我。"

"行。七点。"

"六点半。"

"六点半。行。"赵一鸣踹了一下楼下的垃圾桶。垃圾桶晃了晃。没倒。

楼梯间那个坏了的灯还没修好。顾临风摸黑上楼。走得很慢。每一步都踩稳了。他对这条路已经很熟了——不用看。腿自己知道哪里该抬多高。哪里该拐弯。

进宿舍。室友们都已经躺下了。有人在打电话。有人在打呼噜。有人在磨牙。

顾临风没开手电筒。脱了外套。脱了鞋。躺下来。被子拉到胸口。

窗外还有人在放小烟花。光一闪一闪。照着天花板。

他把手伸进口袋里。摸到了赵一鸣给他的那根烟花。拿出来。放在枕头旁边。然后闭上眼。

2012年。

高考后。

告诉她。
