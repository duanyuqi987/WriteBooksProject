# 第096章 世博新闻



时间：2010年三月下旬，晚饭后的新闻时间



地点：高二十班教室、食堂门口



卷次：第五卷：未来在黑板上倒计时



2010年三月下旬，晚饭后的新闻时间，高二十班教室、食堂门口，小卖部电视里滚动的上海世博新闻把2010年的春夏推到十班面前。上海世博的新闻从小卖部电视里传到教室，十班第一次把远方和自己的未来放在一起说。

郭向阳兴奋地讲城市、展馆和志愿者，段璟却发现自己连下次月考的物理都还没稳住。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

郭向阳也在这段日子里显出自己的位置。爱看新闻，地理好，喜欢把大城市讲得很近，成绩却被数学拖住。群像不是名单，群像是每个人都有自己的难处。

“上海世博开了以后，新闻肯定天天播。”郭向阳说。

“你先把数学卷子开了。”付小川说。

“远方可以看，眼前也不能空着。”常俊凡说。

“我眼前最远的是物理最后一题。”段璟说。

“新闻不是让你们走神，是让你们知道题外面也有世界。”胡向南说。

小卖部电视声音不大，围着的人却不少。有人看新闻，有人只是借着看电视多站一会儿。

上海、展馆、志愿者这些词从电视里飘出来，落到封丘一中的校服袖口上，显得又远又近。

郭向阳讲得兴奋，手指在地图册上划来划去，像一条线能从县城直接划到黄浦江边。

段璟听着，心里却先想到自己的物理卷。远方很亮，眼前的空题也很白。

常俊凡没有笑他想得远。她只是把新闻里听到的一个词写在草稿纸边上，然后继续改错题。

晚自习开始后，电视声被关在小卖部里，教室里只剩笔尖和纸页声。

胡向南说世界很大，但大不是用来吓人的，是用来提醒你们别只把眼睛贴在分数上。

这句话没有让题变简单，却让十班那天的晚自习多了一点看不见的风。

胡向南这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

郭向阳被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

晚饭后那段时间是十班最松的时候。食堂门还没关，小卖部门口的灯泡刚亮，校门口陆续有家长骑电动车送东西。段璟和付小川一起去食堂给饭卡充值，队伍排到门外，饭卡机"嘀"一声短响就进去一个，再"嘀"一声就出来一个，像某种计时的节奏。

"你饭卡还剩多少？"付小川把卡掏出来，拇指在卡面蹭了一下。

"不到二十。"段璟说。

付小川没接话，排到窗口时往卡里充了五十，又转头看了段璟一眼。段璟知道那是什么意思——付小川在算他够不够撑到月底。这个人嘴上从来不说关心，但动作里藏着账本。

段璟也充了五十。卡机短响，余额跳了一位数，他把卡塞回校服口袋，手指碰到口袋底部的线头。这件校服从高一穿到现在，左口袋的线一直没缝。

回到教室时，郭向阳还在讲世博的事。他把地图册摊在段璟桌上，手指按着上海的位置："你看这个展馆，英国馆像蒲公英，晚上会亮。""你先把数学卷子亮了。"付小川从后面路过，丢了一句。郭向阳也不恼，把地图册翻了一页："我数学差了又不是不看新闻就得挨骂。"旁边几个人笑了。郭向阳的地理确实好，好到能把任何一个地名讲得像自己去过。但他的数学也确实差，差到陶建国在班会上专门提过一次。

"郭向阳，你的地理能考年级前十，数学能不能不考年级倒数前十？"当时陶建国站在讲台上，语气不算重，但那种"我先肯定你再提醒你"的方式比直接批评更让人没法顶嘴。郭向阳低着头"嗯"了一声，耳根红到脖子。散会后秦佳禾跟他说了一句"我数学也不稳，可以一起补"，声音很轻，像怕踩到什么。

树莓摊摆在城关镇路口，是三月下旬才出来的。透明塑料盒装红树莓，水珠还没干，一盒两块五。段璟路过时站了一会儿。摊主是个五十来岁的妇人，把零钱压在泡沫箱盖子下面，风一吹纸币角就翘起来，她用手掌按回去。这个动作让段璟想起他妈在菜市场买菜时也这样按着零钱。

他买了一盒。不是特别想吃，是觉得那个妇人手按零钱的样子像一种没说出来但谁都懂的东西——日子就是这样过的，钱就是这样一块两块攒的。

常俊凡从新华书店方向走过来，手里拎着一个塑料袋，里面是练习册和笔芯。她看见段璟站在树莓摊旁边，没有刻意绕开，也没有刻意走近，就走到了摊前。

"你也买树莓？"她问。

"嗯，看着新鲜。"段璟把盒子往她那边递了递。

常俊凡拿了一颗，吃完把籽吐在纸巾上。"酸。""酸的才像真的。"段璟说。常俊凡看了他一眼，那个眼神不算笑，但嘴角有一个很小的弧度——段璟后来反复想过那个弧度到底是什么意思，最后觉得可能没有意思，就是她觉得这句话不无聊。

"你物理补得怎么样了？"常俊凡问。

"还在补，最后一道大题还是不稳。"

"最后一道大题不稳，说明前面基础题的速度还不够。你先把选择填空练到二十分钟以内，后面自然有时间磨大题。"

段璟愣了一下。他不是第一次听人给学习方法，但常俊凡说的时候没有"我比你强"的意味，也没有刻意温柔，就像一个解题的人看到另一道题，顺手把步骤写在旁边。

"我试试。"他说。

"不是试试，是做。"常俊凡把练习册从塑料袋里抽出来，翻到目录页给他看。她自己的进度用铅笔画了记号，一章一章往前推，没有跳，没有赶，每一章的记号都很整齐。

段璟忽然意识到一件事：常俊凡不是那种等着别人来帮的人。她帮别人不是因为想被喜欢，是因为她对自己够确定，确定到可以把多余的那部分分给别人。

这个发现让他心里的喜欢变得比以前更轻，也更沉。轻是因为不再幻想她需要自己来拯救；沉是因为他知道，如果要和她并肩走，他得先把自己的路走稳。

"那你竞赛的事呢？"段璟问。

"名单还没最后定，但辅导老师找过我两次。"常俊凡把练习册合上，"我妈想让我去，我爸说看我。我自己还没想好。""为什么没想好？"

常俊凡没有马上回答。她把树莓盒拿起来看了一眼，又放回去。"竞赛培训要占晚自习，周末也要加课。我不怕多做题，但我不想因为竞赛把其他科拖下来。""你可以的，"段璟说，"你哪科都没落下过。"

"不是落不落的问题，"常俊凡说，"是在哪里花时间的取舍。如果我把时间全部投给竞赛，拿到了奖，那是我的选择。如果拿不到，那是我评估错了，也是我的事。但我不希望别人替我做这个决定。"

她说"别人"的时候没有看段璟，但段璟知道那个"别人"包括她妈、辅导老师、班主任——也包括他。常俊凡在给他划一条线：喜欢你的人会尊重你的选择，而不是替你做选择。

段璟没有再往下问。他把树莓吃完，盒子扔进路边的垃圾桶。风从黄河大堤方向吹过来，带着一点泥土和麦子的味道。三月底的麦子还没完全熟，秆子是青的，但太阳已经够辣。

"走吧，晚自习快开始了。"常俊凡说。

两个人一前一后往校门走。校门口的电动车喇叭声此起彼伏，家长把保温桶、衣服、零花钱往孩子手里塞。有个女人喊了一声"按时吃饭"，声音穿过电动车流，落到一个男生肩膀上，男生回头挥了挥手，动作很匆忙。

段璟看着那个男生跑进校门，忽然想起自己上次回家吃饭是两周前。他爸在饭桌上说了一句"高二了，该定方向了"，没有后面的话。段璟知道那是什么意思——家里不富裕，补课费是一笔账，将来上大学是另一笔账，他不会让家里为难，但也不想现在就认命。

常俊凡走进教室时，邱可可正在往黑板报上贴新一期的排版纸。黑板报的主题是"世博来了"，角落画了一个简笔画的中国馆，红色粉笔涂得不太均匀，余小满在帮忙补色。

"这个馆真的长这样吗？"余小满问。

"新闻里看到的差不多，"郭向阳从座位上站起来，走到黑板报前，"顶上这个倒梯形是它的特征，里面有很多展区。"

"你又知道了。"付小川说。

"我就是知道。"郭向阳把地图册翻开，里面夹着一张从报纸上剪下来的世博场馆示意图，"我以后想去上海。"

这句话他说得很自然，不像宣言，像陈述。付小川没有再说他，而是看了一眼那张剪报。剪报边角已经磨毛了，说明郭向阳翻了很多遍。

段璟坐回自己的位置，把物理卷翻到最后一题。常俊凡的话还在他脑子里——"把选择填空练到二十分钟以内"。他在草稿纸右上角写了一个数字：20min。

邱可可贴完黑板报，用透明胶把翘起的边角重新粘了一遍。透明胶还是翘，她干脆从讲台上拿图钉按上去。这个动作很轻，但教室里安静，图钉钉进木板的声音传了几排。

"可可你轻点，"余小满说，"陶老师路过会说的。"

"陶老师今天不在。"邱可可回头看了一眼门口。

但陶建国刚好从门口经过。他没有进来，只是站在门外看了两秒。教室里原本散开的声音立刻往回收——说话的人把后半句咽回去，翻书的人把动作放轻，连郭向阳都把地图册往桌洞里塞了一下。陶建国没有说什么，走了。他的脚步声在走廊里很稳，一步一步，不急也不慢。

"吓死我了。"邱可可小声说。

"他就是看看，"秦佳禾说，"他不是那种没事找事的人。"

这话不假。陶建国训话从来不在走廊上，也不在教室里随便说。他只在班会上讲，而且每次都准备到能让你觉得他不是在骂你，是在把你从一条更麻烦的路上拉回来。

上次班会陶建国提完郭向阳的数学之后，又说了一段话："你们现在觉得一份卷、一次排名就是天大的事。不是。天大的事是两年后你回头看，发现自己本来可以做的事没有做。不是因为难，是因为你觉得自己不行。"

教室里当时很安静。那种安静跟平时不一样——不是怕挨骂的安静，是每个人都在想自己那件事的安静。

段璟那时候在想自己的物理。常俊凡在想她的竞赛。郭向阳在想他的数学。付小川什么都没想——他是少数几个听完这段话面不改色的人，因为他已经在做了。

玉树捐款箱是这周放到讲台上的。红色纸箱，正面用毛笔写了"玉树加油"四个字，投币口开得不大，里面的纸币被折成小块，硬币沉在底层。胡向南在语文课上提过一次："灾难很远，但你们愿意往里面放一个硬币，就是一种态度。"

段璟往里面放过五块钱。不是很多，但他放的时候想了一下：那个数字等于他在树莓摊买两盒树莓。

晚自习开始前，段璟去水房接水。水房里的暖水瓶排了一排，铝制瓶盖碰在水泥台上发出闷响。常俊凡也在接水，两人站在相邻的水龙头前。

"你说的那个二十分钟，我今天试了一下。"段璟说。

"怎么样？"

"选择题做了二十三分钟。比之前快了三分钟，但是错了两个不该错的。"

"那是因为你还在适应节奏。先练三天，别一天就给自己下结论。"

常俊凡拧紧杯盖，把杯子放进书包侧袋。她的动作很利索，每一样东西都有固定位置——杯子在左边，笔袋在中间，练习册按科目叠好放在右边。

段璟看着那个书包，忽然觉得很踏实。不是因为喜欢她，是因为她的存在本身就像一种参照：你可以把事情做好，可以不慌，可以把生活整理成自己想要的样子。

"你竞赛的事想好了吗？"段璟又问了一遍。

常俊凡这次回答得比之前快："还没完全想好。但我已经跟我妈说了，这个星期天我自己去找辅导老师谈一次。谈完再定。"

"她同意你自己去？"

"不同意。但我说服她了。"

常俊凡说"我说服她了"的时候语气很平常，但段璟听出了里面的分量。她不怕跟家人有分歧，也不逃避沟通。她的边界不是靠冷战划出来的，是靠一句一句说出来的。

段璟想，这大概就是常俊凡最让他喜欢的地方——不是她的成绩，不是她坐在第二排的侧脸，是她知道自己要什么，也知道要为这个"要"承担什么。

晚自习铃响了。段璟回到座位，摊开物理卷。草稿纸右上角那个"20min"还在，他拿笔在下面加了一行：Day 1，23min，错2。

窗外天完全黑了。封丘县城的夜晚没有上海世博新闻里那么亮，但路边灯也在亮，电动车也在跑，麦子也在长。十班的灯管嗡嗡响，笔尖刷刷走，偶尔有人翻书，偶尔有人叹气。

这些声音混在一起，像一个班级自己的节奏——不快，但没有停。

世博新闻之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

世博没有让他们立刻看见世界，只让他们意识到世界并不会等他们把每一道题都想明白。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，小卖部电视里滚动的上海世博新闻还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
