# 第102章 她在一所小学

时间：2015年10月

地点：楚州

卷次：卷六：平行轨道

十月中旬。楚州的秋天很短。

树叶黄了几天就掉光了。

顾临风在图书馆看考研政治。看了两个小时。中间去接了一次水。回来继续看。

手机震了一下。

微信群。高中同学群。平时是死的。偶尔有人往里面扔一条链接。

今天是活的。

许若琳发了三张照片。

第一张：一间教室。墙上有板报。黑板上画着一个巨大的"三年级"。

第二张：一群小孩在操场上做操。队伍歪歪扭扭的。前排有个小胖子动作特别大。

第三张：一个女老师站在讲台上。侧脸。马尾。白衬衫。手里拿着一本语文书。

第三张是苏晚晴。

顾临风的拇指停在屏幕上。

许若琳在照片下面配了几行字。

"今天去楚州一小看晚晴。她在那里实习。教三年级语文。"

"那些孩子很喜欢她。下课都围着她叫苏老师。有个小姑娘说苏老师最漂亮——全班都鼓掌了。"

"晚晴变了好多——比以前会笑了。站在讲台上的样子特别好看。"

群里开始冒泡。

"哇，苏晚晴当老师了。"

"小学老师好啊——有寒暑假。"

"她在哪个小学来着？"

许若琳回复："楚州一小。就在楚州师大后面那条街。"

顾临风盯着这几个字。

楚州一小。

楚州师大后面那条街。

他把照片放大。

苏晚晴站在讲台上。侧脸。马尾扎得比高中时候高了一点。白衬衫袖子挽到手腕。手里拿着一本语文书——大概是三年级的。她嘴在动——应该是正在讲课。嘴角有一点弧度。

他看了很久。久到手机自动锁屏了。

解锁。又看了一遍。

第二张照片里她在操场上。站在队伍前面带操。动作很标准——举手、转身、蹲下。后面的小孩子们跟着做。这个做错了那个做慢了。她不急，一个一个纠正。

阳光很好。她的影子落在塑胶跑道上。旁边是一个小女生的影子——踮着脚在学她的动作。两个影子一高一矮，在红色跑道上排在一起。

他的手指在屏幕上停着。

教室里那张。黑板旁边的墙上贴着一张手抄报——彩色粉笔画的，上面写着"三年级（2）班"。下面贴了几张学生的作业。字迹有大有小。有一张上面画了一个太阳——旁边歪歪扭扭地写着"我的老师苏老师"。

他不知道她在楚州一小实习。不知道她教三年级。不知道她扎马尾的高度变了。不知道那些孩子在作业本上画了她。

现在是知道了。

他把照片看完。退出去。回到桌面。打开浏览器。在搜索框里输了几个字。

楚州一小。

地址：楚州市城西路126号。

他看了一眼。然后关掉浏览器。

把手机翻过来放在桌上。

继续看考研政治。

看了大概十分钟。一个字没看进去。

他重新翻开笔记本。在空白页上写了四个字——楚州一小。

然后划掉了。

划得很重。笔尖把纸划破了。

他把这一页撕掉。叠了两折。扔进桌下的垃圾桶。

垃圾桶是空的。纸落在底上。孤零零的一张。

下午他去食堂吃饭。西红柿炒蛋。土豆丝。一个馒头。跟昨天一样。跟前天也一样。

吃完了把碗端到回收口。

回宿舍的路上经过校门口。他往右拐了一下——那边是往城西路的方向。走了几步停住了。

转身继续往宿舍走。

李博在宿舍里打游戏。"老顾——你怎么脸有点白。"

"没怎么。"

"感冒了？"

"没。"

他坐下。打开台灯。拿出考研政治。

李博隔着桌子看了他一眼。没追问。

晚上十点。熄灯。

他又翻出手机。打开许若琳的朋友圈。三张照片还在。他看了第三张。苏晚晴的侧脸。

然后关掉手机。

天花板上那条裂缝还在。

窗外的路灯穿过窗帘。光很薄。落在地上。

楚州一小。城西路126号。

从他学校骑车过去大概二十分钟。公交车四站。走路四十分钟。

他闭了一下眼睛。

眼前是她站在讲台上的侧脸。马尾比高中高了一点。白衬衫袖子挽到手腕。手拿语文书。嘴在动——在讲课。台下的孩子们仰着头看她。

他没有去。

天花板上的裂缝在月光里。他盯着那道裂缝。盯着。然后翻了个身。脸冲着墙。墙上的小孔还在。黑的。什么都看不到。
