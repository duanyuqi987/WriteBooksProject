# 第225章 重回县一中多年后

所属部卷：第五部：逝水有灯
时间：多年后
地点：县一中翻新的校门

## 诗歌插页 P225：原创仿唐绝句

旧校门前客已稀，
新牌照影换朝晖。
三楼不是谁私藏，
日日书声送客归。

多年后，陆允川写到县一中时，最先想起的不是夏知遥坐过的位置，而是来访登记本。那本登记簿摊在保安亭的小桌上，纸页被风吹起一角，旁边压着一支黑色签字笔。

他觉得这很好。青春如果没有登记本，就容易被人说得无边无际。可真正重回去，姓名、电话、来意、进校时间、离校时间，一栏一栏都要写清楚。那所学校用很普通的方式提醒他：你只是来访，不是归还。

他后来没有再频繁回县一中。偶尔同学群里有人发校门照片，他会点开看一眼，再关掉。三楼教室继续换学生，黑板上的倒计时一年年重写。旧座位不会空着等谁，旧走廊也不负责替谁保存心跳。

写作时，他把这些写进去。不是为了显得冷静，而是为了让小说有真实边界。一个人再怀念，也不能把公共的学校写成自己私人的纪念馆。

窗外傍晚的光落在书桌上，软面抄放在手边。封面蓝线已经磨淡，第一页写着那天买下它的日期。陆允川看着那行日期，心里很安静。
