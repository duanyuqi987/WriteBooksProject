# 第101章 玉树捐款箱



时间：2010年四月中旬，学校组织捐款当天



地点：高二十班教室、讲台前



卷次：第五卷：未来在黑板上倒计时



2010年四月中旬，学校组织捐款当天，高二十班教室、讲台前，讲台上贴着白纸的玉树捐款箱把2010年的春夏推到十班面前。玉树地震后的捐款箱放到讲台前，曹蔚负责登记名单和金额。

有人捐得多，有人只能捐零钱，曹蔚压住起哄，让每个人把名字写清楚就好。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

曹蔚也在这段日子里显出自己的位置。组织能力强，成绩普通，班级杂事常落到他手里，越忙越稳。群像不是名单，群像是每个人都有自己的难处。

“别报金额，自己写在纸上就行。”曹蔚说。

“我只有硬币，响得很。”王子越说。

“硬币也是心意，别笑。”常俊凡说。

“我来排队，别挤讲台。”段璟说。

“捐款不是比赛，认真就够了。”陶建国说。

捐款箱放在讲台上，外面贴着白纸，纸边被胶带压得很平。

那天教室比平时安静。玉树地震的新闻从广播里听过，再落到讲台前，就变成每个人手里的零钱。

曹蔚把登记表放在旁边，名字一行，金额一行，写得很慢，怕把谁看轻。

有人捐纸币，有人捐硬币，硬币落进去时声音很清楚，几个人都下意识低头。

常俊凡把钱放进去，没有往登记表那边多看。她不想让善意也变成比较。

段璟排在后面，手心攥着钱，忽然觉得很多大事都不是离他们很远。

陶建国站在讲台边，少见地没有催作业，只提醒大家把队伍排好。

捐款箱收走以后，讲台上空出一块白印，像那天的安静还留在那里。

陶建国这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

曹蔚被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

常俊凡把硬币轻轻放进捐款箱，水房方向传来暖水瓶碰撞声。十班的热闹还在，只是每个人都比去年更会藏心事。

郭向阳在登记表上写下自己的名字，补课班楼道里有人喊让一让。老师从门口经过，原本散开的声音立刻往回收。

秦佳禾把讲台前的队伍往后理了理，窗台上的金银花茶凉了一半。这样的细节不会被写进红榜，却会留在很多人的夏天里。

何南星听见硬币响时低头看了一眼，讲台粉笔盒里只剩半截白粉笔。这点动作很小，却把那天的压力落到了实处。

邓桥把硬币轻轻放进捐款箱，后墙的错题纸被风掀起一角。旁边的人没有追问，翻书声把难堪遮住。

余小满在登记表上写下自己的名字，竞赛名单的纸角已经微微卷起。常俊凡看见了，没有替谁下结论，只把题号重新写清楚。

曹蔚把讲台前的队伍往后理了理，新华书店塑料袋在手里沙沙响。段璟后来才知道，未来最早就是这样从小事里露头的。

玉树捐款箱之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

大事落进教室时，没有变成口号，而是硬币、纸币、名单和一次安静排队。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，讲台上贴着白纸的玉树捐款箱还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
