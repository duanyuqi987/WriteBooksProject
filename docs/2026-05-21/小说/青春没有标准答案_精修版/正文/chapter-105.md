# 第105章 树莓摊旁边



时间：2010年五月中旬，补课班下课后



地点：补课班楼下、树莓摊旁边



卷次：第五卷：未来在黑板上倒计时



2010年五月中旬，补课班下课后，补课班楼下、树莓摊旁边，付小川手里一张揉皱的补课收据把2010年的春夏推到十班面前。补课班楼道挤满学生，付小川拿着收据不敢回家，因为数学补课费又涨了。

他嘴上说自己以后一定考个贵的分，转身却问段璟值不值得继续补。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

付小川也在这段日子里显出自己的位置。贫嘴、讲义气，成绩忽上忽下，越害怕越会把话说得很轻。群像不是名单，群像是每个人都有自己的难处。

“这张收据比我数学卷子还刺激。”付小川说。

“你真不想补了？”段璟说。

“想补，又怕补了还这样。”付小川说。

“先看你上课有没有真听。”常俊凡说。

“补课不是买分，别把钱花成安慰。”马立群说。

补课班楼道窄，墙上贴着各科提分班的广告，纸边被学生书包蹭得发毛。

下课铃一响，楼道里全是汗味、墨水味和家长在楼下喊名字的声音。

树莓摊就在楼下拐角，甜味被热气一蒸，和烤红薯、尘土混在一起。

补课费、练习册、饭卡和公交钱，平时散在不同地方，这一刻像都堵在楼梯口。

常俊凡走得不快，手里抱着一摞题。她不是不累，只是很少让累先开口。

段璟在楼道里看见许多熟面孔，才明白大家所谓靠自己，其实都背着家里的钱和期待。

老师从楼梯口下来，提醒学生别挤。家长们往上看，像能从每个孩子脸上看出分数。

楼下树莓摊慢慢收摊时，补课班的灯还亮着，像县城夏夜里一块很硬的光。

马立群这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

付小川被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

付小川从来不让人看见自己做题时发呆，但补课费通知发下来的那天，他把那张收据在手里揉了三分钟。

收据是打印的，抬头红字：封丘县新思路教育中心春季补课班收费单。科目：数学（高二提高班）。课时：十六课时。费用：两百四十元。比上学期涨了四十块。

他坐在补课班最后一排靠窗的位置，把收据折成一个小方块，再展开，再折起来。纸面上的折痕越来越密，字迹被揉得有点模糊。下课铃响的时候他是第一个站起来的——平时他不是，平时他最后一个走，因为要帮老师收作业。

今天他没有帮任何人收作业。他抱着练习册挤下楼道，练习册在手臂上滑了一下，他把收据夹进数学书的扉页——夹在了他自己写的班级名单那一页。名单旁边有他估算的每人每月饭卡花销，那些数字他从来没给任何人看过。

补课班楼道里是五月中旬的固定气味：汗味、墨粉味、楼下树莓摊飘上来的酸甜味混在一起。高一学生先涌出去，高二的跟在后面，书包蹭着墙壁，墙上的补课广告纸角被蹭得发毛。有个物理提分班的广告被撕掉了一半，"保提二十分"的"保"字只剩上面半边——像一个承诺被剪了顶。

段璟在楼道口等他。付小川平时下楼时嘴不停——"今天的题太简单了""再讲我都能睡着了""你怎么又只打一个菜"。今天他从四楼下到一楼半句话没说。

"你怎么了？"段璟问。

付小川把手插进校服口袋，收据在口袋里被揉成了球。"补课费涨了。上学期两百，这学期两百四。"

"涨了四十？"

"四十块够我两周的素菜。"付小川的声音不像平时了。没有贫，没有调侃。像一把已经钝了的刀。

段璟没有说话。两个人走到树莓摊旁边，付小川在泡沫箱前停下看着那些透明盒里的红树莓。看了大概十秒钟。然后他转身对段璟说了一句段璟从来没听他说过的话：

"我今天不想做数学。我就想站一会儿。"

段璟站在他旁边。两个人就这么在树莓摊边上站了大概两分钟。摊主的铁盒子压在塑料袋上，纸币角被风吹起来又被压下去。路口的电动车铃声、家长叫名字的声音、补课学生互相喊"明天见"的声音——这些声音围在他们周围，但没有一种声音能替付小川回答那个问题：两百四值不值。

"你觉得值不值得继续补？"付小川终于开口了。他看着树莓摊上的透明盒子，没有看段璟。

"你上课听了没？"

"听了。每一次都听了。"

"那补了有用吗？"

"我数学从年级一百多补到了年级前五十。但我不敢说是因为补课还是因为我自己做得多。"付小川把手从口袋里拿出来，收据球掉在地上，他捡起来，展开。纸面上的字已经看不太清了——折痕太深，墨粉脱落。"我家里不宽裕。我妈在超市收银，一个月一千二。这两百四是我家一个星期的买菜钱。"

段璟第一次听见付小川说自己家里的具体数字。付小川从来不说——他帮人充五十块饭卡、多打一份红烧肉、帮王子越往捐款箱里放硬币，但从来不说自己口袋里有多少。他的慷慨不是因为没有负担，是因为他知道负担是什么样。

"我以后一定考个贵的分。"付小川说，语气恢复了一半平时的贫，但段璟听得出这句话里的东西——不是在贫，是在给自己定锚。

常俊凡从补课班出来时手里抱着物理竞赛训练卷。她看见两个人站在树莓摊旁边——付小川没说话、段璟也没说话——就在旁边站了一会儿。

"你补课费涨了？"常俊凡直接问。她看到付小川手里揉皱的纸。

"涨了四十。"

常俊凡没有说"那就别补了"或者"值得"。她从付小川手里拿过收据，展平，看了一眼上面的课程时间和内容。

"这学期补的是解析几何和立体几何。你的解析几何是强项，上学期就稳了。立体几何你偏弱——空间想象力不够。如果只为了立体几何补这个班，性价比不高。"她把收据还给付小川，"但你已经交了。这学期的钱用完了再决定下学期续不续。这学期先把立体几何补好——班已经在上了，就把课的效果最大化。"

付小川把收据重新折好，这次没有揉成球——折成了一个四方的方块，放进校服口袋。

"你说得对。"他说。

"我没说对，我在分析事实。"常俊凡说完往学校方向走了。她的步子还是一样——不快，每一步都踩实。

段璟和付小川跟在她后面，三个人前后隔着几步的距离。走到校门口时，付小川忽然说了一句话："数学是唯一能让我觉得公平的东西。物理靠直觉，化学靠背，语文靠悟性。数学不靠这些——你做对是因为你算对了，做错是因为你算错了。没有中间地带。"

段璟想起付小川的错题本——四十七道数学错题，每一道都标了错因。付小川的数学好不是因为聪明，是因为他把每一道错题都当成了必须查清的账。

"那你以后会考数学系吗？"段璟问。

"会。考师范大学的数学系。毕业以后回来当老师。"付小川说这句话时没有贫嘴。他知道师范的学费低，知道封丘需要数学老师，知道自己毕业以后能帮家里减轻负担。他的"贵的分"不是想离家远，是离家刚好够近、又刚好够稳。

晚自习前，常俊凡把付小川的数学错题从错题墙上抽出来重新看了一遍。她发现付小川的立体几何错题占了他最近错题的七成。剩下的三成是解析几何——但他的解析几何分数一直在年级前排。补这个班确实性价比低——因为他的弱项只有一个立体几何，而这学期其他十七道题的知识点他本来就已经会了。

她在错题墙上付小川的名字旁边写了一个括号：（立几专项，七成偏弱）。

付小川路过时看到了。他看了一眼那个括号，没有说谢谢，而是在下面加了一行字：Day 1。目标立几七月前结清。

段璟的物理错题本今天翻到了Day 17。多过程分析做了第一道题——斜面接抛体接水平面，过程分三段，每段单独受力分析，然后用运动学连接。做完全对。

他把结果写在错题本上，又翻到扉页。扉页上已经记了三条笔记：陶建国的"捐款不是比赛"，付小川的"她不是要你追"，付小川今天说的"方向比名次重要"。现在他又加了一条——不是别人说的，是他自己总结的：Day 17多过程全对。从做到对的路径可以被复制。

晚自习后，付小川一个人去了充值处。他往饭卡里充了三十块，然后把补课收据从口袋里拿出来。纸已经皱得不成样子，但折痕不再增加。他把收据放进了一个透明文件袋——他专门买来放数学笔记的，现在多放了一张纸。透明袋的封面写着：数学-错题与费用-2010。

他回到教室时，日光灯已经熄了一半。段璟在走廊上等他。两个人没有说话，一起往校门走。

门口树莓摊早就收了。泡沫箱盖子合上，铁盒子已经不在上面。五月中旬的夜风从黄河大堤方向吹来，麦子快熟了，风里有谷壳的味道。

"你下学期还补不补数学？"段璟问。

"看这学期的效果。如果立体几何补上去就不续了——没有立体几何拖后腿，我自己能把分稳住。"付小川把书包带往肩上提了提，"但我以后一定考个贵的分。"

他第二次说这句话的时候，段璟觉得那不再是一句狠话，而是一个已经算好的方程。

回宿舍的路上两个人经过了城关镇路口。树莓摊早就收了，但街口还有灯光——是新华书店的灯。店已经关门了，但橱窗里的灯没关，展示台上放着一本封面反光的《数学竞赛入门》。付小川在橱窗前停了大概五秒钟。

"那本书我看过。书店老板娘说卖六十八块。我没买。"付小川说。

"你想买吗？"

"想。但六十八块够我在食堂多吃十七顿早饭。"

段璟想起自己父亲在电话里说的"树莓卖两块五一盒"。他伸手在校服口袋里摸了一下——饭卡硬硬的边角硌着手指。他想说"我借你钱去买"，但没说出口。因为他知道付小川的逻辑——有些东西不是买不起，是优先级排不上去。那本书排在了十七顿早饭后面。

"等我以后当了老师，我自己教室里的书随便学生看。"付小川说。他转身离开橱窗，把书包带又往上提了一下。那个动作段璟太熟悉了——从高一到现在，付小川每次做完一个决定就会提一下书包带，像一个无声的句号。

第二天早自习前，段璟发现付小川的错题本扉页上多了一行字。字很小，在"数学-错题与费用-2010"那一行的下面，用铅笔写着一句话：妈妈在超市收银，收据能打对就行，不一定要收据好看。

段璟没有问他这句话是什么意思。但他在旁边用橡皮擦了一小块空白，写下了一个数字：Day 18。旁边加了一个向左的箭头，指向付小川的那句话。

郭向阳在早自习时从走廊跑进来，手里拿着地理笔记本。他在树莓摊旁边待了一会儿——不是为了树莓，是为了看别人家电动车后座上的地图挂绳。他回来以后把一张手绘的"上海地铁线路简图"贴到了后墙黑板报上——贴在邓桥的展馆介绍旁边。

"你那个展馆介绍的最后一句话"，郭向阳指着邓桥写的那行小字：以后想亲眼去看一次，不是从网上搜的图片，"这个我也想过。我想的是坐在地铁上从车窗外面看上海。"

付小川看了一会儿那张简图。地铁线路在郭向阳的笔下像一根又一根血管，从市中心往郊区延伸。

"你先把数学考到及格，再画地铁。"付小川说。但他这次接过了郭向阳的笔，帮他把一条画歪的线校正了一下。

下午自习课，常俊凡在错题墙上更新了最后一批分类。她把何南星的数学步骤跳（已解决）归档到一栏，把邓桥的概念混淆（正在改进）归档到另一栏，把段璟的受力分析（Day 17全对）贴到已完成区，把付小川的立体几何（Day 1）贴到最新关注区。

贴完后她退了一步，看着这面墙。四月底到现在，几十张错题纸，十几种颜色的标记，蓝色卡纸压住的透明胶没有翘。她把手上的粉笔末拍了拍，回到靠窗后排的座位上，翻开物理竞赛训练卷，继续做下一道题。

段璟在第四排抬头看了看错题墙。墙上每一个名字后面都连着一张进步曲线。那个"31"还写在竞赛名单旁边——他的位置还没到，但他的方向没有错。

放学铃响的时候，段璟合上错题本。封面上的"物理错题"四个字已经被磨得快看不清了，但扉页上的四条笔记还很清楚。他把错题本放进书包，站起来，没有急着走出教室。



树莓摊旁边之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

树莓摊旁边的甜味压不住补课费的现实，大家第一次把学习和钱明明白白连在一起。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，付小川手里一张揉皱的补课收据还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
