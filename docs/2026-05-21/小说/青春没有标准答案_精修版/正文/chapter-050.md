# 第050章 操场边的单词



时间：2008年十二月下旬，早操后



地点：操场跑道边和十班教室门口



卷次：第三卷：雪落在两栋楼之间



2008年十二月下旬，早操后，操场跑道边和十班教室门口，李慕白贴在掌心的小单词条先把这一天按进了十班的日常里。早操后的冷风里，十班一边跑步一边背单词，月考后的松散被重新拧紧。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

段璟发现自己总把注意力分给常俊凡，英语默写却不会因为喜欢而放过他。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看李慕白贴在掌心的小单词条，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

段璟也在这一天被推到前面。段璟的特点是敏感、自尊、偏理科，喜欢常俊凡却还在学边界。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“这个词我早上背了五遍，还是忘。”李慕白说。

“忘了就第六遍。”马立群说。

“跑步背单词，嘴里全是冷风。”梁一帆说。

“冷风也比空白卷面好。”常俊凡说。

“我现在知道了。”段璟说。

操场边的树叶已经掉得差不多，跑道旁剩下几块没化干净的雪，踩上去会发出轻微的碎响。

早操结束后，大家喘着气背单词，声音断断续续，像每个人都在和自己的懒劲拔河。

李慕白把单词卡贴在掌心，一抬手就能看见，可真正默写时还是会漏掉一个字母。

段璟背到一半忘了词义，脑子里却浮出常俊凡刚才说话的语气，他立刻有点懊恼。

常俊凡不看他，只把自己的单词按词性分成几列，一列一列往下过。

马立群说背书不是喊给风听。她让大家回教室后立刻默写，错一个抄三遍。

梁一帆说冷风都从嘴里灌进去了，英语应该自己冻牢，结果第一个被罚抄。

这一早晨没有浪漫，只有冻红的手指、写错的单词和一排被冷风吹得发抖的人。

操场边的单词没有悬在空处。操场边还没化尽的雪和冻红的手指，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，操场边的单词带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

马立群进门时不一定立刻说话，鞋底在水泥地上一停，操场边的单词带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在段璟快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。操场边的单词越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。李慕白贴在掌心的小单词条让他想说话，又逼他先低头看题。

段璟被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，操场边的单词留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。段璟路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为操场边的单词翻身，谁还在被窝里背单词，旁边人都听得见。

操场外的风掠过树梢，背单词的声音断断续续，像一群人各自给自己打气。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理操场边的单词里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：李慕白贴在掌心的小单词条、一只冻红的手、一句老师点名、一块还没扫干净的雪。

回教室的路上，付小川从后面追上来，把手里的单词条往段璟肩上一拍。"你刚才跑步的时候嘴在动，我以为你在背单词——结果你是在重复'她说了冷风也比空白卷面好'。你完了。"

"我没重复。"

"你重复了三遍。我跑在你后面，全听见了。"付小川把单词条叠成一个小方块塞进裤兜，"你这样月考怎么办？英语默写的时候脑子里全是常俊凡的语气。"

段璟没说话，推开教室门。教室里的暖气片咣当响了一声，靠窗的人喊"关门关门冷风都进来了"。他侧身挤进去，常俊凡已经坐在座位上，正把单词条上的词按词性往笔记本上誊。她的字很小，蓝黑笔，名词一列动词一列，错过的词旁边画一个空心三角。

马立群站在讲台上拍了两下手。"回座位，拿出英语本。默写——现在开始。第一个词：environment。"

全班一阵翻本子的声音。段璟捏着笔，第一个字母写下去，第二个字母就犹豫了。他记得常俊凡刚才说"冷风也比空白卷面好"时的语气——不是安慰，是陈述。像在说一个已经被她验证过的事实。

"第二个：temperature。"

付小川在段璟斜后方发出一声很轻的"嘶"——那是他忘了拼写的标志性声音。梁一帆把本子往旁边挪了挪，试图看清宋乔的答案，被宋乔用胳膊肘挡了回去。

常俊凡写得很快。笔尖几乎没有停顿，只在"opportunity"这个词上停了一瞬——她把两个p写成了一个，用透明胶粘掉，重新写。段璟用余光看见她粘错字时手腕很稳，胶带撕下来时纸面没有毛边。这个细节让他走了神，自己的"opportunity"少写了一个字母。

"交换批改。"马立群说。

段璟把本子往左传。他右边是周明澈，周明澈接过他的本子时低头看了一眼，嘴角动了一下但什么也没说。段璟知道自己的默写本不太好看——字大、行歪、还有几个涂黑的疙瘩。

常俊凡把本子传给宋乔。宋乔翻开以后说了句"你怎么全对"，常俊凡没有接这个夸奖，只说"有一个词我粘过，你看一下拼写"。宋乔又看了一眼，说没错。常俊凡这才把笔帽套上，拧了一下手腕。

课间操的汗还没干透，教室里混着暖气、粉笔灰和校服布料潮潮的味道。后排有人把窗户开了一条缝，冷风立刻钻进来，前排女生集体缩了缩脖子。马立群走过去把窗关上，说谁再开窗就多抄十遍单词。

段璟的默写本传回来时，错题旁边被周明澈用红笔标了正确的拼写。周明澈的字很小很方，像一个一个码好的砖。段璟看着那几个红字，忽然有点感激——不是感激周明澈帮了他，而是感激这个世界上有人能把字写清楚，不让你猜。

午饭铃响的时候，付小川第一个站起来。"食堂今天有白菜炖粉条，去晚了只剩汤。"

"你就知道吃。"宋乔说。

"我还知道英语默写我错了七个。"付小川把默写本卷成筒状塞进书包外侧，"但白菜炖粉条能治愈我。"

常俊凡没有急着去食堂。她把单词条夹进笔记本里，又把笔记本放进透明文件袋。这些动作做完以后，她才站起来，把椅子推进桌底。段璟假装在整理书包，等她走出教室门之后隔了几秒才跟上去。

食堂里热气腾腾，玻璃窗上蒙了一层白雾。打菜窗口排了两条队，左边是白菜炖粉条，右边是土豆烧鸡块。白菜队比土豆队长了三倍——不是白菜更好吃，是白菜便宜一块钱。

段璟排在常俊凡后面隔了四个人。他看见她端着饭盒走到靠窗的位置，坐下，先把筷子摆整齐，再开始吃。她吃饭不快，一口菜一口饭，中间偶尔抬头看一眼窗外。窗外是操场，操场上还有几个没回教室的学生在踢一个瘪了一半的足球。

付小川端着白菜粉条坐到段璟对面。"你又在看她。"

"我在看窗外。"

"窗外有她。"

段璟把一块土豆塞进嘴里，不说话了。付小川也不再追问，只是把自己碗里的粉条往段璟碗里拨了一半。"多吃点，下午还有两节英语课。你不能再走神了。"

下午英语课，马立群把默写成绩报了一遍。没有念具体分数，只说"有人进步，有人退步，退步的人自己知道是谁"。段璟知道说的是自己。他的默写成绩从上周的八十分掉到了六十五分。

常俊凡没有回头。但她在下课后经过段璟桌边时，把自己的单词本放在他桌角。"名词那几页你可以先看。动词的先别抄，等我改完错题再给你。"

段璟拿起那个本子。封面上写着"英语单词分类整理"，下面的名字只有一个字：常。字迹和她的性格一样——不花哨，不犹豫。

教室外面的风停了。傍晚的光从西窗斜进来，把黑板上的粉笔字照得有点发灰。值日生开始擦黑板，粉笔灰扬起来，在斜光里慢慢飘。段璟把常俊凡的单词本放进书包最平整的那一层，拉上拉链时，拉链卡了一下——是书包用了两年，拉链齿磨平了几颗。他蹲下来用手捋了捋拉链，重新拉上了。

晚饭后段璟没有回宿舍，直接去了教室。冬天黑得早，走廊里的声控灯被他踩亮了两盏，第三盏闪了一下没亮。他从书包里拿出常俊凡的单词本，翻到名词那几页。

她的分类方式不是按字母，是按场景。教室相关的词一页：blackboard、chalk、desk、curtain。操场相关的词一页：playground、track、whistle。还有一页专门写了她常弄混的词——environment和government写在一起，旁边画了一个箭头，箭头尾端写着"ment不是men"。段璟笑了一下。这个笑很轻，教室里只有他一个人，没有人看见。

他把自己的默写本翻开，开始抄那些标了空心三角的错词。抄到第三遍时，教室门被推开了。

进来的是何南星。他端着一个搪瓷杯，杯口冒着热气，看见段璟愣了一拍。"我以为教室里没人。"

"我也以为。"

何南星坐到自己座位上，把搪瓷杯放在桌角。段璟闻到一股金银花味——封丘县城很多人家都自己晒金银花泡水，说是清火。"你喉咙不舒服？"段璟问。

"有点咳。我妈说冬天干燥，泡一杯带着。"何南星翻开练习册，又回头看了段璟一眼，"你在抄单词？"

"嗯。今天默写不太好。"

"你平时英语不差啊。"

"今天走神了。"

何南星没有追问为什么走神。他只是把自己的练习册翻到段璟正在抄的那一页，指了指上面一道题。"这道完形填空，我今天错了一半。"

两个人安静地写了一会儿。教室里只有笔尖划过纸面的声音和何南星偶尔吹茶的声音。窗外的路灯亮了一盏，橘黄色的光从玻璃窗透进来，在课桌上铺出一块暖色的四边形。

快到七点的时候，陆陆续续有同学回来了。马晓彤抱着一摞作文本进来，往讲台上一放，拍了拍手上的粉笔灰。白子昂从后门进来，羽绒服拉链拉到最高，一进门就喊冷。有人开始传下午发的数学卷子，有人接水，有人凑在一起对答案。教室里慢慢热闹起来，恢复到晚自习前那种七分喧闹三分安静的状态。

常俊凡进教室时手里提着一个热水瓶。她给宋乔倒了一杯，又给旁边几个女生倒了一圈。经过段璟桌边时她停了一下，看了一眼他桌上摊开的单词本——是她那本。

"动词的先别看。"

"我知道。我在抄名词。"

"抄到哪了？"

"environment。你写的那行小字我看到了——ment不是men。"

常俊凡嘴角动了一下，不太像笑，更像是对自己的标注被发现了这件事有一点意外。"那个词我错过四次。"

"你也会错这么多次？"

"我又不是字典。"

她提着热水瓶继续往前走。段璟低下头，把environment抄了第四遍。这一次他没有写错。

晚自习第一节是数学。数学老师姓孟，戴一副银框眼镜，讲函数的时候习惯用手在空气里画抛物线。段璟听了一半，另一半心思在常俊凡的单词本上——她已经把动词部分改完了，原来画空心三角的地方换成了对钩。他不知道她什么时候改的，可能是下午课间，也可能是在宿舍里打着台灯改的。

第二节晚自习是自由时间。后排有人讨论中午食堂的白菜炖粉条为什么比上周咸了，有人问明天的课表，有人趴在桌上睡觉。段璟把常俊凡的单词本合上，装进书包，然后把自己的物理练习册打开。他强迫自己做了一页题——不是为了复习，是为了证明自己还能把注意力收回来。

下晚自习的铃声响了以后，付小川站在门口等段璟。"走，回宿舍。"

两个人并排走过操场边。操场在晚上几乎全黑，只有跑道旁边的路灯照着一小圈地面。早上的雪已经化得差不多了，但背阴处还有几块硬邦邦的白。付小川把手揣在袖子里，走着走着忽然说："常俊凡今天把单词本给你了。"

"借我抄一下。"

"她从来不借别人。上次宋乔问她借，她说'等我整理完'。但今天你默写六十五分，她就给你了。"

段璟没接话。他不知道怎么说。

"我不是在逗你。"付小川难得正经，"她信你。被信的人不能辜负那份信。"

宿舍熄灯以后，楼道里安静了几分钟，然后开始传来各种声音。有人翻塑料袋找零食，有人用被子蒙着头背单词，隔壁宿舍有人在讲白天的一个笑话——声音压得很低，但笑声还是能透过墙壁传过来。

段璟躺在上铺，把常俊凡的单词本放在枕头旁边。他没有打开手电筒，只是躺着，听外面的风从操场方向吹过来。他想起早操的冷风，想起马立群说"把声音放出来"，想起常俊凡说"冷风也比空白卷面好"。

他把手伸到枕头旁边，摸到单词本的封面。纸面有一点凉。他在心里把今天错的词又过了一遍：environment有两个n，opportunity有两个p，temperature的拼法是tem-per-a-ture。

然后他翻了个身，把被子拉到肩膀上面。明天还会有早操，还会有英语默写。但明天他应该不会再走神了——或者说，至少不会再因为走神漏掉字母。

操场边的单词之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

段璟没有因此变成特别醒目的人，可在段璟眼里，段璟从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，操场边的单词把青春拉回现实：会不会就是会不会，不能靠心事补分。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
