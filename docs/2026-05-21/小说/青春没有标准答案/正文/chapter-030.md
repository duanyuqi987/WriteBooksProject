# 第030章 第一张座位表



时间：十月第一次月考前



地点：高一十班教室后墙



卷次：第二卷：十班的第一张座位表



十月第一次月考前，高一十班教室后墙，贴在后墙的正式座位表把这一天压得很实。奥运海报终于被换下，后墙空出一块给月考目标。临时座位变成正式座位，十班的关系被固定成一张表。楼道里的脚步、窗外的广播和桌面上的表格交叠在一起，每个人都被流程推着往前走。

段璟和常俊凡仍是前后桌，但距离被规则重新命名。段璟没有把这件事想得太轻，也不敢想得太重。他刚从初中那个窄窄的夏天走出来，手里还攥着录取通知书留下的折痕，眼前却已经换成了新的楼道、新的课桌和一群还叫不上名字的人。

常俊凡处理新环境的方式仍然很稳。她不急着和谁显得熟，也不故意疏远谁，先把该交的表交了，该看的通知看完。她的清醒不是冷淡，而是知道一个女生一旦在陌生班级里被过度议论，很多普通动作都会变得不普通。

夏清和也在场。主动帮两个同学换桌脚不平的座位，避免刚开学就闹别扭。十班的群像就是从这些小动作开始的：有人先说话，有人先沉默，有人忙着找宿舍，有人已经开始替全班操心。

“正式座位先坐一个月，谁有意见写纸上，不要在下面嘀咕。”陶建国说。

“写纸上算匿名吗？”付小川说。

“你字我一眼认出来。”陶建国说。

“你还坐后面？”常俊凡说。

“表上这么写。”段璟说。

座位表贴上去时，大家比看成绩还热闹。段璟把手里的笔握紧又松开，笔帽碰到指节，他很快停下来。

有人高兴离黑板近，有人抱怨靠窗风大，有人先看自己旁边是谁。常俊凡把桌角的纸推齐，抬头听了一句，又低头继续做自己的事。

段璟看见自己名字在常俊凡后面，第一反应不是开心，而是心里一紧。夏清和在旁边接了一句，声音不大，却让周围几个人都抬了头。

常俊凡看完表，只把书本往桌洞里重新排了一遍。陶建国敲了敲讲台，粉笔灰浮起来，刚散开的声音又慢慢低下去。

夏清和帮石磊修歪桌腿，几下就把快吵起来的气氛压平。后排有人挪凳子，凳脚擦过地面，前排的同学下意识把卷子按住。

陶建国说座位不是命，但坐在哪里都得把题写明白。楼道里有脚步声过去，夹着脸盆碰到墙角的一声轻响。

十班在这张表上第一次有了清楚形状：谁靠窗，谁靠门，谁总能把话题带偏，谁安静得容易被忽略。食堂方向飘来热气，白菜汤和馒头的味道隔着窗户也能闻见一点。

教室窗台上有没擦干净的粉笔灰，阳光斜斜照进来，把每张桌子的划痕都照得清楚。有人把饭卡翻来覆去看，像看久一点余额就会多出几块钱。

陶建国不爱说漂亮话，他讲规矩时只说三件事：按时、认真、别给别人添乱。段璟想接话，话到嘴边又改成低头翻书，把指尖压在页角上。

宋乔的声音总能把僵住的气氛拉活，周明澈则负责把被拉散的气氛重新收回来。常俊凡把要做的事写在草稿纸背面，字不大，却一行一行排得很清楚。

常俊凡整理桌洞时很安静，课本按科目排好，资料袋贴着桌洞左侧，像先给自己搭了一个小秩序。窗外操场有人吹哨，声音短短一截，很快被教室里的翻书声盖住。

段璟坐在她后面，能看见她翻页，却看不见她表情。这种距离让他既庆幸，又比以前更不会说话。傍晚校门口的三轮车慢慢过去，车斗里铁盆互相碰着，响得很生活。

十班不是一下子熟起来的。它先是一排排不合脚的桌椅，再是一声声试探的借笔、借纸、问路。黑板右上角还留着上一节课的日期，值日生擦得太急，留下浅浅一片灰。

第一张座位表这件小事落下来时，十班每个人都被推到自己的位置上。贴在后墙的正式座位表放在那里，看似普通，却让段璟意识到高中生活已经不是暑假里的想象。门口有人喊了一声报告，声音在走廊里撞了一下，才落进教室。

段璟身上还留着初中留下的笨拙。他想和常俊凡说话，又怕自己的熟悉太冒进；想表现得自然，又总在最简单的动作上露出紧张。常俊凡的水杯贴着桌角放，杯盖拧得很紧，像她处理事情时的分寸。

常俊凡的稳定不是天生没有波动。她只是习惯把波动先收起来，等事情处理完，再用自己的方式消化。段璟把草稿纸翻到背面，背面有一道没擦干净的演算痕迹。

夏清和一开口，旁边的人也跟着动起来。有人接话，有人笑，有人低头找表格，教室一下子不再只剩两个人的沉默。宋乔低头翻笔袋，翻出两支颜色不一样的笔，又把其中一支推给同桌。

陶建国的规矩不总是温柔，却让十班从开学的混乱里慢慢有了边界。边界一出现，人物才知道自己能往哪里站。周明澈把名单压在练习册下面，怕风再把纸吹到地上。

高一开头的日子很碎。交表、领书、找宿舍、排队打饭、问水房、记课程表，每一件都不大，却能把人磨得很真实。付小川小声学老师咳嗽，被旁边人用胳膊肘顶了一下，才收住笑。

段璟以前以为重逢会有一句很特别的话，到了封丘一中才知道，真正的重逢常常被挤在人群里，只来得及点一下头。教室最后一排靠着扫帚，扫帚毛散开，像还没适应新位置的学生。

常俊凡不喜欢被别人替她解释。她说话短，是因为她把很多判断都放在行动里：表交没交，题改没改，东西有没有放好。窗外树叶被风吹得翻白，光落在课桌上，一格一格地移动。

宋乔的热闹、周明澈的稳、付小川的贫、沈曼青的冷静，都在帮十班变厚。段璟越看见这些人，越知道自己不能只盯着一张课桌。段璟听见自己肚子叫了一声，赶紧咳嗽，把那点尴尬盖过去。

食堂窗口的热气、宿舍楼道的脚步、早读铃声和晚自习灯管，把那些心事一层层压回生活里。常俊凡从书页里抽出一张小纸条，看完后没有笑，只把它夹回原处。

封丘县仍然在他们身后。家长的叮嘱、县城小街的文具店、黄河大堤的风，都没有因为他们进了一中就退场。陶建国的皮鞋从过道走过去，鞋底带着一点操场边的细土。

北京奥运的余温像一层浅浅的亮色，落在开学后的教室里。它提醒他们外面有更大的世界，但眼前仍然要先把课本翻开。叶书兰路过门口时往里看了一眼，手里抱着一摞还没批完的作文本。

十班的第一阶段关系还很脆。谁和谁能开玩笑，谁不喜欢被议论，谁成绩好但怕退步，这些都要靠一次次小场面试出来。马立群的英语书页边卷着，她用手指一压，整排读书声就跟着抬高。

段璟开始明白，靠近常俊凡不是把她看得更久，而是学会在她需要边界时停住，在她需要帮忙时把手伸出去。胡向南把旧报纸折成两半，放在讲台角上，油墨味淡淡散开。

常俊凡也开始重新认识段璟。他不是只有敏感和暗恋，也会在被提醒以后改，会在玩笑越界时皱眉，会把错题本翻到新的页。魏守诚在黑板上画线时不说废话，粉笔断了半截，也只是换一根继续。

校服会沾墨水，饭卡会余额不足，座位表会把熟人安排得过近，也会把想说话的人隔得很远。宿舍那边传来一阵铃声，很快被楼管阿姨的钥匙声压下去。

一中的教室并不神圣。桌腿会晃，窗户会漏风，黑板擦会掉灰，粉笔会断。正因为这些不完美，少年人的心事才有地方落下去。水房地面有一小片水渍，路过的人都绕了一下，谁也没大声提醒。

段璟那时还不会说喜欢。他能做的，是多带一支笔，递一张纸，或者在别人开过分玩笑时把话题挡回去。饭盆碰到桌边，发出一声钝响，旁边的人下意识让开半个位置。

常俊凡那时也不会把所有感受都摊开。她会用一句事实替代情绪，比如先交表，比如先看题，比如先把袖口洗干净。段璟把袖口往上卷了一点，露出手腕上一道浅浅的笔印。

十班慢慢有了声音：早读声、起哄声、老师敲讲台声、饭盆碰到桌面的声音。每一种声音都比空泛抒情更能让人相信他们真的在那里。常俊凡写字时手腕很稳，落笔不重，却能把每个横画压平。

座位一旦定下来，很多关系就有了形状。前后桌、同桌、斜前方，这些词比同班更具体。梁一帆从窗边往操场看，被前桌挡了一下，只好把视线收回来。

课本翻久了，页边开始变软，扉页上的名字也从陌生变成每天都会看见的东西。沈曼青抬头问了一个问题，问完就低头记答案，完全不管别人有没有看她。

段璟写名字时笔尖停了一下，因为他听见常俊凡在前面拉开笔袋。罗雨桐把纸边撕得很整齐，递出去时还顺手把桌上的碎纸捏走。

常俊凡不喜欢桌面乱，课本边缘要对齐，水杯放右上角，草稿纸放左侧。蒋一诺把错题本摊开，一边听一边在题号旁边画了一个小小的三角。

付小川把书堆歪，被陶建国从讲台上点名，说你那不是书山，是事故现场。陈柏言的笔记按日期排好，翻页时能看见每页左上角都写着天气。

沈曼青坐得很直，听规则时不插话，等别人问完才补一句关键问题。顾南音小声哼了半句歌，发现旁边人看她，立刻把声音压进喉咙里。

梁一帆总往窗外看操场，身体像被球场拽着，英语书却摊在桌上装无辜。马晓彤在清单上划掉一项，笔尖停了停，又补上一项新的。

十班的笑声来得快，被老师压下去也快。新班级的分寸，就是在这一次次收放里长出来。郑秋实把眼镜往上推，镜片上沾了一点雾，很快又散开。

段璟有时能看见常俊凡在题旁画小方框，那是他从初中就记住的动作。田野把馍掰成两半，先吃小的那半，大的那半留到最后。

靠窗的位置风大，靠门的位置人来人往，谁都能找到一点不满意，也都只能先坐下。林舟把自行车钥匙从口袋里摸出来，又想起还没放学，只好重新塞回去。

陶建国站到讲台边时，刚才还散着的声音立刻低下去。老师的规矩、提醒和偶尔的沉默，把十班从一群新生压成一个真正要运转的班。学生们未必马上喜欢这种压迫，但他们都知道，没有这些规矩，开学的慌乱会把每个人冲散。赵远航说话时总想把地名带出来，像每条路他都比别人熟一点。

食堂也在慢慢成为十班的一部分。白菜豆腐、馒头、稀饭和饭卡余额，把很多人从各自的家里带到同一张长凳上。段璟第一次端着餐盘找位置时，才发现同班关系不是教室里才发生，它也发生在谁往旁边挪了半寸的时候。闻佳把肥皂盒扣上，盒盖没扣严，她又认真按了一下。

宿舍楼里的夜晚让人更快认识彼此。谁家里送来咸菜，谁打电话时声音放低，谁睡前还背单词，谁把想家藏在玩笑里。这些细节不会出现在座位表上，却比座位表更早让一个班有了温度。孙佳怡把电话卡夹在书里，书页合上时边角露出一点蓝色。

常俊凡不是段璟青春里的奖品。她有自己的节奏、压力和判断，她会被老师要求重写作文，也会在成绩条前沉默。段璟越是靠近她，越需要承认她不是为他的成长而存在的人。陆晨把作文纸折起一角，被叶书兰看见后又慢慢压平。

段璟也不能只躲在暗恋里。他要适应住校，要面对月考，要学会和男生宿舍的玩笑保持距离，也要一点点看见十班里其他人的难。他开始变得真实，正是从不再把所有情绪都推给常俊凡开始。方晴说完自己的看法，教室安静了一秒，随后才有人小声接话。

封丘县的生活没有因为他们进了一中就消失。黄河大堤、城关镇小街、金银花、树莓汽水、文具店和校门口公交，仍然一层层围着他们。所谓高中生活，不是从县城里拔出来，而是在熟悉的县城里换一种身份继续长大。韩雨把围巾塞进桌洞，毛线露出一截，被同桌轻轻推了回去。

北京奥运留下的余温也没有立刻散。它挂在后墙海报上，响在校园广播里，被胡向南讲进班会，又被学生带回饭桌和宿舍。大时代没有压住小日子，它只是让小日子多了一点更远的回声。孟晓露低头算饭钱，算到最后叹了口气，又把零钱重新数一遍。

夏清和从旁边递过来一张纸，纸角被汗浸得发软，没人道谢得很郑重，却都把这点帮忙记在心里。

有人从后门探头问这里是不是十班，靠门的同学抬手指了指黑板，又很快低头继续整理书。

常俊凡把橡皮屑拢到草稿纸一角，起身倒进门后的纸篓，回来时顺手把歪掉的凳子扶正。

段璟把桌洞里的书按语数外分开，书脊抵住桌板，发出很轻的一声响。

陶建国把名单夹在教案里，走到窗边看了一眼操场，又转身提醒最后一排把声音放低。

楼道里有人拖着脸盆跑过去，盆底敲着台阶，清脆得像把住校生活提前敲响。

食堂窗口的队伍往前挪一寸，后面的人就跟着挪一寸，饭卡在掌心里被攥得发热。

校服袖口还带着新布的硬，写字时蹭过桌沿，留下几粒不明显的白灰。

窗外广播忽然响了一下，又被人调低，只剩一点断断续续的尾音贴在玻璃上。

男生那边有人小声报分数，报完又装作不在乎，眼睛却一直盯着别人的反应。

女生那边翻书更轻，纸页一页页过去，偶尔夹着一句很低的提醒。

段璟想问一句话，最后只把笔帽扣紧，声音小得连自己都觉得多余。

常俊凡没有回头，只把下一道题的题号圈住，像把混乱的一天先按住一个角。

前排有人借透明胶，后排有人找订书机，一件小东西传过三张桌子，回来时已经沾了粉笔灰。

周围的说话声一阵高一阵低，像水房里忽冷忽热的水，谁都不能完全控制。

段璟把视线落在课本页边，不敢停在常俊凡身上太久，怕被旁边的人看出端倪。

常俊凡抬手把碎发别到耳后，动作很快，快到段璟只能记住一个模糊的轮廓。

黑板槽里堆着断粉笔，有白的，也有半截黄色的，值日生擦完黑板后手指全是灰。

窗台上的搪瓷杯被风吹得轻轻一碰，杯盖响了一下，又被上课铃声盖过去。

付小川在后排憋笑，周明澈回头看他一眼，他立刻把笑咽回去，装作认真翻书。

宋乔从女生那边递来一张纸条，只写了两个字：别慌。常俊凡看完，折好塞进书页里。

楼下卖饭的师傅开始收盆，铁勺敲着桶沿，一声一声传到三楼，像提醒他们天已经黑了。

县城的傍晚落得很快，窗外的树影一暗，教室里的灯就显得更白。

有人小声算还剩几天月考，有人装没听见，手里的笔却比刚才握得更紧。

常俊凡把错题旁边的步骤补完整，写到最后一行时，才轻轻呼出一口气。

段璟看见她放下笔，也跟着把自己那道没想明白的题重新读了一遍。

这样的日子没有谁宣布开始，它只是从一张卷子、一顿晚饭和一次点名里慢慢铺开。

十班的墙上贴着课程表，纸角被风掀起又落下，像很多还没说出口的话。

陶建国经过门口时没有进来，只从窗外看了一眼，教室里的声音便自动矮了一截。

段璟这才发现，班级的规矩有时不靠喊，而靠大家心里同时绷起的一根线。

常俊凡把水杯往桌角挪了挪，给旁边同学让出半本练习册的位置。

这半本练习册的位置不大，却让那天的教室多了一点能让人坐稳的余地。

风从门缝里钻进来，吹动前排女生的试卷边，常俊凡伸手按住，纸才没有翻过去。

到这一天结束时，第一张座位表留下的不是一个夸张转折，而是一串能摸到的东西：桌面的划痕、饭卡的余额、楼道里的脚步声、宿舍熄灯后的低语，以及常俊凡把文件袋抱稳的动作。

段璟后来想起高一开头，最先想起的也不是某句漂亮话，而是这些琐碎的场景。它们让十班从名单变成教室，从教室变成一群人，又从一群人变成他无论走到哪里都绕不开的青春。
