# 第089章 录取通知书旁观

所属部卷：第二部：六月以后
时间：七月下旬
地点：镇上邮政所

## 诗歌插页 P089：原创现代短诗

那时我们都以为，
结束会有清楚的门。
后来才知道，
门只是普通下午的一阵风。
有人走向远处，
有人学着把远处还给远处。

邮政所的织毛衣阿姨叫周秀兰，在这间邮政所工作了二十一年。她看过的录取通知书不下千封——每年七月下旬，那些红色信封会像候鸟一样飞到她的柜台上。

"今年的毕业生里，最急的是那个叫程砚的。"她跟旁边修车摊的大叔聊天，"一天打三个电话来问。我说小伙子你考了五百多分怕什么，肯定有学上。他不信，第二天又打。"

"他通知书到了吗？"

"到了。省城工学院。他接过信封的时候手抖得比我这个老太婆还厉害——我织了二十一年毛衣，手都没他抖。拆开看完以后深呼吸了一口，然后对着天花板说了一句'老子有学上了'。声音大得把旁边寄包裹的一个老太太吓了一跳，以为他在打架。"周秀兰学程砚的语气学得惟妙惟肖——脖子一梗、下巴一抬。修车大叔笑得直拍大腿。

"还有那个陆允川——就是每天早上来问的那个瘦瘦的男生，走路不抬头看人，永远看着脚下的那个。"周秀兰放下毛衣针，拿起凉茶喝了一口，"他通知书到的时候，我特意把他叫住，让他当场拆。他没哭，也没笑，就是很认真地一个字一个字看完。看完以后把通知书折好放回信封。那个动作很轻，好像怕把纸弄皱。走的时候跟我鞠了个躬。不是点头——是鞠躬。"

"这孩子稳。"修车大叔说。

"不是稳。是慎重。他把这份通知书看得很重。"周秀兰重新拿起毛衣针，"还有一个女生，叫叶澄的，考了六百多分。拆信封的时候面不改色，看完点了点头——就三下——然后把通知书收回去说谢谢阿姨。从头到尾没多一个表情。但我知道她心里是高兴的，因为她走的时候鞋跟在地面上磕了一下——不是绊的，是踮了一下脚尖。""你连这个都注意到了？"

"我在柜台后面坐了二十一年，别的本事没有，看人的本事还是有的。"

同一天下午，林曼在她家楼下碰到了夏知遥。夏知遥刚从邮政所回来，手里拿着那个红色信封，额头上有一层薄汗——七月底的中午在外面走路，不流汗是不可能的。

"建筑系？"

"嗯。五年。"夏知遥把通知书晃了晃，信封在空中扇出一阵小风，"比我预期的长一年。我妈说五年太久了，我说建筑就是五年。然后她说'五年出来你都二十三了'，我说二十三怎么了，她说'二十三不好找对象'。我就没再接了。"

林曼笑了。"那你妈妥协了？"

"没妥协。她说'那你好好学，别学到一半又转行'。这就是她的让步方式——不承认失败，但把话题转向你的责任。"夏知遥坐在楼下的长椅上，从兜里掏出一包纸巾擦了擦额头的汗，然后把通知书拆开给她看，"你看这个报到须知——上面写着要自带绘图工具。我昨天在淘宝上看了一下午的绘图板。"

"你已经开始准备了？"

"嗯。反正闲着也是闲着。"夏知遥把通知书收好，抬头看着林曼，"你呢？你的汉语言文学什么时候报到？"

"九月三号。比你晚。"林曼在她旁边坐下，"知遥，我问你一件事。"

"你问。"

"你收到陆允川的点赞了吗？"

夏知遥愣了一下。"什么点赞？"

"你发朋友圈说录取通知书到了，他给你点了赞。"

"我没注意。"

"你没注意是因为你没去找。你要是找了，第一个就能看到。"林曼把胳膊肘搭在膝盖上，看着远处的路口，"他从来没评论过你的朋友圈，但每次都会点赞。从高一开始就这样。"

"你想说什么？"

"我想说，你们俩之间的交流方式真的很奇怪。他点赞，你不回复。你在走廊里跟他说'那挺好的'，他记了三年。你们明明都在互相观望，却谁都不往前走一步。"

夏知遥把绘图板淘宝页面关掉，手机屏幕黑了。"不是不想走。是发现走了以后可能会撞到墙。与其撞墙，不如站在墙的两边各自建自己的房子。"夏知遥把电话收起来，"而且说实话，我自己都不知道去了新环境以后会变成什么样。五年——五年可以发生很多事情。我自己都没搞清楚自己五年后什么状态，怎么去给别人承诺。"

"你这话说了等于没说。"

"对。因为我确实不知道。"夏知遥笑了一下。这次笑跟之前不一样——不是无奈的那种，是承认自己不懂的那种，"建筑要学五年。五年后他在哪个城市、我在哪个城市、我们各自在干什么——我连这个学期要学的课程都不清楚，怎么去预测五年后的事。"

林曼忽然觉得夏知遥不是在回避问题。她是在认真思考——只是答案不是陆允川想听的那种。

"你这个比喻是从哪学的。"

"自己想的。"夏知遥站起来，把通知书夹在胳膊底下，"我去买绘图板了。你要不要一起？"

"我买什么？"

"书店。汉语言文学不用看书？"夏知遥拉了她一把，"走了。趁太阳还没下山，早点买完早点回去。"

林曼被拉起来的时候，手里那支冰棍棒差点掉了。她低头看了一眼——冰棍已经吃完了，只剩下木棍上印的两个字："谢谢惠顾"。

两人一起往街上走。路过邮政所的时候，夏知遥往里看了一眼——柜台上空空的，周阿姨正在织毛衣，旁边那杯凉茶已经喝到只剩茶叶渣。那些红色信封已经全部飞到了该去的人手里，一个都没留。夏知遥没有停下来，只是走路的步子慢了半拍，然后又恢复了正常。
