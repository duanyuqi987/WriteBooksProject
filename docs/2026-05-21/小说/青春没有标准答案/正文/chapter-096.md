# 第096章 世博新闻



时间：2010年三月下旬，晚饭后的新闻时间



地点：高二十班教室、食堂门口



卷次：第五卷：未来在黑板上倒计时



2010年三月下旬，晚饭后的新闻时间，高二十班教室、食堂门口，小卖部电视里滚动的上海世博新闻把2010年的春夏推到十班面前。上海世博的新闻从小卖部电视里传到教室，十班第一次把远方和自己的未来放在一起说。

郭向阳兴奋地讲城市、展馆和志愿者，段璟却发现自己连下次月考的物理都还没稳住。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

郭向阳也在这段日子里显出自己的位置。爱看新闻，地理好，喜欢把大城市讲得很近，成绩却被数学拖住。群像不是名单，群像是每个人都有自己的难处。

“上海世博开了以后，新闻肯定天天播。”郭向阳说。

“你先把数学卷子开了。”付小川说。

“远方可以看，眼前也不能空着。”常俊凡说。

“我眼前最远的是物理最后一题。”段璟说。

“新闻不是让你们走神，是让你们知道题外面也有世界。”胡向南说。

小卖部电视声音不大，围着的人却不少。有人看新闻，有人只是借着看电视多站一会儿。

上海、展馆、志愿者这些词从电视里飘出来，落到封丘一中的校服袖口上，显得又远又近。

郭向阳讲得兴奋，手指在地图册上划来划去，像一条线能从县城直接划到黄浦江边。

段璟听着，心里却先想到自己的物理卷。远方很亮，眼前的空题也很白。

常俊凡没有笑他想得远。她只是把新闻里听到的一个词写在草稿纸边上，然后继续改错题。

晚自习开始后，电视声被关在小卖部里，教室里只剩笔尖和纸页声。

胡向南说世界很大，但大不是用来吓人的，是用来提醒你们别只把眼睛贴在分数上。

这句话没有让题变简单，却让十班那天的晚自习多了一点看不见的风。

胡向南这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

郭向阳被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

李慕白把地图册翻到上海那页，树莓摊旁边的纸盒被太阳晒软。窗外风吹了一下，纸角轻轻动了动，像有人把话收回去。

邱可可在草稿纸边上写下世博两个字，黑板报边角的透明胶又翘起来。十班的热闹还在，只是每个人都比去年更会藏心事。

许念安抬头看小卖部电视的方向，家长的电动车铃声从校门外传来。老师从门口经过，原本散开的声音立刻往回收。

魏然把没做完的物理卷压在新闻下面，饭卡机短短响了一声。这样的细节不会被写进红榜，却会留在很多人的夏天里。

段璟把地图册翻到上海那页，水房方向传来暖水瓶碰撞声。这点动作很小，却把那天的压力落到了实处。

常俊凡在草稿纸边上写下世博两个字，补课班楼道里有人喊让一让。旁边的人没有追问，翻书声把难堪遮住。

郭向阳抬头看小卖部电视的方向，窗台上的金银花茶凉了一半。常俊凡看见了，没有替谁下结论，只把题号重新写清楚。

世博新闻之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

世博没有让他们立刻看见世界，只让他们意识到世界并不会等他们把每一道题都想明白。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，小卖部电视里滚动的上海世博新闻还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
