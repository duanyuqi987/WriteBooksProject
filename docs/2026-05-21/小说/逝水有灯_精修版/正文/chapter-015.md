# 第015章 理科名单多年后

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：新分班后的理科教室

## 诗歌插页 P015：原创七言小诗

分班旧事各西东，
半纸名单映眼红。
年少不知离别重，
错将一瞥作长风。

多年后陆允川重写理科名单，先把自己写得很委屈——写到他看名单时心跳飙到一百二，写到他在课堂上看她的时间加起来能把一本教科书翻十遍——写完从头读，删得只剩下一个发现：高一的时候他们同班。同班而已，不是同一个座位，不是同一道题的搭档，甚至没有真正意义上单独说过一句话。同班只是一个空间事实：她坐在前排，他坐在后排，两人共享墙上那张课表和黑板旁边那张座位表，仅此而已。

多年后他在同学群里看到一张翻拍的旧照片——有人从旧手机里导出了一批高一时期的模糊影像。其中一张拍的是分班后理科一班的第一次班会。照片左下角露出他半个肩膀，右上角有她的侧脸。两个人被教室中间一整块黑板隔开，之间站了十几个人，全都在笑——陈老师正在讲笑话，全班都在笑，他也在笑，她也是。他放大照片看了很久，发现他们同框的第一张照片竟然是在一句老师讲的笑话里。

那年暑假他回老家，顺路去了一趟县一中。公告栏换成了电子屏，滚动播放着"热烈庆祝XX同学获得XX奖"。当年那块贴名单的玻璃公告栏已经拆掉了，原位置改成了一面白墙，上面挂着"明德格物"四个泡沫字。他站在那面墙前面，想起当年他站在同样是这个位置，等着周围的人散开——等了几分钟，就为了看清名单上的一个名字。

他在那个位置停了一分钟，旁边有个学弟匆匆路过，夹克衫里面的校服露出来——校服的款式和十年前一模一样，只是颜色从蓝白换成了灰白。他回头看学弟的背影消失在楼梯口，心想那个年纪的人，书包里背着的除了课本和作业，大概还有一整个公告栏的名字等着去确认。

程砚后来约他吃过一顿饭。两人坐在大学城旁边的小吃街里，桌上堆着烤串的竹签和啤酒瓶。程砚的脸比高中时圆了一圈，说话方式倒是一点没变。

"你还记得分班名单吗？"程砚把一根竹签扔进空盘子里，"我记得你那天站了好久。名单被风吹得啪啪响，你纹丝不动。我还以为你在研究物理竞赛通知。"

"我看的是分班名单。"
"我知道你看的是分班名单。我那天就在你后面三米的地方，看着你从上往下数名字——数到中间的时候手指头停了一秒，然后收了回来。我当时就猜到你看到了什么。"
"那你为什么不说？"
"为什么要说？你那个表情——"程砚想了一下措辞，"像一个刚中了奖又不敢确认的人。我当时觉得说了就破坏了。"

程砚的这句话让他停了很久。原来旁观者看他，比他想象中要善意得多。他以为别人都在嘲笑他的笨拙，但其实别人只是不知道该怎么在不伤人的前提下，帮一只蜗牛从壳里伸出触角。

叶澄在他的修改稿上批了一句话。她读了他的初稿——很多章之后的那一段关于分班的重写。她在旁边用红笔写了四个字："分寸到了。"

"什么叫分寸到了？"
"就是你终于不再把同班当成伏笔了。她分到理科一班，是因为她理科好。和你没有关系。你把她写成因为她理科好才和你同班，这中间的区别你能感觉到吗？"

陆允川感觉到了。十七岁的时候他会在日记里写：命运让我们同班。三十多岁的时候他知道那不是命运——那是教务处的分班算法，按姓氏首字母和上次大考排名交叉排序。她和他的姓氏首字母分别是X和L，差了一个声母的距离。他们在同一张名单上不是因为命运用了心，而是因为拼音排序刚好把他们放进同一个区间，像两枚骰子滚进同一个格子，纯属概率。

但即便是概率，他也保留了对那段概率的感激。因为在那个概率的间隙里，他每天都能经过她的座位，听她回答陈老师的提问，看到她在课间低头改错题，偶尔在食堂里隔两张桌子看见她端起汤碗吹热气。这些都不是命运的安排，只是同一个班级带来的天然利益——但天然利益也是利益，他在那个十六岁的春天实实在在地享受过。

改完这一章之后，他出门散步。小区外面有个男孩骑着共享单车摇摇晃晃地过去，后面的女孩扶着车座说了句"慢点"。他从他们旁边走过，想起有一年春天他也学过骑车——他爸在后面扶着车座，他回头看了一次，发现他爸已经松手了，他在摔下去之前自己骑了大概五米。他现在想起这件事，觉得这段记忆正好能放在这一章的末尾：学会骑车不是学会了怎么不摔跤，是学会了摔跤之后重新上车。

同班也是这样。不是他最终有没有说出口，而是那一年他每天走进同一间教室，坐在同一个角落，看见同一个人——这个事实不需要任何后续就已经足够成立。
