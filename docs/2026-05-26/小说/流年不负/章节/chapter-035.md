# 第035章 补课

时间：2011年8月初

地点：学校

卷次：卷二：窗边人影

八月五号。学校开始暑假补课。

班主任在期末那天说——"说是自愿的。"然后顿了顿。"不来也行。"

但大家都来了。四十来个人的教室里坐了三十七八个。少了三四个——大概是家里有事或者真没来的。

顾临风坐的是原来的位置。倒数第三排。靠过道。

教室里的桌椅没变。黑板上学期最后一节数学课的板书还在——二次函数的对称轴公式被人擦了一半，剩了一半。粉笔灰在地上积了一小层。

人坐齐了。班主任进来。"暑假作业交了没有。没交的明天交。"

没人说话。

第一节课是数学。老师翻开书。说——"我们来复习一下高一的内容。高考不会因为你们刚过完高一就减题。"然后开始讲函数。

风扇嗡嗡响。窗外银杏叶被太阳烤得发亮。蝉叫成了一片。教室里闷热。有人拿本子扇风。

第二排靠窗。

苏晚晴坐在那里。

跟之前一模一样的位子。

马尾——还是那个马尾。发绳是浅绿色的。书包上小鱼挂件在拉链上晃。桌面很干净——她每天会擦一遍。

顾临风从进门的第一秒就看到了她。

暑假两个月没见。她好像没什么变化。头发没长也没短。皮肤还是那个颜色——不白也不黑。看书的姿势还是那样——背挺直。

他又可以每天看到她了。

早读课。语文。

陈老师让大家背课文。教室里一片嗡嗡的背书声。她也在背。嘴一张一合。声音不大——被周围的背书声盖过了。但透过声音的缝隙能听到她的音调——跟朗读《荷塘月色》时一样的沙哑。

上午第二节课后是跑操。

补课期间的跑操没那么严格。体育老师不在。各班自己跑。高一三班在操场东侧集合——跟以前一样。女生在前，男生在后。

顾临风跑在队伍后面。

她的马尾在前面晃。左一下。右一下。阳光照在发绳上发亮。跑了三圈。她没停。跟得很紧。第三圈末她弯腰系了一次鞋带——红色帆布鞋——然后又追上队伍。

跑操结束。回教室的路上她跟许若琳并排走。许若琳在说暑假的事——"我爸妈带我去看海了。坐大巴坐了我六个小时。"

"什么海。"

"舟山。那个沙滩全是人。"

"有贝壳吗。"

"有。我给你带了一个。"

许若琳从口袋里掏出一个东西。一个白色小贝壳。拇指大小。

苏晚晴接过来。在太阳底下看了看。收进了口袋。

"谢谢。"

然后笑了一下。

顾临风从她们旁边走过。隔了一条过道。

中午食堂。菜跟上学期一样——土豆丝、肉末茄子、西红柿鸡蛋。他打了两个菜——土豆丝和西红柿。找个位置坐下。抬头看到她跟许若琳在第三排桌子。她吃包心菜。米饭吃了大半。

下午自习课。教室里安静。外面蝉在叫。窗户开着——风吹进来把窗帘吹鼓起来。

她在做物理题。暑假作业上的。从第一页开始——她应该还没做完。她做完一题翻一页。速度不快——物理大概还是让她头疼。

顾临风在日记本的暑假部分加了一页——用铅笔写。

"补课第一天。她又坐在窗边的位置。两个月没见。她没变。马尾。发绳淡绿色。小鱼挂件还在。每天又能看到她了——这种感觉像是黑白照片突然染上了颜色。"

他写到这里停了一下。把"黑白照片突然染上了颜色"这句划掉——太过分了。不是冷白描该写的。

改成"两个月来第一次觉得哪儿都不想去。"

然后合上本子。

放学后。顾临风从宿舍走到教室拿忘了的书。教室里灯关了。只有走廊的灯从门口照进来。

他走进去。看到她的座位——窗边、空着、第二天她还会坐在那里。

窗外银杏叶在路灯下泛黄。绿色的、还绿着。但叶缘有一点点黄了。

她在的时候教室是满的。她不在的时候教室是空的。一样的桌子、一样的窗、一样的黑板。但不一样。

他拿了书。走出教室。

走廊尽头。八月晚风吹过来。有淡淡的桂花香——不对，桂花还没开。大概是洗衣粉的味道。

宿舍楼下有人在弹吉他。弹的是《老男孩》。跑调了。唱了两句然后停了。

顾临风回到宿舍。赵一鸣躺在下铺翻漫画。

"明天要不要去打篮球。"

"好。"

"你这次别走神了。"

顾临风笑了一下。

"不会。"

赵一鸣翻了一页漫画。

"暑假过得怎么样。"

"还行。"

"看你心情不错。"

"还行。"

顾临风把书包放在床头。翻身。闭上眼睛。宿舍灯灭了。天花板上的那道裂纹印在眼皮上。

明天还能看到她。

这个念头让他觉得今晚可以睡个好觉。
