# 第118章 相亲

时间：2018年2月（农历正月初三）

地点：青阳县 顾临风家

卷次：卷六：平行轨道

正月初三。家里来了亲戚。舅舅、舅妈、表姐带着两岁的儿子。屋里一下多了五个人——一下子从安静的两个人变成了热闹的七个人。小孩在地上爬来爬去，穿着厚厚的棉衣，爬得不太利索，把茶几下面的瓜子翻了一地。瓜子壳蹦到沙发底下——他妈弯腰去捡。

顾临风坐在沙发角上。沙发是布艺的——灰蓝色的罩子洗得有些发白。手里端着水杯——白瓷杯，杯底有水垢。没喝，水凉了。从刚才倒满到现在还没碰过。

舅妈问："临风——工作了？"舅妈坐在沙发另一边——手里剥着橘子。橘子皮撕成小片掉在茶几上。

"代课。还在读书。"

"研究生？"

"嗯。楚州师大——九月开学。"

"好呢好呢。有出息——你妈供你这么多年没白供。"

他点了点头。把水杯放下——杯底磕在茶几上，闷闷的一声。

他妈在厨房和客厅之间来回走。端菜，摆筷子，招呼人——"你们坐你们坐——"——嘴一直没停。额头上一层薄汗。

到了饭桌上才安静一小会儿。圆桌铺了一次性塑料桌布——白底红花——边角用碗压着。舅舅说青阳县的房价涨了——城东新楼盘开盘八千一平米。"比三年前涨了一半——"他夹了块排骨。"咬牙也得买——不买还得涨——现在不给孩子买婚房就是害他。"舅妈接话——"首付得四十多万——咱们几个兄弟姐妹凑一凑——能出来的。"舅妈的声音带着商量的口吻。

舅妈说表姐的单位今年招了新人，表姐被调去带徒弟。"带新人——累得很。什么都不会。从头教——Excel、收发邮件、做报销单——"表姐在旁边叹了一声补充道："现在的应届生连Excel都不会用。我说你大专读了什么。她说'读了电子商务'——我说电商你Excel都不会用——怎么卖东西。"

大家都笑了。顾临风也笑了一下——很轻。筷子在盘子里夹了一颗花生米。

然后舅妈忽然转向顾临风。"你给你妈省点心吧——二十三了还没对象？我们家表姐二十三岁孩子都生了——你呢——"声音里带着那种过年被催婚的惯常语调——半开玩笑半认真。

"他整天忙学习。考研、代课、写论文——没心思谈对象。"他妈替他接话。筷子在盘子里给他夹了一块排骨。"他回来家我问他——他说'忙'——这孩子从小说自己忙——"

"女的不等。"舅妈正色道。"好姑娘谁都抢着要。你等呀等的——到了三十岁——着急了——哭都没地方哭。再说你现在条件也不差——工作稳定了——学历也高了——找个好对象正好。"舅妈把手里的橘子皮放进烟灰缸——然后又拿起来放进桌边的垃圾桶。

他妈放下筷子。看着他。"听你舅妈的。"

他说："知道了。"声音不急。但也不缓。

舅妈又絮叨了几句——"现在小姑娘条件可高了——你要主动出去认识人——不要整天闷在书堆里——"然后话题转回孩子身上——表姐的孩子把勺子扔到地上——不锈钢勺子叮当一声掉在瓷砖上。小孩哇哇哭起来。表姐弯腰去捡——顺手在围兜里掏出一张湿巾擦孩子的嘴。

顾临风趁机去厨房添水。拿起水壶——热水瓶里空了——打开煤气灶重新烧。火苗跳起来窜在壶底——蓝色的。壶底有一层水垢。他妈跟进来。关门——没完全关上，留了一条缝。

"临风。"她的声音放低了——放低到刚好被客厅的电视声和小孩哭声盖过。

他正在倒水——水壶提起来——壶嘴里流出的水在杯子里转着圈。

"你张阿姨说——那姑娘姓苏——苏晚晴。也是青阳的。在楚州一小正编——教语文。你下个月回楚州见一面——只是见个面——喝茶——不是逼你结婚。你张阿姨说她长得好看——性格也好——"

他手里的水杯没有端稳——不是没端稳。是手轻微地缩了一下——像手指忽然碰到烫东西。但水杯里刚才倒进去的热水——反而让他缩了一下——本来稳的、温的，被从壶里加满的热水一冲——那热沿着杯壁传导到指腹。但他真正缩手的原因不是热。是他听到那个名字——他已经准备好好几年的名字——从妈妈嘴里说出来。以"相亲对象"的方式。

水晃了一下——洒了一点在灶台上。灶台上有一层陈年的油渍——水洒上去结成几颗透明的珠子在油渍表面滑着。

他抽出抽纸。擦灶台。擦得很快。抽纸在油渍上擦过去——油变成了几条道子。

"姑娘叫苏什么来着——苏晚晴。名字挺好听的。这名字有一点点像那种古诗词里的——她妈大概也挺有文化的——"他妈接着说。"你张阿姨说过年前问你意见——我说他肯定同意——你肯定同意——你一直想找个老师——人家就是老师——"

顾临风的手停在灶台上。手里的抽纸湿了——热水和凉水混在一起把纸洇透了。抽纸上洇了一小片深色的水印。

"苏晚晴。"

这个名字从别人的嘴里说出来。在正月初三的厨房里——混着炸带鱼的残留油味、烧水壶底的水垢味、抽纸擦灶台的沙沙声。像一张压在抽屉底下八年的旧纸条被翻出来。摊平。用手指抚平上面的折痕。然后——一个字一个字地——读出来。"苏——晚——晴。"不对——这三个字被别人说出来的时候——音节跟音节之间的停顿跟他默念时的位置不同。他说的时候第三个字是轻声——像飘着的。而妈妈说出来——第三声——实实在在落在地上。

他站在厨房里。一只手拿抽纸——抽纸上的水渍慢慢扩大。另一只手扶着灶台边沿——手指的关节有点白。

"不认识。不认识的。"他的声音干干的。像嗓子被什么擦了一下——"不"字出口后喉咙沉了一下。

"不去。我不想去。"他又加了一句。

"为什么？"

"不想相亲。我自己找。"

他妈以为他就是逆反——这个年纪该有的反应——不想爸妈插手感情。笑着拍了他一下——手掌拍在他胳膊上，轻的。"找？你去找去——你要能找到我早抱孙子了。"说完转身打开冰箱——拿出下午要用的肉——"中午吃排骨吃腻了——下午做西红柿牛腩——"

他没回嘴。

把擦过灶台的纸揉成团——扔进了垃圾桶。垃圾桶里有蛋壳和橘子皮和那团湿透的抽纸。

饭吃完。

大家移到客厅喝茶。舅舅在翻手机——大概在看房价新闻。舅妈在看电视里重播的春晚小品——赵本山的旧作，小品里在说"这个可以有——"。表姐在哄孩子睡觉——小孩趴在她肩上，眼皮一开一合的。

顾临风坐在饭桌边上没起身。桌上的盘子快要空了——红烧排骨还剩最后两块孤零零的肉——骨头已被夹干净的。油麦菜剩了三片叶子——浸在蒜粒打的汁里。酱焖豆角剩了一小堆——凉了一两个小时，油结成了块。凉掉的藕盒——面糊壳从酥脆变成了软塌。碗里的米饭——没人吃了——硬了。

他望着那个凉掉的盘子。一个盘子。一个豆角堆。剩菜在盘子里排成某个形状——他盯着看。但看的是另一桌年夜饭。

苏晚晴。

她大概也在青阳过年。她家的除夕桌上——是她妈做的菜。红烧鱼——年年有余。炒青菜——先吃素再吃肉的习惯是她们家的。藕——大概也有藕盒。她爸大概也在加班——或者已经退休了。她妈大概也在桌上说："晚晴——你年纪不小了——该找对象了——二十三了——再过两年就二十五了——三十以前得结婚——"

她大概也说"妈——不急——我自己找——"

一模一样。

两个二十三岁的年轻人——在同一座县城过年。隔了几条街、一棵光秃秃的老槐树、那家关了门改招租的奶茶店、十四步。在不同的餐桌上吃着各自不同的妈妈做的菜。被各自不同的妈妈催婚。独自在心里回答同一个问题：不是不想找。是我喜欢的人你不认识。我也不知道该不该告诉你。因为我已经八年没跟她/他说过话了。

下午五点多。舅舅一家走了。门关上之后屋里安静了好几倍——只剩下电视机里的声音和厨房里的水龙头声。他妈去厨房收拾碗筷——碗盘在水槽里碰着瓷，叮叮当当的。

顾临风一个人坐在沙发上。电视开着——静音——因为遥控器在茶几的瓜子堆下——懒着去拿。画面上在播一档旅游美食节目——介绍一个西藏的酥油茶。酥油在锅里慢慢融化——黄色的——淌出来——倒在碗里。主持人端起来喝——表情很夸张——嘴张得很大——眉毛跳起来——无声的。

他拿出手机。解锁——主屏幕。打开微信。没有未读。往下滑——通讯录。在搜索框里按了一个"S"。自动补全。第一个——"苏晚晴"。头像已经从高中的那张——高三毕业照——换成了现在的——她站在楚州一小操场上，阳光很好，脸上一半在光里，一半在阴影。笑容收着的。跟那张在操场上张嘴大笑的照片——不一样。更安静，更日常。

他盯着看了一小会。点进去。聊天记录——空白的。这个号里——唯一的内容就是朋友圈那些他没有点赞的照片和偶尔的状态。他没有发过一条消息。她没有发过。两个人——互为联系人——却在对话框底下沉默着。

然后退出了。把手机放回口袋里。口袋里还有一颗大白兔奶糖——从教师节那天留到现在的。他一直没吃。

眼睛回到电视屏幕上。酥油茶的镜头已经过了。现在是拍藏民在跳舞——男的穿白色袍子，女的戴红围巾——音乐大概很热闹——节奏轻快的鼓点和拨弦里伴着一两声高亢的吆喝——但电视是静音的。所以画面上的人在无声地跳舞。女的转圈——裙子在空中展开。男的跺脚——靴子在地板上拍着节奏。他们笑着——看起来很开心。然后镜头切走。开始放广告——一个洗发水广告。

他的手指在腿上来回地动。食指在牛仔裤上画一个大圈——然后停下来。然后又开始画。
