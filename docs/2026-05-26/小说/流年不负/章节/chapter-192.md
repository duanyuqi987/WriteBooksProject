# 第192章 一中的教室

时间：2022年10月
地点：青阳县一中
卷次：卷十：不负少年

门卫换人了。不是当年那个矮矮的胖大叔了。现在是个年轻的——三十来岁，戴着眼镜。坐在传达室里看手机。他们走近的时候他抬起头。

"找谁。"

"以前的学生。回来看看。"

"哪一届的。"

"2013届。"

眼镜门卫想了想。大概对不上号。"登记一下。"递过来一个本子。顾临风在上面写了名字、身份证号、来访事由——"回母校参观"。

"进去吧。课间别上楼。上课了可以上去。"

"好。"

校园里。主干道。梧桐树。十二年没回来，梧桐树认不出了。树干粗了两圈——他伸手比了一下。十二年前他张开手掌能盖住树干的一半。现在两只手都合不住。树下的水泥地上有落叶——被扫成一堆一堆的。还没收走。落叶的边缘卷了——已经在地上躺了些日子。踩上去嘎嘣嘎嘣响。干燥的。碎的。

路过的学生穿着校服。红白相间的。跟他们当年不一样——他们那届是蓝白。红色显得更活泼。蓝色更安静。有一个男生边走边看手机——智能手机。大屏的。跟他们当年的翻盖机和滑盖机不一样。旁边的同学拍了他一下:"别被老班看到——他在后面。"男生把手机塞进裤兜里。动作很熟练——大概天天练。顾临风想起自己当年把手机藏在铅笔盒里的日子。铅笔盒是铁的——上课的时候偷偷打开一条缝看QQ。赵一鸣发的消息:"哥们你看苏晚晴没有——她刚才看了你一眼。""没看到。""你瞎吗。"

传达室前面的公告栏。课表换成了电子屏幕。滚动显示着:高二语文公开课、高三模拟考倒计时。倒计时的数字在跳——离高考还有248天。

他看了那个数字一眼。248天。十二年前他的倒计时是多少——忘了。大概差不多。

苏晚晴走在前面。步子不快不慢。经过操场的时候她往跑道那边看了一眼。跑道换成了塑胶的——以前是煤渣。红色跑道。白色标线。有人在跑操。一个班排成四列。体委在前面喊"一二一"。队伍跑过去。地上扬起来的是灰尘还是塑胶碎屑——太远了看不清。

女儿被顾临风抱着。手指指着操场上跑步的人。"跑、跑。"

"嗯。哥哥姐姐在跑步。"

"念念也跑。"

"等你再大一点。爸爸带你来跑。"

高三教学楼——现在不知道还叫不叫这个名字。楼门口挂的牌子写的是"弘毅楼"。以前叫"第三教学楼"。

楼道里还是那种味道。消毒水混合了粉笔灰。还有从厕所飘来的淡淡的氨水味。楼梯扶手是铁的。刷了银灰色的漆。漆面上有一些划痕——大概是学生用钥匙划的。楼梯拐角处的墙上有一块黑印——是以前垃圾桶放在那留下的。

上到三楼。楼道里很安静——在上课。走廊里的灯管换了——LED的。白光。不再是当年那种发黄的日光灯。但走廊的长度没变——从西头走到东头还是那么多步。他走了几步。步数跟当年一样——三十几步。多了一步——腿长了。

能听到从某间教室里传出来的讲课声。英语课。老师在带读单词。"environment——环境。e-n-v-i-r-o-n-m-e-n-t。"学生们跟着念。声音拖得很长——长到最后一个字母被拉成了——"t——"像在叹息。然后接着下一个词。黑板旁边挂着一个高考倒计时牌子——不是电子的。是纸的。上面写着"距高考还有248天"。数字是手写的——今天早上班长刚换过——减了一。

他找到了那间教室。

门口挂的班牌是"高三（3）班"。还是三班。门框上面的油漆翻新过。但门框的木头还是老木头——右下角有一道裂缝，用乳胶填过。他记得那道裂缝。

教室门关着。门上的玻璃窗里能看到里面。一个穿白色衬衫的年轻女老师在讲台上写字——白板笔在白板上写字，吱吱响。黑板上叠了一层白板。讲台换过了。课桌换过了——单人桌，桌面是蓝色的。不是他们当年的双人木桌。

他在门窗外站了几秒钟。

然后往后排看。

倒数第三排靠窗。空的。没有人坐。桌面上堆着书——一摞。旁边挂着一个书包——黑色的。

不是他的座位。但是是差不多的位置。

他又往前排看。第二排靠窗。

坐着一个男生。头发很短。戴着眼镜。桌上摊着物理书。他低着头在写东西——手指指着课本上的一句话，嘴里念念有词。窗外的银杏树在他肩膀上投下来一片晃动的光斑。

男生没有注意到窗外有人在看他。

苏晚晴也站在他旁边。透过门上的玻璃窗往里看。她的眼神停在了第二排靠窗那个位置。

那是她当年的座位。

也坐着一个人。一个男生。但十二年前坐着的是一个女生——扎马尾的，靠窗的，写字小指翘起来的。

那个男生忽然抬头看向窗外——大概感觉到了什么。和她的目光碰了一下。然后又低头继续看书了。对他来说只是一个陌生的女人在走廊里看一看。可能是某个老师。可能是谁的家长。

但对站在走廊里的这个人来说——那扇窗。那个位置。窗外那棵银杏树。还有秋天从窗缝里吹进来的风。

课间铃响了。

教室里的学生呼啦一下站起来。椅子腿刮地板——那个声音和十二年前一模一样。

他们退到走廊另一边。让学生们出去。那个坐在第二排靠窗的男生也出去了。经过他们身边的时候看了一眼。然后走了。

走廊里都是人。说话声。笑声。有人从书包里掏出一包辣条。有人在水龙头前排队。

苏晚晴站在窗边。

她把手放在窗台上。手指碰到的瓷砖是凉的。瓷砖的边角缺了一块——被什么东西磕掉的。她低头看了一下缺角——三角形的。平滑的——大概是某届学生用椅子腿撞的。

窗外是银杏树。操场。远处的水塔。视野和十二年前一样——除了银杏树的树冠高了。树冠遮住了水塔的下半截。

银杏树比当年高了。高一那年这棵树才到二楼。现在是高三——树冠伸到了三楼窗户外面。伸手就能碰到——叶子就在窗外。一伸手就能摘下。叶子正在变黄——一半绿一半黄。有些叶子已经全黄了。有虫子从叶子上爬过——留下了一道弯弯曲曲的白线。风一吹，落了几片叶子。翻着跟头往下掉。掉到第二层的时候停住了——卡在了银杏树的枝杈间。摇摇晃晃的。

她站在窗边。看着窗外。银杏叶的影子在她的脸上移来移去。肩膀上也是。光影斑驳。从鼻梁滑到嘴角。又滑到耳廓。

然后转头看他。

"这棵树。高一的时候还够不着窗户。现在伸进来了。伸进来这么多——它大概也在看教室里坐的是谁。"

他看了一眼银杏。那根伸进窗户的枝条——最前端的新叶子还带着鹅黄的嫩色。已经十月了还在长。又看着她。窗框把她的侧脸框住了——和十二年前一样。只是脸上多了一点点细纹。眼角——笑起来的时候会有两根。嘴角——笑的时候出现的。下巴的线条比当年柔和了一些——从瓜子脸变成了鹅蛋脸。但这些纹路——在他的视线里——像柳树皮的纹。年轮一样的。好看的。

"进去吧。没人了。"

她拉着他的手。推开了教室的门。门铰链没上油——吱呀声很亮。跟当年晚自习后关教室门的声音一样。
