# 第214章 南方出租屋旁观

所属部卷：第五部：逝水有灯
时间：同一周
地点：南方城市出租屋

## 诗歌插页 P214：原创现代诗

隔壁只听见水声
听不见青春
一扇薄门后面
有人拧紧垫圈
也拧松自己

隔壁租户姓梁，在同一层住了两年。她只知道六零三新搬来的男生常加班，夜里回家脚步很轻，门口偶尔放着快递，纸箱上多半是书或打印纸。南方城市里，邻居之间的了解常常止于电梯点头。

那一周，她几次听见六零三的水龙头滴水。声音隔着墙传来，不大，却在夜里格外清楚。后来某天傍晚，滴水声停了，楼道里传来拧螺丝的细响。她出门倒垃圾时，正好看见陆允川蹲在门口，手里拿着一个旧垫圈，额头出了汗。

“修好了？”她随口问。

陆允川抬头笑了一下：“应该好了。再漏就找师傅。”

梁姐点点头，没有多问。她不知道这个年轻人屋里有多少旧照片，也不知道他曾在电脑前坐到凌晨。她只看见一个普通上班族，在周四傍晚修一个不值钱的小故障。

可生活本来就是这样把人往前推的。不是每一次成长都有见证人。更多时候，人只是在一间小屋里把水龙头修好，把湿衣服晾起来，把电脑合上，然后按时去上班。旁人看见的只是动作，动作却足够诚实。
