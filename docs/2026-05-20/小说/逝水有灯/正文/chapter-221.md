# 第221章 重回县一中之前

所属部卷：第五部：逝水有灯
时间：出差顺路回乡
地点：县一中翻新的校门

## 诗歌插页 P221：原创现代诗

校门换了颜色
我站在门外
先登记姓名
再登记来意
青春没有这一栏

陆允川回县城出差，会议结束得早。他拖着行李箱从酒店出来，手机地图提示县一中离这里只有两站路。他站在路口看了一会儿，最后还是上了公交。

车窗外的街道改了很多，原来卖文具的小店变成了药房，邮政所旁边多了连锁咖啡店。县一中的校门也翻新了，门头更高，电子屏滚动着优秀学生名单。陆允川站在门外，差点找不到当年进校的方向。

保安拦住他，问找谁。他说自己是以前的学生，想进去看看。保安让他登记身份证和电话，又问哪一届。陆允川报出年份时，自己也愣了一下。那个数字离现在已经很远，远到要靠登记本确认。

他在来访事由一栏写“校友回访”。四个字端正、客气，和他心里那些乱七八糟的旧画面完全不同。保安把访客牌递给他，提醒不要进教学区影响上课。

陆允川点头，把牌子挂到胸前。进门前，他回头看了一眼街道。曾经他以为重回这里会像打开一扇旧门，可现实先让他登记、刷证、遵守规则。这样很好。学校不是他的回忆仓库，它正在继续教别人读书。
