# 第153章 南湖问答之后

所属部卷：第三部：寒风里的花
时间：七月初
地点：南湖河边

## 诗歌插页 P153：原创仿宋小令

湖静晚风微。
三字纸边，问答各低回。
不敢再将人作钥，
先学空手向前归。

谈完后，叶澄先走了。她还有一份社团总结要交，临走前把那张写了三个词的报名表背面撕下来，递给陆允川。

“别拿这个去找她。”叶澄说。

陆允川愣了一下，“我没想。”

叶澄看着他。

他把纸接过去，改口：“我尽量不想。”

叶澄笑了一下，背着包走向图书馆。陆允川坐在长椅上，把纸折成四折，放进口袋。湖边路灯亮起来，水面把灯影拉长，又揉碎。

他打开手机，苏晚枝的聊天框停在上周的“好”。他看了几秒，退出，点开备忘录，写：道歉不是门票，改变不是筹码，等待不是要求别人回头。

写完后，他没有觉得轻松。相反，心里空了一块。那块空不是释怀，更像把原来堵着门的东西搬开之后，发现屋里很乱。

回宿舍路上，他给祁野带了一瓶冰红茶。祁野接过来，先看了看他，“怎么，今天没发疯？”

陆允川说：“暂时没有。”

祁野拧开瓶盖，“这就算进步。”
