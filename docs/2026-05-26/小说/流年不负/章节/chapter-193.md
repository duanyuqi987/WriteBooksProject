# 第193章 她坐过的位置

时间：2022年10月
地点：青阳县一中高三三班教室
卷次：卷十：不负少年

教室里没有人了。课间学生会去操场或者走廊。一般不会留在教室里。

她走到第二排靠窗的位置。站在那个男生的课桌旁边。桌上摊着物理书——《物理·选修3-4》。翻开的那一页讲的是波的干涉。书页的空白处有几行铅笔记号——频率、周期、波速。字迹很稚嫩，但也算工整。

她低头看了一会儿。

"这个单元我以前也做不好。干涉和衍射老是搞混。"

她用手指在课本上轻轻点了一下。频率的公式——f=1/T。

"我记得你帮我补过物理。"

他站在她身后。离了大概半米。

"不是补。我写了一道题的解法。放在你桌洞里。你没回。"

"我后来看了。"

"你看了。"

"高二下。物理力学的单元测试。有一道力学的题跟你写的解法很像。我想起你放到我桌洞里那两页纸。晚上回去翻出来看了。看懂了。"

"但你从来没提。"

"我不好意思提。你干嘛偷偷摸摸放。"

"怂。"

她笑了一下。很轻。

然后她拉开椅子坐下来了。坐在那个男生的位置上。对着窗外。

从窗口往外看。银杏树的分枝遮住了部分视野。透过树枝能看见操场的一角——红色的塑胶跑道。远处是水塔。水塔还是那个样子——圆筒形，顶上有一个红色的灯。灯白天不亮。晚上会闪。红的。绿的。红的。绿的。

"以前我坐在这里发呆。从没想过有一天你会站在窗外看着我。"

她看着窗外。说这句话的时候没有转头。

他走到她侧面。靠在窗台边。看着她的侧脸。窗外的光打在她的耳朵上。耳廓有一点亮。

"我也没有想过——有一天你会嫁给我。"

她说完了。然后站起来。把椅子推回原位。椅子的四条腿和地板之间发出短促的摩擦声。

"你以前坐在哪。"

"倒数第三排靠窗。"

"去看看。"

他们走到后排。倒数第三排靠窗。座位上坐着一个女生——短头发的，戴眼镜，正在看书。看到有人过来，她抬起头。

"这是你的座位吗。"苏晚晴问她。

"啊。是。"

"我以前坐在前面第二排。"

女生看了她一眼。大概在想这个人为什么跟自己说这个。

"这个位置看外面能看到什么。"苏晚晴又问。

"能看到操场和那边的水塔。还有银杏树。就是看不全——被前面那个楼的角挡住了一块。"

"对。以前也这样。"

苏晚晴弯腰。从后排透过窗户看出去。视角和记忆里的一致——操场占窗户的下面三分之一。水塔在那个位置。银杏树的树冠在左边。被前面教学楼挡掉了一个角。

她站直了。手在椅背上扶了一下。

"这个位置很好。"

"为什么。"

"因为可以看到全班所有人的后脑勺。"

"那你还能看到什么。"

"能看到你的马尾。"

她没说话。

"从左到右。从右到左。一个上午大概晃五十多次。"

"你数过。"

"没数。随便说的。但差不多。"

女儿被顾临风放在旁边的空座位上。她正用手拍着桌子——桌子腿有一点不平衡，拍一下摇三下。她嘻嘻笑。继续拍。

苏晚晴伸手把桌子按住。女儿抬头看她。然后不拍了。

"走吧。快上课了。"

"还有五分钟。"

"那就再坐五分钟。"

他走到自己当年的座位旁边。那个女生还坐在那里。他说"不好意思。可以坐一下吗。"女生站起来:"没事没事。"把桌上的书收进书包里。

他坐下来。

椅子对他一个大人来说有点矮——腿伸不开。膝盖碰到了桌子下面的抽屉。他把腿侧了一下。

从上往窗外看。视角一样。银杏树的叶子在风里翻动。操场上有几个女生并排走着。马尾晃着。

三分钟。

他从抽屉里拿出一本书——不知是谁留的语文课本。翻到扉页。上面没有名字。他把课本放回去。

站起来。正准备走。

低头看到桌面上有一行字。小字。铅笔。被人用橡皮擦过——只留下浅浅的凹痕。他低头凑近了看。

"顾临风。2010。10。"

他愣了一下。抬头看苏晚晴。她站在她当年的座位旁边。离他的位置有四排加两个过道。

"你看这个。"

她走过来。弯腰。顺着他的手指。

"谁写的。"

"不是我。我写在课本上的。刻在桌上的不是我的字。"

她仔细看了看。铅笔的痕迹已经很淡了。但能辨认——笔画很细。不是刻的。是用铅笔写的。而且写出了全名——"顾临风"。

不是他的字。

她认出了这个字迹。很小。很工整。横平竖直。小指翘起来写出来的字——笔尖和纸的角度比常人低。写出来的字有一个特殊的倾斜。

她自己写的。

高一那年。她在后排某个空桌面上写过他的名字。大概是某天下午放学后。教室里没人了。她打扫卫生。扫到他座位旁边的时候。鬼使神差地在桌上写了三个字。

写完之后用橡皮擦掉了。

但橡皮没有擦干净。

"是我写的。"她说。

他看着那三个字的残迹。看了很久。铅笔的石墨在木桌上留下了印子。十二年之后还能看出来。

"为什么写我名字。"

"不知道。那次考完数学。你考得特别差。数学老师当堂骂你。说你态度不端正。你脸红了。耳朵也红了。但你没有顶嘴。你在座位上坐了整整一节课没有抬头。"

"我记得。"

"你下课之后一个人去了操场。我在走廊窗户里看到你走在跑道第五道上。一个人走了两圈。我就回教室打扫卫生。扫到你座位的时候——就写了。不知道为什么要写。"

她把手指放在桌面上。摸了摸那三个字的凹痕。

"后来说了。"他说。

"什么。"

"我问你为什么写我名字。你现在说了。"

"嗯。现在说了。"

她把手指从桌上拿开。

桌面上的铅笔印在下午的光里几乎看不清了。但他们都看清了。
