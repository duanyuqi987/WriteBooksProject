# 第242章 写成小说

所属部卷：第五部：逝水有灯
时间：全稿写作夜里
地点：书桌和窗边

## 诗歌插页 P242：原创仿唐绝句

灯下开篇水色微，
旧名删尽晚风归。
若将往事成书写，
先许他人各自飞。

第一章写得很慢。陆允川原本想从高中教室写起，手指落到键盘上，却先写了一个男生拖着行李去报到。三楼、书皮、草稿纸、薄荷糖，这些细节一个个出来，比“我曾经怎样喜欢她”更稳。

他把夏知遥改成一个虚构名字，把苏晚枝也改成另一个人。改名不是遮羞，而是让她们从旧记录里脱身。她们可以在小说里拥有自己的公交、论文、室友、档案袋、湿伞和不愿承受的压力，而不是只站在陆允川回忆的光里。

写到大学楼下等待时，他停了几次。旧日记里那一段写得很重，几乎每一句都在证明自己的认真。小说里，他改成寒风、石凳、未接电话、宿舍门禁和朋友递来的热水。认真仍在，但不再天然正确。

凌晨两点，窗外安静。陆允川写到未发送的道歉，忽然删掉一大段解释。那一刻他明白，这本小说真正写成，不是因为他终于能把全部话说出来，而是因为他终于知道哪些话不该说给别人承担。

文档自动保存的提示亮起。第一批章节躺在文件夹里，像一排刚刚点亮的小灯。
