# 第145章 她说别追了多年后

所属部卷：第三部：寒风里的花
时间：多年后
地点：图书馆东门台阶

## 诗歌插页 P145：原创仿唐绝句

东门台阶换浅砖，
旧日水瓶不可看。
一句别追非薄幸，
留人退路也留宽。

多年后图书馆东门重修，原来的台阶换成浅色石砖。陆允川站在树荫下，看见学生排队进馆，手里拿着咖啡、电脑和各自的复习资料。

他找不到当年那瓶矿泉水放过的位置。找不到也好。那瓶水本来就不该被保存得太重。

旧稿里，他曾经写“她说别追了”写得很疼，像一场宣判。后来他把那一章改了很多次，终于把苏晚枝的手心、她前一晚的失眠、她挑选措辞的为难都写进去。

写进去以后，那句话仍然疼，却不再只疼他一个人。

陆允川坐在树下长椅上，翻旧日记。原句是：“她不要我了。”他在旁边补：她是在保护自己的生活，也是在给我退路。

这不是替过去找漂亮说法。是他终于承认，边界本来就可以温和，也可以坚定。一个人不愿被追着走，并不需要先把追她的人说成坏人。

离开时，他在自动贩卖机买了一瓶水。水很凉，他喝完，把空瓶投进回收桶。声音很轻，轻得像一个迟到很多年的确认。
