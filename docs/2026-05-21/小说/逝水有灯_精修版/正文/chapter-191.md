# 第191章 夏知遥的消息之前

所属部卷：第四部：失去的美丽
时间：大三秋天
地点：男生宿舍

## 诗歌插页 P191：原创现代诗

我把群头像放回夜色，
不让它替谁证明深情。
风从未说完的话里穿过，
有人低头，学着不再追问。
后来水声慢下来，
灯还亮着，人已经能往前行。

大三秋天的一个晚上，陆允川正在整理电脑桌面——把大一时写的诗稿拖进"旧文件"文件夹。

窗外的杨树叶子落了一大半，剩下的几片在路灯下泛着枯黄的光。秋风从窗缝里挤进来，带着一股干冷的泥腥味——校园里的草坪刚浇过水，湿土的气味被风带上了四楼。他的电脑桌面囤了将近三年的各种文档：论文草稿、诗稿、日记备份、一张很旧的夏知遥高中班级合照。

没错，夏知遥。不是苏晚枝。

夏知遥是陆允川高中的同班同学——坐在他前面三排、每天扎马尾、每次语文课念自己的作文都会被老师表扬的那个女生。陆允川在高二那年第一次意识到自己喜欢她，然后用一整年没有表白。高三毕业那天她跟另一个男生合照，他站在远处看了几秒，没过去。

这是他青春里的第一个遗憾。比苏晚枝更早。更深。因为少年时代的遗憾不是被拒绝——是从来没有开口。

后来考上同一所大学之后，他们偶尔在走廊里点头。高考之后两个人渐行渐远——选了不同的专业，加了好友但几乎不聊天。高中同学群平时也很安静。陆允川把那张合照从手机壁纸换掉——换了之后又打开文件看了三遍，最后拖进了一个叫"高中"的文件夹。

今晚他在清理电脑桌面——把一个叫"旧文件"的文件夹往硬盘移动时，那张合照的缩略图从文件列表中跳了出来。

他停下来。鼠标悬在图标上。然后点开。照片上的夏知遥马尾扎得偏高，校服的领口整理得很整齐。阳光刚好打在右边肩膀上——应该是五一期间在操场主席台前拍的。这张照片旁边还站着另一个男生——就是高三毕业跟她合照的那个。不是陆允川。

他在那张合照上停了三秒，然后点了右上角的X。

手机恰好在这个时候响了几声。高中群忽然热闹起来了。表情包刷得很快。几条消息夹着熟悉又陌生的名字往上跳。有人说"夏知遥在外省准备考试"，又有人接"她是不是很久没回县城了"。

他看着那个名字——夏知遥——心跳只微微加速了一拍。跟当年不一样。当年光是听到她的名字，他能慌整整一节课。现在是轻轻停了一下，然后继续走。

他把群消息退了。手机屏幕反扣在桌上。

宿舍里程砚正在为下周的实验赶预习报告。他听到陆允川把手机扣在桌上的动静，抬头看了一眼。

"怎么了？"

"高中群。看到夏知遥的名字了。"

"疼吗？"

"好像——没以前那么疼了。"

程砚把笔放下。"这叫什么——时间里泡太久了。"

祁野在上铺听到"夏知遥"这个名字的时候停了一下游戏。他知道这个名字——陆允川醉过一次酒，醉后把高中暗恋史全部抖完了。

"你看——你现在看到她的名字，第一反应是什么？"

"想——她应该挺好的。"

"就这些？"

"就这些。"

"那跟三年前的区别是——"

"三年前你只要提她的名字，我能跟你讲故事讲两个小时。现在——没有故事。就一个名字。"

程砚在旁边安静地听着，然后说："时间有时候比朋友会劝。"

陆允川没有回。他把手机设置成免打扰，然后继续拖文件。鼠标在桌面上划过时发出细微的摩擦声——桌上有一层薄灰，大三男生宿舍的桌面永远不可能干净。

那个"高中"文件夹里现在只有合照——没有当年写过的信——那些信早就在高中毕业那天撕了。他记得撕信的那个晚上——高考结束后的第三天，他在自己房间里把几张写满字的信纸叠在一起，先犹豫了一阵，然后从中间撕开。撕纸的声音很轻，但那个动作做完以后，他坐在床边哭了一场。不是因为撕信——是因为他终于承认自己错过了开口的最好时机。

他把文件夹归类到移动硬盘的"归档"目录下。就这三个字。
