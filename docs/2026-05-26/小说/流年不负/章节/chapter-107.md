# 第107章 信

时间：2016年1月

地点：楚州师院宿舍512

卷次：卷六：平行轨道

一月中旬。期末考之前。宿舍空了三个——阿坤搬出去了，小伍去图书馆通宵，李博回家了一趟。

512只剩顾临风一个人。

下午四点。阳光从窗户斜进来，在桌子上摊开一片橘黄色的光。暖气不够足，屋里还是冷。他穿着棉拖鞋。脚趾在鞋里缩着。

桌上有考研政治、一本古代文学史、半瓶墨水。墨水是英雄牌的。蓝黑色。瓶盖上结了一层干涸的墨渍。

他拉开抽屉。

抽屉上了锁。一把小挂锁。钥匙挂在脖子里——从高中养成的习惯。记笔记的本子、写的诗、还有那张高考准考证。

锁弹开。他翻了翻——笔记本。几支铅笔。一枚生锈的校徽。

然后从最底下抽出来一沓稿纸。横条格，20×20，师范学院的专用稿纸。左上角印着楚州师院的校徽。

他铺开稿纸。第一页。拿起那支英雄钢笔。拧开笔帽。笔尖在纸的左上角点了点。墨水洇开一个细小的蓝点。

然后他写了第一行。

"苏晚晴："

写完之后停了一下。看着这几个字。

"你好"太客套。"好久不见"太像要抱怨。"见字如面"太文绉绉。

他划掉了这几个字。重新写。

"很久没跟你说话了。其实高考后就没说过。"

第一段。写考研。

"我大三了。在准备考研。报的是古代文学方向。每天在图书馆坐六个小时。政治背了三遍。英语真题做了十套。专业课看了一大半。不知道能不能考上。"

第二段。写实习。

"去年在楚州二中实习。教初一。站在讲台上的感觉——很奇怪。像小时候玩游戏，忽然自己变成了老师。我对他们说的话，跟我当年的老师对我说的话——差不多。'把手机收起来。'说完之后觉得自己老了。"

第三段。写青阳。

"过年的时候回了趟青阳。经过那条有老槐树的巷子。树还在。比记忆中矮了。大概是树老了，或者是我长高了。巷子口新开了一家奶茶店。卖的东西叫'初恋的味道'。我买了杯柠檬水。"

第四段。

"在书店门口站了一会儿。新华书店——不知道还叫不叫这个名字。以前周六去县城买参考书，每次经过那条巷子都会往里看一眼。"

他写到这里，笔停了。

他知道自己在写什么。写的不是巷子，不是奶茶店，不是考研。他写的是——我还记得你。我记得你家在哪条巷子。我记得你从书店出来跟你妈提东西走进去的样子。我记得你书包上晃着的小鱼。

但他不能写"我还记得你"。

他在纸上写了"我记得"。然后划掉了。在旁边补了一句"青阳县的冬天还是那么冷"。

窗外有鸟叫。可能是麻雀。楚州冬天的麻雀不多。偶尔一只落下来在窗台上跳两下。

他继续写。

第五段。

"不知道你现在怎么样。听许若琳说你在楚州一小实习，教三年级语文。那些孩子应该很喜欢你。你以前就不太会拒绝人——小孩大概看准了这一点。"

赵一鸣前几天发微信说了一句话。大意是晚晴好像谈过恋爱，分开了。

他不能写这个。

他把笔停在半空。一滴墨水从笔尖渗出来落进稿纸——洇成一个小点。圆圆的，边缘有细微的毛刺。

他下面写了三个字——划掉了。又写了几个字——又划掉。

稿纸上有七八处涂改。有的是划掉的。有的是涂黑的。有一个地方他用指甲把纸抠出了一个洞。

第六段。很短。

"其实我一直想给你发个消息。不知道说什么。就算了。"

第七段。更短。

"你还记得我吗。大概记得吧。"

他没有写"我还喜欢你"。没有写"我经常翻你的朋友圈但从来不点赞"。没有写"你穿白衬衫很好看"。没有写"你扎马尾的背影是我的青春"。

他什么都没有写。

他把稿纸翻到第三页。看到最后几行空白。

笔放下。

看了一遍。从头到尾。一字不漏。

然后折好。整整齐齐地——对折、再对折。变成手掌大小的一块。

夹进那本黑皮笔记本。

笔记本放回抽屉。抽屉推上。锁扣摁下去——咔嚓一声。

钥匙放回衣领里面。贴着胸口。凉的。

窗外天已经黑了。路灯亮起来。黄澄澄的一团。

他坐了很久。盯着抽屉。

外面有室友回来了。李博。"老顾——你还没吃饭——"

"来了。"

他站起来。背上书包。

抽屉里的信在黑暗里。那个蓝黑色的圆点——墨水洇渍。大概还没干。
