# 第198章 回乡的暑假之后

所属部卷：第四部：失去的美丽
时间：回乡后的夜晚
地点：镇上老桥

## 诗歌插页 P198：原创仿宋小令

河水浅，晚灯稀。
旧暑上窗扉。
桥头买笔不题谁。
归路有尘微。

晚上家里停了一会儿电。母亲把蜡烛找出来，放在饭桌中央，火苗照着两碗绿豆汤。父亲坐在门口摇扇子，抱怨今年电路又要修。陆允川坐在小板凳上，听他们说邻居家儿子考编、菜市场涨价、二姨的血压。

这些话题很碎，却把他从桥上的旧情绪里拉回来。原来回乡不只是回到一段没完成的暗恋，也是回到父母渐渐慢下来的动作，回到厨房墙皮脱落的角，回到自己曾经忽略的日常。

电来以后，风扇重新转起来。陆允川回到东屋，把旧书从纸箱里搬出来。箱底压着高中时的日记本，封皮已经翘起，里面夹着几张练字纸。他翻到关于老桥那一页，字写得很用力，几乎划破纸背。

他读了两行就合上。不是不敢看，而是觉得不必把一个年轻人的全部疼痛重新演一遍。那页纸已经替当年的他承受过一次，现在可以休息。

他把日记本放到书桌抽屉里，没扔，也没继续读。窗外有人骑车经过，车铃响了两声。母亲在客厅喊他吃桃子，声音隔着门板传来，很近。陆允川应了一声，洗手时看见指缝里还有一点桥栏杆的白漆。
