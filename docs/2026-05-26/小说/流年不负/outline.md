# 流年不负 大纲

## 结构模型

- 作品形态：长篇
- 采用结构：十卷结构（每卷20章，共200章）
- 选择理由：时间跨度长（2010-2023），卷结构自然对应人生阶段切换；每卷20章给予每个阶段充足的展开空间

## 全书主线

> 2010年高一开学，顾临风对同班女生苏晚晴一见倾心，开始了长达三年的沉默暗恋。高考后他终于表白，两人短暂靠近，却被大学分流推向不同城市。此后六年——大学、实习、毕业、工作——他们在各自的世界里成长、经历、偶尔从共同朋友口中听到对方的消息。直到2018年春天，顾临风的母亲给他安排了一场相亲。推开茶餐厅的门，坐在窗边等他的，竟然是当年那个扎马尾的姑娘。这一次，流年没有辜负他们。

## 人物弧线

## 精细章纲

### 卷一：初见秋风（001-020）

#### chapter-001：开学日
- 时间：2010年9月1日
- 地点：青阳县第一中学高一三班教室
- 目标：建立顾临风的视角和世界观，完成"初见"场景
- 场景拆分：
  1. 顾临风从青阳镇坐公交到县一中报到，找教室
  2. 他坐在倒数第三排靠窗的位置，观察陆续进教室的同学
  3. 苏晚晴从教室前门走进来——浅蓝色T恤、马尾、微微低头
  4. 阳光从窗户斜照进来打在她侧脸上，顾临风的心被撞了一下
  5. 班主任点名，他记住了她的名字
- 人物变化：顾临风从"无所谓"到"心里多了一个人"
- 风格与技法：白描初见，不用"美""漂亮"等形容词，用光线和动作写她
- 结尾处理：他低头在课本空白处写下她的名字，然后立刻划掉了

#### chapter-002：第一周
- 时间：2010年9月第一周
- 地点：高一三班教室、食堂
- 目标：建立日常观察模式，展示顾临风如何"收集"她的信息
- 场景拆分：
  1. 他发现她坐在第二排靠窗——那是他在后排可以看到的位置
  2. 他发现她中午吃两个包子，一素一肉，先吃素的
  3. 他发现她写字的时候小指微微翘起来
  4. 她课间发呆看窗外，他在后面看她看窗外
- 人物变化：顾临风从"注意到一个人"到"开始系统性地观察她"
- 风格与技法：细节白描——包子、翘起的小指、窗外的方向
- 结尾处理：他发现自己每天来教室的第一件事，是看她到了没有

#### chapter-003：目光
- 时间：2010年9月中旬
- 地点：教室、走廊饮水机
- 目标：第一次目光对视——埋下"她的目光也停留过"伏笔
- 场景拆分：
  1. 课间顾临风去走廊尽头接水，她也在
  2. 他站在她身后等，她接完水转身——目光碰在一起
  3. 大概一秒钟，她微微愣了一下，然后弯了弯嘴角
  4. 他端着水杯回教室，水洒了一路
  5. 晚上躺在宿舍床上，大脑反复回放那一秒
- 人物变化：他发现她的目光里有"愣了一下"——她注意到他了
- 风格与技法：一秒钟写一整章，用细节撑开时间
- 结尾处理：他第一次想写点什么——但拿起笔只写了日期

#### chapter-004：座位表
- 时间：2010年9月下旬
- 地点：教室
- 目标：引入高中生活结构（座位、课表、考试），侧面展示人物关系
- 场景拆分：
  1. 班主任调整座位，顾临风被调到倒数第二排
  2. 他计算了一下——他和她的距离是四排+两个过道
  3. 新同桌赵一鸣话很多，开始跟顾临风搭话
  4. 顾临风用"嗯""还行"敷衍赵一鸣，但赵一鸣不在乎，继续说
  5. 赵一鸣看出他在看什么方向，笑了一下没点破
- 人物变化：赵一鸣进入故事，成为顾临风唯一的朋友和知情者
- 风格与技法：用座位表写空间关系=情感距离
- 结尾处理：赵一鸣说"那个女生挺好看的"，顾临风没接话，但耳朵红了

#### chapter-005：挂件
- 时间：2010年10月
- 地点：教室
- 目标：第一个关键物品——她书包上的小挂件（全书伏笔005）
- 场景拆分：
  1. 顾临风注意到她书包拉链上挂着一个陶瓷小鱼，黄豆大小
  2. 他观察了三天才确定那是一条鱼，不是别的
  3. 有一次她经过他座位，书包蹭到桌角，小鱼晃了两下
  4. 他盯着那只晃动的鱼，直到她走出教室门
  5. 多年后他会知道——那是她奶奶给她的，奶奶在她小学时去世了
- 人物变化：一个物品让暗恋变得更具体——他喜欢的不是抽象的她，是带着小鱼挂件的她
- 风格与技法：物品作为情感载体，一击两鸣（写鱼=写她=写他）
- 结尾处理：他放学后在校门口小卖部看到一个类似的陶瓷小鱼，站了几秒，没买

#### chapter-006：物理课
- 时间：2010年10月中旬
- 地点：教室、物理实验室
- 目标：展示她的"不完美"——物理不好；展示他的"关注形式"——默默记住
- 场景拆分：
  1. 物理课小测验，卷子发下来
  2. 顾临风看到她拿到卷子后轻轻地叹了口气
  3. 她趴在桌子上，脸埋在胳膊里，大概过了五分钟才抬起头
  4. 物理老师在黑板上讲题，她跟得很吃力，笔记写了擦、擦了写
  5. 顾临风物理还行，他忽然想——如果可以帮她讲题就好了
- 人物变化：他发现了她的脆弱面，暗恋从"仰望"变为"想帮"
- 风格与技法：不写"她很难过"，写"趴在桌子上五分钟""笔记写了擦擦了写"
- 结尾处理：他在自己物理笔记本的最后一页，用很小的字写了一道她可能不会的题的解法。没有给出去

#### chapter-007：跑操
- 时间：2010年10月下旬
- 地点：学校操场
- 目标：场景从教室扩展到校园，展示集体活动中的暗恋
- 场景拆分：
  1. 每天上午第二节课后是跑操时间，全校按班级排队
  2. 高一三班在操场东侧集合，女生在前男生在后
  3. 顾临风跑在队伍最后——这样可以看到前面所有人，包括她
  4. 她的马尾在跑操时会左右晃，阳光照在头发上有细微的光泽
  5. 有一次她鞋带松了，停下来系，他从她身边跑过去，很近
- 人物变化：暗恋渗透到日常生活——连跑操都变成了有意义的事
- 风格与技法：用跑操的队列写"她在前他在后"的距离感
- 结尾处理：跑操结束回教室的路上，她和一个女生并排走，笑了一下——不是对他笑的，但他不介意

#### chapter-008：第一次考试
- 时间：2010年11月初（期中考试）
- 地点：教室
- 目标：用考试成绩建立两人的实力对比（她成绩比他好）
- 场景拆分：
  1. 高一上学期期中考试，三天六门
  2. 成绩公布——苏晚晴班级第八，顾临风班级第三十一
  3. 他看成绩单的时候先找的不是自己的名字，是她的
  4. 赵一鸣说"你考得一般啊"，他说"我知道"
  5. 但他在意的不是自己的名次——他忽然觉得，她那么优秀，他差她太远了
- 人物变化：暗恋中加入自卑——"我不配"的种子种下了
- 风格与技法：用排名数字代替心理描写，数字=情感
- 结尾处理：晚上宿舍熄灯后，他打手电筒做了一页物理题

#### chapter-009：冬天的第一场雪
- 时间：2010年12月初
- 地点：教室、校园
- 目标：季节变化，暗恋随时间加深；第一首诗的萌芽
- 场景拆分：
  1. 青阳县冬天的第一场雪，在十二月的一个周三上午开始下
  2. 下课的时候大家都涌到窗边看雪
  3. 苏晚晴站在窗户前，伸出手去接一片雪花——雪在指尖化了
  4. 顾临风站在她右后方两米处，看着她的侧脸和窗外飘的雪
  5. 这个画面在他脑中盘桓了很多天
- 人物变化：雪的白色和她给他的感觉重合——干净
- 风格与技法：雪景白描，不抒情
- 结尾处理：他周六回家，在房间里写了几行字，第一句是"那年九月风初凉"——这是他三年写诗生涯的起点

#### chapter-010：县城
- 时间：2010年12月，一个周六
- 地点：青阳县城街道
- 目标：暂时离开校园，展示县城生活背景，偶遇她
- 场景拆分：
  1. 周六下午顾临风去县城新华书店买参考书
  2. 在书店门口他看到了苏晚晴——她和一个个子不高的中年女人在一起
  3. 那是她妈妈——他猜的，因为她们走路的姿势很像
  4. 她没有看到他，提着一袋子东西从书店旁边的小巷走进去了
  5. 顾临风站在书店门口，往巷子里看了一眼——很普通的巷子，门口有棵老槐树
- 人物变化：看见她"生活中的样子"——不是在教室、不是在跑操，是一个跟妈妈一起买东西的普通姑娘
- 风格与技法：县城街景白描
- 结尾处理：他后来每次经过那条巷子都会往里看一眼。从来没有告诉过任何人
