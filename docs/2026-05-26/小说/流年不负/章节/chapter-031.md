# 第031章 月考之后

时间：2011年7月初

地点：教室

卷次：卷二：窗边人影

期末成绩贴在黑板旁边的墙上。一张大白纸。上面密密麻麻印着名字和分数。

上午第二节课后贴出来的。

下课后一群人涌过去看。挤了三四排。有人踮脚。有人蹲在前面。有人看到自己的名字后发出各种声音——有的拍桌子，有的长出一口气，有的不说话。

顾临风站在人群外面。

他在等人散开。

赵一鸣已经挤进去了。半分钟后他挤了出来。

"你第二十二名。"

"多少。"

"第二十二。总分478。班级第二十二。年级第一百七十。"

顾临风点了点头。上次是第三十一。从三十一到二十二。进步了九名。

"你呢。"

"第十五。没动。"

赵一鸣说没动的时候脸上没什么表情。但他自己也没退步。

"去小卖部不。"

"你先去。"

赵一鸣走了。

人群慢慢散开。成绩单前面只剩下三四个人。

顾临风走过去。

他从上往下找。

苏晚晴——第六名。总分542。年级第五十二。

第六名。比上学期又进了两名。上次是第八。

他的手指在纸上停了一下。点在她的名字旁边。纸是凉的。圆珠笔印的墨有点洇。

然后继续往下找。

他看到了自己的名字。第二十二名。478分。

旁边的角落里有人站着。

他转过头。

苏晚晴站在成绩单前。离他不到两步远。不知道什么时候过来的。

她在看成绩单。

从左往右看。从上往下看。目光扫过名字和分数。

她看到了自己的名字。停了大概一秒。然后继续往下看。

顾临风站在原地。没动。

她的目光往下移。过了前五名区。过了前十名。过了前十五。到了二十左右的位置。那里有他的名字。

她停了几秒。

大概两秒。也可能三秒。

眼睛在那个区域定了一下。

然后收回目光。

转身走了。

她没有说什么。没有笑。没有皱眉。就只是看了两秒——然后走了。

顾临风站在原地。手放在裤子口袋里。指节在口袋里握紧了又松开。

她在找他的名字吗。

还是只是在随便看。

他不知道。

他走回座位。坐下去。翻开了英语书。翻到单词表。看了三个单词。一个都没记住。

赵一鸣从小卖部回来。手里拿着两瓶雪碧。放了一瓶在他桌上。

"你怎么还没看完。"

"看完了。"

"那走吧。"

"哪去。"

"你不是说去小卖部。"

"不去了。"

赵一鸣看了他一眼。又看了看成绩单那边。座位上只剩他自己了。苏晚晴已经回座位了。

"行吧。"

顾临风把雪碧拉到面前。拉开拉环。喝了口。汽水很冰。喉咙发凉。

晚上他在日记上写：

"她第六名。从上学期第八进到第六。她看成绩单的时候在二十名左右停了大概两三秒。我在想——她是在找我的名字吗。也许只是随意看。但她确实在那里停了一下。"

翻页。

"我从三十一名进步到了二十二名。进步了九名。老师课上提了一句——'顾临风进步比较大'。她没有转头。但她肯定听到了。"

翻页。

"还有两年。我不会只进步九名的。"

合上本子。

窗外银杏叶很茂盛。七月了。叶子是深绿的。没有三月的枝条光秃。

明天考物理。

他翻开物理书。从第一页开始看。

月光从窗帘缝漏进来。

还是那道光。
