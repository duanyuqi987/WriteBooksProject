# 第037章 暑假结束

时间：2011年8月31日

地点：家里

卷次：卷二：窗边人影

八月三十一日。暑假最后一晚。

白天又下了一场雨。下午停的。院子里积了水。傍晚的时候水面反着最后一点天光。橙色的。慢慢变灰。

顾临风把暑假作业摞在桌上。都写完了——数学、物理、英语、化学。赵一鸣那本笔记本夹在他的物理书里没还。明天给他。

晚饭是面条。他妈晚上下班早。做了西红柿鸡蛋面。两个人坐在桌前吃。他妈问"明天开学"，他说"嗯"。他妈又问"钱够不够"，他说"够"。然后没别的话。

吃完饭他洗碗。把锅也刷了。毛巾拧干挂到门后。

回房间。把书包收拾好。课本、作业本、铅笔盒、日记本。日记本的位置换了——以前在最底下，现在夹在物理和数学课本之间。不厚。大概写了二十多页。

赵一鸣的暑假作业本来不重——但加上他自己的所有书摞起来。书包很沉。

他在桌前坐下。窗外夜色已经全上来了。后院的竹竿上还有一件忘了收的衣服。风一吹晃悠悠的。看起来像个站着的人。

他翻开了日记本。

从第一页开始看。

"5月13日。她今天换了一个蓝色的发绳。"

"5月20日。她物理考了72分。"

"6月1日。她穿了一双新鞋。白色帆布鞋。红色鞋带。"

"6月7日。她跟许若琳在走廊聊天。笑过两次。"

"6月15日。放学后她在教室多待了十分钟。"

"7月1日。高一快结束了。还有两年。"

"8月5日。补课第一天。她又坐在窗边的位置。两个月来第一次觉得哪儿都不想去。"

一本日记翻完。他翻到最后一页——空白的。

拿起笔。

"8月31日。暑假最后一晚。"

停了很久。笔尖离纸只有一厘米。没落下。

然后继续写。

"高二了。还剩两年。"

写完。停下来。手指捏着笔转了转。转了七八圈。停下来。笔放在桌上。又拿起来。

在下面加了一句：

"两年后我一定要告诉她。"

写完之后他把这句话看了很久。

两年。听起来很长。但高一已经过去了。三百多天——快得不像话。两年就等于两个高一。两个三百多天。

够吗。

他不知道。但这是他给自己的期限。

他把日记本合上。翻到封面。在第一页那个"1"旁边已经写了"2""3""4"……"21"。他数了一遍——二十一篇日记。写了两个多月。

不多。但每个字都是真的。

他把日记本放进书包夹层。然后把拉链拉好。

走到窗边。推开窗。

外面很安静。远处的田野里有一两个亮着灯的房子。树是黑色的剪影。天上没有月亮。星星也不多——只有西边天上有两颗很亮的。

蝉不叫了。八月结束了。

明天去学校。又能看到她了。

他关了台灯。

躺在床上。手枕在脑后。天花板上的那条裂纹在黑暗里看不清。但他知道它在——像这一年所有的记忆一样。看不见，但一直在。在头顶。在天花板的方向。

明天九月一日。

高二。

他翻了个身。闭上眼睛。

书包在墙角靠着。沉甸甸地。里面装着所有课本、暑假作业、赵一鸣的笔记本。还有一本牛皮纸封面的日记本——最后一页上两行字还没干透。

月光从窗帘缝漏进来。这次有月亮了。细的。蛾眉月。

两年。

还有两年。

他睡着了。
