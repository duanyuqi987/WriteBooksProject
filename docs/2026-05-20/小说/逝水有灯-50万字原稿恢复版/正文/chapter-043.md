# 第043章 诗抄本之后

所属部卷：第一部：初秋三楼
时间：高二冬天
地点：男生宿舍下铺

## 诗歌插页 P043：原创仿宋小令

风过三楼，
纸页小窗明。
欲把旧年藏一笔，
却听铃后有人行。
灯下水声轻。

诗抄本之后，陆允川表面上没有变化。早读照旧，跑操照旧，作业本照旧从前排传到后排，可他自己知道，有一处地方被轻轻碰过。

熄灯铃响后，宿舍陷进一片黑。有人还在被窝里背公式，有人小声讲笑话。陆允川把本子塞到枕头下，仿佛这样就能让心事安静一点。

他把诗抄本里被墨水洇开的句号反复想了几遍，每想一遍都添一点不存在的意思。到第三遍时，他自己也觉得过分，便打开练习册，硬把目光压到题目上。

“你这本子是给语文老师看的，还是给自己壮胆的？”程砚说。“抄诗怎么了？”陆允川说。“抄诗没事，别把人家抄成欠你的。”程砚说。

程砚说完就去洗漱，牙刷杯碰在脸盆架上叮当响。陆允川盯着那句“别欠你的”，心里有点不舒服，却没有反驳。

晚饭时食堂排队很长，铁勺敲在不锈钢盆上，声音一阵一阵。陆允川端着饭找位置，程砚抢先坐下，把一半桌面让给他。

“你别一有事就不吃饭。”程砚把筷子递过来，“感情没进展，胃先退学，不划算。”陆允川接过筷子，笑了一下，笑得很短。

叶澄后来借过这本诗抄本，看见边角密密麻麻的小字，只说了一句，诗可以救人，也可以把人困住，看你怎么用。

夏知遥并不知道这些后续。夏知遥并不知道自己被写进那么多空白。那天她在教室里背政治提纲，手指压着书页，一条一条往下划，神情认真得近乎朴素。她有自己的课表、错题、家里打来的电话和不想解释的疲惫，不可能替陆允川安置所有心情。

这一点，是陆允川很晚才懂的。当时他只懂把一个小动作翻来覆去，像把一颗糖含到没有味道，还舍不得吐掉。

窗外有人倒热水，水流进暖瓶时发出空空的回声。宿管阿姨在楼道里喊别说话了，声音从门缝钻进来，压住一屋子没长大的秘密。

夜里回到宿舍，他在日记本上记了几行，又怕别人看见，把本子压到枕头下面。窗外的风刮过晾衣绳，衣架轻轻碰撞，像有人替他把没说完的话敲碎。

他抄唐诗时喜欢把字写得瘦一点，抄宋词时又忍不住拖长最后一笔。程砚说他这不是练字，是给心事换衣服。陆允川没理他，只把台灯往里挪，免得光照到别人床上。

有一页现代诗写到一半，宿舍突然熄灯。他在黑暗里停住笔，墨水在纸上多洇了一点。上铺有人问谁还有手电，另一边有人说别写了，明天还要早操。

夏知遥的名字从来没有直接出现在诗抄本里，却又到处都是。一个“她”、一处“窗边”、一行“冬天的课桌”，都像绕路写出的地址。多年后陆允川再看，才觉得这种绕路既真诚又危险。

叶澄借本子那天没有翻太久，只停在一首仿宋小令前，说这句还可以，但你别把所有空白都留给同一个人。陆允川当时没答，心里却记了很久。

他把这一天拆成很多小段：哪一句话是真的，哪一个动作是自己添上的，哪一点失落其实来自期待太高。拆到最后，他发现能留下的并不多，可那一点点留下来的东西反而更干净。

寝室灯灭以后，走廊还有脚步声。陆允川闭上眼，脑子里反复出现白天的画面，出现到后来也疲惫了，只剩一道模糊的光。人总会在疲惫里比在激动里诚实一点。

事情过去以后，男生宿舍下铺又恢复原样。台灯还在，钢笔也还在，只有陆允川看它们的眼神变了一点，像有人在同一页纸上多压了一道折痕。

晚自习结束，程砚拍着他的肩说：“别复盘了，再复盘明天早读都开始了。”陆允川把书合上，终于跟着人群往外走。

如果只写疼，这一章会很薄；加上台灯、钢笔、枕头、暖瓶，它才重新有了重量。那重量不是压人，而是让读者相信他们确实在这里生活过。

陆允川把钢笔拿在手里，先把这一页诗抄完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把这页写完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她在教室前排背政治提纲，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

男生宿舍下铺里最明显的不是心跳，而是暖瓶水灌满时的空响。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，泡面调料包被撕开的味道把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。钢笔、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情过去以后，男生宿舍下铺又恢复原样。人群散开，桌椅归位，只有陆允川自己还把诗抄本想得太久，像把一页纸反复折到发白。

夜里他把这件事写进日记本，写到一半停住。原本想写“命运”，最后改成了“今天”，因为今天已经足够具体。

这一段让诗抄本不只装暗恋，也装宿舍的冷、台灯的光和同伴的提醒。诗可以安放心事，但不能替他向别人索要答案。

如果只写疼，这章会很薄；加上钢笔、暖瓶水灌满时的空响和泡面调料包被撕开的味道，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在男生宿舍下铺发呆。”陆允川把钢笔收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

这一章收在平常处：灯灭了，楼道静下来，陆允川睁着眼，终于承认今天不是命运给他的暗号，只是生活递来的一个片段。可片段也会发光，只要他不把光攥得太疼。
