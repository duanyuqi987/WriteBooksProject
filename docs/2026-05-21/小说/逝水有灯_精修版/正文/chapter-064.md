# 第064章 高考后的水旁观

所属部卷：第二部：六月以后
时间：2012年六月八日
地点：县一中校门口

## 诗歌插页 P064：原创现代短诗

那时我们都以为，
结束会有清楚的门。
后来才知道，
门只是普通下午的一阵风。
有人走向远处，
有人学着把远处还给远处。

程砚妈妈在烧烤摊结账的时候，程砚在旁边看着，忽然觉得自己妈今天有点不对劲。

"妈，你没骂我乱花钱？"

"考都考完了。"程砚妈妈把钱递给老板，找了零，仔细对了一遍才揣进包里，"你今天高兴就多吃点。"

程砚一时不知道接什么。他妈平时很抠——不是穷，是抠。买衣服要挑反季打折，买菜要等傍晚降价。今天他点了一百多块钱的烧烤，她居然一句都没说。

"你是不是觉得我考得很好？"程砚试探地问。

"我觉得你考完了。"他妈把钱包拉链拉好，"考完了就行了。考得怎样等到出分再说。"

程砚愣了一下。然后他笑了——那种被看穿之后的笑，不太好意思，但又有点暖。

校门口那边，叶澄其实有人来接。不是她自己叫的——是她爸，骑了一辆电动车过来，后座上还挂着一袋菜。她爸是个沉默的男人，在人群里站了好一阵才等到她。

"你怎么来了。"叶澄走过去。

"你妈让我来的。说你东西多。"她爸看了一眼她的书包——其实不大，就一个书包加一个笔袋。"就这些？"

"就这些。"

她爸把电动车调了个头，把菜袋子挪到前面踏板上，腾出后座。"上来。"

叶澄坐上去，书包抱在腿上。电动车开出去，经过陆允川和他妈旁边——陆允川正在喝那瓶温掉的水。叶澄本想打声招呼，但车已经过去了。

她爸在前面说了一句："你那个同学，叫陆什么的，今天他妈妈也来了。"

"陆允川。"

"对。你以前说他作文老跑题。"

"后来好多了。"

"那就好。"她爸停了一下，像在组织语言，"你们同学以后各奔东西了，关系好的就多联系。"

叶澄没说话。她低头看着手机，班群里正在热烈讨论明天的聚会。程砚已经发了三条消息，每一条都带感叹号。陆允川回了一条，就两个字："几点。"

她打了一行字又删了，最后发出去的是："上午九点学校门口集合。"

夏知遥那一边，情况比这群人要复杂一点。

三轮车到家以后，她母亲在楼下就跟她说了："你爸打电话来了，问你考得怎么样。"

"你怎么说的。"

"我说你说还行。"

夏知遥站在楼梯口，夕阳从楼道窗户里斜照进来，打在她脚边的台阶上。"他还有什么话？"

"他说让你先休息几天。出分以后，再跟你谈报考的事。"她母亲顿了顿，"他说，如果你分数够，可以考虑考到他那边去。"

"他那边是哪边？"

"深圳。"

夏知遥把手里的水瓶放到楼道窗台上。瓶身外面挂满了冷凝水，在灰扑扑的窗台上聚成一小滩。"你让我想想。"

"我不逼你。你自己想。"她母亲上楼之前回过头，"但你爸说的也有道理。深圳学校多，机会也多。"

夏知遥站在楼道里看着窗台上那瓶水。水是母亲买的，瓶盖已经被她拧开过好几次。她想起刚才在校门口看到的一幕——陆允川站在台阶上，他母亲递水给他，他没有说谢谢，但接过去就喝了。很自然，像一个不需要言语的仪式。

她拿起水瓶，上楼。

宿舍那边，周长海的自行车在半路上掉了第二次链子。这次他怎么都挂不上去了，齿轮上全是黑油，手指也蹭得乌黑。他把车推到路边，蹲下来对着链条发呆。

手机响了。是程砚发来的消息：你到家了没，别骑太快掉沟里了。

周长海用沾满机油的手指打了两个字：掉了。

程砚秒回：链条？

周长海：嗯。

程砚：那你推车回去还是我来接你？

周长海：推车。不远了。

程砚：行。你到家说一声。

周长海把手机揣回兜里，把链条往齿轮上对了对，还是没挂上去。他站起来，推着自行车往前走。链条在后轮轴上拖着地面，发出刺耳的金属摩擦声。

走了大概两百米，路边一个修车铺还没关门。光头老板正在收工具，看到他推着车过来，说："链条掉了？"

"嗯。"

"推进来。"

五分钟之后，链条装好了。老板收了五块钱，多给了他一截铁丝。"下次再掉，用铁丝缠一下，能撑一阵。"

"谢谢。"

周长海骑上车往家走。链条虽然装好了，但踩起来还是有咔咔的响。他想起程砚说的——考都考完了。是考完了。明天不用早起，后天也不用。但他现在有点不知道不用早起之后该干什么。

到家以后，他给程砚发了条消息：到家了。链条修好了，五块钱。

程砚回复：五块钱也跟我说，你是不是闲得慌。

周长海笑了一下。手机屏幕的光映在他脸上，在没开灯的房间里格外亮。
