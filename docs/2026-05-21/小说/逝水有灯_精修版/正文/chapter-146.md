# 第146章 食堂争执之前

所属部卷：第三部：寒风里的花
时间：六月末
地点：校外小饭馆

## 诗歌插页 P146：原创现代诗

我把辣椒油放在黄昏里，
不让它替任何人回答。
风从校园的一侧经过，
带走一句没有说稳的话。
后来我学会把手松开，
让灯照着路，也照着空出来的地方。

期末考结束那天，苏晚枝约陆允川去校外吃顿饭。她原话说的是"庆祝建筑初步和结构力学两门都过了，顺便讨论一件事"。吃饭的事一般是陆允川提议，她批准。这次她主动提，说明第二件事比第一件重要。

小饭馆在校门口往东三百米的巷子里，叫"老马饭馆"。红底黄字招牌，门面不大，里面摆了六张方桌。苏晚枝提前到了——他推门进去时她已经坐在靠墙的位置，面前放着一壶茉莉花茶，两个杯子。她倒了茶，推给他一杯。

"先点菜。酸菜鱼还是宫保鸡丁？"

"你请客，你点。"

"那就都点。加一份干煸四季豆。"

点完菜以后，她把手放在桌上，十指交叉，拇指按着茶杯的杯沿。陆允川认识她够久了，知道这个手势意味着"接下来要说的话我准备好了"。

"我爸打电话来了。"她开口。

"祝贺你两门都过了？"

"是。然后他说给他寄一份建筑初步的作业——展览馆的模型照片和透视图。他说想看看我做得怎么样。"

"你给他寄了？"

"寄了。然后他又附加了一句话——'既然是双学位，下学期选课的时候你自己安排。但诗的社团——你能不能考虑减少时间。'"

"他怎么说的？"

"'双学位已经很紧了。再加一个诗社，人会很累。'然后又说：'你自己的时间你自己安排。我说的是建议。不是命令。'"

陆允川把茶端起来喝了一口。茶有点烫，加了茉莉花瓣，闻着香，喝着很淡。

"你怎么回答的？"

"'我会考虑。'"

"就是没说同意还是不同意。"

"对。因为我不确定。他说的有道理——下学期课业更重。结构力学只是第一门。后面还有建筑力学、材料与构造、建筑物理——每一门都需要画图时间。双学位的课在周二和周四晚上加三小时。我算了一下，每周给诗社的时间至少少一半。"

菜上来了。酸菜鱼冒着白汽，干辣椒浮在汤面红色油层上。宫保鸡丁里的花生米炸得焦脆。苏晚枝夹了一块鱼，挑掉上面的花椒。

"你在想什么？"她问。

"在想你爸说的'建议'——本质上是他希望你多给建筑时间。他不再直接反对你写诗，换了种方式。说'你自己安排'——但你要怎么安排才算对？"

"你觉得呢？"

"我觉得不需要减少诗社时间。你做得很好。诗社运转正常。新社员有沈让、小孟和你。你把事往下传就行了。但你说得也对——下学期课多。所以是你需要分配，不是你爸需要你分配。"

"那你帮我分配一下。"

"不是我帮你。是我跟你一起排。"陆允川把筷子放下来。"你把课表发我。诗社的活动固定在周三晚上。社刊组稿改两周一期。树墙维护交给小孟——她已经挂过叶子，知道怎么做。论文和审稿我可以代你看。你只需在需要盖章签字的关键节点出现。这样一周大概省出——两个小时。"

苏晚枝拿着筷子停住了。"你什么时候算过这个？"

"上次你说双学位批准以后。我就排了一下。"

"你排了——但没跟我说。"

"因为你在等考完再谈。我不想在考之前加你的压力。现在考完了——可以把分配方案给你看。"

他从帆布袋里拿出一张折成小方块的A4纸。展开来，上面是手绘的时间表——用铅笔画的，可以擦改。周一到周五，上午下午晚上三段。建筑系的课用蓝色标出，中文系的课用灰色，诗社相关的活动用红色圆圈圈起来。时间表的末尾有一行小小的铅笔字："重新排的。觉得行就试。不行我们再调。——LX"

苏晚枝把时间表看了两遍。然后用手指沿着周三晚上的红色圆圈画过去。

"你给我留了完整一个晚上。"

"周三晚上——诗社专属。不加其他任何事。"

她又在纸的右下角指了一下。"这行字——'觉得行就试'——是我说的格式。"

"我抄你的。但加了'我们再调'。你不知道的后续：如果你觉得不行，不是就完了——是再调。"

饭桌上的菜渐渐凉了。酸菜鱼的汤上凝固了一层薄油。苏晚枝把时间表折起来放进自己的布袋里。

"这个时间表是礼物。"

"不是礼物。是方案。"

"方案就是礼物。"她把茶杯端起来喝完。

然后她说："我爸说的那些——关于减少诗社时间——还有一个更深的。他怕诗社会一直把我拖住。让我没办法专心建筑。他不知道诗社给我的是什么——不是时间负担。是做事的逻辑。我做建筑模型之前在301教室学会了怎样把东西放在精确的位置上。我做展览馆设计的时候用了'空间是邀请'——这话是建筑初步课上学的，但我第一次用是在诗社的树墙上。"

"所以你不需要对诗社说对不起。诗社没有拖住你。"

"对。需要说对不起的是我爸。"

陆允川夹了最后一片四季豆。四季豆干煸得很透，水分几乎没了。

"你可以不说对不起。你只需要在他下次问的时候——把时间表亮出来。'爸，这是我自己排的时间。蓝色的是你的建筑，灰色的是我的中文，红色的是我。'"

苏晚枝看了他大概三秒。然后笑了——不是眼角往下弯的那种笑。是另一种——抿着嘴，嘴角两边往上，不发出声音。

"你这个建议比我预设的所有对话预设都好。因为它不需要我辩护什么。只需要把事实给他看。"

吃完饭走出饭馆时，外面已经全黑了。巷子里的路灯隔好远一盏，光线弱得只够照到自己的脚下。苏晚枝走在前面，踩路边的排水沟盖板走成一排，像小孩子过独木桥。她没有牵他的手，但把手甩得很近。

"明天我去打印两张时间表。一张贴宿舍床头，一张贴301教室讲台。"她说。

"讲台那个位置——是'她的'地方。"

"对。所以贴在那里。"
