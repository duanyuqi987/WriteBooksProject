# 第160章 好久不见

时间：2013年9月，大四开学

地点：江城大学图书馆

卷次：第七卷：实习与远方

暑假过去了。

大四开学那天是九月一日，周一。

林玉起到了图书馆时已经快下午三点了。暑假里图书馆闭馆了两个月，空气里有一股淡淡的灰尘味。管理员把窗户全部打开了，秋风吹进来，吹得窗台上的报纸哗哗地响。

他在靠窗的座位坐下，把考研的书从书包里拿出来。

《中国古代文学史》《现当代文学》《文学理论教程》——三本厚得像砖头的书。他暑假在老家已经看了一遍，现在开始看第二遍。

翻到《诗经》那一章时，有人在他对面坐下了。

他没有抬头。

对方把一本《教育心理学》放在桌上。书很新，封面没有一点褶皱。

林玉起抬起头。

苏安晴坐在对面。

她穿了一件白色的短袖衬衫，头发比上学期短了一些，刚好齐肩。脸晒黑了一点，是暑假在老家的院子里晒的。

"好久不见。"她说。

声音还是那样。不高，不低，刚刚好在图书馆里能听见。

"嗯，好久。"

他看了她一眼。

两个月。暑假两个月。

他暑假在家的时候想过很多次开学。想过在校园里遇见的姿势，想过第一句话该说什么，想过她会不会变样。

想过很多，今天真的遇到了，只说了三个字。

"你考研？"苏安晴指了指他面前的那些厚书。

"嗯。你呢？"

"不考。我准备考编。"

教师编制。

林玉起把笔放在书上。

"想好了？"

"嗯。实习完就差不多想好了。我喜欢教书。"苏安晴翻开《教育心理学》，在第一页上写了自己的名字。字迹和实习时在作文本上写的评语一模一样，笔画很轻，但每一笔都到位。

她也准备考编。

各翻各的书。图书馆里很安静，只听得见翻页的声音和远处饮水机的咕噜声。

过了大概半个小时，苏安晴抬起头来。

"你家那个坡还在吗？"

林玉起愣了一下。然后才反应过来她说的是封丘老家的那个坡。门口那条土路尽头的小山坡，小时候他常去那儿放风筝。

"还在。去年修路的时候差点被推平了，后来村里人不同意，就留着了。"

"我家那个坡早就被推了。修公路的时候。"苏安晴说，语气很平，"暑假回去我找了半天没找到。"

"那你来我家这边放风筝。"

他说完自己愣了一下。

苏安晴也愣了一下。

然后她低下头，继续翻书。

"行啊。等考完试的。"

她的声音很轻，就像刚才这句话只是顺嘴一提。

但林玉起听到了。

他在心里把她这句话收好，放在了某个安全的地方。

下午五点半，图书馆开始清场。管理员在喇叭里喊"闭馆了闭馆了"。苏安晴合上书，站起来。

"明天还来吗？"

"来。"

"那我帮你占位子。"

"不用——"

"我帮你占。"

苏安晴说完就走了。白衬衫的背影消失在图书馆的玻璃门后面。

林玉起在原地坐了一分钟。

然后把书收起来，装进书包。目光落在对面那张桌上——什么都没有，只有一层薄薄的灰尘，还有一小片阳光。

他走出图书馆。秋风扑面，梧桐叶子正一片一片往下落。夕阳把他头顶的枝丫镀上一层金边。

暑假过去了。两个月。

两个月里他都在看书。在自家的院子里，蚊子很多，老妈隔一会儿给他送一块西瓜。他坐在躺椅上，把中国文学史从头翻到尾，又在旁边放了一本教师编制考试的参考书。

那本书是他在县城新华书店买的。买的时候什么都没想。

上了台阶，他掏出手机，想给苏安晴发条信息，但又把屏幕关了。开学第一天，不用太勤快，那句"我帮你占位子"已经够了。

他走在梧桐树下，心里什么都没有想。
