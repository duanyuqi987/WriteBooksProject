# 第220章 旧日记本多年后

所属部卷：第五部：逝水有灯
时间：工作第三年
地点：搬家纸箱旁

## 诗歌插页 P220：原创仿唐绝句

秋灯微明照旧尘，
少年心事隔江春。
回看不见当时路，
一树风声认故人。

多年后他重写旧日记本，没有再把自己写成被命运专门为难的人。他把夸张的句子删掉，把过重的词往回收，只留下能让读者看见的动作和物件。

他当时不知道这一页会被保存那么久。年轻人写东西常常只想熬过当晚，没想到多年以后再翻开，最先醒来的不是名字，而是那一年空气里的温度。所以这一章不只写一件事，也写那件事在岁月里怎样慢慢改变形状。那些形状有时锋利，有时柔软，最后都被他放进更宽的生活里。

楼道里有人提着热水壶经过，壶盖碰着壶身，声音细得像一句没说完的话。陆允川站在那样的声音里，忽然觉得自己并不是故事中心，只是众多走过校园的人之一。这个发现当时没有让他轻松，反而让他更想抓住一点能够证明自己的东西。

多年后陆允川重写这一段，才发现搬家时陆允川翻出旧日记本，里面有三楼、河边、南湖、楼下和许多写重了的话。留下的不是判决，而是一盏很小的灯。他后来反复想起这一段，不是因为它多么宏大，而是因为纸箱里露出的牛皮纸封面。具体的物件比誓言诚实，它不替人辩解，也不替人作证，只安静地留在光里。

她把杯子往桌角挪了挪，动作很轻，却让两个人之间多出一小块空地。他想说的话有很多，可真正到了嘴边，只剩下几句平常的话。年轻时他总觉得平常太轻，装不下喜欢；后来才知道，太重的喜欢常常先把自己压弯。

夜里停电过几分钟，整栋楼忽然安静，连平时最吵的人也压低声音。黑暗让他短暂松了一口气，因为没有人能看清他脸上的表情。这些细枝末节在当时没有被他认真看见。他忙着判断自己有没有被在意，忙着从别人的一句话里找温度，反而忽略了日子本身怎样一格一格向前挪。

“有些话你晚点懂也行，别在没懂的时候先把关系弄坏。”程砚这样说时，语气并不郑重，甚至带着一点不耐烦。陆允川当时听不进去，只觉得朋友站得太远。后来他才知道，太近的人往往只能看见火，站远一点的人才看得见整间屋子正在冒烟。

关系在这里往前挪了一点，也往旁边偏了一点。他没有烧掉，也没有重读到天亮，只把本子放回箱底，到了后来，它终于从伤口变成可以回看的旧路。陆允川那时没有能力准确命名这种变化，只能把它写进纸页，写得越满，越像害怕空白。

程砚后来承认，他也不知道怎样劝才算对，只能尽量不让陆允川把自己弄得太难看。祁野在群里说旧东西别全扔，留点给以后笑自己。朋友的声音并不总是温柔，有时甚至粗糙，可它们让他看见世界并没有随着一段关系变窄。

这一章背后还有一层更普通的生活。班主任会催作业，宿管会查晚归，食堂阿姨会因为饭卡刷不出来皱眉，同学会在群里讨论考试范围。陆允川的心事挤在这些事情中间，没有被世界特别照顾，也没有被世界彻底忽视。这些周边生活把爱情从高处拉回地面。它提醒陆允川，心动不是唯一的课程，失落也不是唯一的作业；人总要在很多不浪漫的事情里，慢慢学会怎样对待一件浪漫的事。

多年后他终于愿意承认，她不需要在他的青春里承担一个完整解释。她可以不解释，也可以继续往前走。苏晚枝不会知道这些页数，夏知遥也不会知道，这样很好。这部分事实后来才被陆允川补进小说。补进去以后，故事没有变得轻松，却变得诚实了一些。

风从窗缝钻进来，把试卷角吹起，又很快压回去，像人心里反复犹豫的念头。他把这一幕和诗歌放在同一页，并不是为了显得文雅。诗对那时的他来说，是一块可以暂时放下重物的石头；重物还在，只是手终于能松开片刻。

原创仿唐绝句写完以后，他在下面空了一行。那一行空白比诗句更安静，像把解释留给未来，而不是逼当时的任何人马上回答。

他写仿唐绝句时，会下意识把情绪往景物里推。秋灯、远水、寒枝这些词替他挡住了最直接的疼，也让疼有了可以安放的形状。因此这些插页不是装饰，也不是为了让青春显得更漂亮。它们更像一排小灯，照着他从自我感动走向自我审查的路；灯光不强，但足够让人看清脚下有水。

很多年以后他回看这页，最刺眼的不是她的冷淡，而是自己把全部生活押在一个回应上的样子。他越长大，越不愿把这段青春说成谁辜负谁。辜负当然存在，可它不是全部；全部里还有胆怯、误会、虚荣、善意和没学会好好告别的年纪。

下午的课间很短，短到只够从教室走到水房再走回来。可他偏偏在这十分钟里想了很多事，想未来，想成绩，想某个名字，也想自己是不是从一开始就把普通日子看得太像预兆。这一类日常在小说里反复出现，是因为陆允川后来相信，真正改变一个人的不是某个爆炸瞬间，而是爆炸之后仍要买饭、交作业、赶公交、给手机充电的无数普通时刻。

多年后重写这一章，他把原来一整段自怜改成了两个人的动作。改完以后，人物没有那么可怜，故事却更宽了。他没有因此变得完全宽容，也没有把所有疼痛都解释成成长礼物。他只是终于愿意承认，自己和别人都曾经笨拙，而笨拙不必永远被写成罪。

他把旧照片重命名，名字取得很普通，普通到像是终于允许它只是照片。这个动作很小，小到放进当年的日记里可能会被嫌不够痛。可真正能留下来的，常常正是这些不够痛的小动作，它们让人相信那一年确实活过。

在这一章的末尾，陆允川没有得到答案。纸箱胶带被拉开时发出很长的一声响。他只是比上一章多看见了一点自己，多承认了一点别人，多放松了一点攥紧的手。

他不再急着证明谁错了。青春里很多错误没有审判席，只有一个人慢慢学会把手从旧门把上松开。那一刻并没有答案，只有风把窗帘吹起一点，又缓缓落下。故事继续往前走，像水流过桥洞，没有替任何人停下，却把所有经过都带成了更低的声音。
