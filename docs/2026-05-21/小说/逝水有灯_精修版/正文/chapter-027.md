# 第027章 第一次说话

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：教室前门和讲台之间

## 诗歌插页 P027：原创仿唐绝句

一语初逢半日迟，
檐前谁记旧年时。
少年不解平常意，
却把闲声唱作诗。

事情真正发生时，比陆允川在草稿纸上排练的任何场景都更短。那天下午课间，物理课代表赵一磊抱着一摞作业本往教室里走——那摞本子大概有二十几本，摞得偏高，最上面那本已经在往外滑了。他在门口被人撞了一下——是一个从后门冲出去接水的男生，手里拿着水杯跑得急，肩膀撞到了赵一磊的胳膊肘——本子从手上滑出去，哗啦啦散了一地，几本滑到了讲台底下，一本飞到了第一排的椅子脚边。陆允川刚好从教室出来准备去上厕所，看到这一幕，弯腰帮捡。他蹲下去捡起最靠近自己的两本——都是封面朝下的，他把它们翻过来，拍了拍封面上的灰。抬头时发现对面蹲下的是夏知遥。她也同时在捡，她的膝盖跪在门槛边沿上，校裤膝盖处沾了一小片灰。她的动作比他快——已经捡了三本摞在腿上，本子封面朝上，书脊对齐，正伸手去够第四本。

她的手指尖快碰到那本作业本的封面时，和他的手碰上了。不是指尖碰指尖——只是她的手背轻轻蹭到了他的大拇指外侧，凉凉的，可能因为刚从水房出来洗过手。两个人同时把手收回去，又同时说了一句"你捡"。然后停了半秒——都不知道谁该继续捡。

陆允川先说了一句话，声音不大——他甚至不确定自己有没有把嗓子里的那口气顺出去——像把一块石子轻轻地放到地面上："你捡吧。"她的手指顿了一下，然后继续动作——她把那本捡起来，书角在拇指上刮了一下，摞进自己手里，站起来。站起来的时候膝盖上的灰印子还在，她没拍。她说"谢谢"。两个字很平，没有任何多余的语气——就像对任何一个帮她捡东西的同学那样，尾音很短，说完就收了。但她说话的时候看了他一眼——不是那种正视，是说话时习惯性地把眼睛往声音来源那边投了一下，零点几秒。陆允川也站起来，把自己捡的两本放到她摞的顶上——放的时候手指刻意避开了她的手——说了声"没事"。然后她转身走回座位把本子放下，接着去水房了——她的水杯还放在水房的窗台上，杯盖敞着，里面泡着一袋茶包。

这是他们之间第一次说话。完整的对话只有三个词——"你捡吧""谢谢""没事"。加起来六个字，时间不超过四秒钟。但陆允川把这六个字拆成了无数个片段：她的"谢谢"里有没有笑？没有，但声音比平时软了半度。他的"没事"音量够不够？够，不过听上去像是先吞了半个音节才说出口的。

程砚在教室里目睹了整个过程。他本来在座位上喝酸奶，看到门边的本子散落事件后把酸奶放下了——吸管还插在嘴里，眼睛一眨不眨地看完了四秒半的全程。等陆允川回座位后，他先沉默了一会儿，然后说了一句话："六个字——你捡吧、谢谢、没事。你知道你今天的外交发言用了多少字吗？六。你知道你昨天草稿纸上排练的对话加起来多少字吗？大概两百。"陆允川把课本翻开，翻错了页，翻到了上周的课文，又往回翻："你数过了？""我没数你的草稿纸，但我数过对方——她说完'谢谢'就转身去水房了。你觉得她会在意那六字台词吗？她不会。她的水杯还在窗台上等她，她的下一节课笔记还没抄完，她的脑子里百分之九十九装的是物理作业截止日期。所以那六个字对她来说是日常，对你来说是里程——你们俩对同一件事的级别定义不一样，这是正常的。"程砚继续说，"但恭喜你。你终于正式跟她说话了。从今天开始你算是在她的世界里正式注册了——用六个字。注册费是零，体验版，永久免费。"陆允川没从课本上移开眼睛，但他翻书的速度明显慢了——那页物理课本上的受力分析图一个箭头都没看进去。

那天晚自习前，陆允川在日记本上写了很长一段，措辞极其隆重——用了"命运""转折""奇迹"这些词，还写了一句"我们的手背在作业本的边际线上短暂相触，像两艘船在同一个纬度错身而过"。写完以后自己看了两遍，耳朵烧得厉害，把这一页撕掉了——撕的时候纸页在胶装脊上留了一小截锯齿边。他重新翻开新的一页，深吸一口气，只写了一行字：今天下午课间，帮她捡了三本物理作业本。她说谢谢，我说没事。字迹很淡——他怕自己又写多了，刻意放轻了笔压。他把这一页合上，不再往下写。那六个字不需要续写任何句子——它们本身就是完整的，像一枚已经铸好的硬币，再往上加任何东西都是多余的刻痕。
