# 第054章 一瓶水

时间：2012年10月中旬

地点：教室

卷次：卷三：此心藏久

下午第二节课。体育课。高三体育课。

老师让大家跑了两圈，然后做了十节体操。剩下的时间自由活动。女生基本都回教室了——有的在写作业，有的趴在桌上补觉。男生一部分去打篮球，一部分在操场边坐着聊天。

顾临风跑完步之后去了小卖部。

小卖部在食堂旁边。一间平房。门口支着一把大遮阳伞。夏天的遮阳伞还没收——伞面上落了一层灰。卖东西的是个中年女人，圆脸，不爱说话。拿东西。找钱。下一个。

顾临风在冰柜前面站了站。冰柜里是冰的饮料——可乐、雪碧、冰红茶、矿泉水。他拉开冰柜门，手伸进去。碰到一瓶农夫山泉——冰的。他拿起来看了看。放回去了。然后绕到旁边的常温货架上，拿了一瓶常温的农夫山泉。

一块五。他付了两个钢镚儿——一个一块、一个五毛。

水瓶外面有点凉——常温货架挨着冰柜，还是沾了点冷气。他用手掌把水瓶捂了一会儿。直到瓶身不再凉了。

回教室的路上。他把水瓶拿在手里。右手的拇指在瓶盖上来回摩了好几遍。瓶盖是白色的塑料。瓶身是透明的。农夫山泉四个字是红色的。标签上画着两座山。

到教室楼下的时候。他停了一下。

他在楼下的水龙头前洗了个手。用洗手液搓了两遍。然后甩干。

上楼。每层十五个台阶。五层。七十个台阶。他一阶一阶往上走。手里攥着那瓶水。瓶身上的标签被汗沾湿了一点——他手心里在出汗。

他到教室门口。先往里面看了一眼。

教室里有三四个人。两个女生在写作业。一个男生趴在桌上睡觉。苏晚晴的座位空着——她还没回来。

他走进去。脚踩在地板上。地板响了一声——很老的木地板。有些地方松了。踩上去总会响。他停住。等了几秒。没有人抬头。走到她的座位旁边。

她的桌子收拾得很整齐。课本摞在左上角。练习册摞在右边。铅笔盒是铁皮的——和开学时是同一个，面上印着一只小花猫。她从高一就开始用的那个。桌面上有一个很浅的水印——大概是上午放水杯的时候留下的。

他把那瓶水放在桌面的正中央。

动作很轻。塑料底碰到木桌面，几乎没有声音。他放好之后，瓶身晃了一下。他用手扶住。等它稳了。退后一步。看了那个位置一眼——正中央。她回来一坐下就能看到。

然后他转身。走回自己的座位。

坐下去的一瞬。心脏在胸腔里撞了一下。很重。他深呼吸了一口。翻开英语书。找到一个看起来正在读的页面——第七十三页——然后把手指放在一行字上。假装在默念。他脑子里一个字都没有看进去。他的余光一直在往右前方向扫描。

第一个人推门进来——男生。后排的。手里拿着篮球。满头是汗。从后门进来。第二个人——语文课代表。从前门进来。手里抱着一摞作业本。第三个人——

苏晚晴。

她从后门进来。体育课后——脸上有点汗。鼻子尖上亮着。她用袖子擦了擦额头。马尾有点歪——大概是跑步的时候松了。她把发绳解下来重新扎了一下。用牙齿咬着发绳，两只手把头发拢到一起。然后绑好。

她走向自己的座位。

走到桌前。低头。看到了那瓶水。

她站住了。

她看了它几秒。然后环顾教室——前门。后门。讲台。窗边。她的目光扫过了教室里每一个人。最后。扫到了顾临风的方向。

他正在看英语书。第七十三页。目光固定在那行字上。脸上没有任何表情。但他的耳朵——耳廓的上半部分。正在以他自己都控制不了的速度变红。从耳垂红到耳尖。像有人拿红色水彩笔从下往上涂。

他把英语书立起来。挡住了自己的脸。

教室那边。她拿起了水瓶。翻过来看标签。农夫山泉。常温的。她拧了一下瓶盖——密封的。没打开过。

"谁放的？"

她的声音不大。但教室里很安静。所有人都听到了。没有人回答。写作业的女生抬头互相看了一眼。打篮球的男生在后面说"不是我"。语文课代表摇头。

"这瓶水谁放我桌上的。"

又问了一遍。还是没人回答。

她沉默了一会儿。把水放到桌子右上角。和她的课本放在一起。然后坐下。翻开物理卷子。

她没有喝。

但也没有扔掉。

顾临风把英语书立高了一点。翻了一页。第七十四页。全都是单词。他一个也看不进去。他的右耳全红了。耳朵烫得像刚从开水里捞出来一样。他用左手假装整理衣领——手指蹭到了耳廓。烫手。

赵一鸣从后门走进来。手里拿了一根冰棍。绿豆的。一元钱。他站在讲台旁边吃。吃着吃着看到了苏晚晴桌上的水。然后扭头看了顾临风一眼。

顾临风没有看他。但赵一鸣是懂的人。

他把冰棍吃完。把棍子扔进垃圾桶。慢慢走到座位上。坐好。然后侧过头。

"你放的。"

不是问句。是陈述。声音压得很低——只能让顾临风一个人听到。

顾临风没有说话。也没有看赵一鸣。他的眼睛还在英语书上——第七十四页。那个单词他看了一分多钟——abandon。放弃。但这个词现在跟他想的完全不是一回事。

赵一鸣没有再说什么。他靠在椅背上。摇了摇头。然后把自己的化学练习册翻开了。

下午第三节课。历史——高三还在上历史，因为会考成绩可以用到自主招生。没人认真听。一大半人在做数学题。一小半人在睡觉。

顾临风在做函数题。一道含参量的二次函数。对称轴。开口方向。根的分布。他用参数分离法——把参数挪到一边，函数挪到另一边。画了两个草图。确定了参数的取值范围。他不是真的在思考参数的问题。他脑子里转的角度只有一个——她到底喝没喝。

他忍了大半节课。忍不住了。

他把头往右前方偏了一下——角度很小。从历史老师的后脑勺旁边看过去。从第一排和第二排头之间的缝隙穿过去。看到了她的桌面。

那瓶水还在桌子右上角。瓶盖没有拧开过。标签还是翘着一角。她刚才拿起来又放下了。放的位置稍微偏了一点——从正中央挪到了右上角。靠墙——不容易被碰倒。

她在听历史课。笔在笔记本上走。左手搭在桌面上。中指指尖离瓶盖大概三厘米。喝水——倒不到——伸手就能够到。但她一直没有动它。

晚自习。晚自习的时间很长。从六点半到九点四十。中间有十分钟休息。

顾临风做了物理选择题——全部单选——然后做了一道物理大题。接着做了化学。做了数学。他把今天所有科目都过了一遍。然后拿出草稿纸——在右下角写了一行字。"

她还没喝。"

写完看了几秒。用笔杠掉了。

晚自习结束后。同学们陆续往外走。椅子在地板上摩擦的声音。拉链拉上的声音。关柜子门的声音。有人说了句"明天谁擦黑板——"有人应了一声。

顾临风故意收拾书包收拾得很慢。一支笔一支笔地放回铅笔盒。一本书一本书地摞整齐。然后站起来。

苏晚晴也在收拾东西。她把物理卷子夹进文件夹里。把课本一本一本放进书包。那瓶水还在桌上。书包拉链拉上了。她站起来。把椅子推到桌子底下。然后伸出了手。

她的手指碰到瓶盖。

拧开了。

白色的瓶盖被她拧下来，放在桌面上。她拿起水瓶，仰头——喝了一口。然后又喝了一口。然后拧上瓶盖。把水瓶子放进书包侧袋里。背着书包。走出了教室。

她的背影走过走廊。马尾在书包上面晃悠。

顾临风站在原地。看着她的背影消失在门外的走廊里。他在教室里站了大概十秒。然后把椅子放好。关灯。关门。走过走廊。下楼梯。

走到一楼的门口。外面的路灯亮了。橘黄色的一层光打在水泥地上。赵一鸣在路灯下面等他。手里拿着手机在看。

"喝了？"

顾临风看着他。

"你怎么知道。"

"你的耳朵。现在还是红的。"

顾临风下意识地用手摸了一下右耳。确实是烫的。

两个人在路灯下面站了一会儿。赵一鸣不想催他。让他自己开口。

"她喝了两口。"

赵一鸣说。"两口够了。你再放一瓶——她就知道是你了。"

顾临风没有说话。他低头看着路灯下的影子。两个人——他的影子和赵一鸣的重叠在一起。他往旁边让了一步。影子分开了。

"不会再放了。"

"为什么。"

"放太多。她会不舒服。"

赵一鸣沉默了一会儿。

"也是。她不是那种需要很多人献殷勤的女生。"

"嗯。"

"她需要的是一个人。安静地。记住她喜欢常温还是冰的。"

顾临风没有回答。但他的左手插在口袋里，摸到了今天小卖部找的那个五毛钱钢镚儿。钢镚儿被他的汗捂热了。

两个人走回宿舍。路灯的光晕里。小飞虫还在绕圈子。
