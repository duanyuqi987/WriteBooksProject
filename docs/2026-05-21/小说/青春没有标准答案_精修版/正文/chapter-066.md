# 第066章 校门外的公交



时间：2009年一月底，期末考试前两天



地点：校门外公交站、文具店门口



卷次：第三卷：雪落在两栋楼之间



2009年一月底，期末考试前两天，校门外公交站、文具店门口，文具店新买的透明笔袋先把这一天按进了十班的日常里。期末前两天，学生们在文具店买笔芯和2B铅笔，公交站比平时更沉默。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

邱可可平时笑得最大声，这天却反复检查准考证袋。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看文具店新买的透明笔袋，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

邱可可也在这一天被推到前面。邱可可的特点是外向，笑点低，能把尴尬变成笑声。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“我是不是少买了一支铅笔？”邱可可说。

“两支够，橡皮在这里。”常俊凡说。

“准考证袋放书包内层。”段璟说。

“你们俩今天像考务老师。”邱可可说。

“紧张就检查清楚，不要只在嘴上紧张。”胡向南说。

校门外的公交站到了冬天会显得更窄。学生、铺盖卷、饭盒和家长的电动车挤在一起，谁都想快一点。

路边小店烤红薯的香气很重，热气从铁桶里冒出来，把排队的人脸烘得忽明忽暗。

常俊凡把书包背带往肩上提了提，没有把手里的袋子递给谁。

段璟站在她旁边半步远的位置，不靠太近，也不故意退开。

宋乔嫌车慢，付小川嫌风大，周明澈则一直看路口，像他看久了公交就会出现。

车来时，人群往前涌，常俊凡先扶住旁边一个差点滑倒的女生，再自己上车。

段璟跟在后面，看见车门边的泥水溅到鞋面上，忽然觉得冬天的现实感很重。

公交开走后，校门口很快空了一半，剩下的人继续等，像每个人都有不同的回家速度。

校门外的公交没有悬在空处。公交站泥水、烤红薯热气和校服袖口，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，校门外的公交带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

胡向南进门时不一定立刻说话，鞋底在水泥地上一停，校门外的公交带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在邱可可快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。校门外的公交越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。文具店新买的透明笔袋让他想说话，又逼他先低头看题。

邱可可被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，校门外的公交留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。邱可可路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为校门外的公交翻身，谁还在被窝里背单词，旁边人都听得见。

公交从县城路口慢慢拐过来，车窗上蒙着白气，里面的人影一晃一晃。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理校门外的公交里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：文具店新买的透明笔袋、一只冻红的手、一句老师点名、一块还没扫干净的雪。

许念安站在公交站牌下跺脚，书包带勒得肩膀发酸。这样的细节没人会写进成绩单，却会留在同学之间。

魏然看见车灯从路口慢慢拐过来，先把同学的书包往里推。窗外风一过，纸页轻轻动了一下，像替谁把话咽回去。

马晓彤把冻硬的月票攥在手心，等车门开了才松开。这个小动作很快过去，却让教室更像一个真正在运转的班。

韩泽避开路边化开的雪水，鞋尖还是溅上一点泥。旁边的人没有特意看，可那一瞬间还是被段璟记住了。

何南星站在公交站牌下跺脚，书包带勒得肩膀发酸。常俊凡没有接话，只把手里的事做完。

余小满把冻硬的月票攥在手心，等车门开了才松开。冬天把这些小动作放大，让每个人的慌张都有了形状。

校门外的公交之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

邱可可没有因此变成特别醒目的人，可在段璟眼里，邱可可从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，常俊凡帮她把东西按顺序放好，段璟也学着把慌张变成动作。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
