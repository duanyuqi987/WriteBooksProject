# 第034章 暑假

时间：2011年7月-8月

地点：青阳县家中

卷次：卷二：窗边人影

暑假漫长。

七月的青阳镇很热。早晚凉快一点，中午太阳把地面晒得冒热气。树上的蝉从早上叫到天黑。

顾临风每天早上六点多起来。

先帮他爸搬东西。他爸养猪。院子里有个猪圈——三头猪，泥地，气味总是有的。他换上旧衣服穿雨鞋进去打扫。铲子、水管、石灰。干到八点出一身汗。然后回屋洗把脸。吃早饭。

白粥咸菜。他爸坐在对面。两个人没什么话。吃完他爸碗筷一推出了门——去镇上打牌或者去邻村帮忙盖房子。

上午他做暑假作业。坐在自己房间的桌子前。桌子靠着窗户。窗外是后院——竹竿上晾着衣服，墙角堆着柴。

做一会儿。停一会儿。

笔还是那支笔——笔帽上有指印的那支干了芯的黑色中性笔。笔帽还在铅笔盒最里层。他写作业用另一支——在镇上小店买的，五毛一支，出水不太匀。

中午做饭。他妈在镇上的纺织厂上班。中午不回来。他自己炒个菜——大多是土豆丝炒青椒。有时候加点肉末。

吃完饭洗碗。然后睡午觉。但大部分时候睡不着。

躺在床上。看天花板。天花板上也有裂纹——跟宿舍那个挺像。

下午三点以后阳光没那么烈。

他骑自行车出门。

家里的自行车是二八大杠。黑漆磨掉了大半。链条松。踩三圈响一次——咯吱。

他骑车穿过镇上的街道。街道两边是两层的楼房，底层是店铺——杂货铺、理发店、摩托修理铺。下午街上人不多。几个老人坐在家门口扇扇子。一只黄狗趴在树荫下。

他从主街拐进一条小巷。然后放慢了速度。

巷子尽头有一棵老槐树。树冠遮了大半个巷口。树干很粗——一个人抱不住。树下有块石板，被晒得很烫。

这是苏晚晴家那条巷子——他上次在县城书店门口看到她提着东西走进来了。后来回学校查了一下高一开学时填的信息表。她是青阳县的。不是镇上——是县城人。

他骑过巷子的时候往里面看了一眼。

巷子很深。两边是青砖墙。墙上有爬山虎。尽头是个院子门。门前有个水龙头。没人。

他骑过去了。

然后掉头。又骑了一遍。

还是没人。

槐树叶子一动不动。没有风。

他骑回家了。

暑假里他去了那条巷子很多次。每周两三次。每次下午三四点。每次骑车经过的时候往里看一眼。

从来没碰到过她。

有一次巷子里有个老人在门口择菜。看了他一眼。"找谁啊。"

"走错了。"

老人低头继续择菜。

他把车骑走了。骑得比平时快。

回到家。坐在桌前。翻开一个没用完的练习本。在最后一页写了几句。

"城中相见不知期，巷外老槐空自垂。"

写完看了一遍。用笔把"自垂"改为"待谁"。然后再改为"绿满枝"。

然后把诗的那一页撕下来对折放进抽屉。抽屉里已经有两张这样的纸了。

八月初。镇上下了几场大雨。猪圈的泥更烂了。他爸念叨了两天。最后买了新雨鞋。

雨过天晴那天下午他又骑车出去了。

巷子里有水洼。他把车停在槐树旁边。没骑进去。站在巷口往里看了一眼。

院子里传来电视的声音。少儿频道。大概是邻居家小孩在看动画片。

他站了一分钟。然后骑走了。

回到家。在日记本上写了几行字——但暑假在家不能写得太多。本子放在书包最底层。家里只有他一个人时他才拿出来。

"暑假过了一半。两个多月没见到她。还有二十天开学。"

"我今天去她巷子了——觉得自己像个小偷。"

"写了一首诗。写得很差。全撕了。"

合上本子。放回抽屉最里面。上面盖着课本。

他走到院子里。夕阳把猪圈的棚顶染成了橙色。空气里混着泥土和青草的味道。远处稻田有人在放水。水流的声音很轻。蝉已经不叫了。

九月一日快点来吧。
