# 第237章 同学群的婚讯

所属部卷：第五部：逝水有灯
时间：工作日午后
地点：公司茶水间

## 诗歌插页 P237：原创仿唐绝句

群中红帖午灯斜，
旧友纷纷问近涯。
不向姓名深处去，
一声祝福到谁家。

陆允川在群里回了一句“新婚快乐”。很快被新的祝福盖过去。有人问夏知遥最近在哪里，有人说好像在外省工作，去年回过一次县城。又有人提到苏晚枝，说她可能在南方城市，具体不清楚。

这些名字出现得很随意，像聊天时顺手翻到旧照片。没有人知道它们曾在陆允川那里占过多大位置。群里的人只是说近况，说不确定的消息，说“有空一起聚”。成年人的消息常常这样，半真半旧，带着一点热闹，也带着很多不知道。

陆允川的手指停在输入框上。他本能地想问一句“她现在怎么样”，又把手收回。知道了又能怎样？不知道也并不妨碍他祝福别人结婚、回会议室开会、晚上按时下班。

董向南在群里发语音，声音很高兴，感谢大家到时候来喝喜酒。陆允川点开听完，回了一个红包。金额不大，备注写“祝顺利”。

茶水间门口有人叫他：“评审会开始了。”他把手机锁屏，端着咖啡往会议室走。屏幕暗下去前，群里还在刷表情。那些旧名字没有消失，只是终于不再需要他追着确认。
