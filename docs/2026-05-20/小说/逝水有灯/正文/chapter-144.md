# 第144章 她说别追了旁观

所属部卷：第三部：寒风里的花
时间：六月下旬
地点：图书馆东门台阶

## 诗歌插页 P144：原创现代诗

她把话说得很轻，
轻到怕伤人，
又重到必须被听见。
台阶上的水瓶晒热了，
她的手心也在出汗。

苏晚枝说那些话前，一夜没睡好。她在床上翻了几次身，室友问她是不是期末压力大，她说有点。

压力当然有，更多的是陆允川。她不讨厌他，甚至愿意承认他敏感、细心、可靠。可这些优点和压力并不矛盾。一个人可以很好，也仍然让另一个人喘不过气。

早上看见他拿着两瓶水站在东门，她心里那根线终于绷紧。她知道再不说，陆允川会继续把照顾做得更细，而她会越来越难拒绝。

她没有接水。拒绝一瓶水很小，可那是她能做的第一步。

说“别追着我走”时，她手心在出汗。她怕陆允川难堪，也怕自己说得太软，对方听不见。她尽量不用“烦”“怕”这些字，尽量把话落在动作上：等、找、带水、变脸。

陆允川问“那我以后不找你了”时，她差点心软。可她知道这句话把问题推向了另一个极端，好像她只要设边界，就成了要断绝关系的人。

所以她说：“不是不说话。是别追着我走。”

这已经是她能说出的最清楚也最不伤人的版本。说完以后，她进图书馆，找到座位，翻开书，却半页都没看进去。
