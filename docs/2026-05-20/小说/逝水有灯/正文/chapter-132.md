# 第132章 她和别人说笑

所属部卷：第三部：寒风里的花
时间：六月初
地点：校园超市门口

## 诗歌插页 P132：原创仿唐绝句

门前汽水碰轻声，
笑语随风过短亭。
莫把旁人寻常事，
误听成己不公平。

苏晚枝看见陆允川时，笑意还没完全收住。她自然地打招呼：“你也买水？”

陆允川点头，“嗯。”

旁边那个男生把其中一袋东西递给她，“那我先去活动室，你慢点。”

“好，辛苦。”苏晚枝接过袋子。

男生走后，陆允川看见袋子里是社团活动要用的胶带、纸杯和几瓶矿泉水。事实很普通，普通到他的脸色显得没有理由。

苏晚枝问：“你怎么了？”

“没事。”

她看着他，没有立刻接。超市玻璃门开合，冷气一阵阵漏出来。陆允川手里的矿泉水已经结了一层水珠，顺着指缝往下滴。

“你刚才不是没事的表情。”苏晚枝说。

陆允川想说没有，想说只是热，想问那个男生是谁。可话到嘴边，他自己先觉得难看。最后他说：“我以为你今天没空。”

苏晚枝眉心轻轻动了一下，“我没空去听讲座，不代表我不能做别的事。”

这句话很平。越平，越让陆允川无处躲。他把矿泉水放到收银台上，扫码付款，连祁野要的冰红茶都忘了。

走出超市时，苏晚枝没有追问。她拎着袋子往活动室方向走，步子比平时快一点。陆允川站在门口，看见汽水瓶在袋子里轻轻撞了一下，声音很小，却让他记了很久。
