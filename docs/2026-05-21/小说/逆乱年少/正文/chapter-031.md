# 第031章 三句话

时间：2011年2月下旬，返校火车上

地点：K152次列车九号车厢

卷次：第二卷：冬天的图书馆

火车过了松林站，窗外的田野还是灰绿色的。

林玉起坐在苏安晴旁边，膝盖上放着书包。那个中年男人下车以后，他和她之间隔着一个空出来的座位扶手——铁的，冰凉，但他的手肘不敢挨上去。

刚才的对话停在了四楼的话题上。

"四楼比较安静。二楼人太多了。"

苏安晴说完这句话就低头翻书了。林玉起不确定她是真的想继续看书，还是跟他一样——只是需要一个低头不看对方的理由。

车厢里的广播开始卖盒饭。"旅客朋友们，午餐时间到了，餐车供应热菜热饭——"

"你饿吗？"他问她。

"不饿。带了面包。"

她从书包侧袋里掏出一个面包——超市里那种三块钱一袋的红豆面包。包装袋上印着一只卡通兔子，跟她这个人不太搭。

她撕包装袋的动作很利落——沿着锯齿口一撕到底，面包完整地露出来，没有碎屑掉在衣服上。林玉起注意到她的手指——不是那种不沾阳春水的手，指甲剪得很短，指尖有一点干裂，是冬天用冷水洗手留下的痕迹。

"你在中文系都读什么书？"她咬了一口面包，边嚼边问。她的嘴不大，嚼东西的时候腮帮子微微鼓起，然后很快平复下去。

"这学期主要读现代文学。鲁迅、沈从文、张爱玲。"

"张爱玲好。"

这三个字让林玉起意外了一瞬间。他没想她会读张爱玲——教育学专业的人，他以为只看教育原理和教育心理学。

"你读过她的什么？"

"《十八春》。"苏安晴把面包咽下去，拧开矿泉水瓶喝了一口。"图书馆借的。看完以后好几天心里不舒服。"

"因为结局？"

"不是因为结局。"她想了一下。"是因为她说'我们回不去了'。不是回不去那个地方，是回不去那个时间——人在，时间没了。"

林玉起没有说话。他把头靠在座椅靠背上，看着对面车窗上映出来的模模糊糊的人影。苏安晴说这句话的时候语气很平，像在复述一个跟自己无关的知识点。但他总觉得她在说别的东西——或者什么都没说，只是他过度解读了。

火车又钻进一个隧道。车灯闪了一下。

"你写诗吗？"

苏安晴忽然问他。林玉起转过头，在隧道出口漏进来的光里，她的脸被照得很白。

"你怎么知道？"

"我去过诗社。何南星念过你写的东西。"

林玉起的手心又一热。不是因为她去过诗社，而是因为她记住了。她记住了他写的诗，记住了何南星念过他的名字，记住了他是中文系的。

"写得不好。"他说。

"挺好的。"她说，"有一首写图书馆四楼的。说有人在对面翻书，风吹进来，你闻到皂角味。"

林玉起屏住了呼吸。

那首诗。第一首诗。他第一次在诗社读的那首——"银杏还没黄的十月，我坐在图书馆四楼，你在对面翻书"。那首诗里写的"你"就是她。而她当着面把这首诗念了出来，虽然她不知道那个"你"就是她自己。

至少他不确定她知不知道。

"你还记得那首诗？"他的声音稍微有点干。

"最后一句可以不要。"苏安晴学何南星的语气说了一遍，然后笑了笑。

这个笑很轻。嘴角弯了一点，眼睛也弯了一点，然后又收了回去，像冬天早晨水洼里一层薄冰被太阳照了一下——亮了一瞬间，融化的速度比亮起来还快。

林玉起见过她在图书馆的样子，见过她在超市的样子，见过她在校门口小卖部的样子。这是第一次见她笑。

火车的速度开始变慢。广播里说前方到站江城站，播音员的声音带着标准的火车站腔，每个字都咬得很重。

车厢里的人开始站起来拿行李。苏安晴把面包包装袋团成一团，塞进书包侧袋里，然后站起来。她比他矮一个头——坐着的时候不觉得，站起来了才发现。她的头顶大概到他的下巴。

"开学见。"她说。

"开学见。"

她走了。过道里的人很多，她的背影在人群里时高时低。林玉起坐在原位没有动。窗外的楼越来越密——江城到了。

三个半小时。他们说了大概三句话——关于封丘、关于图书馆四楼、关于张爱玲。每一句都很短，每一句都停留在日常的边界内，没有越过去一毫米。

但够他回想很久了。

他站起来拿行李的时候，才发现自己的书包拉链开着一半。刚才太紧张，忘了拉上。里面那本软皮笔记本的封面露了出来，第二卷第一页写着四个字——

"冬天的图书馆"。
