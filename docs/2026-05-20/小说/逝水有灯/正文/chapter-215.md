# 第215章 南方出租屋多年后

所属部卷：第五部：逝水有灯
时间：多年后
地点：南方城市出租屋

## 诗歌插页 P215：原创仿唐绝句

旧屋空余白漆痕，
雨声曾替夜关门。
行囊不带当年痛，
只记窗前一盏昏。

几年后，陆允川搬离那间出租屋。房东来验房，先看墙面，再看水电，最后指着窗边那块补过的墙皮说：“这里你自己弄的？”陆允川说是，房东摆摆手，说不扣钱。

他把最后一箱书搬到楼下，回头看了一眼。房间空下来后显得更小，折叠桌留下四个浅浅的印子，窗边仍有一点潮味。那一年雨季、加班、旧照片和滴水声，都被空屋子轻轻放开。

新租客下午来看房，是一个刚毕业的女孩，背着电脑包，问网络稳不稳。房东说挺稳，陆允川站在门口补了一句：“水龙头换过垫圈，暂时不漏。”女孩笑着说谢谢。

他没有告诉她，自己曾在这张桌子前整理旧日，也没有告诉她哪一晚雨最大。屋子会接住下一个人的生活，不需要继续保存他的全部情绪。

多年后写到南方出租屋，陆允川没有把它写得很惨。它的确潮、窄、租金高，可它也让他学会了把过去放进一个文件夹，学会了修一点小东西，学会了在没有观众的夜里安静下来。
