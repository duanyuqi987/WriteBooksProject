# 第007章 黄河大堤上的风



时间：报志愿前的一个下午



地点：黄河大堤边和通往县城的土路



卷次：第一卷：九年级的志愿表



报志愿前的一个下午，黄河大堤边和通往县城的土路。自行车后座上晃来晃去的资料袋把这一天的开头压得很实。大堤下麦田一片发亮，远处有人在地头晾晒金银花，苦香味被风推过来。段璟站在旁边，手上还有铅笔芯蹭出的黑印，耳朵却一直留意着常俊凡那边的动静。那时候他们还没有后来那么多说不清的关系，只是同一场中考里两个被分数推着往前走的人。

大家都说报考是分数的事，可骑到大堤上才发现它也是离开和留下的事。大堤上的路不平，自行车轮压过碎石，车把会轻轻晃一下。他们说话也像骑车，不敢一下子冲太快，怕摔，也怕把真正想问的事撞出来。

常俊凡想确认自己不是因为一时赌气才选封丘一中。她做事不急，先把笔袋摆正，再把要问的问题写在草稿纸右上角。段璟后来才明白，那不是冷淡，也不是故意把人挡在外面。她只是很早就知道，别人可以热闹，可以起哄，可以替她猜测，但真正要承担结果的人只有她自己。

段璟想跟她并排骑一段，却总被路上的石子和自己的心虚打乱节奏。他不是不会说话，只是一碰到常俊凡，平常那些贫嘴和硬话就像被粉笔擦抹了一遍，剩下一点不成句子的灰。他怕自己显得太在意，又怕自己真的一点也不在意。初三的男生常常这样，明明心里已经绕了好几圈，开口却只剩一个嗯。

他们的对话没有谁故意说得漂亮，都是初三学生能抓住的那些实话。“你们都报一中？那我要是不报，显得我很没出息。”赵远航说。“别拿别人给自己定出息。”常俊凡说。“你真不怕考不上？”段璟说。“怕。可是怕也不能替我填表。”常俊凡说。话说完以后，谁也没有立刻把目光放到对方脸上，仿佛只要盯着桌面、墙面或手里的纸，心里的动静就不会被看出来。

赵远航也在场。骑车从坡下冲上来，嘴上喊着不累，脚下一直没停。这种小动作比豪言壮语更像那一年：大家都说自己不怕中考，可水杯里的水少得很快，草稿纸一张接一张被揉皱，课桌边缘全是指甲抠出来的小白印。

常卫东修车铺门口挂着旧轮胎，常俊凡出门前帮父亲递了两把扳手，手指还有一点机油味。家庭没有在他们的青春里大声喊口号，它只是藏在饭盒、车票、洗好的校服和一遍遍叮嘱里。封丘县的日子也是这样，街道不宽，消息传得快，谁家孩子能不能上一中，很快就会成为亲戚饭桌上绕不开的话。

段璟握紧车把，第一次明白常俊凡的清醒不是不害怕，而是害怕的时候还愿意往前走。这个动作没有被任何人郑重记录下来，甚至连旁边的人都未必注意到。但段璟记住了，因为它让他第一次意识到，所谓喜欢不是突然出现的一句表白，而是很多个没说出口的瞬间慢慢叠起来。

县城里的夏天靠近得很快。午后窗台烫手，作业本边缘卷起一点毛边，水房里有人一边接水一边背英语单词。广播里偶尔提到北京奥运，老师提到中考，家长提到一中，学生提到放假。所有大事小事挤在同一张课桌上，谁也没有多余的地方安放自己的慌张。

大堤上的风把他们的校服吹得鼓起来，三辆自行车一前一后，像三张还没写完的志愿表。段璟那时还不会把这种场面写成故事，他只会把它塞进心里，等很多年以后再慢慢认出：原来青春最真实的部分，不是后来回忆时替它加上的光，而是当时那些尴尬、具体、说错话又来不及补救的细节。

黄河大堤上的风比校园里的风宽。人在上面骑车，很容易把话说得比平时远一点。赵远航和邱可可他们一会儿冲到前面，一会儿又落到后面，给段璟和常俊凡留下短短一段并排的路。

堤下有人翻晒金银花，白黄相间的一片摊在竹席上。常俊凡停下来看了一眼，说小时候总觉得这种花泡水像药。段璟说我妈也泡，苦。她纠正他说，不是苦，是清。

远处的麦地被风压出一层层暗纹，像试卷上没擦干净的铅笔印。段璟想问她如果去了不同学校怎么办，可话到嘴边，只问了一句你热不热。常俊凡看他一眼，说还行。

赵远航在旁边留下的小动作，也让这一章不只围着段璟和常俊凡转。封丘县的学生多数早早学会看大人的脸色：家里宽裕不宽裕，老师看好不看好，自己能不能把一次机会抓住。这些判断没有写在卷面上，却时时刻刻挤在卷面旁边。

常俊凡很少把情绪说满。她说可以，就是可以；说不改，就是不改。这种短句有时会让段璟不知道怎么接，可也正因为短，才不像临时编出来的温柔。她给人的距离感，不是拒绝所有人，而是提醒别人别替她安排人生。

段璟的笨拙则更容易被看见。他会在该说话时沉默，在该沉默时又憋出一句不合适的话。付小川他们偶尔笑他，他也会顶嘴，可常俊凡一看过来，他的顶嘴就像被人抽掉了半截，只剩下耳朵发热。

那时候大家还没有智能手机可以随手查消息，更多事情靠公告栏、电话、老师通知和同学传话。一个消息从办公室传到教室，往往会变出好几个版本。有人说分数线会涨，有人说一中要扩招，有人说住校床位紧，越传越像一场小小的风。

封丘县的夏天有自己的声音。三轮车从巷口过去，车斗里响着铁盆；早餐摊收摊时，塑料凳一只套一只；远一点的地方，麦场上有人拍打编织袋。这些声音落进少年人的心里，不会立刻变成故事，却会在多年后突然浮出来。

大堤上的路不平，自行车压过碎石时，车把会轻轻偏一下。段璟不敢骑得太快，怕追得太明显；也不敢落得太远，怕常俊凡和前面的人说话时自己听不见。

堤下有人翻晒金银花，竹席铺了一片，白黄相间。常俊凡停下来看，说小时候总觉得金银花茶像药。段璟说我妈也泡，苦。她纠正他说，不是苦，是清。

树莓摊旁边支着一把旧伞，摊主把竹篮下面垫了湿毛巾。赵远航买了一小包，刚吃一颗就皱脸。常俊凡说酸就别逞强，段璟在旁边笑，又怕笑得太明显。

大堤上的风比校园里的风宽，话也容易被吹远。段璟想问她如果没考到同一所学校怎么办，可这问题太像把害怕摊开给人看，他最后只问了一句你水够不够。

常俊凡骑车时背挺得直，资料袋被她夹在车筐里。风几次要把袋口掀开，她都伸手按住。段璟看着那个动作，忽然觉得她选择一中，大概也会这样按住自己的不安。

远处麦田被风压出一层层暗纹，像试卷上没擦干净的铅笔印。赵远航他们骑到前面去，回头喊快点。段璟应了一声，却还是把速度放在常俊凡旁边。

县城在大堤那边显得很小，校门、文具店、补课班都被风吹成一个模糊方向。常俊凡说，去哪里都得先把眼前的题做完。段璟说你说话像老师。她说老师至少知道重点。

他们没有在大堤上说什么大话。没有永远，也没有一定。那时他们能抓住的，只是一段并排骑车的路，一小包酸甜的树莓，还有封丘一中这四个越来越清楚的字。

这段日子里，封丘县像被中考轻轻拧紧了一圈。街上还是那些卖菜的三轮车、修车铺和文具店，可学生走过去时，总会下意识把书包带往肩上提一提，好像这样就能把未来背稳。

常俊凡不是不会累。她只是习惯先把事情排好，再允许自己短暂地松一口气。文件袋、笔袋、准考证、通知书，每一样都被她放在固定位置。段璟看见这些，才慢慢明白她的要强不是给别人看的。

段璟的心思则更乱。想到封丘一中时，他会想到校门；想到常俊凡时，他会想到她写字时压住纸角的手。两种念头挤在一起，让他有时分不清自己到底是在向学校靠近，还是向她靠近。

大人们的担心没有停止。生活费、住校、路费、孩子能不能适应，这些问题每天换一种说法出现。可他们也没有把孩子往后拉。很多时候，父母的支持只是沉默地把钱数好，把被褥晒干，把一句注意身体重复好几遍。

赵远航的存在让这段前史多了一点旁边的声音。有人贫嘴，有人算分，有人说自己无所谓，真正到了选择面前，又都把无所谓收了回去。群像不是热闹的装饰，它提醒段璟，每个人都在用自己的办法过关。

北京奥运的新闻越来越密，电视里常出现倒计时。县城大人谈起奥运时带着兴奋，学生们听见倒计时，却总会想到另一个倒计时：开学、报到、新班级，以及还不知道会不会同班的那些人。

如果只把这几章写成喜欢与被喜欢，就会太窄。真正让人信服的，是喜欢旁边还有考试、家境、选择和县城日常。青春不是悬在空中的情绪，它有尘土、有饭菜味，也有不得不填下去的表格。

段璟开始学着不把每一次沉默都理解成拒绝。常俊凡沉默时可能在算题，可能在想家里的话，也可能只是累了。一个人越是真实，就越不该只被另一人的心事解释。

常俊凡也不是冷冰冰的答案。她会提醒别人别吃太凉，会把借来的东西归整好，会在别人起哄时皱眉。这些小动作没有一句像表白，却比表白更能说明她是什么样的人。

到了夜里，县城慢慢安静下来，只有远处车声和院子里的虫鸣。段璟躺在床上，把当天的场面一遍遍翻出来，最后发现最难忘的不是某句漂亮话，而是常俊凡把自己的选择握稳的样子。

封丘一中这个名字在他们嘴里出现得越来越频繁。起初它只是宣传单上的校门，后来变成老师讲过的分数线、家长算过的生活费、同学拿来互相试探的一句你报不报。段璟每次听见这四个字，都会下意识去看常俊凡。

有人把离别说得很夸张，好像初中一结束，所有人马上就会被命运吹散。也有人故意说得很轻，说县城就这么大，早晚还能遇见。常俊凡听见这些话，通常不反驳，也不附和，因为她知道遇见容易，重新站到同一条路上并不容易。

段璟家离县城不算远，却也不是抬脚就到。早晨路上有卖菜的三轮车，有背书包赶车的学生，有刚从地里回来的大人。那些味道和声音混在一起，让封丘县不像地图上的名字，而像每天都要经过的生活。

常俊凡的学习方法有一种笨拙的可靠。她不喜欢把笔记涂得花里胡哨，只在最容易错的地方画小方框。如果一道题错了三遍，她会在旁边写明错因：看漏条件、公式混用、心急。

段璟偷偷学过这种写法，写到第三天就乱了。页边全是自己补给自己的借口。他看着那些借口，忽然有点佩服常俊凡，也有点不好意思。喜欢有时不是被一个人的光照到，而是被她的方法照出自己的毛糙。

同学们说话没轻没重，常把未来挂在嘴边。可真正落到自己身上时，谁都不轻松。有人怕考不上，有人怕上了也跟不上，有人怕住校，有人怕家里花钱。段璟第一次发现，青春不是只有热闹，也有很多不能轻易说出口的计算。

这种计算并不丢人。县城少年长大得早，早在他们懂得浪漫以前，就先懂了路费、饭费、资料费和家长眼里的期待。常俊凡清醒，段璟敏感，他们正是在这些现实里慢慢靠近。

所以九月还没到，九月已经在很多细节里提前出现了。新买的笔芯，晒过的被褥，压在字典里的宣传单，电视里反复出现的奥运倒计时，都在提醒他们：初中快结束了，下一段路不会等谁想明白以后才开始。

少年人把许多事都说得像玩笑，是因为正经起来太难为情。赵远航开了一个不轻不重的玩笑，大家笑过以后，又各自看向别处。常俊凡没有拆穿任何人，段璟却从那阵笑里听出了害怕：害怕考砸，害怕分开，害怕自己其实没有想象中勇敢。

风在这一段里不是装饰。它吹起校服下摆，也吹乱资料袋里的纸。段璟追着一张纸跑了两步，常俊凡先他一步踩住纸角，弯腰捡起来递给他。她说拿好，别什么都等风停了再做。段璟没接上话，只把纸夹得更紧。

那几天他们都在学习一种新的分寸。不是所有关心都能问出口，不是所有同行都能被叫作约定。常俊凡往前走的时候很少回头，段璟就学着不把每一次回头都当成答案。这份笨拙的克制，比直接表白更像当时的他们。

有些场面如果只写一句他们都很紧张，就会轻得像没发生过。真正让人相信它发生过的，是铅笔芯断掉以后谁递了一把小刀，是水房排队时谁往旁边挪了半步，是车铃响起时谁下意识护住了文件袋。段璟和常俊凡的靠近，也只能从这些动作里慢慢长出来。

饭盒、脸盆、准考证套和透明文件袋，都是那一年反复出现的东西。它们不抒情，却比抒情可靠。段璟记得常俊凡把文件袋抱在胸前走过楼梯口，也记得自己把饭盒塞进书包时，里面的勺子撞出一声闷响。生活先把他们推到这些物件面前，再让他们慢慢明白选择的重量。

大人们谈未来时，常常把声音压低，可压低并不等于学生听不见。段璟听见过父亲说要不就让孩子拼一回，也听见过母亲说住校可别冻着。常俊凡听见过母亲问学费，也听见父亲说姑娘想去就让她去。那些话没有直接出现在课堂上，却一直跟着他们进考场。

同学之间的关系也在这种时候变得微妙。平时一起打闹的人，到了报志愿时会突然认真；平时不怎么说话的人，会在排队时递过一支笔。赵远航有自己的紧张，也有自己的玩笑。群像不是背景，它让段璟知道，自己的心事只是青春里许多心事中的一个。

晚上电视里播新闻，奥运的画面和彩色广告交替出现。段璟一边看，一边给铅笔削尖；常俊凡可能也在另一个屋子里整理书包。他们并不知道彼此正在做什么，却都被同一个夏天推着往九月走。

如果多年以后再删掉这些细节，只留下他们喜欢过彼此，故事会一下子轻很多。真正让这段青春站起来的，是他们还没懂爱情时先懂了考试、家境、县城和选择。这些东西不浪漫，却给浪漫留出了可信的地面。

常俊凡不喜欢别人把她写成遥远的光。她会皱眉，会算错，会因为母亲一句别太累而突然红眼圈，也会在第二天照常把错题整理完。段璟喜欢的不是一个被想象出来的完美女孩，而是这个会把文件袋抱得很紧、却仍然往前走的人。

段璟也不应该只站在暗恋的位置上。他要面对的同样有成绩、家里、钱和选择。他如果只是看着常俊凡发呆，这个故事就会变窄；他真正开始长大，是从意识到她也有自己的难处，而不是只在他的青春里发亮开始。

段璟把书包带往肩上提了提，布料磨过校服领口，声音很轻。

常俊凡回头看了一眼，又很快转回去，像只是确认身后有没有人挡路。

风把路边的纸屑吹到墙角，谁也没有立刻去捡。

夏天还在往前走。

远处有人喊了一声，声音很快被热风推散。
