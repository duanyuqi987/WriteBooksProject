# 第060章 一个好字

时间：2011年六月中旬，周五下午

地点：图书馆二楼靠窗倒数第三个座位

卷次：第三卷：春天的诗

诗集拿回来是在三天后。周五下午，图书馆，同一个位置。

苏安晴把绿色硬壳笔记本放在桌上，推过来的方向跟他那天推过去的方向完全一样——从左边往右边推，手没有离开桌面。

"看完了。"她说。

林玉起把诗集接过来，没有立刻翻开。他把它放在手边，离自己的右手大概五厘米。封面上多了一个很浅的指印——不是脏，是用手指蹭过很多遍以后留下的那种淡灰色痕迹，像一片非常薄的云。

"你觉得——"

"你自己看。"

苏安晴说完就低下头继续看她的《教育心理学》。她的铅笔在笔记本上写了几行字，林玉起没看清楚写的是什么。

他翻开诗集。

第一页没有变。第二页没有变。翻到第二十页的时候，他注意到铅笔写的四个字——"我也喜欢"——写在《花谢了以后》那首诗的旁边。字很小，铅笔的颜色很淡，几乎要凑近了才能看到。她的字跟她的说话方式一样——干净、简洁、不拐弯，每个笔画的起落都很果断。

他继续翻。在第三十五页——《不是巧》那首诗旁边——她又写了几个字："上课认真听讲"。这句话跟诗的内容毫无关系，但他知道她在回答什么。那天他写的最后一句是"把整个春天走成了一根朝向你的指针"。她写"上课认真听讲"——不是批评，是一种很轻的提醒，像把一杯水放在一个人旁边的桌子上，什么都没说，只是说"渴了就喝"。

翻到最后一页。第四十七首诗的下面——那首写于三天前、他犹豫了很久才放进去的诗——空白处多了一个字。

铅笔写的。

"好。"

一个字。没有感叹号，没有句号，没有下划线，没有圈起来。就是一个字——"好"。写在页面右下角，很小，大概三号字大小，但笔画非常端正，每个转角都交代得很清楚。

林玉起盯着那个字看了很久。久到窗外的梧桐叶被风吹翻了好几片，白色背面朝上，像翻过一页书。久到楼下有人喊了一声"马文超你快点要迟到了"。久到他对面位置的人换了一拨——一个戴眼镜的大三女生走了，来了一个拎着笔记本电脑的研一男生。

"好"是什么意思。

可以是很满意。可以是还不错。可以是字面意义上的"好"——这个字写得好，这首诗写得好，这个想法好。可以是"我看到了"的另一种说法。可以是"我知道了"的另一种写法。

也可以什么都不是——就是一种礼貌的确认，像批改完作业在最后一页签个名。

林玉起抬头看了一眼苏安晴。她正在看她的《教育心理学》，铅笔停在手指之间，大概在回忆刚才看过的那个概念。她的表情很平静，没有任何"等你看完评价"的期待。

她写了一个字，然后就把这件事放下了。像合上一本书放进书架，那么自然。

林玉起把诗集慢慢合上。那个"好"字被夹在了第47首诗和封底之间的空白页上，不会再被任何人看见，除非他自己翻开。

他把诗集放进书包，拉上拉链。

"谢谢。"他说。

苏安晴没有说"不客气"。她只是从书页里抬起头，看了他一眼。那个眼神里没有任何多余的内容——不是在传递什么信息，不是在暗示什么意思，也不是在回避什么。就是看了一眼，然后继续低头看书。

但林玉起知道，那个"好"字会跟他一辈子。

不是因为它代表了什么承诺。只是因为它是由她的手写上去的，用的是铅笔，笔画很轻，却压住了那本诗集里所有的忐忑和犹豫。

窗外的梧桐树叶又翻了一片。他把《中国诗学史》重新翻开，翻到第125页——前天停在的那一页。从第124页到第125页，中间隔了三天，隔了一个"好"字。

> 世间千万言
> 不及一字轻
> 铅笔写过不留声
> 却把整本心事
> 压住了三分
