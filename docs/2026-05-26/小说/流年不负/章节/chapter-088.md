# 第088章 没有偶遇

时间：2014年1月

地点：青阳县

卷次：卷五：各自南北

寒假期间，他路过老槐树巷口很多次。

不是刻意的——他跟自己说。去新华书店要经过那里。去菜市场要经过那里。去赵一鸣家也要经过那里。青阳县城就三条主街，这一条是主街之一，绕不开的。但每次经过的时候——他都会放慢脚步。慢到几乎是停了下来，然后再走。

第一次——腊月二十五。早上去给他爸买鞭炮。卖鞭炮的铺子在老城区西边，走这条路最近。早上的寒气穿着棉袄也挡不住——脸被冻得有点木。雾很浓——整个县城像被泡在白蒙蒙的雾气里。路边的法国梧桐枝干在雾中若隐若现——梧桐的青灰色树皮在雾气中变成深灰——树皮上的一块块剥落处在雾中还能看清边界。他走过农资公司的铁门——铁门上有锈条，挂着"春节期间照常营业"的牌子，门缝里冒出一个老头的脑袋——戴着棉帽子，手里拿着一个暖水袋。走过菜市场——肉铺大叔正在往铁钩上挂半扇猪——猪皮白白净净的，猪肉上凝了一层薄霜。大叔的手套上全是血水——手套黑了。经过巷口的时候晨雾还没散。老槐树的枝干在雾里只看得清轮廓——黑黑的、曲曲折折的，枝条的细部全被雾气模糊了。树身上贴着一张红纸——大概是哪家贴的"出门见喜"——红纸边角上沾了露水，红色洇成了粉色。

巷子深入很安静。收音机在放黄梅戏——《天仙配》，"树上的鸟儿成双对 绿水青山带笑颜——"，咿咿呀呀的女声从砖墙缝里飘出来。巷子口停着一辆旧自行车——前轮瘪了，车座上的皮裂口露出黄海绵。旁边地上搁着一块抹布——上面印着一个楼盘的广告，广告字褪得快看不清了。最里面的枇杷树——冬天的枇杷叶子是墨绿的，不落叶，叶子厚厚地在雾气中沉沉的。

他站在巷口看了大概十几秒。没有走进去。然后继续往鞭炮铺子走。

第二次——腊月二十八。去买酱油。第三次——正月初三。去赵一鸣家拜年——两个人喝了杯茶，聊了聊各自大学的情况。赵一鸣在省城读书——学了计算机，"大学里全是码农"他说的时候吃着一根麻花——麻花屑掉在茶几上。第四次——正月十五。没什么事，就是出门走走。正月初五街上放了很多鞭炮——脚底下铺了厚厚一层红纸屑，踩上去软绵绵的。巷口有几个小孩在放小鞭——点着一根香、往地上一扔就跑——炸开的红色纸屑飞到鞋上——他踢掉。第五次——初七下午。第六次——初九。第七次——初十。数不清了。

每次动作都一样——脚步慢下来，往里看一眼，然后继续走。

有时候巷口有个卖烤红薯的老头——推着铁皮炉子。炉子外面熏得焦黑，在冬天的冷空气里焦黑的铁皮上会结薄霜。煤块在里面红透透地烧着。红薯的甜味飘了半条街。老头坐在一个木头箱子上——穿军大衣，戴棉帽子——帽子两个耳扇放下来系在下巴。他不买红薯——他喜欢站在那个位置——那个位置是看巷子深处枇杷树最好的角度。枇杷树下有个小花盆——种着文竹——冬天文竹有点黄——大概冻的。

有时候巷口空无一人。只有老槐树的光枝条被风吹得互相刮擦——干涩的吱嘎声——像用了太久的门轴。地上枯叶在风里打转。墙上电线杆贴着小广告——"疏通下水道""专业修锁"——电话号码被撕掉了——"139"开头，后面不知道了。

一次都没有"偶遇"。

不是真的想偶遇。他只是——经过。也许只为了确认那个地方还在。确认路灯还是橘黄色的。确认栅栏还是生了锈的。确认那棵枇杷树还是墨绿的。确认那条巷子——那个住着她的小世界——还在时间缝隙里安静地运转。他每次离开巷口的时候——身后有什么东西被留在那里——说不上来是什么。

初七那天他站得最久。

那天下午下着小雨。青阳县的老街路面湿漉漉的——石板路滑得发亮。没带伞的行人用手遮着头跑过去——有人把书包举在头上、有人把塑料兜顶在脸上。他没带伞——棉袄帽子拉到了头上。帽子沿耷下来遮住半边眼睛。走到巷口的时候停下了。巷口卖烤红薯的老头今天没出摊——铁皮炉子不在了，只剩地上一块黑圈——是炉子放久了烤黑的痕迹。大概是下雨天生意不好。

雨落在老槐树的枝干上——顺着树皮的裂纹往下淌——树皮上那层灰被雨水冲掉了，露出里面干净的褐皮。雨滴挂在树枝上——透明的，停几秒，滑下去。巷子深处没有人。谁家的晾衣绳上还有一件没收——红毛衣被雨淋得往下滴水珠子。汪。一只黄狗趴在谁家屋檐下——把头搁在前爪上，看了他一眼，眨了眨眼。狗的毛湿了一块——贴着身上。没有叫。

他站在雨里。从巷口看进去——巷子的尽头是雾蒙蒙的雨水帘。枇杷叶顶上的雨滴——嗒嗒嗒——叶面的蜡质让水珠先凝成球再滑下去。

站了大概有十分钟。久到巷子左边那栋楼的二楼窗户开了——一个老太太探出身子——灰白头发，戴着老花镜——眼镜腿缠着胶布。老太太身后电视里放着《还珠格格》——"紫薇紫薇——"——小燕子的声音从二楼窗缝里往下漏。老太太看着他。

"小伙子——你找谁。"

"不找谁。"

"那你在雨里站这么久。"老太太眉头皱了皱——看得出来她不放心。

"躲雨。"

老太太抬头看了看天——雨不大，就毛毛雨。她又看着他——"你躲雨站了十分钟。"

他没说话。他把帽子往下拉了拉——挤出一个招呼——那个表情算不算笑自己也不知道。老太太看了他几秒——把窗户合上了——玻璃关上之前又补了一句"你往前走五十步有个超市——超市门口有挡雨棚——比站这儿强"。窗户在她身后合上——"啪"——声音被雨盖过了。

他没有去超市。但他转身走了。走的时候棉袄肩膀那块已经深了一圈。帆布鞋尖湿了——灰蓝色的布面变成深蓝色。帽檐上积了一小滩雨水——低头时水流下来滴在门槛上。进了家门——他妈在织毛衣，电视里在放选秀节目，评委说的话很夸张。她抬头看他——"你不是说出去走走吗，怎么一身水。"他说"下雨了。"他妈说"不知道怎么找地方避一避。"他说"嗯。"

换了干衣服——灰棉毛衫、厚裤子。湿衣服搭在暖气片上——暖气片上起了蒸汽。窗外雨还在下——檐水滴在水泥地上——叮。叮。叮。赵一鸣发来QQ消息——"明天出来吃饭。一中门口那家面馆下午两点。"他回"行。"赵一鸣又发来——"你寒假见到她了没。"他回"没。"赵一鸣说"没去找她吗。"他回"没有。"赵一鸣说"你也真能忍。"他回"不是忍。"赵一鸣问"那是什么。"他打了三个字——"遵守约定。"赵一鸣发了一个省略号。然后又发了一句——"你这种人活着太累了。"

他看着屏幕。没有回复。把手机扣在枕头边——屏幕朝下。天花板上的墨点还在——是他高一时甩圆珠笔不小心甩上去的，三年多了。墨点淡了。但还在。墨点周围有一圈洇开的蓝色。

他闭上眼睛。脑子里是那条巷子——灰砖墙、青苔、老槐树皮、枇杷叶上的雨滴。黄狗的耳朵垂下来了。二楼的窗户关了——玻璃上有老太太手指留下的雾气手印。那条巷子的深处——她的窗户在哪一扇——他不知道。但那都不重要。重要的是——站在巷口就够了。墙在。风还会吹来煤炉子和烤红薯的气味。

"遵守约定"——听起来像口号。对他不是。那是一道墙。墙这边是他。那边是她。他不能翻过去。也不想翻过去。他每天来墙边站一会儿。确认墙还在。确认那个方向的风还是同一个味道。

第六次。第七次。以后可能还有无数次。

一次都没有走进去过。
