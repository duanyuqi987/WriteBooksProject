# 第064章 下学期见

时间：2011年七月上旬，放假前最后一天

地点：图书馆二楼

卷次：第三卷：春天的诗

放假前最后一天，林玉起在图书馆收拾东西。他把借了一个学期的书整理好——《中国古代文学史》《现代文学三十年》《何其芳诗选》《中国诗学史》——摞成一摞，搬到二楼还书处。

图书馆里人很少。期末考试结束以后，平日挤满了人的自习区空了一大半，椅子推回桌底、桌面上清清爽爽的，像被退潮的海水冲刷过的沙滩。只有几个考研的学生还坚守在窗边，桌上摆着肖秀荣政治和考研英语真题，他们是大三的，没有寒暑假。

林玉起把书递给图书管理员方慧芳——一个戴着圆框眼镜的年轻女老师，永远在工位上看《收获》杂志。"都还了？"方老师扫条形码的时候问。

"都还了。"

"你借得最多的不是这些吧？你是不是自己带书来的？"

"嗯，有些是自己的。"

方老师笑了一下，没有追问。她见过太多这样的学生——来图书馆不是借图书馆的书，是借图书馆的安静。

林玉起回到靠窗倒数第三个座位上收拾自己的东西。两本自带的《唐诗三百首》和《宋词选》、一支按动式黑色中性笔、一本用了一大半的笔记本、一个喝水用的玻璃瓶。他把这些东西一件一件放进书包——放之前先吹了吹桌子上的灰，不是桌子脏，是他觉得这个位置陪了他一个学期，走之前该跟它好好打声招呼。

收拾到最后一本书的时候，他听见脚步声。

不用抬头，他知道是谁。右脚比左脚重一点——那个人自己说过的。

苏安晴也在收拾。她把她经常坐的那个位置上的书搬走了——《教育学原理》《教育心理学》《大学英语综合教程》，还有一本从图书馆借的《教育研究方法导论》。她把每一本书都摞整齐，边角对齐，书脊朝外，像书店里陈列的新书。

"收拾好了？"林玉起问。

"差不多了。"

苏安晴把最后一本书放进书包，然后把桌面上贴的那张考试安排表撕下来，揉成一小团，握在手心。她看了一眼桌面——那个空白的、曾经放着她水杯、笔记本和考试安排表的桌面——然后拿起书包。

"走吧。"

两个人一起走出图书馆。门口的老周头儿叼着没点着的半截烟，在翻一份旧报纸。他抬头看了他们一眼，点了一下头，算是道别。

外面阳光很烈。文华路的梧桐树影斑斑驳驳地落在地上，风吹过时，影子也跟着晃动，像水面上的波纹。自行车铃声从远处传来，夹杂着行李箱滚轮拖过水泥路面的声音——很多学生今天走，拖着箱子往校门口赶。

走到图书馆和宿舍楼之间那条岔路口——就是上个月那条小路上苏安晴说"你走路右脚比左脚重一点"的地方——苏安晴停下了。

她转过来，不是那种正式的面谈姿势，只是侧过身子，书包带从肩膀上滑下来一点。

"下学期见。"

三个字。语气跟她说"叠比折轻"时一样——干净、确定、平静。不是那种依依不舍的告别，也不是那种敷衍了事的客套。就是一种确认：下学期还会见面，还会在这个校园里，还会在图书馆、诗社、食堂、某条小路上遇见。

不是"再见"——"再见"太郑重了。也不是"回头见"——"回头见"太随便了。"下学期见"——刚好。具体到时间，模糊到地点。有承诺的壳，没有承诺的核。足够近，也足够远。

林玉起站了两秒。

"下学期见。"他说。

苏安晴点一下头，转身往教育学院宿舍楼走了。她的书包在背上轻轻晃着，拉链上挂着的那个小吊坠——一只蓝色的塑料海豚，林玉起上学期就注意到了——在阳光下反了一下光。

林玉起往反方向走。走到中文系宿舍楼拐角的时候，他回头看了一眼。岔路口已经没有人了，阳光直直地打在地面上，把梧桐叶的影子晒成深黑色。树上的一只蝉突然停了叫声，周围安静了大概三秒，然后又有一只新的蝉叫起来，比刚才那只更高更尖。

郭向阳在宿舍收拾行李，床上的被子捆成了一个不规则的球形，用编织袋套着。"回家！封丘的热干面，我想死它了。"他拍着林玉起的肩膀说。

林玉起没接话。他在想三个字。

"下学期见。"

这三个字是苏安晴对他说的。她对别人说什么——是"开学见"还是"再见"还是"回头联系"——他不知道。但对他，她说的是"下学期见"。这是她和他之间的一种语言，不包含任何字面以外的承诺，却包含了字面上全部的含义。

他把书包装好，拉上拉链。拉链还是之前那个——断了半边的那根。明天下午一点四十，他会坐上回封丘的大巴。三个半小时，穿过省道两旁的桉树和水田，穿过夏天最热的那段时间。他不知道苏安晴会坐在哪个位置，但他知道她也在车上。
