# 第065章 高考后的水多年后

所属部卷：第二部：六月以后
时间：2012年六月八日
地点：县一中校门口

## 诗歌插页 P065：原创七言小诗

灯过长廊日影深，
一行旧字压衣襟。
少年未解离分意，
先把沉默写到今。

多年后陆允川重写高考后的水，先删掉最重的怨气，再删掉最亮的自我感动。页面空下来以后，母亲递来的矿泉水瓶反而更清楚。

高考结束后，陆允川被人群推到校门外，母亲递给他一瓶水，他却不知道该高兴还是失落。这件事没有随着时间消失，只是从伤口变成了可以被描述的旧场景。

他重新补回校门口的东西：透明笔袋、矿泉水瓶、校门警戒线、家长的遮阳伞。这些东西不负责抒情，却负责让记忆落地。

夏知遥被人群推远前，回头看了一眼校门。陆允川不知道她在看学校，还是看这三年。正因为不知道，他后来没有替她解释。

他也给夏知遥补回生活。夏知遥从另一侧人群里出来，身边跟着她的母亲和同伴。她看起来也累，额前碎发贴着汗，笑的时候不是轻松，倒像终于可以松开一根绷了很久的线。她不是他青春的答案，也不是他小说里的奖品，她只是一个也在往前走的人。

程砚和叶澄的声音也被留下。一个把玩笑递过来，一个把边界说清楚。没有他们，陆允川很容易把孤独误认为深情。

写到这里，他把日记本合上，去倒了一杯水。母亲包里提前买好的面包的味道从记忆里浮出来，提醒他那一年并不只有疼。

所谓温柔现实，就是不替过去粉饰，也不把过去判死刑。它承认陆允川曾经笨拙，也承认夏知遥有权不接住他的笨拙。

陆允川后来愿意留下这一笔，是因为它不漂亮。透明笔袋不漂亮，矿泉水瓶也不漂亮，可它们让人相信，这段青春不是从空中落下来的。

他没有再给夏知遥安排额外台词。她已经在校门口完成了自己的动作，剩下的沉默，应当由陆允川自己消化。

程砚把话说得粗，叶澄把话说得稳，两种声音都把他往现实里拉。陆允川当时不领情，很多年后却知道自己被救过几次。

那天的母亲包里提前买好的面包没有特别好吃，却因为太具体，抵住了那些过大的形容词。人有时候正是靠这些具体的味道，才没有被回忆吞没。

家长们举着伞，学生从警戒线里一拨一拨涌出来。这些声音把高考后的水从一个人的心事里拉出来，变成许多人共同挤过的一天。陆允川不再只听见自己胸口的动静，也听见椅子、脚步、塑料袋和远处的喊声。

夏知遥被母亲拉到阴影下擦汗。这个动作很小，却让她从陆允川的想象里退回到真实的人：她会忙，会累，会被家里和未来推着走，也会在必要的时候保持沉默。

程砚隔着人群晃可乐瓶，说话还是老样子，不好听，却有效。陆允川皱了皱眉，还是照着他说的做了一点，因为朋友有时不是负责安慰，而是负责把他从自我感动里拽出来。

叶澄背着书包站在树下回班群消息。她没有把话说得太满，只把边界摆到桌面上。陆允川那时不一定立刻懂，却会在后来某个晚上想起这句话，知道自己当年被提醒过。

母亲把瓶盖重新拧紧，像怕他连水都拿不稳。家里的声音一进来，青春就不再只是校园里的风。它变成饭点、车费、睡眠、行李和一句不太会表达的关心。

他低头看着矿泉水瓶，忽然意识到它不是命运给他的标记，只是那天确实存在的东西。正因为确实存在，它比任何夸张的形容都更能留下来。

太阳晒热的柏油路和瓶装水的塑料味混在一起，让记忆有了可以落脚的地方。陆允川后来写到这里，没有再急着解释自己多难过，而是先把气味、光线和手上的动作写清楚。

他把透明笔袋收好，又拿出来检查了一遍。这个来回不像英雄，也不像受害者，只像一个年轻人不知道怎样安放自己，于是先安放手边的东西。

多年后他最想删掉的，是那些把自己写得过于正确的句子。正确有时比错误更可怕，因为它会把别人挤成背景。

他把高考后的水重新写了一遍，决定不给任何人判刑。疼痛是真的，笨拙是真的，错过也是真的；但这些真实不需要互相证明。

这一笔补进去以后，章节没有变得更响，反而更稳。高考后的水不再只是一个标题，而是有矿泉水瓶、有透明笔袋、有人说错话也有人及时停住的真实一天。

母亲把面包递给他，又说不想吃就先放包里。陆允川接过来，才看见她手背晒红了一块，原来这场考试也让门外的人等了很久。

夏知遥被人群挤得往后退了一步，她母亲赶紧扶住她的肩。陆允川看见这一幕，突然明白她不是从试卷里走出来的符号，她也是别人心里提了一下午的人。

程砚说晚上要去吃烧烤庆祝，声音很大。陆允川点头，却先回头看了一眼校门，门口那条警戒线还没撤，像三年最后一点边界。

校门口里的光线一点点变暗，警戒线和遮阳伞都还在原处。陆允川把透明笔袋收好，动作慢得像怕惊动什么。可真正需要被惊动的不是别人，是他自己。

他后来重写这一章时，特意把矿泉水瓶留在中间。不是为了让它变成象征，而是因为人只有摸到具体东西，才知道自己曾经怎样站在具体生活里。

多年后的陆允川终于承认，自己当年并不比别人更懂爱。他只是更擅长记住矿泉水瓶，也更擅长把沉默误解成命运。

多年后他想起母亲递水的手，才明白那一天不是只有他的结束。门外的人也跟着考完了一场漫长的试，他们没有答题卡，却同样等到嗓子发干。

本章最后，他没有写“如果当初”。他只写下：那时有家长喊名字的声音、车喇叭声和塑料瓶被捏响的声音，有一个人慢慢学会把别人还给别人。
