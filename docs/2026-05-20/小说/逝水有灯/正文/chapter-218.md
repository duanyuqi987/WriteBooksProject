# 第218章 旧日记本之后

所属部卷：第五部：逝水有灯
时间：搬家当天
地点：搬家纸箱旁

## 诗歌插页 P218：原创仿宋小令

箱未满，纸微黄。
旧字有余伤。
封口胶带响一场。
不寄到谁旁。

搬家师傅来得比约定时间早。门铃响起时，陆允川正把旧日记本重新叠好。师傅看了看满屋纸箱，说书挺多。他笑了笑，说以前读书留下的。

日记本放在哪一箱，他犹豫了很久。放进“书籍”箱里，似乎太普通；单独带走，又显得太郑重。最后他把它们和旧相册、毕业证复印件、几本不用的工具书放在一起，在箱外写了四个字：旧纸资料。

写完这几个字，他心里反而轻了一点。旧日记本不是圣物，也不是证词。它们只是旧纸资料的一部分，需要防潮，需要搬运，需要在新屋里找到一个不碍事的位置。

楼下车位不好停，师傅催他快点封箱。胶带被拉开，发出很长的一声。陆允川把箱口压平，一圈一圈缠住。那一刻他突然想到，自己曾经那么想把某些话寄给某个人，其实只是怕没人知道自己痛过。

现在不需要了。知道的人可以只有他自己。箱子被搬走时，他伸手扶了一下，纸板边缘硌到掌心。师傅问这箱重不重，他说：“还行，都是纸。”
