# 《逝水有灯》大纲

## 体量规划

- 总目标：约 50 万字
- 章节：245 章
- 每章：约 2000-2300 字符量级
- 诗歌：245 首插页诗，超过 100 首要求

## 五部结构

1. 第一部：初秋三楼。县一中初遇、分班、暗恋、诗抄本和高三倒计时。
2. 第二部：六月以后。高考结束、迟到表白、升学、大学南湖和新关系开端。
3. 第三部：寒风里的花。等待、电话、误会、朋友劝告和被拒绝后的失衡。
4. 第四部：失去的美丽。牵手误会、删除线、道歉未发送、毕业照里的体面距离。
5. 第五部：逝水有灯。工作后整理旧日记、重回校园、诗歌索引和写成小说。
