# 第125章 照片

时间：2018年3月

地点：楚州，手机

卷次：卷七：流年未断

张阿姨发来了照片。

短信里附了一张图。他点开。加载圈转了两秒。

照片亮了。

苏晚晴站在楚州一小的校门口。背后是校门——浅红色的砖墙，铁栅栏，门柱上挂着学校的牌子。天很蓝。大概是去年秋天拍的。校门两侧的银杏树叶子黄了一半。

她没怎么变。

马尾还在。浅蓝色衬衫——跟高中时候的颜色几乎一样。黑色长裤。肩上的包是米白色的——不大。左手拿着一个文件夹，右手垂在身侧。

比高中时瘦了一点。脸上的轮廓比以前清晰了——颧骨和下巴的线条。但眼睛还是亮晶晶的。

笑容比以前松弛了。高中时她笑的时候总是抿着嘴，好像怕笑得太大。这张照片里她的嘴角是自然弯的——不是在拍照的人面前刻意控制的那种笑。

从容了一些。她的站姿不紧绷——重心在左脚，右脚微微往前，肩膀是放松的。眼睛看着镜头。目光跟高中时候不一样——那时候她看人总是很快移开，像怕被人发现自己在看什么。

现在她的目光稳稳地落在他身上——落在屏幕上，落在他的眼睛里。

他看了很久。

屏幕暗了一次。他按亮。手指在屏幕上碰了一下——画面动了。又暗了一次。又按亮。拇指按在home键上。指纹解锁。屏幕亮起——照片还在。

手指悬在屏幕上。滑到照片的左侧去看校门口的细节。校名是铜字，有点反光——"楚州市第一小学"七个字。第一个字"楚"的左边偏旁磨得有点花了。她站的台阶上有个小水洼——大概是前一天下过雨。水洼不大。映着半边蓝天和一小角银杏树。她穿的是平底鞋——黑色的帆布鞋。鞋头有点磨白了。右脚鞋带系得比左脚紧。

跟高中时候一样的鞋。高中她就穿黑色帆布鞋。穿了三年。鞋底磨薄了再去买一双一样的。

他把照片缩小。两根手指在屏幕上一捏——照片变小了。她的身影退远了一点。又放大——两根手指往外一推。照片放大了。她的脸占满了屏幕。眼睛。鼻子。嘴唇。放大后的像素有点模糊。但那双眼睛还是亮晶晶的。又缩小。

然后他把手机放在桌上。屏幕朝上。照片慢慢暗下去——自动锁屏了。

手指交叉握在一起。十指交叉——左手拇指压在右手拇指上面。手背上的青筋比高中时候明显多了。代课这两年，捏粉笔捏的。手掌根部有一层淡黄色的茧——粉笔一直握在那里。

他把手机又拿起来。

打开相册。新建了一个文件夹。手指在输入框里点了一下。键盘弹出来。他打了一个字——一。然后点确定。文件夹建好了。

只有一张照片。

他退出相册。打开通讯录。翻到张阿姨。打了一行字——"照片收到了。谢谢张阿姨。"大拇指在发送键上停了一下。按下去了。发出去。发送成功的小勾在消息旁边亮起来了。

然后他把手机翻过来，屏幕朝下放在桌上。桌面上有个茶杯印——圆形的。他的手机刚好放在那个圆圈里。

手指在桌面上轻轻敲了两下。食指和中指——轮流。嗒。嗒。

停了。

过了一会儿。他又把手机翻过来。屏幕亮了——用指纹。点开相册。点开"一"。照片亮起来。

她站在校门口。阳光很好。银杏叶黄了一半。有几片叶子正在往下落——刚好被镜头捕捉到了。她的马尾安静地垂在肩上。马尾的尾部搭在浅蓝色衬衫的衣领上。阳光在她头发上切了一道亮与暗的分界线。亮的部分是褐色。暗的部分是深棕。

八年。

他记忆里的她跟这张照片重合了。高一开学的那个九月——她从前面走进教室，浅蓝色T恤，深蓝色牛仔裤，扎马尾。九月阳光照在她侧脸上。她的头发在光里是褐色的。她往教室里扫了一眼，然后走向第二排靠窗的位置。

那时候他的心脏好像漏跳了一拍。但他没有告诉任何人。甚至没有告诉自己。

现在他看着这张照片——同一个姑娘。同一个马尾。同一个颜色的衬衫。同一个——

但他不一样了。

他也瘦了。比以前高了几厘米。肩膀宽了一些。胡茬硬了。手指上有了粉笔的茧。眼睛底下有青色的印子——失眠好几天的结果。

他不知道她还认不认得他。

照片里她的眼睛亮晶晶的。看着镜头。看着八年后的他。

他把手机屏幕按灭。

房间暗了。

窗外那棵樱花树还在。花瓣快落完了。剩下的几朵在枝头轻轻晃。

他又按亮屏幕。看了一眼照片。然后把手机放进口袋里。

站起来。

走到窗前。

樱花落了满地。白色的花瓣铺在灰色的地砖上。明天扫地的阿姨会来清理。

他靠在窗框上。手插在口袋里。手指碰到手机的边缘。

周六下午三点。

一茶一座。

她坐在窗边。

他推开门。

这是他想了八年的画面。

但明天——不是"想"。是真的。

他站在窗前。窗外的路灯亮着。对面楼的窗户里有人在拉窗帘——浅粉色的。拉上了。窗帘布后面是人影——模糊的。在屋里走来走去。

他低头看了看自己的手。左手食指上有一道粉笔划的印子——白色的。从指尖斜着划到第二个关节。他用拇指擦了擦——没擦掉。蘸了口水擦——还是没擦掉。算了。

口袋里的手机沉甸甸的。不重。但他能感觉到它的存在——贴着大腿外侧。长方形的。坚硬的一小块。

里面有一张照片。

她站在一小的校门口。阳光很好。马尾被风吹起来一点点。耳边的碎发飘起来了——在阳光里是透明的。她看着镜头。表情平静。

好像在说——好久不见。

他把窗帘拉上。窗帘布在滑轨上发出很轻的沙沙声。房间暗了一些。手机在口袋里。他没有再拿出来。走到床边。坐下。床垫嘎吱了一声。然后把鞋脱了。两只鞋落在地板上——咚。咚。左脚先。右脚后。

他躺下来。被子拉到胸口。闭眼。窗帘缝里透进来一条路灯光——细长的。橘色的。落在靠近门口的地板上。

他翻了个身。面朝墙。

他翻了个身。面朝墙。墙上的涂料又起皮了。这次他没抠。被子拉上来盖住了肩膀。枕头还是那个荞麦皮的——中间的凹槽刚好卡住后脑勺。手机在枕头底下。屏幕朝下。里面有她的照片。文件夹名字是"一"。只有一张。一张就够了。

窗外的樱花树在风里晃了一下。最后几瓣花落下来了——轻飘飘的。有一瓣贴在玻璃上。停了一会儿。然后被风吹走了。天快亮了。或者天还没亮——他分不清。大概是凌晨。凌晨的光是灰的。什么都看不清。但他知道她在照片里。站得端端正正的。马尾。浅蓝色衬衫。平底黑帆布鞋。在楚州一小的门口。等着周六下午。

窗外的路灯灭了——天亮了。光线从灰变白。窗台上落了一层灰。他用手指画了一下。画了一道线。线上面的灰被抹掉了。线下面的灰还在。那道线在朝阳里很清楚——浅灰和深灰的分界。他看着那道线。想起照片里她站在一小的门口——阳光从左边照过去。她半边脸在光里。半边脸在暗里。也是这样的分界。
