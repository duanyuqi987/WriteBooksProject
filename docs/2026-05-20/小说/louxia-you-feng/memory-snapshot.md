## 最近章节摘要

- chapter-036《回到校园》：许亦安出差顺路回到学校，校园翻新了一半，老网吧变成水果店，小吃街的摊位换了人。 过去没有消失，只是换了招牌。
- chapter-037《楼下没有人》：许亦安站在曾经等过的楼下。楼已经改成研究生宿舍，门口的花坛还在，只是月季长高了。 他终于承认，那些等待大多等的是自己的想象。
- chapter-038《青禾的背影》：许亦安远远看见一个像林青禾的人。她和孩子说话，低头整理小外套。后来他确定那就是她。 他没有上前，不是逃避，而是尊重彼此已经拥有的生活。
- chapter-039《便签迁移》：许亦安把旧手机便签导出来，一条条看完，再存进电脑里一个普通文件夹。文件夹名字叫“大学”。 他不再删除，也不再反复打开。保存就是告别的一种。
- chapter-040《天明已是昨》：许亦安把这段故事写成小说。他写到楼下的风、网吧的照片、没接的电话，也写到自己终于不再把遗憾交给别人负责。 青春被重新放回温柔的位置：不是失败，不是胜利，只是经过。

## 活跃伏笔

- 楼下的风：从等待意象转为释怀意象。
- 旧手机便签：从执拗记录转为温柔保存。
- 沈安晴的信：帮助许亦安看见自己的含糊和不成熟。
