# 第097章 一个女生

时间：2014年10月

地点：楚州师院图书馆门口

卷次：卷五：各自南北

十月中旬。文学社迎新茶话会——一间贴着诗词条幅的小教室。桌上摆着橘子、瓜子、泡很久的菊花茶——菊花的黄已经把水染成了淡黄色。新社员十来个——大一，眼里带着还没褪完的新鲜感。老社员坐靠窗那排——顾临风坐最边上——手里剥橘子——橘子皮有点厚——指甲缝里塞满了黄色皮屑。

茶话会结束——大家三三两两散了。十月的天暗得早——六点多路灯就亮了。他正要往图书馆走。

"顾临风。"

回过头。一个女生站在教室门口——穿米色针织衫——头发直的到肩膀。手里抱黑色封面笔记本——跟他那本有点像——边角磨白了。她是一年级新社员的——刚才自我介绍时坐在后排。

"我是周雨——文学社新社员——大一——你刚才茶话会说你写日常——我挺感兴趣的。"

他刚才发言很短——"日常——写身边的东西——灯——走廊——操场。"说完就坐下。没想到有人记得。

"嗯——随便写的。""我可以看一下吗。"他想了想："没有带。""下次带来。""嗯。"

她笑了笑——牙齿整齐——眼下皮肤有几点浅浅雀斑——四五颗——笑得很直接——"那说定了——下次带过来。"转身走了——头发在肩上甩半圈——步子不快。

之后每次文学社活动她都在——坐第三排——离他不远不近。活动前五分钟到——书包里总有零食——饼干或巧克力豆——分给旁边人。第一次递过来他说"不用"——她说"你每次都这么说"——第二次他接了一片——牛奶味的。茶歇时她会走过来——"今天这首诗觉得怎么样"——"还行"——"第三段有点太直了"——"是有点。"

她的散文写得好——有细节——宿舍楼下流浪猫、食堂阿姨打菜多放一片肉、奶奶织的围巾有一针松了一直没补。有条灰围巾她戴过——左边靠下那块确实松了一针。跟她相似——都会观察。

阿坤在社团室外瞄了一眼——"那个叫周雨的——是不是对你有点意思。""不知道。""'不知道'——你心里清楚得很。"他没接话。

十月底。下午——他从图书馆出来——抱着五六本书。周雨站在图书馆门口台阶上——大概在等他。

"顾临风——我有话跟你说。"他停下。她把笔记本抱在胸前——抿嘴唇——然后说："我喜欢你。"

大方——没结巴——没低头。阳光从梧桐叶缝漏下来——打到她侧脸——雀斑在光里很淡。他想了几秒——不是惊喜的几秒——是"要不要试一试"的几秒——然后点头。

"好。""好——是什么意思。""试试。"她又笑了——眯起眼——牙齿露出来——干净的笑。"以后图书馆帮我占座。""嗯。"她伸出手——他握了一下——她的手比苏晚晴的凉——他手掌还是汗多。

心里有一句话没说出来——不是因为喜欢她——是因为觉得应该试一试。也许能忘记呢。也许呢。

周雨转身走的时候——回头看了他一眼。笑了一下。然后抱着笔记本走进了梧桐大道。梧桐叶开始落了——叶子在她身后飘下来——不是那种漫天飞舞的飘——是一片两片的。有一片落在她肩膀上。她没有察觉——继续往前走——叶子在棕色头发上停了一会儿——然后被风带走了。

第二天中午。食堂。他在第三个窗口排队。周雨从旁边走过来——端着一个铁盘——铁盘上是番茄炒蛋和米饭。她停在他旁边——"你每次都吃第三个窗口的。""你怎么知道。""因为我也每次排第三个窗口。"她笑了一下——端着铁盘走了。坐到靠窗的角落——那个角落是他平时坐的位子。

当天下午。他去了图书馆。习惯性地走到三楼靠墙的第四个座位——桌面有一块被台灯长期照出的色差。他把书包放下。刚翻开笔记本——背后有人。"第三排第四座——你说是随便坐的。"回头一看——是周雨——她端着杯子在找座位。水杯是保温杯——粉色的，杯盖上有兔耳朵。他说"对"。她在他旁边坐下——第三个座位。安静了几秒。她说"我不会打扰你的。我就是来看书。"

他写了几行字。旁边有纸张翻动的声音。她带的是那本黑色封面笔记本——翻开以后低着头写她的散文。笔尖沙沙地划过纸面——偶尔停下来——她把笔尾咬在上下门牙之间——想了十几秒——又继续写。她的专注是安静的专注——跟他写字时的专注是同一种。图书馆里的灯亮了——台灯把她的头发边缘照出很淡的铜色。她写着写着忽然抬头看了一眼窗外——又低下头继续写。他的笔停了几秒——看了她的侧脸一眼。然后他也继续写。窗外槐树枝光秃秃的。十一月的阳光很短。

这是他们第一次一起上自习。后来变成了三十多次。

那晚躺床上——看着天花板——想很久——关于这个决定。然后不想了。窗外有电车开过——嘀的一声。灯灭了。黑暗里李博问了一句——"今天在图书馆你跟那个女生坐一起的。"阿坤抢着说——"我早就发现了——她还给你递过饼干。"小伍从上铺探头——"是你之前说的文学社社员吗——叫什么——周雨。"他"嗯"了一声。室友们安静了两秒。然后阿坤说——"也好——你也该谈谈了。总比每天一个人去操场走圈强。"李博说"阿坤——你少说一句。""我这不是关心他嘛。"然后大家都沉默了。阿坤把键盘切换到静音模式——键帽压下去几乎没有声音，只有手指在键帽上滑动的那种沙沙响。窗外又一辆电车开了过去——比刚才那辆更远，嘀的一声更轻。黑暗里只有呼吸声和偶尔的翻身声。

他没有回答。翻了个身。脸冲着有洞的墙。墙是凉的。他把手贴在墙上——掌心感觉到的凉意沿着手掌往手腕爬。手机在枕头下面——今天收到的短信有一条是周雨发的——"谢谢图书馆的座位。明天我还来。"他还没有回。他把手机从枕头下拿出来——打了两个字——"好的。"发送。发送成功的提示音在黑暗中响了一下——嘀。屏幕按灭。闭上眼睛。天花板的裂缝在路灯光里还是很淡的一条线。明天她会坐在第三个座位。他会坐在第四个座位。明天他还是会吃第三个窗口的包子。明天他还是会在笔记本上写下一些关于走廊、关于站台、关于操场的句子。但明天旁边会多一个人——一个写散文的、说话声音直接、眼下有雀斑的大一女生。他说不清这是什么感觉——不是替代，不是新的开始——是试一试。看看能不能在那条固定的路线上，偶尔换一条道。偶尔在图书馆的角落不是一个人。偶尔收到一条不是10086的短信。只是一试。
