# 第140章 叶澄的劝告多年后

所属部卷：第三部：寒风里的花
时间：多年后
地点：南湖凉亭

## 诗歌插页 P140：原创仿唐绝句

南湖旧座对斜晖，
一句平声隔岁归。
若把心门修作狱，
谁人经过不知违。

多年后陆允川再看诗抄本，叶澄那句“她不是钥匙”被他用黑笔圈了起来。圈得很重，纸背都有印。

他坐在南湖凉亭，旁边有学生练主持词，声音时高时低。石桌上摆着奶茶和打印稿，和当年的报名表很像。

旧稿里，他曾经把叶澄写成一个清醒得近乎冷的人。后来他删掉了那些形容，补上她的活动表、红笔、社团压力和被纸边勒出的手腕。这样一来，她不再像一段道理，而像一个真实的同学。

真实的人说出真话，才会有重量。

陆允川把那页拍照存档，文件名写成“南湖劝告”。他没有发给叶澄，也没有发朋友圈。那是他欠自己的一次整理。

离开凉亭时，风把桌上一张学生的稿纸吹起来。他伸手按住，对方道谢。他笑了笑，说没事。

他终于明白，按住一张纸就只是按住一张纸。不必每个小动作都被写成命运。
