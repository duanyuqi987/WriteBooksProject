# 第083章 我不知道她怎么想

时间：2011年10月下旬，某个夜晚

地点：图书馆四楼到操场的路上

卷次：第四卷：暑假的信

郭向阳问了那个问题之后，林玉起想了整整一周。

他不知道她怎么想。这不是借口，是真的不知道。苏安晴对他到底是什么意思——是礼貌，是巧合，是老乡之间的自然亲近，还是也有那么一点不一样——这一点是最难的。你没有办法拿着尺子去量别人的感觉。也不是完全没有线索。

线索一：上学期诗社活动结束，她说他的诗"写得很好"。但她也夸过其他人，何南星念诗的时候她也说好。那只是礼貌。

线索二：开学她主动喊了他。但那是主动喊一个认识的人帮忙搬东西。如果换成一个不认识的男生站在那里，她大概不会喊。所以他属于"认识"的范畴——离"特别的"还差多远，不知道。

线索三：在图书馆碰面，她会说"你也在啊"。语气很自然，没有任何刻意的成分。不是惊喜，不是回避。就是陈述一个事实——你在，我也在。如此而已。

还有最让他拿不准的一次：上周在食堂，她和一个男生坐在一起吃饭。那男生个子很高，戴着运动手环，看上去像体育系或教育学院的。两个人各吃各的，偶尔说几句话。她在笑。那种笑不像是客套，但也不像是亲密——就是一个正常的、和熟人吃饭时会有的笑。

林玉起从旁边走过去，看见她没往这边看。他打好饭坐到三楼靠角落的位置。他在想那个男生是谁——同学？社团伙伴？还是别的什么。然后他想到自己没有资格去想这个问题，又觉得这个念头本身就不对——他凭什么觉得自己可以有资格？他从来不是她的什么。

周四晚上，他从图书馆出来，没有直接回宿舍。他绕过操场走了一圈。操场上有人在夜跑，有人在草坪上坐着聊天，有人在单杠上练引体向上。路灯把树影投在地上，影子被风吹得一晃一晃。

走第二圈的时候，他在看台边碰见了何南星。社长穿着一件深绿色的旧外套，坐在台阶上喝茶。他永远在喝茶。

"这么晚还在外面晃？"

"想了点事。"林玉起在他旁边坐下来。

"什么事——算了，你的表情说了：又是那个女生。"

林玉起没说话。何南星递过保温杯。"喝一口——不是什么好茶，超市买的菊花。"

他接过来喝了一口。苦的，但苦得刚好。

"何南星，你看过一个人很久但不知道她怎么想——怎么办？"

何南星没有马上回答。他把保温杯拧紧，看着操场对面那排亮着灯的宿舍楼。田径场上的脚步声此起彼伏，像一首没有旋律的歌。

"去年这个时候，你的诗里全是'如果'。"何南星慢慢地说，"如果你来、如果你在、如果你笑、如果你回头。今年的诗里出现了'你'——不是如果了。是确定的'你'。你进步了。"

"我问的不是诗。"

"我知道你不是问诗。但我只能回答诗的问题——因为我不够了解那个女生。我只能说，从你写的诗看，一年前你在藏，现在你在找。藏是自我保护，找是主动。但你找的方式是坐在图书馆里等——那不是找，那是守。"

他把"守"字说得很轻，但林玉起听到的时候，胸口像被什么东西撞了一下。

守。他在图书馆四楼守着的不是一个座位——是一道门，一个时间，一个不知道还会不会来的可能。

> 守和等有什么不同
> 等是站在原地
> 守是站在原地还说服自己不是站在原地

何南星站起来拍了拍裤腿上的灰。"我说的不一定对。毕竟我只比你高一届。你自己想吧。不过我建议你——如果有一天你不想守了，就往前多走一步。"

他走远了。保温杯在手电简的光里一晃一晃。操场的灯灭了又亮，因为是感应灯，他走过的地方亮了，停住的地方又暗下去。
