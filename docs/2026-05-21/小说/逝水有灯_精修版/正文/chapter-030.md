# 第030章 第一次说话多年后

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：教室前门和讲台之间

## 诗歌插页 P030：原创七言小诗

初语犹在人已移，
六字如山各自知。
年少不知寻常重，
却将捡本比旧诗。

多年后陆允川重写第一次说话，先把自己写得很委屈——写到他蹲下去帮她捡作业本的时候觉得那是命运安排的一个位置，写到手背碰到她手指的零点一秒等同于整个宇宙压缩成一道光。写完以后把那一页按住，没有撕——这是第一次他在重写时没有删自己十七岁的心事。因为他终于承认了一件事：那六个字尽管在时间长度上不值一提，但在他的生命周期里确实占据了一个节点——那是他和她在没有任何第三方建议、没有任何排练、没有任何理由的情况下自然发生的第一段对话。发音清楚、没出错、也没有事后让自己想钻地缝。十七岁的他再也做不出比这更好的六个字了。

多年后他在办公楼电梯里跟人抢按开门键时，手指碰到了对方同事的手——一个三十三岁的已婚男人，手上戴着婚戒。那人说"你按"，他说"你按"，两人在电梯门要关的瞬间同时按下去，然后各自退开。面无表情，各看各的楼层指示。他忽然在那零点一秒的指关节相碰里想起了高一的那次。但没有任何心跳——不过是两个人同时在同一个位置按同一个按钮，纯属概率。人在三十三岁的时候终于能做到面无表情地处理肢体碰触，而十七岁的时候——十七岁他花了三节晚自习来回想她手指的温度。

程砚后来问他："你还记得那六个字吗？""记得。""说实话，一般人问'你记得跟谁说的第一句话吗'，大部分人早忘了。""因为一般人的第一句话太长了——'你叫什么名字''你从哪个初中毕业的''你现在住校吗'——太多了。六个字好记。"

叶澄在修改稿上批了一段更长的。她说："你第一次和她说话，不是因为你想说话。是因为地下洒了本子。你不需要准备好话题去接近她——环境替你创造了接触，你只需要做一个人最基本的反应——弯腰帮她捡起来。这件事之所以成立，不是因为你有多勇敢，而是因为你在不做任何准备的情况下，做的第一反应是帮人捡东西。这不是什么浪漫桥段，但比浪漫更难——因为它证明你在她的世界里出现时，本能是好的。十七岁的你也许不够勇敢，但你够善良——你该少骂自己一点。"

这段话陆允川看了很多遍。每一遍都没划线，直接贴在了修改稿最后一页的背面，折了角——折角代表这句话是他会反复回来翻看的位置。这么多年过去，那些他觉得最笨拙、最不够大方的瞬间，在别人眼里不过是一个善良的人在做一件很小的事。那些他从头复盘到每一帧画面的片段，在真实的世界里只是一次没有人注意的弯腰。那些他自己嫌走路步幅太小的日子，在别人看来他只是在走——走出门，走下蹲，捡起本子，站起来，看着她的眼睛说一句"你捡吧"。他终于知道那六个字就够了。
