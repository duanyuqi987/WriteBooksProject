# 第188章 诗社告别之后

所属部卷：第四部：失去的美丽
时间：大三开学后
地点：宿舍 / 篮球场

## 诗歌插页 P188：原创仿宋小令

书脊边，夜色平，
半窗旧影入疏灯。
问当年、几回执念，几回无凭。
不把归人留梦里，
水声明日照常生。

退社之后的第一周，陆允川发现自己多出了很多时间。

宿舍的日历翻到了九月。窗外梧桐树的叶子还是很绿，但边缘已经开始有一点点发黄——不是秋天来了，是夏天快结束了。下午的太阳还是很毒，但傍晚的风已经带了一丝凉意，吹在脸上不再是热浪，更像一条湿毛巾轻轻擦过。

以前每周三晚上诗社活动，他吃完饭就往活动室走，来回加上活动至少三个小时。这三个小时现在空出来了。他用第一个周三去操场跑步——跑了两圈就喘不上气，因为太久没运动了。第二个周三去听了哲学系的旁听课——讲康德，不太听得懂，但坐在最后一排发呆比坐在诗社里提心吊胆轻松。第三个周三跟祁野打了一场篮球——输了，因为祁野也并不会打篮球，两个人在场上主要是追着球跑，偶尔球撞到篮板上发出一声闷响。

"你退社之后比退社之前还要忙。"祁野喘着粗气坐在篮球架下。

"之前是忙着想。现在是忙着不做那种想。"

"效果怎么样？"

"还行。晚上偶尔还是会想翻那个日记本。但翻到了第一页就放下了。"

"为什么不翻？"

"因为第一页写的是'我对她好她应该回应'——看了就不想再看了。"

"那你翻出什么了没翻进去，就算进步。"

"嗯。不过我今天看到诗社群里发消息说下周开始招新。苏晚枝发了一张新生咨询群的二维码，配字：热爱诗歌的可以进来聊。叶澄回了'好的社长'。夏知遥回了'奶茶店可以赞助吗'。我什么都没回。"

"忍得住吗。"

"忍住了。"

"好。忍住一次，下次就容易了。"祁野把球捡起来投了一个——没进，"退社最难的不是离开——是离开了还看得见他们在干什么，还能忍住不选'回群聊'按钮。"

"你什么时候这么懂了？"

"不是懂你。是懂退出工会。我退了好几个工会，每次退了之后还在潜水看他们聊副本。后来我把工会开除了，群也退了。感觉比以前爽多了——不是因为他们不好，是因为我在里面不是打副本，是找人。"

陆允川看着篮球框。球从框边缘弹出去，滚进了场边的排水沟。祁野跑去捡，踩到了沟边的碎石。

回到宿舍他打开了"不再分析的日常"文件夹，加了一行：

"退诗社一周。忍住没在群里回复。这不是冷落——这是练习把'自然碰面的空间'还给苏晚枝。"

保存。

后来苏晚枝在食堂碰到他。两人各自端着餐盘。

"群里怎么不说话？"苏晚枝问。

"退社的人不应该在招新群里太活跃。影响新生的判断。"

"谁规定退社的人不能帮招新——"

"没人规定。但我想先静一静。"陆允川把筷子放在碗上，"你跟老社员们忙招新——加油。需要发朋友圈宣传的话我可以转发——那个不突兀。"

苏晚枝看了他几秒。"好。谢谢。"

食堂打饭窗口那边传来大师傅敲锅沿的声音——大概是菜快没了，在报备。排队的人群往后退了半步，有人踮脚看还剩什么菜。

她把餐盘端起来往前走了两步又停下。"对了——那天你在活动室还的那本诗集里夹了一张主持词草稿。我帮你收起来了。收在社史箱里。写的是——'第三段念慢一点，灯光切换要等五秒'。"

"那是我大二时写的。"

"嗯。字很潦草。但备注那句写得挺好的——像在预测自己的未来。"

陆允川没有追问她是什么意思。只是说了句"谢谢"，然后继续吃饭。苏晚枝也继续吃饭。她走回到靠窗那张桌子——还是她习惯的老位置，粥碗旁边放着一本翻开的复习资料。食堂的声音盖住了所有的延伸。筷子碰碗沿、铁盘叠铁盘、刷卡机"滴"一声接一声——这些声音合在一起，变成一种持续不断的嗡嗡声，像一台巨大的洗衣机在轻柔模式下运转。
