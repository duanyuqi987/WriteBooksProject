# 第088章 没有偶遇

时间：2014年1月

地点：青阳县

卷次：卷五：各自南北

寒假期间，他路过老槐树巷口很多次。

不是刻意的——他跟自己说。去新华书店要经过那里。去菜市场要经过那里。去赵一鸣家也要经过那里。青阳县就那么大——绕不开。

但每次经过的时候，他都会放慢脚步。

第一次是腊月二十五。早上去买鞭炮——他爸让买的，说过年要放。卖鞭炮的铺子在老城区那边，走那条路最近。早上的雾很浓——整个县城笼在白蒙蒙的水汽里。路边的法国梧桐光秃秃的，枝干在雾里若隐若现。他走过农机公司——铁门上挂着"春节期间照常营业"的牌子。走过菜市场——肉铺的大叔正在往铁钩上挂半扇猪。肉上面凝着一层白霜。经过巷口的时候晨雾还没散。老槐树的枝干在雾里只看得清轮廓——黑黑的、曲曲折折的。树身上贴着一张红纸——大概是哪家贴的"出门见喜"，被露水打湿了，红纸洇出了粉色。

巷子深处很安静。谁家的收音机在放黄梅戏——《天仙配》，咿咿呀呀的女声从砖墙缝隙里传出来。巷子里停着一辆旧自行车，前轮瘪了。车把上搭着一块抹布。车座上的皮裂了——露出黄色的海绵。

他站在巷口看了大概十几秒钟。巷子大约三十来米深。左右两边是两排老式的砖瓦房。墙根长着青苔——冬天的青苔是褐色的，干巴巴的。最里面那家门口种了一棵枇杷树。枇杷树的叶子是墨绿的——冬天不落叶。门口还放着一把竹扫帚——靠在门框上。门是暗红色的铁门。门上贴着一张倒过来的"福"字。

她在不在家。不知道。

他没有走进去。

第二次是腊月二十八。去买酱油。第三次是正月初三——去赵一鸣家拜年。第四次是正月初五——没什么事，就是出门走走。初五那天县城放了很多鞭炮。地上铺了一层红纸屑。巷口有小孩在放小鞭——一根香点着一个，往地上一扔就跑。炸开的红色纸屑飞到他的鞋上。他把纸屑踢掉。第五次是初七下午。

他每次经过巷口，动作都一样——脚步慢下来。往里看一眼。然后继续走。

有时候巷子口有个卖烤红薯的老头，推着铁皮炉子。铁皮炉子外头熏得焦黑——大概是用了很多年了。炉子里头的炭火烧得红红的。老头坐在一个木头箱子上——穿着军大衣，戴着棉帽子。烤红薯的甜味飘了半条街。他经过的时候红薯的甜味飘过来——他没有买。他只是在那个位置站了一下——那是看巷子最深处的枇杷树最好的角度。枇杷树下面有个小花盆——种着文竹。文竹有点黄。

有时候巷子口什么人都没有。只有风。风吹得老槐树的光枝丫互相刮擦，发出干涩的声音。地上有几片枯叶——卷着的，风一吹就在地上打转。巷子口的电线杆上贴着小广告——"疏通下水道""专业修锁"。电话号码被撕掉了几条。有一条只剩"139"三个字。

一次都没有"偶遇"。

不是真的想偶遇。他自己也说不清为什么要经过。也许只是为了确认那个地方还在。确认她的窗户还在。确认那棵老槐树还活着。确认枇杷树还是绿的。确认那条巷子——那个住着她的小世界——还在时间的某条缝隙里安静地运转。

初七那天他站得最久。

那天下午下着小雨。青阳县的老街路面湿漉漉的，石头缝里积着水。没带伞的行人用手遮着头跑过去。他没带伞。棉袄的帽子拉到头上。走到巷口的时候停下来。巷口的烤红薯老头今天没出摊——铁皮炉子不在。大概是下雨天生意不好。

雨落在老槐树的枝干上，顺着树皮的纹理往下淌。雨滴挂在树枝上——透明的，停几秒，然后滑下去。巷子里没有人。谁家晾的衣服还没有收——一件红毛衣挂在晾衣绳上，被雨打得往下滴着水珠子。一只黄狗趴在谁家门口的屋檐下，把头搁在前爪上，看了他一眼，然后眨了眨眼睛。黄狗的毛是湿的——贴在身上。没有叫。

他站在雨里。

雨水从他的帽檐滴下来——一滴一滴的。地上溅起细小的水花。他的帆布鞋鞋尖湿了一块——颜色变深了。他不觉得冷。或者说，不记得冷。只是站着。看着巷子深处的枇杷树——雨打在枇杷叶子上，叶子一颤一颤的。

站了大概有十分钟。久到巷子左边那栋楼的二楼窗户打开了——一个老太太探出头来看着他。老太太头发灰白，戴着一副老花镜。眼镜腿用胶布缠着。身后屋里传出电视的声音——在放《还珠格格》，小燕子在说"紫薇紫薇"。老太太看了他一会儿。"小伙子——你找谁。"

"不找谁。"

"那你在雨里站这么久。"

"躲雨。"

老太太抬头看了看天——雨不大，就是毛毛雨。"你躲雨站了十分钟。"

他没说话。他把帽子往下拉了拉，露出了一个大概是笑的表情。老太太看了他几秒。把窗户关上了。然后窗户又开了一条缝——"你往前走五十步有个超市，超市门口有挡雨棚。比站这儿强。"窗户又关上了。

他没有去超市。但他转身走了。

回家的时候棉袄已经有点湿了。肩膀那块颜色深了一圈。帽檐上积了一小滩水——他低头的时候水流下来，滴在门槛上。他妈在客厅里织毛衣——一件蓝色的毛衣，针脚很密。电视开着——在放一个选秀节目，评委在说"你的声音很有辨识度"。她抬头看他："你不是说出去走走吗，怎么一身水。"

"下雨了。"

"不知道自己找地方避一避。"

"嗯。"

他换了身干衣服。灰色的棉毛衫、深蓝色的厚裤子。湿衣服搭在暖气片上。暖气片是铸铁的——上面刷了银粉，有些地方银粉掉了露出铁皮。他坐在自己房间的床垫上。窗外雨还在下。檐水一滴滴落在水泥地上，打出细小的声音。窗外的柿子树在雨里站着。枝丫上挂着的最后两颗干柿子被雨打湿了——颜色从黑变成了深褐。

赵一鸣发来QQ消息——"明天出来吃饭。我在一中门口那家面馆。"

"行。"

"你寒假见到她了没。"

"没。"

"没去找她吗。"

"没有。"

"你也真能忍。"

"不是忍。"

"那是什么。"

"'遵守约定'。"

赵一鸣发了一个省略号。然后说："你这种人活着太累了。"

顾临风看着屏幕。没有回复。

雨打到窗玻璃上，水珠子一颗一颗往下滑。有的水珠滑得快——往下窜了一截就散了。有的滑得慢——停在一个位置很久才动。他把手机扣在枕头边。天花板上有他在高一那年画的一个墨点——是他甩圆珠笔的时候不小心甩上去的。三年多了，墨点淡了一点。但还在。墨点周围有一圈淡淡的蓝色——是墨水洇开的痕迹。

他闭上眼睛。脑子里是那条巷子。巷子最深处的枇杷树。二楼窗户上挂着一串晒干的玉米，玉米粒是黄褐色的。门口停着的那辆旧自行车——前轮瘪了。黄狗的耳朵垂下来。收音机里的黄梅戏还在唱——肯定是那个老太太放的，从她开着的窗户飘出来。她每天放同一盘磁带。他每次经过都能听到同一段。"树上的鸟儿成双对——绿水青山带笑颜——"

"遵守约定"——听起来像一句口号。但不是。对他来说是一道墙。墙的这边是他。那边是她。他不能翻过去。也不想翻过去。他只是每天来墙边站一会儿。确认墙还在。确认那个方向的风还是那个味道——煤炉子味、烤红薯味、老槐树的树皮味。

他把这些路过的次数存在脑袋里。第一次。第二次。第三次。第四次。第五次——初七的雨。以后可能还有第六次、第七次。

但一次都没有走进去过。

一次都没有。
