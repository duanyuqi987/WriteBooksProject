# 第231章 诗歌索引之前

所属部卷：第五部：逝水有灯
时间：深夜书桌前
地点：自己的书房

## 诗歌插页 P231：原创现代诗

我把旧校徽放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

诗歌索引发生之前，生活先把场面摆好。

陆允川在整理书稿附录的时候，发现在正文中散布的四十五首诗歌——每章开头一首——从来没有被统一列过清单。它们散在各章的标题页上，像无人认领的信件。

书房里只有台灯亮着——是他大学时那盏旧台灯，塑料灯罩已经发黄了，但灯泡换了好几次，亮度反而比以前更稳。灯座底部贴着一张当年苏晚枝写给他的便签边角，上面只有一个"好"字的半边——"女"字旁被撕掉了，只剩下"子"。他不记得为什么只剩半边，大概是搬家时不小心蹭掉的。

他决定做个索引。

但做索引之前，他先翻乱了书桌。旧诗抄本——封面是牛皮纸的，和日记本同款——打开以后第一页是他大一时用圆珠笔抄的第一首仿唐绝句，字迹稚嫩得像小学生的描红。散装的稿纸、几张便签——其中一张上写着"今晚记得买牙膏"，下面又画了一盏洋葱灯——不知道是苏晚枝什么时候随手添上去的。一枚旧校徽从笔筒底部滚了出来。那枚校徽是他大一时别在书包上的，后来书包换了，校徽取下来扔进文具盒，跟着他搬了无数次家。校徽的金属面有点氧化了，边缘泛着暗色的斑，但上面的字还很清楚：县第一中学。

他把校徽翻过来。背面的别针已经锈了，用手按一下还会弹起来。弹力还在。

书桌上的电脑开着，屏幕上是文档的总目录：45个md文件，从chapter-001到chapter-245，中间缺了几十个号码——那是不需要诗歌的中间章节。每五章一个诗歌循环：现代诗、仿唐绝句、仿宋小令、现代诗、仿唐绝句。这个格式他从头坚持到尾，没有跳过任何一组。

他开始拉一个表格：编号、诗体、中心意象、写作日期。前两个好填，后两个需要查日记。

他翻开那本牛皮封面的日记本。日记的最后一页夹着一张纸——一张便利贴上写着几行随手记的诗草稿。字迹潦草，有几个字被涂掉了。

便利贴下面还有一张：是苏晚枝画的灯泡。很丑，不像灯，像个倒过来的洋葱。

他把这张便利贴抽出来，放在键盘旁边，和旧校徽并排。

这时候才发现——他的笔记本上依次列着的这些诗歌编号，正好对应着他从19岁到29岁的十年。从第一首"信纸之外"到最后一首"灯慢慢亮了"，意象在变，但有一个东西没变：每一首诗都在跟一个人说话。即使那个人的名字从未出现在诗里。

他没有给这本诗集起过书名。直到程砚有一次翻他的稿子，说："你每首诗都在写'灯'。为什么不直接叫'逝水有灯'？"

"逝水有灯"不是他的创意。是程砚的。

但此刻整理索引的时候他觉得这个名字是对的。因为"逝水"是他改不了的事——时间过去，关系变迁，人各一方。而"灯"是他唯一能做的事——把当时的光写下来。

他起身去厨房倒水。水龙头响了一下，关掉。回来的时候，便利贴上的洋葱灯被风从桌上吹到了一旁的地上。他弯腰捡起来，夹回日记本里。

不是随手夹。是夹在毕业照那一页。

然后继续做索引。表格里已经填了前二十首。还差二十五首。

他重新倒了一杯水——顺便去洗了把脸。卫生间镜子里的自己看起来有点疲惫，眼角有血丝，但眼睛里有光——不是文学作品里说的那种光，是熬夜盯着屏幕的物理反光。他用冷水拍了拍后颈，冬天水管里的冷水激得他打了一个哆嗦，但清醒了很多。

回到书桌前，他继续填表。第21首到第30首的意象连起来读，像一首精简版的个人成长史：松线（毕业）、白漆（出租屋）、辣椒酱（母亲寄来的包裹）、牛皮封面（旧日记本）、旧螺丝孔（回一中）、新绿漆（南湖凉亭）。他写这些意象的时候并没有预设它们会组成一个序列，但此刻回头看，每一个意象都恰好标记了他离开大学以后的一个关键节点。不是设计好的——是生活自己排的序。

填到第35首的时候他停了一下。这首诗的意象是"残粉"——咖啡粉。写于发现喻可婚讯那天。他把这首诗的草稿从日记里翻出来，草稿纸背面还粘着那天在茶水间撕的速溶咖啡包装袋——空的，折了一个角，贴在本子里当书签用。他把咖啡包装袋取下来，对着台灯看——透明塑料，内侧还粘着几粒没有化开的咖啡粉末，深褐色的，细得像沙子。

他忽然意识到，这些诗不只是章节的装饰。它们是他在四十五个不同瞬间里截下来的情感切片。每一首都是在当时当刻写的——没有等到冷静下来再修改，没有用"现在回头看"的滤镜去美化。这就是为什么第一版的诗总比后来的版本粗糙——但也更真实。

他决定在诗歌索引的序言里加一句话："这些诗均系创作当下所写。未经时间过滤，保留原初的情感温度。请注意——有些地方可能太烫。"

写完这句备注以后他看了一眼时间：凌晨四点。窗外还是黑的，但远处已经有一两声鸟叫——冬天的鸟叫特别轻，像怕惊醒了还在睡觉的人。

今晚不睡了。表格还剩最后十五首。
