# 第003章 初秋三楼之后

所属部卷：第一部：初秋三楼
时间：2010年九月
地点：男生宿舍与三楼教室

## 诗歌插页 P003：原创仿宋小令

灯影轻，水声平，
新书翻处有秋声。
问谁把、半张草稿，
藏过三更。
少年不识寻常重，
一字无心也动情。

陆允川把那张草稿纸夹进语文书以后，反而不敢再翻那一页。

第二天早读，他把书立在桌上背课文，眼睛却总往书脊那条缝里落。纸夹得太靠前，露出一点角，像一个不肯完全藏好的秘密。程砚坐在旁边啃馒头，嘴里含糊不清地背：“君子曰，学不可以已。”背完自己先笑了，“我觉得我可以已。”

陆允川没笑。

程砚顺着他的视线看过去，伸手就要抽他的书。陆允川一把按住。

“干嘛？”程砚挑眉。

“别乱翻。”

“里面有金条？”

“草稿。”

“草稿你藏得跟情书似的。”

陆允川松开手，脸有些热。他知道这件事说出来很可笑。一张别人顺手帮忙按住的草稿纸，没有内容，没有约定，连谢谢都是他先说的。可正因为它轻，他才更不知该把它放到哪里。扔掉太刻意，留下又显得自己荒唐。

那天上午他在数学课上走神，老师叫他回答问题，他站起来时椅子腿在地上磨出一声尖响。黑板上是一道集合题，他本来会做，可脑子里空了一下，只看见粉笔灰落在讲台边。老师皱眉：“刚开学就飘了？”

班里有人笑。陆允川低下头，说：“对不起。”

夏知遥没有回头。她坐得很直，笔尖停在本子上。陆允川在那一刻忽然觉得庆幸，又有一点说不清的失落。庆幸她没有看见自己的难堪，失落也是因为她没有看见。年轻时人的心就是这样不讲理，既想藏好，又想被某个人单独发现。

中午他没和程砚一起去食堂。程砚问他是不是被老师骂傻了，他说不饿。等教室里人走得差不多，他才从书里抽出那张草稿纸。纸面上那道错题还在，角落里有一块浅浅的油渍，是程砚的馒头蹭上的。他盯着看了一会儿，忽然觉得自己也像一道没算完的题，条件太少，心事太多。

他拿出一个硬皮本。那本子本来是母亲给他记英语单词用的，封面印着一只蓝色海豚，纸页有淡淡的化学味。他翻到最后几页，在空白处写下第一句诗：新楼风浅拂秋尘。

写完他又划掉。

太像从书上抄来的。

他重新写：我把清晨放进书包。

这句也不算好，可像他自己的话。他又往下写，写车窗、写稻田、写三楼的灯。写到“我还不知道谁会走进来”时，笔尖停住了。他没有写夏知遥的名字。不是因为克制，而是因为不敢。他怕名字一落到纸上，事情就真的变重了。

下午放学，程砚在楼梯口等他：“你今天不对劲。”

“没有。”

“你们这些说没有的人，最有事。”

陆允川把本子往书包里塞：“真没有。”

程砚看了他几秒，没再追问，只把一颗薄荷糖扔给他。糖砸在他胸口，又掉进手心。陆允川撕开糖纸，凉味很快在嘴里散开。他们并肩下楼，楼道里挤满人，谁的校服袖子擦过谁的书包，谁又在后面喊借过。陆允川被人群推着往前走，忽然觉得自己的心事并没有占满整栋教学楼。它只占了他胸口很小的一块，别人看不见，也不会因此绕路。

晚上宿舍熄灯后，他躺在下铺，听见有人还在小声背单词，有人把耳机塞进被子里听歌。窗外有虫声，风扇叶片慢慢转，像在给黑暗计数。陆允川把硬皮本从枕头下摸出来，用手机屏幕照着，又把白天写的那几行看了一遍。

他没有觉得自己文采好。相反，他越看越觉得幼稚。可是幼稚也有幼稚的用处。它让一个不敢开口的人，至少有地方把一口气放下来。

第二天清晨，他把草稿纸从语文书里取出来，夹进硬皮本最后一页。夹进去的时候，纸角露出一点。他用手指把它压平，像按住一阵不听话的风。

这件事没有改变任何关系。夏知遥还是坐在靠窗第三排，程砚还是会在课间抢他的笔，数学老师还是讲得很快。陆允川也还是那个从镇上来的新生，听课、排队、打水、在晚自习后急着回宿舍洗漱。

只是从那天起，他多了一个本子。

后来很多年，他翻到这几页时，最想删掉的不是诗句，而是当年那种把微小回应无限放大的急切。可他最终没有删。因为一个人若连自己曾经怎样笨拙都不肯承认，就很难真的变得温柔。

他把本子合上，放回抽屉。纸页合拢时有一点轻响，像三楼教室门口，某个人短暂停下脚步。
