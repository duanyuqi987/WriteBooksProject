# 第146章 食堂争执之前

所属部卷：第三部：寒风里的花
时间：六月末
地点：校外小饭馆

## 诗歌插页 P146：原创现代诗

饭馆风扇转得很慢，
啤酒泡沫贴着杯壁。
有些话还没变成争执，
先在筷子尖上停了一下，
油花微微晃动。

期末前一周，祁野拉陆允川去校外小饭馆吃饭。同行的还有两个室友，一个刚考完高数，一个刚和女朋友吵架。大家都想找个地方喘口气。

小饭馆在学校东门外，门口贴着“学生套餐十二元”。风扇挂在墙上，叶片沾着灰。老板娘把啤酒开好，往桌上一放，瓶身很快起雾。

陆允川原本不想来。苏晚枝那句“别追着我走”还在心里，他这几天尽量少说话，少看手机，也少去诗社活动室。

祁野说：“你再闷下去，人都要发霉。出来吃饭。”

饭桌上先聊考试，聊宿舍楼下新开的奶茶店，聊谁欠谁两块钱。陆允川偶尔笑一下，筷子夹着花生米，放到嘴边又放回碟子里。

其中一个室友提到诗社活动，随口说：“其实你和那个苏晚枝真不太合适。人家看着就挺有分寸，你这人一认真就吓人。”

桌子上的声音短了一下。风扇还在转，啤酒泡沫一点点往下滑。

陆允川把筷子放下，声音不大：“你了解什么？”
