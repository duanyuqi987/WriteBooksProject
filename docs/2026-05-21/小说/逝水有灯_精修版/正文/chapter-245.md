# 第245章 写成小说多年后

所属部卷：第五部：逝水有灯
时间：多年后的清晨
地点：书桌和窗边

## 诗歌插页 P245：原创仿唐绝句

书稿微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

多年后陆允川重读写成小说那一章，没有改一个字。

那是一个普通的周六早晨。妻子出门买菜了，厨房的灯还亮着，灶台上搁着一只洗了一半的锅。楼上有人在弹钢琴——《致爱丽丝》的前四小节，弹完就停了，过了一分钟又从头弹了前四小节。可能是哪个孩子在练琴，也可能是个成年人刚学。

不是因为它完美。是因为他已经没有需要修改的地方了。该删的删了，该加的人加了，该还给当事人的原话还了，该用物件收束的结尾用了一支旧中性笔。

他打开窗户。清晨的风很凉。和写完那天一样。书桌上的一切也在原处：终稿——封面上已经有了几道折痕，是妻子打扫卫生时不小心压到的。诗歌索引表格——左下角被程砚用铅笔写了一个小小的"准"。旧日记本——封面的牛皮纸已经磨出了底层的灰色纤维。毕业照——照片背面记着日期，字迹褪成了浅褐色。出租屋照片——墙上那块白漆剥落的位置刚好在照片右下角。苏晚枝画的洋葱灯便利贴——便利贴的胶早就失效了，他用透明胶从背面粘了一层，贴在书架内侧的木板上。程砚用粉笔写过字的柱子的照片——柱子的新绿漆在阳光下反着光。

这些物件不是纪念品。是他在多年里一样一样收捡起来的证据——证明他真的经历过那些日子，那些日子不是想象出来的，那些日子切实地发生在真实的街道、面馆、凉亭、出租屋、学校走廊。

他把窗户完全推开。阳光斜斜地照在书桌的边角，照在那支旧中性笔上。笔杆上有一道细裂纹——去年夏天热胀冷缩出来的。他一直没用胶带缠，因为缠上了就不好握。

手机最后一次震了。

是程砚、祁野、叶澄、夏知遥、苏晚枝五个人同时在一个群里发了消息。没有人建过这个群——这是程砚今早刚拉的。群名叫"逝水有灯读者群"。

程砚说："书读完了。今天天气好，突然想建个群。没有目的。就是想告诉老陆：你的灯照亮的不止你一个人。"

祁野跟着说："我在读书的时候看见那根松线。牛逼。从大学到现在都是牛逼。"

叶澄说："诗索引第36首'水声仍向远天生'——这一句适合赠给所有人。"

夏知遥说："偏差清单最后一项是零。鲈鱼刺的偏差你不改。我不追究。"

苏晚枝最后说："我在成都。银杏绿了。今天是阴天。但你们说的灯——我看见了。"

陆允川看着五条消息。没有逐一回复。他打了群里的回复："谢谢你们。书结束了。但人不结束。"

手机震动。群里的六个头像同时亮着。

他走到阳台上。河在远处。阳光把水面切成碎碎的光。厨房里水开了——他在烧水，准备泡杯茶。茶叶是昨天买的龙井，钱塘产。用新换的橱柜里的新水壶烧。

水开了。开关跳起来的声音干脆利落。

他泡好茶，回到书房。把终稿从书架上取下来，放在书桌中央。《逝水有灯》——书名字体是程砚选的黑体。旁边还有一张便签，写着"归给那盏一直没灭的灯"。

四十五首诗歌。十年。五十万字。一群人。一个人写。一群人收。

窗外的风慢慢停了。书页安静下来。电脑屏幕自动变暗了。厨房里的水还在壶里，温温的，不烫。刚好喝。

他把杯端到窗边。看着那条河。多年前他在面馆后面看到一段同样的水光——不是河，是路灯反在湿路面上的一小片亮片。他当时以为那是为某个人而亮的。

现在知道它不是为某个人。它就是亮。不是灯。是光的本身。

杯里的水喝完了。他回到书桌前。旧中性笔还在键盘旁边。他把笔拿起来，写了今天日记的第一行。

日期。天气。一句话：

"今天无事。书已完成。朋友们都安好。把水壶放进新橱柜里。"

合上日记。日光在书页上慢慢移。

他靠在椅背上，看着窗外那条河。河水在午后的阳光下闪着细碎的亮片——不是灯光，是自然光。楼下有人在遛狗，狗跑得快，人被拽着踉跄了几步。对面楼有个老太太在阳台上晾被单，白色的被单在风里鼓起来，像一张快要升空的帆。这些都是平常的事——平常到如果不是此刻停下笔来看，根本不会注意到它们正在发生。

但他现在注意到了。不是因为书写完了他终于有空——是因为书写完了以后，他发现那些"平常的事"本身就是答案。不需要注释，不需要升华，不需要把它们写成比喻。它们就是你绕着走了一圈又回到原点之后，重新看见的东西。

手机又震了一下。是群里的消息。程砚把群名改成了"逝水有灯读者群（已完结）"。祁野说"为什么加括号"。程砚说"不加括号感觉还会连载"。叶澄说"你们能不能安静一分钟"。夏知遥说"我还在统计偏差"。苏晚枝没有说话，但她在群里发了一张照片——一棵刚浇过水的香菜，种在一个白色的花盆里，放在她工位的窗台上。阳光刚好打在香菜的叶子上，叶片上的水珠反着光，像一粒粒很小的灯。

陆允川看着这张照片。他没有在群里回复。但他在日记里加了一句话："苏晚枝在成都种了一棵香菜。不是因为我说了香菜叶太多遍。是因为香菜本来就好养。"

人没有在等什么。只是把水喝完了。然后继续过着。
