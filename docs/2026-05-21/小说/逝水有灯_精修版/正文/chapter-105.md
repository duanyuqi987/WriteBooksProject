# 第105章 诗社传单多年后

所属部卷：第二部：六月以后
时间：大一社团周
地点：图书馆前广场

## 诗歌插页 P105：原创仿唐绝句

红章微凉近晚风，
少年心事隔帘中。
不知旧梦归何处，
一盏疏灯照水东。

搬家公司在上午十点到。三个工人，每人扛着两个纸箱，从六楼往下走。陆允川蹲在书房地板上分拣剩下的杂物，膝盖压着一卷宽胶带。

"这箱是什么？"搬家公司的人指着一个旧的牛皮纸箱问。

"等一下我自己搬。"陆允川说。

那是大学时期的东西。毕业十二年，搬了五次家，每次这箱东西都会跟着走，但也很少打开。纸箱的一角已经塌了，露出里面几本书的书脊。

他撕开封口的胶带。最上面是毕业照。A4大小，压了膜，边角微微卷。他站在第三排左起第七个，穿着学士服，额前头发被风吹出一个不自然的弧度。

照片下面是几本教材：《古代汉语》《中国文学史》《外国文学简编》。书页侧面已经泛黄，有几本的书脊和书芯分了家。

教材下面是诗抄本。

棕色硬壳，书脊磨出白边。他拿起它的时候，从第47页掉出来一张纸。

传单。

"诗酒趁年华"——标题还在。纸质已经变得很脆，折痕处几乎要断开。红印退了颜色，从原来的朱红变成浅褐，洇出的毛边也模糊了，像被水泡过又晾干。社团章歪着，和他记忆中一模一样。

他把传单翻过来。背面的社刊摘录里那首四行短诗还在：

"陌生人在桥上停下/不是为了看河/是因为桥上的风/比岸上的凉"

落款是个不认识的名字。大概是当年某个社员的习作。

陆允川蹲在地上，把传单举到眼前对着光看。光线透不过纸，只能看到背面诗的影子。传单边角的褐色印迹不是旧，是当年叶澄手心出汗洇开的。

他记得叶澄。后来叶澄考了教师资格证，在老家教中学语文。三年前同学聚会在县城的饭店里吃了一顿饭，她带着女儿来的，女儿穿着粉色的棉袄，在饭桌下面钻来钻去。叶澄说"你还写诗吗"，他说"偶尔"，她说"当年诗社那些事你还记得吗"，他说"记得"。两个人都没有再往下说。

他把传单小心地放到一边，继续翻诗抄本。

第47页空着——不对，不是完全空着。在右上角，铅笔写了三个字："允，信也。"字迹很淡，像是写了之后很快被擦过。

那是苏晚枝说的。《说文》里的一句话。

多年后陆允川已经不记得苏晚枝的声音。他记得她说话的内容，但不记得音高、音色、语速。这是一个很奇怪的记忆现象：一个人说的话留下来了，但说话的声音消失了。像是一首歌的歌词还记得，但旋律怎么都想不起来。

他把诗抄本往前翻。第23页，《风铃》。高中写的。他在操场看台上写的，旁边坐着程砚。程砚看完说"太文了"。第35页，《旧车站》。高考前一周写的，那时候觉得人生最大的一件事即将发生，写完了回头看，也不过是四个路口外一个废弃的站台。

第47页。传单从这里掉出来。

他把传单重新夹进第47页，想了想，又拿出来。

传单右下角有一个笔迹他没认出来——不是他自己的字，比他的更圆更软。是苏晚枝写的。她当年在他填表的时候在传单空白处写了两个字，他没注意到。

"来了。"

就两个字。大概是随手记的。可能是当天诗社的报名人数统计，也可能是给其他社员的备注。

但陆允川停在这两个字上很久。

"来了。"

十八岁那年，他在大学广场上接过一张传单，填了一张报名表，被一个学姐解读了名字。二十年后他蹲在搬家前的地板上，发现她在传单上写了"来了"。

他忽然觉得这两个字像一句回话。像是对当年他自己的一个简单的确认：你来了，我记下了。

电话响了。搬家工人在楼下催："陆先生，还有东西吗？要发车了！"

"来了！"陆允川喊了一声。

喊完他自己愣了一下。

他把传单夹回诗抄本第47页，把诗抄本放进牛皮纸箱，把牛皮纸箱抱进电梯。

电梯里只有他一个人。镜子里的自己：四十多，头发还没少，但鬓角白了。穿一件灰T恤，上面沾着打包时落的灰。

他想起那天广场上的帐篷。白色的，一排一排撑开。扩音器的声音很大。祁野在篮球社那边填表，叶澄在发传单，苏晚枝坐在桌子后面说"你的名字是哪几个字"。

他记得那天太阳很晒，传单的纸边在手里越捏越软。

电梯到了。他把纸箱放进搬家车的后厢，工人帮他挪出旁边的一块空间。他坐进驾驶座，发动车，空调还没凉下来。

副驾驶上搁着那本诗抄本。他伸手翻开第47页。

传单还在。红印还在。

他从手套箱里找出一支笔，在传单空白处——在苏晚枝写的"来了"旁边——写了两个字。

"嗯。"

车开出小区大门。导航里女声说"前方三百米右转"。

他把诗抄本合上。
