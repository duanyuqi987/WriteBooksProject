# 第102章 饭卡余额



时间：2010年四月下旬，晚饭前



地点：食堂充值处、操场西侧



卷次：第五卷：未来在黑板上倒计时



2010年四月下旬，晚饭前，食堂充值处、操场西侧，段璟饭卡里突然多出来的二十块钱把2010年的春夏推到十班面前。段璟发现父亲多打了二十块饭费，却在电话里听见家里说今年小麦收成不稳。

他想把钱省下来，又不想让同学看出自己在算饭菜价格。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

段璟也在这段日子里显出自己的位置。敏感、自尊、理科偏强，喜欢常俊凡，却越来越怕自己的喜欢变成拖累。群像不是名单，群像是每个人都有自己的难处。

“钱够不够？不够再说。”父亲说。

“够，食堂也不贵。”段璟说。

“你今天怎么只打一个菜？”付小川说。

“不饿。”段璟说。

“不饿和舍不得不是一回事。”常俊凡说。

食堂窗口前总有人把饭卡拍得很响，像那样余额就会多出来一点。

充值处的队伍贴着墙排，前面是打菜的热气，后面是学生翻找零钱的声音。

饭卡上的数字很小，却比成绩还具体。少一块钱，窗口就会停一下，后面的人也会跟着看过来。

常俊凡不喜欢在这种时候追问。她知道有些窘迫越解释越难堪。

段璟端着餐盘站在旁边，第一次觉得自己平时说不出口的自尊，其实别人也有。

食堂师傅把勺子在盆边敲了敲，提醒后面的人快一点，队伍又慢慢往前挪。

午饭并不浪漫，白菜、豆腐、米饭和饭卡余额，却能把人的日子照得很清楚。

吃完饭回教室时，后墙的错题区还在那里，像另一种账本。

胡向南这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

段璟被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

段璟是在充饭卡的时候发现多出二十块的。他把卡插进读卡器，屏幕上跳出一个数字：54.3元。上周充了三十，吃了一周，按理说余额应该在三十出头。他把卡拔出来，又插进去，数字没变。

他站在充值窗口旁边，手里捏着饭卡。充值处的队伍还在往前挪，后面的人催了一句"快点"，他才把卡塞回口袋，往食堂门口走。多出来的二十块只有一种可能——他爸多打了。

食堂里正在晚饭高峰。打菜的窗口前热气蒸腾，白菜豆腐的气味跟红烧肉的酱油味混在一起。段璟排在窗口前，先看了看今天的菜价——素菜一块五，荤菜三块，米饭五毛。他算了一下：如果只打一个素菜加米饭，两块。一天三顿都这么吃，一周能把二十块省回来。

"今天怎么只打一个菜？"付小川端着餐盘从旁边走过，看了一眼段璟的盘子。

"不饿。"

付小川没有追问。但他把自己的餐盘放到段璟对面，坐下来。付小川盘子里是两个菜——一个土豆丝一个红烧肉。他吃饭的方式很规律：先吃完米饭的一半，然后夹菜，最后把剩下的米饭和菜汁拌在一起吃完。

"你爸上周给的钱不够？"付小川头也不抬。

"够。"

"那你只打一个菜干嘛。"

段璟没有回答。付小川把一块红烧肉夹到段璟盘子里，动作快得像在递一张卷子。

"我不饿。"段璟说。

"不饿也吃。你今天下午物理课站了三次，第一次是做题，第二次是去水房接水，第三次是站在后排揉腿。揉腿是因为饿的。"付小川说完继续吃，没有看段璟的反应。

段璟把红烧肉夹回付小川盘子里。"你的。"

"你夹回来我也不会夹回去。不是嫌你，是这块肉本来就是多打的。"

段璟知道付小川的"多打"是什么意思——不是食堂多给了，是他自己多打了一份。付小川的饭卡里从来不缺钱，但他从不炫耀这个。他的帮助方式跟常俊凡不同：常俊凡是把问题拆开讲给你听，付小川是直接把东西放你面前然后立刻转移话题。

晚上，段璟用宿舍楼下的公用电话给家里打了一个电话。铁壳电话掉了一块漆，投币口旁边有人用圆珠笔写了一个"快"字。他投了一块钱硬币，拨了家里的座机号。

"爸，饭卡里多了二十块。"

"嗯，我多打的。你妈说这周补课费涨了怕你饭钱不够，我想着多打点，你别省着吃。"父亲的声音在电话里有点沙，背景里有拖拉机的声音——可能是刚从地里回来。

"小麦今年怎么样？"

电话那头沉默了两秒。"还行。就是今年雨水不太平均，拔节的时候旱了一阵，现在灌浆又怕倒伏。你好好念书，地里的事别管。"

段璟手指卷着话筒线。他知道"还行"是什么意思——对种地的农民来说，"还行"就是不太好但还没到最差。他听得出那个停顿。每次父亲要说真话之前都有一小段停顿，然后换成一句更轻的话。封丘四月底的风大，黄河大堤方向吹过来的风把电话亭的铁门吹得咯吱响。

"我饭钱够。多出来的我给你打回去？"

"不用。留着，买点水果。你妈说你们学校门口有卖树莓的，两块五一盒，去买一盒。"

段璟挂了电话以后在电话亭里站了一会儿。铁壳上的漆又掉了一块——就在他手扶着的位置。他爸不知道县城补课费到底涨了多少，但知道树莓卖两块五一盒。有些数字父亲不问你，但他自己会找到。

回到教室时晚自习还没开始。常俊凡坐在第二排，正在翻一本物理竞赛辅导书。她的草稿纸边上写着一行小小的数字——不是做题的演算，是日期和进度标记。

段璟路过她座位时看了一眼那个标记，不小心碰到了她桌角放着的杯子。杯子晃了一下，没倒。

"你手冷。"常俊凡说。

"外面风大。"

"你晚饭吃了吗？"

"吃了。"

常俊凡没有继续问。但她从书包侧袋里拿出一个塑料袋，里面是两个煮鸡蛋。她把袋子放在课桌上，往旁边推了推——推的方向是段璟的方向。她没有说"给你"，也没有抬头。这个动作小到只有段璟自己能看到。

"我吃过晚饭了。"段璟说。

"鸡蛋不是饭。"

段璟拿了一个鸡蛋，在桌角磕了一下，开始剥壳。蛋壳很薄，一磕就裂成了几条缝。他剥得很慢，蛋白被剥下来一小块，他把那一小块也捡起来吃了。常俊凡全程没有抬头，但她的笔停了一秒——段璟剥完鸡蛋开始吃的那一刻。

第二天中午，段璟又去了食堂充值处。这次他没有插卡查余额，而是站在充值窗口前，看了看旁边的墙上贴着的菜价表。菜价表是手写的，粉笔字，每周更新。这周素菜多了个炒豆芽——一块钱。他把豆芽和米饭的组合在心里算了一遍：一块五。一天三顿四块五。一周三十一五块。

"你一直在看菜价表，看了大概两分钟。"常俊凡的声音从背后传来。段璟转过身，她站在食堂门口，手里端着茶杯。

"我在想豆芽和白菜哪个便宜。"

"豆芽一块，白菜一块五。"常俊凡说，"但你如果只吃豆芽，一周以后你上物理课会犯困。蛋白质不够。"

段璟没有说话。常俊凡走到他旁边，没有看菜价表，而是看着充值处的队列。"你爸多打的钱是因为家里宽裕了还是因为想让你别省着？"

"后者。"

"那你省回去没用。他多打钱已经是一种表态——'家里撑得住，你不要省'。你把钱省回去等于在说'我不信你撑得住'。"

段璟把这句话想了好几秒。他想起父亲在电话里的停顿——那个停顿不是为了藏坏消息，是为了让他不要分担。父亲的选择是撑住，他如果省回去，等于推翻了父亲的选择。

"你今天午饭打两个菜。"常俊凡说完就走了。茶杯盖子拧紧，杯子放进书包侧袋——左边，笔袋在中间，右边空着。

段璟在窗口打了两个菜：豆芽和土豆丝。两个菜加起来两块五，比只打一个菜多了一块，但比荤菜便宜。他把餐盘端到付小川对面的位置，坐下来。

"今天两个菜了。"付小川说。"嗯。""常俊凡跟你说了什么？""她说如果只吃豆芽，蛋白质不够。"

付小川点了点头，不是对常俊凡的观点认可，是对"段璟终于愿意听人劝"这个事实认可。

下午第一节是胡向南的语文课。他讲了《陈情表》的最后一段，然后突然换了一个话题："李密写《陈情表》不是卖惨。是把自己真实的处境讲清楚——祖母多病、自己进退两难。讲清楚不等于求同情。你们以后写作文也好，跟家里沟通也好，记住一点：说事情，不要演情绪。"

段璟把这句话在语文课本的空白处写了下来。他忽然觉得这句话也可以反过来理解：他之前想在食堂省饭钱，其实也是一种"演情绪"——演的是"我能扛"，但家里希望的不是他扛，是他如实地说够不够。

下课后他在走廊上给家里打了一个电话。这次用的时间很短——几十秒。

"爸，饭钱够。但要是下个月补课费真的涨了，你提前跟我说，我这边能省的地方提前省，不会饿着。"

电话那头他爸沉默了两秒——不是找话，是在接收这句话。"行。那你好好吃饭。""嗯。"

挂了电话，段璟走出电话亭。四月底的风从黄河大堤方向吹来，带着麦子灌浆时特有的青草味。麦子快熟了，但还没熟。这个阶段最怕倒伏——一场大风就能让麦秆贴地，贴地的麦子收成减半。

他回到教室时，付小川的错题本摊在桌上。数学栏那一列已经记到了第四十七道错题。段璟看了一眼，发现付小川也在记——不是给别人看，纯粹是自己用。

"你数学还需要记错题？"段璟问。

"我数学好是因为我记错题，不是我不记错题。"

段璟在自己的错题本上翻到今天的新页。Day 11，物理受力分析全对。他在"全对"两个字下面画了一条线，然后翻到第五章的综合题——斜面加弹簧那道。

晚自习的时候，段璟做完了弹簧那道综合题。做完以后他把答案跟常俊凡的对照了一下——她没有主动给他看，是他路过她座位时她草稿纸正好摊着。解题步骤完全一致，唯一不同的是最后一步：她用了简化的公式，他用了完整的推导。答案一样，但她的方法更快。

段璟在错题本上把那道题的两种解法都记了下来。左边是自己的推导，右边是她的简化公式。中间用一道竖线隔开。

第二天下午补课班下课后，段璟路过城关镇路口的树莓摊。摊主还是那个妇人，泡沫箱盖子掀开，透明盒里的红树莓水珠还没干。他站了一会儿，掏了两块五，买了一盒。

"上次跟你一起那个女同学呢？"摊主找钱时问了一句。

"在教室。"

"你们一中的学生，天天补课，辛苦。"

段璟没有接话。他把树莓拿回教室，放在常俊凡桌上。常俊凡正在改错题，抬头看了一眼树莓，又看了一眼段璟。

"你买的？"

"嗯。我爸让我买的。"段璟顿了顿，"他知道树莓两块五一盒。"

常俊凡把树莓盒打开，拿了三颗。她没有说谢谢——不是因为不礼貌，是因为在她看来，有些东西说了"谢谢"就变成了交换。她只是把树莓放到桌上，跟她的物理竞赛辅导书平齐，像这个下午的一个坐标。

秦佳禾路过时看了一眼树莓，笑了一下。"段璟你什么时候学会买水果了？""从我爸知道树莓卖两块五的那天。"

秦佳禾没有追问。她知道这句话里面有什么东西。她家里也这样——爸爸知道她在学校食堂打一个菜还是两个菜，但从来不问，只是每个月往卡里多打十块。无声的多打，是封丘家长最常用的表达方式。

放学后，段璟在教室门口跟付小川站了一会儿。天色还没完全暗，路灯刚亮，街口树莓摊还没收。付小川把书包往肩上拉了拉，看着远处说："你爸是那种不会说好话但会做事的人。""你怎么知道？""因为他多打了钱没跟你商量。如果商量，你一定说不用。他就直接打了。"

段璟低头看着自己的鞋——鞋底磨平了一小块，是每天从教学楼走到食堂那三百米磨出来的。

"我一直觉得钱不够是家里的问题。现在觉得不是。家里能给的已经给了。剩下的是我能不能把给的变成对的。"段璟说。

付小川把这句话听完了。没有点头，没有"你说得对"。他只是把书包带子又拉了一下，说："走，晚自习要开始了。"

两个人一前一后往教室走。教学楼里的灯已经全亮了，日光灯管嗡嗡响。

教室里何南星正站在错题墙前面，拿着龙门专题比对自己的错题纸。他把上周的步骤跳错题重新做了一遍贴在墙上——每一道题旁边现在都标了步骤编号。邓桥趴在桌上用透明胶把错题墙的边角又贴了一次，上次黑板报留下来的蓝色卡纸还剩一张，他剪成小块压在透明胶下面做了补丁。

常俊凡正在收拾书包。她把今天做过的物理竞赛题按日期整理好，每一张草稿纸的右上角都写了日期和完成状态。段璟注意到她的草稿纸堆里有一张颜色不一样的——是上次那张便签，上面写着"物理竞赛基础篇第三章到第五章"。

那张便签她还留着。压在物理竞赛辅导书的扉页里，露出一个小小的角。

晚自习铃响的时候，段璟翻开错题本，在新的一页上写了Day 12。物理第五章综合题——斜面加弹簧——今天第一次在二十分钟内做完，全对。他在"全对"旁边画了一个正字的第三画。正字还差两画，但他已经知道最后两画会来。

窗外黄河大堤方向的风还在吹。四月快结束了，麦子灌浆的声音听不见，但风里有土味。那种土味跟操场上的不一样——操场上的土是干的粉的，大堤方向吹来的土是湿的重的，带着黄河水和麦秆的混合气息。

段璟想到了父亲电话里说"还行"之前的那两秒停顿。那两秒里装着一整个灌浆期的焦虑——怕倒伏、怕旱、怕一场雨下在不该下的时候。但父亲还是选择了说"还行"。这不是撒谎，是给儿子留一个安静的晚自习。

他把这道题收进错题本，合上书页。封面上的"物理错题"四个字已经被手指磨得起了毛边。





饭卡余额之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

饭卡余额让他第一次认真想，所谓未来要从家里一分一角地长出来。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，段璟饭卡里突然多出来的二十块钱还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
