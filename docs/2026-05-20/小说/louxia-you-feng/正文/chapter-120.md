# 第120章 天明已是昨之后



天明之后，许亦安没有再打开文档。电脑合着，屏幕黑下来，窗外的光落在外壳上，像一层很薄的水。

房间被早晨慢慢填满。台灯关了，水杯洗过，桌上的纸页被压在书下，只露出一个空白角。

楼下有人笑着经过，声音从窗缝里进来，很快又远了。

他站在书柜前，看见旧日记本安静地放在第二层。没有拿出来，也没有伸手摸。

如果非要对过去说一句话，他想，大概也只是：放在那儿吧。

他换了衣服，把垃圾袋拎到门口。袋子里有废纸、空水瓶和一小截用完的笔芯。

林青禾被留在故事里，也被还给自己的生活。这个还字很轻，却是许亦安写了很久才学会的。

他终于不需要她理解这本书，也不需要她为他的青春盖章。

结尾不需要响。真正落下来的东西，往往只是一个人关灯、出门、把钥匙放进口袋。

周一鸣后来问他下一本写什么。许亦安说不知道。这个不知道听起来不焦虑，反而像一扇窗开着。

曹原说别写我坏话。许亦安回已经写了。曹原骂了一句，群里又热闹起来。

门口的鞋带松了，他蹲下重新系好。鞋带穿过指间，普通得让人安心。

他把旧校徽放进一个小铁盒，和几枚不用的钥匙扣在一起。盒盖扣上时，声音很轻。

这不是爱情的胜利，也不是失败后的漂亮说辞。它只是一个人终于能把旧物放好，把门带上。

走廊窗户开着，风把公告贴吹得贴紧墙面，又慢慢松开。

青春被重新放回温柔的位置：不是失败，不是胜利，只是经过。经过之后，人还要下楼，还要吃饭，还要去新的地方。

他没有再给林青禾发消息。不是因为忘了，而是因为记得也可以很安静。

出门前，他检查了一遍钥匙、手机和钱包。没有检查旧日记本，也没有回头看电脑。

楼下风很轻，早餐摊前排着三个人。许亦安站到队尾，听见锅里的油轻轻响。

豆浆递到手里时很烫。他换了一只手，低头吹了吹，像吹散一个终于不用再解释的昨天。

人群往各自的方向走，没有人知道他刚刚写完一本关于青春的小说。这很好，故事不必总被看见才算存在。

书桌留在身后，电脑、旧校徽、日记本都在自己的位置。位置合适，心就不再被它们牵着走。

后来再有人问起林青禾，他能笑着说，是很久以前的同学。不是轻描淡写，也不是隐藏伤口，只是时间终于给了他一个平常说法。

旧日记本没有成为遗物，也没有成为圣物。它只是一本旧本子，放在书柜第二层。

最后留下来的不是一句誓言，而是一阵从楼下吹上来的风。

天明已是昨，不是说昨天消失了，而是说昨天终于不再冒充今天。

这一章真正留下来的，是他出门时没有回头。

许亦安拎着早餐往前走。风从楼下经过，吹动路边树叶，也吹过他空出来的手心。故事到这里停下，生活没有停。
