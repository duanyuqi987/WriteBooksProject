# 陆允川

男主角。乡镇中学出身，考入县一中后敏感、自尊、沉默，擅长把话写进诗抄本而不是说出口。

## 核心欲望

他想被看见，想证明自己从乡镇来到县城并不是偶然，也想证明自己那种笨拙而漫长的喜欢有价值。

## 核心恐惧

- 害怕自己只是普通人，普通到连喜欢也不会被记住。
- 害怕付出没有回声，于是不断寻找证据。
- 害怕被朋友看穿，又渴望有人把他拉住。

## 行为习惯

- 把想说的话写进诗抄本。
- 情绪越重，语气越假装平静。
- 遇到关系问题时先分析、猜测、记录，最后才开口。

## 声纹

早期说话绕，喜欢用“没事”“随便”“都行”遮掩真实愿望；后期句子变短，更愿意承认“不知道”和“我也有错”。

## 人物弧线

- 初期：把一次心动视为命运，把暗恋写成自我证明。
- 中期：在大学里把被需要误认为被爱，把喜欢推向执拗。
- 后期：承认自己并不了解对方，也没有资格要求别人回应自己的全部期待。
- 结局：把旧日记和诗歌整理成小说，温柔地承认青春已经过去。
