# 第144章 "那年九月风初凉"
时间：2018年4月中旬
地点：楚州月湖边
卷次：卷八：旧诗新读

周六下午。他骑车去接她。

一辆旧自行车。二八寸的，链条有点松，骑起来咔咔响。他在楚州一小门口停下来，脚尖点地。

苏晚晴从教学楼里走出来。手里拎着一个帆布袋。看到他，步子快了一点。

"你真骑车来的。"

"你叫我骑的。"

她看了看后座。后座上有一层薄灰。他用手擦了擦——手心里全是灰。

"上来吧。"

她侧身坐上去。手放在自己膝盖上，没有扶他。

"扶着我。路上有坑。"

她的手从膝盖上抬起来，放在他的腰侧。很轻——只碰到了衣服。

链条咔咔地响。风从耳边过。四月的风不凉不热，刚刚好能吹起头发。她的头发飘起来，有一缕扫到了他的后颈。

他没说话。

"往左拐。"她说。

"不是去月湖吗。"

"先去个地方。前面红绿灯左拐。"

他按她说的走。拐了两个弯，到了另一条路。路边种的是法国梧桐，树干上斑驳的白皮。

"就是这里。停一下。"

他停下来。她下了车，站在路边一棵梧桐树下。树冠很大，把整条人行道都罩在阴影里。

"这棵树的叶子跟青阳的不太一样。"她仰头看了看，"青阳的是中国梧桐。"

他站在旁边，等她。她看了大概半分钟，说"走吧"。

到了月湖。下午的月湖人不多。水面平静。他把自行车停在入口的栏杆上，锁了。

"锁什么。"她说。"没人偷这种车。"

"习惯了。"

他们沿着湖边慢慢走。今天走的是逆时针方向——先过水杉林，再过亭子。亭子里今天有人——一个老人在拉二胡。拉的是《二泉映月》——音不准，但拉得很慢，像一个一个字在念。

他们在亭子外面听了一会儿。

"走吧。"她说。

柳树下面有一排石凳。她挑了一个靠水的坐下。他坐在她旁边。两个人中间隔了一个人的距离。

她看着水面。湖面上的柳条影子在水里晃。晃了很久。她开口了。

"那年九月风初凉。"

声音很轻。但每个字都很清楚。

那年——九月——风——初——凉。

五个字。

湖面上的柳条影子还在晃。亭子里的二胡停了。一只鸟从水面上飞过去。

顾临风坐在石凳上。手放在膝盖上。手指蜷起来，又伸直。指尖碰到石凳的凉意。

他不知道过了多久。可能五秒。可能更久。他听到自己的心跳声——不是说像什么敲鼓。就是能听见。左胸口那里，咚，咚，咚。

"那年。"他说。

"九月。"她说。

"风初凉。"

"学府门开菊未黄。"

她说的是第二句。

他转头看着她。她没看他。还看着湖面。风吹过来，把她的刘海吹到眼睛上。她抬手拨了一下——指尖从额头上划过去，把头发别到耳后。

"你怎么——"

"知道这两句。"

后面两个字他说不出来了。

她还看着湖面。阳光从柳树叶子间漏下来，在她脸上落下碎碎的光。她的表情很安静。不是平静——是安静。像翻一本旧书，翻到某一页，看到页脚折了一个角。

"高二那年。十月。"

她终于转过头来了。

"你坐在靠窗那排。第三排——靠走廊那边。你们班的教室在我们班楼下。那天课间操结束之后，我经过你们教室门口。"

她说的每一个字都像在念一段写好的文字。不快。不慢。没有多余的词。

"你的笔记本摊在桌上。翻开的。蓝色封面。那一页写着这首诗。"

"你看到了。"

"看到了第一句。和最后两句。"

他记得那个笔记本。蓝色封面，右下角用黑色水笔写了一个"风"字。封面磨得有点白了——他从高一开始用，一直用到高三。

"我进门之后，你还没写完。正在改第三段。你低头写字，没看到我。"

"你在门口站了多久。"

"大概三秒。"

三秒。

八年。

他又听到了自己的心跳。左胸口那里。咚。咚。咚。

"你为什么没有——"

"没有告诉你。"她接过他的话。

"嗯。"

"因为你低着头在写字。我看着你的笔尖在纸上停了一下，又划了两下——改了两个字。然后把最后一段画了一个圈。"

"我不知道。"

"你当然不知道。你从来没抬过头。"

他说不出话。

湖面上的风停了一瞬。柳条垂到水面，不动了。有只鱼从水下游上来，嘴唇碰到柳叶尖——啵一声，又沉下去了。

"那个笔记本——"他说。

"后来你还写。一直写到高三。体育课的时候。课间操的时候。自习课的时候。你总是在写。"

"你一直看着。"

"不是一直。是偶尔经过你的教室——你的笔记本总是摊开的。有时候翻到了新的一页，有时候是旧的一页。有一页上的字被橡皮擦过好多次。纸擦得有点皱。你在上面写了又改，改了又写。"

他的手放在石凳上。指节发白。

亭子里的二胡又响了。这次拉的是《梁祝》——还是音不准，还是拉得很慢。

"那首诗。"他说。

"什么。"

"你知道名字吗。"

她停了一下。"《初见》。"

他没说话。她什么都知道。

又沉默了一会儿。风开始大了。柳条摆动得更厉害。湖面上的倒影碎成千万片。

她把双手交叠在膝盖上。手指交叉着，大拇指互相摩挲。

"为什么现在才告诉我。"

"因为——"她停了一下。"你该知道了。"

"知道什么。"

"知道不是只有你一个人在记住那些事。"

他从石凳上站起来。走了两步。又走回来。在她面前站住。

"苏晚晴。"

"嗯。"

"那首诗。我可以念给你听吗。"

她抬头看他。眼睛里有光——很细碎的一点。不是眼泪。是湖面上反射过来的阳光。

"好。"
