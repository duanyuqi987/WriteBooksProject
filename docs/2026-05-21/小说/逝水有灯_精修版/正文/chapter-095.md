# 第095章 夏天的聚餐多年后

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P095：原创七言小诗

尘过长廊日影深，
一行旧字压衣襟。
少年未解离分意，
先把沉默写到今。

工作第二年夏天，陆允川出差路过县城，待了一晚。

程砚正好也回来——厂里年中检修，放了三天假。两个人在电话里约好了，去那家川菜馆再吃一顿。

"那家店还在不在？"陆允川问。

"在。换了个老板。菜的味道没变。"

晚上七点，两个人坐到了当年升学宴那张桌子旁边的卡座。玻璃转盘还在，但比以前小了一圈。菜单压了一层塑料膜，上面用手写加了一道"老板推荐：水煮牛肉"。

程砚没有穿衬衫——穿了一件厂里发的文化衫，领口洗得有点松了。他比以前胖了一点。陆允川也胖了一点。

"你后来跟哪些人还有联系？"程砚一边翻菜单一边问。

"叶澄。你。没了。"

"林曼呢？"

"群里有她的朋友圈转发。没聊过私信。"

"夏知遥呢？"

"没有。"陆允川把筷子从纸套里抽出来，"你上次问我这个问题是什么时候？"

"六年前。大二寒假。你那时候说没有。现在还是说没有。"

"因为真的没有。"

菜上来了。水煮鱼、回锅肉、麻婆豆腐、一盆酸菜粉丝汤。跟六年前一模一样的四道菜。程砚把鱼转到陆允川面前。"你先吃。你上次帮我挑了刺，今天我帮你挑。"

"你会挑鱼刺？"

"不太会。但可以学。"程砚拿起筷子，夹了一块鱼，开始低头挑刺。他的动作很笨——手指大，刺又细，挑一根要花十秒。陆允川看着他挑了一会儿，笑了。"你好像一只熊在拆炸弹。"

"熊能把炸弹拆了那是天才。"程砚头也不抬。

鱼刺挑了一半，程砚忽然说："其实我一直觉得，那次升学宴是你高中时代的一个重要节点。"

"为什么？"

"因为你那天晚上第一次在很多人面前没有低头。我三叔问你土木工程是干什么的，你说'就是造桥修路的'。那是我第一次看到你用一句话回答一个人，而不是用三句话解释自己。"程砚把挑好刺的鱼肉夹到陆允川碗里，"你后来在大学里做presentation、答辩、面试的时候，是不是都是用的那个语气？"

陆允川想了想。好像是。大一第一次在专业课上做报告，他站在讲台上想到了程砚妈妈那天的笑脸，想到了她说"你回去告诉你妈妈，你这孩子是我们程砚最好的朋友"。然后他的紧张就消了一半。

"你妈那天说的话，我记到现在。"陆允川说。

"她说什么了？"

"她说我是你最好的朋友。我回去告诉我妈了。我妈说，程砚妈妈人真好。我说对。"

程砚放下筷子看了他一眼。然后又把筷子拿起来，夹了一块回锅肉塞进嘴里。"肉还是那个味。够咸。"

吃完饭，两人在店门口站了一会儿。街对面的水果店还在——但换成了连锁品牌，门头亮着LED灯。水果店隔壁的便利店也换了。只有那家川菜馆还在，招牌换了"老张川菜"，但辣椒还是一样辣。

"你觉得下次我们回来，这家店还会在吗？"程砚问。

"会。只要老板还炒得动菜。"陆允川把打包的酸菜粉丝汤拎在手里，"走吧。你回去帮我跟你妈说，她当年那句话我记了十年。"

"她会笑死。她最得意的就是那次升学宴。"

两人在街口分开。一个往南，一个往北。路灯把他们的影子拉成跟六年前一模一样的角度。只不过这次影子比那时候宽了一点——两个人都胖了。
