# 第123章 教室的灯之后

所属部卷：第三部：寒风里的花
时间：同一天夜里
地点：二号教学楼空教室

## 诗歌插页 P123：原创仿宋小令

楼静水痕长。
灯照空台，粉字未成霜。
一页纸轻人跑远，
门禁声里夜微凉。

陆允川回到宿舍时，祁野正在洗袜子。水盆放在地上，洗衣粉泡沫堆到盆沿。

“你又去哪儿了？”祁野问。

“送文件。”

“文件送出去了？”

陆允川把书包放下，没有立刻答。祁野抬头看他一眼，明白了，低头继续搓袜子。

“人家说不用找，你是不是还去了？”

陆允川拉开椅子坐下，“我只是顺路。”

“二号楼和宿舍不是一条路。你当我没腿吗？”

这话不重，却比安慰有用。陆允川打开书包，文件还在里面。那页纸原本平整，现在角上有一道折痕。他忽然觉得自己这一天所有解释都很薄，薄到被朋友一戳就破。

他没有再给苏晚枝发消息。群里她已经把签名照片传上来，叶澄标了明天装订时间。事情被处理得清清楚楚，只有他多跑了一趟。

夜里熄灯后，陆允川听见上铺翻身，床板轻响。他在手机备忘录里写：不是所有空着的地方都在等我去填。

写完他看了很久。那句话没有让他立刻舒服，却比“我很难过”更接近事实。
