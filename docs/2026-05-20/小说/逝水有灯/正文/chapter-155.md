# 第155章 南湖问答多年后

所属部卷：第三部：寒风里的花
时间：多年后
地点：南湖河边

## 诗歌插页 P155：原创仿唐绝句

旧椅新漆认湖风，
三字当年纸背空。
问到后来方觉浅，
原来答案在停中。

多年后陆允川整理旧文件，在一本社团资料夹里翻出那张折成四折的纸。纸上是叶澄写的三个词：道歉、改变、等待。

墨迹有点淡，折痕处快要裂开。他把纸摊平，放在窗边拍了一张照片。照片里看不出南湖，也看不出当年的狼狈，只是一张旧纸。

他曾经以为那次问答会给他一个方法，让他重新靠近苏晚枝。后来才知道，叶澄真正教他的不是怎么挽回，而是怎么停下。

停下听起来太简单。可对当时的陆允川来说，停下比说一百句道歉都难。他总想做点什么，让别人看见他的后悔。可很多时候，真正的后悔就是先不再制造新的负担。

他把那张纸放进透明文件袋，标签写成“南湖问答”。不是纪念某种清醒，而是提醒自己：很多问题问到最后，都不是要别人回答。
