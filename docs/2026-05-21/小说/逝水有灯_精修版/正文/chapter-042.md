# 第042章 诗抄本

所属部卷：第一部：初秋三楼
时间：高二秋天
地点：教室后排的课桌抽屉

## 诗歌插页 P042：原创仿唐绝句

字里藏灯未可明，
少年心事上诗名。
当时不解寻常意，
却将抄写作倾听。

事情真正发生时，并没有陆允川想象得那么郑重。语文老师让大家把第一次抄的诗交上来检查。陆允川选了一首余光中的《乡愁》，小的时候乡愁是一枚小小的邮票——他抄得很慢，每一笔都端正得像刻在石碑上。抄的时候用的是0.5毫米黑色中性笔，笔尖在纸上划过时几乎没有声音，只留下微微凹下去的笔画痕。语文老师在翻他的本子时说了一句"字不错"，然后把本子传给前排在投影上讲评。本子被投影放大在屏幕上——第三排的夏知遥抬头看了一眼。他发现她的目光在屏幕上停了大概三秒，然后低头继续抄笔记。那三秒里有她的反应：没有评价，只是看了一眼他的字——横平竖直，力道均匀，大概和她开学时在扉页上写名字时他自己心里的感觉一样。

她的诗抄本封面是深蓝色布面的，和她的黑色水笔一样的颜色。她放在桌上时从来不折角，而是用一片从作业本上撕下来的纸条当书签——极为朴素的逻辑。陆允川在传作业时偶然看到了她的书签：一条从英语卷子上撕下来的空白边缘，上面写了一个P——大概是代表草稿页。那个P写得很快，笔尖从纸上滑过去时留下了淡淡的墨痕。

课间他去讲台交作业——顺路正好经过她的座位。他看见她的诗抄本今天打开着放在作业堆旁边，翻到了第一页——她今天抄的是李清照的《声声慢》前四句。他的目光停了一瞬，在"寻寻觅觅冷冷清清凄凄惨惨戚戚"那一行——他以前在课本上读过，读时感觉是古代的忧愁。这会儿他再看，觉得这行如果是要形容自己每天在教室里的状态——在人群中搜一个人的背影，找一个说了六字多一点的人再主动多一两句话——那就不止是古代的愁了。不过他说这一个念头按停，因为她和她抄的词不一定是同样的心境——他不能通过一首词的选目就去替她建立心境档案。这只是她语文老师布置的作业。

午饭时程砚聊起了各自的诗抄本。"我抄的是李黑的——仰天长啸出门去不是不是写错，是仰天大笑。"程砚自己笑了，"这个诗抄本的目录页我已经废了——第一页我抄错字用了修正液，然后修正液没干就往下抄，第二句粘了半句白。老师批下来给我写了三个字：可以重抄。""我觉得你的作风不太适合入选诗集选。""对，但我适合入选搞笑集——期末评语上老师可能会写：该生诗抄本参与度较高，但作品建议归类为物理伤害。"

晚上陆允川拿起笔在诗的后面加了一行批注："2011年9月，高二开学第二周。窗外法桐的新一茬叶子还没开始掉，窗内的日光灯管今天换成暖光灯了——旧的那根冷光灯闪了一整个星期，总务处终于派人来换了，换灯的师傅踩在课桌上拧灯管的时候全班都在底下扶着桌腿。她第一首诗抄李清照的《声声慢》。"写完停下笔——这份批注他没有交给老师检查，这是他自己的记录。他知道自己这样也许小题大做，但这种小题是其题的世界里唯一可以操作的记录方式。
