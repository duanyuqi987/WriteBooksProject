# 第018章 除夕

时间：2011年2月

地点：顾临风家中（石桥镇）

卷次：卷一：初见秋风

除夕那天从中午开始有人在放炮仗。小孩扔的摔炮——"啪"——扔在石板路上——白烟冒一小团——没了。还有人在点彩珠筒——"吱——"一声上天——"砰"——炸了——红色和绿色的火星落下来——灰掉在屋顶的瓦片上——簌簌。窗玻璃被远处的鞭炮声震得轻轻颤——窗框的旧木头间发出细微的咝咝响。

他妈在厨房里忙了一整天。围裙系着——蓝色的粗布围裙——系带在后腰打了个结。菜刀在砧板上——切萝卜丝——咔咔咔——刀口翻出白萝卜片。灶台里是土灶——烧柴——火映在厨房的墙上是橙色的——墙上的铝锅盖反着那团光。他爸在堂屋里擦桌子——那张四方八仙桌——桌面原木——用水——拧干毛巾再擦——擦完手指在桌面上抹了一把——桌面干净了。摆碗筷——三个人的碗——三双筷子——筷子是竹筷子——底部有点发黑——筷筒里的用了好多年。窗花是他姐从省城寄回来的——红色的薄纸——十二生肖——两只兔子的那对贴在堂屋窗户上——左边一只兔子——右边一只兔子——兔子的耳朵是椭圆——裱在红纸里——纸面上粘了金粉。

灶台上面蒸汽弥漫——白雾滚滚——从灶台窜到天花板——天花板上的椽子都淹在白气里——看不清木头纹路。猪肉炖粉条在铁锅里咕嘟咕嘟——汤面上鼓起一个个透明的气泡——破了——又鼓——粉条在褐汤里向外拉着——一颤颤的。醋溜白菜——醋倒了两次——第一次倒少了——尝了一筷子——又倒了一次——酸味从锅边飘过来——刺得鼻子里有点痒。红烧鱼——鱼是活的——他爸早上拎水桶去镇上菜市买的——用桶提回来——桶里的水洒了一路——鱼在桶里游到中午——脸盆换过两遍水。下午他妈杀鱼——菜刀刀背拍在鱼头上——鱼尾猛弹——水溅到她的围裙上——他躲开了——上楼去翻了两页书——又下来——鱼已经躺在盘子里——眼睛变白——鱼身划了几道斜刀——姜片码在刀口里——葱丝盖在鱼腹——酱油淋上去的时候——酱色顺着鱼鳞沟流——汇在盘底。旁边贴锅菜——饺子——整整齐齐排在盖帘上。盖帘是高粱秆子编的——用了好多年——秆子皮都磨滑了——泛黄——吃手的时候手指划过秆子——有一道一道细细的纹路。他妈包的饺子和过年饺子一样——每个褶掐得很紧——皮边是花瓣形——一个个立在盖帘上——有只饱满的——面皮太薄——隐约能看见里面的芹菜猪肉馅——葱绿色透着面皮。他在旁边帮忙按剂子——从面盆里拿出面团——搓成擀面杖粗细的长条——切成小段——在案板上压扁——压得不是太圆——但能用。他妈擀皮——擀面杖在那按好的剂子上面——两滚一个——皮子飞过来——他接住——放馅——开始包——包出来的饺子不太立——跟她的比——他的是塌的——扁——"你这个——馅太少了——"他妈说——他往里面加了点——包下一个——捏边——慢了——他妈已经做了五个——他做了两个。面扑在案板上像白雾——四周都沾了——手指也白。

天黑了。石板路灰了。石桥的拱影沉进了夜色。外面的爆竹声密起来了——开始是此起彼伏——然后连成一片——整个石桥镇噼里啪啦的——分不清谁家放谁家的——声音从东头传到西头——窗玻璃不停轻颤。天空不时亮一下——烟花在远处炸开——金一朵——红一朵——然后是金红的菊——在头顶轰——轰——又散了——小孩的尖叫声夹在炮声中间——隔着玻璃——"啊——"长声——然后又是炮声。

七点钟。他爸把电视机打开了。CCTV春晚——开场歌舞——十几个人穿着各种颜色的衣服在台上转圈——长裙子甩——扇面——镜头里台下在鼓掌——主持人走出来——"向全国各族人民——致以新春的祝福——"然后那个女主持接着说——"向港澳同胞、台湾同胞——"她妈正好从厨房进来——手里端着一盘菜——往电视看了两眼——"今年这舞台好像比去年大——"他爸说"是——去年那个舞台小——"

年夜饭端上桌了。七碟八碗的——摆满了八仙桌。中间的是红烧鱼——鱼身完整——葱丝和姜丝盖在上面——酱油色深——蒸汽从鱼肚里往外冒——鱼头朝着他爸——这是规矩——鱼头得对着家里最年长的人。旁边——一盘饺子——猪头肉——卤的——薄——酱油蘸料在小碟子里——炸花生米——金红色的——夹的时候筷子在碟里拨——小油粒沾在手指。酒杯里有半杯白酒——他爸的——酒气——粮食酒的透瓶香。他妈喝橘子水——汇源——橙色的铁罐——打开的时候用拉环——拉环弹了一下——汽在里面滋滋微响。他的面前是一杯茉莉花茶——茶叶沉在杯底——花骨朵——片叶舒展开——蒸汽卷着茉莉的香——微微含着苦。他夹了一筷子猪头肉——放进嘴里——嚼——卤汁咸——有一粒花椒——咬碎了——麻麻热热。

一家三口坐下来——桌子下炉火暖暖地烤着腿和脚。他爸举起酒杯——嘴张了张——没说出话。然后喝了一口——酒在喉咙里咕噜一声。他妈给他夹了一个饺子——筷子伸过来——搁在碗里——"芹菜馅的——你尝尝——这锅没破——"他夹起来——吹了吹——咬了一口——确实是没破的——面皮厚度正好——很有嚼劲——馅是芹菜猪肉——有汁水——从破口顺着嘴角流了一点——他用舌头抿回去了。然后是鸭腿——卤鸭——夹给他——"吃——平常在学校没啥吃的——"

饭吃到一半，手机震动了一下。是他爸那部老诺基亚——黑色——放在桌角——振的时候在白瓷杯旁边嗡嗡。他爸拿起来——喂——"嗯——新年好啊——李厂长——是啊——石桥——没出门——新年好——"那边说了好多话——他爸就"嗯嗯嗯"——然后挂了。没多久他妈也接了一个——"老婶子过年好——身子骨好——临风？挺好挺好——没感冒——期末考得还行——"她笑着说了好一阵子才挂了。

顾临风没有电话。他自己有一部手机——他爸淘汰给他的——诺基亚1100——黑灰色——旧——右边的壳上有一道很长的划痕——是掉在石头上蹭的。屏幕上光本来就不亮——OLED那种蓝绿光——在小字——"无SIM卡服务——"存不了几个号。他很早就知道这部手机打不出电话——卡还没办——他说下学期办——但他爸说——"再等等——放暑假再说。"所以——他到现在也没有卡。晚上手机放在床头充电——充电器插上发出嘎的一声——脆的电子声。手机屏幕亮——显示时间——和"无SIM"。

通讯录里存了三个号码——但没插卡也打不出去。存在"爸"——"妈"——"姐"——三个名字。没有第四个人。

吃完饭。他妈收拾碗筷——把鱼端走——放到橱柜里——明天继续吃——年年有余——年年剩。他爸坐到沙发上——人窝进去——春晚继续——赵本山带着徒弟们在小品里说了一句——"公鸡中的战斗鸡——"他爸笑了一声——那种喉咙里出来的——呼吸笑——手在大腿上拍了一下——然后又歇下去。他妈在厨房里喊——"声音小点——"他爸把电视遥控器按了按——音量往下两格。

顾临风坐在旁边的木椅子上。这把木椅是家传的——旧——扶手被磨得黑亮。他妈把瓜子盘端过来——放在他跟前——盘底磕在桌面上——瓜子是炒的——咸——壳上面有灰色的盐分。"吃吧——"他抓了一把。嗑了一个——门牙压——壳裂——舌尖把瓜子仁勾出来——嗯——空的——壳里什么都没有——扔进旁边的铁皮炉——空心壳烧了——一股青烟——小小。第二个——嗑开——放进嘴里——瓜子仁是苦的——苦瓜子的那种——可能是火大了——吐出来——放在掌心里——扔进垃圾桶。第三个——正常——咸香——嘴里的唾液遇到瓜子变得浓浓的。一连嗑了半盘子——磕得门牙有点打滑。喝了口茶。嘴里的盐和茶味混在一起。

快零点的时候鞭炮声突然大了起来。比七点那阵还猛——响声变了一个频率——不是劈啪——是轰——轰隆隆的——远处的近处的——全都盖过来了。成片——像暴雨砸铁皮——墙都在响——墙里的砖缝传着声音——地板底下也有微振。他爸出去放炮仗——把一整盘五千响的大地红从蛇皮袋里掏出来——红纸皮——挂着上面金丝的细导索——铺在门外——挂在门口的竹竿上——竹竿的尖端弯了——炮拖在石板地上——长长一条蜿蜒——像红色的蛇。他爸从兜里摸出打火机——然后用烟头去点引线——引线露出——捻子——火焰碰到药捻子——花火从炮竹上窜出去——白——"咝咝咝咝咝——"他爸往后退了几步——快。炮响了。红光一闪一闪——在门口的墙上——人影被炸得恍恍——门框上的红对联在光下抖着。空气中是火药的味道——那种硫和硝的——还有纸烧焦的气——青烟往上冒——一口口地在院子里散。

零点。电视里的倒计时——"五四三二一——"钟声响了——主持人——欢呼——笑声——"过年了——"镜头移到演播室外面——千万条烟花炸满了天空——北京——然后是西安——然后是西湖——全是烟花。

顾临风站在院子里。周围火药烟绕——在冷空气里飘——他被烟火光影打在身上——蓝的——红的一阵——绿的一阵。隔壁的院子里有人在放烟花——放的是那种一大桶——"砰——砰砰——"连着升空——炸出的星一点一点往下滑——金色的尾——落在房脊上——熄了。院子的墙头上落了一片炮仗屑——红碎纸——卷的——有些还在冒细烟。雪地里有个很深的脚印——他爸踩的——脚跟很明显——后掌比前掌深。

远处的鞭炮声汇成巨大的轰隆——像天上滚过一个实心的东西——分不出哪一声是远的、哪一声是近的。村子方向那边有人在大声唱——唱的是《难忘今宵》——调子跑了好几个音——唱到"共祝愿祖国好——"忽然被一个烟花炸断了——然后又接上——嗓音破了。

他把手伸进口袋里。口袋里什么都没有。手机在楼上充电——衣服口袋里只有一张超市小票——团着——还有一毛硬币——冰的。他把手拿出来——对着手心哈了口气——白色的——手上的寒气被白雾冲走了——很快又凉了。

他抬头看着烟花。烟花在暗空中展开——落下。他看着——嘴张开了——很小。没有发出声音。只是嘴动了一下——两个字。像在念一个名字。口型——苏晚晴——嘴唇碰在一起——再张开——气流——但周围鞭炮声太大——他自己也听不见。只有嘴唇动了。

烟花又炸了一个——金色——炸开之后化成很多小星星——小星星向下——在半空中就灭了。然后天空又恢复黑暗——等着下一轮——黑暗很轻——烟花的硝烟在天上静着——又被新一轮炸散。

他又站了一会儿——脸冷——手揣回口袋——转身进去了。

堂屋里——电视机还响着——主持人的声音小了些——他妈靠在椅子上——眼皮在打架——嗑的瓜子壳还堆在她的膝盖上——用手扫了扫——瓜子壳落在她围裙上——针线篮子搭在椅子旁边。他爸把炉子捅了捅——火钳从炉门伸进去——夹住煤饼——抖了两下——灰从炉底掉下去——火立刻旺了一些——橙色加深——暖气迎到脸上。炉子外面的铁壶里的水在低低地咕噜。堂屋的灯没全开——只有电视机和炉火的亮度。他在他们旁边站了会儿——然后把脸盆拿出水缸边——刷牙——挤牙膏——薄荷辣味——漱口——把水吐进院子——水洒在石板缝里——随即被冷风吹成冰。

上了二楼。窗外烟花还在放——比刚才稀了些——一声——两声——隔着远。他把手机从充电线上拔下来——拿在手心里——屏幕亮——时间——零点十二分——日期已经跳到正月初一。没有未接来电——没有短信——信号图标的那个位置——一直是个叉。

把手机放在桌面上。窗外又一蓬烟花——红——透过窗户——照在天花板的水印上——那片夏天漏雨的痕迹。水印被映红了——像脸的影子——眼下那个灰深的点——在红色光中反倒暗。然后变绿——变白——窗外恢复了黑暗。他躺在床上——枕头有点高——他把枕头下面的书拿出来——物理书——放旁边——平躺——手搭在胸口。外面的鞭炮声渐渐远去——偶尔蹦出一个响——不知道谁家零星的炮。

他闭上了眼睛。今晚——苏晚晴应该也在某个房间——跟她爸妈一起——看春晚——吃了饺子——刚才她在窗口站一站——看烟花——天冷——窗户缝里有个烟花可以看——也许她的窗也高——也许她的烟花也是金色。他翻了个身——脸侧着——想了一阵——然后不想了。睡着了。
