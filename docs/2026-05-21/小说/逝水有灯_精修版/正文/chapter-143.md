# 第143章 她说别追了之后

所属部卷：第三部：寒风里的花
时间：六月下旬
地点：图书馆东门台阶

## 诗歌插页 P143：原创仿宋小令

石阶边，晚声轻，
半窗灯影落无凭。
问旧年、几回执念，几回分明。
人去不须追远路，
水流仍向晓光生。

期末最后一周，陆允川回了一趟老家。不是因为夏知遥的邮件——是因为他妈打电话来说家里的狗生病了，让他回去看看。他妈打电话从来不废话，"豆豆病了，你回来一趟"说完就挂了。狗是老狗了，养了十年——从他初二开始养的。叫豆豆，是一只黄色的串串，腿短，耳朵一只竖一只耷拉。他记得豆豆刚来的时候只有巴掌大，现在牙齿掉了两颗，眼角有了白毛。

陆允川坐大巴回去的路上，把诗抄本摊在膝盖上翻看。大巴是老式的金龙客车，空调出风口坏了一个，冷气不均匀，靠窗的位置一会儿冷一会儿热。他把诗抄本翻到目录页——目录是他自己写的，用铅笔，每个标题旁边标了页码和写作日期。从高考前到现在，一共写了五十多首。

第35页：《旧车站》。高考前一周写的。"废弃的站台不等人/只等下一次雨。"他当时觉得这句很好。现在看——太文了。"废弃的站台"是文艺的默认选择。实际上老家的长途汽车站一直在用，从来没有废弃过。他选了"车站"这个意象不是因为看见了车站——是因为需要一个停靠的意象。先有寓意，后找事物。顺序反了。

回到家，豆豆趴在院子里，尾巴摇了摇，力气不如以前，但还是认得他。他妈说兽医院开了药，再观察一周。他把豆豆抱起来，狗毛在他衣服上蹭了一层浅黄色的绒。

晚上他在高中同桌的QQ群里看到消息——是林曼发的。她说路过老学校看到施工围挡已经围起来了。有人在群里发了一张照片：教室走廊的水磨石地面，被磨了三年还是老样子，灰白底子嵌着细小的彩色石粒。门上方的绿色标牌"高三（4）班"——那个"4"的右下角被磕掉了一小块漆。黑板旁边贴着高三时程砚画的最后一张黑板报——画的是高考加油，用粉笔画的帆船，船帆上写满了每个人的名字。程砚的字又大又歪，周长的字小得像蚂蚁，夏知遥的名字被林曼写在了桅杆位置。

他保存了那张照片。然后给程砚发了消息："学校的楼要拆了。"

程砚几乎是秒回——他现在上班摸鱼的本事更精进了。"知道。有人发了。你看到黑板报了吗？"

"看到了。船帆上的名字有我的。在你的毛笔字下面。你的字还是全班最大。"

"我画的那张。颜料不够红——那罐红粉笔被隔壁班借走了没还——所以我用了橘色代替。结果帆的颜色偏橘。老师看了说'红色的帆不够红，不吉利'。"

"红色——你画不出红色。你总是画成橘色的。"

"没办法，从小色弱。"

"你自己不知道？"

"去年体检才知道。测色盲的图册翻到第二页医生就让我停了。说你这个不是色盲是色弱——红绿色弱。我才知道原来我一直看到的'红色'跟别人不一样。以前以为橘色就是红色。就像有人以为绿豆糕是黄豆糕。"

陆允川在黑暗的房间里对着手机笑了一声。豆豆趴在床脚，听到他笑抬起头看了一眼，又趴回去了。程砚是色弱——画了三年黑板报——从来不知道自己在用橘色代替红色。全班也没人告诉他。大概是因为那个橘色也好看——偏暖的橘，在黑板底色上反而更亮。

"那那些被你用橘色写的名字——也算红色吗？"

"算。因为在我的视觉里那一直是红色。只是别人的视觉里不是。"程砚打了一串字又删掉。陆允川能看到那边的"对方正在输入……"闪了好几次。最后只发了两个字："算了。"

"别算了。我觉得你那句话有意思——在别人的视觉里不是。但对你来说是。那到底算不算？"

"要看是谁判断。如果我判断——是。如果让全班投票——可能橘色就是橘色。但这种东西不是投票决定的。"

陆允川把这番对话记在了诗抄本的空白页上。他用铅笔写得很小，挤在第35页《旧车站》的旁边。写完后给第35页画了一道箭头，箭头旁边新写了几个字："后来知道那个车站一直在用。是当时我需要一个停下来的意象，而不是车站真的废弃了。"

然后在同页加了一行，用蓝笔——不是铅笔——以示区分："程砚说在他的视觉里橘色就是红色。我想：什么是客观的？什么是主观的？如果两个人看到的颜色不同但用了同一套名字——是不是也在一套共同的参照系里？那些被橘色粉笔写的名字，它们自己是不知道的。但它们被写下来的那一刻，有一个人的手在握着那支粉笔。那个人的手是客观存在的。"

写完后他翻到第47页。传单还在——边角被反复摩挲得起了毛。"允，信也。"铅笔字淡得快看不见了，像从纸上渗了进去。他用铅笔重新描了一遍。描的时候力道很轻，怕把纸戳破。

回学校的大巴上，晨光从车窗斜进来，他打开了夏知遥的最后一封回信。她在邮件最后写了一行字："我看了你的回信，里面说的那个人——苏晚枝——她替你找到了能用字表达的东西。这个比我给你的多。所以我不用觉得高中欠你什么了。你没有欠我。我也没有欠你。那个废弃站台的诗可以换题目了。"

"旧车站——可以改了。"陆允川对着窗外后退的行道树默念。行道树是杨树，树干上刷了半截白石灰——跟他们县公路两边的树一模一样的刷法。

回到学校那天，他行李都没放回宿舍，直接拖着行李箱去了301教室。门没锁——诗社的教室永远不锁。树墙上多了一片新叶子，是苏晚枝贴的。叶子的形状剪得比平时的叶子宽一点，像一个横着的感叹号。叶子上的字是：

"今晚灯只有一盏。给每一个不再追了的人。——诗社，期末周"

他在那片叶子下面站了片刻。行李箱的轮子在地板上发出轻微的咕噜声——然后停了。他用铅笔在叶子背面补了两个字：谢谢。写得很小，刚好嵌在那片叶子背面的叶脉纹路里。

苏晚枝不在教室。但她留下的这行字和他需要的恰好重叠。他不知道她是提前知道他会看到这片叶子——还是只是在她自己的逻辑里推导出了"期末周需要一个句号"。两种可能他都接受。
