# 第068章 家长会以后



时间：2009年一月底，期末成绩公布前夜



地点：男生宿舍、女生宿舍和走廊电话旁



卷次：第三卷：雪落在两栋楼之间



2009年一月底，期末成绩公布前夜，男生宿舍、女生宿舍和走廊电话旁，魏然写到一半又揉掉的检讨纸先把这一天按进了十班的日常里。期末前夜，魏然因为顶撞老师被要求写说明，宿舍里没有人再笑他。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

他嘴硬说不怕家长知道，真正拿起电话时却沉默很久。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看魏然写到一半又揉掉的检讨纸，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

魏然也在这一天被推到前面。魏然的特点是有棱角，偶尔顶嘴，心里其实怕拖累班级。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

"我没说错。"魏然说。

"没说错，也要看怎么说。"陶建国说。

"你要不要先别打电话？"段璟说。

"迟早得打。"魏然说。

"那就把话说清楚，别赌气。"常俊凡说。

家长会后的教室总有一种奇怪的安静，像每张课桌都刚被成年人坐过，留下不属于学生的重量。

桌面上有人忘了带走矿泉水瓶，有人把成绩分析表压在书下，像压住一封不想看的信。

余小满把母亲签过字的表格折了两次，折痕正好压在总分那一栏。

段璟回到座位时，发现桌洞里的草稿纸被父亲整理过，乱糟糟的一叠忽然变得很规矩，他反而不适应。

常俊凡的家长没有在教室多停，问完老师问题就走。她站在后门边看了几秒，脸上没有明显表情。

宋乔说我妈坐我座位上比我还像学生，大家笑了一下，很快又各自安静。

宿舍电话旁排队时，许多人把同一句话改了好几遍：考得还行、还可以、下次会更好。

这类话听起来像模板，可每个人说出口时，喉咙里都有自己的卡顿。

家长会以后没有悬在空处。家长会后桌面上留下的矿泉水瓶，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，家长会以后带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

陶建国进门时不一定立刻说话，鞋底在水泥地上一停，家长会以后带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在魏然快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。家长会以后越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。魏然写到一半又揉掉的检讨纸让他想说话，又逼他先低头看题。

魏然被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，家长会以后留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。魏然路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为家长会以后翻身，谁还在被窝里背单词，旁边人都听得见。

家长们从城关镇的小街散开，棉鞋踩过路边薄冰，把学校里的话带回各自家里。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理家长会以后里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：魏然写到一半又揉掉的检讨纸、一只冻红的手、一句老师点名、一块还没扫干净的雪。

走廊尽头的IC卡电话机在家长会这天晚上从来没闲过。话筒被前面的人握得温热，下一个接过来时还能感觉到上一个人的手汗。

魏然排在第三个。他前面是邓桥，邓桥正在跟家里人汇报化学成绩，声音压得很低，但封丘话的尾调还是从话筒边漏了出来："化学八十七，老师说了，这个分数能上重点线。"邓桥顿了一下，那边大概是在问他语文。他改口说："语文也还行。"

段璟站在魏然后面，没有出声。走廊的声控灯灭了，被邓桥的声音重新点起来，灯光在头顶嗡嗡地响，带着一层灰黄色的频闪。

轮到魏然时，他把电话卡插进去，手指在数字键上停了好几秒。那个号码他每个周末都打，但在今晚，那十一位数字忽然变得很重。他按完最后一个数字，听见那边传来"嘟——嘟——"的声音，每一下都让他的喉结往上顶一次。

"喂？妈，是我。"魏然说，声音比平时低了一档。

"家长会开完了。陶老师跟你说了？"他听着那边的话，眼睛盯着电话机上的显示屏，屏幕上跳动的计费数字一秒一秒地减。

"我知道，"魏然说，"检讨我写了一半了。不是因为不服——是当时说话没过脑子。"

话筒里传来一阵很长的说话声，魏然没有打断。走廊里等电话的人自动往后退了两步，给他多空出一点地方。段璟看见魏然握着话筒的手在发抖，不是冷的那种抖，是用力的那种——他在用握话筒的力气压着自己的声音。

"行。我知道了。你跟我爸说别担心。"魏然挂了电话，把卡拔出来，没有看任何人的眼睛，直接往宿舍方向走。

段璟跟上去，在楼梯口叫住他。

"你还差多少字？"段璟说。

"差三百。"魏然没回头，"但我不知道怎么往下写。写'我错了'三个字就够了吧？为什么要写八百字？"

"因为你只写三个字，陶老师会觉得你在应付。"常俊凡的声音从后面传过来。她刚从女生宿舍那边过来，手里拿着一个暖水袋，外面裹着一条旧毛巾。

"常俊凡你怎么来了？这是男生宿舍这边。"魏然说。

"走廊不分男女。"常俊凡把暖水袋换到另一只手里，"魏然，你检讨里写了什么？"

"写了'我不该顶撞刘老师'。"

"然后呢？"

"然后没了。"

常俊凡没有立刻说话。她把暖水袋放在走廊窗台上，从校服口袋里掏出一张叠好的纸。那是她晚上帮魏然列的一个提纲，上面用铅笔标了几行小字：第一段写事情经过，不辩解；第二段写为什么当时没忍住；第三段写补救办法。"你不用照抄，"她说，"但至少要让人看出来你想过。"

魏然接过那张纸，低头看了两眼。走廊的灯光不够亮，他往窗边挪了一步，借着从宿舍门缝漏出来的光又看了一遍。

"为什么帮我？"魏然说。

"因为你把检讨纸揉了五次，"常俊凡说，"说明你在意。在意的人值得帮。"

魏然把提纲折好放进裤兜里，嘴唇动了一下，像是想说谢谢，最终说出来的却是："那你帮我看看我写的那一半。"

段璟在旁边听着，忽然觉得常俊凡今晚做了一件比她考年级十七更累的事。她帮魏然不是因为她跟魏然熟——他们平时几乎不讲话。她帮魏然只是因为下午开家长会时，她看见魏然的妈妈坐在最后一排，从头到尾没有抬头看黑板。她一直低着头看自己的手，手指在膝盖上交握，骨节捏得发白。

男生宿舍熄灯前的半小时是整栋楼最吵的时候。水房里有脸盆碰撞的声音，走廊里有人追着还拖鞋，上铺的人在翻身子时压得铁床架嘎吱响。但305寝室今晚比平时安静。

魏然坐在下铺，把检讨纸摊在膝盖上写。段璟坐在他对面，拿着化学卷子改错题。付小川从上铺探出半个脑袋，嘴里叼着一根棒棒糖，含含糊糊地说："魏然，你写完了我帮你数字数。"

"不用你数。"魏然说。

"我数数特别准。"

"你上次数自己物理分数，把五十八数成了八十五。"

"那次是光线不好。"付小川把棒棒糖换到另一边嘴角，"但今天不一样。今天是我为数不多的、真心想帮忙的一天。"

魏然抬头看了他一眼，付小川笑了一下，没有把玩笑继续开下去。他知道今晚不是开玩笑的时间。

周明澈从被窝里探出头来，手里拿着一个诺基亚2610。屏幕蓝光照在他脸上，按键音在安静的寝室里格外清楚。他在给家里发短信。

"你发的是什么？"付小川问。

"跟我妈说我期末还行。"

"你妈怎么回？"

"她说明天让你爸给你带一双厚袜子，别冻了脚。"周明澈把手机翻过来放在枕头上，屏幕朝下，光灭了。

305寝室里安静了几秒。然后付小川说："我妈今天也来了。她开完会就走了，没跟我说话。"

"为什么？"周明澈说。

"因为她在教室门口看见我，只说了四个字：'物理太差'。"付小川把棒棒糖咬碎了，嘎嘣嘎嘣地嚼，"然后就去赶末班车了。"

段璟把化学卷子翻了一面，听见付小川说这句话时声音里有一种他平时藏得很好的东西。不是难过，不是愤怒，是一种"我明知道你说得对但我不想听"的钝痛。

女生宿舍306那边也有自己的安静。宋乔坐在常俊凡的床边，把英语笔记摊在膝盖上，但她没有在背单词。她低头看着自己袜子上的一个破洞，大脚趾从里面露出来。

"我妈今天在教室坐了四十分钟，出来以后跟我说了两件事。"宋乔说。

"哪两件？"常俊凡坐在她旁边，正在整理明天要交的化学作业。

"第一件，我的英语作文走题了。第二件，我的袜子破了，她明天让跑县城的长途大巴司机帮我带一双新的。"

"那是不是好消息？"

"我不知道，"宋乔用大脚趾在袜洞里动了一下，"她说袜子的时候语气比说作文温柔。但我不知道那是关心还是不想跟我谈分数。"

常俊凡把化学作业放在一边，没有再说关于分数的事。她只是说："明天拿到新袜子，你先把这只破的扔了。穿破袜子久了，脚会磨泡。"

宋乔没说话，但她把脑袋靠在了常俊凡的枕头上。枕头上有一股很淡的洗衣粉味道——雕牌，封丘一中女生宿舍的标准配置。

走廊里的电话终于空下来了。最后一个打电话的人是余小满。她把电话卡插进去时，卡槽发出咔的一声，然后她按下了家里的号码。电话接通后，她说的话不多，大部分时候在听。挂掉电话以后，她站在走廊里没有马上走。头顶的声控灯灭了，她一个人站在黑暗里，吸了一下鼻子，然后回了寝室。

段璟从水房出来时正好看见她往回走的背影。他没有叫她。他知道有些话只适合跟电话那端的人说，不适合被第三个人听见。

熄灯后，走廊里只剩应急灯那一小团绿色的光。305寝室的上铺传来翻身的声音。魏然还在被窝里用手电筒照着检讨纸，笔尖在纸上走得很慢。他写到"我反思"三个字时停了一下，在手电筒的光柱里把那三个字划掉，改成了"我想过以后怎么做"。

段璟躺在自己的铺位上，没有睡着。他想起下午父亲坐在他座位上，把他桌洞里那些卷子按科目叠好，物理在上面，化学在下面，中间夹了一张他上次改错的便签纸。父亲没有动那张便签纸，只是把它挪到了化学卷子的第一页。

窗外的雪不知道什么时候又开始下了。很细，几乎听不见声音，但在路灯的光柱里能看见它们斜着往下落。段璟裹紧被子，在心里把今天的事过了一遍——魏然的检讨、付小川的棒棒糖、常俊凡的提纲——然后他意识到，家长会带来的东西，不全是压力。也有些东西，是压力压过之后才会露出来的。

第二天早读前，食堂的萝卜包子蒸得比平时多了一屉。因为家长会开完，不少家长走之前给住校的孩子留了早饭钱。食堂窗口的阿姨一边盛粥一边对每个人说"今天多喝一碗"，像是在替那些已经坐上长途大巴的家长把话传到。

魏然走进食堂时眼睛下面有两团青灰色，检讨显然写到很晚。他把检讨纸折好放在书包最外层，准备早自习交到陶建国办公室。付小川看见他就招手，手里举着一个萝卜包子："魏然，给你留的。萝卜馅的，我尝过了，不咸。"

魏然接过包子，没说话，咬了一口。他嚼了两下，忽然说："付小川，你的物理不是五十八，是六十一。"

"什么？"

"我昨晚帮你查的成绩条。六十一，及格了。"

付小川愣了一秒，然后整个人从凳子上弹起来："我就说光线不好！——不对，是六十一！及格了！"他高兴得差点把粥碗打翻，旁边的周明澈眼疾手快地把碗扶住，粥洒出来一小半，在桌上摊成一个不规则的圆。

常俊凡坐在女生那一桌，面前摆着一碗小米粥和一个白水煮蛋。她把蛋壳剥得很慢，手指被蛋壳硌出红印。宋乔坐在她对面，新袜子已经换上了——一双藏青色的棉袜，她妈托长途大巴司机天不亮就送到了学校门卫室。

"袜子合适吗？"常俊凡问。

"大了一点，但暖和。"宋乔把脚伸出来看了看，"我妈在里面塞了一张纸条。"

"写的什么？"

"'作文注意审题，袜子别舍不得换。'"宋乔把纸条念完，嘴角往上翘了一下，又很快收回去，"她用了两张纸写我爸的腰椎间盘和奶奶的血压，写我的只有这一句。"

"一句也是写给你的。"常俊凡把剥好的鸡蛋掰成两半，蛋黄完整的半给了宋乔。

段璟端着自己的粥坐在隔壁桌，和常俊凡隔了一把椅子的距离。他听见了她和宋乔的对话，没有插嘴，但他把自己碗里的萝卜包子拨出来一个，放在公共的搪瓷盘里，往女生那桌推了推。

常俊凡看见了那个包子，也看见了段璟收回去的手。她没有说谢谢，只是把包子夹到自己碗里，对宋乔说："段璟给的。你一半我一半。"

走廊里的IC卡电话机在早读前又响了一次。是余小满的妈妈从家里打过来的——她到学校时忘了把棉背心留给女儿。余小满接到电话时眼眶红了一下，但她对着话筒说："没事妈，我不冷。教室里暖气挺热的。"

挂掉电话后她抱着棉背心往回走，走到教室门口时已经恢复了平时的样子。但路过常俊凡桌子时，她停下来说了句："我妈让我跟你说谢谢。上次你借我的化学笔记，我这次方程式题全对了。"

常俊凡抬头看了她一眼，点了下头。然后又在自己的化学笔记本上翻到新的一页，用铅笔记下一行字：余小满方程式全对。她在旁边又加了一个圆，和给段璟那张便签纸上画的圆一模一样——小小的，没有意义，只有她自己知道为什么画。

家长会以后之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

魏然没有因此变成特别醒目的人，可在段璟眼里，魏然从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，段璟看见有棱角的人也有怕，常俊凡则提醒大家别把同学当笑料。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
