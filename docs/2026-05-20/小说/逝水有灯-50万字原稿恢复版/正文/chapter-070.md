# 第070章 墙角的试卷多年后

所属部卷：第二部：六月以后
时间：高考后第二天
地点：陆家小院的东屋

## 诗歌插页 P070：原创七言小诗

窗过长廊日影深，
一行旧字压衣襟。
少年未解离分意，
先把沉默写到今。

多年后陆允川重写墙角的试卷，先删掉最重的怨气，再删掉最亮的自我感动。页面空下来以后，墙角那摞卷起边的试卷反而更清楚。

陆允川把课本和卷子堆在墙角，三年变成一摞纸，他看了很久。这件事没有随着时间消失，只是从伤口变成了可以被描述的旧场景。

他重新补回东屋里的东西：旧书包、草稿纸、墙角纸箱、用秃的笔芯。这些东西不负责抒情，却负责让记忆落地。

纸箱被推到墙角时发出沉闷一声。陆允川蹲下来，把最上面那张卷子压平，忽然觉得青春也许就是这样：费很大力气，最后被叠成一摞。

他也给夏知遥补回生活。夏知遥没有出现在这个房间里，却藏在许多纸页的边角：某次排名旁边的名字，某页草稿背面的一句短诗，某张班级通知上被他无意识圈出的座位号。她不是他青春的答案，也不是他小说里的奖品，她只是一个也在往前走的人。

程砚和叶澄的声音也被留下。一个把玩笑递过来，一个把边界说清楚。没有他们，陆允川很容易把孤独误认为深情。

写到这里，他把日记本合上，去倒了一杯水。母亲切在碗里的西瓜的味道从记忆里浮出来，提醒他那一年并不只有疼。

所谓温柔现实，就是不替过去粉饰，也不把过去判死刑。它承认陆允川曾经笨拙，也承认夏知遥有权不接住他的笨拙。

陆允川后来愿意留下这一笔，是因为它不漂亮。旧书包不漂亮，草稿纸也不漂亮，可它们让人相信，这段青春不是从空中落下来的。

他没有再给夏知遥安排额外台词。她已经在东屋里完成了自己的动作，剩下的沉默，应当由陆允川自己消化。

程砚把话说得粗，叶澄把话说得稳，两种声音都把他往现实里拉。陆允川当时不领情，很多年后却知道自己被救过几次。

那天的母亲切在碗里的西瓜没有特别好吃，却因为太具体，抵住了那些过大的形容词。人有时候正是靠这些具体的味道，才没有被回忆吞没。

院子外有人骑摩托经过，远处小孩在喊卖冰棍。这些声音把墙角的试卷从一个人的心事里拉出来，变成许多人共同挤过的一天。陆允川不再只听见自己胸口的动静，也听见椅子、脚步、塑料袋和远处的喊声。

夏知遥只剩在试卷边角和排名表里的一个名字。这个动作很小，却让她从陆允川的想象里退回到真实的人：她会忙，会累，会被家里和未来推着走，也会在必要的时候保持沉默。

程砚在电话那头催他出去打球，说话还是老样子，不好听，却有效。陆允川皱了皱眉，还是照着他说的做了一点，因为朋友有时不是负责安慰，而是负责把他从自我感动里拽出来。

叶澄在群里提醒大家留好准考证。她没有把话说得太满，只把边界摆到桌面上。陆允川那时不一定立刻懂，却会在后来某个晚上想起这句话，知道自己当年被提醒过。

母亲端来西瓜，看见满床纸没有催他扔。家里的声音一进来，青春就不再只是校园里的风。它变成饭点、车费、睡眠、行李和一句不太会表达的关心。

他低头看着墙角试卷，忽然意识到它不是命运给他的标记，只是那天确实存在的东西。正因为确实存在，它比任何夸张的形容都更能留下来。

旧纸、晒热的被单和西瓜汁混在一起，让记忆有了可以落脚的地方。陆允川后来写到这里，没有再急着解释自己多难过，而是先把气味、光线和手上的动作写清楚。

他把纸箱收好，又拿出来检查了一遍。这个来回不像英雄，也不像受害者，只像一个年轻人不知道怎样安放自己，于是先安放手边的东西。

多年后他最想删掉的，是那些把自己写得过于正确的句子。正确有时比错误更可怕，因为它会把别人挤成背景。

他把墙角的试卷重新写了一遍，决定不给任何人判刑。疼痛是真的，笨拙是真的，错过也是真的；但这些真实不需要互相证明。

这一笔补进去以后，章节没有变得更响，反而更稳。墙角的试卷不再只是一个标题，而是有墙角试卷、有纸箱、有人说错话也有人及时停住的真实一天。

陆允川把一张张卷子从书包里抽出来，先按科目分，后来又按时间分，最后全乱了。三年不是几摞纸能分清的，它们混在一起，像他没学会整理的心。

母亲问这些书卖不卖废纸，他说再等等。她没有追问，只把西瓜碗放到窗台上，说热了就吃。这个不追问，比任何安慰都让他松了一口气。

程砚第二次打电话来，背景里有篮球砸地的声音。他说你再不出来，球场都要毕业了。陆允川终于笑了一下，把最上面那张卷子压进纸箱。

东屋里的光线一点点变暗，旧书包和西瓜碗都还在原处。陆允川把纸箱收好，动作慢得像怕惊动什么。可真正需要被惊动的不是别人，是他自己。

他后来重写这一章时，特意把墙角试卷留在中间。不是为了让它变成象征，而是因为人只有摸到具体东西，才知道自己曾经怎样站在具体生活里。

多年后的陆允川终于承认，自己当年并不比别人更懂爱。他只是更擅长记住墙角试卷，也更擅长把沉默误解成命运。

多年后他才懂，墙角那摞试卷不是废纸，也不是纪念品，而是一段用力过猛的时间。它们被放在那里，慢慢落灰，替他完成了最朴素的告别。

本章最后，他没有写“如果当初”。他只写下：那时有院子里的鸡叫、风吹窗纸和远处摩托车经过的声音，有一个人慢慢学会把别人还给别人。
