# 第107章 封校的周末

所属部卷：第二部：六月以后
时间：大一秋天
地点：大学南门和围墙边

## 诗歌插页 P107：原创仿唐绝句

南门锁后晚风斜，
纸样新成递暮霞。
不是相逢多曲折，
一条窄路到灯花。

操场后面的小路很窄，一边是围墙，一边是低矮灌木。墙外打印店老板骑着电动车等他们，车筐里放着包好的社刊样张。

“南湖诗社是吧？”老板把纸包从栏杆缝里递进来，“下次早点说，封门麻烦。”

苏晚枝连声道谢，接过纸包时没拿稳，最上面几张滑出来。陆允川蹲下去捡，看见封面标题印偏了一点，右下角还有一小块墨迹。

“要重印吗？”他问。

苏晚枝翻了几页，“不用，样张就是看问题。正式版再改。”

他们沿着围墙往回走。纸包不轻，陆允川说他来拿。苏晚枝没有客气，把一半分给他，“那你拿这摞，我拿封面。”

这种分法很普通，却让陆允川心里亮了一下。不是被照顾，也不是被拒绝，只是共同完成一件小事。他问她为什么进诗社，苏晚枝想了想，说高中时作文总被老师说太平，后来反而想学着把话说准一点。

“你呢？”她问。

陆允川说：“以前写得很乱。”

“乱也可以改。”

这句话没有安慰的姿态，只像一个编辑对稿子的判断。陆允川却记住了。到活动室门口时，叶澄已经等着，接过纸包就翻问题，第一句是：“封面颜色偏了。”

苏晚枝叹气，“我就知道。”

陆允川站在旁边，看她把袖口往上卷，拿红笔在样张上圈错字。那一刻他喜欢的不是浪漫，而是她处理事情时的认真。只是他当时分不清这两者之间的距离。
