# 第115章 第一次一起吃饭多年后

所属部卷：第二部：六月以后
时间：多年后
地点：食堂二楼靠窗桌

## 诗歌插页 P115：原创仿唐绝句

旧桌临窗雨色微，
当年餐盘已无归。
记人小事须知止，
莫把温柔认作围。

多年后学校食堂翻修，二楼靠窗的位置改成了自选餐区。陆允川站在入口处刷临时卡，听见机器“滴”的一声，忽然想起自己当年的饭卡余额：二十三块六。

他端着餐盘走到窗边。现在的窗口已经不问要不要葱，调料都放在一旁，自己取。他看见一小盆葱花，绿得新鲜，旁边有夹子。

陆允川没有夹，也没有刻意避开。他只是点了一份面，端到靠窗的位置。窗外不是当年的雨，桌子也换成了浅色木纹，餐盘边缘干干净净。

旧稿里，他曾经写苏晚枝“不爱吃葱”写了很多次，仿佛记得这件事就能证明自己深情。后来他删掉大半，只留下她把葱花拨到碗边，留下她坚持各自付账。

这两个动作已经足够。一个说明她接受了他的顺手帮忙，一个说明她保留了自己的边界。再多写，反而会把她推回他想象里的位置。

吃完饭，他把餐盘送到回收处。旁边一个新生低头找饭卡，急得额头冒汗。陆允川把自己刚才领的临时卡递过去，让他先刷。新生连声道谢。

他笑了笑。细心如果不要求回报，就只是细心。到了这一步，它终于轻下来。
