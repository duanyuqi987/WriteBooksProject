# 第171章 最重的话之前

所属部卷：第四部：失去的美丽
时间：大二冬天深夜
地点：男生宿舍书桌前

## 诗歌插页 P171：原创现代诗

我把删除线放回夜色，
不让它替谁证明深情。
风从未说完的话里穿过，
有人低头，学着不再追问。
后来水声慢下来，
灯还亮着，人已经能往前行。

陆允川翻开日记本的时候，祁野刚打完一局游戏。

时间是晚上十一点四十。程砚还在实验室，宿舍里只有两个人。台灯只照亮半张桌子，窗外有人打水回来，拖鞋声在走廊里一下一下远去。

"你在写什么？"祁野从床铺上探下头来。

"日记。"

"又是三千字的那种？"

"不知道。"

"你知道。"祁野把椅子转过来，"你现在这个表情，跟还书那晚一模一样。不要骗我。"

陆允川没有回答。他把笔帽拔开，在日记本最顶空白处写下日期——日期写得很重，笔尖几乎戳穿纸面。然后停住了。

他今天写到的是苏晚枝。但不是平时那种"想不通"的纠结——今晚他心里有恨。

不是想不通——是愤怒。愤怒她为什么不接受他。愤怒她跟别人站在一起时那么自然。愤怒自己为了她付出了那么多却被一句"我会很累"拒绝。愤怒台阶那夜他被冻了将近两个小时，她只说了一句"你别总在楼下等我"。愤怒他送给她的七本书和那么长的赠言，她只回了一个铅笔写的"好"。

他写：

"你今天跟别人站在一起的时候，有没有想过我站在不远处看到了你——"

写到这里笔停了。他把笔按在纸上，墨在笔尖下洇开一个小黑点。然后他划掉了整行。不是轻轻画一条线——是用笔来回划了很多次，黑墨压得很深，纸背凸起一条硬棱。

下一页他又写：

"你不值得我——"

又划掉了。这一笔更重，几乎划破了纸。

桌角的泡面桶还没扔。是祁野今晚吃完后搁在那里的，汤面上浮着一层红油，已经凝固成橙色的膜。苍蝇撞过灯罩后嗡嗡两声，又安静了。

"陆允川。"祁野忽然喊他。

"嗯。"

"你刚才写了几行，划了几行？"

"——你怎么知道我在划。"

"因为你的笔在纸上走路的方式不对。写字是流线，划掉是来回搓。搓得那么重，整个宿舍都听得见。"

陆允川低头看着自己压在纸上的手。掌心下的墨迹还没干，蹭出一小片模糊。

"我只是在写一些不能发出去的话。"他说。

"写给谁的？"

"苏晚枝。"

"什么话？"

"——很难听的。"

"具体是什么？"

陆允川沉默了。祁野把游戏关掉了——陆允川认识祁野以来，除了系统强制维护外，这是第二次有人关掉游戏，而且是为他。第一次是大一他妈复查结果不确定的那天。

"你不用跟我说。"祁野站起来走到他旁边，屁股靠在桌沿上，低头可以看到他划掉的那些行,"但你用那么大的力气——你想要那个本子替你流血吗？"

"我就是——憋不住。"

"憋不住可以跟我说。先说出来，再决定要不要记在日记里。先过嘴，再过笔——记得吗？这些都是你自己过去总结的。"

"我没力气过嘴。"

"那就过我这张嘴。"祁野拿过日记本，把那一页翻了一下——密密麻麻的删除线，黑墨把下面的字完全盖住了，像一块一块的黑色伤口。

他合上本子。"你在这里面待了多久？"

"不知道。大概二十分钟。"

"二十分钟写了这么多，没有一个字是给苏晚枝发出去的？"

"没有。"

"这些重话不是给苏晚枝写的——是给你自己写的。你在用写字来惩罚自己。"

陆允川没有否认。他觉得祁野说得对。

祁野把日记本放在桌角——泡面桶的旁边。泡面桶里的红油膜轻轻晃动了一下，什么都没影响。

"假设——假设苏晚枝今晚突然出现在我们宿舍门口，你会把这些读给她听吗？"

"不会。"

"为什么？"

"因为太难听了。她听到会很难过。"

"所以你自己知道这些话是错的——但你还是写了。"

"对。"

"为什么？"

陆允川低头想了想。"因为不说出来我会更难受。写出来的瞬间会松一口气。但写完——看到那句——我就知道我错了。所以我划掉。"

祁野安静了几秒。走廊里又有人打水回来了，拖鞋声远去了。

"所以这个过程不是恨她——是你把自己的疼捏成针，扎了一排，然后你又觉得不对，把这些针一根一根拔出来。拔出来的痕迹就是那些删除线。"

陆允川靠在椅背上，看着天花板。上面贴着的那张周杰伦海报已经卷边了，垂下来一角。

"疼是真的。但疼的方向是错的。"

"对。"祁野说，"你感觉你在气她。其实你是在气自己。气自己为什么得不到她的回应。气自己为什么是个'好人但不被选的人'。那不是恨——是不甘。"

他把日记本推回到陆允川手边——然后拉了一张凳子坐在他旁边，没有走。

"现在想听专业意见吗。"

"谁的？"

"我的。"

"——你什么时候有专业意见了。"

"我泡面都戒了。脑子很清醒。"祁野指着日记本，"你现在要做的不是继续写，也不是继续划。是把本子合上。睡一觉。明天等心情平了——再打开，看一遍你写了什么。到时候你就知道自己真正想表达的是什么。不是恨——是别的。"

"是什么？"

"等你明天睡醒再告诉我。"

陆允川看着日记本。台灯照着纸页，那些删除线在灯光下显得格外立体——像纸面上埋着一截一截不长的断轨。他用手掌压住那一页，掌心下的墨迹蹭出一小片模糊的污迹。

他松开手。然后合上本子。

"好。"

"晚安。"

祁野爬上床，很快发出均匀的呼吸声。陆允川关了台灯。黑暗里他还能感觉到掌心里那片墨迹——湿的，黏的，凉的。
