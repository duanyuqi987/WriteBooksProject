# 第148章 食堂争执之后

所属部卷：第三部：寒风里的花
时间：六月末
地点：校外小饭馆

## 诗歌插页 P148：原创仿宋小令

残油边，晚声轻，
半窗灯影落无凭。
问旧年、几回执念，几回分明。
人去不须追远路，
水流仍向晓光生。

三周以后。暑假开始，大部分学生回了家。校园里安静得只剩下蝉鸣和操场洒水器的喷水声。

苏晚枝留在学校——建筑系暑假有短学期，两周的测绘实习。她每天早上背着三脚架和测量仪去学校对面那座老宅测绘，中午回来把数据导进CAD画图。下午两点到五点，建筑系馆的机房里开着空调，她戴耳机画图，画累了就趴在键盘上睡十五分钟，口水流在空格键上。

诗社的事按照新方案执行了。周三晚上的活动由小孟主持，苏晚枝只参加了月初选题会。社刊改成了月刊。树墙的维护交给了小孟和一位新加入的女——大一中文系新生，叫顾言。

"顾言做事认真。"苏晚枝在电话里对陆允川说，"但是她有点怕我。上周我路过301教室推门进去，她正在给树贴新叶子。手抖了——把一片叶子贴歪了。她赶紧撕下来想重新贴，我说不用撕，歪的叶子是真叶子。自然的树没有每一片都正的。"

"她说什么？"

"'学姐你说的是真的还是为了安慰我？'我说两个都是。她就笑了——一个人站在树墙前面自己笑的。"

陆允川听着电话里她的声音。比以前放松了一点。不是语速上的放松——是她说话时留的空隙变长了。以前她的话是连续的、完整的、一句接一句没有停顿的。现在会在句子之间留一个呼吸的空隙。

"你这个暑假最大的收获是什么？"他问。

"不是我画的那些图——是把钥匙交出去了。交给了小孟。然后把灯也只留了一盏。"

"老郑留下的那盏。"

"对。老郑从来不问我们为什么只用一盏灯。他自己也是这样的。每层只开一盏走廊灯。他说'看得清路就行'。"

"你现在晚上还去301吗？"

"有时候去。不是去检查——是去坐。坐在讲台下面。不开灯。就坐在黑暗里听外面的蝉鸣。昨天听到了一只蝈蝈——可能是从生物系实验室跑出来的。叫得很有规律。每隔四秒钟叫一声。"

"你在数它的间隔。"

"职业病。建筑测绘做多了，会对所有声音的间隔产生精度要求。"

挂掉电话后，陆允川把暑假留下来需要看的三本书——都是下学期中文系的必读——从图书馆搬回宿舍。宿舍里只有他一个人。徐远回家了，祁野在老家帮体校带暑假班的训练。

他把三本书排在书桌上。古代文学、现代汉语、文艺理论。文艺理论的编者序里有一句话："真正的理论不是提供答案——是提供更好的问题。"

他把这句话拍下来发给苏晚枝。附文："这句话像你教我的。不是给答案——是把问题问得更准确。"

苏晚枝拍了自己的电脑屏幕回他。屏幕上是CAD图纸——一座老宅的正立面实测图，每一条线都有精确的尺寸标注。图纸最下面有一行用CAD文本工具打的字："测绘人：苏晚枝。助手：三脚架、测距仪、一瓶矿泉水。"然后她又加了一行："问题：这栋房子的主人为什么把正门开在了西边而不是南边？回答：因为西边有三棵构树，夏天遮阳，冬天落叶。"

"你看，建筑也可以是一个好的问题。"她说。

"三棵构树——你量了树的胸径吗？"

"量了。最粗的那棵胸径四十七厘米。大概跟我同一年出生的。"

"跟树同龄——不容易。"

"对。我站在树前站了十分钟。测绘师傅说我浪费时间。我说这不是浪费时间——是确定为什么门开在西边。"

暑假的最后一周，她完成了测绘报告。交报告那天，她发了一条朋友圈。图是那栋老宅的正立面手绘透视图——她用针管笔绘的，线条干净得看不清任何重叠的痕迹。配文："这栋房子的正门开在西边，不是搞错了方向——是西边有三棵构树。建房的人愿意为三棵树改变门的朝向。"

陆允川点了个赞。评论："这是第三部最好的结尾。"

"还没结尾——还有一周暑假才完。"

"那就提前预支。"

暑假末的一个晚上，苏晚枝回到301教室。小孟在树墙下面贴了一张便条："晚枝学姐：这周新长了五片叶子。一片是诗，两片是建筑系学生的设计说明（他们说墙上的设计纲要很好用），一片是叶澄的实验数据之三——'酵母的最佳盐浓度是0.8%。'一片是祁野写的短句：'篮板球教会我的：落下来比跳起来需要更多的耐心。'"

苏晚枝看完便条后，在树墙树根位置加了一片很小的叶子。那片叶子上只有两行。第一行："控制权不是能力。信任也是。——当值最后一夜。"第二行是用红色马克笔补的，字迹更小："信任已经在用。好用了。——当值之后三周。"

然后她没有关灯——今晚她想去开后排那盏一直不用的灯。打开后排灯的开关。日光灯闪了三次，然后亮了。整间教室第一次同时亮了两盏灯。

她站在两盏灯中间的位置，看了看前墙和后墙，看了看左边那六张绘图桌。然后关了前灯，只留后灯。

"一盏就够。但可以换一盏。"她对着空教室说。
