# 第203章 最后一次争论之后

所属部卷：第四部：失去的美丽
时间：同一天傍晚
地点：校外面馆

## 诗歌插页 P203：原创仿宋小令

汤渐冷，雨还斜。
旧话落谁家。
一声听见已无他。
灯下各归霞。

苏晚枝先走。她撑开伞，伞骨有一根微微弯着，雨水顺着边缘往下滴。陆允川坐在原位，看她穿过马路，没有追，也没有喊她名字。面馆玻璃上凝着水汽，她的背影很快被红绿灯和行人隔开。

老板过来收碗，见他那碗还剩大半，问：“不合口味？”陆允川摇头，说：“挺好，今天没胃口。”老板笑了一下，把碗端走，又给他倒了杯热水。

他坐到雨小些才离开。东门外的路面有积水，车灯映在里面，被行人踩碎。他走得很慢，脑子里反复出现苏晚枝那句话：感动不能替我答应。过去他也知道这个道理，只是不愿让它落到自己身上。轮到自己时，道理就不再是道理，而是一道必须跨过去的坎。

回宿舍后，祁野问他怎么淋湿了。陆允川把伞挂起来，说：“碰见她了。”

祁野停住：“吵了？”

“说清了一些。”陆允川擦头发，毛巾盖住眼睛，“也算吵了。”

祁野没有继续追问，只把桌上的感冒冲剂推给他。陆允川端着杯子坐下，热气扑到脸上。他忽然觉得，这一次最难的不是失去希望，而是承认希望从一开始就不该由别人替他保管。
