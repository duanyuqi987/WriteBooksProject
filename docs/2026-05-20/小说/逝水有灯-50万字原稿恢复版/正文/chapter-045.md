# 第045章 诗抄本多年后

所属部卷：第一部：初秋三楼
时间：高二冬天
地点：男生宿舍下铺

## 诗歌插页 P045：原创七言小诗

窗落三楼课后深，
一声铃响动衣襟。
少年不解寻常意，
错把微光作远音。

多年后陆允川重写诗抄本，先把自己写得很委屈，写完又全部删掉。删到最后，页面上只剩诗抄本里被墨水洇开的句号，以及一个年轻人不太体面的执拗。

多年后他翻到这本诗抄本，发现真正值得留下的不是那些指向她的暗号，而是一个少年试图把混乱整理成句子的努力。

他坐在出租屋的小桌前，窗外有车灯慢慢扫过墙面。旧日记本摊开，纸页发脆，里面没有惊天动地的证词，只有几行用力过猛的句子。

他在旁边补写：陆允川开始抄诗，唐诗、宋词、现代诗混在一本本子里，所有空白处都隐约指向夏知遥。这句话写出来以后，比记忆本身更平静。平静不是不疼，而是终于不再要求那一年替自己负责。

手机相册里还留着几张学校旧照片。三楼、操场、礼堂、公告栏，全都被像素磨得发虚。陆允川放大看了看，又缩回去，觉得自己不必再从模糊里找答案。

窗外有人倒热水，水流进暖瓶时发出空空的回声。宿管阿姨在楼道里喊别说话了，声音从门缝钻进来，压住一屋子没长大的秘密。

程砚后来在同学群里发过一张聚会照片，笑说谁还记得当年谁暗恋谁。陆允川没有接这个话，只回了一个“看着都没变”，其实每个人都变了，只是照片替大家保留了一点旧样子。

叶澄私下问他，真的还要写这段吗？陆允川说要写，但不会再写成控诉。她回了句，那就让她也像个人，不要只像你青春里的答案。

这句话让他停了很久。于是他给夏知遥补回了日常：补回她的课本、朋友、沉默和选择，也补回她没有回应的权利。

他终于承认，喜欢不是法庭，青春也不是判决书。那些没得到回应的部分，可以难过，可以遗憾，却不能被写成对方的罪。

章节末尾，他合上日记本，去楼下买了一瓶水。便利店门口的灯照着湿漉漉的地面，风很轻。他忽然觉得，过去没有被洗白，也没有被毁掉，只是安静地退回了过去的位置。

他抄唐诗时喜欢把字写得瘦一点，抄宋词时又忍不住拖长最后一笔。程砚说他这不是练字，是给心事换衣服。陆允川没理他，只把台灯往里挪，免得光照到别人床上。

有一页现代诗写到一半，宿舍突然熄灯。他在黑暗里停住笔，墨水在纸上多洇了一点。上铺有人问谁还有手电，另一边有人说别写了，明天还要早操。

夏知遥的名字从来没有直接出现在诗抄本里，却又到处都是。一个“她”、一处“窗边”、一行“冬天的课桌”，都像绕路写出的地址。多年后陆允川再看，才觉得这种绕路既真诚又危险。

叶澄借本子那天没有翻太久，只停在一首仿宋小令前，说这句还可以，但你别把所有空白都留给同一个人。陆允川当时没答，心里却记了很久。

他在修改稿旁边写下四个字：不要审判。写完又补了两个字：也别粉饰。温柔现实不是把疼痛擦干净，而是让疼痛和饭菜味、雨水声、普通人的疲惫放在同一个世界里。

多年后最难写的不是遗憾，而是分寸。写少了像逃避，写多了像控诉。陆允川试着把句子放轻一点，不是因为事情不重要，而是因为重要的东西经不起太用力的抓。

多年后他把这一段重写时，特意保留了台灯和钢笔。这些东西没有抒情能力，却能证明那一年不是一团雾，而是一间有桌椅、有灰尘、有铃声的地方。

他没有再问如果当时怎样，因为问题问得太久，人会忘记继续生活。于是他把句子收住，让诗抄本留在原处，也让夏知遥从他的解释里走出去。

如果只写疼，这一章会很薄；加上台灯、钢笔、枕头、暖瓶，它才重新有了重量。那重量不是压人，而是让读者相信他们确实在这里生活过。

陆允川把钢笔拿在手里，先把这一页诗抄完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把这页写完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她在教室前排背政治提纲，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

男生宿舍下铺里最明显的不是心跳，而是暖瓶水灌满时的空响。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，泡面调料包被撕开的味道把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。钢笔、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

多年后他重写这一段，先删掉审判，再删掉自怜。留下诗抄本，留下男生宿舍下铺，也留下那个还不会把喜欢放轻的自己。

他终于知道，温柔不是替过去辩解，而是承认每个人都只拥有当时的能力。能写清这一点，比把谁写成坏人困难得多。

这一段让诗抄本不只装暗恋，也装宿舍的冷、台灯的光和同伴的提醒。诗可以安放心事，但不能替他向别人索要答案。

如果只写疼，这章会很薄；加上钢笔、暖瓶水灌满时的空响和泡面调料包被撕开的味道，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在男生宿舍下铺发呆。”陆允川把钢笔收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

所以诗抄本多年后留下来的，不是“如果当初”，而是“原来如此”。原来人会把小事看得很大，也会在很久以后学会把它放回小事里。
