# 第130章 未接电话多年后

所属部卷：第三部：寒风里的花
时间：多年后
地点：男生宿舍阳台

## 诗歌插页 P130：原创仿唐绝句

旧机无声照指纹，
几行通话隔年昏。
若知铃响能惊客，
少拨当时第四轮。

多年后陆允川在旧手机备份里看见那晚的通话记录。四通电话挤在十一分钟里，时间短得惊人，也难堪得清楚。

他把截图导出来，本想放进素材文件夹，最后删了。通话记录不需要被保存成证据。它已经说明得够明白：当年他不是不知道自己急，只是把急交给了对方承担。

宿舍阳台早已拆掉防盗网，新楼刷了浅灰色外墙。陆允川站在楼下，看见阳台上挂着一排衣服，风一来，衣角同时动了动。

他想起祁野那句“连续打电话不是深情，是催债”。年轻时觉得难听，现在觉得准确。准确的话常常不温柔，却能省掉后来许多更不温柔的事。

离开前，他在旧日记旁边补了一句：未接电话不一定是冷落，连续拨号却一定会制造压力。

写完后，他把笔帽盖上。那一晚终于不再只属于他的慌张，也属于苏晚枝被打断的会议、被震动追赶的手和那张还没填完的页码表。
