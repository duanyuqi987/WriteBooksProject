# 第160章 短暂和好多年后

所属部卷：第三部：寒风里的花
时间：多年后
地点：诗社活动室

## 诗歌插页 P160：原创仿唐绝句

旧刊封处纸尘微，
一饼当年不必追。
若把轻松看作诺，
晴窗也会转阴灰。

多年后陆允川在旧照片里看到那天的活动室。照片是叶澄拍的，几只纸箱堆在窗下，桌角放着半包苏打饼干。

他盯着那半包饼干看了一会儿，笑了。年轻时他曾经把那块饼干写得很重，像一段关系回暖的信号。后来他把那段删掉，只留下“中午，她分给我一块饼干”。

这样反而更好。

那天确实轻松过。轻松是真的，边界也是真的。一个人愿意自然相处，不等于愿意重新承担过去所有压力。陆允川用了很久才学会不把善意往自己需要的方向拖。

他把照片重命名为“整理旧书”，没有写“短暂和好”。因为真正值得保存的不是和好这个判断，而是几个人在晴窗下把旧刊整理进纸箱的下午。

那天下午已经足够完整，不需要再被他加上更多意义。
