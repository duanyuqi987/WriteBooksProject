# 使用说明

## 开始新小说

```text
用 $novel-writing-workbench 帮我写一本 50 万字的修仙小说。主角是废柴弟子，意外获得上古传承。
先帮我确定世界观和主角设定，再规划第一卷大纲。
```

如果只有灵感碎片，可以这样说：

```text
用 $novel-writing-workbench 帮我头脑风暴：我想写一个关于"时间商人"的故事，主角能用寿命交换任何东西，但发现自己的寿命正在被未知的人买走。目标是 30 万字。
```

## 继续写作

```text
用 $novel-writing-workbench 继续写小说。
```

系统会自动读取记忆系统，恢复到上次中断的位置。

## 大纲规划

```text
用 $novel-writing-workbench 为我的小说规划第一卷大纲。
```

大纲采用四层展开：字数 → 卷结构 → 粗略章节 → 精细章节（前5章），后续章节滚动展开。

## 审查文稿

```text
用 $novel-writing-workbench 审查 chapter-012，重点检查伏笔一致性和技法使用。
```

审查包含六层闸门：读者视角 → 禁用词兜底 → Canon 检查 → 技法检查 → 升级感检查 → 声纹检查。

## 角色管理

```text
用 $novel-writing-workbench 为我的主角创建角色卡，需要包含身份张力、声纹档案和弧光规划。
```

## 伏笔追踪

```text
用 $novel-writing-workbench 检查当前所有未兑现的伏笔，哪些应该在下一章推进。
```

## 起点热门分析

```text
用 $novel-writing-workbench 分析当前起点修仙小说的热门趋势。
```

需要先运行爬虫脚本获取数据：
```bash
python scripts/qidian_analyzer.py
```

## 语音转文字

```text
用 $novel-writing-workbench 把 ./录音/ 文件夹批量转成写作素材。用 medium 模型，输出小说素材格式。
```

```text
用 $novel-writing-workbench 把这段会议录音转文字，生成 SRT 字幕。
```

需要先安装 Whisper 和 ffmpeg：
```bash
pip install openai-whisper
```

## 小说数据集下载

```text
用 $novel-writing-workbench 下载中文网文小说数据集，用来做写作参考。
```

也可以直接运行脚本：
```bash
python scripts/novel_dataset_downloader.py
```

## 双轨输出（Markdown + HTML）

```text
用 $novel-writing-workbench 把刚写完的章节同步成 HTML 阅读版。
```

## 查看进度

```text
用 $novel-writing-workbench 查看小说进度。
```

## 指定写作风格

系统内置三种风格。写作时可通过如下方式指定：

```text
用 $novel-writing-workbench 写下一章，使用冷白描风格。
```

```text
用 $novel-writing-workbench 写下一章，使用系统爽文风格。
```

```text
用 $novel-writing-workbench 写下一章，使用怪诞悬疑风格。
```

## 连续写作模式

如果不想每章都确认：

```text
不用每章问我，按计划持续分批写，直到本卷写完。遇到关键风险、设定冲突或质量问题时再停下来说明。
```

## 字数统计

```bash
python scripts/count_text.py path/to/chapter.md
```

脚本会输出 CJK 字符数、英文词数和粗略总量。
