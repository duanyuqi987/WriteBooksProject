# 第104章 一篇课文

时间：2015年11月

地点：楚州二中初一三班教室

卷次：卷六：平行轨道

十一月中旬。实习最后一周。

周三上午。第三节课。初一三班。

课表上写着——《背影》。朱自清。

顾临风把课文翻了翻。这篇他上中学时学过。那时只觉得啰嗦——一个人送儿子坐火车，翻来覆去地写买橘子的事。

现在再看。不一样了。

上讲台。他把书放在桌上。不用看教案——这篇课文他读了不下二十遍。

"同学们。今天我们学习朱自清的《背影》。"

底下的翻开书。哗啦啦一片。

"朱自清写这篇散文是在1925年。当时他在北大教书。有一天他收到父亲的一封信。信上说——'我身体平安，惟膀子疼痛厉害，举箸提笔，诸多不便，大约大去之期不远矣。'他读完信之后写下了这篇《背影》。"

黑板。粉笔。背影 朱自清。他的粉笔字比刚来的时候好了一些。

"这篇写的是——一个儿子对自己父亲的回忆。回忆里最清晰的不是父亲的相貌。不是声音。也不是他说过的话。而是——一个背影。"

他开始读课文。前两段——写徐州、写家、写父亲赋闲。

然后是月台。

"我看见他戴着黑布小帽。穿着黑布大马褂。深青布棉袍。蹒跚地走到铁道边。慢慢探身下去。尚不大难。可是他穿过铁道。要爬上那边月台。就不容易了。"

"他用两手攀着上面。两脚再向上缩。他肥胖的身子向左微倾。显出努力的样子。"

他读到这里。忽然停了。

手按在课本上。指节有点白。

教室很静。学生等着他往下讲。

他想到的不是朱自清的父亲。

他想到的是高一那堂语文课。

陈维中讲《荷塘月色》。提问。苏晚晴站起来朗读。她的声音不高。但每个字都清楚。像下午四点的阳光照进教室——不刺眼。刚好能看清所有东西。

那是他第一次近距离听她的声音。

那天她站的位置——离他的座位大概五排。但他觉得自己像坐在第一排。

"这时我看见他的背影。我的泪很快地流下来了。"

他继续往下读。声音跟刚才一样——不响。咬字清。

"我赶紧拭干了泪。怕他看见。也怕别人看见。"

班上有几个学生在偷笑。大概是听到"泪很快地流下来了"觉得好笑——在他们这个年纪，"哭"还是一件不好意思的事。更别说是"父亲哭"。

顾临风没有停。

读完全文。他合上书。

"这篇课文我们不分析句子结构。也不考修辞手法。"

底下有轻轻的骚动。

"我们只讲一个问题——作者写到最后说'我的眼泪又来了'。你们觉得他第二次哭——跟第一次有什么不一样。"

没人举手。

他等了一会儿。

后排有个女生犹豫地把手举起一半。

"你说。"

"第一次是看到爸爸的背影心疼。第二次是——想他了。"

顾临风点了点头。

"再看课文最后一段——'我不知何时再能与他相见。'这是1925年写下的。他父亲1917年去世。也就是说——他再也没有见到他。"

教室里安静了。

安静了很久。

下课铃响的时候，他说："今天的作业——周末回家跟爸妈一起吃顿饭。不要写作文。不用交作业。只是吃顿饭。"

学生们看着他。有那么几秒——没人动。

然后有人开始收拾书包。有人往外走。那个曾经低头玩手机的男生，在出教室门的时候回头看了顾临风一眼。

然后走了。

顾临风坐在讲台旁边。等教室空了。

他把课本收进包里。把黑板上写的字擦掉。

粉笔灰在阳光下飘。落在他的袖口上。

朱自清的父亲买完橘子走了。他看着他爬上月台。肥胖的身子。努力的样子。

朱自清的父亲再也没有见过儿子。

而顾临风的父亲还在青阳县机械厂上班。

苏晚晴的母亲还在县城开超市。

他们两个还在同一座城市。

只是隔着一条街。二十分钟的车程。两年多的沉默。

他拿着黑板擦站了很久。

最后一粒粉笔灰落下来。

他把黑板擦放回槽里。背着书包走了。
