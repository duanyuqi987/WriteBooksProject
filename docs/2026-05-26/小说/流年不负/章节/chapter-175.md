# 第175章 婚礼
时间：2019年5月1日
地点：青阳县月湖边酒店
卷次：卷九：执手之约

婚礼不大。

十来桌人。亲戚占了一大半——顾家镇上的，苏家县城的，两边的叔叔姑姑舅舅阿姨，全来了。高中同学坐了一桌。大学同学也有一桌。同事零零散散凑了半桌。其余的宾客是两家父母的朋友邻居。

顾临风站在台上。台下的面孔他差不多都认识——有些十年来见，有些上周刚一起吃过饭。苏晚晴站在他右边。婚纱的裙摆在地上铺成很小的一圈。白雏菊捧花给了许若琳之后她手里空着——两只手交握在腰前。

古筝曲停了。大厅里嗡嗡的人声小了。但没完全安静——角落里还有老人在聊天，在说"新娘子真俊。"

高中同学那桌最热闹。

化学课代表——现在在某药企做研发——推了推眼镜。扭头对旁边的人说:"当年你们俩可是咱们班最大的悬案。"

旁边的人是班长。班长的头发比高中少了一些，但声音一点没变。他接了一句:"谁不知道顾临风喜欢苏晚晴啊。就是没人敢当面说。"

另一个同学——当年坐苏晚晴后面的那个男生，现在在青阳县城开了家手机维修店——拍了一下桌子:"我敢说——我高一就说了。我说'顾临风那边一直往这个方向看——'。你们全不信。"

"不是不信。"化学课代表接过去。"是不敢信。顾临风这个人——你当面问他吃没吃饭他都想半天才回答。让他表白——那不是要他命。"

班长笑了。"所以他今天站上面，我到现在还不太习惯。"

旁边另一个女生——苏晚晴隔壁班的闺蜜——说:"你们男生知道什么。我跟苏晚晴同宿舍两年。她从来没说过喜欢谁。但她的课本第一页夹着一张纸——上面画了一个男生侧脸。高一的。"

一桌人都安静了。

"我见过。画得不怎么样。但——侧脸的线条，跟你们班顾临风一模一样。"

化学课代表推了推眼镜。"所以——"

"所以。两个人都这样。一个偷看。一个偷画。"

一个夹在课本第一页。一个写在笔记本最后一页。都以为对方不知道。但十年后的这张桌子上，所有人都知道了。

台上。顾临风看着这张桌子。他看到他们在聊天——嘴在动，听不到说什么。但能看到化学课代表推了三次眼镜，班长拍了两次桌子，手机店老板晃着他的手机在说"你看你看——"。那些面孔比高中时多了细纹，多了黑眼圈，有人的鬓角早白了几根。但在那张桌子上，在五月的月湖边，他们都是十八岁。

他拿起麦克风。

"谢谢大家来。"

大厅安静下来了。连角落里聊天的两个老人都停了。

"我等这一天等了很久。"

他停了一下。手指攥紧了话筒。话筒上缠了一条红丝带——喜气的红，他的手指却是白的——因为攥得太紧了，血都跑到手掌另一面去了。

"没有更多的话。"

台下安静了一瞬。

然后鼓起了掌。啪啪啪啪——不是那种稀稀拉拉的午后茶会掌声。是齐的。一开始不太齐——后来慢慢齐了。从高中同学那桌开始——整桌人全部站起来。然后是大学同学那桌。然后是两边亲戚——顾母站起来最慢，因为她在擦眼泪，纸巾还在手里攥着。苏父也站起来了——他本来想坐着拍手的，但看了一圈周围的人都站了，就也站起来了。动作很慢——从椅子上起来，膝盖响了一下。然后站直了。

苏晚晴看着台下。看着这些站起来拍手的人。

她高中同学在十年前也这样——不过当时拍手是在运动会和元旦晚会上。现在拍手是在婚礼上。十年前他们十六岁。穿着校服，头发剪得规规矩矩。现在有的已婚、有的带娃、有的单身、有的创业。但拍手的姿势跟十年前一样——手掌张开，落到一起。

他们拍的是一段暗恋的结局。

高中同学那桌上。化学课代表眼眶红了——但没哭出来，因为他还在推眼镜——推了第四次。班长把手拍红了——他夫人坐在旁边，用手肘撞了他一下，"你轻点。"手机店老板对着台上有样学样地喊了一句"早生贵子！"——被他夫人从后面打了后脑勺一下。

苏晚晴隔壁班的那个闺蜜没站起来。她坐在位置上。对着台上的苏晚晴竖了个大拇指——手举到肩膀那么高，停了大概三秒钟。苏晚晴看到了。朝她弯了一下嘴角。

交换戒指。

顾临风从西装内袋里掏出戒指盒——红色丝绒，边角有一点磨白。跟去年十一月在月湖边打开的那个不一样——这个是婚礼版的。两枚戒指并排放在盒子里:一枚是那圈碎钻，一枚是素铂金。两枚戒指都是一样的颜色——月光的颜色。

他把碎钻戒指拿起来。苏晚晴伸出手——左手。无名指。戒指已经戴过一次了，但还是同一个手指，同一个人，同样是碎钻在灯光下碎成无数个小光点。他轻轻套进去。戒指经过指节——很顺，比上次更顺。

轮到她了。她把素铂金环拿起来。他伸左手。无名指上戴过一次的痕迹还在——一个浅浅的印子，像是手指已经记住了戒指的宽度。她把戒指推进去。素铂金环挨着那个浅印，严丝合缝。

"好了。"

两个字。

台下的掌声又起来了。但这次没有之前长。因为大家都等着看新人吻一下——这是一个婚礼。但顾临风没有按照婚庆公司的流程走。他低下头。没有吻她。而是把额头抵在她的额头上。

两个人前额碰前额。眉毛碰眉毛。眼睛离得太近——什么也看不见。但他能感觉到她的呼吸。她能感觉到他睫毛在动。他的睫毛扫到了她的眉毛。

这个动作比任何仪式的亲吻都长。

台下有一个小孩——苏家三叔的孙子，大概四岁——忽然问了一句:"他们在干什么？"

他妈妈回答:"在说悄悄话。"

小孩说:"不是说悄悄话——他们没动嘴。是在碰额头。"

没人再回答他。

额头分开。顾临风又把麦克风拿起来。

"还有一件事。想给一个人看。"

他从西装内袋掏出那张纸。折了两折的，纸边上有一点皱——跟去年十二月求婚时一样。其实不是"一样的那张"——是另写的一张。同样的一首现代诗，不同的纸张，同样的字迹。他打开纸——展开很大，比上次展得平。

"这首诗。是去年跪在月湖边写的。苏晚晴听过。赵一鸣也看过一部分。"

他顿了一下。

"但还有一个人。她从来没看过。但我觉得她应该看。"

他看着台下一个方向。顾母的方向。

"妈。"

顾母愣住了。眼泪从中午就在眼眶里转——但现在才真的掉下来。她把老花镜往上推——镜腿被胶布缠过的地方被她手指捏着。

"你以前偷偷看过我的笔记本。说看到一句'九月风初凉'。你说看不懂。"

他把纸举起来——对着顾母坐的方向。台上的灯光穿过纸背，让那些钢笔字变成了半透明的黑色。

"今天念给你听。"

他念了。跟去年冬天在月湖边对着苏晚晴念的那遍不一样。那次是紧张。这次是平静。声音缓慢，像在语文课上带学生朗读。每个字都咬得很准——比上课时还准。因为他知道台下这个人——这个偷看了他笔记本的女人——等这一次等了十年。

念完了。

大厅里又安静了一下。这次比前面两次都长。

顾母没站起来。她脱了老花镜。拿着老花镜的手在抖——镜片在手里晃，反着台上的灯光。她嘴唇动了好几下——想说什么，但没说出来。最后从桌上拿起茶杯。茶杯里是茉莉花茶——酒店给每桌上的那种最便宜的茶水。她喝了一口。放下杯子。对旁边的人说:"这孩子——从小就这样。闷着。什么都不说。但全写下来了。"

她说这句话的时候声音是哑的。但嘴角在笑。那种笑——是从眼泪里挤出来的。

苏父坐在另一张桌子旁。他看着台上拿诗的年轻人——上次这个人坐在他家沙发上，手心全是汗。现在站在台上，手心还是湿的——话筒上那个红丝带被他攥出了汗渍——但是声音很稳。

苏父端起茶杯。紫砂壶里的茶倒进紫砂杯——动作和在家里一样。然后把茶端过来。喝了一口。对苏母说了一句话:"这茶的杯数，我今天多喝了一杯。"

苏母没听懂。苏父也没解释。

婚礼结束。所有流程都走完了——该说的都说了，该念的也念了。比来宾们预期的要短——因为没有司仪的串场词，没有扔捧花的环节，没有切蛋糕的仪式。就是把该做的做了。说几句话。交换戒指。念首诗。没了。

但所有人都不觉得缺了什么。有些人走出酒店的时候回头看了一眼台上的两个人——新郎和新娘并排站着，手里各自拿着红色的结婚证和一张诗稿。西装袖口和白婚纱——黑和白——被窗外的晚霞染成了一样的橙色。

酒店大厅朝湖的落地窗外。月湖的水面在落日下发着金光。远处的水塔还站在那里。水塔顶上的灯还没亮。但快了。

桌上有一盘没吃完的喜糖——红色袋子上印着描金喜字，里面装着大白兔奶糖和徐福记花生酥。有些来宾把喜糖放进包里带回家。有些忘了拿——红色的袋子散落在桌布上，像一朵一朵小花。

高中同学那桌出来的时候还在聊。化学课代表说:"下次聚会——你们说下次是什么时候？"班长说:"不知道。但希望别再十年了。"手机店老板接了一句:"下次我请客——在我店里。虽然不大——"然后被他夫人又打了一下后脑勺——"人家结婚你谈生意。"
