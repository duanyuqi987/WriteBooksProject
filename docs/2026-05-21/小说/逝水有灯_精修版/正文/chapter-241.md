# 第241章 写成小说之前

所属部卷：第五部：逝水有灯
时间：多年后的清晨
地点：书桌和窗边

## 诗歌插页 P241：原创现代诗

我把修订稿放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

写成小说发生之前，生活先把场面摆好。

清晨六点，陆允川坐在书桌前。窗户开着，风吹进来，把书稿最上面那页掀起一角，又轻轻放下。他没有立刻去压。任由它起落了几次。纸张被风翻动的声音和远处早班公交的刹车声叠在一起。

他面前是五十万字的终稿。分成四十五个主章和一个诗歌索引。最上面一行是书名：《逝水有灯》。

这个名字是程砚取的。不是他。但这个名字是对的。他在写完最后一章的那个晚上给程砚发过一条消息："你取的书名。归你。"程砚回："不用归我。归那盏一直没灭的灯。"

现在看着这整摞稿子，他开始做写成小说前的最后一件事情：删。

不是大幅删。是删掉那些还在文末残留的小总结。他写了一份清单：

- 第XX章结尾："他终于明白" → 改成具体动作
- 第XX章结尾："写作到最后也许只是" → 删掉整句
- 第XX章结尾："这不是彻底释怀的神话" → 删掉。不需要否定
- 第XX章结尾："结尾最怕用力" → 删掉。这话本身就是在用力

他逐章逐行地检查。这项工事持续了将近三个小时。每删掉一句模板化的总结，他就在旁边空白处用铅笔标一个"已修"。铅笔的笔芯断过两次。最后一次断的时候他把笔放下，去厨房倒水。喝完回来继续。

手机响了。妹妹。

"哥，你书什么时候写完？"

"今天。"

"今天？"

"大概中午之前。"

"那写完以后你要干嘛？"

"休息两天。然后看能不能帮妈把厨房那个橱柜换了。"

"那个柜子从你高中用到现在了。"

"所以该换了。"

电话挂了。他回到书桌前。最后一个需要删的句子在倒数第二章的末尾。原句写着："他不再急着安排一个漂亮结尾。"他想了很久。然后把整句删掉。改成："他把一页被风掀起的稿纸压平。"

书稿翻开新的一页。是最后一章的开头。他读了自己十年前的笔迹。十年前在诗社集会后写下的第一版"灯还没有灭"。白纸上有泪水滴过的褪色。不是伤感——是没及时擦掉的痕迹。他在退色上面画了一个极小的标记。然后继续阅读。

写完清单上最后一个"已修"的时候，书房的闹钟响了。六点半。该起来了——但他已经坐了三个小时。

他在书桌角摸到了那支旧中性笔。笔帽松了。手指稍微用力就能转几圈。这支笔是他大学时在图书馆文具机里买的——两块钱。陪他写完了笔记、信稿、诗、四版修改。

再两小时，这部书就完成了。不是"终于"完成。就是完成。

风又把纸吹起来。这次他伸手压住了。
