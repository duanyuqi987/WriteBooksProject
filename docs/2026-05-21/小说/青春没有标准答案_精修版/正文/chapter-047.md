# 第047章 月考排名



时间：2008年十二月中旬，月考出分当天



地点：高一十班后墙红榜前



卷次：第三卷：雪落在两栋楼之间



2008年十二月中旬，月考出分当天，高一十班后墙红榜前，一张用透明胶贴歪的月考排名表先把这一天按进了十班的日常里。成绩贴上后墙，十班第一次被分数清清楚楚地切成不同位置。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

段璟想看常俊凡名次又怕被发现，常俊凡则不愿自己只被一个数字概括。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看一张用透明胶贴歪的月考排名表，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

邓桥也在这一天被推到前面。邓桥的特点是理科反应快，嘴上不服输，考试后会第一时间算排名。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“我英语怎么又少两分？”邓桥说。

“听力第二题你上次也错。”刘青岚说。

“排名贴这么高，是怕我们看不见吗？”段璟说。

“看见不等于看懂。”常俊凡说。

“别只看名次，看错在哪儿。”马立群说。

排名表贴上去的时候，教室先安静了两秒，随后才有人装作随便地往后墙走。

有人只看自己，有人先看同桌，有人嘴上说无所谓，眼睛却在前二十名里来回找。

段璟找到自己的名字后，手心里出了一点汗。他没立刻找常俊凡，怕自己找得太明显。

常俊凡看排名很快，看完就回座位，把试卷翻到错题页，像把自己从墙上那张纸里领回来。

邓桥在旁边一遍遍算单科差距，铅笔头敲着桌面，敲得前排都忍不住回头。

刘青岚把英语卷子折好，声音很轻地说，进步也得有人看见才行。

马立群把听力题重新放了一遍，磁带机卡了一下，班里没人笑，因为大家都知道笑完还得听。

这一天的分数把十班切开，又把他们拉回同一间教室。谁都不能只靠名次活，也不能假装名次不存在。

月考排名没有悬在空处。后墙红榜的纸角和同学们压低的呼吸，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，月考排名带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

马立群进门时不一定立刻说话，鞋底在水泥地上一停，月考排名带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在邓桥快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。月考排名越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。一张用透明胶贴歪的月考排名表让他想说话，又逼他先低头看题。

邓桥被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，月考排名留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。邓桥路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为月考排名翻身，谁还在被窝里背单词，旁边人都听得见。

有人说将来要去新乡、郑州，也有人只盯着红榜上那一行小字，先把下一张卷子改完。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理月考排名里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：一张用透明胶贴歪的月考排名表、一只冻红的手、一句老师点名、一块还没扫干净的雪。

余小满站在红榜前数自己的名字，数到一半又从头看起。冬天把这些小动作放大，让每个人的慌张都有了形状。

曹蔚把错题本翻到新一页，先写下这次少掉的两分。十班的热闹没有停，只是换成了更低的声音。

何嘉树捏着成绩条不说话，纸角被指甲按出一道白痕。这样的细节没人会写进成绩单，却会留在同学之间。

邓桥没有因此变成特别醒目的人，可在段璟眼里，邓桥从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，排名让前后桌之间的轻松变窄，也让十班第一次真正讨论学习方法。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
