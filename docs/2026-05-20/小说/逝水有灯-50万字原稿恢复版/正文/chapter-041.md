# 第041章 诗抄本之前

所属部卷：第一部：初秋三楼
时间：高二冬天
地点：男生宿舍下铺

## 诗歌插页 P041：原创现代诗

我把早晨放低，
让一层楼从课桌边慢慢经过。
你的名字没有被我喊出，
只在诗抄本里被墨水洇开的句号旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写诗抄本抵达之前的坡度。冬天的宿舍很早就黑，窗缝漏风，铁床架一碰就凉。陆允川坐在下铺，把被子搭在膝盖上，台灯罩出一块小小的黄光。

他把唐诗抄在前面，宋词抄在中间，自己写的短句藏到最后。写到某个字时笔尖顿住，墨水慢慢洇开，像一句话没忍住。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。

那天真正的事还没有发生，诗抄本里被墨水洇开的句号却先被他记住了。它太小，小到放在任何人的日子里都不显眼，偏偏在陆允川这里有了重量。

“你这本子是给语文老师看的，还是给自己壮胆的？”程砚说。“抄诗怎么了？”陆允川说。“抄诗没事，别把人家抄成欠你的。”程砚说。

程砚说完就去洗漱，牙刷杯碰在脸盆架上叮当响。陆允川盯着那句“别欠你的”，心里有点不舒服，却没有反驳。

叶澄后来借过这本诗抄本，看见边角密密麻麻的小字，只说了一句，诗可以救人，也可以把人困住，看你怎么用。

课间十分钟被铃声切得很碎。有人挤去小卖部买面包，有人趴在桌上补觉，有人拿着水杯一路跑向水房。陆允川站在这些声音中间，心里那点紧张没有消失，却被生活稀释了一些。

夏知遥这时还只是人群里的一个名字。夏知遥并不知道自己被写进那么多空白。那天她在教室里背政治提纲，手指压着书页，一条一条往下划，神情认真得近乎朴素。陆允川看见了，却没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子，又一行行划掉。留下来的，是教室里真实的光、真实的灰、真实的不知道该把手放在哪里。

窗外有人倒热水，水流进暖瓶时发出空空的回声。宿管阿姨在楼道里喊别说话了，声音从门缝钻进来，压住一屋子没长大的秘密。

如果说喜欢在这里有了开端，也不是因为命运突然发声，而是因为一个少年开始过分留意普通事物。他还不懂，留意可以是温柔，也可能慢慢变成对自己的消耗。

他抄唐诗时喜欢把字写得瘦一点，抄宋词时又忍不住拖长最后一笔。程砚说他这不是练字，是给心事换衣服。陆允川没理他，只把台灯往里挪，免得光照到别人床上。

有一页现代诗写到一半，宿舍突然熄灯。他在黑暗里停住笔，墨水在纸上多洇了一点。上铺有人问谁还有手电，另一边有人说别写了，明天还要早操。

夏知遥的名字从来没有直接出现在诗抄本里，却又到处都是。一个“她”、一处“窗边”、一行“冬天的课桌”，都像绕路写出的地址。多年后陆允川再看，才觉得这种绕路既真诚又危险。

叶澄借本子那天没有翻太久，只停在一首仿宋小令前，说这句还可以，但你别把所有空白都留给同一个人。陆允川当时没答，心里却记了很久。

这一章要把脚步放慢，慢到能听见课桌被拖动时的短促摩擦声。事情还没到来，可人物已经在日常里摆好了位置：谁靠窗，谁靠门，谁总替别人递一张纸。

陆允川后来才知道，所谓开始并不是一个明确的按钮，而是很多个微小选择叠在一起。他没有说话、没有靠近、没有确认，可他已经在心里给那个人留出了一块地方。

事情还没有真正发生时，陆允川已经在心里排练了好几种反应。可排练没有用，现实总会把人推到更笨的位置，让他先拿错台灯，再说错一句无关紧要的话。

那天放学，他没有立刻回宿舍，而是在男生宿舍下铺外多站了一会儿。风把暖瓶吹得轻轻晃动，他忽然意识到，自己等待的并不是答案，只是一个能让明天继续期待的理由。

陆允川把钢笔拿在手里，先把这一页诗抄完，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把这页写完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她在教室前排背政治提纲，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

男生宿舍下铺里最明显的不是心跳，而是暖瓶水灌满时的空响。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，泡面调料包被撕开的味道把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。钢笔、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情还没到来，诗抄本已经先进入他的视线。他当时不懂这种提前的留意有多危险，只觉得只要自己足够安静，就不会打扰任何人。

放学后他在男生宿舍下铺外多站了一会儿，听见楼下有人喊集合。那一声很普通，却把他从想象里拉出来，让他记起自己还要回宿舍、洗杯子、准备明天的早读。

这一段让诗抄本不只装暗恋，也装宿舍的冷、台灯的光和同伴的提醒。诗可以安放心事，但不能替他向别人索要答案。

如果只写疼，这章会很薄；加上钢笔、暖瓶水灌满时的空响和泡面调料包被撕开的味道，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

本章最后，事情仍悬在半空。陆允川收好书，跟着人群往外走。楼道里风从窗缝挤进来，把他的校服袖口吹起一点，他下意识按住，像按住一句还没成形的话。
