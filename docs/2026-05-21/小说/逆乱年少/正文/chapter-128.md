# 第128章 抵达

时间：2012年7月5日，傍晚

地点：太行山李家沟村小

卷次：第六卷：雪落无声

从火车站到李家沟还有四十里山路。

来接他们的是村长李德厚，五十六岁，脸上晒得黑红，开一辆半旧的农用三轮车。车斗里铺了两层化肥袋子，八个人就坐在袋子上面，手抓着车斗边缘，颠了一路。

山路是土路，前两天下过雨，路面坑坑洼洼的。三轮车每过一个坑，车斗里的人就集体弹起来一下，屁股离开化肥袋子然后又落回去。苏安晴坐在车斗最里面，两只手抓着车斗边沿，指节都捏白了。林玉起坐在她对角，每次车颠簸的时候他都下意识地看她一眼——然后又移开。

山上比山下凉快。

到的时候太阳快落山了。李家沟小学在村子的最高处，三间平房，一个土操场，操场边上竖着一根旗杆，旗杆顶上飘着一面褪了色的红旗。

教室的窗户缺了两块玻璃，用塑料布蒙着。课桌是旧的，桌面被刻得坑坑洼洼，桌腿下面垫着碎砖头——地不平。

林玉起站在教室门口往里看了一眼。黑板不是黑板，是一面墙上刷的黑漆，漆面磨掉了一大片，露出底下灰白色的墙皮。讲台上放着一盒粉笔，粉笔只有半截，最长的那根也不到一根手指的长度。

陈维中带着男生住一间教室，女生住另一间。床是几张课桌拼起来的，上面铺一层硬纸板，再铺一层褥子。

苏安晴把大包打开，里面的书一本一本地拿出来——二十几本旧课本、几本作文书、一本翻烂了的《新华字典》。她把书码在讲台边上，码得很整齐。

"这些书够吗？"林玉起问。

"不够。但比没有强。"

她站起来拍拍手上的灰，看着窗外。窗外的山脊被晚霞染成了暗红色，一层一层往远处叠，叠到最远的地方变成了紫色。

"我小时候也在这个学校上过学。"村长李德厚蹲在教室门口，点了一根烟。"不过那时候还是土坯房。现在好多了，前年扶贫盖的砖房。"

"有多少学生？"陈维中问。

"报名了四十二个，但能来的大概三十几个。有些娃家住在更深的山沟里，走路要一个多钟头。农忙的时候就不来了。家里要人干活。"

苏安晴看着窗外没有说话。

夕阳的最后一丝光从窗户外面透进来，照在讲台上那摞书上。她码好的那排书被光照出一个整齐的影子，影子落在水泥地面上，像一截微型的阶梯。

晚饭是村长老婆做的——土豆炖粉条，一大锅，搁了点肉末。苏安晴端着碗站在教室门口吃，看着操场上那根旗杆。

林玉起端着碗走过去，站在她旁边。

"在想什么？"

"在想明天。"她说，"四十二个，只来三十几个。"

"来了就教。"

她看了他一眼，嘴角动了一下——不算笑，但也不算没笑。

天黑之后山里安静得吓人。没有路灯，没有车声，只有远处的虫鸣和偶尔一两声狗叫。林玉起躺在课桌拼成的床上，透过窗户看到外面的天——星星密密麻麻的，比在江城看到的多得多。

苏安晴的灯先灭了。然后是陈维中的呼噜声。

林玉起翻了个身，从枕头下摸出一个小本子。没有灯，他就借着窗外透进来的星光，用圆珠笔在纸上写了几个字。

看不清字迹。但知道写了什么。

他在本子上歪歪扭扭地画了一个圆——一个不太圆的圆。然后合上本子，塞回枕头底下。枕头上是化肥袋子的味道，混合着褥子上淡淡的潮气。窗外的虫鸣连成一片，像山里有一条看不见的河，河水不是水，是声音。

>
村深星作灯，
山远月为邻。
明日新台上，
将谁唤作生。
>
