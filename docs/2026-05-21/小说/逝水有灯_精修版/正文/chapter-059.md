# 第059章 最后一场模拟考旁观

所属部卷：第一部：初秋三楼
时间：高三春天
地点：县一中考场

## 诗歌插页 P059：原创现代短诗

那时我们都以为，
结束会有清楚的门。
后来才知道，
门只是普通下午的一阵风。
有人走向远处，
有人学着把远处还给远处。

程砚觉得陆允川这次模拟考有一个进步：他发呆的次数显著减少了。

"你注意到了吗？"午休的时候，程砚对叶澄说。两人在教室后排的窗边，叶澄正在整理英语答题卡，程砚靠着窗台嗑瓜子。

"注意到什么？"

"陆允川。他这次考试期间没有老往夏知遥那边看。"

叶澄把目光从答题卡上抬起来，想了想。好像确实没有——陆允川这次考完之后，做的最多的事是对答案和改错题，而不是站在走廊里等夏知遥经过。

"可能是考试压力太大了，顾不上。"叶澄说。

"我觉得不是。"程砚把瓜子壳吐进手心里，又倒进旁边的垃圾桶，"我觉得他是真的开始紧张高考了。这跟之前不一样。之前他那叫暗恋，这次是真慌了。"

"你慌吗？"

"慌啊。谁说我不慌。"程砚拍了拍手上的瓜子屑，"我慌是因为我妈说了，要是考不上二本，就让我去她厂里打工。"

"那你考一个给她看看不就行了。"

"你说得轻巧。"程砚苦笑了一声，但很快又恢复了平时的表情，"不过陆允川比我麻烦。他又要考大学，又要惦记人。CPU不够用。"

"你少用这种比喻。"

叶澄这么说，但嘴角还是微微动了一下。

另一边，夏知遥和杜雨薇在走廊栏杆边上透气。杜雨薇把胳膊肘搭在栏杆上，看着楼下小卖部进进出出的人。"你说这次模拟考到底准不准？"

"不太准。"夏知遥把手里的矿泉水瓶翻了个面，"题太偏了。历史那道太平天国的题，考的是教材上的小字注释。高考不会这么考。"

"你怎么知道？"

"我做了五年真题，没见过这么出的。"

杜雨薇佩服地看着她。"你连真题都做完了？"

"没做完。做了三年的。"夏知遥喝了一口水，目光落在远处操场上，"我只是觉得，与其一直猜高考会出什么题，不如先把自己不会的补上。"

"你这么说好像很有道理。但我还是忍不住想猜。"

"猜也可以。猜完记得继续做题。"

杜雨薇笑了。她有时候觉得夏知遥说话的方式很特别——不热情，不冷淡，就是刚好够用。这让她想起夏知遥曾经跟她说过的一句话：我不喜欢让别人猜我在想什么，但我也不想解释太多。

回到教室以后，杜雨薇在自己的座位上偷偷观察了一会儿陆允川。陆允川正在改错题，左手按着卷子，右手拿红笔在草稿纸上演算。旁边程砚凑过去看了一眼，说了一句什么，陆允川皱了皱眉，用笔帽戳了一下程砚的手背。程砚缩手，笑着说"好好好不打扰你"。

杜雨薇收回视线，在自己的笔记本上写了一个字。然后划掉了。那个字是"陆"。

她不是喜欢他。她只是偶尔觉得，这个人跟自己有点像——都是那种不太会把话说清楚的人。区别在于，他想对夏知遥说点什么，而她想对他说点什么。但两个人都没说。

傍晚的时候，叶澄和夏知遥在楼梯口碰到。

"你历史那道选择题选了什么？"夏知遥问。

"B。"

"我也是。"

两人并肩下楼。叶澄忽然说："陆允川也是选的B。"

夏知遥看了她一眼。"你怎么知道？"

"他对答案的时候说的。"叶澄语气很平，"他这次考得还行，班主任说他进步了。"

"那挺好的。"

"你觉得好？"

"我们班多一个人上一本，对谁都好。"夏知遥把书包带子往肩上拉了拉，"班主任不是说，今年我们班一本率要比上届高吗。"

叶澄没再说什么。她知道夏知遥的回答滴水不漏——永远从班级整体出发，永远不会让人抓到可以延伸解读的缝隙。这不是冷漠，这是一种很多人学不会的自我保护方式。

走到教学楼门口时，夏知遥停了一步。"叶澄。"

"嗯？"

"你帮我看一下，我书包拉链没拉好。"她转过身背对着叶澄。书包最外层敞着，露出半截草稿纸。叶澄帮她拉上拉链，在那一瞬间看到了草稿纸上写的东西——不是英语单词，不是数学公式，是一行很小的铅笔字：我想考去一个有江的城市。

叶澄的手指顿了一下。她把拉链拉好，说："好了。"

"谢谢。"夏知遥头也没回，走进了放学的夕阳里。

叶澄站在原地，看着那个背影愣了好几秒。然后她忽然想起——上次程砚问她陆允川想去哪，程砚转述陆允川的答案是："想去一个有江的城市"。

一样的话。

叶澄把这件事压在心底，没有告诉任何人。那天晚上她写日记时，写下了一行：有些事情，可能当事人自己都不知道。旁观的人看清楚了，也不能去说。因为这些东西太轻，一说就碎。
