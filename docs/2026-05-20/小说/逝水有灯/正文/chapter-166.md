# 第166章 牵手的误会之前

所属部卷：第四部：失去的美丽
时间：大二冬末
地点：学校东门外小路

## 诗歌插页 P166：原创现代诗

小路边的灯坏了一半，
影子比事实更长。
他还没走近，
心里已经替别人，
写完了一场不公平。

冬末的东门外在修路，学生们都从旁边小路绕行。那条路窄，一边是施工围挡，一边是卖烤冷面的小摊。

陆允川那天去打印店取社团材料。路上碰见叶澄，她匆匆往学校里走，手里抱着一摞讲座海报。

“苏晚枝刚去东门那边取横幅。”叶澄说，“你要是顺路，帮忙看看她有没有拿错尺寸。”

“好。”

陆允川走到小路口时，天已经暗下来。路灯坏了一盏，施工围挡的影子压在地上。他远远看见苏晚枝和一个男生站在一起。男生手里提着横幅，另一只手像是扶了她一下。

他的脚步停住。

那一下很短。短到如果他走近一点，就会看见苏晚枝只是踩到一块松动的砖，身体晃了一下，对方扶住她手腕，很快松开。

可陆允川没有立刻走近。他站在坏掉的路灯下，让那一下在心里变长，变重，变成一个足够刺痛自己的画面。
