# 第095章 分科前最后一晚



时间：2009年十二月中旬，正式分班名单公布前夜



地点：高二十班教室、宿舍楼道、操场边



卷次：第四卷：分科以前的夏天



2009年十二月中旬，正式分班名单公布前夜，高二十班教室、宿舍楼道、操场边，梅朵桌上那张没有写完的同学录纸把这段日子推到十班面前。正式分班名单公布前夜，十班许多人都在装作平常，梅朵却悄悄写同学录。

段璟想问常俊凡会不会还在同一个班，常俊凡先说别为了任何人改选择。这不是突然降下来的大事，而是从早读、课间、饭卡、家长电话和后墙红榜里一点点长出来的。

段璟坐在教室里，常常先看见常俊凡的背影，再意识到自己其实该看卷子。从这个学期开始，这种走神不再只是喜欢，也会影响一道题、一次排名和一个选择。

常俊凡没有变成谁的答案。她会犹豫，会烦，会被家里问得沉默，也会在别人替她下结论时把话说得很硬。

梅朵也在这件事里显出自己的位置。安静，画小花，成绩普通，习惯在别人热闹时把细节留下来。十班从来不是只由两个人组成，它是四十多个人一起挤出来的青春。

“你们谁要写同学录？我只带了几张。”梅朵说。

“还没分呢，你就开始送别。”付小川说。

“有些话提前写，也不算坏。”常俊凡说。

“你确定了吗？”段璟说。

“确定了。但不是因为任何人，是因为我自己。”常俊凡说。

分科前最后一晚，教室比平时安静。不是没人说话，而是每句话都像怕碰到什么。

后墙上的座位表还没换，大家却已经开始想象谁的桌子会空出来。

梅朵把同学录纸压在练习册下面，问一句写不写，又马上说不写也没事。

付小川嘴上嫌酸，手却第一个把纸拿过去，写到最后只憋出一句别忘了借我橡皮。

常俊凡整理书包时，把理科资料和语文摘抄分成两摞，动作很慢，却没有犹豫。

段璟想问她是不是还在十班，话到嘴边又换成明天早读带不带英语书。

宿舍楼道里有人搬书，纸箱蹭过水泥地，声音像这三个月一直压着的选择终于动了。

操场边有风，灯光照着跑道。十班没有真的散，可每个人都知道，明天以后很多事会换一种形状。

陶建国这段时间管得比高一更细。不是因为学生忽然变坏，而是因为分科、晨检、国庆活动和期中考试挤在一起，谁一松就会被推着往后退。

梅朵被叫到名字时，先看了一眼周围同学，再看老师。这个小动作让段璟发现，班里很多看起来不在乎的人，其实都怕被公开放到台面上。

封丘县的校园生活没有电影里那么整齐。校门外有卖烤红薯的，有修自行车的，有家长骑电动车来送饭，也有学生在路边背单词。

黄河故道那边的风有时会吹进县城，吹动校服袖口，也吹得人想起更远的地方。可更远的地方暂时还只是试卷最后一道大题旁边的空白。

国庆六十周年的新闻、甲流晨检的表格、分科确认的回执，都没有替他们解决任何一道题，却让他们知道，学校外面的时间也在往前走。

宋乔会把女生那边的情绪说出来，付小川会用玩笑把男生那边的沉默撬开，周明澈则常常在最乱的时候把表格、名单和班费重新理一遍。

段璟有时羡慕这种确定。他自己更像一张写了太多草稿的纸，明明知道下一步该写什么，却还是忍不住看常俊凡会不会也停在同一个地方。

常俊凡并不总是回头。她更多时候只是把自己的事情做好：改题、接水、记账、扶凳子、把别人的话挡回去，然后继续低头写下一行字。

孙佳怡把同学录纸推给旁边的人，又假装只是随手一放，校服袖口蹭过桌沿，留下一道浅浅的粉笔印。这样的细节不会出现在成绩单上，却会留在很多年后的记忆里。

段璟整理书包时停了停，把一本练习册留在桌洞里，操场上的哨声隔着窗户传进来，显得很远。这个动作很小，却比一句大道理更能说明当时的慌。

常俊凡站在后门看了一眼座位表，才发现自己已经把名字记熟，窗外的树影晃了一下，像把话题轻轻推远。旁边的人没有追问，教室里的翻书声把尴尬遮住。

郑天宇把想说的舍不得写成好好学习，写完自己都笑了一下，有人把椅子往里收，木腿擦过地面发出短短一声。常俊凡看见了，也没有立刻说什么，只把自己的题继续写完。

蒋一诺从操场边回来，鞋底带着一点潮气，踩在教室地面上很轻，同桌抬头看了一眼，又像什么都没看见一样低头。段璟后来想起这一幕，才知道青春里很多变化一开始都很安静。

薛北辰把同学录纸推给旁边的人，又假装只是随手一放，粉笔灰落在袖口上，拍了两下也没有拍干净。窗外风一吹，纸角轻轻动了一下，像有人把话又咽回去。

唐小满整理书包时停了停，把一本练习册留在桌洞里，小卖部那边的铃声远远传来，很快被上课铃盖住。十班的热闹还在，只是每个人都比高一时多了一点分寸。

方晴站在后门看了一眼座位表，才发现自己已经把名字记熟，红笔帽没有扣紧，轻轻碰到桌面滚了一圈。老师从门口经过，所有人的声音立刻低了半截。

分科前最后一晚之后，十班没有立刻发生巨变。真正的变化通常不是一声响，而是某个同学开始少和旧同桌一起吃饭，某个名字开始出现在另一张名单上。

第四卷收在一场安静的夜里：有人去文科班，有人留理科班，十班没有散，却再也不是原来的十班。段璟把这件事记得很清楚，不是因为它多戏剧，而是因为它让他第一次看见选择有后果。

他也慢慢懂得，喜欢不能替常俊凡做决定，也不能把她的每一次沉默都解释成和自己有关。

常俊凡要走的路，首先属于她自己。段璟若真想靠近，就得先学会站在不挡路的位置。

高二就是这样来的：没有高一开学那样热闹，也没有后来倒计时那样压迫，却把许多关系悄悄推到更真实的位置。

晚自习结束时，梅朵桌上那张没有写完的同学录纸还留在教室里，像一枚小小的钉子，把这一天钉在十班的时间线上。

灯一盏一盏暗下去，楼道里传来脚步和水瓶碰撞声。段璟走到门口时回头看了一眼，发现常俊凡还在收书。

他没有催，也没有等得太明显。只是把书包带往肩上提了提，跟着人群往楼下走。
