# 第111章 等成绩

时间：2017年2月

地点：青阳县

卷次：卷六：平行轨道

二月。寒假。青阳。

考研成绩要二月中旬才出。等待的这段时间像一条很长很直的马路——看不到头。

顾临风回了家。他家住在青阳县机械厂家属院，五楼的两居室。他妈把他高中那间屋子收拾了一遍——换了新床单，擦过桌子，窗帘也换了。深蓝色，拉上的时候屋里很暗。

早上他睡到九点多。起来吃一碗稀饭配咸菜。然后翻翻手机。再躺回去。

他妈在客厅打电话，跟张阿姨聊了一个多钟头。谈话内容他听不清楚——大概是"你儿子也回来过年了吧""我家那个还在读书呢""考研——不知道能不能考上"。

下午他去县城转了转。

青阳县这些年变了不少。主干道拓宽了，路两边新开了好几家店。那个修鞋的大爷还在老地方——坐在马扎上，面前摆着个木箱子，箱子里装着钉子、锤子、碎皮。大爷老了很多。头发全白了。

新华书店没变。门脸还是那个门脸。门口的台阶磨得发亮——不知道多少人踩过。他在门外站了一会儿。没进去。

往前走。经过那条有老槐树的巷子。

树还在。冬天的槐树光秃秃的。枝干伸向灰色的天。巷子口的奶茶店关了——门上贴了张纸。"旺铺招租。联系电话***。"旁边那家包子铺还在。门口的蒸笼冒着白汽。

他在巷子口站着。

风吹过来，巷子底传来什么东西被风卷起来的声音。沙沙的。

有人从巷子里走出来。一个中年女人——个子不高，穿深蓝色羽绒服。手里提着一袋子菜。

是苏晚晴的妈妈。

他认出来了——走路的姿态没变。高二那年他在新华书店门口见过一次，苏晚晴跟她走在一起。母女俩走路的姿势很像——都是稍微内八。

苏妈妈拐了个弯，往东边走了。

他没动。

过了一会儿。又有人从巷子里往外走。

白色羽绒服。马尾。

苏晚晴。

她手里提着一个超市塑料袋，里面装了牛奶、面包和几袋速冻水饺。她站在巷子口，低头在塑料袋里摸了一下——大概是摸到钱包确认了一下。

然后她拐进了老槐树巷子。

隔着一条街。

十四步。

他从街这边看过去。她的马尾比照片上低了一点——不是扎高了，是普通的高度。白色羽绒服领子立起来。耳朵露在外面——冻得有一点红。

她在巷子里走了大概三十米——拐了个弯。进去了。

他站在马路对面。没有任何动作。没有喊"苏晚晴"。没有跑过去。没有从口袋里摸出手机。

只是站着。

公交站广告牌在身后。上面贴着某品牌饮料的广告——一个穿红裙子的女人举着玻璃杯笑。

风钻进他的领口。他把围巾紧了紧。

她拐进去的那个弯——巷子里第三家。门口有一盆铁皮文竹。那大概就是她家。

他站在街这边。看着那盆文竹。竹叶黄了两片。

十四步。

他想——如果他走过马路。走向那条巷子。敲开那扇门。她会有什么反应。她妈妈会有什么反应。

"阿姨你好。我是顾临风。是苏晚晴的同学。高中同学。青阳一中。2010级。2013届。"

然后呢。

他笑了。很轻。笑自己还跟高中时一样——把所有可能性在脑子里排练一遍。然后选最不可能的那种。

他转身走了。

往家的方向。往南。

经过新华书店的时候停了一下。书店门口贴了张海报——"寒假特惠，全场八折"。海报右下角有店长的手写——"寒假没有结束——你可以继续假装在学习。"下面画了个笑脸。

他看了两秒。然后继续走。

回到家的时候天色已经暗了。他妈还在厨房。炒菜的声音嗞啦啦的。屋里有油烟味。他换了鞋，把羽绒服挂进衣柜，在床边坐下。

电话响了。他妈接的。

"喂——张阿姨。嗯——在家呢。挺好的。嗯——你上次说的那个姑娘——"

顾临风把耳机塞进耳朵。放了首孙燕姿的《开始懂了》。声音调大。压过客厅的电话声。

窗外青阳县城的路灯亮起来。他在这里过了十七个寒假。这是第十八个。街景几乎一样——路灯、电线、广告牌。唯一不同的是她刚才走进巷子里的背影。

白色羽绒服。马尾。

隔着一条街。

十四步。

他翻了个身。把被子拉过来。蒙住头。

那些没有喊出来的句子又在脑子里打转。像抽屉里那封没寄出去的信。像高三那年空白的草稿纸。像朋友圈里没有按下去的赞。

他闭上了眼睛。

客厅的电话还在说。张阿姨的声音透过门缝传进来。"苏晚晴——那姑娘可好了——当老师——长得好——"

他听到了。

耳机里的歌放到了副歌。孙燕姿唱："不能回头——怎么回头——"

他翻了个身。膝盖撞到墙。疼了一下。
