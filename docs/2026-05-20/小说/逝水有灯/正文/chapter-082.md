# 第082章 迟到的表白

所属部卷：第二部：六月以后
时间：六月末
地点：河边护栏旁

## 诗歌插页 P082：原创仿唐绝句

河风吹热旧栏杆，
一句迟来出口难。
谢意轻轻非允诺，
少年才识路回弯。

陆允川说得比想象中短。

他没有把三年从头讲起，也没有说那些在诗抄本里绕来绕去的话。他只是低头看着护栏，说：“我喜欢过你很久。高考也结束了，我想至少告诉你一声。”

夏知遥没有立刻回答。

河边有人经过，脚步声从他们身后过去。远处小卖部的冰柜嗡嗡响，晚风带着热气。陆允川握着护栏，手心被铁皮烫了一下，却没有松开。

过了一会儿，夏知遥说：“谢谢你告诉我。”

这句话很轻，也很稳。陆允川抬头看她。

她继续说：“但我不知道怎么接。你很好，我也不讨厌你。只是……我没有想过我们之间是这样的。”

陆允川喉咙发紧：“嗯。”

“我怕我说什么都会让你误会。”夏知遥声音低了一点，“所以我只能说，谢谢。但我给不了你想要的答案。”

这不是拒绝里最锋利的一种，却仍然疼。陆允川点点头，努力让自己看起来平静：“我知道。”

其实他不知道。他只是终于不能再装作不知道。

夏知遥看了看手机：“我该回去了。”

“好。”

她走出几步，又停了一下：“陆允川，祝你以后顺利。”

“你也是。”

夏知遥离开后，陆允川还站在护栏边。河水没有因为这场表白改变方向。小卖部冰柜还在响，路灯一盏一盏亮起来。世界太平常了，平常得让他的难过无处表演，只能落回身体里，慢慢发沉。
