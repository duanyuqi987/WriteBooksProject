# 第149章 三十七行

时间：2012年12月25日至12月28日

地点：江城大学校园各处

卷次：第六卷：雪落无声

圣诞节之后雪还没化。

不是连续的雪——隔两天下一场，不大，但停不下。旧雪还没清干净新雪又盖上去，校园里的人行道上踩出了一层坚硬的冰壳，走上去咯吱咯吱响。路边的雪堆已经变成了灰黑色——铲雪车推过的痕迹，汽车的轮印，行人踩出的泥泞。但屋顶上的雪还是白的。树上的雪也还是白的。高处的白和低处的脏，泾渭分明。

林玉起用三天写了他写诗以来最长的一首诗。

不是在书桌前写的——是在走路的时候。从宿舍到图书馆。从图书馆到食堂。从食堂到教学楼。每一步都在想。想好了就在手机上按一行，回到宿舍用钢笔抄到本子上。

关于三年。

不是写"三年"这个词。是写这三年里发生过的每一个具体的时刻。大一在文学社教室里第一次看见她——她坐在第三排靠窗的位置，低头在写着什么。她抬起头来的时候，阳光正好从窗户照进来，照在她半边脸上。那一刻他只想到一个词："亮"。

大二的那本手抄诗集。他在图书馆坐了三个晚上，抄坏了三根笔芯。封面用的是他去文具店挑了一个小时的布面本子——灰色的，触感粗糙，适合握在手里。他不知道她有没有认真看。但现在想来应该是看了的。

支教的那个月。她站在讲台上写自己的名字。铁蛋举手。家访的山路。她崴了脚。乒乓球台上她说"你答案不像在骗我"。最后一节课她哭了，他把那颗糖含着，含了二十分钟，直到糖全部化成水。

秋天的走廊。她靠在墙上问他诗在哪。他说改好了但不是改好了。她在图书馆自习的那张桌子——她耳朵红了的那一瞬间。

生日钢笔。他吸饱墨水，在那张白纸上写"谢谢"。字很轻。怕压坏笔尖。

雪地里的笑声。她躺在雪上笑。说"雪里看你——还挺帅的"。手套少了一只。他帮她找到了。

这些全部。全部。他一个字一个字地在纸上写出来。诗不是连贯的叙事——是碎片拼接。一段一段，一节一节。每节写到三五行就换一个意象。用雪做骨，用光做皮。没有一个字写"喜欢"，但每一个字都在说同一件事。

写到深夜。郭向阳睡了。宿舍里只剩下钢笔在纸上划过的声音——细细的，沙沙的，像雪落在地上。

十二月二十八号凌晨两点。他写完了最后一节。

搁下笔。把本子举起来，对着台灯的光，从头到尾看了一遍。

三十七行。

是他写诗以来最长的一首。

他把本子合上。关了台灯。宿舍暗下来。窗外的雪还在下——隔着玻璃看不清雪片，只能看见路灯周围那一小圈光里有无数的白点在飘。不停地飘。不知道什么时候停。

他想。这首诗写了三年。

其实不需要三十七行。三行就够了。但他不敢只写三行。三行太直接了。三十七行——可以藏在里面。可以躲在雪后面、躲在光后面、躲在意象后面。可以不被看出来——或者被看出来但不用承认。

苏安晴之前说他"可以写得更直接一点"。他不是不会。是不敢。

诗可以很勇敢。人不可以。

或者说——诗是人的掩护。因为有了诗，人就可以不用那么勇敢。把想说的话都塞进意象里——雪是冷的、光是暖的、你是远的。读的人读到什么就是什么。写的人不负责解释。

他躺回床上。闭上眼睛。脑子里还是那三十七行字——每一个字都在自己的位置上，不偏不倚。

窗外的雪没有停。

>
三年写一纸，三十七行诗。
只敢寄雪去，不敢送君知。
