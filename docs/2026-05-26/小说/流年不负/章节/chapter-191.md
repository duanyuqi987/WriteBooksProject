# 第191章 回青阳

时间：2022年10月
地点：青阳县
卷次：卷十：不负少年

十月。国庆假期。天气晴好。

车开进青阳县城的时候是上午十点。路还是那条路——从省道下来，过一座桥，桥下是青阳河。河水比往年浅了一些，露出了河床上白色的石头。桥头的限速牌换了一块新的，但限速没有变——还是四十码。

女儿坐在后排的安全座椅里。手里攥着一包小馒头干——吃了一半。睡着了。小拳头还攥着饼干袋子。饼干碎屑掉在想推上。

他开车。苏晚晴坐在副驾驶。车窗开着一条缝。风从缝里灌进来——十月的风，干的，带着一点烧稻草的烟味。青阳到了。

她深吸了一下。没有说什么。

先回老槐树巷子。巷子在老城区。导航找不到了——上次回来是两年前，路怎么走已经记不太准确。他靠着回忆拐进了两条街，对了——巷口的电线杆上面还歪着那面"老槐树巷"的旧铁牌，铁锈更多了，牌子上的字掉了半边漆。巷口的槐树还在。树比记忆里粗了。树皮上的裂纹更大更深，像老人的手背。树下停着几辆电动车，地上落了些槐树叶子——十月的叶子边缘发黄。

她下车。把女儿从座椅上抱下来。女儿醒了。揉眼睛。

老房子。外墙刷了新漆。以前是灰砖，现在抹了白色的涂料。门也换了——以前是木门，现在是防盗门。窗台上的花盆还在，但里面的花换成了仙人掌，不是以前的文竹了。

胡同里出来一个大婶。她不认识苏晚晴。苏晚晴也没认她。两人互相看了一眼。大婶走了。

"变了好多。"她说。

"树没变。"

她抬头。槐树的树冠遮住了半条巷子的天。树枝上还挂着几串槐角——风干的，褐色的，在风里轻轻碰出声音。

他抱起女儿。让女儿的手去摸槐树皮。女儿摸了摸。然后把手缩回去——太糙了，硌手。

"这是妈妈小时候爬过的树。门口这棵。以前不敢爬——爷爷会骂。有一回偷偷爬上去，爬到第一个分叉就不敢再往上了。奶奶在下面喊——晚晴你给我下来。奶奶说你要摔了腿我就打断你的腿。"

她说着。声音里没有伤感。只是陈述。

"你后来下来了没有。"

"下来了。但下来的时候膝盖蹭破了皮。奶奶一边骂一边给我抹紫药水。紫药水抹在膝盖上——紫色的，洗两天洗不掉。"

女儿不知道他们在说什么。她只是站在槐树底下，抬头看树叶沙沙响。一片叶子飘下来，落在她头顶上。她伸手去摸——没摸到。叶子又飘走了。

他又把车往东开。经过县一中。

从车上能看到一中的围墙和围墙后面的操场。操场上有班级在上体育课。有人在跑步。白色的校服晃来晃去。跑在队伍前面的那个女生扎着马尾。马尾从左晃到右，从右晃到左。

他把车停在了路边。

隔着围栏。教学楼的外墙重新贴了瓷砖——以前是白色，现在是浅黄色。但楼的形状没变。六层。长条形。每层有八个窗户。从左边数第三个窗户——高三三班。他的教室。也是她的。

有铃声。下课铃。

然后学生们涌出教学楼。人声从安静变成了嘈杂——就像倒了一桶豆子在铁皮上。冲在最前面的是几个男生。手里转着篮球。后面的女生三三两两走在一起。有一个女生把碎发别到耳后。

动作和她一模一样。

"这是哪里啊。"女儿趴在车窗上看着铁围栏。手指抓着围栏的漆面。

"这是一中。"苏晚晴说。"爸爸和妈妈认识的学校。"

"认识？"

"嗯。以前一起在这里上学。"

"爸爸也在这里上学？"

"对。比妈妈差一点。成绩不好。坐在后排。"

"妈妈坐在哪里。"

"前面。靠窗。"

"为什么妈妈坐在前面。"

"因为妈妈成绩好。"

顾临风把方向盘握了握。然后松开了。把车熄火。打开车门。站在围栏外面。手扶着铁栏杆。栏杆被太阳晒得温热。

他转头看苏晚晴。她已经下了车。站在他旁边。

女儿也下了车。站在他们中间。一手拉一个。

教学楼的窗户在下午的阳光里反射着白光。看不清窗里面的情形。但他们都记得。

"进去看看。"

她看了他一眼。"能进吗。"

"门卫不在。门开着。"

"你怂不怂。"

"怂。但这次不怂了。"

他把女儿抱起来。三人往校门走去。
