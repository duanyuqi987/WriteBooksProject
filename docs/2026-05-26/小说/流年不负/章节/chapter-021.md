# 第021章 新座位

时间：2011年3月

地点：理科三班教室

卷次：卷二：窗边人影

三月。高一下学期开学。

顾临风到教室的时候门口的座位表换了。班主任贴了一张新的——按上学期期末成绩重新排位。他从下往上找自己的名字。

倒数第三排靠窗。

他把书包放好。拉开拉链。拉链还是卡。

教室里人还不多。日光灯亮着。吊扇没开。三月的天还有点凉。

赵一鸣还没到。他的新座位在第三排——离顾临风远了。赵一鸣期末考了第十五名，往前挪了。

门被推开。

苏晚晴走了进来。

灰色外套。深蓝色牛仔裤。马尾扎得很紧。书包上那个小鱼挂件还在——他隔了三排也看到了。她往教室前面走。

第二排靠窗。

她坐下了。

顾临风忽然意识到一件事——从这个位置，他能看到的不再是她的侧脸。是她的后脑勺。马尾。脖子。

她拉开书包。翻出课本。语文书。封面是白的。

窗外有一棵银杏树。三月的银杏还没长叶子，光秃秃的枝条伸在窗框里。

她坐在窗与银杏之间。

阳光从枝条间漏进来。落在她桌上。她往旁边挪了挪课本。

顾临风盯着那个方向看了几秒。

然后低下头。翻开数学书。第一页。第二页。第三页。他没看进去一行字。

赵一鸣从后门进来的时候拍了他肩膀一下。

"换座位了。你在看什么呢。"

"没看什么。"

赵一鸣看了看他。又看了看前面的方向。

"哦。"

他没再说别的。

顾临风把笔从铅笔盒里拿出来。小虎队的贴纸又磨白了一块。

上午四节课。语文、数学、英语、物理。

语文课老师没叫她朗读。

数学课她上黑板做了一道题。做对了。下来的时候马尾摆了两下。

英语课她被点起来背单词。背了五个。全对。

物理课她皱眉了。

顾临风记了下来。

下午的阳光换了个角度。从银杏枝条间照进来。照在她右肩上。

她的右肩比左肩亮一格。

他把物理笔记本往前翻了一页。在边角画了一条线。然后停了笔。

下课铃响了。

她站起来，绕到讲台那边接水。回来的时候从后排过道经过。

离他不到一米。

小鱼挂件在书包拉链上摇了摇。

她走过去。

顾临风把画的线涂掉了。

晚自习前他去走廊尽头站了站。从那里能看到教室的窗户。从这个角度只能看到窗边一排人的头顶。他认出了她的马尾。

天还没完全黑。窗外的银杏枝像个剪影。

赵一鸣走过来。

"你最近老往走廊跑。"

"透透气。"

赵一鸣没追问。递给他一根棒棒糖。橘子味的。

"我妈塞我书包里的。太多了。给你一根。"

顾临风接过来。剥开糖纸。

橘子味。

晚自习她一直在做物理题。做到九点还没停。教室里人走得差不多了。他故意收拾书包收拾得很慢——把笔一支一支放回铅笔盒，把课本一本一本摞好。

她终于站起来。

他比她晚十秒出门。

走廊里的灯有点暗。她的影子在前面拉得很长。到楼梯口她往左拐。他往右——回男生宿舍。

走过转角的时候他又回头看了一眼。

银杏树还在窗外。

教室的灯灭了。
