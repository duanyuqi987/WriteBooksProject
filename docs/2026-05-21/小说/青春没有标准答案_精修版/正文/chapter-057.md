# 第057章 高二楼的反击



时间：2009年一月上旬，大雪继续下的下午



地点：两栋教学楼之间、食堂侧门



卷次：第三卷：雪落在两栋楼之间



2009年一月上旬，大雪继续下的下午，两栋教学楼之间、食堂侧门，砸在十班值日牌旁边的雪团先把这一天按进了十班的日常里。高二楼反击更猛，雪团越过空地，十班值日牌被砸歪。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

梁一帆不服，赵远航想带人绕到食堂侧门，段璟第一次主动拦住同学。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看砸在十班值日牌旁边的雪团，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

梁一帆也在这一天被推到前面。梁一帆的特点是爱运动，反应快，雪仗里第一个按不住手。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“他们砸咱们牌子。”梁一帆说。

“从食堂那边绕过去。”赵远航说。

“别绕，绕过去就真成打架了。”段璟说。

“段璟说得对。”常俊凡说。

“谁再下楼，下午自习全站后面。”陶建国说。

高二楼的反击更有准头，几个雪球从二楼斜着飞下来，落点都在高一楼门前那块最热闹的地方。

十班值日牌被砸歪时，梁一帆脸色一下变了，像被砸中的不是牌子而是班级脸面。

赵远航说可以从食堂侧门绕过去，高宇马上说我知道那条路，几个人眼睛都亮了一下。

段璟拦住他们时声音并不大，但他站在台阶中间，没有让开。

常俊凡看了他一眼，马上接话，说现在绕过去就不是雪仗，是找事。

陶建国赶到时，鞋底带着雪水。他没有先骂，只让所有人把手里的雪放下。

高二楼上还有人不服气，隔着空地喊了几句，被他们自己的老师吼回去。

那片空地忽然安静下来，只有雪还在下，像刚才那些喊声并不是它弄出来的。

高二楼的反击没有悬在空处。值日牌旁边散开的雪渣，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，高二楼的反击带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

陶建国进门时不一定立刻说话，鞋底在水泥地上一停，高二楼的反击带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在梁一帆快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。高二楼的反击越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。砸在十班值日牌旁边的雪团让他想说话，又逼他先低头看题。

梁一帆被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，高二楼的反击留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。梁一帆路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为高二楼的反击翻身，谁还在被窝里背单词，旁边人都听得见。

高二楼的喊声压过来，高一楼这边也不肯立刻低头，楼道里全是湿鞋印。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理高二楼的反击里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：砸在十班值日牌旁边的雪团、一只冻红的手、一句老师点名、一块还没扫干净的雪。

罗雨桐看见值日牌被砸歪，先伸手扶了一下再回头喊人。这样的细节没人会写进成绩单，却会留在同学之间。

赵远航想绕到食堂侧门，脚迈出去又被旁边人挡住。窗外风一过，纸页轻轻动了一下，像替谁把话咽回去。

陈柏言把雪从袖口里抖出来，才发现手腕已经红了一圈。这个小动作很快过去，却让教室更像一个真正在运转的班。

张若楠站在楼道口听高二那边起哄，脸上笑意慢慢收住。旁边的人没有特意看，可那一瞬间还是被段璟记住了。

刘青岚看见值日牌被砸歪，先伸手扶了一下再回头喊人。常俊凡没有接话，只把手里的事做完。

王子越想绕到食堂侧门，脚迈出去又被旁边人挡住。老师没有立刻点名，那一秒沉默反而让人更知道该收住。

高二楼的反击之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

梁一帆没有因此变成特别醒目的人，可在段璟眼里，梁一帆从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，高二和高一的玩笑开始有了年级对抗的味道，老师们往楼下赶。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
