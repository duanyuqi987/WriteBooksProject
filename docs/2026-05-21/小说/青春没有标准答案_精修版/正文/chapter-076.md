# 第076章 甲流晨检



时间：2009年九月初，高二开学第一周



地点：高二十班教室门口、操场入口



卷次：第四卷：分科以前的夏天



2009年九月初，高二开学第一周，高二十班教室门口、操场入口，门口那张甲流晨检体温登记表把这段日子推到十班面前。高二开学后，甲流晨检成了进班第一件事，十班在体温表和分科余波里重新集合。

高宇嫌晨检麻烦，梁一帆借机起哄，常俊凡提醒他们别把别人的担心当笑话。这不是突然降下来的大事，而是从早读、课间、饭卡、家长电话和后墙红榜里一点点长出来的。

段璟坐在教室里，常常先看见常俊凡的背影，再意识到自己其实该看卷子。从这个学期开始，这种走神不再只是喜欢，也会影响一道题、一次排名和一个选择。

常俊凡没有变成谁的答案。她会犹豫，会烦，会被家里问得沉默，也会在别人替她下结论时把话说得很硬。

高宇也在这件事里显出自己的位置。个子高，爱出风头，物理不错但粗心，最怕被当众说幼稚。十班从来不是只由两个人组成，它是四十多个人一起挤出来的青春。

“一天量两回，有必要吗？”高宇说。

“你不怕，不代表别人家里不怕。”常俊凡说。

“高宇体温正常，嘴温偏高。”梁一帆说。

“先签名，后面还排着。”段璟说。

“晨检不是形式，谁发热先报告。”陶建国说。

晨检表贴在教室门口，表格上方写着日期、姓名、体温和是否咳嗽。

体温计在同学手里传来传去，有人嫌凉，有人嫌慢，还有人故意报一个夸张数字逗笑。

陶建国站在门口盯着，谁也不敢真把这事当玩笑。甲流两个字让家长比学生更紧张。

常俊凡负责帮忙核对名字，笔尖在表格上一行一行往下走，没签的人会被她叫回来。

段璟去开窗，早晨的风从操场那边吹进来，带着一点消毒水和湿土的味道。

高二刚开学，课程表比高一密，晨检又把每天开头切成一道规矩的门槛。

有人咳一声，周围立刻安静半秒，然后又假装什么都没听见。紧张并不总是大喊大叫，也可能是忽然变礼貌。

第一节课铃响时，晨检表已经写满，教室终于回到课本、粉笔和没讲完的题里。

陶建国这段时间管得比高一更细。不是因为学生忽然变坏，而是因为分科、晨检、国庆活动和期中考试挤在一起，谁一松就会被推着往后退。

高宇被叫到名字时，先看了一眼周围同学，再看老师。这个小动作让段璟发现，班里很多看起来不在乎的人，其实都怕被公开放到台面上。

封丘县的校园生活没有电影里那么整齐。校门外有卖烤红薯的，有修自行车的，有家长骑电动车来送饭，也有学生在路边背单词。

黄河故道那边的风有时会吹进县城，吹动校服袖口，也吹得人想起更远的地方。可更远的地方暂时还只是试卷最后一道大题旁边的空白。

国庆六十周年的新闻、甲流晨检的表格、分科确认的回执，都没有替他们解决任何一道题，却让他们知道，学校外面的时间也在往前走。

宋乔会把女生那边的情绪说出来，付小川会用玩笑把男生那边的沉默撬开，周明澈则常常在最乱的时候把表格、名单和班费重新理一遍。

段璟有时羡慕这种确定。他自己更像一张写了太多草稿的纸，明明知道下一步该写什么，却还是忍不住看常俊凡会不会也停在同一个地方。

常俊凡并不总是回头。她更多时候只是把自己的事情做好：改题、接水、记账、扶凳子、把别人的话挡回去，然后继续低头写下一行字。

孟凡在晨检表上写下体温，写完还回头确认小数点有没有错，有人把椅子往里收，木腿擦过地面发出短短一声。窗外风一吹，纸角轻轻动了一下，像有人把话又咽回去。

夏清和把体温计递给后面的人，指尖碰到冰凉的玻璃，同桌抬头看了一眼，又像什么都没看见一样低头。十班的热闹还在，只是每个人都比高一时多了一点分寸。

许照临听见有人咳嗽，手里的笔停了一下又继续签名，粉笔灰落在袖口上，拍了两下也没有拍干净。老师从门口经过，所有人的声音立刻低了半截。

任思齐把教室窗户推开一条缝，冷风立刻吹动前排试卷，小卖部那边的铃声远远传来，很快被上课铃盖住。这样的细节不会出现在成绩单上，却会留在很多年后的记忆里。

丁遥被老师叫回门口补登记，脸上有点挂不住，红笔帽没有扣紧，轻轻碰到桌面滚了一圈。这个动作很小，却比一句大道理更能说明当时的慌。

秦越在晨检表上写下体温，写完还回头确认小数点有没有错，走廊里有人喊了一声报告，很快又安静下去。旁边的人没有追问，教室里的翻书声把尴尬遮住。

顾南音把体温计递给后面的人，指尖碰到冰凉的玻璃，黑板右上角的日期被擦得只剩一点白影。常俊凡看见了，也没有立刻说什么，只把自己的题继续写完。

甲流晨检之后，十班没有立刻发生巨变。真正的变化通常不是一声响，而是某个同学开始少和旧同桌一起吃饭，某个名字开始出现在另一张名单上。

高二的第一周没有想象中的新鲜，更多是规矩、体温、座位和突然变重的课程表。段璟把这件事记得很清楚，不是因为它多戏剧，而是因为它让他第一次看见选择有后果。

他也慢慢懂得，喜欢不能替常俊凡做决定，也不能把她的每一次沉默都解释成和自己有关。

常俊凡要走的路，首先属于她自己。段璟若真想靠近，就得先学会站在不挡路的位置。

高二就是这样来的：没有高一开学那样热闹，也没有后来倒计时那样压迫，却把许多关系悄悄推到更真实的位置。

晚自习结束时，门口那张甲流晨检体温登记表还留在教室里，像一枚小小的钉子，把这一天钉在十班的时间线上。

灯一盏一盏暗下去，楼道里传来脚步和水瓶碰撞声。段璟走到门口时回头看了一眼，发现常俊凡还在收书。

他没有催，也没有等得太明显。只是把书包带往肩上提了提，跟着人群往楼下走。
