# 第224章 重回县一中旁观

所属部卷：第五部：逝水有灯
时间：十年后秋天
地点：县一中→异城

## 诗歌插页 P224：原创现代诗

我把旧螺丝孔放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

夏知遥没有回一中。

她在陆允川发朋友圈说"今天回母校"的时候，正在另一个城市的会议室里给客户演示方案。会议室在写字楼二十三层，落地窗能看到大半个城市的天际线——高架桥、滨江路、远处还在施工的几栋新楼，塔吊的臂在灰蒙蒙的天空里缓慢旋转。演示到第四页的时候，她的手机震动了一下，余光扫到那条动态，手指在触摸板上顿了一拍，然后继续翻页。

"怎么了？"客户问。

"没事。系统卡了一下。下一页——这是第三季度的预算分配。"

演示结束以后，她一个人坐在会议室里，把高跟鞋脱了，光脚踩在酒店的地毯上。厚地毯，脚趾踩上去陷进去大概一公分。她把手机翻出来，重新点开那条朋友圈。

陆允川发的图：校门口的桂花树。文案只有一句话——"旧螺丝孔还在。"

她见过那几个螺丝孔。高三那年的某个下午，她路过校门口，陆允川蹲在门牌下面，说在捡掉下来的钥匙。她当时觉得奇怪——校门口的门牌下面能掉什么钥匙？后来她才明白他只是想找个借口在校门口多待几分钟。

那时候他应该是在等人。等苏晚枝。

她点了个赞。

然后又点进去看了一遍。

鲁萱给她发了条微信："你今天见到高中同学了？"

"没有。就是刷到朋友圈了。"

"那个陆允川的？"

"对。"

"他还在写小说？"

"还在。上次面馆那章他发给晚枝看了。晚枝说写得好。"

"你看了吗？"

"看了。写的就是五年前面馆发生的事。一字不改的那种。"

"一字不改，那不怕当事人有意见？"

"当事人已经没意见了。"夏知遥把腿蜷起来，整个身体陷进会议椅里，"他们和好了。"

"什么叫和好？"

"就是——不是仇人了。也不是恋人。就是——可以点赞朋友圈、偶尔聊两句、说话不用再绕弯子的关系。"

"那你觉得这种关系好还是不好？"

夏知遥想了很久。窗外是这座城市的CBD，玻璃幕墙反射着下午的太阳，把对面楼照成一个巨大的光斑。

"挺好的。"她最后说，"比以前任何一个阶段都好。"

"为什么？"

"因为不累。"

酒店楼下的咖啡厅里，她点了一杯美式，不加糖。等咖啡的时候，她打开手机备忘录，找到了一篇很久以前写的日记。不是她的日记——是她帮叶澄整理旧资料时无意翻到的一页摘抄。摘抄的内容是某次诗社集会后叶澄写的随笔：

"陆允川今天在会上读了首诗，说旧日子像灯。大家没太听懂，但都鼓了掌。苏晚枝没来。他没有提她名字，但每句话都在往她身上靠。我坐在第三排，觉得这个人以后要么成为一个很好的作家，要么成为一个很好的人。也可能两者都是。"

夏知遥把摘抄关上，咖啡端上来了。美式的苦味很淡——这家店的豆子是浅烘的。

她忽然想发一条消息给叶澄。她们有一阵子没联系了。

"在？"

"在。怎么了？"

"今天在朋友圈看到陆允川回一中了。"

"我知道。程砚跟我说了。他们两个一起去的。"

"你觉得他回去是为了什么？"

"为了找材料。他小说快收尾了。可能觉得需要回一趟原点。"

"找到了吗？"

"程砚说他找到了一组旧螺丝孔。"

"螺丝孔？"

"对。门牌下面的。旧门牌拆了换新的，螺丝孔没填。"

夏知遥看着"螺丝孔"这三个字。她觉得这就是陆允川——他能在一整面刷了新漆的墙上发现那两个没填的旧螺丝孔。而大多数人经过的时候只会看到新的门牌。

"你觉得他会把螺丝孔写进书里吗？"

"百分之百。"叶澄秒回。

夏知遥关了聊天窗口。咖啡喝了半杯。她站起来，穿上高跟鞋，走出咖啡厅。玻璃幕墙上映出她的影子——一个穿着职业装、拎着公文包的女人。和十年前那个穿着白校服、站在走廊上背课文的女生已经完全不同了。

但那个女生还在。不是留在旧螺丝孔里。是留在她没有删掉的备忘录里。

手机又震了一下。是工作群里催下一份方案的deadline。

她回了：收到。

高跟鞋接着往前走。酒店的旋转门推出一阵凉风。
