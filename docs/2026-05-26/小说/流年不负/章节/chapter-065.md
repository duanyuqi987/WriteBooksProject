# 第065章 物理题

时间：2013年6月8日晚上

地点：月湖边小路

卷次：卷四：夏夜湖光

他们走的那一圈，前五分钟谁都没说话。

湖风把柳条吹起来。柳梢擦过水面。水面上的月亮碎了一下，又合上。碎的时候月亮变成了几十片小光点——在水面上跳了一下。合上的时候恢复成一个完整的圆——比原来偏了一点。风又来了。又碎了。

她的白布鞋踩在石板上。啪嗒。啪嗒。他穿了双旧运动鞋——鞋底磨得差不多了，走在石板上闷闷的。两种脚步声交替着。布鞋轻，运动鞋沉。轻的在前，沉的在后——差半拍。他试了一下调整步伐跟上她的节奏。踩了两次之后发现跟不上。她的步子比他快一点点。放弃了。

顾临风在心里数了大概九十步。湖边的石板路他走过很多次——以前一个人走的时候数步子。现在不用数了。但还是在数。习惯。数到第九十一步的时候。他开了口。

"你物理最后一道大题做出来了吗。"

她愣了一下。转过来看他。眼睛里有光照到——水面的反光。亮晶晶的。

然后她笑了。

"没有。太难了。"

她的笑很轻。不是哈哈大笑。是那种忽然被戳中笑点、但只是在安静环境里才露出来的笑。眼睛弯了一小下。肩膀也跟着轻轻耸了耸。

"那道题——电磁感应——我想了十五分钟。然后放弃了。"

"我也是。"

"你写到第几步。"

"写了一半。把公式列了，求了一个问。"

"我也是。"她说。"列了公式，写了两行。第三行写不出来。"

"第三行要转换一下。"

"后来想通的？"

"出了考场才想通的。"他说。

她又笑了。

"你出了考场还在想物理题？"

他嗯了一声。没说是因为在找她的时候顺便想的。

他们继续走。

"你考得怎么样。"她问。

"还行。"

"还行是好还是不好。"

"589——估的。"

"挺准的。"

"你呢。"

"612吧。也不一定。英语作文不知道扣多少。"

"不会扣太多。"

她看他。"你怎么知道。"

他顿了一下。"你英语一直很好。"

她没接这句话。走了几步。然后说："你一直在看。"

不是问他。

是陈述句。

他把手塞进裤子口袋。裤兜里面有一个包装纸——薄荷糖——上次赵一鸣塞给他的。还没吃。他碰了碰那个糖。没拿出来。

又走了几步。

"你选文还是理。"她问。

"理。"

"我也是。"她停了停。"但我想当语文老师。"

他偏过头看她。

她继续说："所以我要读师范。教育学。楚州师大。"

"楚州。"

"嗯。楚州。"

"我也去楚州。"

她转头看了他一眼。没问为什么。

但是她的脚步慢了半拍。然后恢复正常。

湖边的灯越来越稀。他们走到了南岸。南岸比较偏。灯柱之间隔着五十米。隔在中间的地方就靠水面的反光照路。

"你呢。"她问。"想去哪里。"

"楚州师院。"

"学什么。"

"中文。"

她点点头。像在确认什么。

"中文好。"她说。"你想当老师吗。"

"语文老师。"

她停了一下。转过来正眼看他。看了几秒。

"怎么了。"他问。

"没什么。"她转回去了。"就是觉得——"

"觉得什么。"

"觉得挺合适的。"

一只青蛙从路边的草丛里跳到水里——噗通。水花很小，但声音在安静的晚上很脆。他们都往声音的方向看了。

走过了几十米的暗处。

前面有一个路灯。路灯下放了一张木椅——油漆剥了，露出底下原木的颜色。椅子面向湖水。椅背斜斜的。

"坐会儿？"她问。

"好。"

她先坐下。拉了拉裙摆。他坐在她旁边——隔了一个拳头的距离。

从后面看过去：椅子。两个人。灯光在他们头上。湖在他们面前。湖面上一层一层的波纹。柳条垂在画面右边。

风吹过来的时候她头发飘了一下。发梢碰到他的手背。

他把手挪开了几厘米。

"物理题之后呢。"她说。

"怎么了。"

"高考完了。除了对答案，还想聊什么。"

他说："想聊高中。"

"聊高中什么。"

"随便。什么都行。"

她往后靠在椅背上。"聊什么都可以——三年你大概都看到了。"

这句话他不是马上接的。

他看着湖面有十几秒。蛙鸣声很近。大概在南边的草丛里。

"你中午吃饭先吃素的包子，再吃肉的。"

她没说话。

"你写字的时候这个小指——"他在自己的手上比了一下。

她把右手拿起来。看了一眼。小指微微翘着。

"奶奶教的。她教我写毛笔字。说小指要顶住。后来写硬笔也改不掉。"

"挺好看的。"

"同学说像古代小姐。"

"不是。"

她偏头看他。

"不是。"

她偏头看他。

他盯着湖面。水上有一个漂浮物——一个饮料瓶盖。绿色的。在波浪里一上一下。他盯着那个瓶盖——盯得很认真。其实他根本看不见什么细节。就是需要一个落点。

"同学说错了。"

她静了大概五秒。右手的手指在膝盖上慢慢地敲了一下。又敲了一下。第三下停了。她把手放回腿侧。

"赵一鸣是你同桌？"

"嗯。"

"他话多吗。"

"多。"

"他一直跟你坐？"

"高一下学期开始。后来调过几次。他一直坐我旁边。"

"他对你挺好。"

顾临风点了头。

"他今晚一直在给你使眼色。"

顾临风的手停住了。

"看到了？"

"看到了。"她说。"你们大概觉得我看不到——赵一鸣第二次给你眼色的时候差点把饮料瓶都甩出来了。"

顾临风用手扶了一下额头。

然后他也笑了。他很少笑。笑的时候声音低低地从鼻子出来——哼了一下。嘴角动了一下。

"你怎么什么都看到了。"

"一个人在你后面看了三年。"她说。"她的后脑勺是能看到很多东西的。"

她这句话一说出口，两个人都安静了。

风从湖面灌过来。凉飕飕的。八点多的湖风已经有点凉意了——六月晚上。

"其实我也在看你。"他说出口之后自己都顿了一下。他没想到自己会说这句话。这句话在他脑子里转了三年。今天晚上忽然就滑出来了——像一个放在桌边的东西。一下子就滚下去了。捡不回来。

她把头发别到耳后。这个动作今晚他看到了好几次。每一次都不一样。第一次是在防风灯下——手指夹住碎发。第二次是在湖边——风吹起来的碎发。这一次——她说"我知道"的时候——手指顺着耳廓滑下来。很慢。比平时慢。

"我知道。"

然后她站起来。椅子上的漆皮掉了一块在椅子上。她坐过的地方——椅面上有一个微弱的温度。

"再走走吧。"

他也站起来。

又走在石板路上。这次脚步声同步了。轻和沉叠在一起。

从湖心亭走到九曲桥。从九曲桥走到南岸广场。从广场绕回北岸。

月湖被他们快走完了。

"你想当什么样的老师。"

"站在讲台上不会被学生讨厌的那种。"她说这话的时候嘴角动了一下。"我妈是老师。她每天回家都很累。有时候学生不听话。她在房间里哭过一次。我在门外听到的。"

"你妈妈是小学老师？"

"嗯。教语文的。"

他们走到北岸的时候聚会的人已经散了，只剩赵一鸣在收拾。赵一鸣把塑料布卷起来——卷成了一个卷。地上的饮料瓶捡了五六个——夹在腋下。防风灯提在手里。一个人在那儿忙。看到他们回来——抬头。嘴张了一下。又闭上了。继续卷他的塑料布。

苏晚晴往路灯那边走。走了几步。停下。回头。

她站在路灯下。灯光从头顶打下来。头发边缘全是光。她整个人像被描了一个光圈。

"你们经常来月湖？"

"来过几次。都是赵一鸣组织的。"

她点了点头。然后说："下次再来。"

不是问句。是陈述句。

然后她走了。这次没有回头。许若琳在远处的面包车旁边等她——车门开着。里面亮着灯。她弯腰进去。门关上了。面包车发动的声音在夜里很响——突突突突。然后渐远。尾灯消失在转弯处。

顾临风在路灯下站到面包车尾灯彻底看不见为止。
