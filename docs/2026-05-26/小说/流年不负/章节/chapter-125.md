# 第125章 照片

时间：2018年3月

地点：楚州，手机

卷次：卷七：流年未断

张阿姨发来了照片。

短信里附了一张图。他点开。加载圈转了两秒。

照片亮了。

苏晚晴站在楚州一小的校门口。背后是校门——浅红色的砖墙，铁栅栏，门柱上挂着学校的牌子。天很蓝。大概是去年秋天拍的。校门两侧的银杏树叶子黄了一半。

她没怎么变。

马尾还在。浅蓝色衬衫——跟高中时候的颜色几乎一样。黑色长裤。肩上的包是米白色的——不大。左手拿着一个文件夹，右手垂在身侧。

比高中时瘦了一点。脸上的轮廓比以前清晰了——颧骨和下巴的线条。但眼睛还是亮晶晶的。

笑容比以前松弛了。高中时她笑的时候总是抿着嘴，好像怕笑得太大。这张照片里她的嘴角是自然弯的——不是在拍照的人面前刻意控制的那种笑。

从容了一些。她的站姿不紧绷——重心在左脚，右脚微微往前，肩膀是放松的。眼睛看着镜头。目光跟高中时候不一样——那时候她看人总是很快移开，像怕被人发现自己在看什么。

现在她的目光稳稳地落在他身上——落在屏幕上，落在他的眼睛里。

他看了很久。

屏幕暗了一次。他按亮。又暗了一次。又按亮。

手指悬在屏幕上。滑到照片的左侧去看校门口的细节。校名是铜字，有点反光。她站的台阶上有个小水洼——大概是前一天下过雨。她穿的是平底鞋——黑色的帆布鞋。

跟高中时候一样的鞋。

他把照片缩小。又放大。又缩小。

然后他把手机放在桌上。

手指交叉握在一起。手背上的青筋比高中时候明显多了。代课这两年，捏粉笔捏的。

他把手机又拿起来。

打开相册。新建了一个文件夹。文件夹名字：一。

只有一张照片。

他退出相册。打开通讯录。翻到张阿姨。打了一行字——"照片收到了。谢谢张阿姨。"发出去。

然后他把手机翻过来，屏幕朝下放在桌上。

手指在桌面上轻轻敲了两下。

停了。

过了一会儿。他又把手机翻过来。点开照片。

她站在校门口。阳光很好。银杏叶黄了一半。她的马尾安静地垂在肩上。

八年。

他记忆里的她跟这张照片重合了。高一开学的那个九月——她从前面走进教室，浅蓝色T恤，深蓝色牛仔裤，扎马尾。九月阳光照在她侧脸上。她的头发在光里是褐色的。她往教室里扫了一眼，然后走向第二排靠窗的位置。

那时候他的心脏好像漏跳了一拍。但他没有告诉任何人。甚至没有告诉自己。

现在他看着这张照片——同一个姑娘。同一个马尾。同一个颜色的衬衫。同一个——

但他不一样了。

他也瘦了。比以前高了几厘米。肩膀宽了一些。胡茬硬了。手指上有了粉笔的茧。眼睛底下有青色的印子——失眠好几天的结果。

他不知道她还认不认得他。

照片里她的眼睛亮晶晶的。看着镜头。看着八年后的他。

他把手机屏幕按灭。

房间暗了。

窗外那棵樱花树还在。花瓣快落完了。剩下的几朵在枝头轻轻晃。

他又按亮屏幕。看了一眼照片。然后把手机放进口袋里。

站起来。

走到窗前。

樱花落了满地。白色的花瓣铺在灰色的地砖上。明天扫地的阿姨会来清理。

他靠在窗框上。手插在口袋里。手指碰到手机的边缘。

周六下午三点。

一茶一座。

她坐在窗边。

他推开门。

这是他想了八年的画面。

但明天——不是"想"。是真的。

他站在窗前。窗外的路灯亮着。对面楼的窗户里有人在拉窗帘——浅粉色的。拉上了。

他低头看了看自己的手。左手食指上有一道粉笔划的印子。他用拇指擦了擦——没擦掉。

口袋里的手机沉甸甸的。

里面有一张照片。

她站在一小的校门口。阳光很好。马尾被风吹起来一点点。她看着镜头。表情平静。

好像在说——好久不见。
