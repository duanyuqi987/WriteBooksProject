# 第099章 分手

时间：2015年1月

地点：楚州师院

卷次：卷五：各自南北

一月十五号。期末考试完第一天。楚州下了一场雪——不大——薄薄一层白——盖在梧桐秃枝上。

周雨发来消息——"今天下午有空吗——老地方。"

老地方——图书馆门口台阶——他们第一次说话的地方。他到的时候她已经在那里——深灰大衣——红围巾——手里拿着那本诗集——海子《活在珍贵的人间》——他写过"生日快乐"的那本。

他走过去。站在她面前。两人中间隔了一级台阶——她在上面——他在下面——视线齐平。

她抬起眼睛。没有哭。眼下那几颗雀斑在冬日冷光里比平时深了一点。

"顾临风——我们分手吧。"

他看着——没接话——等她说完。

"你心里有别人。"

手冻得有点僵。她把诗集递过来——他没有接。诗集在两人之间的空气里悬着——封面上落了雪。

"跟你在一起的时候——你像在出神。我不知道你在想什么——但我知道你想的不是我。"声音很稳——没拔高——没颤——像准备了很久——默念过无数遍。

"一开始以为你只是慢热。三个月——还是慢——慢到能猜到你出神的时候在想谁。"

他没有反驳。

"对不起。"他说。

"你不用对不起。你没做错什么。你对我挺好——生日礼物——陪自习——陪逛操场——。"她把诗集放在台阶上——放了以后退了一步——好像那个动作需要一些距离。台阶上薄雪——诗集放在上面——封面上立刻有了一层白。"但你的心不在。"

周雨往后退了一步。围巾往上拽——盖住下巴。她呼出的白汽在红围巾边缘散开。被冻得有点发红的皮肤上——有很细的白毛——是围巾蹭上去的丝。

"想过跟你说'你改一改'——但这不是'改'的事。你改不了——你根本没打算改。"

"对不起。"他又说了一次。

"我走了。"

转身——走下台阶——靴子踩在薄雪上——咯吱——咯吱——咯吱。走了一半——身体顿了一下——像拉了什么东西在身后。她没有回头。红色围巾在灰白背景里一点一点变淡——变模糊——最后拐进教学楼走廊那头——不见了。

他捡起诗集。书壳被雪水打湿一小块——扉页上"生日快乐"洇成模糊的蓝色。周雨在最后一页用铅笔写了字——他之前没看到——字很小——轻的——像怕用力太大把纸划破：

"我知道你写了很多诗。但不给我看的那些——才是真的。再见。"

把诗集夹进书包。站在台阶上——雪落在肩上——落在诗集封面——手握着书——很凉。

回去路上。梧桐道在雪下染成黑白。脚印在雪地上一深一浅。

512只有他一个人。阿坤电脑黑着——李博书架码得整整齐齐——小伍床头贴了家里寄来的照片——四个人站在砖楼前——他妈他爸他妹他家的狗。蓝色格子被——灰床单——枕头下压手机。

坐椅子上——没开灯——坐了半小时。窗外的天从灰变成深灰——他坐在暗里——没有动。楼道声控灯自己亮了——有人跑过去脚步声啪啪响——开门——关门——安静了。

拿出诗集——周雨的铅笔字——很轻——"再见"。书放桌上——起身洗脸。走廊水池牙膏泡沫——跟去年一样。镜子里的自己没有表情——眼皮有点肿——大概是冻的。水龙头——冷水——往脸上泼几把。擦脸时棉袄左胸口那块——凉——周雨放诗集的位置。

回宿舍——开灯。阿坤回来了——"今天怎么这么早。""嗯。"阿坤看了一眼他的脸——然后没有问。

李博回来得更晚。进门的时候帽子上落了一层雪——他摘下来在外面抖了抖——雪屑在走廊里飞散。"晚上吃了吗。""不饿。""怎能不饿——我给你带了包子。"李博把塑料袋放在他桌上——韭菜鸡蛋的——跟食堂早上的一样。塑料袋里还有一杯豆浆——豆浆已经不烫了，温的。

"谢了。"

"不谢。"

李博坐下来打开台灯——翻开一本没看完的书。小伍还没回来。宿舍里只有翻书页的声音和阿坤压低了的键盘声。安静得有些不正常——平时阿坤的键盘是啪啪响的，今晚他在用红轴模式——声音软了很多。李博看了一会儿书，头也没抬地说——"外面雪停了。"

顾临风站起来走到窗前往下看。楼下的路灯还是橘色的。灯光下的雪地不是白色的——是淡黄的。梧桐树的光枝干上挂着一层薄雪——那些雪在灯光下有几处反着亮晶晶的光。地上有一排一排的脚印——往不同方向走的——往图书馆方向——往食堂方向——往女生楼方向。他看了几秒。视线从女生楼方向移回来。关了窗户。风从窗缝里灌进来——把桌上的纸页吹翻了一页。

阿坤摘了耳机。耳机挂在脖子上——里面还有游戏的BGM在响。他站起来——走到顾临风的桌边——把一罐可乐放在他面前。"冰的。"他说。"不用。""喝——又不是药。""不是药才喝。"他拉开拉环——呲——气泡冒出来——白色泡沫顶在罐口。喝了一口——冰的。气泡在喉咙里炸开。"谢了。"阿坤点了下头——什么也没说——回去继续打游戏。键盘声从软的红轴换回了青轴——啪啪啪——又恢复了他惯常的节奏。

顾临风坐在桌前。包子吃了一个。豆浆喝了两口。手指在桌上敲了两下——然后停下来。视野落在桌角那本诗集上。封面是旧的——黄底黑字——"活在珍贵的人间 海子著"。翻到最后一页——周雨的铅笔字——"再见。"字非常轻——有些笔画已经模糊了。他看了几秒。把书合上——放进抽屉里——推到最里面。抽屉的木头摩擦声低沉。他站起来——去刷牙。水池那——还是那面有斜裂缝的镜子。镜子里的自己没太多表情。他盯着自己的眼睛看了几秒。然后低头吐掉牙膏沫。水冲下去——白沫在水池里转了几圈——被排水口吸走了。

上床。躺下。枕头下手机硌着。拿出来——按亮——收件箱空着。通讯录里S栏——苏晚晴。他看了一秒。然后按灭。天花的板——四条裂缝。呼吸慢慢变平。窗外的光在窗帘上印着一片模糊的橘。下雪之后的世界特别安静——连摩托车的声音都消失了。他在这安静里——什么都没有想——也什么都在想——然后慢慢——睡着了。
