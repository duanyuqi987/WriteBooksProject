# 第105章 朋友圈

时间：2015年12月

地点：楚州师院宿舍

卷次：卷六：平行轨道

十二月了。楚州入冬。

宿舍没有暖气。晚上冷得手指尖发僵。窗玻璃上结了一层薄薄的雾气——白天晒进来的那点热量到了夜里全散了，空气干燥而扎人。李博买了个暖水袋。灌满热水，抱着睡觉。暖水袋是橡胶的，灌满水后鼓鼓的像半个西瓜。

顾临风没有。他裹着被子看书。被子是旧的蓝格子被——他妈缝的。棉絮洗过几次，有些地方变薄了，有些地方团成了块。他把最厚的那块挪到胸口。

手机放在枕头旁边。屏幕黑着。旁边是耳机线，缠成一团。

考研政治背到第三章——马原。生产力决定生产关系。他背了三遍。第一遍划出来——荧光笔，浅黄色。第二遍默读——嘴唇不动，但嗓子后面在发音。第三遍闭眼回忆一遍。然后翻到下一页。书页有点皱了——被翻了很多次，折角被拇指磨出了毛边。

窗外的风很大。呼呼地灌进玻璃缝里，玻璃轻微地振着。窗帘在动——深蓝色，不太厚，风从缝里挤进来的时候布料鼓起来又瘪下去。他把被角又掖了掖。脚趾在棉袜里蜷着。

十点十五。隔壁宿舍有人放音乐——周杰伦的《晴天》。前奏的钢琴从墙壁传过来——被砖墙滤掉了一层高频，变得闷闷的。李博已经睡了。呼吸声不太响但有节奏。暖水袋从被窝里滑出来掉在地上——咚的一声——他没醒。暖水袋在水泥地上慢慢变凉。

顾临风背完五页。合上书。书脊在枕头边碰了一下——轻轻的一声"啪"。

拿起手机。

他本来只是看时间。十点十六。

然后他点开了微信。

朋友圈。许若琳。

九张照片。

隔着屏幕能看到九宫格的小图。最中间那张有一个白影子。他的拇指在屏幕上顿了一下。

第一张：一棵圣诞树。大概是商场里的。装满了彩灯。树顶上的那颗金星歪了——大概是被人碰到过，没扶正。

第二张：一盆火锅。红汤的。筷子在里面捞东西。锅底的红油泛着光。旁边摆了七八个小碟子——毛肚、黄喉、藕片、土豆。碟子边缘沾了油渍。

第三张：三个人在火锅店端着饮料碰杯。橙汁。许若琳在最左边。中间是另一个不认识的女生。右边的人是——

第四张——

他停了下来。

第一张——是这组里最大的一张。合唱照片。苏晚晴和许若琳。苏晚晴穿着白衬衫，最上面两颗扣子没扣——领口露出一小截锁骨。马尾。站在操场上。阳光很亮。亮得她脸上的纹路都被冲淡了。她在笑。不是高三毕业照上那种收着的笑——嘴唇抿到刚好露出一点牙。是松开的。嘴巴张着，眼睛弯弯的。眉毛往下压了一点——不是皱眉，是笑到眼睛必须合拢时眉毛自动跟着动的那个角度。

她旁边是一群孩子。穿校服。在做操。灰白相间的校服——上衣有点大，袖口盖住了半个手掌。

楚州一小操场。红色的塑胶跑道，绿色的假草坪。阳光被塑胶跑道吸收了一部分——跑道上的颜色发暗。假草坪上有些绿色的碎屑被踩得翘了起来。

第五张。苏晚晴站在大课间的队伍前面。领操。双臂平伸。一个标准的侧平举。后面的孩子们跟着做——有的手举高了，有的慢了半拍——有一个小男孩在队伍最后面完全做错动作——该蹲下的时候他在跳。

第六张。教室里。苏晚晴蹲在一个小女孩旁边，指着她桌上的作业本。嘴动着——大概在讲题。小女孩咬着笔头。笔头上的齿痕清晰可见。苏晚晴的膝盖抵着课桌腿，白色的裤子上蹭了一点粉笔灰。

第七张。黑板。黑板上写着"祝苏老师生日快乐"——粉笔字，歪歪扭扭的，一看就是孩子们写的。"祝"字特别大，"日"字被写成了"目"。旁边画了一个蛋糕。

第八张。苏晚晴捧着一束纸折的花站在黑板前面。眼泪在眼眶里——没掉下来。水光在眼睑边缘反着光。她在笑。又哭又笑的那种。嘴唇弯着，但还在抖。

第九张。所有人合影。她站在最中间，被孩子们围着。一个女生在背后给她比了个兔耳朵——两根手指竖在她头顶上方。

顾临风把第四张照片放大。

她的笑容比以前松弛了。高三那年在考场外面她笑过一次——收着、浅的、嘴弯了一点点。她从楼道里出来的时候嘴角往上提了一下然后很快收回去了。他站在花坛后面看到了。现在不是。现在她站在那里，阳光铺在脸上，笑得像一个没有心事的人。不是"像"——是真的在那一秒里没有心事。

他的拇指在屏幕上来回滑动。放大、缩小、再放大。放大到她的脸占据了整个屏幕——鼻梁上一小块被阳光照亮的光斑。缩小回到整个操场的全景——她又变成了一枚白色的点在红绿相间的操场上。

许若琳的配文：

"去楚州一小找晚晴蹭教师食堂。伙食不错！看到她被一群小萝卜头围着叫苏老师，老母亲般的欣慰。晚晴真的适合当老师——温柔又有耐心。生日快乐呀我的苏老师。"

他在底下看到了几条点赞。赵一鸣。两个高中同学。苏晚晴本人没有评论——大概是忙。或者看到了，没有回。

他看了很久。久到手机屏幕暗下去。他又点亮。又看了一会儿。

没有点赞。

拇指在"赞"那个小红心的左边停了一下。然后滑过去了。他用拇指往上推了一下屏幕——把朋友圈刷新了。许若琳的动态被推到了上面。然后他往下拉回来，又看了一遍。

他把手机关掉。翻过去。放在枕头旁边。屏幕朝下。枕头边上有一小块压出来的凹痕——他晚上常常把手机放在那里。

窗外风还在刮。窗帘一下一下地鼓起来。风从窗缝挤进来——呜呜的声音，很高很细。

他躺在黑暗里。天花板上那条裂缝应该还在——太黑了看不见。但是他知道在哪里。从灯座往墙角延伸。两年了。每次熄灯之后他都会往那个方向看。

十二月。楚州的冬天很冷。

她的手大概也冷。站在操场上带操。初冬的风吹过来——楚州的风是从北边来的，穿过楚河的时候带了一层水汽，然后从城西的空地那边灌进一小的操场。白衬衫下面应该加了保暖衣。她领操的时候手伸得很直——照片上看得出来。手指并拢，小指那个改不掉的习惯——微微翘着。

那些孩子叫她苏老师。声音大概很响——一大群小学生喊起来，操场上的回音让"苏老师"三个字在不同的离墙之间弹来弹去。

她收到了纸折的花。收到了写歪了的"生日快乐"。收到了一个五年级男生偷偷塞给她的糖。糖大概是草莓味的——小孩子的口袋里最常装的就是草莓味水果糖。那个男生塞完大概跑了——往教室门口的方向，边跑边回头看她的反应。

她大概很开心。开心得在被一群孩子围在中间的时候，忘了自己二十三岁。忘了一个人在这里教书。忘了一些事。

他的嘴角动了一下。在黑暗里。不是笑。嘴角往外拉了一点点又收回来。脸上的肌肉自己完成的这个动作。他自己没有察觉。

然后翻了个身。脸冲着墙。墙上那个小孔还在——在黑暗里看不到。但手能摸到。他用食指尖碰了一下那个小孔。冰凉的水泥。粗糙的边缘。像一粒硬掉的米。

手机里那些照片还在。那个放大过的笑容还在。

他没有点赞。

那天晚上他做了个梦。梦见了楚州一小的操场。阳光很亮。操场上有广播体操的音乐——是那种老式的扩音喇叭放出来的，声音有点破。她站在队伍前面带操。穿着白衬衫。马尾。他在校门口站着。隔着铁栅栏。她没有看到他。然后下课铃响了。孩子们涌过来——灰色的校服、白色的校服、红领巾。好多。把栅栏挡住了。她不见了。他往旁边挪了挪想换个角度。但铁栅栏一直接到路的尽头。

早上醒来他洗了把脸。水龙头的水很冷——冬天的自来水，手伸进去觉得手指骨都在疼。他往脸上泼了两把。抬起头。镜子里的脸有点白。下巴上有几点水珠往下滑。眼睛下面有点青。他擦了擦脸上的水。毛巾是旧的——有点硬，擦在脸上沙沙的。

背上书包去了图书馆。书包的拉链没拉好——露出来一截考研政治的封面。他边走边把拉链拉上。拉链头有点卡——他用手指把夹在里面的布扯出来。
