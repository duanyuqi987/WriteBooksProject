# 第069章 墙角的试卷旁观

所属部卷：第二部：六月以后
时间：高考后第二天
地点：夏知遥家中

## 诗歌插页 P069：原创现代诗

她也在整理书，
把试卷分成两摞。
一摞卖掉，
一摞先留着。
未来还没来，
但房间已经催她
腾出一个角落。

夏知遥高考后第二天也在整理书。

她家的房间不大，书桌靠着窗，下面塞满了旧卷子。母亲说趁有空收一下，别等通知书来了还乱成一团。夏知遥蹲在地上，把试卷分成几摞：要卖的、暂时留的、可能还要查分用的。

她不像陆允川那样坐在纸堆前发很久的呆。她动作更快，因为家里还有别的事等着。父亲要去镇上办手续，母亲让她顺便列一下要买的生活用品。志愿还没填，分数还没出，可离开已经在日常里提前开始。

林曼打电话来，说晚上有人约着去网吧通宵，问她去不去。

夏知遥笑了笑：“不去了，我妈不放心。”

“考完还管这么严？”

“也不是严。”她把一摞卷子捆起来，“家里事多。”

电话挂断后，她从英语书里翻出一张小纸条，上面写着高三时背过的作文句子。纸条背面还有一个词：keep moving。

她看了一会儿，把它夹进暂时留下的那摞里。

她也会空，也会不舍，也会不知道接下来去哪儿。只是她没有把这些都说出来。她更习惯先把书收好，把地扫干净，把能做的事一件件做完。很多女孩的告别并不声势浩大，它们藏在捆书的细绳里，藏在母亲从门外递进来的饭碗里。
