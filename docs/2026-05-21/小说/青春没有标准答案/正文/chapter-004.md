# 第004章 地震后的默哀



时间：2008年5月中旬



地点：初中操场和教室前门



卷次：第一卷：九年级的志愿表



2008年5月中旬，初中操场和教室前门。一个贴着白纸的捐款箱把这一天的开头压得很实。操场土跑道被晒得发白，广播线在主席台后面绕了几圈，声音有些发闷。段璟站在旁边，手上还有铅笔芯蹭出的黑印，耳朵却一直留意着常俊凡那边的动静。那时候他们还没有后来那么多说不清的关系，只是同一场中考里两个被分数推着往前走的人。

远方灾难忽然落进县城小学校，所有学生第一次感觉课本外的世界并不遥远。刘海峰站在操场前排，没有像平时那样催队伍站直。广播里传出的声音有些破，却把每个人都按在原地，连平时最爱说笑的学生也把手垂了下去。

常俊凡想让自己的难过有一个具体去处，哪怕只是几张皱巴巴的零钱。她做事不急，先把笔袋摆正，再把要问的问题写在草稿纸右上角。段璟后来才明白，那不是冷淡，也不是故意把人挡在外面。她只是很早就知道，别人可以热闹，可以起哄，可以替她猜测，但真正要承担结果的人只有她自己。

段璟想说点安慰的话，却发现任何话都太轻。他不是不会说话，只是一碰到常俊凡，平常那些贫嘴和硬话就像被粉笔擦抹了一遍，剩下一点不成句子的灰。他怕自己显得太在意，又怕自己真的一点也不在意。初三的男生常常这样，明明心里已经绕了好几圈，开口却只剩一个嗯。

他们的对话没有谁故意说得漂亮，都是初三学生能抓住的那些实话。“默哀三分钟，帽子摘下来，手里的东西先放下。”刘海峰说。“你有一块钱零钱吗？我想把整钱换开。”常俊凡说。“有，两张五角。”段璟说。“名字写不写都行，但数要对。别让好心变成糊涂账。”沈曼青说。话说完以后，谁也没有立刻把目光放到对方脸上，仿佛只要盯着桌面、墙面或手里的纸，心里的动静就不会被看出来。

沈曼青也在场。把班里捐款名单重新誊了一遍，数字对得很慢，也很认真。这种小动作比豪言壮语更像那一年：大家都说自己不怕中考，可水杯里的水少得很快，草稿纸一张接一张被揉皱，课桌边缘全是指甲抠出来的小白印。

晚上回家时，段璟父亲把电视音量调得很小，全家人围着新闻吃饭，谁都没有多夹菜。家庭没有在他们的青春里大声喊口号，它只是藏在饭盒、车票、洗好的校服和一遍遍叮嘱里。封丘县的日子也是这样，街道不宽，消息传得快，谁家孩子能不能上一中，很快就会成为亲戚饭桌上绕不开的话。

段璟看见常俊凡把钱放进箱子里时，手背碰了一下纸边，她没有立刻抬头。这个动作没有被任何人郑重记录下来，甚至连旁边的人都未必注意到。但段璟记住了，因为它让他第一次意识到，所谓喜欢不是突然出现的一句表白，而是很多个没说出口的瞬间慢慢叠起来。

县城里的夏天靠近得很快。午后窗台烫手，作业本边缘卷起一点毛边，水房里有人一边接水一边背英语单词。广播里偶尔提到北京奥运，老师提到中考，家长提到一中，学生提到放假。所有大事小事挤在同一张课桌上，谁也没有多余的地方安放自己的慌张。

风把操场边的杨树叶吹得发响，三分钟很短，短到铃声一响大家又要回去做卷子；三分钟也很长，长到段璟第一次觉得沉默可以压住整个校园。段璟那时还不会把这种场面写成故事，他只会把它塞进心里，等很多年以后再慢慢认出：原来青春最真实的部分，不是后来回忆时替它加上的光，而是当时那些尴尬、具体、说错话又来不及补救的细节。

默哀结束以后，操场并没有立刻恢复热闹。有人低头擦眼角，有人盯着鞋尖，更多人只是沉默地往教室走。常俊凡走得很慢，手里攥着还没投进捐款箱的零钱，硬币边缘在掌心压出一圈印。

回到教室，黑板上还留着没讲完的函数题。刘海峰没有马上接着讲，只让大家把新闻里听到的地名写在纸上。段璟写得磕磕绊绊，忽然意识到课本上的地图不是画给考试看的，它真的连着许多人的家。

那天食堂排队也比平时安静。铁勺敲在盆沿上，声音显得格外硬。常俊凡端着饭盆站在队尾，宋乔想说两句轻松话，张了张嘴又闭上了。

沈曼青在旁边留下的小动作，也让这一章不只围着段璟和常俊凡转。封丘县的学生多数早早学会看大人的脸色：家里宽裕不宽裕，老师看好不看好，自己能不能把一次机会抓住。这些判断没有写在卷面上，却时时刻刻挤在卷面旁边。

常俊凡很少把情绪说满。她说可以，就是可以；说不改，就是不改。这种短句有时会让段璟不知道怎么接，可也正因为短，才不像临时编出来的温柔。她给人的距离感，不是拒绝所有人，而是提醒别人别替她安排人生。

段璟的笨拙则更容易被看见。他会在该说话时沉默，在该沉默时又憋出一句不合适的话。付小川他们偶尔笑他，他也会顶嘴，可常俊凡一看过来，他的顶嘴就像被人抽掉了半截，只剩下耳朵发热。

那时候大家还没有智能手机可以随手查消息，更多事情靠公告栏、电话、老师通知和同学传话。一个消息从办公室传到教室，往往会变出好几个版本。有人说分数线会涨，有人说一中要扩招，有人说住校床位紧，越传越像一场小小的风。

封丘县的夏天有自己的声音。三轮车从巷口过去，车斗里响着铁盆；早餐摊收摊时，塑料凳一只套一只；远一点的地方，麦场上有人拍打编织袋。这些声音落进少年人的心里，不会立刻变成故事，却会在多年后突然浮出来。

早晨校门口的早点摊还没完全收，油条筐上盖着一块发硬的白布。段璟经过时，总能听见老板娘喊谁的豆浆没拿。这些声音和一个贴着白纸的捐款箱一起留在他记忆里，让地震后的默哀不像一句章节名，而像一个能闻到油烟味的上午。

课间十分钟没有想象中轻松。有人冲去水房，有人围着老师问题，有人站在走廊里用手扇风。沈曼青从人群里挤过去时说了一句让让，常俊凡把资料袋往怀里收，段璟则下意识侧身，给她留出半步。

午饭时，食堂的白菜豆腐热得烫嘴，大家还是吃得很快。段璟把饭盒放到桌上，听见旁边有人讨论分数线，筷子停了一下。常俊凡没有加入，只把自己碗里的葱花挑到一边，动作慢得很稳。

下午最容易犯困，窗台被太阳晒得烫手，试卷边角卷起来一点。刘海峰让他们把错题重新抄一遍，班里顿时响起一片很小的叹气声。常俊凡没有叹，她只是换了一支笔，像换一种力气继续往下写。

傍晚放学时，县城街灯一盏盏亮起来，小摊开始把塑料凳搬到路边。段璟有时会远远看见常俊凡和几个女生并排走，她们说话声音不大，经过文具店时会停一下，看看新到的笔芯和文件袋。

晚自习里的时间比白天慢。吊扇转得吱呀响，黑板上的粉笔字被灯管照得发白。段璟低头写题，偶尔听见常俊凡翻页，声音很轻，却足够让他知道她还在前面某个具体的位置。

回家路上，自行车铃声从身后过来，又很快被风推远。封丘县的路灯不算密，明暗交替落在路面上，像一格一格还没写完的选择题。段璟骑得不快，脑子里却一直在想一中、成绩和常俊凡。

夜里院子里有蛙声，远处偶尔传来货车经过的低响。段璟躺下以后，会把白天没说好的话重新想一遍。想得再顺，第二天见到常俊凡时，还是会卡在喉咙里。

文具店老板认识不少初三学生，看见他们进门，不用问就把黑色笔芯和透明文件袋往外推。段璟买东西时会先看价签，常俊凡也一样。两个人都没把这种节省说出口，却都知道高中还没开始，钱已经先变成了一道题。

小卖部门口的冰柜嗡嗡响，最便宜的雪糕总是先卖完。沈曼青说考完一定要吃两根，常俊凡提醒他先别把嗓子吃坏。段璟听着他们说话，忽然觉得这种普通的玩笑比任何保证都让人踏实。

水房排队的时候最能看出一个人的性子。有人插队，有人抱怨，有人默默把瓶子往里挪。常俊凡通常不抢，也不让自己吃亏。轮到她时，她把瓶口对准热水龙头，手腕很稳，像对待一道必须小心计算的题。

作业清单贴在黑板左侧，每天都被人用粉笔补上一行。段璟有时看着那几行字发愣，觉得它们像一道矮墙，把所有人围在中考前的日子里。常俊凡则会把清单抄到本子上，完成一项划掉一项。

隔壁班偶尔传来一阵哄笑，刘海峰就会走到门口看一眼。本班立刻安静，连付小川也把嘴闭上。常俊凡低头写题，段璟看见她的耳朵被窗外的光照得有点红，很快又觉得自己不该看这么细。

家长打来的电话常常很短。吃饭了吗，别熬太晚，表填了吗，钱够不够。这些句子听上去都不新鲜，却能把一个学生从自以为很大的心事里拉回现实。段璟和常俊凡都在这样的电话里，学着不把烦躁说给家里听。

有一次下雨，走廊地面被踩出一串泥点。常俊凡从书包侧袋里拿出一张旧纸，垫在文件袋外面，怕水溅上去。段璟看见以后，也把自己的资料往怀里收了收。他不是忽然懂事，只是忽然知道有些东西应该被好好护着。

班里有人把封丘一中说得像一座很远的城，也有人说它就在县城里，没什么稀奇。常俊凡听到这两种说法都不争。对她来说，远不远不是地图决定的，是自己有没有真的走进去。

段璟偶尔会把草稿纸写满，再从边角撕下一小条算题。他不舍得浪费纸，也不舍得承认自己在意常俊凡。于是很多心事都跟公式挤在一起，写得乱七八糟，最后被他揉进桌洞。

常俊凡的沉默并不空。她沉默时会做事：削铅笔、抄题、整理文件袋、把借来的资料按原来的顺序夹好。段璟慢慢看懂，真正清醒的人不一定说很多道理，她们只是把每一步都走得不含糊。

操场边的杨树叶在热风里翻动，叶背一闪一闪发白。体育课集合时，大家站得歪歪扭扭，心思却还留在教室里的卷子上。段璟听见有人喊常俊凡的名字，回头太快，被付小川看见，笑了半天。

那种笑没有恶意，却让段璟明白，喜欢一旦露出一点边，很快就会被同龄人看见。他开始学着把眼神收慢，把话说少，也把自己真正想问的问题压到更深的地方。

可越是压着，细节越会自己冒出来。一支笔放在桌沿，一页资料被风翻起，一句平平常常的提醒，都能让段璟在很多年后想起那年夏天的温度。

封丘县的傍晚来得不急。太阳落下去以后，屋檐和树影还热着，路边摊的煤炉慢慢红起来。学生们背着书包从街上走过，校服后背被汗洇出浅浅一片，谁也不觉得狼狈，因为每个人都差不多。

常俊凡走路时很少东张西望，段璟却会把周围的动静都看一遍。他们像两种不同的学习方法：一个先定方向，一个不断确认旁边有没有路。这样的不同，后来也一直影响着他们。

所以这一段不能只写成暗恋。暗恋只是表层，底下还有县城学生对升学的谨慎、普通家庭对花销的计算，以及两个少年第一次认真面对选择时的笨拙。这些东西合在一起，才让他们的靠近有了真实重量。

这段日子里，封丘县像被中考轻轻拧紧了一圈。街上还是那些卖菜的三轮车、修车铺和文具店，可学生走过去时，总会下意识把书包带往肩上提一提，好像这样就能把未来背稳。

常俊凡不是不会累。她只是习惯先把事情排好，再允许自己短暂地松一口气。文件袋、笔袋、准考证、通知书，每一样都被她放在固定位置。段璟看见这些，才慢慢明白她的要强不是给别人看的。

段璟的心思则更乱。想到封丘一中时，他会想到校门；想到常俊凡时，他会想到她写字时压住纸角的手。两种念头挤在一起，让他有时分不清自己到底是在向学校靠近，还是向她靠近。

大人们的担心没有停止。生活费、住校、路费、孩子能不能适应，这些问题每天换一种说法出现。可他们也没有把孩子往后拉。很多时候，父母的支持只是沉默地把钱数好，把被褥晒干，把一句注意身体重复好几遍。

沈曼青的存在让这段前史多了一点旁边的声音。有人贫嘴，有人算分，有人说自己无所谓，真正到了选择面前，又都把无所谓收了回去。群像不是热闹的装饰，它提醒段璟，每个人都在用自己的办法过关。

北京奥运的新闻越来越密，电视里常出现倒计时。县城大人谈起奥运时带着兴奋，学生们听见倒计时，却总会想到另一个倒计时：开学、报到、新班级，以及还不知道会不会同班的那些人。

如果只把这几章写成喜欢与被喜欢，就会太窄。真正让人信服的，是喜欢旁边还有考试、家境、选择和县城日常。青春不是悬在空中的情绪，它有尘土、有饭菜味，也有不得不填下去的表格。

段璟开始学着不把每一次沉默都理解成拒绝。常俊凡沉默时可能在算题，可能在想家里的话，也可能只是累了。一个人越是真实，就越不该只被另一人的心事解释。

常俊凡也不是冷冰冰的答案。她会提醒别人别吃太凉，会把借来的东西归整好，会在别人起哄时皱眉。这些小动作没有一句像表白，却比表白更能说明她是什么样的人。

到了夜里，县城慢慢安静下来，只有远处车声和院子里的虫鸣。段璟躺在床上，把当天的场面一遍遍翻出来，最后发现最难忘的不是某句漂亮话，而是常俊凡把自己的选择握稳的样子。

封丘一中这个名字在他们嘴里出现得越来越频繁。起初它只是宣传单上的校门，后来变成老师讲过的分数线、家长算过的生活费、同学拿来互相试探的一句你报不报。段璟每次听见这四个字，都会下意识去看常俊凡。

有人把离别说得很夸张，好像初中一结束，所有人马上就会被命运吹散。也有人故意说得很轻，说县城就这么大，早晚还能遇见。常俊凡听见这些话，通常不反驳，也不附和，因为她知道遇见容易，重新站到同一条路上并不容易。

段璟家离县城不算远，却也不是抬脚就到。早晨路上有卖菜的三轮车，有背书包赶车的学生，有刚从地里回来的大人。那些味道和声音混在一起，让封丘县不像地图上的名字，而像每天都要经过的生活。

常俊凡的学习方法有一种笨拙的可靠。她不喜欢把笔记涂得花里胡哨，只在最容易错的地方画小方框。如果一道题错了三遍，她会在旁边写明错因：看漏条件、公式混用、心急。
