# 第056章 高一楼先动的手



时间：2009年一月上旬，大雪午后



地点：高一楼走廊、楼前空地和操场边



卷次：第三卷：雪落在两栋楼之间



2009年一月上旬，大雪午后，高一楼走廊、楼前空地和操场边，梁一帆掌心里越捏越硬的雪球先把这一天按进了十班的日常里。高一楼开始反击，十班几个男生冲到楼前，把雪团往高二楼二层窗口扔。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

沈曼青看出这事快失控，常俊凡也提醒大家别把热闹变成处分。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看梁一帆掌心里越捏越硬的雪球，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

沈曼青也在这一天被推到前面。沈曼青的特点是冷静，提问直接，遇到热闹会先看风险。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“别往窗户扔，真碎了谁赔？”沈曼青说。

“我瞄的是墙。”梁一帆说。

“你瞄墙，雪可不一定听你的。”常俊凡说。

“往后退一点。”段璟说。

“哪个班的？都给我站住！”王启民说。

高一楼的反击一开始还有分寸，雪球大多砸在墙上、栏杆上和空地中央。

可喊声越大，分寸就越薄。梁一帆手里的雪球越捏越硬，像把整个年级的面子都捏进去了。

沈曼青站在台阶上没有参与，她看的是窗户、楼梯、老师会从哪边过来。

付小川喊十班只反击不挑事，结果扔得比谁都兴奋，雪球擦着高二楼窗台飞过去。

常俊凡往后退了两步，拉住旁边一个想往前冲的女生，说别挤到台阶。

段璟看见一个雪球飞偏，砸在高二年级老师的棉衣肩上，周围的笑声突然断了一拍。

王启民从楼侧出来时，脸色比雪还冷。大家像被同时按住开关，手里剩下的雪团都不知道往哪儿放。

高一楼先动的手这句话后来传了很久，每个人都记得自己没先动，却都记得自己笑了。

高一楼先动的手没有悬在空处。台阶上的雪泥和越捏越硬的雪球，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，高一楼先动的手带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

王启民进门时不一定立刻说话，鞋底在水泥地上一停，高一楼先动的手带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在沈曼青快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。高一楼先动的手越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。梁一帆掌心里越捏越硬的雪球让他想说话，又逼他先低头看题。

沈曼青被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，高一楼先动的手留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。沈曼青路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为高一楼先动的手翻身，谁还在被窝里背单词，旁边人都听得见。

雪团砸在墙上散开时，封丘一中的冬天终于不只是冷，也有了少年人收不住的响声。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理高一楼先动的手里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：梁一帆掌心里越捏越硬的雪球、一只冻红的手、一句老师点名、一块还没扫干净的雪。

沈曼青把雪球攥得太紧，掌心冻得发疼也没立刻松开。窗外风一过，纸页轻轻动了一下，像替谁把话咽回去。

罗雨桐刚冲下两级台阶又被同学拽回去，鞋底在雪泥上滑了一下。这个小动作很快过去，却让教室更像一个真正在运转的班。

赵远航抬头瞄二楼墙面，雪球飞出去时自己先闭了眼。旁边的人没有特意看，可那一瞬间还是被段璟记住了。

陈柏言听见老师喊停，手里的雪团一下子不知道该往哪儿放。常俊凡没有接话，只把手里的事做完。

张若楠把雪球攥得太紧，掌心冻得发疼也没立刻松开。老师没有立刻点名，那一秒沉默反而让人更知道该收住。

高一楼先动的手之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

沈曼青没有因此变成特别醒目的人，可在段璟眼里，沈曼青从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，一个雪球擦过高二年级老师的肩膀，喊声终于变了味。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
