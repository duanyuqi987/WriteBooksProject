# 第074章 网吧通宵旁观

所属部卷：第二部：六月以后
时间：高考后的夜晚
地点：县城老网吧

## 诗歌插页 P074：原创现代短诗

那时我们都以为，
结束会有清楚的门。
后来才知道，
门只是普通下午的一阵风。
有人走向远处，
有人学着把远处还给远处。

网管叫刘洋，二十六岁，光头，戴黑框眼镜。他在这个网吧做了四年，看过的毕业生通宵不下几百场。

"每年六月都一样。"他一边擦柜台一边跟旁边一个熟客聊天，"考完那天晚上肯定爆满，第二天晚上也爆满，第三天就开始少了——有些人去旅游了，有些人被家长扣在家里填志愿。"

"你觉得他们哪个最奇怪？"熟客问。

刘洋想了想，用下巴指了指角落那三台机子。"那个穿灰T恤的。他昨晚打开聊天软件看了好几次，每次鼠标停在同一个名字上面，停几秒又移开。一个字都没发。"

"暗恋吧。"

"应该是。但我觉得他不只是暗恋。"刘洋把泡面调料包码整齐，"他那个样子像是想说什么，但又在等什么——等时间到？等自己准备好？谁知道。"

凌晨三点的时候，刘洋去收泡面桶。走到角落那排，灰T恤的男生正好把聊天窗口关掉。屏幕上只剩一个空白的文档——上面什么都没写。

"同学，泡面吃完了没？"

"吃完了。"男生把泡面桶递给他。

刘洋注意到他的手指还放在鼠标上，食指微微弯着，像随时准备点开什么。但他没有点。他把鼠标推到一边，靠回椅背上，看着窗外发呆。

刘洋收了泡面桶就走了。他在网吧见惯了各种人——有打通宵游戏打到哭的，有跟网友吵架吵到砸键盘的，有看着聊天框发呆一整个通宵的。第一类人最多，第二类人其次，第三类人最少。但第三类人给他的印象最深，因为不发消息比发消息更需要力气。

凌晨五点半，刘洋开始打扫。他把烟灰缸倒干净，用抹布擦键盘缝——里面全是烟灰和零食渣。擦到灰T恤男生那台机子时，他看到键盘缝里夹着一张便签纸。抽出来一看，上面没写字，只画了一个问号。

他本来想扔，但想了想，把便签纸压在柜台下面。说不定那个男生会回来找——问号这种东西，画出来就是等着被回答。

但那个男生没有回来找。天亮以后他跟另外两个男生一起走了，走得很快，脸色都不太好——通宵的脸。

叶澄那晚没有去通宵。她在自己房间里整理作文存档，弄到凌晨一点。睡前看了一遍班群——程砚在群里发了一张网吧照片，配文"今晚的战场"。背景里能看到陆允川的侧脸，他正在看电脑屏幕，屏幕上开着聊天软件。

叶澄看了一眼照片就关掉了群。她知道陆允川在看谁的聊天窗口，但她没有发私信去问。有些事情旁观者看得再清楚，也不能替当事人做决定。

她关掉台灯，躺在床上。手机屏幕亮了一下——是周长海在群里回复程砚："你拍照能不能先美颜，我脸都是糊的。"程砚回："你脸本来就这样，不怪相机。"

叶澄笑了一下。然后关掉手机，闭眼睡了。

杜雨薇那晚也没有出门。她在家刷了一部电影，讲的是几个高中毕业生在暑假最后一天去海边。电影里的主角在沙滩上写了好几个人的名字，海水涨上来把名字冲掉，第二天那些名字就不存在了。只有一个人，主角把她的名字写在了一块石头上。

杜雨薇看完电影后，拿起手机翻到陆允川的聊天窗口。她和陆允川不熟，整个高中三年只说过几句话。但她一直记得，有一次物理课上，陆允川回答了一道很难的题，回答完之后看了夏知遥一眼——那种眼神她很熟悉，因为她也用同样的眼神看过别人。

她打了一行字："你们今晚去通宵了？"

删掉了。

又打了一行："高考出分后你有什么打算？"

删掉了。

最后她把手机关掉，翻了个身。电影里那句台词还在脑子里转——"有些名字适合写在沙子上，因为被冲掉之后你就不会一直想去找。"但她觉得主角说错了。被冲掉的名字往往记得最久。

她闭上眼睛，很快也睡着了。

那天晚上县城的星空很亮——比平时亮，大概是白天下了阵雨，空气被洗干净了。老网吧的灯管亮了一整夜，菜市场门口那条路上不断有人经过：通宵的毕业生、准备出摊的早点铺老板、扫街的环卫工人、失眠的老人。每个人都在自己的夜晚里，做着各自的事，想着各自的人。

网吧角落里那台机子的键盘缝里，曾经夹过一个画着问号的便签纸。后来被网管收到柜台下面，再后来被新的泡面调料包压在底下，慢慢被遗忘。但那个问号——那个没有发出去的消息——留在陆允川记忆里的时间，比他想象的要长得多。
