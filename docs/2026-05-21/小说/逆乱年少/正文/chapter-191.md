# 第191章 最后一晚

时间：2014年6月，离校前最后一晚

地点：图书馆四楼

卷次：第八卷：没有答案的答案

离校前最后一个晚上，林玉起又去了图书馆四楼。

不是刻意去的。他下午把最后一行李塞进编织袋，弯腰去够床底下的鞋盒时，后背的汗把T恤洇湿了一大片。郭向阳说走之前再去吃一顿后街的烧烤，他应了一声"好"，但等宿舍里的人都出去了，他还是绕了个弯，走去了图书馆。

图书馆今晚开到十点。他刷卡进去——学生卡再过三天就失效了，闸机上的绿灯亮了一下，像是在做最后一次确认——这人还是这里的学生。

四楼没有人。不是几乎没有人，是真的没有人。整层楼只有日光灯管在头顶嗡嗡地响，书架像一排排沉默的灰墙。靠窗的位置空着——第四张桌子，面向书架，背对走廊。他坐下来，把椅子调了调——这张椅子有条腿短了两毫米，坐上去会轻微地晃，他从来不去换。

因为苏安晴坐这里的时候从来不抱怨。

他把手放到桌面上，手心贴着桌面——五月底的木桌触感是凉的，但是边缘被前人的手肘磨出了一小块光滑的区域。他摸到了那个区域，心想苏安晴大概也摸过这里。

坐了一会儿，楼梯间传来脚步声。不是管理员——管理员的脚步更重，而且会顺手开灯。这个人的脚步很轻，速度不快不慢。

苏安晴从楼梯拐角走出来的时候，手里握着一个保温杯。粉色的，还是宋乔送她的那个，杯身被磕掉了一小块漆。

她没有问他为什么在这里。她拉开他对面的椅子坐下来，把保温杯放在桌上。

"我以为你今晚会去后街。"

"郭向阳他们在那边，"林玉起说，"我想先来这坐坐。"

"嗯。"

她拧开保温杯，喝了一口。水蒸气从杯口升起来，在她脸前散开。她把杯子放下来，两个手围在上面暖着。

"大二那年冬天，我在这复习文学史。"林玉起指了指自己坐的位置。"背陶渊明，背到'久在樊笼里，复得返自然'的时候，你从书架那边走过来，在我对面坐下了。你拿了那本《教育学原理》，翻到第五章，然后你在笔记本上写了一段话——写了大概五分钟。我一个字也没看进去。"

"你每次看其他人复习，都说'一个字也没看进去'。"苏安晴说，"你这个借口用了四年。"

林玉起愣了一下，然后笑了——是那种被戳穿以后不好意思的笑。苏安晴说得对，这确实是借口。不是"一个字也没看进去"，是"只看到你坐下了，就开始集中注意力了"。不是不能集中注意力，而是注意力全都在她身上，忘了书。

窗外的月光很淡，被日光灯压得几乎看不见。梧桐树的影子投在窗玻璃上，风一吹，影子像在微微地摆手。

"以后没有图书馆四楼了。"林玉起说。

苏安晴抬起头看了他一眼。她的目光在日光灯下显得格外安静。

"以后有别的地方。"
