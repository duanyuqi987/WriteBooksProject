# 第018章 前排和后排之后

所属部卷：第一部：初秋三楼
时间：高一春天
地点：物理课后的教室

## 诗歌插页 P018：原创仿宋小令

本过三层，
一瞬也关情。
欲把墨痕藏半页，
却因雨后日光明。
照见旧心惊。

前排和后排之后，陆允川表面上没有变化。每次传作业他还是把自己的本子递出去，前排的人还是接了摞在下面继续传。但他在心里把整件事拆成了很多小步骤——递出本子的速度现在控制得很稳，不快不慢，刚好让前桌能自然接过去。他算过：太快了显得敷衍，太慢了又太明显，最优速度是让前桌的手在空中等不超过零点三秒。多停留零点几秒是因为手上有粉笔灰，不是因为想多看什么。他甚至为此故意在上课前摸一摸黑板边缘让自己手上沾灰。那层灰很薄，每次只能沾一点点，所以他得在上课前至少摸两次黑板——一次不够白，两次刚好。

程砚在课间问他："你最近怎么每次传作业前都摸一下黑板？"陆允川说："手上有汗，沾点粉笔灰好翻纸。""你这理由我很怀疑——你手上哪有那么多汗？冬天就算了，春天你又不是汗手。"陆允川没接话。程砚又看了看他，说："但你每次传作业时手上的确没抖，我先信一半。"他说完从抽屉里掏出一包饼干，拆开了递给陆允川一片，"来，吃。补充点热量，免得你下次编理由的时候血糖不够。"

午饭时程砚发现陆允川今天打了一个荤菜两个素菜，比平时的量多了一半。"你受什么刺激了？""我今天真饿了。""终于知道饿了？你上学期有一周每天只打一个菜，我以为你要练辟谷。"陆允川夹了一块红烧肉放进嘴里，嚼了两下，觉得确实比上次好吃。

"你最近有没有人跟你说过话？"程砚试探性地问。他把筷子搁在餐盘边上，摆出一副"我就是随便问问"的表情。
"谁？"
"前前排的。"
"说过了啊——传作业的时候她回头问她同桌那道题的答案，我听到了。"
"她对她同桌说的——你听到了——你管这叫'说过了'？"程砚端起碗，看陆允川的眼神像在看一张填错栏目的表格，"老陆，你对'说过话'的定义是不是和别人不太一样？"他把碗放下来，用筷子敲了敲陆允川的餐盘边缘，"真正的'说过话'是你对她说了什么，她对你回了什么。两个方向，有来有回。你那种叫'单方面接收信号'，和收音机没什么区别。"

晚自习中间，物理老师来教室巡课。他腋下夹着那摞作业本，走到后排时放慢了脚步。陆允川以为他要说作业格式的问题——结果物理老师走到他旁边停下来，翻了翻他桌上的练习册说："你的受力分析步骤写得很清楚，继续保持。"程砚在桌子底下踢了他一脚，意思是"夸你了"。

陆允川觉得这个夸奖和他上次的作业本被叠在她本子上面有关系，但又觉得这完全是两件不相干的事——老师夸的是步骤，他叠本子是运气。运气和实力没有因果关系，但在十七岁的叙事里，运气是最好的因果。

夏知遥对这些毫不知情。物理课后她和何敏去水房接水，水房在走廊尽头，地上总是湿的，墙角竖着几把拖把。何敏问她受力分析第三题的摩擦力到底往哪边，她拿过何敏的杯子，倒了一点凉水进去，用湿手指在瓷砖上画了一个箭头形说"你看，相对运动趋势是斜向左下，摩擦力斜向右上"。何敏凑近了看，又用手指沿着那个水箭头描了一遍，然后恍然大悟。夏知遥画完把手指上的水甩了甩，拿起杯子走了。那个瓷砖上的箭头很快干了，什么痕迹都没留下。只有墙角拖把上滴下来的水还在地上积着，一滴一滴地响。

夜里宿舍熄灯后，程砚突然说："我今天观察了全班的作业传递路线，咱们这列是最短的——只有四排。所以本子交到第三排的时候大概只需要三秒。三秒能想什么？三分之一道选择题。"

"你睡前就算这个？"陆允川从被子里闷声说。
"对。我帮你算好了——三秒，不够紧张。所以你该睡就睡，明天还要交化学作业呢。"

陆允川闭上眼，在脑子里把那个"啪"声又过了一遍。三秒很短，但三秒钟有时候刚好够一个念头从形成到被压制——像一颗石子从手心滑出去，还没落地就被另一只手接住了。

第二天一早早读，阳光还没完全照进教室，日光灯管嗡嗡地亮着，有几盏在不停地闪。语文老师让全班齐读《荆轲刺秦王》中的一段。陆允川张嘴跟着读，读到"风萧萧兮易水寒"时发现自己跑到了下一段的"壮士一去兮不复还"。他把书往回翻了一页，程砚在旁边抿着嘴憋笑——憋得肩膀一抖一抖的。前排有人回头问"谁读到壮士去了"，陆允川耳朵热了，程砚替他回了句"壮士要出发了，你们读慢一点"。教室里压着嗓子笑了几声。语文老师看了程砚一眼，说"程砚你先把课文读完再当壮士"，笑声又压下去。

早读结束，窗外雨后的阳光照进来，在作业本上落了一道明亮的横线。他把那道横线对齐了桌边，翻开数学课本，找了道微积分题开始做。他现在做数学题的速度比上学期快了将近一倍——不是因为智商提高了，而是因为解数学题的时候脑子不用拐到第三排去。这是唯一的休息方式。
