# 第006章 物理课

时间：2010年10月中旬

地点：高一三班教室

卷次：卷一：初见秋风

物理老师姓李，瘦高个，戴一副银边眼镜。眼镜腿用白胶布缠了一圈。上课从不点名——不是不管，是不在乎。他进了教室就翻书，翻完书就往黑板上画图。画圆不用圆规，手腕一转就是一个。画完图开始讲。谁听懂了、谁没听懂，他不管。讲完了靠回讲台边上喝一口茶。茶杯是玻璃的，里面的茶叶是立着的。

十月中旬一次课堂小测。

不是什么大考。就一张A4纸，正面四道题，反面三道。全是力学题——物体受力分析、摩擦力方向判断、牛顿第二定律应用。卷面是手刻油印的。字是蓝色钢笔写的，印的时候油墨没匀，有的地方淡得只剩下半个笔画。第一道题的斜面上画了一个小方块，摩擦力的箭头方向空着让你填。

李老师把卷子往讲台上拍了一下。"四十分钟。做完交。不会的空着。"然后把卷子从前往后传。传卷的时候前面那个女生把正反面搞反了——顾临风指了指——她翻过来了。

顾临风接了卷子。纸质比较薄，下午的阳光能透过去看到反面的字。他看了第一道——物体放在三十度斜面上。重力。弹力。摩擦力。不难。

他在草稿纸上列公式。第一步：画受力图。重力mg竖直向下。斜面支持力N垂直于斜面向上。摩擦力f方向待定。第二步：正交分解——沿斜面方向与垂直于斜面方向。mg·sin30°。mg·cos30°。

做到第三题的时候，教室里有人叹气。从前排传过来的。然后是翻纸声——纸张在桌面上刮过的声音。靠窗的那个女生在看窗外。左手支着下巴。笔在右手，没动。做到最后一道题的时候，前面有人站起来了。

是苏晚晴。

她把卷子放到讲台上。讲台上已经交了三四张了——摆得很乱，有的正有的反。她把自己的卷子放在最底下。卷子没放稳，从讲台边缘滑了一下——李老师的茶杯差点被打翻。她伸手按住了。放好。然后走回座位。头没抬。脚步很轻。

李老师从茶杯后面看了她一眼。没说话。

顾临风看过去。她桌上的笔搁在草稿纸旁边——笔盖在旁边，没盖上。草稿纸上画了不少。有涂改——黑色的水笔画了一个力的分解图。重力箭头往下——竖直的。斜面支持力倾斜向上——角度大概也是三十度。但摩擦力的方向画反了——本来是沿斜面向上（阻止物体下滑），她画成了向下。她改了一遍。箭头涂黑了——涂成了一个黑色的小三角，旁边画了一道竖线。然后画了一根新的箭头。新的向右上方。还是错的。正解是沿斜面指向上方——平行于斜面，与重力沿斜面的分力方向相反。她把整张草稿纸揉成一团。停下了。然后从抽屉里抽了一张新的草稿纸。

他继续写自己的卷子。做题——受力分析——公式带入——计算结果——复核——写答案。他做到最后一道题的时候，时间还剩九分钟。最后一道题是牛顿第二定律——已知合外力求加速度。他写了一个a=2m/s²。然后检查了一遍。把卷子反过来又查了正面。第三题的一个数字有点糊——是把6写成了9又改回去。他在旁边标了个记号——等于6m——写了个"=6"。

下课铃响之前他把卷子交了。卷子放在讲台上——他放得很正，对齐了讲桌的边。

第二天物理课。李老师拿着一摞卷子进来。卷子对折的，竖着夹在他胳膊底下。他进来的时候门没关——有人把门带上了。

"念一下成绩。"

他没念名字，只念分数。从高到低。翻一张，念一个。翻过去的卷子放在讲台另一边。"九十二。八十八。八十五。八十三。八十——"

顾临风得了七十八。不算差。比他预想的高一点。他期中模拟那套卷子才七十一。

他往下听。手指夹在物理课本的页缝里——食指夹着。

"六十五。六十三——六十一。"

停了。李老师翻到最后一张卷子。看了上面的名字。然后把卷子翻过来——对着全班正面。卷子的总分栏上写了一个数字：五十九。

没念名字。但全班都看到了。红笔写的59。下面有道题只写了公式没算出结果——空格还空着。

然后李老师开始讲题。第一题——物体在斜面上的受力分析——他在黑板上画了一个标准的斜面（他的圆画得还是那么圆）。"这道题大部分人都做对了。注意摩擦力的方向——"他用红色粉笔在斜面上方画了一个向上的箭头。红色箭头比白色箭头显眼。

讲到第三题的时侯他看了下面一眼。"这道题错的人不少。受力分析方向搞反了——摩擦力的方向是阻止物体相对运动的方向。我再画一遍——"

顾临风没在听。

他往左前方看。四排+两个过道之外。

苏晚晴趴在桌子上。

脸埋在胳膊里。头发散开了——马尾的皮筋有点松了，头发从侧面漏出来搭在手腕上。能看到她的肩膀——校服下面的一小块。肩膀微微动了一下——呼吸。趴了大概五分钟。周围有同学在翻卷子、在抄黑板上的题。她旁边的女生推了她一下。没反应。又推了一下。她直起身。用力吸了一下鼻子。是鼻子不通的那种吸法。

然后她翻开物理笔记本。拿起笔。在手心里转了一圈——笔在拇指和食指之间滚了两转。然后开始写。

写了几个字。划掉了。划掉的那一笔很重——纸面戳透了，透到下一页。翻过一页。继续写。又停了。笔放在桌上——圆珠笔在桌面上轻滚了一下。手指在桌面上敲了两下。很轻。食指第一关节轻敲木桌面。然后重新拿起笔。

她在物理笔记本上写了擦、擦了写。

顾临风低下头。翻开自己的物理笔记本。纸张的页边用尺子打了一道竖线——分笔记区和旁注区。这是他自己的习惯。翻到倒数第二页。那页是上次物理课的笔记本上怎么都记错了——牛顿第二定律的公式F=ma，他写成了F=mv。自己没发现，旁边的赵一鸣看了一眼。"你写的这是什么。"然后他改了。改完之后在旁边标了一个小星号——提醒自己下次注意。

翻到最后一页。空的。横格纸。格线是浅蓝色的。右上角有点折，折过之后被压平了。

他拿起笔，在最下面写了几行字。字很小。小到他自己要凑近了才能看全——字的高度大概就是横格线宽的一半。用铅笔写的，不是圆珠笔，这样擦起来容易。

写的是一道力学题的解法。物体在斜面上——重力分解：G∥=mgsinθ，G⊥=mgcosθ。摩擦力f=μN=μmgcosθ。若μ<tanθ，物体加速下滑。斜面角度三十度。μ取0.3。他写完之后在旁边画了一个受力图——箭头画得不比物理老师的圆规圆，但方向是对的。

写完。他把那页纸折了折——折成四分之一大小——压在本子的最后一页和封底之间。

没撕下来。没给出去。字太小了。除了他自己，没人会注意到最后一页有这么几行字。

那页纸在物理笔记本的最后一页夹了很久。

后来物理笔记本换了一本新的——这本写到第六十多页就写满了。旧的那本塞在宿舍床头的一个纸箱里。纸箱原来装的是方便面——康师傅红烧牛肉面。箱子外面印着牛肉面的图。现在装了旧书和旧笔记本。最后一页的字还在。字很小。铅笔写的字过了一两个月没擦——铅笔粉已经渗进了纸纹里面。即使擦了也会留印子。折痕压得平整——四分之一大小，折角对着右下角。

周末。宿舍人走了大半。周六下午刘洋去了操场打球。赵一鸣回家了。剩下两个——一个姓陈的躺在上铺看电子书。还有一个不知道去哪了。

顾临风坐在床上翻那本旧笔记本。从纸箱里掏出来。封面有点灰——擦了一下。一张一张翻过去。公式。例题。受力图。板书的抄写。有些地方有涂改——涂改带的颜色比纸张本身的白色还白。翻到最后一页。手指在那道题解上按了按。铅笔字在手指下面微微地凸起来——写字的时候下笔重了。然后他把笔记本放回纸箱。盖上。纸箱盖子上压了一本《新华字典》。字典的封面是塑料的，边角磨掉了颜色。

窗外有人在打篮球。球砸在水泥地上——咚、咚、咚。节奏不快。偶尔球弹偏了，滚到一边去。有人在喊——"这边、这边——""投啊——""——不进——"哄笑声。然后是球重新弹地的声音。

他把纸箱推到床底下，往里推了推。床底下有灰——他用脚把灰往里面蹭了蹭。袜子忘了脱。袜子底沾了层灰——在地上走路的时候留下了浅灰的脚印。

第二天上课。数学课。老师在黑板上讲二次函数顶点式——y=a(x-h)²+k。对称轴x=h，顶点坐标(h,k)。他在黑板上写了一个例题——a=2,h=-1,k=3。顾临风用铅笔抄在自己的数学笔记本上。顾临风翻着自己的物理笔记本。翻到最后一页——被他用铅笔写了又折过的那个位置。折着的纸被翻过一次之后自然弹开了一些。题解露出了一部分。他看了两秒。翻过去。然后翻回来。又看了两秒。手指在折角处抹了抹。

下课的时候他站起来伸了个腰。伸到一半改成了系鞋带——右脚那只鞋带松了。蹲下去系了一个蝴蝶结。站起来的时候头不小心碰到了赵一鸣桌上挂着的书包。没碰掉。然后他直起身，往左前方看。

她在写什么。桌子上摊开的是一本新买的物理练习册——封面是蓝色的，粗体的"高中一年级物理同步练习"印在封面二分之一的位置。纸面和数学练习册材质不同，是胶版纸。她翻开第一页——这本练习册是按照课本一章一章排的。第一个章节是"运动的描述"。她拿起笔。写了一行字——"错误分析"。然后在下面分行列了几项数字。搁下笔。趴在桌上了。三秒。然后直起身——甩了一下头——马尾扫在椅背上。继续写。

赵一鸣从外面回来——手里端了两杯水。一杯给他。顾临风接过来。喝了一口。水是温的。"你是不是老在看前面啊。"赵一鸣坐下之后低头翻化学书。漫不经心地说。

"没有。"

"行。"赵一鸣的水喝了一大口，咽下去的时候喉结动了一下。"那你自己看着办。"

上课铃响了。化学课。化学老师端着一只烧杯进来——里面装的不是水，是泡了半天的胖大海。胖大海胀得像一只棕色的水母。化学老师的嗓子沙哑——"今天讲氧化还原反应——"她用教鞭戳了戳黑板。粉笔在黑板上画了一个电子转移的箭头示意图。顾临风翻开化学课本。听课。做笔记。但偶尔还是往左前方漂一眼——第二排靠窗的人在做着什么。
