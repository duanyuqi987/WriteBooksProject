# 第112章 青禾的背影之前



青禾的背影之前，校庆晚会还没开始。礼堂外的广场挤满人，孩子手里的蓝色气球在灯光下轻轻晃。

广场上人声很杂，有老同学喊名字，有志愿者维持队伍，还有孩子因为气球线缠住手腕小声抱怨。重逢没有给他准备安静背景。

蓝色气球在人群上方晃了一下，又被一只小手拉低。

他确认那是林青禾，比自己想象中平静。平静不是不震动，而是震动之后，他仍然知道自己该站在哪里。

他在心里说了一句，好久不见。没有发出声音，也没有期待回应。这句问候留在自己这里，已经足够完整。

林青禾弯腰替孩子扣好扣子，动作很熟练。许亦安忽然意识到，她的人生早已长出许多他不知道的枝叶，而这些枝叶不需要经过他的允许。

很多年后再回头看，林青禾不再只是许亦安记忆里那个忽近忽远的人。她也有自己的城市、工作、家人和难以向旧同学解释的疲惫，只是这些后来都没有进入他当年的日记本。

她没有把这些话说出口，因为说出口就太像责怪，而她并不想伤害他。

校园里的风总是比外面慢半拍，吹过楼道、花坛和公告栏，最后才吹到人的心里。

电话挂断后，许亦安在广场边站了一会儿。周一鸣没有再追问，他也没有急着解释自己为什么不过去。那种沉默很短，却刚好够他把手机放回口袋。

校庆群里有人问他到哪了，他回已经在回去路上。消息发出去以后，群里继续聊合影和晚饭，没有人知道他刚刚看见了谁。

礼堂侧门的灯很亮，亮得每个人的影子都变短。

他从口袋里摸出校庆胸牌，背面的别针有点松。他把别针扣好，又放回桌上，像把这一晚也扣回它该在的位置。

他后来才懂，喜欢一个人不该让自己变得越来越不像自己。

晚会结束后，他从礼堂另一侧离开。手机里没有新增联系人，也没有新的聊天框。夜风吹过广场，他觉得这种空白很体面。

蓝色气球不承担重逢的意义。它只是被孩子牵着，上浮一点，又回到那只小手里。

校庆胸牌的别针有点松，扣上时发出很轻的一声响。

他没有上前，重逢就停在远处。这个距离不是遗憾的装饰，而是他终于学会的尊重。

他没有给林青禾发消息。不是逃避，而是尊重彼此已经拥有的生活。祝福有时候最好的样子，就是不去打扰。

他没有把克制表演出来。只是等人群把她带远，转身从礼堂另一侧离开，去赶一辆普通的车。

第二天醒来，他没有像年轻时那样反复复盘每个眼神。只记得蓝色气球、孩子的小外套和自己没有走过去。

楼下早餐店的豆浆装在纸杯里，没有塑料袋，也没有谁故意让他吃辣。他站在队伍里等了一会儿，忽然笑了笑。原来有些味道不是被忘记，只是终于从某个人身上退回到食物本身。

上班路上，他看见地铁广告牌换了一轮又一轮，像许多来不及说出口的话被新的生活覆盖。可覆盖不是抹掉，底下的痕迹还在，只是不再要求被所有人看见。

广场人群把她带远，也把许亦安留在原地。原地不再是失败，它只是一个人选择不过去的位置。

蓝色气球后来在他的记忆里留得很清楚。它轻轻往上，又被孩子拉住，像所有属于别人的生活，都有自己的线。

这一次他没有把重逢讲成命运，也没有把不上前讲成遗憾。他只承认：看见了，祝福了，然后离开。

回去的路上，他看见写字楼门口换了新的海报。旧海报被卷起来放在墙角，露出背面泛白的纸边。许亦安看着它，忽然觉得记忆也可以这样被收起，不必撕碎，也不必展示。

他没有把这些想法告诉林青禾。那时候他还以为，真正深的感情应该自己消化，不能给对方添麻烦。后来他才知道，不能说的感情很容易在心里变形，变成委屈，变成审问，变成某个夜晚突然冒出来的重话。

那晚他又打开文档，把旧日记里的几句话重新排了一遍。写到一半，他听见楼下有人关车门，忽然明白生活并没有因为他迟来的释怀而停住。

那一行字写着：我没有过去。多年后他知道，这不是胆怯，是边界。

多年后他重写这一段时，特意删掉了几个太用力的词。年轻时他靠重话证明疼痛，后来才相信轻一点也能说清楚。

重读这一页时，他把自己从句子的中心往旁边挪了挪。故事不是为了替谁判案，而是把当时所有人的无能为力放在同一盏灯下。

林青禾终于从他的故事中心走出来，成为广场上一个真实的人。她有自己的孩子、外套、气球和路，许亦安只远远看见一次，就够了。

这一章真正留下来的，不是他当时以为最重要的那句话，而是蓝色气球升起来一点，又被孩子拉回手里。那画面还没有成为记忆，只是很安静地停在那里。

这一页日记后来被他保存了很多年，保存到里面的怨气都慢慢变轻。许亦安合上电脑，听见窗外有人笑着从楼下经过。那笑声很快远了，房间重新安静下来。他坐在书桌前，觉得这段旧事好像没有结束，又好像早就结束了。
