# 第054章 一张小纸条



时间：2009年一月上旬，座位表公布当天



地点：十班教室前门和后墙



卷次：第三卷：雪落在两栋楼之间



2009年一月上旬，座位表公布当天，十班教室前门和后墙，贴在新座位表旁边的一张小纸条先把这一天按进了十班的日常里。新座位表公布，有人在旁边贴了别哭两个字，十班笑成一团。它落得很具体，具体到谁先抬头、谁先躲开、谁把手里的笔停了一下。

何嘉树承认纸条是自己贴的，原本想安慰被调开的同桌，却让陶建国点名。段璟坐在自己的位置上，先看见事情表面，再慢半拍意识到它会落到谁身上。他喜欢常俊凡，但这份喜欢在冬天里不能只停在心里发热，它要学会避开别人的难堪，也要学会承认分数和规矩。

常俊凡没有把自己放在热闹中心。她先看贴在新座位表旁边的一张小纸条，再看周围人的反应，像往常一样把问题拆开：谁需要提醒，谁需要安静，谁只是嘴上不服气。她不是没有情绪，只是习惯先处理能处理的部分。

何嘉树也在这一天被推到前面。何嘉树的特点是安静细致，作文好，常把班里的暗流看得很清楚。十班的群像就是这样慢慢厚起来的：每个人都不是点名册上的名字，而是会出错、会怕、会嘴硬、也会替别人递一把伞的人。

“我就是开个玩笑。”何嘉树说。

“玩笑贴在座位表旁边，就会变成态度。”陶建国说。

“别哭这两个字也太欠了。”宋乔说。

“有些人真会哭，只是不让你看见。”常俊凡说。

“我没哭。”段璟说。

小纸条在十班不是稀罕事。有人传题号，有人借橡皮，有人写下课去不去水房。

真正麻烦的是，纸条一旦被起哄，就会从一件小事变成谁也不好收拾的热闹。

段璟看见纸条角露出来时，第一反应不是好奇，而是上一次被误会后常俊凡皱起的眉。

他把纸条递回去，手指没有多停。这个动作太普通，普通到旁边人找不到起哄的缝。

宋乔看了他一眼，像在确认他这次终于学聪明了。

付小川张了张嘴，又自己闭上，把玩笑咽回去，低头装作研究橡皮屑。

常俊凡接过纸条时只说了声谢谢，声音不大，却把边界放得很清楚。

十班从这些小事里慢慢学会，热闹不是想要就要，别人的难处也不是拿来传的。

一张小纸条没有悬在空处。小纸条折痕旁边沾着的一点粉笔灰，把它压回了十班每天都要经过的地方。

封丘县的冬天落到学生身上，常常不是一句冷，而是写字慢下来的手、袖口上的白灰和饭后迟迟不愿离开的热汤。

教室窗户关不严，一张小纸条带来的动静在门口和窗边来回传。靠窗的人先喊冷，靠门的人嫌吵，后排又有人想把紧张说成玩笑。

陶建国进门时不一定立刻说话，鞋底在水泥地上一停，一张小纸条带来的浮躁就会矮下一截。

宋乔把女生那边的消息带得快，周明澈把男生这边的混乱收得稳，付小川则负责在何嘉树快绷不住时递上一句不太正经的话。

常俊凡的桌面仍然整齐，错题本、草稿纸、水杯和透明文件袋各有位置。一张小纸条越乱，她越像要先把自己这一小块地方守住。

段璟的桌洞则总有点乱，卷子、笔芯、单词纸和没还的橡皮挤在一起。贴在新座位表旁边的一张小纸条让他想说话，又逼他先低头看题。

何嘉树被别人叫到名字时，下意识先看老师，再看同桌。这个动作很小，却让段璟看见十班里普通人的怕和要强。

食堂窗口排队时，一张小纸条留下的尴尬会变轻一些。白菜汤一端上来，人就先顾着别烫到手。

水房地面常有湿脚印，暖水瓶排成一列。何嘉树路过时把自己的瓶子往里挪，给后面的人空出一点地方。

宿舍里的夜晚比教室诚实。熄灯后，谁因为一张小纸条翻身，谁还在被窝里背单词，旁边人都听得见。

县城晚上暗得早，小店门口的灯一亮，纸条上的玩笑就显得没有白天那么轻。

北京奥运已经过去几个月，后墙海报的颜色淡了一点，可年度回顾的新闻还会从小卖部电视里传出来。

那些大事件没有替他们做题，也没有替他们处理一张小纸条里的难处，只是在某个抬头的瞬间提醒他们：外面确实有更大的世界。

可眼前的世界仍然很具体：贴在新座位表旁边的一张小纸条、一只冻红的手、一句老师点名、一块还没扫干净的雪。

座位表是上午第二节课间贴出来的。陶建国把它贴在教室后墙红榜旁边，高度与前几次完全一致——左上角对齐红榜的第一根钉，右下角用手掌压平。他贴完之后没有立刻走，而是转过身来面对全班，把嗓门压到刚好盖过后排的窃窃私语。

"座位表旁边如果多出任何东西，我会查是谁贴的。"他说完这句话，用食指关节敲了两下后墙上的铁皮报栏，发出咚咚的闷响，然后走出了教室。

全班安静了大概三秒钟。然后，坐在第三组倒数第二排的何嘉树，从桌洞里撕了一角草稿纸，用黑色签字笔在上面写了两个字，趁大家都在往后墙涌的混乱中，把那角纸用透明胶粘在了座位表左边三厘米的位置。

那张小纸条上写的是：别哭。

两个字，笔画很轻。但用的不是开玩笑的歪斜字体——何嘉树的字一向很工整，这两个字写得尤其认真：三点水连着右边那个字的一横，像一个已经准备好的理由。

最先看到纸条的是余小满。她被调到了靠走廊的位置，和原来的同桌分开了。她挤到后墙前找自己名字时，先看到的是纸条上那两个字。她愣了一下，然后笑了一声。但那声笑没持续多久——她找到自己的名字后，脸上的表情从笑变成了抿嘴。

"谁写的？"她回头问。

没有人承认。但有几个人的目光同时投向何嘉树。何嘉树站在人群外围，手里攥着刚才撕纸剩下的那大半张草稿纸，纸的边缘有撕扯时留下的不规则锯齿。

付小川是第二个笑的。他站在余小满后面，看到纸条后笑出了声，但那笑声很轻，像是笑到一半发现不太好笑。"别哭——谁这么缺德？把人写哭了怎么办？"他嘴上这么说，眼睛却已经在人群里锁定了何嘉树。付小川认出字迹了——上周何嘉树帮他抄过英语笔记，那个"别"字的三点水偏旁，何嘉树习惯把第二点写成一个小勾。

"何嘉树。"付小川压低声音，用手肘碰了碰何嘉树的胳膊，"你干的？"

何嘉树没有承认也没有否认。他把那大半张草稿纸折了两道，塞进裤兜里。"余小满昨天跟我说，她和她同桌从初一就是同桌。现在调到走廊的另一端，两个人中间隔了整整一排。"

"所以你就写了别哭？"

"我只是觉得——"何嘉树把目光从后墙收了回来，"有些人的难过，没人替他们说，他们就不说了。"

后排的笑声还没完全收住，陶建国就已经回来了。他站在后门口，看了一眼墙上多出来的纸条，然后把目光移到了人群里。

"纸条谁贴的？"他的语气很平，但音量比平时低了一级——段璟后来觉得，陶建国每次真正生气的时候声音反而会压低，像是在给犯事的人留一个自己承认的缝隙。

何嘉树从人群里走了出来。"我。"

陶建国看了看他，又看了看纸条上的字，然后做了一件全班都没料到的事。他把纸条从墙上揭下来——不是撕，是用指甲小心翼翼地挑起透明胶的边缘，整张纸条完好地揭了下来。他把纸条放在讲台上，压在粉笔盒下面。

"跟我去办公室。"他说。

何嘉树跟在陶建国身后走出了教室。前排有人伸长了脖子往门口看，后排有人趴在桌上侧耳听走廊里的动静。段璟看见常俊凡从前排转过来看了一眼门的方向，然后转回去，继续做她的题。但她翻页的动作比平时慢了半拍——她的手指在纸的边缘停了片刻，像在等什么。

办公室里，陶建国让何嘉树坐在他办公桌对面的椅子上。那是一把硬板凳，木面已经被前面无数个被叫来的学生磨得发亮。陶建国没有坐下，而是站在窗户边，背对着光线，把他本来就不高的影子投在办公桌上。

"你帖子条是想安慰余小满对吧？"

何嘉树点了点头。

"余小满和你原来的同桌是不是一个人？"

"不是。但昨天她在水房洗脸的时候哭了。没有人知道。我是去接水的时候碰巧看见的。"

陶建国沉默了一会儿。他把窗台上的一盆兰花——那盆花已经快要枯死了，叶子黄了一半——往里挪了挪，然后转过来面对何嘉树。"你的出发点是好的。但贴在座位表旁边，不合适。"

"我知道。"

"你知道但你还是贴了。"

何嘉树低着头，看着自己膝盖上被校服裤子盖住的骨头形状。"因为如果我不贴，就没有人在那个时间点上告诉她——座位变了不要紧，有人看到了她的难过。"

陶建国走到办公桌前，拉开抽屉，从里面拿出一张白纸。他把那张白纸对折了一下，用钢笔在上面写了几行字。"下周班会，我会拿出五分钟，你在全班面前读一下这个东西。"

何嘉树接过那张纸，低头看了一遍。上面的内容不是检讨书，是一段关于"同学之间互相照顾"的倡议——陶建国的措辞很官方，但在最后一行，他用比前面小一号的字写了一句话："建议在表达关心前，先确认方式是否合适。"

"这不是惩罚。"陶建国把钢笔套拧上，"是你既然能注意到别人的难过，就应该也学会用最合适的方式让那个人接收到。"

何嘉树从办公室出来时，手里除了那张纸，还拿着他从墙上揭下来的小纸条。他把纸条重新折好，放进了校服胸前的口袋里。

他走过走廊时，余小满在水房门口拦住了他。她把一杯冒着热气的水递到他面前。"你用纸条安慰我，我用热水还你。"

何嘉树接过搪瓷杯，热气扑在脸上，镜片起了一层白雾。"你怎么知道是我？"

"你的三点水勾得太有特色了。"余小满说完自己笑了——不是刚才在红榜前那种笑到一半收住的笑，是真的从嘴角咧到眼角的笑。"而且全班只有你会在下课以后去水房看谁在洗脸。"

何嘉树低头喝了一口热水。金银花的味道很淡，更多是水本身被烧开后那种微微发甜的味道。他把杯子递回去时，手指在杯沿上停了一下。"下次别在水房哭。"

"那在哪儿哭？"

"教室也行。但别一个人。"

余小满把杯子抱在怀里。"好。"

第三节课间，段璟去水房接水时碰上了何嘉树。两个人站在左右两排水龙头下面，中间隔了一个正往暖水瓶里灌热水的低年级生。等那个低年级生走开后，段璟才开口。

"你今天干的事，换我是做不到的。"

何嘉树低着头冲杯子。"不是为了逞能。是真的觉得应该有人替她说一句话。"

"但你写完'别哭'之后，你知不知道全班人有大半都会笑？"

"知道。"何嘉树把水龙头拧紧，多余的水从杯沿溢出来，滴在水池边缘上。"但笑是一时的。而且笑完以后——你注意到没有，后来大家的笑声都不太自然。不是因为不好笑，是因为笑到一半发现，自己其实也怕被调开。"

段璟想了想，确实。当时后排的笑声此起彼伏了大概五秒钟，然后像被一道无声的浪拍了回去。不是有人喝止，而是笑的人自己收住了。因为笑完之后，他们都转身去找自己的新座位——新座位的旁边坐的人跟昨天不一样了。

"小纸条这种东西，"何嘉树端着水杯往门外走，在纱门旁边停了一下，"作用不是让所有人都不哭。作用是让哭的那个人知道有人看到了。哪怕只有一个。"

晚自习前，陶建国把那两行字从讲台上拿起来，在全班面前展开。"今天有人在后墙贴了张纸条。纸条上写了什么，我想大部分人都知道了。我不说这个做法对不对——让何嘉树自己说。"

何嘉树从座位上站起来。全班的目光从四面八方打在他身上。他把陶建国给他的那张倡议书展开，念了前面陶建国写的官方部分，念到最后一行时，他停了一下，然后把陶建国写的那行小字也念了出来。

念完之后，他加了一句自己的话："下次如果有同学因为调座位难过，不要贴纸条。直接去找那个人说。纸条会被风刮走，但当面说的话，不会被陶老师从墙上揭下来。"

全班笑了——这次是真的笑，时间很长，连陶建国都微微弯了一下嘴角。

下课后，段璟经过后墙时，发现有人在原来贴纸条的地方用铅笔写了两个字：好的。

字迹不熟。但段璟觉得，那可能是余小满。

他正要走开，付小川从后面拽住了他的书包带子。"你发现没有，今天调座位最倒霉的不是被调开的人——是被调到第一排的人。"

段璟顺着付小川的目光往前看。常俊凡正坐在第一排靠窗的位置，背挺得很直，手里捏着一支红笔在批自己的英语卷子。她没有回头看任何人，但从段璟的角度，能看到她的肩膀线条——校服棉袄的领口后面露出一点点深灰色的围巾边角，围巾上那一小块墨渍还在，刚好在肩胛骨的位置。

"她被调到第一排以后，一整天都没往后看过。"付小川把声音压得很低，"不是不想看，是知道自己在一排的第一排，回头等于告诉全班自己在意后面。"

"你怎么知道她在意？"

"因为课间我经过一排去接水，看到她草稿纸上写了一道数学题。那道题是孟老师上午布置的，她自己早做完了。但她旁边那页纸上，画了好几个三角函数的图像——sin和cos的曲线。你知道谁三角函数最烂吗？"

段璟没有回答。

"你。"付小川拍了拍他的肩膀，"她在帮你整理三角函数。坐在第一排，全班人都能看到她的桌面，她还是在你不知道的时候把你最烂的题型画了一遍。"

段璟站在原地，看着第一排那个背影。他想走过去说一句什么，但走了两步又停下来——不是因为怕被人看到，而是因为他忽然想到，她会在全班最显眼的位置偷偷帮他整理题型，说明这件事在她心里比在他心里更重要。不是量上的重，是质上的重——她选择了在最容易被看见的地方做一件不能被看见的事。

值日生开始擦黑板。粉笔灰在傍晚的斜光里慢慢落，黑板上的数学公式被擦掉了一半，剩下半个函数的曲线还依稀可辨。段璟帮值日生把讲台后面的垃圾桶拎到门口——今天轮到他值日倒垃圾。他拎着垃圾桶经过第一排时，常俊凡刚好抬头。

两个人对视了不到一秒。段璟先开了口："三角函数我今晚会做一遍。"

常俊凡合上草稿纸，把那一页画了曲线的纸压在英语课本下面。"知道公式就行，不用全做。"

"你写了那么多。"

"不是写给你一个人的——"她停了一下，把笔帽套上，"我们组的数学都不太好。我写完复印几份传下去。"

段璟知道这个解释不完全是真的。但他没有追问。他拎着垃圾桶走到教室门口时，常俊凡忽然又叫了他一声。"段璟。"

"嗯？"

"倒完垃圾早点回来。今晚马老师要抽查第八单元单词。"

他点了点头。拎着垃圾桶走过走廊时，纱门在身后弹簧的作用下咣当一声合上了。那声合门的回音在空荡荡的走廊里拖了两三秒，然后被水房那边传来的暖气管哐当声盖了过去。段璟把垃圾桶放在垃圾站旁边，往回走时在水房门口停了一下。何嘉树说的那句话还在他的脑子里转——纸条会被风刮走，但当面说的话不会被从墙上揭下来。他想，常俊凡刚才那句"倒完垃圾早点回来"，也许就是她版本的当面说话。没有写纸条，没有起哄，只是一个名字加一个提醒，刚好够他听出里面有一层比单词抽查更多的东西。

晚上回到宿舍，付小川已经从别人那里听说了何嘉树被陶建国叫到办公室的全过程。他躺在下铺的床上，把鞋子踢到床底，双脚搁在叠好的被子上。"何嘉树这个人，"他对着天花板说，"平时在班里不声不响，作文写得好但从来不当面念。今天忽然站出来承认纸条是他贴的——我都替他捏一把汗。"

"但他说了实话。"段璟蹲在地上刷牙，牙膏沫从嘴角溢出来，他用手背蹭了一下。

"实话是最难的。"付小川翻了个身，"我爸上次问我考得怎么样，我说还行。其实那次我考得更差。但我没办法说'更差'两个字。何嘉树能当着全班的面说出'贴纸条是觉得应该有人替她说一句话'——这种实话我恐怕一辈子都说不出来。"

段璟把漱口水吐进洗脸盆里。搪瓷盆内壁积了一层浅黄色的水垢。"那你以后尝试说说看。不用先从你爸开始，先从我。"

付小川从上往下看了他一眼，然后又把脸扭向天花板。"那我先说一件——今天调座位，我被调到梁一帆旁边。他的桌子边沿翘了一块木片，老刮到我胳膊。"他把右手举起来，小臂外侧确实有一条浅红的印子。"但我没说换座位，因为梁一帆上次数学考了全年级第三，我问了他一道题，他讲了三种解法。我觉得这个比胳膊上一条印值。"

段璟笑了一声。"这个实话挺好的。"

"还有一件事——"付小川把被子拉过头顶，声音从棉花和涤纶的纤维里闷闷地透出来，"今天何嘉树念完那个倡议书以后，我才发现我是那种笑完之后自己心里也在怕的人。我跟梁一帆其实不算熟，我怕的是换了个不熟的同桌以后，没人帮我讲数学题。你看，我怕的东西这么具体，具体的程度连我自己都不好意思说。"

段璟把搪瓷盆里的水倒进下水道，水花溅起来打在小腿上，凉得他打了个寒噤。"怕具体的事不可耻。怕抽象的事才可怕。"

"你怕的什么？"

段璟直起腰，把搪瓷盆搁回架子上。"我怕的东西也很具体。但我今晚不说。"

付小川从被子里探出半张脸，看了他一眼，又把脸缩了回去。"行。等你愿意说的那天，我不收你听实话的费用。"

一张小纸条之后，十班的很多关系都发生了一点小变化。不是突然亲近，也不是突然疏远，而是大家知道了谁会先冲出去，谁会先劝住人，谁会在最后留下来收拾。

何嘉树没有因此变成特别醒目的人，可在段璟眼里，何嘉树从这天起不再只是旁边一张脸。

常俊凡也不是只负责清醒。她会笑，会烦，会因为分数皱眉，也会在别人越界时把话说得很硬。

段璟慢慢明白，喜欢一个人不是替她解释一切，而是在她自己已经说清楚时，别再添上一层自己的想象。

十班最动人的地方，恰恰在这些不整齐里：有人懂事太早，有人幼稚太久，有人嘴硬，有人沉默，却都被同一间教室暂时收在一起。

到这一天结束时，座位调整把许多关系重新摆开，也把段璟和常俊凡的距离变得更清楚。十班没有因此变得更完美，只是每个人都比早上多知道了一点自己。

晚自习铃声照常响起，窗外的冷风也照常贴着玻璃走。段璟低头写题时，听见前排翻页的声音，忽然觉得这个冬天还很长。
