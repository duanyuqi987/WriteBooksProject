# 第107章 封校的周末

所属部卷：第二部：六月以后
时间：大一秋天
地点：大学南门和围墙边

## 诗歌插页 P107：原创仿唐绝句

墙草微凉近晚风，
少年心事隔帘中。
不知旧梦归何处，
一盏疏灯照水东。

周六早上七点半，陆允川被宿舍走廊里的说话声吵醒。

"封了！真的封了！南门铁栅栏上了锁，保安在旁边坐着，说不让出去不让进来。"隔壁寝室的人在走廊里打电话。

"你昨晚没回？那你在哪？"有人回。

"在网吧通宵。现在只能翻围墙。北边那段矮一点——"

"你别翻了，上个月有人翻墙摔骨折。"

陆允川坐起来。徐远还在睡，被子蒙着头，只露出一截充电线延伸到枕头底下——他插着耳机睡了一夜。祁野的床已经空了，被子叠成方块放在床尾，球鞋不在鞋架上。

手机上有两条消息。

一条是苏晚枝六点五十二分发的，在诗社群里："今天下午的读诗会照常。地点改到东操场看台，因为教学楼周末不开放。带一本你喜欢的诗集来，不是手机截图，是实体书。"

一条是祁野七点十五分发的："我在体育馆。北边围墙有段矮的，好多人在那看外面。你来不来？"

陆允川穿好衣服，去水房洗脸。水房里有三个男生，一个在刮胡子，满嘴泡沫，对着镜子仔细修鬓角；一个在洗袜子，洗衣液的泡沫从指缝里滴到拖鞋上；还有一个靠在门框上刷牙，眼睛还是闭着的。

刮胡子的男生说："今天整个学校都封了，你刮胡子给谁看？"

"给我自己看。"

"你连女朋友都没有。"

"我长得帅不行吗？"

陆允川洗完脸，对着镜子拨了一下额前的头发。镜子边缘写着"水房文明公约"几个字，第三条是"节约用水"，底下画了一个哭脸。他往宿舍走的时候在走廊窗边停下来看了一眼北围墙方向——果然能看到几颗人头在围墙内侧晃来晃去，像是在看墙外。

北围墙在体育馆后面那一截确实矮。墙头一米八，下面垫了几个旧砖头，被历届学生踩得光滑发亮。陆允川走到那里时，大约有七八个人聚在墙根下。

一个穿格子睡衣的男生站在砖头上，手扒着墙头，只露出额头和眼睛看外面。旁边的人在下面催："看到啥了？"

"看到了个卖煎饼的摊。"

"还有呢？"

"还有城管在追他。"

祁野坐在旁边双杠上，球鞋踩在下面的横杆上。"来啦？"他拍拍旁边的空位，"坐这儿。"

陆允川没有坐双杠，靠在旁边的铁制爬梯架上。"你早上几点起的？"

"六点。"祁野说，"操场跑完五公里。然后看到通知说体育馆周末开放但限流——一次最多二十个人。"

"那你打球了没？"

"打了。占了个半场，练了一个小时投篮。现在脚踝有点酸。"他低头转了转脚踝，"不过不影响。"

这时墙头的睡衣男生喊了一声："煎饼摊被赶走了！城管的车开过来了！"

底下的人笑起来。有人起哄："实况转播啊，兄弟。"

陆允川和祁野沿着围墙往北走。北围墙有段区域长满了构树，树冠从墙外伸进来，枝丫上挂着褐色的果子。地上落了一片，踩上去黏鞋底。

"你下午去诗社活动？"祁野问。

"嗯。东操场看台。"

"苏晚枝组织的那个'词与物的距离'？"

"对。"

"她真会取名字。'词与物的距离'——这不就是'你写的东西到底像不像真东西'的意思？"

"差不多。"

"那我也去听。"

陆允川看了他一眼。

"干嘛？我一个体育生不能听诗？"祁野说，"我今天没安排。下午不打球，晚上不训练，封校又不能出门，闲着也是闲着。多了解一点不同领域的知识。"

"你带诗集了没有？实体书。"

"没有。你不是有吗？借我一本。"

宿舍里陆允川翻出三本诗集：《艾青诗选》《海子诗全集》和程砚寄来的那本还没拿到的《里尔克诗选》。他把《艾青诗选》给了祁野。

"这个好读吗？"

"'为什么我的眼里常含泪水，因为我对这土地爱得深沉'——艾青的。你应该听过。"

"听过。高考作文里被引用烂了的那句。"祁野翻了两页，"不过他写土地确实很实在。不是瞎抒情。"

下午两点，东操场看台。

来看台参加读诗会的只有五个人：苏晚枝、叶澄、陆允川、祁野，还有一个大四的学长叫沈让，戴一顶灰色的鸭舌帽，帽檐压得很低，据说是上一届诗社的副社长，已经退社了，今天是"路过来看看"。

苏晚枝坐在看台第二排，旁边放了一个布袋，里面是三本诗集和一本硬壳笔记本。她今天没戴眼镜，头发扎了低马尾。陆允川注意到她没戴眼镜的时候眼睛其实很大，只是常年戴眼镜让人忽略了这个事实。

"人比我想的多。"苏晚枝开口。

"五个算多？"沈让把帽檐往上拨了一下。

"我以为只有两个。"

"你自己加叶澄？"沈让说。

"我自己加叶澄，再加一个陆允川。三个。"苏晚枝指了指祁野，"这位是编外人员。"

"体育教育，祁野。"祁野自我介绍，"我对诗不太懂，但我对'词与物'这个话题有兴趣。因为最近发现很多词语在日常生活里其实是不准的。"

"比如？"苏晚枝问。

"比如'坚强'。别人总说我坚强。但我不觉得我坚强，我只是不方便在球场哭。"

大家都安静了一下。然后苏晚枝笑了——不是平时那种礼貌的短笑，是真的一下子被逗到了。

"好。"她说，"我们今天就从'坚强'开始。"

读诗会持续了一个半小时。五个人轮流读自己带的诗集，然后讲一段为什么选这首诗。沈让读了博尔赫斯的一首《雨》，说他喜欢那首诗里"雨"是具体的、有声音的、能淋湿人的，而不是一个象征。叶澄读了顾城的一首短诗，讲了她对"门"和"窗"的理解。陆允川读了艾青的《树》，说两棵树在地面上分开很远、在地底下根缠在一起——这很像人和人之间的距离。

苏晚枝最后读的是她自己写的诗。她先说明"这不是名家作品，是我上周写的，还在改"，然后念了。诗很短，写的是她高二的冬天，有一天放学后站在学校天台上看雪，旁边有只流浪猫蹲在通风管下面。诗的最后两句是：

"雪落在我和它之间/谁也不问谁的名字。"

念完以后没人说话。

沈让第一个开口。"那只猫是真的吗？"

"真的。回家路上我还给它买了火腿肠。"

"那就不需要改了。"沈让说。

祁野走的时候对陆允川说："你那个学姐的脑子，跟我们不一样。"

"怎么不一样？"

"她能把高二的猫和雪写到一首诗里，而且不煽情。"祁野把《艾青诗选》还给陆允川，"我觉得我大概懂了一点什么叫'词与物'。就是——猫是真的猫，雪是真的雪，那个天台上真的有一个她——这就够了。不用再加一只'代表孤独的猫'或者'象征纯洁的雪'。"

天快黑时，陆允川独自走了一遍围墙边的窄路。围墙根泥土里有野草被踩平的痕迹，有几只被雨打蔫的烟头。一个女生蹲在围墙边用树枝挖着什么，陆允川路过时她抬头说："我在找蚯蚓，实验课要用。别告诉我这是封校期间不让挖蚯蚓。"他说"不告诉"，继续往前走。

走到东门附近时看到外卖骑手们又在递东西。有个骑手举着一盒奶茶过门缝，喊："谁点的杨枝甘露？常温还是冰的——喂——这单没写——"

门内侧一个女生举手："我的我的，常温。"

陆允川在门边站了一会儿，然后转身往食堂走。今天晚上食堂继续有"封校特供窗口"，铁板豆腐的师傅已经认识祁野了，远远看见他就喊："今天辣椒自己加，免费。"
