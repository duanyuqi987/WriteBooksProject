# 第220章 旧日记本多年后

所属部卷：第五部：逝水有灯
时间：多年后
地点：陆允川书房

## 诗歌插页 P220：原创仿唐绝句

旧封微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

多年后陆允川重写旧日记本，先删掉了一句话。

原稿写着："那个日记本是他青春时代的遗物，每一次翻开都是一次挖掘。"他删掉了"遗物"和"挖掘"。改成："那个日记本是他搬家时从纸箱里翻出来的。封面磨白了，里面写了很多重复的话。"

第二次修订时，他把"很多重复的话"改成了具体的内容："她在三楼食堂出现了二十三次。每次他都记了时间。"

第三次修订时，他删掉了"二十三次"，改成"很多次"。不是怕数字显得琐碎，是他数了三遍，三遍的数量都不一样。他不能确定那是不是全部的碰面。那本日记只记录了他意识到的，没记录他没有意识到的。

所以他不能把"二十三次"写成一个确定的答案。那不是事实。

程砚有一次问他："你不觉得反复改一个数字很浪费时间吗？"

他反问："你觉得事实重要还是效果重要？"

程砚说："你的小说你做主。"

"那事实重要。"

"那就数清楚。"

"数了三遍。都不一样。所以不是二十三次，是很多次。"

程砚点了点头，没有再说话。

这本小说改到最后一轮的时候，陆允川把所有从日记本里引用的内容都核对了一遍。他发现旧日记里有些时间标注错了——比如2016年3月14日记的是"周六下午"，但查日历，那天是周二。他不知道当时是写错了还是记错了。两种可能都有。

他在小说里保留了"周六下午"。不是因为懒惰，是因为那个错误本身就是信息——年轻的他在日记里把周二过成了周六的样子。那种时间感的错乱，比准确的日历更接近他当时的心理状态。

错误的日期不需要被修正。只需要被理解。

苏晚枝在小说定稿前收到了他的消息。不是关于日记本的——是关于书稿的最后确认。

"面馆那章之后还有几章。写到你了。你看不看？"

"看。发过来。"

他发了出租屋那几章。

她看完以后问："你墙上真的有块剥落的漆？"

"有。"

"你真的没补？"

"没补。有瓶补墙膏在抽屉里。放了三年。"

"后来呢？"

"后来搬家的时候扔了。"

"补墙膏还是漆？"

"补墙膏扔了。漆还在墙上。我走的时候它还在。"

"你觉得那面墙现在已经补上了吗？"

"不知道。可能补了。可能没有。"

"那你希望它补了还是没有？"

他想了一会儿。"希望它还在。但不是为了我的小说。是因为那是那间出租屋的印记。不是我的。"

苏晚枝没有回复这一句。

过了三分钟，她发来一张照片。照片是她自己的出租屋里的墙。角落里有一道很细的裂缝，从天花板一直延伸到插座旁边。

"我这儿也有一个。不是剥落。是裂缝。住了两年。不打算补。"

他问："为什么不补？"

她回："因为它刚好在床头。每天晚上关灯的时候能看见。"

他把这张照片存在手机里。不是放在"写作素材"文件夹。是放在"日常"文件夹——和辣椒酱洒了的照片、楼下豆浆摊的照片、搬家车装满纸箱的照片放在一起。

日记本和裂缝。出租屋和毕业照。所有这些东西，在他的电脑里最终被放进同一个根目录。目录名叫"逝水有灯"。

他打开这个目录，看了一眼文件数量：四十五个章节文档。从头到尾，从"信纸之外"到"打字声轻了"。每修改一版，文件后面就多一个版本号的后缀。现在所有后缀都删掉了。只留最终的干净的文件名。

他打开最后一个文档。写最后一句。

光标停在一行字的结尾。

他打了句号。

存了档。关了电脑。

窗外的路灯亮着。楼下没有敲铁勺的声音，没有空调滴水，没有隔壁的煤气费谈判。新公寓的夜晚安静得像一道空白的页面。

但空白的页面不是空白的。它等着一行新的开始。
