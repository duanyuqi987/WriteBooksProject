# 第055章 高三倒计时多年后

所属部卷：第一部：初秋三楼
时间：多年后
地点：旧照片文件夹

## 诗歌插页 P055：原创仿唐绝句

黑板余痕照片中，
倒时数字已成空。
少年各自奔前路，
不只相思在晚风。

陆允川在电脑里找到一张高三教室照片。

照片里没人正对镜头，大家都在低头做题。黑板右上角写着 178，粉笔字有点歪。那时他大概只是随手拍，想留住高三的某个午后。多年后再看，最先看见的不是谁的座位，而是整间教室压低的肩膀。

每个人都很累。

这个认识来得很晚。年轻时，他总把自己的心事放大，觉得自己一边喜欢，一边备考，格外艰难。现在看，那间教室里没有谁轻松。夏知遥不轻松，程砚不轻松，林曼不轻松，连总考第一的人也不轻松。

倒计时对所有人都一样。它不因为谁暗恋就慢一点，也不因为谁疲惫就停一停。

陆允川把照片放进“高三九月”的文件夹。里面还有作文模板、旧课程表、程砚发来的错题照片。他没有再单独截出夏知遥的座位。那样做会把一整间教室缩窄，缩成他一个人的青春。

他打开硬皮本，写下：

高三不是背景。
它是所有人共同经过的一段窄路。

写完以后，他想起夏知遥在黑板前改数字的样子。她写下 278，粉笔断了一截。那一幕以前被他理解成温柔，现在更像一种用力生活的安静。

他合上电脑。屏幕暗下去时，照片里的教室也消失了。倒计时早已结束，可那些人各自往前走的样子，忽然比任何心事都更清楚。
