# 第104章 黑板报边角



时间：2010年五月上旬，竞赛名单公布后



地点：教室后墙、年级办公室门口



卷次：第五卷：未来在黑板上倒计时



2010年五月上旬，竞赛名单公布后，教室后墙、年级办公室门口，贴在黑板报旁边的物理竞赛名单把2010年的春夏推到十班面前。物理竞赛名单贴出来，周明澈入选，段璟差一名没有上榜。

周明澈不敢高兴得太明显，段璟也不想显得小气，常俊凡把两个人都拉回题目。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

周明澈也在这段日子里显出自己的位置。稳，能管事，成绩均衡，习惯先照顾集体，自己的压力常放到最后。群像不是名单，群像是每个人都有自己的难处。

“可能只是这次题型刚好。”周明澈说。

“上了就是上了，别解释。”段璟说。

“解释没用，错题有用。”常俊凡说。

“入选的准备竞赛，没入选的准备下一次考试。”胡向南说。

“老师这碗水端得真平。”付小川说。

竞赛名单贴在年级办公室门口，纸不大，围着的人却一圈又一圈。

入选的名字被人很快找到，没入选的人也会把名单从头看到尾，像名字会突然补上去。

周明澈站在旁边，没有立刻笑。段璟看见自己的名字不在上面，先把书包带往肩上提了提。

常俊凡没有安慰得太轻。她知道落选最烦的就是别人替你把失落说得不值一提。

胡向南把几张训练卷发下来，说竞赛是另一条路，不是所有人的路，也不是唯一的路。

晚自习时，入选的人去办公室听安排，教室里空了几个座位，大家才真正感觉到差距。

段璟低头改错题，发现自己差的不是一个名单，而是几道题里没补上的步骤。

竞赛名单贴了三天，纸角开始卷起来，心里的那点不服却没有那么快平。

胡向南这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

周明澈被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

何南星站在竞赛名单前看了很久，竞赛名单的纸角已经微微卷起。十班的热闹还在，只是每个人都比去年更会藏心事。

邓桥把训练卷卷起来塞进书包，新华书店塑料袋在手里沙沙响。老师从门口经过，原本散开的声音立刻往回收。

余小满在错题旁边补上一步推导，桌洞里的练习册露出红色封皮。这样的细节不会被写进红榜，却会留在很多人的夏天里。

曹蔚从办公室门口慢慢走回教室，树莓摊旁边的纸盒被太阳晒软。这点动作很小，却把那天的压力落到了实处。

周明澈站在竞赛名单前看了很久，黑板报边角的透明胶又翘起来。旁边的人没有追问，翻书声把难堪遮住。

付小川把训练卷卷起来塞进书包，家长的电动车铃声从校门外传来。常俊凡看见了，没有替谁下结论，只把题号重新写清楚。

何嘉树在错题旁边补上一步推导，饭卡机短短响了一声。段璟后来才知道，未来最早就是这样从小事里露头的。

竞赛名单让十班明白，优秀也会制造距离，落选也要体面地继续学。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，贴在黑板报旁边的物理竞赛名单还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
