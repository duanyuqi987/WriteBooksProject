# 第096章 大二

时间：2014年9月

地点：楚州师院

卷次：卷五：各自南北

九月。大二了。

新生报到那天——他在宿舍窗口往下看。楼下同样的场景——拉着行李箱的家长和学生从红色横幅下走过去。横幅换新了——"欢迎2014级新同学"——布上的金边还没褪。去年这个时候——他站在那个横幅下面——一个人——拎着蛇皮袋。今天他从五楼窗口往下看。

李博在背后整理书架——书脊朝外排得整整齐齐。"看什么呢。""新生。""感慨了？""还好。"

时间过得快。大一一年——从九月陌生到六月习惯——教室、食堂、图书馆、操场——走过同一条路线。操场跑道跑出了磨痕。电话亭玻璃换了新的——鸟粪痕迹没了。不联系苏晚晴——已经一年多。三百九十多天——没发过一条短信——没打过一次电话——没路过师大北门。301路每天从师院南门发车——早上六点到晚上九点——每十二分钟一班——他从来没坐过。

许若琳大二更忙了——英语系课多——专业课加二外日语。文学社来得少了——偶尔QQ发一句——"最近社团怎么样"——"还行"——"新社长是写诗那个师兄"——"对"。不再频繁提苏晚晴——他也不问。话题从"晚晴最近在做什么"变成了"日语比英语难学一百倍"。

暑假支教过去四个月了。李家沟小学——太行山深处——两排平房、土操场、一根旗杆。教室没灯——白天靠窗户自然光。宿舍是学校腾出的教室——木板隔成两间——床是课桌拼的——被子是学校老师从家拿的。夜晚山里冷——七月太行山晚上得盖两床被。他教了十四天语文——二十一个孩子——每天四节课。孩子们坐在歪斜的木课桌后——字歪歪扭扭——但每个都很用力。讲《静夜思》——"床前明月光"——有个孩子举手——"老师你家离这里多远"——他说不远——但其实三百多公里——要转三趟车。走那天——小女孩拉着他书包带——"老师你明年还来吗"——他说"来"。

大二课比大一深。古代文学进入唐宋——李白杜甫白居易。现代汉语进入语法结构——"主谓宾""偏正结构"。书法课从颜体转到行书——他每天在宿舍水泥地上铺报纸——毛笔蘸清水临《兰亭序》。阿坤有一次踩到他报纸——"兄弟你这练习环境也太艰苦了。""省钱。""你不是省钱——你是不想让自己闲着。"他没反驳。

九月下旬。收拾书桌——抽屉最深翻出高中的暗恋日记。蓝色封面——边角卷了——压在一堆笔记下一年多。

翻开第一页。"2010年9月13号——她今天换了发绳——以前是黑的——今天是浅蓝色的。"往后翻——"2011年5月20号——她物理72分——卷子发下来咬了咬嘴唇——我觉得她有点难过。"她今天穿什么颜色的衣服——她笑了几次——她考了多少分——她跟谁说了什么。不是日记——是暗恋者的记账本——怕忘记——怕哪天醒来不记得。

最后一篇停在2013年8月29号——"她说了那句话——暂时不要联系了——我答应了。"

合上日记。放回原位。窗外有新生在军训——喊口号声很远——透过玻璃只听得见嗡嗡共振。打开手机——通讯录——S——苏晚晴。电话号码还在——没有删——这辈子不会删。但也不会打。

放下手机。翻开专业书。继续看《古代文学》。翻到了李白的《将进酒》——"君不见黄河之水天上来，奔流到海不复回。"他在诗行旁边的空白处写了几个字——"两年了"。字迹很轻——铅笔写的。写完以后把铅笔横过来——用铅笔的橡皮那头在字上擦了两下——字变淡了，但没有完全消失。留下了灰色的痕迹。

窗外新生军训的口号又响了一阵——"一二一——一二一——"——声音从操场方向传过来，经过两栋楼的折射，到512的时候已经变了形，"一"拉长了，"二"缩成了短音。他们去年也是这样——阿坤军训第一天就被晒脱了皮，小伍的解放鞋太大，踢正步的时候鞋掉了一次。那个时候才开学——宿舍里四个人谁也不认识谁。阿坤带来的键盘还是旧的薄膜键盘，李博的值日表还没打印出来，小伍写的信纸还是从家里带的横线纸——不是熊猫信封。

三百九十多天过去了。阿坤换了机械键盘。李博的书架从一格变成了两格。小伍的信已经攒满了一个鞋盒。他的暗恋日记停在两年前的最后一行。

他把专业书翻到下一页。李白之后是杜甫——"飘飘何所似，天地一沙鸥。"看完这几行——他把书合上了。沙鸥。沙鸥是在江上看不清岸的鸟。他在本子上画了一只鸟——用铅笔画了几笔。鸟在水上面。然后他看看——把鸟改成了点。然后合上笔记本。明天还有课。睡吧。

但是他没有马上睡。

他坐在床边。脱了鞋。脱了袜子。赤脚踩在凉水泥地上——九月末的地面已经不热了，从脚底凉上来。他弯腰捡起今天走圈时掉在地上的一颗红色跑道颗粒——小小的、扁扁的。他看了看——把它放进了床脚的鞋盒里。鞋盒里的红色颗粒已经堆成了薄薄的一层——有些颗粒还有棱角，有些已经磨圆了。他关上鞋盒。抬头看了看窗外——窗外对面那栋灰色的宿舍楼还亮着几个窗户，一格一格的，有的窗户里能看见人影在走动。

他的手机亮了。屏幕的光在暗处打在他的手指上。是赵一鸣发来的QQ消息——"你俩最近怎么样了。"他看了几秒钟——打了三个字："老样子。"赵一鸣回："还不联系？"他说"不。"赵一鸣说："你这种人——电视剧里活不过三集。"他回了一个字"嗯"。然后按灭屏幕。

躺下。天花板的裂缝在路灯光里若隐若现。两年了——七百多天。每一个早晨的闹钟。每一个包子和豆浆。每一个第三排靠窗的座位。每一页笔记、每一圈跑道、每一盏绿色台灯。每一天都在走同一条路——从宿舍到教室，从教室到食堂，从食堂到图书馆，从图书馆到操场，从操场回宿舍。每一次路过南门的公交站——站牌下的绿铁皮被太阳晒出了新的锈斑。301路的路线他倒着都能背出来。那些站名像刻在脑壳里面的——抠不掉。

他翻了个身。把被子拉到肩膀。手机压在枕头下面——屏幕是暗的。明天早上李博还是第一个起床——叠被子的声音还是有节奏的啪啪啪。阿坤还是最后一个起——闹钟响了三次才把头从被子里拔出来。小伍还是会吸鼻子。他还会坐第三排靠窗。日子是重复的。重复里面藏着等——等着"暂时"变成"可以"——等着十二公里从不能走变成可以走——等着那条路线不再只是脑子里的一条线。

他闭上眼睛。呼吸慢慢变平。窗外偶尔有人晚归——脚步声由远到近——由近到远——声控灯亮了一下——灭了。然后整个走廊、整栋楼——都安静了。
