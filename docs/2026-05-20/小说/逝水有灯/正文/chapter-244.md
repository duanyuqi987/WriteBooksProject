# 第244章 写成小说旁观

所属部卷：第五部：逝水有灯
时间：朋友读稿后
地点：书桌和窗边

## 诗歌插页 P244：原创现代诗

朋友读完
没有说你终于赢了
他说这里还疼
但疼不是证据
也不是奖章

程砚读得比陆允川预想中认真。他在周末晚上打来电话，第一句不是夸，也不是安慰，而是：“前半段好多了，后半段还有几处你太想替自己结案。”

陆允川打开文档，按他说的页码跳过去。程砚指出几处：写苏晚枝沉默时，语气还像在等她解释；写夏知遥回乡时，还带着一点“她不知道我多难过”的影子；写朋友劝告时，有些地方把朋友写成工具。

“你别急着让读者原谅你。”程砚说，“也别急着让读者觉得你成长了。你把事写清楚，剩下的让他们自己看。”

叶澄的反馈更短。她只在文档旁批注了一句：这里让她自己选择，不要替她温柔。那条批注停在苏晚枝拒绝之后的章节旁，像一枚小小的红灯。

陆允川没有辩解。他把几处作者味太重的段落删掉，换成动作：她关掉手机，她把文件夹抱紧，她在雨里撑伞离开，她保存照片后继续工作。

旁观者的价值就在这里。他们不负责替作者舒服。他们把那些还想美化自己的地方指出来，让小说回到人，而不是回到陆允川的自我证明。
