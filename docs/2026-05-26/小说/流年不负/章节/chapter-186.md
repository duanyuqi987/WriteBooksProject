# 第186章 女儿

时间：2021年5月
地点：楚州妇幼保健院
卷次：卷十：不负少年

走廊里的椅子是铁的，扶手冰凉——医院里的空调开得足。椅子坐久了屁股硌得疼。他把外套叠了叠垫在屁股底下。薄薄的——没什么用。

顾临风坐在第四张椅子上。旁边是一个饮水机。每隔几分钟就有人来接水，纸杯撕下来的声音——咔。水桶里冒泡的声音——咕嘟。咕嘟。桶里的水快见底了。还剩大概一拳高。没人来换。他盯着那个水桶看了很久——水面上有个小小的气泡。从桶底升上来。升到一半。破了。

他从上午十点坐到了下午一点。三个小时。墙上有个钟——圆形。白底黑针。秒针走得不准——跳一跳停半秒。然后继续跳。他把手机掏出来看了一眼。跟墙上的钟对了一下。慢了大概两分钟。

中间站起来了两次。一次去厕所——洗手的时候看了眼镜子。镜子里的那个人眼眶下面发青。胡茬冒出来了。用手蹭了一下下巴——扎手。一次去窗户边——外面在下雨，五月的雨不大不小。雨滴从窗玻璃上滑下来——弯弯曲曲的。他看了大概五分钟的雨。想：生下来以后如果下雨，就叫念念——念念不忘。如果晴天——还叫念念。反正已经想好了。

旁边来了一个中年男人。手里也攥着一顶帽子。帽檐被手指掐出了一个凹。两个人互相看了一眼。没有说话。

走廊尽头的手术室。门关着。上面的灯是红色的。

赵一鸣打来电话。他接了。

"怎么样。"

"还在等。"

"你别紧张。"

"我不紧张。"

"你声音不像不紧张。"

"信号不好。"

他把电话挂了。攥着手机。屏幕亮了。又灭了。又亮了——他妈发来一条消息:"生了没有"。他打了三个字:"还在等"。发送。

然后把手机放进口袋。

又拿了出来。看了一眼。放回去。

一点二十分。手术室的门开了。铰链响了——润滑剂大概干了。吱呀一声。一个护士出来——口罩拉到了下巴。手里拿着一个夹子。夹子上夹着几张纸。她低头看了夹子上的名字——抬头扫了一眼走廊。

"苏晚晴家属。"

他站起来。膝盖有一点软。像蹲久了猛地站起来那种——大腿前面没力。他用手扶了一下饮水机——水桶晃了一下。稳住了。

"在。"

"母女平安。顺产。六斤四两。恭喜。"

"她——"

"在里面。等一会儿转移病房。你先去办手续。出门右拐。窗口。拿着这个——"她撕了一张纸条给他。"产妇目前状态稳定。别担心。"

"好。"

护士转身进去了。门又关上了。吱呀——铰链又响了。

他站在原地。手里握着手机。手机硌着手心。手汗把屏幕糊了一层——字看不太清。他低头看了一眼——手在抖。不是紧张——是刚才攥了太久的拳头忽然松开了。血液回流。手指发麻。微信的输入框里打了一串乱码——smaoshjd——他把乱码删了。打了四个字:"母女平安"。发给了妈。发给赵一鸣。发给她妈。发给所有联系人。

然后去办手续。在窗口排队。前面还排着三个人。他站在原地等了七分钟。他把队排完了。交钱。签字。签字的时候笔从手里滑了一下——手心里全是汗。他把笔捡起来。签完了。工作人员看了看他的名字:"顾临风——是'临风'不是'林峰'？""是。"

手续办完。回到病房。

门虚掩着。

他推开。

房间里有两个人。

苏晚晴靠在病床上。枕头叠了两个，垫在腰后面。脸上全是汗。头发贴在额头上和脸颊上。有些头发从马尾里散出来了——贴在脖子上。脸色不是白的。是薄——跟孕吐时一样，皮肤底下透出青青的血管。

但她眼睛是亮的。

她怀里抱着一个白色的小包裹。包裹里露出一个小脑袋——头发不多，几根，黑的，贴着头皮。脸皱皱的。眼睛没睁开。嘴巴动了一下——很小很小的幅度，像在梦里吃东西。

她抬头看了他一眼。

满脸是汗。

然后弯了弯嘴角。

笑的样子和高中时一模一样。先弯眼角。嘴角慢了半拍。

他想说点什么的。

但他说不出来。嗓子像被什么东西堵住了。

他走到床边。没有坐下。站着。低头看着那个小包裹。小脸上有一层淡白色——胎脂。小拳头攥着。五根手指，每根都只有他大拇指指甲盖那么大。

指甲是透明的。

他伸出手。用食指摸了一下她的拳头。食指比她的拳头大很多——他的手指盖住了她整个手背。

小拳头张开了一下。五根手指张开了——像一朵慢慢绽开的花。然后又攥紧了。把他的食指尖攥在了手心里。攥得很有力——不像是个刚出生几个小时的婴儿。手指甲盖是透明的——能看到底下的毛细血管。一丝一丝的粉红。

他低头看着那一小团手指包着自己的食指。看了很久。久到她把手松开了——她睡着了。手指慢慢松开。他的食指尖从她手心里滑出来。

他愣了很久。

抬起头。看苏晚晴。

嘴唇动了动。又闭了。又动。最后说出来了。声音比平时低——不是刻意的。是声带被什么东西压住了。

"辛苦了。"

她说:"不辛苦。"

她说的时候，声音有一点哑。嗓子眼里像是还有没倒干净的空气——大概是刚才用力的时候喊过。产房里的声音——他听不见。手术室的门太厚了。但现在她的声音很轻。轻到他耳朵里只剩这三个字。她清了清嗓子。然后把怀里的小包裹往他这边斜了斜。包裹动了——里面的小脸皱了一下。嘴巴张了张。打了个哈欠。

"你抱抱。"

他伸出手。护士在旁边教他——"这个手托着头，这个手托着腰——对——托着脖子——脖子太软了——对——再往上一点——"

他托着。两只手臂像个支架。硬邦邦的。不敢弯——怕弯的角度不对。

小包裹比他想象的重。但不太重——六斤四两。一个五公斤的哑铃。但比哑铃软。比哑铃暖。包裹里面有热气往外冒——他把包裹贴近胸口。

他弯腰。把脸凑过去。凑得很近。婴儿的呼吸喷在他鼻尖上——凉凉的。闻到了一股味道。不是香。是一种淡淡的、带一点奶味和血的混合味道。还有一点消毒水——医院里。还有一点——她说——"是胎脂的味道。不好闻但以后你闻不到。"

他轻轻吸了一下。那股味道进入鼻腔。他记住了。

然后把包裹还给了她。不是不想抱。是他的手抖得太厉害了。他怕自己手滑。她接过去的时候——手稳的。刚生完孩子的手。稳的。他知道她的手为什么稳——因为她抱着的是比他重一百倍的东西。

"叫什么名字。"

"And you——already have one picked out."

她说的英文——大概是产房里跟医生说的，还没切换回来。发音有一点含糊。她可能自己都没意识到在说英文。话出口之后她自己顿了一下——然后笑了。很轻——嘴角弯了一下。

他愣了一拍。

然后她说:"名字不是你想好了吗。"

"我想了几个。等你选。"

"先叫她——"她低头看着那个小脸。包被里露出来的小耳朵——小小的。耳廓上的软骨还是软的——折了一下。她用手指轻轻把那片小耳朵拨开。拨顺了。"念念。先叫她念念。"

"顾念。"

"嗯。顾念。念念。你取的名字。"她念了一遍。"像一首诗。"

"就是诗。"

"哪首。"

"还没写。第一句:你出生那天下雨。第二句:爸爸在走廊里等了三个小时。"

她笑了一下。笑得很轻。医生说了别用力——连笑都不能太用力。

他坐在床边那张硬椅子上。椅子离床大概半米。他把椅子往前拉了拉——拉到床边。椅子腿蹭在地板上——吱——他赶紧停住了。怕吵到隔壁床。然后坐下来。膝盖贴着床沿。左手放在床沿上——离她的手大概十厘米。她把手放过来——搭在他手腕上。

窗外还在下雨。雨点敲在空调外机上——嗒嗒嗒。嗒嗒嗒。嗒嗒嗒。雨比刚才小了一些。窗台上的一个小盆栽里积了一点水——水面上倒映着窗外的天。灰色的。

病房里很安静。隔壁床的产妇在睡觉。轻微的鼾声。陪床的是她丈夫——坐在折叠椅上。手里拿着一本育儿手册。大概是看过了——因为他的眼睛是闭着的。书上翻开的那一页有一张图——新生儿脐带护理。脐带的示意图——蓝色和粉色的。他看了一眼。把目光移回女儿身上。

她在给女儿喂奶。第一次。姿势不太熟练。她把女儿横着抱——护士在旁边指导——"头再高一点——对——让她的嘴张大——不是你自己往上凑——你放松——"。她咬着下嘴唇。眉头拧着——很认真。像做高考理综的物理最后一道大题。女儿找了一会儿——嘴里发出吧嗒吧嗒的声音。小嘴在襁褓上蹭来蹭去——找错了地方。她把女儿的嘴轻轻挪过去——"在这里——这里——对——"。找到了。开始吃。很慢。很轻。小嘴在慢慢地哺——喝进去一口。咽下去。然后停一会儿。然后又喝。不着急。

她低头看着女儿。眼睛半眯着——里面有点湿。大概是累的。大概是别的。

他看着她们。窗外有救护车开过去——警笛响了。由近到远。声音穿过雨——变哑了。她没抬头。女儿也没醒来。这个房间里——只有吮吸的声音。只有雨。只有他。

窗外的雨声小了一些。大概是云飘走了。外头的梧桐树被雨水洗得很绿。叶子上挂满了水珠——一闪一闪的。天边裂了一道缝——窄窄的——透出一点淡金色。天快晴了。

他把手放进上衣内袋里。摸到了一个小本子——巴掌大小。封面是牛皮纸。他从护士站借的笔还在口袋里——一支蓝色的圆珠笔。笔盖咬得有牙印——不是他的。大概是护士的。他本来想今天写的。但发现写不出来。手放到本子上。又放下。又放上去。钢笔尖戳在纸上的第一行空了一秒。然后他把本子合上了。

有些东西不该写成诗。

有些东西，就是活着的样子。

他把本子放回口袋里。站起来。走到窗边。天边的裂口更大了——光从云缝里打下来。一道。两道。三道。照在远处楼顶的水塔上。水塔的灯白天不亮——晚上才亮。红色的。绿色的。红的。绿的。跟月湖边那个一模一样。不是在青阳。是在楚州。每个城市都有这样的水塔。但只有他知道——有一个水塔。在人最紧张的时候可以用来数灯:一。二。三。四。

他回到床边。坐下来。看着念念吃奶。窗外的光正好打在她脸上。小脸上的奶汁还没擦干净。一点白色的。挂在嘴角。她伸出舌头顶了顶自己的嘴角——没顶到。然后放弃了。继续睡。

五月。女儿出生。
