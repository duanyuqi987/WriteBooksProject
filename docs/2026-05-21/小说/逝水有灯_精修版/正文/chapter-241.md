# 第241章 写成小说之前

所属部卷：第五部：逝水有灯
时间：多年后的清晨
地点：书桌和窗边

## 诗歌插页 P241：原创现代诗

我把修订稿放到晨光里，
不再让它替谁作证。
风从旧年的缝隙穿过，
纸页轻响，名字安静。
后来灯慢慢亮了，
人也终于学会往前行。

写成小说发生之前，生活先把场面摆好。

清晨六点，陆允川坐在书桌前。他是被闹钟叫起来的——闹钟设的是六点，但他在五点半就醒了。书稿还剩最后一轮删除就完成了，脑子自己提前激活了。外面的天还是灰蓝色的，街道上只有零星几盏路灯还在亮着，远看像一排还在值夜班的哨兵。

窗户开着，风吹进来，带着早晨特有的一股湿凉——楼下园林工人正在给花坛浇水，水洒在泥土上泛起一阵淡淡的土腥味。风把书稿最上面那页掀起一角，又轻轻放下。他没有立刻去压。任由它起落了几次。纸张被风翻动的声音和远处早班公交的刹车声叠在一起。

他面前是五十万字的终稿。分成四十五个主章和一个诗歌索引。书稿摞在桌上，像一小座白色的山——每一页都被他翻过至少三遍以上，页角有些磨毛了，有些页边沾了茶渍和咖啡渍。最上面一行是书名：《逝水有灯》。

这个名字是程砚取的。不是他。但这个名字是对的。他在写完最后一章的那个晚上给程砚发过一条消息："你取的书名。归你。"程砚回："不用归我。归那盏一直没灭的灯。"

现在看着这整摞稿子，他开始做写成小说前的最后一件事情：删。

不是大幅删。是删掉那些还在文末残留的小总结。他写了一份清单：

- 第XX章结尾："他终于明白" → 改成具体动作
- 第XX章结尾："写作到最后也许只是" → 删掉整句
- 第XX章结尾："这不是彻底释怀的神话" → 删掉。不需要否定
- 第XX章结尾："结尾最怕用力" → 删掉。这话本身就是在用力

他逐章逐行地检查。这项工事持续了将近三个小时。每删掉一句模板化的总结，他就在旁边空白处用铅笔标一个"已修"。铅笔的笔芯断过两次。最后一次断的时候他把笔放下，去厨房倒水。喝完回来继续。

手机响了。妹妹。

"哥，你书什么时候写完？"

"今天。"

"今天？"

"大概中午之前。"

"那写完以后你要干嘛？"

"休息两天。然后看能不能帮妈把厨房那个橱柜换了。"

"那个柜子从你高中用到现在了。"

"所以该换了。"

电话挂了。他回到书桌前。最后一个需要删的句子在倒数第二章的末尾。原句写着："他不再急着安排一个漂亮结尾。"他想了很久。然后把整句删掉。改成："他把一页被风掀起的稿纸压平。"

书稿翻开新的一页。是最后一章的开头。他读了自己十年前的笔迹。十年前在诗社集会后写下的第一版"灯还没有灭"。白纸上有泪水滴过的褪色。不是伤感——是没及时擦掉的痕迹。他在退色上面画了一个极小的标记。然后继续阅读。

写完清单上最后一个"已修"的时候，书房的闹钟响了。六点半。该起来了——但他已经坐了三个小时。

他在书桌角摸到了那支旧中性笔。笔帽松了。手指稍微用力就能转几圈。这支笔是他大学时在图书馆文具机里买的——两块钱。陪他写完了笔记、信稿、诗、四版修改。

再两小时，这部书就完成了。不是"终于"完成。就是完成。

风又把纸吹起来。这次他伸手压住了。

手掌按在纸页上的触感很熟悉——打印纸的表面有点涩，是那种七十克普通A4纸的质感。他大学时用的一直是这种纸，毕业后也从来没换过——不是买不起更好的，是用习惯了。人的某些习惯像指纹一样，一旦形成就很难改：咖啡一定要速溶的，煮面一定要过凉水，写信一定要手写第一稿。

他低头看了一眼掌心下面压着的那一页。是倒数第二章——写成小说那一章的第一段。十年前他在这里写下了"把一页被风掀起的稿纸压平"——那是一句对未来的描述。十年后他做着完全相同的动作——把手掌按在纸上，压住被风掀起的一角。十年前他不知道自己会真的经历这个瞬间。他只是写了一个他觉得合适的意象。但现在他活进了自己写的句子里。

这大概就是写作最奇怪的地方：你不是在描述生活，你是在给未来的生活提供一个模板。然后有一天你忽然发现自己按照那个模板活了一遍。

他把手从纸上移开。纸没有再次被吹动——风已经停了。清晨的风就是这样：一阵一阵的，来的时候把窗帘鼓得老高，走的时候什么痕迹都不留。

还差最后几个"已修"标注。他重新拿起旧中性笔——笔杆上的裂纹在指腹下有一道很细的粗糙感。他把最后几个章节的结尾重新过了一遍。第XX章的"他终于明白"已经改成了"他把杯子放进水槽"。第XX章的"结尾最怕用力"已经整个删掉。第XX章的"这不是彻底释怀的神话"——删掉，改成"他泡了杯茶。茶叶是昨天买的龙井。钱塘产。"

他觉得这个结尾比之前任何一版都好。因为它没有告诉读者该怎么理解这本书。它只是说——茶泡好了。喝不喝是你的事。
