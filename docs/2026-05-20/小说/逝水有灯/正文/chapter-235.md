# 第235章 诗歌索引多年后

所属部卷：第五部：逝水有灯
时间：多年后
地点：自己的书房

## 诗歌插页 P235：原创仿唐绝句

索引翻残纸页轻，
百灯不为一人明。
诗行若问当年事，
先照他人各自生。

多年后，诗歌索引成了陆允川最常翻的文件。不是因为它比正文重要，而是它记录了每一次克制。哪一首删掉了控诉，哪一章换成了物件，哪一个名字被改成动作，表格里都有痕迹。

有读者后来问他，为什么每章都放诗。他没有说这是为了显得特殊，只说：“有些章节需要一盏小灯，照一下就够，不能照太久。”说完他自己也笑了，觉得这话仍有点像作者自述。

真正的答案其实更简单：诗歌让他学会少说。现代诗允许停顿，仿唐绝句逼他删繁，仿宋小令教他把话收到半处。那些限制把他从漫长的自我解释里拉出来，让他不得不面对场景本身。

索引最后一页，有一行叶澄提醒过的话：她们不是意象。陆允川一直保留着，没有删。每次修到苏晚枝或夏知遥相关章节，他都会先看一眼。

书房的灯换过几盏，索引文件也迭代了很多版。可那行字像一枚很小的钉子，把整本书钉在现实里。
