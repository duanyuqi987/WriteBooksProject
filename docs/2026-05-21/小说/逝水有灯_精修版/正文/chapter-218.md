# 第218章 旧日记本之后

所属部卷：第五部：逝水有灯
时间：工作第三年
地点：新住处

## 诗歌插页 P218：原创仿宋小令

旧封边，晓风清，
半窗纸页动微灯。
问旧年、几番执念，几番无凭。
人各有程休再问，
水声仍向远天生。

余波没有立刻散开。

读过旧日记本之后的周末，陆允川正常上班、加班、吃外卖、洗衣服。日记本放在书架上，他没有再去翻。不是因为不敢，是因为该看的内容已经看完了。剩下的是把看到的东西——那些戳出凹痕的笔画、那些涂改了十几遍的句子、那些精确到分钟的碰面记录——消化成可以放进小说里的东西。

周三下午，他在公司收到一条快递消息。老妈寄的。一箱苹果、两袋腊肉、一罐自己做的辣椒酱。箱子打开的时候，辣椒酱的盖子没拧紧，洒了一点在旁边的文件袋上。他擦掉辣椒酱的时候想起来：上次家里的酸菜，已经在冰箱里放了两个月，还没吃完。

他给老妈打了个电话。

"苹果收到了。腊肉也收到了。辣椒酱洒了一点。"

"洒了？盖子没拧紧吗？我拧了三圈。"

"可能快递路上颠的。没事，就洒了一点点。"

"那你把剩下的放冰箱。腊肉冷藏，辣椒酱冷冻。苹果放阴凉处。"

"知道了。"

"你自己做饭还是叫外卖？"

"各半。"

"各半是什么意思？"

"就是一半自己做一半叫外卖。"

"自己做的都做什么？"

"炒青菜。番茄炒蛋。面条。"

"你爸说炒青菜也算饭。"

"不算吗？"

"不算。青菜是菜。你加点肉。"

"好。下次加肉。"

挂了电话以后，他站在厨房里看着冰箱。冰箱门上贴着几张外卖单和一张便利贴。便利贴上写着：辣椒酱冷冻。

他打开冷冻层，把辣椒酱塞进去。冷冻层里还有一袋冻了一个月的饺子。旁边的冷藏层里，辣椒酱洒到的地方被他用湿抹布擦了三遍，还是有一小块浅红色的印子。

他不讨厌那个印子。它证明辣椒酱确实来过。

晚上写稿的时候，他写到了日记本相关内容的那一章。他描述封面"边角磨白"，描述笔迹"戳出凹痕"，描述那个涂改了十几遍的句子。写到一半，他停下来，去书架上把那本日记抽出来，翻到通信草稿那几页，确认了一句原话。

确认完以后，把日记本放回了原处。

不是抽屉底。是书架上。可以看见，不需要翻。

然后他注意到手机上有条未读消息。是苏晚枝，三小时前的：辣椒酱洒了擦不干净，用牙膏试试。我上次也是这么擦的。

他盯着这条消息看了几秒。然后回：你怎么知道我辣椒酱洒了？

她回：我不知道。我是说我自己的。昨天我也洒了辣椒酱。

他回：哦。牙膏有用吗？

她回：有用。擦了三遍。

他回：我也擦了三遍。没用牙膏。

她回：那加牙膏再擦。

他去厨房加了牙膏又擦了一遍。这次干净了。

回来以后他给苏晚枝发：干净了。

她回了个大拇指。

这件事不值一提。但它发生了。两个在不同城市的人，在同一天洒了辣椒酱，用同样的方式处理了同一个问题。

他在小说的草稿里记了一笔：不是伏笔，不是象征，只是当天确实发生了的事。保留。

睡前，他关了大灯，留了台灯。日记本在书架上的位置刚好在台灯光圈的边缘，牛皮封面的一角被光照到了，其他部分在暗处。

他没有再看。关了台灯。

明天还要上班。
