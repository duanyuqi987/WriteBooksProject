# 第137章 叶澄的劝告

所属部卷：第三部：寒风里的花
时间：六月午后
地点：南湖凉亭

## 诗歌插页 P137：原创仿唐绝句

湖水微凉近晚风，
少年心事隔帘中。
不知旧梦归何处，
一盏疏灯照水东。

七点。301教室。

苏晚枝推开门时，陆允川已经在了。他坐在讲台下面的位置——那个她说"考试周复习"的位置。前排灯亮着，后排黑着。瓶盖门帘在门框上被开门的气流推了一下，铝盖互相嗑了一声。

"来得早。"她把布袋放到讲台上。袋子里装着模型材料——轻木条、U胶、砂纸，还包着半截没粘完的框架。她的手指上沾了U胶留下的白印。

"下午在凉亭跟叶澄聊了。"

"她找你了？"

"嗯。她把你在凉亭跟她说的话转达了——关于你不知道该怎么办。"

苏晚枝拉了一张椅子坐下来。铁架的椅子腿在水泥地上刮出一声短响。

"她是擅自转达的。不是我让她说的。"

"但她转达的是真话。"

"是。我说了。我不知道怎么办。"苏晚枝把手放在膝盖上，手背上的U胶白印在台灯的照射下像一片微缩地图。"你约我来说'那件事'——你要说的是什么？"

"我想说——你不用一个人想。有些问题需要两个人一起想。"

苏晚枝抬起眼。她的眼睛在台灯光下颜色很浅——不是平时那种因为戴眼镜显得深的样子。

"什么问题？"

"'一年见一次面够不够'——这个问题不应该一个人回答。你问叶澄'他会不会觉得不够'——你应该问我。"

"那我问你。"苏晚枝把话接得很快，像是已经在心里问了无数遍。"如果接下来一年我大部分时间在画图和考试和双学位的课，你见我最多一周一次——你觉得够吗？"

"不够。"陆允川说。没有犹豫。"但够不够不是全部的问题。还有另外一半。"

"哪一半？"

"你愿不愿意在不够的情况下还试。"

教室里安静下来。那根日光灯管在头顶嗡了一声，又被老郑换过的镇流器压下去了。

"我在想的是另一面。"苏晚枝说。"不是'够不够'——是'公不公平'。我觉得我不能让你进入一个我没办法全心投入的状态。你对我太认真了。从诗社传单那天开始就很认真。我不能收这份认真，然后因为自己没时间就只给半份。"

"半份是多少？你自己算的？"

"我算了。课表排出来以后，下一年的空闲时间折半——周三晚上诗社活动，周末可能会有一个下午。再多就是挤的。挤出来的时间不是放松的时间——是从考试前、交图前、期末前偷出来的。偷时间的人会紧张。"

"紧张什么？"

"紧张又一次没做好。"

"又一次？"

苏晚枝把椅子往后推了推——不是退避，是给自己多一点空间说后面的话。

"我从小到大都是做什么事就做好。高中是。诗社是。建筑课也是。我以为如果我做好所有事的程度够高，我爸就能不说话。后来发现不是——他只是换了一种方式不说话。但我已经养成了这个习惯：任何一件我签了名的事，必须做好。做不好就不配签名。"

"所以你怕你签不了名。"陆允川说。

"怕我签了之后做得不好。这对你不公平。"

"但你已经在做了。瓶盖门帘——那不是给诗社做的。301教室的树——树干上你说'空间不是容器是邀请'——那是写给每一个进入这个空间的人的，包括我。你凌晨三点写的信——'并排的两棵树'——也不是给诗社写的。"

苏晚枝把手里一个轻木条折了一下。木条太脆，断了一小截。

"'两棵树'——那封信我写的时候也在想：这个比喻是不是太好听了。太好听的东西可能不太真。"

"那你觉得真的是什么？"

她放下断木条。"真的是——当你今天说'不够但我愿意试'的时候，我心里在想的回答是：'那好。试。但我可能需要你提醒我——提醒我不用把每件事都做完才签自己的名字。'"

"签在什么上面？"

"签在——"她抬眼看着树墙。"——那棵树的树根上。那行字旁边：'献给每一个在夜里找空教室的人'。在旁边加一行：'苏晚枝——在这里。在试着。'"

陆允川站起来，走到树墙前面，在树根位置的空白墙上用手比了比。"这里可以写。"

"你觉得我家人会怎么看？"

"你爸？"

"对。他说'考试及格再跟我说'。"

"等你考过建筑初步和结构力学。我猜你两门都能及格。"

"如果能及格——如果他看到了成绩单，他可能会愿意坐下来听我说：诗社和建筑可以同时做好。和你也可以同时做好。"

"那你需要的是时间证明。"

"需要。但我需要的不是一个人证明。需要你在旁边——"她想措辞。"不是帮我搬桌子。是不搬桌子的时候也在。"

走廊里老郑的拖把车推过来了。车轮在水泥地上发出熟悉的闷响。老郑从门口探了一下头，看到两个人，没有进来打扰，只是把门口地上的一团纸屑扫走了。

苏晚枝走到树墙前面，站在陆允川旁边。

"今天你说的那两句话——"

"哪两句？"

"'不够'和'但我愿意试'。加起来比'够了'更让我觉得安全。因为是真的。"

"'够了'可能是装的。'不够但试'装不出来。"

"对。"她用手背擦了擦眼角——可能是台灯刺眼，可能是别的。手的背面还有U胶的白印。"那你从现在开始不需要观察我了。"

"什么意思？"

"观察是你收集证据的阶段。但证据够了。结论我也说了。以后你不需要再站在超市门口数瓶盖，不需要再发消息问程砚怎么看，不需要再在阳台上坐塑料凳读信。"

"那我需要做什么？"

"做——当你来接我的时候，不用再给叶澄说'顺路'。就直接说：'我等她。'"

陆允川把她的手拉过来。U胶白印摸起来粗粗的，像晒干的浆糊。手的温度还是比他低。

"我等她——"他重复了一遍。"不加'顺路'。"

苏晚枝笑了一下。眼角往下弯。这一次是近的、是深的、是小心的。

老郑推着拖把车折回来敲了敲开着的门。"你们两个——门帘挂起来是好事。但要锁门。上次你们走后没锁——我帮你们锁的。这把钥匙给你们备用——别告诉后勤处。"

他把一把新的钥匙放在讲台上，推着车走了。拖把桶的响声渐渐飘远。

陆允川把钥匙收进口袋。然后他说："你刚才说你从小到大做什么事都做好。你知道你是从什么时候开始不用'做好'来证明的吗？"

"什么时候？"

"你在信里写'不想争夺同一块土'的时候。"

"那是跟你说的那句话——'两棵树并排长'。"

"对。树不用做好。树只要长。"

苏晚枝看了他片刻。然后走向讲台拿起布袋，把断掉的轻木条放进袋子里。

"那个模型明天要交。我先回去粘。粘完给你发照片。"

"好。"

走到门口时她让瓶盖门帘从手指间滑过去。铝盖的毛刺还没打磨——他上次被扎过的那个橙色的还在那里。

"你上次被这个毛刺扎了一下。"她说。

"对。"

"你觉得那个刺是提醒你'是真的'。我后来想——它提醒我的东西不一样。"

"提醒你什么？"

"提醒我——就算我不完美，你也会伸手。毛刺不伤人，只是在。"

（Author's Note: 这一章是第三部最重要的转折——陆允川和苏晚枝从"观察和验证"阶段正式进入"在一起"阶段。对话中的"不够但我愿意试"是两个人关系的核心定义。这不是浪漫的理想化告白，而是两个都怕做不好的人互相给对方空间。）
