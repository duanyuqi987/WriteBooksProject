# 第083章 室友

时间：2013年9月中旬

地点：楚州师院宿舍512室

卷次：卷五：各自南北

宿舍四个人。李博、阿坤、小伍。顾临风。

李博是第一个到的。他是洛阳人，说话的时候偶尔会冒出一两个河南话的音——"中"。他每次进门都敲门——虽然他自己有钥匙。指节在门上敲三下——笃、笃、笃——然后等一秒，再开门。这个习惯从第一天就开始了。开学第三天他贴了一张纸在门后面——"512宿舍卫生值日表"。打印的。字体是宋体。下面还画了一条线——用圆珠笔画的，不太直，中间有点歪。他用透明胶带贴了四个角——然后退后一步看了看，觉得歪了，撕下来重新贴。反复贴了三次。

"你们谁要是值日那天没倒垃圾——"李博指了指那张纸。他的指甲剪得很短——能看到指甲缝里的皮。"上面有名字啊。"

阿坤看了一眼。阿坤是摊在自己铺位上看的——腿翘在床架的护栏上。"班长。你放过我们吧。"

"我不是班长。"

"你马上就是了。"

李博把嘴抿了一下。没说话。过了两天——系里开班会选班干部。大家推举了李博。他的竞选发言只有一句话——"我会做好该做的事。"全班三十多个人，他得了二十六票。当天晚上阿坤说"我说了吧"。李博说"你闭嘴"。阿坤笑着又打他的魔兽。

阿坤叫陈坤。但他让所有人叫他阿坤。他带来的行李最少——一个背包、一个手提袋。背包里装了三件东西：电脑、充电器、一条黑色的耳机。第一条晚上就通宵打游戏——魔兽争霸。键盘敲得啪啪响。屏幕的蓝光照着他的脸——把他的颧骨和鼻梁照得很清楚。他的脸在暗处的宿舍里像一个浮在蓝光里的面具。凌晨三点他还在戴着耳机说"奶妈加血奶妈加血"——声音没控制住，从耳机漏出来。李博被吵醒了，翻了三次身。第三次翻身的时候他说"阿坤"——阿坤没听见。他又说了一次"阿坤"——然后阿坤的耳机里大概是打完了团战，他摘下一只耳机"啊？"——"你这键盘也太响了"——"哦——我轻点"——然后他换了一个更软的按法，但没一会儿手指条件反射又重了起来。

他的桌子最乱。泡面盒摞了三层，最上面那盒里还有半碗汤——汤上凝了一层白色的油脂。可乐罐排了一排——绿色的、蓝色的、红色的。瓜子壳掉在键盘缝里——要用镊子才能夹出来。有一次李博路过说"你桌子上的细菌比食堂的还多"。阿坤说"细菌也是生命"。李博说"那你去跟细菌住"。阿坤说"我行"。李博每天路过他的桌子都皱眉头。"阿坤你那个泡面盒子都长毛了。"阿坤头也不抬。"明天倒。"明天又说后天。后天变成了下星期。最后是小伍看不下去了帮他倒的——倒的时候泡面盒的底下粘了一层黄绿色的霉。

小伍叫伍思远。崇阳人。他家在崇阳县下面一个镇——名字很长，他说了三遍室友才记住。第一次住校。第一晚翻来覆去睡不着。床板在他身下咯吱咯吱响了大概四十分钟。凌晨一点多从上铺探头下来。探头的时候先露出来的是一只手——扶着护栏——然后是头发，然后是眼睛。他看了几秒——大概在确认顾临风是不是也没睡。

"顾临风——你睡了吗。"

"没。"

"我有点想家。"他的声音有点紧——像喉咙里卡了什么东西。不是哭。就是紧。

"嗯。"

"你家住得近吗。"

"青阳县。三个小时。"

"三个小时也不近。"

沉默了一会儿。小伍把脑袋缩回去了——缩回去的时候头发擦过护栏的铁管，发出很轻的沙沙声。过了大概十分钟——上铺传来吸鼻子的声音。很轻。然后是翻身的动静——床板又咯吱了一下。然后没声音了。

小伍胆子最小。第一次去食堂打饭，站在那里不知道点哪个窗口——六个窗口排在他面前，他像一根柱子一样杵在中间。后面的人从他左边绕过去了他没动。顾临风走过去指了中间那个。"番茄鸡蛋面还行。"小伍点头——点头的幅度很小，就是下巴往下低了低。打好面以后坐在他对面。把筷子从筷子筒里抽出来——竹筷子，一根长一根短。他用两根筷子互相蹭了蹭——刮掉可能有的毛刺。吃了两口，说"比我妈做的淡"。然后继续吃。吃完了。碗里的汤也喝干净了——他端起碗，仰头。碗放下来的时候碗边沾了一片番茄皮。

小伍每周给家里写一封信。信封上画着熊猫——他爸是邮局的，家里攒了一摞熊猫信封。他写得很慢。台灯拧到最低档。笔尖在纸上像在描图——一个字一个字地描。字迹工整——像小学生抄课文。每一个撇和捺的角度都差不多。有时候写到一半会抬头问——"临风，'适应'的'适'怎么写"——顾临风在空气里给他比划了一下。先写舌——再写走之底。小伍的手指跟着在桌子上画了一下。然后低头继续写。有的信他读了一遍以后又叠好放回信封里——然后又拿出来——改了两个字。改好以后又放了回去。信放进信封之前他会用嘴唇碰一下信纸的边角——可能是觉得这样比较郑重。

开学第二周。晚上十点。熄灯铃响了——不是铃，是一段音乐，电子合成的《致爱丽丝》，从走廊尽头那个音箱传过来的。音量开得不大——但整栋楼都听得见。日光灯啪一下灭了。宿舍陷入暗处。窗外的路灯光透过窗帘的缝隙——在地面上切出一条细细的白色。然后进入卧谈时间。不知道是谁先开的头——大概是阿坤。

阿坤先开口的。"咱们班那个女生——三班的——叫什么来着，刘什么的——好看。"声音在黑暗里飘——看不见表情，但能听出笑意。

"刘思琪。"李博说。说的时候他的床铺响了——他翻了个身。

"对对对。眼睛挺大的。"

"你怎么什么人都认识。"

"咱班花名册我看了一遍。"

"你看那玩意儿干什么。"李博的语气是那种"又来了"的语气。

"提前做功课。"

小伍从上铺翻了个身——床板咯吱响。"你们高中谈过恋爱吗。"他的声音从上铺传下来，在黑夜里没有方向感——就像从整个天花板压下来的。

阿坤说。"谈过。高二。谈了三个月就分了。就三个月。她嫌我打游戏太多。"说的时候情绪很平——像在讲别人的事。"她最后说了一句话——'你跟你的电脑结婚吧'——然后走了。我后来打了三天三夜的魔兽。想明白了。我确实更爱我的电脑。"室友们都笑了。笑完以后有几秒的安静——大概大家都在想这句话到底是真心的还是自嘲。

李博说。"没。高中忙。没顾上。"

阿坤说。"班长说谎。你一看就是没谈过。"

"我真没谈过。"

"所以才说'没顾上'。谈过的人会说'没有'。"

沉默了两秒。李博翻了个身——用沉默承认了阿坤的判断。

"临风你呢。"阿坤把声音往他那边转了。

"没有。"

"暗恋过吗。"

顾临风没有接话。窗外的路灯光透过窗帘的缝隙——在地面上切出一条细细的白色。那条白色的光从窗户脚一直延伸到床底下。他侧着身。脸朝着墙。墙上有那个小洞——他不知道前一个学长钉钉子挂过什么。钟。海报。照片。十字架。洞里是黑的。他把食指伸进洞口摸了摸——内壁粗糙，凉凉的，像水泥。

"临风？"阿坤的声音提高了一点。

"没有。"

"你们都不诚实。"阿坤说。"我看人很准的。李博一看就是没谈过——但心里有个人。临风一看就谈过——或者差一点就谈了。是差一点——就差那么一点。小伍一看就是——算了，小伍我不看了。他太单纯了。"

小伍说"我也没有——"。然后补充了一句"我高中没敢跟女生说话"。阿坤说"那你高中白上了"。

李博说。"你先把你的魔兽打好吧。情场失意，战场也失意——你两头都不占。"

阿坤笑了。他的笑在黑暗里闷闷的——大概是用枕头挡着脸。笑完以后他说"行——班长今天赢了"。

又聊了一会儿——关于军训。阿坤说站军姿的时候他故意站歪想偷懒——结果教官让他多站了十分钟。小伍说他的解放鞋不合脚——脚后跟磨掉了一层皮。话题又转到食堂——李博说食堂的红烧肉不如他高中的红烧肉，他高中食堂是全省示范食堂，有九个窗口。话题又转到哪个班的女生好看。阿坤说师院有个校花——大二的，叫什么他忘了，"反正是学音乐的"。小伍说音乐系的女生都好看。李博说"你们一天到晚就是女生女生"。阿坤说"不然呢，难道聊食堂的饭？今天晚上的西红柿炒蛋比昨天的咸了——这个有价值吗"。李博笑了一声。说"行——你们聊"。

声音渐渐低了。阿坤最后一个说完——"明天有没有人叫我——六点——我要去机房占电脑位置——"然后没人回答了。然后宿舍安静下来。安静里有几种声音——阿坤的呼吸变重了，李博偶尔翻一次身——床板响一下，小伍把被子往上拉了拉。

顾临风听到小伍又吸了一下鼻子。跟第一晚一样——很轻，怕人听见。但没有第一晚那么重。

然后一切沉默了。

他翻了个身。枕头底下的手机硌着脑袋——硬硬的，长方形的棱角顶着后脑勺。他没有把手机拿出来。只是把手伸到枕头下面——凉。碰了碰手机壳的金属边。凉的。他在黑暗中用手指按了一下屏幕——没按亮，按的是侧面的按钮。不知道按到了什么。

外面有人在走廊里趿着拖鞋走过去。脚步声由近到远——拖——拖——拖——一步一步的。大概是去上厕所。拖鞋底在地板上发出的摩擦声在安静的五楼走廊里被放大了。声控灯亮一下——走廊里的光从门缝漏进来一瞬——灭了。

他闭上眼睛。

那些他没有说的话——还在脑子里转。关于高一的教室。关于二排靠窗的位置。关于一个扎马尾的女生——马尾上面的皮筋有时候是黑的，有时候是浅蓝色的。关于一封没有递出去的信——信在书包里放了三年，信封的边角磨破了。关于月湖边的那个晚上——风是凉的，她的声音很轻。关于她说"暂时不要联系"——他说"好"——然后松开手。关于那条301路公交路线——十一个站名，十二公里，两块钱——他没有坐过一次。但每天在脑子里坐两遍。一遍是去。一遍是空着座位回来。

他什么都不会说。这些话在黑暗里转了几圈——然后就沉下去了。像石子沉进水里。水面恢复了原来的平静。

他把被子往上拉了拉。蒙住半张脸。呼吸落在被子里面，有洗衣粉的味道——是碧浪牌的，他妈从镇上买回来的大袋装。被子里是暖的。呼吸把被窝里的湿度提高了。

第二天早上。李博第一个起床。闹钟是六点整——振动的，没开声音。他按掉闹钟。坐起来。床板给了他一声短促的咯吱。他下床——光脚踩在凉水泥地上，脚趾蜷了一下。然后走到窗边拉开窗帘。哗——外面天刚亮，操场上有一层薄薄的雾。几个晨跑的人影在雾里若隐若现。雾被阳光切成了一绺一绺的——透出很淡的金色。

他叠被子的时候很有规律——用手提起两边被角、在空中抖一抖、对折、再对折、用手掌在叠痕上压了压、放在床尾。动作精确——不知道是从哪里学来的。然后他去洗漱。水龙头打开的声音从走廊尽头传过来——先是水声，然后是漱口声，然后是杯子放在水池边沿的声音。他回来的时候拎着四个包子——用塑料袋装的。塑料袋上印着"师院食堂"四个字。

"谁的包子。"

"我的。"阿坤从上铺跳下来——落地的时候整个人晃了一下。

"那是临风的。你的在塑料袋里。"

"班长。你过分了。到底谁才是你的上铺。"

顾临风接过包子。塑料袋还热着——透过塑料能感觉到包子的热气。韭菜鸡蛋的，包子皮上能看到韭菜的绿渗出来的汁印。他咬了一口。皮有点厚——馅不太多。他突然想起高中食堂的包子——那种个头更小但馅更多的，五毛钱一个。每次她排在他前面四五个人的位置。中间隔了大概三四人。她能挡住他的视线——他只要看她的背影就行了。她买两个包子、一碗小米粥。包子先吃素的那个——韭菜鸡蛋的。吃的时候会站在食堂左边的窗前——可以看到操场上晨跑的人。她侧脸的轮廓在晨光里有很淡的金边。有一次她吃完包子，用纸巾擦了擦嘴角——擦了两下——然后把纸巾折了两下扔进垃圾桶。折了一下——对折。又折了一下——再对折。他发现她折纸巾的动作用的是左手。她是左撇子吗。不是——她写字用右手。但折纸巾用左手。她可能是两个手都会。

他咬了一口包子。咀嚼。咽下去。

李博递过来一杯豆浆——纸杯装的。"老顾——你刚才走神了。走神走了大概十秒钟。"

"在想事。"

"'在想着韭菜鸡蛋的事'——你这个借口太烂了。"

阿坤在旁边笑。牙膏沫还挂在嘴角——白白的，一左一右。他用手背擦了擦。

上午的课是中国古代文学。老师姓孙，四十多岁，戴金丝边眼镜——金丝很细，在太阳光下会闪。讲话的尾音拖得很长。"屈原——的代表作——《离骚》——是第一首——文人独立创作的——长篇抒情诗——"每两个字之间好像都有一个小停顿——像在检查自己讲得对不对。底下有人趴在桌子上睡觉——口水流在课本上。有人在看手机——藏在桌子底下，拇指飞快滑动。顾临风坐在第三排。笔记本摊开。但他写的东西跟《离骚》没什么关系。他在本子边角写了几个字。然后停了一下。然后把那几个字涂掉了——用圆珠笔横着来回划，直到那几个字被蓝墨水完全盖住。墨水把一个字的偏旁洇成了墨团。他看了看那个墨团。然后提起笔——在那个墨团上又加了几笔。加了一点、一横、一撇、一捺。墨团变成了一个字的偏旁。然后又加了一竖——变成了一棵树。树的根扎在墨团上，树枝往笔记本的格子线外面伸。树没有叶子——就是光秃秃的几根枝。

下课后李博凑过来看了一眼他的笔记本。"你画的是什么。"他指着那个墨团上的树。

"树。"

"不是树。是个字。你把字涂成树了。"李博的眼镜片反着教室的日光灯，像两块白色的小圆片。

顾临风合上笔记本。放进书包里——拉链在涩的那个角度又卡了一下。

那个中午他没有去食堂。在操场走了一圈。中午的操场很晒——太阳直直地打在塑胶跑道上。跑道边缘冒着热气——空气在热浪里微微扭曲。跑道上有上体育课的学生在测五十米。体育老师拿着秒表喊"预备——跑"——然后按秒表的拇指落下去——脚步声啪啪啪地落在跑道上。一个跑得特别快的男生冲过终点线以后继续跑，多跑了好几步才停下来。另一个跑得慢的追到终点以后弯腰扶着膝盖喘气。他走过的时候一阵风把沙坑里的沙子吹起来——沙子打在裤腿上。他拍了拍裤腿。继续走。脚底透过鞋底能感觉到塑胶跑道的温度——比体温高一点。

走完以后回了宿舍。512里三个人都在。阿坤在打游戏——屏幕上的魔兽世界里一个骑马的战士在追一只怪物。李博在看书——那本《中国历代政治得失》已经翻到"宋代"那章了，他用铅笔在页边画了线。小伍在写信——今天的信纸是浅黄色的，他在写"展信佳"的"展"字——第一笔有点歪，他用橡皮擦掉了重写。橡皮擦出来的碎屑他吹了一下——碎屑飞到了床单上——他低头一颗一颗捡起来。他在自己的铺位上躺下来。脱了鞋——左脚、右脚。枕头下面的手机硌着后脑勺。后脑勺的骨头隔着枕头能感觉到手机的轮廓——长方形的、硬的、四角是圆的。他没有挪动。只是枕着。手机壳慢慢地从凉变温——从体温里吸收了热度。天花板上的裂缝还在。他看了几分钟。然后闭上眼睛。
