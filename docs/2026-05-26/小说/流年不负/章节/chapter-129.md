# 第129章 一茶一坐

时间：2018年3月周六下午2:45

地点：楚州"一茶一坐"茶餐厅

卷次：卷七：流年未断

他提前了十五分钟。

但透过茶餐厅的玻璃窗，他已经看到她了。

三号桌。靠窗。和张阿姨说的一模一样。

她低着头在看手机。马尾安静地垂在肩上。浅蓝色上衣——跟照片里那件一样，颜色比高中时那件深一点点。不是刻意搭配的，只是她一直喜欢这个颜色。

茶餐厅的装修是日式简约风格。原木色的桌椅。每张桌子上方吊着一盏暖黄色的小灯。灯罩是竹编的——光从竹篾缝里漏出来，在桌面上印了一道道细长的条纹。靠窗有一排卡座。卡座的坐垫是深灰色的亚麻布。有一处脱线了——露出来的海绵是黄的。

窗边有一盆绿萝。养得很好。叶子肥厚，深绿色，藤蔓从花盆边沿垂下来，快垂到地上了。阳光从玻璃照进来，落在桌面和她的侧脸上。她的手指在手机屏幕上慢慢滑动——大概在看新闻，或者朋友圈。拇指划一下，停两秒，再划一下。手机壳是透明的——发黄了。用了很久。

和高中时一模一样的坐法。

她坐在靠窗的位置——这八年时间里，无论是青阳一中高一三班的教室、青阳一中高三六班的教室、楚州师大的图书馆、楚州一小的办公室——她永远坐在靠窗的位置。

好像靠窗坐是一种本能。

顾临风在茶餐厅门外站住了。

玻璃窗隔开了他们。

他在窗外。她在窗内。中间隔着玻璃的厚度、八年的时间、和从高一教室后排走到她课桌旁边的那段距离。

她在看手机。没有抬头。手机屏幕的光映在她脸上——淡淡的，白里透一点蓝。她的睫毛在光里很长。两道眉毛微微蹙着——大概在看不太有意思的东西。

手指划了一下屏幕。停。又划了一下。

她不知道窗外有人在看她。

就像高中时候——她不知道走廊上假装看窗外的那个人其实在等她经过。不知道课间操散场后有人故意走慢两步好排在队伍最后面——这样可以看她的背影走进教学楼。不知道每次月考成绩出来，有人比她先找到她的名字。

现在她还是不知道。

她在等他。

手机拿在左手。右手放在桌上——手指自然弯着，中指上有一个很小的创可贴。大概是上班的时候纸划的。当小学老师——每天要翻很多本作业，纸页切口锋利。

桌上已经放了一杯水。透明的玻璃杯。水喝了一半。杯壁上凝了一层细细的水珠。杯底有一个很淡的圆形水印——在木桌面上。旁边放着一张纸巾——叠过的。纸巾上有一小片被撕掉的创可贴——肉色的。她把创可贴撕了。

她端起杯子喝了一口。嘴唇碰到杯沿的时候——杯沿上印了一个很浅的口红印。她用拇指擦了一下。没擦干净。放下去。手指在杯壁上轻轻转了一下——习惯性的动作。茶还没上。她在等茶。也在等人。

然后她抬头。

往门口看了一眼——大概是在看他来了没有。看了一眼就收回来了。大概没看到认识的人。

然后她又低下头继续看手机。

没有认出窗外的人。

顾临风的喉咙动了一下。手在裤兜里攥成拳头又松开。松开又攥成。

路边的樱花树在他身后投下一块影子。影子在阳光下很淡。

有行人从他身边走过。一个拎着菜篮子的老太太。两个说说笑笑的女学生。一个送外卖的——电动车停在路边，人拎着塑料袋跑进了隔壁的面馆。

他没有动。

他在窗外看着她。

阳光从她的侧后方打过来。头发是褐色的——和八年前走进高一教室时一模一样。教室里的阳光也是从这个角度打过来的——那时候他的座位在倒数第三排靠过道，她的座位在第二排靠窗。阳光穿过窗户先照到她，穿过她再照到他的桌面上。

他比这个城市的任何人都更早看到她。

比楚州一小的同事。比师大的室友。比青阳老家的媒婆。比她小学三年级的任何一个学生。

但他也是最晚开口的那个。

手指在口袋里攥紧。指节发白。

门上的风铃响了——不是他推开的，是有客人出来。一个男的，拎着公文包，把门带上了。风铃又响了一声。

她没抬头。

他把手机拿出来看了一眼。两点四十八。还有十二分钟。

他没有进去。提前十五分钟——他本来打算提前五分钟就够了。但他走到了茶餐厅门口。透过窗看到了她。

然后他就不动了。

像一只脚踩在门槛上——但抬不起来。

茶餐厅里的音乐很轻。听不太清是什么曲子。暖黄色的灯光在下午的光里显得多余。木制的桌子擦得很亮。吧台里有个人在往茶杯里续水——细长的、冒着热气的水柱落进茶杯里。

她又拿起手机。换了另一只手。右手大概累了——那个创可贴让她握手机的姿势有点别扭。她用食指和中指夹着手机，拇指在屏幕上点。大概在回消息。

回谁的。同事。学生家长。媒婆。

她答应相亲的时候停了一下。张阿姨说的。

他站在门外。看着那个停顿——如果停顿可以看见的话。

两点五十。

他走了两步——往茶餐厅门口。路过玻璃窗的时候脚步慢了。她离他不到两米——隔着玻璃。近到他能看见她后颈上几根细碎的头发——扎马尾时没绑进去的。

她没有注意到他。她的注意力在手机上——睫毛偶尔闪一下。阳光又往左边移了一点。窗外的樱花树下，一只麻雀在啄面包屑。茶餐厅的玻璃反光里能看到一个穿浅灰衬衫的男人的倒影。

站在门外。手在口袋里。看着她。

两点五十五。他转了身。走向茶餐厅的门。两步路。他走了四步——每一步都踩得很慢。脚底能感觉到地砖的缝隙。地砖是灰色的——有点脏了。上面有一块口香糖的印子——黑色的。被踩扁了。嵌在砖缝里。

手搭在门把上——门把是铜的，冰凉。冰凉的铜在掌心里像一枚硬币。门把上有些细微的划痕。不知道多少人推过这扇门。但今天——是他。

风铃还没有响。他的手掌心在出汗。他没有推门。手搭在门把上大概十秒钟。十秒钟里他想了什么——不知道。脑子是空的。又好像是太满了——满到所有念头挤在一起。什么也看不清。路边有辆车按了一下喇叭——很响。一个小孩子从旁边跑过去——手里举着一个气球。气球是红色的。上面画着一只兔子。

他把手从门把上拿下来。在裤子上擦了擦。裤子的料子是棉的——吸汗。掌心的汗在裤子上留下了一小块深色的印子。

然后又搭上去。手指慢慢收紧。能感觉到门把上的每一条细纹。铜的温度慢慢变暖——被他手心的热捂暖了。

他深吸了一口气。不是深呼吸。就是吸了一口气。跟平时一样。然后推。

风铃响了。清脆的。一排金属管互相碰撞——叮叮当当。声音在下午安静的茶餐厅里特别清楚。吧台里的服务员抬起头。角落里的客人往门口扫了一眼。窗边那个人——

也抬起了头。

两点五十八。他推门的前一秒钟。脑子里什么都没有了。空了。那些想了八年的开场白——全空了。只剩下一个念头——她在里面。窗边。三号桌。马尾。浅蓝色上衣。透明水杯。跟八年前一模一样的坐法。跟八年前不一样的是——这次他知道她的名字。她也知道他的。这次他可以光明正大地走到她面前。不用假装看窗外。不用假装路过。就走到她面前。叫她的名字。然后坐下。
