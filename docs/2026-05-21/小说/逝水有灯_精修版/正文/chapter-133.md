# 第133章 她和别人说笑之后

所属部卷：第三部：寒风里的花
时间：六月初
地点：校园超市门口

## 诗歌插页 P133：原创仿宋小令

汽水边，晚声轻，
半窗灯影落无凭。
问旧年、几回执念，几回分明。
人去不须追远路，
水流仍向晓光生。

第二天下午，陆允川在图书馆二楼看期末论文的资料时收到苏晚枝的消息："下来一趟。超市门口。有事。"

不是"有事吗"——是"有事"。

他到超市门口时，苏晚枝站在冰柜旁边，手里拎着一袋汽水瓶——空瓶，大概七八个，是从冰柜旁边的回收袋里捡出来的。她把瓶盖一一拧下来，用一根绳子穿起来，做了一个简易的门帘。瓶盖被穿了孔，铝质软，用钥匙就能钻眼。

"你干嘛？"

"诗社门口需要一道门帘。301教室的日光灯太冷，门帘挡一下光——不是挡全部光，是让光碎成小片的。瓶盖颜色不一样：橙色的、柠檬色的、青苹果色的。光透过这些颜色会变暖。"

"这几个瓶盖是——"

"有你的橙子瓶盖，有我的柠檬瓶盖。"她把瓶盖串拎起来抖了抖，铝盖互相撞出清脆的声音。"何屿白上次喝剩下的那个青苹果味的瓶盖也在。我跟他要的。我说诗社要交废品。他就给了。"

陆允川看着那串色彩斑驳的瓶盖门帘。柠檬味——她。橙子味——他。青苹果味——何屿白。被她一起串在了一根绳子上，成了一样东西，不再代表谁对谁的占有。

"你不需要做这个。"他说。

"不需要。但我想做。"她把门帘收好放进布袋，"我想说的是：你以为何屿白会把你的位置挤掉。但门帘上有三种颜色。橙子有位置。柠檬有位置。青苹果也有位置。不同的颜色不互相挤。"

"你在用瓶盖给我做心理辅导。"

"对。因为你需要的不是道理。是实物。"她拉上布袋的拉链，"道理你已经自己总结了——'她的快乐不全都跟我有关'。但道理是抽象。你需要一个可以摸的答案。这个瓶盖门帘就是可以摸的。"

陆允川接过她手里的瓶盖门帘。铝盖的孔边缘毛刺没打磨干净，扎了一下手指。他缩了一下手。

"毛刺——我忘了处理。"苏晚枝说。

"没事。这个刺——刚好。"

"刚好什么？"

"刚好能感觉到是真的。"

苏晚枝看了他手指上被扎出的那个小红点。"你这个人——连被扎了都要赋予意义。'刚好是真的'——这种话你要写进诗里可以，但首先它只是个没处理的瓶盖毛刺。不用过度解释。"

"你说话越来越像祁野了。"

"祁野教了你直接表达感受。我觉得他教得对。"苏晚枝靠在冰柜上，"不如你尝试一次——对我也直接说。不用哲学包装。像你对祁野说的那样。"

陆允川手里抓着那串瓶盖，感受铝的冰凉和边缘毛刺的粗糙。他想了片刻。

"昨天从超市回去以后，我睡不着。一直在想你那句'小心才是真的'。我想，你对我小心——是因为你在乎我的看法。你在乎我对你的判断。这个在乎让我觉得自己不是随便一个人。"

"刚才那句带了一个比喻——'不是随便一个人'。去掉比喻——剩下的核心是什么？"

"核心是——"他停了半秒。"你让我觉得重要。"

"这个就是直接的表达。"苏晚枝笑了一下——很浅的笑，但眼角弯了。"你看。不难。"

她把瓶盖门帘从他手里拿回去，挂在自己的布袋边上。这个布袋跟了她两年半，从大一到现在。现在它的侧面多了一串彩色铝盖，每走一步响一声。

回教学楼的路上，碰到何屿白。他从建筑系馆出来，怀里抱着一捆模型材料。看到苏晚枝他停下来打了个招呼，然后看到陆允川。

"你是诗社的对吧？"何屿白说。

"嗯。"

"晚枝上次让我帮她改透视图，提到了你。她说你们在做一个'诗与空间'的主题。那个树墙我去看过——在301教室后面。你们用纸做了树。我们建筑系做的东西是硬的——混凝土、钢、玻璃。但树墙是软的——纸、胶、灯光。我觉得挺好。"

"谢谢。"

"不用谢。我是夸她的主题——不是你的。"何屿白笑了一下转身走了，模型材料撞到门柱上发出沉闷的声响。

苏晚枝在旁边说："他人不错。就是话有点直。"

"跟你差不多。比祁野还差一点。"

"我没那么直。"她说。然后补了一句："我只是对你小心。对别人可以随意。"

陆允川决定把这句话也记下来。不是用笔记本——是在心里。小心才是真的。对别人随意——对你小心。

诗社的瓶盖门帘在傍晚挂上了301教室的门框。老郑推着拖把车经过时停下来看了半天。

"这个比树好看。有颜色。"他说。

"是废品。"苏晚枝说。

"废品排好了就是门帘。你这个同学很会过日子。"老郑推着车走了。水管在车上颠得叮当响。
