# 第141章 秋忙

时间：2012年9月下旬至10月初

地点：江城大学

卷次：第六卷：雪落无声

大三确实是分水岭。

九月底，辅导员在群里发了一条消息：中文系今年的推免名额下来了，四个。想保研的同学准备好材料。林玉起看了一眼就关掉了——他的成绩在班里排第七，前面有六个人。保研轮不到他。

郭向阳的成绩倒是够。但他不想读研。"码农读什么研，出去干活才是正经。"他已经在准备大厂的秋招了，每天刷算法题刷到十二点。

何南星偶尔在QQ上冒个泡。他毕业之后在省城一家出版社做助理编辑，每天看稿子看到眼睛疼。"唯一的缺点是钱少。但每天跟书打交道。适合咱这种人。"他给林玉起发了一个链接——省城那家出版社明年招实习生。"你明年春天可以试试。我可以帮你递简历。"

林玉起把链接收藏了。

十月初。文学社开了第一次例会。新社长是一个大二的女生叫程雨，说话软绵绵的，但做事利索，一上来就定了这学期的四个活动——诗歌朗诵会、小说创作坊、征文比赛、元旦特刊。她把任务分配得很清楚，林玉起负责诗歌朗诵会。

"林学长，你今年要上台朗诵一首。"程雨说。

"我？"

"你是文学社里写诗最多的。不上台说不通。"

散会后林玉起在教学楼后面的长椅上坐了一会儿。十月的风开始凉了。梧桐叶在变黄，一片两片地往下落。

他打开QQ，看见苏安晴的头像是亮的。

他打了一行字，又删了。又打了一行，又删了。

然后苏安晴先发了消息过来。

"陈老师把支教总结群发了，你看到没？"

"看到了。"

"他把我们八个每个人都夸了一遍。"

"陈老师人好。"

"嗯。"

过了大概十秒。

"林玉起。最近写诗了吗？"

"写了。"

"给我看看。"

"还没改好。"

"你每次都这么说。"

林玉起看着屏幕上这六个字。他想象苏安晴打这行字时的表情——大概不是责备。可能是笑了。

"改好了给你看。"他打过去。

"好。"

然后她的头像灰了。

林玉起把手机放回口袋里。秋风把一片梧桐叶吹到他的膝盖上，干了，边缘卷起来，叶脉清晰。

>
秋风不问人，落叶自叩门。
欲寄诗三首，又把字改昏。
>
