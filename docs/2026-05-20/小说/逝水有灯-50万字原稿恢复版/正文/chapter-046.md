# 第046章 元旦晚会之前

所属部卷：第一部：初秋三楼
时间：高二元旦
地点：学校礼堂后排

## 诗歌插页 P046：原创现代诗

我把早晨放低，
让一盏灯从课桌边慢慢经过。
你的名字没有被我喊出，
只在礼堂后排一张折皱的节目单旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写元旦晚会抵达之前的坡度。元旦晚会那晚，礼堂门口挂着红色横幅，胶带贴得不太牢，边角被暖气吹得轻轻掀起。后排座位冷，塑料椅一坐下去就响。

陆允川把节目单折成窄窄一条，又展开，折痕越来越深。轮到朗诵节目时，他才发现自己掌心出了汗。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。

那天真正的事还没有发生，礼堂后排一张折皱的节目单却先被他记住了。它太小，小到放在任何人的日子里都不显眼，偏偏在陆允川这里有了重量。

“你别把节目单揉碎了，等会儿还要抽奖。”程砚说。“我没用力。”陆允川说。“纸都快替你紧张了。”程砚说。

程砚嘴上打趣，灯光暗下去时却没有再说话。他知道有些时刻不能起哄，起哄会把真正的东西弄脏。

叶澄坐在前几排，跟着主持人鼓掌。她回头看了一眼后排，看见陆允川坐得笔直，像被一束他自己想象出来的光照住。

课间十分钟被铃声切得很碎。有人挤去小卖部买面包，有人趴在桌上补觉，有人拿着水杯一路跑向水房。陆允川站在这些声音中间，心里那点紧张没有消失，却被生活稀释了一些。

夏知遥这时还只是人群里的一个名字。夏知遥站在舞台中央，校服外面套着白色毛衣。灯光照得她眼睛很亮，她握着话筒，开口第一句比陆允川想象中平稳。陆允川看见了，却没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子，又一行行划掉。留下来的，是教室里真实的光、真实的灰、真实的不知道该把手放在哪里。

后台有人搬道具，木板磕在地上发闷响。音响偶尔啸叫，主持人笑着圆场。礼堂外卖烤红薯的小车还没走，甜味从门缝飘进来，和舞台上的亮片味混在一起。

如果说喜欢在这里有了开端，也不是因为命运突然发声，而是因为一个少年开始过分留意普通事物。他还不懂，留意可以是温柔，也可能慢慢变成对自己的消耗。

礼堂里暖气开得太足，后排同学把校服拉链拉开，椅子腿在地面上磨出刺耳声。主持人念错一个班级，台下笑成一片，紧张才被冲淡了一点。

夏知遥上台前站在幕布边，低头看稿。灯光还没打到她身上，她只是一个准备节目的学生，袖口有一点皱，发卡也没有完全别正。正是这些小地方，让陆允川多年后仍觉得可信。

朗诵到中段，话筒轻轻啸叫了一声。夏知遥停了不到一秒，继续往下念，声音没有抖。陆允川那时第一次明白，他喜欢的也许不是想象出来的柔弱，而是她面对台下几百双眼睛时的稳。

晚会散场后，地上到处是瓜子皮和彩带。程砚弯腰捡到一张节目单，塞给陆允川，说留着吧，反正你刚才已经把自己的揉烂了。陆允川接过来，没有否认。

这一章要把脚步放慢，慢到能听见课桌被拖动时的短促摩擦声。事情还没到来，可人物已经在日常里摆好了位置：谁靠窗，谁靠门，谁总替别人递一张纸。

陆允川后来才知道，所谓开始并不是一个明确的按钮，而是很多个微小选择叠在一起。他没有说话、没有靠近、没有确认，可他已经在心里给那个人留出了一块地方。

事情还没有真正发生时，陆允川已经在心里排练了好几种反应。可排练没有用，现实总会把人推到更笨的位置，让他先拿错红横幅，再说错一句无关紧要的话。

那天放学，他没有立刻回宿舍，而是在学校礼堂后排外多站了一会儿。风把舞台灯吹得轻轻晃动，他忽然意识到，自己等待的并不是答案，只是一个能让明天继续期待的理由。

陆允川把节目单拿在手里，先把节目单摊平，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先好好看节目，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她站在幕布边低头确认稿纸，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

学校礼堂后排里最明显的不是心跳，而是话筒轻轻啸叫后又恢复平稳。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

晚会散场时，礼堂门口烤红薯的甜味把人从心事里拽出来。程砚说再大的事也得先回队伍，陆允川把节目单折好，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。节目单、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情还没到来，节目单已经先进入他的视线。他当时不懂这种提前的留意有多危险，只觉得只要自己足够安静，就不会打扰任何人。

放学后他在学校礼堂后排外多站了一会儿，听见楼下有人喊集合。那一声很普通，却把他从想象里拉出来，让他记起自己还要回宿舍、洗杯子、准备明天的早读。

这一段把晚会从光里放回人群。节目会结束，椅子要归位，地上的彩带要被扫走，陆允川也要把她从自己的比喻里还出来。

如果只写疼，这章会很薄；加上节目单、话筒轻轻啸叫后又恢复平稳和礼堂门口烤红薯的甜味，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在学校礼堂后排发呆。”陆允川把节目单收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

本章最后，事情仍悬在半空。陆允川收好书，跟着人群往外走。楼道里风从窗缝挤进来，把他的校服袖口吹起一点，他下意识按住，像按住一句还没成形的话。
