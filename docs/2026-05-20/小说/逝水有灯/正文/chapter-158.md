# 第158章 短暂和好之后

所属部卷：第三部：寒风里的花
时间：七月中旬
地点：诗社活动室

## 诗歌插页 P158：原创仿宋小令

纸箱封声长。
一块苏打，碎屑落晴窗。
不敢多求今日暖，
怕把轻松压作伤。

晚上回宿舍，陆允川把手上的胶带印洗了很久。透明胶黏过的地方泛红，水冲过去，还是有一点粘。

祁野问：“今天怎么样？”

“还行。”

“还行是好，还是你又开始脑补？”

陆允川把毛巾搭好，“只是还行。”

这次他说得比较稳。祁野看了他一眼，没有继续损他，只把桌上的泡面推过去，“吃点。你中午肯定没正经吃。”

陆允川坐下吃面，想起苏晚枝递来的那块饼干。以前他会把这件事写成“她还在意我”。现在他只写：她给我一块饼干，我收下了，没有多说。

这句话很短，却让他心里踏实一点。

诗社群里，叶澄发了整理结果。苏晚枝在下面补充：“第三箱旧刊已封好。”陆允川看见，回：“目录已同步。”

没有私聊，也没有多余的话。短暂和好最难的地方，不是重新说笑，而是不把一次说笑当成重新开始。
