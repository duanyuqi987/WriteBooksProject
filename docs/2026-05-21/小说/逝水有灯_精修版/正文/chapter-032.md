# 第032章 雨天值日

所属部卷：第一部：初秋三楼
时间：高一下学期雨季
地点：放学后的教室

## 诗歌插页 P032：原创仿唐绝句

雨打窗棂扫地时，
一盆清水各东西。
年少不知同行重，
只将扫帚作人梯。

事情真正发生时，并没有陆允川想象得那么郑重。星期三的雨果然很大。放学铃响时窗外的雨已经积了快一整天，操场的排水沟满了，水往上漫，低年级的体育课早就取消了。教室里的值日小组开始打扫——扫地、拖地、倒垃圾、擦黑板。

夏知遥负责扫地。她在前排弯腰扫着，扫帚的竹枝刮过地砖发出沙沙声。陆允川负责拖地。他拿着拖把从后排开始拖，一下一下拖得很慢——不是因为要认真拖干净水鞋印，而是因为他每拖一排都要确认一下和她之间的距离。拖到第三排时他和她的距离最近——大概两个课桌宽。她正在扫第四排的过道，扫帚尖从他拖湿的地砖边擦过去，扫帚和拖把的方向正好相反。

"那个——你这边已经拖过了？"她问。
"拖过了。"
"那我扫帚过去不会把灰带上来吧。"
"不会，地已经半干了。"

又是一次正常对话——这一次七个字加四个字，总共十一个字，比捡作业本那次多了五个字。内容和值日有关：地、扫帚、灰、拖过。没有任何心跳加速的词汇。但陆允川在心里又一次做了笔录——她关心地砖会不会被他弄脏，这是对环境有责任心，不是对他有任何特别的态度。他把这个分析结论写在心里的备忘录栏里——"她在意卫生标准"，备注："可能是习惯，也可能是被她妈妈要求的，同样适用于任何人。"

程砚负责倒垃圾。他从厕所那边的垃圾桶把垃圾袋扯出来，经过教室时看见陆允川正在用拖把按着一个位置反复拖，拖了大概三四遍了，那块地砖的纹理都被拖出了高光。"老陆，那块砖拖四遍了——它快被你拖亮成镜子了。""湿拖要拖两遍——地砖吸水不均匀。""那不叫吸水不均匀，那叫你心理不均衡。"

黑板前，另一个组员在擦数学老师留下的板书——函数图像的粉笔灰擦起来成片成片地飞，空气中白色的微尘飘着。夏知遥扫完了地板，把扫帚放到教室后门角落，回头去拿抹布擦窗台。窗台已经积了厚厚一层雨季尘——那种灰很细，沾了雨水以后结成一坨，得用湿抹布使劲擦才能擦掉。她擦窗台时认真到每个角落都不放过——左到右，上到下，像在写一个端正的立方体展开图。陆允川在拖地板的同时用余光观察的她的抹布轨迹：从讲台开始，到第三扇窗，再到后门——如果她按照这个路线继续走，她会在后门口和他撞上。他的拖把正横在后门附近。

她果然走到后门时停下脚步——抹布还攥在手里，她低头看地上的拖把，然后抬头看他。她轻轻把拖把的杆子往旁边推了半寸，然后从那个半寸的间隙里跨过去，继续擦墙。这推拖把的动作发生在零点三秒内，是她和拖把，也是——通过拖把——和他的交界。陆允川弯下身把拖把拎起来拧了水，重新放下去。他不知道那零点三秒算不算"接触"，但他觉得至少算——拖把替他接了她的手。

值日结束后，教室里只剩下最后几个收拾书包的人。窗外雨还没停，但小了——从之前的大雨转到均匀的中雨。陆允川拿上伞往楼下走，走到二楼时看见夏知遥在教学楼门洞下面站着——手里没有伞。她抬头看了看天空，表情很平，像在判断雨几时能停。陆允川握着伞柄走下了最后几步台阶。他的嘴张了一下——想说"要不要撑我的"——但只说了个"要——"就顿住了，后面的字被他自己吞回去了。夏知遥没听见，因为雨声太大了。她转过头看见他时，他只来得及说了一句"你怎么不走"。"没带伞。""蒋雯的伞呢？""蒋雯下午请假了。"陆允川把手里的伞柄往前递了两寸——但不是直接递到她手上，而是递到了她面前的门框旁边，意思很清楚：你如果要用，这把可以。她看了伞一眼，然后摇了摇头说"不用，我等我爸来送"。

他收回了伞。两人在教学楼门洞下面站了大约一分半钟——雨水顺着门洞的檐口往下滴，滴在水泥地上打出白色的水花。两把伞都在她们头顶上一尺的地方，各自握着自己那把的伞柄。谁也没说更多。直到夏知遥的爸爸骑着自行车从校门口进来——披着雨衣，雨衣的帽兜低低地压在额头上——她走下台阶接过父亲递来的伞，回头说"拜拜"。陆允川回了一声"拜拜"。然后各自消失在不同的雨水方向里。

晚自习的教室里，座位空了几个——离家近的人雨太大直接走了。陆允川坐在位置上写英语作文，题目是"A rainy day"。他写了值日生拖地的经历——写到了地砖的纹理，写到了雨水顺着窗台往下流的路线，写到了拖把被拧干时的声音。写完之后他发现自己从头到尾没提到她——不是故意的，是因为那一分半钟的等待里，值得写进英语作文里的东西确实只有雨、地和拖把。但她站在那里——在教学楼门洞下面和他一起等——就已经是他那篇作文的隐藏主题了。
