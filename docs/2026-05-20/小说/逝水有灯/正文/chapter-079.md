# 第079章 毕业照边缘旁观

所属部卷：第二部：六月以后
时间：六月中旬
地点：学校操场主席台前

## 诗歌插页 P079：原创现代诗

她站在第二排，
只想着别闭眼。
太阳很晒，
校服领口有点紧。
有人在照片边缘看她，
她在照片中央
也只是一个快毕业的人。

夏知遥拍毕业照时，最担心自己闭眼。

她不太擅长拍照。摄影师一喊看镜头，她就容易紧张。林曼站在旁边，小声提醒她笑自然点。夏知遥说我已经很自然了，林曼说你现在像在听物理讲评。

两个人差点笑场。

拍照时，夏知遥没有注意后排。全班那么多人，太阳又刺眼，她连镜头都看不太清。快门响过后，她第一反应是终于结束，第二反应是校服外套快热得穿不住。

拍完集体照，林曼拉她去主席台旁边拍单独照。她们拍了几张，姿势都不熟练。林曼说以后到了大学要多拍照，不然青春都没证据。夏知遥笑，说青春难道靠证据吗。

可她还是认真看了照片。照片里她和林曼站在红色横幅下面，笑得有点拘谨。她觉得这张不算好看，却舍不得删。

陆允川有没有在远处看，她不知道。即使知道，她也只能继续处理自己的告别。和老师合影，和同桌合影，去打印店登记尺寸，回家把照片给母亲看。她的毕业照不是为了进入谁的收藏，而是为了证明她也从这所学校走出去了。

晚上，她把全班照存进手机。照片里每个人都很小，包括陆允川。她看见他站在边上，旁边是程砚。她只是停了一秒，想：原来他站在那里。

然后她把手机放下，开始收拾第二天要带去填志愿咨询会的资料。
