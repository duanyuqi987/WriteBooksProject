# 第033章 雨天值日之后

所属部卷：第一部：初秋三楼
时间：高一下学期雨季
地点：放学后的教室

## 诗歌插页 P033：原创仿宋小令

雨过空堂，
拖痕渐已干。
欲把同行藏半页，
却因门洞等风残。
伞下半分寒。

雨天值日之后，陆允川表面上没有变化。但雨季继续——下一周的星期三又是他的值日，又是雨天。这次他提前多带了一把伞——家里那把旧蓝色伞塞在书包里，外面的塑料布还没拆。他想好了一个理由：如果她又没带伞，就说这是他爸买多了送过来的。

程砚发现了他书包里多了一把伞。"你带两把？"
"一把备用。"
"你以前从来不带备用伞。去年有天下暴雨你没带伞，直接冲回宿舍，头发湿得像落汤鸡——然后你在宿舍里打了一夜的喷嚏。那天你怎么没带备用？"
"那天我看天气预报说是多云。"
"多云——行。"

这次值日夏知遥带了伞。她午饭后就从家里拿了大伞——蒋雯在门口喊她说外面已经开始打雷了。值日结束后她拎起伞就走了，门洞下面没有等任何人。陆允川拿着两把伞站在那里，看着她的伞面消失在雨幕里，一把多出来的伞柄冰冷地攥在手心。

晚饭时程砚终于憋不住问了出来："你今天怎么还是带了两把伞？""怕下雨。""她带了吗？""带了。""那你的备用伞不就白背了——背了大半个教材楼的重量，就为了证明你有两把伞。""这不是重点。"程砚端碗喝了一口汤，然后说："重点是——你愿意背两把伞走一天路。你为她焦虑整个雨季，每一次雨点打在教室窗上都能让你提前心跳加速。这些虽然她不知道，但它们都是真的——只不过它们不是给她的，它们是给你自己的。"陆允川夹了一块土豆，嚼了两口，觉得程砚说得比物理老师讲动量守恒还准。

夜里回宿舍，上铺的人问他值日累不累。他说还行——其实特别累，不是因为拖地，而是因为每次值日都是集中注意力训练。他要估算她和自己站位之间的距离，要控制走过去的步速，要控制拖把的方向不让水溅到她裤脚——所有这些计算耗掉的脑力比做一套物理卷子还多。

晚自习后他从走廊经过——楼道的灯把地上的水迹照得发白。他把那两把伞从书包里拿出来，一把是自己的——边角翻毛那把。另一把是爸买多的——塑料布还没拆，全新的，没有人碰过。他把全新的那把放回抽屉，把旧伞挂在椅背上。下次下雨，他还是会带多的那把。不是因为有必要，而是因为他需要让自己在雨中被需要——这是一种不太讲理的自我安慰，但安慰只要能安慰到，就是有效的。
