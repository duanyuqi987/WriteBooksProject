# 第050章 元旦晚会多年后

所属部卷：第一部：初秋三楼
时间：多年后
地点：旧节目单前

## 诗歌插页 P050：原创仿唐绝句

节目单黄字未残，
旧灯重看已无欢。
当年一语藏冬夜，
今日知它只是寒。

多年后，陆允川在诗抄本里找到那张元旦节目单。

纸已经发黄，第三个节目那一行仍有明显褶皱。夏知遥的名字印在上面，旁边还有他当年用铅笔写下的几个字：挺好的。

他看着那三个字，想起礼堂后门外的冷风，想起程砚说要捐门票，想起夏知遥抱着话筒线说谢谢。那时他把这一幕反复擦亮，以为自己终于靠近了一点。现在再看，那只是同班同学之间非常普通的一次对话。

普通并不等于没有意义。

它的意义不在于证明关系，而在于证明他曾经从沉默里往外走了一小步。他没有说错话，没有吓到谁，也没有把自己的心事强塞给别人。他只是说了一句“挺好的”，然后把那晚带回了自己的本子里。

陆允川把节目单从诗抄本里取出来，放进文件袋。标签写：高二元旦。

他没有再把它单独放在夏知遥相关的那一格。因为那晚也属于很多人：属于怕忘词的夏知遥，属于嗑瓜子的程砚，属于跑调的老师，属于后台缠成一团的话筒线。

一段记忆被放宽以后，反而更耐看。

他拍了一张节目单照片发给程砚。程砚回：我那时候真嗑瓜子了吗？

陆允川回：还差点被我瞪死。

手机屏幕亮着，他笑了一会儿。窗外新年的城市灯火很远，和县一中的礼堂没有关系，却也让那张旧节目单安静地亮了一下。
