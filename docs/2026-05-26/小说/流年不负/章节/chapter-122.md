# 第122章 电话那头

时间：2018年3月中旬

地点：楚州，顾临风公寓

卷次：卷七：流年未断

第二天下午。他妈又打来了。

"张阿姨那边确认了。姑娘同意了。"

顾临风刚下课。走廊里有学生在跑，鞋底打在水泥地上，咚咚咚地响。

"哦。"

"你什么态度。"他妈的声音硬了。

"我说好。"

"'一茶一座'三号桌。张阿姨帮你订的，靠窗。"

"嗯。"

"你到底是去还是不去。"

"去。"

"你声音能不能大点。"

他走到走廊拐角处，靠墙站着。墙上有个消火栓，玻璃罩上贴了一张红色的勿动标识。

"我会去的。"他说。声音控制得很平。

"那就行。衣服准备好了没有。"

"准备好了。"

"你穿什么。"

"衬衫。"

"什么颜色的。"

"浅灰。"

他妈沉默了两秒。"你那件灰的领口都磨毛了。去买件新的。"

"好。"

"头发。"

"理了。"

"什么时候理的。"

"昨天。"

又沉默了一下。电视的声音从听筒里传来——好像是天气预报。

"楚州明天有雨。"他妈说。"出门带伞。"

"知道了。"

电话挂了。

他把手机从耳边拿下来。屏幕上的通话时长——三分二十一秒。他盯着那个数字看了几秒。然后把手机放进裤兜里。

站在走廊拐角。背靠着墙。墙上的消火栓玻璃映出他变形的脸。下课铃响了，学生从他身边跑过去。有人在喊"等等我"，有人在笑。一个男生差点撞到他——往旁边闪了一下，说了句"老师对不起"，然后继续跑了。他没有应。手在裤兜里把手机攥得很紧。

他走回办公室。

办公室不大。四张桌子。另外三个老师都不在。他的桌子靠窗，桌上堆着作文本和教案。窗台上放着一盆绿萝——对桌刘老师养的，叶子有点黄了。

他把手机放在桌上。

然后站起来。在办公室里走了一圈。

办公桌上有一叠期中考试的卷子。他翻了翻——不是他的科目。又放回去。

走到窗边。窗外是操场。有学生在打篮球。球砸在篮板上的声音传过来——砰。砰。砰。

他又走回桌前。坐下。

拿起手机。

翻通讯录。

联系人从A到Z。他翻得很慢——一个一个看过去。赵一鸣。刘老师。陈老师。王主任。妈。爸。姐。李博。阿坤。周磊。小伍。刘阿姨。张阿姨。

没有苏晚晴。

他把通讯录又翻了一遍。从A到Z。还是一样。

六年了。他删掉那个号码是在大一那年冬天。十二月。删完之后他躺在上铺，盯着天花板看了两个多小时。室友问他怎么了。他说没事——复习累了。

后来换了手机。旧手机还放在家里——青阳老家的抽屉里，电池早就鼓包了。那个手机的通讯录里也没有她了——最后一条记录是高考后那天晚上拨出的。通话时长：七分二十四秒。

他把手机放下。

窗外的篮球声停了。下课铃又响了。学生在走廊里跑。

他拿起作文本继续批改。

改了几本。停下笔。

在草稿纸上写了三个字——苏晚晴。

字迹很轻。铅笔写的。几乎看不清。

他看了看这三个字。然后用橡皮擦掉了。

纸面上只剩一点铅笔的灰色痕迹。他用手抹了一下——没了。

他把草稿纸折起来，放到作文本最下面。

下午的日光从窗户照进来，在桌面上落了一块长方形。绿萝的影子歪歪地印在作文本上。他盯着那片影子看了一会儿。

然后拿起下一本作文。

翻开。

第一行字：我的理想。

他看了一分多钟。一个字都没批。

把红笔放下。

站起来。走到饮水机前接了一杯水。饮水机咕咚响了一声。水面在杯子里晃了一下。

端着水杯走回桌前。坐下。

喝了一口。水是凉的。

他把杯子放在桌上。手还握着杯壁。杯壁是塑料的，薄薄的。手里的温度慢慢传到杯子上。

窗外有人在喊口令——体育课。一二一。一二一。

他松开杯子。

拿起笔。

批完剩下的作文。

他把作文本摞整齐，放进抽屉里。抽屉推到底的时候卡了一下——里面有太多东西了。他用力一推，咣当一声关上。

窗外夕阳已经落尽了。操场上的篮球架只剩下一个黑色的轮廓。远处的教学楼亮着几盏灯——晚自习的教室。有学生在背书——声音断断续续地飘过来，听不清在背什么。大概是《岳阳楼记》。或者是《出师表》。反正都是一些要默写的篇目。

他关了办公室的灯。灯管灭的时候——嗞一声。然后安静了。把门带上——锁拧了两圈。锁芯有点涩。铜钥匙在手里转得很慢。楼道里声控灯亮了。他的脚步声在空荡荡的楼道里回响——嗒。嗒。嗒。每一步之间的间隔比平时长。

走到楼下的时候，他又拿出手机。没有未接来电。没有新消息。他看了一眼通讯录——又翻到S那一栏。空的。

S前面是R。后面是T。中间那一段只有三个人——邵老师。沈主任。孙会计。没有苏晚晴。

他把手机放回口袋。走出校门。校门口的传达室亮着灯。保安在看电视——屏幕的光一闪一闪的。保安抬头看了他一眼。他点了点头。保安也点了点头。然后他走出了校门。

街上已经黑了。路灯亮了一排。他的影子从一根灯柱下走到另一根灯柱下——先拉长，再缩短，再拉长。像夜晚里一条沉默的、反复伸展的尺。一家小吃店的灯还亮着。锅里冒着热气。白色的蒸汽在路灯下飘了很远。
走出校门的时候传达室的保安在吃盒饭——一次性筷子掰开了。饭盒里是土豆丝和红烧肉。保安抬头看了他一眼——点了点头。他也点了点头。校门外的路上学生已经散了。只有几个值日的学生推着自行车往外走。一个男生的书包拉链没拉。语文书露出一角——快掉了。旁边那个女生提醒了他。他停下来拉上拉链。

顾临风往前走了二十几步。经过学校东门那个公交站。公交站牌上有14路。从楚州一小过来——经过楚州师大——到楚州二中。转一趟车就能到。他站在站牌前看了一会儿。14路的路线旁边画着一排小点——每一站。楚州一小是起点站。楚州师大是第四站。楚州二中是第十一站。跟她上班的路是相反的。每天早上她在那边上车——他在这边上。两个方向。两条路。隔着七个公交站的距离。

一辆14路开过来了。车门打开——下来了三个中学生。校服是深蓝色的——跟楚州一小不一样。学生下了车就往学校方向走了。车门又关上了。开走了。他的目光跟着那辆公交车看了几秒。然后转身往回走。脚步比来时快了一些。经过小吃店的时候没停。经过水果店也没停。经过理发店的时候——门口的碎头发还在。理发师在屋里给一个老头剃头——剃刀在头上推过去。白头发落下来。落在围布上。
回到公寓。开门。鞋柜上的旧字典还是歪的——早上出门的时候忘了扶。他换了拖鞋。外套脱了搭在椅背上。在床边坐了一会儿。又站起来。走到窗前。拉开窗帘。对面楼的灯亮着——好几家。有一家在炒菜——葱姜蒜的味道。有一家在放音乐——很老的歌。邓丽君的。甜蜜蜜。从窗缝飘进来。

他拿出手机。又翻到S那一栏。还是三个人——邵老师。沈主任。孙会计。

周六。还有三天。
