# 第137章 他变了

时间：2018年3月周六下午

地点：楚州"一茶一坐"茶餐厅

卷次：卷七：流年未断

她说"你比以前话多了"。

他笑了笑。手指无意识地转了一下空茶杯——杯底的茶叶末跟着转了一圈。

"当老师练的。站在讲台上不能一句话不说。"

"你以前真的不太说话。"

"嗯。高一的时候——整个学期可能跟前后桌说过不到十句话。"

"我知道。"

他看了看她。她好像什么都知道。知道他语文好。知道他写过诗。知道他在走廊上看她。知道他以前不爱说话然后忽然变得能站在讲台上讲一节课。

"你呢——你当老师以后话变多了吗。"

"我本来就比你爱说话。"她笑了笑。"只不过那时候不太敢跟你说。"

"为什么不敢。"

她停了一下。端起杯子——已经空了。她把杯子又放下了。手指在杯沿上抹了一下。"不知道。"声音很轻。

窗外有人放鞭炮——很远，大概是哪家店开业。鞭炮声闷闷的。噼里啪啦响了十几秒。然后停了。青烟从远处飘过来——很淡。

她说"你还写东西吗"。

他没有想到她会问这个。

他的左手在桌面上。右手握着杯子——杯子的把手在拇指和食指之间慢慢转了一下。"偶尔。"

"写什么。"

"诗。"

"还是古体的。"

"也有现代的。"

她点了点头。没有追问。但眼睛里有什么想说没说。

"想看吗。"他问了。

"方便吗。"

他从夹克的内侧口袋里摸出一个本子——很小，巴掌大。黑色封面，线装的。皮面的角已经磨白了。是本旧本子。翻开——里面是钢笔写的字。很密。有些页写得比较满，有些页只有几行。字迹有变化——前面的比较稚嫩，后面的比较稳。

她接过去。翻开第一页。

第一篇的日期——2016年9月。那时候他考研刚考完第一轮。不是学生，也没当上老师——在公寓里看书看累了，就写了。她看了一会儿。翻了一页。又看了一会儿。再翻。有时候看好几分钟才翻。有时候扫一眼就翻。

翻到其中一页她停了。看了很久。大概十几秒。然后抬头看了他一眼。眼神里有一种——他知道她读懂了的表情。

但他没有解释。她也没有问。

她把本子合上——很轻。递还给他。指尖碰了一下他的手指——比刚才在壶上碰到的那次多停了一点时间。"挺好的。"

"哪首。"

她想了一下。"最后那一首。"

他记得那首。写的是一个站在讲台上的人回头看到窗外有一只鸟停在了电线上。最后两句——"我讲了很多课 / 没有一个字是你"。他写的时候知道自己写的是谁。现在她正在看——他也知道她知道。

她把右手从桌面上拿下来，放在膝盖上。然后用左手拿起杯子——空的——又放了下去。

"你写过——跟我有关的吗。"

她问得很慢。每个字之间都有很短的停顿。好像在估算这句话说出去以后的后果。问完以后她没有低头——也没有看他。只是看着桌上两人的茶杯——一个绿茶的杯，一个茉莉花茶的杯。

他说"写过"。声音很低。

他没有说"我写的每一首诗都跟你有关"。他没有说本子里第一个她相关的意象能追溯到2010年——高一的秋天——她的马尾上夹过一个蓝色的发夹。那个发夹后来没了。他注意到了。但从来没问过。有人抄在诗稿的边缘。有一个词用来描述那个发夹的颜色——已经褪色了。

她等他继续说。他没有。她也没有追问。只是轻轻点了一下头。

服务员过来问要不要续杯。他说不用了。然后服务员把桌上的空壶收走了——两个壶叠在一起，壶嘴互相敲了一下，叮。

窗外的路灯已经很亮了。街上的车多了——下班时间到了。一辆公交车缓缓靠站——刹车声透过玻璃传进来。车上人很多。

她说"我该回去了。晚上还要备课。"

他点头。招手叫服务员买单。

"AA。"她说。

"我来。"

"AA吧。"

"下回你请。"

她停了一下。然后点了点头。把掏出来的钱包放回包里。他看着她的手把钱包放回包里的动作——拉链拉开，钱包滑进去，拉链拉上。这个画面他记了很多年。

他付了钱。服务员找零——几张纸币和一个五毛硬币。他把硬币在指间翻了一下——然后放进口袋里。

两个人站起来。椅子腿在地板上刮了一下。

从三号桌走到门口——这段路两个人走了大概十步。她的马尾在他前面轻轻晃。浅蓝色的衬衫后领还是翘着。窗外的灯光从侧面打了过来——把她头发边缘照成了一层很淡的轮廓。他走在她后面大概两步，像高中时——课间操散场后的距离。但这次——他知道她会回头。

她走到门口，回头看了他一眼。"走吧。"声音自然。

他跟上去了。

门上的风铃又响了。这次不是因为有人推门——是门外的风灌进来，吹动了风铃。
