# 第081章 迟到的表白之前

所属部卷：第二部：六月以后
时间：六月末
地点：河边护栏旁

## 诗歌插页 P081：原创现代诗

河边的铁栏晒得发热，
他把手放上去，
又收回来。
一句话迟到了三年，
迟到的东西
总带着不合时宜的汗。

六月末，陆允川约夏知遥去河边见面。

消息打出去前，他删改了很久。不是因为内容复杂，只是“有空出来走走吗”这几个字一旦发给她，就不再只是走走。程砚坐在旁边，看他打字，看到最后问：“你想清楚没有？”

“想清楚了。”

“想清楚什么？说出来，还是要结果？”

陆允川没答。

程砚把手机还给他：“你可以说，但别逼人家接。她要是说不方便，你就停。”

夏知遥回得不算快。过了半个多小时，她说可以，傍晚在河边护栏那边。

陆允川看着那行字，心跳快了一阵，又慢下来。真正等到傍晚，他反而没有想象中勇敢。河边风很热，护栏被太阳晒了一天，手碰上去发烫。他提前到了，站在树荫下，把准备好的话在心里过了一遍。

说三年太重。说喜欢又太轻。说不想留下遗憾，像把自己的不安推给对方。

夏知遥到的时候，穿着白色短袖，手里拿着手机。她看见他，停了一下，走近：“你找我有事？”

陆允川点头。河水在下面慢慢流，岸边有人散步，有人推着自行车经过。这个场景太普通，普通到容不下他给自己安排的那些盛大开场。

他忽然明白，迟到三年的话，未必更有分量，只是更难放下。
