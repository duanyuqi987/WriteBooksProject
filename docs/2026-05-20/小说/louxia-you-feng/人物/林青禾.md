# 林青禾

## 角色张力

- 来处：普通大学校园里的年轻人，生活由宿舍、教学楼、网吧、小吃街和手机消息组成。
- 当下：被喜欢的人，亲近但不承诺，温柔又有回避。
- 欲望：自由、体面、不被同学议论
- 恐惧：被过度期待、被追问、被绑定
- 内在矛盾：想靠近别人，又不知道怎样不伤害别人。

**一句话灵魂**：一个不是反派的女孩，她只是没有义务回应全部深情。

## 基本信息

- 身份：大学生 / 青年记忆中的关键人物
- 叙事功能：推动许亦安理解喜欢、边界和释怀
- 性格色彩：真实、年轻、不完美

## 声纹

- 句长偏好：玩笑轻快，常用回避和反问化解压力
- 发言长度：中短句为主
- 特色词汇：校园口语、玩笑、回避或直接表达
- 情绪外露：不完全外露，常通过动作表现
- 绝不说的话：过度文学化、像总结报告一样的话
