# 第197章 最后一首诗

时间：2023年1月
地点：楚州
卷次：卷十：不负少年

诗写完了。纸上有七处涂改——一个字的平仄不对，画掉了；两个字的韵脚调过，旁边注着小字；一处是墨水洇了，纸皱了一小块。

他从头到尾又念了一遍。不出声。嘴唇动着。像背课文。

第一句。那年九月风初凉。2011年同样的七个字写在石桥镇二楼的书桌上。窗外是干的河床——河底露着青黑色的石头，缝隙里长出野草。钢笔是英雄牌——笔尖有一点弯，写着写着墨水会断。他写到半夜。写完把那页纸对折再对折——塞进旧词典。藏起来。怕人看到。怕被误解。怕被她看到——更怕她看不到。

现在2023年。他坐在自己的书房里。门外客厅里。女儿在垒积木——她已经能垒到十二块了。妻子在批作业——红笔在对勾的拐弯处留下了一小片反光。同样的七个字。不怕人看到了。反而是怕她看不到——但已经不怕了。她已经看了。而且会一直看下去。

第二句。笔底春秋墨未央。未央——没有尽头。从第一首《初见》到这首《不负》，中间隔了十二年。十二年他从一个少年变成了一个中年。从"此心已在阑珊处"到"流年未必无情物"。阑珊是灯火稀疏。而现在是"未央"。未央——不只有尽头。才刚刚开始。还有很长很长的路。写了很多首诗。有些她看过了——花了整整一个通宵。有些她还没看。有些只有他自己知道——在草稿纸上写了她的名字，然后又划掉了。很多次。那些她永远看不到。

第三句。少日藏心无一字。年少时把所有的喜欢都藏了起来。一个字都不敢说。赵一鸣问他写了什么——他说"没什么"。赵一鸣说"你笔记本上全是字"。他说"那是物理题"。走廊饮水机那次碰到她的目光——杯子里的水洒了。烫了手。起了泡。他把杯子背到身后。退了两步。把烫伤的手藏进裤子口袋里。

第四句。中年执手话偏长。现在每天坐在同一张餐桌前。早上一个煎糊了蛋的。晚上一个批完作业的。她说起班上的学生调皮——有一个男孩把蝌蚪带进教室了，蝌蚪从瓶子里跳出来在地板上一跳一跳的。全班女生都站到椅子上。她说的时候在笑——蝌蚪跳到了讲台上。他说起高三班里有个学生作文写得好——写他的同桌。"那个人笑起来眼睛眯成一条线——像月牙。"他给学生写的评语是——"你写的人很像一个我认识的人。那个人也扎马尾。也笑。笑的时候先弯眼角。嘴角慢半拍跟上。"

第五句。曾经别路千山外。2012年到2019年。七年。楚州到南城。一千公里的距离。他在。她不在。他在楚州的走廊里晾衣服的时候想起她——一件一件地晾。衣架勾在晾衣杆上。衣服的边角还在滴水。楼下有人在跳绳——绳打在地上的啪嗒声传上来。他每次晾衣服都会站一会儿——往南边看。南边的天空没什么特别的。但南边有她。他不知道她在哪个城市的走廊里做着同样的事——她在南城。师范生。教育学。也站在走廊里。也往北边看。

第六句。到底归途一水旁。月湖边上。2019年春节。他再次见到她。湖还是那个湖。柳树粗了一圈但还在——柳叶扫在水面上，画着一圈一圈的波纹。长椅换了铁艺的但还能坐两个人。她坐在他旁边——还是隔了一个拳头的距离。跟高三那年一样。但这一次隔得很短。她在湖边说他写的诗押韵不对——"晓"和"霜"不在同一个韵部。他说不重要。她说对——不重要。押韵对不对比不上一句一直没说出来的话——现在终于说出来了。

第七句。流年未必无情物。

他把笔放下了。写到这里的时候他停了很久。流年——流逝的时间。他一直以为时间是他们的敌人。高一的时候时间太快——他还没来得及跟她说一句话，三年就过去了。大学的时候时间太慢——分开的日子被拉得很长。

但时间让该回来的回来了。

第八句。不负当时一片心。

不负。不是"不要辜负"——还没有发生的。不是"不能辜负"——自己要求自己的。是"没有辜负"——已经完成了的。那颗当时的心——2010年9月1号他在课本空白处写下"苏"字时的那颗心。铅笔尖在纸上折了一小下——笔芯断了。他又削了。又写了一个完整的"S"。那个"S"歪歪的——因为心跳得太响，手在抖。那颗心跳了二十几下就停了——因为他赶紧把书合上了。然后又跳了。在放学以后。在写《初见》的时候。在月湖边说了那四句话的时候。在楚州重逢的时候。在结婚那天。在女儿出生的那一刻——一直都在跳。没有停过。没有辜负过。

他把纸上的涂改处用食指抹了抹——墨水干了。抹不平。改的那些痕迹还在。

他把诗放在一边。从旁边的抽屉里拿出一本新的笔记本——浅灰色封面的。今年新买的。翻开第一页。空白。他把这张纸用胶水贴在第一页上。用手掌按了按。按了大概二十秒。胶水干了之后纸页有点皱。但不碍事。

然后他在纸的下方用很小的字写了日期:2023年1月6号。楚州。晚。

他把笔记本合上。浅灰色的封面。放在书架第二层靠左。四本本子并排。从左到右:深绿色旧词典、浅蓝色笔记本（给她的）、牛皮纸笔记本（给女儿的）、浅灰色笔记本（今年的）。

他站起来。退后一步。手没有碰它们。

"顾临风。你在看什么。"

苏晚晴不知道什么时候站在他身后了。手里还握着那支红笔。笔帽没套。红色的笔尖差一点就蹭到了她的裤子。

"没什么。理理书架。"

"你那书架有什么好理的。四本本子挪来挪去。"

她看了一眼那排本子。目光在浅灰色那本上停了停。

"新买的。"

"嗯。"

"写了什么。"

"诗。"

"给我看。"

他把浅灰色笔记本抽出来。翻到第一页。递给她。她接过去。低头看。

第一句。她念出来了——"那年九月风初凉"。

然后她不念了。眼睛往下走。一行一行地走。看到第四句的时候她嘴唇动了动——像在记什么。看到第五句的时候她的手指在纸上划过。看到第六句的时候她闭了一下眼睛。看到最后一句的时候她停下了。

"不负当时一片心。"

她把这句话念了两遍。第一遍是正常的音量。第二遍很轻——像是说给自己听的。

然后把笔记本合上。递回给他。

"好。"

一个字。跟她说"我愿意"时一样轻。一样准。

"改了七回。"他说。

"改出来的好。"

"第三句改了四个字。最开始写的是'少日藏心无一语'——后来改了'无一字'。因为'一语'跟第四句的'话偏长'重复了意思。"

"你用功了。"

"陈老师教的。写诗要推敲。"

她嗯了一声。转身要走。走了两步——又转回来。

"这个'一水旁'——"

"月湖。"

"我知道。为什么不说'月湖'。"

"太直白了。不好。"

"你的诗里终于有了一个不是直白的地方。"

他愣了一下。

她说完去厨房倒水了。水龙头的水哗哗响。

他站在原地。手里捏着那本浅灰色的笔记本。封皮上有一点手汗印——被他的手指摁出来的。

他把笔记本放回书架第二层靠左。

然后走到厨房门口。她正在喝水。背对着他。马尾偏在左肩。

"苏晚晴。"

"嗯。"

"以后我不写诗了。"

她放下杯子。转头看他。

"为什么。"

"写了十二年。写到了这首。够了。"

她看了他一会儿。然后点了点头。没有说"为什么不再写"。没有说"你的诗很好看"。只是点了点头。

然后她把杯子放在沥水架上。

"明天吃什么。"

"你做什么我吃什么。"

"那你明天别做。我做。鸡蛋不煎焦了。"

"煎焦了也好吃。"

"不好吃。你别安慰我。"

"没安慰。是真的。"

窗外。天黑透了。院子里的冬青树上还有一点残雪。没化的。白色的。在路灯下反着一点光。
