# 第063章 高考后的水之后

所属部卷：第二部：六月以后
时间：2012年六月八日
地点：县一中校门口

## 诗歌插页 P063：原创仿宋小令

风入旧窗轻，
纸上有余声。
欲问前程何处去，
水边灯影自分明。
回首不须惊。

烧烤摊摆在县城东边那条巷子里，门面小，桌子全都支在路边。程砚早早就到了，占了一张圆桌，点了四十串烤肉、二十串土豆片、五串烤馒头，还有三瓶汽水。

"你点这么多谁付钱？"叶澄坐下的时候看了一眼满桌的盘子。

"AA。"程砚理直气壮。

"那你还点这么多。"

"我高兴。"

陆允川坐下的时候，汽水已经开了两瓶。程砚给他倒了一杯，杯子是塑料的一次性杯，杯壁上印着啤酒广告。

"干杯。"程砚举起杯子。

"为什么干？"陆允川问。

"为高考结束。"

"万一我没考好呢？"

"那也是结束。结束本身就值得干杯。"

陆允川想了想，拿起杯子碰了一下。汽水冒泡，溅了一点在桌面上。

周长海最后一个到，自行车骑得太急，停的时候差点撞翻桌子。他喘着气坐下来，先把一串土豆片塞进嘴里。"我刚回了一趟家，我妈非让我算分。"

"你算了？"程砚问。

"算了一半就算了。越算越烦。"

"那你还来。"

"就是因为烦才来。"周长海又拿了一串肉，"吃东西就不烦了。"

五个人吃到中途，话渐渐多了起来。程砚问了一圈人的志愿打算。叶澄说她报师范，以后想当英语老师。"我妈说女孩子当老师稳定，但我是真喜欢教书。我高中三年，最开心的事就是在讲台上领读。"

"你那个领读的表情特别严肃，"周长海说，"我每次都不敢不跟你读。"

"那是你本来就懒。"

陆允川说他还没有完全想好，但大概会报一个有江的城市的学校。"不一定是最好的学校，但想离开县城看看。"

叶澄问："你爸妈怎么说？"

"我妈说随便我。我爸还没表态。"

"我爸表态了。"程砚把汽水喝完，又给自己倒了一杯，"他说要是我考上本科，就买一台电脑给我。要是考不上，就让我去他朋友开的汽修厂当学徒。"

"那你一定要考上。"陆允川说。

"我尽力了。剩下的事不归我管。"

吃完烧烤已经快九点了。五个人沿着巷子往外走，路灯昏黄，飞蛾在灯泡周围扑腾。程砚走在最前面，忽然回头问："你们说，以后我们还能这样一起吃饭吗？"

"为什么不能？"叶澄说。

"大学各去各的。你在A城，他在B城，我在C城。一年能见一次就不错了。"

周长海叹了口气。陆允川没说话。

过了一会儿，叶澄开口了。"见不见面，跟交情没关系。跟想不想有关系。"

程砚安静了几秒，然后笑了。"你这话说得跟班主任似的。不过有道理。"

走到巷口，五个人在路灯下分开。周长海骑自行车先走了，车链子掉了又下来装上，骂了一声。叶澄往东走，走之前回过头对他们说："出分以后在群里说一声。"

"行。"

陆允川往西走回家。路上他经过学校门口，那条横幅还挂着，在夜色里看不太清楚，像一个没来得及擦掉的问题。警戒线已经拆了，只剩两根铁杆靠在墙边。校门口的传达室亮着灯，保安大叔在里面看手机。

他站了一会儿，没进去。然后转身，继续往家走。

回到家，母亲还在厨房里。桌上放着给他留的菜——保温罩盖着，掀开以后热气冒上来。红烧肉、炒青菜、一碗米饭。母亲从厨房探头看他。"烧烤吃饱了？"

"又饿了。"

"你吃烧烤向来吃不饱。"母亲把筷子递给他，"坐下来吃。"

他坐在饭桌前吃，母亲坐在旁边看他吃。电视开着，声音调得很小，是新闻频道，正在播高考相关的报道。一个记者站在某考点门口对着镜头说："今天是全国高考最后一天，考生们陆续走出考场——"

母亲拿起遥控器换了台，换成电视剧。

"为什么不看新闻？"

"考都考完了，看什么新闻。"她换到一个古装剧频道，"吃你的饭。"

陆允川低头继续吃。红烧肉有点咸，大概是加热次数太多，汁已经收干了。但这是他从小吃到大的味道，每一块都一样咸。

吃完饭他去洗澡。洗澡的时候听见母亲在客厅打电话，跟她姐姐——他的大姨——说他今天考完了，看起来心情还行，作文这次没跑题。她的声音穿过水声传进来，断断续续，但语气很平稳。

洗完澡他躺在床上，把手机打开。班群里已经炸了——有人在发红包，有人在约明天的KTV，有人把数学答案贴出来引发了一波惨叫。

程砚在群里发了一张烧烤桌面的照片，配文：庆祝陆允川作文没跑题。

底下叶澄回复：他不是说这次没跑吗。

程砚：就是庆祝没跑啊！

陆允川笑了。他打了两个字：幼稚。然后翻了个身，把手机充上电。

窗外的巷子里有人在放烟花——不是正规的烟花，是小孩子玩的那种手持烟花棒，在空中画圈，画出来的光很快就灭了，然后又画一圈。

他想起下午在校门口看到的夏知遥。她坐在三轮车上，被挤在母亲和另一个女生中间，车拐弯的时候她伸手扶了一下车棚的金属杆。就是这个动作——很普通，普通到不值得特别记住。但陆允川还是记下来了。

他把被子拉到下巴，闭上眼睛。今天不用背书，不用做题，不用算倒计时。明天不用早起。后天也是。

这是他三年以来第一个不欠任何题目的夜晚。
