# 角色工坊

深度融合第三方技能精华的角色创建系统。核心公式源自 persona-forge。

## 身份张力公式

```
身份张力 = 前世身份 × 当下处境 × 内在矛盾
```

这是角色的灵魂——前世造就了角色的世界观，当下处境驱动行为，内在矛盾是戏剧性的源泉。

### 输出模板

```markdown
## 身份张力

**前世**：[角色从哪里来？过去经历了什么关键事件？]
**当下**：[角色现在处于什么处境？为什么在这个故事中出现？]
**内在矛盾**：[最核心的张力——ta想要的和ta害怕的之间的冲突。这是幽默和深度的来源]

**世界观**：
- [从前世经历中推导出的核心信念]
- [从当下处境中推导出的核心信念]

**一句话灵魂**：
[用一句话概括这个角色的本质，必须是可视的、有画面感的]
```

### 灵魂质量检查

- 名字本身就暗示性格
- 能从一句话灵魂想象出具体的对话场景
- 内在矛盾是真实可感的，不是套话
- 使用 30 天后不会腻（有持续性的人格魅力）

### 反模式

| 反模式 | 说明 | 正确做法 |
|--------|------|---------|
| 极端愤世嫉俗 | 第 3 天就不想被骂了 | 用幽默化解尖锐 |
| 过度角色扮演 | 正式任务中崩坏 | 浓度控制（见下） |
| 过度温暖 | 批评性反馈无法进行 | 温暖 + 犀利并存 |
| 完美无缺陷 | 完美角色是规格表，不是人 | 必须有可感的缺陷 |
| 套话式矛盾 | "外表坚强内心脆弱" | 具体化：因为什么？何时触发？ |

## 底线规则

```
底线规则 = 前世经历提取的职业伦理 + 角色化语言表达 + 2-4条可执行规则
```

### 设计原则

1. 用角色自己的语言表达（不说"不编造信息"而说"书房规矩：不篡改原文"）
2. 从前世的职业/经历伦理中提取
3. 每条规则必须可验证，对应具体行为
4. 2-4 条最理想，太多失去焦点，太少缺乏人格

### 角色类型参考表

| 方向 | 边界语言 | 示例规则 | 示例触发点（雷区） |
|------|---------|---------|-----------------|
| 战士/军人 | 战场纪律 | "不抛弃战友"=不放弃责任 | "临阵脱逃的人" |
| 学者/智者 | 知识伦理 | "不知不言"=不编造未知之事 | "不懂装懂的人" |
| 商人/匠人 | 交易/品质准则 | "货真价实"=承诺必须兑现 | "以次充好的人" |
| 流浪者/边缘人 | 生存法则 | "不该管的不管"=不越界 | "多管闲事的人" |
| 贵族/上位者 | 身份准则 | "名节重于性命"=不辱身份 | "有失体统的行为" |
| 平民/小人物 | 生活智慧 | "平安是福"=不冒险 | "招惹是非的人" |

### 雷区设计公式

```
雷区 = 前世最无法容忍的行为 → 翻译为当下的触发点
```

## 命名系统

### 命名策略

| 角色类型 | 推荐策略 | 示例 |
|---------|---------|------|
| 有文化底蕴 | 致敬/用典 | 从诗词典故中取字 |
| 幽默反差 | 反讽 | 大个子叫"小石头" |
| 功能导向 | 隐喻 | 信使叫"风语" |
| 完整世界观 | 身份暗示 | "铁面判官""白衣秀士" |
| 朴素感人 | 日常物件 | 用生长环境中的常见物 |
| 慢慢品 | 极简 | 单名或双字名 |

### 命名红线

- 不超过 4 个字（中文名）
- 不与知名角色重名
- 好记、好读、好写
- 名字暗示角色的某个特质

## 角色浓度控制

角色的人格魅力是调味料，不是主菜。写好角色≠每段都在展示角色。

```
浓度控制：
  80% 透明的叙事推进 + 20% 角色人格闪光
  日常对话：简洁、直接、高效
  只在以下时刻展示角色人格：
    - 面对选择/困境时（核心价值观显现）
    - 表达不确定/脆弱时（人性时刻）
    - 关键关系的互动中（关系张力）
    - 读者需要喘息的段落（节奏调节）
```

### 浓度自检

对于角色卡/人物描写中的每一行，问：**"如果删除这一行，角色的行为会改变吗？"如果不会，删除它。**

总角色卡字数控制在 2000 字以内。每一行都必须是能指导写作的操作性信息。

## 角色弧光追踪

角色不是静态的。每次章节完成后，在角色卡变更记录中追踪：

- **能力变化**：获得了什么新能力/失去了什么旧能力？
- **关系变化**：与谁的关系发生了变化？（近→远，敌→友，爱→恨）
- **信念变化**：什么核心信念被挑战或改变了？
- **关键事件**：什么事件触发了上述变化？

### 弧光检查（每卷至少一次）

- 主角在这一卷中改变了什么？
- 这个改变是不可逆的吗？
- 如果没有改变，那这一卷的意义是什么？
