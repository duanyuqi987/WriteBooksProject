# 第022章 操场边的单词

所属部卷：第一部：初秋三楼
时间：高一晚自习前
地点：操场西侧看台

## 诗歌插页 P022：原创仿唐绝句

看台日影落书边，
一页单词反复翻。
年少不知眸落处，
只将背影作文言。

事情真正发生时，并没有陆允川想象得那么郑重。晚自习前的操场，跑道被晒了一天，塑胶味混着草腥味飘在看台上。陆允川在看台上背单词。夏知遥和蒋雯在跑道边散步——她们走得不快，偶尔停一下，停的时候蒋雯会仰头看天，夏知遥会低头看地——走累了坐到看台第二层台阶上喝水。她离他大概五米——不是在他旁边，是在他斜下方，隔着一层台阶和两个水泥墩。水泥墩上有人用涂改液写了两行小字——"XXX到此一游"和"今晚考试必过"——字迹已经褪了色，被雨水冲得起白边。

这五米是他在室外离她最近的物理距离。五米——如果在走廊里擦肩而过，距离不到半米，但那不算，因为在走廊里他来不及看清任何东西。在这里，五米，他可以借着背单词的姿势安安稳稳地观察：五米足够他看清她头发被汗水贴在耳后的几缕碎发——头发不长，刚好到肩膀，发尾有点翘，不知道是剪坏了还是自然卷。也足够他听清蒋雯对她说的话——"明天英语小测你这次肯定能及格，你上次只差三分"。蒋雯的声音很大，大概是怕操场的风把话吹散。夏知遥应了一声"希望吧"，声音小得多，然后拧开矿泉水瓶盖喝了一口，仰头时喉咙动了一下。太阳刚好照在水瓶上，瓶身反了一下光。她今天穿了一件淡蓝色的运动外套，袖子捋到小臂，左手腕上戴着一根黑色的发圈。

陆允川手上的单词书翻到了"absorb"。他心里一直在默念这个词——不是因为它好用，而是因为"吸收"这个词正好能描述他此刻的状态：把五米外的每一个画面都吸进脑子里，存起来，等回宿舍以后再慢慢回放。

"你那个absorb已经念了三遍了，"程砚从旁边伸过头来，因为他刚从跑道跑完三圈回来，额头上还渗着细汗，呼吸不平，"你是不是在把这个单词翻译成'吸她'——同音不同义，你别乱翻译。"陆允川把书合上："你跑步的时候能不能把嘴也跑累？""不能。嘴是用来呼吸的，跑步的时候你不用嘴说话，只用来吸气，太浪费了。"

他坐下来，从书包里掏出水杯灌了半瓶，然后靠在看台护栏上往下面扫了一眼。他扫到了夏知遥坐在下面——然后收回目光，像什么都没看到一样翻开单词书继续背，声音大了一些："absorb——吸收；abstract——抽象的。"陆允川配合着往下背。

夏知遥和蒋雯在下面聊明天的小测。夏知遥说"阅读理解第三篇我总是看不懂"，蒋雯说"那篇是考古类生词多，你不要从头读，先读题再定位"。蒋雯还用手指在膝盖上比划了一下做题顺序——先看问题划关键词，再回去扫读找定位句。夏知遥想了一下说"行我今晚试试"。两人之间还传了一下那瓶水——夏知遥递给蒋雯，蒋雯喝了一口又递回去。这是最普通的高中生对话——考试、生词、试试看。但对陆允川来说，这也变成了他的课后资料之一：她阅读理解第三篇不太行。他默默存了下来，准备以后如果有机会讨论考试题可以用得上——虽然他不知道什么时候有用，但先存着总没错。他甚至想好了开场白："你第三篇是不是也觉得考古类的生词偏多？"——这个开场白到现在还没用过一次。

晚自习铃响了——铃声是从教学楼顶那个大喇叭传出来的，在操场上听起来有点远，闷闷的，像隔了一层棉被。操场上的人往教学楼方向走，看台上的也收书起身。有人把单词书往书包里一塞就跑，拉链都没拉，书包张着口。陆允川站起来把单词书塞进书包，拉链拉到一半，看见夏知遥从下面台阶站起来时手里那个矿泉水瓶没拿稳——瓶盖可能没拧紧，滚下了台阶，当当当碰了三层水泥台阶，最后翻了一圈停在草丛边。她在原地愣了一下，然后弯下腰去捡，背弯成一道弧，发尾从肩头滑下来。她握着瓶子上来了，瓶身沾了草屑和灰，她用手拍了两下瓶底，左右看了看没找到垃圾桶，就把瓶子夹在腋下继续走了。

"走吧，别想了。"程砚把他的水杯递过来，"明天还要小测，你的absorb还没背熟。"陆允川接过水杯喝了一口。程砚又说："背熟的意思是要会用——不是让你每天来操场上absorb。"陆允川差点呛到。程砚一笑："行了，开玩笑的。走了，食堂还有五分钟关门，赶紧去买个茶叶蛋。"

两人往教学楼走。脚下踩过跑道边缘的煤渣碎石子，嘎吱嘎吱响。陆允川回头看了一眼看台——空荡荡的台阶，夕阳的光已经从最上面一层移到了最下面，最后一点金色收在台阶的缝隙里，像被什么东西吸进去了。程砚在前面喊了一声"走快点食堂要收了"，他加快了步子。明天这个时候，他还会在这里，她也会在跑道上走几圈。这不是约定，只是规律——规律就是暗恋的人最需要的东西。约定会被打破，规律不会。太阳每天都会从同一个方向照过来，跑道上每天都会有人散步，他的单词书每天都会翻到同一页。
