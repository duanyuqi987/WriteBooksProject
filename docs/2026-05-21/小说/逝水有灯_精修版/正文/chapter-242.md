# 第242章 写成小说

所属部卷：第五部：逝水有灯
时间：多年后的清晨
地点：书桌和窗边

## 诗歌插页 P242：原创仿唐绝句

书稿微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

事情落到当天，没有被命运单独照亮。

陆允川把终稿从头到尾通读了一遍。五十万字，从第一页到最后一页，花了一整个上午。窗外的光线从灰蓝变成淡金，又变成白，中午时分的太阳正好打在他的书桌上，把纸页照得稍微有些刺眼——他起身把百叶窗转了个角度，光线被切成等宽的条纹落在书稿上。中午的时候他停了一次——吃了一碗泡面，又把程砚上次留的烤鸭骨头重新热了一分钟。微波炉转的时候骨头在盘子里劈啪响了几声，油星溅到了转盘上，他懒得马上擦。吃完继续读。读到下午三点，终于读完。喉咙有些干——期间忘了喝水。

读完以后他做了三件事。

第一件事：把稿子里所有人物的对话又重新对了一遍。程砚的话，每一句都像他——短，直接，夹杂实用的建议。祁野的话，每句都有"牛逼"或某个近似词——在关键时刻制造松弛。叶澄的话，旁观者清，不抢戏但每一句都准确。夏知遥的话，干净利落，带着一种职业女性的节奏。苏晚枝的话——他在面馆那一章得到了她的原话。不是他写的。是她说的。他做的事只删掉了多余的"了"和"的"。

他还对照了那天苏晚枝确认的几句台词。面馆的核心对话——他说了"三年前"，她的"十九岁的女生不知道怎么拒绝一个对她特别好的人"——是原话一字不改。她允许。

第二件事：把描写中所有残留的"似乎""好像""也许""仿佛"过了一遍。不是全部删除——有些地方需要犹豫。但大多数是写作者的自我防御——怕笃定了，怕负责了。他留下的只有五个"也许"。

第三件事：确认所有章节日期、地标、事件顺序没有逻辑漏洞。毕业照那章日期对上日记。出租屋那章日期对上水电费单。重回一中那章日期对上校友卡办办当天。南湖那章日期对上草浆味——那天校园确实刚剪过草坪，校报新闻有记载。婚礼那章坐第八桌——喻可本人确认。

做完这三件事以后，他把全部终稿存进三个地方：电脑本地硬盘、外接移动硬盘、加密云端。三个备份。防止任何一种意外。

然后他给所有该收到这个书稿的人发了邮件。程砚一个。祁野一个。叶澄一个。夏知遥一个。苏晚枝——最后发。不是刻意放在最后。是她的邮箱地址在联系人列表的最下面。

邮件正文每一封都一样：

"《逝水有灯》终稿。不要求回复。不要求提意见。只要求收下。谢谢你曾在这里。"

发给苏晚枝的加了一句："你在这里的篇幅最多。但你是唯一一个——被我写成'人'而不是'符号'的。谢谢你的原话。"

发完邮箱。电脑的散热风扇转了几圈。然后安静了。

他站起来，走到窗边。外面不是南湖，不是面馆，不是出租屋，不是一中校门。是他现在的城市——一栋普通公寓楼，楼下一排香樟树，一个小超市，一个公交站。

但这些地方都被他写下来了。那些在旧日子里的光亮，此刻就在书桌的书稿里——没有一个字是最终被篡改的。

程砚可能在十分钟后打开邮件。叶澄可能会在今晚。苏晚枝——应该已经在看了。

他把窗户打开得更大了一点。下午的风比早晨的暖。书桌上一支旧中性笔被吹得滚了几圈，停在键盘旁边。他捡起来，插进笔筒。

小说的最后还有一页空白。不是留给任何人补偿的。是留给自己的。

他把那张空白页翻过去。封底。

窗外下午的阳光已经变成了金黄色——那种冬天特有的、温和而不刺眼的颜色。阳光打在对面的香樟树上，把树冠的轮廓镶了一道金边。树下有一个老头在遛狗——一只黄色的土狗，慢慢悠悠地跟在老头后面，尾巴摇得很懒。老头走几步停一下，狗也走几步停一下。两个速度完全匹配，像是被同一个时钟校准了。

陆允川在窗边站着看了一会儿。然后转身回到书桌前，把《逝水有灯》的书稿重新整理了一下。书稿的右下角有一点茶渍——大概是昨天喝龙井时不小心洒的。他把那一页抽出来，用纸巾吸干了残余的水分。茶渍不会消失——水可以吸干，颜色会留下来。但这不重要。书稿上的茶渍就像旧日记本上的删除线——它们都是真实发生的证明。

他把书稿装进牛皮纸文件袋。封面写：《逝水有灯》终稿。日期是今天。然后他把文件袋平放在书架上——和牛皮日记本、毕业照、出租屋照片、诗歌索引并排。同一个格子，终于填满了。

他退后一步，看着这一格书架。四十五首诗。五十万字。十年。一个格子。

他忽然想起程砚那天在拉面馆说的话——"你是在给自己办一场迟到的毕业典礼。"现在这场典礼结束了。没有学士服，没有抛帽子，没有"我们毕业了"的横幅。只有一支旧中性笔、一盏旧台灯、一个终于填满的格子。

全书终。
