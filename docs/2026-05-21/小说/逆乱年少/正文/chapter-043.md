# 第043章 银杏叶

时间：2011年5月初，周五下午

地点：图书馆三楼文学区

卷次：第二卷：冬天的图书馆

书还回来的时候，林玉起没有立刻翻开。

周五下午，同样的图书馆三楼。苏安晴把书递过来，说了一句"谢谢，很好看"，然后就去教育学的书架那边了。她的书包还是装得整整齐齐，拉链拉得严严实实。

林玉起把书拿在手里，封面上多了一个透明的塑料书套。不是图书馆用的那种，是小卖部里卖的那种——一块钱一个，最便宜的那种。

他愣了一下。他把书借出去的时候封面有些旧，书的边角也有些磨损。现在书还在那个塑料套里，平平整整的，像被小心对待过的样子。

他在靠窗的位置坐下，翻开书。扉页还在，他写在扉页上的日期还在——"2009年8月购于封丘新华书店"。然后他翻到了中间。

一片银杏叶。

金黄色，压得很平，脉络清晰得像地图上的河流。夹在张九龄《感遇》那一页。

林玉起把叶子拿起来，对着窗外的光看了看。去年秋天，十月底的时候，他在文华路金黄的银杏树下捡的。他捡了好几片，夹在不同的书页里，然后忘了。真的忘了——不是刻意藏在某个地方等谁来发现，就是忘了。

但现在它在这里。在"草木有本心，何求美人折"那一页。

他不知道苏安晴翻到这一页的时候想了什么。她有没有把叶子拿起来也对着光看？有没有想这片叶子夹在这里多久了？有没有想——他是故意的，还是忘了？

如果是故意的，那这首诗选得太巧了，那片叶子夹得太准了，准到像在说一个不能说出来的意思。但事实是——他真的忘了。他连自己夹了这片叶子都不记得。

这个"忘了"比"故意的"更让他不自在。故意的至少是他在控制，忘了——连他自己都控制不了。

他把叶子重新夹回去，合上书。窗外文华路的银杏树现在是绿的，新的叶子刚刚长满枝头。去年秋天的那片金黄，已经在这个书页里睡了整整一个冬天和一个春天。

晚上回到宿舍，郭向阳趴在桌上看电影。周明和陈磊在讨论明天的篮球赛。没有人注意林玉起多看了那本书两眼。

他把书放在床头——那个位置原来是他的笔记本。现在笔记本被挤到了枕头下面。

> 放下书
> 抬起头
> 发现自己
> 把一片遗忘的秋天
> 留在了春天

他在笔记本上写下这四行，然后划掉了后面两行。何南星说过——"你已经用整首诗说了，不需要最后再总结"。

关灯以后，他躺在黑暗里想：她包了书皮。她可能只是觉得书有点旧，她习惯把东西保护得很好。就像她把筷子摆齐，把书包收拾整齐——这只是她的习惯，跟他没关系。

但银杏叶呢？她翻到那一页的时候，有没有停下来读那首诗？

他没有答案。但他知道那片叶子还夹在书里，和那首诗待在一起。
