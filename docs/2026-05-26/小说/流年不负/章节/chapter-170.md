# 第170章 领证
时间：2019年1月
地点：青阳县民政局
卷次：卷九：执手之约

一月中旬一个周三的早晨。青阳县民政局门口。

八点四十。门口已经有人在排队了。不是长假，不是节日，但结婚登记处从来不缺人。前面有两对——第一对很年轻，看起来像刚毕业的学生，男生紧张得一直在看手表；第二对年纪大一些，大概三十出头，女生抱着一个文件夹，里面装着各种证件。

顾临风和苏晚晴排在第三。

天冷。一月的早晨气温很低——嘴里呼出的气在空气里变成白雾。苏晚晴穿了一件深蓝色的大衣，围巾还是那条灰色围巾，围了两圈。顾临风穿的是黑色羽绒服——他妈前年买的，袖子有点短了，但他一直没换。

她站在他左边。手里捏着两个人的身份证和户口本。身份证在手里来回翻着——正面看看，反面看看。她的照片是大学时候拍的，刘海有点厚，笑容很淡。他的照片是工作后拍的，头发比大学短了很多，表情很严肃——像在拍监考照。

"照片上的人都很年轻。"她说。

"你现在的照片也很年轻。"

她没接这个话。把两张身份证并排放在一起。两个名字——"苏晚晴""顾临风"——挨得很近，只隔了身份证塑料壳的厚度。照片上的两个年轻人都没什么表情。但他们马上要把名字写在同一张证书上了——并排，不隔任何东西。

前面排队的第二对进去了。轮到他们的时候快九点半了。

婚姻登记处的窗口不大。柜台是米黄色的人造石，台面上放着一盒印泥、一个笔筒、一瓶胶水——胶水瓶上沾着干涸的白色胶渍。办事员是个四十多岁的女人，短发，戴眼镜。眼镜腿上挂着一根防掉绳——那种老式的黑色弹簧绳。

她把两个人的材料接过去。身份证、户口本、照片——一张一张地摊在桌面上，像铺牌一样。然后拿起两张照片对着灯光看了一眼。

"正面免冠。合格。"

声音像在念课文——但比课文更平。

苏晚晴坐在柜台外面的椅子上。她把大衣口袋里的双手拿出来，放在膝盖上。膝盖并拢——跟第一次去他家时一样。顾临风站在她旁边——没有坐。站着可以看到柜台里面。办事员在电脑上敲字，键盘的声音很脆——哒哒哒哒。

"出生日期。"

"1993年11月——"

"你们两个分别报。"

"顾临风。1993年11月8号。"

"苏晚晴。1994年3月21号。"

"职业。"

"教师。"

"教师。"

"确认一下。双方都是自愿结婚？"

"是。"

"是。"

"近亲关系——"

"没有。"

"没有。"

办事员抬起头看了他们一眼。眼镜片反着电脑屏幕的光，看不清她的表情。但她看完了两个人之后点了下头——像是在确认什么东西。

"材料没问题。去旁边那个窗口交一下工本费。"

工本费九块九。顾临风在牛仔裤口袋里翻了半天——找了十块。收钱的阿姨找了两个五毛的硬币。硬币掉在柜台上转了两圈，他用手指按住。

回到登记窗口的时候。办事员已经把所有材料收好了。她拿出两个红本本——封面印着金色的国徽和"中华人民共和国结婚证"几个字。她翻开一本，把照片贴上去，放进旁边一个黑色的小机器里。机器"滋滋"两声——钢印压下去的声音。

然后拿起第二本。同样的动作——贴照片、放机器、"滋滋"。

她把两本红色的结婚证合上。递过来。

"恭喜。"

两个字。跟她说"材料没问题"时的语气差不多。但在推过结婚证的时候，她嘴角动了一下——一个很淡的、职业性的微笑。不是假笑——是那种见过了几百对新人之后还愿意笑的笑。

顾临风接过红本本。

沉。很轻——一张纸的重量。但他接过来的时候觉得沉。像接过期末考试卷子时那种沉——不是因为拿不动，是因为上面写着东西。写着名字。写着两个人的名字。

他把结婚证打开。

里面左页是结婚证编号、发证机关、有效日期。右页是照片——两个人的合照，他深蓝色衬衫、她白色毛衣，并排坐在红色的背景布前。照片下面是两个人的名字。

苏晚晴。

顾临风。

两个名字并排印在一起。不是手写的——是打印的宋体字。笔画清晰，间距均匀，每一个字的墨色都一样深。就像两个名字天生就该放在一起——从笔画结构到视觉距离，一切都刚刚好。

他看了很久。

久到办事员开始处理下一对。久到苏晚晴在旁边轻轻碰了一下他的手肘。

"走不走。"

"等一下。"

他又看了一遍。确认了一遍——名字没错、日期没错、照片没错。照片上她的嘴角微微往上弯——不是笑，就是弯了一点。他也没笑。但他的眼睛里有一点光。

她把属于她的那本接过去。翻开看了一眼。然后合上。再翻开看一眼。再合上。

来回翻了三次。

"走啦。"

他站起来。把红本本放进羽绒服内袋里——胸口的位置。跟他放诗稿的位置一样。口袋里还有那张被汗浸湿了一角的诗。红本本贴着诗稿——一个是在最紧张的时候写给她的，一个是在最不紧张的时候签下的。

走出了登记处的门。穿过走廊。推开民政局大门的玻璃门。一月的阳光照进来。

阳光很好。

不是那种刺眼的白光。是冬天特有的淡黄色阳光——不强烈，不咄咄逼人。照在脸上有一点暖意，但不会出汗。地上的影子很淡——淡到几乎看不出来。

她站在民政局的台阶上。把结婚证从大衣口袋里拿出来。对着阳光看——光线透过红皮封面，把封皮映成了半透明的橘红色。国徽的金色在太阳底下反光——一小团金。

"我们结婚了。"

她说这话的时候没有看他。看着结婚证。声音很轻。轻得跟昨晚说"好"一样。轻得跟六年前说"我需要想一想"一样。

但这次是这四个字——"我们结婚了"。

"嗯。"他说。

然后笑了。

不是大笑。不是咧嘴笑。就是嘴角往上弯了一点，眼睛里面的光比平时亮了一点。那种笑很自然地——不是刻意的、不是对着镜头摆的、不是结婚照上那种"一、二、三、笑"的笑。是自然而然的。是他说完"嗯"之后没法控制的那一点点面部变化。

眼睛里有光。

不是眼泪。不是反光。就是一种光——像是早上拉开窗帘的那一瞬间，阳光照进来然后散在空气中的那种光。很小。很淡。但存在。

她看着他。看到了那个笑。看到了他眼睛里的光。她没说话。也弯了一下嘴角——那种弯，跟他的一模一样。

两个人站在民政局门口的台阶上。各自拿着各自的红本本。一月的太阳照在两个人身上。后面是不断推门进出的新人——有的紧张，有的笑着，有的在打电话给家里报喜。

他们没报喜。他们只是站在那里。

然后她把红本本收起来——放回大衣内袋里，拉上拉链。这个动作很慢。拉链从下往上拉，拉到胸口位置的时候停了一下。然后继续往上，拉到头。

"中午吃什么。"

"你说。"

"红烧鱼。"

"上次在我家吃的那种？"

"嗯。我妈昨天教我了。"

"你学会了吗。"

"没有。但可以试试。"

"炒糊了呢。"

"那你吃。"

她笑了一下。不是弯嘴角——是真的笑了一下。笑得很短，大概一秒。但那是他今天看到的第二个笑。第一个是他自己的。

他走下台阶。她跟在旁边。两个人并排走在青阳县的街道上。一月的街道没什么人。路边梧桐树的叶子全掉光了——光秃秃的枝丫指着天空。阳光穿过枝丫，在路上画出很多交叉的线条。

她的肩膀碰到了他的手臂。羽绒服和棉衣之间隔着两层填充物。没有温度交换。但她没有挪开。就那样贴着，走过了整条街。

从民政局回家。坐公交。半小时的车程。青阳到楚州的班车。周六乘客不多。他们坐在倒数第二排。她把头靠在窗户上。红本本从大衣口袋里露出一角。他帮她把那一角塞回去。手指碰到她的大衣——冰凉的。但里面贴着她体温的那本结婚证，是暖的。

公交车在省道上开着。两边的田里什么都没有——一月的江南，稻田里只有收割过的稻茬，整整齐齐地排列着。阳光照在稻茬上，每一根都有一道细细的影子。

窗外的天空很蓝。冬天的晴空没有云。蓝得很透。像高中时候画画用的那种蓝色颜料——加了很多水调开，然后薄薄地涂在纸上。能看到纸底下的白色。现在天空底下是冬天的稻田——黄色和蓝色的交界面是一条很直的地平线。

她在公交车上睡着了。头从窗户滑下来，靠在他的肩膀上。他肩膀没动。一整个车程都没动。右臂麻了——从肩膀到手指都不是自己的了。但他一直没动。

到楚州的时候是下午一点。太阳偏西了一点——但还是很亮。月湖的湖面上反射着太阳的光，碎钻一样的波纹一波一波地荡。远处的水塔还在。塔顶的灯在白天不亮——只有一根金属杆子立在那里。

他碰了碰她的肩膀。

"到了。"

她睁开眼睛。迷糊了两秒钟。然后低头看了看怀里的红本本——还在。

"我们回来了。"

"嗯。"

她站起来。拿起包。他把她的围巾递过去——刚才掉在座位上了。她接过去围好。公交车停在月湖路口的站牌。车门打开，一月的冷风涌进来。

她第一个走下去。站在站台上等他。手里紧紧捏着那本红色的结婚证。封面在冬天的风里被吹动了一角——但她按住了。

他下车。站在她旁边。

"走吧。回家。"

"回家。"

两个字。以前也有过一个"家"——他在楚州租的小公寓。她在青阳的老房子。但现在这两个字不一样了。现在"回家"是去同一个地方。一室一厅。一个阳台两盆花。一个厨房一个卫生间。一张桌子两边写字。

那条路他们走了很多遍——从月湖路口到公寓楼下，大概步行十分钟。今天走了十二分钟。走得很慢。不是因为累。是因为不太想走快。

路边的梧桐树也在掉叶子——最后几片叶在冬天的风里打着旋。有一片落到她头上。他没去摘。那片叶正好落在她的发卡上——银色发卡顶着枯黄叶子，像一朵花。
