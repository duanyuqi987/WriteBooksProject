# 第077章 毕业照边缘

所属部卷：第二部：六月以后
时间：六月中旬
地点：学校操场主席台前

## 诗歌插页 P077：原创仿唐绝句

风色微明照旧尘，
少年别后识前因。
一声铃尽人潮散，
半页风光不问春。

"三——二——一——"

快门响的那一刻，有人笑场了。

是程砚。他站在最后一排中间，突然来了一句"你们觉不觉得摄影师长得像物理老师"。周围几个人全笑了。摄影师从相机后面探出头来，表情很复杂："同学，能不能严肃一点？这是毕业照。"

"对不起对不起。"程砚举起双手表示歉意，但眼睛里还是笑意。

"重来。三——二——一——"

快门又响了。这次没有人笑场。但在快门按下的那一瞬间，陆允川的校服袖口被一阵风突然吹起，袖口翻过来露出里面的白色内衬。他的目光下意识地跟着袖口往下落，所以在照片里他的视线是微微低垂的——不是看着镜头，是看着某个不在镜头里的方向。

夏知遥站在第二排中间。快门响的时候她刚好把头发别到耳后，手指还停在耳侧。这个动作被镜头捕捉下来，成了她在整个高中最后一张集体照里的样子——认真的、有一点紧张的、但带着很浅的笑意。

"好了。解散。"摄影师挥了挥手，"底片发到你们班主任那里。需要加印的找教务科。"

队伍立刻散了。有人冲过去跟老师合影，有人拽着好朋友在主席台前面不走，有人在班里最漂亮的女生旁边排着队等合照。体育委员拿着手机到处录视频，抓到谁就问："你对高中三年有什么感言？"大部分人的回答都是"不知道""太长了""就那样吧"。

程砚从最后一排跳下来，跑到陆允川身边。"我刚才说你没看镜头。"

"我知道。"

"你怎么不看镜头？"

"风吹的。"

"你放——"程砚把"屁"字吞了回去，"风吹的能让你的眼睛往下看？你明明在看——算了，不问了。"

陆允川没说话。程砚拍了拍他的肩膀，掏出手机。"站好，我给你单独拍一张。"

"不用。"

"什么不用。以后你翻集体照想放大看某人，结果自己连张清楚的单人照都没有，多丢人。"程砚把手机举起来，对准陆允川，"往左站一步。对。别挡到光。笑一下。"

陆允川笑了一下。

"你管那叫笑？"

"笑了。"

"行，反正不是我拍自己。"程砚按下快门，然后翻回来看了看效果。"还行。比你高一入学照好看。你那时候跟包子似的。"

叶澄在另一边帮前排女生收拾凳子。那些蓝色塑料凳叠起来很重，她一个人搬了两趟。夏知遥过来帮她，两个人一人一头抬着一摞凳子往器材室走。经过主席台边的时候，夏知遥的凉鞋踩到了地上的横幅边缘——横幅没贴牢，滑了一下。叶澄伸手扶住她的胳膊。

"没事吧？"

"没事。"夏知遥站稳了，看了一眼地下的横幅，"下次应该用双面胶。透明胶不行。"

"你跟班长说，让他反馈给教务处。"

"算了。反正我们也用不到了。"

两个女生把凳子放进器材室。出来的时候夏知遥说："谢谢你刚才扶我。"

"客气什么。"叶澄用手背擦了一下额头的汗，"你报哪个学校？"

"还没定。等分数出来再说。你呢？"

"师范。第一志愿英语。"

"你肯定能考上。"夏知遥说得很笃定，"你英语是我们班最好的。"

叶澄看了她一眼，没有谦虚，也没有自夸，只是点了下头。

操场上的人渐渐散光了。横幅被取下来卷好，蓝色凳子被收回器材室，三脚架被折叠起来装进摄影师的箱子里。陆允川站在操场边上，看着主席台前面那块空地——刚才他还站在上面，脚尖踩着一道白线。现在白线还在，人已经都走了。

程砚从后面走过来，把剩下的半瓶矿泉水递给他。"喝点水。走了。我妈说中午吃饺子，你一起来。"

"你妈又没叫我。"

"我叫你了就等于我妈叫你了。走吧。"

陆允川接过矿泉水瓶，喝了一口。然后转身跟上程砚。操场上的热气从地面往上蒸，把两个人的背影蒸得有一点变形。
