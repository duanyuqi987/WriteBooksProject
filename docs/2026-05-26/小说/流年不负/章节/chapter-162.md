# 第162章 她爸的茶
时间：2018年9月
地点：青阳县苏家
卷次：卷九：执手之约

苏晚晴的母亲是在下午四点半回来的。

钥匙插进锁孔的声音响了三下——转一圈，回半圈，再转。门开了。一个穿着碎花衬衫的中年女人站在门口。手里提着塑料袋——芹菜、一条鱼、一袋冻排骨。

她看到顾临风的时候愣了一下。然后笑了。

"小顾——是小顾吧？"她把菜放进厨房。"坐坐坐，别站着。苏晚晴，你怎么让小顾站着。"

"他刚站起来。"苏晚晴说。

"妈好。"顾临风站着没动——不知道该叫什么，就跟着苏晚晴叫了。

"叫阿姨就行。"苏母从厨房走出来，围裙已经系上了。"在楚州回来的？坐车累不累？"

"不累。开车回来的。"

"你买车了？"

"前几个月买的。二手的，代步。"

"好好好。有车方便。"

苏母话多。进了厨房就开始说——问他在楚州住哪里，学校离苏晚晴教的小学远不远，平时自己做饭还是在外面吃。问一句就切一刀菜。芹菜切完了，又问了他家里人身体怎么样，他父母多大年纪了。

顾临风站在厨房门口，一一回了。

苏晚晴在客厅里择菜。青菜买了两把，她一片一片掰下来，黄的扔进垃圾桶，好的放进盆里。

苏父还没回来。苏母说他去菜市场后面的卤菜店了——"那家的酱牛肉好，买了多少年了。"

半个小时后苏父回来了。手里提着两个塑料袋子。一个装的是酱牛肉，一个装的是凉拌素菜——木耳拌黄瓜，多加了一把花生米。

四个人在厨房和客厅之间来回倒腾了一个小时。苏母亲自掌勺——红烧鱼、糖醋排骨、芹菜炒肉、家常豆腐。锅铲的声音一直没断过。油烟机太老了，声音很大——嗡——但苏母说话的声音更大。

"苏晚晴小时候最讨厌吃芹菜。你猜怎么着？后来居然习惯了。"

"妈——"

"我又没说错。"

顾临风在客厅里坐着。茶几上的紫砂壶还放在那里。茶已经凉了。

苏父从厨房端出五盘菜——茶几不够大，又加了一张折叠桌。苏晚晴把茶几上的茶具收起来，放在电视机旁边的柜子上。

"吃饭吧。"苏母端出最后一道菜。拿碗、摆筷子。四双筷子摆在四个方向。苏晚晴的碗跟他挨着——两个人的碗沿碰到了，发出轻轻一声"叮"。

红烧鱼的酱油味。糖醋排骨的酸甜味。芹菜炒肉的蒜香。汤是紫菜蛋花汤——很家常，蛋花打得碎，紫菜沉在碗底。

"小顾你吃。别客气。"苏母用筷子指了指鱼。"这条鱼清蒸是他爸喜欢的，但今天人多，就红烧了。你尝尝。"

顾临风夹了一筷子鱼肉。刺很少。酱油放得不多，能吃出鱼肉本身的鲜味。

"好吃。"

苏母满意地点了点头。

饭桌上苏母一直在说话。说苏晚晴小时候的事——小学一年级一个人走了三公里路去学校，她和苏父满县城找；初中的时候在学校里跟男生打架，因为那个男生扯了她的辫子；高中三年学习太拼了，瘦得下巴都尖了。

苏晚晴低头吃饭。筷子在碗里拨来拨去。

苏父没怎么说话。吃饭很慢——一口饭嚼很久。偶尔抬起眼看顾临风一眼，又回到碗里。

饭吃到一半。苏母忽然问了一句。

"小顾，你和苏晚晴的事，你爸妈知道吗。"

"知道。他们也挺高兴的。"

"见过面了？"

"还没。打算下次带她回去。"

苏母点点头。又夹了一块排骨放进苏晚晴碗里。

"这丫头，从高中起就不怎么跟家里说这些事。大学四年，问她有没有谈朋友，每次都说'没有'。"

"妈——"

"现在有了。挺好的。"

苏晚晴的筷子停了一下。然后继续夹菜。

顾临风在桌子底下碰到她的手。凉凉的。像高考后在月湖边第一次牵她的手时一样。

她没抽开。

苏父把碗里的饭吃完了。放下筷子。端起茶杯喝了一口——不是茶，是白开水。他往杯子里兑了点热水，摇晃了一下杯子，杯底的水垢翻起来又沉下去。

"小顾。"他第一次主动开口。

"叔叔您说。"

"我问你几个问题。不是为难你。是想听你说实话。"

苏母看了苏父一眼。但没说话。

"你为什么要娶我女儿。"

顾临风放下筷子。手掌在膝盖上摊开。

"因为我喜欢她。从高一开始。"

苏父没有任何反应。

"高一到高三。三年。加上大学四年。加上毕业之后。一共八年多。中间有六年没联系。但我没喜欢过别人。"

"不是没人介绍。"

"是有人介绍。也试过跟别人相处。不行。"

"为什么不行。"

"心里放不下。"

苏父看着他的眼睛。看了大概三秒钟。

苏晚晴的筷子停在碗口。没有动。

"你们现在都在楚州。你是中学老师。她是小学老师。"苏父说。"工作稳定。但生活不光是工作。以后有了孩子——"

"想过。"顾临风说。"房子打算在楚州买。两个人的公积金加起来够还月供。她教小学，以后孩子上学方便。我这边评上中级职称之后工资会涨一些。两边老人——青阳到楚州开车一个小时，随时能回来。"

苏父听着。从口袋里摸出烟盒。打开，拿出一根。这次他点了。

打火机"啪"的一声。烟头红了。

他吸了一口。烟雾往上飘。苏母扇了扇——但没说什么。

"你说的这些都是计划。"苏父说。"计划是好的。但过日子不按计划走。"

"我知道。"

"你知道什么。"

"我知道以后会有变数。她工资会不会涨，我职称能不能评上，房价会不会涨，两边老人身体好不好——这些都不是现在能确定的。"

苏父没说话。

"但是我确定一件事。"

"什么。"

"不管以后怎么变。她在哪。我在哪。"

苏父把烟头按进烟灰缸里。按了两下。然后端起茶杯。杯子里还有半杯水。

他喝了一口。

然后"嗯"了一声。

一个字。跟他说"坐"的时候一样。跟他说"喝吧"的时候一样。

但顾临风听出了区别。

那声"嗯"后面，没有问题了。不是"嗯？"——是"嗯。"

苏母在旁边笑了一下。然后又夹了一大块鱼放进顾临风碗里。

"小顾你别光说话，吃菜。红烧鱼凉了就不好吃了。"

苏晚晴从碗边抬头看了她爸一眼。她爸正低头喝茶。杯子里已经没有水了，但他还在举着杯子。

客厅里吊扇在头顶转——三挡，不快不慢。扇叶的影子在墙上转圈。

楼上的小孩子又开始跑了。这次脚步更轻——像是穿了袜子在地板上滑。那个女人的声音又响起来——"再跑就把你的玩具收起来。"

然后是沉默。脚步声停了。

苏母站起来去盛汤。紫菜蛋花汤只剩底了，她把锅底的紫菜全倒进了顾临风碗里。

"多吃点。太瘦了。老师也辛苦。"

苏晚晴看着碗里的汤。又看了看她妈。

"妈，他已经吃了两碗了。"

"两碗多吗？年轻人吃三碗正常的。你爸年轻时一顿能吃四碗。"

苏父把茶杯放在桌上。站起来。

"我先去洗澡了。"

他走过顾临风身边的时候，停了一下。手拍了拍顾临风的肩膀。

就一下。然后走了。

那只手很重。掌心有老茧——常年在厂里做车工留下的。拍在肩膀上的声音很闷——噗的一声。

顾临风没回头。

但他在桌子底下握住了苏晚晴的手。这次她的手不像高考后那么凉了。

她反握了一下。

厨房的水龙头响了。苏母在洗碗。碗碟碰撞的声音和她不成调子的哼唱混在一起。

窗外的天已经暗了。楼下院子里的灯亮了——一盏老式白炽灯，灯光昏黄。

老槐树的叶子在晚风里动着。
