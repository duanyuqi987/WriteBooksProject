# 第022章 春天的走廊

时间：2011年4月

地点：教学楼走廊

卷次：卷二：窗边人影

四月的走廊风是暖的。

青阳一中的教学楼是个"回"字形。中间是块空地，种了两排冬青。三楼的走廊在东侧，从尽头可以看到对面教室的窗户。

顾临风发现了这件事。

那天课间他去走廊尽头接水。饮水机在一排窗户前面。他接完水，抬头看了一眼——对面是理科三班的教室。隔着中间的空地，大概二十米的距离。

第二排靠窗。

苏晚晴在。

她正低着头写字。马尾垂在肩上。后背挺得笔直。

从这个距离看不到她的脸。但能看到轮廓。能看到她写字的手在动。

他端着水杯站在饮水机旁。水是热的，杯子有点烫手。

上课铃响了。

她抬起头。往窗外看了一眼。

他往后挪了半步。

她没看到他。

后来他开始去走廊尽头站着。

课间十分钟。以前他会跟赵一鸣去小卖部。买辣条、买冰棍、买可乐。现在不去了。

赵一鸣问"你怎么不去小卖部了"，他说"最近肚子不舒服"。

借口用了三天。第四天赵一鸣没说别的。自己带了两包辣条回来，扔了一包在他桌上。

他在走廊尽头一站就是十分钟。

有时候能看到她——在做题，在看书，在和同桌说话。有时候看不到——她的座位空着，大概是去接水了。

有一次她站起来往窗边走。推开窗户。风吹进来，把桌上的卷子吹翻了。她伸手按住。

她对着窗外站了几秒。

他站在对面走廊看着她对着窗外站。

然后她转身回了座位。

他继续站了一分钟。直到上课铃响。

春天的风把她的马尾吹起来一点点。他看到了。

赵一鸣有天中午问他。食堂里，两个人面对面坐着吃饭。

"你最近老往走廊那边站。"

"嗯。"

"那边有什么好看的。"

顾临风夹了一块土豆。

"没什么。透透气。"

赵一鸣吃了口饭。

"对面是咱们班的教室。"

顾临风没说话。

赵一鸣也没再说话。把盘子里的鸡腿夹给了他。

"多吃点。你最近好像瘦了。"

顾临风把鸡腿吃了。

四月下旬。走廊里有风。顾临风站在走廊尽头，看对面窗边的她。

她在翻一本课外书。不是课本。封面是蓝绿的。他认不出是什么书。她翻了大概十分钟。然后合上。放进书包。

她往窗外看了一眼。

这次他没躲。

隔着二十米的空地，她看不到他——距离太远了。但他能看到她看窗外的样子。

她没有笑。也没有皱眉。就只是看着窗外。看了大概五秒钟。然后低下头继续做自己的事。

顾临风把杯子里的水喝完了。

水是凉的。

上课铃响。

他回教室之前又看了一眼——她已经在做物理题了。笔在草稿纸上画着，头低着，马尾搭在一边的肩膀上。

那个画面他记住了。

很多年后他还记得——四月的走廊、远处的窗、窗里的她。
