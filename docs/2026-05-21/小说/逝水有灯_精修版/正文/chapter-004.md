# 第004章 初秋三楼旁观

所属部卷：第一部：初秋三楼
时间：2010年九月
地点：县一中三楼东侧教室

## 诗歌插页 P004：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
窗外法桐落下的第一片黄叶还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，初秋三楼会显得没有那么沉重。教室、走廊、操场里站着许多人，每个人都有自己的迟到、作业、饭卡和回家路。

叶澄后来跟人说起陆允川，用的词是"认真得有点累"。她说开学那天就注意到他填表格时反复核对的样子，像在做中考卷子。"他连写个家庭住址都要先在草稿纸上打一遍底，我就觉得这个人心事重。"叶澄的看法来自多次观察：他整理课桌时书脊一定要对齐，做眼保健操时手指按穴位的位置比广播里示范的还准，交作业时总是把本子正面对着老师的方向。

"这种人不难相处，但他不会主动让你靠近。"叶澄这样总结。

程砚的看法更直接。他和陆允川分到同一个宿舍，第一天晚上就发现陆允川睡前要把明天要穿的衣服叠好放在床头——不是乱塞，是整整齐齐，连衣领都翻好了。"我说兄弟你这样累不累？他回了我一句'习惯了'。我说那你习惯得有点过头。"程砚不喜欢绕弯子。他在宿舍里开过玩笑："你们注意到没，老陆看那个谁的时候笔都不动了——他还以为自己藏得很好。"室友们笑了一阵，陆允川当时不在，所以这个笑话暂时绕过了它该绕的那个人。

报到那天，程砚从后排观察了一下午。他看到陆允川的眼神在教室里绕了一大圈，最后总是收在一个方向。程砚没有当面戳穿，但私下跟叶澄说："我觉得他这次可能来真的。""来真的？第一天就来真的？""第一天还不够？你觉得感情需要质检吗？"叶澄被他问住了，想了想说："别乱讲。万一人家真的放在心上，你这样说他以后更不敢说了。"程砚挑了挑眉："我就是因为知道他放在心上，才先说给你听。说给他听他有压力。"

夏知遥在这个故事里完全无辜。她坐靠窗那一列，旁边两个女生一个叫蒋雯一个叫何敏，三人在开学第一周就搭了伙。蒋雯负责记课表，何敏负责提醒大家带饭卡，夏知遥话最少，但她的意见通常最重——比如物理老师讲错一个公式变形，何敏和蒋雯还在讨论，夏知遥已经翻到了课本对应页码，指给她们看。

她不知道陆允川的每一次停顿。就像陆允川也不知道她的书包里塞着一封没写完的信，不知道她每周五放学后要在校门口等一辆去外婆家的城乡公交，不知道她上个月因为分班的事和家里吵了一架。她有时会因为题目做不出皱眉头，有时会在蒋雯讲笑话时笑得趴在桌上，有时也一个人站在走廊尽头看操场上的人跑步——那些时刻和陆允川毫无关系。

叶澄晚自习后对陆允川说过一句话。那天教室里只剩几个人，她走过去，没有寒暄，直接说了一句："喜欢不是错。但你把它当秘密扛着，扛久了对谁都不公平。"她说完就去收拾讲台了。陆允川站在座位边，手里捏着一本卷了角的练习册，好一会儿没有动。

程砚偶尔在宿舍里开导他——虽然"开导"两个字用在程砚身上不太准确，因为他的方式更像"顺手敲一敲"。"老陆，你知道你这人最大的缺点是什么？""你别说了。""你太怕别人看出来，结果搞得所有人都看出来了。"陆允川把头埋进被子里，闷声说："睡觉。""行，睡觉。但明天吃饭的时候眼珠子别又定在六点钟方向。"

从旁观者的角度看，陆允川的心事既不神秘也不可耻，只是太年轻，所以每个动作都像在演默片——他以为别人看不懂，其实每个旁观者都在心里给他配了音。

叶澄在作文里写过一句话，被语文老师圈出来当范文念："青春就是把很小的事看得很大，又没办法承认它其实很大。"她没写任何人名，但程砚听了以后回头看了陆允川一眼，陆允川正低着头看书，完全没对上这个信号。

夏知遥也许隐隐察觉到一点什么——她的感知不是主动的，而是被动的，像窗户开着一条缝，风自然会透进来。有一次发作业本，陆允川的本子被传到她手里，她看了一眼名字，顿了一瞬，然后把本子放到他桌上，说了句"你的"。那是她对他说的第一句话，两个音节，语气平得像念天气预报。陆允川说了声"谢谢"，声音小到他自己都没听清。夏知遥已经转身走了。

程砚在走廊里撞见这一幕，回到宿舍后对陆允川说："人家跟你说了两个字，你回了两个字，加起来四个字——你怎么就不敢多说两个？""加什么？""比如'谢谢'后面加一句'今天作业多吗'，很难吗？"陆允川不做声，把被子拉过来蒙住头。"行，你又睡觉。你这招在我这儿已经失效了，明天换一个新的。"

旁观不是冷漠。程砚的嘴欠里有在乎，叶澄的平静里有分寸，室友们的玩笑里有不愿明说的关心。陆允川不喜欢被旁观，因为被旁观意味着他的心事不够特殊——而少年人最怕的，就是自己的深情不够独一无二。

但正是这种不独一无二，让后来的出路变得可能。大家都年轻过，都笨拙过，都在某间教室里因为某个人的一个动作心跳过快。如果每一份笨拙都要被写成悲剧，那满大街走着的人就都别想过好自己的生活了。

有一天晚自习前，叶澄把表格交完回来，经过陆允川座位时说了一句话，她的声音很轻，像顺手把一件东西放到桌上："别把你还没说出口的戏，提前让她替你演完。"

陆允川抬起头，叶澄已经走过去了，正在和后排的女生讨论明天交什么作业。窗外起风，窗框震了一下，头顶的日光灯轻轻晃了晃。他把桌上的数学卷子翻了个面，继续做选择题。铅笔在纸上画出一个"C"，笔迹很浅，像是犹豫了一下才落笔。

他不确定这句话是什么意思，但他把它记住了。
