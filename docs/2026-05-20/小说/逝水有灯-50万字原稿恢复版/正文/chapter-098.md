# 第098章 省城南湖之后

所属部卷：第二部：六月以后
时间：大一九月
地点：大学南湖凉亭

## 诗歌插页 P098：原创仿宋小令

风入旧窗轻，
纸上有余声。
欲问前程何处去，
水边灯影自分明。
回首不须惊。

省城南湖之后，陆允川没有立刻变得清醒。他只是比前一天更安静，像把一场很大的声音关进了抽屉。

陆允川来到大学，湖水、垂柳和喷泉让他短暂安静，他开始想试着过另一种生活。这句话在他心里反复出现，每出现一次，就少一点戏剧，多一点具体。

湖边有人练吉他，和弦断断续续。陆允川听了一会儿，没觉得像谁，只觉得这座学校有自己的声音，不需要借旧人来证明。

程砚看他不说话，没再开玩笑，只把新校园卡往他面前推了推，说：“先处理眼前的，别把人想散了。”

叶澄的提醒更直接：“你可以记住难过，但别用难过去改写别人。”陆允川听着不舒服，却没有反驳。

夏知遥那边仍然有自己的事情。夏知遥在另一个城市，头像偶尔亮起，朋友圈里出现新的教学楼和军训服。陆允川看见了，没有点赞，也没有反复点开。这并不冷酷，只是生活真实的样子。

他把南湖凉亭边晃动的柳影想了很久，终于发现真正值得留下的不是疼本身，而是疼旁边的动作：收拾书、喝水、回消息、等车。

喷泉落水声、迎新帐篷拆架子的声音和远处社团招新的喇叭声让这一切继续往前。哪怕他站在原地，世界也不会陪他站太久。

那天晚上他在日记本上写：今天没有那么想她。写完又觉得这句话太像宣誓，便改成：今天风从湖上来。

夜里他打开日记本，写下省城南湖四个字。写完后没有继续抒情，只补了一句：今天仍然要睡觉，明天仍然要起床。

陆允川后来愿意留下这一笔，是因为它不漂亮。新校园卡不漂亮，行李箱轮子也不漂亮，可它们让人相信，这段青春不是从空中落下来的。

他没有再给夏知遥安排额外台词。她已经在南湖边完成了自己的动作，剩下的沉默，应当由陆允川自己消化。

程砚把话说得粗，叶澄把话说得稳，两种声音都把他往现实里拉。陆允川当时不领情，很多年后却知道自己被救过几次。

那天的食堂窗口第一顿热汤面没有特别好吃，却因为太具体，抵住了那些过大的形容词。人有时候正是靠这些具体的味道，才没有被回忆吞没。

迎新帐篷在湖边收架子，新生拖着行李找宿舍。这些声音把省城南湖从一个人的心事里拉出来，变成许多人共同挤过的一天。陆允川不再只听见自己胸口的动静，也听见椅子、脚步、塑料袋和远处的喊声。

夏知遥的朋友圈出现了另一座学校的军训服。这个动作很小，却让她从陆允川的想象里退回到真实的人：她会忙，会累，会被家里和未来推着走，也会在必要的时候保持沉默。

程砚在消息里问他省城怎么样，他回湖挺大，说话还是老样子，不好听，却有效。陆允川皱了皱眉，还是照着他说的做了一点，因为朋友有时不是负责安慰，而是负责把他从自我感动里拽出来。

叶澄问他适不适应，他第一次回得不那么勉强。她没有把话说得太满，只把边界摆到桌面上。陆允川那时不一定立刻懂，却会在后来某个晚上想起这句话，知道自己当年被提醒过。

母亲打电话问他被子够不够厚，他站在湖边说够。家里的声音一进来，青春就不再只是校园里的风。它变成饭点、车费、睡眠、行李和一句不太会表达的关心。

他低头看着南湖柳影，忽然意识到它不是命运给他的标记，只是那天确实存在的东西。正因为确实存在，它比任何夸张的形容都更能留下来。

湖水、热汤面和新宿舍木柜子的味道混在一起，让记忆有了可以落脚的地方。陆允川后来写到这里，没有再急着解释自己多难过，而是先把气味、光线和手上的动作写清楚。

他把新校园卡收好，又拿出来检查了一遍。这个来回不像英雄，也不像受害者，只像一个年轻人不知道怎样安放自己，于是先安放手边的东西。

过了那一阵，大学南湖凉亭又恢复成普通地方。陆允川却还停在原处，像一个人走出考场以后，耳边仍有铃声。

他试着复盘每一句话，后来发现复盘没有尽头。真正能做的，是把新校园卡放回该放的位置，再去面对下一件事。

这一笔补进去以后，章节没有变得更响，反而更稳。省城南湖不再只是一个标题，而是有南湖柳影、有新校园卡、有人说错话也有人及时停住的真实一天。

湖边的长椅有些潮，陆允川坐下时裤脚沾了一点水。他没有立刻站起来，只把行李箱拉近，听喷泉落回湖面。

新室友在远处喊他去吃饭，他答应了一声。那一声答应很普通，却像把自己从旧日子里往外牵了一步。

晚上回宿舍，他把校园卡放到枕头边，又觉得太幼稚，塞进钱包。手机屏幕亮了一下，不是夏知遥，他也没有失望太久。

南湖边的光线一点点变暗，行李箱和热汤面都还在原处。陆允川把新校园卡收好，动作慢得像怕惊动什么。可真正需要被惊动的不是别人，是他自己。

他后来重写这一章时，特意把南湖柳影留在中间。不是为了让它变成象征，而是因为人只有摸到具体东西，才知道自己曾经怎样站在具体生活里。

事后最难的不是疼，而是不把疼继续喂大。陆允川试着把注意力放回新校园卡，放回吃饭和睡觉，放回一个人必须自己负责的明天。

回宿舍的路上，他经过社团招新摊位，有人递来传单。他接了一张，没有马上扔。那张纸后来夹在书里，像省城递给他的第一件小事。

多年后再看，这种笨拙比豪言更可信。因为一个人真正从青春里走出来，常常不是靠一句漂亮总结，而是靠第二天照常把行李箱轮子放回原处。
