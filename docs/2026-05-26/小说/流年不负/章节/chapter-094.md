# 第094章 有人追她

时间：2014年6月初

地点：支教社团聚餐

卷次：卷五：各自南北

六月初。支教社团出发前最后一次聚餐。学校后门一家东北菜馆。老板是东北人——菜分量很大——锅包肉摞得像小山——金黄外壳裹着肉片，淋着糖醋汁。地三鲜汪了一层油——茄子和土豆发亮。三桌人——十来个人。男生喝了四扎啤酒——啤酒装在塑料扎壶里——黄液体上浮一层白沫。

店里空调坏了——老板搬了两台落地扇对着吹——扇叶呜呜转。有人脱了外套搭椅背。

顾临风坐在角落。啤酒杯在手里转了十几圈——没怎么喝。玻璃杯凝了一层水珠。许若琳坐对面——正夹一块锅包肉——咬下去嘎嘣脆——筷子放下端起啤酒喝了一口。

旁边一个第二分队的女生——姓陈——随口问了一句。声音不大——但店里天花板低——声音聚拢："若琳——你们宿舍那个——叫苏什么的——是不是有人在追她。"

顾临风手里的筷子停了一下。夹的一颗花生米停在盘子和嘴中间。

许若琳一边吃肉一边说。"你说晚晴？""对对——苏晚晴。""嗯——有。师大大二的一个男生——姓周——追了快一个学期了。""成了吗。""不知道——她没怎么提。""长得帅不帅。""我没见过——听说条件挺好——学体育的——打篮球的——个子应该挺高的。"

旁边女生"哦"了一声——拿起一串烤羊肉——开始聊别的话题——支教点有没有蛇、能不能洗澡、要带花露水——"六神的好用"。

顾临风还是低着头。筷子夹那颗花生米——夹很久没夹起来。花生米掉在盘子外面——在白色塑料桌布上滚了一小段。捡起来放骨碟里——骨碟里有鸡骨头和一团用过的纸巾。又夹一颗——这次夹住了——放进嘴里——嚼——盐炒的——有点咸。

喝了半口啤酒——温的——泡没了——液面平静像隔夜茶。杯底在桌布上印了个湿圆圈。有人叫老板加拔丝地瓜——老板说了"好嘞——"。

然后他去了一趟洗手间。

洗手间日光灯很白。瓷砖缝里有黄色污渍。不锈钢洗手池——水龙头有点漏水——一滴一滴打在陶瓷面上——声音在封闭空间里被放大——滴——嗒。滴——嗒。排气扇嗡嗡转。他洗了手——凉水——动作很慢——冲湿、按洗手液、搓手背、搓指缝、冲干净。手指在凉水里冲了很久——冲到手背发红。抬头看镜子里的自己——脸色没什么变化——但嘴闭得很紧——下唇比平时用多些力。关水龙头。抽两张褐色再生擦手纸——粗糙——有碎屑——擦干后纸粘手心里。丢进垃圾桶——桶满了——纸巾堆成小山。

走出洗手间。回到桌上——许若琳正在讲支教要带什么——"充电宝——那边不一定有插座——手电筒——湿巾——因为可能没水洗脸。"有人问山区有没有信号——另一个人说"移动的有一个——联通的没有"。许若琳忽然转过头——"临风——你手机什么卡。""移动。""那没事——有信号。""嗯。"

他又坐了一会儿——看着桌上锅包肉、地三鲜、花生米、拍黄瓜、拔丝地瓜——拔丝地瓜糖浆拉出很长的丝——有人站起来夹——丝不断——旁边人帮忙用筷子剪断。每个人都在说话——笑声——碰杯声。老板在厨房喊"锅包肉两份——"——炒锅翻菜声。但那些声音像隔了一层水——模模糊糊的。

十点多散席。大家往外走——有人在门口互相加QQ——"你支教回来把照片发我"。许若琳她们往女生宿舍走——笑声在夜风里飘来。顾临风一个人往男生宿舍走——影子在路灯下时短时长。回512——阿坤在打副本——键盘啪啪响——屏幕技能特效红红绿绿闪着。李博看书——台灯亮着。小伍写信——熊猫信封——钢笔沙沙声。

顾临风脱鞋——鞋带解了两次——死结——蹲下慢慢解。坐床上——手臂枕在头下。天花板裂缝还在。看了很久——大概两个小时。没睡着。

有人追苏晚晴。姓周。大二。学体育。打篮球。追了快一个学期。

也就是说——从他上大一下的第一天起——那个男生已经在追她了。他在操场走圈的时候——那个人可能在师大的操场上打篮球。他在图书馆写诗的时候——那个人可能在师大的教室走廊里等她下课。他查301路公交路线的时候——那个人可能骑电动车从师大南门骑到北门。那个人可以站在她面前说话——可以看到她笑——可以帮她拿东西——可以在她膝盖摔破时递创可贴。他没有生气的资格——没有难过的资格——是他说遵守约定的——是他一次都没有联系。

翻身——把被子蒙住头——被子里一股洗衣粉味——很暗——什么都看不到。闭上眼睛——脑子里是她被追的样子——模糊的——师大的梧桐树下——两个人并肩走——女生白色外套——马尾——男生比她高一个头——看不清脸——只有两个轮廓在绿色梧桐叶下面移动。

不知道什么时候睡着的。

第二天早上。闹钟响了——默认的嘀嘀嘀。按掉——手指滑了两下才按准。洗脸——凉水冲脸上——眼睛涩。刷牙——牙龈出了点血——泡沫里淡红色。去食堂——包子和豆浆——韭菜鸡蛋馅——咬了一口。去教室——坐第三排靠窗——窗外有新生在军训——喊口号声嗡嗡传进来——跟去年一样。听课——记笔记——"现代汉语——动词的体"。写完在页码旁边画了个圈。跟每天一样。什么都没说。

但是课间的时候，他在笔记本的空白处写了一行字——他不知道自己为什么会写这个。

"学体育的。打篮球的。姓周。"

写完以后看着那行字。笔尖在纸上停了一会儿。然后画了一条横线——把整行字划掉。蓝墨水把那几个字盖住了。他把笔帽盖上。揣进笔袋。然后翻开下一页。继续听课。老师讲到了动词的完成体——"了"字放在动词后面表示动作完成。他在本子上写了一个例句——"她拒绝了那个人。"

写完以后又划掉了。这次划得更用力——纸被圆珠笔尖划出了一道细细的口子。翻到下一页。
