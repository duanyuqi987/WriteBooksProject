# 第190章 小鱼挂件

时间：2022年8月
地点：楚州
卷次：卷十：不负少年

八月。暑假。天很热。空调外机嗡嗡转。客厅里凉凉的。

女儿一岁三个月。会走了。走得不太稳——步子一小步一小步地往前蹭，手臂张开像在走平衡木。每次快倒的时候就弯腰，手撑地，然后爬起来继续走。

下午。苏晚晴在整理旧物。

一个纸箱子。从父母家搬过来的，放在阳台上两年了。纸箱的胶带已经泛黄，上面用黑色记号笔写着"晚晴高中物品"——是她妈妈的字。

她在客厅地板上拆开纸箱。

女儿在旁边看。然后伸手进去抓。抓出了一个铁盒子。饼干盒——蓝白条纹的，盖子有一点瘪，表面生了几点褐锈。女儿把铁盒子抱在怀里。摇了摇。里面叮叮当当响。

"打开。"

苏晚晴帮她打开。

铁盒子里的东西。一个高中时的发绳——黑色弹力的，弹力已经没了，一扯就断。女儿扯了一下。断了。她拿着断了的发绳看了看。扔了。

一张成绩单——高一下学期期中。纸已经泛黄，折痕处发白。上面印着:苏晚晴，高二三班，全班第八。英文旁边有一行铅笔字——"晚晴你这回物理又退步了。加油。"是她妈妈写的。她看了两秒。然后放在一边。

一支黑色中性笔。笔帽上有手指的痕迹——常年握着磨出的凹痕。不是她的笔。

是顾临风的。

她拿起那支笔。旋开笔帽。笔尖是干的。笔芯底部有一小截干涸的墨油。她旋上笔帽。把笔放在手里捏了捏。

然后女儿从铁盒子里又抓出了一个东西。

很小的。黄豆大小。陶瓷的。形状是一条鱼——尾巴歪着，眼睛是点的一个黑点。小鱼的背上有几个细微的裂纹。时间太久了——陶瓷表面的釉已经褪了，露出了底下略微粗糙的胎。

苏晚晴接过来。放在手心里。

低头看着它。看了很久。小鱼躺在她手心的纹路里——生命线和智慧线之间。黄豆大小。刚刚好能填满生命线分叉的那个小缺口。

女儿伸手去够。踮着小脚。手指伸得直直的。她说"别动"。声音比平时轻。比"吃饭了"轻。比"天冷了加衣服"轻。轻到女儿听懂了——把手缩回去了。缩回去之后站在旁边。看着她妈妈手心里那个小东西。手垂在身体两侧。不动了。

这是她奶奶在她六岁那年给她的小鱼。陶瓷的。巴掌大小——不，只有黄豆大小。奶奶的手很大。指节突出。青筋从手背爬到手腕。小鱼放在那样的手心里——更小了。在青阳老家的巷子口地摊上买的。三毛钱——她记得。奶奶从蓝布口袋里摸出三张皱巴巴的一毛钱。递给摊主。摊主是个老头——胡子白的。把小鱼放在她手心里。奶奶说"挂着，保平安"。红绳串着。她串在书包上。从小学串到初中——小学书包是红色的，初中书包是蓝色的。从初中串到高中——高中书包是浅蓝色的。跟那件T恤一个颜色。一直串到大一那年——九月三号。在火车上。火车从青阳开到楚州。她低头一看——红绳断了。小鱼掉在了座位下面。她趴下去捡——手指摸到了。小鱼的尾巴硌了一下指腹。她就把小鱼收进了这个铁盒子里。从此再也没有挂过。

二十多年了。

铁盒子在。边角生锈了。小鱼在。裂纹多了两道。奶奶不在了。

她把手掌合上。小鱼在手心里——硌着。温的。被掌心的温度捂暖了。她把小鱼握在手心里。握了一会儿。然后又摊开手。小鱼的尾巴在掌心上轻轻翻了一下。

"顾临风。"

他正在阳台上收衣服。听见她叫。走进客厅。手里还提着一件她的衬衫。

她把小鱼托在掌心上给他看。他低眼看了一下。认出来了。

"你书包上那个。"

"你怎么知道。"

"我记得。"

他把衬衫挂在椅背上。走近看。手指碰了一下小鱼的尾巴——很轻，像怕碰碎。裂纹比当年多了一道。以前只有两道。

"我以为你丢了。"

"没有。红绳断了。就收起来了。"

他又看了一眼小鱼。然后看着她。

"你奶奶给的。"

"嗯。"

"你说过大一那年红绳断了。然后你收起来了。就没再挂过。"

"嗯。"

"有一次在月湖边——你从书包上拿下来。放在手心里。看了一眼。又放回去了。我在后面看到的。"

"你连这个都看到了。"

"后来——工作后有一次你跟我路过一个饰品店。在新华书店旁边。你停下来看了好一会儿。橱窗里有一排红绳。不同粗细的。你看了大概两分钟。我没问你在看什么。"

她低头看着手掌里的小鱼。小鱼也在看她——那个黑点的眼睛。点上去的。但好像在看她。

"在看红绳。想买一根。又觉得没必要。反正已经不挂了。走了。"

"今天去。"

他站起来。从衣架上扯下自己的T恤——刚才叠了一半的。套上。然后拉她的手。

"现在去。"

她站起来。把女儿交给隔壁邻居阿姨——阿姨在走廊里择菜。说"一个钟点就回"。

两人走了一站路。路边的小饰品店。店面不大。墙上挂满了发卡、耳环、手链。收银台旁边一个旋转货架上挂满了各种颜色的绳子。红的、黑的、蓝的、黄的、彩色的。

他拿起一卷红色。问"有没有细的"。店员从下面抽屉里又拿了一种。细的。中国结的红。

"这个。"

"给我两根。"

"两根？"

"对。两根。"

他付了钱。五块钱两根。

回到家。他从抽屉里翻出剪刀。把红绳剪了一小截——长约十五厘米。然后他把小鱼拿过来。穿过鱼身上的小孔——孔很小，他眯着眼睛，穿了三次才穿进去。然后把红绳的两头对齐。打了个死结。又打了一个。

他把串好的小鱼放在手心里。看了一眼墙上的钟。

"好了。给你妈妈。"

他叫女儿过来。女儿正蹲在地上玩。听到爸爸叫，跌跌撞撞地走过来。他蹲下来。把红绳小鱼举到她面前。

"这是你妈妈小时候奶奶给她的。现在给你。"

女儿伸手抓。他躲开。"不能抓。挂起来。"他帮她挂在小书包的拉链上——那个小书包是蓝色的，帆布材质。小鱼挂在拉链头上，垂下来。黄豆大小。

跟他当年在教室后排看到的那个小鱼。一模一样的位置。

书包挂在鞋柜的挂钩上。女儿站在书包前面。看着那只小鱼。

小鱼微微晃了一下。

苏晚晴站在厨房门口。手上的洗菜水还没擦。看着那个小鱼在女儿的书包上慢慢地转。红绳。陶瓷。裂纹。还有那个黑点——鱼的眼睛。

她想起奶奶的手。粗糙的。指节很大。那条巷子。老槐树下。奶奶蹲下来，把小鱼挂在她书包上:"挂着，保平安。"

她想起高一那年。走进教室。书包上的小鱼晃来晃去。她不知道后排有个男生在看着这只小鱼。看了三年。

她想起那只小鱼在书包上蹭到课桌时发出的声音——很轻。像米粒掉在木头上。

现在这个声音在女儿的背包上传了出来。一样的轻。

她把湿手在围裙上擦了一下。围裙上湿了两道。然后走过去。蹲下来。从女儿的书包上把小鱼拿起来——放在手心里。用手指摸了一下小鱼的背。那道裂纹——新的。那道旧裂纹——奶奶当年系的时候就有了。还有一个黑点——鱼的眼睛。她摸了很久。指腹在陶瓷的粗胎质上感觉不到什么。但她知道它在那里。

然后她把小鱼挂回女儿的书包上。站起来。手在女儿的头上轻轻拍了一下。女儿的头发很软。发丝在手心里滑过去——像水。很像。

"好看。"

女儿仰头看着她。笑了。嘴张着。上下四颗牙——两个门牙之间有条缝。

晚上。女儿睡了。睡姿很难看——一张小床她滚了三百六十度。头朝东变成了头朝西。被子踢到了床脚。

苏晚晴坐在客厅里。手里还攥着另一根红绳。他多买的那根。茶几上的台灯光从侧面打过来——红绳在手心里反着丝光。她把红绳放在手腕上比了比。取了。放在茶几上。又拿起来。比了比。取了。第三次——她系上了。

她把红绳系在自己手腕上。打了个活结。试了试松紧——不勒不松。手腕内侧能看到静脉血管——青色的。红绳在青色的血管上面——像一条很细的线。她转了转手腕。红绳跟着转。她对着台灯看了看。然后放下袖子。遮住了。

第二天早晨。顾临风在餐桌上看到她的手腕上多了一根红绳。没有小鱼。只是一根红绳。

她没有刻意露。给他盛粥的时候袖口往上滑了一点——红绳从袖口滑出来。然后又滑回去了。

他没有问。她把粥端过来。放在他面前。碗底磕在桌上的时候发出闷闷的声音——陶瓷碰木头。然后她在他对面坐下。拿起自己的筷子。开始吃。

他低头喝粥。喝了两口。抬起头。又低头喝粥。然后又抬起头。这次看着她的左手腕。袖口遮住了。但能隐约看到皮肤上面有一个细细的红线——不是绳子。是隔着袖口透出来的那一点点颜色。

好看的红色。

"好看。"

她嗯了一声。给他夹了一筷子咸菜。手腕上的红绳从袖口里滑出来一点——露出一个结。那个结系得不太好看。因为是她自己系的——单手系活结总会有点松垮。歪的。但并不影响它是一个结。
