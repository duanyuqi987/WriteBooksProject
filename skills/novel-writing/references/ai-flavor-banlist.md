# 去 AI 味禁用词与排版红线

融合第三方精华：三遍校对法 + 6 大类 AI 检测体系。

## 第一遍：内容检查（逻辑/事实/结构）

- [ ] 事实准确？（日期、地名、角色名、前文事件不矛盾）
- [ ] 逻辑清晰？（没有前后矛盾）
- [ ] 结构合理？（不跑题、不注水）
- [ ] 信息完整？（本章任务说明中的要点全覆盖）
- [ ] 没有编造？（所有数据和例子都真实存在于小说世界中）

## 第二遍：风格检查（6 大类 AI 腔检测）

### 1. 套话与口水词

```
在今日的、在这个、众所周知、值得注意的是、可想而知
不得不说、总而言之、综上所述、从某种意义上说
```

修复：直接切到要点，不绕弯。

### 2. AI 句式

```
不仅……而且……（出现 3 次以上）
一方面……另一方面……
不是……而是……（过度使用成对连词）
```

修复：拆成短句，用多样表达。

### 3. 正式词汇（产生距离感）

```
显著提升、大幅增强 → 用具体数字
充分利用 → 用好、用上
进行操作 → 直接用动词（点击、输入、放下）
```

修复：口语化、具体化。写小说不是写公文。

### 4. 机械结构

```
过度使用"首先、其次、最后"
每 200 字就来一个小标题
列表强迫症（什么都要列个一二三）
"以下几个方面""以下几点"
```

修复：自然叙述，只在确实需要时用列表。

### 5. 中立态度（没有立场——小说最致命的 AI 味）

```
总是两端说："既有好的一面，也有不好的一面"
不敢判断："取决于具体情况""这也是一种可能"
每句话都加保险："也许""或许""在某种程度上"
```

修复：明确态度，敢下判断，敢于让角色有鲜明的立场。

### 6. 缺乏细节（抽象不落地）

```
大词："很多""一些""显著""大量"（没有具体数字）
缺失真实感受和具体经验
没有时间线、没有场景、没有身体感知
```

修复：加入具体细节、具体数字、可感知的身体经验。

## 情绪/心理禁用词

不禁、缓缓、微微、淡淡、轻轻、默默、静静、深深、渐渐

## 连接/过渡禁用词

然而、此外、与此同时、不仅如此、另一方面、总而言之

## 解释性禁用短语

这意味着、也就是说、换句话说、显然、众所周知、很简单

## 空洞修饰禁用词

仿佛、宛如、好似、似乎、犹如、如同

## AI 式总结/抒情禁用句

- "命运的齿轮开始转动"
- "真正的考验才刚刚开始"
- "他不知道的是……"
- "而这一切，只是开始"
- "从此，他的人生彻底改变"
- "多年以后回想起来……"

## 排版红线

- 禁止碎片化分行（全文碎片化分行 >5% → 重写）
- 禁止省略号泛滥（一段 >2 个 → 小修）
- 禁止连续 3 个以上单词语行 → 合并为正常段落
- 禁止连续 3 个以上相同句式
- 禁止连续 3 段相近段落长度

## 章末红线

- 禁止以抒情/总结/感慨/预告收尾
- 禁止连续两章使用相同结尾模式
- 必须用动作、发现、选择或危机形成钩子
- 禁止出现"开始了""结束了""转动了""新的"等空洞词

## 积极规则

优先使用：
- 具体动作代替情绪标签
- 可见细节代替抽象气氛
- 潜台词代替解释性对话
- 选择和代价代替单纯事件经过
- 短句、白描、留白

## 第三遍：细节打磨（标点/排版/节奏）

- [ ] 句长合适？（主体 15-25 字，最长不超过 40 字）
- [ ] 段落不太长？（手机上 3-5 行，阅读器上 100-150 字一段）
- [ ] 标点自然？（多句号，少逗号连长句）
- [ ] 节奏变化了？（快慢结合，不单调）
- [ ] 加粗适量？（1-2 处/200-300 字）

### 第三遍方法

1. 朗读感受节奏（AI 写的文章读起来一定有不顺的地方）
2. 找超过 30 字的句子，拆分
3. 检查段落长度，拆分过长段落
4. 调整排版，去掉过度格式化
