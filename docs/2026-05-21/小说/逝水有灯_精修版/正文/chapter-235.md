# 第235章 诗歌索引多年后

所属部卷：第五部：逝水有灯
时间：多年后
地点：陆允川书房

## 诗歌插页 P235：原创仿唐绝句

校徽微明照旧尘，
少年回首已非春。
风过水边无问处，
一灯还照后来人。

多年后陆允川重写诗歌索引序言的时候，在文件最底部发现了一行被隐藏的批注。

是PDF注释功能里的。程砚在某个深夜用电子笔写的："这些诗我每一首都能背。不是因为我记忆力好——是因为我见证了它们被写出来。我看见过你凌晨三点改最后一句的样子。你问过我'还是不对吧'六十七次。每次我都说对。但其实有一半的时间我也听不出来对不对。我只是知道你会继续改下去。"

陆允川对着屏幕笑了很久。然后把这段批注截图发给程砚："你什么时候写的？"

程砚回："忘了。可能半年前拿你的PDF看的时候随手划了一笔。"

"为什么不用笔直接告诉我？"

"电子批注也是一种笔。新时代。你要与时俱进。"

"我谢谢你。"

"不客气。"

他把那行批注保留在终稿的PDF里。不是正文。是注释层。只有打开PDF的高级模式才能看到。这是一个隐秘的彩蛋——只有少数几个会点开高级模式的人会发现。

书的最后一件事，是整理扉页。

他写了好几版扉页的献词。

第一版："献给我的大学，以及大学里遇见的每一个人。"

删了。太像获奖感言。

第二版："献给那些在我的青春里路过的人——无论是过客还是留下者。"

也删了。太像墓志铭。

第三版："这本书是为你们写的。我不需要说出你们的名字。"

想了很久。然后也删了。因为他觉得说"为你们写的"还是太重——像是给一份很贵的礼物，对方不得不表现出开心。

最后一版只有一行："这里有我所有的灯光。谢谢你和你们。"

不再改了。

扉页下面的第二页，是诗歌索引的总目录。四十五首，三体，四十五行。一行不多，一行不少。他检查了最后一遍——每一首的日期、意象、诗体——然后点了"打印"。打印机吸进去第一张纸的时候，停顿了一下。像一声深呼吸。然后一页一页地吐出这本书的附录。

把打印好的索引钉在书稿最后一章后面的时候，已经是傍晚了。他把书稿整理好，平铺在书桌上。窗外有鸽哨——不是鸽子，是隔壁顶楼在修空调，安装师傅吹哨子叫同伴。但听起来确实像鸽哨。

他翻到诗的最后一页——第四十五首，"灯慢慢亮了"。

全诗只有六行。但他写了十年。第一版是大二时，在诗社集会后写的。当时题目是"灯还没有灭"，最后一句是"所以她还能看见"。那是写给一个人看的。

第四版，最终版。题目改成"灯慢慢亮了"。最后一句改成"人也终于学会往前行"——不，他又改了一次。"终于"删掉。"人也学会往前行"。

六行。不多不少。

他把这最后版本的诗发给了所有该发的人。程砚、叶澄、祁野、夏知遥。苏晚枝。没有邮件正文。只有附件。

然后关了电脑。站起来，走到厨房。打开水龙头，接了一杯水。水龙头的声音在安静的傍晚里干净清脆。

喝完水，杯子放回沥水架上。窗外安装师傅又吹了一次哨。这次是真的鸽哨——隔壁楼顶不知谁养的一群鸽子刚好散了盘。翅膀声呼啦一下，又远又近。

他把书稿装进一个牛皮纸文件袋。封面写：《逝水有灯》终稿。下面一行小字日期。然后放在书架上——和牛皮日记本、毕业照、出租屋照片并排。

四个物件。横跨十年。书架这一个格子终于填满了。
