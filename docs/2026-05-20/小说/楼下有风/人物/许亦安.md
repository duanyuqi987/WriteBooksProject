# 许亦安

## 角色张力

- 来处：普通大学校园里的年轻人，生活由宿舍、教学楼、网吧、小吃街和手机消息组成。
- 当下：敏感、执拗的主角，用日记本保存自己对林青禾的喜欢。
- 欲望：被确认、被需要、被温柔对待
- 恐惧：自己在别人生活里可有可无
- 内在矛盾：想靠近别人，又不知道怎样不伤害别人。

**一句话灵魂**：一个把等待当作爱的男生，终于学会把爱还给自由。

## 基本信息

- 身份：大学生 / 青年记忆中的关键人物
- 叙事功能：推动许亦安理解喜欢、边界和释怀
- 性格色彩：真实、年轻、不完美

## 声纹

- 句长偏好：短句和自嘲较多，真正受伤时会开玩笑
- 发言长度：中短句为主
- 特色词汇：校园口语、玩笑、回避或直接表达
- 情绪外露：不完全外露，常通过动作表现
- 绝不说的话：过度文学化、像总结报告一样的话
