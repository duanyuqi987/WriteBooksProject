# 第198章 怕这是最后一次

时间：当晚九点多，即将到达封丘

地点：大巴上→封丘汽车站

卷次：第八卷：没有答案的答案

苏安晴睡了大概四十分钟。

四十分钟里林玉起想了很多事情。想省城的公交卡要去哪里办，想研究生宿舍是几人间，想从河这边到河那边的那座桥叫什么名字。想他和苏安晴在省城第一次吃饭会去哪家馆子。想他自己写的那些诗——哪些可以留着继续改，哪些应该直接扔掉。想他爸在封丘工厂里上班的样子——他爸每天六点出门，六点回来，手上永远有一层洗不掉的机油味。想他妈做饭的时候放盐总放多——这四年来每次回家他妈都会问一句"食堂的饭咸不咸"。

然后他想到了苏安晴靠在他肩膀上的这个场景——是不是最后一次。

他不是悲观。他是太清楚生活怎么运转了。生活不是小说——不会因为今晚她靠在你肩膀上睡着了，明天你们就在一起了。生活是具体的。开学日期不一样，课表不一样，报到的地方不一样。从省城的师范大学到她的综合大学是九站路——不远，但如果不专门走，也可以永远不走。生活是一天一天堆积起来的，而每一天都有足够多的理由让两个人越走越远。

他怕这是最后一次。怕以后想见她需要打电话约时间。怕以后没有操场上随便一圈就碰到的自由。怕以后她的东西放在她的宿舍，他的东西放在他的宿舍，没有走廊把他们连起来，没有水房让他们偶遇。

怕毕业以后他们的距离不会变成零，只会从走廊的两头变成城市的两头。

苏安晴醒了。

不是被颠醒的——是她睡够了，自然地睁开眼睛。她醒了大概两秒钟以后才意识到自己靠在哪里，然后很平稳地把头从他的肩膀上抬起来，坐直了身体。她没有慌张，没有后退，没有说"不好意思"。

她只是睁开眼睛，直起身子，然后说："我睡了多久？"

"四十多分钟。"

"你的肩膀酸了吧？"

"没有。"林玉起说了假话。他的肩膀酸得发僵，稍微动一下肩膀关节会发出咔的一声。

苏安晴一眼就看出来了。"你连这个也要——"

她没说完。她伸手碰了一下他的肩膀——用食指和中指，轻轻地，隔着T恤的布料。像是在碰一下确认他没有骨折。然后她把手收回去。

他们两个之间从来不拥抱。不牵手。不触碰。唯一的一次是苏安晴拉了一下他的袖子——往前站一排。今晚她靠在他肩膀上睡了一觉，醒来以后用手指碰了碰他的肩。这大概就是他们全部的身体接触了。

但她碰了一下。轻轻的。够了。

大巴减速了。前面的路牌上写着"封丘"两个字——白底黑字，反光漆在车灯的照耀下亮得刺眼。司机高师傅用带着口音的普通话在前面的喇叭里说："封丘快到了，大家准备一下。"

林玉起把遮阳帘拉开。外面是他熟悉的县城——不高不矮的楼房，大多数是六层。街道两旁排列着招牌大大小小颜色不一的店铺。路灯不多，有的路段很亮，有的一段黑。封丘县城不是大城市，晚上九点多街上已经没什么人了。

汽车站到了。车停下来，司机打开了行李舱。车厢里有几个乘客开始收拾东西。

林玉起站起来，把苏安晴的布袋递给她。她接过去的时候，手指和他的手指碰到了一起——不是刻意碰的，是交接布袋的过程里自然的触碰。她的手指是凉的，大概是刚睡醒之后体温还没完全回上来。

她说："到了。"

他说："到了。"

他们下了车。封丘的夜风迎面扑上来，没有栀子花了，只有汽车尾气和远处烧烤摊飘过来的孜然味。天上的星星比江城多一些——小县城没有那么多灯光污染，能看到北斗七星，勺子柄朝南。

行李舱里两个人的箱子还挨在一起。林玉起把自己的箱子拖出来，再把苏安晴的箱子也拖出来。她把拉杆接过来的时候说："我自己来吧。"他说"好"。她接过去，两个人的手指擦了一下。这次不是"交接布袋的过程里自然的触碰"——这次谁都注意到了，谁都当没注意到。

站在汽车站门口的广场上，看着各自要去的方向。林玉起的家在城西，苏安晴的家在城东。两个方向。

"明天——"林玉起说。

"明天再说吧。"苏安晴说，"先回家。"

"好。"

苏安晴拖着箱子往城东方向走了。走了几步，回过头来。汽车站门口的那盏灯照着她，光不够亮，看不清她的表情，但她的声音能听清。

"肩膀回去拿热毛巾敷一下。"

"知道了。"

她走了，背影被深蓝色行李箱遮掉了一半，然后在黑暗里越变越小。

林玉起站在汽车站门口，看着那个方向看了很久。他想了很多事——四年前他一个人从封丘出发去江城，不知道自己会遇见一个什么样的世界；四年后他回来了，知道自己遇见了什么——一个女孩，从大一到大四。但更多的他还是不知道。

他拖着旧箱子往城西方向走。箱子的轮子在地上骨碌碌地滚，在空旷的街道上特别响。
