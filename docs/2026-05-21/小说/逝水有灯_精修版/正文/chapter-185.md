# 第185章 道歉未发送多年后

所属部卷：第四部：失去的美丽
时间：多年后
地点：公寓 / 云端

## 诗歌插页 P185：原创仿唐绝句

微光微明照晚尘，
少年旧梦隔河春。
回看风声无定处，
一灯还与后来人。

那个2021年的文档一直存在陆允川的旧电脑里，跟其他大学时期的资料一起备份到了云端。

多年后的一次清理，他无意间点开了它。那是一个落着灰的旧文件夹，里面混着本科论文、求职简历、早期租房合同——全是过往生活留下的数码层积。文件名：2021.12.15.docx。打开第一行还是"苏晚枝："。

窗外的城市在黄昏中慢慢亮起灯光，远处的高架桥上车辆往来不息，车灯拉成一条一条的金色虚线。书房里的台灯把旧电脑的屏幕衬得格外白。

他往下翻。信尾的落款是空着的——本来应该写名字的地方，光标后面什么都没有。当时他便没有写自己的名字，就像连姓名都不忍心多添一句。

他看完正文，想象了几秒如果当时真的发了这封信会发生什么。是苏晚枝会回一封同样长的邮件？还是会直接打电话？或者什么都不回？

但他现在的想象不是浪漫的。他知道答案：她大概率会感到很累，夏知遥会发一条消息说"他又写信了"，程砚会叹气，叶澄会在他发出去之前做最后的挽留。然后自己又后悔。

往下的结局不必多想。感谢当时那个没发——让它自己在那天图书馆四楼的电脑前停住了。

他把文档关了。没有删——但也不准备发给任何人看。

苏晚枝在多年前的朋友圈发过一条她城市的日暮照片，色调是偏青的。他在后面留言发了一个赞。那是近五年来她唯一一条他留言过的内容。没有多余的互动。

不需要。那些年欠她的千丝万缕——已经不必再一笔一笔亲手还到面前。

他在"不再分析的日常"文件夹里新起了一行，加了当晚的最后一条：

"那封道歉信的核心只在三个字——'不发'、'了'。'不发了'。没有撤回。没有补发。"——后面又跟了一行："补在备忘录那段话里：有些成长不需要被见证。被见证的成长容易退化成表演。"

写完他关掉了文件夹。时间已经不早了。

那晚他做了个梦。梦见大学图书馆四楼——他坐在原先那个靠窗的位置。窗外是冬天的枯枝，阳光从西南方向斜斜地照进来，在桌面上切成明暗两半——就像那个十二月的下午一模一样。苏晚枝恰巧在他对面坐下，两人各自看书，偶尔抬头对上一下。然后她收拾东西走了——临走时在他桌角轻叩一下。他低头看——是一个小纸片，上面画了两个圆珠笔小人牵着手，不是恋人的那种，是幼儿园小朋友郊游过马路时手拉手的样子。画的线条歪歪扭扭——不知道是她什么时候画的。纸的发黄程度像放了二十多年。

他早上醒来后把这个梦写进了日记。不是"不再分析的日常"文件夹。是那本旧日记本——最后一页空白处。铅笔。用的是跟苏晚枝一样的力道——很轻。一笔一画。

写完之后他把日记本重新放回书架。旁边还是那本《中国现代诗选》——苏晚枝铅笔写的"好"还在。两种笔迹现在在同一层书架的不同高度上——旧日记本高一些，批注低一些。互相对望。

他关掉书房的灯，回到卧室。窗外城市的夜晚很安静。没有什么需要再发。没有什么需要再收回。就这样。

他躺在床上，妻子已经在旁边睡着了，呼吸平稳得像远处的潮水。他没有马上闭眼，而是盯着天花板上透过窗帘的微光——城市的夜空从来不是真正的黑暗，总有路灯、广告牌、邻居家的阳台灯，把黑暗稀释成一种淡淡的灰蓝色。他想起了大学宿舍的天花板——那块漏水痕迹的轮廓。他现在闭上眼睛还能画出那个形状。

那封没有发出的道歉信还在硬盘里。三十年后它还会在——只要他不删，它就会一直存在于某个云端服务器的某个扇区里，跟几亿封其他人的邮件和照片挤在一起。一封从来没有送达的信，在数据层面跟一封已经被读过千万遍的信没有区别——都是0和1。但它没有被读。这是它唯一特殊的地方。

他翻了个身。妻子动了一下，把被子往他那边推了推。这个动作是无意识的——她在梦里也知道他盖得不够。他把被子拉上来盖住肩膀，然后闭上了眼睛。

明天是周一。他要去公司开早会，要给客户改方案，要记得在午休的时候给程砚回消息——程砚昨天晚上发了一段话，说他去了一趟大学，南湖凉亭的浅绿漆已经开始褪色了，最左边那根柱子木纹还是直的。"你的书比那层漆持久。"程砚说。

他当时没有回复。现在在黑暗里他想好了回什么：持久不是书的功能。是那盏灯还在亮。

然后他睡着了。那封没有发送的信安静地躺在文件夹深处，像一个被折起来的小纸船——它永远不会漂到收信人手里，但它也不会沉。
