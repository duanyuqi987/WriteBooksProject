# 第134章 她和别人说笑旁观

所属部卷：第三部：寒风里的花
时间：六月初
地点：校园超市门口

## 诗歌插页 P134：原创现代诗

我把汽水瓶放在黄昏里，
不让它替任何人回答。
风从校园的一侧经过，
带走一句没有说稳的话。
后来我学会把手松开，
让灯照着路，也照着空出来的地方。

何屿白觉得苏晚枝是自己见过的学妹里最奇怪的一个。他认识她是在建筑初步课上——她坐在第一排靠窗的位置，笔记本摊开，上面的笔记不是纯文字，而是一半文字一半小图。文字记录老师讲的要点，小图画的是教室窗户光线的角度变化。她告诉他这叫"双轨记录法"——左脑记概念，右脑记空间。

她可以用三种不同的语言说话：对建筑系的同学用专业术语——"采光井""日照分析""风压系数"，句子短得像施工指令；对诗社的人用评论——"这句节奏不对""意象过曝了""少写一个字更好"，像在批改一首已经投了三次稿的诗；对超市门口的冰柜用——"柠檬只剩一瓶了，留给明天"，语气跟预订建筑材料一样郑重。

他对苏晚枝没有任何超出学长的关心。他有女朋友，异地，在成都读材料学，每周六晚上视频一次——视频内容已经从刚开始的"今天干了什么"演变成了"这周考试考了多少分"和"毕业以后你想去哪个城市"。他们在一起三年，感情稳定得像混凝土养护到期的试块。所以他帮苏晚枝改透视图纯粹是出于学长的责任感——她的线条很干净，用针管笔勾的，但空间比例偶尔会偏（楼间距总是画得比实际大一点），他帮她用硫酸纸重描一次剖面线。硫酸纸盖在图纸上，用铅笔重新确认一遍退距。

那天买汽水是因为太热了。六月下午两点从建筑系馆出来，太阳隔着衣服能把肩膀晒疼。他先拿了一瓶青苹果味的，问她喝什么，她说柠檬。他拧开瓶盖递过去，她接过去喝了一口。

"你每次帮人改图都请汽水？"

"不是。今天是太热了。而且帮我刷过油漆的学弟我也只请他喝矿泉水。汽水是多出来的预算，一般是考完试才喝的。"

"那今天为什么——"

"因为你刚才交上来的板图得了优。马老师说你用马克笔表达材质的方式比往届都好。就当是提前庆祝。"

苏晚枝笑了。眼角往下弯，下巴微微缩。何屿白觉得这个笑是由衷的，但不是给他的——是因为板图得了优。他的恭喜只是一个载体。

后来何屿白在建筑系走廊里远远见过她和那个诗社的男生走在一起——男生手里拿着卷着的图纸，和她并排走，步速一致。苏晚枝指了一个什么东西给他看，他凑近看。何屿白没有上前打扰。但他认出了那个男生——就是后来在超市门口遇到诗社活动时，站在苏晚枝旁边、手里握着瓶盖门帘的那个人。

何屿白低头看了看自己手里那瓶青苹果汽水的瓶盖。他本来想扔进垃圾桶的——垃圾桶在冰柜旁边。但苏晚枝跟他说"瓶盖给我——诗社需要废品"。他不明白诗社为什么会需要瓶盖，但没多问就给了。

几周后，他在301教室看到了那串门帘。三种颜色的瓶盖串在绳子上，挂在门框上，被日光灯一照——柠檬色的盖子透光最多，变成浅黄；橙色的透光少，变成暗橘；青苹果的最薄，光透过时泛一层淡淡的绿。三种颜色混在一起，把那道冷白光切成了细碎的暖片。

他站在门帘前面，数了数：柠檬的有四个，橙子的有三个，青苹果的只有一个。他想——青苹果是自己送出去的那一个。被串在了中间偏左的位置。和其他盖子之间隔了大概三厘米，刚好是两根指头的宽度。

"这个青苹果盖子隔得很开。"他回头对苏晚枝说。

"故意的。因为颜色跳——青苹果偏绿，紧挨着柠檬会看不清区别。"

"你真讲究。连瓶盖的颜色距离都要控制。"

"这是我的强迫症。"她说完想了想，又加了一句，"就跟你看剖面线的粗细一样。"

何屿白笑了。

"下次如果有红色的瓶盖——我帮你留着。应该挂在最右边。"

"什么味的？"

"西瓜。只有夏天有。"

"那叫限时色号。"

"对。建筑系的说法：季节性材料。"

两人都笑了。然后何屿白收起笑容说："我觉得你们这个社团——不是在玩。她是认真在做空间。"

"她说空间不是容器——是邀请。"陆允川刚好走进来，接了这句。

何屿白回头。"这句是从建筑初步的教材里来的。"

"对。她把教材上的话贴在了墙上。"陆允川指了指树干上那片新叶子。

何屿白走近看了看那片叶子上的小字。"她把'容器'改成了'邀请'？"

"嗯。原文是'空间不是容器。'她加上了'是邀请'。"

"厉害。"何屿白把这片叶子拍了张照。"我要告诉马老师——他的教材被诗社改装了。"

后来何屿白每周去301教室一次。不是为了诗社的活动——是为了看墙上多了什么新的叶子。那片"空间不是容器。空间是邀请。"是他最喜欢的。因为那是他教材上的句子，被苏晚枝改了两个字，比原来更准。他想：能把一个句子改准而不破坏原意的人——不管在哪一行，都会做得不错。
