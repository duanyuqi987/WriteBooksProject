# 第022章 春天的走廊

时间：2011年4月

地点：教学楼走廊

卷次：卷二：窗边人影

四月的走廊风是暖的。

青阳一中的教学楼是个"回"字形——中间是块天井——种了两排冬青——常年没人修剪——长得高矮不齐——有些枝条伸到了走廊栏杆上面——很绿——叶面上有一层灰。三楼的走廊在东侧——从尽头能看到对面教室的窗户。天井里常有鸟——麻雀——在冬青上跳——有时跳到走廊上——有人走过——它们又飞走。天井地上的砖缝里长着青苔——背阴处——湿漉漉的。

顾临风发现了对面教室能看到这件事。

那天课间他去走廊尽头接水。饮水机在一排窗户前面——靠走廊栏杆——太阳从对面的方向打过来——水桶里的水被照成半透明的淡蓝。他接完水——把杯盖拧上——抬头。

对面——隔着天井——大约二十米——是理科三班的教室。教室的窗开着——从走廊尽头看过去——那扇最左的窗户——正好对准第二排靠窗。

苏晚晴在。她正低头写字——看不清细节——但轮廓是她的——马尾——肩膀。后背挺得笔直——头微侧——右手在动——笔尖从纸上划过。桌上的课本是物理——蓝色封面——从这个距离判断是物理——因为数学课本的封面颜色比物理浅一些。她的左手按着本子——不时松开——翻过去——接着写。

他端着水杯站在饮水机旁——水是早上灌的开水——放了一节课——现在温了——杯子有点烫手。他站在那——没走——看了几秒。对面教室有人从她身后走过去——也是理科三班的人——一个男生——手里拿着篮球——拍了一下——在地上弹一下——那个男生走到后门出去了。

上课铃响了——刺耳的——和上学期一样的铃声——不管春天夏天。她抬起头——往窗外看了一眼——脸朝向他的方向——但中间隔了二十米和天井——她看到的只能是走廊的一排窗户和一个模糊的人影——看清是谁几乎不可能。顾临风往后挪了半步——退到了饮水机侧边——她的目光从窗外收回去——低下头——把桌上的书合上——换了下一节的课——英语。

后来他开始去走廊尽头站着。

课间十分钟。以前他会跟赵一鸣去小卖部——在操场后面那个铁皮棚子——买辣条、买冰棍、买可乐、买五毛钱的干脆面——赵一鸣每次都买两包——一包自己吃——一包塞给他——"你不要跑那么慢——"那时候还是跑操的日子。现在不去了。

第一次——赵一鸣说"走——下课了小卖部——"他说"我肚子不太舒服——不去——"赵一鸣看了他一眼——"行——"自己去了——回来带了包辣条扔他桌上——"要是胃疼就别吃——"他没吃——放在桌洞——下午才吃——辣条已经不脆了。

第二次——赵一鸣又喊——他说"还是不去了——"赵一鸣没说啥——回来的时候带了两根冰棍——橘子味——一人一根——然后他们俩靠着后门墙边——咬了一口——凉得牙齿酸。

第三次——赵一鸣没再喊他——自己去了——然后回来。

他站在走廊尽头——一站就是十分钟。有时候能看到她在教室里——在做题——在看课本——在和同桌许若琳说话——说话的时候侧脸对着走廊——嘴在动——许若琳点了头——她也点头。有时候看不到——她的座位空着——大概是去接水了——或者去办公室交作业——或者上洗手间。空座位的时候他会低头看一下自己手里的杯子——水面平静——杯壁凝的水珠——然后用手指在杯壁上划一道——把水珠刮成一条竖线——然后继续等——等到她回来——坐下——才走——走回教室——正好上课铃响。

有一次她站起来——往窗边走——推开窗——铝合金窗框沿着滑槽——嗤——窗户开了——四月的风从窗口灌进教室——把她桌上的试卷吹翻了——白色卷子翻到地上——她弯腰——伸手按住——卷子上踩了一脚——不知道谁的——捡起来——放在桌上——压上那个陶瓷鱼的铅笔盒——盒底压住卷子角。她对着窗外站了大概几秒——没有看什么特定的方向——就是看着外面——春天的天——淡白——阳光不热——风有点大——把她的刘海吹开了——露出额头。然后她转身回了座位——把卷子反过来——继续做。

他站在对面走廊——看着她对着窗外站的那几秒。她没有看到他——她眼睛里只有窗外——是空的。

他继续站了大概一分钟——直到上课铃响。春风吹过来——天井里的冬青叶子在响——哗——鸟从灌木里飞出来——翅膀扑过——从三楼底掠过。他看到她的马尾被风带起来一两根碎发——翘了一下——然后落回去。

赵一鸣有天中午在食堂问他。两人面对面坐着——赵一鸣吃鸡腿饭——鸡腿夹到他盘里——他吃了——赵一鸣往嘴里塞了口米饭——然后问:"你最近老往走廊那边站。"

"嗯。"

"那边有什么好看的。"

顾临风夹了一块土豆——土豆炖得很烂——筷子夹起来的时候破了——掉到盘子里——"没什么——透透气。"

赵一鸣吃了一口饭——瞄了他一眼——把盘里的鸡腿夹起来——又放到顾临风盘子里——"多吃点——你最近好像瘦了。"他看了看——夹起来——咬了一口——嚼——然后把鸡骨头放在盘子边上——手指沾了油——在纸巾上擦。

赵一鸣嚼着卷心菜——突然停下——"对面——是咱班的教室。"

顾临风没说话——把饭吃了——米饭拿筷子往嘴里送——赵一鸣也没再问——把空盘放在一起——站起来——"走了——"

四月下旬。走廊里的风一天比一天暖——天井的冬青花开了——白色——小小的——一簇簇——不香——淡到闻不见。走廊栏杆上——有蜘蛛网——在栏杆缝隙结的——网丝很细——在阳光下泛着微光——一只小飞虫粘在网上——不动了。

那天课间——他照常去走廊尽头——接水——然后站在那位置。她没在做题——在翻一本课外书——不是课本——封面是蓝绿色的——上面有字——太远——看不清。是那种借阅室的文学书——可能——也可能是她自己买的。她翻着——一页一页——不快——大概看十来分钟——然后合上——放到桌肚里——打开书包——把书包拉链拉一下——再翻桌上的物理练习本。

然后她往窗外看了一眼。

这次他没躲。二十米远——隔着天井——她看不清他的脸——光线从他背后打过去——他在走廊的暗处——她是逆光看的。但她看窗外的那几秒——他站在那——没动——看着她看窗外——她的眼睛是空的——没有确定的焦点——就是人偶尔做的一个很松散的动作——看着外面——不为了什么。她没有笑——也没有皱眉——就是看着窗外。大约五秒钟——五秒。然后低下头——继续做自己的事。

顾临风把杯子里的水喝完了。最后一口水——在嘴里——凉——咽下去——喉结动了。拧上杯盖——把杯子放在栏杆边——手松开——一会儿再拿起来。上课铃响的时候他回教室——最后那一瞥——她在做物理题了——笔在草稿纸上画——头低着——马尾搭在右肩——春日下午四点的光——从窗外斜进来——照着她的后背——毛衣领口——上面有一层细细的毛——被光打得泛着绒白色。他记住了这个画面。

很多年后他还记得——四月——春天——走廊——风——对面窗——窗里的她——低头写着什么——然后抬头看了一下外面——她没看见他。他也没让她看见。但他在看她。

他回到理科三班后门口——握着杯子——回到座位——翻开化学书。赵一鸣在旁边——抬头看了他一眼——嘴角动了动——什么都没说。
