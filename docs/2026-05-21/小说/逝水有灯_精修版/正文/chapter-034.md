# 第034章 雨天值日旁观

所属部卷：第一部：初秋三楼
时间：高一下学期雨季
地点：放学后的教室

## 诗歌插页 P034：原创现代短诗

那时我不懂边界，
只懂把沉默写满。
值日表上被雨溅湿的名字还在，
像一枚小小的证词。
它不替谁说爱，
只证明我们曾经年轻。

如果把镜头从陆允川身上移开，雨天值日对大多数值日生来说是件很烦的事。雨天意味着地上水多、伞多、泥多——拖地要拖三遍而不是两遍，垃圾桶里有泡烂的纸巾粘在底部不容易拎出来，黑板上的粉笔灰被潮气浸湿了以后擦起来糊了一片。空气里到处都是潮湿的布味——是从拖把池那块永远不干的抹布散发出来的，混着洗衣粉和水的铁锈味。星期三值日组里有五个人，其中三个都巴不得快点做完走人——程砚倒垃圾的时候脚下一滑差点摔进雨水坑，他边打滑边骂了声"这雨能不能下完了"，手里的黑色垃圾袋甩了半圈，差点泼出来，旁边的劳动委员赶紧扶了他一把，说"你小心点，这袋子底下破了个洞"。

对夏知遥来说，值日是例行任务。她在家里也做家务——她妈要求她每周扫一遍自己房间，所以她扫地的动作比一般高一女生利落得多。她不太在意别人在拖地的时候有没有看她——因为她不在意任何人在值日时的眼光。她扫完一排回头看有没有遗落垃圾时，刚好看见陆允川在第三排把拖把反转过来往墙角怼灰——那个动作很仔细，蹲下去用手把落在墙角的粉笔头捡起来扔进垃圾袋。她心里闪了一句"这个人做事挺认真"，然后继续扫她的地。这个念头持续了大概三秒，然后信息覆盖——化学作业压过来了。

程砚看见夏知遥推拖把杆的那零点三秒。他没说话，但他在宿舍里之后用这件事分析陆允川："她的手不是碰了你——她是碰了你的拖把杆。你觉得这有什么不同？""不同在于那不是物理接触。""对，拖把是你们的中间介质。就像那时候你传作业本隔着一摞书挨着她的本子——你是通过物体和她接触。你们之间的物理方程永远多一个项——一个物体项。"

叶澄那天来串教室拿物理卷子——文科班的物化会考在即，她来借陈老师的复习资料。天在下雨，她站在一楼门洞下面等伞，旁边一个理科班的女生也同样在等。她俩聊了几句雨——那女生说"这雨下得跟上个月一模一样"，叶澄说"今年雨季好像比去年长"。门洞下面的穿堂风吹得她的刘海贴在额头上，她撩了一下头发，低头看了眼自己的鞋——帆布鞋的鞋头已经湿了一小块，颜色深了一个色号。然后她抬头看见陆允川正从楼梯口拿着伞往下走——他的表情里有种不恰当的紧张，像握住伞柄的力气大了点。她没有去提醒他——她只是默默地想："他是不是又带了两把伞。"——叶澄分析这种心态时的说法是"备用焦虑"——陆允川会为所有可能发生的事做准备，从备用的透明胶带到备用的橡皮到备用的伞，好像多准备一点就可以让她少麻烦一点。这份焦虑本身不浪漫，但它是一种更底层的东西——责任感。

蒋雯那天也在教室里等雨。她刷着手机看雨什么时候能停——天气预报说还有一小时。她偶然抬头一眼瞧见陆允川和夏知遥分别站在门洞两侧，中间隔着大概一臂的距离，都拿着东西。她觉得那个画面有点莫名的好看——两个人都站着等雨，一个拿着拖把一把拿着扫帚，画面构图刚好对称。然后她低头继续看手机——因为雨还没停，化学作业还没交。
