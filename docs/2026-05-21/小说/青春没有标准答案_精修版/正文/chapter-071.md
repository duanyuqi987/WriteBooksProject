# 第071章 文理表发下来



时间：2009年三月，高一下学期开学后的第二周



地点：高一十班教室后排、办公室门口



卷次：第四卷：分科以前的夏天



2009年三月，高一下学期开学后的第二周，高一十班教室后排、办公室门口，一张淡蓝色的文理倾向摸底表把这段日子推到十班面前。陶建国把文理倾向摸底表发下来，十班第一次认真谈到分科。

段璟想知道常俊凡会选什么，却不敢让自己的问题显得像跟随；郑天宇则在理科和体育训练之间摇摆。这不是突然降下来的大事，而是从早读、课间、饭卡、家长电话和后墙红榜里一点点长出来的。

段璟坐在教室里，常常先看见常俊凡的背影，再意识到自己其实该看卷子。从这个学期开始，这种走神不再只是喜欢，也会影响一道题、一次排名和一个选择。

常俊凡没有变成谁的答案。她会犹豫，会烦，会被家里问得沉默，也会在别人替她下结论时把话说得很硬。

郑天宇也在这件事里显出自己的位置。跑得快，数学不差，常把选择说得像比赛，其实最怕被父亲一句话定下来。十班从来不是只由两个人组成，它是四十多个人一起挤出来的青春。

“这不是正式分科表，先写倾向，也先问问自己。”陶建国说。

“老师，倾向能不能写看情况？”郑天宇说。

“人生可以看情况，表格不行。”陶建国说。

“你想选理吗？”段璟说。

“我想先看清楚自己能走哪条路。”常俊凡说。

春天刚到，教室里还没有完全暖起来，摸底表传到后排时，纸边被很多手摸得发软。

有人在理科下面画勾，有人把笔悬在文科和理科之间，像那一格会忽然替自己说话。

常俊凡没有急着写。她把表格放在练习册上，先把姓名和班级填好，最后才看选择栏。

段璟看见她的笔停住，自己也停住。物理题里最难的是受力分析，这张纸上的力却看不见。

郑天宇把表格翻来覆去，说自己跑八百米都没这么累。周明澈让他别把表折坏，老师还要收。

办公室门口排着几个人，都是来问分科的。陶建国一边批作业一边回答，语气比平时慢。

窗外操场有篮球落地的声音，春天让人想往外跑，可教室里每个人都被一张纸按在座位上。

常俊凡最后没有问段璟选什么。她只把表格收进书里，说正式填以前还要再想一次。

陶建国这个学期管得比上学期更细。不是因为学生忽然变坏，而是因为分科摸底、月考和家长电话挤在一起，谁一松就会被推着往后退。

郑天宇被叫到名字时，先看了一眼周围同学，再看老师。这个小动作让段璟发现，班里很多看起来不在乎的人，其实都怕被公开放到台面上。

封丘县的校园生活没有电影里那么整齐。校门外有卖烤红薯的，有修自行车的，有家长骑电动车来送饭，也有学生在路边背单词。

黄河故道那边的风有时会吹进县城，吹动校服袖口，也吹得人想起更远的地方。可更远的地方暂时还只是试卷最后一道大题旁边的空白。

分科确认、甲流晨检和国庆黑板报还在后面，可文理倾向摸底已经先把他们往前推了一步。

宋乔会把女生那边的情绪说出来，付小川会用玩笑把男生那边的沉默撬开，周明澈则常常在最乱的时候把表格、名单和班费重新理一遍。

段璟有时羡慕这种确定。他自己更像一张写了太多草稿的纸，明明知道下一步该写什么，却还是忍不住看常俊凡会不会也停在同一个地方。

常俊凡并不总是回头。她更多时候只是把自己的事情做好：改题、接水、记账、扶凳子、把别人的话挡回去，然后继续低头写下一行字。

高宇把摸底表压在练习册下面，笔尖在选择栏上方停了很久，作业本被翻到下一页，纸声比说话还清楚。老师从门口经过，所有人的声音立刻低了半截。

于晓航低头算自己这次物理和历史的分差，算完又觉得不该只看分，旁边的铅笔滚到桌沿，被人伸手按住。这样的细节不会出现在成绩单上，却会留在很多年后的记忆里。

白子昂把父亲说过的话写在草稿纸边上，写完又用橡皮擦掉，桌洞里的试卷露出一角，上面还压着没改完的题。这个动作很小，却比一句大道理更能说明当时的慌。

乔鹿去办公室门口问了一句，出来时比进去前更沉默，校服袖口蹭过桌沿，留下一道浅浅的粉笔印。旁边的人没有追问，教室里的翻书声把尴尬遮住。

程予安把表格折了一下又展开，折痕正好压过文理两个字，操场上的哨声隔着窗户传进来，显得很远。常俊凡看见了，也没有立刻说什么，只把自己的题继续写完。

孟凡把摸底表压在练习册下面，笔尖在选择栏上方停了很久，窗外的树影晃了一下，像把话题轻轻推远。段璟后来想起这一幕，才知道青春里很多变化一开始都很安静。

夏清和低头算自己这次物理和历史的分差，算完又觉得不该只看分，有人把椅子往里收，木腿擦过地面发出短短一声。窗外风一吹，纸角轻轻动了一下，像有人把话又咽回去。

他也慢慢懂得，喜欢不能替常俊凡做决定，也不能把她的每一次沉默都解释成和自己有关。

常俊凡要走的路，首先属于她自己。段璟若真想靠近，就得先学会站在不挡路的位置。

这个学期就是这样往前走的：没有高一开学那样热闹，却已经开始把很多关系推到更真实的位置。

晚自习结束时，一张淡蓝色的文理倾向摸底表还留在教室里，像一枚小小的钉子，把这一天钉在十班的时间线上。

灯一盏一盏暗下去，楼道里传来脚步和水瓶碰撞声。段璟走到门口时回头看了一眼，发现常俊凡还在收书。

他没有催，也没有等得太明显。只是把书包带往肩上提了提，跟着人群往楼下走。
