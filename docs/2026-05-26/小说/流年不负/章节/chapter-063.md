# 第063章 月湖

时间：2013年6月8日晚上

地点：青阳县月湖公园

卷次：卷四：夏夜湖光

月湖在县城南边。

不算大。绕湖走一圈大概四十分钟。湖边种了一排柳树。柳条垂到水面。夏天晚上的风从湖面吹过来，带着水草的气味——有点腥，有点凉。

顾临风到的时候已经是傍晚七点半。

天色还没全暗。西边剩一抹橙红。湖水是灰蓝色的。远处湖心有个亭子，亭子里的灯是黄的。

他沿着湖边走。

聚会的地方在湖北岸的草地上。赵一鸣在那里铺了一块塑料布。上面放着几瓶饮料——可乐、雪碧、矿泉水。还有两包薯片和一袋瓜子。

人到了七八个。

都是理科三班的。赵一鸣叫的人。有几个已经在喝了——可乐倒进一次性杯子里，泡沫冒上来。一个叫陈伟的男生往可乐里扔了一颗薄荷糖——味道刺鼻——然后端起来喝了一口。

"你是不是傻。"旁边的人说。

陈伟说："好喝。"

大家笑了。

赵一鸣看见顾临风，站起来挥手。

"来啦。"

"嗯。"

顾临风在塑料布边上坐下来。腿盘起来。屁股底下是草——草有点扎。赵一鸣给他递了一瓶矿泉水。他接过来，拧开盖子。没喝。

"苏晚晴会来。"赵一鸣压低了声音。

顾临风的手顿了一下。

"你怎么叫的她。"

"我叫的许若琳。许若琳叫的她。"

"嗯。"

顾临风把矿泉水瓶在手里转了一圈。水瓶在手心滚了一下。

赵一鸣看了他一眼。没说话。只是往他旁边坐近了一点。

人越来越多。

刘浩带了把吉他来。坐在草地上随手拨了两下弦——声音在湖面上飘出去，像被水吸走了——不太响。他弹了一首《同桌的你》。弹了前奏就停了——"忘了后面怎么弹"——然后自己笑了。

三个女生坐在一块。其中一个是王敏。王敏声音很大——"我家里已经给我报了驾校了。暑假考驾照。"旁边的人问考驾照干嘛。她说："我哥说大学开学不会开车怎么谈恋爱。"

男的一块笑了。

赵一鸣大声说："我不考驾照。我要去打工。去广东。我表哥在那边厂里。"

"真的假的。"

"真的。挣点钱。大学开学前回来。"

顾临风看了赵一鸣一眼。赵一鸣之前没跟他提过这个。

赵一鸣注意到他的目光。笑了一下。"还没跟你说。今晚才说的。"

顾临风没说话。

塑料布上饮料瓶越来越多。空瓶子被风吹得滚了两下。一个雪碧瓶滚到了顾临风脚边。他用鞋尖把它拨回去了。有人把瓜子壳往湖里扔。瓜子壳在水面上浮着——白花花的一片。一个瓜子壳被风吹翻了——翻了个面。

刘浩又弹了一首《老男孩》。这次弹对了——和弦没忘。前奏弹了三遍才开唱。旁边的人笑他。"你到底唱不唱。"他没理。继续弹。唱了两句——嗓子哑了——"嗓子不行了今晚喊太狠了"——把吉他搁在膝盖上。不弹了。有人递了瓶矿泉水给他。他接过去——没拧开。放在了一边。

有个男生掏出一个翻盖手机——NOKIA的。按了半天键。对着月湖拍了一张照。闪光灯亮了一下。湖面被照亮了一小块——又暗了。"拍糊了。""晚上拍能有什么效果。"——没删。把手机合上了。

天完全黑了。

湖面变成黑色。水面上的波纹推着月光往前走——推到一半被柳条的影子截断了。远处县城的楼里有灯——白的、黄的。灯光在水面上拉了长长的影。有盏灯忽明忽灭的。大概接触不良。从远处看像一颗星星在眨眼睛。

湖边草丛里有虫子在叫。不是蝉——是蟋蟀。叫得很急。一只叫了，另一只就跟着叫。然后全草丛的蟋蟀都叫起来。过了几分钟又一起停了。安静得像有人按了暂停键。然后又开始叫。

有人点起了防风灯。灯放在草地中间。橘色的光把周围人的脸照得半明半暗。

苏晚晴来了。

她是从北岸那条小路上走过来的。小路两边是矮冬青——修剪过的。冬青叶子在灯光下泛着暗绿色。远处的县城方向有人在放烟花——不是放是试——一颗烟花飞上去。炸开。红色的。不大。飞到一半就灭了。可能是卖烟花的在试炮。

顾临风先看到的是她的影子——被防风灯的光拉长了，落在他边的草地边上。

然后他抬起头。

她穿了一条白色的短袖棉裙。头发放下来了，不是马尾。头发披在肩上。大概刚洗过——发梢还有点湿。风一吹，碎发贴在脸上。她用手指拨开了。光从防风灯打在她脸的侧面上——鼻梁的侧影，嘴唇的弧线。

他注意到她换了裙子。不是校服。在学校他看了三年校服。这是第一次看她穿裙子——白的。棉的。裙摆到膝盖。她走路的时候裙摆轻轻晃。

许若琳挽着她的胳膊。

"我们来啦。"许若琳说。

赵一鸣站起来。"坐坐坐。"

他让出塑料布上靠中间的位置。顺便踢了一脚旁边的空可乐瓶——瓶子滚到了草地外面。

苏晚晴坐下了。她先拉了拉裙摆，然后盘起腿。坐得很规矩。膝盖并在一起。脚踝交叠。脚上是一双白色的帆布鞋——鞋带是新的。大概第一次穿。

许若琳坐在她旁边。把她往中间挤了挤。

苏晚晴往周围看了一下。目光扫过顾临风。停了一下。

他点了一下头。

她也点了一下头。

大概就是一秒。

但这一秒里——他的手心出汗了。不是热。是汗。

顾临风把视线移开了。手里握着矿泉水瓶子。瓶子被他握得有点扁。塑料瓶壁陷进去了一块。他松了松手。瓶子弹回来——啪的一声。很轻。他怕她听见。把瓶子放在了草地上。

赵一鸣开始主持。

"大家举杯——"

有人举起可乐杯。有人举起矿泉水瓶。有人举起空瓶子。

"敬高考。敬我们。敬高中三年。"

杯子碰在一起，发出闷闷的响声——塑料碰塑料，不脆。

顾临风举起矿泉水和赵一鸣碰了一下。他的目光往左前方偏了一下——苏晚晴在喝一瓶橙汁。瓶盖拧了半天没拧开。

许若琳帮她拧开了。

她把橙汁放在膝盖上。没喝多少。

此后的一个小时，顾临风一直在原地坐着。

他听别人说话。刘浩弹吉他——后来弹了一整首《那些年》。大家都跟着唱。调子跑了，歌词也忘了两句——胡彦斌的不是胡彦斌的——有人说是胡彦斌的，有人说是胡的。没争明白。

苏晚晴也跟着轻轻地唱了几句。声音不大。但顾临风听到了。

她的声音很轻。像羽毛落在水上。

赵一鸣在旁边用胳膊肘捅了顾临风一下。

"去啊。"

"去干嘛。"

"别装了。"

"我没装。"

赵一鸣叹了口气。把可乐瓶往草地上一顿。"顾临风。高考都结束了。你还等什么。等你大学毕业？等她结婚？等到猴年马月。"

顾临风把矿泉水瓶在草地上放定。瓶子在草地上歪了一下——草太软，瓶子站不住。他又放了一次。还是歪。第三次他用力摁了一下——瓶底嵌进草泥里了。站住了。

"她旁边全是人。"

"那又怎么样。"

"不怎么样。"

赵一鸣不说话了。过了一会儿，他说："你今晚要是不跟她说一句话，你就真的废了。"

顾临风没回答。

远处的同学在唱歌。另一拨三四个在湖边用手机灯光照着捞东西——"癞蛤蟆！"——"那是癞蛤蟆吗？那是青蛙。"——"那它为什么叫得这么难听。"

湖风南来。

柳条飘起来又落下去。

顾临风握着瓶子。瓶底磕在膝盖上。一下。两下。三下。

他抬头看了一眼湖面。

月亮出来了。不是满月——缺了一块。月光洒在水面上。银白色。细碎的。风吹过去的时候光碎了又聚拢，聚拢了又碎。

他站起来。

赵一鸣抬头看他。

他往人群外走了两步。不是走——是挪了几步。

然后停下来。

又坐下来。

赵一鸣把脸埋进手心里。没说话。

防风灯的火苗被风吹得忽明忽暗。橘色的光在草地上晃来晃去。火苗歪向左边——又弹回来。防风灯的玻璃罩上沾了一只小飞虫。扑了两下翅膀。飞走了。

苏晚晴还在原来的位置坐着。喝了一口橙汁——瓶子离开了嘴唇，瓶口上留了一个浅浅的唇印。她拿纸巾擦了擦瓶口。然后抬头看月亮。

顾临风看到她的侧脸。月光打在她脸上。鼻尖有一小点光。下巴也是一小点。头发披在肩上——有几根落在锁骨的位置。锁骨在棉裙领口上方——很窄的两条线。

她把碎发别到耳后。手指顺着耳廓滑下来——指尖在耳垂上停了一下。然后放下了。

这个动作他看了三年。

从高一看到现在。在走廊上看过。在操场上看过。在食堂里瞥到过。每一次都一样——右手的食指和中指夹住碎发，往后拨，拨到耳后。然后指尖在耳垂上停一瞬。

三年。没变过。

这一次离得不远——隔着七八个同学、一块塑料布、和一阵湖风。

他低下头。

看着自己手里的矿泉水瓶。

瓶底上的字被他磨得看不清了。日期——大概去年春天的。他握了三年矿泉水瓶。课间喝水的时候如果她也在走廊——瓶底一定对着自己的脸。这样眼睛可以越过瓶底往她那边看。

远处的歌声还在继续。

"那些年错过的大雨，那些年错过的爱情——"

跑调的。刘浩在唱。旁边有人在笑。刘浩没管。继续唱。唱到"那些年"三个字的时候嗓子劈了——变成了气声。然后又接上了。

顾临风听着。

跑调的每一个字。他都听进去了。
