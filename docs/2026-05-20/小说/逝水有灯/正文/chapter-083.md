# 第083章 迟到的表白之后

所属部卷：第二部：六月以后
时间：六月末
地点：河边小卖部门口

## 诗歌插页 P083：原创仿宋小令

河灯起，晚风低，
一句谢谢隔东西。
问少年、几分轻放，几分难离。
汽水开时声很短，
不成回音也成题。

夏知遥走后，陆允川去小卖部买了一瓶汽水。

老板问要冰的还是常温的，他说冰的。瓶盖打开时，气泡冲上来，溢出一点，流到手背上。他没有立刻喝，只坐在门口的塑料凳上，看着河边的人来来往往。

手机响了，是程砚。

“说了？”程砚问。

“说了。”

“她怎么说？”

陆允川把汽水瓶放到地上：“她说谢谢，但给不了答案。”

电话那边安静了几秒。程砚没有骂，也没有开玩笑，只问：“你有没有逼她？”

“没有。”

“那就行。”程砚说，“疼就疼一会儿，别把疼变成账。”

这句话陆允川当时听着很难受。因为他确实有一瞬间想把三年都拿出来称一称，想证明自己这么久不该只换来一句谢谢。可他也知道，那样不对。喜欢不是押金，不能到期要求退还。

挂断电话后，他把汽水喝完。冰得太快，胃有点疼。他站起来往回走，路过护栏时没有再停。

回家后，他打开诗抄本，本来想写很多，最后只写了三句：

她没有错。
我也没有赢。
夏天很热。

这三句不像诗，却比他那些绕来绕去的句子更诚实。
