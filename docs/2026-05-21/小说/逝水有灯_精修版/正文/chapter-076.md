# 第076章 毕业照边缘之前

所属部卷：第二部：六月以后
时间：六月中旬
地点：学校操场主席台前

## 诗歌插页 P076：原创现代诗

我把水放到掌心，
让它照见一页没有说完的夏天。
有人从人群里走远，
也有人在原地学会松开。
照片边缘被风吹起的校服袖口没有替谁回答，
只把那一年轻轻照亮。

拍毕业照的通知是昨天晚上发到班群里的。班主任艾特了所有人：明天上午九点，操场主席台前集合，穿校服。

程砚在群里回了一句："校服都洗了三遍了，就等今天。"

班主任回了一个笑脸。

陆允川早上六点就醒了——不是紧张，是热。东屋的窗户对着东边，太阳一出来就直直地打在床上。他翻了两个身，索性起床。洗漱的时候从镜子里看了看自己：头发有点长，刘海快扎到眼睛了。要不要剪一下再拍？想了想，算了。毕业照以后翻出来看，头发长就是那时候的流行。

母亲在厨房煮粥，看到他起来，说今天怎么这么早。陆允川说今天拍毕业照。母亲哦了一声，多煮了一个鸡蛋。

他把校服从衣柜里拿出来。外套已经洗得有点褪色了——高一刚发的时候是深蓝色，现在变成了浅灰蓝。拉链修过一次，换了个银色拉链头，跟原来那个不一样。他穿上校服，拉链拉到胸口的位置。对着镜子看了看，又把拉链拉到头。

出门的时候，母亲在门口站着，手里拿着一把梳子。"头发梳一下。后脑勺翘了一块。"

陆允川低下头，母亲用梳子帮他把后脑勺按下去。梳子沾了点水，凉凉的。

"好了。去吧。"

操场上已经来了不少人了。主席台前面摆了四排蓝色塑料凳子，后面是站人的台阶。横幅已经从教学楼楼上拿下来了——高三六班，红底黄字，被风鼓起来一角，有人在用透明胶把它贴回墙上。

摄影师是个光头的中年人，穿着摄影马甲，正蹲在三脚架后面调相机。他让先到的同学站上去试光，然后就开始用喇叭安排队形。

"高个子往后站。你——对，你，站到最后一排。矮个子往前坐。女生坐前面两排凳子。"

程砚站在队伍外面，手里拎着一瓶没开的矿泉水，正在跟周长海聊天。"你说毕业照有什么意义？三十年以后拿出来看，谁还认得出谁？"

"你能认出现在。"陆允川走到他旁边。

"三十年以后呢？"

"三十年以后你拿出来看，能认出你自己。"

"那倒是。"程砚把瓶盖拧开喝了一口，"我这张脸还是比较有辨识度的。"

"对，脸最大。"

"你骂我。"

两人还没说完，摄影师就用喇叭喊了："最后一排的同学往左边靠一靠，不要超出镜头边缘。"程砚被安排到最后一排中间偏左的位置——他个子高。陆允川站到了最后一排的最右边。

"你站那么边上？"程砚歪着头从人群里喊他。

"安排就是这样。"

"那你等会儿别眨眼。边上容易虚。"

快门按下之前，有人从主席台侧面的器材室跑过来——是两个迟到的女生。叶澄从队列里探出头朝她们招手："这边！凳子给你们留着。"两个女生气喘吁吁地挤进前排坐下，其中一个的校服扣子扣错了位置，叶澄帮她把扣子重新扣好。

夏知遥从主席台另一侧走过来，她旁边是林曼。林曼在跟她说家里的猫昨天离家出走了，今天早上又在厨房窗台上发现它在偷鱼吃。夏知遥被逗得笑起来，那个笑还没落下去，摄影师就开始喊"各就各位"。

陆允川站在最右边，脚尖踩着地上那条白线。校服袖口被风吹起来，扫过手腕。他想把袖子按住，手动了动又放弃了。
