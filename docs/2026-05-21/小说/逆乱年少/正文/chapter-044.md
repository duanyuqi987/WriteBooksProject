# 第044章 周三晚上

时间：2011年五月上旬

地点：第二教学楼

卷次：第二卷：花谢的时候

发现苏安晴每周三晚上在二教自习，纯属偶然。

那天晚上林玉起去二教找一个高中同学——付小川，物理系的，说要借一本线性代数的辅导书。林玉起不用学线性代数，但付小川说中文系也要学逻辑，学逻辑不如学数学，不由分说把书塞给了他。

他抱着书从三楼下来，经过二楼的时候，从走廊尽头的窗户里看见了苏安晴。

207教室，靠窗第三排。她低着头，面前摊着一本很厚的教材，右手握着笔，左手压着一本笔记。桌上的台灯调到了最低的亮度，刚好照亮桌面那一小块地方。她的耳机线从衣领里伸出来，不知道在听什么。

林玉起停在走廊上。他不敢停太久——被人看见的话，解释不了为什么对着一个教室发呆。但他忍不住多看了一秒。她写字的时候嘴唇微微动着，好像在默念什么。这个细节他看到过一次就不会忘。

他下楼，出教学楼，走到文华路上。夜风凉凉的，路灯把银杏树的影子打在地上。他走了一半，又折回去。

不是回去看苏安晴。是回去确认一件别的事。

他走到二教大厅的布告栏前。布告栏上贴着各学院的课表——教育学院，周三晚上，在207教室上"教育心理学"。晚上六点半到八点半。

也就是说，苏安晴不是在自习，是刚下课后还留在教室里继续看书。

他把付小川那本线性代数塞进书包里。书包还是乱七八糟的。他拉上拉链，走出二教大门。

第二天早上，他查了自己的课表。周三下午他有中国古代文学史，四点下课。然后——他原本的安排是去操场跑步。

从下周开始，他在自己的日历上写了一个新的安排：周三晚上，图书馆自习。

他没有写"二教"——那是故意不写的。就好像只要不写，这个改动就不算数。

周三来得很快。四点五分下课，林玉起回宿舍洗了把脸，换了件干净的T恤。郭向阳在打游戏，头也不回地问："跑步？"

"去图书馆。今天不跑。"

"你上周不是还说跑步，这周就换图书馆了？"

"换换。天热了。"

郭向阳没再问。但林玉起出门的时候，听见郭向阳在后面小声说了一句："换得倒是挺勤快。"

他没有回头。

图书馆四楼，靠文学区的窗户可以看见二教的灯。207教室的灯八点半以后还亮着。林玉起把书翻开，一个字也没看进去。他在等那盏灯灭掉。

八点四十分，207的灯灭了。他合上书，收拾书包。从图书馆走回宿舍，要穿过文华路。走过银杏树的时候，他站了几秒。去年秋天他把叶子夹在书里，以为只是捡了一片叶子。现在每一棵银杏树都绿了，而那片叶子在书里睡了一整个冬天，等着被一个人翻开。
