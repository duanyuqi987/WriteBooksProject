# 第031章 雨天值日之前

所属部卷：第一部：初秋三楼
时间：高二开端
地点：三楼走廊和水房

## 诗歌插页 P031：原创现代诗

我把早晨放低，
让一线水从课桌边慢慢经过。
你的名字没有被我喊出，
只在走廊尽头那把滴水的蓝伞旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写雨天值日抵达之前的坡度。那场雨来得很急，最后一节课还没下，窗玻璃已经被雨点敲得发白。值日表贴在门后，纸角被潮气卷起来。

陆允川拎着拖把去水房，鞋底一路带水。他看见夏知遥站在走廊边望雨，书包抱在胸前，像在计算雨小下来的可能。他没有把这些动作想成什么预兆，只是一个刚学着适应县城节奏的学生，尽量让自己看上去不慌。

那天真正的事还没有发生，走廊尽头那把滴水的蓝伞却先被他记住了。它太小，小到放在任何人的日子里都不显眼，偏偏在陆允川这里有了重量。

“你自己的伞呢？”程砚说。“放走廊了。”陆允川说。“你人也差不多快放走廊了。”程砚说。

程砚嘴上损他，还是把自己的伞往他肩上一塞，说别装大方装到感冒，明天跑操你替我请假？

叶澄倒完污水回来，看见那把蓝伞靠在窗台下，伞尖还在滴水。她没有问是谁的，只把伞往里挪了挪，免得被人踩到。

课间十分钟被铃声切得很碎。有人挤去小卖部买面包，有人趴在桌上补觉，有人拿着水杯一路跑向水房。陆允川站在这些声音中间，心里那点紧张没有消失，却被生活稀释了一些。

夏知遥这时还只是人群里的一个名字。夏知遥问同桌有没有多余的伞，同桌摇头，说我妈今天也来晚。她便笑了一下，说那就等一等。陆允川看见了，却没有资格把这种看见说得太重。

他后来在日记本里写到这里，先写了很多夸张的句子，又一行行划掉。留下来的，是教室里真实的光、真实的灰、真实的不知道该把手放在哪里。

水房的水龙头关不严，一滴一滴落进铁桶里。拖把挤出的水带着粉笔灰，顺着地砖缝往低处流。陆允川握着拖把杆，手心被木刺扎了一下，疼得很具体。

如果说喜欢在这里有了开端，也不是因为命运突然发声，而是因为一个少年开始过分留意普通事物。他还不懂，留意可以是温柔，也可能慢慢变成对自己的消耗。

雨水从走廊窗缝飘进来，靠窗那排地砖很快湿了一片。陆允川拖地拖到一半，拖把越拖越脏，只好拎着桶去水房重新涮。桶里的水晃出来，溅到裤脚上。

夏知遥站在窗边等雨小，肩上背着书包，手里还拿着没做完的英语卷子。她同桌说要不一起冲，她摇摇头，说卷子湿了明天交不了。

陆允川把蓝伞放到走廊尽头时，故意没有回头。他怕一回头，就会把这件事做得太像表演。程砚看见了，叹了口气，说你这人连做好事都偷偷摸摸。

后来那把伞被谁拿走，他没有问。第二天早读，伞出现在教室后门旁边，伞柄上挂着一张便利贴大小的纸，上面只写了两个字：谢谢。陆允川看了一眼，立刻把目光移开。

这一章要把脚步放慢，慢到能听见课桌被拖动时的短促摩擦声。事情还没到来，可人物已经在日常里摆好了位置：谁靠窗，谁靠门，谁总替别人递一张纸。

陆允川后来才知道，所谓开始并不是一个明确的按钮，而是很多个微小选择叠在一起。他没有说话、没有靠近、没有确认，可他已经在心里给那个人留出了一块地方。

事情还没有真正发生时，陆允川已经在心里排练了好几种反应。可排练没有用，现实总会把人推到更笨的位置，让他先拿错拖把，再说错一句无关紧要的话。

那天放学，他没有立刻回宿舍，而是在三楼走廊和水房外多站了一会儿。风把雨鞋印吹得轻轻晃动，他忽然意识到，自己等待的并不是答案，只是一个能让明天继续期待的理由。

如果只写疼，这一章会很薄；加上拖把、水桶、窗台、雨鞋印，它才重新有了重量。那重量不是压人，而是让读者相信他们确实在这里生活过。

陆允川把拖把拿在手里，先把拖把涮干净，才敢让目光短短离开纸面。这个动作比任何独白都诚实：他并没有多成熟，只是在学着把心事放到事情后面。

程砚看他慢吞吞的样子，压低声音说：“你先把值日做完，别又把一分钟想成一整年。”陆允川想反驳，手却已经动了起来。

夏知遥那边也有自己的节奏。她抱着书包在走廊边等雨小，随后被同伴叫了一声，便把注意力收回到眼前的事上。她不是谁的谜面，只是一个正在过自己一天的人。

叶澄经过时没有评价，只提醒他们别耽误下一节课。她的语气很平，像把一杯水放到桌上，不烫人，却能让人清醒。

三楼走廊和水房里最明显的不是心跳，而是水滴落进铁桶的一下一下。陆允川后来愿意保留这种声音，因为它让那一年从情绪里长出地面。

临近吃饭的时候，食堂窗口冒出的白雾把人从心事里拽出来。程砚说再大的事也得先吃饭，陆允川端着餐盘，没有再把沉默装得很深。

他不是忽然变得懂事，只是在一件件小事里被迫往前挪。拖把、铃声、作业、饭卡，全都不懂他的难过，却全都要求他继续生活。

事情还没到来，蓝伞已经先进入他的视线。他当时不懂这种提前的留意有多危险，只觉得只要自己足够安静，就不会打扰任何人。

放学后他在三楼走廊和水房外多站了一会儿，听见楼下有人喊集合。那一声很普通，却把他从想象里拉出来，让他记起自己还要回宿舍、洗杯子、准备明天的早读。

这一段的温度在雨水和拖把之间。善意可以很轻，轻到只是一把伞；也正因为轻，才不该被他写成索取。

如果只写疼，这章会很薄；加上拖把、水滴落进铁桶的一下一下和食堂窗口冒出的白雾，它才像一段真正生活过的时间。疼仍在，只是不再独占整页纸。

程砚又催了一句：“走吧，别在三楼走廊和水房发呆。”陆允川把拖把收好，跟着人群往外走，心里那点没说出的东西被脚步声慢慢压低。

本章最后，事情仍悬在半空。陆允川收好书，跟着人群往外走。楼道里风从窗缝挤进来，把他的校服袖口吹起一点，他下意识按住，像按住一句还没成形的话。
