# 第001章 开学日

时间：2010年9月1日

地点：青阳县一中高一三班教室

卷次：卷一：初见秋风

九月一号。青阳县一中。

校门口挂了一条红横幅，上面写着"热烈欢迎2010级新生"。布是新的，红得扎眼。家长和学生从横幅下面挤过去，扛着行李、拎着暖壶。

顾临风是一个人来的。

他爸把他送到镇上汽车站，塞了两百块钱。"省着花。"他爸说。然后就转身走了。顾临风背着书包上了车，靠窗坐下。车窗外的树一棵一棵往后退。

到县城是上午九点。车站门口有个卖烤红薯的老头，推车上挂了个锈铁皮烟囱。烟囱口冒着白气。红薯的甜味混在柴油味里。他买了一小个，五毛钱。剥皮的时候烫了手指，换了两回手才吃完。

他问了两次路，找到一中大门。路上经过了一条河，河面上漂了一层绿藻，两头鸭子在绿藻里刨食，屁股朝天。

高一三班在教学楼三楼。走廊很窄，两边是教室门。门上贴着班级号——高一（一）、高一（二）、高一（三）。顾临风在第三个门前停了一下，然后走了进去。

教室不大。四十来张课桌排成七八列。已经坐了二十多人。有人在聊天，有人在擦桌子，有人趴在桌上。日光灯亮着，天花板上有两个吊扇在转。

他选了倒数第三排靠过道的位子坐下。

桌面上有上届留下的圆珠笔印子。他用手抹了抹，没抹掉。

书包是灰色的，拉链有点卡。他拔了两下才拉开。里面一本语文书、一本数学书、一个铁皮铅笔盒。铅笔盒是他姐用过的，面上印着小虎队的贴纸，边角磨白了。

他把书拿出来摞在桌角。语文书封面上有上一届学生的名字——张磊。用蓝水笔写在右上角。他拿橡皮擦了擦。没擦干净。

又过了一会儿。

一个女生从前门走了进来。

浅蓝色T恤。深蓝色牛仔裤。扎马尾。

九月的阳光从窗户斜打进来，刚好落在她侧脸上。她的头发在光里是褐色的。她往教室里扫了一眼，然后走向第二排靠窗的位置。

顾临风的手放在铅笔盒上，没动。

她坐下。把书包挂到椅背上。书包拉链上有个小东西晃了一下——看不清是什么。然后她拿出一本书翻开了。

教室里闹哄哄的。有人在笑，有人在拍桌子。风扇嗡嗡响。

班主任姓王，四十多岁，男的，戴眼镜。他走进来敲了两下讲桌。安静了一些。

"我姓王。你们的班主任。"

他念名单。

"顾临风。"

"到。"

"苏晚晴。"

"到。"

声音从第二排靠窗的位置传来。不响。刚好能听见。

顾临风在"苏晚晴"三个字下面用指甲划了一道印。

下午发了课本。十几本。语文、数学、英语、物理、化学、生物、政治、历史、地理。本子好几摞。他把书垒在课桌左边，本子放右边。

晚自习前回了一趟宿舍。六人间，上下铺。他的床位靠门，下铺。被子从家里带来的，蓝格子，洗得发白。枕套上有他妈补的一块补丁。

室友陆陆续续来齐了。对面下铺的是个戴眼镜的胖男生，抱了一箱方便面放床底下。他自我介绍叫孙磊——"孙悟空"的"孙"，"三个石头"的"磊"。上铺那个叫刘洋，瘦瘦的，话不多，进门打了个招呼就躺上去看手机了。另外两个人住靠近窗口的两张床。

宿管老师姓周，五十来岁。他挨个敲门——"电灯十点关，水房十点停水。"重复了三遍。门牌上贴了值日表，蓝色圆珠笔写的。星期一是孙磊，星期二是刘洋，星期三排到顾临风。星期四和星期五是那两个人——他还不知道人家的名字。

晚自习九点四十下课。回到宿舍，灯十点整灭。

窗帘没拉严实。月光漏进来一条。

顾临风没睡着。

他从书包侧兜里摸出一支笔。翻开语文课本的封底内页。在右下角写了两个字——苏晚晴。

写完看了几秒钟。然后拿笔在名字上划了两道横线。

纸破了。

他把书合上，放到枕头底下。

走廊里有人在唱歌。唱的是《星空》。声音越来越远。然后没了。
