# 第092章 夏天的聚餐

所属部卷：第二部：六月以后
时间：升学宴前后
地点：县城小饭馆

## 诗歌插页 P092：原创仿唐绝句

风色微明照旧尘，
少年别后识前因。
一声铃尽人潮散，
半页风光不问春。

菜上到第四道的时候，话题从大学转到了高中。

"你们还记不记得高二那次运动会？"程砚夹了一块回锅肉，筷子上全是红油——肉片切得很薄，卷着蒜苗和豆瓣酱，"周长海跑接力赛最后一棒，眼看就要第一了，结果他跑错道了。"

"我没跑错！隔壁班的抢了我内道！"周长海用筷子敲了一下碗边，"我到现在都记得隔壁班那个男生的脸。理平头的那个，叫什么来着——"

"刘阳。"叶澄说。

"对！就他！"周长海激动得差点把茶杯碰倒，"他后来高考考了三百多分，去当兵了。"

"你怎么还记着人家的高考分数？"林曼问。

"不是我记。是成绩单贴在校门口的时候我正好看到了。当时我就觉得——老天是公平的。"

"你记仇记了两年？"杜雨薇筷子停在半空。

"不是记仇。是那场比赛我真的能拿第一。我跑最后一棒的时候跟第二名的差距至少三个身位。结果他抢了我的内道，我被迫往外绕，绕完以后就被三个人超了。"周长海把肉塞进嘴里，嚼了两下，"体委哭了一整节课。"

"体委哭是因为摔伤了膝盖，不是因为成绩。"叶澄纠正，她用勺子舀了一碗酸菜粉丝汤，汤里的花椒被她小心地拨到了碗边。

"你记得也太清楚了。"

"我是运动会记录员。所有成绩都要记。那天下午四乘一百米接力，高二三班第三名，高二五班第四名。体委的膝盖是他在起跑的时候自己摔的——跟任何人的道次都没关系。"

全桌人安静了一秒。然后祁野开口："所以你刚才纠正了一个你记了整整两年的事实。"

"对。因为我耳朵听不了错误信息。"叶澄喝了一口汤，面不改色。

圆桌中间的玻璃转盘被人转了一圈，菜盘子从这头转到那头。回锅肉、水煮鱼、麻婆豆腐、炒青菜、一盆酸菜粉丝汤——汤面上浮着几颗花椒，祁野舀汤的时候没注意，喝到了麻椒籽，表情瞬间变了形。程砚哈哈大笑："你中招了！"祁野灌了两口茶水才缓过来。

陆允川坐在程砚旁边，默默吃菜。桌上聊到了夏知遥。是林曼先提起来的——她说夏知遥今天不能来是因为她妈那边的亲戚从深圳过来了，说要给她庆功。她爸也从深圳打了电话回来，说了很多话。

"她爸很久没联系她了？"叶澄问。

"差不多两年。中间就过年打过一次电话。但这次考上大学，他突然很热心。说让她研究生考到深圳去，深圳机会多。"林曼放下筷子，手指在桌布上无意识地划了一道，"知遥说他还记不住她具体读的是什么专业。说是建筑设计，他说成'建筑工人'。纠正了三次，每次电话里都还是说错。后来知遥就不纠正了。"

一桌人安静了一瞬。转盘上那道麻婆豆腐的油在灯光下泛着红光，没人去动。

"她爸真这么说？"程砚皱起眉头，他那双筷子往桌上一拍。

"真的。知遥跟我说的时候在笑——那种笑你们见过吗？嘴角是翘的，但眼睛不弯。她说'我爸问我毕业以后能不能帮他盖房子'——他以为学建筑就是学盖房子。也不是不对，但他问的方式让人很难受。"林曼的声音放低了，"我听着一点都不好笑。"

陆允川放在筷子上的手指动了一下。然后他夹了一块鱼，低头挑刺。鱼是草鱼，刺多而细，他低着头一根一根往外拨。拨出来的刺整整齐齐排在自己碟子边上，像在给什么东西列清单。

"她不在乎她爸怎么说。"杜雨薇忽然开口，声音不大但很坚定——今天她说话比平时少，但每次开口都踩在点子上，"她跟我说过——他想说什么就说什么，但她的路她自己走。她说这话的时候正在画一张图，铅笔断了，她削了继续画。削铅笔的时候手特别稳。""你怎么知道？"林曼问。

"因为也问过她类似的问题。我说你爸这样——她说你不用替我难过。我爸十年不在我身边，我的生活也正常过下来了。说明他不在不是问题。他突然回来才是问题。但问题是他的，不是我的。"杜雨薇把筷子放下，"她说这些话的时候语气很平，像是在陈述一个定理。不是在抱怨。"

叶澄喝了一口水，没有接话。但她的表情里有一种无声的认可。杜雨薇今天穿了一条碎花裙子，说话的时候筷子还举在半空中。

"你怎么知道？"林曼问。

"因为我也问过她类似的问题。她说她爸对她的了解，停在她十岁那年。但问题是她现在已经十八岁了。十年之间的空白，不是打几个电话就能填回来的。"杜雨薇把筷子放下，"她说这些话的时候语气很平，像是在陈述事实，不是在抱怨。"

叶澄喝了一口水，没有接话。但她的表情里有一种无声的认可——她知道杜雨薇说的是真的。那次在楼梯口她帮夏知遥拉书包拉链时，看到的那行字——"想考去一个有江的城市"——或许不是为了浪漫，而是为了远离一些需要远离的东西。

程砚举起茶杯。"那为她说一句——前程似锦。"

全桌人都举起了杯子。玻璃转盘停了。酸菜粉丝汤里的热气慢慢地升起来，把所有人的脸都罩了一层薄薄的白雾。

陆允川碰了杯，喝了茶。然后他放下杯子，把筷子拿起来，继续挑鱼刺。这次挑得很仔细——一根一根，全部挑干净。然后他把那块鱼肉放进程砚碗里。"你刚才一直说想吃鱼。"

程砚低头看了看碗里的鱼肉。"你把刺都挑了？"

"嗯。"

"靠。你这样我都不好意思损你了。"

"那你别损。"

"不行。我忍不住。你这人就是太好了——对谁都好。你要是坏一点，我还能多骂你两句。"程砚把鱼肉塞进嘴里，含含糊糊地说，"好吃。你自己也吃。"

陆允川夹了一块鱼放到自己碗里。这次没有挑刺，直接吃了。
