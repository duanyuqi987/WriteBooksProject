# 第094章 有人追她

时间：2014年6月初

地点：支教社团聚餐

卷次：卷五：各自南北

六月初。支教社团出发前最后一次聚餐。在学校后门的一家东北菜馆。老板是个东北人，菜的分量很大。锅包肉一盘摞得像小山——金黄的外壳裹着肉片，上面淋了糖醋汁。地三鲜的油汪了一层——茄子和土豆在油光里发亮。桌子坐了三桌人——十来个人。男生喝了四扎啤酒。啤酒装在塑料扎壶里——黄色的液体上面浮着一层白沫。

店里的空调坏了。老板搬了两台落地扇对着吹。扇叶呜呜地转。有人脱了外套搭在椅背上。

顾临风坐在角落。啤酒杯在手里转了很久——转了大概十几圈，没怎么喝。玻璃杯上凝了一层水珠。许若琳坐他对面，正夹着一块锅包肉。咬了一口——嘎嘣脆。她把筷子放在碗上，端起啤酒喝了一口。

旁边一个女生——第二分队的，姓陈，跟许若琳很熟——随口问了一句。她的声音不大，但是店里的天花板低，声音聚拢。"若琳，你们宿舍那个——那个叫苏什么的——是不是有人在追她。"

顾临风手里的筷子停了一下。没有抬头。筷子夹着一颗花生米——停在盘子和嘴之间的半空中。

许若琳一边吃肉一边说。"你说晚晴？"

"对对对。苏晚晴。"

"嗯。有。师大的一个男生——大二的，叫什么来着——姓周。追了快一个学期了。"

"成了吗。"

"不知道。她没怎么提。"

"长得帅不帅。"

"我没见过。"许若琳把肉吞下去了。用拇指擦了擦嘴角的酱汁。"不过听说条件挺好的。学体育的。打篮球的。个子应该挺高的。"

旁边女生"哦"了一声。然后从桌上拿起一串烤羊肉——吃了一口。开始聊别的话题——关于支教点有没有蛇——"山里蛇肯定有"——关于能不能洗澡——"哪有地方给你洗澡"——关于要不要带花露水——"六神的好用"。

顾临风还是低着头。手里的筷子夹了一颗花生米。夹了很久没有夹起来。花生米掉在了盘子外面——在白色塑料桌布上滚了一小段。他把它捡起来放在骨碟里。骨碟里有几根鸡骨头和一团用过的纸巾。他又夹了一颗花生米——这次夹住了。放进嘴里。嚼。花生米是盐炒的——有点咸。

他继续坐着。喝了半口啤酒。啤酒是温的。泡沫已经消掉了。液面平静得像一杯隔夜茶。他把酒杯放下。杯底在桌布上印了一个湿圆圈。又坐了一会儿。有人让老板加了一份拔丝地瓜。老板说"好嘞——"。

然后他去了一趟洗手间。

洗手间的灯是日光灯。白色的。很亮。墙上的瓷砖是白色的。缝里有些黄色的污渍。洗手池是不锈钢的。水龙头有点漏水——一滴一滴地打在洗手池的陶瓷面上，声音在封闭的空间里被放大了。滴——嗒。滴——嗒。墙角的排气扇在转——嗡嗡响。

他洗了手。用的凉水。洗手的动作很慢——先冲湿、按洗手液、搓手背、搓指缝、冲干净。手指在凉水里面冲了很久——冲到手背发红了。然后抬头看镜子里的自己。镜子里的人脸色没什么变化。跟来吃饭时一样。但嘴闭得很紧。下嘴唇比平时多用了些力。

他关掉水龙头。抽了两张擦手纸。纸是褐色的再生纸——粗糙的，上面有碎屑。擦干手以后纸粘在手心里——有五处湿掉了。他把纸丢进垃圾桶。垃圾桶里已经满了——纸巾堆成了小山。

走出洗手间。

回到桌上。许若琳正在讲支教时候要带什么——"充电宝一定要带，那边不一定有插座。还有手电筒。还有湿巾——因为可能没水洗脸。""有人问"山区有没有信号"，另一个人说"移动的有信号，联通的没有"。

"临风——你手机是什么卡。"许若琳忽然问他。她的眼睛在灯光下面有点亮——大概是喝了半杯啤酒。

"移动。"

"那没事。有信号。"

"嗯。"

他又坐了一会儿。看着桌上的锅包肉、地三鲜、花生米、拍黄瓜、拔丝地瓜。拔丝地瓜的糖浆拉出了很长的丝——有人站起来夹，丝不断，旁边的人帮忙用筷子剪断了。每个人都在说话。笑声。碰杯声。老板在厨房里喊了一嗓子——"锅包肉两份——"——然后传出一阵炒锅翻菜的声音。

但他什么都听不太清楚。那些声音像隔了一层水。模模糊糊的。

十点多散席。大家往外走。有人在门口互相加QQ——"你支教回来把照片发我"。有人在找自己的伞。许若琳她们往女生宿舍方向走——几个人边走边聊天，笑声在夜风里飘过来。顾临风一个人往男生宿舍走。他的影子在路灯下时短时长——每经过一盏路灯，影子就从脚底拉出去一段。

回到512。阿坤在打副本——今晚大概是团战，他戴着耳机，键盘啪啪啪地响。电脑屏幕上的技能特效红红绿绿地闪着。李博在床上看书——床头的台灯亮着，书页在他手里翻过一页。小伍在写信——信封又换了，这次是熊猫的。他的钢笔尖在纸面上走得很快——沙沙沙的。

顾临风脱了鞋。鞋带解了两次——一次没解开，鞋带结打得太死了。他蹲下来用手指慢慢解。鞋带松开以后他把鞋放在床底下。换了睡衣。躺在床上。把手臂枕在头下面。枕头里的棉花有点结块了——按下去一块凸起来一块。天花板上的裂缝还在。他看了很久——大概两个小时。中间翻了一回身。李博把床头灯关了。阿坤的键盘声也停了。黑暗里面只剩下小伍偶尔翻身的声音和窗外偶尔开过的摩托车引擎声。

他没睡着。

有人追苏晚晴。姓周。大二。学体育的。打篮球的。追了快一个学期了。

快一个学期。

也就是说——从他开始上大一下学期的那一天起，那个人就已经在追她了。他在操场走圈的时候，那个人可能在师大的操场上打着篮球。他在图书馆写诗的时候，那个人可能站在师大的教室走廊里等她下课。他查301路公交路线的时候，那个人可能骑着电动车从师大的南门骑到北门。那个人可以站在她面前说话。可以看到她笑。可以帮她拿东西。可以在她膝盖摔破的时候递创可贴。

他没有生气的资格。没有难过的资格。是他说遵守约定的。是他一次都没有联系。

他翻了个身。把被子蒙住头。被子里闷着一股洗衣粉的香味。他的呼吸很轻——被子里面很暗，什么都看不到。他闭上眼睛。脑子里是她被追的样子——模糊的。没有脸的。师大的梧桐树下，两个人并肩走。女生穿白色外套。马尾。男生个子高——比她高一个头。看不清脸。看不清表情。只有两个轮廓在绿色的梧桐叶子下面移动。

他不知道自己什么时候睡着的。

第二天早上。闹钟响了——手机铃声是默认的那个，嘀嘀嘀的。他按掉闹钟。手指在屏幕上滑了两下才按准。洗脸。凉水冲在脸上——眼睛有点涩。刷牙。牙龈出了一点血——泡沫里有很淡的红色。去食堂——包子和豆浆。包子的馅是韭菜鸡蛋——咬了一口。去教室——坐第三排靠窗。窗外又有新生在军训——喊口号的声音嗡嗡地传进来。听课。记笔记。笔记上写了"现代汉语——动词的体"。写完以后在页码旁边画了一个圈。跟每天一样。什么都没说。
