# 逆乱年少

## 项目底座

### 基本信息
- 书名：逆乱年少
- 类型：校园青春小说
- 字数目标：20万字（约200章）
- 时代背景：2010-2014年
- 主要场景：大学校园（江城大学）

### 核心主题
- 暗恋：从大一到大四，一场漫长的、克制的暗恋
- 成长：大学四年中，从少年到成人的蜕变
- 诗歌：唐诗宋词与现代诗歌贯穿全文，是情感表达的载体
- 现实：生活费、课表、车票、电话卡——青春被具体的事物压着

### 风格定位
- 第三人称有限视角（以男主林玉起为主视角）
- 情感克制：不直接说"喜欢"，用动作、细节、场景传达
- 短段落、场景切换、感官细节丰富
- 诗歌插入：每章开头或关键处插入原创诗词
- 核心句式："不是A，只是B"
- 拒绝戏剧化冲突，日常中见真情

### 主要角色
- 林玉起：男主，来自封丘县城的普通家庭，中文系，敏感、内向、爱写诗
- 苏安晴：女主，来自同县城的文科女生，教育学专业，干净、独立、清醒
- 郭向阳：室友，计算机系，话多、讲义气，最早发现林玉起秘密的人
- 付小川：高中同学兼大学校友，物理系，嘴贫但心细
- 宋乔：苏安晴室友，新闻系，消息灵通，性格开朗
- 何南星：同班同学，安静、爱喝茶，林玉起的另一个倾诉对象

### 技法偏好
- 白描：用最少的笔墨勾勒人物和场景
- 不写之写：最重要的情感不直接写出
- 草蛇灰线：诗歌作为伏笔，前後呼应
- 春秋笔法：在平淡叙述中含褒贬
- 一击两鸣：一个细节同时写两个人物
