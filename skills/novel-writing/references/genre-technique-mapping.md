# 创作取向与技法匹配

本表用于在规划和起草时选择技法。它不是分类牢笼：同一部小说可以混合多种取向。选择技法时先看本章功能，再看题材标签。

## 基础原则

- 技法服务故事，不服务炫技。
- 每章聚焦 1-3 个主要技法即可。
- 文学向不等于慢，网文向不等于粗糙。
- 没有明确风格时，默认使用 `styles/通用中文小说.md`。

## 按创作取向匹配

| 创作取向 | 核心需求 | 推荐技法 | 慎用 |
|---|---|---|---|
| 现实/文学 | 真实、余味、人物复杂度 | 白描、不写之写、春秋笔法 | 过度反转、金句堆砌 |
| 悬疑/推理 | 信息控制、线索、公平反转 | 草蛇灰线、一击两鸣、不写之写 | 线索重复强调、解释过早 |
| 言情/关系 | 情绪递进、潜台词、关系变化 | 暗透法、反衬法、繁章波折法 | 直接替角色讲心情 |
| 历史/时代 | 时代质感、命运感、价值判断 | 春秋笔法、白描、层峦迭翠 | 现代口吻、资料堆砌 |
| 幻想/奇幻/仙侠 | 世界规则、成长、奇观 | 草蛇灰线、层峦迭翠、白描 | 设定说明过长 |
| 都市/职场/家庭 | 日常真实、关系压力、选择代价 | 白描、一击两鸣、反衬法 | 说教、标签化人物 |
| 怪诞/恐怖 | 违和感、未知、认知不安 | 不写之写、草蛇灰线、暗透法 | 直接解释恐怖全貌 |
| 网文/连载 | 追读动力、阶段回报、节奏稳定 | 繁章波折法、反衬法、一击两鸣 | 套路重复、无铺垫爽点 |

## 按章节功能匹配

| 章节功能 | 推荐技法 | 目标 |
|---|---|---|
| 开篇建场 | 白描、一击两鸣 | 同时建立人物、处境和问题 |
| 人物出场 | 白描、反衬法 | 让人物用行动被记住 |
| 关系变化 | 暗透法、反衬法 | 写出未说出口的东西 |
| 冲突升级 | 繁章波折法、一击两鸣 | 让选择和代价递进 |
| 伏笔埋入 | 草蛇灰线、白描 | 线索自然混入场景 |
| 伏笔兑现 | 草蛇灰线、层峦迭翠 | 让读者回想起前文 |
| 情绪余波 | 不写之写、白描 | 留出读者感受空间 |
| 主题显影 | 春秋笔法、层峦迭翠 | 让主题从事件中浮出 |
| 过渡压缩 | 避繁文法、一击两鸣 | 跳过无效重复 |
| 高潮段落 | 繁章波折法、白描 | 让行动、情绪和信息同时爆发 |

## 风格文件建议

| 用户描述 | 推荐风格文件 |
|---|---|
| 未指定文风、只要自然中文小说 | `通用中文小说.md` |
| 克制、现实、乡土、战争、历史、少解释 | `冷白描.md` |
| 系统、升级、打脸、强连载回报 | `系统爽文.md` |
| 怪谈、悬疑、规则、诡异、认知不安 | `怪诞悬疑.md` |

## 选择算法

1. 先判断本章功能。
2. 再判断作品取向。
3. 选择 1 个主技法和 0-2 个辅技法。
4. 如果技法会破坏作品气质，以作品气质为准。
5. 审查时只检查实际使用的技法，不要求每章覆盖技法表。
