# 第165章 彩礼
时间：2018年10月
地点：青阳县
卷次：卷九：执手之约

提亲之后第二个周末。两家人又坐到了一起。这次不在酒店——在苏家。理由是苏母说"没必要每次都花钱在外面吃，在家做点家常的就很好。"

顾母带了一只老母鸡。活的。装在蛇皮袋里。到了苏家楼下，母鸡在袋子里扑腾了两下——顾母拍了拍袋子："别动，马上就到了。"

苏母接过母鸡，拎到阳台上拴了。她说"太客气了"，但是笑得眼睛眯成一条缝。她说这只鸡刚好——明天苏晚晴她姥姥要来，正愁没什么好菜。

顾父带了两个瓶子。一瓶白酒——跟上次酒店里喝的一样，洋河大曲。一瓶黄酒——说给不喝白酒的人备着。

茶还是苏父泡的。龙井。紫砂壶。三只杯子。动作跟上次一模一样——烫杯、放茶叶、冲水、倒掉、再冲水。

顾临风坐在老位置上。沙发太软，陷下去还是那截。但这次他坐得比上次松弛了一些——后背没挺那么直。

两家人围坐在茶几周围。苏父泡茶，苏母端出瓜子花生和切成小块的苹果。顾母把带的东西往茶几下面推了推——她说"别占地方。"

苏晚晴坐在沙发最左边。她今天没怎么说话。手里拿着一个橘子在剥——橘子皮撕成一片一片的，放在茶几上的小碟子里。剥得很慢。

茶几上摊开了几样东西。男方家带来的一张三合板的红纸——上面写着聘礼单。顾母的字歪歪扭扭的，但每个字都写得很认真。

苏母看了一眼红纸。又看了一眼苏父。苏父在泡茶，没看。

"老苏——"苏母叫他。

"嗯。"

"你看看。"

苏父把茶杯放在茶盘上。拿起红纸。眼镜在电视机旁边的柜子上，苏晚晴帮他拿过来。他戴上眼镜，看了大概十秒钟。

然后把红纸放下。

"写这些做什么。"

顾母愣了一下。"这是规矩——"

"规矩是做给别人看的。"苏父端起茶杯。"孩子们过得好就行。不讲究那些。"

顾父在旁边坐着。他本来话就少，这种场合话更少。但听完苏父的话，他点了一下头。

"老苏说得对。但该有的还是要有的。"顾母坚持。

苏母也说话了。"老苏——人家是尊重我们。你这是什么态度。"

苏父没理她。端起茶杯喝茶。

苏晚晴在旁边剥橘子。橘子剥完了。她把橘子掰成两半，一半放在茶几上，一半递给顾临风。顾临风接过去的时候碰到她的手指。

"姐姐那边是多少。"苏母忽然问。

"我姐姐前年结的。八万八。"顾母说。

"那也是正常。青阳这边现在都差不多——"

"但是我们家不讲究跟别人比。"顾母赶紧说。"临风刚工作两年，攒了些钱，但不多。主要是以后买房子——楚州的房子也不便宜。"

"房子不急。"苏母说。"两个人都有公积金。慢慢攒。"

"对。我也是这么想。先租房也行——"

"租房好。"苏母接过去。"我们家苏晚晴不会做家务。让她先学两年。"

"临风会做。他洗碗洗了好多年了。"

两个妈妈一下子聊开了——从彩礼聊到房子，从房子聊到谁做饭，从谁做饭聊到以后谁带孩子。声音越来越大，把苏父泡茶的水声都盖住了。

苏晚晴和顾临风坐在中间。听两个妈妈把他们的生活安排得明明白白。苏晚晴又把橘子掰开——这次递给了她妈半瓣。她妈接过去，边吃边说"这孩子什么事都自己扛，以后得有人帮她扛——"

苏父放下茶杯。杯子落在茶盘上，"嗒"的一声。

客厅安静了。

"够了。"他说。

他看着顾母。又看着顾父。

"我说过了。孩子过得好就行。不讲究那些。"

然后他停了一下。

"但你们既然问。"

他从口袋里掏出一张纸。折了两折的，打开。上面写了几个数字——字很小，很工整。是他自己在家算的。

"我这些年攒了一些。不多。五万块钱。给苏晚晴做嫁妆。你们给多少彩礼，不够的我自己补。"

苏母愣住了。她显然不知道苏父还准备了这个。

苏晚晴也愣住了。橘子瓣从她手里滑下来，掉在茶几上。她没捡。

顾母的眼眶又红了。这次没拿纸巾——眼泪直接流下来了。一滴落在茶几上。她用手背擦掉。

"老苏——"

"别叫我老苏。叫苏建国。"

"苏建国——"顾母擦了眼睛。"你这——你这——我们真的不好意思——"

"没什么不好意思的。"苏父说。"她是我女儿。"

五个字。

苏晚晴站起来。去了阳台。

阳台上的文竹在风里微微摇动。那盆吊兰——从高中到现在换了三次盆的那盆——长得很绿。她把手放在文竹的叶子上。叶子很软，像羽毛。

后面传来脚步声。

顾临风站在阳台门口。

"你怎么也来了。"

"你爸把茶壶拿了。不用我倒茶。"

她没回头。手指在文竹叶子上轻轻地动。阳台外面能看到远处那棵老槐树的顶——树叶黄了很多。十月底的风吹过来，黄色的叶子往下落。

"他说我是他女儿。"苏晚晴说。

"我听到了。"

"他以前从来没说过这种话。"

顾临风没说话。站在她旁边。

"我小时候。他从来没说过'我爱你'。也从来没说过'你是我女儿'这种话。"苏晚晴的声音很轻。"他就是每天早上给我做早饭。白粥、馒头、一个鸡蛋。鸡蛋是煎的，煎两面——他怕不熟。天天如此。从小学一年级到高三。"

顾临风看着她。她的侧脸在傍晚的光里。

"我一直不确定他是不是认可我。"她说。

"现在呢。"

她把文竹叶子上的一个小黄斑摘下来。放在花盆的泥土上。

"现在知道了。"

屋内传来苏母的笑声。苏父又在倒茶了。他把顾临风那杯倒得很满——又到了那个差一点就要溢出来的分寸。

顾母在茶几边上看苏父倒茶。她说"老苏——建国——你这倒茶的手艺跟谁学的。"

"自己琢磨的。"

"你倒得好。不倒出去一丝。"

苏父"嗯"了一声。

苏晚晴和顾临风从阳台走回来。苏晚晴坐回沙发最左边。顾临风坐回他原来的位置。沙发垫子上的凹陷又恢复了那个深浅的区别。

苏父端起他那杯茶，喝了一口。然后把茶壶推到桌子中间。

"你们定吧。"他说。"日子。酒店。请多少桌。你们商量。我不懂。"

然后他站起来。拍了拍裤子。走到走廊里面的房间去了。

房间里传出他打开收音机的声音。调频。沙沙沙。然后停在一个台——在播黄梅戏。一个女声在唱《天仙配》里的"夫妻双双把家还"。

苏母看着他的背影。摇了摇头。

"他这个人就这样。心里什么都想好了。嘴上就是不说。"

顾母笑着。

"不说也行。倒茶的时候说了。"

苏晚晴低头。看着茶几上她爸推过来的那把紫砂壶。壶身上有光。是这么多年茶叶水淋出来的光。

她把手覆上去。壶还温着。

顾临风也把手覆上去。覆在她的手背上。

苏母和顾母在对面看到了。两个妈妈对视一眼。谁都没说话。但都在笑。苏母笑得嘴角往上弯——弯得幅度比以前都大。顾母笑得眼眶又红了——这次没掉泪。拿手背按了按眼角。按完了把手放下去——继续看。

阳台上拴着的母鸡"咕咕"叫了两声。窗外有鸽子飞过去。翅膀扑棱的声音从三楼传出去很远。鸽子飞到对面的屋顶上——收起翅膀。在屋脊上走了几步。雪白的羽毛在夕阳里变成浅橘色。
