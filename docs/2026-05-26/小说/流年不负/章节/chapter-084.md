# 第084章 遵守约定

时间：2013年9月至10月

地点：楚州师院

卷次：卷五：各自南北

大学生活在九月下旬进入了轨道。

每天早上六点半起床。手机闹钟——嘀嘀嘀。按掉。洗脸——凉水泼在脸上，把眼睛里的睡意洗掉。刷牙——高露洁，薄荷味，泡沫在嘴里凉凉的。去食堂。包子和豆浆——每天都一样。包子是韭菜鸡蛋的。豆浆是淡的——糖放得很少。卖包子的阿姨认识他了——他第一次去的时候阿姨问他"什么馅的"，他说"韭菜鸡蛋"。第二天再去的时候阿姨看到他就拿起装韭菜鸡蛋的蒸屉，问"几个"。"两个。""豆浆也来一杯。""好的"——塑料袋装着包子，纸杯装着豆浆。七点四十到教室。坐第三排靠窗的位置。那排桌椅的桌面上有一道圆珠笔划的印子——大概是谁在上面写过字，被管理员擦掉了，但痕迹没擦干净。听课。记笔记。圆珠笔蓝色的油墨在纸上一行一行地留下字迹。下课。食堂。下午的课。下课。食堂。晚自习——图书馆三楼靠墙第四个座位。绿色台灯。纸上写满字。十点回宿舍。十一点睡觉——熄灯。躺在黑暗里。天花板上的裂缝在路灯光里若隐若现。

日子是重复的。每天早上的闹钟声、包子的味道、豆浆的温度、第三排靠窗的座位、图书馆的绿色台灯、回宿舍路上经过的电话亭——都差不多。一天一天之间唯一的区别是笔记本的页码在增加——昨天翻到第18页，今天翻到了第22页。

他让自己忙起来。

周一的课排到下午五点四十。周二晚上有书法选修课——毛笔字，从颜体练起。周三晚上中国古代文学——讲《诗经》和《楚辞》。周四晚上西方哲学导论——讲柏拉图。周五晚上历史地理——讲黄河改道。周六上午两节书法。

他的课表比宿舍里任何一个人都满。

小伍有一回看了一眼他的课表——那张A4纸被折了两折，但是上面的字还是很清楚，密密麻麻的——"临风——你是不想让自己闲着吗。"

"只是想多学点。"

"学这么多不累吗。"

"不累。"

李博在宿舍复习近代史纲要。李博复习的时候有一个习惯——用左手捏着耳朵。他说这样能提高注意力——不知道从哪里看来的。他翻了翻顾临风的笔记本——几本都写了半满。字是行书，写得很整齐。每一页都分了区——主要知识点在左半边，扩展和补充在右半边。有些地方在旁边用红笔加了批注——"此处可与诗经对比""参考读史记"。

"你这笔记记得比上课讲的还多。"

"怕忘了。"

"忘不了。考试又不考这些。"

顾临风没答。他把笔记本合上，放进书包里。书包拉链在涩的那个角度又卡了一下。

十月的一个周二。书法课。教书法的是个退休返聘的老先生——姓钱。头发全白了——那种很干净的白色。但手很稳——拎笔的时候都不需要支撑，手悬在空中，手指捏着笔杆，定定的，一丝都不晃。他在教室里走了一圈，一个一个地看。走到顾临风背后的时候脚步停下了。站了几秒钟。顾临风能感觉到他在看自己的肩膀和手腕。"腕力不够。"声音不高——像在自言自语。顾临风把腕往下压了半寸。再写一横——这一次横的起笔和收笔之间的力量更稳了。钱先生说"好"。然后走到下一个学生旁边。

下课以后他还在练。教室里的人走光了。最后走的人把灯关掉了一排——只剩他这一排的日光灯还亮着。毛笔在纸上走——横、竖、撇、捺。笔锋在宣纸上转折的时候有一种轻微的滑动——笔毛分开又合拢。墨汁的味道散开来——飘在空空的教室里，混着旧桌椅的木蜡味。窗外有风吹进来——从半开的窗户缝里挤进来，吹得宣纸的边角一掀一掀的，像一个活的东西在呼吸。他用镇纸压住——镇纸是铜的，压上去一声闷响。继续写。写了大概二十几个字，手开始有点酸——从手腕一路酸到小臂。把手甩了甩——手腕的关节咔嗒响了一下。又写了十几个字，酸到肩膀了。把毛笔搁在笔架上——笔架是陶制的，中间凹下去，刚好可以放笔。揉了揉右前臂。看了看写的字——多是多，像也不像。"永"字那个捺的角度还是差了一点。把写得不满意的几张揉成纸团——扔进角落的废纸篓里。纸团在篓底弹了一下。

十月初。天气开始转凉。楚州的秋天比青阳干。昨晚的阵雨让路面上有些小水坑。他走路的时候要绕开——水坑里映着灰蒙蒙的天。路上的梧桐叶子卷着边，有风的时候在地上打转——三两片、五六片，沙沙的干涩声音。他每次从教学楼走到食堂的路上，会踩到脆的叶子——咔嚓一声。落叶在脚下碎成褐黄色的粉末。

晚饭以后他偶尔去操场走圈。

操场的跑道是塑胶的——师院操场跟一中不一样。一中的跑道是煤渣铺的，跑完鞋底是黑的。师院的是红色的塑胶——踩上去有轻微的回弹，像踩在一块硬一点的橡皮上。八条跑道。白色的分界线画得很直，但有些地方已经磨淡了。中间是足球场——草是假的，绿色的塑胶草皮，踩上去有点硬，脚感跟鞋子有关。操场边上有十几盏大灯——铁杆的，高高的，每个灯柱上装了大概十来颗灯泡。晚上九点半准时打开——先是几声电流的嗡嗡，然后灯泡一颗一颗亮起来。灯光是白色的——把跑道照得跟他脚下的路一样清楚。跑道边上有一排香樟树——树影在灯光下面拖得很长。

他开始走路。不是跑步。是走。步子不快——大概一秒一步多一点，两手插在裤兜里。

从第一道开始——最内圈。走完一圈回到起点。然后往外走到第二道——跨过两条白线。再走一圈。走到第三道。再一圈。

走到第五道的时候他停了一下。脚步顿住了——一只脚踩在白色的分界线上，另一只还落在塑胶跑道上。

第五道。高中那年在操场走圈也是第五道——那次是他以为她有男朋友那次。那次他决定高考完就表白——走了一个多小时走到脚底起泡。那次之后他就一直在走第五道。每次跨过第四道和第五道之间的那条白线——都像是在跨过一道隐形的时间边界，从"现在"跨回"高三那年"。一样的红色跑道。一样的夜空。一样的一个人。但操场的尺寸变大了——从中等到了大。周边的人变了——从青阳一中穿校服的同学变成了师院穿T恤和运动服的大学生。但那些都不是关键。关键是——在这条跑道上走着的这个人，心里那个方向一直没变。

现在也在走第五道。他低下头看了看脚下的白线——五号道的分界线在灯光下是灰白色的。他继续走。左脚、右脚、左脚、右脚。

操场里有跑步的人超过他——有男生有女生。一个穿黑色背心的男生跑得很快——脚步声啪啪啪的，从他身边掠过去，带起一小阵风。有人戴着耳机跑——耳朵里的音乐声透过耳机壳漏出来，是周杰伦的《晴天》。有人两个人一起跑——女的跑在男的前面几步，男的追在后面喊"慢点"。有人手牵手在散步——走的是一道和二道的分界线。足球场上有人在踢球——喊声和哨声混在一起。"传——""打——""好——"——足球打在塑料草皮上的声音是沙沙的，打在铁门框上是一声震响。

他继续走。到了第三圈。呼吸还是均匀的。

第五道一圈四百多米。他走三圈。走完第三圈的时候脚底有点热——隔着鞋底能感觉到塑胶跑道被他的脚步磨出了温度。他停下来。站了几秒钟。呼吸在鼻子里进出——有点急，但很快恢复了正常。然后走出操场。

操场门口的饮水机。不锈钢的台面——上面有水渍干了以后留下的白圈。三个水龙头——一个坏了，用透明胶带缠着。最左边的龙头上贴着一张纸条——"出水量大，小心溅湿"。他弯腰在没坏的龙头上接了一口水。水是热的——因为这个饮水机接的是开水房的水管。含着一股铁锈味——大概学校的管道年久了。他喝了两口，把剩下的倒掉了。水洒在地上——在地上摊开一片深色的湿痕，然后就干了。

回宿舍的路上经过一个电话亭。师院的电话亭是黄色的——铁皮做的，像一个倒扣的碗。里面亮着日光灯。电话挂在机身上——银色的听筒，听筒线是黑色的螺旋线。电话亭的玻璃是透明的——但内面上蒙了一层灰。

他停下来。手插在裤兜里。裤兜里是手机。手机里存着一条路线——301路，11站，12公里。手机里还存着一个号码——排在S那一栏。那个号码从存进去以后就从来没有被拨过。存在通讯录里——像一个沉默的坐标，证明某个方向还没变。

十月晚上有点冷。他能看到自己呼出来的白气。很薄——从嘴里呼出来以后在半空里停了一下，然后就散了。手在裤兜里攥了攥。

他没有走进电话亭。继续往前走。

李博在宿舍看《新闻联播》。电视是他从家里带来的——一台十四寸的旧电视，外壳是灰色的，顶上那根天线用锡纸包了一截。画面有点糊——雪花点在主持人的脸上跳跃。女主播的声音很标准——"今天是10月11日星期五——"。阿坤在打副本——今晚好像是什么团本，键盘敲得特别响。他的手指在键盘上跳——脸上的表情在屏幕蓝光的映照下很专注，嘴唇被屏幕光照得发紫。小伍在写信——这次的信纸是带花边的。他说是他妈寄来的新信纸。信纸是浅粉色的，边角有印上去的玫瑰图案。

"临风——你跑步去了？"阿坤头也没抬。屏幕上他的魔兽角色正在被怪物追——他在按技能键。

"走圈。"

"'走圈'——你这种说法老年人用的。"阿坤终于打完了那波怪——他松了口气，把后背靠到椅背上。

"老年人也挺好。"

顾临风坐在床边。脱掉运动鞋。鞋底沾了塑胶跑道的红色颗粒。他用手指拨掉了两颗——颗粒掉在地上，弹了一下，滚到了鞋盒的方向。

阿坤的游戏打完了。他把耳机摘下来——挂在脖子上——"咱们宿舍也该集体活动一下。老是各干各的。我看别人宿舍都一起吃饭喝酒——我们宿舍太老干部了。"

李博合上书。用书签夹住那一页。"你想干什么。"

"出去吃一顿。师院后门新开了一家烤鱼。听说是活鱼现杀的。"

"贵不贵。"

"AA。"

小伍从上铺探头说"好"。李博说"周五晚上"。顾临风说"行"。

周五晚上四个人去了那家烤鱼店。从师院后门那条小巷子穿出去——巷子窄，两个人并肩走就有点挤。巷子两边是居民楼——一楼的窗户亮着灯，能看见里面有人在看电视。烤鱼店在巷子尽头——门面不大，门口立着一块红字招牌——"万州烤鱼"——字是用霓虹灯管弯的，"烤"字的灯管有一截不亮了。店里摆了七八张方桌。墙上贴着红纸菜单——手写的，字有点歪。靠墙的鱼缸里养着十几条草鱼——在水里游得很慢，偶尔嘴唇碰到缸壁发出咚的一声。老板是个胖胖的中年人——系着围裙，围裙上全是油点子和辣椒渍。

烤鱼装在铁盘子里端上来——铁盘下面垫着木托盘，木托盘被烤得发黑了。鱼是草鱼，上面盖了一层辣椒和花椒——干辣椒是红油浸过的，花椒是一颗一颗的暗红色的小珠子。鱼身下面铺了豆皮、豆芽、藕片、金针菇——浸在红油汤里，冒着泡。阿坤用筷子拨开辣椒，露出鱼肉——白白的冒着热气。鱼肉上划了花刀——刀口整整齐齐。李博叫了四瓶啤酒。老板娘用开瓶器撬开瓶盖——瓶盖弹到地上。

小伍喝了一瓶脸就红了——红到脖子根，连耳朵都是红的。他吃了两口烤鱼——举起杯子又喝了一口。鱼有点辣——他喝了一口啤酒，辣劲没下去——又喝白开水。李博也喝了不少——两瓶下去脸不红气不喘，但话比平时多了。说了很多洛阳的事——"洛阳水席你们吃过吗——二十四道菜——"。阿坤喝了三瓶开始讲他的网恋故事——"那个女生打奶妈特别厉害——每次团战大家都喊她——她在游戏里有一个公会——大家一起打了三个副本——有一次她为了给我加血自己死了——"。李博说"打游戏认识的能算恋爱吗"。阿坤说"怎么不算——她给我加过一百多次血——你女朋友给你加过一百多次血吗"。李博说"我没女朋友"——阿坤说"那不就是了"。

大家都笑了。顾临风也笑了。笑得很轻——嘴角往上动了动。

吃完饭他们沿着学校外面的街走回去。晚上的街道很安静——偶尔有电动车开过去，车灯扫在路面上。路灯间隔不均匀——有的路段很亮，有的路段很暗。经过一段暗路的时候阿坤差点踩到路边一只猫——猫叫了一声跑了。经过师院南门的时候，顾临风看了一眼公交站牌。301路。站牌在路灯下面——绿色的铁牌子被橘色的路灯照成了灰绿色。没有人等车。公交站长椅空着。长椅的木板有两条裂了——中间那条凹下去一块。

阿坤他们走前面——三个人的影子在路灯下被拉得时短时长。"快点临风——"阿坤回头喊的时候脸在路灯下是发黄的。

"来了。"

他快步跟上。脚步在柏油路面上轻响。

回到宿舍。512的门推开——李博去开了灯——日光灯闪了两下才亮。洗漱。李博第一个——然后是顾临风——然后是阿坤——小伍最后一个。水龙头开了又关——水池里留下了牙膏沫和头发。熄灯。黑暗落下来。

窗外的风带进来一股食堂做饭的气味——炒洋葱、酱油、大锅饭的蒸汽。又到了晚饭的点了。但他已经习惯了这种味道——从九月到现在，每天都要闻到两三次。习惯到闻到这个味道就知道几点了。

他把鞋整齐地放在床底下。鞋带松开——白色的鞋带沾了一点土。躺在床上。拿出手机。屏幕亮了一下——蓝光在黑暗里投在他的脸上。没有什么新消息。收件箱是空的。QQ头像全黑着——好友都是离线状态。他把手机翻过去。屏幕光灭了。黑暗重新落下来。

闭上眼睛。

师大的操场——跑道是什么颜色的。大概也是红色的。塑胶的。八条跑道。中间是假草坪。她会不会也在走圈。她走路的速度应该比他还慢——她高一在走廊里走路的步子是那种匀速的，不紧不慢。她走第几道。大概第一道——她嫌外道太晒。或者不走的——她本来就不爱运动。

他想了一会儿。然后不想了。把脑子里那些画面清理干净——像在黑板上用板擦擦掉粉笔字。黑板干净了。但粉笔灰还在空气里飘着——落下来需要时间。

遵守约定——就是不联系。不找她。不打电话。不发短信。不去她的学校。不在QQ上点她的头像。这些事说起来简单——但具体到每一天里就不简单了。每一天都有三十四万八千秒。每一秒都可能是打破约定的那一秒。但每一秒他都没有打破。

他做到了。

一天一天地做。早上起来——站在镜子前——用冷水泼到脸上——看着自己眼睛里的红血丝——把手机拿出来——按亮屏幕——确认没有不由自主给她发了短信。然后刷牙。然后去食堂。然后上课。然后图书馆。然后操场。然后宿舍。然后熄灯。然后闭上眼睛。一天完成。明天再重复。

到今天为止——已经四十二天了。
