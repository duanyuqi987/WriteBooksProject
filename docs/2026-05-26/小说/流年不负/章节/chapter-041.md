# 第041章 她的一句话

时间：2011年10月

地点：教室走廊

卷次：卷三：此心藏久

十月中旬。下午第二节课下课。

走廊里挤满了人。有人在接水，有人在聊天，有男生在走廊尽头比谁跳得高——手掌拍天花板，拍到了就笑。拍不到的再跳一次。

顾临风从厕所往回走。手里攥着半截粉笔——物理课代表让他帮忙拿到办公室。他没去办公室。忘了。

走廊不宽。他从人群中穿过。侧身。低头。

然后他听到了一个声音。

"我觉得学习好的人很厉害。"

是她的声音。

他在人群里停了一步。不是站住，是没迈出去。

苏晚晴和许若琳站在教室门口。靠在门框上。她手里拿着一个水杯，那个透明的，能看到里面还剩半杯水。许若琳在跟她说班上谁谁谁的成绩。

"就是那种，不用费很大劲就能考很好的人。"许若琳说。

苏晚晴摇了摇头。

"不是。我说的是那种很努力的人。知道自己想要什么。"

她的声音不大。走廊很吵，这两句话在嘈杂里其实听不太清楚。但顾临风听清楚了。

他没有继续往前走。

转身回了厕所。

站在洗手台前。拧开水龙头。水哗哗地流。他接了一把凉水泼在脸上。十月的凉水。手撑在台面上。台面是白的。瓷砖缝是灰的。

他看着镜子里的自己。

十七岁。刘海有点长，快遮到眉毛了。眼睛不大。鼻梁上有点油。校服领子翻得不太整齐。

镜子里那个人看了他几秒。

然后他拧上水龙头。甩了甩手上的水。走出厕所。

走廊里的人少了一些。苏晚晴已经不在门口了。她坐回了自己的位置——他经过教室门口的时候往里面扫了一眼。她在看语文书。低着头。马尾搭在右肩上。她翻了一页。手指在纸面上捋了一下——把折了的那一页按平。

下午的太阳从西边窗户照进来。她的侧脸在光里。光从她的额角照下来——经过眉骨的弧度、鼻梁的侧线、下巴小小的尖。地上投了一小片她的影子。脖子那里影子最深——因为头低着。下巴快要碰到锁骨。

顾临风回到座位。把粉笔放到桌角。赵一鸣在抄英语作业，头也不抬。

"你脸上怎么是湿的。"

"洗了把脸。"

"这个天洗冷水脸。你是不是有病。"

他没回答。

翻开物理课本。第十六页。匀变速直线运动。公式写在黑板上，他看了一遍。然后又看了一遍。然后把公式抄在笔记本上。字很小。笔画很用力。

晚自习九点四十下课。

回到宿舍。赵一鸣在跟对面铺的人打闹。有个人在泡面，满屋子都是老坛酸菜的味道。上铺的在打电话。

十点整。灯灭了。

窗帘没拉严。月光进来一条。

顾临风从枕头底下摸出手电筒。很小的那种，塑料壳，五块钱。按键有点接触不良，按了两下才亮。

光打在被子内侧。一小片。白得刺眼。

他翻开物理练习册第四十六页。把被子拉到头顶，裹成一个帐篷。手电筒的光圈在纸面上晃了一下。

第一道题。匀加速直线运动。初速度零。加速度两米每二次方秒。求三秒后的位移。

他开始算。

被子里不透气。闷。热。塑料手电筒的外壳慢慢变烫。

他做完了三道题。

然后翻到数学。函数。定义域。值域。奇偶性。他把之前做错的两道题重新做了一遍。

手电筒的光开始泛黄——电池快没电了。他拍了拍手电筒，光又亮了一点。

外面有人起夜。拖鞋声从走廊那头拖到这头。厕所门开。关。

顾临风关了手电筒。等脚步声过去。等了大概三分钟。

重新打开。光更暗了。勉强能看到字。

他翻开英语书。单词表。背了八个单词。每个默写三遍。

手电筒彻底没电了。按了几下，不亮了。塑料壳上有一个小裂缝——不知道什么时候磕的。裂缝从上到下。光就是从那里漏掉的。白费了一个多月的电池。

他把手电筒塞回枕头底下。把练习册合上。被子里闷了很久，掀开的时候凉风灌进来。凉风从脚底一直窜到胸口。他打了个冷战。但又觉得凉快——被子里闷了太久。脸是烫的。脖子后面的汗凉了一下。

窗外的月光还在。

他翻了个身。上铺的板子响了一下。

他说了一句"学习好的人很厉害"——在脑子里把这句话又听了一遍。不是他听见的那个版本。是他想象中她说这句话的样子。马尾。安静的声音。不是在跟谁比。是在说一个事实。

他不知道自己能不能变成那样的人。

但他想试试。

闭上眼。

第二天早晨醒来。枕头又在地上。

手电筒压在枕头下面。他把枕头捡起来的时候顺便摸了一下手电筒——塑料壳还是热的。

早上六点半。去教室上早自习。路上买了两个馒头。边走边吃。

到教室的时候苏晚晴已经在了。她每天早上都到得很早。坐在第二排靠窗的位置，面前摆着英语书。她的嘴唇在动——在背单词。

他坐到自己座位上。把昨天没做完的物理题翻开。

继续做。

赵一鸣从后门进来，看了他一眼。又看了他桌上摊开的练习册。

"你最近打鸡血了？"

"没。"

"作业都没收你还在做。"

顾临风没说话。笔在纸上走。一个字一个字地写。

赵一鸣看了他几秒。然后坐下，把自己的作业也翻开了。

窗外银杏开始变黄了。十月中旬的银杏，叶子边缘先黄，中间还绿着。有一片叶子落下来，贴在窗户玻璃上。

苏晚晴伸手把窗户推开了一点。叶子被风吹走了。

她的马尾晃了一下。

顾临风在物理题的最后一行写下答案。数字很小。几乎看不见。

他把笔放下。手心里全是汗。
