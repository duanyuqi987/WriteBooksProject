# 第011章 理科名单之前

所属部卷：第一部：初秋三楼
时间：高一下学期
地点：新分班后的理科教室

## 诗歌插页 P011：原创现代诗

我把早晨放低，
让一线水从课桌边慢慢经过。
你的名字没有被我喊出，
只在公告栏边被胶带粘住的名单角旁边轻轻停过。
多年以后，我学会松手，
也学会把亮处还给生活。

这章仍写理科名单抵达之前的坡度。分班那天，公告栏前挤得像食堂窗口。旧的竞赛通知还没撕干净，只剩半张白纸粘在玻璃上，新名单贴上去时边角翘起，风从走廊尽头吹过来，纸角啪啪地响，像有人在反复翻同一页书。

陆允川站在人群外面。他先装作看旁边那张"化学竞赛报名通知"——其实高一根本没人报，通知在玻璃上已经贴了两个月，字迹褪成了淡蓝色。等人群松动了一些，他才往理科一班那张名单挪了两步。他没有把这些动作想成什么预兆，只是一个习惯在别人看不到的时候才敢暴露自己在乎什么的人。

那天真正的事还没有发生，名单角被胶带粘住的那一小片纸却先被他记住了。黄色的旧胶带已经失去黏性，名单角翘起来，露出下面还残留着痕迹的玻璃。他觉得那个翘角像被翻开又放下的书签，放的人走得太急，忘了夹回去。

"你报的理科还是文科？"旁边一个戴眼镜的男生问道，手里拿着自己的分班表，表格被他折成了豆腐块。"理科。"陆允川说。"我也是理科。你是哪个班的？""原来三班。""三班？那你们班主任是方老师？听说他管得很细。"陆允川点头。那个男生正要继续说，旁边有人叫他去办公室签字，他便把表格往书包里一塞跑走了，临走还回头说"以后可能就是同班了"。陆允川说了声"好"，心想对方大概率没听见。

公告栏前的人终于散了一些。他站到名单正面，从上往下一行行看。理科一班四十二个人，名字按姓氏拼音排列。他先看到自己的名字——陆，排在中间偏下的位置。然后开始找另一个名字。从A到Z，一个一个看，像在做一篇需要逐行比对的阅读理解。找到了——XI，夏。她也在理科一班。

他把这个发现装进脑子里，没有表露任何情绪。旁边有人在感叹自己被分到二班跟好朋友分开了，有人在大笑说"我们还在一个班"，有人抱怨分班规则莫名其妙。陆允川从这些声音中间默默退出来，走回原来的教室去拿剩下的课本。他走得比平时快了一点——不是因为高兴，而是因为他怕自己再多站一秒，就会有人看出他刚才在找什么。

原来的教室正在清空。桌椅被搬到走廊上，抽屉里的废纸和空笔芯散了一地。叶澄站在门边核对名单，她的新班是文科一班，从此以后和陆允川不在同一栋教学楼了。她看见陆允川走进来，喊住他说："理科一班在对面楼的二楼西侧。同桌名单在门后面的公告上，你去找自己的位置。以后作业收交了别记错时间——你们新班主任姓陈。"陆允川点了一下头，点完才想起这可能是他和叶澄之间最后几次面对面说话了。从那以后他们只能靠课间跑楼梯才能碰上一面。

程砚也选了理科，也是理科一班。他从走廊里冲进来，一把拍在陆允川肩上："我们还在一个班！""你轻一点。""我怕什么轻？以后物理卷子靠你了。""你的物理也挺好。""我的物理是选择题猜得好，大题是乱写。"程砚拿上书包，"走，去新教室占座位——你说我们是占后排还是占靠窗？""后排靠窗。"陆允川不假思索。程砚看了他一眼，笑了一下，那个笑里有种"我就知道"的意思。

新教室比原来的宽敞一些，但还没安顿好。讲台上堆着没拆包的粉笔箱，黑板没擦干净，上一届的数学作业答案还留着半行没擦掉的痕迹。陆允川在后排靠窗的角落坐下，把书包放在旁边给自己占了个位置。窗外是一棵更大的法桐树，树枝刚好碰得到二楼窗台，风一吹就轻轻刮擦玻璃，发出沙沙的摩擦声。他听了一会儿，觉得这个声音比三楼的更近。

新同学陆续走进来，有人拖桌子，有人问"这儿有人吗"，有人误把书包放在别人占好的座位上被提醒了。理科一班里有原来三班的人——程砚、他自己，还有其他几张认识但没说过话的脸。但他第一眼还是去确认了她的位置：前排偏左，靠近走廊，和两个女生坐在一起。

他坐下来，把笔袋里的笔重新排列了一遍，笔帽统一朝上。他知道自己又开始做那些让自己看上去不慌的小动作了。但他没有停下来，因为停下来他会想得更多——比如她刚才走进来时看到这个班有什么表情，比如她对这个新班级有什么期待，比如她和旁边那两个女生是本来就认识还是第一天坐在一起。
