# 第100章 树莓摊旁边



时间：2010年四月中旬，周末补课后



地点：城关镇街口、树莓摊旁边



卷次：第五卷：未来在黑板上倒计时



2010年四月中旬，周末补课后，城关镇街口、树莓摊旁边，余小满手里一小盒还带水珠的树莓把2010年的春夏推到十班面前。周末补课后，余小满在树莓摊旁边等母亲，听见家长们谈起补课和成绩。

余小满想买一盒树莓，却先数了数口袋里的零钱；常俊凡没有替她付钱，只陪她等了一会儿。这不是一句未来就能概括的事，而是从饭卡余额、补课收据、竞赛名单和家长饭桌里一点点长出来的。

分科以后，十班看起来还是十班，课桌、黑板、食堂窗口都没变，可每个人手里的压力换了重量。

段璟越来越清楚，喜欢常俊凡不能只靠看她一眼。她要面对的不是他的心事，而是自己的成绩、家庭和将来。

余小满也在这段日子里显出自己的位置。容易想家，靠清单学习，家里节省，情绪常被一件小东西牵动。群像不是名单，群像是每个人都有自己的难处。

“我妈说这周别乱花钱。”余小满说。

“那就先不买，闻一下也不收费。”常俊凡说。

“你这安慰有点实在。”段璟说。

“一中学生吧，刚下补课？”摊主说。

“周末补课结束就回家，别在路口逗太久。”陶建国说。

树莓摊摆在城关镇路口，红色小果子装在透明盒里，水珠还没干。

补课结束的学生从楼道里涌出来，书包重，汗也重，谁都先想买点凉的。

家长们站在摊边聊天，话题绕不开成绩、补课费和谁家孩子最近又进步了。

常俊凡站在树荫下，没有急着插话。她知道家长的话落到学生耳朵里，常常比老师批评还重。

段璟闻到树莓的酸甜味，忽然觉得县城夏天不只是热，也有很多说不清的压力藏在甜味后面。

摊主认得一中的校服，问他们是不是又补课。有人点头，有人装作没听见。

一小盒树莓很轻，拿在手里却像要先掂量家里给的钱够不够。

路口电动车一辆辆过去，卷起灰尘，把补课楼下的热闹慢慢带散。

陶建国这段时间讲得最多的不是口号，而是把眼前的事做具体。竞赛要名单，偏科要错题墙，补课要时间表，家里期待也要能说清楚。

余小满被推到这件事里时，先露出的不是优点，而是慌张。十班很多人都是这样，越想显得没事，越会在一个小动作上漏出真实。

封丘县的夏天来得直接。麦子熟过以后，路边热气一层一层往上冒，校服贴在背上，练习册也像被晒重了。

新华书店、补课班楼道、树莓摊、食堂充值处和教室后墙，慢慢成了他们这一阶段最常经过的地方。

上海世博新闻从电视里传出来，玉树捐款箱被放到讲台上，远方的大事一件件落进县城校园，最后都要和作业本放在同一张桌上。

常俊凡不再只是把自己守好。她会帮人分错题，会把起哄挡回去，也会在别人想替她安排未来时明确说不。

段璟看见这些以后，心里的喜欢少了一点自我感动，多了一点笨拙的克制。他还不会完全做好，但已经知道不能再只想着靠近。

十班也在变。有人去竞赛培训，有人补偏科，有人被家里催得喘不过气，有人第一次意识到努力是要花钱、花时间、花掉许多轻松的。

段璟站在树莓摊旁边数零钱，讲台粉笔盒里只剩半截白粉笔。十班的热闹还在，只是每个人都比去年更会藏心事。

常俊凡把树莓盒拿起来又放下，后墙的错题纸被风掀起一角。老师从门口经过，原本散开的声音立刻往回收。

郭向阳避开路边晒热的砖缝，竞赛名单的纸角已经微微卷起。这样的细节不会被写进红榜，却会留在很多人的夏天里。

秦佳禾闻到酸甜味时先低头笑了一下，新华书店塑料袋在手里沙沙响。这点动作很小，却把那天的压力落到了实处。

何南星站在树莓摊旁边数零钱，桌洞里的练习册露出红色封皮。旁边的人没有追问，翻书声把难堪遮住。

邓桥把树莓盒拿起来又放下，树莓摊旁边的纸盒被太阳晒软。常俊凡看见了，没有替谁下结论，只把题号重新写清楚。

余小满避开路边晒热的砖缝，黑板报边角的透明胶又翘起来。段璟后来才知道，未来最早就是这样从小事里露头的。

何南星闻到酸甜味时先低头笑了一下，家长的电动车铃声从校门外传来。窗外风吹了一下，纸角轻轻动了动，像有人把话收回去。

树莓摊旁边之后，十班没有立刻变成更成熟的班级。真正的成熟来得很慢，常常先表现为没人再随便笑别人的饭卡、座位、家长电话和落选名单。

县城的夏意从树莓摊旁边冒出来，甜味很近，压力也很近。段璟把这件事记得很清楚，因为它让未来从一句漂亮话变成了能拿在手里的东西。

他也开始明白，常俊凡不是他青春里的奖品。她有自己的路、自己的判断，也有不想被任何人替她说完的沉默。

如果他真的想和她一起走，就不能只问她去哪里，还要问自己有没有能力把脚下的路走稳。

2010年的春夏就这样压过来：世博新闻很远，补课班很近；未来很亮，眼前的题也仍然要一笔一笔写完。

晚自习结束时，余小满手里一小盒还带水珠的树莓还留在记忆里，和粉笔灰、热风、饭卡声一起混在这个夏天。

楼道灯亮起来，学生一拨一拨往宿舍和校门走。段璟背着书包，没有急着说话。

常俊凡走在前面，步子不快。两个人中间隔着一点距离，那距离不再只是暧昧，也像现实留给他们的空白。
