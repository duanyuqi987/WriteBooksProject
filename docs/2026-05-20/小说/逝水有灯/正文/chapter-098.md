# 第098章 省城南湖之后

所属部卷：第二部：六月以后
时间：大一九月
地点：大学南湖凉亭

## 诗歌插页 P098：原创仿宋小令

湖上晚风轻。
新舍灯明，旧梦未全醒。
一键消息归静处，
水边人影各分明。

那晚回宿舍，陆允川把鞋放到阳台，袜子搭在椅背上。祁野在下面给家里打电话，说饭菜还行，床也还行，就是澡堂排队太长。

轮到陆允川给母亲报平安时，他站在楼道尽头。母亲问他被子潮不潮，同学好不好相处，钱够不够。他一一说够，说好，说别担心。电话挂断后，楼道里的声控灯暗下去，他站着没动，等了一会儿才咳一声把灯唤亮。

宿舍桌上摊着校园地图。南湖的位置被他用手指压出一道折痕。旁边是新发的学生手册，第一页写着作息、请假、门禁和补办校园卡流程。新生活并不诗意，它先要求他记住澡堂开放时间和食堂窗口。

他打开高中群，消息已经堆到九十九加。有人分享大学社团招新，有人说想回县城吃面。他把群折叠起来，又把夏知遥的聊天框从置顶取消。没有删，也没有拉黑，只是不再放在最上面。

祁野回头看见，挑眉，“有进步。”

“你看见什么了？”

“看见你像个人了。”

陆允川拿起枕头砸过去，两个人笑了一阵。笑完他坐回桌前，在草稿纸上写下“南湖”两个字，旁边画了一个很小的方框。那不是忘记的开始，只是他终于给旧事换了一个不挡路的位置。
